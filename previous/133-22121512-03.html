Step

Action

2

Program Registration + Read more ...

Select Program Registration Tab

Program Registration + Read more ...

  • Select Add Program

Add New Bank Details for GPTP program in the Organisation Register

Table 3

Step

Action

1

Level of HPOS Organisation Register access + Read more ...

Ask the caller which level of HPOS Organisation Register access they have. (This question assumes user already has been attributed HPOS access in the first instance) and has relevant PRODA Access.

To Create New bank details for the Organisation Site Record the user will require

  • HPOS Org-Fin Admin with:
    • HPOS Org-Admin

The combination of access the user has been assigned will determine what they are able to action as follows:

  • HPOS Org Fin Admin + HPOS Org-Admin – Allows user to Create New Bank Details and update payment instructions
  • HPOS Org Fin Admin + HPOS Org-Site-Admin – Allows user to only select existing bank details and to add update payment instructions. This Access DOES not allow the user to CREATE NEW BANK Details

If the user is unsure what access they have and are not the PRODA Organisation Director or do not have Owner Access they:

  • can check their membership/attributes in PRODA, or
  • a Service Officer can check users access/attributes using PRODA DIGITAL ACCESS and where required Organisation Linking via Medicare Portal, following comprehensive security check

2

Log on to PRODA and HPOS + Read more ...

  • Log on to PRODA using user name and password
  • My linked Services will display
  • Go to HPOS tile and select ‘Go to Services’

If HPOS tile does not appear under linked services, see Program Registration, Step 7 in Table 1

Log on to HPOS on behalf of the Organisation:

  • Choose an organisation to act on behalf of from list displayed
  • Select Continue

3

Access the Organisation Register + Read more ...

Select Organisation register green tile on the HPOS main page

The Organisation Register-Organisation Record page will display

  • Select the Associated sites tab
  • Select the Organisation Site ID number for the enquiry

4

The Organisation Site Record + Read more ...

Organisation Site Record Displays.

Select Banking Details tab.

5

Banking details + Read more ...

Select Add on the right hand side under the Action column.

Add Banking Details page will display. User select:

  • Enter New Bank account, go to Step 9

6

Enter a new account + Read more ...

A pop up modal will open that enables the user to enter new bank account details:

  • Branch BSB
  • Account number
  • Account name

The user must complete the Acknowledgement by ticking the ‘I acknowledge that’ box.

Select Confirm and Submit.

The summary page will display. The user can scroll to the end of the page.

Select Save to finalise the process.

Program Registration and adding payment instructions for a Supervisor or Registrar

Table 4

Step

Action

1

Log on to PRODA + Read more ...

  • Log on to PRODA
    • Under My Linked Services the HPOS tile will display
    • Select Go to service

User logs onto their individual HPOS account:

  • Select No organisation – Proceed as an individual only
  • Select Continue

The individual HPOS home screen appears with the all of the users services. Select:

  • My details tile (dark purple)
  • My Personal details

2

My Banking Details + Read more ...

A new screen will appear.

  • Select My Banking details
    • The My banking details screen displays
  • Select GPTP on the left hand side under Program Column
  • Select Add from under the Action column

3

Add Banking details + Read more ...

The add banking details screen displays.

Program details will populate. Location and Location ID fields will not populate as GPTP banking details are added against provider stem and not the location.

  • Select Enter a new account complete Account Details, BSB, Account Number and Account Name,

The user must complete the Acknowledgement by ticking the ‘I acknowledge that’ box.

Select Submit and save update.

Note:

  • Reset will return to pre-entry details and user can recommence entering new details, or
  • Cancel will return user to the banking details main page

4

Home banking details + Read more ...

User will be returned to the home banking details page.

A message will display confirming action taken with banking details.

New bank account details will display for the Supervisor or Registrar on the following day.

Adding payment instructions using the existing bank account details for an individual

Table 5

Step

Action

1

Search organisation record + Read more ...

Log onto the Medicare Portal which will open the eBusiness Gateway page.

Use the dropdown for Source office codes. Service Officers should select their source office code. E.g. MLDD

Use the dropdown for eBusiness System and:

  • select Provider Directory Service
  • select Submit
  • Go to Organisation Register on the left hand side of the page
  • Use the drop down to select either the Organisation or Organisation Site Record
  • Select View
  • Enter the Organisation ID or Organisation Site ID
  • Use the drop down next to ID Type
  • Select ID type according to the search ID being used
  • Select Search

If search is using:

  • the Organisation ID, go to Step 2
  • the Site ID, go to Step 3

2

Organisation record + Read more ...

The Organisation Record displays with all the details for the Organisation in view mode.

Scroll down to Associated Sites:

  • Select from Current Organisation Sites the relevant Organisation site ID (there may be multiple sites under the organisation record)
  • Select View from the Action Column
  • Go to Step 4

3

Organisation Site Record + Read more ...

Organisation Site Record displays in view mode

  • Scroll to the end of the page
  • Select Program Registration button

4

Program Registration/s + Read more ...

Program Registration/s will display in Create/Amend Mode

Check if relevant program is listed (e.g. GPTP) on screen and if:

  • Relevant program is listed with no payment instruction assigned, direct the caller to HPOS to add their bank account details
  • Relevant program is listed, with current bank account details, select bank details for more information
  • Relevant program is not listed and does not display on screen, direct the caller to go to HPOS and register the program by selecting from the new program drop down GPTP. Bank account details can now be added after registration is complete
