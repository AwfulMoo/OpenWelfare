This page contains details of transfer situations involving Safety Net, rules regarding suppressed services, rules for same day delete transaction, group rules for selected services group, selection limits, radiotherapy items, Report Management and Distribution System (RMDS) reports, rules for quality control, rules applying to ROMI, Safety Net and substantiation, refine adjustment information, the Patient claim adjustment - unpresented cheque letter template, and contact details. Links on this page may be unavailable to external audiences.

Suppress service suppress rules

Table 2: This table lists the rules regarding suppressed services.

Rule

Description

1

The result of a suppress action is always an overpayment. The patient is not responsible for the debt on suppressed services.

2

The transfer and assessing screens are bypassed when suppressing services, except in circumstances where joint family Safety Net registrations apply.

3

The details of services to be suppressed cannot be changed and are not reassessed during the adjustment.

4

The result of the adjustment is calculated automatically when the services are selected with the action codes:

  • SF - Suppress unrelated services - Fraud
  • SH - Suppress unrelated services - Medicare
  • SM - Suppress unrelated services - MOL

5

Service Officers cannot perform another adjustment on suppressed services.

Rules for same day delete transactions

Table 3: This table lists rules for a same day delete transaction. Any adjustment updated using the NHHI transaction to suppress or transfer services can be deleted.

Rule

Description

1

The same date delete must occur:

  • on the same day
  • by the same Service Officer

2

To transfer claim lines from 1 history to another, the claim lines are selected from card number 1 and transferred to card number 2.

3

When a transfer is deleted, it must be actioned in the same order as originally processed.

  • Card number 1 is to be entered into the control line to retrieve all associated claim lines for both card numbers 1 and 2.

4

When card number 2 is used in the control line, no information related to the transfer is displayed.

The Process page contains the procedure to delete a same day adjustment.

Rules for selected services group

Table 4: This table lists group rules for selected services group.

Rule

Description

1

Referred/request item numbers with associated AH, SR, DI or PA lines.

2

Multiple procedures and notional charges, including PAYP items.

3

Relative Value Guide (RVG) - when the whole group is rejected selecting 1 line with rejected services.

Note: when a group includes paid and rejected services and rejected lines sit within the group, any line selected will return the whole group.

4

Diagnostic imaging multiple services rule (DIMSR)/ Multiple vascular ultrasound services site rule (MVUSSR) - the message 8-NOTL CHG appears when these items are processed with individual charges (similar to NHRI).

Note: these rules apply to action codes TH, SH, SM and SF. Subsequent adjustments cannot be performed where action codes SH, SM and SF appear or services are transferred to another Medicare card.

Selection limits

Note:

  • Service centre Service Officers may perform 1 adjustment
  • Zone office Service Officers may perform an additional 2 adjustments (or 3 where a service centre has not made a previous adjustment)
  • where a 4th or further adjustment is required, it must be referred to Medicare Claims Helpdesk Support

See also: Adjust Medicare patient claim on a previously adjusted service.

Rules for selection limits

Table 5: This table describes the latter day adjustment (LDA) rules that apply to selection limits associated with action code TH.

Rule

Description

1

No more than 25 history lines can be selected with using the action code TH.

2

Patient claims statements cannot accommodate more than 50 lines.

Radiotherapy items

Note:

  • Service centre Service Officers may perform 1 adjustment
  • Zone office Service Officers may perform an additional 2 adjustments (or 3 where a service centre has not made a previous adjustment)
  • where a 4th or further adjustment is required, it must be referred to Medicare Claims Helpdesk Support

See also: Adjust Medicare patient claim on a previously adjusted service

Rules for radiotherapy items

Table 6: This table describes the latter day adjustment rules (LDA) that apply to radiotherapy items.

Rule

Description

1

When radiotherapy Medicare Benefits Schedule (MBS) items are selected for transfer not more than 16 lines can be adjusted at any 1 time.

2

All radiotherapy items are derived fees, therefore 32 items are required (base item and derived items).

Table 7: This table describes the latter day adjustment rules that apply to Report Management and Distribution System (RMDS) reports.

Rule

Description

RMDA report - transferred services that result in an overpayment are written to the current MCA Medicare Overpayment reports.

2

  • where a 4th or further adjustment is required, it must be referred to Medicare Claims Helpdesk Support

Description

Rules for Safety Net and verification

Table 10: This table describes the latter day adjustment (LDA) rules that apply to Safety Net and verification.

Rule

Description

1

Cannot verify and transfer the service on the same day. The message 1-SUBS TDY displays.

2

Cannot LDA and then verify on the same day. The message 8-LDA TDAY displays.

3

Verification must occur on the following day.
