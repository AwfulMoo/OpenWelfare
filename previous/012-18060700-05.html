Organisation details not saving when amending or creating a provider number with a 3GA placement

An attempt must be made to key organisational details at first point of keying. Follow the steps if unsuccessful.

For existing 3GA placements, take a screen snip of provider number location page (for reference), and then:

  • remove all data relating to 3GA placements e.g. specialty code or information in provider placement tabs, and save
  • go back into ‘edit’ mode and key ABN and business structure information, and save
  • go back into ‘edit’ mode and rekey data relating to 3GA placements

For new 3GA placements:

  • key address, contact information, provider number start and end dates, ABN and business structure details, press save
  • go back into ‘edit’ mode and key 3GA placement information

If issues persist, escalate to a Team Leader with the medical practitioner’s details, the error occurring and snips of the screens.
