TFES - Registration and change of details form received

Table 1

Step

Action

1

TFES - Registration and change of details form (TAS020) + Read more ...

Businesses use the TAS020 form to:

  • register to claim assistance with TFES
  • updating their business details
  • register the business and users for Centrelink Business Online Services (CBOS)

After creating a new claimants record, upload the TAS020 on the claimant documents tab on the record.

For existing claimants, upload the TAS020 on the claimant document tab if received via email, post or fax. Claimants with CBOS access can upload the document to their claims document tab.

If a claimant fills in an old version of the form, ask them to complete a new form, direct them to the Services Australia website to access the current version.

Email TAS020 forms received outside the National Business Gateway, to TFES support.

2

Check the form is completed + Read more ...

For new claimants, see Create a CRN for an Organisation in Customer First table in Register a TFES claimant.

For existing claimants updating details, see Registration and Change of Details (TAS020) form received table of Update TFES claimant information.

3

Locate online activity in TFES Assessor Tool + Read more ...

After TAS020 is processed, an activity is created in the TFES assessor tool for the online registration, if the TAS020:

  • has been submitted to register a new TFES claimant, or
  • is adding or removing signatories from an existing TFES claimant record

The activity status is changed to awaiting internal advice to ensure the online registration is not actioned until the QA officer has completed the required checks.

If the QA officer approves the new claimant eligibility or updates to the existing claimant record, they:

  • search on the TFES assessor tool for the online registration activity for the claimant, and
  • change the status of the activity to QA back to assessor

The online activity can then be selected for action from the allocation list in the TFES tool.

Create organisation or user ID in RAMB

Table 2

Step

Action

1

Confirm claimant details in myHSI + Read more ...

Before creating an Organisation in Registration and Access Management for Business (RAMB), confirm the details in myHSI.

In myHSI, open TFES Claimant:

  • Enter claimant code, select search
  • Select claimant code to enter the View Claimant Details screen
  • Select Comments tab to confirm the record is eligible and approved by QA officer

Do the claimant comments show the claimant is eligible?

  • Yes, go to Step 2
  • No, open TFES Assessor tool:
    • select the Online Registration activity
    • in Comments/Details field, enter ‘Claimant ineligible, no online registration required’
    • change activity Status to Completed
    • process ends here

2

Searching for an organisation RAMB + Read more ...

Before creating a new organisation, search on RAMB to see if the business is already registered.

In myHSI:

  • Open TFES Claimant
  • Enter Claimant Code, select search
  • Select Claimant Code to enter the View Claimant Details screen
  • Copy the CRN

Open RAMB:

  • Under Manage Organisation, select View/Update Organisation
  • In Organisation Search paste the CRN of the business, select Continue

Did the text ‘Your search has not returned any records. Please review your search criteria and try again.’ appear?

  • Yes, this means the organisation is not registered for TFES CBOS, go to Step 3
  • No, this means the organisation is registered for TFES CBOS, go to Step 6

3

Creating an organisation in RAMB + Read more ...

In myHSI:

  • Open TFES General, Claimant Details screen
  • Open another tab to access Registration and Access Management for Business (RAMB)
  • Under Manage Organisation select Create Organisation

In Organisation Details fill in the following fields. If the details are on the Claimant Details screen, copy the information and paste it in the field:

  • Organisation Type: select Small business not ABN – TFES only
  • Organisation CRN: paste the CRN
  • Organisation Name: paste the Claimant Name
  • Organisation ID: Service Officer creates and keys an authentic Organisation ID
    • Preference is to use the first 3 letters of the claimant name followed by one number
    • Must be 4 characters
    • May include both letters and numbers
  • Contact Name: –paste Contact Name
  • Contact Number: paste Business Phone Number

4

Allocate Role to the organisation + Read more ...

After creating the organisation in RAMB , allocate the roles:

  • Organisation Online Mail, select the tick boxes:
    • OOM Basic Access
    • OOM Tasmanian Bulk Upload System

  • Tasmanian Freight Equalisation Scheme, select the tick box:
    • TFES online claimant

Select Continue.

Check the following 3 accesses are ticked:

  • OOM Basic Access
  • OOM Tasmanian Bulk Upload System
  • TFES online claimant

Did the warning ‘the Organisation ID already exists’ appear in red?

  • Yes, enter an alternative Organisation ID
  • No, select Finalise. Go to step 5

5

Create the Work Unit + Read more ...

Enter the Work Unit information by selecting the following:

  • Work Unit ID: TFES Bulk Upload Service (TBUS) – 001 from the dropdown menu
  • Work Unit Name: key TBUS
  • Work Unit Description: key TBUS
  • Select Continue
  • Check that all details have been entered correctly and then Finalise

6

Register user in RAMB + Read more ...

Prepare the email templates to send to the signatory to advise of User ID and temporary password:

  • TFES Centrelink Business Online Services Advice of Password – Email#2
  • TFES Centrelink Business Online Services Advice of User ID – Email#1

The email templates are in TFES support mailbox in the Stationery folder.

In myHSI:

  • Open TFES General, Claimant Details screen, copy the Customer Reference Number (CRN)
  • Open another tab to access Registration and Access Management for Business (RAMB):
  • Under Manage User select Register User

In RAMB:

In Organisation Search, paste CRN, select Continue.

User Details screen shows the Organisation Name, Organisation CRN and the Org ID.

In Enter User Details key the following details:

  • First name: copy and paste from Claim Signatories in myHSI
  • Last name: copy and paste from Claim Signatories in myHSI
  • Employee ID: key TFES (this is the same for every RAMB user)
  • Contact number: copy and paste Business Phone Number from myHSI
  • Email: key email address for user as shown in section 16 on TAS020. Note: if the details of the contact person are not listed in section 16, key the details entered in section 9 – 12 on TAS020
  • Select Continue

7

Give user access to work units + Read more ...

In RAMB under the heading Universal

  • Under Organisation Online Mail, select the tick boxes:
    • OOM Basic Access
    • OOM Tasmanian Bulk Upload System

  • Under Tasmanian Freight Equalisation Scheme, select the tick box:
    • Tasmanian online claimant

  • Under Work Units, select the tick box:
    • TBUS

Select Continue

Check the following 4 accesses are ticked:

  • OOM Basic Access
  • OOM Tasmanian Bulk Upload System
  • TFES online claimant
  • TBUS

The next screen shows the following:

  • Account Identification
  • User ID
  • Temporary Password
  • Account creation date

Select Finalise and fill in the details in the email templates prepared in step 6.

Send both emails to the email address entered for the signatory on the TAS020.

To add additional users, select Update Registered user and select Add another user. Repeat steps 6 and 7.

Note: when adding additional users, a warning ‘There are some existing users that may be a match to the new user being registered’ appears.

  • Check to ensure the user is not already registered
  • Select the check box ‘Registered user does not already exist’

8

Complete online activity in the TFES Assessor Tool + Read more ...

  • Open TFES Assessor tool
  • Select online activity
  • Change the status to complete
