Going to and viewing the Service payment summary screen + Read more ...

Go to the Service payment summary screen using any of the following:

  • Main menu (Hamburger button)
    • Select the Main menu icon
    • Select Service Search
    • The Service Search screen displays
    • Key the service’s details
    • Select Search
    • Select the hyperlinked Service NAPS ID in the search results table
    • The ACSP Service in Context home page displays
    • Select Service payment summary under the Payment statements section
    • The Service payment summary screen displays
  • Quick link navigation bar
    • Select the Service payment summary hyperlink in the Quick links navigation bar
    • The Service payment summary screen displays
  • Quick find navigation bar
    • Type Service payment summary in the Quick find search bar
    • Select Go
    • The Service payment summary screen displays
  • Payment statements menu tab
    • Select the Main menu icon
    • Select Service search under the Aged Care Staff Portal section
    • The Service search screen displays
    • Key the service’s details
    • Select Search to return search results
    • Select the hyperlinked Service NAPS ID
    • The ACSP Service in Context home page displays
    • Select the Main menu icon
    • Select Service payment summary under the Payment statements section
    • The Service payment summary screen displays
    • Hover the mouse pointer over the Payment statements menu tab
    • A pop up window displays, select Service payment summary
    • The Service payment summary screen displays

Viewing the Service payment summary screen

Note:

  • PDF format is not available for residential care payment statements
    • See the Resource page for the manual work around to this issue

The Service payment summary screen displays the following sections:

The Description and Amount section

  • Displays for all care types

The Subsidy/supplements section

  • Displays for all care types
  • The various supplements displayed are dependent on the service in context’s care type

The Adjustments for current period section

  • Commonwealth unspent amount (for home care only)
  • Manual adjustments
  • Means/income tested subsidy reduction (for residential care only)
  • Means/income tested subsidy review refund (for residential care only)
  • Income tested subsidy reduction (for home care only)
  • Income tested subsidy review refund (for home care only)

The Adjustments for previous periods section

  • Displays for all care types
  • The supplements displayed in this section are dependent on the service in context’s care type

The Outstanding balance and advance/ad hoc section

  • Total subsidy paid
  • Less Outstanding balance from (previous claim period)
  • Less Advance (for flexible care and residential care services)
  • Less Ad-hoc payment (for home care services only)
  • Total amount paid to the provider
  • For home care services only
    • Income tested subsidy review refund
    • Income test reduction
    • CW Unspent amounts
    • Payment Due/Payment Held over amount

The Special payments description section

  • Displays for residential care services only
  • Displays the date of when Special payments were made to the residential care service Note: Service Officers can select the hyperlinked Special payments summary and the Special payments summary screen displays

From the Service payment summary screen, Service Officers can:

  • change the Claim month
    • select the month from the Claim month drop down list Note: the Claim month defaults to the current payment statement
  • download the payment statement in CSV, XML or PDF format
    • select the relevant format button
  • view historical payment statements

Procedure ends here.

  • Care recipient details
    • ID
    • Family name
    • Given name
  • Admission details
    • Entry
    • Departure

    • Type

    • Reduced

  • Rate per day
  • Total Entitlement Available

    • Paid leave days

    • First, Previous, Next or Last options, or
    • the relevant page number
  • select the relevant format option to download the payment summary in CSV, XML or PDF format

2

Create Payment Statement Notes + Read more ...

Note: payment statement notes can only be edited prior to claim approval.

The 2 types of payment statement notes are:

Global payment statement notes - used when Services Australia need to communicate broadly with all service providers. While Global payment statement notes cannot be modified at the service level, they can be excluded from appearing on a payment statement for a service.

Service BP payment notes - used when Services Australia need to communicate with a specific service provider. Service BP payment notes can be created/modified any time up to the point of approval of the claim.

  • Select Claim Events from the left hand side menu
    • The Service Orders screen will display
  • Select Payment Statement Notes
    • The Payment Statement Notes screen will display
  • Select New

Key the following fields with information:

  • Note Type - 'Payment Statement by Service BP’ or ‘Payment Statement Global Notes’
  • Care Type - this field is optional. The relevant care type can be chosen from the drop down list.
  • Visible on Payment Statement - if this box is unchecked, the note is excluded from appearing on a payment statement for a service
  • Notes - it is a free text field. Enter relevant information that will display in the payment statement
  • Effective From - start date of the claim month. It can be a future date. This is a mandatory field
  • Effective Till - last date of the claim month. This is a mandatory field
  • Service ID - this field is optional. To apply the note to specific service(s), enter the ID here. Select 'Insert' to add more rows. If left empty, all the services for the selected care type and selected state are considered
  • Select Save.

Procedure ends here.
