Debt Recovery only Engagement Fast Notes

Table 1

Fast Note

To be used:

Closed/open

Unsuccessful Engagement

Used for all unsuccessful contacts for example:

  • number disconnected
  • no answer
  • unable to confirm identity
  • customer not available

Closed

Unsuccessful Engagement - Refusal to POI

Use when the call is answered and the person refuses to establish Proof of Identity (POI).

Closed

Successful Engagement - Refusal to pay

Use when the customer completes POI but refuses to pay.

Closed

Successful Engagement - No Arrangement

Use when customer is aware of the debt, but has not made a commitment to pay today. For example: customer to consider options, finances, requests a statement, will call back, etc.

Closed

Successful Engagement - Transferred call

Used where the customer is made aware of the debt and the call is transferred to another area to discuss the debt - such as appeals, family assistance, External Collection Agent (ECA), etc.

Closed

Successful Engagement - Payment Arrangement

Used if script is not working: For all types of payment options including payment arrangement, payment in full, partial lump sum and withholdings.

Closed

Recovery options exhausted

Used when all contact attempts/all recovery options have been exhausted.

Closed

Fast Notes used by debt staff when referring debt recovery

Table 3

Fast Note

To be used:

Closed/open

Garnishee customer service

Used when requesting a garnishee be ceased or varied, or to request a quality check.

Open

Wage Garnishee Referral Fast Note

Used when requesting a garnishee of continuing income (for example, employment)

Closed

Bank Garnishee Referral Fast Note

Used when requesting a garnishee from a financial institution

Closed

High Value Debt Team

Used when referring a file for Civil recovery action. Prior to making a civil recovery referral, debt staff must ensure the file contains an asset/s that could be recovered through civil action.

Open

Departure Prohibition Order (DPO)

Used when referring a file for DPO. Prior to making a DPO referral, debt staff must ensure the file contains debts that are suitable for DPO action.
