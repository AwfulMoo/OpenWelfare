Sanction notification

Table 1

Step

Action

1

Notification received from the Aged Care Quality and Safety Commission + Read more ...

Aged Care and Allied Health (ACAH) Branch receives email notification from the Aged Care Quality and Safety Commission via their positional mailbox.

Services Australia register the sanction in the systems which enables correct payments during the sanction period.

  • Residential care sanctions are processed by ACAH
  • Home care sanctions are processed by the Aged Care Programmes Branch’s Operations (Home Care team)

All sanction conditions are included in the notification. This includes any exceptions to the sanction. For example, an exception is when a care recipient who has already entered into a care agreement before the sanction period but not yet entered into care.

Identify the relevant payment system + Read more ...

If the sanction applies to:

  • a home care and flexible care services, see Table 3
  • a residential care service, see Table 4

Manage sanctions for home care or flexible care in ACMPS

Table 3

Step

Action

1

Log on to ACMPS + Read more ...

Note: if the Sanctions tab is not visible on the screen, select Display tabs icon and select the tab from the list.

Only Subject Matter Experts (SME) and above in the Aged Care and Allied Health (ACAH) team have the delegation to process a sanction in ACMPS.

  • Log on to ACMPS
  • From the Home screen, select Service Search
  • Enter the service ID stated on the template in the Service ID field
  • Select Sanction Details tab
  • All sanction details display under the Sanction Details tab

2

Code/update sanction details + Read more ...

Note: if the checkbox remains unchecked, when approving the claim, the warning message ‘Sanction exists, check service claim for new care recipients entered’ displays on the Multiple Claim Approval screen. This warning prompts the checks needed before approving the claim.

  • Select the Edit List icon
  • Enter the required information in the fields provided:
    • Sanction Start date (select field icon to display the calendar)
    • Description
    • Sanction End Date
    • Proposed End Date
    • Inactive checkbox - if the checkbox is unchecked, there is a current sanction in place, if the checkbox is checked, the sanction has ended
  • Select Save

3

Monitor monthly claims + Read more ...

If a sanction is in place, monitor the status of monthly claims for the duration of the sanction. This includes identifying (any) new care recipients who entered into care during the sanction period. This must be done before the claim being approved.

4

Service search screen check + Read more ...

On the:

  • Home screen, select Service Search
  • enter the service ID stated on the template in the Service field
  • select Active Care Recipients

Identify the existing care recipients from new care recipients by cross checking the dates from the Active From column against the sanction period.

Care recipients whose Active From date:

  • falls within the sanction period, including the commencement date, are considered to be new care recipients, go to Step 5
  • is prior to the commencement of the sanction period are considered to be existing care recipients, go to Step 6

5

New care recipients entered into care + Read more ...

  • Contact the Aged Care Quality and Safety Commission via email or phone to determine why new care recipients were entered during the sanction period (as this means sanction requirements may not been met)
  • The Aged Care Quality and Safety Commission will give instructions if any other action is required
  • Once actioned, go to Step 6 to approve claim

6

Care recipients existing + Read more ...

If all the care recipients are existing, the claim can be approved.

Advise the Aged Care Payments Team within the Aged Care and Allied Health Branch that claim can be approved as sanction requirements have been met.

Use the Register sanction details screen + Read more ...

Note:

  • A warning message will display when:
    • the sanction start date is 28 days or more in the past
    • the sanction period is less than 28 days between the sanction start and end dates
  • If the sanction is for a set period, the End date field must be entered
  • All fields are optional unless marked with an asterisk

On the Register sanction details screen, complete the following fields:

  • Received date* field – key the date (DD.MM.YYYY), or select from the calendar (select the calendar icon to display the calendar)

  • End date field – key the date (DD.MM.YYYY), or select from the calendar (select the calendar icon to display the calendar)
  • Case ID* field
  • Sanction type* field – the drop down menu only displays ‘No new entries during sanction period’
  • Additional information field – document clear and accurate comments detailing the action taken. There is a maximum 500 character limit

Manual adjustment of a sanction for residential care in ACSP

Table 8

Step

Action

1

Apply manual adjustment for care recipients in a sanctioned service + Read more ...

If a care recipient entered:

  • into permanent care during the sanction period:
    • call the Aged Care Quality and Safety Commission to find out why they were entered, as sanction requirements have not been met
  • residential respite care before the start of the sanction period and then became a permanent care recipient at the same service after the sanction start date:
    • check that the care recipient is listed in the Aged Care Quality and Safety Commission’s email, they must be listed
    • apply a manual adjustment, to manage a residential care manual adjustment, see the Using the Aged Care Staff Portal to manage manual adjustments for residential care (Service in Context) table
    • add a care recipient note against the care recipient’s record with details of the action taken, see the Using the Aged Care Staff Portal to manage residential care manual adjustment (Service in Context) table
