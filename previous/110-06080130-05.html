3

Review Result Reason Codes applicable to Review Result ‘PRT’ (Present Rate to Continue) for Review Type EII (Earned Income Intervention) + Read more ...

The following codes and hierarchy rules must be applied for Review Type EII (Earned Income Intervention) where the Review Result is PRT (Present Rate to Continue).

Determine which of the following circumstances is applicable to the intervention and code it appropriately. If more than one circumstance is relevant, the code highest on the following list must be used:

  • if a debt is attributed to the intervention, code - ‘Debt but no change in rate’ (DBT)
  • if working credit/income bank is updated by Compliance Officer (CO) responsible for the intervention, code - ‘Income not declared correctly’ (NDC)
  • if the customer is placed on stimulus reporting, code Customer placed on stimulus (STM)
  • if compensation was identified as part of the intervention by the CO responsible for the review (or by a compensation Service Officer on behalf of the reviewing CO) - code ‘Compensation update’ (CMM)
  • if a qualification related Families Assistance update or debt was created by the CO responsible for the intervention, code - ‘Family Payment related outcome’ (FRO)
  • if the customer has correctly declared their income, code - ‘Income declared correctly’ (DDC)
  • if the customer has contacted and advised they did not commence work, code - ‘No change in circumstances’ (NCC)
  • if the customer:
    • has failed to contact after CRB suspension action, OR is restored/suspended/cancelled outside of CRB, AND
    • a Review Result of SUS/CAN cannot be claimed, code - ‘Information not provided but no change to rate’ (IND)

5

Review Reason Codes applicable to Review Result 'CAN' (Cancellation), 'SUS' (Suspension) or 'REJ' (Rejection) + Read more ...

IRS will display the Review Result Reason as the actual reason the customer's payment was 'SUS'pended, 'CAN'celled, or 'REJ'ected in the host system.

Other acceptable values are any valid cancellation, suspension or rejection reason from host processing.
