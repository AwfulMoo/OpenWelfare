Workforce Supplement financial year percentage rate

Enter a Workforce Supplement financial year percentage rate

Table 1: This table describes how to enter the Workforce Supplement yearly rate in System for the Payment of Aged Residential Care (SPARC).

Step

Action

1

Log on + Read more ...

Log on to SPARC.

2

Select option 9 + Read more ...

From the Main Menu:

  • select option 9 (System Administration)
  • press [Enter]

Result: the Administration Menu is displayed.

3

Select option 3 + Read more ...

Select option 3 (System Constants Maintenance Menu) and press [Enter].

Result: the System Constants Maintenance Menu is displayed.

4

Select option 68 + Read more ...

Select option 68 (Workforce Supplement Percentage) and press [Enter].

Result: the Workforce Supplement Percentage is displayed.

5

Key details + Read more ...

Key:

  • Date of effect and press [Tab]. Result: Validates and the cursor moves to percentage field.
  • Percentage rate and press [Enter]. Result: Validates the percentage rate.

6

Save percentage rate + Read more ...

Press [F5] (Commit) twice to save the percentage rate.

Verify percentage rate

Verify the Workforce Supplement financial year percentage rate

Table 2: This table describes how to verify the Workforce Supplement financial year percentage rate. Rates must be verified by a Service Officer other than the Service Officer that was responsible for the most recent update to the rate.

Step

Action

1

Log on + Read more ...

Log on to System for the Payment of Aged Residential Care (SPARC).

2

Select option 9 + Read more ...

From the Main Menu:

  • select option 9 (System Administration)
  • press [Enter]

Result: the Administration Menu is displayed.

3

Select option 3 + Read more ...

Select option 3 (System Constants Maintenance Menu) and press [Enter].

Result: the System Constants Maintenance Menu is displayed.

4

Select option 68 + Read more ...

Select option 68 (Workforce Supplement Percentage) and press [Enter].

Result: the Workforce Supplement Percentage is displayed.

5

Check details are entered correctly + Read more ...

Have the date of effect and percentage rate been entered correctly?

  • Yes, go to Step 6.
  • No, request the originating Service Officer to update the system.

6

Verify + Read more ...

Press [F6] to verify.

Result: Information message displays advising percentage rate has been successfully loaded into SPARC.

Load Workforce supplement data

Load the Workforce Supplement data to the DSS mainframe dataset

Table 4: This table describes how to load the Department of Social Services (DSS) Workforce file into the Mainframe Time Share Option (TSO).

Step

Action

1

Log on + Read more ...

Log on to DSS mainframe.

Key the:

  • DSS LAN ID
  • password

Press [Enter].

Result: the Solve Access - Primary Menu is displayed.

2

Key details + Read more ...

  • Key M (Access Services) and press [Enter]. Result: the MAI Primary Menu is displayed.
  • Select TSO and press [Enter]. Result: The TSO/E Logon menu is displayed.
  • Key the:
    • DSS LAN ID
    • password
  • Press [Enter]. Result: The 'Ready' prompt is displayed.

3

Select as required + Read more ...

Select:

  • Send Files to Host (4th icon on menu bar). Result: the Send Files to Host - Session A window is displayed.
  • Browse from the PC File Name field. Result: the Browse window is displayed.

From the Browse window, select:

  • and double click required file in the network drive or removal storage
  • Open

Result: the Send Files to Host - Session A window is displayed.

4

Send Files to Host + Read more ...

From the Send Files to Host - Session A window:

  • Click on selected file in the Transfer list. Result: The PC file name, Host file name and Transfer list panes are populated.
  • Change the Host File name:
    • Key name of file.
    • Note: use 'single' quote marks and SPARC user ID. For example, 'WHQPE.XCS9.WS.LOAD.PROD'.

From Transfer Type drop down menu, select:

  • Text
  • Update in List

Result: the Transfer List - PC File Name will display the File Type as text.

From the Send Files to Host - Session A window:

  • check the Logical Record Length
  • select Options

Result: the File Transfer Settings window is displayed.

5

Check Logical Record Length field + Read more ...

From the File Transfer Settings window, is the Logical Record Length field 256?

  • Yes, select OK. Go to Step 7.
  • No, go to Step 6.

6

Navigate to Send Files to Host - Session A window + Read more ...

From the Transfer Type drop down menu:

  • select text
  • in the Logical Record Length field:
    • key 256
    • select OK

Result: the Send Files to Host - Session A window is displayed.

7

Send information + Read more ...

Double click Send.

Results: the Send-File status - Session A window is displayed, then the Ready prompt screen is displayed.

8

Log off + Read more ...

From the Ready prompt screen, key logoff and press [Enter].

Workforce supplement file

Load the Workforce Supplement file

Table 5: This table describes how to load the Workforce supplement file in the System for the Payment of Aged Residential Care (SPARC) production.

Step

Action

1

Log on + Read more ...

Log on to DSS mainframe.

Key the:

  • DSS LAN ID
  • password

Press [Enter].

Result: the Solve: Access - Primary Menu is displayed.

2

Key M + Read more ...

Key M and press [Enter].

Result: the Solve Access - Primary Menu is displayed.

3

SPARC Production + Read more ...

Select SPARC Production and press [Enter].

Result: The SPARC Logon is displayed.

Key the:

  • DSS LAN ID
  • password

Press [Enter] twice.

See System for the Payment of Aged Residential Care (SPARC).

Result: The Medicare Australia Corporate Menu is displayed.

4

Select appropriate option + Read more ...

  • Select 2 (SPARC) and press [Enter] twice. Result: The SPARC Main Menu is displayed.
  • Select 9 (System Administration) and press [Enter]. Result: The SPARC Administration Menu is displayed.
  • Select 5 UTL (Ad Hoc Report/Batch Job Utility) and press [Enter]. Result: The SPARC Administration Ad Hoc Report/Batch Job Utility window is displayed.

5

Key fields + Read more ...

Key fields:

  • Rule name: NHSUTLRU_WS
  • (2) JCL Model: STANDARD
  • (3) Mode: UPD
  • (4) No. of Parameters: 0
  • (5) Output settings:
    • Print to Screen: N
    • Job Class: S
    • Print Destination: IPR specific to location
    • Print Class: A
    • Copies: 1

Press [F6] to submit the job.

Result: REPORT NHXUTL BATCH JOB SUBMITTED is displayed at the bottom of the AD HOC REPORT/BATCH JOB UTILITY.
