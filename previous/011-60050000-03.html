Step

Action

Confirm that the PBS prescriber requires immediate assistance + Read more ...

Note: a HPOS delegate’s identity cannot be verified over the phone. Staff therefore cannot conduct patient checks or registrations for delegates. See ‘Handle calls relating to the CTG PBS Co-payment Program’ on the Resources page for more information.

Does the PBS prescriber require immediate assistance with their eligible patient who is in consultation with them, and who has given consent to search for and be registered on the program?

  • Yes, go to Step 2

Login to Medicare – eBusiness Gateway + Read more ...

Open the Services Australia Intranet page.

    • Service Officers select their own source office code
  • eBusiness System
    • select Provider Directory System

The Provider Directory System Home Page displays.

4

Security check the prescriber in Provider Directory System (PDS) + Read more ...

To conduct a Basic Search using a Prescriber Number or a Provider Stem:

  • From the left-hand menu, drop down Provider, and select View
  • Select Basic Search
  • Select ID Type (Prescriber Number or Provider Stem)
  • Enter the Prescriber Number (preferred) or Provider Stem (first 6 numbers, no letters)
  • Select Search
  • Ask caller to confirm their full name

Have the details returned a result?

  • Yes, go to Step 6
  • No, go to Step 5

5

Conduct a Detailed Search using prescriber’s personal details + Read more ...

  • Select Detailed Search
  • Enter Family Name
  • Enter Given Name
  • Enter Profession type from the drop-down box, as supplied by the caller

Have the details returned a result?

  • Yes, go to Step 6
  • No,
    • Advise the caller that you cannot verify them on Medicare systems, as per Handle calls relating to the CTG PBS Co-payment Program on the Resources page
    • Procedure ends here

6

Confirm caller details + Read more ...

Ask the caller to confirm their:

  • date of birth, and
  • place of qualification or current practice location address

Are these details correct?

  • Yes, go to Step 7
  • No,
    • Advise the caller that you cannot verify them on Medicare systems
    • See Handle calls relating to the CTG PBS Co-payment Program on the Resources page
    • Procedure ends here

7

Open the Consumer Directory Maintenance System (CDMS) + Read more ...

Open the Services Australia Intranet page.

Select:

  • the Customer Service tab
  • Medicare Portal tile

The Medicare – eBusiness Gateway Home Page screen displays.

From the dropdown menus:

  • Source office
    • Service Officers select their own office code
  • eBusiness System
    • select Consumer Directory Maintenance System
  • Select the green Submit button from the bottom right corner

The Welcome to the Consumer Directory Maintenance System screen displays.

Note: CDMS session timeouts are set for 30 minutes. If this happens, log in again.

8

Retrieve patient details within CDMS + Read more ...

  • Select Amend Consumer Details
  • Select Basic Search
  • Program – Medicare
  • ID Number Type- Medicare Card Number
  • Enter the patient’s Medicare number (not including the Individual Reference Number)
  • Select Entitlement

Have the details returned a result?

  • Yes, go to Step 9
  • No, go to Step 10

9

Confirm the correct patient + Read more ...

Ask the provider to confirm the patient’s:

  • full name, and
  • date of Birth

Once verified, select the correct patient on the radio button, and select the Closing The Gap tab.

The patient’s status will either show an Active or Inactive CTG status.

Go to Step 11.

10

Conduct a Detailed Search using patient’s personal details + Read more ...

  • Select Detailed Search
  • Enter Family Name
  • Enter Gender
  • Enter Date of Birth or Age Range
  • Select Entitlement
  • Select correct patient on search results and Entitlement

Once verified, select the correct patient on the radio button, and select the Closing The Gap tab.

The patient’s status will either show an Active or Inactive CTG status.

11

Confirm patient status + Read more ...

Release information to the prescriber.

If the patient is already registered for CTG, the prescriber can issue a prescription.

If the patient is not registered for CTG, go to Step 12.

12

Register a patient for CTG + Read more ...

Confirm with the prescriber that they would like to register the patient for the CTG PBS Co-payment Program

  • Enter the current date in the Start Date field
  • Enter a Comment to include Prescriber Surname, Prescriber/Provider number and the reason why the prescriber was required to complete a phone registration (for example, Dr X, Pres #1234567, prescriber was in a remote location with no internet access)
  • Select Validate
  • Select OK on the Information Message pop-up
  • Select Validate
  • Select Update to save, or
  • select Change to make a further edit

The CTG enrolment is complete

Note: if a registration is completed in error, escalate the details to National Office. See Escalation templates on the Resources page.

Withdrawing from the CTG PBS Co-payment Program

Table 2

Step

Action

Patients wanting to remove themselves from CTG + Read more ...

Note: a PBS prescriber cannot withdraw a patient who is already registered for the CTG PBS Co-payment Program.

If a patient wants to remove themselves from CTG first complete steps 7-10 in Table 1.

2

Withdraw a patient for CTG + Read more ...

Confirm with the patient that they would like to withdraw themselves from the CTG PBS Co-payment Program.

  • Enter today’s date in the End Date field
  • Enter a Comment to include reason for updating (e.g. By patient [insert patient name] request)
  • Select Validate
  • Select OK on the Information Message pop-up
  • Select Validate
  • Select Update to save or Select Change to make a further edit
  • Complete the Escalation template on the Resources page if a patient has complex enquiries regarding their CTG registration.

The CTG withdrawal is complete.

Pharmacists dispensing and claiming CTG prescriptions

Table 3

Step

Action

1

Pharmacist dispenses the PBS item + Read more ...

The pharmacist must submit a CTG 6-character value, either manually or generated by their dispensing software, into the software system for CTG patients.

The pharmacist dispenses the PBS item as per normal dispensing procedures.

The dispensing software will verify a patient’s CTG status in real time and calculates the:

  • Co-payment to be charged to the patient
  • PBS Safety Net amount to count towards the family's Safety Net records

Note: eligible pharmacies can check a patient’s CTG status within HPOS. See Delegate and Pharmacy Organisation Steps to Access HPOS for Closing the Gap on the Resources page.

2

Non-concessional patient/Concessional patient + Read more ...

If the prescription is for a:

  • Non-concessional patient, ensure that the prescription is included in the General bundle for the next PBS claim once it has been receipted by the patient or agent
  • Concessional patient, ensure the prescription is included in the Concessional bundle for the next PBS claim once it has been receipted by the patient or agent

Note:

  • RPBS prescriptions must be included in the Repat bundle as per normal
  • Do not include under Co-payment prescriptions in the PBS claim bundles (unless, as per normal practice, they are on the same prescription form as claimable items)
  • Under Co-payment prescriptions are those written for non-concessional patients for PBS items priced at the time of supply below the general patient Co-payment, or concessional patients for PBS items priced at the time of supply below the concessional patient Co-payment
  • The Under Co-payment Additional Surcharge (UCPAS) cannot be charged to patients under this program
