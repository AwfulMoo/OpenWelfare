Claim level PPL cancellation

Step

Action

1

PPL claim and component level status + Read more ...

There are two status levels for PPL:

  • PPL claim: This is the overall PPL claim status. It is displayed on the customer’s benefit status line
  • Component level: These are available from 1 July 2020. It is made up of PPL period, connected Flexible PPL days and separate Flexible PPL days. These are displayed on the PPL Calculation Summary (P1CS) screen.

For a complete list of PPL claim and component level cancellations, see Cancellation, rejection, not effective, assessed and current zero rate codes for Parental Leave Pay (PPL).

2

Automatic or manual cancellation + Read more ...

Determine if the cancellation should be coded automatically or manually. In most cases, PPL is cancelled and coded automatically

  • If the customer is advising a change of circumstances, manual cancellation should not be undertaken. Record the change in circumstances. Procedure ends here
  • To continue with a manual cancellation of PPL for reason:
    • CRQ (customer request), go to Step 3
    • Another reason, go to Step 4

3

PPL/CAN-CRQ (Customer Request) + Read more ...

Discuss why the customer has asked to have their PPL cancelled. Check if a change of circumstance (e.g. transferring some of their payment to another person, such as a partner) would be more appropriate. If the decision to cancel CRQ is correct, tell the customer once PPL has been cancelled:

  • they lose any remaining PPL period and Flexible PPL days for the child
  • they cannot reclaim any remaining PPL for the same child at a later date

Tell the customer the CRQ cancellation cannot be backdated. The earliest date the cancellation can take effect is from date paid to + 1 (DPT+1).

Cancelling the PPL CRQ is an administrative decision, and not made under the PPL Act. Customers cannot appeal the CRQ cancellation reason if they later change their mind. A cancelled PPL payment may be restored if the customer has maintained eligibility. If restoring the payment, check with the customer for any change in circumstances (for example, return to work). If after cancellation there has been a change in circumstances, these changes must be coded in the restoration activity. This may result in PPL cancelling after restoring for another reason. See Restoration of Parental Leave PAY (PPL).

To code the manual cancellation on the P1CA screen:

  • Action: CAN (cancel)
  • Reason: CRQ (customer request)
  • Effect Date: on or after DPT + 1
    • to determine the DPT+1, check the PS screen for the last Entitlement Period End Date (EPED). For employer payments, the EPED may be a future date. This is the correct date, even if the payments have not yet been issued to the customer
  • Level: Claim
  • Use Fast Note - select Auto Text, use Families > SUS/CAN/RES > PPL Cancelled CRQ, record all relevant details of the cancellation
  • attach the Fast Note to the CAN activity
  • 'S'elect the CAN activity and complete the Assessment Results (AR) process

Check the activity and the date of effect to ensure no debt is generated.

A Q999 is not required, as an automatic letter will issue to the customer.

Go to Step 5

4

Manual cancellation + Read more ...

To manually code a PPL cancellation select the Claim Contact > Paid Parental Leave Claim summary option.

Note: manual cancellation should not be undertaken when a customer advises a change in circumstances. Record the change in circumstances

The PPL Claim Summary (P1CS) screen will display with the current PPL claim information. Code an 'A' to navigate to the PPL Claim Action (P1CA) screen to code the cancellation details.

On the P1CA screen, code:

  • Action: CAN (cancel)
  • Reason: select relevant reason for cancellation.
    (If using reason code of OTH (other), add notes to the activity to prevent an error. CAN-OTH activities will be reviewed for coding accuracy).
  • Effect Date: must be the date of event
  • Level: Claim
  • Use Fast Note - select Auto Text, use Families > SUS/CAN/RES > PPL Cancelled to record details relevant to the cancellation
  • attach the Fast Note to the CAN activity
  • 'S'elect the CAN activity and complete the Assessment Results (AR) process

5

Finalise activity + Read more ...

Check details before finalising activity on the Assessment Results (AR) screen for the assessment letter to be issued.

If the customer is not eligible for PPL for the child, check if another person (e.g. their partner) is eligible and invite the claim if appropriate.

If another person has also claimed PPL for the child, check if their entitlement needs to be reassessed.

If a Family Tax Benefit (FTB) customer's PPL has been cancelled 'Transfer to Partner', check if FTB Part B should be stopped.

Once PPL is cancelled it can only be restored without the need to reclaim if it is established that the payment should not have been cancelled and the customer was eligible. See Cancellation, rejection, not effective, assessed and current zero rate codes for Parental Leave Pay (PPL).

Record details on a DOC.
