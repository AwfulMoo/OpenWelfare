This page contains the process for identifying and resolving failed claim transmissions

Failed claim return codes

Mainframe failed claim return codes

This table lists failed claim return codes and descriptions, and describes how Service Officers resolve those codes.

Failed claim code

Description and resolution by the eBusiness Service Centre team

0102 HEC

Payee provider is not on file + Read more ...

Check:

  • Provider Directory System (PDS) to confirm when provider number was created
  • QPLI to see when the last update was made

Note: Mainframe takes 24 hours to update.

0103 HEC

Payee provider practice was closed more than 30 days prior to the claim date for the current claim + Read more ...

Advise the customer:

  • the location was closed more than 30 days ago
  • to call Provider Eligibility on 1300 302 122 to discuss further
  • manual claim may need to be submitted. Refer to 132 150 for more information on manual claiming

0104 HEC

Payee provider is not eligible to Medicare Direct Bill + Read more ...

  • check PDS to ensure the provider number is for a provider (not hospital or billing agent)

Practice staff to:

  • amend details of the claim, substitute another payee provider at the same location and retransmit claim

0107 HEC

Payee provider cannot be the same as Service provider at a different location + Read more ...

Advised the customer claims cannot have different provider numbers for the same Payee/Servicing provider at a different location

0108 HEC

Payee provider not registered on PDS + Read more ...

Check PDS for any recent updates

  • If there are recent updates, confirm details and ask practice to retransmit claim
  • If there are no recent updates, advise customer that Provider:
    • completes Banking Details Online Claiming form (HW052), or
    • nominate a different payee provider

The Resources page includes a link to the HW052 form.

0109 HEC

Payee provider is registered on PDS but not at this site ID + Read more ...

Check PDS for any recent updates

  • If there are recent updates, confirm details and ask practice to retransmit claim
  • If there are no recent updates, advise customer that Provider:
    • completes Banking Details Online Claiming form (HW052), or
    • nominate a different payee provider

The Resources page includes a link to the HW052 form.

0202 HEC

Service provider is not on file + Read more ...

Check:

  • PDS to confirm when provider number was created
  • QPLI to see when the last update was made

Note: mainframe takes 24 hours to update.

0203 HEC

Service provider practice was closed more than 30 days prior to the Payee provider's claim date for the current claim + Read more ...

Servicing Provider to:

  • send the claims manually or contact Provider Registrations on 1300 302 122 to extend their provider location number
  • for manual claiming refer to 132 150

0205 HEC

The payee provider is closed at date of transmission + Read more ...

Payee provider to:

  • send the claims manually or contact Provider Registrations on 1300 302 122 to discuss further, or
  • nominate a different payee provider

0502 HEC

Provider's claim date is in the future + Read more ...

Practice staff to:

  • contact their software vendor to reset the dates within the claim header

0503 HEC

Provider's claim date is too far in the past + Read more ...

Practice staff to:

  • contact their software vendor to reset the dates within the claim header

8006 HEC

Duplicated claim - previously registered through Electronic Data Interchange (EDI) + Read more ...

eBusiness Service Centre to:

  • confirm claim with same claim number has been PAID previously

9016 HEC

Transmission date is in the future + Read more ...

Practice staff to:

  • contact their software vendor to reset the dates within the claim header

9059 HEC

Transmission date is too old (more than 7 days) + Read more ...

Practice staff to:

  • contact their software vendor to reset the dates within the claim header
