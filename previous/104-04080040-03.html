Assessing a statistical request made Services Australia

This table describes the process for making a statistical request and the assessment process completed by the Internal and External Gateway Section.

Step

Action

1

Outside organisation requesting information + Read more ...

If an outside organisation wants to request statistical information, they need to send their request to statistics@servicesaustralia.gov.au

Note: requests must be made via email to allow a trail of correspondence to be forwarded to other agencies or government departments as needed.

2

Registration + Read more ...

The request is registered and a unique reference number issued (for example: MI2222).

3

Initial review of request + Read more ...

The Internal and External Gateway Section will assess the request and make a decision whether the request can be responded to immediately or if further information is required.

If further information is required, go to step 4.

4

Documentation sent to organisation for completion + Read more ...

The appropriate template is sent to the requestor to be complete.

When completed template is returned, go to Step 5.

5

Health, Centrelink and Child Support Program Requests + Read more ...

If the request relates to Health statistics it will need to be assessed by the External Request Evaluation Committee (EREC) who ensure the provision of health information is in accordance with legislation and privacy principles.

The Internal and External Gateway Section will work with Information Service Branch to progress the request to EREC.

  • Once clearance is received, go to Step 6
  • If the request relates to Centrelink statistics, the requestor is referred to data.gov.au. If the data that is required is unavailable, the Internal and External Gateway Section will provide assistance
  • If the request relates to Child Support Program statistics, the Internal and External Gateway Section will provide assistance

6

Quote + Read more ...

The Information Services Branch will contact the organisation to discuss the quote.

7

Extraction + Read more ...

When the quote has been accepted, extraction of data will begin.

8

Clearances + Read more ...

When extraction of data is completed clearances may need to be obtained from all relevant Policy Departments and/or business areas.

For current processes, refer to the Internal and External Gateway Section.

9

Completing the request + Read more ...

Once clearances have been obtained, data is sent to the requestor via email, registered post or physical drop off, in a secure manner.

10

Invoicing + Read more ...

The Internal and External Gateway Section will prepare the invoice and send it through to CFO.SUPPORT.HCI@servicesaustralia.gov.au for processing.
