Determine if the location or a satellite clinic of the location is listed on a 19(2) or 19(5) Direction

Table 2

Step

Action

1

Checking 19(2) and 19(5) Directions + Read more ...

  • Open application allocated via PaNDA
  • Select the provider stem hyperlink at the top of the stem page to open a list of provider numbers issued to the health professional
  • Check the practice address on the application
  • Go to Resources page and select the link for the COAG 19(2) Directions and/or open the current Indigenous and South Australia Health 19(2) or 19(5) Directions
  • Check the practice name and address on the application against the 19(2) or 19(5) Directions noting the address may not be on the Direction
  • Where the practice name and address on the application matches a location listed on a 19(2) or 19(5) Direction:
    • make a note of which Direction the location is on. Go to Step 2
  • Where the practice name on the application matches a practice name on a 19(2) or 19(5) Direction:
    • this practice is considered a satellite clinic and falls under a 19(2) or 19(5) Direction. Make a note of which Direction the practice name is on. Go to Step 2
  • Where the practice name on the application does not match practice name on a 19(2) or 19(5) Direction:
    • Check the Registered business name and Account held in the name(s) of (under Bank account details) listed on the application. If one of these names match a practice name on a 19(2) or 19(5) Direction, this practice is considered a satellite clinic and falls under a 19(2) or 19(5) Direction. Make a note of which Direction the practice name is on. Go to Step 2
    • Check the Registered business name and Account held in the name(s) of (under Bank account details) listed on the application. If both names do not match a practice name on a 19(2) or 19(5) Direction, conduct a Google search on the practice name on the application
    • Where the website clearly indicates the practice is associated with a practice listed on a 19(2) or 19(5) Direction, this practice is considered a satellite clinic and falls under a 19(2) or 19(5) Direction. Make a note of which Direction the practice name is on. Go to Step 2
    • If website does not indicate the practice name on the application is a satellite clinic of a practice listed on a 19(2) or 19(5) Direction, the location does not fall under a section 19(2) or 19(5) Direction. Continue to process according to the appropriate processing Operational Blueprint page

2

Check health profession on 19(2) or 19(5) Direction + Read more ...

Check the 19(2) or 19(5) Direction to determine if the health professional identified in Table 1 is listed on the 19(2) or 19(5) Direction.

Where the health professional’s profession:

  • is listed on the 19(2) or 19(5) Direction, go to Table 3
  • not listed on the 19(2) or 19(5) Direction, contact a Programme Support Officer (PSO)

Step

Action

1

Direction location falls under + Read more ...

Which Direction is the location listed on or is a satellite clinic of a location listed on?

  • Indigenous Direction, go to Step 2
  • South Australia Health Direction, go to Step 3
  • COAG Direction, go to Step 4

2

Flagging location listed on an Indigenous Direction + Read more ...

  • Select Amend against the provider number listed on the provider number application
    • Select Assoc Details tab
    • Select ATSIHS in Location Type
    • Select Add
    • Select AMS in Location Type
    • Select Add
    • Select Save
  • Repeat if additional locations are requested that are listed on an Indigenous Direction
  • Go to Step 3 if additional locations are requested that are listed on the South Australia Health Direction

3

Flagging location listed on the South Australia Health Direction + Read more ...

  • Select Amend against the provider number listed on the provider number application
    • Select Assoc Details tab
    • Select AMS in Location Type
    • Select Add
    • Select Save
  • Repeat if additional locations are requested that are listed on the South Australia Health Direction
  • Go to Step 4 if additional locations are requested that are listed on a COAG Direction

4

Flagging location listed on a COAG Direction + Read more ...

  • Select Amend against the provider number listed on the provider number application
    • Select Assoc Details tab
    • Select AMS in Location Type
    • Select Add
    • Select Save
  • Repeat if additional locations are requested that are listed on a COAG Direction
