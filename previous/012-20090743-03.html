Assessing business structure and EFT details

Table 1: this table describes how to assess if a health professional needs to provide business structure and/or bank account details in line with the level of access they have requested.

Step

Action

1

Check the application + Read more ...

Open the application via current workflow procedures. Make sure the application is an:

  • Application for a Medicare provider number and, or prescriber number for allied health and non- medical health professionals (HW093), or
  • Application for a Medicare provider number and/or PBS prescriber number for a midwife or nurse practitioner form (HW088), or
  • Application for a Medicare provider number and, or prescriber number for a medical practitioner (HW019), and/or
  • approved 3GA placement

Check if the application is complete. Is the application complete?

  • Yes, go to Step 2
  • No, continue to assess the application and go to Step 10

2

Check Provider Directory System (PDS) + Read more ...

Open PDS:

  • Select Provider
  • Select View
  • Select Detailed Search
  • In the Family name field enter the first 3 letters of the last name followed by a %
  • In the Given name field enter the first 3 letters of the first name followed by a %
  • Enter date of birth in Date of Birth field
  • Select Profession
  • Select Search. A list of health professionals with a matching name displays
  • Select View against the health professional that matches the application

3

Identify speciality codes and eligibility for Medicare benefits + Read more ...

Check PDS for the speciality codes located at the bottom of the health professional’s stem page.

If the health professional is:

  • an allied health, non-medical health professional, go to Step 4
  • a nurse practitioner or midwife, go to Step 5
  • a medical practitioner subject to section 19AB of the Health Insurance Act 1973 (the Act), go to Step 6
  • a medical practitioner subject to section 19AA only and is on an approved 3GA placement, go to Step 7
  • a medical practitioner with access to Medicare benefits at stem level, go to Step 8
  • a medical practitioner with refer, request and assist at private operations only access, go to Step 9

4

Allied health or non-medical health professional + Read more ...

Check the form provided is Application for a Medicare provider number and, or prescriber number for allied health and non-medical health professionals (HW093). If an incorrect form is used, go to Step 10.

Check application for answer to ‘Are you providing services that are Medicare benefit eligible?’

Applicant has ticked ‘Yes’

Are business structure and banking details provided in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, banking details are not provided in full
    • Service rule the location, and issue provider number for no claiming
  • No, only business structure details are not provided in full
    • Make 1 attempt to call the health professional to get the business structure. See Perform telephone security check for Medicare health professionals for more details
    • If staff get the details over the telephone, issue a provider number for access to Medicare benefits
    • If staff do not get the details over the telephone, service rule the location and issue a provider number for no claiming

Are business structure and banking details provided in full?

  • Yes, issue a provider number for access to Medicare benefits
  • No, service rule the location and issue provider number for no claiming

No boxes ticked

Are business structure and banking details provided in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, service rule the location, and issue provider number for no claiming

Both boxes ticked

Are business structure and banking details provided in full?

  • No, only business structure details are not provided in full

Nurse practitioner or midwife + Read more ...

Check the form provided is Application for a Medicare provider number and/or PBS prescriber number for a midwife or nurse practitioner form (HW088). If the incorrect form is used, go to Step 10.

Note: check processing notes in PaNDA to confirm the level of access approved at the location.

Location is approved with ‘full access to Medicare benefits’

Are business structure and banking details provided in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, banking details are incomplete:
    • Service rule the location, and
    • Issue provider number for refer and request only
  • No, only business structure is incomplete:
    • Make 1 attempt to call the health professional to get the business structure. See Perform telephone security check for Medicare health professionals for more details
    • If staff get the details over the telephone, issue a provider number for access to Medicare benefits
    • If staff do not get the details over the telephone, service rule the location and issue a provider number for no claiming

Location is approved for ‘refer and request only’

Are business structure and banking details provided in full?

  • Yes, enter:
    • business structure
    • banking details
    • service rule the location, and
    • issue provider number for refer and request only
  • No, service rule the location, and
    • issue a provider number for refer and request only

Medical practitioner subject to section 19AB of the Act

Check the form provided is Application for a Medicare provider number and, or prescriber number for a medical practitioner (HW019).

If the incorrect form is used, go to Step 10.

Check application for answer to 'Which one of the following do you want to do at this location'.

Refer and request only + Read more ...

Are business structure and banking details supplied in full?

  • Yes, enter business structure and banking details,
    • service rule the location, and
    • issue provider number for refer and request only
  • No, service rule the location and issue provider number for refer and request only

Refer, request and claim Medicare or Department of Veterans’ Affairs (DVA) rebateable services + Read more ...

Are business structure and banking details supplied in full?

  • Yes, determine eligibility to Section 19AB, see Provider number applications for medical practitioners and their requirements under the Health Insurance Act 1973. If medical practitioner satisfies Section 19AA and/or 19AB, issue provider number for refer, request and Medicare rebateable services
  • No, only business structure is incomplete:
    • Make 1 attempt to call the medical practitioner to get the business structure. See Perform telephone security check for Medicare health professionals for steps to perform a security check when calling to get details over the telephone:
    • If staff get details over the telephone, determine eligibility to section 19AB. If the medical practitioner satisfies section 19AB, when issuing a provider number, issue for refer, request and Medicare rebateable services
    • If staff do not get details over the telephone, go to Step 10
  • No, banking details are incomplete:
    • Make 1 attempt to call the medical practitioner to get business structure and/or banking details in writing (complete HW019), to email Medicare tier 2 support within 24 hours. See Perform telephone security check for Medicare health professionals for steps to perform a security check when calling to get details over the telephone
    • If contact is successful, pend the work item until close of business the following working day to allow the medical practitioner to provide details. Determine eligibility to section 19AB once details are provided. If the medical practitioner meets section 19AA and/or 19AB, issue provider number for refer, request and Medicare rebateable services
    • If contact is unsuccessful or details not received in writing by close of business the following working day, go to Step 10

Note: if a section 19AB(3) exemption is issued by Department of Health and Aged Care, contact a Medicare Health Programme Support Officers (PSO). The PSO will email Provider Services Support to notify the Department of Health and Aged Care

Refer, request and assist at private operations only + Read more ...

Are business structure and banking details supplied in full?

  • Yes, determine eligibility against Section 19AB. If medical practitioner meets Section 19AB issue provider number for refer, request and assist at private operations only
  • No, only business structure is incomplete:
    • Make 1 attempt to call the medical practitioner to get the business structure. See Perform telephone security check for Medicare health professionals for steps to perform a security check when calling to get details over the telephone:
    • If staff get details over the telephone, determine eligibility to section 19AB. If the medical practitioner satisfies section 19AB, when issuing a provider number, issue for refer, request and assist at private operations only
    • If staff do not get details over the telephone, go to Step 10
  • No, banking details are incomplete:
    • Make 1 attempt to call the medical practitioner and advise to provide business structure and/or banking details in writing (complete HW019), to email Medicare tier 2 support within 24 hours. See Perform telephone security check for Medicare health professionals for steps to perform a security check when calling to get details over the telephone
    • If contact is successful, pend the work item until close of business the following working day to allow the medical practitioner to provide details. Determine eligibility to section 19AB once details are provided. If the medical practitioner meets section 19AB, issue provider number for refer, request and assist at private operations only
    • If contact is not successful or details are not received in writing by close of business the following working day, go to Step 10

Note: if a section 19AB(3) exemption is issued by Department of Health and Aged Care, contact a Medicare Health Programme Support Officers (PSO). The PSO will email Provider Services Support to notify the Department of Health and Aged Care

Apply Location Service Rule + Read more ...

A service rule is applied when a health professional does not satisfy business structure or 19AB legislative requirements for the location on the application form:

  • Allied health and non-medical health professionals will not be able to claim a Medicare benefit at this location (chiropractors, osteopaths, physiotherapists, and podiatrists are eligible to request a limited range of diagnostic and/or pathology services)
  • Medical practitioners will only be able to refer and/or request at this location

  • Go to Loc Service rules
  • In Service Rule Number field, key 618
  • In the Activity Code field, select Yes
  • Enter the Start Date of the location
  • Do not key an End Date
  • Select Add New
  • Select Cancel

If no business structure details are supplied, go to Step 3

If partial business structure details are supplied, go to Step 2

Note: where partial business structure details are supplied, enter these as per the application provided

2

Record location organisation details + Read more ...

On the Organisation tab

  • Select the Employment Type from the drop down menu as per the form:
    • Employee Salaried form field is the Employee - Payroll Provider field in PDS
    • Employer - Employer field in PDS is not used
  • Select Add under Organisation Details
  • Enter the ABN and select Search
  • Select relevant record:
    • If multiple ABN options return, select the matching record
    • If results don't match the form, select the first result returned
  • Enter Trading As details as per the Registered Business Name on the form
  • Select the Business Type from the drop down menu as per the form
    • Associateship and Government or Other Public Body fields are not used in PDS
  • Select Premises Type from the drop down menu as per the form

Step

Action

1

Check provider details + Read more ...

  • access PDS
  • Select Provider
  • Select View
  • Select Detailed Search
  • In the Family name field enter the first 3 letters of the last name followed by a %
  • In the Given name field enter the first 3 letters of the first name followed by a %
  • Enter date of birth in Date of Birth field
  • Select Profession

  • assess the search results and confirm the applicant’s details on application match PDS
  • select View on required provider and check details on application match PDS

  • access PDS
  • select Program Registration

  • select View on required provider and check details on application match PDS

  • go to Step 2

2

Check practice details and access EFT registration in PDS + Read more ...

  • In the provider location screen, select Program Registration button at the bottom of the page
  • At the end of the MCARE row in the table, select the View hyperlink in Action column
  • Go to Step 3

3

Account matching EFT details in the form + Read more ...

  • Select the matching bank account information
  • In the Payment Instruction Start Date either type a ‘T’ for today or use the calendar to enter the date of processing, even if the provider number does not open until a future date
  • The end date must be left blank
  • Select Save
  • Bank details will now be registered to the provider number, repeat the appropriate Steps if more than 1 provider number requires banking details to be registered
  • Go to Step 5

4

No Account matching EFT details in the form + Read more ...

  • Select Create New Bank Detail button
  • Key the BSB number, Account Number and Account Name as specified on the form in the appropriate fields in the dialogue box
  • The start date must be date of processing (this date will auto-populate)
  • The end date must be left blank
  • Check the details keyed to ensure accuracy
  • Select Create button
  • Select the drop down menu next to Account in current payment instruction and select the bank account information keyed
  • In the Payment Instruction Start Date either type a ‘T’ for today or use the calendar to enter the date of processing, even if the provider number does not open until a future date
  • do not indicate anything within the Hold tick box
