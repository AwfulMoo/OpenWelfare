This page contains information about the cancellation, suspension and rejection codes for Assistance for Isolated Children (AIC) Scheme.

Code descriptions

Cancellation, suspension and rejection codes for Assistance for Isolated Children (AIC)

This table describes information about the cancellation (Can), suspension (Sus) and rejection (Rej) codes for AIC, a description of the code and what action can be taken using that code. It also gives information on when the system will issue a letter or the staff member needs to issue one manually, and notes on the action required to complete the Can/Sus/Rej and the letter type required.

The cancellation, suspension and rejection of AIC payments can only be done by AIC Smart Centre staff.
