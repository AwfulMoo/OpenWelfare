Multiple same day consultation restrictions

Restricting item is a standard consultation item

Table 1: This table describes how to process multiple same day consultation claims.

Item

Description

1

Services performed before 12 February 2016 + Read more ...

Is evidence of separate attendance provided?

  • Yes, pay with PI 42
  • No, check Prior Approval and Referral System (PARS). When prior approval:
    • has been provided pay with PI 42
    • has not been provided, reject the second and subsequent services on the same date of service (DOS) with reason (RSN) code S257 - Service already paid - no separate attendance evident on claim

2

Services performed after 12 February 2016 + Read more ...

When processing claims where more than 1 consultation is claimed for the same patient on the same day, check the Prior Approval and Referral System (PARS) for prior approval. Has prior approval been provided?

  • Yes, pay with PI 42
  • No, reject the second and subsequent services on the same date of service with reason (RSN) code 159 - Item associated with other service which is payable.

Note: 2 consultations cannot be claimed on the same day, for the same patient, except when claiming physiotherapy services PH23 and PH27 where treatment, not exceeding 5 days, is required for a short term acute onset condition. When claiming, providers should indicate AM or PM next to each consultation or the time of each consultation.
