Searching for a DVA file number using VNSI

Table 1

Step

Action

1

Navigate to the VNSC screen + Read more ...

Key 'VNSI' and press [Enter] and the VNSC screen is displayed.

2

Perform an exact name search + Read more ...

Key:

  • veteran's Surname and First Name (mandatory)
  • Age (an age range can be keyed)
  • '0' in the Postcode field (Australia wide search)
  • Sex
  • Birth-Date (any date combination can be keyed)
  • 'e' for exact in the Surname Match and First Name Match fields

Press [Enter].

3

Check search result + Read more ...

If the search result displays:

  • only 1 matched patient, go to Step 6
  • more than 1 matched patient, go to Step 4
  • no matched patient, go to Step 5

4

Limit the search + Read more ...

Limit the search by keying either:

  • Sex (M or F)
  • Birth-Date
  • Postcode range:
    • 0000 – 9999: Australia wide
    • 2000 – 2999: NSW
    • 3000 – 3999: VIC
    • 4000 – 4999: QLD
    • 5000 – 5999: SA/NT
    • 6000 – 6999: WA
    • 7000 – 7999: TAS

Is the patient found?

  • Yes, go to Step 6
  • No, go to Step 7

5

Expand the search + Read more ...

Perform a nationwide search:

  • Key '0' in the Postcode field. Dash out all other numbers (for example 0--- ---- ) and press [Enter]
    Note: Can only be used when complete surname and first name are used
  • Change the Surname Match and First Name Match fields to 'P' for possible match and press [Enter]
  • Change the First Name field to the veteran's first name initial only. Ensure 'P' is in the First Name Match field and search by postcode or state. Press [Enter]
  • If no first name provided, search by surname. Filter search further with exact postcode or date of birth (can search with just the day, month or year of birth). Press [Enter]

Is the patient found?

  • Yes, go to Step 6
  • No, go to Step 7

6

Select the patient + Read more ...

Select the patient by keying:

  • 'S' in the SEL field
  • 'VEXI' in the TRAN field

Press [Enter].

Note:

  • Check that at least 2 of the search criteria match the search result. For example, veteran's name and birth date or veteran's name and address
  • If there are more than 5 matched patients, press [PF8]-Forward to display the next 5 and so on
  • If DVA file number found then use reason code 249 when finalising the claim

7

Unable to locate correct patient + Read more ...

If unable to locate the correct patient reject the service with RSN code 376 - Patient cannot be identified from information supplied.

Note: For trainee Service Officers or during periods of high quality control errors, a second Service Officer must:

  • perform a VNSI name search
  • confirm the patient cannot be identified
  • for Manual Claims only - key appropriate comments under processing notes in PaNDA with their details and reason code 376

How to determine eligibility 2-NIL ELIG

Table 3

Step

Action

1

Check file number + Read more ...

If another file number is displayed at the bottom of the pend processing screen, process the claim using this new file number.

  • Key P at the end of the control line and press [Enter] twice
  • Overkey the FILE No field with the new file number and press [Enter]
  • Place the cursor on the correct patient and press PF5
  • Apply Reason (RSN) code 249 and continue assessing

Check VHOI

Press Page Up key to swap screens. On the second screen key 'VHOI,Veterans File no' and press [Enter]. Press PF10 to determine if a different, eligible card number has been used to process previous services. If an eligible card is present, process the claim using this new file number.

  • Key P at the end of the control line and press [Enter] twice
  • Overkey the FILE No field with the new file number and press [Enter]
  • Place the cursor on the correct patient and press PF5
  • Apply Reason (RSN) code 249 and continue assessing

If unable to locate an active file number, go to Step 2.

2

Check PARS + Read more ...

Check Prior Approval and Referral System (PARS) for an approval.

  • If PARS is present and payment is:
    • approved for item claimed, pay with PI 3L or 3X to force match to approval
    • not approved for Item claimed, reject the services with RSN code 289
  • If PARS is not present, go to Step 3.

3

Check VEXI + Read more ...

Check VEXI for additional details.

Key 'VEXI,Veterans File no' and press [Enter].

If the card type is a Personal Treatment Entitlement Card (PTEC) or Specified Treatment Entitlement Card (STEC) and:

  • the veteran is under 16 years of age, pay with PI 3L
  • the address is overseas, the postcode is listed as 0099 or the patient is over 16 years of age, complete an Eligibility query form and refer to the Programme Support Officer (PSO)
  • if no card type is present and a date of death is recorded, complete an Eligibility query form and refer to the VAP Programme Support Officer (PSO)

The Resources page contains a link to the Eligibility query form

If the above criteria are not met, and PSO are unable to provide further information, go to Step 4.

4

Reject claim + Read more ...

If the patient does not have any eligibility, reject all services with RSN code 211.

  • Pelvic Infection

  • Abdominal Pelvic Surgery
  • Prolapsed Invertebral Disc
    • Yes, pay with PI 3L

Item range MT53 – MT57 (ADF Firefighter scheme) + Read more ...

Are the items in the range MT53-MT57 for condition displayed in VEXI or VIEW?

  • TREATMENT ONLY – SCREENING FOR COLORECTAL CANCER AND MELANOMA<>
    • Yes, pay with 3L
    • No, reject with RSN 583

If items not in range MT53 – MT57, go to Step 7.

7

Determine patient entitlements + Read more ...

In the claim pend screen press PF10 to determine patient entitlements.

If see CDB for further disabilities is displayed, complete an Eligibility query form and send to the VAP Programme Support Officer (PSO).

Do the services relate to the patient’s entitlements?

Note: for dental, podiatry or optical check, go to Step 7 for further information.

  • Yes, pay with PI 3L
  • No, go to Step 8

Note: condition must be specific to area being treated e.g. Left or Right Knee. The only exception is neck and back conditions. A broad description of these disabilities is sufficient for payment of services.

For STEC card holder with TMN - DVA health card holders with White Card eligibility code TMN (treatment for malignant neoplasia) are entitled to payment for treatment of cancer regardless of the type of cancer. General wording of TMN or Cancer must be indicated, the provider does not need to specify which type of cancer (excludes Dental, Optical, Podiatry & Orthotist)

If the provider indicated:

  • TMN or Cancer check VEXI for TMN
    • TMN present, pay PI 3L
    • TMN not present, reject RSN 583

If the provider has left condition treated field blank, reject with RSN 581

If PAMT – Provisional Access Medical Treatment is displayed, the provider must indicate 1 or more of the 20 specific conditions approved for PAMT. For more information, see FAQs: PAMT – Provisional Access to Medical Treatment in Resources.

Has the provider indicated the PAMT condition treated and does it match 1 or more of the following 20 conditions listed below?

  • Yes, pay with 3L
  • No, Reject with RSN 581 (if no condition has been indicated) or RSN 583 (if condition stated does not match PAMT condition list)

Note: podiatry services must also indicate PAMT condition treated.

The 20 conditions include 16 musculoskeletal, 2 hearing and 2 skin conditions, and consist of the following:

  • Achilles Tendinopathy and Bursitis
  • Chondromalacia Patella
  • Cut, Stab, Abrasion and Laceration
  • Dislocation
  • Fracture
  • Internal Derangement of the Knee
  • Intervertebral Disc Prolapse
  • Joint Instability
  • Labral Tear
  • Lumbar Spondylosis
  • Non-melanotic Malignant Neoplasm of the Skin
  • Osteoarthritis
  • Plantar Fasciitis
  • Rotator Cuff Syndrome
  • Sensorineural Hearing Loss
  • Shin Splints
  • Solar Keratosis
  • Sprain and Strain
  • Thoracic Spondylosis
  • Tinnitus

SCI eligibility

If a white card holder has any of the below conditions as an accepted disability on VEXI:

  • Fracture dislocation of thoracic 4-5 vertebrae below t-12
  • CS Tetraplegia
  • T 4/5 complete paraplegia and oblique fracture through proximal end
  • Head injury with neurological deficit and a fracture of the thoracic spine with complete paraplegia
  • Incomplete paraplegia
  • Quadriplegia (tetraplegia)
  • Crush(es) of the 4th and 6th thoracic vertebrae with paraplegia

The veteran is covered for treatment of the conditions listed in the Manifestation list.

Manifestations covered for treatment:

  • Muscle paralysis or impaired motor function below the level of the injury
  • Sensory loss below the level of the injury
  • Bladder dysfunction/neurogenic bladder   - urinary retention, incontinence, incomplete emptying, urgency, frequency, need for catheterisation (indwelling or intermittent)
  • Bowel dysfunction/neurogenic bowel – constipation, abdominal distention, abdominal pain, faecal retention and impaction, faecal incontinence
  • Sexual dysfunction
  • Erectile dysfunction
  • Autonomic dysreflexia/hyperreflexia
  • Heterotopic ossification
  • Infertility (males only)
  • Megacolon
  • Muscle contractures
  • Muscle spasticity, spasms (for injuries at T12 and above)
  • Neurogenic pain
  • Pressure ulcers/sores, skin necrosis
  • Thermoregulatory dysfunction – inability to regulate body temperature leading to hyperthermia or hypothermia

Is a PARS record present?

  • Yes, and the payment is:
    • approved for item claimed, pay with PI 3L or 3X to force match to approval
    • not approved for item claimed, reject the services with reason (RSN) code 289
  • No, go to Step 9

Claim rejection + Read more ...

Press PF2 to return to the pend screen. Is the treated condition noted?

  • Yes, however if the condition noted does not match PF10 entitlement, reject claim with Reason (RSN) 583
  • No, reject claim with RSN 581

Note: a subsequent specialist consultation can still be paid, even when associated services are rejected with RSN 583.
