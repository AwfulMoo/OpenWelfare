Diagnostic imaging service types and processing indicators

Table 12

Service Type

Processing indicator

Self-determined R-type services

When an R-type diagnostic imaging service has been self-determined, a diagnostic imaging request (DI) line is not to be keyed for that service.

Processing indicator 11 (diagnostic imaging service has been self-determined) must be keyed in the PI field against each R-type item(s) that has been self-determined.

NR-type services

When an NR-type diagnostic imaging service has been claimed, a diagnostic imaging request (DI) line or processing indicator 11 (diagnostic imaging service has been self-determined) must not be keyed for that service.

If NR-type services are processed with a DI line or processing indicator 11 (diagnostic imaging service has been self-determined), the system will display an appropriate error message.
