SW01, SW02, SW04, SW06, SW11, SW13

Initial consultation – rooms, in-home, private hospital + Read more ...

Initial consultation - Rooms; In-home; Private Hospital – 1st Patient; Residential Aged Care Facility (RACF) – 1st Patient; Private Hospital – 2nd and Subsequent Patients; Residential Aged Care Facility (RACF) – 2nd and Subsequent Patients.

After 1 October 2019

Rules

  • Limit of 1 service in 14 days from any provider
  • Limit of 12 services in 365 days from any provider

Restrictions

If an initial consultation restricts with:

  • itself or another social worker initial consultation item, reject with RSN code S 160
  • any other item, reject the item with the lower fee with RSN code S 159

Before 1 October 2019

Rules

• only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient

Restrictions:

If item restricts with:

• itself or another initial consultation, reject with RSN code S 160

• any other item, reject item with lower fee with RSN code S 159

SW03, SW05, SW10, SW12

Initial consultation – public hospital, RACF + Read more ...

Initial consultation - Public Hospital – 1st Patient; High care RACF – 1st Patient; Public Hospital – 2nd and Subsequent Patients; High Care RACF – 2nd and Subsequent Patients.

Services Australia will only pay for health care services carried out in Public Hospitals and high care Residential Aged Care Facilities in exceptional circumstances and only when DVA has given prior financial authorisation.

Rules

• initial consultation limit of 2 in 365 days

• only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient

Requires prior financial authorisation.

Check PARS for an approval.

When prior financial authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283

Social Worker Treatment Cycle Report

Table 3:

Item number

Description and resolution

SW90

Allied Treatment Cycle Report + Read more ...

Provision of report to GP on outcomes of Treatment Cycle related to the referral.

Rules

  • An initial consultation item in the allied fee schedule is claimed
  • Any number from 1 to 11 subsequent consultation items from the same fee schedule have been claimed
  • Only 1 report item is payable for each initial consultation
  • The provider must have claimed 1 initial and 1 subsequent consultation
  • Cannot be paid twice on the same day
  • Limit of 1 report item payable within 14 days
  • Limit of 12 report items payable within 12 months

Restrictions

  • Associated consult not claimed, reject with RSN code S 748
  • If the item restricts with itself or any other item, reject item. Use lower fee with RSN code S 159

Social Worker direct supply to DVA item

Table 4:

Item number

Description and resolution

SW99

Social Worker – report or service specifically requested by DVA + Read more ...

Rules

Providers may use item number SW99 only when DVA contacts them directly to request that they provide a:

  • written report; or
  • consultation or assessment to eligible veterans or war widows/ers, either separately or in conjunction with a written report

For example, this may occur when DVA requires a second opinion concerning treatment for a veteran. DVA will give financial authorisation and advise the fee at the time of the request, according to the relevant schedule items. The kilometre allowance is included in the fee, and is not to be claimed in addition to the fee.

Note: this item does not cover the supply of clinical notes, care plans or other information requested by DVA as part of monitoring activities, as these are provided free-of-charge under DVA requirements.

Requires prior financial authorisation:

Check PARS for an approval.

When prior financial authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X.
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283
