Submitting an application for a 19AB(3) exemption to Health

Table 1: This process commences after the application has been assessed under 19AB class exemptions for medical practitioners in Medicare and Provider number applications for medical practitioners and their requirements under the Health Insurance Act 1973.

Step

Action

1

Check Supporting documents + Read more ...

Does the application include:

  • A complete Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)
  • A copy of a current Visa for a temporary resident medical practitioner, or
  • Permanent resident details recorded in PDS, or a copy of a permanent resident visa for a permanent resident practitioner
  • A letter of support from the employer, stating:
    • medical practitioner’s name
    • the term and location(s) of employment, and
    • why provider number with access to Medicare benefits is required
  • Australian Health Practitioner Regulation Agency (Ahpra) registration for any conditions or limitations

For temporary resident medical practitioners requesting assisting at operations, make sure the following documents are provided:

  • A complete and signed Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) with related business structure and banking details
  • Copies of pages from their passport including personal details and Australian visa
  • Evidence of current registration with the medical board. (if this is their first application for a provider number, or they hold limited registration)
  • Letter of support from their employer or the hospital stating:
    • the reason they require access to Medicare benefits for assisting at operations and the employment period, or
    • a copy of their employment contract, which references assisting at private operations as part of their employment
  • If the visa is not a 407 and the letter of support is not from the hospital, then one of the following documents is also required:
    • letter of support from the hospital, or
    • training schedule, or
    • medical registration that states the medical practitioner is training on the ‘competent authority’ pathway

Note: Applications including 3GA placements do not require a letter of support.

  • Yes, go to Step 2
  • No, go to Step 3

2

Check PaNDA for duplicate application + Read more ...

Make sure there are no existing applications for the medical practitioner for the same location(s) requested before submitting a new application to Health for a 19AB(3) exemption.

Is there a duplicate application for the medical practitioner for the same location in PaNDA?

  • Yes, go to Step 4
  • No, go to Step 5

3

Missing Documents + Read more ...

Make one attempt to call the medical practitioner to get missing documents.

If contact is unsuccessful, return using incomplete letter template Z2306 or Z2305 and include a copy of the submitted documents:

  • Complete appropriate fields of letter template
  • Print letter
  • Save letter as a PDF
  • Upload letter to PaNDA
  • In PaNDA, ensure the Surname First Name, Provider Number and Case ID fields are completed and correct
  • Add any appropriate notes to the PaNDA file
  • Select Complete
  • Select Request Rejected in the Complete Reason field
  • Select Save
  • Send letter

Procedure ends here.

4

Check other work item(s) + Read more ...

Has the duplicate work item been actioned, i.e. already sent to Health?

  • Yes, add a note in PaNDA to your work item, ‘NFA – Duplicate see work item ID…’. Procedure ends here
  • No, close the duplicate work item off as already completed. Go to Step 5

5

Check PaNDA categorisation of current application + Read more ...

Make sure the current PaNDA work item has been re-categorised to the work type of Health and Save.

6

Open the spreadsheet + Read more ...

  • Open 19AB Spreadsheet template from 26062019 located on the Resources tab
  • Save 19AB Spreadsheet template from 26062019 to your team’s folder under 19AB_Spreadsheet SUBMISSION
  • The spreadsheet is required to be named YYYYMMDD.19AB.Service Officer’s first name. For example, 20201201.19AB.JANE
  • Save the Application for a Provider number (HW019) and supporting documents to the teams folder under the medical practitioner’s SURNAME, FIRST NAME DDMMYYYY (DDMMYYYY is date of assessment)

Note: for supporting documents reduce the file size of the PDF’s before saving.

The date is the date the application is sent to Health.

7

Complete the spreadsheet + Read more ...

  • The 19AB Spreadsheet is set out into different sections including:
    • Applicant Details
    • Preferred Contact Details
    • Practice Details
    • Visa Information
    • Medical Registration
    • Qualification
    • 3GA Placement
    • Assistance at Operations
    • Spousal
    • AGPTP
    • Comments
  • The field requirements are also provided under the individual sections. Enter the data required as set out in Tables 2 – 12
  • Do not enter additional spaces or lines before or after the text where not required
  • If a medical practitioner is applying for multiple practices with the same requested start and end dates:
    • all mandatory fields are required to be completed for the first practice, and
    • columns K to R, AE (if applicable) to AH and Comments (if applicable) are required to be completed for the subsequent locations
  • These locations will upload as a single application within Health’s system
  • If the applicant is requesting several locations with different start and end dates, all mandatory fields are required for each practice the medical practitioner is applying for, as each location will upload as a separate application on Health’s system

8

Action supporting document requests + Read more ...

There can be instances where supporting documentation provided will indicate action is required on another medical practitioner’s record. For example where a replacement exemption has been requested, a statutory declaration from the medical practitioner being replaced may have been provided, with a request to close their provider number.

Is there any action required on another provider number as a result of this application?

  • Yes, go to Step 9
  • No, go to Step 10

9

Action work item + Read more ...

  • Extract document from current work item ID
  • Create new PaNDA work item
  • Add notes to both PaNDA files stating extracted document see work item ID or extracted from work item ID
  • Action the extracted documents file. For example: statutory declaration, closure request. See closing a provider number in Provider location details for health professionals in Medicare

10

Pend the work item + Read more ...

  • Make sure the Surname, First Name, Provider Number and Case ID fields are completed and correct
  • Add notes to PaNDA. For example: entered on DoH s/s dated (date sent to Health)
  • Select Status Pended
  • Select pend reason Department of Health Deallocate
  • Save or Save and Next

Processing the decision of request of a section 19AB(3) exemption from Health

Table 13: This process starts when a Service Officer is allocated a work item in PaNDA.

Step

Action

1

Review application and section 19AB(3) exemption + Read more ...

  • Open all documents under the PaNDA work item – the approval or decline notice will be attached. A decline notice will detail the reasons for the decline
  • Check the section 19AB(3) exemption against the documentation received
  • Does the section 19AB(3) exemption match the documentation:
    • Yes, go to Step 2
    • No, contact a Programme Support Officer (PSO)

2

Open Provider Directory System (PDS) + Read more ...

Open PDS and Select:

  • Provider
  • View
  • Detailed search
  • Type the practitioner’s surname in the appropriate field with a % sign either side (this is a wild card function)
  • Type the practitioner’s first name in the applicable field with a % sign at the end of the name
  • Select Profession
  • Select Search

Assess the search results returned:

  • Select View against the matching medical practitioner, or
  • note if no match is found

3

Health decision + Read more ...

What is the decision of request of a section 19AB(3) exemption from Health:

  • 19AB(3) exemption is declined, go to Step 4
  • 19AB(3) exemption is approved, go to Step 5

4

Processing a declined section 19AB(3) exemption + Read more ...

If a match (existing stem) was found in Step 2, in PDS:

  • Select Edit
  • Select Notes
  • Select New Note
  • In Notes Subject, key ‘19AB declined – (locality)’
  • In Notes Description, key ‘19AB declined for (full practice name and address), signed by (DoH delegate) on (date declined). For example, 19AB declined for Darwin Medical, Darwin NT on 12/12/2020 signed by Jane Bloggs
  • Select Add New
  • Select Save
  • Continue to Complete 19AB Exemption declined letter

If no match (existing stem) was found in Step 2

  • Complete appropriate fields of the 19AB Exemption declined letter Z2716
  • Save letter as a PDF
  • Upload letter to PaNDA
  • In PaNDA, make sure the Surname First Name, Provider Number and Case ID fields are completed and correct
  • Print the declined letter, then send to the medical practitioner
  • Add any appropriate notes to the PaNDA file
  • Select Complete
  • Select Request Rejected in the Complete Reason field
  • Select Save

Procedure ends here.

5

Update stem + Read more ...

  • If there is no stem recorded, or if current stem details need updating as per the submitted application form see Processing provider number applications for overseas trained or temporary resident medical practitioners Table 2 Process an initial stem for refer and request only for an OTD or TRD, then return here
  • If there is an open 181 specialty code on stem:
    • Check if the 19AB exemption has been issued as a General Practitioner, close the 181 code the day before the 19AB exemption start date and add a 180 specialty code with the start date matching the 19AB exemption start date
    • If the 19AB exemption has been issued as a Specialist or for assisting at operations do not close the 181 code or add a 180 code
  • If there is a 110 specialty code on stem:
    • and the 500 specialty code end date is prior to the date of permanent residency, make sure the end date of the 519 specialty code is the day before permanent residency was granted
    • and the 500 specialty code end date is after the date permanent residency, make sure the end date of the 519 specialty code is in line with the 500 specialty code end date
  • If the section 19AB(3) exemption issued is for a temporary resident assistance at operations and specialty code 505 is not present on the stem:
    • Specialty tab
    • Select Add Specialty Code, key 505 in Specialty Code Description
    • key the date the exemption was granted from in Start Date
    • no End Date is required to be keyed
    • Select Add New
    • Select Cancel
    • Select Save

  • Select the Provider’s stem number (orange hyperlink) - existing locations will display.

Note: if the medical practitioner is a permanent resident or Australian citizen, make a note of the last day the medical practitioner is subject to section 19AB (500 specialty code).

  • Check for an existing provider number for the requested practice/hospital name or street address. For example, Prince Charles Hospital Rode Rd and Prince Charles Hospital 627 Rode Rd are considered a match
    • If a match exists go to Step 6
    • If a match for practice/hospital name or street address is found see Issuing a second provider number or contact a Programme Support Officer (PSO) for help
    • If no provider number exists for the requested practice/hospital or the requested street address continue assessment, go to Step 11

6

Update existing provider number – contact details and flags + Read more ...

Select Amend against the provider number to be updated.

Address tab:

  • Update address details where additional information has been supplied on the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)

Contact tab:

  • Update telephone, mobile, fax and email fields as per the application
  • Start Date – no amendment is required
  • End date:
    • If there is an end date on the exemption, key the exemption end date
    • If there no end date on the exemption, and there is an end date on the HW019, key the end date from the form
    • If there no end date on the exemption and the provider number application form, no end date is required

Assoc Details tab:

See Section 19(2) and 19(5) Directions for links to Directions.

If Section 19(2) and 19(5) directions is listed select:

  • ATSIHS and AMS in Location Type for locations on the 19(2) and 19(5) directions
  • AMS in Location Type for locations on COAG direction
  • Select Add

If the location is not listed no action required.

7

Loc-Spec Codes tab: + Read more ...

Add or update 519 specialty code.

Has the medical practitioner previously had a class exemption or a section 19AB(3) exemption granted to the location:

  • Yes, and there is a gap in the medical practitioner satisfying section 19AB:
    • make a note of the period the medical practitioner did not satisfy section 19AB
    • Select Add Specialty Code
    • Key 519 in Specialty Code Descriptor
    • Start Date – key the date the exemption was granted from
    • End Date: If the exemption end date is within the period the medical practitioner is subject to section 19AB, key exemption end date, or if the exemption is granted beyond the period the medical practitioner is subject to section 19AB, key last day the medical practitioner is subject to section 19AB, or if the medical practitioner is a temporary resident and the exemption does not have an end date, no end date is required to be keyed
  • Yes, and there is no gap in the medical practitioner satisfying section 19AB:
    • Select Amend against the respective provider number
    • Specialty Code Descriptor – no amendments required
    • Start Date – no amendments required
    • End Date: if the exemption end date is within the period the medical practitioner is subject to section 19AB, key exemption end date, or if the exemption is granted beyond the period the medical practitioner is subject to section 19AB, key last day the medical practitioner is subject to section 19AB, or if the medical practitioner is a temporary resident and the exemption does not have an end date, no end date is required to be keyed
  • No:
    • Select Add Specialty Code
    • Key 519 in Specialty code
    • Start Date – key the date the exemption was granted from
    • End Date: if the exemption end date is within the period the medical practitioner is subject to section 19AB, key exemption end date, or if the exemption is granted beyond the period the medical practitioner is subject to section 19AB, key last day the medical practitioner is subject to section 19AB, or if the medical practitioner is a temporary resident and the exemption does not have an end date, no end date is required to be keyed
  • Select Add New
  • Select Cancel

When the section 19AB(3) exemption is issued as a Temporary resident assistance at private operations:

  • Key 505 in Specialty Code Descriptor
  • Start Date key the date the exemption was granted from
  • End Date key exemption end date

If an application for Other Medical Practitioners (OMPs) program is received or a 186 specialty code at the location that is open or expired within the previous 3 months see Other Medical Practitioners programs for Medicare. Process the OMPs and go to Step 8 below. Otherwise, go to Step 8

8

Update existing provider number - 3GA Placements + Read more ...

Placements tab:

If the section 19AB(3) exemption issued with a post 1 January 2019 Budget 3GA:

  • Select the Description field:
    • key an A if a ACRRM placement
    • key an M if a MDRAP placement, or
    • key an R if a RACGP placement
  • Select the appropriate placement description from the Specialty codes for MDRAP/ACCRM/RACGP tables in 3GA placements post 1 January 2019.

Note: the main ACCRM placement for the implementation of the new budget is the ACCRM Independent pathway. The code to be used for this placement is IA-IP-ACCRM (472 spec code).

  • Start Date – key the date the exemption was granted from, this cannot be earlier than the start date of the placement, if it is contact a Programme Support Officer (PSO)
  • Key the end date as per the exemption, if this does not match the placement end date contact a PSO
  • Select Add Placement
  • Check the placement appears correctly in the Placements at this Provider Location field

Does the placement appear correctly in the Placements at this Provider Location field?

  • Yes,
    • Select Save
    • Go to Step 9
  • No, continue below

If an error message is returned for overlapping placement dates, see Table 14 before starting the following:

  • Select the Description field:
    • key A for an ACRRM placement
    • key M for an MDRAP placement, or
    • key R for an RACGP placement
  • Select the appropriate placement description from the Specialty codes for MDRAP/ACCRM/RACGP tables in 3GA placements post 1 January 2019.

Note: The main ACCRM placement is the ACCRM Independent pathway. The code to be used for this placement is IA-IP-ACCRM (472 spec code).

  • Enter the Start Date as the day after the previous 3GA placement ended or date of lodgement of the 3GA placement with Services Australia, whichever is later
  • Enter the End Date as per the end date on the placement
  • Select Add Placement

Update existing provider number - Provider Service Rule (PSR) + Read more ...

Loc-Service Rules

  • Select Add Service Rule, the Add New pop up box will display
  • Enter Service Rule 618 in the Service Rule Number field
  • Select Yes in the Activity Code field
  • Start Date is the date after their previous exemption expired
  • End Date is the date before the new 19AB(3) exemption is approved
  • Select Add New
  • Select Cancel

10

Notification of removal from a 3GA placement may be in the form of:
