Assess MRI provider eligibility in Provider Directory System (PDS)

Table 1: This table describes how to check a provider’s details (location and speciality codes) in PDS prior to linking the MRI provider to an LSPN. This will enable MRI services to be provided at the same location as the eligible MRI. Service Officers will need to use this process to assess whether the provider can be added to the LSPN.

Step

Action

1

Search for provider + Read more ...

  • Open the Provider Directory System (PDS)
  • Select Provider from the side menu
  • Select View
  • Key provider stem in the Provider Stem field on the Basic Search tab
  • Select Search

2

Compare provider details + Read more ...

Check the name and provider number recorded in PDS matches the information provided on the LSPN form.

Note: some providers have more than one stem and these will be listed on the top of this screen.

3

Check registration and comments field + Read more ...

Check Provider's medical registration is valid (open within current date)

Note: if Ahpra has restrictions/notations noted on the provider’s medical registration, check the Ahpra website for details and determine if provider is eligible to provide services at the requested LSPN location/s.

  • Check the provider’s specialty codes for eligibility restrictions. Are restrictions present?
    • No restrictions, go to Step 5
    • Yes, provider is subject to 19AB (must have exemption, spec code 500 and 519), go to Step 4

4

Check speciality codes 1 + Read more ...

Check provider stem and location. Is the provider eligible to access Medicare benefits?

  • Yes, (provider has billing rights at the location/s) go to Step 5
  • No, return the application, advising that the provider could not be linked and that the provider will need to contact Medicare Provider Registration for more details
  • Note: Provider eligibility should not be discussed with LSPN practice staff.

5

Check specialty codes 2 + Read more ...

Is there is a current Radiology specialty code 046 or 047 at stem level?

  • Yes, proceed to Add provider on LSPN register in PDS
    • Take note of any end dates on the 046/047 as these will need to be used when adding the provider to the LSPN Register, so that the specification code 118 will also end the same dates
  • No, do not process. Return the application advising the provider they are not eligible for MRI services and they should contact Medicare Provider Registration eligibility requirements

Add provider on LSPN Register in PDS

Table 2: This table describes how to link a provider on the LSPN Register to the provider’s location record in PDS. This link will lock the address details in PDS so that changes cannot be made. See Unlink provider on LSPN register from PDS record table for a workaround.

Only link the provider once you have assessed MRI provider eligibility as per the MRI provider eligibility in Provider Directory System (PDS) table.

Step

Action

1

Access LSPN Register in PDS + Read more ...

  • Log into PDS
  • Select Organisations from the left hand side menu
  • Select Amend
  • Key the LSPN on the Basic Search facility
  • Select Search

LSPN record will display in Amend, ensure it is the correct LSPN that the MRI provider is being added.

2

Add provider details + Read more ...

Select the Associates tab:

  • Select the Add Associate button
  • Select MRI provider from the dropdown list
  • Key in the provider stem number and location indicator
  • Compare the LSPN Address to the Provider Location Address to ensure they match. That is, PDS and LSPN are referring to the same location
  • Key the date the application was received by Services Australia in the Start Date field or a future date if requested/ in line with provider location start date
    • Note: if the LSPN/provider requests backdating of MRI eligibility, the request will need to be referred to Provider Eligibility and Accreditation Section for consideration. Send the request via Team Leader to Provider Services Support
  • If there is an End date on the location in the Provider Directory System (PDS), key that date in the End date field. Otherwise leave the End Date field blank.
  • Select Add new
  • Repeat these steps for any additional providers
  • Select Cancel to exit the add associates modal
  • Select Save

3

Check MRI speciality code at stem level + Read more ...

Confirm there is a 118 code on the provider stem.

If applicable, add an end date to the 118 code to align with the end date on the 046/047 code.

Note: Speciality Code 118 is a location specific code and only pays Medicare benefits when recorded at a specific practice location.

4

Check MRI speciality code at practice level + Read more ...

Select the orange provider stem hyperlink located on the provider’s stem page in PDS.

The Provider Location Search screen displays with a list of provider locations.

From the list, select the view link that corresponds with the relevant provider number location indicator and/or address that matches the LSPN address.

Confirm there is a 118 code on the provider stem. Start dates should match what was keyed when linking the provider to the LSPN at Step 2.

If applicable, add an end date to the 118 code to align with the end date on the 046/047 code.

Note: if the provider requests backdating of MRI eligibility refer to Provider Eligibility and Accreditation Section for consideration. Send the request via Team Leader to Provider Services Support.

5

Send letter + Read more ...

Eligibility Approval letter to be sent to the provider:

  • Go to Standard Letter Templates
  • Select ‘Provider and third party Letters’
  • Select Diagnostic services
  • Select ‘Provider approval for MRI eligibility (z0752)’ and follow the macro prompts
  • Address and signatory:
    • The Letter is to be addressed to the Authorise Representative who signed the form and sent to the LSPN mailing address (postal address) listed in PDS
    • Enter the LSPN number
    • Enter the date they are eligible from
  • Letter Options:
    • In Diagnostic imaging practice name field – enter the Practice name
    • In provider number field – enter the providers stem (e.g, 123456)
    • In the Location field – enter the last 2 characters of the provider number (e.g. 5D)
    • In the Date Eligible from field – enter the date they are eligible from
    • Continue to add the new providers that have been linked
  • Post or email via HPOS only, and save a copy to the MRI application file in the Workload management tool (PaNDA)

If the form has been received via HPOS form upload, all correspondence is to be sent via HPOS.

Note: when the practice uploads the application form in HPOS they receive an automated message acknowledging receipt of the form.

The Resources page contains a link to the Standard Letter Templates page on the Intranet

Note: for ineligible machines link the providers but instead of sending the confirmation letter, send the general amendment confirmed email.

Confirmation of processing + Read more ...

Using the email address recorded for the diagnostic imaging practice on the LSPN record, email a confirmation of receipt. If no email address has been provided, no acknowledgment is required.

See the email template on Resources for the appropriate wording in the email. The email is required to be sent from the Medicare Provider Registration Tier 2 support positional email box.

Note: if the request has been received via HPOS form upload then correspondence is tobe delivered/sent via HPOS.

Requests received via HPOS form upload can be identified as the naming convention will contain the 10 digit RA number of the sender. Also, the HPOS Reference Number field will contain a 9 digit number that can be used to search in HPOS to reply. For example, Business name: 9987654321, and the HPOS Reference Number field: 123456789-1. For further instructions see Health Professional Online Services Support Mail Centre – Managing emails from health professionals.

Table 4. This table describes how to end date the 118 on provider stem and locations. This process is likely to be used as required.

Action

Access Provider Stem in PDS + Read more ...

  • Select Provider from the left hand side menu
  • Select Amend
  • Key the Provider Stem on the Basic Search facility
  • Select Search

  • Select the End At Locations link against the 118 specialty code

3

Note: this step is for national office staff only:

  • Once the 118 has been end dated at the location and stem, if applicable, proceed to end date the 046/047 specialty code.
  • When end dating a 046/047 check if the 655 needs to be end dated as well (in addition to the 118)

Unlink provider on LSPN Register from PDS Provider location to enable address update

If the provider location and LSPN location are an exact address match, PDS will lock the provider location address so that it cannot be amended. If the LSPN address is updated, the system will recognise that the provider location address is no longer an exact match and will unlock the address for amendment.

Ensure MRI details are correct – processed by National Office

Table 6: This table describes how to ensure MRI equipment details are correct. National office will use this process to determine whether an MRI can be added to an LSPN.

Step

Action

1

Check LSPN form details + Read more ...

Check that:

  • the details of the MRI have been completed on the relevant form
  • the application has been signed by the proprietor or authorised representative; and
  • practitioner details have been provided where MRI eligibility is required

Are the relevant sections complete?

  • No, contact the practice for clarification
  • Yes, go to Step 2

2

Access register + Read more ...

Log on to PDS:

  • Select Organisations from the side menu
  • Select View
  • Search for the LSPN

3

Compare address details + Read more ...

Are the address details listed on the application the same as the address recorded on the LSPN Register?

  • Yes, go to Step 4
  • No, contact the practice/authorised representative for clarification. Do not:
    • change any details on the LSPN Register, or
    • continue processing

4

Navigate to the modality accreditation screen + Read more ...

Select the Accreditation tab

Does the site have an MRI accreditation and is current for dates the MRI equipment is being recorded for?

  • Yes, go to Step 5
  • No, check the Accreditation Reports to see if an update has been missed/not actioned. If the report has not been actioned the register should be updated with the accreditation details
    • If no accreditation report contact the practice/authorised representative to advise that MRI accreditation is required for Medicare benefits to be paid. It’s possible that the site has MRI accreditation but Services Australia has not received the report

5

Check MRI details + Read more ...

Select the Equipment tab

Is an eligible, partial or mobile eligible MRI recorded at the site and is current (MR1,MR2,MR3)?

  • Yes, go to Step 7
  • No, contact the practice/authorised representative for clarification. Do not:
    • change any details on the LSPN Register, or
    • continue processing

6

Check for MRI providers + Read more ...

Select the Associates tab.

Have any providers already been registered at the LSPN?

  • Yes, check that provider dates are still current, update if necessary
  • No, go to MRI provider eligibility in PDS table.

Temporary transfer of MRI eligibility – processed by National Office

Table 7: This table describes how to action a temporary MRI transfer, this is processed by National Office staff. This table can be used when Services Australia receives email notification from Health that there has been a temporary MRI eligibility transfer.

Note: a form with the MRI details may be required if the information has not been recorded anywhere else on the LSPN register.

Step

Action

1

Check that the advice from the Department of Health and Aged Care is complete + Read more ...

Check that:

  • details of the MRI equipment (for both LSPN sites) are complete with serial number, make and model, date of manufacture or date first installed in Aust, LSPN address and start and end dates. An LSPN form is not required for short term eligibility transfers between sites unless MRI provider eligibility is required; and
  • details of the MRI equipment for the return of eligibility at the original site is also complete

Are details complete?

  • No, if details are not complete National Office will contact Health
  • Yes, details are complete, go to Step 2

Note: where new equipment is being installed along with the serial number, make and model, the manufacture date or first date installed in Australia also should be provided for capital sensitivity requirements.

2

Access register + Read more ...

Log on to PDS:

  • Select Organisations from the side menu
  • Select View
  • Search for the LSPN

3

Compare address details + Read more ...

Are the address details listed on the advice from Health the same as the address recorded on the LSPN Register?

  • Yes, go to Step 4
  • No, National Office will contact Health for clarification

Do not:

  • change any details on the LSPN Register, or
  • continue processing

4

Check MRI equipment details + Read more ...

Select the Equipment Tab and check eligible MRI is recorded at the site.

Is the MRI equipment and serial number listed on advice from Health currently recorded at the site?

  • Yes go to step 5
  • No, National Office will contact Health for clarification. Do not:
    • change any details on the LSPN Register, or
    • continue processing

5

Amend equipment details + Read more ...

Select the Amend link for the relevant MRI.

In the Date removed field, key in the date the eligibility is to cease, as provided by Health and select Add.

Is MRI eligibility to be transferred back to the same LSPN at a later date (same or different equipment)?

  • Yes, go to Step 6 to Add equipment
  • No, go to Step 7

6

Add equipment details + Read more ...

Select the Add Equipment button and:

  • select relevant MRI equipment codes
  • enter MRI details being added
  • click Add
  • click Save

7

Confirm with Health + Read more ...

National Office will send an email to Health confirming the MRI transfer has been completed.
