This page contains information about the process for requesting a general withdrawal from Inland Revenue, New Zealand (NZIR) for a case that has been transmitted to Child Support and where the core + 10% liabilities have been collected in full.

Request General Withdrawal from NZIR

This table describes how Service Officers request a General Withdrawal from NZIR.

Step

Action

1

General Withdrawal Request + Read more ...

A general withdrawal request is sent to NZIR when the core + 10% liabilities have been collected on a case that has been transmitted by NZIR to Child Support (NZ In Case)

Create intray, go to Step 2.

2

Complete intray (Service Officer) + Read more ...

Create a new intray: New Zealand Withdrawal Request

  • Route the intray to:
    • Region: INTERNATIONAL
    • Office: INTERNATIONAL HOBART
    • Steam/Team: MC S005 NZPROJECT TM 1
    • Position: Z - AUTO REQUESTS NZ

Case Selection will produce a report to be sent to NZIR as part of the weekly exchange based on the intray type routed to the nominated Cuba position.

3

Response received from NZIR + Read more ...

Received within 15 working days of Weekly General Withdrawal Request spreadsheet sent to NZIR.

CA team to delete the intray when a withdrawal or amended certificate is processed.
