Request online claiming statement in Mainframe

Table 1

Step

Action

1

Log on to Mainframe

2

Request or view statement + Read more ...

To:

  • Find date of lodgement, key 'DCEI,PROVIDER NUMBER,CLAIM ID'
  • Request duplicate bulk bill statement, key 'DPSI,PROVIDER NUMBER,CLAIM ID, DOL =DDMMYY,R'. Note: requests are sent to the printing facility and mailed to the payee provider at their current mailing address. Statements are printed daily and should be received within 10 working days
  • View duplicate bulk bill statement, key 'DPSI,PROVIDER NUMBER,CLAIM ID, DOL =DDMMYY'

Request Medicare Easyclaim statement in PDS

Table 2

Step

Action

1

Log on to PDS + Read more ...

  • Go to the Medicare eBusiness gateway and select the APPLICATIONS icon
  • Select eBusiness Logon (including CDMS)
  • Select Provider Directory System and appropriate source office code and select Submit

2

Select Payments or Processing + Read more ...

In the Provider Directories menu, expand the Reports tab and select either:

  • Payment
  • Processing

3

Display report + Read more ...

  • Enter the required details such as payee provider number, date range and type format for the report to be downloaded
  • Select the Format type option - either Portable Document Format (PDF) or Excel
  • Select Request

Result: the Payment or Processing report is displayed.

Note:

  • Statements are posted and can only be emailed or faxed under Team Coach's discretion due to site's circumstances
  • Allow at least 4 business days from the date of lodgement before requesting reports to allow time for processing. Claims will not appear on the report until they have been finalised
