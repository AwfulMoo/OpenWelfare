Action

Check eligibility + Read more ...

The medical practitioner may be either an Australian citizen or permanent resident, or a temporary resident who is subject to section 19AB of the Health Insurance Act 1973 when the medical practitioner enrols in the 5 Year Overseas Trained Doctors (OTD) Scheme.

The References page contains a link to the Act.

When the Department of Health and Aged Care advises Services Australia Medicare of completion of the 5 Year OTD Scheme, the following criteria must be met:

  • Have become a permanent resident of Australia
  • Be recognised on PDS as having GP recognition (530 or 540)Completed the required term of service and the effective date of completion

Note: if any of the above criteria have not been met, escalate to a Programme Support Officer (PSO).

Transfer of location on the 5 Year OTD Scheme

Table 3: this table describes the process to complete when the Department of Health and Aged Care notifies Services Australia Medicare of a transfer of location for a provider subject to section 19AB on the 5 Year OTD Scheme.

Step

Action

1

Check Provider Directory System (PDS) + Read more ...

Open PDS:

  • Select Provider
  • Select View
  • Select Detailed Search
  • In the Family name type the practitioner’s surname with a % sign either side (this is a wild card function)
  • In the Given name  type the practitioner’s first name with a % sign at the end of the name
  • Select Profession
  • Select Search
  • A list of health professionals with a matching name displays

Is there a PDS match for the medical practitioner?

  • Yes, go to Step 3
  • No, there are no previous records that match the applicant. Go to Step 2.

2

No PDS record located for provider + Read more ...

In PaNDA:

  • Add a note stating ‘Provider not found on PDS, NFA’
  • Select Complete
  • Select Save or Save and Next, as appropriate

3

Check PaNDA and PDS + Read more ...

Check locations to determine if a provider number has been issued for the new location.

If a provider number has been issued for the new location, go to Step 4

If a provider number has not been issued for the new location, check PaNDA to determine if an application has been received for the new location.

Where an application has been received for the new location, assess the application:

  • If the application is incomplete, return the application in line with current procedures
  • Refer to Class Exemption Guidelines to determine a class exemption can be applied. Process provider number

Determine application needs to be sent to Department of Health and Aged Care for a 19AB Exemption. Complete spreadsheet, and file appropriately in PaNDA.

4

Update PDS record + Read more ...

If correspondence from the Department of Health and Aged Care is received notifying of a transfer of location, end date the location the provider is transferring from. If the previous location is listed on the correspondence:

  • Navigate to the appropriate record
  • Select Edit
  • Key Location End Date: key the day prior to the start date for the new location on the Department of Health and Aged Care correspondence
  • Select Save
  • Make one attempt to call the provider to advise:
    • The provider number for the location transferred from has been closed
    • The date the provider number has been closed from
    • Add a note to PaNDA of the details of the phone call
  • go to Step 5

If the previous location is not listed on the correspondence:

  • Add a note to PaNDA stating ‘No previous location indicated. NFA’
  • Select Complete
  • Select Save or Save and Next, as appropriate

5

Finalise + Read more ...

A letter must be sent confirming provider number closure.

  • Select the appropriate letter template
  • Complete letter template
  • Print and send letter
  • Save letter as a PDF and upload to PaNDA

In PaNDA:

  • Add a note stating ‘Transfer of location under 5 Year OTD Scheme processed’
  • Select Complete
  • Select Save or Save and Next, as appropriate
