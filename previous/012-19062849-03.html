Assessing a change in residency status notification

Table 1: this table describes how to assess a change in residency status notification from temporary resident to Australian permanent resident or citizen for a medical practitioner in the Provider Directory System (PDS).

Step

Action

1

Ensure correct evidence is supplied + Read more ...

Once it has been identified on the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) there has been a change in the residency status of the medical practitioner, the following evidence is required to amend the residency status in PDS.

Australian Permanent Resident

  • Australian permanent resident visa, or
  • visa email confirmation, or
    • VEVO confirmation (print out/screen shot)
    • visa label in passport and the bio-data (picture) page of their passport

Australian Citizenship

  • Australian citizenship certificate, or
  • Bio-data (picture) page of their Australian passport

Where the medical practitioner has supplied an Australian citizenship document or an Australian passport and there is no permanent residency date recorded in PDS, contact the applicant and request they supply their permanent residency visa so PDS can be updated accordingly.

PDS must reflect the medical practitioner is no longer a temporary resident by adding comments on the registration to reflect the date Australian citizenship was obtained.

Do not amend the residency status field on PDS to permanent resident until evidence has been submitted.

Note: New Zealand passport holders who become Australian citizens may not be issued with a permanent residency visa. In this instance, the date of Australian citizenship (from Australian passport or citizenship certificate) is used as the commencement date of Australian citizenship.

If the application is incomplete or documents not supplied go to Step 2.

If correct documents supplied go to Step 3.

2

Incomplete application + Read more ...

Where an application for a provider number indicates a change in residency status from temporary resident to permanent resident (or Australian Citizen) and the permanent residency visa has not been supplied:

  • Attempt to contact the medical practitioner on supplied contact information
  • Perform a security check
  • Request the medical practitioner submits a copy of the visa within 24 hours or the application will be returned
  • Application is to be pended with notes
  • A team leader will check the provider registration inbox for the documentation
  • If received, go to Step 3

If unable to contact provider or documentation not received, return the application using the incomplete letter template Z2306 and include a copy of the submitted documents:

  • Complete appropriate fields of letter template
  • Save letter as a PDF
  • Upload incomplete letter to PANDA
  • In PaNDA ensure the Surname First Name, Provider Number and case ID fields are complete and correct
  • Add a processing note to PaNDA detailing reasons for return
  • Change file status to Complete in PANDA
  • Select ‘Request Rejected’ in the Complete Reason field
  • Select Save or Save and Next, as appropriate

3

Access the medical practitioner details in PDS to determine the update process + Read more ...

Access PDS:

  • Open PDS
  • Select Provider
  • Select View
  • Select Detailed Search
  • Type the medical practitioner’s surname in the appropriate field with a % sign either side (this is a wild card function)
  • Type the medical practitioner’s first name in the applicable field with a % sign at the end of the name
  • Select the Profession
  • Select Search
  • Select the appropriate medical practitioner from the list of names

Check PDS specialty codes

Medical practitioner does not have access to Medicare benefits if only the below specialty codes are current:

  • 181 with no end date
  • 500 start date must match first date of registration with Ahpra, end date will be generated by PDS and will be 10 years from start date minus one day
  • 519 with no end date
  • See Table 7

Medical practitioner satisfies 19AA with post graduate qualification:

if any of the specialty codes, 001-099, 132, 530, 540, 532, 801-899 appear on the stem and are current along with:

  • 500 specialty code start date must match first date of registration with Ahpra, end date will be generated by PDS and will be 10 years from start date minus one day
  • 519 specialty code with no end date
  • See Table 3

Medical practitioner satisfies 19AA with 3GA placement:

  • has an open 3GA specialty code on location/s
  • 500 specialty code start date must match first date of registration with Ahpra, end date will be generated by PDS and will be 10 years from start date minus one day
  • 519 specialty code with no end date
  • See Table 4

Medical practitioner has access but does not satisfy section 19AA if only the below specialty codes are current:

  • 180 specialty code with no end date
  • 500 specialty code start date must match first date of registration with Ahpra, end date will be generated by PDS and will be 10 years from start date minus one day
  • 519 specialty code with no end date
  • See Table 5

Medical practitioner has access for assisting at operations only approved by Department of Health and Aged Care if only the below specialty codes are current:

  • 181 specialty code with no end date
  • 500 specialty code start date must match first date of registration with Ahpra, end date will be generated by PDS and will be 10 years from start date minus one day
  • 519 specialty code with no end date
  • 505 specialty code current at stem and location
  • See Table 6

If a 110 specialty code is reflected on the file, the residency status has been previously updated.

If Australian citizen details need to be recorded see Table 7. Otherwise, continue processing the provider application.

Processing a change in residency status for a medical practitioner without access to Medicare benefits

Table 2: this table describes how to process a change in residency status notification from temporary resident to Australian permanent resident or citizen without Medicare benefits for a medical practitioner in PDS.

Step

Action

1

Update/process change in residency on PDS + Read more ...

If permanent Australian residency details are already recorded and only citizen details need to be recorded see Table 7.

If permanent Australian residency details need to be recorded select Edit to update residency details:

  • Select Personal tab
  • Under Residency Details
    • Select Australian Permanent Resident from the Residency Status dropdown menu
    • Key date of permanent residency according to documents supplied
    • Update Passport Number and Country of Issue if details are provided
    • Key Visa Type according to documents supplied
  • Select Specialty codes tab to update the Specialty Code
    • End date the 181 specialty code the day before permanent residency
    • Key the 110 specialty code from the date permanent residency was granted
    • If the 500 specialty code end date is in the future, end date 519 specialty code in line with 500 specialty code
    • If the 500 specialty code is in the past, end the 519 specialty code the day before permanent residency was granted
  • Select Save

Note: if the update is only to residency status, go to Step 2. Otherwise, continue processing the application.

2

Create confirmation letter + Read more ...

  • Complete the 19AB-Visa Letter
  • Save letter as a PDF
  • Upload confirmation letter to file in PaNDA
  • Ensure the Surname, First Name, Provider Number and Case ID fields are complete and correct
  • Print the letter and send to the medical practitioner
  • Add any appropriate comments to the PaNDA file
  • Select Complete
  • Select Request Approved
  • Select Save or Save and Next, as appropriate

Processing a change in residency status for a medical practitioner who satisfies 19AA with post graduate medical qualification

Table 3: this table describes how to process a change in residency status notification from temporary resident to Australian permanent resident or citizen who satisfies section 19AA with postgraduate qualification in PDS.

Step

Action

1

Update/process change in residency on PDS + Read more ...

Where permanent residency details are already recorded and only citizen details need to be recorded, see Table 7.

If permanent residency details need to be recorded select Edit to update residency details:

  • Select Personal tab
  • Under Residency Details
    • Select Australian Permanent Resident from the Residency Status dropdown menu
    • Key date of permanent residency according to documents supplied
    • Update Passport Number and Country of Issue if updated passport details were provided
    • Key Visa Type according to documents supplied
  • Make a note of the 500 specialty code end date on the stem
  • Select Edit
  • Select Specialty tab
  • Select End at Location(s) hyperlink via 519 specialty code in Specialty tab
    • If no open 19AB exemptions are found go to Amend stem specialty codes
    • If the 500 specialty code end date is in the future select every line that has a 19AB exemption past the noted 500 specialty code date
    • Key end date in line with 500 specialty code end date in the End Date to be applied field
    • Select End Specialty code at provider level – this end dates the specialty code on the stem and locations
    • Select Amend Select Rows
    • Review selection and Apply End Date
    • Select Save
    • If the 500 specialty code is in the past select every 19AB exemption that expires after the 500 specialty code
    • Key the end date as the date before permanent residency in the End Date to be applied field
    • Select End Specialty code at provider level – this end dates the specialty code on the stem and locations
    • Select Amend
    • Select Rows
    • Review selection and Apply End Date
    • Select Save

Amend stem specialty codes

  • Select Specialty codes tab on stem to update the Specialty Code
    • End date the 180/181 specialty code the day before permanent residency was granted
    • Key the 110 specialty code from the date permanent residency was granted
    • If the 519 specialty code was not closed as part of the ending 519 on locations then end date the 519 specialty code as follows
    • If the 500 specialty code end date is in the future, end date 519 specialty code in line with 500 specialty code end date. If the 500 specialty code is in the past, end the 519 specialty code the day before permanent residency was granted

Where only an update to a medical practitioner’s residency is undertaken, go to Step 2. Otherwise continue processing the provider application.

Processing a change in residency status for a medical practitioner who has access to Medicare benefits but does not satisfy 19AA

Table 5: this table describes how to process a change in residency status notification from temporary resident to Australian permanent resident or citizen who currently has access to Medicare benefits and does not satisfy section 19AA in PDS.

Step

Action

1

Update/process change in residency in PDS + Read more ...

If permanent residency details are already recorded and only Australian citizen details need to be recorded see Table 7.

Select Edit to update residency details:

  • Select Personal tab
  • Under Residency Details
    • Select Australian Permanent Resident from the Residency Status dropdown arrow
    • Key Date of Permanent Residency according to documents supplied
    • Update Passport Number and Country of Issue in updated passport details was provided
    • Key Visa Type according to documents supplied
    • Select End at Location(s) hyperlink via 519 specialty code in Specialty tab
    • Select every provider number row that has a 19AB exemption granted
    • Key end date as the date before permanent residency in the End Date to be applied field
    • Select Amend Select Rows
    • Review selection and Apply End Date
    • Save
    • Select Validate
    • Select Save
    • Repeat process where the provider has an Other Medical Practitioner’s (OMP) program at the location/s. OMPs end date is the day before permanent residency was granted
  • Select Specialty codes tab on stem to update the Specialty Code
    • End date the 180/181 specialty code the day before permanent residency
    • Key the 110 specialty code from the date permanent residency was granted
    • If the 500 specialty code end date is in the future, end date 519 specialty code in line with 500 specialty code end date. If the 500 specialty code is in the past, end the 519 specialty code the day before permanent residency was granted
    • If any specialty codes between 647 and 650 or 662 and 665 (MHST, PSC, FPS) are present these must also be end dated using the date before Permanent Residency

Select the orange provider stem hyperlink at the top of the page

Individually check each provider number that has a Status of OPEN

Select view in the Action filed at the end of the provider number row

  • If the provider number has a 519 code, add a 618 service rule with a start date the day after the 519 ended
  • If the provider number has 19AB class exemption listed in the Location Type, add a 618 service rule with a start date of permanent residency
  • If the provider number has neither a 519 or 19AB class exemption recorded, add a service rule with a start date in line with the provider number start date

Access to Medicare benefits has ceased due to residency status change, a phone call to the medical practitioner is required

  • Use the mobile number on the stem or phone number on the principal place of practice
  • Complete a security check
  • Advise the medical practitioner access to Medicare benefits has ceased due to change in residency status
  • Advise the medical practitioner the requirements of meeting section 19AA

If contact is unsuccessful, add note to PaNDA.

Where only an update to a medical practitioner’s residency is undertaken, go to Step 2. Otherwise, continue processing the application for a provider number.

  • Under Residency Details

    • Key Date of Permanent Residency according to documents supplied

Processing a change in residency status notification from Australian permanent resident to Australian Citizen

Table 7: this table describes how to assess a change in residency status notification from Australian permanent resident to Australian Citizen for a medical practitioner in PDS.

Step

Action

1

Update/process change in residency on PDS + Read more ...

Select Edit to update residency details:

  • Select Personal tab
  • Under Residency Details
    • Select Australian Citizen from the Residency Status dropdown menu, do not over key Date of Permanent Residency
    • Update Passport Number and Country of Issue if updated passport details were provided
    • Select Registration tab
    • Select Amend on any line, key in Comments
    • If Australian citizenship document is supplied key ‘Australian Citizen ddmmyy’
    • If application has citizenship date in full and no supporting documents key ‘declared Australian Citizen ddmmyy’
    • If Australian passport supplied and no dates indicated on the application key ‘Australian passport issued ddmmyy’
    • Select Update
    • Select Save

Note: New Zealand passport holders who become Australian citizens may not be issued with a permanent residency visa. In this instance, the date of Australian citizenship (from Australian passport or citizenship certificate) is used as the commencement date of Australian citizenship

Where only an update to a medical practitioner’s residency is undertaken, procedure ends here. Otherwise, continue processing the application for a provider number.
