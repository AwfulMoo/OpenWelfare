2

Parental Leave Pay (PPL) and Dad and Partner Pay (DAP) + Read more ...

For PPL payments delivered by Services Australia and DAP payments, the default tax rate is 15%.

Customers can nominate a different tax rate between 0% and 50% when claiming PPL/DAP or at a later time.

Note: for PPL payments delivered by their employer, these customers will need to contact their employer to discuss their tax deduction rate.

Can the customer complete via self-service? + Read more ...

5

Create a tax deduction in Process Direct + Read more ...

Tax deductions can be created in Process Direct by:

  • entering ‘PITX’ in Super Key field
  • select [+] to add Tax Instruction

The Create Tax instruction (PITX) screen will display

  • Code Start Date
  • Code End Date where a specific date is advised, if no date is provided the end date will default to 31/12/9999
  • Code deduction amount in either whole dollars (minimum amount is $1, max $99999) or percentage
  • Select Benefit Group from drop down options
  • Save
  • Update Header Data by confirming receipt date and channel and save
  • Select ‘Assess’
  • Select ‘Finish’ – add comments and finalise

Note: following ‘Assess’ being actioned any changes must be completed in Customer First.

6

Create a tax deduction in Customer First + Read more ...

Action the request on the Payment Instruction Summary (PINS) screen.

  • Ensure the correct system is accessed. For example:
    • For DAP, type SVDAP in the Next: field and ISS in SYSTEM
    • For PPL, type SVPPL in the Next: field and ISS in SYSTEM
    • For FHA, type SVFHA in the Next: field and ISS in SYSTEM
  • Enter 'PINS' in the Next: field
  • Enter 'Y' in Add New Deduction: field
  • Code 'T' in Type: field

The Tax Instruction (PITX) screen will display

Note: for PPL and DAP customers, only the whole percentage (%) can be coded for tax deductions and not a dollar ($) amount.

  • Enter either the:
    • Amount of the fortnightly deduction, in whole dollars, requested by the customer in the Requested Deduction $: field, (minimum amount is $1, max $99999) or
    • Percentage of the fortnightly deduction requested by the customer in the P.F %: field. The amount will be rounded down to the nearest dollar
  • Enter a valid code for the relevant benefit group in the Benefit Group: field
  • Enter the date in the Effect Date: field. This is the date on which the payment instruction takes effect.
    • Note: this date field cannot be earlier than today's date and must be within the next 28 days. If left blank, the date will default to the first available payday after the request has been made.
  • Enter the Source: field with a valid code:
    • 'PHO' if advised over the phone
    • 'COS' if advised over the counter
    • 'LET' if advised by letter
    • 'SU66' if advised by ‘SU066’ Tax Deduction Authority form (NSS ONLY)
  • The ACTION field will automatically code 'I'
  • Code the DOR: field
  • Press [Enter]
  • Enter 'Y' in the Confirm: field, press [Enter]

The PINS screen is displayed.

  • Enter 'Y' in the Finished: field and press [Enter], which will navigate to the Assessment Result (AR) screen
  • Enter 'Y' on the AR screen to finalise activity

Go to Step 9
