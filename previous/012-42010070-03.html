This page contains procedures to search for a provider in the Provider Directory System (PDS), how to assess and process an application for a Medicare provider number for an intern.

On this page:

Assess an application for a Medicare provider number for an intern

Process a Medicare provider number application for an intern

Process an additional Medicare provider number application for an intern

Assess an application for a Medicare provider number for an intern

Table 1: this table describes how Service Officers assess applications for an initial Medicare provider number for an intern to determine the specialty codes used in PDS.

Step

Action

1

Assess the application and Ahpra registration + Read more ...

Check the Initial application carefully. Ensure all required sections have been completed and the form has been signed.

The application must not be processed if the signature is missing. Return the application as per incomplete application process

Check the provider’s Australian Health Practitioner Regulation Agency (Ahpra) Medical Board registration is attached. This may be a copy of a letter, a print out from the Ahpra website or copy of an email from Ahpra containing the practitioner’s name, registration number registration status and effective date. If Ahpra document attached go to Step 2

If the registration is not attached, search the Ahpra website for the provider’s registration. The Resources page has a link to the website.

Search the register by the provider’s last name and first name. Note: take a screenshot of the search result and upload as a PDF to PaNDA. Go to Step 2

If registration details are not attached and the Ahpra website does not have the intern registered, the application is incomplete and must be returned with an accompanying letter. Return the application as per incomplete application process.

2

Assess qualification + Read more ...

Check the qualification section of the application to determine the practitioner’s primary medical qualification.

Qualification – includes details of where the medical practitioner obtained their medical qualification

Was the qualification obtained from a university in Australia?

  • Yes, go to Step 3
  • No, degree from a New Zealand university, see New Zealand medical practitioners - provider eligibility and registration
  • No, degree from a university other than Australia or New Zealand see Processing provider number applications for overseas trained or temporary resident medical practitioners

3

Assess intern registration + Read more ...

Is the Ahpra registration provisional?

  • Yes, see Process a Medicare provider number application for an intern
  • No, see Applications for a provider number for medical practitioners and sections 19AA/19AB requirements

Process a Medicare provider number application for an intern

Table 2: this table describes how Service Officers process applications for a Medicare provider number for an intern in the Provider Directory System (PDS).

Step

Action

1

Search PDS for a provider registration + Read more ...

Check if the practitioner is existing within the system or has a shell record:

  • Open PDS
  • Select Create
  • In the Registration Number field type the medical practitioners Ahpra registration number
  • Select Search

If a shell record is found this can be identified if in the Action field it shows View or Promote only

If a stem is found this can be identified if in the Action field it shows View, Amend or History

If no stem or shell record is found, PDS will return Service officer to the Create page

Is there a shell record or an existing Stem?

  • Yes,
    • shell record – Promote the shell and go to Step 2
    • existing stem – select View and go to Step 2
  • No, go to Step 4

2

Check provider stem details in PDS + Read more ...

Check the following tabs in PDS against the information in the application form (HW019):

Personal:

  • Last name and first name must appear in PDS as it is on their Australian Health Practitioner Regulation Agency (Ahpra) registration. If the provider has put a middle name on the application that does not appear on Ahpra, key the middle name
    • do not key abbreviated names, for example Alex instead of Alexander
  • Gender
  • Date of birth
  • Languages select language if available from drop down box on PDS. If language is not listed on PDS make a note on PaNDA file ‘language not listed’
  • Title
  • Residency details if provider indicates Australian born select Australian citizen from the Residency Details drop down box in PDS. If non-Australian born complete all sections

Qualifications:

  • key qualifications, the accepted abbreviation is ‘MBBS’
  • place of base qualifications - select from the dropdown list
  • year of qualifications
  • ‘No’ must be selected in the sonographer field
  • if the provider has ticked ‘yes’ to a prescriber number on the application form, select ‘Prescriber’ in ‘Prescriber status’

Contact:

  • key contact details as supplied on the application form. If the Rapid Addressing Tool (RAET) suggests changes, ‘accept’ the changes

Registration:

  • Add Comments
  • Select Amend
  • If there are any conditions listed on Ahpra (mentioned at the bottom of the Ahpra registration website screen), key the word ‘conditions’
  • If the medical practitioner has ticked Born in Australia on the application form key “declared Australian born”
  • If the medical practitioner has provided a New Zealand passport key passport details and expiry date i.e. “NZ passport x010121” (see note)
  • For temporary resident visa holders, key the visa type and the expiry date e.g. ‘v457 x010121

Note: if the medical practitioner submits a New Zealand passport and the place of birth is not a New Zealand suburb or city, check the date of issue. The date of issue must be at least 4 years prior to the Medical Practitioner finishing medical school. If the date of issue is less then 4 years prior to the qualification date, make one attempt to contact the provider for evidence of New Zealand permanent residency or citizenship from that year. If unsuccessful, return application as per incomplete application process

If the following error message is displayed see FAQ on the Resources - error.0100 Prescriber number cannot be allocated. (Prescriber does not have a required and/or current specialty code OR registration.)

Make amendments where required, go to Step 3

3

Check the Specialty tab + Read more ...

Check for the following combinations of specialty codes to match the application. Amend for existing stems and add for shell records where required:

  • 160 for Australian born/citizen/permanent resident
  • 162,181 and 185 for a New Zealand citizen/permanent resident when enrolled in an Australian accredited medical school.
  • 181, 162, 500 and 519 for current temporary resident
  • 110, 162, 500, 519 for medical interns who were not Australian or New Zealand permanent residents or citizens when enrolled in Medical school, that have indicated they are now an Australian permanent resident or citizen
  • The start date is the same as the registration start date
  • End Date: no end date is required

Where indicated on application form the medical practitioner has signed a Bonded Program agreement with Department of Health and Aged Care add specialty codes:

  • 501 for Medical Rural Bonded Scholarship (MRBS) scheme
  • 507 Bonded Medical Places (BMP) specialty code,

The start date is the same as the registration start date

Once these have been confirmed/saved, go to Step 5

4

Create provider stem + Read more ...

Key the following tabs from the application information:

Personal:

  • Last name and first name must appear in PDS as it is on their Australian Health Practitioner Regulation Agency (Ahpra) registration. If the provider has put a middle name on the application that does not appear on Ahpra, key the middle name. Do not key abbreviated names, for example Alex instead of Alexander
  • Gender
  • Date of birth
  • Languages select language if available from drop down box on PDS. If language is not listed on PDS make a note on PaNDA file ‘language not listed’
  • Title
  • Residency details if provider indicates Australian born select Australian citizen from the Residency Details drop down box in PDS. If non-Australian born complete all sections.

Qualifications:

  • key qualifications, the accepted abbreviation is ‘MBBS’:
  • place of base qualifications - select from the dropdown list
  • year of qualifications
  • ‘No’ must be selected in the sonographer field
  • if the provider has ticked ‘yes’ to a prescriber number on the application form, select ‘Prescriber’ in ‘Prescriber status’

Contact:

  • key contact details as supplied on the application form. If the Rapid Addressing Tool (RAET) suggests changes, ‘accept’ the changes

Registration:

Select ‘Add New Registration’ and key:

  • Area: AUS
  • Registration Number: as listed on Ahpra registration
  • Status: select ‘Intern’ if Ahpra registration status shows Provisional
  • Start Date: as stated on Ahpra registration
  • End Date: leave blank

Comments:

Select Amend

  • If there are any conditions listed on Ahpra (mentioned at the bottom of the Ahpra registration website screen), key the word ‘conditions’
  • If the medical practitioner has ticked Born in Australia on the application form key “declared Australian born”
  • If the medical practitioner has provided a New Zealand passport key passport details and expiry date i.e. “NZ passport x010121 (see note)”
  • For example for temporary resident visa holders, key the visa type and the expiry date e.g. ‘v457 x010121

If the medical practitioner submits a New Zealand passport and the place of birth is not a New Zealand suburb or city, check the date of issue. The date of issue must be at least 4 years prior to the Medical Practitioner finishing medical school. If the date of issue is less then 4 years prior to the qualification date, make one attempt to contact the provider for evidence of New Zealand permanent residency or citizenship from that year. If unsuccessful, return application as per incomplete application process.

Specialty:

Select ‘Add New Speciality’ and key specialty codes:

  • 160 for Australian born/citizen/permanent resident
  • 162,181 and 185 for a New Zealand citizen/permanent resident when enrolled in an Australian accredited medical school
  • 181, 162, 500 and 519 for current temporary resident
  • 110, 162, 500, 519 for medical interns who were not Australian or New Zealand permanent residents or citizens when enrolled in Medical school, that have indicated they are now an Australian permanent residents or citizens
  • The start date is the same as the registration start date
  • End Date: no end date is required

Where indicated on application form the medical practitioner has signed a Bonded Program agreement with Department of Health and Aged Care add specialty codes:

  • 501 for Medical Rural Bonded Scholarship (MRBS) scheme
  • 507 Bonded Medical Places (BMP),

The start date is the same as the registration start date

See the Resources page for a link to the Department of Health and Aged Care website

Start Date:

  • select Add New, cancel
  • select Save
  • respond to any return messages (e.g. mandatory fields not keyed) and
  • select Save
  • the record is created and a stem number allocated
  • select the orange number hyperlink at the top of the record

Note:

  • if the following error message is displayed see FAQ in Resources error.0100 Prescriber number cannot be allocated. (Prescriber does not have a required and/or current specialty code OR registration.)”
  • all start dates should coincide with the start date of the Ahpra registration. No end date should be recorded on the stem page

Go to Step 5.

5

Create new location + Read more ...

Select Create New Provider Location.

Check for an existing provider number for the requested practice/hospital name or street address. E.g. Prince Charles Hospital Rode Rd and Prince Charles Hospital 627 Rode Rd are considered a match.

  • If no provider number exists for the requested practice/hospital or the requested street address continue assessment.
  • If a match for practice/hospital name or street address see Issuing a second provider number or contact a PSO for assistance.

Address:

  • key postcode in Locality or Postcode as per the application. Select the appropriate suburb and postcode from the drop down menu. If the postcode does not return a result or does not match, key the suburb in Locality or Postcode. Select the appropriate suburb and postcode from the drop down menu. If the postcode and locality do not match, contact a Programme Support Officer (PSO)
  • key street number and street name in Address Line
  • select address that matches the application
  • key practice name (if supplied) in Building or Department (26 character limitation in Building or Department line)

Key relevant fields in the Contact tab:

  • phone
  • fax
  • email
  • key location start date from the application or date of Ahpra registration whichever is later

Note: an end date is not required if it has not been specified on the application.

Assoc Details Tab

Check the 19(2) determinations in the Provider Registration resources. If it’s determined that the location being processed is on a 19(2) determination flag the location as directed below:

There are 2 location flags that are used depending on which direction a location falls under. If a location is listed on:

  • An Indigenous Direction – Select ATSIHS and AMS
  • The COAG Direction – select AMS

6

Business structure + Read more ...

Are business structure details present on the application form?

Yes, select Organisation Tab

  • Select the Employment Type from the drop down menu as per the form:
    • Employee Salaried form field is the Employee - Payroll Provider field in PDS
    • Employer – Employer field in PDS is not used
  • Select Add under Organisation Details
  • Enter the ABN and select Search
  • Select relevant record:
    • If multiple ABN options return, select the matching record
    • If results don't match the form, select the first result returned
  • Enter Trading As details as per the Registered Business Name on the form
  • Select the Business Type from the drop down menu as per the form
    • Associateship and Government or Other Public Body fields are not used in PDS
    • Select Premises Type from the drop down menu as per the form
    • Select save
    • Go to Step 7

No go to Step 7

7

Register the provider for EFT + Read more ...

Are EFT details present on the application form?

  • Yes, select Program registration
    • Add banking details, see Step 2 in the Register for EFT payments table then go to Step 8
  • No, go to Step 8

8

Assess 19AB restrictions + Read more ...

Does the intern have the combination of 181, 162, 500 and 519 speciality codes?

  • Yes, go to Step 10
  • No, go to Step 9

9

Assess if service rule required + Read more ...

Were business structure and EFT details provided in full?

  • Yes, go to Step 11
  • No, go to Step 10

Note: see FAQs on the Resources page for information regarding access to Medicare benefits including assisting at operations.

10

Apply Location Service Rule + Read more ...

A Service Rule will need to be applied to prevent access to Medicare billing

  • Go to Loc-Spec Codes Add Location Service Rule:
  • Select Add Service Rule
  • In Service Rule Number field, key 618
  • In the Activity Code field, select Yes
  • Enter the Start Date of the location
  • In End Date, leave blank
  • Select Add New
  • Select Cancel

11

Confirmation Letter + Read more ...

Select and complete the Z2701 letter template from the letters index.

  • Complete appropriate fields of letter template ensuring to refer patients to specialists and consultant physicians is the level of access selected
  • Save letter as a PDF
  • Upload the letter to PaNDA
  • Ensure the Surname, First Name, Provider Number and Case ID fields are completed and correct
  • Print the letter and send to the medical practitioner
  • Add any appropriate comments to the PaNDA file
  • Select Complete
  • Select Save

Process an additional Medicare provider number application for an intern

Table 3: this table describes how Service Officers process additional applications for a Medicare provider number for an intern in the Provider Directory System (PDS).

Step

Action

1

Search PDS for a provider registration + Read more ...

  • Open PDS
  • Search PDS to check if the practitioner is existing within the system:
  • Select Provider
  • Select View
  • Select Detailed Search
  • Type the practitioner’s surname in the appropriate field with a % sign either side (this is a wild card function)
  • Type the practitioner’s first name in the applicable field with a % sign at the end of the name
  • select the Profession
  • select Search

Note: the provider must be registered on PDS with the name they have registered with on the Ahpra Website. See the Resources page for a link to the website.

  • Where a provider stem has not been previously issued go to Step 1 in Table 1
  • Where a provider stem has already been issued go to Step 2 in Table 2
