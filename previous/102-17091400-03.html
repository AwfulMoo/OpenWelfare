3

Report Identity Fraud + Read more ...

Identity fraud and attempts to commit identity fraud occur when a person uses an identity other than their own legal name to:

  • create a CRN
  • make a claim, and
  • receive a payment they are not entitled to receive

To prevent Identity fraud, check for indicators that may show a deliberate attempt to commit fraud, such as:

  • Different payment destinations provided
  • Different mobile phone numbers provided
  • Not declaring a previous name/s
  • Multiple myGov accounts for the initial linking of records and ongoing transactions
  • Making a second claim when already on a benefit
  • Identity documents lodged with similar and recent issue dates

If a staff member identifies fraud, they must Report Suspected Fraud.

If there is no identified fraud or after actioning the Fraud referral, go to Step 4.

Dependent Youth Allowance (YA) + Read more ...

Where Parental Income Test (PIT) links already exist on the Multiple CRN, they will need to be either deleted or ended.

Go to the PIT Parent Selection (NPPS) screen. Select the relevant Current PIT parent. If the PIT Link Start Date is:

  • before the most recent YA Start Date, end date the PIT link:
    • End Date: field, the day prior to the current YA Start Date
    • End Reason: field, NLR - No longer required
    • Action: field, C - Correct or I - Insert
    • Enter
  • on, or after, the most recent YA Start Date, delete the PIT link:
    • Action: field, D - Delete
    • Enter

Repeat the steps for each Current PIT parent, and then Finalise the activity via Assessment Results (AR) screen.
