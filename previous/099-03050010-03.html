Explanation of reason codes fields used on this page

Table 1

Item

Description

1

Term / Code + Read more ...

Three letter code used on the PEN system to reject, suspend or cancel.

2

Description + Read more ...

Expanded meaning of the code.

3

Valid Action Codes + Read more ...

  • REJ = rejection
  • SUS = suspension
  • CAN = cancellation

4

Automatic or manual action + Read more ...

Man = reasons that can be manually input on the Benefit Action (BA) screen.

Auto = automatically occurring cancellations, rejections or suspensions (cannot be keyed manually).

Man/Auto = can be manually inserted or automatically initiated.

Note: on the BA screen use field help ('?') in the reason field for a list of codes available.

5

Restoration + Read more ...

  • Rejections can never be restored.
  • Suspensions can always be restored.
  • Cancellations can sometimes be manually or automatically restored.
