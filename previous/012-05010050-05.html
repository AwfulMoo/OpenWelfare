This page contains the claim deletion rules, information requirements and contact details to handle claims that cannot be deleted.

Claim deletion rules

Claim deletion rules for each claim type

Table 1: This table outlines the status a claim must have before it can be deleted.

Claim type

Rule

Medicare Online

A claim can be deleted if either:

  • PAID (processed) on the same date
  • Not fully processed (for example, status WAITPEND)

Note: for claims in WAITBKGD, advise the practice the claim cannot be deleted and they must call back later.

Medicare Easyclaim

A claim can be deleted if either:

  • Payee code 2 - Unpaid account
  • Payee code 6 - Bulk bill. Note: for claims in WAITBKGD, advise the practice the claim cannot be deleted and they must call back later

Note:

  • Payee code 7 - PAID by EFT to Claimant, cannot be deleted
  • Easyclaim patient claims must not be deleted after 5.30pm (Eastern Standard Time). Deleting claims after this time impacts the system's ability to process banking reports appropriately. Advise practices to submit a request for a Latter Day Adjustment in this instance

Medicare Bulk Bill Webclaim

A claim can be deleted if either:

  • PAID (processed) on the same date before the claim is sent to the overnight payment run
  • Not fully processed (for example, status WAITPEND)

Medicare Patient Claim Webclaim

A claim can be deleted if either:

  • PAID (processed) on the same date before the claim is sent to the overnight payment run
  • Not fully processed (for example, status WAITPEND)

Claim deletion information requirements

Required information for each claim type before deletion

Table 2: This table lists the details for each claim type that must be provided to action a claim deletion request.

Claim type

Details required

Patient claims

Bulk bill individual vouchers
  • Medicare card number
  • Patient name
  • Servicing provider number
  • Dates of service
  • Item numbers

Bulk bill claims except individual vouchers
  • Minor ID (Medicare Online only)
  • Claim ID
  • Payee provider number
  • Date of Lodgement (DOL)

Handle claim that cannot be deleted

If the claim has been processed on an earlier day or cannot be deleted for other reasons, advise the caller an adjustment is required. Service Officers can contact the relevant area for support or warm transfer the caller.

Medicare Online, Medicare Easyclaim and Medicare Bulk Bill Webclaims Bulk Bill Claims already sent to the payment run or processed on a previous day

Complete

  • Application for Bulk Bill claim adjustment form, and
  • A new assignment of benefit form that is completed with the correct information and signed by the patient

See Bulk bill adjustments for claims under 2 years

Medicare Online patient claims and Medicare Patient Claim Webclaims already sent to the payment run or processed on a previous day

All requests for adjustment must:

  • be documented in a fax or letter on the health professionals site letterhead and include:
    • claimant's Medicare card number
    • patient's Medicare card number
    • patient's name or IRN
    • date of service
    • item number
    • servicing provider number
    • amount charged
  • Duplicate account and receipt with reason for the change and notated with the practitioners signature

Note: health professionals must advise patient of the error

Requests for adjustments can be faxed to Assessing and Benefits.

Medicare Easyclaim Payee code 7 claims

All requests for adjustment must:

  • be documented in a fax or letter on the health professionals site letterhead and include:
    • claimant's Medicare card number
    • patient's Medicare card number
    • patient's name or IRN
    • date of service
    • item number
    • servicing provider number
    • amount charged
  • Include a duplicate account and receipt with reason for the change and notated with the practitioner's signature

Note: health professionals must advise patient of the error

Requests for Easyclaim adjustments must be faxed to Assessing and Benefits.
