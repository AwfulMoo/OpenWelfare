Initial consultation items

Table 1:

Item no.

Description and resolution

CD01, CD04, CD09, CD14

Initial consultation - rooms; in-home; private hospital; low care residential aged care facility (RACF) + Read more ...

After 1 October 2019

Rules

  • Limit of 1 service in 14 days from any provider
  • Limit of 12 services in 365 days from any provider

Restrictions

If an initial consultation restricts with:

  • itself or another dietitian initial consultation item, reject with RSN code S 160
  • any other item, reject the item with the lower fee with RSN code S 159

Prior to 1 October 2019

Rules

  • up to 2 initial consultations can be claimed in a 12 month referral period. Each initial consultation must be for a new episode of care or a new and unrelated condition

Restrictions:

If item restricts with:

  • itself or another initial consultation, reject with RSN code S 160
  • any other item, reject item with lower fee with RSN code S 159

CD07, CD11

Initial consultation - public hospital, high care RACF + Read more ...

The agency will only pay for health care services carried out in public hospitals and high care residential aged care facilities in exceptional circumstances and only when DVA has given prior financial authorisation.

Rule:

  • up to 2 initial consultations can be claimed in a 12 month referral period. Each initial consultation must be for a new episode of care or a new and unrelated condition

Requires Prior Financial Authorisation

Check PARS for an approval.

When Prior Financial Authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283

Group session item

Table 4:

Item no.

Description and resolution

CD17

Group session – per patient + Read more ...

Rule:

  • a group session item cannot be claimed on the same day as an initial or extended consultation item
  • a group session item can be claimed on the same day as a subsequent consultation

Restrictions

If item restricts with:

  • any initial or extended consultation item, reject item with lower fee with RSN code S 159
  • any subsequent consultation item, check for evidence of a separate attendance (e.g. am/pm):
    • if evidence present, pay with PI 42
    • if evidence is not present, reject item with lower fee with RSN code S 159

Diabetes Treatment Cycle Report

Table 5:

Item no.

Description and resolution

CD90

Allied Treatment Cycle Report + Read more ...

Provision of report to GP on outcomes of Treatment Cycle related to the referral.

Rules

  • An initial consultation item in the allied fee schedule is claimed
  • Any number from 1 to 11 subsequent consultation items from the same fee schedule is claimed
  • Only 1 report item is payable for each initial consultation
  • The provider must have claimed 1 initial and 1 subsequent consultation
  • Cannot be paid twice on the same day
  • Limit of 1 report item payable within 14 days
  • Limit of 12 report items payable within 12 months

Restrictions

  • Associated consult not claimed, reject with RSN code S 748
  • If the item restricts with itself or any other item, reject item. Use lower fee with RSN code S 159

Diabetes Educator direct supply to DVA item

Table 6:

Item no.

Description and resolution

CD99

Diabetes educator – report or service specifically requested by DVA + Read more ...

Rule:

Providers may use item number CD99 only when DVA contacts them directly to request that they provide a:

  • Written report; or
  • Consultation or assessment to eligible veterans or war widows/ers, either separately or in conjunction with a written report. For example, this may occur when DVA requires a second opinion concerning treatment for a veteran. DVA will give financial authorisation and advise the fee at the time of the request, according to the relevant schedule items. The kilometre allowance is included in the fee, and is not to be claimed in addition to the fee.

Note: this item does not cover the supply of clinical notes, care plans or other information requested by DVA as part of monitoring activities, as these are provided free-of-charge under DVA requirements.

Restrictions:

Requires Prior Financial Authorisation.

Check PARS for an approval.

When Prior Financial Authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283
