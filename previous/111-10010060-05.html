Classification Guidelines

The following tables provide specific classification information for form codes and unstructured documents. This is a support document for Classification Officers undertaking manual classification of Centrelink customer documents. Classification Officers can escalate the activity to a Classification Support Officer at any stage if they need support. Note: remember to check every single page number of each digital image to ensure that all pages are for the same customer. If the form code is not listed for selection escalate to Classification Support.
