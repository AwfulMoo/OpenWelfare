Using the OPDL screen in the Debt Management and Information System (DMIS) to locate debts

Item

Description

1

Locating debts on the OPDL screen in Customer Record + Read more ...

Note: the fields on the OPDL screen are display only, and field help ('?') cannot be used to expand the acronyms.

Key 'OPDL' in the Nxt field to go to the OPDL screen:

  • The OPDL screen is a summary with fields as display only. Up to 99 pages of debts can be viewed
  • Select debts from the list using either 'S' or 'U':
    • 'S' will select the debt for viewing
    • 'U' will restart Family Tax Benefit (FTB) debts where recovery action has been paused. Only use this option if a customer asks to repay the debt while waiting for updated taxable income from the Australian Taxation Office (ATO)
  • A maximum of 10 selections can be made. Use function keys to scroll through the selected debts:
    • [PF24] (Shift+F12) to move forward to the next debt, and
    • [PF23] (Shift+F11) to move back to the previous debt
  • Debt specific screens are protected for started NDB or DBT activities. Select the activity from the Activity List (AL) screen to make updates to debt specific screens
  • If a debt is selected from the OPDL screen, the General Information (OPGI) screen will display. To go to a specific debt screen enter the screen name in the Nxt field and press [Enter]

Note: see Debt screens in Process Direct for debts managed in Process Direct.

2

Debt Id, Ben Type and Reason fields + Read more ...

Debt Id field - Displays the unique 8 character identifier number.

Ben Type field - Displays the 3 letter code of the payment type that was overpaid.

Reason field - Displays the 3 letter code of the reason for the debt.

Full details of the benefit type and reason is on the OPGI screen.

3

Debt Total $ field + Read more ...

The Debt Total $ field displays the total calculated debt. This is the amount of the debt raised.

4

Interest Chg $ field + Read more ...

The Interest Chg $ field Displays the Interest Charge applied for each individual debt.

The system shows the interest charged at the individual debt level and the CRN total amount of interest charged. To view the Interest Charge history, go to the Debt List within Customer First using the Search field. The Debt List is located under the Financial Customer Care allocation block.

Click on a debt interest charge hyperlink to show:

  • the period start date and end date
  • the debt amount
  • the interest rate
  • the interest amount
  • the duration (days)
  • the reason
  • any exclusions

5

Balance Outstanding $ field + Read more ...

The Balance Outstanding $ field displays the current amount owed on the debt. It is the Total Debt $ field, plus any Interest Chg $, less all amounts, adjusted, repaid or waived.

The balance outstanding is recalculated whenever:

  • a repayment amount is recorded on the Repayment History (OPRH) screen
  • information is entered in the Correct Amount $ field on the Adjust Repayment History (OPAH) screen. This screen is accessed by selecting a payment from the OPRH screen
  • an adjustment is recorded on the Adjust Repayment (OPAR) screen
  • a recalculated amount is recorded on the OPDA screen
  • a waiver amount is recorded on the Add Waiver (OPWAA) screen
  • a waiver amount is reversed on the Waiver (OPWA) screen
  • a refund is reallocated via the Debt Transfer Details (OPRTD) screen

6

Status field + Read more ...

The Status field displays the status of the debt:

  • Update the status of an undetermined debt on the Add Debt (OPAD) screen and Modify/Assess Debt (OPMA) screen
  • View the status of the debt on the General Information (OPGI) screen

7

CRN Totals field + Read more ...

  • Total Debt - is the sum of all debts raised on the customer's record
  • Total Interest Charged - is the sum of all interest charged to the customer's record
  • O/S Balance column - is the total sum of outstanding debt balances. It includes the amount of interest charged

These totals display at the bottom of each page of the OPDL screen. Viewing more than one page of debts will not change the totals.

8

Select View field + Read more ...

The Select View field shows the OPDL screen for different types of debts. Leaving the field blank will display all debts other than parked or deleted debts.

  • Key 'PRK' and press [Enter] and the OPDL screen will display all debts with a parked status
  • Key 'Del' and press [Enter] and the OPDL screen will display all debts deleted in the past

9

Arrangements field and the Payment Arrangement List screen + Read more ...

The Payment Arrangement List (OPAL) screen shows a history of recovery arrangements including any current arrangements.

Each arrangement shows the Type, status (STS), commencement (Add Date), End Date and Amount.

The Arrangements field on the OPDL screen will show:

  • the current arrangement's type code, and
  • the status code, for example VOL/CUR

Otherwise the field will display 'No active arrangements'.

Type codes are:

  • CSH Regular Cash Agreement
  • IRR Irregular Cash Arrangement
  • GRN Garnishee
  • WHS Withholdings
  • AGY Agency Recovery
  • DSC Discount
  • VOL Direct Debit
  • AWH Trigger Auto-Withholding processing
  • SRW Standard Reconciliation Withholding

Status (STS) codes:

  • CUR Current
  • PND Pending
  • PVL Provisional
  • FUT Future
  • FIN Finalised
  • BKN Broken
  • CAN Cancelled

If there are no arrangements in place, then 'no current arrangements' will display.

10

Departure Prohibition Order (DPO) status + Read more ...

If there is an 'Approved' DPO in place, the Arrangements field will show 'DPO Active'. When there is no active DPO, this field will be blank.

Click the CRN link in Customer First to view the Customer Summary page. A DPO alert is also displayed on Money You Owe and the Centrelink Express Plus app.
