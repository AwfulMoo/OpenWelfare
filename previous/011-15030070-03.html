Grant or refuse an extension request

Table 1

Step

Action

1

Evaluate – Extension + Read more ...

This process follows on from Search for an existing claim table.

When an extension request is received via telephone or in writing:

  • Select Phone enquiry or Statement by claimant from the dropdown menu

Note: different options will display depending on the selection made.

  • Select Have you authorised the caller and selected the relevant claim? or Does the claim selected match the details in the document? button
  • Select Is an extension being requested? or Is an extension to be processed button
  • Select Continue

A message displays Have you entered a case note? Would you like to continue processing Statement by claimant task or assign to the queue?

If the Service Officer is:

  • proficient:
    • select Keep assigned to me
    • select Ok
  • not proficient:
    • select Assign to the queue
    • select Ok

2

Capture extension details + Read more ...

Is the request for a reconsideration of decision?

  • Yes, claim to be transferred to APS4 (or above) for action. See Escalations and case notes
  • No,
    • enter extension request received date
    • the extension decision date defaults to today's date
  • If the request is:
    • granted, select Do you grant the request?, go to Step 3
    • refused, select Continue, go to Step 4

3

Grant the extension request + Read more ...

  • New due date field displays and the new due date auto populates
  • Select Continue

Note: the due date can be amended if granting an extension of more than 28 days.

4

Issue correspondence – Z2804 + Read more ...

  • Select the Notice type from the dropdown menu
  • Select Extension from the Letter purpose dropdown menu
  • If the decision made is to refuse, select the relevant reason
  • Select the correct recipient and select Preview to view the letter content
  • Select Submit to issue the letter
  • Add a case note to document this process

Issue a Notice of refusal (Z2806)

Table 2

Step

Action

1

Evaluate – Statement by claimant + Read more ...

This process follows on from the Search for an existing claim table.

  • Select Statement by claimant or Notice of Past Benefits request from the dropdown menu
  • Select Does the claim selected match the details in the document? button
  • Select Do you want to refuse to issue a NOPB? or Do you want to issue a NTC? button
  • Select Continue
  • A message displays Are you sure you want to refuse to issue a Notice of Past Benefits? or Are you sure you want to refuse to issue a Notice to claimant?, select Yes

Note: select the document type based off the document received, the options displayed will auto populate depending on the document type.

2

Who sent the request for a new NOPB? + Read more ...

If from the:

  • injured person, claimant or authorised third party, go to Step 3
  • notifiable person or their solicitor, go to Step 4

3

Issue correspondence – Z2806 + Read more ...

  • Select Notice of refusal from the Heading dropdown menu
    Note: if reconsideration of decision is selected claim is to be action by an APS4 (or above), use transfer assignment to transfer claim to the appropriate delegation
  • Select the tick box against the relevant reason for the notice of refusal being issued
  • If Failed to submit the statement within 2 years from DOJS reason is selected, additional fields appear:
    • Enter Date request was received
    • Enter Settlement date
  • Select the correct recipient and select Preview to view the letter content
  • Select Submit to issue the letter
  • Add a case note to document this process
  • Procedure ends here

4

Skip correspondence – Z2806 + Read more ...

  • Select Skip Correspondence button
  • Select Submit
  • Message displays Are you sure you want to proceed skipping the correspondence?, select Yes

Issue a Review of decision – Extensions (Z2804)

Table 3

Step

Action

1

Evaluate – Extension + Read more ...

When reviewing an extension decision:

  • Select Statement by claimant from the dropdown menu
  • Select Does the claim selected match the details in the document? button
  • Select Is an extension to be processed? button
  • Select Continue

A message displays Have you entered a case note? Would you like to continue processing Statement by claimant task or assign to the queue?

If the Service Officer is:

  • proficient and has APS4 delegation:
    • select Keep assigned to me
    • select Ok
  • not proficient or is not delegated:
    • select Assign to the queue
    • select Ok

2

Capture extension details + Read more ...

Extension request received date defaults to the original extension received date.

If the review decision is:

  • substituted or varied
    • select Do you grant the request? button
    • select Submit
    • go to Step 3
  • affirmed
    • do not select the Do you grant the request?, select Submit
    • go to Step 3

3

Issue correspondence – Z2804 + Read more ...

  • Select the Notice type from the dropdown menu
  • Select Extension from the Letter purpose dropdown menu
  • Select the Review decision from the dropdown menu
  • Select the Original extension decision from the dropdown menu
  • Depending on data entered select the relevant reason for the decision
  • Select the correct recipient and select Preview to view the letter content
  • Select Submit to issue the letter
  • Add a case note to document this process

Table 4

Action

APS4 (or above) to action a reconsideration of decision + Read more ...
