Assessing value of assets as security for the Scheme

Example

Description

1

Real asset value calculation + Read more ...

To determine the real asset value of the real estate offered as security for a loan under the Scheme, the current market value is reduced by the deductions and Nominated Amount (if any). The real asset value is worked out using the following formula:

Real asset value = current market value minus (Nominated Amount + deductions)

Example:

  • Current market value = $350,000
  • Nominated Amount = $45,000
  • Deductions = $17,500

Real asset value = $350,000 – ($45,000 + $17,500) = $287,500

2

Maximum Loan Amount calculation – single customer + Read more ...

A 70 year old single person offers a property valued at $180,000 as security for the loan. A mortgage of $80,000 is owed on the property. The maximum loan amount is calculated as follows:

  • Age component amount based on age 70 = $3,080
  • Current market value = $180,000
  • Deductions = $80,000
  • Real asset value: $180,000 - $80,000 = $100,000

Maximum Loan Amount = $3,080 x ($100,000 / $10,000) = $30,800

3

Maximum Loan Amount calculation – couple + Read more ...

A couple aged 72 and 68 years of age, offer property valued at $240,000 as security for a loan to each of them. The property is jointly owned by the couple (50% each), with a mortgage of $90,000.

The younger partner's age is used for the age component, and the maximum loan available for each member of the couple is calculated as follows:

  • Age component amount based on age 68 = $2,850
  • Current market value = $120,000 each (i.e. $240,000 divided by 2. This is the value of each partner’s share of the asset).
  • Deductions = $45,000 each (i.e. $90,000 divided by 2. This is the value of each partner’s share of the allowable deductions)
  • Real asset value = $120,000 - $45,000 = $75,000. After rounding down to nearest $10,000, this equals $70,000

Maximum Loan Amount: $2,850 x ($70,000 / $10,000) = $19,950
