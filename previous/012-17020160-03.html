All speech pathology items SH01-SH99

Processing Rules + Read more ...

  • Maximum time payable = 150 minutes
  • Times need to be rounded to the nearest 15 minutes. E.g. 48 mins would be keyed as 0:45hr; 78 mins would be keyed as 1:15HR
  • Key the time in the FEE field as H:MMhr and press [ENTER]
  • If the provider has claimed an amount without times, reject with RSN code S 333
  • If time claimed exceeds max limit, reduce time to max limit and insert RSN code 362

Note: services that are rendered away from the provider's rooms attract a loading in addition to the normal time-based fee.

This loading is equal to one quarter of the normal hourly rate.

The system will automatically apply the loading for applicable items.

SH28, SH29, SH74

LSVT LOUD – Lee Silverman Voice Treatment Program + Read more ...

Fee based item

  • Limit of 18 services in 365 days from any provider
  • Time does not need to be keyed for manual claims, it will pay as a standard fee

Note: online and webclaim providers can key/select any time (in 15 minute increments) in the TIME DURATION field.

Restrictions

  • If the item restricts with itself, check PARS for an approval
  • If approval is present pay service with PI 45
  • If approval is not present reject with RSN code S 160

Loading

Out of Rooms loading + Read more ...

Out of Rooms loading is automatically added to consultations delivered outside of rooms. Only 1 loading per visit is permitted, and it is expected that providers will perform the evaluation of assessments in their rooms.

SH90

Allied Treatment Cycle Report + Read more ...

Provision of report to GP on outcomes of Treatment Cycle related to the referral.

Rules

  • An initial consultation item in the allied fee schedule is claimed
  • Any number from 1 to 11 subsequent consultation items from the same fee schedule are claimed.
  • Only one report item is payable for each initial consultation
  • The provider must claim 1 initial consultation and 1 subsequent consultation
  • Cannot be paid twice on the same day
  • Limit of 1 report item payable within 14 days
  • Limit of 12 report items payable within 12 months

Restrictions

  • Associated consult not claimed, reject with RSN code S 748
  • If the item restricts with itself or any other item, reject item. Use lower fee with RSN code S 159

SH99

Report or service specifically requested by DVA + Read more ...

Requires Prior Financial Authorisation.

Is only used when DVA contacts the providers directly to request:

  • a written report
  • a consultation or assessment to eligible veteran or war widows/ers

The kilometre allowance is included in the fee and is not to be claimed separately.
