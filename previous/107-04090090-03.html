Completing an activity that is triggering Family Tax Benefit (FTB) reconciliation where an undetermined FTB qualification debt is included in the result + Read more ...

Identifying undetermined debts included in the reconciliation result:

  • Code 'U' next to the reconciliation result on the Assessments Results (AR) screen. The FTB Confirm Reco Amounts Summary (FCRAS) screen will display Undetermined Debts in the Type field

or

  • Code 'V' next to the reconciliation result on the AR screen
  • 'S'elect the reconciliation result on the FAO Reconciliation & Claim Summary (FRCS) screen
  • On the FTB Calculation Result (FACRF) screen, scroll down [PF8] and 'S'elect the Total Auto/Man/Special Adjustment line which will have an amount in the ‘Reco’ field
  • To view the FTB Special adjustment Periods/Reasons on the FTB Total Adjustment Period Display (FTAPD) screen, code 'Y' in the Go to FCRAS Screen
  • The FCRAS screen shows Undetermined Debts in the Type field

Identifying qualification debts included in the completed reconciliation result: + Read more ...

  • 'S'elect the reconciliation result on the FAO Reconciliation & Claim Summary (FRCS) screen

  • To view FTB Special Adjustment Periods/Reasons, on the FTB Total Adjustment Period Display (FTAPD) screen, code 'Y' in the Go to FCRAS Screen. This shows the FTB Special Adjustment Periods/Reasons field
  • The FCRAS screen shows Undetermined Debts in the Type field
  • Go to Step 4

4

Is it a qualification debt? + Read more ...

A qualification debt is a debt other than a debt due to changes in income or maintenance income.

FTB undetermined debts appear:

  • on the Activity List (AL) screen, and
  • within STAarted reconciliation activities on reconciliation screens. For example FRCS

If the qualification debt has already been included in a completed FTB reconciliation result, select the debt from the OPDL screen in DMIS.

View the Activity Management Reference (AMR) number that created the debt, to identify whether an undetermined debt relates to qualification or income.

Is it a qualification debt?

  • Yes, go to Step 5
  • No, the activity can be finalised as income related debts are included in the reconciliation process. Procedure ends here

Actioning an undetermined qualification debt

Note: due to the pause on debt raising and recover, Steps 1, 2 and 3 will not apply until further notice unless it is necessary to raise the qualification debt to apply a waiver.

Table 2

Step

Action

1

Refer to Debt Raising Team + Read more ...

Refer to the Debt Management Team to finalise the undetermined FTB qualification debt(s). Send an ACTDOC using the Debt Raising MIV FFRECO Request Fast Note Template:

  • Include the relevant debt ID number(s)
  • Transfer to region code XWA

The Debt Management Team will determine the debt shell within 3 working days. An ACTDOC will be returned to the Smart Centre for the original activity to be re-coded and finalised.

2

Advise the customer: + Read more ...

  • the timeframe to have their FTB reconciliation finalised
  • they will be sent a letter once their FTB reconciliation has been completed or they can view the results using Online Services

3

Check for an ACTDOC from Debt Raising Team + Read more ...

Regularly check Workload Management. Finalise the reconciliation activity as soon as the XWA team have determined the debt/s.

The Service Officer/Service Support Officer is required to regularly check Workload Management in Customer First to ensure that the reconciliation activity is finalised as soon as the Debt Raising Team has determined the debt/s.

4

Debt waiver provisions + Read more ...

Raise a manual debt if the debt meets waiver provisions contained in Section 97 or 101 of the Family Assistance (Administration) Act 1999. Include the same information as the debt that was incorrectly included in reconciliation.

See:

  • Create a new debt record manually on the Debt Management Information System (DMIS)
  • Administrative error waivers
  • Special and unusual circumstances waivers
  • Waiving Centrelink debts

Refer to Debt Management Team

Send an ACTDOC using the Fast Note Template:

  • Include the debt ID number(s) and all relevant information
  • Transfer to region code XWA

The Debt Management Team will manually raise the debt/s and return an ACTDOC to the relevant site.

5

Check for ACTDOC from Debt Management Team + Read more ...

Regularly check Workload Management to ensure the waiver process is finalised as soon as the Debt Management Team have raised the manual debt/s.

6

Waiving the manually raised debt + Read more ...

Conduct a FTB 'Refresh' (FSH) activity on the Benefit (BA) screen, once the manual debt is ready for waving. Use 1 July of the relevant financial year as the date of effect.

At the Assessment Results (AR) screen, code 'U' next to the applicable FTB line. This will go to the FTB Confirm Reco Amounts (FCRA) screen.

Before completing the activity, accept the adjustment on the FCRA screen.

Period From and Period To fields show the period of the waived debt. Accept the adjustment by using a manually raised debt code.

Check the result on the FRCS screen.

FTB will re-reconcile once all actions are complete. The re-reconciliation result will issue the debt amount back to the customer.

Complete the activity on the AR screen.

Waive the manually raised debt.
