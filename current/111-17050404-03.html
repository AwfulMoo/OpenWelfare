Search work items and Assign to Me

Table 1: how staff can search documents and assign to themselves in Work Optimiser - Medicare.

Step

Action

1

Navigate to PaNDA Search + Read more ...

  • Launch Work Optimiser - Medicare desktop icon
  • Select the PaNDA Search tile under the Health Services menu

2

Enter search parameters + Read more ...

Enter mandatory fields:

  • Program
  • Work Type

Depending on the Program and Work Type, extra search fields will display.

Enter any of the following details if known:

  • Status
  • Case ID
  • Date of Lodgement

Refine the search results by entering more information.

3

Search results + Read more ...

  • select Search
  • view results in the Results pane

Select a work item to view more details and image.

4

Select action + Read more ...

  • Select a work item from the search results
  • Select Assign to Me to assign selected work item(s) to self
    • The Work Item Detail screen will change from read-only to editable
    • A message at the bottom of the screen will advise the work item was successfully assigned

See Worklist in Work Optimiser for staff to view and manage all work assigned in Work Optimiser - Medicare.

Create new work items

Table 2: how staff use the Document Upload function to create a new work item in Work Optimiser - Medicare.

Step

Action

1

Access Document Upload + Read more ...

Launch Work Optimiser - Medicare desktop icon.

Select PaNDA Document Upload tile under the Health Services menu.

2

Upload document + Read more ...

Upload a document to create a new work item:

  • select Create New Work Item from Upload Type field
  • select Browse, a pop up window will display, select a file
  • Date of Lodgement defaults to today’s date, correct if needed
  • select Program from the drop down menu
  • select Work Type from the drop menu
  • Case ID and Processing Notes may be added but are not mandatory
  • for remaining mandatory fields (*) refer to the relevant program Operational Blueprint
  • select Upload to submit the new work item into the PaNDA allocation pool

A message will display at the bottom of the screen advising that the document was successfully uploaded.

Add a document to an existing work item

Table 3: how staff use the Document Upload function to add a document to an existing work item in Work Optimiser - Medicare.

Step

Action

1

Document upload + Read more ...

A document can be uploaded and attached to both an open and completed work item.

To add a document to:

  • an open work item, there are 2 ways:
    • PaNDA Document Upload tile under the Health Services menu, go to Step 2
    • Upload feature in the work item, go to Step 3
  • a completed work item, use the Upload feature in the completed work item. Go to Step 4

2

Access Document upload + Read more ...

Launch Work Optimiser - Medicare desktop icon.

Select PaNDA Document Upload from the Health Services menu.

Upload a document and add to existing work item:

  • select Add to Existing Work Item from Upload Type field
  • select Browse, a pop up window will display, select a file
  • Date of Lodgement defaults to today’s date, correct if needed
  • work items from the Worklist will display as a list with radio buttons to the left
  • to add a document, select the radio button to the left of the Work Item ID
  • select Upload

A message at the bottom of the screen says the document was successfully uploaded to the work item.

Procedure ends here.

3

Access work item for upload + Read more ...

Launch Work Optimiser - Medicare desktop icon.

Select Worklist from the Work management menu.

Select work item for uploaded documents to be attached.

Upload a document and add to existing work item:

  • select Upload
  • select Browse, a pop up window will display. Select a file
  • Date of Lodgement defaults to today’s date, correct if needed
  • select Upload

A message at the bottom of the screen says the document was successfully uploaded to the work item.

Procedure ends here.

4

Access completed work item for upload + Read more ...

Launch Work Optimiser - Medicare desktop icon.

Select PaNDA Search from the Health Services menu:

  • Enter mandatory search parameters and select Search
  • Select work item to display the Work Item Details page

Upload a document and add to existing work item:

  • select Upload Document
  • select Browse, a pop up window will display. Select a file
  • Date of Lodgement defaults to today’s date, correct if needed
  • select Upload

A message at the bottom of the screen says the document was successfully uploaded to the work item.

Procedure ends here.

Associate document to a work item

Table 4: how staff associate documents within a worklist in Work Optimiser - Medicare.

Step

Action

1

Access work item + Read more ...

Launch Work Optimiser - Medicare desktop icon.

Select the Worklist tile from the Work Management menu.

Documents can be associated from open or completed work items to other open or completed work items.

To associate:

  • an open work item to another open or completed work item, go to Step 2
  • a completed work item to another open or completed work item, go to Step 3

2

Associate documents from an open work item + Read more ...

Select the work item for association.

This will open the Work Item Details page:

  • select the Associate Documents button. The default option will display all of the open work items in the Worklist with tick boxes to the left
  • select the box to the left of the work item to be associated
  • select the Selected Work Items button to view the work item chosen for association
  • select Associate

A message at the bottom of the screen says the work item was successfully associated.

All documents will now display in the Work Item Details page for the originating work item.

This will remove all documents from the work item selected for association. The status of the (orphaned) work item will change to Cancelled.

This action can be reversed. See Table 5.

3

Associate documents from a completed work item + Read more ...

Select the completed work item for association.

This will open the Work Item Details page:

  • Select the Associate Documents button
  • In the Type drop down menu, select Completed Work Items
  • Search for completed work item using the following criteria:
    • Program prefills based on the work item to be associated
    • Work Type prefills based on the work item to be associated
    • Case ID
    • Primary Identifier
    • Secondary Identifier
  • Select the box to the left of the work item to be associated
  • Select the Selected Work Items button to view the work item chosen for association
  • Select Associate

A message ay at the bottom of the screen says the work item was successfully associated.

All documents will now display in the Work Item Details page for the originating work item.

This will remove all documents from the work item selected for association. The status of the (orphaned) work item will change to Cancelled.

This action can be reversed. See Table 5.

Remove documents from a work item

Table 5: how staff can remove documents within a work item in Work Optimiser - Medicare. This feature can be used on work items with 2 or more documents.

Step

Action

1

Access work item + Read more ...

Launch Work Optimiser - Medicare desktop icon

Select the Worklist tile from the Work Management menu.

Select the work item from the Worklist.

This will open the Work Item Details page.

2

Unassociate documents + Read more ...

In the Documents table:

  • Select the Delete button in the Delete column that is aligned with the document for removal.
  • A warning will ask the user to confirm they want to remove the document.
    • Select Delete to remove the document
    • Select Cancel to go back

A record of the removed document will be added to the Work Item History screen with a FileNet link to the original document.

Splitting work items

Table 6: how staff can split documents to a work item in Work Optimiser - Medicare.

Step

Action

1

Document identified as needing splitting + Read more ...

If a document needs splitting based on business expectations and needs, use Adobe to extract pages from the document.

The extracted pages will upload to create a new work item in PaNDA.

2

Access Adobe Acrobat Reader + Read more ...

Use Adobe to extract needed page/s from the original work item document.

Save the extracted pages to desktop.

To upload the newly extracted work item, see Table 2

Date of Lodgement: use the date from the original work item.

3

Delete from desktop + Read more ...

After uploading the new document and creating a new work item in PaNDA:

  • delete the extracted pages from desktop
  • on the original PaNDA work item, add a processing note to indicate that documents in this file have been extracted and uploaded as a new work item
