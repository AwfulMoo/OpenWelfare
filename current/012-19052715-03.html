Assess an initial or subsequent application for a provider number

Table 1

Step

Action

1

Assess applicant details + Read more ...

Ensure the mandatory information below has been provided:

  • Application type
    If Subsequent selected, key the Medicare Provider number if provided. If not provided, continue assessing the form to determine if the health professional can be identified from other application details
  • Applicant's details
    • Family name, given name
    • Date of birth. If not provided, the application cannot be completed. Go to Table 8

If only the surname field has been completed check the registration to see if the applicant has only one name:

  • One name only, continue assessing the form. Go to Step 2
  • More than one name and registration numbers:
    • match, insert missing details. Go to Step 2
    • do not match, the application cannot be completed. See Table 8

2

Assess health profession and qualification details + Read more ...

Ensure the mandatory information below has been provided:

  • Health profession indicated
    If multiple selected, return the form and advise that separate applications are required for each profession. See Table 8
  • Profession qualification
  • Place obtained
  • Year obtained

If any of this information is not provided the application cannot be processed. See Table 8.

3

Assess contact details and registration + Read more ...

Update the personal contact details and registration if provided.

  • Postal address
  • Phone, mobile and email details are not mandatory
  • Registration and/or membership number
    Note: not applicable to Aboriginal Health Workers

4

Assess location details + Read more ...

If the health professional has indicated they are applying for more than 1 location, ensure that:

  • all details in the required location section are attached for each additional location, and
  • questions are all completed for each additional location

If the location details are incomplete, see Table 8.

Alternative address requests

If the application indicates an alternative location address, ensure the:

  • applicant has lodged a written request, and
  • all required information supplied. See Provider number for health professionals in Medicare for what is needed for an alternative address request

If a written request was not received, or does not contain all details needed to make an assessment:

  • make one attempt to call the health professional to obtain the outstanding details(s). Perform telephone security check
    • If the information is not obtained over the phone, See Table 8
    • If the information is obtained over the phone, assess the health professional's eligibility to use an alternative address

If the request meets all eligibility criteria below, process the application using the alternative address location:

  • Premises type - must be Home or Mobile
  • Consultations - only telehealth or mobile. No face-to-face consultations at the home location
  • Reason - is due to safety concerns

Note: for information on recording alternative addresses, see Step 1 in Table 3.

If the request does not meet all of the above criteria:

  • make one attempt to call the health professional. Perform telephone security check
    • Discuss the reason they are unable to use another location address
    • Ask the health professional if they still want the provider number application processed
    • If no provider number is needed, see Table 8
    • If provider number is needed, process the application using the physical location address
    • If unable to make contact with the applicant over the phone, see Table 8

Claiming Medicare benefits

Ensure either Yes or No is indicated for claiming Medicare benefits from the location/s.

Applicant has ticked 'Yes'

Is business structure and banking details provided in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, make one attempt to call the health professional to obtain business structure and/or banking details. Perform telephone security check
    • If information is obtained over the phone, issue provider number for access to Medicare benefits
    • if unable to obtain information over the phone, see Table 1 on the Resources page

Applicant has ticked 'No'

Is business structure and banking details provided in full?

  • Yes, issue provider number to access to Medicare Benefits
  • No, service rule the location and issue provider number for refer and request only

No boxes ticked

Is business structure and banking details provided in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, make one attempt to call the health professional to obtain business structure and/or banking details. Perform telephone security checks
    • If information is obtained over the phone, issue provider number for access to Medicare benefits
    • If unable to obtain information over the phone, see Table 1 on the Resources page

Both boxes ticked

Is the business structure and banking details provide in full?

  • Yes, issue provider number for access to Medicare benefits
  • No, make one attempt to call the health professional to obtain structure and/or banking details. Perform telephone security check
    • If information is obtained over the phone, issue provider number for access to Medicare benefits
    • If unable to obtain information over the phone, see Table 1 on the Resources page

Note: location start date and end date are optional.

5

Assess employment details + Read more ...

Assess the below employment information:

  • Employment status
    This is mandatory if claiming Medicare benefits. If not completed, or multiple options selected, go to Table 8
  • ABN/ACN and Registered business name
    This is mandatory if claiming Medicare benefits. If not completed, see Table 8
  • Trading As
    Not required if employment status is Sole Trader, Individual proprietor, or Company. Trading name will be individual's name
  • Business type and Premises type
    This is mandatory if claiming Medicare benefits. If not completed, see Table 8

If the application form is received requesting access to Medicare benefits and the business structure information has not been supplied:

Try to contact the health professional on supplied contact information.

Is contact successful?

  • Yes,
    • Perform a security check
    • Confirm the business structure and record the details in PDS
    • Add a note to the PANDA file
  • No,
    • Add note to PANDA file indicating call unsuccessful and contact not made
    • Continue processing the application according to current business processes
    • Send provider number confirmation letter

6

Medicare Online/Easyclaim and bank details + Read more ...

If the health professional is claiming Medicare benefits, the details below must be completed:

  • Medicare Online (Y/N). If the health professional indicated Yes, ensure the Minor ID is provided
    • If there is no Minor ID, a default Minor ID will be used. For example, AHP30001 for Victoria. See Major and Minor IDs for more details
    • If Minor ID is invalid, use the relevant allied health professional minor ID when processing the application. See Major and Minor IDs for more details
  • Medicare Easyclaim (Y/N)
    • If the health professional indicated Yes, ensure the EFTPOS device supplier has been provided.
    • If there is no device supplier, a default Minor ID will be used. For example, AHP30001 for Victoria. See Major and Minor customer identifiers (ID) for eBusiness Service Centre
    • If it is invalid, use the relevant allied health professional minor ID when processing the application. See Major and Minor customer identifiers (ID) for eBusiness Service Centre
  • Bank account details
  • Try to contact the provider on supplied contact information

Is contact successful?

  • Yes,
    • Perform a security check
    • Confirm the business structure and record the details in PDS
    • Add a note to the PaNDA file
  • No,
    • Add note to PANDA file indicating call unsuccessful and contact not made
    • Continue processing the application according to current business processes
    • Send provider number confirmation letter

7

Prescriber number and declaration + Read more ...

Check if the health professional indicated wants to apply for a Prescriber number (if applicable):

If Yes, ensure health professional is eligible - see Prescriber number eligibility.

If the health professional is not eligible the application cannot be processed. see Table 8

Declaration must be signed. The date is optional.

If not completed, return the form as incomplete. See Table 8

For more details about prescriber numbers, see Prescriber number for health professionals in Medicare.

8

Evidence of accreditation/registration + Read more ...

For initial application, check that evidence of accreditation/registration is attached.

  • Where certification is not supplied for an Ahpra registered health professional:
    • access the Ahpra window if a shell record is present, or
    • go to the Ahpra website to confirm registration. See Resources page.
  • Where certification is not supplied for a non-Ahpra health profession, the application must be returned

See also PDS field codes for allied health and non-medical health professionals

Note: if the application is for a subsequent location, evidence of accreditation/registration is not required. Ahpra and other professional bodies, such as Dietitians Australia (DA), notify Services Australia of any changes to registration/eligibility.

Additional health professional accreditation information can be found on the individual allied health and non-medical health professional Operational Blueprint pages. For example, Osteopath.

Has the required registration evidence been provided (for initial applications only)?

  • Yes, see Table 2
  • No, and the health professional is:
    • Ahpra accredited professional, go to Step 9
    • Non-Ahpra accredited professional, see Table 8
    • If certification is not supplied for a non-Ahpra health profession, the application must be returned

See also PDS field codes for allied health and non-medical health professionals

9

Confirm Ahpra eligibility + Read more ...

If the Ahpra certificate is not attached, check the Ahpra Register of practitioners:

  • If certification is not supplied for an Ahpra registered health professional, access the Ahpra window if a shell record is present or go the Ahpra website to confirm registration
  • Enter details in the following fields in the APHRA website:
    • Name
    • Registration number, and
    • Health profession, select Search. Advanced Search allows users to search as above and with the locality. Take a screenshot of the search result and add it to the electronic file

If the Ahpra registration search is:

  • successful, go to Step 10
  • not successful, the application needs to be returned as incomplete. See Table 8

10

Compare Ahpra registration to application form + Read more ...

  • Select View Details to open up the registration record
  • Check the following details match the applicants:
    • Name
    • Profession (for example, Psychologist, Osteopath, Podiatrist)
    • Registration Number
  • Registration Status should be Registered
  • Registration Type should be either Current, Provisional or Limited
    • Psychologists with Provisional or Limited registration types are not eligible for Medicare
  • Registration Expiry Date should be current

Note: for Psychologists only - If the Endorsements field shows Approved area of practice – clinical psychology register the applicant as a Clinical Psychologist.

  • Initial applications, see Table 2
  • Subsequent applications, see Table 3

Process an initial provider number application

Table 2

Step

Action

1

Check for an existing provider stem + Read more ...

Conduct a search in Provider Directory System (PDS) to see if there is an existing stem:

  • Open PDS
  • Select Provider
  • Select View
  • Select Detailed Search
    • In the Family name key the practitioner's surname
    • In the Given name field key given name
    • Enter date of birth in Date of Birth field
    • Select gender
    • Select profession
    • Select Search.
    • A list of health professionals with matching details will show
  • Check if the health professional on the application matches any previous records

Note: the health professional's name in PDS must match the name registered with their accreditation body (for example, Ahpra)

Does the health professional have a previous record?

  • Yes, check for an existing stem. If the existing stem is for:
    • a different profession, a new stem must be created for this application. Contact a Programme Support Officer (PSO) for further assistance if needed. Go to Step 2
    • the same profession, update the provider details. Go to Step 5
  • No, go to Step 2

2

Generate new stem details + Read more ...

In PDS:

  • select Provider
  • select Create
  • key the Family Name and Given Name as per the name details of registration/accreditation and select Search
  • if a shell record is available, view and confirm it is the appropriate shell record for the provider and profession requested
    Note: if no shell record is available for dental hygienists, dental therapists and oral health therapists cease processing and contact a Programme Support Officer.
  • for health professionals other than dental hygienists, dental therapists and oral health therapists select Create New Provider and manually key the stem details

3

Record personal details + Read more ...

The applicant’s name must appear in PDS exactly as their registration/accreditation.

If the health professional has put a middle name on the application but it does not appear on registration/accreditation, key the middle name as per the application.

  • Key the Title, Date of Birth and Gender as per the application
  • Key the language if provided
  • All other fields in the Personal tab are left blank

4

Record qualifications and issue a prescriber number + Read more ...

Enter the qualification in full as per their health profession:

  • Key the relevant Qualification code in the Qualifications field
  • Select Place of base qualifications from the drop down
  • Enter the Year obtained as per the application
  • Select No in Sonographer field
  • All other fields in the Qualifications tab are left blank

Note: applicants who satisfy the criteria of more than 1 health profession will be allocated a separate provider stem for each profession. The health professional must supply a separate application for each profession.

Issue a prescriber number

Any allied health and non-medical applicants requesting a prescriber number must be assessed for eligibility.

For a list of eligible allied health and non-medical health professionals, see Prescriber number eligibility

See also PDS field codes for allied health and non-medical health professionals

5

Record the contact details + Read more ...

  • Begin by keying into the Locality or Postcode field. The rapid address tool will show
  • Repeat the process with the address details. For example 123 SMI for 123 Smith Street
  • If a non-standard address has been provided, select Manual Entry and key the address manually. This overrides the rapid address tool

6

Record the registration details + Read more ...

  • Select Add New Registration
  • Record the Area
  • Key the Registration Number:
    • Ahpra registration number, for example PSY##########, or
    • Non-Ahpra registration number as per evidence of registration information. For example, accreditation certificate ID Number
  • Record the Status
  • Key the Registration Start Date
  • Select Add New, then Cancel

Note: for dental hygienists, dental therapists and oral health therapists registration should only be pre-populated by promoting the Ahpra window. Do not amend the details populated.

7

Record the specialty details + Read more ...

  • Select Add New Specialty
  • Key the health professional specialty code in Specialty Code field
  • Key the Start Date as the same date recorded in the Registration details tab (Step 6)
  • End Date: no end date is required
  • Select Add New, then Cancel

Step

Create location in Provider Directory System (PDS) + Read more ...

To create a location in PDS:

    • In the Given name field key given name

5

Apply Provider Service Rule (PSR) to location + Read more ...

A PSR is applied when an applicant indicates they do not intend to claim Medicare benefits at the location provided on the form. Where eligible, the health professional will only be able to refer and/or request at this location.

If the applicant is:

  • claiming Medicare benefits, go to Step 6
  • not claiming Medicare benefits, ensure a PSR is added at the location level
    • If the health professional is not eligible, the application cannot be processed. See Table 8

To add the PSR in the Loc-Service Rules tab in PDS:

  • Select Add Service Rule, the Add New pop up box will show
  • Enter PSR 618 in the Service Rule Number field
  • Select Yes in the Activity Code field
  • Start date is the same as the start date of the location
  • End date is not required

6

Record location organisation details + Read more ...

  • Select the Employment Type from the drop down as per the form
    • Employee Salaried form field is the Employee - Payroll Provider field in PDS
    • Employer - Employer field in PDS is not used
  • Select Add under Organisation Details
  • Enter the ABN provided and select Search
  • Select relevant record:
    • if multiple ABN options return, select the matching record
    • if results don’t match the form, select the first result returned
  • Enter Trading As details as per the form
    If blank, enter the Registered Business Name in Trading As field
  • Select the Business Type from the drop down menu as per the form
    Associateship and Government or Other Public Body fields are not used in PDS
  • Select Premise Type from the drop down menu as per the form
  • Select Save

If the ABN and trading name are the same:

  • Select the Business Type from the drop down as per the form. Associateship and Government or Other Public Body fields are not used in PDS
  • Select Premises Type from the drop down as per the form
  • Select Save

Multiple practices

There can be multiple practices at the same location with the same ABN, however the trading name would be different.

Where a health professional has requested:

  • updates to an organisation linked to a specific location, access the Organisation tab
  • make the necessary changes as indicated by the health professional
  • to remove an organisation, escalate to a Team Leader

7

Save the record + Read more ...

  • Select Save
  • Action any return messages, for example Mandatory fields non populated
  • Select Save

The location is now created.

  • File the application as processed in line with current workflow procedures

    • In the Given name field key the first 3 letters of the first name followed immediately by a %

  • Yes, go to Step 3

Is the registration line Current (no end date listed)?

  • Yes, contact the health professional by phone and tell them they are currently registered for Medicare purposes and no further action is required:
    • Add note about the phone call on the PaNDA record
    • Select: Complete
    • Select Save
  • No, go to Step 4

4

Update PDS + Read more ...

Select the Registration tab. Assess if there is a gap on the provider stem.

For example, close of registration was 01/07/18, start date is 03/07/18; gap is on 02/07/18.

  • If there is a gap:
    • Select Add new registration
    • Select the Area from the drop down
    • Registration Number - key the number provided
    • Record the Status
    • Key the Registration Start Date
    • Comments: Add a note stating 'Gap as per registration advice from <accrediting body>'. For example, 'Gap as per registration advice from Australian Mental Health Nurses Association (ACMHN)'
    • Select Update
    • Select Save
    • Go to Step 5
  • If there is no gap:
    • Select Amend on the closed registration line
    • Record the Status (change from Other)
    • Remove the end date
    • Comments: Add a note stating 'No gap as per registration advice from <accrediting body>. For example, No gap as per registration advice from Australian Mental Health Nurses Association (ACMHN)
    • Select Update
    • Select Save
    • Go to Step 5
