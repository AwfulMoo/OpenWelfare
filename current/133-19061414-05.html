OLS1 account history codes used in Consumer Directory Maintenance System (CDMS)

Code

Explanation

Added program password - program OLS reason FIRST

Completed registration for Medicare online account.

Program interaction - program OLS type PREQD

Status changes to 'deleted' once PSENT process completed.

Program interaction - program OLS type PSENT

Password sent to customer by mail.

Amend program password - program OLS reason CRESET

Password reset by customer:

  • they created the password during logon or in their Medicare online account
  • they requested a new password letter online via the Forgotten Password page

Added program questions and answer - program OLS

Secret question set up.

Program interaction - program OLS type ESENT

Email sent to customer to reset password.

Program interaction - program OLS type ERESET

Password reset by customer using instructions in email.

Deleted program password - program OLS reason ORESET

Password reset by staff member. 'P' number shows in the next column.

Deleted program entitlement - program OLS type OLS

Medicare online account registration cancelled.

OLS2 account history codes used in CDMS

Code

Explanation

Added program registration - program OLS2

The customer linked Medicare services to their myGov account by confirming their existing Medicare online account.

Added program entitlement - program OLS2 type OLS2

The customer linked Medicare services to their myGov account by confirming their Medicare record.

Amended program entitlement - program OLS2 type OLS2 (CANCELOLSREGISTRATION)

The customer linked Medicare services to their myGov account. They have since cancelled their Medicare online account.

They can reactivate their account by re-linking Medicare services to myGov.

Note: if they chose to cancel and prevent access, a staff member will need to reset access before they can re-link Medicare services to myGov.
