Item

Description

PH10, PH11, PH15, PH19

Initial consultation - rooms; in-home; private hospital; low care residential aged care facility (RACF) + Read more ...

After 1 October 2019

Rules

Allied Health treatment cycle

  • Limit of 1 service in 14 days from any provider
  • Limit of 12 services in 365 days from any provider
  • Treatment Cycle does not apply to DVA clients who have a Totally and Permanently Incapacitated (TPI) Gold Card

Restrictions

If an initial consultation restricts with:

  • itself or another physiotherapy initial consultation item, reject with RSN code S 160
  • any other item, reject the item with the lower fee with RSN code S 159

Prior to 1 October 2019

Rules:

  • initial consultation limit of 3 in 365 days
  • only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient

Restrictions:

If an initial consultation item restricts with:

  • itself, reject with RSN code S 160
  • any other item, reject item with lower fee with RSN code S 159

PH12, PH16, PH14, PH18

Initial consultation - public hospital; high care RACF + Read more ...

Services Australia will only pay for health care services carried out in public hospitals and high care residential aged care facilities in exceptional circumstances and only when the Department of Veterans' Affairs (DVA) has given prior financial authorisation.

Rules

  • only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient

Requires Prior Financial Authorisation.

Check PARS for an approval.

When Prior Financial Authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283

PH13, PH17

Initial consultation - rooms; in-home; private hospital; low care residential aged care facility (RACF) + Read more ...

After 1 October 2019

Rules

Allied Health treatment cycle does not apply to allied health treatment provided to DVA clients while they are admitted to hospital

  • Limit of 12 services in 365 days from any provider
  • Treatment Cycle does not apply to DVA clients who have a Totally and Permanently Incapacitated (TPI) Gold Card
  • Payable with either PH23 or PH27 on the same day, to a limit of 2 services per day

Restrictions

If an initial consultation restricts with:

  • itself or another physiotherapy initial consultation item, reject with RSN code S 160
  • any other item, reject the item with the lower fee with RSN code S 159

Prior to 1 October 2019

Rules

  • initial consultation limit of 3 in 365 days
  • only 1 consult item can be claimed per day

Standard consultation items

Table 2:

Item

Description

PH20, PH21, PH25, PH29

Standard consultation - rooms; in-home; private hospital; low care residential aged care facility (RACF) + Read more ...

Subsequent consultation items

Rules

  • Only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient
  • From Dates of service after 12 February 2016, prior approval is required when claiming more than 1 consultation per patient per day
  • In certain exceptional circumstances, DVA may pay for 2 subsequent consultations provided on the same day in circumstances where it is for treatment of a short term acute onset condition such as treatment of an admitted entitled person with acute pneumonia or for joint mobilisation immediately following surgery. The length of time for providing these services should not exceed 5 days without contacting DVA. When claiming, providers should indicate AM or PM next to each consultation or the time of each consultation to allow processing by Services Australia

Restrictions

If item restricts with itself on same date of service, check PARS for an approval:

When approval:

  • present, pay service with PI 42
  • not present, reject with RSN code S 160

If item restricts with any other physiotherapy item on the same day, Check PARS for an approval. When approval:

  • present, pay service with PI 42
  • not present, reject with RSN code S 159

PH22, PH26, PH24, PH28

Standard consultation - Public Hospital; High care RACF + Read more ...

The department will only pay for health care services carried out in public hospitals and high care residential aged care facilities in exceptional circumstances and only when DVA has given prior financial authorisation.

Rules

  • cannot be claimed on the same day as an initial consultation for the same patient
  • 2 subsequent consultations cannot be claimed on the same day

Requires Prior Financial Authorisation.

Check PARS for an approval.

When Prior Financial Authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283

PH23, PH27

Standard consultation - private hospital + Read more ...

Subsequent consultation items

Rules

  • 2 private hospital subsequent consultation items can be claimed per day for the first 5 days from admission
  • Can be paid with an initial (either PH13 or PH17) on the same day
  • A third consultation on the same day cannot be paid and must be reviewed by DVA
  • 2 consultations can be claimed on the same day, not exceeding 5 days from date of admission to hospital, for a short term acute onset condition. When claiming, providers should indicate AM or PM next to each consultation or the time of each consultation
  • The Allied Health treatment cycle does not apply to allied health treatment provided to DVA clients while they are admitted to hospital

Restrictions

When 2 services are claimed on the same day - check for AM/PM or the time of each consultation:

  • Times present, Yes - only 2 x consultations per day allowed for the first 5 days from hospital admission, check history:
  • Less than 5 days of 2 x consults claimed = ok to pay PI 42
  • More than 5 days of 2 x consults claimed = reject RSN 160
  • Times not present – Reject with RSN 160

If item restricts with another initial and subsequent on same date of service, check PARS for an approval.

If Prior Financial Authorisation is:

  • present, pay service with PI 45
  • not present, reject service with RSN 160

Physiotherapy Treatment Cycle Report

Table 6:

Item

Description

PH90

Allied Treatment Cycle Report + Read more ...

Provision of report to GP on outcomes of Treatment Cycle related to the referral.

Rules

  • An initial consultation item in the allied fee schedule is claimed
  • Any number from 1 to 11 subsequent consultation items from the same fee schedule is claimed
  • Only 1 report item is payable for each initial consultation
  • The provider must have claimed 1 initial and 1 subsequent consultation
  • Cannot be paid twice on the same day
  • Limit of 1 report item payable within 14 days
  • Limit of 12 report items payable within 12 months

Restrictions

  • Associated consult not claimed, reject with RSN code S 748
  • If the item restricts with itself or any other item, reject item. Use lower fee with RSN code S 159

Physiotherapy consumable and appliance items

Table 7:

Item

Description

PH92, PH93, PH94

Clinically required consumables, small medical aids and appliances + Read more ...

Physiotherapists can claim for consumables. Multiple requests for treatment aids, consumables and small appliances can be paid on the same date of service provided the combined total amount of the items does not exceed the maximum daily fee limit. Check QITI for fee amount.

Rules

  • only 1 consumable/small aid item should be claimed per day, The Department of Veterans' Affairs (DVA) will allow multiple consumable/small aid items may be paid providing daily fee limit is not exceeded

Restrictions

If a provider claims an amount for a single PH92/PH93*/PH94 that exceeds the maximum fee:

  • pay the maximum fee allowable and apply RSN 288

If a provider claims multiple PH92/PH93*/PH94 items, check that the total amount claimed does not exceed the daily fee limit. If the daily fee limit is:

  • exceeded, reduce the amount of the items to the daily limit, insert RSN 288, and reject any additional consumable services with RSN code S 160
  • not exceeded, pay the items with PI 45

If item restricts with any other item:

  • reject with RSN code S 159

Note: item PH93 contains a GST component. Any reference made to the 'maximum fee' is excluding GST. If the provider has added GST to the invoice, then the fee must be reduced to the non-GST amount when processing.

PH95, PH96

Static splint/cast, dynamic splint + Read more ...

Physiotherapists can claim for small medical aids, appliances and dynamic splint. Treatment aids, consumables and small appliances can be paid on the same date of service provided the combined total amount of the items does not exceed the maximum daily fee limit. Check QITI for fee amount.

Rules

Prior Financial Authorisation is required when the cost of the splint or cast exceeds the maximum daily fee.

Restrictions:

If item exceeds the daily maximum fee and returns error message 8-CHG HIGH, Check PARS for an approval.

When:

  • present, pay amount as per approval with [PF9]
  • not present, reduce the amount of the item to the daily limit, insert RSN 288 and pay claim
  • if item restricts with any other item reject PH95/PH96 with RSN code S 159

PH98

Small medical aids and appliances - postage/freight + Read more ...

Rules

PH98 is only payable when claimed for the same date of service as item PH94.

Restrictions

When error message 6-EPC NOT is returned, this indicates that PH94 has not been claimed for the same date of service. Reject item PH98 with RSN code S 550.

Physiotherapy direct supply to DVA item

Table 8

Item

Description

PH99

Physiotherapy - report or service specifically requested by DVA + Read more ...

Rules

Providers may use item number PH99 only when the Department of Veterans' Affairs (DVA) contacts them directly to request that they provide a:

  • written report; or
  • consultation or assessment to eligible veterans or war widows/ers, either separately or in conjunction with a written report

For example, this may occur when DVA requires a second opinion concerning treatment for a veteran. DVA will give financial authorisation and advise the fee at the time of the request, according to the relevant schedule items. The kilometre allowance is included in the fee, and is not to be claimed in addition to the fee.

Note: this item does not cover the supply of clinical notes, care plans or other information requested by DVA as part of monitoring activities, as these are provided free-of-charge under DVA requirements.

Restrictions

Requires Prior Financial Authorisation, Check PARS for an approval:

When Prior Financial Authorisation is:

  • present, pay service with PI 3Y, or force match an approval with PI 3X
  • not present, reject service with RSN code S 279
  • not covered by the current approval, reject service with RSN code S 283
