CH31, CH32, CH41, CH42

Initial consultation - public hospital; high care residential aged care service + Read more ...

Services Australia will only pay for health care services carried out in public hospitals and high care residential aged care services in exceptional circumstances and only when DVA has given prior financial authorisation.

Rules

  • Initial consultation limit of 3 in 365 days
  • Only 1 consult item can be claimed per day and cannot be claimed on the same day as an initial and subsequent consultation for the same patient

Requires prior financial authorisation. Check PARS for an approval.

  • Yes, pay service with PI 3Y, or force match an approval with PI 3X
  • No, reject service with RSN code S 279
  • Not covered by the current approval, reject service with RSN code S 283

Rules

DVA providers must register with DVA if they, or any eligible claimants providing chiropractic services under arrangements with the Department, wish to perform and assess x-rays at their practice location(s). Relevant documentation should be forwarded to DVA.

Restrictions

Any item, reject item with lower fee with RSN code S 159.

CH90

Allied Treatment Cycle Report + Read more ...

Provision of report to GP on outcomes of Treatment Cycle related to the referral.

Rules

  • An initial consultation item in the allied fee schedule is claimed
  • Any number from 1 to 11 subsequent consultation items from the same fee schedule is claimed
  • Only 1 report item is payable for each initial consultation
  • The provider must have claimed 1 initial and 1 subsequent consultation
  • Cannot be paid twice on the same day
  • Limit of 1 report item payable within 14 days
  • Limit of 12 report items payable within 12 months
  • Restrictions
  • Associated consult not claimed, reject with RSN code S 748
  • If the item restricts with itself or any other item, reject item. Use lower fee with RSN code S 159

CH99

Chiropractor - report or service specifically requested by DVA + Read more ...

Rules

Providers may use item number CH99 only when DVA contacts them directly to request that they provide a:

  • written report; or
  • consultation or assessment to eligible veterans or war widows/ers, either separately or in conjunction with a written report

For example, this may occur when DVA requires a second opinion about treatment for a veteran. DVA will give financial authorisation and advise the fee at the time of the request, according to the relevant schedule items. The kilometre allowance is included in the fee, and is not to be claimed in addition to the fee.

Note: this item does not cover the supply of clinical notes, care plans or other information requested by DVA as part of monitoring activities, as these are provided free-of-charge under DVA requirements.

Restrictions

Requires prior financial authorisation. Check PARS for an approval.

Is approval present?

  • Yes, pay service with PI 3Y, or force match an approval with PI 3X
  • No, reject service with RSN code S 279
  • Not covered by the current approval, reject service with RSN code S 283
