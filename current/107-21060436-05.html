Coronavirus Supplement (CVS) debt frequently asked questions

Table 3:

Step

Action

1

What happens to the CVS debt when the Income Support Payment (ISP) debt is not legally recoverable? + Read more ...

If the corresponding ISP debt is Finalised No debt (FND) because it is not legally recoverable, the CVS debt also needs to be FND.

2

The customer has retained at least $1 (this includes amounts under $1, for example $0.48) of eligible ISP payment for the entitlement period/s however ADEX has identified a CVS debt/s. What do I do? + Read more ...

Review the CVS debt/s to determine if the customer remained qualified to receive one or all of the CVS amounts identified by ADEX. If the customer remains qualified for:

  • one but not all of the CVS payments identified, adjust the undetermined debt shell accordingly
  • all the CVS payments identified, FND the debt using the reason code NDC

Note: auto CVS debts cannot be amended. To adjust the amount and/or period paid, FND the auto CVS debt shell. Create a manual debt shell to raise the correct CVS debt.

3

How should I action a debt if it contains both an ISP amount and a CVS amount in the one debt shell? + Read more ...

Use the existing debt shell to raise the ISP component. Create a manual debt shell to raise the CVS component separately.

4

What date should I use when raising a CVS debt? + Read more ...

The date identified in ADEX, for example 1 day. This is the day the CVS was paid for by the system.

5

Can a CVS debt be waived? + Read more ...

Yes. See Waiving Centrelink debts.

Note: if the ISP debt is waived Less than Fifty (LTF) investigate the CVS debt separately.

6

Are CVS arrears legally payable after a reassessment (for example, after a customer or secretary initiated review)? + Read more ...

CVS is an add-on payment. If the ISP is reassessed and arrears are legally payable, then a customer's eligibility to CVS for that period should also be reassessed and paid if applicable.

7

Is the Approved Program of Work Supplement (APWS) included as a supplement and add-on payment for CVS? + Read more ...

No. APWS payments are not considered a supplement or add-on payment for CVS purposes.

8

Why are the customers ISP and CVS entitlement periods different? + Read more ...

A customers ISP entitlement period determines payability for CVS. It ensures the customer only receives 1 CVS payment for each period.
