Assess a provider number application for a medical practitioner

Table 1: this table describes how to assess an application for a Medicare provider number and/or prescriber number for a medical practitioner (HW019).

Step

Action

1

Check the application + Read more ...

  • Open the application via current workflow procedures
  • Ensure the application form received is an Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)
  • Ensure the form has been signed

Note: if proof of residency or citizenship is previously recorded on PDS, these documents are not required.

Is the application a HW019, and signed by the medical practitioner?

  • Yes, go to Step 2
  • No, see Incomplete, inconsistent, or rejected applications. Procedure ends here

2

Subject to section 19AB + Read more ...

Was the medical practitioner registered with an Australian medical board before 1 January 1997?

  • Yes, medical practitioner is not subject to 19AB, go to Step 5
  • No, go to step 3

3

Medical school accreditation and Ahpra registration type + Read more ...

Check the Australian Health Practitioner Regulation Agency (Ahpra) website to determine the medical practitioner’s registration type.

Key the medical practitioner's name or registration number as per the application to open their registration details.

Make a note of the registration type.

Did the medical practitioner obtain their primary medical qualification at an accredited Australian or New Zealand medical school?

  • Yes, and has current Provisional Ahpra registration type. Go to Processing provider number applications for Interns in Medicare
  • Yes and has current Ahpra registration other than Provisional type, go to Step 4
  • No, medical practitioner is subject to section 19AB:
    • If you are unable to process 19AB applications, raise proficiency on the PaNDA file
    • If you are able to process 19AB applications, re-categorise the work type of the item in PaNDA to Health and Save
    • See Processing provider number applications for overseas trained or temporary resident medical practitioners

See Resources for a link to the Australian Medical Council Limited (AMC) website for a list of accredited medical schools.

4

Medical practitioner a permanent resident or citizen + Read more ...

Was the medical practitioner an Australian or New Zealand permanent resident or citizen when they enrolled in an accredited Australian or New Zealand medical school? Check date of permanent residency.

  • Yes, go to Step 5
  • No, Medical practitioner is subject to section 19AB
    • If you are unable to process 19AB applications, raise proficiency on the PaNDA file
    • If you are able to process 19AB applications, re-categorise the work type of the item in PaNDA to Health and Save
    • See Processing provider number applications for overseas trained or temporary resident medical practitioners

Note: when the submitted Application for a provider number form (HW019) is an initial application, evidence of residency status at the time of enrolment in an accredited Australian or New Zealand medical school must be provided.

Types of acceptable Residency documents

Australian Permanent Resident

  • Australian permanent resident visa
  • Visa email confirmation
  • VEVO confirmation (print out/screen shot), or
  • Visa label in passport and the bio-data (picture) page of their passport

Australian Citizenship

  • Australian citizenship certificate, or
  • Bio-data (picture) page of their Australian passport

See Supporting documents required for more information

Note: if proof of residency or citizenship is previously recorded on PDS, residency documents are not required.

5

Check PDS + Read more ...

Open PDS:

  • Select Provider
  • Select View
  • Select Detailed search
  • Key practitioner’s Surname in the appropriate field with a % sign either side (this is a wild card function)
  • Key practitioner’s First name in the applicable field with a % sign at the end of the name
  • Select Profession
  • Select Search

Assess the search results. Was a match found?

  • Yes, go to Step 6
  • No, if a provider number has been supplied:
    • Key the first six digits in the Search ID field under Basic Search
    • Select Search

If there are no results, go to Processing provider number applications for Interns in Medicare

6

Confirm provider details on PDS + Read more ...

  • Select View for the matching medical practitioner
  • Assess the provider file
  • Check Place Obtained is an accredited Australian or New Zealand medical school
  • Check Residency Details (if applicable) and Registration Comments (if applicable) state the medical practitioner was an Australian or New Zealand permanent resident or citizen when enrolled in an accredited medical school in Australia or New Zealand, for example, **NOT SUBJ TO 19AB 010410**
  • Specialty Code:
    • Open 104, go to Table 2
    • Open 181 and 185, go to Table 2
    • Open 162 with 181 and 185, see Assess additional provider number requests for interns
    • Open 110, go to Table 2
    • Open 160, see Assess additional provider number requests for interns

If the application includes an approved 3GA placement see:

  • 3GA placements post 1 January 2019
  • 3GA placements pre 1 January 2019

Note: if the 19AB restrictions recorded on PDS and the application/supporting documents are different, contact a PSO.

Step

Action

1

Identifying speciality codes and eligibility to access Medicare benefits + Read more ...

Check for the speciality codes located at the end of the bottom of the provider’s stem page.

Providers who have an open ended speciality code of 110 do not have access to Medicare benefits other than assisting at operations if they do not have one of the following open post graduate speciality codes:

  • between 001-099
  • 132, 130, 530, 532, 540. or
  • between 801-808, 898 or 899

Providers who have an open ended specialty 104 and 198 codes have access to Medicare benefits when they have one of the following open post graduate speciality codes:

  • between 001-099
  • 132, 130, 530, 532, 540. or
  • between 801-808, 898 or 899

Note: check for return of service restrictions in the notes section of PDS if there are the following open specialty codes:

  • 502 Medical Rural Bonded Scholarship (MRBS) return of service, or
  • 508 Bonded Medical Places (BMP) Scheme return of service

The Department of Health and Aged Care may have added restrictions on locations that the health practitioner is eligible to work in. If needed contact a PSO for assistance.

See Provider specialty codes and the Provider Directory System (PDS) for code descriptions.

Process a provider number request for a medical practitioner

Table 3: this table describes how to process a provider number request for standard medical practitioners in the Provider Directory System (PDS).

Step

Action

1

Amend information on provider stem + Read more ...

Check all personal, registration and speciality details on PDS against the application and amend if required by selecting Edit at the bottom left of the screen.

Select the relevant tab at top of page for editing:

  • Complete housekeeping functions where additional information is supplied on the application for a provider number, such as update title, adding additional given names
  • Check date of birth matches PDS - Unique Identifier Hyperlink (when Date of Birth Accuracy is Verified) or passport. The passport may be used to confirm correct date of birth
  • Update contact details in PDS to match the Application for a provider number form

Registration tab

If an AUS registration line is not present:

Does the medical practitioner have a Unique Identifier Hyperlink?

  • Yes:
    • Select hyperlink
    • Key registration details in PDS to match the details in the Registration details shown in the Ahpra Medical Practitioner Information window
  • No:
    • End the current state registration line the day before processing
    • Date and key a new AUS registration line using the details on the Ahpra Website

Note: if the registration status is to be amended take a screenshot of the search result and add to the file. If there are limitations noted on the Ahpra Website, please read carefully and assess if this will affect the application being processed, for example limited practice locations. Contact a PSO if assistance is required.

Status of registration

The registration status is Current and there is an open ended 160 specialty code present:

Does the medical practitioner have a Unique Identifier Hyper link?

  • Yes:
    • Select hyperlink
    • Make note of the date of General – Registration under registration details
    • Select Amend on the 160 specialty
    • Key an end date as day before the general registration start date
    • Select Update
    • Select Add specialty code
    • Key 110 in specialty code and add a start date the day after the 160 was end dated
    • Select Add New
    • Select Cancel
    • Select Save
  • No:
    • Make note of the date of processing
    • Select Amend on the 160 specialty
    • Key an end date as day before date of processing
    • Select Update
    • Select Add specialty code
    • Key 110 in specialty code and add a start date the day after the 160 was end dated
    • Select Add New
    • Select Cancel
    • Select Save

Note: if the registration status is Provisional or Limited no amendments are required to the specialty codes.

2

Assess new Location + Read more ...

At the top of the PDS:

  • Select the provider stem hyperlink to open a list of provider numbers issued to the practitioner

3

Assess if location already exists or is a new location + Read more ...

  • Check for an existing provider number for the requested practice/hospital name or street address. For example, Prince Charles Hospital Rode Rd and Prince Charles Hospital 627 Rode Rd are considered an exact match
  • If no provider number exists for the requested practice/hospital or the requested street address continue assessment, go to Step 4
  • If a match for practice/hospital name is located and the street address is different or vice versa see Issuing a second provider number or contact a PSO for assistance

4

Create provider number + Read more ...

Select Create New Provider Location

Address tab:

  • In the Locality or Postcode field, key either the suburb or postcode
  • Select the appropriate suburb and postcode from the drop down menu. If the postcode and locality do not match, contact PSO
  • In Address line, key street number and street name
  • Select address that matches the practice address listed on the application from the drop down menu
  • In Building or Department, key practice name, if supplied (26 character limitation)
  • In Property, key additional practice information, such as shopping centre or building name, if supplied (30 character limitation)
  • Key Suite/Unit Type and Suite/Unit Number and Floor/Level Type and Floor/Level Number if provided

If a non-standard address has been provided, select Manual Entry and key the address details in the appropriate fields.

Contact tab:

  • Key details in telephone, mobile, fax and email fields as per the application
  • Start Date – key as per the application (use date of lodgement if no date quoted)
  • End date- key as per the application (leave blank if no date quoted)

5

Section 19(2) and 19(5) + Read more ...

Select Assoc Details tab

See Section 19(2) and 19(5) Directions for links to Directions.

If the location is listed select:

  • ATSIHS and AMS in Location Type for locations on the 19(2) and 19(5) directions
  • AMS in Location Type for locations on COAG direction
  • Select Add

If the location is not listed no action required. Go to Step 6

6

Assess eligibility for access to Medicare benefits + Read more ...

Check application for the answer to ‘Which one of the following do you want at this location’?

See Business structure and bank account details for EFT for health professionals for correct processing of access requests.

7

Save the location + Read more ...

  • Select Save
  • Respond to any return messages (mandatory fields not keyed etc)
  • Select Save

The provider number is then created.

8

Additional Provider locations + Read more ...

Is the medical practitioner requesting any additional provider locations?

  • Yes, go to Step 2
  • No, go to Step 9

9

Confirmation Letter + Read more ...

  • Select and complete the Medical practitioners provider number letter Z2701
  • Complete appropriate fields of the letter template for all provider numbers/locations, ensuring the level of access granted is identified

10

Complete the work item + Read more ...

  • Save letter as a PDF
  • Upload the letter to PaNDA
  • Ensure the Surname, First Name, Provider Number and Case ID fields are completed and correct
  • Print the letter and send to the medical practitioner
  • Add any appropriate comments to the PaNDA file
  • Select Complete
  • Select Request Approved
  • Select Save or Save and Next, as appropriate
