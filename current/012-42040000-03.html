Assess AHOMPs application

Table 1: This table describes the steps to assess an After Hours Other Medical Practitioner program (AHOMPs) application.

Step

Action

1

Assess the application + Read more ...

  • ensure the document has been signed and a copy of the accreditation certificate is attached
  • check section 1 - if any tick boxes in section 1 are incomplete, for example:
    • no box ticked, return with letter Z2306
    • have ticked 'No', return with letter Z2305, or
  • if no Accreditation certificate has been attached, return with AHOMPS - Incomplete or Ineligible' letter

2

Check the accreditation certificate + Read more ...

There are 2 types of accreditation certificates:

  • Australian General Practice Accreditation Limited (AGPAL) and
  • GPA Accreditation Plus (GPA)

Check the certificate is current and valid. Note: practices must be fully accredited:

  • if a certificate is received with a status of Registered or Provisional, the request is incomplete
  • return the application as incomplete with AHOMPS - Incomplete or Ineligible letter

Note: practices have until 31 December 2021 to finalise their accreditation if it has expired or is due to expire before 30 June 2021. See Background for more information.

3

Location eligibility criteria + Read more ...

  • assess the eligibility of the location requested on the application
  • check the practice details on the Accreditation Certificate
  • ensure the name and address of the practice on the certificate matches details in Part 5 of the AHOMPs application
  • if the information does not match, the application is incomplete
  • return the application with AHOMPS - Incomplete or Ineligible letter. Specify on PaNDA the information supplied is inconsistent

4

Check medical practitioner eligibility + Read more ...

Search the medical practitioner’s details using information provided in the application in Parts 2-4. Select the correct practitioner to further assess the file.

Open Provider Directory System (PDS):

  • select Provider
  • select View
  • select Detailed Search
  • in the Family name field, key the first 3 letters of the last name, followed by a %
  • in the Given name field, key the first 3 letters of the first name, followed by a %
  • enter date of birth in Date of Birth field
  • select Profession
  • select Search
    • a list of health professionals with a matching name displays
  • select the medical practitioner. If no records match, contact a PSO

5

Check previous OMPs program registration + Read more ...

If the application was lodged after midnight 1 January 2019 (e.g. after 31 December 2018 11:59pm) check if the medical practitioner had specialty codes 186, 188, 199 or 615 open with an end date later than 1 January 2019 on at least one location.

Do these circumstances apply?

  • Yes, go to Step 6
  • No, add notes to PaNDA – not eligible after 1 January 2019 and re-categorise to National Office escalations

6

Check provider numbers + Read more ...

  • select the orange hyperlink of the medical practitioner’s stem in PDS to return a list of provider numbers
  • check a provider number with access to billing has been issued for the location on the AHOMPs application
  • if the eligibility requirements have been satisfied go to Step 1 in Table 2, the Resources page provides specialty codes to determine eligibility
  • if the application is not eligible return it as incomplete with 'AHOMPS - Incomplete or Ineligible letter' stating reason

Process AHOMPs application

Table 2: This table describes the steps to process an After Hours Other Medical Practitioner program (AHOMPs) application.

Step

Action

1

Process at Stem + Read more ...

In PDS, return to medical practitioner’s stem.

Does a current 199 specialty code exist?

  • Yes, go to Step 2
    Note: if a 199 specialty code exists at stem level do not extend the end date
  • No, select Edit > Specialty tab > Add specialty code button

Key in the following fields:

  • Dialog box: specialty code 199
  • Start date: key date of lodgement of application
  • End Date: key 6 years from start date or 30 June 2023, whichever is earliest
  • select Add new button
  • close dialog box
  • select Save button

Example of start date entry: Application was received on 06/02/2015, 199 start date is 06/02/2015 end date 05/02/2021.

2

Process on Provider Number + Read more ...

  • select the orange hyperlink of the medical practitioner’s stem in PDS to return a list of provider number locations
  • select Amend button on the provider number that requires the AHOMPs update
  • select Edit > Loc-Spec Codes tab>Add spec code tab

Key in the following fields:

  • Dialog box: specialty code 199
  • Start date: key date of lodgement of application or start date of Medicare billing eligibility whichever is later
  • End date: 6 years from start date on stem, end date of billing eligibility, or 30 June 2023, whichever is earliest
  • select Add new >close the dialog box>
  • select Save button
  • repeat for all eligible locations requested on the application

AHOMPs program cannot be backdated. Do not offer this option to medical practitioners

3

Preparation of letter + Read more ...

  • Z1674 letter of confirmation must be sent to the medical practitioners confirming the updates
  • convert the letter into PDF format and upload to the original file. Save changes and close the file
  • file the application as processed in line with current workflow procedures

Step

Action

1

Assess the document + Read more ...

Check the document includes:

  • Medical practitioner’s name
  • Approved from and to dates
  • Specifies the Instrument is an AHOMPs program Extension approval

  • return the request Z2305 no access to Medicare rebates

Process AHOMPs extension from Department of Health and Aged Care

Table 4: This table describes the steps to process an After Hours Other Medical Practitioner program (AHOMPs) extension received from Department of Health and Aged Care.

Step

Action

1

Process at stem + Read more ...

In PDS, return to medical practitioner’s stem.

If specialty code 199 exists on stem:

  • Select Edit > Specialty tab > Amend specialty code button

Key in the following fields:

  • Dialog box: specialty code 199
  • Start date: leave as is
  • End date: key end date as per Department of Health and Aged Care approval
  • Select Add new button
  • Close dialog box
  • Select Save button

Select Notes tab

  • New note
  • Notes subject: key A/Hours OMPs extension and the last two digits of the provider number e.g. A/Hours OMPs extension XX Only
  • Notes Description: key A/Hours OMPs extension XX only and approval dates

2

Process on Provider Number + Read more ...

  • Select the orange hyperlink of the practitioner’s stem in PDS to return a list of provider number locations
  • Select Amend button on the provider number that requires the AHOMPs update
  • Select Edit > Loc-Spec Codes tab

If there is a gap from the end of the 199 to the new approval start date:

  • Select: Add specialty code
  • Key 199
  • Start date: as per approval date
  • End date: as per approval date
  • Select Add new
  • Select Save

Note: AHOMPs approval dates are required to be in line with the medical practitioner’s access to Medicare benefits.

If there is no gap from the end of the 199 to the new approval start date:

  • Select Amend on specialty code 199
  • End date: over key end date with the date on the approval
  • Select Add new
  • Select Save

Note: AHOMPs approval dates are required to be in line with the medical practitioner’s access to Medicare benefits.

  • Repeat for all eligible locations requested on the application

3

Preparation of letter + Read more ...

  • A letter of confirmation must be sent to the medical practitioner confirming the updates
  • Select Z2535 3GA and OMPs Special Circumstances Extension Approved letter
  • Convert the letter into PDF format and upload to the original file. Save changes and close the file
  • File the application as processed in line with current workflow procedures

Assess MOMPs application

Table 5: This table describes how to assess a Medicare Plus for Other Medical Practitioners (MOMPs) application.

Step

Action

1

Assess the application + Read more ...

  • assess the mandatory fields which must be completed, including Parts 1-6, and 8
  • if all sections have been completed, go to Step 3
  • if the mandatory application fields have not been completed, access the MOMPs - Rejection letter template

2

Check practitioner eligibility + Read more ...

  • search the practitioner’s details using information provided on application Parts 2 - 4 of form

Open PDS:

  • select Provider
  • select View
  • select Detailed Search
  • in the Family name field key the first 3 letters of the last name followed by a %
  • in the Given name field key the first 3 letters of the first name followed by a %
  • enter date of birth in Date of Birth field
  • select Profession
  • select Search
    • a list of health professionals with a matching name displays
  • select the medical practitioner. If no record matches, contact a PSO

Check previous OMPs program registration

Check if the medical practitioner had specialty codes 186, 188, 199 or 615 open with an end date later than 1 January 2019 on at least one location.

Do these circumstances apply?

  • Yes, continue processing
  • No, add notes to PaNDA not eligible after 1 January 2019 and re-categorise to National Office Escalation

Note: if the application was lodged prior to midnight 1 January 2019 (e.g. On or before 31 December 2018 11:59pm), continue processing.

Eligibility Criteria

Check the specialty codes on the medical practitioner’s stem for a previous MOMPs registration:

  • scroll to specialty code sections and look for code 188:
    • if a 188 code is recorded on stem go to Step 3
    • if a 188 code is not recorded on stem go to Step 2

Medical practitioner must meet all the following criteria:

  • must have held Medical board registration with an Australian board prior to 01/11/1996
  • must have been eligible to access Medicare benefits prior to 01/11/1996
  • must not have previously been on the Vocational Register or Register of Fellows list

Check the following on PDS to determine eligibility:

  • date of earliest registration prior to 1/11/1996
  • specialty codes that allow Medicare rebates (other than 130, 132 or 532) with a start date prior to 01/11/1996
  • provider number locations with a start date prior to 01/11/1996 that are not for a public hospital
  • ensure there are no current or previous 130, 132, 530, 532 or 540 specialty codes on practitioner’s stem

If all the above criteria have been met, the medical practitioner is eligible. Go to Step 3.

If any of the above criteria are not met, return incomplete application and MOMPs - Rejection letter template.

File in line with current workflow procedures.

3

Determine location eligibility + Read more ...

Part 5 of the form contains the requested practice location. To check eligibility see the Department of Health and Aged Care website for Health Workforce Locator.

  • Search the map function
  • Select Distribution Priority Area (DPA)
  • Key full street address of the practice in Find Address Search box
  • Select Search Location - the geographic information will be returned

Check the result of DPA:

  • if Yes, location is eligible, take a snip of the DPA page, upload to PaNDA. Go to Step 4
  • if No, location is not eligible, take a snip of the DPA page, upload to PaNDA and return incomplete application task card and utilise MOMPs rejection letter
  • file in line with current workflow procedures

The Resources page provides a link to the Health Workforce Locator on Department of Health and Aged Care’s website.

4

Check provider numbers + Read more ...

  • select orange hyperlink of medical practitioner’s stem at the top of the page to return a list of all provider number locations
  • check that a provider number with access to Medicare has been issued for the location on the MOMPs application

If a provider number has not been issued, or the medical practitioner doesn’t have Medicare billing access at the location:

  • add a note to PaNDA advising no provider number/access to benefits
  • access letter templates Z2305 Provider Registration - Reject letter
  • file in line with current workflow procedures

If a billing provider number has been issued, go to Table 6.

Assess and process OMOMPs application

Table 7: This table describes how to register a medical practitioner for Outer Metropolitan Other Medical Practitioners (OMOMPs).

Step

Action

1

Assess the paperwork + Read more ...

Medical practitioners seeking to participate in the OMOMPs must complete the OMOMPs program application form available via Health’s website. Submit with supporting documents. The application is then assessed and approved by Health.

Health assess applications for eligibility in accordance with requirements set out in the guidelines in the Health Insurance Act 1973.

When approved by Health, the OMOMPs instrument is sent to Medicare for processing.

When assessing the instrument approval from Health ensure it is signed by the Delegate.

2

Search for the medical practitioner on PDS + Read more ...

In order to register the 615 specialty code the medical practitioner must have an existing Medicare provider number or supply an application form for a Medicare provider number.

Perform a search in PDS to ensure the medical practitioner has an existing stem:

Open PDS:

  • select Provider
  • select View
  • select Detailed Search
  • in the Family name field key the first 3 letters of the last name followed by a %
  • in the Given name field key the first 3 letters of the first name followed by a %
  • enter date of birth in Date of Birth field
  • select Profession
  • select Search
    • a list of health professionals with a matching name displays
  • select the medical practitioner. If no record matches, contact a PSO

Note: in most instances when searching for the medical practitioner on PDS a stem will have been already registered.

3

Check previous OMPs program registration + Read more ...

Check if the medical practitioner’ stem had specialty codes 186, 188, 199 or 615 open with an end date later than 1 January 2019 on at least one location.

Do these circumstances apply?

  • Yes, go to Step 4
  • No, add notes to PaNDA not eligible after 1 January 2019 and re-categorise to National Office Escalation team

4

Check for existing provider number + Read more ...

  • select the orange hyperlink at the top of the page to go to the provider number location list screen
  • check the returned list to see if the provider number has been previously allocated
  • if a provider number has been issued go to Step 6
  • if a provider number has not been issued search for an application in PaNDA
  • if no application is received, go to Step 5
  • if an application is received, assess and process the request in line with current policy and procedures
  • go to Step 6

5

Return incomplete application + Read more ...

  • if the medical practitioner is not found on PDS and has not submitted an application for provider number, the request cannot be processed
  • return the request with Z2306 incomplete application letter

6

Check stem for 615 specialty code + Read more ...

  • check if the 615 specialty code is at the stem level prior to adding to a location
  • on the bottom of the medical practitioner’s stem screen check for 615 code under the table titled specialty code
  • if the 615 specialty code is at stem level go to Step 8
  • if the 615 specialty code is not at stem level go to Step 7

7

Add specialty code 615 at stem level + Read more ...

Return to PDS:

  • select Edit button
  • select the Specialty tab located on top of the screen
  • select the Add specialty button, located on left of screen
  • key in the 615 specialty code
  • the start date recorded on PDS is the date stipulated on the instrument – the end date recorded on PDS is 30 June 2023
  • select Save
  • the 615 specialty code will be updated and displayed on the stem
  • go to Step 8

8

Add specialty code 615 to a location + Read more ...

Return to PDS:

  • select the orange hyperlink stem at the top right of the page to display all location(s) for the medical practitioner
  • search for the approved location as per the OMOMPs instrument
  • when the location on PDS is found, ensure it is an exact match of the location quoted on the Instrument
  • also check the specialty code 615 has not been added previously. If the specialty code 615 has been added previously for the period specified on the instrument, no further action is required. Add a note to PaNDA stating no further updates

To add 615 specialty code on the existing location on PDS:

  • select Amend
  • select the Specialty Codes tab
  • select the Add Specialty button, located on left of screen
  • key in the 615 specialty code
  • key Start date as indicated on the instrument
  • key End date as indicated on the instrument, or 30 June 2023, whichever is earliest
  • select the Save button
  • the 615 specialty code is updated and displayed on PDS

9

Preparation of letter + Read more ...

A 3GA and OMPs approval letter Z2528 must be sent to the medical practitioner confirming the updates:

  • convert the letter into PDF format and upload to the original file
  • save changes and close the file
  • file the application as processed in line with current workflow procedures

Assess ROMPs application

Table 8: This table describes how to assess a Rural Other Medical Practitioners (ROMPs) application.

Step

Action

1

Check the application + Read more ...

Ensure application is complete and check the following information has been completed on the form:

  • medical practitioner’s personal details
  • current status (optional)
  • practice locations and provider numbers (provider numbers optional)
  • mailing address (optional)
  • contact details (optional)
  • expression of interest (Note: it is a requirement of the ROMPs program that applicants express an interest in undertaking FRACGP or FACRRM)
  • declaration
  • if section 6 is incomplete (no box ticked) return application as incomplete stating the ‘Expression of Interest not ticked’ ROMPs - Unable to Process Application letter

2

Determine location eligibility + Read more ...

ROMPS location eligibility criteria:

  • RRMA 4-7 - Eligible

or

  • RRMA 3 with DPA Status - Eligible

Check RRMA Classification at Health Workforce Locator on Health’s website.

The Resources page has a link to the website. Note: even if RRMA is noted on application, check and confirm.

RRMA Search

  • select Search the map
  • tick RRMA 1991
  • enter full street address of practice and select address from drop down list
  • print a copy to PDF and upload to PaNDA record

The Geographic information will be returned. Only where a practice address returns a result of RRMA 3 - is it then required to confirm the location is in a Distribution Priority Area.

DPA Search

Enter full street address of practice in mini menu:

  • select DPA for GPs
  • select the tick box to confirm
  • enter fill street address of practice and select the address from the drop down list
  • the Geographic information will be returned
  • print a copy to PDF and upload to PaNDA record
  • if ineligible return application with appropriate ROMPs ineligible letter
