Using the External staff access request search screen + Read more ...

Note:

  • The External staff access request search screen is an accordion menu
    • An accordion menu is a menu type display where Programme Officers can select the Arrow button in the Search external staff access request details section to either display or hide the section
  • This screen is available for Programme Officers from either Services Australia or the Department of Health and Aged Care
    • See the Resources page for the business roles needed to use this screen
  • Programme Officers can key the date, or select the date from the calendar (select the calendar icon to display the calendar) for the following fields
    • Requested date from
    • Requested date to
    • Expiry date from
    • Expiry date to
  • Programme Officers may need to refine their search criteria if they see the following information messages
    • The No records available information message
    • The Your search returned X requests, but no more than 200 can be displayed information message
  • All fields are optional unless marked with an asterisk

Use the External staff access request search screen to view all external users’ access requests.

The External staff access request search screen will display the Search external staff access request details section.

To generate search results, complete the following fields:

  • Request status - defaults to Awaiting approval. Select the drop down menu for either Select All, Approved, Deleted or Rejected
  • Role - defaults to Please select. Select the drop down menu to display the roles relevant to the external user
  • Email
  • User
  • Organisation* - defaults to Department of Health and Aged Care. Select the drop down menu to display the department relevant to the external user if applicable
  • Requested date from - this field is mandatory if the Requested date to field has been completed
  • Requested date to - this field is mandatory if the Requested date from field has been completed
  • Expiry date from
  • Expiry date to

Select Search.

Is the search successful?

  • Yes, go to Step 7
  • No,
    • Programme Officers must refine their search criteria if they cannot locate the relevant letter from the search results
    • select Clear to clear the search criteria. Repeat this step (Step 6)

Note:

  • Programme Officers can also select Go to external staff user search and the External staff user search screen will display. Go to Step 10

    • First, Previous, Next or Last options, or
    • the relevant page number

Using the Register external staff access request screen + Read more ...

Note:

  • Programme Officers from the Department of Health and Aged Care with the appropriate business roles can register access requests for someone else from their department
    • See the Resources page for the business roles needed to use this screen
  • The Role* field will display roles relevant to the external user’s department
    • For example, the Social Security Compensation Officer role will only display to Programme Officers from the Department of Health and Aged Care
  • The Notes* field has a 500 character limit
  • All fields are mandatory

Use the Register external staff access request screen to register an external user’s access request.

Complete the following fields:

  • Email*
  • Given name*
  • Family name*
  • Department* - select the drop down menu for Department of Health and Aged Care. Note: Programme Officers must not select either the Aged Care Quality and Safety Commission or Department of Veterans’ Affairs option. This is because ACSP access is not yet available for these external users
  • Role* - select the drop down menu to display the relevant roles
  • Expiry date* - key the date, or select the date from the calendar (select the calendar icon to display the calendar) for the following fields
  • Notes* - key clear and accurate reasons for the request

To complete the Register external staff access request process

  • Select Submit
  • The SUCCESS: External staff user request is being processed information message within this tab
  • The Request details tab will display the access request information
  • Go to Step 15

Alternatively, to cancel the Register external staff access request process:

  • Select Cancel
  • The Cancel? pop up notification will display
  • Select Yes
  • The External staff access request search screen will display

Using the Update external staff access request screen + Read more ...

Note:

  • Programme Officers from the Department of Health and Aged Care with the appropriate business roles can update access requests for someone else from their department
    • See the Resources page for the business roles needed to use this screen
  • The Role* field will display roles relevant to the external user’s department
    • For example, the Social Security Compensation Officer role will only display to external users from the Department of Health and Aged Care

    • This must include the line manager’s approval
  • All fields are mandatory

Use the Update external staff access request screen to make any updates to an individual access request.

To update an access request:

  • Select Update on the Viewing the Request details tab
  • The Update external staff access request screen will display

Make the updates to relevant fields:

  • Email*
  • Given name*
  • Family name*
  • Department* - select the drop down menu for Department of Health. Note: Programme Officers must not select either the Aged Care Quality and Safety Commission or Department of Veterans’ Affairs option. This is because ACSP access is not yet available for these external users
  • Role* - select the drop down menu to display the role(s) relevant to the external user
  • Expiry date* - key the date, or select the date from the calendar (select the calendar icon to display the calendar) for the following fields
  • Notes* - key clear and accurate reasons for the request

To complete the Update external staff access request process:

  • Select Submit
  • The SUCCESS: Update of external staff user request is being processed information will display
  • The Request details tab will display the updated information
  • Select Go to external staff request detail
  • The individual View external staff access screen will display
  • Select either
    • The Attachments tab to upload any relevant documents associated with the updates and go to Step 17
    • The Notes tab to include any additional notes associated with the updates and go to Step 18
  • Procedure ends here

  • Select Cancel
  • The Cancel? pop up notification will display
  • Select Yes
  • The External staff access request search screen will display

Using the Notes tab + Read more ...

Note:

  • Once the Add note process has been completed, there is no option to remove/delete the note
  • When Add note is selected, the updated information will display as an annotation
    • That is, the annotation (the updated content) will display in additional to the original Notes information
    • The annotation will not override (replace) the original Notes information
  • The Notes* section has a 500 character limit
  • All fields are optional unless marked with an asterisk

Use the Notes tab to Add note.

To Add note:

  • Select Add note in the Notes tab
  • The Create note dialog box will display
  • Complete the Notes* section
    • Key clear and accurate reasons for the request
  • Select Submit
  • The note has been saved
  • This completes the ACSP process for the external users

Programme Officers from the Department of Health and Aged Care must:

  • send a completed Services Australia access request form - Aged Care Systems form to the Department of Health and Aged Care Policy Operations Team via their positional mailbox
  • send their line manager’s approval as an email attachment to the Department of Health and Aged Care’s Aged Care Payments Team via their positional mailbox

The Department of Health and Aged Care’s Aged Care Payments Team will email all forms and email attachments to the Aged Care Programmes Branch’s Operational (Change and Correspondence) team. All emails are sent via positional mailboxes. See the Resources page for a copy of this request form.

All external access requests are approved by the Aged Care Programmes Branch’s Operations Change and Correspondence Team. Approval is sent via email notification.
