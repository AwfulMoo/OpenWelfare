Request type

Additional information

Specialty code C90 225.

(Medical grade footwear)

Might take longer to action. These requests have to be looked at by the Rehabilitation Appliance Program area for determination.

Online claims and claims relating to specialty codes C83 225 and C83 226.

(Community nursing)

Referred to the Provider Services Communication email with subject line 'VAP (site) Provider Update Request' for forwarding to the VAP Helpdesk.

Specialty code C63 111.

(Optical Dispenser)

Specialty code C63 111 (Optical Dispenser)

Will only be considered where the request is to back date the qualifications to within a 4 week period of the current start date. Requests to transfer to code C63 111 from another location will not be considered. Providers are required to apply for a contract with Department of Veterans' Affairs (DVA) for each individual location where C63 111 is to be applied.

Backdating the provider specialty code past the effective date.

(EFF From) stored on VPUI

Will not be considered. If there are extenuating circumstances the request is sent to Provider Services Communications email with 'VAP (site) Backdating Speciality Code' in the subject line, which is then forwarded by the VAP team leader to the VAP Helpdesk. This request will then be forwarded to Medicare Liaison and Contract Management (MALCM) if deemed necessary. If MALCM are unable to complete the request a notification will be sent advising of the reason.
