Recording unreasonable to live at home (UTLAH) assessments for customers under 18 years of age - social work 003-12130080

  • Background
  • Process
Expand all


If a child is in immediate danger or risk of harm you must act, and a delegate must call 000.

For Services Australia social workers only.

This page contains procedures for recording Unreasonable to Live at Home (UTLAH) assessments in the Social Work Recording Application (SWRA) in Process Direct and the Social Work Information System (SWIS) in Customer First.

Process Direct

On this page:

Unreasonable to Live at Home (UTLAH) assessment recording in the SWRA in Process Direct

Unreasonable to Live at Home (UTLAH) assessments - not pursued recording in the SRWA in Process Direct

Unreasonable to Live at Home (UTLAH) assessment recording in the SWRA in Process Direct

Table 1

Step

Action

1

Assessment is not going to be resolved during the first contact + Read more ...

If the assessment is not going to be resolved during the first contact, in Customer First create a Fast Note. Use Auto text template UTLAH Assessment in progress:

  • key the template name in the Search field then select Search, or
  • select Auto text. Select:
    • Category 1: Social Worker
    • Category 2: Update
    • Template: UTLAH Assessment in progress

Exit from Customer First and open Process Direct.

2

Recording the assessment + Read more ...

In Process Direct

  • Select the Social Work Recording Application on the Task Selector screen
  • Reports and case notes summary is default selected. Select Start
  • On the Reports and Case Notes Summary screen select New
  • Assessment type: From the drop down list select Report - UTLAH and either:
    • ABSTUDY
    • DSP
    • Under 18
  • Decision: From the drop down list select either Granted or Not Granted
  • Is this assessment related to an existing referral?
    • For all new assessments select No
    • If the UTLAH has resulted from an existing referral for a Crisis Payment (CrP) assessment completed in the SWRA select Yes
      Note:       if selecting Yes a referral ID must be entered. The referral ID can be located in the existing referral in SWIS in Customer First.
  • Select Next

3

Complete Consent to record and share + Read more ...

Note: The date of consent entered into SWRA must be later than the last consent date recorded in SWIS to prevent a system error.

On the Consent to record and share screen:

  • complete all of the mandatory fields
  • add third party details where required. Do this by selecting Add then Save. Social workers need to select Add for each third party
  • select Next

4

Record the Referral record information + Read more ...

Some fields on the Referral record screen will pre-populate. Review all fields and make appropriate changes.

  • Area and Location fields will pre-populate based on the social worker's user profile. Update if required
  • Referral date: will prepopulate with today's date. Update if required
  • Customer Status: will not pre-populate. Select appropriate option:
    • Job Seeker
    • Student
    • Not on Payment
    • Other
  • Cultural background: will pre-populate based on the customer’s information. Update if required
  • Interpreter used: select Yes or No
  • Referral allocated through workload manager: select Yes or No
  • Referral source: select Select referral source. Then from the drop-down options select:
    • Customer if an online claim was completed, or
    • DHS. This will produce a second list. Select appropriate option, that is, Booked Appointment, Service Centre, Smart Centre or Processing Team
  • Customer currently on income management?: select Yes or No. If Yes, select the relevant measure from the Initiative drop down list
  • Referral issues: select Select referral issues
    Note: The Referral issues options that present in the pop-up box will pre-populate based on the selection made earlier on the Assessment type screen. These cannot be changed unless the Assessment type is also changed. To change the Assessment type click Cancel then select Previous twice and select another option in the Assessment type drop down list. Referral issues include:
    • Personal and/or Family Issues
    • Accommodation
    • DSP Delegated UTLAH Assessments - DSP/UTLAH
    • ABSTUDY Delegated UTLAH Assessments - ABSTUDY UTLAH
    • Under 18 - select applicable options:
      Delegated UTLAH Assessments- SpB or
      Delegated UTLAH Assessments- YA Under 18
  • Select Next to go to the Contact Record screen

5

The Contact Record screen + Read more ...

  • Area and Location fields will pre-populate based on the social worker’s user profile. Update if required
  • Contact date: field will pre-populate with today's date. Update if required
  • Contact Type field will pre-populate to File/Paper Assessment and cannot be changed
  • Entitlement Decision, Social Work Intervention, and Outcome of Intervention fields will pre-populate based on the information recorded on the Assessment Type screen and cannot be changed
  • Additional outcome of Intervention field should be completed depending on the social worker’s delegation:
    • Non delegated social workers select Assessment in progress. When the assessment transfers over to SWIS in Customer First it will save as a draft report
    • Delegated social workers select Assessment completed. When the assessment transfers over to SWIS in Customer First it will save as a Final report

Note: Social Work Support Managers recording the delegation of decisions for non-delegated social workers must access the report via SWIS in Customer First after all other contacts have been recorded.

6

Recording the UTLAH Report + Read more ...

All Under 18 UTLAH assessments must have a report completed and transferred to SWIS in Customer First. This is with the exception of rejections due to insufficient information, because the customer did not attend a social work interview.

  • Complete all fields on the UTLAH Report screen with the appropriate information, including the report in the Social Worker Assessment Summary field
  • The Doc: will prepopulate with specified information relating to the social worker completing the report. This information cannot be changed and should not replace a grant or rejection DOC

7

Transfer the assessment information to SWIS in Customer First + Read more ...

All U18 UTLAH assessments must have reports completed and transferred to SWIS.

  • When all mandatory information has been recorded in the SWRA in Process Direct select Submit for the assessment to be transferred over to SWIS
  • When the UTLAH referral has successfully transferred the additional contacts must be entered in SWIS in Customer First

For non-delegated social workers:

The report will transfer over in draft mode and a Social Work Support Manager (SWSM) will still be required to set the report to final in SWIS.

For delegated social workers:

The report will transfer over as final in SWIS.

8

Record additional contacts in SWIS in Customer First + Read more ...

Social workers must record additional contacts in the original referral created by the SWRA in Process Direct using SWIS in Customer First. Mandatory contacts are to be recorded for the interview with the customer, parents and any third parties.

Record contact with the customer

  • Go to the customer's record in SWIS in Customer First
  • Select the referral created in SWRA in Process Direct
  • Select Create a new contact, complete Consent process mandatory fields and select continue.
  • Contact Type: field select either
    • Interview in CSC - Face to Face
    • Phone Interview
  • Entitlement Decision: field, leave blank as was completed in SWRA in Process Direct
  • Social Work Intervention: Select Advice – DHS services and two or more other options depending on the intervention provided. For Example Counselling/Support, Participation Related Intervention or Referral Out to Reconnect or Accommodation etc.
  • Outcome of Intervention: field select Information Collected and Assessment in Progress

9

Recording parent and third party correspondence + Read more ...

Social workers should record a contact in SWIS if they send a Q465 letter to a parent. They should also record a contact if they send a UTLAH questionnaire to a parent or third party.

  • Go to the customer's record in SWIS in Customer First
  • Select the referral created earlier in SWRA in Process Direct
  • Select Create a new contact, complete Consent process mandatory fields and select continue.
  • Contact Type: field
  • Select Correspondence
  • Entitlement Decision: field. Leave blank
  • Social Work Intervention: field
    • Select Discussion with Third Parties or Contact with Parents/Guardians (this is compulsory)
    • Select Advice – DHS Services
  • Outcome of Intervention: field. Select Provision of Information and Assessment in Progress

10

Recording parent and third party contacts + Read more ...

  • Go to the customer's record in SWIS in Customer First
  • Select the referral created earlier in SWRA in Process Direct
  • Select Create a new contact, complete Consent process mandatory fields and select continue.
  • Contact Type: field. Select: UTLAH - Third party or parent phone interview
  • Entitlement Decision: field. Leave blank
  • Social Work Intervention: field.
    • Select Discussion With Third Parties or Contact with Parents/Guardians (this is compulsory)
    • Two other options may be chosen depending on what has been undertaken during the contact with the parent/s and who the third party is, for example, Advice - DHS Services, Counselling Support or Referral Out to Reconnect or Family Relationship Centre.
    • If there was liaison with the area SW Manager in regards to lack of third party verification Referral Out and SW Manager- No third Party Verification must be selected
  • Outcome of Intervention: field. Select Information Collected and Assessment in Progress

The name, address and phone number of parents and third parties who are contacted as part of the UTLAH assessment should be recorded in the social work report. This helps with review and reassessment processes..

11

For Social Work Support Managers recording the delegation of decisions for non-delegated social workers + Read more ...

Social Work Support Managers must complete delegations in SWIS in Customer First.

  • Go to the customer's record in SWIS in Customer First
  • Select the referral created earlier in SWRA in Process Direct
  • Select Create a new contact
  • Contact Type: field select File/Paper Assessment
  • Entitlement Decision: leave blank
  • Social Work Intervention: field select Delegating SW Assessments
  • Outcome of Intervention: field select Assessment Completed

6

Recording the decision + Read more ...

Once all information has been received and a decision has been determined, a new contact will need to be created in the original referral.

For delegated social workers

  • Select the original referral and then select Create a new contact
  • Contact Type: field. select File/Paper Assessment
  • Entitlement Decision: field. Select Delegation Assessments then select the applicable decision for the customer's claim in the drop down box:
    • UTLAH 18+ Granted
    • UTLAH 18+ Not Granted
    • Claim Not Pursued 18+
    • UTLAH ABSTUDY Granted
    • UTLAH ABSTUDY Not Granted
    • UTLAH DSP Granted
    • UTLAH DSP Not Granted
    • UTLAH Under 18 Granted
    • UTLAH Under 18 Not Granted, or
    • Claim Not Pursued Under 18
      Claim Not Pursued 18+ and Claim Not Pursued Under 18 are used for customers who have discussed with a social worker the eligibility criteria for claiming UTLAH, but have decided not to continue with the claim
  • Social Work Intervention: field select Eligibility Assessment Undertaken
    • 2 other options need to be selected depending on the intervention provided by the social worker, for example: Counselling/Support, Related Intervention
  • Eligibility Assessment Undertaken should be used only once for each referral, but there are exceptions to this, where a customer is also claiming a Crisis Payment. With delegated assessments, it should be when the contact is reported (File/Paper Assessment)
  • Outcome of Intervention: field select Assessment Completed

All UTLAH assessments, grants and rejects must have a SWIS Report written. This excludes rejections due to insufficient information, because the customer did not attend a social work interview.

For non-delegated social workers

The process is the same as for delegated social workers, with the exception of the last step:

  • Outcome of Intervention: field select Assessment in Progress

For Social Work Support Managers recording the delegation of decisions for non-delegated social workers

Select the original referral and then select Create a new contact.

  • Contact Type: field select File/Paper Assessment
  • Entitlement Decision: field leave blank
  • Social Work Intervention: field select Delegating SW Assessments
  • Outcome of Intervention: field select Assessment Completed
