Reverse a deleted line adjustment

Reversing a deleted line adjustment

Table 1: This table describes how Service Officers reverse a line deletion for bulk bill LDA claims in Medicare.

Step

Action

1

Key control line + Read more ...

  • key control line with ‘DHSI, provider no, claim ID, DOL=DDMMYY’.
  • Press [Enter].

The display defaults to card number order.

2

Code reversal + Read more ...

Key X over the R in the A field of the restated line.

Press [Enter].

3

Check result + Read more ...

The restated line is removed and the O line is restored. This includes:

  • removing the O from the A field
  • displays the message REV DEL OK

Reversing an inserted line adjustment

Table 2: This table describes how Service Officers reverse an inserted line for bulk bill LDA claims in Medicare.

Step

Action

1

Key control line + Read more ...

  • key control line with ‘DHSI, provider no, claim ID, DOL=DDMMYY’.
  • Press [Enter].

The display defaults to card number order.

2

Code reversal + Read more ...

Key X over the N in the A field of the inserted line.

Press [Enter].

3

Check result + Read more ...

The system removes the new line from the screen and returns the message REV INS OK.

Grouped lines adjustment

Reversing an adjustment on grouped lines

Table 4: This table describes how Service Officers reverse an adjustment on grouped items for bulk bill LDA claims in Medicare.

Step

Action

1

Key control line + Read more ...

  • key control line with ‘DHSI, provider no, claim ID, DOL=DDMMYY’.
  • Press [Enter].

The display defaults to card number order.

2

Code reversal + Read more ...

Key X over all the R in the A field for all lines in the group.

Press [Enter].

3

Check result + Read more ...

All restated lines are moved and all O lines are restored. This includes:

  • removing the O from the A field
  • displays the message REV ADJ OK

Close a claim

Closing a claim after adjustment

Table 5: This table describes how Service Officers close a claim when all adjustment details have been entered for bulk bill LDA claims in Medicare.

Step

Action

1

Close claim + Read more ...

Press [PF3]-CLOSE.

The message CLM CLOSED HIT ENTER is returned. The status of the adjustment claim is now ADJ COMP.

Delete a claim

The adjusted claim may be deleted any time prior to closing the claim, up to and including the time when the status of the claim is ADJ COMP.

Deleting a claim before assessment

Table 6: This table describes how Service Officers delete a claim before assessment for bulk bill LDA claims in Medicare.

Step

Action

1

Code deletion + Read more ...

Key:

  • DHSI,PROVIDER No, CLAIM ID,DOL=DDMMYY,DELETE, or
  • DHSC,PROVIDER No, CLAIM ID,DOL=DDMMYY,DELETE

Press [Enter].

2

Check result + Read more ...

The message ADJ DEL OK displays confirming the claim has been deleted from the system, including any actions processed in the claim and the registration of the claim that can be seen on DCEI.
