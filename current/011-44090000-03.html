MES incomplete or missing information

Table 1: actions Service Officers are required to follow when missing documents or incomplete information for MES applications has been received.

Item

Document type

1

Certified documents + Read more ...

Medicare Entitlement Statement (MES) supporting documents do not need to be certified.

Do not return the application to the applicant.

Continue to assess and process the application.

Accepting non-certified documents is an internal process only. All external reference material (MES form and website information) will refer to certified copies.

When answering enquires, advise applicants to provide certified copies.

2

Missing supporting documents + Read more ...

If it has been established that the application is incomplete or documents are missing, check if missing information can be verified via other sources before returning the application. Sources include:

  • Visa Entitlement Verification Online (VEVO)
  • Immigration Movement Record (IMR)
  • Consumer Directory Maintenance System (CDMS)
  • Processing and National Demand Allocation (PaNDA)
  • MES positional email box

After all sources have been checked and an assessment cannot be made, return the application to the applicant with an appropriate letter, see Access VEVO (Visa Entitlement Verification Online) program.

3

MES application forms + Read more ...

Before processing an application check if the applicant has completed the acceptable version of the MES application forms below. Version numbers can be identified on the bottom left corner of the form:

  • MS015.2205
  • MS015.2008

If an application is submitted on an old MES application form (that is, not listed above), return the application to the applicant and:

  • complete standard letter template z2456 – MES application needs to be resubmitted on the current form
  • add copy of letter to the application in PaNDA

Do not assess the application or enter the applicant’s details into the MES database before returning the application.

4

Incomplete application forms + Read more ...

The following questions on the form must be completed by the applicant:

  • applicant's name
  • questions relating to applications for permanent residency
  • questions relating to prior place of residence. Exception- staff can verify questions 13 and 14 of the MES application form by phone
  • declaration ticked (electronic or hand written are permitted) including their:
    • full name (middle name/s not mandatory), and
    • date (not mandatory)

Note: for previous form MS015.2008, the applicant must sign (electronic signatures are permitted).

If any of the above questions are incomplete, check original application in email and upload to PaNDA.

If the questions listed above are not completed:

  • return the application to the applicant with standard letter template z0132; and
  • add a copy of the letter and attach it to the application in PaNDA

If questions other than those listed above have not been completed:

  • make 2 attempts to contact the applicant by phone to verify details

If contact has been made with applicant:

  • verify required information with applicant
  • add comments in the MES database about the type of information verified with applicant
  • complete a separate document stating type of information verified, the date then add to file in PaNDA
  • continue assessing application

If contact could not be made with the applicant:

  • add comments in the MES database ‘application form not completed unable to make contact with applicant’ see Table 5
  • complete a separate document stating application form not completed, unable to contact applicant. Then add to file in PaNDA
  • return the application to the applicant with standard letter template z0132
  • add copy of letter to application in PaNDA

MES database

Table 4: how Service Officers search the MES database to identify existing records and determine the impacts to the MES application outcome.

Step

Action

1

Search the MES database + Read more ...

  • Select File
  • Select Open
  • Key applicant’s details in the following fields:
    • Surname
    • Date of birth
  • Select Search

When searching punctuated names, replace punctuation with a space, for example, O’Connor should be O Connor.

To expand search parameters, the first 3 characters of the last name can be entered followed by percentage symbol (%), for example, for Smith, key Smi%.

Make sure all variants of name searches are searched.

Is a match returned?

  • Yes, go to Step 2
  • No, the application is ready to be processed. See Table 5

2

Matched records + Read more ...

Is the matched record for the same claim period as the current application?

  • Yes, further investigation is required. Go to Step 3
  • No, if a previous MES application has a status of:
    • Reject or Pend, open the application in database and further investigate by reading the processing notes. Determine if the status of previous applications will impact the current application
    • Approved, open to view previous residence details, if previous residence indicated is RHCA country may impact current application, further checks may be required
  • If all checks have been completed and there is no impact, the application is ready to be processed. See Table 5. Procedure ends here
  • If trained in general layer MES processing or unable to determine the impact, escalate the application to a more experienced Service Officer by raising the proficiency or re-categorise correctly in PaNDA. See Work Optimiser for staff
  • If there are impacts and the Service Officer has the required skill tag and all checks have been complete the application is ready to be processed, see Table 5

3

Determine the application outcome + Read more ...

After all checks are complete and the application does not have to be escalated or reassigned, determine the application outcome:

  • Certified (eligible for MES – full or partial period), go to Step 4
  • Approved (eligible for MES - full or partial period. Application processed on the same day of checking but not yet certified.), go to Step 4
  • Pend (further information or missing information), go to Step 7
  • Reject (not eligible for MES), go to Step 8

4

Duplicated application + Read more ...

Determine if the application is a duplicate.

A duplicate is where the below information is the same on the Certified or Approved application and the application that is assessing:

  • financial year being claimed
  • address information
  • Tax Agent details are the same (if applicable)

Is the application a duplicate:

  • Yes, go to Step 5
  • No, go to Step 6

5

Verify the processing date + Read more ...

If the application is Certified, Service Officers must verify the processing date.

If the application was processed:

  • under 2 weeks from the processing date:
    • no further action is required, the MES is still in the mailing process
    • notate the application number in the Case Id field in PaNDA
    • notate in the ‘add process notes’ section ‘Dup application –NFA’
    • Note: comments not required in MES database
  • over 2 weeks from the processing date:
    • see Table 8
    • key comment ‘Dup application MES reprinted and sent’ in Comments field, in MES database

If the application is Approved:

  • no further action is required, Duplicate application processed on the same day
  • notate the application number in the Case Id field in PaNDA
  • notate in the ‘add process notes’ section ‘Dup application – NFA’

Comments not required in MES database.

6

Determine the next course of action + Read more ...

Service Officers must check the following information to determine the next course of action:

If the postal address on the database is:

  • different to the application form, see Table 8 (a new statement will need to be issued)
  • not different to the application form, go to next question

If the approved period on the database is:

  • different to the request period on the application form, further investigation is required to determine MES eligibility is correct, see Table 12
  • not different to the request period on the application form, go to next question

If the Tax agent details on the database are:

  • different to the application form, see Table 8 (a new statement will need to be issued)
  • not different to the application form, if there are other discrepancies escalate the application to a more experienced Service Officer by raising the proficiency in PaNDA. See Work Optimiser for staff

7

Application status is Pend + Read more ...

Check Comments tab to see why the application was pended.

Has the application been returned with all the requested documentation?

  • Yes, proceed with processing, see Table 5
  • No,
    • Search PaNDA and email to make sure documents haven’t been uploaded separately
    • After searching PaNDA and requested documents have not been submitted, return the application to applicant with letter z0132 re-requesting the additional information
    • If the missing information relates to application for Permanent Residency document not being submitted, and the application has been already returned twice, do not return to the applicant, request this information by sending an email to Tier 2 – Investigation of incorrect or missing historical information. Service Officers who manage this email box will request this information directly from the Department of Home Affairs. Applications can only be returned for Permanent residency documents twice
    • Procedure ends here.

8

Application status is Reject + Read more ...

Further investigation is required, consider the following:

  • If the information:
    • differs between the 2 applications a reassessment may be required
    • does not differ, resend the original rejection letter z2110
  • Escalate application where necessary to a more experienced Service Officer by raising the proficiency in PaNDA. See Work Optimiser for staff

Note: if a written request for reassessment is included, escalate application to SME to action.

MES database application process

Table 5: how Service Officers process an application into the MES database.

Step

Action

1

Open MES database + Read more ...

  • Select File
  • Select New

2

Name tab + Read more ...

Key the applicant’s name as per their passport.

Processing information

  • No Title – if the applicant has not provided a title on the application form, this field can be left blank on the database. If the system returns an error message (edit error box appears), it can be overridden, select Accept
  • Surname/Given Names – always enter names as per passport, even if the names slightly differ from the information on the application form
  • Names including punctuation and hyphens – replace any punctuation with a space, for example, apostrophes, hyphen – replace O’Connor with O Connor
  • Single name (only name) – key the full name in the Surname field. The first alpha from the only name must be entered in the Given Name field
    Even though there is a provision for the Surname and Given Name on the passport id page, if only one field has been used for the name reference, this is referred to as a single name
    • if unable to determine given name from surname, key the name as a single name (only name)
      Note: no provision for the surname and given name will be indicated on the passport id page
    • names that exceed the allowable number of characters - key as much of the surname as possible, then key the applicant's full name in the Comments field
    • passport and application name differ - where an applicant has indicated one name on their application form, but the photo ID page of their passport confirms a different name, search all passport pages for notation of a change of name. Where there is no evidence of a name change in the passport, return the application and request evidence of name change, such as marriage certificate, change of name certificate etc
  • Change of name – if a change of name document has been provided to support notation on passport, key name on database as indicated on the MES application form

3

Name tab - Date of Birth and Phone Number fields + Read more ...

  • Date of Birth – key as indicated on passport
  • Phone Number – this field is not mandatory, however where a phone number is provided, key in the number as indicated on application form. Do not key overseas phone numbers

4

Name tab – address fields + Read more ...

  • Home and postal address are different – key address information as indicated on application form
  • Home address and postal address are the same – key address in Home Address field and select same as Home Address tick box in the Postal Address field
  • Home address not provided – the postal address must be keyed in the Home Address field and select same as Home Address tick box in the Postal Address field
  • If the postal address:
    • is the tax agent’s address (as identified on the form), key the postal address as indicated in the Postal Address field in Application tab and enter the tax agent’s details in the Agency tab
    • has not been provided and tax agent details are provided – only key address in the Home Address field, select Same as home address tick box option and enter the tax agent’s details in the Comments tab
  • Abbreviations – if an address exceeds allowable number of characters, abbreviate the address accordingly
  • Overseas addresses – the overseas address must be keyed as indicated on the form, in either the Home address or Postal address field. Key as much of the address as possible, the system will cut the details if it exceeds a certain number o`f characters. An approved MES with an overseas address is extracted and then sent out manually by the programme area
  • Non-standard postcodes (overseas postcodes) – the Postcode field will only allow Australian standard postcodes. Key overseas postcode on the last line of the address in the Suburb or Town field

5

Application tab + Read more ...

The Application tab is where the person’s eligibility information is keyed.

  • Country prior to Australia – select country as indicated on application from the drop down box. If the country is not listed in the drop down box, key ‘other’ in the residency field and provide the country name in the Comments field.
  • Years and months – key details as indicated on application form
    Note: Previous residence must be 6 months or more. If the shorter period is quoted, return the application.
  • Permanent residency applied for?

If the applicant has indicated on their MES application form:

  • Yes, they have applied for permanent residency:
    • key ‘Y’ in the Permanent residency applied for? field
    • key Date of Lodgement of Permanent residency in the Date field
    • key 'Y' in the Still Current? Field if the Permanent residency application is ongoing. Key 'N' in the Still Current? Field if the Permanent residency application is no longer i.e granted
    • key ‘Y’ in the Permission to Work field. Note: always key Y even if the person has work rights or not
    • key ‘N’ in the Spouse, parent, child Australian citizen field. Note: always key N
  • No:
    • key ‘N’ in the Permanent residency applied for? field
    • leave Date field blank
    • key ‘Y’.in the Permission to Work field. Note: Y must always be keyed regardless if the person has work rights or not
    • key ‘N’ in the Spouse, parent, child Australian citizen field. Note: N must always be keyed
  • Period claimed – key appropriate MES eligibility dates. If the eligibility period is split, select Next to enter another eligibility period. Note: split eligibility periods must be within the same eligibility period. If the departure date is:
    • less than 30 days, e.g. the Date signed field is earlier than the departure date, change the date signed date to the departure date. If it is not changed it will default to the date signed date, which may impact the MES eligibility period
    • more than 30 days, e.g. the Date signed field is earlier than the departure date but falls to the next financial year, return the application to the applicant with the appropriate letter z0132
  • Current visa number – leave blank
  • Date signed –usually keyed as indicated on form. Where date signed is not completed, key date of lodgement or processing
  • Signed by – select Applicant
  • Visa sighted – select the tick box even though visa information is not always required
  • Passport sighted – if passport is submitted, select the tick box. If not submitted do not select the tick box
  • Other error – leave blank (DO NOT USE)

6

Agency tab + Read more ...

Complete the Agency tab where the application has been completed by a tax agent.

  • Tax Agency Name – key company name. Abbreviate name where appropriate
  • Tax Agent Number – key ‘0’
  • Tax Agent – key the name of the tax agent who completed the application form. If the name of the tax agent is the same as the tax agency name, key the full name into Tax Agency Name field and ‘T/A’ in Tax Agent field
  • Phone number – key in the tax agent’s phone number as indicated on form. If no number has been provided key ‘0’

Note: if postal address does not belong to the tax agent but details have been provided on MES application form, key details in the Comments tab – (T/A name, phone number). Do not key details in the Agency tab.

When no permission indicated on the application form, key details in the Comments tab – agency details and no permission given.

7

Comments tab + Read more ...

Enter any relevant comments related to the application in the Comments tab. Information in Comments tab assists with any enquiries received for the application.

Make sure comments are clear so other Service Officers can determine a status of an application.

8

Processing application outcomes + Read more ...

Once all the application information has been keyed in the database, finalise the application with the appropriate outcome of:

  • Approved, see Table 6
  • Rejected, see Table 7
  • Pended, see Table 7

Validate application + Read more ...

Select Validate.

The MES database system validation process will assess information based on business rules.

Rejected or pended MES applications + Read more ...

An application is rejected where the applicant is not entitled to MES, whereas an application is pended when there is insufficient information to determine MES.

The reject function of the MES database is no longer used due to outdated functions. Rejected applications are forced into a ‘pend’ status and comments must be added to identify that the application has been rejected.

Pend application + Read more ...

If the box is not unticked, a letter may still be issued by the system even though the MES application has been pended.

The pended application screen will return an application number.

9

Email letter + Read more ...

Rejected MES application

This is mailed or emailed to the applicant (see Table 17 for the email response process). Quality assurance completed by accredited officer.

10

Upload copy of letter in PaNDA + Read more ...

Insufficient Information – PaNDA status ‘Complete’

Even though the application has not been approved, the assessment is now complete.

When the MES application is returned with the requested information/documents, it will be assessed as an originally submitted MES application.

Rejected MES Application – PaNDA status ‘Rejected’.

Reprint MES

Step

Action

1

Locate application + Read more ...

  • to locate the correct application, conduct a search, see Table 4
  • open application:
    • select File
    • select Open
    • key the first 3 letters of the surname and ‘%’ in the Surname field, for example, for Smith, key ‘Smi%’, this will expand the search parameters
    • Key date of birth of the applicant
    • select Search

For:

  • phone requests, go to Step 2
  • non phone requests, where there is:
    • a new address, go to Step 8
    • the same address, go to Step 10

2

Identify the caller as the applicant or tax agent + Read more ...

If the caller is:

  • tax agent, go to Step 3
  • applicant, go to Step 4

3

Confirm the authorised tax agents details + Read more ...

Confirm the authorised tax agent details under the Agency tab or Comments tab in the MES database:

  • can be confirmed, go to Step 4
  • cannot be confirmed, application details cannot be discussed, this includes the status of the MES application. Advise the tax agent the applicant must contact Services Australia about their application. Procedure ends here

4

Perform a security check + Read more ...

Confirm the applicant's details in the MES database:

  • full name
  • date of birth
  • address
  • period being claimed

Check the above details against the MES database. If the full name (including title), date of birth, or period being claimed is not correct, the statement details will need to be re-keyed in full. The reprint process ends here and a new MES statement will be issued.

5

Postal address + Read more ...

When the recorded postal address in the MES database is:

  • correct (the same), go to Step 6
  • incorrect, go to Step 7

6

Determine how long it has been since the original statement was printed + Read more ...

  • Select Actions
  • Select History
  • Refer to Print date column to identify the print date

If the:

  • date has been 2 weeks or more since the original statement print date, go to Step 11
  • original statement print date is under 2 weeks since the original statement print date, advise the applicant the application has been processed and should be received shortly. Procedure ends here

7

Establish who the caller is + Read more ...

If the requester is the:

  • tax agent, advise the tax agent that the statement cannot be reprinted unless the request comes from the applicant, procedure ends here
  • applicant, is the postal address the same?
    • Yes, go to Step 9
    • No, go to Step 8

8

Enter new postal address + Read more ...

Before updating the address, confirm with the applicant if the new postal address is their tax agents.

In the MES database:

  • Select Action
  • Select Addresses
  • Enter new postal address
  • Select Store New Address
  • If the system returns an error message, either address is too long and needs to be abbreviated or symbols have been used that are not acceptable, select Close

The address will not change on the original name tab as the application has been previously certified.

Updated address details will only be visible once MES application has been totally closed, for instance selecting ‘X’ to close the application.

Change of address details can be viewed via Actions>Addresses

Where the new postal address is:

  • the tax agent’s address, go to Step 9
  • not the tax agent’s address, go to Step 10

9

New postal address is the tax agent’s address + Read more ...

Tax agent details cannot be updated in the Agency tab in the MES database once an application is certified.

Enter tax agents details in the Comments tab in the MES database, include:

  • Tax Agency
  • Tax Agent
  • Tax agent phone number

10

Add comment + Read more ...

  • Comments must be updated in Comments tab with action taken:
    • Statement reprinted
    • Address updated – statement reprinted
    • Tax agent - address updated – statement reprinted
    • Indicate the channel of reprint being sent e.g. post, email or both
  • Once all details have been keyed and verified on the duplicate statement, in the MES database, select Save and close the tab

11

Reprint statement + Read more ...

Reprinting a statement is a manual process and cannot be requested through the database.

To reprint a statement, complete standard letter template MES reprint statement z2442:

  • Enter the applicant’s full name, address and application number

Table 13: reasons for revoking a MES and steps delegated staff follow when revoking a MES.

1

Revoking a MES + Read more ...

A MES application may need to be revoked when the application has been certified and it has later been identified that it has been incorrectly assessed and approved.

This could mean that the application was:

  • approved and issued for the whole financial year but should have been issued for a partial period, or
  • approved for whole or partial financial year but the applicant should not have been issued with a MES as they are ineligible

If an incorrect assessment has been identified, escalate the MES application to a more experienced Service Officer or Programme Support Officer (PSO) by recategorising the work item.

If it is a new application for a different financial year and the assessment has identified previous application/s may have been approved incorrectly and will possibly need to be revoked, follow the steps below:

  • Key the new application in the MES database and force into Pend status, see Table 5
  • Add processing notes 'application escalated as a possible revoke (state assessment reasons)' in PaNDA
  • Escalate application:
    • General layer staff will un-assign the application in PaNDA and select reason Proficiency Raised and notate in Add processing notes why unable to action
    • Staff with skill tag ‘established’ will reassess to determine if further escalation is required
    • Established staff skill tag will pend the MES file as ‘Previous Work in Progress Extended’ and send email to MES processing team with the MES applicants name, application number in the subject field and details of assessment reasons for revoking application in body of email
  • A Service Officer with the required skill tag will assess the application to determine if the application is to be revoked. To make an accurate assessment they may need to:
    • retrieve previous MES applications
    • seek further information from the Department of Home Affairs via DHA Medicare helpdesk
    • check International Movement Records and/or VEVO, see Access VEVO (Visa Entitlement Verification Online) program
    • seek advice from Medicare Consumers Program Support - Medicare Consumers Section - Medicare and Veterans Health Branch (Tier 3 MES helpdesk National Office)
