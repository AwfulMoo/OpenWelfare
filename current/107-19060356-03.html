Writing off a debt

For Debt Staff only.

Table 1

Step

Action

1

Benefit status + Read more ...

A withholding arrangement cannot be updated if the customer's Child Care Subsidy (CCS) status is Current zero rate (CZR).

If the customer has a voluntary arrangement in place and they later become CCS current:

  • 20% withholdings from their CCS will automatically commence
  • the customer is under no obligation to honour the voluntary arrangement, in this situation

Go to the 'Benefit History' tab to check the status of CCS.

Is the status CZR?

  • Yes:
    • a CCS withholding arrangement cannot be updated
    • add a voluntary arrangement
    • procedure ends here
  • No, go to Step 2

2

Identify customer and Debt Case ID + Read more ...

When a customer has debts in both Process Direct and DMIS, action general write off reasons in both systems at the same time. This will create a Manual Follow-up (MFU) activity. The MFU will finalise when the write off has been actioned in both systems.

Identify the customer:

  • Select customer in Customer Summary
  • Select the Debt icon to launch Debt Management
  • The Debt List screen will display debts that are:
    • undetermined (will not display in DMIS)
    • determined
    • written off
    • waived
    • referred to an external collection agent, and
    • finalised
  • Select the Debt Case ID number for the debt that needs to be written off

3

Add Write Off + Read more ...

Ensure the selected Debt Case ID number is correct:

  • Select > Write Off Debt

Choose the reason for the write off from the drop down menu.

The End date field defaults to the maximum allowed period. Change this date to match the relevant write off reason. The date cannot exceed the maximum period for write off reason.

Write offs that apply to debts in DMIS and Process Direct must be applied in both systems. See the Resources page for more information.

4

Key notes and add documents + Read more ...

Key write off details in the Summary field.

Key extra details in the Content field.

Does a supporting document need to be added?

  • Yes:
    • Select to attach supporting documents for the write off
    • Select Browse to locate the file to upload
    • Complete the Document Name field. Note: this must be keyed
    • Add any extra information in the Description field
    • Go to Step 5
  • No, go to Step 5

5

Submit and confirm action + Read more ...

To complete the update:

  • Select Submit to display the Confirmation screen
  • Select:
    • Yes, to continue. This will show a confirmation message. Go to Step 6
    • No, to return to the Add Write Off Debt screen. Select Cancel to cancel all updates

6

View + Read more ...

The Debt details screen will now display the write-off.

Select the Write Offs icon to show details of all write offs that have been applied to this debt.

Re-refer debt to collection agent + Read more ...

Only debts with the following temporarily written off reasons can be manually re-referred:

  • DOS (Overseas)
  • NCE (Not cost effective), or
  • WUN (Whereabouts unknown)

The debt must also be:

  • DET (Determined), or
  • DWO (Written Off), and
  • previously been referred to an ECA

Select > Re-refer to Collection Agent.

On the Re-refer Debt to Collection Agent screen, select an ECA from the drop down menu.

Note: select the same ECA previously used for the customer.

An ECA will be assigned if nothing is selected. Go to Step 4

3

Recall debt from collection agent + Read more ...

On the Recall Debt from Collection Agent screen, complete the following fields:

  • Reason
  • Summary

    • Yes, to continue. Go to Step 5
    • No, to display the Re-refer Debt to Collection Agent screen. Select Cancel to cancel all updates

Reallocation of repayments

For Debt Staff only.

Table 5

Step

Action

1

Identify customer and repayment details + Read more ...

Identify the customer:

  • Select customer in Customer Summary
  • Select the Debt icon to launch Debt Management
  • On the Debt List screen, select the Repayments icon to display:
    • repayment source
    • allocated amount
    • over recovered amount
    • refunded amount, and
    • reversed amount

Select:

  • Repayment Details icon on the repayment line
  • To:
    • reallocate to CCS debt, go to Step 2
    • reallocate to DMIS, go to Step 3
    • refund, go to Step 4

Note: from October 2020, refunds for over recovered Child Care Subsidy (CCS) debts are auto paid to the customer. The auto refund will be actioned after:

  • 7 days if the Repayment Source is 'Cheque' or 'Credit card'. Note: a manual refund can be processed if the customer contacts between 2 and 7 days
  • 2 days in all other cases

The manual refund process remains in place for all other over recovered debts.

2

Reallocate to CCS debt + Read more ...

On the Repaid components screen, select the Reallocate icon on the specific component line.

Select the debt to reallocate the repayment to.

The Reallocate amount will default to the maximum repaid component amount up to the value of the selected debt. Do not reduce the amount more than the value of the debt selected. An error message will display if the amount entered is incorrect. The reallocate amount cannot be greater than the outstanding amount of the selected debt.

Select Reallocate to populate the Details field. Select:

  • Submit if details are correct, or
  • Cancel to cancel updates or make changes

Check details are correct and select Submit.

Select Submit on the Confirmation screen.

Procedure ends here.

3

Reallocate to DMIS, add notes and submit + Read more ...

Only reallocate repayments received from CCS Withholdings to DMIS child care debts (CCB/CCR/CCF).

Basic Details - Reallocate Type:

  • Full reallocation: defaults to full repayment amount
  • Partial Reallocation: key amount to reallocate

Add notes:

  • Key the withholding details in the Summary field
  • Record extra details in the Content field

Select Submit. If there is:

  • an outstanding balance in DMIS, confirmation message displays
  • no outstanding eligible balance in DMIS available, an error message displays:
    • 'E277: Cannot reallocate to DMIS, as there are no eligible debts'
    • select Cancel to return to previous screen and cancel again to cancel all updates

Procedure ends here.

4

Refund Repayment, add notes and submit + Read more ...

Repaid Components displays. Outstanding DMIS debts balance will display before refund.

Basic Details:

Select from the following 2 refund methods from the drop down menu options:

  • EFT Electronic Fund Transfer. Select:
    • full refund, or
    • partial refund, and
    • key Bank details from the drop down menu
  • RFM – Refund Manual (SF030). Prepare a Notice to prepare a Refund Account (SF030), see Refunding debt repayments. Select either:
    • full refund, or
    • partial refund – key the amount to refund
    • use Browse to locate and upload the completed SF030
    • the Document Name field must be keyed. Add any extra information in the Description field

Add notes:

  • Key the withholding details in the Summary field
  • Record extra details in the Content field

To complete the update:

  • Select Submit to display the Confirmation screen. Note: error message E347: Payment Lock (Bank details invalid), will display if the customer has had money returned from the selected bank account. If so:
    • contact the customer
    • update bank account details
  • Select:
    • Yes, to complete refund. Go to Step 5
    • No, to return to the Refund screen as no action is taken

5

Repayments screen + Read more ...

Check the Confirmation displayed on the Repayments screen. For refunds via:

  • SF030, the changes will be applied when ESSentials team processes the SF030 manually
  • EFT, the payment will be made in the next payment run cycle

The update to the Repayments screen will display based on the refund method and when this is finalised.

Viewing debt repayment withholdings from an ongoing CCS entitlement

Table 8

Step

Action

1

Withholdings from an ongoing CCS entitlement + Read more ...

  • The 20% (or nominated percentage) withholdings will be calculated from the entitled amount after the 5% customer withholding amount is applied
  • The withholdings will be made weekly, or as session reports are received
  • Withholdings can be deducted from Additional Child Care Subsidy (ACCS) payments - including Child Wellbeing (CWB)
  • The 20% (or nominated percentage) withholdings will be calculated as an amount from the total CCS Entitlement for the family for the relevant week. Customers will need to discuss the amounts paid with their providers

Launch the Debt Management App from the Debts icon in Process Direct.

Select the Arrangements tab to view the arrangement type for the customer. If there are withholdings from CCS entitlement:

  • the arrangement type will show 'withholding' and
  • the Benefit Type will show 'CCS'

Select the Repayments tab. Take note of the receipt date and paid amount fields.

2

Child Care page + Read more ...

Go to the session reports twisty on the Child Care page. Review the submitted date of the session reports. The debt withholding will show as recovered the day the session report is processed.

For example, if the session report is received for week ending 13/10/2019 - submitted date 14/10/2019, processed date 14/10/2019. The receipt date of the withholding in the Debt Application is 15/10/2019. This indicates the withholding for 15/10/2019 is for the week ending 13/10/2019.

The entitlement amounts in Session Reports received and Service Payment amounts under the Weekly Subsidy twisty will not be reduced by this withholding amount. The amounts displayed are the customer's full entitlement. When discussing CCS amounts paid to the provider, the Paid Amount in the Repayments tab of the Debt Application must be deducted from this amount.

3

Customer Online View + Read more ...

From the Menu in Customer First, select My Family > Child Care > Child Care Subsidy Summary.

From the Child Care Subsidy Summary, select the View details option for the child. If the customer receives CCS for more than one child, view the details of each child.

Select View CCS Recent Payments. This will display the two amounts for the relevant week. The smaller amount will match with the repayment amount in the Debt Application Repayment screen.

Customers will need to discuss amounts paid with their provider.

The Resources page contains an example of viewing a customer's withholdings from CCS.
