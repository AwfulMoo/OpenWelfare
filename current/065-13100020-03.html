Manage a positive manual adjustment

Table 1: all positive manual adjustments are to be verified and approved by Subject Matter Experts (SME) or Team Leaders.

Step

Action

1

Positive manual adjustment options + Read more ...

Note: for the relevant security roles needed to perform each step, see the Resources page in the Aged Care Staff Portal (ACSP) – Access and using the home page.

To:

  • create a home care or flexible care positive manual adjustment, go to Step 2
  • update an existing home care or flexible care positive manual adjustment, go to Step 3
  • manage a residential care positive manual adjustment, see the Using the Aged Care Staff Portal to manage residential care manual adjustment (Service in Context) table

2

Create positive manual adjustment + Read more ...

Log on to ACMPS:

  • On the Home screen, select Payment Additions option from the Create menu on the left hand side menu
  • The Select Transaction Type screen will display
  • Select the relevant positive adjustment transaction type
    Note: if the relevant transaction type cannot be located, select the Forward option to locate
  • The [Care type] Positive Manual Adjustment: New screen will display
  • In the General Data tab, complete the below fields:
    • the Main Partner (Service ID)
    • Adjustment Type
    • Start Date – the start date must correspond with the month in which the adjustment is to be paid (will default to the current claim month)
    • End Date – the end date must correspond with the month in which the adjustment is to be paid (will default to the current claim month)
  • Select the Entitlement Items tab
  • The Entitlement Items screen will display
  • In the Entitlement Items screen, select New
  • The [Care type] Positive Manual Adjustment: New - Create Entitlement Item screen will display
    • Select Manual Adjustment in the Payment Type field
    • Enter the start date in the Start Date field
      Note: the start must correspond with the month in which the adjustment is to be paid (will default to the current claim month)
    • Enter the end date in the End Date field
      Note: the end date must correspond with the month in which the adjustment is to be paid (will default to the current claim month)
    • Enter the entitlement amount in the Amount field
  • Select Back
  • The Entitlement Items screen will display
  • Select Payment Note tab
  • Select New icon
  • Enter notes giving a brief description of the reason for the manual adjustment
    Note: there is a character limit and notes will be displayed in the payment statement
  • Select Save

Go to Step 4.

3

Update an existing positive manual adjustment + Read more ...

Note: this step only applies when the manual adjustment has not been verified and approved by SME or Team Leaders.

  • Log on to ACMPS:
  • On the Home screen, select Payment Additions option on the left hand side menu
  • The Payments Additions screen will display
  • Complete the below fields to search for a claim:
    • Service ID
    • Select the relevant manual adjustment type from the Category Type drop down list, for example:
      Positive Manual Adjustment for home care positive manual adjustments
      STRC - Positive Manual Adjustment for Short-term Restorative Care positive manual adjustments
      TC - Positive Manual Adjustments for Transition Care positive manual adjustments
    • Select the Status drop down box and select required status to narrow the search for manual adjustment with desired status
    • Payment Addition ID
  • Select Search
  • Select the relevant positive manual adjustment (Payment ID hyperlink) to update
  • The relevant positive manual adjustment details will display
  • Select Undo Decision Assessment option
  • Select the Entitlement Items tab
  • In the Entitlement Items screen, select the Edit option
  • The [Care type] Positive Manual Adjustment: New - Create Entitlement Item screen will display
  • Complete the relevant fields
  • Select Back
  • The Entitlement Items screen will display
  • Select Payment Note tab
  • Select New icon
  • Enter notes giving a brief description of the reason for the manual adjustment
    Note: there is a character limit and notes will display in the payment statement
  • Select Save

4

Decision assessment + Read more ...

In the Positive Manual Adj. New screen:

  • Select Entitlement Determination to calculate the claim amount. This enables the Decision Assessment option (calculation must be made before getting verified)
  • Select Decision Assessment
  • Select Save

Service Officers will advise the SME or Team Leaders via email that a manual adjustment has been created or updated and needs approval.

Note: once the adjustment has been added, SME or Team Leaders must verify the manual adjustment. To verify the manual adjustment, see Table 3

2

Create a negative manual adjustment + Read more ...

  • Log on to ACMPS
  • On the Home screen, select Payment Deductions option under the Create menu on the left hand side menu
  • The Select Transaction Type screen will display
  • Select the relevant negative adjustment transaction type, for example:
    • ZNAD Negative Adju for home care negative manual adjustments
    • ZSND STRC - Negative Adju for Short-term Restorative Care negative manual adjustments
    • ZTND TC - Negative Adjus for Transition Care negative manual adjustments
  • The [Care type] Negative Manual Adju: New screen will display
  • Under the General Data tab, complete the below fields:
    • Main Partner* – the ID of the service for which the negative adjustment is needed. This field is mandatory
    • Description - advise the reason for the negative adjustment
  • Under the Items tab:
    • set the Item Category as Withholding Deductions from the drop down option
    • Select Edit icon
  • The Deduction Plan Item Details screen will display
  • On the Deduction Plan Item Details screen, complete the below fields:
    • Start Date and End Date
      Note:       the start/end dates must correspond with the month in which the adjustment is to be deducted (will usually be the current claim month)
    • Deduct from Party - the ID of the service for which the negative adjustment is needed. This is pre-populated to the Service ID selected as the Main Partner in Step 2
    • Payment Family - choose the relevant option from the list
    • Due date rule - choose from the list (corresponding to the Payment Family value selected)
    • Priority - always set as '999'
    • Deduction amount - the amount to be deducted
    • Partial Deduction - this check box must be set. If the entitlement amount is less than the deduction amount, and if this box is unchecked, then the entitlement amount will not be adjustment with the deduction amount during the net calculation document (NCD) calculation
    • Description - enter the adjustment reason
      Note: there are limits to the number of characters used in ACMPS when notes are entered in the Description field
  • Back
  • Decision Assessment
    • Select Note option
    • Select New icon
    • Enter notes giving a brief description of the reason for the manual adjustment
      Note: there is a character limit and notes will display in the payment statement
    • Save

Service Officers will advise the Subject Matter Experts (SME) or Team Leaders via email that a manual adjustment has been created and requires approval.

Note: once the adjustment has been added, SME or Team Leaders must verify the manual adjustment. To verify the manual adjustment, see Table 3.

Procedure ends here.

3

Updating an existing negative manual adjustment + Read more ...

  • Log on to ACMPS
  • On the Home screen, select Payment Deductions option on the left hand side menu
  • The Search: Payment Deductions screen will display
  • Complete the below fields to search for a claim:
    • Service ID* - this field is mandatory
    • Service Name
    • Status - select the drop down box and select required status to narrow the search for the negative manual adjustment with desired status
    • Payment Deduction ID
  • Select Search
  • Select the relevant Payment ID hyperlink to update
  • The relevant negative manual adjustment details will display
  • Under the Items tab, select Edit icon of the hyperlink in the Item field
  • The Deduction Plan Item Details screen will display
  • In the Entitlement Items screen, select the Edit option
  • Update the entitlement amount in the Deduction Amount field
  • Select Back
  • The Negative manual adjustment details screen will display
  • Select Note tab
  • Select New icon
  • Enter notes giving a brief description of the reason for the manual adjustment (notes will display in the payment statement)
  • Select Save

Service Officers will advise the SME or Team Leaders via email that a manual adjustment has been updated and needs approval.

Note: once the adjustment has been added, SME or Team Leaders must approve the manual adjustment. To verify the manual adjustment, see Table 3.

Search positive manual adjustment + Read more ...

3

Verify or delete positive manual adjustment + Read more ...

  • Select the relevant manual adjustment to view
  • Select the General Data tab to view the adjustment details for example, Service ID, start date, end date etc
  • Select the Gross Entitlement Amounts tab to view the payment amount

Is the SME or Team Leader satisfied with the positive manual adjustment?

  • Yes:
    • Select the drop down menu of the More tab
    • Select Verify
    • Select Save
    • Procedure ends here
  • No:
    • Select the drop down menu of the More tab
    • Select Delete
    • Select Save
    • Go to Step 4

Note: selecting Cancel cancels all the performed actions.

4

Advise outcome to the Service Officer + Read more ...

After deleting the incorrect manual adjustment, send an email to the Service Officer to advise that the manual adjustment has been deleted and the reason for deletion.

It is the Service Officer’s responsibility to enter a new manual adjustment, see Table 1.

Procedure ends here.

Approve or reject negative manual adjustment + Read more ...

  • Log on to ACMPS
  • On the Home screen, select Payment Deductions from the left hand side menu
  • The Search: Payment Deductions screen will display
  • Complete the below fields to search for a claim:
    • Service ID* - this field is mandatory
    • Service Name
    • Status - select the drop down box and select required status to narrow the search for the negative manual adjustment with desired status
    • Payment Deduction ID

  • Select the relevant negative manual adjustment
  • The relevant negative manual adjustment details will display

Check the details for correctness prior to approval.

Is the SME or Team Leader satisfied with the negative manual adjustment?

  • Yes:
    • Select For Approval option to approve the negative manual adjustment
    • Procedure ends here
  • No:
    • Select Reject option to reject the negative manual adjustment
    • Select Save
    • Go to Step 6

6

Advise outcome to the Service Officer + Read more ...

After rejecting the incorrect manual adjustment, send an email to the Service Officer to advise that the manual adjustment has been rejected and the rejection reason.

It is the Service Officer’s responsibility to enter a new manual adjustment, see Table 2.
