Amend Contracted Service Provider (CSP) Officer record in Healthcare Identifiers (HI) Service 012-03070040

  • Process Summary
  • Process

Expand all


This page contains procedures for a Service Officer to amend the Contracted Service Provider (CSP) Officer details registered in the Healthcare Identifiers (HI) Service. The CSP Officer is the registered individual responsible for maintaining the accuracy of the CSP Organisation record details in the HI Service.

Update CSP Officer details registered with the HI Service

Table 1: This table describes how Service Officers amend the CSP Officer details registered with the HI Service. Note:

  • if acting on behalf of another person known to the HI Service, for example a CSP Officer, first search for the individual in the eHealth Program (EHP) prior to searching for the EHP individual
  • for the purpose of this procedure we are not acting on behalf of another individual and will search directly for the individual

    Step

    Action

    1

    Service Officer acting on behalf of another person? + Read more ...

    Is the Service Officer acting on behalf of another person known to the HI Service, for example a CSP Officer, in response to a telephone request?

    • Yes:
      • Complete 'Act for' an EHP individual (procedure)
      • Go to Step 2
    • No, go to Step 2

    2

    Search for CSP Officer + Read more ...

    From the EHP, perform a search for the CSP Officer.

    See Searching for records in the Healthcare Identifiers Service using the eHealth Program (EHP).

    The Search for Individual screen is displayed with one or more matching individuals under search results.

    3

    Navigate to Amend Administrator Details + Read more ...

    • Select 'Amend' in the Action column of the relevant record
    • The Amend Administrator Details screen is displayed with details of all registered CSP Officers in the Recorded Names section of the screen

    4

    Change details via relevant tabs + Read more ...

    • Select the relevant tabs and make the changes requested

    See also:

    • Update name details
    • Code EOI details
    • Update personal details
    • Update address details
    • Update electronic communication details
    • Update linked organisation details

    5

    Check validation + Read more ...

    • Select Submit once all changes are complete. Are there validation errors?
      • Yes, correct errors as prompted
      • No, the Confirm Amend Administrator Details screen is displayed

    6

    Confirm + Read more ...

    If the details are:

    • incorrect, select Edit to return to the Amend Administrator Details screen
    • correct, select Save

    A confirmation message is displayed.

    Add or amend CSP Officer name details

    Amend CSP Officer name details for the HI Service

    Table 2: This table describes how Service Officers add name details to a CSP Officer of an organisation registered with the HI Service. Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request. See 'Act for' an EHP individual (procedure)

    Step

    Action

    1

    Navigate to Name Details tab + Read more ...

    • Select the Name Details tab of the Amend Administrator Details screen
    • The Name Details tab is displayed listing current details

    2

    Name deletion required? + Read more ...

    Does a name displayed in the Recorded Names table need to be deleted?

    • Yes, go to Step 3.
    • No, go to Step 4.

    3

    Delete + Read more ...

    • Select Delete in the Recorded Names table

    • A system confirmation message appears
    • Select OK and go to Step 9

    4

    Add or amend name + Read more ...

    To:

    • Add a new name for the CSP Officer, go to Step 5
    • Amend an existing name displayed in the Recorded Names table, go to Step 6

    5

    Create new name + Read more ...

    • Select Create New Name to add a new CSP Officer name
    • A blank Name Details tab is displayed.
    • Go to Step 7

    6

    Amend name + Read more ...

    • Select Amend in the Action column of the relevant record
    • The Name Details tab is displayed with relevant details listed.

    7

    Complete name details + Read more ...

    Enter the information to be added or amended into the relevant fields. The following fields are mandatory:

    • Family Name
    • Name Usage

    Note:

    • Name Usage Start Date (defaults to today’s date)
    • Start Date Accuracy Indicator (defaults to Day, month and year are accurate)

    8

    Check validation + Read more ...

    • Select Amend once all changes are complete

    Are there validation errors?

    • Yes, correct errors as prompted.
    • No, the Confirm Amend Administrator Details screen is displayed.

    9

    Further amendments required? + Read more ...

    Do more details need to be amended?

    • Yes, select another tab to update.
    • No, go to Step 6 of the Update CSP Officer record table.

    Amend EOI details for a CSP Officer

    Code EOI details for a CSP Officer of a CSP organisation

    Table 3: This table describes how Service Officers amend evidence of identity (EOI) details for a CSP Officer of a CSP Organisation registered with the HI Service.

    Note:

    • Always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request. See 'Act for' an EHP individual (procedure)
    • EOI details already recorded in the HI Service can be deleted and new EOI details can be added to the record.

      Step

      Action

      1

      Navigate to EOI details + Read more ...

      Select the EOI Details tab of the Amend Administrator Details screen.

      The EOI Details tab is displayed, listing current EOI evidence for the CSP Officer.

      2

      Delete selected record + Read more ...

      • Choose the EOI type to be deleted from the record
      • Go to the Action column of the EOI Type and select delete
      • This action will delete the selected record
      • To continue select Ok

      Navigate to Evidence of Identity field + Read more ...

      • Select Create New EOI
      • Empty Evidence of Identity fields are displayed

      4

      Complete information + Read more ...

      • Select the information from the drop-down box in the relevant fields. The EOI Document/s field is mandatory
      • Select Add
      • Select Create New EOI to continue adding EOI information, if required

      5

      Amend further details if required + Read more ...

      Do more details need to be amended?

      • Yes, select another tab to update
      • No, go to Step 6 of the Update CSP Officer record table

      Amend CSP Officer personal details

      Update CSP Officer personal details for the HI Service

      Table 4: This table describes how Service Officers amend the personal details for a CSP Officer of a CSP Organisation registered with the HI Service.

      Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request. See 'Act for' an EHP individual (procedure)

      Step

      Action

      1

      Navigate to Personal Details + Read more ...

      • Select the Personal Details tab of the Amend Administrator Details screen
      • The Personal Details tab is displayed listing existing details

      2

      Update information + Read more ...

      Enter the information to be amended into the relevant fields (overtype existing details where necessary). The following fields are mandatory:

      • Date of Birth
      • Date of Birth Accuracy Indicator (defaults to Day, month and year are accurate)
      • Sex

      3

      Check CSP Officer details + Read more ...

      If the CSP Officer is:

      • identified as a potential replica record, go to Step 4
      • deceased and the date of death needs to be entered, go to Step 5

      If amendments to personal details are complete, go to Step 6 of the Update CSP Officer record table.

      4

      Possible replica record? + Read more ...

      If the CSP Officer is identified as a potential replica:

      • Select Record Potential Replica
      • Complete Identify and resolve replica for EHP individual
      • Go to Step 6

      5

      Update death details if required + Read more ...

      If a date of death needs to be entered:

      • Select Retire. Date of death fields are enabled
      • Update date of death details
      • Go to Step 6

      6

      Amend further details if required + Read more ...

      Do more details need to be amended?

      • Yes, select another tab to update
      • No, go to Step 6 of the Update CSP Officer record table

      Add or amend CSP Officer address details

      Update CSP Officer address details for the HI Service

      Table 5: This table describes how Service Officers add or amend address details for a CSP Officer of a CSP Organisation registered with the HI Service.

      Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request. See 'Act for' an EHP individual (procedure)

      Step

      Action

      1

      Navigate to Address Details + Read more ...

      • Select the Address Details tab of the Amend Administrator Details screen
      • The Address Details tab is displayed listing current details

      2

      Deletion of address required? + Read more ...

      Does an address displayed in the Recorded Addresses table need deleting?

      • Yes, go to Step 3
      • No, go to Step 4

      3

      Delete address + Read more ...

      • Select Delete in the Recorded Addresses table

      • A system confirmation message appears
      • Select OK and go to Step 8

      4

      Create or amend address + Read more ...

      To:

      • Create a structured or unstructured Australian or international address, go to Step 5
      • Amend an existing address displayed in the Recorded Addresses table, go to Step 6

      5

      Navigate to blank Address details + Read more ...

      • Select Create New Address to create a structured or unstructured Australian or international address.
      • A blank Address Details tab is displayed
      • Go to Step 7

      6

      Navigate to relevant address details + Read more ...

      • Select Amend in the Action column of the relevant record
      • The Address Details tab is displayed with relevant details listed
      • Go to Step 7

      7

      Address details + Read more ...

      • Select Address Details tab
      • The Unstructured Address Indicator tick box will be automatically selected. See Address formatting in Healthcare Identifiers
      • Enter the address in the Address Search box, starting with the number of the address. The rapid addressing tool will predict the address and display it as the number and street name starts to be entered. If the address that appears is the correct address, select it
      • The following information will pre-populate into the relevant fields:
        • Address Lines
        • Suburb/Town/Locality
        • State
        • Postcode
      • Select Address Purpose
      • Address Start Date (defaults to today’s date)
      • Start Date Accuracy Indicator (defaults to Day, month and year are accurate)
      • Select Add

      8

      Amend further details if required + Read more ...

      Do more details need to be amended?

      • Yes, select another tab to update
      • No, go to Step 6 of the Update CSP Officer record table

      Update CSP Officer linked organisation details for the HI Service

      Table 7: This table describes how Service Officers amend linked organisation details for a CSP Officer of a CSP Organisation registered with the HI Service.

      Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request. See 'Act for' an EHP individual (procedure)

      Step

      Action

      1

      Navigate to Amend Administrator Details + Read more ...

      • Select the Linked Organisation Details tab of the CSP Officer screen
      • The Amend Administrator Details screen of the Linked Organisation Details tab is displayed

      2

      Expand list of organisations + Read more ...

      • Select View Linked Organisations to expand the list of organisations to which the CSP Officer is linked
      • All organisations to which the CSP Officer is linked are displayed

      3

      • the CSP Officer record is identified as a potential duplicate record, go to Step 4

      If the CSP Officer is identified as a potential duplicate:

      • Select Record Potential Duplicate

      • Go to Step 6

      5

      Amend links + Read more ...

      To amend links to organisations:

      • Complete Link or unlink Contracted Service Provider (CSP) Officer to a CSP organisation record in Healthcare Identifiers (HI) Service
      • Go to Step 6

      6

      Amend further details if required + Read more ...

      Do more details need to be amended?

      • No, go to Step 6 of the Update CSP Officer record table
