Annual practice registration renewal

Table 1: This table describes how to assess and process an Application to register or amend a diagnostic imaging or radiation oncology practice form (HW061) when a practice wants to extend their registration.

Step

Action

1

Assess application + Read more ...

  • Review the HW061 form to make sure Questions 1-5 and any other relevant sections of the form are complete. The form must be signed by a current authorised representative or the proprietor
    • Make sure the correct declaration has been signed
    • If the HW061 form requests an amendment and is received while the Renew LSPN button is active (usually 3 months before the End Date), continue to process the amendments and renew the LSPN record. Requests received prior to this timeframe, are processed without any change to the end date

Note: on the declaration form there are tick boxes to indicate whether the signatory is the proprietor or the authorised representative.

If the HW061 form is incomplete or incorrect return the application with a Z2374 letter to the practice outlining all incomplete or incorrect information.

2

Verify authorised representative + Read more ...

Log in to PDS and: select Organisations

  • select View
  • in basic search, key the LSPN and select ID Type LSPN
  • select Search
  • the LSPN record will display

Verify a current recorded authorised representative, or proprietor has signed the declaration. Enter new authorised representative details if these are nominated by an authorised person.

Note: return the application as incomplete if the declaration is not signed by:

  • a current authorised representative
  • the proprietor, or
  • a responsible senior officer who has also provided a signed letter

3

Extend practice registration + Read more ...

Registration can only be extended 3 months before the End Date, up until a delegate cancels the registration. The Renew LSPN function is not available outside this timeframe.

If registration can be extended:

  • select the Renew LSPN button at the bottom of the LSPN record
  • a pop up message asks to confirm if an authorised representative has confirmed the details are current and up to date
  • select the checkbox and Confirm
    • PDS extends the registration End Date by 12 months
    • a HPOS notification is sent to all HPOS Administrators advising of the renewal. That is, where at least one Administrator is linked to the LSPN record, and an electronic messaging opt out is not in place
    • the Comments are updated with a message which includes the new End Date and the P number of the officer that renewed the registration of the LSPN record
  • If a renewal request is received for an LSPN record which has a Cancelled status:
    • forward the request to a Programme Support Officers (PSO). The Pathology and Diagnostic Imaging Team consider these requests

4

Update other primary information + Read more ...

If relevant, update the business, contact, equipment and/or MRI provider details as requested (see Amend details of a diagnostic imaging practice for assistance).

5

File the application + Read more ...

  • Add a note to the PaNDA file stating ‘Actioned”
  • Select Complete and Save

6

Form has been received via HPOS + Read more ...

If Services Australia receive the form via HPOS, deliver return correspondence via HPOS.

Forms received via HPOS can be identified, as the naming convention will contain the 10-digit RA number of the user. The HPOS Reference Number field will contain a 9-digit reference number to search in HPOS to reply. For example, Business name: 9987654321, and the HPOS ref # field: 123456789-1.

Service Officers will need to amend the business name from the RA reference to the LSPN to ensure applications can be easily found.

For further instructions, see Health Professional Online Services Support Mail Centre – Managing emails from health professionals.

Add or amend the practice’s accreditation

Table 2: This table describes how to add and amend accreditation details in the Accreditation tab in the register.

Step

Action

1

Access the LSPN register in PDS + Read more ...

  • Log into PDS
  • Select Organisations from the left hand side menu
  • Select View
  • in basic search, key the LSPN and select ID Type LSPN
  • Select Search
  • LSPN record displays in View.

2

Verify practice details + Read more ...

Ensure the correct LSPN record is being updated.

From the LSPN view screen, verify the practice/trading name and address, by comparing it to the information recorded on the accreditation report.

Do the details match?

  • Yes, select Edit to display the LSPN record in Amend, go to Step 3
  • No, do not action. Email the accreditor advising details on the report that do not match with Services Australia records

3

Determine the type of accreditation action required + Read more ...

Select the Accreditation tab.

Compare the information on the accrediting body’s report, determine which of the 3 scenarios (there may be more than 1 scenario on the report for different modalities) which need to be actioned. If:

  • no start and end dates have previously been recorded for a modality type listed in the report, go to Step 4
  • there is a change of accreditor, go to Step 5
  • if there is an N on the report for a specified modality, go to Step 6
  • if dates for a modality type needs to be amended, go to Step 7

4

Create new modality record + Read more ...

On the Accreditation tab select Add Modality Accreditation. Input the following details in the relevant field titles on the Add Accreditation modal as per the accreditation report:

  • Status - status type, key either:
    • A - accreditation against full suite of standards
    • Y - accreditation against entry level standards
  • Start Date - key in start date/date of accreditation/transfer
  • End Date - key the end date as provided on the report
  • Accreditor - key the accrediting body (either NATA, QIP or HDAA)
  • End reason - select ACR – Accreditation Ended
  • Select Add – continue to add modalities as per the report if required
  • Select Cancel or X to close the Add accreditation pop up modal
  • Select Save

5

Record modality for a new accreditor + Read more ...

Before adding new accreditation details, check the existing accreditation for all modalities. Make sure the End Date is before the new accreditation period begins. Where there is no cessation report for a previous accreditation, the previous accreditation body must be contacted to get a copy of the cessation report before processing the new modality report.

  • On the Accreditation tab, select the Manage link for the relevant modality accreditation
  • The Manage Accreditation modal will display
  • Select Add Accreditation
  • Input the following details in the relevant field titles on the Manage Accreditation pop up modal as per the accreditation report: Status - status type, key either:
    • A - accreditation against full suite of standards
    • Y - accreditation against entry level standards, or
  • Start Date - key in start date/date of accreditation/transfer
  • End Date - key in end date provided on the report
  • Accreditor - key in accrediting body (either NATA, QIP or HDAA)
  • End reason - select ACR – Accreditation Ended
  • Select Add
  • Select Cancel or X to close the Manage accreditation pop up modal
  • Continue this process for all modalities listed on the report
  • Select Save

6

Ceasing accreditation + Read more ...

  • On the Accreditation tab, select the Manage link for the relevant modality The Manage Accreditation (Mange) modal will display
  • Select the Amend link for the relevant accreditation period being end dated
  • Key the End Date as advised by the accreditor
  • Select Update

Repeat the above process for each modality as per the report

  • Select Cancel or X to close the Manage Accreditation modal
  • In the Comments tab, select Add Comment
  • Leave a comment advising:
    • Which modalities were end dated
    • The end date recorded
    • The reason (extension/change of accreditor/ceased)
  • Select Add
  • Select Save

7

Amend existing date + Read more ...

On the Accreditation tab:

  • select the Manage link for the relevant modality being amended
  • the Accreditation modal will display:

For accreditation extensions that are to be continuous:

  • Update existing accreditation with overlapping dates, making the End Date the day before the new accreditation period begins. See Step 6 for instructions on ceasing accreditation
  • Add a new accreditation line:
    • Select the Add accreditation button in the Manage Accreditation modal
    • Enter the details that appear on the current report
    • Select Add
    • Select Cancel or X to exit the Amend Accreditation (Manage) modal
    • Follow these steps for all modalities included on the report
    • Select Save

For corrections, select the Amend link for the relevant accreditation period being updated:

  • Amend the information in line with the accreditation report in the Manage Accreditation modal
  • select Update
  • Select Cancel or X to exit the Amend Accreditation (Manage) modal, and
  • Select Save

8

Delete an existing entry + Read more ...

On the Accreditation tab:

  • select Manage link for the relevant modality accreditation being amended
  • the Manage Accreditation modal will display, select the Delete link for the relevant accreditation period being deleted
  • select the Delete Selected Accreditation checkbox on the Delete Accreditation (single) modal
  • select Confirm Delete
  • select Cancel or X to exit the Amend Accreditation (Manage) modal
  • select Save

Note: if required there is an option to delete all accreditation periods for a particular Modality:

  • From Amend Accreditation (Manage) modal select Delete All
  • On the Delete Accreditation (All) modal select the Delete all accreditation for this modality checkbox
  • Select Confirm Delete
  • Select Cancel or X to exit the Amend Accreditation (Manage) modal
  • Select Save

Amend equipment details of a diagnostic imaging or radiation oncology practice in the LSPN register

From 1 July 2018, transducer information is no longer required.

Step

Action

1

Equipment tab + Read more ...

On the Equipment tab:

  • if adding equipment:
    • click on Add Equipment button
    • complete the equipment details

    • select Save

Step

Action

Organisation tab + Read more ...

Process practice registration closure request – including relocations

Table 9: This table describes how Service Officers process a registration closure request, including when there is a change of address.

Step

Action

1

Confirm closure/ relocation date + Read more ...

The Application to register or amend a diagnostic imaging or radiation oncology practice form (HW061) can be completed to notify Services Australia that an existing Practice is subject to closure or relocation to a new address.

See the Background page for more information on applications for relocation.

2

Close previous practice registration on the DI register + Read more ...

In Amend:

  • Update the Organisation tab
    • Select End Reason Code (C/S/-) as Closed. If Closed is not selected as an End Reason Code then the equipment at the LSPN will not be automatically removed
    • Enter closure date in the End Date field
    • Note: if the date is input before selecting the end reason code, re-check that the date input still remains
  • select Save

All equipment without a date removed, or date removed in the future automatically updates to match the LSPN end date. Check the LSPN view screen that these dates have been correctly applied.

Note:

  • end dating equipment is important as it cannot be listed on another LSPN record if still listed on the LSPN record of the closing practice
  • if a new application is received on the same form as the closure request, ensure that a separate Work Item is created for each LSPN record. For example, a separate Work Item for the closed LSPN record and another for the new LSPN record

3

Relocations - Register the new practice on the register + Read more ...

The new practice details must be completed on the Application to register or amend a diagnostic imaging or radiation oncology practice form (HW061) to complete the relocation. See Register for a Location Specific Practice Number (LSPN) for Medicare.

Respond to equipment list requests

Table 10: this table describes how to assist a caller who wishes to access an equipment list for a practice.

Step

Action

1

Proprietor or authorised contact + Read more ...

Perform a security check for the LSPN record.

Is the caller a proprietor or authorised contact?

  • Yes, go to Step 2
  • No,
    • guide them on how to register for PRODA, and
    • advise that an authorised person needs to call to organise an HPOS link

2

Assist caller in accessing equipment list on HPOS + Read more ...

If the caller:

  • is linked to the LSPN record in HPOS with Staff or Associate access, advise this information is available online
  • already has a PRODA account but is not linked to the LSPN record, offer to link them
  • does not have a PRODA account, advise them to:
    • sign up for PRODA, and
    • call the Provider Registration line to be added as an associate
  • is unsure how to access their equipment list through HPOS, advise they can access the guide by searching LSPN HPOS on the Services Australia website

3

Unable to access equipment list + Read more ...

The authorised representative/proprietor/responsible executive officer may submit an email to the Pathology and Diagnostic Imaging Team.

The email should include why the practice has been unable to access their equipment list using HPOS.

These requests are assessed on a case-by-case basis.

Suspend practice registration – Provider Services Support or National Office only. APS5 delegation required

Table 12: This table describes how to suspend practice registration. Service Officers do not action these requests, escalate for delegate approval if received in error.

Step

Action

1

Access register + Read more ...

APS5 delegation required.

Log on to PDS and:

  • Select Organisations from the side menu
  • Select View
  • Search for the LSPN
  • Confirm the practice’s registration has not yet been renewed

2

Confirm reminder letters were sent + Read more ...

  • Select Correspondence from the side menu of PDS
  • Enter the LSPN in the Search ID field
  • Select LSPN as the ID Type
  • Select Search
  • Verify at least one reminder letter was sent

Note: Sections 23DZW and 23DZZV of the Health Insurance Act 1973 requires any requests at least 28 days prior to suspension.

3

Check for outstanding or rejected applications in PaNDA + Read more ...

  • In PaNDA, search the LSPN Application and LSPN Accreditation work types, using the LSPN as the search term
  • For applications received from 3 months prior to the LSPN End Date to the current date:
    • Check to see if it was a valid request
    • For rejected applications, verify the rejection decision was correct
  • If a valid request was sent, make sure:
    • all details are updated correctly, and
    • renew the practice registration. See the instructions in Table 1
  • If no valid request was received, go to Step 4

4

Suspend practice registration + Read more ...

  • Return to the LSPN record in PDS and select the Suspend LSPN button
  • Select the Suspend Selected LSPN tick box in the Suspend modal
  • Select Confirm
  • The system automatically actions the following:
    • the Status field changes to Suspended
    • a letter generates and sends to linked Associates, or the practice’s Mailing Address informing them of the suspension
    • a comment saves confirming the suspension, that a letter was sent (including which channel), who it was sent to and when this occurred

Cancel practice registration – Provider Services Support or National Office only. EL1 delegation required

Table 13: This table describes how to cancel the registration of a suspended practice. Service Officers do not action these requests, escalate for delegate approval if received in error.

Step

Action

1

Access register + Read more ...

EL1 delegation required

Log on to PDS and:

  • Select Organisations from the side menu
  • Select View
  • Search for the LSPN
  • Confirm the LSPN’s registration status is suspended

2

Confirm reminder letters were sent + Read more ...

  • Select Correspondence from the side menu of PDS
  • Enter the LSPN in the Search ID field
  • Select LSPN as the ID Type
  • Select Search
  • Verify:
    • at least one reminder letter was sent
    • the suspension letter was sent

Note: Sections 23DZX and 23DZZW of the Health Insurance Act 1973 requires a practice to be informed of its suspension before a decision to cancel can be made.

3

Check for outstanding or rejected applications in PaNDA + Read more ...

  • In PaNDA, search the LSPN Application and LSPN Accreditation work types, using the LSPN as the search term
  • For applications received from 3 months prior to the LSPN End Date to the current date:
    • Check to see if they were a valid request
    • For rejected applications, verify the rejection decision was correct
  • If a valid request was sent, ensure all information has been updated correctly and renew the registration as per the instructions in Table 1
  • If no valid request was received, go to Step 4

4

Cancel LSPN registration + Read more ...

  • Return to the LSPN record in PDS and select the Cancel LSPN button
  • Select the Cancel Selected LSPN tick box in the Cancel modal
  • Select Confirm
  • The system automatically actions the following:
    • the Status field changes to Suspended
    • a letter generates and sends to any linked Associates, or the practice’s Mailing Address informing them of the cancellation
    • a comment is saved confirming the suspension, that a letter was sent (including which channel), who it was sent to and when this occurred

Note: Section 23DZZC of the Act prevents a proprietor from registering a practice at a cancelled location for 12 months without approval from an EL2 delegate.
