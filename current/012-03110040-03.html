This page contains the procedures that outline how to amend a Healthcare Provider Identifier - Organisation (HPI-O) record in the Healthcare Identifiers (HI) Service.

Initial steps to amend the details of an organisation

Table 1: describes the first actions Service Officers take to amend the details of an organisation registered with the HI Service.

Step

Action

1

Check person known to Healthcare Identifier + Read more ...

Is the Service Officer acting on behalf of another person known to the Healthcare Identifiers (HI) Service (Responsible Officer (RO) or Organisation Maintenance Officer (OMO)), for example, in response to a telephone request?

  • Yes, complete Handle 'Act for' call in Healthcare Identifiers (HI) Service, then go to Step 2
  • No, go to Step 2

2

Search for the associated organisation + Read more ...

From the eHealth Program (EHP), search for the associated organisation.

3

Select Amend + Read more ...

From the Search for Organisations screen, select Amend in the Action column of the relevant record.

One of the below displays:

  • Amend organisation Seed Organisation screen, defaulting to the Public Officer Details tab
  • Amend organisation Network Organisation screen, defaulting to the Organisation Name Details tab

4

Navigate to each relevant tab and make the requested changes + Read more ...

Select the relevant HPI-O procedure:

  • Add or amend organisation name details
  • Amend organisation details
  • Add or amend organisation service details
  • Add or amend address details
  • Add or amend electronic communication details
  • Add or amend organisation ELS details
  • Amend linked organisation details
  • Add or remove linked provider details
  • Add or amend OMO details
  • Add or amend HI HPD entries
  • Amend certificate information

5

Select Submit + Read more ...

Are there validation errors?

  • Yes, correct errors as prompted, then select Save and go to Step 6
  • No, go to Step 6

6

Check details are correct + Read more ...

Check that the details on the Confirm Amend Organisation Maintenance Role screen displayed are correct. If the details are:

  • Incorrect or additional updates are required, select Edit and amend the relevant details on the Amend Organisation Maintenance Role screen. Go to Amend HPI-O procedures and select the relevant update required
  • Correct, select Save. A confirmation message is displayed

Add or amend the name details of an organisation

Table 2: follows on from the Initial steps to amend an organisation record table and describes how Service Officers add, amend or delete name details for an organisation registered with the HI Service.

Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request.

Step

Action

1

Search for name in the Recorded Organisation Names table + Read more ...

Select the Organisation Name Details tab of the Amend HPI-O Seed/Network Organisation screen.

Does a name displayed in the Recorded Organisation Names table need to be deleted?

  • Yes, go to Step 3
  • No, go to Step 4

2

Evidence of Identity (EOI) and authority to act documentation + Read more ...

If the organisation name that is changing has been previously verified via the EOI and authority to act documentation, updated business EOI and authority to act is required.

Is the organisation name changing from what is currently used on the EOI and authority to act documentation?

  • No, go to Step 3
  • Yes, if the new EOI and authority to act documentation has:
    • been provided, go to Step 3
    • not been provided, procedure ends here. Updated business EOI and authority to act is required in order to complete the name change

3

Delete name + Read more ...

Select Delete in the Recorded Organisation Names table.

When a system confirmation message appears, select OK and go to Step 8.

4

Amend or add name + Read more ...

To:

  • amend an existing name displayed in the Recorded Organisation Names table, go to Step 5
  • add a new name for the organisation, go to Step 6

5

Amend information + Read more ...

Select Amend in the Action column of the relevant record.

Depending on what type of amendment is being carried out, amend or complete the required field in the system as shown below.

In the:

  • Organisation Name field, correct an error
  • Organisation Name Usage section, select the appropriate tick box
  • Name Usage End Date, key the end date as either:
    • date before the processing date
    • date provided at Question 14
  • End Date Accuracy field, key date, month and year are unknown

See also: Create organisation name details for a seed organisation in Create Healthcare Provider Identifier - Organisation (HPI-O) record for network organisation in Healthcare Identifiers (HI) Service.

Go to Step 7.

6

Enter new information + Read more ...

Select Create New Name to add a new organisation name.

When the blank Organisation Name Details fields display, key the information to be added into the relevant fields. The below fields are mandatory:

  • Organisation Name
  • Organisation Name Usage
  • Name Usage Start Date (defaults to today’s date)
  • Start Date Accuracy Indicator (defaults to Day, month and year are accurate)

To set a new name as the preferred name, select the Preferred Name Indicator tick box.

Where an organisation name needs to be added and no start date has been provided:

  • Leave Name Usage Start Date as the default date
  • Enter Start Date Accuracy as date, month and year are unknown

7

Save information + Read more ...

Select:

  • Add, if added a new organisation name
  • Amend, if amended an existing organisation name

8

Return to previous table + Read more ...

Return to Step 5 of the Initial steps to amend an organisation record table.

Action

Amend certificate information for an organisation

For Tier 3 staff only.

Table 12: follows on from the Initial steps to amend an organisation record table describes how Service Officers amend certificate information details for an organisation registered with the HI Service.

Note: always use 'Act for' functionality if acting on behalf of another person, for example, in response to a telephone request.

Step

Action

1

Select Certificates tab + Read more ...

Select the Certificates tab of the Amend HPI-O Seed/Network Organisation screen.

The Certificates tab displays with the below 4 buttons:

  • Link Existing Certificate - Search for an existing certificate using the individual’s existing certificate Registration Authority (RA) number
  • Initiate New Certificate - Create a new Public Key Infrastructure (PKI) certificate by entering various details (for example, token type or operating system)
  • Manually Link Certificate - Enter certificate details (this option is only available to Tier 3)
  • Request Nash Certificate - Request a National Authentication Service for Health (NASH) PKI certificate for Healthcare Provider Organisations (this option is only available to Tier 3)

2

Check if amendment required + Read more ...

Does certificate information need to be amended?

  • Yes, go to Step 3
  • No, return to Step 5 of Initial steps to amend an organisation record table

3

Amend certificate information + Read more ...

Complete Create or link Public Key Infrastructure (PKI) certificate for Individual or Organisation in Authentication required to access digital health.

Return to Step 5 of Initial steps to amend an organisation record table.
