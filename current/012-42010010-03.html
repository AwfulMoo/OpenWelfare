Incomplete application received by mail, email or fax

Table 1.

Step

Action

1

Issuing a letter to the provider + Read more ...

Select Z2306 Provider Registration - We need more information and complete appropriate fields:

  • Letter must include details advising the outcome of their application
  • Print and send letter to the applicant
    • Where appropriate include a printed copy of the application with the return letter

Note: there may be more than 1 reason the application cannot be processed. It is important where possible, all reasons are communicated to the applicant.

See the Resources page for a link to the letter templates.

2

Finalising the application + Read more ...

In PaNDA:

  • Save letter as a PDF
  • Upload letter to PaNDA
  • Ensure that the Surname, First Name, Provider Number and Case ID fields are completed and correct
  • Add a Processing Note advising the reason the application was returned
  • Select Complete
  • Select Save

Checking registration in the Ahpra window and on the Ahpra website

Table 4.

Step

Action

1

Check PDS + Read more ...

Open PDS:

  • Select Provider
  • Select View
  • Select Detailed Search
  • In the Family name field key the first 3 letters of the last name followed by a %
  • In the Given name field key the first 3 letters of the first name followed by a %
  • Key date of birth in Date of Birth field
  • Select Search
  • A list of health professionals with a matching name displays

Is there a PDS match for the health professional?

  • Yes, go to Step 3
  • No, go to Step 2

2

Check if shell record is present + Read more ...

In PDS:

  • Select Provider
  • Select Create
  • Key surname in Family Name field
  • Key first name in Given Name field
  • Select Search

Is there a PDS match for the health professional?

  • Yes, go to Step 3
  • No, go to Step 4

3

Check health professional's details + Read more ...

In PDS:

  • Select View against the matching health professional's details
  • Select the Unique Identifier hyperlink
  • The Ahpra Information window will display
  • Confirm the details on the application match details in the Ahpra Information
    • If the details do not match, go to Step 4

4

Navigate to the Ahpra website + Read more ...

The Resources page contains a link to the Ahpra website.

5.

Access Registers of Practitioners menu + Read more ...

  • Select Advance Search
  • Key the health professional's surname in the Family name field
  • Key the health professional's first name in the Given name field
  • Select Search

Is a result returned for the health professional?

  • Yes:
    • Select View Details against the appropriate health professional, check name, registration number and any registration conditions
    • Attach a print of the Ahpra registration to the PaNDA record
  • No, return application as rejected
