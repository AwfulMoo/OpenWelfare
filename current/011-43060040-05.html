Insert and delete lines in FAPI

Use the A field on the FAPC screen to insert and delete lines in FAPC.

Note: actions can be keyed for many service lines but they must be of the same type, that is, all ‘D’ or all ‘I’.

Insert and delete line values, descriptions and explanations

Table 3: This table lists the valid values in the A field. Key one of the following values in the A field and press [PF6].

Value

Description

Explanation

I

Insert a service line.

Insert a blank service line under this line and move all subsequent service lines down one line.

D

Delete a service line.

Delete this service line and move all subsequent service lines up one line.

FAPI PF Key functions

FAPI PF (function key) descriptions

Table 5: This table contains a list of FAPI PF (function) keys that can be used on the FAPI suite of screens.

Function key

Function

PF1-HELP

Displays an explanation for a return message or a specific field.

PF3-NEXT ERR

Displays the next error in the claim.

PF4-SHOW TXT

Position cursor on the T column field and press [PF4] will display the item text.

PF6-INS/DEL

Used to perform action(s) specified in the action field. The actions that can be performed are insert a blank line and delete a line.

PF7-BACKWARDS

Used to move backward to previous history lines in potential conflict with items being assessed.

PF8-FORWARDS

Used to move forwards to additional history lines in potential conflict with items being assessed.

PF8-BROWSE

Displays the next patient/provider segment.

PF10 - INS

Add operator text.

PF12-CANCEL

Cancels transaction.
