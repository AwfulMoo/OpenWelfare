Shared Drives

Use the NZ Secure folder to store and manage all embargo files. Go to:

  • G:> TAS > HOB-188-Collins > NZ_Secure > Embargo

Go to mySupport and request shared folder HOBR-NZ-SECURE-RW. In

Request Details, in the following fields:

  • Reason, key ‘Processing NZ embargoes’
  • Site, key ‘HOB-188-Collins’
  • Branch, leave unchanged
  • Folder, use ‘HOBR-NZ-SECURE-RW (NZ)’

Use Local Scripts files to manage, create arrears debt coding work items:

  • G: > Software > Scripts > Local > International > B > BatchJobs > MagicKey > NZ Embargo DOC

Folders

Note: YY is financial year and XX is sequential schedule number.

Use the following folders:

  • Embargo > Incoming:
    • YY Lists (folder for each new financial year)
    • LIST YYXX (sub-folder for each list in financial year (YY is financial year)
    • Originals (sub-folder for storing original schedule (as received) and receipts)
    • Vendors (sub-folder for screenshots of PAPN for International Finance Team to create vendors)
  • Embargo > Outgoing:
    • store GDN reports

File naming conventions

The filename for:

  • FULL LIST YYXX, is a spreadsheet copy of original schedule from NZ for used for adding notes to cases (completed, on hold etc.) and ensuring each schedule is fully actioned
  • Embargo List is a spreadsheet with all details of individual embargo. For example:
    • Voucher number
    • CUS details (CRN, name etc.)
    • Amount received from NZ (AUD)
    • Amount retained by Services Australia (AUD for CUS/PTR)
  • <CUS - CRN>, the name for PAPN screenshot files

Approved text for CUS/PTR debt scenarios

Use the following approved text for CUS/PTR debt scenarios:

  • Customer debt fully recovered:
    • Letter Text for Accounts Payable letter to customer: Please note: You are not required to repay this amount, it has been deducted from your New Zealand pension arrears
  • Partner debt, fully recovered:
    • Letter Text for Accounts Payable letter to partner: Please note: You are not required to repay this amount, it has been deducted from your partner’s New Zealand pension arrears
  • Adjust Repayment (OPAR) screen:
    • Text for each list: FROM NZ ARREARS RECEIVED [date] ON LIST [list number]

Vendor codes and invoice instructions

Vendor codes are used with voucher numbers when paying excess amounts to customers through INFOLINK FI:

  • General Ledger (GL) codes:
    • Age Pension is 70547
    • Disability Support Pension is 70548
  • Cost centre code is 2000691
