Step

Action

1

Check the application + Read more ...

  • Open the application via current workflow procedures

Check Provider Directory System (PDS) + Read more ...

  • Provider
  • View
  • Detailed search

  • Yes, select View for the matching medical practitioner and go to Step 3
  • No, if a provider number has been supplied,
    • enter the first 6 digits in the Search ID field under Basic Search
    • press Search
  • If no results are found, take note and go to Step 3

3

Residency documents supplied + Read more ...

Does PDS have an open 180 or 181 specialty code on a stem or was no stem found at Step 2?

  • Yes,
    • If the application indicates residency status is temporary resident, then a copy of their current temporary resident Visa is required
    • If the Visa details supplied are different to those recorded on PDS, update PDS with the new Visa details
    • If the application indicates residency status is permanent resident, then a copy of their current permanent residency documents are required
    • See Processing a change in residency status for a medical practitioner then go to Step 4

    • Go to Step 4. If contact is unsuccessful, go to Step 29
  • No, go to Step 4

Determine if a class exemption can be applied + Read more ...

Determine if a class exemption can be applied using the order of assessment and referring to the medical practitioner’s stem and documentation received.

6

Specialist and Specialist trainee class exemption, additional specialty codes + Read more ...

If the medical practitioner has any additional specialist or consultant physician specialty codes other than the specialty codes above and a letter of support was received indicating what specialty the medical practitioner is working in, and:

  • the medical practitioner is only working in the specialty listed on the class exemption, the Specialist and Specialist trainee class exemption can be applied, see Table 2
  • the medical practitioner is working in an additional specialty that is not listed on the class exemption, the application needs to be sent to the Department of Health and Aged Care requesting a Section 19AB(3) exemption, see Applying for and processing a section 19AB(3) exemption with the Department of Health and Aged Care

If a letter of support has not been received, attempt to contact the medical practitioner to determine which specialty or specialties they are applying under. Was contact successful?

  • Yes, perform security check. See the Security check when a health professional or their staff call regarding provider/prescriber number or Medicare billing eligibility table. Process for Provider Registration telephony calls from health professionals in Medicare then ask the medical practitioner which specialty they will be billing under.
    • Medical practitioner confirms they are only working in the specialty covered by a class exemption. Make a note of the conversation including date and time of call into the PaNDA comments field. The Specialist and Specialist trainee class exemption can be applied, see Table 2
    • Medical practitioner confirms they are not working in the specialty covered by a class exemption or they are working in both capacities. Make a note of the conversation including date and time of call into the PaNDA comments field. Advise the medical practitioner the application needs to be returned to them. The application will need to be resubmitted with a letter of support clearly confirming the speciality or specialities they will be working under, return the application as incomplete, go to Step 29
  • No, return the application as incomplete, go to Step 29

Note: if none of the above apply, the Specialist and Specialist trainee class exemption can be applied, see Table 2

7

Northern Territory class exemption + Read more ...

Is the practice address indicated on the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) for a practice address located in the Northern Territory:

  • Yes:
    • If the medical practitioner is on an approved 3GA placement go to Step 8
    • If the medical practitioner is a temporary resident, the Northern Territory class exemption can be applied, see Table 2
    • If the medical practitioner is a permanent resident or Australian citizen and satisfies Section 19AA of the Health Insurance Act 1973 with a post graduate qualification (specialty code 001-099, 130, 132, 530, 540, 532 or 801-899 on the medical practitioner’s stem), the Northern Territory class exemption can be applied, see Table 2
    • If the medical practitioner is a permanent resident or Australian citizen and does not satisfy Section 19AA of the Health Insurance Act 1973 with a post graduate qualification (specialty code 001-099, 130, 530, 540 132, 532 or 801-899 on the medical practitioner’s stem), go to Step 8
  • No, go to Step 8

8

Aboriginal and Torres Strait Island Primary Health Care Services class exemption + Read more ...

Is the medical practitioner any of the following and is applying for a practice address listed on an Aboriginal Community Controlled Health Service Section 19(2) Direction?

  • A general practitioner (130, 132, 530, 540 or 532 specialty code on the medical practitioner’s stem),
  • A specialist or consultant physician (specialty code within the range 001- 099 or 801-899)
  • A non-specialist (181 or 180 specialty code on the medical practitioner’s stem), or
  • not registered on an approved 3GA placement

  • Yes, an Aboriginal and Torres Strait Island Primary Health Care Services class exemption can be applied, see Table 2
  • No, go to Step 9

Note: the Aboriginal and Torres Strait Islander Primary Health Care Service class exemption cannot be applied to satellite locations. The location must be listed on the 19(2) or 19(5) direction before the class exemption can be applied.

  • Select ‘GPs’
  • Key the full practice address in the Find Address field
  • Select ‘Search Location’
  • Select practice address from the drop down options
  • The search result will display. If the practice address is:
    • DPA Yes for GPs, the DPA(MMM3-4 and exceptional circumstances) and DWS class exemption can be applied. See Table 2
    • DPA No for GPs, continue below.
  • Under ‘Classifications’, tick Modified Monash Model
  • Select ‘2019’
  • Key the full practice address in the Find Address field.
  • Select “Search Location”
  • Results will be displayed. Take a snip of the results and upload to PaNDA as a PDF.If the practice address is:
    • MM3-4, the DPA(MMM3-4 and exceptional circumstances) and DWS class exemption can be applied. Take a snip of the result and upload to PaNDA as a PDF, then see Table 2
    • MM1-2, continue below.
  • Take note of the ‘Catchment’ listed under the ‘Distribution Priority Area’ results
  • Open the Distribution Working Group page on the Department of Health and Aged Care website
  • Search for the catchment within the Exceptional circumstances reviews and outcomes.
  • Results will be displayed. Take a snip of the results and upload to PaNDA as a PDF. If the catchment is:

After-Hours specialist class exemption + Read more ...

Check if the medical practitioner is recognised as:

  • A general practitioner (130, 132, 530, 540 or 532 on the medical practitioner’s stem)
  • Anaesthesia – 051
  • Cardiology – 084
  • Diagnostic radiology - 047
  • General surgery - 032
  • Medical oncology 097
  • Obstetrics and gynaecology – 053
  • Ophthalmology – 054
  • Psychiatry – 099

If the medical practitioner is not recognised in one of these specialties, go to Step 24

Check if a letter of support stating the medical practitioner is working in an after-hours capacity been received with the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)

Note: after hours is defined as anytime on a Saturday, Sunday or public holiday, or before 8.00am or after 6.00pm on any day other than a Saturday, Sunday or public holiday.

Has the medical practitioner met this criteria?

  • Yes, the After-Hours specialist class exemption can be applied. See Table 2
  • No, go to Step 24

24

After-Hours temporary resident class exemption + Read more ...

If the medical practitioner is a permanent resident or Australian citizen, specialist or consultant physician, or has not included a letter of support stating they will be working in an after-hours capacity, go to Step 25.

If the medical practitioner is a temporary resident, check the following:

  • The medical practitioner is a temporary resident
  • The medical practitioner is not recognised as a general practitioner (130, 132, 530, 540 or 532 specialty code on stem) or a specialist or consultant physician (001-099 or 801-899 specialty code on stem)
  • The medical practitioner has supplied a letter of support stating they will be working in an after-hours capacity along with the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)

Note: after hours is defined as anytime on a Saturday, Sunday or public holiday, or before 8.00am or after 6.00pm on any day other than a Saturday, Sunday or public holiday.

Has the medical practitioner met the above criteria?

  • Yes, the After-Hours temporary resident class exemption can be applied, go to Table 2
  • No, go to Step 25

World Health Organisation (WHO) class exemption + Read more ...

Check if the medical practitioner:

  • was a permanent resident or Australian citizen prior to 1 January 1997
  • is recognised as a general practitioner (130, 132, 530, 540 or 532 on stem) or a specialist or consultant physician (001-099 or 801-899 on stem), and
  • had commenced study outside Australia prior to 1 January 1997 at a medical school listed in the World Health Organisation’s World Directory of Medical Schools (6th edition)*

*Including updates contained in the World Health Organisation’s World Forum (up to and including Volume 17 Number 3, 1996). See Resources.

Has the medical practitioner met the above criteria?

  • Yes, the World Health Organisation (WHO) class exemption can be applied. See Table 2
  • No, go to Step 28

28

Class exemption cannot be applied + Read more ...

If the medical practitioner is not eligible for a class exemption check that the application includes

  • A complete Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019)
  • A copy of a current Visa for a temporary resident medical practitioner or
  • Permanent resident details recorded on PDS or a copy of a permanent resident visa for a permanent resident practitioner
  • A letter of support from the employer, stating medical practitioners name, the term and location(s) of employment and why a need a provider number with access to Medicare benefits (An application that includes a 3GA placement do not require a letter of support)
  • Have you confirmed AHPRA registration for any conditions or limitations

If all documents have been received, go to Applying for and processing a Section 19AB(3) exemption with the Department of Health and Aged Care. Procedure ends here.

If documents are missing, make 1 attempt to call the medical practitioner to get the missing documents. If contact is unsuccessful, go to Step 29

Process a provider number and class exemption

Table 2

Step

Action

1

Update provider stem + Read more ...

Update personal contact details if required as per the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019).

Is a 519 specialty code present on the stem?

  • Yes, go to Step 2
  • No,
    • under Specialty tab, select Add Specialty Code, key 519 in Specialty Code Description, key the date of first registration in Start Date
    • If the medical practitioner is an Australian permanent resident or Australian citizen, key the 519 specialty code end date as per the 500 specialty code in the End Date
    • Select Add New, select Cancel. Go to Step 2

2

Class exemption and 3GA placement pre 1 January 2019 + Read more ...

Is the class exemption issued in conjunction with a Pre 1 January 2019 3GA placement?

  • Yes,
    • If the corresponding 3GA specialty code is present on the stem, go to Step 3
    • If the corresponding 3GA specialty code is not present on the stem, under Specialty tab, select Add Specialty Code, key the corresponding 3GA specialty code in Specialty Code Description, key the date the exemption was granted from in Start Date, no End Date is required to be keyed. Select Add New, select Cancel. Select Save. Go to Step 3
  • No, go to Step 3

3

Check Mental Health Skills Training and Pregnancy Support Counselling + Read more ...

Does the application include any of the following placements?

  • Australian General Practice Training (AGPT) – RACGP
  • Remote Training – RACGP
  • Practice Experience Program - RACGP
  • Australian General Practice Training (AGPT) – ACRRM
  • Remote Training – ACRRM
  • Independent Pathway – ACRRM
  • Australian General Practice Training (AGPT) placement or
  • Remote Vocational Training Scheme (RVTS) placements
    • Yes, go to Step 4
    • No, go to Step 5

4

Check specialty codes + Read more ...

Are specialty codes 662, 663, 664 or 665 open on the stem?

  • Yes,
    • If a 662 code is recorded, add a 647 code (if not already recorded)
    • If a 663 code is recorded, add a 648 code (if not already recorded)
    • If a 664 code is recorded, add a 649 code (if not already recorded)
    • If a 665 code is recorded, add a 650 code (if not already recorded)
    • Key Start Date – start date of 3GA placement
    • Select Add new
    • Select Cancel
    • Go to Step 5
  • No, go to Step 5

5

Check provider location + Read more ...

Select the medical practitioner’s stem number (orange hyperlink). Existing locations will appear.

  • Check for an existing provider number for the requested practice/hospital name or street address. E.g. Prince Charles Hospital Rode Rd and Prince Charles Hospital 627 Rode Rd are considered a match
  • If no provider number exists for the requested practice/hospital or the requested street address continue assessment, go to Step 13
  • If a match for practice/hospital name or street address is found see Issuing a second provider number or contact a PSO for assistance.
  • If a match exists go to Step 6

Note: where the medical practitioner is a permanent resident or Australian citizen, make a note of the last day the medical practitioner is subject to Section 19AB.

6

Update existing provider number + Read more ...

Select Amend against the provider number to be updated.

Address tab:

  • Update address details where additional information has been supplied on the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) or Approved 3GA placement document

Contact tab:

  • Update telephone, mobile, fax and email fields as per the application or 3GA placement
  • Start Date – no amendment is required
