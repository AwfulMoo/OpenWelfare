Registration and Change of Details (TAS020) form received

Step

Action

Business notifies of change to their details + Read more ...

Check the TAS020 form, has an Authorised Signatory completed the declaration in Section 20?

  • Yes, go to Step 2
  • No,
    • upload the TAS020 to the Claimant Document tab
    • enter detail in the claimant comments

2

Add the activity to the TFES tool + Read more ...

  • Select Create New in TFES Tool
  • Enter the:
    • date received
    • date uploaded, and
    • claimant code
  • In Claim ID field enter ‘Update Details’

Allocate activity to in TFES Tool:

  • Selecting Allocate
  • Select the activity from the unassigned records:
    • select service officer from dropdown menu
    • select Save Allocations
  • Go into work allocation, update Work Type to Updated Claimant Details
  • Update the comments section with the details of the updates made

If claimant record goes to ‘Pending Approval’ after updates are made in myHSI, change the status in the TFES tool to ‘Awaiting QA officer’.

3

Compare details + Read more ...

Are the details on the TAS020 the same as the details on the claimant record?

  • Yes,
    • upload the TAS020 to claimant documents tab
    • in the claimant comments, document ‘TAS020 received, no updates required’
    • procedure ends here
  • No, go to Table 2

Note: if a section of the TAS020 is blank, it is assumed the detail in that section has not changed.

  • No, go to Section 17

20

Declaration + Read more ...

This Section must be completed to certify changes to the record.

Has Section 20 been completed by an authorised person? (see Acceptable Signatories)

  • Yes, complete updates, upload the TAS020 to the claimant documents tab
  • No, TAS020 cannot be actioned. Do no update business details on myHSI.
    • Contact current nominated claimant signatory, in myHSI.
    • Ask them to complete a new TAS020
    • Document the contact on the claimant comments
    • upload copy of all forms received

Claimant online update received

Table 3: This table details the process of approving a claimant record when the status has change from ‘eligible’ because they have updated their details online.

Step

Action

1

Locating work item + Read more ...

If a claimant updates their details through the Third Party Portal, an activity may appear on the Activity List TPP Bank/Signatory Update.

2

Contact the claimant + Read more ...

Tell the claimant:

  • the online update they submitted has been received
  • a TAS020 needs to be certified by an authorised person as listed in section 20
  • to upload the TAS020 to their claimant documents tab using online services

3

TAS020 received + Read more ...

TFES admin officer locate TAS020 by:

  • searching Claimant Document Upload in the activity list, or
  • going directly to the Claimant Documents tab

TFES admin officer check the details entered online by the claim against the TAS020.

Are the details correct?

  • Yes, record the details in the Claimant Comments
  • No:
    • put details in Claimant Comments
    • contact claimant to verify
    • ask for a new TAS020 if required

If the claimant posts or emails the TAS020 without uploading it, scan and upload the TAS020 to the Claimant Documents Tab.

4

Create activity in TFES Assessor tool + Read more ...

Open TFES Assessor Tool:

  • Select Create new
  • Key the Date received and Date uploaded
  • Key ‘Claimant code’ in Claimant Code field
  • Key ‘Agent code’ in Agent Code field (if applicable)
  • In Claim ID field key ‘TAS020 Updates’
  • Select Save
  • Select Allocate
  • In Allocate to, select assessor name from the dropdown
  • Locate the activity in unassigned records, select Allocate box
  • Select Save Allocations
  • Select activity and change Work Type to ‘Update Claimant Details’
  • In Comments/Details field, document the changes made to claimant record, noting the claimant made the changes online
  • Update the activity status, select Awaiting QA Officer from the dropdown menu, select Save

5

QA Officer action + Read more ...

The QA officer verifies the details, see Tasmanian Freight Equalisation Scheme (TFES) quality checking.

Are the details correct?

  • Yes, the QA officer will approve the updated details in myHSI. The record will then be eligible
  • No, the QA Officer will not approve the activity in myHSI and will change the status to ‘QA back to Assessor’ in TFES tool the Service Officer to follow up
