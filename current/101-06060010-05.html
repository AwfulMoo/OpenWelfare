Pensioner Concession Card (PCC) name conventions

Name conventions and examples to be displayed on PCCs

Table 1: This table contains the set name rules for the PCC with examples of how the name is displayed on the customer's record and how it is displayed on the PCC. Only legal names will be on the cards.

Name rules for PCC

Example from mainframe

Example from card print out

Characters are in UPPER case

JOHN ALEX CITIZEN

JOHN A CITIZEN

Maximum of 32 characters are printed

ALEXANDER BARRINGTON CITIZEN

ALEXANDER B CITIZEN

Middle name is always initial only

JAMES PETER CITIZEN

JAMES P CITIZEN

Spaces and hyphens are considered one character

MARY JANE ELIZABETH CITIZEN

MARY JANE E CITIZEN

Where a name is greater than 32 characters long and regardless of whether hyphens and/or spaces are used in the name, only 32 characters will be printed based on the following rules:

Truncate first name then family name when 32 characters are exceeded.

SARINA-CHERYLLYNNE CITIZEN-CITIZEN

MOHAMMED KHALRAZ RASHEED CITIZEN

SARINA-CHERYLL CITIZEN-CITIZEN

MOHAMMED KHALRAZ R CITIZEN
