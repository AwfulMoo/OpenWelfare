Check access to online account + Read more ...

Check the customer is registered with an active Centrelink online account.

If an active account does not exist, the customer must confirm their identity at the local service centre to register for a Centrelink online account.

Starting an online claim + Read more ...

Electronic messaging + Read more ...

Claim progress + Read more ...

The Claim navigation page shows the sections of the claim the customer must complete. Sections must be completed in order before the next can be started.

Customers select Start to access sections.

Each section has modules with questions. On each page, the customer can select Previous to go back to the previous question.

Customers can read the Oliver Digital Assistant text for help with the questions on the current page.

After completing each section, the customer returns to the Claim navigation page to select the next section. There is an Edit option to correct answers in previous sections. To return to the Claim navigation page select the arrow under the last dot on the side navigation and Continue.

Customers can use the dots on the side navigation of each page to go between pages in the online claim. If a question is answered incorrectly, a red cross will appear in the dots on the right of the page. Select the red cross to go to the page with the error.

A claim does not need to be completed in one session. Customers can leave the Claim navigation page at any time. The responses will save to be continued later.

Customers must answer all compulsory questions before submitting the claim.

Customers will be advised if they are have a reduced question set. They will see a summary of the following sections they can update as necessary:

  • Your personal details
  • Your circumstances

3

Determining a new FST start date (backdating) + Read more ...

To cover medical, pharmaceutical or other related expenses already incurred, the start date of an FST may be backdated to a particular date of medical service or treatment given to the foster child.

The customer must provide evidence of the medical service or treatment to support the request for backdating the start date with the new claim.

Customers will need to keep records of expenses to claim reimbursement from Medicare after the FST is granted. The customer can take the receipt/s and concession card to Medicare to request a refund of medical expenses.

Claims may only be backdated for a period in which eligibility is satisfied. As the card is for the child, the claim may cover a period during which the child lived with 2 or more carers.

If eligibility is broken during the period, for example, the child returned to live with their parent, the single claim covers all periods but each period needs to be processed separately. Suppress the letters from the Assessment Results (AR) screen.

Note: to backdate the start date for an FST, the Date of Receipt (DOR) for the claim activity must equal the date of medical service or treatment and must not be more than 40 weeks earlier than today's date.

The start date field on the LIC Start Date (LSD) screen defaults to the DOR. This date may be changed during a new claim activity and can be no earlier than 40 weeks prior to today's date.

If backdating to an earlier date, refer to the Level 2 Policy Help Desk - Concessions via the Online query form to request a manual override when all evidence has been gathered.

When completing the online query form, ensure the query is for 'Additional Products and Services' and the category is 'Concessions'.
