Definition of terms on the QMA Sampling Override Tool

Table 1

Term

Details

QMA VARIANT

Identifier against which all QMA sampling parameters are configured.
For a list of the QMA variants, see Table 2.

DESCRIPTION

Description of work item type sampled under the QMA Variant

USER ID

Staff Logon ID

MANUAL OVERRIDE RATE

If a manual override is activated then the auto sampling rules will be bypassed and the sampling will happen at this rate until the auto sampling has been recommenced.

Options are limited to:

  • 100 – to activate the sampling override
  • 0 – to remove the override and recommence auto sampling

OVERRIDE REASON CODE

If a manual override rate has been selected, an override reason must be recorded.

Options are limited to:

  • Performance support
    The staff member is subject to a performance improvement plan
  • History override
    The staff member has restarted the work after a significant period of other tasks or long term leave
  • Recommence auto sampling
    The sampling override is no longer needed. The staff member will return to the auto sampling model
