This page contains information about deactivating a Contracted Service Provider (CSP) organisation record in Healthcare Identifiers (HI) Service.

Deactivate CSP organisation record

CSP organisation record deactivation

This table describes how to deactivate a CSP organisation that has a status of 'Active'. Note: if acting on behalf of another person known to the HI Service (for example, a Responsible Officer, Organisation Maintenance Officer or CSP Officer), a search for the individual in the eHealth Program (EHP) must be conducted prior to searching for the organisation.

Step

Action

1

Check if Service Officer is acting on behalf of another person + Read more ...

Is the Service Officer working on the behalf of another person known to the HI Service in response to a telephone request?

  • Yes, complete 'Act for' an EHP individual (procedure) then go to Step 2
  • No, go to Step 2

2

Search for CSP organisation + Read more ...

From the eHealth program (EHP), search for the CSP organisation being deactivated.

From the Search for Organisation screen, select Amend in the Action column of the relevant record.

The Amend CSP organisation screen is displayed.

3

Navigate to organisation details + Read more ...

From the Amend CSP organisation screen, select the Organisation Details tab.

The Organisation Details tab displays the selected record.

4

Deactivate record + Read more ...

Select the Deactivate button.

Result: a warning message is displayed, advising that this action will end-date all links to HPI-O records.

  • select OK
  • enter a comment in the ‘Reason’ text box and select Validate

The Organisation Activity Status changes to ‘Deactivated’ and the Status Date field automatically updates with the current date and time stamp.

5

Submit and check validation + Read more ...

Select Submit.

Are there validation errors?

  • Yes, correct errors as prompted then go to Step 6
  • No, go to Step 6

6

Correct errors if required and finalise + Read more ...

Review the details on the Confirm Amend CSP organisation screen. If the details are:

  • Incorrect, select Edit to return to the Amend CSP organisation screen. Return to Step 3
  • Correct, select Save. A confirmation message displays
