Assessing YA and ABSTUDY PIT review letters

Table 1

Step

Action

1

YA/ABSTUDY PIT Review + Read more ...

PIT reviews are sent to dependent Youth Allowance (YA) and ABSTUDY customers to confirm their family circumstances and parent(s)/guardian(s) income. This information is used to determine the customer’s payability and rate of payment.

There are 4 types of PIT reviews:

  • Annual PIT
  • Employment Income NIL rate (EIR)
  • Parental Income Test (PIR)
  • Change in Circumstance

If the customer, or parent(s)/guardian(s) contacts to:

  • request a reissue of the YA and ABSTUDY PIT review , go to Step 2
  • request an extension of YAL/ABSTUDY PIT review, go to Step 5
  • complete the PIT review verbally, see Step 1 in Table 2
  • enquire about who are the parents or what is applied under the PIT, see Parental Income Test (PIT)

Where a PIT review is allocated via Workload Manager, see Step 1 in Table 2.

2

When is a PIT review to be reissued? + Read more ...

Prior to issuing a replacement, check if the customer's Household and Income Assets (NHI) screen has already been updated with verified income.

The Parental Income Test (PIT) review only needs to be reissued where:

  • the original PIT review is not received
  • it has been misplaced
  • the parent is unable to lodge the PIT review online, by phone or at a service centre

Is a replacement PIT review needed?

  • Yes, and the child is in receipt of:
    • Youth Allowance (YA), go to step 3
    • ABSTUDY, go to step 4
  • No, NHI has been updated with verified income. Procedure ends here

3

Reissue Youth Allowance (YA) PIT letter + Read more ...

Service Officers must reissue Youth Allowance PIT letters in Customer First:

  • Next: field, HS locate the line with the relevant PIT review letter
  • Sel Hs column, select Print Central or Print Local
  • select Continue
  • Final: field, enter ‘Y’
  • Printer: field, enter:
    • ‘CENTRAL’ to print through to the mailing house, or
    • Your local printer ID to print locally
  • Continue. The PIT Review will be issued centrally through the mailing house or locally
  • Discuss a reasonable timeframe for them to provide the information
  • create a review via the Activity Registration (AYR) screen using the following attributes:
    • Service Reason: ‘YAL’
    • Source: ‘SY012’
    • Activity Type: 'MFU'
    • Date of Receipt: Applicable date
    • Review Reason: 'YAR'
    • Resubmit Date: Date determined suitable
    • Keywords: 'YARyyyy. Staff must capture ‘yyyy” as the appropriate year, for example, in 2021 the keyword is YAR2021
    • Proceed with Activity?: ‘N’
  • Notes: PIT Review Extended until DD MMM YYYY. Suspend Payment with Reason YAR if not supplied by new due date
  • Select Continue

If the NSSREM advice has been archived, reissue the letter via an AYR activity, (which will provide a new One Time Access Code and start the review timeline again), create a review on the Activity Registration (AYR) screen by coding the following fields:

  • Service Reason: 'YAL'
  • Source: ‘SY012’
  • Activity Type: 'YAR'
  • Date of Receipt: Applicable date
  • Review Reason: 'YAR'
  • Action: 'FMI'
  • Keywords: 'YARyyyy' staff must capture ‘yyyy' as the appropriate year, for example, in 2021 the keyword is YAR2021
  • Proceed with Activity?: ‘Y’
  • select Continue

Note: staff must make sure they record all information relating to the PIT review in a DOC.

The Resources page contains a link to the SY012 (A) form if a parent says they cannot complete the PIT Review online. See Completing Parent(s)/Guardian(s) details for the Base Tax Year and Current Tax Year (MOD JY) for more information.

Procedure ends here.

4

Reissue ABSTUDY PIT letter + Read more ...

The PIT review letter may be reissued via an AYR activity or via the History Summary (HS) screen.

If the letter has not been archived, the letter can be reissued to the customer and/or their correspondence nominee.

Service Officers must reissue ABSTUDY PIT reviews in Customer First or Customer Record.

To reissue the form via HS screen:

  • Next: field, HS locate the line with the relevant PIT review letter
  • Sel Hs column select Print Central or Print Local
  • select Continue
  • Final: field, enter 'Y'
  • Printer field, enter:
    • CENTRAL to print through the mailing house, or
    • LOCAL to print through a local Printer ID
  • select Continue
  • discuss a reasonable timeframe for them to provide the information. If the information is not returned, the customers payment will revert to a zero rate

Record details in a DOC.

If the PIT review letter has been archived, retrieve the letter from the Archiving and Culling Engine (ACE).

Procedure ends here.

5

Extension to return the PIT review + Read more ...

If the customer or their parent(s)/guardian(s) contact requesting an extension to provide the information, Service Officers can extend the date if there is sufficient information recorded on the NHI screen. For example:

  • if a reasonable PIT estimate has been recorded, or
  • an extension to the Annual PIT review has been requested and can be provided prior to 1 January

If contact is made by parent(s)/guardian(s), Service Officers should consider obtaining and coding a parental income estimate to prevent cancellation. See Assessing and coding parental income for ABSTUDY, Assistance for Isolated Children (AIC) Scheme and Youth Allowance (YA).

If a PIT review letter/reminder is reissued via the AYR screen, the due date will automatically appear on the Future Activity List (FAL) screen.

If an extension is needed:

  • go to the Future Activity List (FAL) screen
  • highlight the YAL Dependent Review or ABY PIT Review activity
  • select To VIEW a future activity via the FAY screen
  • select Continue
  • on the Future Activity Details (FAY) screen, open the Actions section and go to the Action Date: field
  • key the new date agreed on for the return of the PIT information in the Date: field
  • select Continue

If payments are:

  • suspended go to Step 6
  • current, record details on a DOC. Procedure ends here

6

Restoration + Read more ...

Customer's payments are suspended and the parent(s)/guardian(s) have requested an extension. Go to the Benefit Action (BA) screen and complete the following fields:

  • ServRsn: select the appropriate payment type, YAL - Youth Allowance or ABY - ABSTUDY
  • Action: RES - Restore
  • Effect Date: the date of suspension
  • select Continue
  • Finalise the activity via Activity Results (AR)
  • record details in a DOC, see Getting it Right (GIR) Minimum Standards

Procedure ends here.

Assessing Youth Allowance (YA) and ABSTUDY Parental Income Test (PIT) reviews

Table 2

Step

Action

1

Lodgement of Parental Income Review + Read more ...

In most cases, where the parental income information has been lodged using the Advise Parental Income (API) online service, PIT reviews will be completed automatically.

Where the system has not automatically completed a submitted PIT review, or more information is required, Workload Manager (WLM) will allocate the review for processing by Service Officers in Process Direct.

Where a PIT review has been allocated in WLM, go to Step 2

Where a Request for Information (RFI) has fallen due, and the requested documents have:

  • been provided, go to Step 3
  • not been provided, and the customer is on YA, suspend the payment:
    • go to the Benefit Action (BA) screen
    • Service Reason: field, appropriate payment type
    • Action: field, SUS – Suspension
    • Reason: field, YAR – Failed to return PIT review form
    • Effect Date: field, Due date of RFI
    • Source: field, INT - Internal
    • Receipt Date: field, system date
    • select Submit Change
    • Finalise the activity via the Assessment Results (AR) screen, go to Step 8
    • Set the CoC status as Completed via Interaction History
  • not been provided, and the customer is on ABSTUDY:
    • do not record any details on the NHI
    • Finalise the review
    • Set the CoC status as Completed via Interaction History
    • Ensure the ABSTUDY payment has reverted to a zero rate, go to Step 8

2

Advise Parental Income (API) Online Service + Read more ...

Where parent(s)/guardian(s) have completed the PIT review using the API online service, a pre assessment check must be completed.

Parent(s)/guardian(s) can submit multiple PIT reviews using their OTAC. Service Officers may be allocated more than one PIT review. If this occurs, Service Officers must check all PIT reviews and compare the details.

In Process Direct:

  • On the Transaction Summary (TS) screen, select the Claim Overview to review the information provided
  • Check the Transaction Notes as they may provide details on why the update was not automatically applied to the record
  • Check if any supporting documentation has been uploaded
  • Check the FAO Income Previous Year (FIPY) screen in the parent(s)/guardian(s) record for any confirmed data

Is more information required to assess the PIT review?

  • Yes, go to Step 4
  • No, the PIT review can be processed, go to Step 5

3

Verbal, SY012 or MOD JY PIT review + Read more ...

Where parent(s)/guardian(s) contact to complete their PIT review verbally, an SY012/MODJY or supporting documents have been uploaded, Service Officers must complete a pre assessment check.

If a MOD JY or SY012 has been provided, Service officers must ensure each section of the form has been completed.

In Customer First:

  • go to the workspace in Customer First, and Select Document Tools under Quick Links or New Document Tools – Digitisation
  • select the SY012 or MODJY, to review the information provided
  • check if any supporting documentation has been uploaded
  • check the FIPY screen in the parent(s)/guardian(s) record for any confirmed data

Where parent(s)/guardian(s) contact to provide their income details verbally, Service Officers must still complete the above pre assessment checks.

Is more information required, or are there multiple PIT reviews with different details?

  • Yes, go to Step 4
  • No, the PIT review can be processed, go to Step 6

4

Further information if needed + Read more ...

If more information is needed, attempt to contact the parent/guardian verbally to get the information.

Where the parent(s)/guardian(s) has not answered a question(s) properly, get the answers verbally from the parent(s)/guardian(s)

If the contact is successful:

  • confirm with the parent(s)/guardian(s) that:
    • the details they have provided are correct, and
    • they acknowledge that giving false or misleading is a serious offence
  • If the parent/guardian declares the information provided is true and correct, record details in a Fast Note, using the Auto Text function:
    • Category 1:
    • Students, ABSTUDY & Apprentices for YA or ABSTUDY (STU/APT)
    • Jobseekers for YA (OTH/JSK)
    • Category 2: claims
    • Template: Verbal MOD JY
    • Select Confirm
    • Edit auto text with the details provided by the parent(s)/guardian(s)
    • Select Continue to complete the DOC
  • Process the PIT Review:
    • for verbal, SY012/MOD JY PIT reviews, go to Step 6
    • for Online PIT Reviews, go to Step 5

If the contact is unsuccessful, or more documents are required:

  • issue a Request for Information
  • ensure the PIT review is placed on hold until the documents are provided, or until the due date
  • go to Step 8

For information on what documents are accepted as verification, see Documents accepted as evidence of parental income.

5

Processing Annual PIT Review in Process Direct + Read more ...

Where a PIT review has been allocated via WLM, Service Officers must process this work item in Process Direct.

To assess the PIT review in Process Direct:

  • Check the transaction notes as it may provide details on why the update was not automatically applied to the record
  • Review parent responses in the transaction overview
  • Update the PIT Parent Selection (NPPS) screen with any changes to the parent(s)/guardian(s) provided

Check the following fields on the NHI screen:

  • Superkey: NHI
  • Event Date: field, for:
    • Annual PIT reviews, ensure the event date reflects 1 January of the relevant year (Where an increase has applied, the system will automatically apply 1 October)
    • Where an EIR/PIR/CoC has been issued, an NHI entry will already be recorded with the appropriate date
  • Financial Year: the Base Tax year of the calendar year for which the review was issued
  • Estimate Indicator: field
    • No, where actual income has been advised
    • Yes, where estimated income has been advised
  • Review Date: the date provided by the parent(s)/guardian(s). If the parent(s)/guardian(s) have not supplied the date the evidence is expected, get the date from the parent. If the parent cannot be contacted, code the review date 28 days in the future
  • Parent CRN: select the relevant PIT linked parent from the options provided. Ensure the appropriate parent(s)/guardian(s) are PIT linked via the PIT Parent Selection (NPPS) screen
  • Update the following income fields, with the information provided within the PIT review:
    • Taxable Income
    • Overseas Income
    • Maintenance Paid
    • Net Business/Investment Loss
    • Reportable superannuation contributions
    • Exempt reportable fringe benefits
    • Other reportable fringe benefits
  • Finalise the activity, select Assess, the Entitlements (ELD) screen will display
  • If any errors present, see the Digital Assistant Roxy before escalating to ICT
  • If an overpayment occurs, see Raising Parental Income Test (PIT) debts
  • If a restoration is needed, go to Step 7
  • If no restoration is needed, select Finalise, a pop-up window will display where a DOC is required, go to Step 8

Note: where appropriate, update all dependent sibling records.

PIT Review CoC transaction will automatically be set as 'Completed'.

6

Processing PIT review in Customer First or Customer Record + Read more ...

PIT reviews must be completed in Customer First or Customer Record where the parent(s)/guardian(s) are supplying their parental income:

  • verbally
  • via a Youth Allowance Parental Income Assessment (SY012), or
  • via a Parents/Guardians details for the Base Tax Year and Current Tax Year for Dependent Youth Allowance or ABSTUDY customers (MOD JY) form

In Customer First or Customer Record:

  • select the PIT review activity from the Activity List (AL) or Future Activity List (FAL) screen
  • Update the PIT Parent Selection (NPPS) screen, where needed
  • Next: field, key 'NHI'
  • Event Date field, for:
    • Annual PIT reviews, ensure the event date reflects 1 January of the relevant year (Where an increase has applied, the system will automatically apply 1 October)
    • Where an EIR/PIR/CoC has been issued, an NHI entry will already be recorded with the appropriate date
  • Financial Year: field, the Base Tax year of the calendar year for which the review was issued
  • Estimate Indicator: field,
    • No, where actual income has been advised
    • Yes, where estimated income has been advised
  • Review Date: field, the date provided by the parent(s)/guardian(s). If the parent(s)/guardian(s) have not supplied the date the evidence is expected, get the date from the parent. If the parent cannot be contacted, code the review date 28 days in the future
  • Update the following income fields, with the information provided within the PIT review:
    • Taxable Income
    • Overseas Income
    • Maintenance Paid
    • Net Business/Investment Loss
    • Exempt reportable fringe benefits
    • Other reportable fringe benefits
  • If the payment is suspended, go to Step 7
  • If the payment is current, finalise the activity via the Assessment Result (AR) screen. If an overpayment occurs, see Raising Parental Income Test (PIT) debts, go to Step 8

Note: Service Officers must ensure all dependent sibling records are updated with relevant information where appropriate.

7

Restoration needed + Read more ...

Within the PIT review, go to the Benefit Action (BA) screen and complete the following fields:

  • ServRsn: select the appropriate payment type (YAL - Youth Allowance or ABY - ABSTUDY)
  • Action: RES - Restore
  • Effect Date: the date of suspension/cancellation
  • in Customer First:
    • select Continue
    • Finalise the activity via the Assessment Results (AR) screen. Go to Step 8
  • in Process Direct:
    • select Assess
    • the outcome will be recorded via the Finalisation Notes. Go to Step 8

Payment cannot be restored if YA has been Cancelled - Failed to Verify Parental Income (CAN-PIE). A CAN-PIE will occur 13 weeks after the payment was suspended for the same reason (SUS-PIE). The customer must reclaim.

8

Record Details in a DOC + Read more ...

If PIT information has been obtained verbally, Service Officers must record details in a Fast Note, using the Auto Text function:

  • Category 1:
  • Students, ABSTUDY & Apprentices for YA or ABSTUDY (STU/APT)
  • Jobseekers for YA (OTH/JSK)
  • Category 2: Claims
  • Template: Verbal MOD JY
  • Select Confirm
  • Edit auto text with the details provided by the parent(s)/guardian(s)
  • Select Continue to complete the DOC

Otherwise, record details in a DOC including any of the following information where appropriate:

  • additional information or documents requested
  • rate reduction/increase or overpayment
  • suspension
  • restoration
  • cancellation if parental income too high (CAN/PIH)

It is important to record any additional information as per the GIR DOC Minimum Standards.

Where Service Officers have the relevant skills, they must select all relevant activities/scans to complete on the customer's record. Once all work is completed, make sure relevant work items are completed.
