Step

Action

1

Verifying a student’s Residential Costs Option (RCO) + Read more ...

If the student:

  • wishes to accept an Indigenous Commonwealth Accommodation Scholarship (CAS), the Indigenous Student Success Programme (ISSP) for relocation and/or accommodation, Department of Veteran Affairs (DVA) Relocation Scholarship or other equivalent Commonwealth scholarship instead of RCO, go to Step 2
  • is discontinuing RCO for a vacation period (short or long vacation) and wishes to access RCO upon return to the residential college in the following semester, go to Step 3
  • no longer wishes to receive RCO, go to Step 4
  • is no longer eligible for RCO, go to Step 5
  • wishes to commence receiving RCO or requesting their residential fees be paid, go to Step 8
  • is leaving the residential college early and needs to pay exit fees, see ABSTUDY Additional Assistance

2

Accepting an Indigenous Commonwealth Accommodation Scholarship (CAS) or the Indigenous Student Success Programme (ISSP) for relocation and/or accommodation + Read more ...

Students receiving RCO or the ISSP for relocation and/or accommodation and elects to accept an Indigenous CAS in the same year cannot receive RCO for the same scholarship period. This means RCO must be end dated from the commencement of the Indigenous CAS or the ISSP in the year and may result in an overpaid amount of RCO.

Note: the RCO is usually a more beneficial financial package than receipt of an Indigenous CAS or the ISSP and associated ABSTUDY benefits. A student must carefully consider their best outcome. See Resources page for examples of determining most beneficial outcome.

RCO is to be ended:

  • On the Accommodation (AC) screen, update the Sharer Status: field to remove the RCO code
  • Update the Event Date: field with the date of the beginning of the scholarship period
  • Code the balance of the AC screen as appropriate based on the new circumstances of the student. For further information, see Recording accommodation details
  • Record details on a DOC. Close the Display on Access DOC which identifies receipt of RCO
  • Go to Step 6

3

Discontinuing RCO over a vacation period + Read more ...

A student may wish to discontinue RCO over a vacation period in which they temporarily leave the residential college, and continue with RCO upon recommencing at the college in the following semester. The student must be asked whether they plan to return to the residential accommodation.

For the periods the student temporarily returns home during their scheduled vacation period, the AC screen must be updated so the student's normal rate for ABSTUDY Living Allowance at the away from home rate or independent rate, or the normal rate of ABSTUDY PES, is payable for the periods during which residential fees are not being charged.

The dates of these periods can be provided by the student, or alternatively can be provided by the college or hostel in respect to each term/semester. If exact dates are unavailable, use the formal semester commencement/end dates for the courses at the student's education provider.

Note: if a student opts to remain in their residential college or in a hostel during semester breaks, RCO can continue to be paid. Living Allowance and ABSTUDY PES will continue to be payable at the appropriate reduced or nil rate, and a DOC with relevant details must be placed on the student record.

The AC screen must be updated to reflect each of these periods, and there may be several through the period of study, as:

  • Address Type used for RA Calc: field - code 'HOM'
  • Event Date: field - code the first date the Residential Fees are not being charged
  • Sharer Status: field - code 'EXE'
  • Type of Rent: field - code ‘NRP’
  • Net Rent Amount: field - leave blank
  • Rent Verification: field - leave blank

Obtain the date the student intends to return to the residential college or hostel, and the date the Residential Fees begin again, as the AC screen must be updated in advance to:

  • Address Type used for RA calc: field - code 'TER'
  • Event Date: field - code the date the Residential Fees begin again, that is, when the student returns to the college/hostel
  • Home Ownership: field - code NHO (non-homeowner)
  • Type of Rent: field - code BOA (board and lodgings) or LOD (lodgings)
  • Sharer Status: field - code 'RCO' (Residential Costs Option)
  • Net Rent Amount: field - code the weekly amount being charged by the residential facility
  • Rent Verification: field - code EXB (customer in Boarding Style accommodation)

Note: to obtain the weekly Net Rent Amount, divide the total amount being charged by the amount of weeks the invoice is covering

Go to Step 6

4

No longer wishes to receive RCO payments + Read more ...

Where the customer has not lost eligibility to RCO payments, but no longer wishes to receive RCO, update the AC screen as follows:

  • Sharer Status: remove the RCO code, and update with the customer’s new sharer status
  • Event Date: code using the day after the period covered by the most recent residential fees payment

Code the remaining fields of the AC screen as appropriate based on the new circumstances of the student. For further information, see Recording accommodation details

Annotate the RCO Display on Access (DOA) DOC with details, and expire the DOC.

Go to Step 6

5

No longer eligible for RCO payments + Read more ...

Where the customer is no longer eligible for RCO, for example due to:

  • ceasing study
  • changing from full-time to part-time studies
  • no longer boarding with the accommodation provider

Update the AC screen as follows:

  • Sharer Status: remove the RCO code, and update with the customer’s new sharer status
  • Event Date: code the date of the change in circumstances

Code the remaining fields of the AC screen as appropriate based on the new circumstances of the student. For further information, see Recording accommodation details

Record details on a DOC and close the Display on Access DOC which identifies receipt of RCO

Go to Step 6

6

RCO payments made to the accommodation provider + Read more ...

RCO payments to the accommodation provider can be identified as:

  • Tertiary student - 'RCP' (Residential Costs Payments) amount paid via the Supplementary Education Payments (EPSP) screen
  • Secondary non-schooling student - a DOC on student record

Have there been any RCO payments made to the accommodation provider for the current year?

  • Yes:
    • Create a Fast Note, select Auto text. Use Students, ABSTUDY & Apprentices > Update > ABSTUDY action requests > Confirm
    • Advise in the Fast Note that the student has ceased receiving the Residential Costs Option, and the EPSP screen must be checked for any excess Residential Costs Payments which may have been paid to the accommodation provider. Procedure ends here
    • Go to Step 7
  • No, procedure ends here

7

Residential Costs Payments to be recovered from the accommodation provider + Read more ...

As the student has ceased the RCO prior to the end of studies for the current year, there may be excess Residential Costs Payments which must be recovered from the accommodation provider.

The accommodation provider can retain any payments which have been charged for the accommodation. This includes any period after the student ceases to qualify for RCO. Some examples of ceasing to qualify for RCO include:

  • ceasing study
  • changing from full-time to part-time
  • no longer boarding with the accommodation provider

The accommodation provider may have a refund policy for when a student leaves early. Refunds must be recovered.

To determine the coding required, check the EPSP screen, contact the accommodation provider and confirm the payments are correct, taking into account any refund. This includes charges for a period after they ceased to qualify for RCO.

Adjust the Amount paid: field on each screen to reflect updated charges/refunds. Any periods not already coded may require an additional EPSP screen to be coded.

If a new EPSP screen must be coded, see Step 1 in Table 3.

Finalise activity on the Assessment Results (AR) screen and record details in a DOC on the student's record. Procedure ends here.

8

Eligibility for RCO + Read more ...

Approved residential arrangements, can include:

  • on campus accommodation owned/leased by the education provider
  • off campus accommodation owned/leased by the education provider
  • on campus accommodation owned by private enterprise (for example, Unilodge Bundoora, Unilodge ANU)
  • off campus accommodation owned by private enterprise, only where there is a clear partnership with the education provider to relieve demand on campus accommodation
  • a hostel
  • staying in an overseas residential college or equivalent that is attached to their Australian educational provider

The student will self-identify on their ABSTUDY claim, in a note or verbally where they are living, and verification is not required for this information. The residential charges do not have to include meals for approval of RCO.

Note: for at least one fortnight over the period they are claiming RCO,  the student must receive at least $1 of:

  • Living Allowance, including any amounts of Rent Assistance (RA), Pharmaceutical Allowance (PhA)
  • Energy Supplement (CES)
  • PES, or
  • DFISA-like payment

Is the student eligible for RCO?

  • Yes, go to Step 10
  • No, due to:
    • not receiving an amount of Living Allowance, go to Step 9
    • other reasons, student is not eligible for RCO. Record details in a DOC. Procedure ends here

9

Determining if the student is receiving an amount of Living Allowance + Read more ...

To meet RCO eligibility the student must receive at least $1 of Living Allowance including any amounts of Rent Assistance (RA), Pharmaceutical Allowance (PhA) or Energy Supplement (CES).

This means prior to moving into the residential facility the student may be on a zero rate of payment due to the effects of the personal, partner or parental income tests.

To determine if the student is in receipt of at least $1 of Living Allowance, provisionally code the AC screen:

  • Address Type used for RA Calc: field - code TER (term)
  • Event Date: field - code the date the student started living at the college/hostel
  • Home Ownership: field - code NHO (non-homeowner)
  • Type of Rent: field - code BOA (board and lodgings) or LOD (lodgings)
  • Sharer Status: field - code RCO (Residential Costs Option)
  • Net Rent Amount: field - code the weekly amount being charged by the residential facility
  • Rent Verification: field - code EXB (customer in Boarding Style accommodation)
  • Date of Receipt: field (DOR) - where the Event Date (DOV) is:
    • today or in the future, code the DOR as the date the customer contacts
    • in the past, code a DOR equal to the DOV. This may result in a negative adjustment on the AR screen. For New Claims, this will require the original NCL activity to be cancelled and re-indexed using the DOR that RCO is to be coded from

Note: to obtain the weekly Net Rent Amount, divide the total amount being charged by the amount of weeks the invoice is covering.

Go to the Assessment Results (AR) screen and check if Living Allowance, RA, PhA or CES is now paying.

Is the student now receiving at least $1 of Living Allowance?

  • Yes, go to Step 10
  • No, student is not eligible for RCO. Remove provisional coding and record details in a DOC. Procedure ends here

10

Relocation Scholarship (RS) + Read more ...

A RS payment can be identified on the One Off Payment (OOP) screen. An indigenous CAS can be identified on the Higher Education Scholarship Summary (HESS) screen.

Has the student already been paid a Relocation Scholarship (RS), Indigenous Commonwealth Accommodation Scholarship (CAS), Indigenous Student Success Programme (ISSP) for relocation and/or accommodation costs, DVA Relationship Scholarship or equivalent Commonwealth scholarship for the same scholarship period?

  • Yes, go to Step 11
  • No, see Step 1 in Table 2

11

When is RCO payable? + Read more ...

RCO is not payable for any period where a person has received (or will receive) an RS, ISSP for relocation and/or accommodation costs, Indigenous CAS or other equivalent scholarship. Unless the offer of a place in a residential college or hostel is received after the RS, Indigenous CAS, ISSP for relocation and/or accommodation costs or equivalent scholarship has been paid for a year.

Was the offer of the place in a residential college received after the qualification date for the RS, Indigenous CAS, ISSP for relocation and/or accommodation costs or other equivalent scholarship?

A written or verbal advice from the residential college or hostel identifying the date of the offer of the accommodation will be required.

  • Yes, see Step 1 in Table 2
  • No, and the student elects to continue to receive RS, ISSP for relocation and/or accommodation costs or an Indigenous CAS, the student is ineligible for RCO. Record details on a DOC. Procedure ends here
  • No, and the student elects to take-up RCO, then:
    • RS is to be raised as a debt, and
    • Indigenous CAS is to be cancelled, see Step 4 in Table 2

Consider the following when a student may apply for RCO + Read more ...

The student may apply for RCO. While this option will be financially advantageous to many students, it will not be a benefit to all, particularly those on the maximum rate. The following must be considered:

  • Where eligible Living Allowance recipients take up the RCO, they will receive the lesser of:
    • $41.60 per fortnight, or
    • The student's normal fortnightly Living Allowance entitlement, including any amounts of Rent Assistance, Pharmaceutical Allowance or Energy Supplement (that is, the amount they would receive after the application of the personal, partner and parental income tests). A student receiving ABSTUDY PES will forfeit their ABSTUDY PES payments entirely for the period when their residential costs are being met by ABSTUDY. Their pension payment is not affected by this arrangement

Note: the residential fees component of the RCO is not subject to a means test.

Non-refundable deposits

To secure accommodation some Residential Colleges or Hostels will require a student to pay a deposit in advance. These are deemed as non-refundable deposits. They can be paid prior to the RCO payments commencing. Students must be advised if they do not take up accommodation with the college or hostel, an overpayment would be raised that the student would have to pay back. See Create a new debt record manually on the Debt Management and Information System (DMIS).

A student whose residential costs are higher than their loss of entitlement would benefit financially if they take RCO.

The amount a student may receive for a Relocation Scholarship, ISSP for relocation and/or accommodation costs or an Indigenous CAS must also be considered when looking at the total payable to the student.

See the Resources page for examples.

Additional Assistance and/or Meals Allowance are not available to supplement the reduced rate of Living Allowance or PES.

The Resources page contains examples of these situations.

3

Coding the students record + Read more ...

The RCO is initially coded on the student's record. Coding of the student's record is the same for both tertiary and secondary level students.

The Education Details (EDC) screen must contain accurate course details; the course level for tertiary students is TGD/TGC/TSC/TGB/TGA as appropriate and SNS/SSN/UPC for secondary level students.

The Address Details (AD) screen must have a term address recorded for the period of residence at the college/hostel and the Address type: field updated with the code 'TER'.

The Accommodation (AC) screen must be updated:

  • Address Type used for RA Calc: field - code 'TER' (term)
  • Event Date: field - code the date the student started living at the college/hostel
  • Home Ownership: field - code NHO (non-homeowner)
  • Type of Rent: field - code BOA (board and lodgings) or LOD (lodgings)
  • Sharer Status: field - code 'RCO' (Residential Costs Option)
  • Net Rent Amount: field - code the weekly amount being charged by the residential facility
  • Rent Verification: field - code EXB (customer in Boarding Style accommodation)
  • Date of Receipt: field (DOR) - where the Event Date (DOV) is
    • today or in the future, code the DOR as the date the customer contacts
    • in the past, code a DOR equal to the DOV. This may result in a negative adjustment on the AR screen. For New Claims, this will require the original NCL activity to be cancelled and re-indexed using the DOR that RCO is to be coded from

Note: to obtain the weekly Net Rent Amount, divide the total amount being charged by the amount of weeks the invoice is covering.

Code all other fields as appropriate. For further information, see Completing Accommodation Details (AC) screen and assessing Rent Assistance (RA).

The RCO code in the Sharer Status: field stimulates the system to pay the appropriate amount of Living Allowance or ABSTUDY Pensioner Education Supplement (PES) to the student during periods of residence.

Note: a debt shell for Relocation Scholarship (RS) will occur for a student who has previously been paid RS in the same scholarship period. In circumstances where RCO is payable because the offer of the place in the residential accommodation occurred after a RS has been paid, the RS is not a debt.

Advise the student the college or hostel is required to send an account or invoice to the ABSTUDY Smart Centre processing centre advising the residential charges for each term/semester. The account or invoice must include exact dates charged for accommodation and meals and an itemised list of charges including whether they are compulsory or refundable fees.

Record a DOC containing details of claim for RCO. Record a Display on Access DOC identifying the student is in receipt of RCO.

Is the student in receipt of an Indigenous CAS or ISSP for relocation and/or accommodation costs?

  • Yes, go to Step 4
  • No, go to Step 5

4

Cancelling an Indigenous Commonwealth Accommodation Scholarship or ISSP for relocation and/or accommodation costs + Read more ...

If a student is in receipt of an Indigenous CAS (recorded at HESS screen) or the ISSP for relocation and/or accommodation costs for the scholarship period:

  • Advise the student to contact their higher education provider and retrospectively cancel the Indigenous CAS or ISSP for relocation and/or accommodation costs from the beginning of the scholarship period
  • Provide confirmation from the higher education provider of the cancellation date of the Indigenous CAS or ISSP
  • End date the Indigenous CAS or ISSP on the HESS screen
  • Record details on a DOC. Record a Display on Access DOC identifying the student is in receipt of RCO

  • Students name
  • Exact dates charged for accommodation and meals
  • Itemised list of charges, including if they are compulsory or refundable fees
  • Yes, and the student has contacted through:

System coding for payment of RCO for ABSTUDY and PES

Table 3: Portions of this process can be performed by ABSTUDY Smart Centre Processing staff only.

Step

Action

1

Payment of Residential fees + Read more ...

An account or invoice for the residential fees has been received at the ABSTUDY Smart Centre Processing site in respect of each term/semester.

Payments of residential fees in respect of tertiary students are paid via the Supplementary Education Payments (EPSP) screen.

When the college/hostel claims the residential costs, update the Supplementary Education Payments (EPSP) screen by coding the fields as follows:

  • Institution: field - this field will auto populate from the institution details on the current Education Course Details (EDC) screen
  • Event Date: field - use the first day of the period claimed by the residency provider. If this is prior to the commencement of ABSTUDY, use the first date of entitlement to Living Allowance
  • Payment Type: field - code 'RCP' (residential cost payments)
  • Amount Claimed: field - code an amount here only if the amount claimed is not the amount to be paid
  • Amount Paid: field - code the amount to be paid. Usually this will be the amount the residency provider claims for the full period charged
  • Payee: field - code 'TPT' (third party). Note: no other code can be entered in this field for RCO
  • Claim details: field - free text field, code relevant information including invoice number and period of payment
  • Frequency: field - defaults to 'ONE' - do not change this field
  • Source: and DOR: fields - code as appropriate

If residential fees are to be paid to:

  • more than one residential institution, and the total payment exceeds the ABSTUDY system arrears limit:
    • annotate the open Residential Costs Option Invoice Rec’d DOC to reflect the second or subsequent invoices need to be paid
    • place the open Residential Costs Option Invoice Rec’d DOC on hold for system date +1, to action the second or subsequent invoices the following day
    • action the oldest invoice first, select Continue. Go to Step 2
  • one residential institution, select Continue. Go to Step 2

Note: students who have already paid some of the compulsory non-refundable fees on the invoice such as an online application fee or acceptance fee, must be included when paying the residential costs so the accommodation provider can reimburse the student.

2

The screen will flow to the Payment Instruction Summary Screen (PIPS) + Read more ...

Link to the accommodation provider CRN on the Payment Instructions (PIPI) screen, see Payments to third parties and institutions for ABSTUDY.

If the accommodation provider does not have a CRN create one by referring to Creating and updating Organisation records for ABSTUDY purposes. The accommodation provider must not be linked to a payee that has been manually created on the Payment Instructions Payee Details (PIPD) screen.

This will generate the payment to the college/hostel.

Amounts of over $20,000 will reject overnight, these will have to be reissued the following working day. See Reissuing a rejected payment.

Note: upon reissuing the rejected payment, if the payment amount is too large to issue in one payment, the system will automatically break into 2 or more payments. These will issue on consecutive days.

  • Ensure the DOC is completed
