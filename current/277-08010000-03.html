Description

Obtaining addresses for the USA states + Read more ...

USA state information has been removed from the Central Authority Address list. This is because the USA maintains a list of the State and Federal Authorities on their website which is updated monthly.

2

Tracing a customer in the USA + Read more ...

If attempting to locate (trace) a customer in the USA, issue a 'Request for information (tracing)' letter from the International Transmittal Letters Macro.

Open the Intergovernmental Reference Guide (IRG) to find the address for the state required.

Note: the US Central Authority (Washington DC) have advised they are only able to identify which state a customer is in. They are unable to provide other details.

When a request is:

  • returned 'unknown' from a state within the USA, resend the request to Washington DC. Email the International Transmittal Team
  • for California and Texas, if there is a possible location for a customer, to fast-track the process also send the request to the International Transmittal Team

Active duty Military

For active duty Military members with no current address, send a request to the Cuyahoga County office. See Central Authority Address Listing.

3

Transmitting a case for enforcement + Read more ...

USA addresses can be accessed in the Intergovernmental Referral Guide. To access information in the IRG:

From the HOME page:

  • enter the site as a public user ensuring the STATE radio button is selected
  • select the state from the drop down list
  • select GO

The next page displayed requires the selection of the address type.

  • Select the CONTACT tab
  • Select CENTRAL REGISTRY CONTACT as the address type
  • Select VIEW

The contact details for the authority will be displayed.

About the Central Authority Address Listing

Table 3

Item

Description

1

How have the Central Authority addresses been obtained? + Read more ...

The addresses have been obtained from several places, addresses:

  • that have a link to the National Child Support Enforcement Authority (NCSEA) website have been obtained from a list published by the NCSEA, mostly concerning UNCRAM countries. This list was last published in 2004
  • with a link to the Hague website are recorded as the nominated Central Authorities for the purpose of the Hague Convention
  • with a reference to the "Old List" have been copied over from the list applicable up until 07/04/2011. These addresses were largely obtained from a list given to the Child Support Agency by the Attorney-General's Department in 2000

2

What does 'confirmed' mean? + Read more ...

The reference to 'confirmed' means the Central Authority has been confirmed as the appropriate authority for management of child support claims, either through:

  • a reliable website
  • success in enforcement
  • replies to correspondence, or
  • confirmation from that Central Authority

Those authorities without a 'confirmed' status have not been recently verified, but appear to be the appropriate body on the basis of historical information.

If correspondence is sent to an unconfirmed address and a reply received confirming them as the appropriate authority:

  • escalate to the Child Support Program Branch
  • copy in Child Support International

3

What does 'The Competent Authority' mean? + Read more ...

Where an appropriate child support administration in a country has not been identified (usually because these authorities are not party to any specific convention), address correspondence with the governments of those countries to:

  • 'The Competent Authority'
  • see the UQ45.05 - DFAT covering letter and the DFAT guidelines

4

An address has changed + Read more ...

If the address of a Central Authority has changed, or the Service Officer receives a request for registration from a Central Authority not recorded on the list:

  • escalate to the Child Support Program Branch
  • copy in Child Support International
