Table 1: This table describes how Service Officers amend the details of a Contracted Service Provider (CSP) organisation record registered with the Healthcare Identifier (HI) Service.

Step

Action

1

Acting on behalf of another person know to the HI Service + Read more ...

Is the Service Officer acting on behalf of another person known to the HI Service, for example, CSP Officer, in response to a telephone request?

  • Yes, complete Handle 'Act for' call in Healthcare Identifiers (HI) Service, then go to Step 2
  • No, go to Step 2

Note:

  • When acting on behalf on another person known to the HI Service, for example, a CSP Officer, first search for the individual in eHealth program (EHP) prior to searching for the organisation
  • For the purpose of this procedure, the Service Officer is not acting on behalf of another individual and will search directly for the organisation

2

Search for the associated organisation + Read more ...

From the EHP, perform a search for the associated organisation.

Select Organisation and perform a search for the CSP organisation.

The Search for Organisation screen is displayed with 1 or more matching organisations under search results.

See also: Searching for records in the Healthcare Identifiers Service using the eHealth Program (EHP).

3

Amend CSP organisation details + Read more ...

Select Amend in the Action column of the relevant record.

The Amend CSP organisation screen is displayed.

Select the relevant tabs and make the changes requested.

Select Validate once all required details have been updated.

See also:

Add or amend CSP organisation name details

Amend CSP organisation details

Add or amend CSP address details

Add or amend CSP electronic communication details

Add or amend CSP organisation ELS details

Amend linked organisation details

4

Check for errors + Read more ...

Select Submit once all changes are complete. Are there validation errors?

  • Yes, correct errors as prompted
  • No, the Confirm Amend CSP organisation screen is displayed

5

Amend errors and complete the procedure + Read more ...

When details are:

  • incorrect, select Edit to return to the Amend CSP organisation screen
  • correct, select Save

A confirmation message is displayed.

  • Initiate New Certificate - create a new Public Key Infrastructure (PKI) certificate by entering various details, for example, token type or operating system
