This page contains information on how to assess an increased study load for Austudy and Youth Allowance (YA) students.

On this page:

Study details

System Coding

Study details

Reviewing study details

Table 1: This table describes how to action an increase in study load for Austudy and Youth Allowance (YA) students.

Step

Action

1

Has the student changed their course of study or institution? + Read more ...

  • Yes, see Changing or correcting course of study or educational institution for Youth Allowance, Austudy and Pensioner Education Supplement students
  • No, go to Step 2

2

Is the student studying in an approved course at an approved institution? + Read more ...

See Approved courses of study for Centrelink administered payments for students

  • Yes, go to Step 3
  • No, the customer is considered to be a trainee or job seeker. See Part-time or full-time study as an approved activity. Procedure ends here

3

Has the student who was previously studying at less than 75% of a full time study load increased their study load to at least 75%? + Read more ...

Note: a student who is already undertaking at least 75% of the normal full-time study load is considered to be full-time. Any increase from their current study load will not affect their payments.

  • Yes, go to Step 5
  • No, go to Step 4

4

Concessional study load has not already been considered + Read more ...

If a concessional study load has not already been considered, the student would have to test their eligibility for a either a concessional study load or another Centrelink payment.

  • If the customer is 25 years of age or older and meets the qualification for Austudy, see:
    • Assessing eligibility for the 66% concessional study load for Austudy or
    • Assessing eligibility for the 25% concessional study load for Austudy
  • If the customer is under 25 years and meets the qualification for Youth Allowance, see Assessing concessional study loads for Youth Allowance (YA)

Record any relevant information on a DOC. Procedure ends here.

5

What type of payment is the customer receiving? + Read more ...

  • YA (job seeker), see Youth Allowance (YA) job seeker or Australian Apprentice returning to full-time study
  • Austudy or YA (student) (concessional study load), see Step 1 in the Updating study details table
  • JobSeeker Payment (JSP), see JSP job seekers returning to full-time study

System Coding

Updating study details

Table 2: This table describes how to code an update to a student's study details.

Step

Action

1

The customer already has a course coded as part-time or concessional + Read more ...

End the old course by completing the following:

  • access the customer's record in the NSS system for Austudy and YA, and for Pensioner Education Supplement (PES) the ISS system
  • go to the Education Details History (EDCH) screen
  • select the appropriate (current) course from the EDCH screen by inputting 'S' next to the correct line and pressing [Enter]
  • on the Education Course Details (EDC) screen with the 'old' course, code the date the student finished the previous course in the Student End Date: field
  • code in the appropriate Source and Date of Receipt, and in the Action: field code 'C' for Correct.
    Note: proof of enrolment is not required unless there is doubt about the student's enrolment status

2

Add a new EDC screen + Read more ...

  • create a new EDC screen by coding 'EDC' in the Nxt: field and press [Enter] at the Education Course History (EDCH) screen
    • if old course details were given an end date, whilst in the same activity and if this action returns back to the Activity List (AL) screen, select the activity again, code 'EDC' in the Nxt: field and when returned to the EDCH screen, press [Enter] again
  • enter the new details coding the Participation Status Event Day: field as the day the student changed to full-time study (that is, the day after part-time study ceased)
  • in the Enrolment Verification: field, code 'VNR' (Verification Not Required)
  • in the Participation Status: Course Level: and Field of Study: fields enter information as advised by the student and from their proof of enrolment. Select the correct code from field help
  • obtain relevant details from the student to update the Inst Course Start Date:, Inst Course End Date:, Institution course code: and Student ID: fields
  • update the Student start date: field with the day the student commenced the full-time study load
  • code in the appropriate Source and Date of Receipt, and in the Action: field enter 'I' for Insert

For students in receipt of Austudy or YA, go to Step 3

3

Update the Newstart Education Status (NES) screen + Read more ...

  • Go to the NES screen
  • The first page on the NES screen should show Full-time Student in the Student Status: field. If not create a new screen:
    • Event date: should be the same as the participation status event date on the EDC screen
    • in the Student Status: field enter 'FTS' for full-time student
    • Educ Level Attained: is the last level the customer has successfully completed
    • in the Action: field, input 'I' for Insert

4

Finalising the activity + Read more ...

Finalise the activity on the Assessment Results (AR) screen, after ensuring there are no incorrect back payments or debts raised, and record the reason for the decision and action on a DOC.
