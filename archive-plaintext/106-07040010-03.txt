|                                   | documents + Read more ...         |
|                                   |                                   |
|                                   | -   Do not photocopy identity     |
|                                   |     documents. Documents should   |
|                                   |     only be scanned onto the      |
|                                   |     record if required            |
|                                   | -   Do not request documents if   |
|                                   |     details are already on the    |
|                                   |     Identity Summary screen or    |

|                                   |     at the Reg/Serial No: field.  |

|                                   | -   Code the State: field with a  |
|                                   |     valid Australian State, or    |
|                                   |     'NAT' for National. If this   |

|                                   |     Australia                     |

|                                   | -   Once coded, return the        |

|                                   |     Australian payments under     |
|                                   |     International Agreements.     |
|                                   |     Procedure ends here           |
