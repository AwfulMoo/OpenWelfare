Eligibility for FAA for vacation travel

Table 2: This table describes the circumstances under which an ABSTUDY
tertiary or secondary non-schooling student can be eligible for FAA for
vacation travel during the study year.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if student is eligible for  |
|                                   | FAA for vacation travel + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check if the student has          |
|                                   | previously claimed FAA for this   |
|                                   | travel this year as vacation      |
|                                   | travel may only be claimed once   |
|                                   | per year for a course.            |
|                                   |                                   |
|                                   | Has FAA for vacation travel       |
|                                   | already been claimed?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Eligible for FAA + Read more ...  |
|                                   |                                   |
|                                   | Advise the student, that they are |
|                                   | approved and entitled to          |
|                                   | undertake vacation travel.        |
|                                   | Determine if the student requires |
|                                   | FAA to be:                        |
|                                   |                                   |
|                                   | -   authorised and booked in      |
|                                   |     advance of travel, or         |
|                                   | -   reimbursed after the travel   |
|                                   |     has been undertaken The       |
|                                   |     student will be required to   |
|                                   |     complete an ABSTUDY Fares     |
|                                   |     Allowance Claim (SY032).      |
|                                   |     Procedure ends here.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Ineligible for FAA + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Advise the student they are not   |
|                                   | eligible for FAA for vacation     |
|                                   | travel.                           |
|                                   |                                   |
|                                   | If the travel request has been    |
|                                   | received by ABSTUDY Smart Centre  |
|                                   | telephony, record details of      |
|                                   | discussion on a DOC. Explain      |
|                                   | their review and appeal rights.   |
|                                   | See Initial contact about a       |
|                                   | decision and the review of        |
|                                   | decision process.                 |
|                                   |                                   |
|                                   | If the travel request has been    |
|                                   | received by the ABSTUDY Smart     |
|                                   | Centre Travel Team, finalise the  |
|                                   | Travel Booking status as          |
|                                   | 'Rejected' in the Travel          |
|                                   | Management Application (TMA).     |
|                                   | Record details of the decision in |
|                                   | the Notes field. See Coding       |
|                                   | pre-booked Fares Allowance (FAA)  |
|                                   | entitlements for ABSTUDY and      |
|                                   | ABSTUDY Pensioner Education       |
|                                   | Supplement (PES)                  |
|                                   |                                   |
|                                   | Check if the student's            |
|                                   | circumstances will entitle them   |
|                                   | to access alternate FAA travel    |
|                                   | types.                            |
|                                   |                                   |
|                                   | Record details on a DOC.          |
+-----------------------------------+-----------------------------------+
