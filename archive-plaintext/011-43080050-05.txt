Assessing and Benefits

Verifying a claim

+-----------------------------------+-----------------------------------+
| Screen                            | Method                            |
+-----------------------------------+-----------------------------------+
| NPOI screen                       | The out of pocket amount is       |
|                                   | verified automatically at the     |
|                                   | time of processing when the       |
|                                   | Service Officer keys the partial  |
|                                   | payment amount in the Fee field   |
|                                   | with a P prefix.                  |
+-----------------------------------+-----------------------------------+
| NSCI screen                       | The out of pocket amount can be   |
|                                   | verified after a claim has been   |
|                                   | processed using:                  |
|                                   |                                   |
|                                   | -   NSCI,card number              |
|                                   |                                   |
|                                   | Service Officers must ensure that |
|                                   | where multiple items are waiting  |
|                                   | verification, the correct item is |
|                                   | selected.                         |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Claims breaking the Medicare  |
|                                   |     Safety Net threshold with a   |
|                                   |     keyed partial payment cannot  |
|                                   |     be automatically verified.    |
|                                   |     Service Officers are prompted |
|                                   |     by the message 8-SUB REQ      |
|                                   |     appearing.                    |
|                                   | -   Service Officers are to rekey |
|                                   |     a zero partial payment and    |
|                                   |     then verify the items using   |
|                                   |     the NSCI screen.              |
+-----------------------------------+-----------------------------------+
