Acceptable proof of birth documents

The following original and unaltered identification documents are
accepted as proof of birth to enrol a child in Medicare:

Original birth certificate. Note: commemorative certificates are not
acceptable

If applicant is:

-   the birth mother the Proof of Birth declaration on the back page of
    the Newborn Child Declaration (FA081) form
-   not the birth mother, all 4 pages of the Newborn Child Declaration
    (FA081) form including the Proof of Birth declaration on the back
    page

Passport or naturalisation certificate

Adoption papers (where migration status is known)

Documents issued by the Department of Home Affairs

For a hospital birth, a letter (on hospital letterhead not older than 6
months from the newborns date of birth) from the doctor or midwife who
attended the birth, or another appropriate hospital representative (for
example, Privacy Officer, Maternity Ward clerk, Administrative Officer)
stating:

-   mother's full name
-   child's sex
-   child's date of birth
-   child's place of birth

For a home birth, a letter (not older than 6 months from the newborns
date of birth) from the midwife who attended the birth, stating:

-   midwife's name and qualifications
-   mother's full name
-   child's sex
-   child's date of birth
-   child's place of birth

Correspondence or documentation from Centrelink which verifies the
child's full name and date of birth and gender

An Electronic Proof of Birth (EPoB) from an approved hospital as part of
the Newborn Enrolment Trial

-   To identify newborns enrolled as part of this trial, go to the
    Consumer Directory Maintenance System (CDMS) and check the
    eligibility document type is Enrolment Trial - Proof of Birth

Script - newborn with indeterminate sex

Table 1: This table describes the script Service Officers need to follow
when dealing with a request to enrol a newborn with indeterminate sex.

Due to the sensitivity around this type of request, Service Officers
need to advise the parent about this process using the following script
before enrolling the newborn.

+-----------------------------------+-----------------------------------+
| Situation                         | Script to follow                  |
+-----------------------------------+-----------------------------------+
| Complete introduction and         | 'I am calling to discuss the      |
| identify confirmation             | application we received from you  |
|                                   | to add (child's name) to your     |
|                                   | Medicare card.'                   |
+-----------------------------------+-----------------------------------+
| Verify gender details             | 'I am calling to verify (child's  |
|                                   | name)'s date of birth and         |
|                                   | sex/gender?'                      |
+-----------------------------------+-----------------------------------+
| The gender details are the same   | 'Thank you.'                      |
| as the application                |                                   |
|                                   | 'I am in the process of adding    |
|                                   | (child's name) to your Medicare   |
|                                   | card, however our ICT systems     |
|                                   | only have male or female          |
|                                   | sex/gender option. We are working |
|                                   | to update these systems so that   |
|                                   | we can correctly record (child's  |
|                                   | name)'s gender in the future.     |
|                                   |                                   |
|                                   | 'So that I can add (child's name) |
|                                   | to your Medicare card today, I    |
|                                   | will need to record their gender  |
|                                   | as male or female.                |
|                                   |                                   |
|                                   | 'Would you prefer to have         |
|                                   | (child's name)'s gender recorded  |
|                                   | as male or female for Medicare)?' |
+-----------------------------------+-----------------------------------+
| The gender details are not the    | 'Thank you.                       |
| same as the application           |                                   |
|                                   | 'The gender on the application    |
|                                   | form was recoded as               |
|                                   | (indeterminate).'                 |
|                                   |                                   |
|                                   | Follow the process for handling   |
|                                   | forms containing incorrect        |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| Record gender as male or female   | 'I have now recorded (child's     |
| and follow process as outlined in | name)'s gender as (male or        |
| Change of sex or gender for       | female) for the purpose of        |
| Medicare enrolments               | creating their Medicare record.   |
|                                   |                                   |
|                                   | 'I have also recorded a note on   |
|                                   | (child's name)'s record to state  |
|                                   | that they are (indeterminate).    |
|                                   | Once the Medicare ICT systems     |
|                                   | have been updated to capture      |
|                                   | non-binary, indeterminate         |
|                                   | information we will update        |
|                                   | (child's name)'s details.'        |
|                                   |                                   |
|                                   | Note: no title is to be recorded  |
|                                   | in the newborn's record such as   |
|                                   | Miss or Master unless requested   |
|                                   | by the parent.                    |

FAQ - Medicare digital self service newborn enrolment

Table 2: This table provides frequently asked questions and their
answers for Service Officers and customers about the Medicare digital
self service newborn enrolment process.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: How can a customer      |
|                                   | enrol their newborn digitally?    |
|                                   |                                   |
|                                   | [] Customers can access the       |
|                                   | ‘Enrol a Newborn’ service from    |
|                                   | their Medicare online account and |
|                                   | the Express Plus Medicare mobile  |
|                                   | app.                              |
|                                   |                                   |
|                                   | Medicare online accounts can be   |
|                                   | accessed via myGov. For more      |
|                                   | information on creating, using    |
|                                   | and troubleshooting Medicare      |
|                                   | online accounts and myGov, see    |
|                                   | Using Medicare self service and   |
|                                   | How customers create a myGov      |
|                                   | account and link services.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Who can enrol their     |
|                                   | newborn?                          |
|                                   |                                   |
|                                   | [] Anyone who is currently        |
|                                   | eligible to enrol a newborn can   |
|                                   | use this service. Depending on    |
|                                   | their relationship to the         |
|                                   | newborn, they may need to upload  |
|                                   | additional documentation to       |
|                                   | support their application.        |
|                                   | Information about enrolment       |
|                                   | requirements can be located in    |
|                                   | the Medicare entitlement type of  |
|                                   | newborn table.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: What documents can      |
|                                   | customers upload?                 |
|                                   |                                   |
|                                   | [] Customers can upload a variety |
|                                   | of documents to satisfy proof of  |
|                                   | birth and if required proof of    |
|                                   | relationship to the child.        |
|                                   | Documents include but are not     |
|                                   | limited to all or part of the     |
|                                   | Newborn Child Declaration (FA081) |
|                                   | form, a birth certificate, proof  |
|                                   | of legal guardianship and more.   |
|                                   | See the Acceptable proof of birth |
|                                   | documents table.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Have any changes been   |
|                                   | made to legislation and policy    |
|                                   | regarding the enrolment of        |
|                                   | newborns in Medicare?             |
|                                   |                                   |
|                                   | No. The only thing that has       |
|                                   | changed is the channel in which   |
|                                   | we receive applications. By       |
|                                   | adding a new digital enrolment    |
|                                   | option, parents/guardians have    |
|                                   | greater flexibility to do         |
|                                   | business with Medicare. The same  |
|                                   | eligibility requirements remain   |
|                                   | in place.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Are there any changes   |
|                                   | to CDMS or PaNDA?                 |
|                                   |                                   |
|                                   | Yes, the Consumer Directory       |
|                                   | Maintenance System (CDMS) has a   |
|                                   | new Process Digital Newborn       |
|                                   | Enrolment screen. Information     |
|                                   | that customers enter in their     |
|                                   | digital enrolment appears here.   |
|                                   | Service Officers review and       |
|                                   | assess this information to make a |
|                                   | decision about the enrolment.     |
|                                   |                                   |
|                                   | Some enrolments can be            |
|                                   | automatically accepted. See       |
|                                   | Process newborn child enrolment   |
|                                   | (MOA) from CDMS Provisional       |
|                                   | Registration screen table for     |
|                                   | more information. This will       |
|                                   | create a new consumer record      |
|                                   | without any further manual key    |
|                                   | strokes.                          |
|                                   |                                   |
|                                   | Documents can be located in       |
|                                   | Processing and National Demand    |
|                                   | Allocation (PaNDA) under program: |
|                                   | Medicare Eligibility, work type:  |
|                                   | Digital Newborn Enrolment.        |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: What happens if I       |
|                                   | receive an enquiry about this     |
|                                   | service?                          |
|                                   |                                   |
|                                   | Refer to Customer enquiries about |
