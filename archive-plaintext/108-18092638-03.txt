Determining if the customer is a homeowner or a non-homeowner for the Accommodation Details (AC) screen

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Principal home + Read more ...    |
|                                   |                                   |
|                                   | A customer is considered to own   |
|                                   | their principal home when they    |
|                                   | and/or their partner:             |
|                                   |                                   |
|                                   | -   own the property outright     |
|                                   | -   are purchasing the property   |
|                                   | -   have an interest in the       |
|                                   |     property, for example, they   |
|                                   |     own a percentage of the       |
|                                   |     property with someone other   |
|                                   |     than their partner or by way  |
|                                   |     of a Sale leaseback           |
|                                   |     agreement)                    |
|                                   | -   have the right to use their   |
|                                   |     principal home for life which |
|                                   |     was established by a deceased |
|                                   |     estate or third party         |
|                                   | -   leave the principal home they |
|                                   |     own to enter aged care or     |
|                                   |     other care situation, and     |
|                                   |     retain ownership of the       |
|                                   |     vacated principal home        |
|                                   | -   have a deemed interest in the |
|                                   |     principal home for example,   |
|                                   |     the home is owned by a Trust  |
|                                   |     or Company in which the       |
|                                   |     customer has an interest and  |
|                                   |     that interest gives the       |
|                                   |     customer reasonable security  |
|                                   |     of tenure                     |
|                                   | -   sell their principal home and |
|                                   |     intend to repurchase and      |
|                                   |     proceeds are determined to be |
|                                   |     exempt an asset               |
|                                   | -   own their principal home but  |
|                                   |     temporarily live elsewhere    |
|                                   |                                   |
|                                   | Note: this includes ownership of  |
|                                   | mobile accommodation such as a    |
|                                   | caravan, tent or boat, when it is |
|                                   | their principal home and the      |
|                                   | customer has reasonable security  |
|                                   | of tenure. See Assessing          |
|                                   | caravans, boats and park homes to |
|                                   | determine home ownership          |
|                                   |                                   |
|                                   | Does the customer and/or their    |
|                                   | partner meet one of the above     |
|                                   | criteria?                         |
|                                   |                                   |
|                                   | -   Yes, customer is a home       |
|                                   |     owner, see Step 1 in Table 3  |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Amounts paid for the right to     |
|                                   | live in accommodation for life +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The customer can financially      |
|                                   | secure a right to live in their   |
|                                   | accommodation for life (for       |
|                                   | example, granny flat interest,    |
|                                   | enter a retirement village or     |
|                                   | live in shared equity housing)    |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   paying, or agreeing to pay a  |
|                                   |     lump sum amount (for example, |
|                                   |     entry contribution or shared  |
|                                   |     equity), or                   |
|                                   | -   transferring, or agreeing to  |
|                                   |     transfer assets               |
|                                   |                                   |
|                                   | Has the customer financially      |
|                                   | secured a right to live in their  |
|                                   | accommodation for life in one of  |
|                                   | the above ways?                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, customer is a             |
|                                   |     non-homeowner, see Step 1 in  |
|                                   |     Table 2                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Assessment of entry               |
|                                   | contribution + Read more ...      |
|                                   |                                   |
|                                   | To determine home ownership       |
|                                   | status, compare:                  |
|                                   |                                   |
|                                   | the entry contribution and        |
|                                   |                                   |
|                                   | the Extra Allowable Amount (EAA)  |
|                                   |                                   |
|                                   | -   the EAA is the difference     |
|                                   |     between the homeowner and     |
|                                   |     non-homeowner assets limits   |
|                                   |     at the time the arrangement   |
|                                   |     was created                   |
|                                   |                                   |
|                                   | If the entry contribution is more |
|                                   | than the EAA, the customer is a   |
|                                   | homeowner.                        |
|                                   |                                   |
|                                   | Is the entry contribution more    |
|                                   | than the EAA?                     |
|                                   |                                   |
|                                   | -   Yes, customer is a homeowner, |
|                                   |     see Step 1 in Table 3         |
|                                   | -   No, customer is a             |
|                                   |     non-homeowner, see Step 1 in  |
|                                   |     Table 2                       |
+-----------------------------------+-----------------------------------+

Selecting an AC screen option for non-homeowners

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Non-homeowner + Read more ...     |
|                                   |                                   |
|                                   | A customer is a non-homeowner     |
|                                   | when they do not have a right or  |
|                                   | interest that gives them          |
|                                   | reasonable security of tenure in  |
|                                   | their principal home.             |
|                                   |                                   |
|                                   | Coding the correct non-homeowner  |
|                                   | option in the Home Ownership      |
|                                   | field on the AC screen will       |
|                                   | ensure the non-homeowner assets   |
|                                   | test will apply to the customer.  |
+-----------------------------------+-----------------------------------+
| 2                                 | Non-homeowner lives in a special  |
|                                   | residence + Read more ...         |
|                                   |                                   |
|                                   | Special residences include:       |
|                                   |                                   |
|                                   | -   granny flats when entry       |
|                                   |     contribution is equal to or   |
|                                   |     less than EAA                 |
|                                   | -   retirement villages when      |
|                                   |     entry contribution is equal   |
|                                   |     to or less than EAA           |
|                                   | -   sale - leaseback homes, for   |
|                                   |     more information see          |
|                                   |     References                    |
|                                   | -   shared equity housing when    |
|                                   |     entry contribution is equal   |
|                                   |     to or less than EAA           |
|                                   |                                   |
|                                   | Is the customer a non-homeowner   |
|                                   | who lives in a special residence? |
|                                   |                                   |
|                                   | -   Yes, the correct              |
|                                   |     Non-homeowner option to       |
|                                   |     select is Lives in Special    |
|                                   |     Residence - SRN. Go to Step 7 |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Non-homeowner life interest or    |
|                                   | right to accommodation for life + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | This applies if the customer is:  |
|                                   |                                   |
|                                   | -   paying for a life interest or |
|                                   |     right to accommodation for    |
|                                   |     life                          |
|                                   | -   living in a private residence |
|                                   |     that is to be their principal |
|                                   |     home, and                     |
|                                   | -   entry contribution is equal   |
|                                   |     to or less than the EAA       |
|                                   |                                   |
|                                   | Do all of the above apply?        |
|                                   |                                   |
|                                   | -   Yes, the correct              |
|                                   |     Non-homeowner option is       |
|                                   |     Purchased life tenancy - PLT. |
|                                   |     Go to Step 7                  |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Non-homeowner in government       |
|                                   | funded aged care + Read more ...  |
|                                   |                                   |
|                                   | The Aged Care Processing Team     |
|                                   | enters this code. An Aged Care    |
|                                   | Assessment may be needed. Service |
|                                   | Officers not in the Aged Care     |
|                                   | Processing Team, see Change of    |
|                                   | address to an aged care home.     |
|                                   |                                   |
|                                   | Is the customer a non-homeowner   |
|                                   | in a government funded bed in an  |
|                                   | accredited aged care home?        |
|                                   |                                   |
|                                   | -   Yes, the correct              |
|                                   |     Non-homeowner option is Non   |
|                                   |     homeowner in Govt funded aged |
|                                   |     care - GFN. Go to Step 7      |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Non-homeowner lives in nursing    |
|                                   | home + Read more ...              |
|                                   |                                   |
|                                   | Is the customer a non-homeowner   |
|                                   | who lives permanently in a care   |
|                                   | facility that is listed below?    |
|                                   |                                   |
|                                   | -   multi-purpose service         |
|                                   |     facility                      |
|                                   | -   non-subsidised (unfunded)     |
|                                   |     bed, in an accredited aged    |
|                                   |     care home                     |
|                                   | -   non-accredited aged care      |
|                                   | -   National Torres Strait        |
|                                   |     Islander Flexible Aged Care   |
|                                   |     (NATSIFAC) program            |
|                                   | -   hospital as a nursing home    |
|                                   |     type patient awaiting         |
|                                   |     admission to an aged care     |
|                                   |     home                          |
|                                   |                                   |
|                                   | If:                               |
|                                   |                                   |
|                                   | -   Yes, the correct              |
|                                   |     non-homeowner option is Lives |
|                                   |     in Nursing Home - NHN. Go to  |
|                                   |     Step 7                        |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Customer is a non-homeowner +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If none of the above              |
|                                   | circumstances apply, the correct  |
|                                   | Non-homeowner option is           |
|                                   | Non-homeowner - NHO.              |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Other considerations + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Was the customer previously a     |
|                                   | homeowner?                        |
|                                   |                                   |
|                                   | Yes, no matter whether they have  |
|                                   | sold the previous home or kept    |
|                                   | it:                               |
|                                   |                                   |
|                                   | -   an update to real estate      |
|                                   |     details may be needed         |
|                                   | -   an update to other assets may |
|                                   |     be needed for example,        |
|                                   |     financial assets              |
|                                   | -   Include asset changes on the  |
|                                   |     DOC                           |
|                                   | -   go to Step 8                  |
|                                   |                                   |
|                                   | No, go to Step 8                  |
