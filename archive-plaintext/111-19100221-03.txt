Right of reply process

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Right of reply process - original |
|                                   | checker + Read more ...           |
|                                   |                                   |
|                                   | The original QOL/QMA checker      |
|                                   | reviews the QC decision           |
|                                   | overturned - right of reply       |
|                                   | email. If they:                   |
|                                   |                                   |
|                                   | -   Agree with the overturned     |
|                                   |     decision (dispute is upheld)  |
|                                   |     request the local Service     |
|                                   |     Support Officer (SSO) or Team |
|                                   |     Leader enter supporting       |
|                                   |     comments within the dispute   |
|                                   |     form and update the Status    |
|                                   |     Field with 'Finalised QC      |
|                                   |     decision overturned'.         |
|                                   |     Procedure ends here           |
|                                   | -   Disagree with the overturned  |
|                                   |     decision, discuss with the    |
|                                   |     local SSO or Team Leader.     |
|                                   |     Give the SSO/TL a detailed    |
|                                   |     explanation, including        |
|                                   |     operational resources, of why |
|                                   |     the outcome is not accepted.  |
|                                   |     Request they enter these      |
|                                   |     supporting comments within    |
|                                   |     the dispute form. Once these  |
|                                   |     comments have been added to   |
|                                   |     the form select Submit        |
|                                   |                                   |
|                                   | The original QOL/QMA checker must |
|                                   | add their response to the dispute |
|                                   | form within 5 working days of the |
|                                   | Date of Decision. If no response  |
|                                   | is provided within 7 working      |
|                                   | days, the SSO/CAO who completed   |
|                                   | the dispute may 'Finalise' the    |
|                                   | outcome.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Right of reply - Service Support  |
|                                   | Officer + Read more ...           |
|                                   |                                   |
|                                   | If the original checker disagrees |
|                                   | with the overturned decision the  |
|                                   | dispute status will update to     |
|                                   | 'Escalated' and an addition email |
|                                   | sends automatically to the        |
|                                   | relevant staff.                   |
|                                   |                                   |
|                                   | The SSO must review the comments  |
|                                   | provided by the original checker. |
|                                   | If the SSO:                       |
|                                   |                                   |
|                                   | -   Agrees with the additional    |
|                                   |     comments the original result  |
|                                   |     should be upheld. The SSO or  |
|                                   |     Team Leader should update the |
|                                   |     comments within the dispute   |
|                                   |     form and update the Status    |
|                                   |     Field with 'Finalised - QC    |
|                                   |     decision upheld.' Procedure   |
|                                   |     ends here                     |
|                                   | -   Disagrees with the additional |
|                                   |     comments the dispute may      |
|                                   |     require escalation to an      |
|                                   |     independent third party       |
|                                   |     review                        |
|                                   |                                   |
|                                   | Before seeking a third party      |
|                                   | review, SSOs should discuss the   |
|                                   | outcome. If no agreement can be   |
|                                   | reached, the overturning SSO must |
|                                   | seek a third party review via the |
|                                   | relevant program or policy team   |
|                                   | to obtain a resolution. If unsure |
|                                   | of the relevant stakeholder       |
|                                   | required to complete this review, |
|                                   | contact Quality and Technical     |
|                                   | Support with the details.         |
|                                   | Requests for third party reviews  |
|                                   | will only be actioned when they   |
|                                   | contain all required supporting   |
|                                   | information.                      |
|                                   |                                   |
|                                   | The current status of the dispute |
|                                   | stands until any agreement is     |
|                                   | reached or a third party review   |
|                                   | is finalised.                     |
|                                   |                                   |
|                                   | SSOs should not alter the         |
|                                   | decision coded by another SSO in  |
|                                   | the SharePoint escalation form    |
|                                   | without completed and required    |
|                                   | consultation.                     |
