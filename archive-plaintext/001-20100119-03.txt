Assessing medical certificates and incapacity exemptions for job seekers in COVID-19 areas with reduced or suspended mutual obligation requirements

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessing medical certificates    |
|                                   | and incapacity exemptions for job |
|                                   | seekers in COVID-19 areas with    |
|                                   | reduced or suspended mutual       |
|                                   | obligation requirements + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This process applies only to job  |
|                                   | seekers with reduced or suspended |
|                                   | mutual obligation requirements    |
|                                   | due to COVID-19.                  |
|                                   |                                   |
|                                   | For all other job seekers lodging |
|                                   | medical certificates, follow      |
|                                   | usual process to determine if:    |
|                                   |                                   |
|                                   | -   an exemption from mutual      |
|                                   |     obligation requirements can   |
|                                   |     be granted                    |
|                                   | -   the job seeker requires       |
|                                   |     referral to an Employment     |
|                                   |     Services Assessment (ESAt)    |
|                                   |                                   |
|                                   | Participation requirements for    |
|                                   | all DSP customers aged under 35   |
|                                   | are currently voluntary. There is |
|                                   | no need for DSP customers to      |
|                                   | lodge a medical certificate while |
|                                   | the participation is voluntary.   |
|                                   |                                   |
|                                   | Job seekers who are unable to     |
|                                   | work due to a medical condition   |
|                                   | can seek a temporary incapacity   |
|                                   | exemption from their mutual       |
|                                   | obligation requirements by        |
|                                   | lodging a valid medical           |
|                                   | certificate. There are temporary  |
|                                   | rules for job seekers living in   |
|                                   | COVID-19 areas. These rules allow |
|                                   | job seekers to be granted a       |
|                                   | special circumstances (major      |
|                                   | personal crisis) exemption where  |
|                                   | a temporary incapacity exemption  |
|                                   | cannot be granted.                |
|                                   |                                   |
|                                   | Regardless of the assessment      |
|                                   | outcome for the medical           |
|                                   | certificate, job seekers in a     |
|                                   | COVID-19 area should also be      |
|                                   | referred for an ESAt where        |
|                                   | required, to allow them to access |
|                                   | suitable payments and services.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Job seeker in a COVID-19 area     |
|                                   | with reduced or suspended         |
|                                   | requirements lodges a medical     |
|                                   | certificate + Read more ...       |
|                                   |                                   |
|                                   | Based on the information from the |
|                                   | treating doctor, assess if the    |
|                                   | job seeker can be granted a       |
|                                   | temporary incapacity exemption -  |
|                                   | see Assessment of a medical       |
|                                   | certificate to determine          |
|                                   | exemption from mutual obligation  |
|                                   | requirements for a temporary      |
|                                   | incapacity.                       |
|                                   |                                   |
|                                   | If the job seeker is eligible for |
|                                   | a temporary incapacity exemption, |
|                                   | an exemption can be granted an    |
|                                   | exemption for the period of       |
|                                   | incapacity on the medical         |
|                                   | certificate up to:                |
|                                   |                                   |
|                                   | -   a maximum of 13 weeks as a    |
|                                   |     general rule                  |
|                                   | -   12 months if the job seeker   |
|                                   |     meets the criteria for a      |
|                                   |     serious illness exemption     |
+-----------------------------------+-----------------------------------+
| 3                                 | Manifestly medically eligible for |
|                                   | Disability Support Pension + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a medical certificate shows a  |
|                                   | job seeker is clearly manifestly  |
|                                   | medically eligible for DSP, they  |
|                                   | should be invited to claim        |
|                                   | Disability Support Pension. This  |
|                                   | includes where the job seeker is  |
|                                   | terminally ill.                   |
|                                   |                                   |
|                                   | See Granting Disability Support   |
|                                   | Pension (DSP) (manifest).         |
+-----------------------------------+-----------------------------------+
| 4                                 | Employment Services Assessment    |
|                                   | (ESAt) referral + Read more ...   |
|                                   |                                   |
|                                   | Consider if an ESAt referral is   |
|                                   | required to assess the job        |
|                                   | seeker's work capacity and        |
|                                   | support needs.                    |
|                                   |                                   |
|                                   | An ESAt should be considered if   |
|                                   | the job seeker does not have a    |
|                                   | current ESAt, or their            |
|                                   | circumstances have changed since  |
|                                   | their last ESAt, and available    |
|                                   | information indicates they:       |
|                                   |                                   |
|                                   | -   may have an ongoing partial   |
|                                   |     capacity to work, because     |
|                                   |     they are unable to work at    |
|                                   |     least 30 hours per week       |
|                                   |     within the next 2 years       |
|                                   | -   cannot be granted a temporary |
|                                   |     incapacity exemption and may  |
|                                   |     have a temporary reduced work |
|                                   |     capacity of less than 30      |
|                                   |     hours per week                |
|                                   | -   may benefit from referral to  |
|                                   |     Disability Employment         |
|                                   |     Services                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Special circumstances (major      |
|                                   | personal crisis) exemption + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a temporary incapacity         |
|                                   | exemption cannot be granted for a |
|                                   | job seeker in a COVID-19 area     |
|                                   | with reduced or suspended         |
|                                   | requirements, a special           |
|                                   | circumstances (major personal     |
|                                   | crisis) exemption may be granted  |
|                                   | for the period of the medical     |
|                                   | certificate, up to a maximum of   |
|                                   | 13 weeks. This includes where     |
|                                   | the:                              |
|                                   |                                   |
|                                   | -   job seeker is able to do some |
|                                   |     work of at least 8 hours per  |
|                                   |     week                          |
|                                   | -   incapacity for work is        |
|                                   |     clearly not temporary         |
|                                   |                                   |
|                                   | Job seekers are not eligible for  |
|                                   | an incapacity exemption if their  |
|                                   | incapacity for work is mainly     |
|                                   | because of their drug or alcohol  |
|                                   | dependency/misuse. In this case,  |
|                                   | a special circumstances (major    |
|                                   | personal crisis) exemption may be |
|                                   | considered due to COVID-19, if    |
|                                   | special circumstances not wholly  |
|                                   | or predominantly due to misuse of |
|                                   | drugs or alcohol mean that job    |
|                                   | seekers are unable to comply with |
|                                   | their requirements.               |
|                                   |                                   |
|                                   | For more details, see Exempting a |
|                                   | job seeker from mutual obligation |
|                                   | requirements due to special       |
|                                   | circumstances.                    |
