|                                   | job seeker compliance. Job seeker |

|                                   | Framework.                        |

|                                   | failure with an event date during |
|                                   | 2020 or 2021, staff must check to |

|                                   | the job seeker on the event date  |
|                                   | of the failure.                   |
|                                   |                                   |
|                                   | If reviewing a rejected claim or  |
|                                   | a job seeker compliance decision  |
|                                   | made since March 2020, check:     |
|                                   |                                   |
|                                   | -   Historical dates and          |
|                                   |     locations, refer to Mutual    |
|                                   |     obligation pause and          |
|                                   |     resumption dates table on the |
|                                   |     Resources page                |
|                                   | -   Current Operational Messages  |
|                                   |     for any current altered       |
|                                   |     arrangements                  |

Community Development Program (CDP) - suspension of mutual obligation requirements

For Participation Solutions Team (PST) - skilled Service Officers only

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Temporary measures to support CDP |
|                                   | job seekers + Read more ...       |
|                                   |                                   |
|                                   | Where mutual obligation           |
|                                   | requirements are suspended in CDP |
|                                   | regions due to COVID-19           |
|                                   | outbreaks, the following          |
|                                   | temporary measures are put in     |
|                                   | place to support CDP job seekers  |
|                                   | in these regions:                 |
|                                   |                                   |
|                                   | National Indigenous Australians   |
|                                   | Agency (NIAA) instructs CDP       |
|                                   | service providers in impacted     |
|                                   | regions not to submit any         |
|                                   | Provider Reports for              |
|                                   | non-compliance during the         |
|                                   | suspension of mutual obligation   |
|                                   | requirements                      |
|                                   |                                   |
|                                   | CDP job seekers are not required  |
|                                   | to attend face-to-face            |
|                                   | appointments with their provider  |
|                                   | (appointments can be conducted    |
|                                   | over the phone)                   |
|                                   |                                   |
|                                   | The Department of Employment and  |
|                                   | Workplace Relations (DEWR)        |
|                                   | automatically lifts all           |
|                                   | suspensions (manual intervention  |
|                                   | by Services Australia is approved |
|                                   | when payment has not              |
|                                   | automatically restored)           |
|                                   |                                   |
|                                   | Services Australia takes the      |
|                                   | following actions:                |
|                                   |                                   |
|                                   | -   A consolidated strategy is    |
|                                   |     put in place by the Job       |
|                                   |     Seeker Compliance Team to end |
|                                   |     existing penalties for CDP    |
|                                   |     job seekers in impacted       |
|                                   |     regions.                      |
|                                   | -   If the job seeker contacts    |
|                                   |     regarding a:                  |
|                                   |     - submitted Provider Report,  |
|                                   |     (i.e. Connection failure      |
|                                   |     (CNN), Reconnection failure   |
|                                   |     (RCN), Provider Appointment   |
|                                   |     Report (PAR) No Show, No Pay  |
|                                   |     (for failure to attend a job  |
|                                   |     I/V) (NSP)), go to Step 2     |
|                                   |     - Non-attendance Report       |
|                                   |     (NAR), go to Step 3           |
|                                   |     - Unemployment Non Payment    |
|                                   |     Period (UNPP), go to Step 4   |
|                                   |     - Comprehensive Compliance    |
|                                   |     Assessment (CCA), go to Step  |
|                                   |     5                             |
|                                   |     - Serious Failure (8 week     |
|                                   |     non-payment periods), go to   |
|                                   |     Step 6                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Submitted Provider Report - CNN,  |
|                                   | RCN, PAR, NSP + Read more ...     |
|                                   |                                   |
|                                   | The penalties that were           |
|                                   | previously applied must be ended. |
|                                   | The date of effect must be the    |
|                                   | business date immediately before  |
|                                   | the start of the suspension of    |
|                                   | mutual obligation requirements.   |
|                                   |                                   |
|                                   | If the failures were not ended    |
|                                   | through the consolidated          |
|                                   | strategy, a PST-skilled Service   |
|                                   | Officer with the WNPPD security   |
|                                   | resource must end the penalty     |
|                                   | under Serious Failure Hardship    |
|                                   | Provisions.                       |
|                                   |                                   |
|                                   | -   Select the job seeker excuse  |
|                                   |     reason: Not required - reject |
|                                   |     due to system issues. This    |
|                                   |     will automatically            |
|                                   |     pre-populate the rejection    |
|                                   |     code (GIE)                    |
|                                   | -   On the Decision and Penalty   |
|                                   |     screen record:                |
|                                   |     'Failure is rejected due to   |
|                                   |     NIAA direction as a result of |
|                                   |     COVID-19 contingency measure' |
|                                   | -   Finalise activity and make    |
|                                   |     sure the job seeker's payment |
|                                   |     is displaying correctly       |
+-----------------------------------+-----------------------------------+
| 3                                 | Non-attendance Reports (NAR) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Department of Employment and  |
|                                   | Workplace Relations (DEWR) lift   |
|                                   | Non-attendance Reports (NAR)      |
|                                   | suspensions on a daily basis      |
|                                   | during the suspension of mutual   |
|                                   | obligation requirements.          |
|                                   |                                   |
|                                   | If a CDP job seeker contacts      |
|                                   | about a NAR suspension that       |
|                                   | hasn't been lifted and is more    |
|                                   | than 24 hours old, PST-skilled    |
|                                   | Service Officers should lift it   |
|                                   | manually.                         |
|                                   |                                   |
|                                   | -   Select: re-engagement link    |
|                                   |     from Compliance Action screen |
|                                   | -   Record:                       |
|                                   |     'No re-engagement, unable to  |
|                                   |     attend appointment or contact |
|                                   |     provider (RNU)'               |
|                                   | -   Record free text on the       |
|                                   |     Activity Result (AR) screen   |
|                                   |     on the job seekers record:    |
|                                   |     'No reengagement required due |
|                                   |     to NIAA direction as a result |
|                                   |     of COVID-19 contingency       |
|                                   |     measure'                      |
|                                   | -   Finalise activity, clear out  |
|                                   |     of the record                 |
|                                   | -   If the system does not        |
|                                   |     automatically restore the     |
|                                   |     payment within a few minutes, |
|                                   |     manually restore on the       |
|                                   |     Benefit Action (BA) screen    |
+-----------------------------------+-----------------------------------+
| 4                                 | Unemployment Non Payment Periods  |
|                                   | (UNPP) + Read more ...            |
|                                   |                                   |
|                                   | UNPP failures and hardship        |
|                                   | provision determinations are      |
|                                   | managed under current business    |
|                                   | processes.                        |
|                                   |                                   |
|                                   | Outstanding failures              |
|                                   |                                   |
|                                   | Any outstanding UNPP failures     |
|                                   | must be investigated by a         |
|                                   | PST-skilled Service Officer with  |
|                                   | a WNPPD security resource.        |
|                                   |                                   |
|                                   | See Unemployment due to a         |
|                                   | voluntary act or misconduct.      |
|                                   |                                   |
|                                   | If the UNPP has commenced         |
|                                   |                                   |
|                                   | A UNPP may be waived under        |
|                                   | hardship provisions if the CDP    |
|                                   | job seeker:                       |
|                                   |                                   |
|                                   | -   falls within a class of       |
|                                   |     person specified by the       |
|                                   |     Secretary of the Department   |
|                                   |     of Employment and Workplace   |
|                                   |     Relations (DEWR), and         |
|                                   | -   would be in severe financial  |
|                                   |     hardship as a result of the   |
|                                   |     UNPP                          |
|                                   |                                   |
|                                   | See Applying an Unemployment      |
|                                   | Non-Payment Period (UNPP) and     |
|                                   | determining hardship provisions.  |
+-----------------------------------+-----------------------------------+
| 5                                 | Comprehensive Compliance          |
|                                   | Assessment (CCA) + Read more ...  |
|                                   |                                   |
|                                   | All outstanding CCAs must be      |
|                                   | finalised without penalty (FWO) - |
|                                   | Not persistent Non-compliance.    |
|                                   | The CCA should not be conducted   |
|                                   | with the job seeker.              |
|                                   |                                   |
|                                   | Finalising a CCA without penalty  |
|                                   |                                   |
|                                   | Select the CCA via the Compliance |
|                                   | Investigation Workflow            |
|                                   |                                   |
|                                   | Select 'Yes' to the question 'Is  |
|                                   | the CCA Valid?' on the Review     |
|                                   | Record screen                     |
|                                   |                                   |
|                                   | On the CCA Details screen, select |
|                                   | 'No' to the notification question |
|                                   | and record free text:             |
|                                   | 'CCA finalised without penalty,   |
|                                   | not conducted with job seeker due |
|                                   | to NIAA direction as a result of  |
|                                   | COVID-19 contingency measure'     |
|                                   |                                   |
|                                   | On the Customer Contact screen,   |
|                                   | record:                           |
|                                   |                                   |
|                                   | -   Incoming customer initiated   |
|                                   |     contact - other for method of |
|                                   |     contact                       |
|                                   | -   Other for where the customer  |
|                                   |     is calling from               |
|                                   | -   Record free text in the       |
|                                   |     contact attempt section:      |
|                                   |     'CCA finalised without        |
|                                   |     penalty, not conducted with   |
|                                   |     job seeker due to NIAA        |
|                                   |     direction as a result of      |
|                                   |     COVID-19 contingency measure' |
|                                   | -   Yes to the question 'Was this |
|                                   |     contact attempt successful?'  |
|                                   | -   No to the question 'Has a CCA |
|                                   |     Interview been booked?'       |
|                                   |                                   |
|                                   | On the CCA commencement screen,   |
|                                   | tick yes to the Privacy Notice    |
|                                   | and record 'Yes' to the question  |
|                                   | 'Has the customer attended the    |
|                                   | CCA?'                             |
|                                   |                                   |
|                                   | On the Job Seeker Circumstances   |
|                                   | screen - as the CCA has not been  |
|                                   | conducted, do not update any of   |
|                                   | the CIC. Select continue and      |
|                                   | click OK when the pop up presents |
|                                   |                                   |
|                                   | On the Decision Screen:           |
|                                   |                                   |
|                                   | -   Record the outcome as:        |
|                                   |     Finalise CCA and do not apply |
|                                   |     a penalty                     |
|                                   | -   Reason Code: Not persistent   |
|                                   |     non-compliance                |
|                                   | -   Summary of Decision text:     |
|                                   |     'CCA finalised without        |
|                                   |     penalty, not conducted with   |
|                                   |     job seeker due to NIAA        |
|                                   |     direction as a result of      |
|                                   |     COVID-19 contingency measure' |
|                                   |                                   |
|                                   | Finalise the CCA workflow         |
+-----------------------------------+-----------------------------------+
| 6                                 | Serious failure (8 week           |
|                                   | non-payment period) + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The penalties that were           |
|                                   | previously applied must be ended. |
|                                   | The date of effect must be the    |
|                                   | business date immediately before  |
|                                   | the start of the suspension of    |
|                                   | mutual obligation requirements.   |
|                                   |                                   |
|                                   | Waiving a Serious failure (8-week |
|                                   | non-payment period)               |
|                                   |                                   |
|                                   | -   Select the serious failure    |
|                                   |     via the Compliance            |
|                                   |     Investigation Workflow        |
|                                   | -   Select Review Decision        |
|                                   | -   Record the date review        |
|                                   |     requested: Date of contact    |
|                                   | -   Select Actions required:      |
|                                   |     Waive                         |
|                                   | -   Provide details to support    |
|                                   |     the reason for making this    |
|                                   |     decision, record:             |
|                                   |     'Penalty waived due to NIAA   |
|                                   |     direction as a result of      |
|                                   |     COVID-19 contingency measure' |
|                                   | -   Who initiated the waiver      |
|                                   |     discussion: Customer          |
|                                   | -   What was the date of contact? |
|                                   |     Record the business date      |
|                                   |     immediately before the start  |
|                                   |     of the suspension of mutual   |
|                                   |     obligation requirements       |
|                                   | -   Does the customer have the    |
|                                   |     capacity to undertake a       |
|                                   |     Serious Failure Requirement?  |
|                                   |     No                            |
|                                   | -   Will serving the non-payment  |
|                                   |     period cause the customer     |
|                                   |     severe financial hardship?    |
|                                   |     Yes                           |
|                                   | -   Select the reason for waiving |
|                                   |     the penalty period            |
|                                   |                                   |
|                                   | If the Penalty period has         |
|                                   | commenced                         |
|                                   |                                   |
|                                   | -   Select - Serious Failure      |
|                                   |     Period Hardship - Partial     |
|                                   | -   Select Actions required:      |
|                                   |     Adjust Penalty Period         |
|                                   | -   Please indicate why the       |
|                                   |     penalty needs to be adjusted: |
|                                   |     Yes - Actual Penalty Period   |
|                                   |     Finishes / Starts Earlier     |
|                                   |     Than System Calculated Period |
|                                   | -   Adjusted penalty period start |
|                                   |     date: do not change this      |
|                                   |     date. The penalty start date  |
|                                   |     remains the same.             |
|                                   | -   Adjusted penalty period end   |
|                                   |     date: the business date       |
|                                   |     immediately before the start  |
|                                   |     of the suspension of mutual   |
|                                   |     obligation requirements       |
|                                   | -   Provide details to support    |
|                                   |     the reason for making this    |
|                                   |     decision: Record:             |
|                                   |     'Penalty waived from          |
|                                   |     [business date immediately    |
|                                   |     before the start of the       |
|                                   |     suspension of mutual          |
|                                   |     obligation requirements] due  |
|                                   |     to NIAA direction as a result |
|                                   |     of COVID-19 contingency       |
|                                   |     measure'                      |
|                                   | -   Source: INT, Date: T          |
|                                   | -   Continue through the ripples  |
|                                   |     then finalise activity on AR  |
|                                   |     screen                        |
|                                   |                                   |
|                                   | If the Penalty period has not     |
|                                   | commenced                         |
|                                   |                                   |
|                                   | -   Select - Serious Failure      |
|                                   |     Period Hardship - Full        |
|                                   | -   Provide a summary of why this |
|                                   |     waiver decision has been      |
|                                   |     made, record the following:   |
|                                   |     'Penalty waived due to NIAA   |
|                                   |     direction as a result of      |
|                                   |     COVID-19 contingency measure' |
|                                   | -   Source: INT, Date: T          |
|                                   | -   Continue through the ripples  |
|                                   |     then finalise activity on AR  |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Finalise all outstanding          |
|                                   | requirements on the Participation |
|                                   | Summary with a result of 'No' and |
|                                   | 'No appointment available /       |
|                                   | required'.                        |
