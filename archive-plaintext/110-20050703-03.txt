| 8                                 | Payment eligibility + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Group members may be eligible for |
|                                   | a settlement payment if they have |
|                                   | paid money towards a debt that    |
|                                   | was based in full, or in part, on |
|                                   | averaged ATO income information   |
|                                   | or if all of the following        |
|                                   | applies:                          |
|                                   |                                   |
|                                   | -   the debt was based in full or |
|                                   |     in part on averaged ATO       |
|                                   |     income information            |
|                                   | -   the debt was recalculated     |
|                                   |     based on information you gave |
|                                   |     us, like payslips             |
|                                   | -   the group member paid back    |
|                                   |     more than the recalculated    |
|                                   |     amount                        |
|                                   |                                   |
|                                   | Not every group member will be    |
|                                   | eligible for a settlement         |
|                                   | payment:                          |
|                                   |                                   |
|                                   | -   under the terms of            |
|                                   |     settlement, we will make      |
|                                   |     payments similar to interest  |
|                                   |     to eligible group members. It |
|                                   |     is based on the amount of     |
|                                   |     debt repayments you made and  |
|                                   |     for the period you were       |
|                                   |     without the money. Group      |
|                                   |     members who paid back more    |
|                                   |     and were without this money   |
|                                   |     for longer, will get larger   |
|                                   |     payments                      |
|                                   | -   group members who paid back a |
|                                   |     small amount or were refunded |
|                                   |     soon after they made the      |
|                                   |     payment will receive a        |
|                                   |     smaller payment               |
|                                   | -   most settlement payments will |
|                                   |     be between $50 and $300. Some |
|                                   |     eligible group members who    |
|                                   |     only repaid small amounts may |
|                                   |     get less than $10             |
|                                   | -   Services Australia aims to    |
|                                   |     deliver settlement payments   |
|                                   |     as quickly as possible to     |
|                                   |     eligible group members. The   |
|                                   |     settlement process will take  |
|                                   |     time                          |
|                                   | -   the agency expects to start   |
|                                   |     making payments by mid-2022.  |
|                                   |     These will be spread out over |
|                                   |     a number of months            |
|                                   | -   there is no need for the      |
|                                   |     group member to call or visit |
|                                   |     us unless they have been      |
|                                   |     asked to                      |
