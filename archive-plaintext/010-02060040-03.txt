Payment of arrears and recording outcome

Table 2: This table describes the process involved when the Agreement
has been returned, payment of arrears and recording the outcome.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | When forms returned + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Check the 2 returned ABSTUDY      |
|                                   | Standard Hostels Agreement to     |
|                                   | ensure that:                      |
|                                   |                                   |
|                                   | -   terms of the agreement have   |
|                                   |     not been altered              |
|                                   | -   no pages have been replaced   |
|                                   | -   all relevant certificates and |
|                                   |     approvals have been provided  |
|                                   |     as per clause 3.1 of the      |
|                                   |     Agreement                     |
|                                   | -   each page has been initialled |
|                                   | -   both copies have been signed  |
|                                   |     by the Chairperson or Manager |
|                                   |     of the hostel, or an AHL      |
|                                   |     representative, and their     |
|                                   |     signature has been witnessed  |
|                                   |                                   |
|                                   | Has the Agreement been returned   |
|                                   | satisfactorily completed?         |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Agreement has not been            |
|                                   | satisfactorily completed + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Contact the hostel to advise that |
|                                   | an aspect of the Agreement has    |
|                                   | not been satisfactorily           |
|                                   | completed. Return the Agreement   |
|                                   | for satisfactory completion, and  |
|                                   | advise that payments of ABSTUDY   |
|                                   | away from home allowances will    |
|                                   | not be payable via any method     |
|                                   | until the Agreement is returned   |
|                                   | satisfactorily completed.         |
|                                   |                                   |
|                                   | Note: payments should not be made |
|                                   | via the fortnight-in-arrears      |
|                                   | method until the Agreement is     |
|                                   | returned completed, as it will    |
|                                   | affect the capacity to pay        |
|                                   | ABSTUDY boarding instalment       |
|                                   | payments once the Agreement is    |
|                                   | completed.                        |
|                                   |                                   |
|                                   | Once the agreement is returned    |
|                                   | satisfactorily completed, go to   |
|                                   | Step 4.                           |
|                                   |                                   |
|                                   | If the hostel advises it will not |
|                                   | sign the Agreement, go to Step 3. |
+-----------------------------------+-----------------------------------+
| 3                                 | Fortnight in arrears payment      |
|                                   | method + Read more ...            |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Hostels not seeking payment via   |
|                                   | ABSTUDY boarding instalments, or  |
|                                   | not willing to accept the         |
|                                   | conditions of the Agreement must  |
|                                   | have payment of ABSTUDY           |
|                                   | allowances directed to the hostel |

|                                   | for the correct coding of a       |
|                                   | student's record to ensure        |
|                                   | ABSTUDY away from home allowances |
|                                   | are directed to the hostel        |
|                                   | fortnight-in-arrears.             |

|                                   | Create a DOC on the hostel's      |
|                                   | organisation record detailing the |
|                                   | contact with the hostel and that  |
|                                   | an Agreement will not be entered  |
|                                   | into by the hostel for the        |
|                                   | current 3 year period.            |
|                                   |                                   |
|                                   | If required, request update (via  |
|                                   | Operational Blueprint feedback)   |
|                                   | to remove the hostel from the     |
|                                   | Hostel Agreement list in          |
|                                   | Resources.                        |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   | hostel.                           |
