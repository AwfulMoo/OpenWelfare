This document outlines Healthcare Provider Identifier - Organisation
(HPI-O) in the Healthcare Identifiers (HI) Service.

Healthcare Provider Organisation

A healthcare provider organisation is an organisation where healthcare
is provided.

There are two types of healthcare provider organisation in the HI
Service:

-   Seed Organisation. See also: Create Healthcare Provider Identifier -
    Organisation (HPI-O) record for seed organisation in Healthcare
    Identifiers (HI) Service
-   Network Organisation. See also: Create Healthcare Provider
    Identifier - Organisation (HPI-O) record for network organisation in
    Healthcare Identifiers (HI) Service
