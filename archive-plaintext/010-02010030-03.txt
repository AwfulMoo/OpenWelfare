Determining who should be the ABSTUDY applicant

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Person wishing to claiming        |
|                                   | ABSTUDY + Read more ...           |
|                                   |                                   |
|                                   | If the person wishing to claim    |
|                                   | ABSTUDY is:                       |
|                                   |                                   |
|                                   | -   a correspondence nominee      |
|                                   |     acting on behalf of a person  |
|                                   |     wishing to claim ABSTUDY, go  |
|                                   |     to Step 8                     |
|                                   | -   an Australian Apprentice,     |
|                                   |     they will be the applicant,   |
|                                   |     go to Step 8                  |
|                                   | -   a student, go to Step 2       |
+-----------------------------------+-----------------------------------+
| 2                                 | Student is the applicant + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Students who meet one of the      |
|                                   | following will be considered the  |
|                                   | applicant:                        |
|                                   |                                   |
|                                   | -   aged 16 years or over. See    |
|                                   |     the Resources page for an     |
|                                   |     example                       |
|                                   | -   independent under an ABSTUDY  |
|                                   |     criteria                      |
|                                   | -   applying for tertiary or      |
|                                   |     postgraduate study            |
|                                   | -   receiving a Commonwealth      |
|                                   |     Government pension, for       |
|                                   |     example, Parenting Payment    |
|                                   |     Single (PPS) or Disability    |
|                                   |     Support Pension (DSP)         |
|                                   |                                   |
|                                   | Does the student meet one of the  |
|                                   | above?                            |
|                                   |                                   |
|                                   | -   Yes, the student will be the  |
|                                   |     applicant, go to Step 6       |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Disputes over who is the          |
|                                   | applicant + Read more ...         |
|                                   |                                   |
|                                   | Service Officers need to make a   |
|                                   | decision about who is the most    |
|                                   | suitable person to be the ABSTUDY |
|                                   | applicant. When determining whom  |
|                                   | the most suitable person is,      |
|                                   | consider:                         |
|                                   |                                   |
|                                   | -   whether the student lives     |
|                                   |     with the applicant and has    |
|                                   |     done so in the last 12 months |
|                                   | -   whether the student lives     |
|                                   |     with the applicant during     |
|                                   |     vacations                     |
|                                   | -   who is responsible for the    |
|                                   |     care of the student, for      |
|                                   |     example, providing food and   |
|                                   |     shelter                       |
|                                   | -   who contributes regularly and |
|                                   |     significantly to the          |
|                                   |     student's living and          |
|                                   |     education costs               |
|                                   | -   who receives Family Tax       |
|                                   |     Benefit (FTB) or other        |
|                                   |     allowances in respect of the  |
|                                   |     student                       |
|                                   | -   who has been awarded custody  |
|                                   |     of the student                |
|                                   |                                   |
|                                   | Where more than one person has    |
|                                   | applied for ABSTUDY on behalf of  |
|                                   | a student or there is doubt as to |
|                                   | who is the parent/guardian,       |
|                                   | investigations will need to be    |
|                                   | carried out to determine who the  |
|                                   | most suitable person to apply is. |
|                                   |                                   |
|                                   | Is there a dispute over who the   |
|                                   | applicant is?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Determine most suitable           |
|                                   | applicant + Read more ...         |
|                                   |                                   |
|                                   | Service Officers need to make a   |
|                                   | decision about who is the most    |
|                                   | suitable person to be the ABSTUDY |
|                                   | applicant, this can involve:      |
|                                   |                                   |
|                                   | -   both parties, and/or (with    |
|                                   |     the applicants' knowledge)    |
|                                   | -   independent authorities (with |
|                                   |     the applicantâ€™s knowledge)    |
|                                   |     such as a school or welfare   |
|                                   |     worker                        |
|                                   | -   advice from the student       |
|                                   |                                   |
|                                   | Where difficult cases present, it |
|                                   | may be beneficial to speak to an  |
|                                   | Indigenous Service Officer (ISO)  |
|                                   | before making the final decision. |
|                                   |                                   |
|                                   | Once the applicant has been       |
|                                   | determined, go to Step 8.         |
+-----------------------------------+-----------------------------------+
| 5                                 | The ABSTUDY claim must be lodged  |
|                                   | by an approved applicant + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The applicant may be:             |
|                                   |                                   |
|                                   | -   either of the natural or      |
|                                   |     adoptive parent(s) with whom  |
|                                   |     the student lives             |
|                                   | -   the natural or adoptive       |
|                                   |     parent with whom the student  |
|                                   |     normally lives, where their   |
|                                   |     natural or adoptive parents   |
|                                   |     are separated                 |
|                                   | -   a guardian with whom the      |
|                                   |     student lives who is          |
|                                   |     responsible for their care    |
|                                   | -   the foster parent or guardian |
|                                   |     of a student who is in State  |
|                                   |     care and has been placed in   |
|                                   |     their care                    |
|                                   | -   an officer of the relevant    |
|                                   |     State/Territory authority,    |
|                                   |     where the student is in State |
|                                   |     care and is living in an      |
|                                   |     institution, or               |
|                                   | -   a person, or official of an   |
|                                   |     institution, who has care of  |
|                                   |     the student, where the young  |
|                                   |     person would, but for reasons |
|                                   |     of age, qualify as an orphan  |
|                                   |     or as homeless                |
|                                   |                                   |
|                                   | Does the applicant meet one of    |
|                                   | the above circumstances?          |
|                                   |                                   |
|                                   | -   Yes, they are considered an   |
|                                   |     approved applicant, go to     |
|                                   |     Step 8                        |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Claim lodgement + Read more ...   |
|                                   |                                   |
|                                   | Can the student lodge the claim?  |
|                                   |                                   |
|                                   | Note: a suitable reason for not   |
|                                   | lodging the claim would be        |
|                                   | exceptional circumstances such as |
|                                   | serious illness.                  |
|                                   |                                   |
|                                   | -   Yes, the student will be      |
|                                   |     considered the applicant, go  |
|                                   |     to Step 8                     |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Student unable to lodge claim +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | An interim applicant may lodge a  |
|                                   | claim for ABSTUDY on behalf of a  |
|                                   | student if they:                  |
|                                   |                                   |
|                                   | -   have temporary care of the    |
|                                   |     student                       |
|                                   | -   can make an informed decision |
|                                   |     about the student's           |
|                                   |     Aboriginality or Torres       |
|                                   |     Strait Islander status, and   |
|                                   | -   are following the wishes of   |
|                                   |     the student or                |
|                                   |     parent/long-term guardian in  |
|                                   |     respect of their payments     |
|                                   |                                   |
|                                   | Does the person meet all the      |
|                                   | above circumstances?              |
|                                   |                                   |
|                                   | -   Yes, consider them an interim |
|                                   |     applicant. They can lodge an  |
|                                   |     ABSTUDY claim for the         |
|                                   |     student. Go to Step 8         |
|                                   | -   No, do not accept a claim.    |
|                                   |     Advise them of the decision   |
|                                   |     and record details on a DOC.  |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 8                                 | Lodge appropriate ABSTUDY form +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Advise the applicant or           |
|                                   | correspondence nominee they can   |
|                                   | lodge the appropriate claim for   |
|                                   | ABSTUDY. For information about    |
|                                   | the most appropriate ABSTUDY      |
|                                   | claim, see Claiming ABSTUDY.      |
|                                   |                                   |
|                                   | Record details on a DOC.          |
