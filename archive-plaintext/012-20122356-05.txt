FAQs - Ahpra pandemic sub-register

Table 1: frequently asked questions about the Australian Health
Practitioner Regulation Agency (Ahpra) pandemic sub-register.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Who can health          |
|                                   | professionals contact about their |
|                                   | Ahpra pandemic sub-register       |
|                                   | status?                           |
|                                   |                                   |
|                                   | Answer: Health professionals      |
|                                   | should contact Ahpra.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Is a health             |
|                                   | professional on the Ahpra         |
|                                   | pandemic sub-register?            |
|                                   |                                   |
|                                   | Answer: Search Ahpraâ€™s public     |
|                                   | register of practitioners.        |
|                                   |                                   |
|                                   | Health professionals will have a  |
|                                   | green flag and a notation on      |
|                                   | their Ahpra registration to       |
|                                   | indicate they are on the          |
|                                   | sub-register.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: What is the impact on   |
|                                   | Medicare?                         |
|                                   |                                   |
|                                   | Answer: Health professionals in   |
|                                   | the sub-register:                 |
|                                   |                                   |
|                                   | -   will have their registration  |
|                                   |     with Medicare re-instated     |
|                                   |     which may enable them to      |
|                                   |     access Medicare benefits      |
|                                   | -   may be able to use their      |
|                                   |     existing open Medicare        |
|                                   |     provider numbers              |
|                                   |                                   |
|                                   | Under locum tenens provisions,    |
|                                   | eligible health professionals can |
|                                   | use one of their existing         |
|                                   | provider numbers at a different   |
|                                   | location for up to 2 weeks.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: What can be managed in  |
|                                   | Health Professional Online        |
|                                   | Services (HPOS)?                  |
|                                   |                                   |
|                                   | Answer: Service Officers should   |
|                                   | encourage unrestricted health     |
|                                   | professionals to manage their     |
|                                   | Medicare provider numbers online  |
|                                   | via HPOS where they can:          |
|                                   |                                   |
|                                   | -   view the status of their      |
|                                   |     Medicare provider numbers     |
|                                   | -   open closed locations         |
|                                   | -   create a Medicare provider    |
|                                   |     number for new locations      |
|                                   | -   update bank details           |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: How are Ahpra           |
|                                   | sub-register access issues        |
|                                   | escalated?                        |
|                                   |                                   |
|                                   | Answer: Escalate access issues to |
|                                   | a Programme Support Officer (PSO) |
|                                   | if the health professional is:    |
|                                   |                                   |
|                                   | -   on the Ahpra sub-register,    |
|                                   |     and                           |
|                                   | -   their Ahpra registration is   |
|                                   |     valid                         |
