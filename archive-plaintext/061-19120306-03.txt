Customer contact regarding Proof of Life Certificate (AUS061)

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer contacts + Read more ... |
|                                   |                                   |
|                                   | Before releasing any information, |
|                                   | it is imperative that the         |
|                                   | customer is authenticated as the  |
|                                   | person to whom the information    |
|                                   | relates or a person the customer  |
|                                   | authorised to receive the         |
|                                   | information. For more             |
|                                   | information, see Authenticating a |
|                                   | Centrelink customer.              |
|                                   |                                   |
|                                   | This must include checking the    |
|                                   | customer's current address and    |
|                                   | telephone number.                 |
|                                   |                                   |
|                                   | When it is established that a     |
|                                   | customer's address or telephone   |
|                                   | number must be updated, then all  |
|                                   | current addresses (including      |
|                                   | postal address) and telephone     |
|                                   | details must be checked and       |
|                                   | updated via the Change in Contact |
|                                   | Details workflow.                 |
|                                   |                                   |
|                                   | If the customer has an            |
|                                   | interpreter indicator and an      |
|                                   | interpreter is required, refer to |
|                                   | Guidelines for using interpreters |
|                                   | and Booking an Interpreter and    |
|                                   | ensure the interaction and action |
|                                   | taken regarding engaging the      |
|                                   | services of an interpreter are    |
|                                   | recorded.                         |
|                                   |                                   |
|                                   | Customers contacting from         |
|                                   | overseas via the published        |
|                                   | international contact number will |
|                                   | initially reach the Centrelink    |
|                                   | International Services (CIS)      |
|                                   | Smart Centre. Customer general    |
|                                   | enquiries, unrelated to the Life  |
|                                   | Certificate and Proof of Life     |
|                                   | (POL) intervention, should be     |
|                                   | answered and issues resolved at   |
|                                   | the first contact. Service        |
|                                   | Officers should action unrelated  |
|                                   | POL enquiries and provide basic   |
|                                   | background information regarding  |
|                                   | Proof of Life certificates.       |
|                                   |                                   |
|                                   | Is the customer/third party       |
|                                   | contacting about a life           |
|                                   | certificate?                      |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   for their Chilean pension, go |
|                                   |     to Step 2                     |
|                                   | -   for other foreign pensions    |
|                                   |     see Assisting customers to    |
|                                   |     maintain an existing foreign  |
|                                   |     pension                       |
|                                   | -   issued for a Proof of Life    |
|                                   |     intervention, go to Step 3    |
|                                   |                                   |
|                                   | No, procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Life certificates for Chilean     |
|                                   | pensioners in Australia + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Give the customer a Proof of Life |
|                                   | Certificate 'Certificado de       |
|                                   | Supervivencia' (AUS061ES) to      |
|                                   | complete. The Resources page      |
|                                   | contains a link to the form.      |
|                                   |                                   |
|                                   | When the AUS061ES has been        |
|                                   | returned check that the form has  |
|                                   | been completed.                   |
|                                   |                                   |
|                                   | Send the form via internal mail   |
|                                   | to Centrelink International       |
|                                   | Services (CIS).                   |
|                                   |                                   |
|                                   | CIS staff are to complete the -   |
|                                   | Australia/Chile Agreement on      |
|                                   | Social Security - Liaison form    |
|                                   | (AUS187CL) and send with the      |
|                                   | AUS061ES to Superintendencia de   |
|                                   | Pensiones                         |
|                                   |                                   |
|                                   | For more information see:         |
|                                   |                                   |
|                                   | -   Chile Agreement and foreign   |
|                                   |     pension information           |
|                                   | -   Assisting customers to        |
|                                   |     maintain an existing foreign  |
|                                   |     pension                       |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Proof of life intervention        |
|                                   | enquiries + Read more ...         |
|                                   |                                   |
|                                   | Advise the customer of the        |
|                                   | purpose of the intervention:      |
|                                   |                                   |
|                                   | 'Thank you for contacting us      |
|                                   | today. As the letter states, we   |
|                                   | need to make sure you are         |
|                                   | receiving the correct amount of   |
|                                   | payment and we need to review     |
|                                   | your eligibility. As part of this |
|                                   | review, please complete the Proof |
|                                   | of Life Certificate and have it   |
|                                   | verified by one of the authorised |
|                                   | persons named on the              |
|                                   | certificate.â€™                     |
|                                   |                                   |
|                                   | POL enquires:                     |
|                                   |                                   |
|                                   | -   Customer needs help to        |
|                                   |     complete the form - Staff can |
|                                   |     use the form to assist        |
|                                   |     customer, see the Resources   |
|                                   |     page for a link to the        |
|                                   |     AUS061. Procedure ends here   |
|                                   | -   Form can't be returned by due |
|                                   |     date - If the customer is     |
|                                   |     unable to complete and return |
|                                   |     the form by the due date,     |
|                                   |     transfer the customer to the  |
|                                   |     Compliance team for an        |
|                                   |     extension to lodge form to    |
|                                   |     prevent unnecessary reminder  |
|                                   |     letters or suspension. Go to  |
|                                   |     Step 4                        |
|                                   | -   If an AUS061 is returned to a |
|                                   |     service centre, the Service   |
|                                   |     Officer must scan the         |
|                                   |     document. Procedure ends here |
|                                   | -   Customer contacts in relation |
|                                   |     to suspension or cancellation |
|                                   |     of their payment due to not   |
|                                   |     lodging the AUS061. Go to     |
|                                   |     Step 4                        |
|                                   | -   Customer has returned to      |
|                                   |     Australia, go to Step 4       |
|                                   | -   Customer deceased, go to Step |
|                                   |     5                             |
+-----------------------------------+-----------------------------------+

|                                   | Compliance team + Read more ...   |

|                                   | Warm transfer via softphone. Use  |
|                                   | the Proof of Life transfer button |
|                                   | under Compliance.                 |

|                                   | deceased.                         |
