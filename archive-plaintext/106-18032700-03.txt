| 6                                 | Subsequent contacts to support a  |
|                                   | claim for a vulnerable customer + |

|                                   |                                   |
|                                   | If a customer cannot submit their |
|                                   | claim for a payment or concession |
|                                   | card within 14 days of their      |
|                                   | first contact, they may be        |
|                                   | eligible for an extension of time |
|                                   | to submit. A customer may be      |
|                                   | granted an extension of up to 13  |
|                                   | weeks to submit their claim from  |
|                                   | the date of the first contact for |
|                                   | a medical condition or due to     |
|                                   | special circumstances.            |
|                                   |                                   |
|                                   | Extensions are granted based on   |
|                                   | the customer's individual         |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Medical condition extensions must |
|                                   | be supported by a medical         |
|                                   | certificate. The customer must    |
|                                   | submit a medical certificate as   |
|                                   | evidence to support the extension |
|                                   | when submitting their claim. The  |
|                                   | medical certificate is to confirm |
|                                   | that the customer, or their       |
|                                   | partner, or the person for whom   |
|                                   | they provide care, had a medical  |
|                                   | condition which:                  |
|                                   |                                   |
|                                   | -   was continuous between the    |
|                                   |     day of the contact and the    |
|                                   |     day the claim is submitted,   |
|                                   |     and                           |
|                                   | -   had a significant adverse     |
|                                   |     effect on the ability to      |
|                                   |     submit the claim within 14    |
|                                   |     days after the contact date   |
|                                   |                                   |
|                                   | Special circumstances can         |
|                                   | include, but are not limited to:  |
|                                   |                                   |
|                                   | -   A customer having practical   |
|                                   |     difficulties in meeting the   |
|                                   |     14 day time frame as a result |
|                                   |     of living in a remote area,   |
|                                   |     or                            |
|                                   | -   A customer residing in a      |
|                                   |     country that Australia has a  |
|                                   |     social security agreement     |
|                                   |     with, and there are reasons   |
|                                   |     why the customer could not    |
|                                   |     submit within the 14 day      |
|                                   |     period, or                    |
|                                   | -   An emergency interfering with |
|                                   |     a customer's plans to submit  |
|                                   |     within the 14 day period,     |
|                                   |     such as a natural disaster    |
|                                   |                                   |
|                                   | When an extension is granted:     |
|                                   |                                   |
|                                   | Tell the customer:                |
|                                   |                                   |

|                                   |     provided with their claim to  |
|                                   |     support a medical condition   |
|                                   |     extension                     |
|                                   | -   they must submit their claim  |
|                                   |     by the end of the extension   |
|                                   |     period, to be assessed from   |
|                                   |     the first date of contact     |
|                                   |                                   |
|                                   | Record the contact with the       |
|                                   | customer by creating a closed DOC |
|                                   | using the Fast Note - select Auto |
|                                   | Text, use Generic > Claims > ITC  |
|                                   | vulnerable customer extension:    |
|                                   |                                   |
|                                   | -   Select (tick) Display on      |
|                                   |     access box                    |
|                                   | -   Select Confirm                |
|                                   | -   DOA information will present  |
|                                   |     for example: Reason SCC –     |
|                                   |     Special Contact               |
|                                   |     Considerations and Expiry     |
|                                   |     Date equal to 182 days from   |
|                                   |     date of DOC                   |
|                                   | -   Select Confirm                |
|                                   | -   Edit auto text. Note: when    |
|                                   |     recording the vulnerable      |
|                                   |     circumstance do not include   |
|                                   |     details of a sensitive or     |
|                                   |     personal nature, including    |
|                                   |     medical abbreviations. See    |
|                                   |     Online Document Recording     |
|                                   |     (ODR)                         |
|                                   | -   Select Continue               |
|                                   |                                   |
|                                   | Expire any previous current DOA   |
|                                   | DOCs on the customer record       |
|                                   | relating to this intended claim   |
|                                   |                                   |
|                                   | When an extension is not granted, |
|                                   | record the reason(s) in a DOC and |
|                                   | tell the customer they must       |
|                                   | submit their claim as soon as     |
|                                   | possible to be paid from the      |
|                                   | earliest date.                    |
