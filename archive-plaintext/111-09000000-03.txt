Customer Record - Medicare

Table 2: This table contains information about how Medicare staff access
Customer Record from Customer First - FOH, features in Customer Record
and how to log off Customer Record.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Accessing Customer Record + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To access Customer Record:        |
|                                   |                                   |
|                                   | -   Launch the Customer First -   |
|                                   |     FOH desktop icon              |
|                                   | -   Select Medicare from the task |
|                                   |     bar at the top of the screen  |
|                                   | -   Select the                    |
|                                   |     ZCFS_CSO_MCA-Customer First   |
|                                   |     MedicareCSO Role business     |
|                                   |     role if prompted. This will   |
|                                   |     launch the Customer First     |
|                                   |     Workspace in a new window     |
|                                   |     behind                        |
|                                   | -   Select the new window         |
|                                   | -   Select Customer Record from   |
|                                   |     the left hand menu            |
|                                   | -   Enter OPOQ in the first       |
|                                   |     editable field                |
|                                   | -   Select ENTER from the numeric |
|                                   |     keypad. This will switch      |
|                                   |     Customer Record to payment    |
|                                   |     mode                          |
|                                   |                                   |
|                                   | Standard keyboard mapping is used |
|                                   | by Customer Record. See the       |
|                                   | Resources page for a printable    |
|                                   | keyboard legend.                  |
|                                   |                                   |
|                                   | To view additional features, go   |
|                                   | to Step 2.                        |
|                                   |                                   |
|                                   | To log off, go to Step 3.         |
+-----------------------------------+-----------------------------------+
| 2                                 | Features of Customer Record +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Features of Customer Record       |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   The Medicare Mainframe ruler  |
|                                   |     line has been replaced with a |
|                                   |     striped page. Use             |
|                                   |     CTRL+Shift+A to turn the      |
|                                   |     striped page on or off        |
|                                   | -   The display size can be       |
|                                   |     changed by using CTRL+ mouse  |
|                                   |     scroll                        |
|                                   | -   Select CTRL+N from an         |
|                                   |     existing Customer First       |
|                                   |     window to launch an identical |
|                                   |     window. Note: only one        |
|                                   |     session of Customer Record    |
|                                   |     can be open at any time       |
|                                   | -   Overwrite and insert          |
|                                   |     functions are available. The  |
|                                   |     option in use is displayed in |
|                                   |     the lower right corner of the |
|                                   |     screen as OVR for overwrite   |
|                                   |     or INS for insert             |
|                                   | -   Select the END key to swap    |
|                                   |     between OVR and INS options   |
+-----------------------------------+-----------------------------------+
| 3                                 | Logging off Customer Record +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Clear out of customer record.     |
|                                   |                                   |
|                                   | Enter CESF and select ENTER from  |
|                                   | the numeric keypad.               |
|                                   |                                   |
|                                   | Select the close button           |
|                                   | [/pages/Separated-parents\ima     |
|                                   | ges\icon-button-close-window.PNG] |
|                                   | in the top right corner.          |
+-----------------------------------+-----------------------------------+
