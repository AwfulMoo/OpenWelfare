This page contains information on how to assist a customer when payment
summary information is not available or incomplete in the ATO myTax
service.

Customer advises payment summary information is not available

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | A customer advises their          |
|                                   | information is not available in   |
|                                   | the ATO myTax service + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   The original Centrelink PAYG  |
|                                   |     (Pay As You Go) payment       |
|                                   |     summary information is sent   |
|                                   |     to the ATO for the financial  |
|                                   |     year just ended in early July |
|                                   | -   Amended data for the          |
|                                   |     financial year that has just  |
|                                   |     ended is sent to the ATO each |
|                                   |     week, commencing from the     |
|                                   |     second Wednesday in July      |
|                                   |                                   |
|                                   | Has the current financial year    |
|                                   | ended?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, advise customer Payment   |
|                                   |     Summary information has not   |
|                                   |     been finalised and            |

|                                   |     original Payment Summary will |

|                                   |     Process ends here             |
|                                   |                                   |
|                                   | Note: for a financial year/s      |
|                                   | before the latest completed       |
|                                   | financial year, the amended       |
|                                   | payment summary is sent to the    |
|                                   | customer but not to the ATO.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Was a payment summary             |
|                                   | generated? + Read more ...        |
|                                   |                                   |
|                                   | Go to the Tax Payment Summary     |

|                                   | relevant payment summary.         |
|                                   |                                   |
|                                   | Is the payment summary available? |
|                                   |                                   |
|                                   | -   No, go to Step 3              |
|                                   | -   Yes, go to Step 4             |
+-----------------------------------+-----------------------------------+
| 3                                 | Did the customer receive a        |
|                                   | reportable payment? + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Go to the Tax Year to date        |
|                                   | (TXYTD) screen and check the      |
|                                   | payment/s the customer received   |
|                                   | during the relevant financial     |
|                                   | year.                             |
|                                   |                                   |
|                                   | Is the payment reportable? Note:  |
|                                   | for more information, see Taxable |
|                                   | and non-taxable (tax exempt)      |
|                                   | payments for Centrelink Payment   |
|                                   | Summary                           |
|                                   |                                   |
|                                   | Yes, raise a Service incident     |
|                                   | through the ICT Service Desk via  |
|                                   | mySupport, include reportable     |
|                                   | payment information and periods   |
|                                   | and outline identified issue      |
|                                   |                                   |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | No, inform the customer a payment |
|                                   | summary will not be issued, as    |
|                                   | they did not receive a reportable |
|                                   | payment.                          |
|                                   |                                   |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Is the customerâ€™s information     |

|                                   |                                   |
|                                   | Centrelink and ATO require the    |
|                                   | same customer information to be   |
|                                   | recorded on both systems to be    |
|                                   | able to match the customer        |
|                                   | between departments.              |
|                                   |                                   |
|                                   | Confirm with the customer the     |
|                                   | correct information is recorded   |
|                                   | on the system, including:         |
|                                   |                                   |
|                                   | -   Tax File Number (TFN)         |
|                                   | -   Name                          |
|                                   | -   Address                       |
|                                   |                                   |
|                                   | Is the information correct?       |
|                                   |                                   |
|                                   | Yes, check the following:         |
|                                   |                                   |
|                                   | -   Whether an error has occurred |
|                                   |     during the calculation of a   |
|                                   |     customers' Centrelink payment |
|                                   |     summary. See General          |
|                                   |     Information on Manual         |
|                                   |     Follow-up (MFU) tax           |
|                                   |     activities Procedure ends     |
|                                   |     here                          |
|                                   | -   Whether the payment summary   |
|                                   |     is visible and has not been   |
|                                   |     issued. See Issuing payment   |
|                                   |     summaries. Procedure ends     |
|                                   |     here                          |
|                                   |                                   |
|                                   | No, advise the customer the       |
|                                   | incorrect information was the     |
|                                   | reason their information has not  |
|                                   | been pre-filled in myTax. Once    |
|                                   | this information is updated,      |
|                                   | reissue the Centrelink payment    |
|                                   | summary                           |
|                                   |                                   |
|                                   | -   Procedure ends here           |
