Uploading Earnings Apportionment Tool (EAT) calculations sheets

+-----------------------------------+-----------------------------------+
| Uploading EAT calculations sheets | All EAT calculation sheets used   |
|                                   | must be saved as an .XLSX file    |
|                                   | for them to be uploaded to the    |
|                                   | customer record.                  |
|                                   |                                   |
|                                   | To save the EAT as an .XLSX file: |
|                                   |                                   |
|                                   | The Compliance Officer (CO) must  |
|                                   | save the file to their H Drive as |
|                                   | an 'Excel workbook' rather than   |
|                                   | an 'Excel Macro-Enabled           |
|                                   | Workbook'. To do this, select:    |
|                                   |                                   |
|                                   | -   File                          |
|                                   | -   Save As                       |
|                                   | -   Browse to locate the H Drive  |
|                                   | -   Change the Save as type to    |
|                                   |     Excel workbook to save the    |
|                                   |     file as an .XLSX file         |
|                                   |                                   |
|                                   | The sheets must be uploaded to    |
|                                   | the customer's record using the   |
|                                   | Upload Document function on the   |
|                                   | Customer First Tools screen.      |
|                                   |                                   |
|                                   | See Attaching electronic          |
|                                   | documents to a Centrelink         |
|                                   | customer's record using Document  |
|                                   | Tools in Customer First.          |
|                                   |                                   |
|                                   | All EAT calculation sheets must   |
|                                   | be uploaded using the reason code |
|                                   | DBTCAL.                           |
|                                   |                                   |
|                                   | After all calculation sheets have |
|                                   | been uploaded, the saved files    |
|                                   | must be deleted from their H      |
|                                   | drive.                            |
|                                   |                                   |
|                                   | For multiple employers, save each |
|                                   | calculation with the Customer     |
|                                   | Reference Number (CRN) and        |
|                                   | employer number as outlined in    |
|                                   | the example below:                |
|                                   |                                   |
|                                   | EAT sheet for:                    |
|                                   |                                   |
|                                   | -   1st employer save as -        |
|                                   |     123456789A01                  |
|                                   | -   2nd employer save as -        |
|                                   |     123456789A02                  |
+-----------------------------------+-----------------------------------+
| Deleting EAT calculation sheets   | If a decision is changed because  |
| if they are returned in Quality   | of the QOL checking process, COs  |
| Online (QOL)                      | must:                             |
|                                   |                                   |
|                                   | -   Upload the corrected          |
|                                   |     calculation sheet to the      |
|                                   |     customer record               |
|                                   | -   Arrange for the incorrect     |
|                                   |     calculation sheet to be       |
|                                   |     deleted by lodging an ECRM    |
|                                   |     Image Deletion Request form   |
|                                   |                                   |
|                                   | This ensures only the correct     |
|                                   | calculation sheet remains         |
|                                   | attached to the customer record.  |
|                                   | This is important as customers    |
|                                   | can access this information under |
|                                   | Freedom Of Information (FOI).     |
+-----------------------------------+-----------------------------------+
| Reassessment or Review of         | Where a calculation sheet needs   |
| decision                          | to be amended because of a        |
|                                   | reassessment or review of         |
|                                   | decision, the CO can:             |
|                                   |                                   |
|                                   | -   Copy information from the     |
|                                   |     calculation sheet,            |
|                                   |     (previously uploaded to the   |
|                                   |     customer record)              |
|                                   | -   Paste the required            |
|                                   |     information into a new Excel  |
|                                   |     Macro-Enabled Workbook        |
|                                   |                                   |
|                                   | Once the reassessment has been    |
|                                   | completed, the CO will need to    |
|                                   | upload the new calculation sheet  |
|                                   | to the customer record as         |
|                                   | outlined above.                   |
|                                   |                                   |
|                                   | The CO is not required to delete  |
|                                   | the original calculation sheet,   |
|                                   | as this is a new decision         |
|                                   | resulting from the reassessment.  |
