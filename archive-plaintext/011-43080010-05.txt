Examples of eligibility

| Eligibility type                  | Example                           |

| Eligible concessional individual  | Dylan, daughter Grace and partner |
|                                   | Sam register as a Medicare Safety |
|                                   | Net family. Dylan has a           |
|                                   | concession card, so is a          |
|                                   | concessional individual.          |
|                                   |                                   |
|                                   | Dylan is eligible for the         |
|                                   | Concessional Safety Net           |
|                                   | threshold.                        |
|                                   |                                   |
|                                   | Grace and Sam don’t have          |
|                                   | concession cards. This means Sam  |
|                                   | and Grace are not eligible for    |
|                                   | the Concessional Safety Net       |
|                                   | threshold.                        |
+-----------------------------------+-----------------------------------+
| Eligible concessional family      | Dylan, daughter Grace and partner |
|                                   | Sam register as a Medicare Safety |
|                                   | Net family. Dylan has a           |
|                                   | concession card, so is a          |
|                                   | concessional individual. Dylan’s  |
|                                   | dependant, Grace has a concession |
|                                   | card too.                         |
|                                   |                                   |
|                                   | Dylan and Grace form a            |
|                                   | concessional family and are       |
|                                   | eligible for the Concessional     |
|                                   | Safety Net threshold.             |
|                                   |                                   |
|                                   | Sam doesn’t have a concession     |
|                                   | card. This means Sam isn’t part   |
|                                   | of the concessional family and    |
|                                   | isn’t eligible for the            |
|                                   | Concessional Safety Net           |
|                                   | threshold.                        |
