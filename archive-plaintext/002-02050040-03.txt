Table 10: for BIR trained officers to process Reconciliation debt (FHA)
(RCN) raised before Royal Assent 24 July 2021.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Nil rate reconciliation debt +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers with a Nil rate         |
|                                   | pre-commencement debt will have   |
|                                   | the choice to 'trade' available   |
|                                   | clock days to reduce or waive     |
|                                   | their debt. Some customers may    |
|                                   | choose to continue to receive FHA |
|                                   | rather than reduce their debt by  |
|                                   | trading days.                     |
|                                   |                                   |
|                                   | This applies if the debt has been |
|                                   | repaid in full or partially       |
|                                   | repaid.                           |
|                                   |                                   |
|                                   | Customers can choose to have      |
|                                   | their:                            |
|                                   |                                   |
|                                   | -   BIR debt fully waived (if     |
|                                   |     they have enough available    |
|                                   |     days to trade)                |
|                                   | -   BIR debt partially waived (by |
|                                   |     trading some of their         |
|                                   |     available days), or           |
|                                   | -   No debt waiver applied and    |
|                                   |     will retain the days          |
|                                   |     available on their clock      |
|                                   |                                   |
|                                   | The trade calculation is a        |
|                                   | standard $45 per day regardless   |
|                                   | of rate of payment received. This |
|                                   | means for every clock day a       |
|                                   | customer chooses to trade in      |
|                                   | against their existing RCN debt,  |
|                                   | the debt balance will reduce by   |
|                                   | $45. For example, $900 debt       |
|                                   | divided by $45 equals 20 clock    |
|                                   | days.                             |

|                                   | customer reduces the original     |

|                                   | over recovered debt amounts will  |
|                                   | be refunded to the customer.      |
|                                   |                                   |
|                                   | If the customer chooses not to    |
|                                   | trade clock days or does not have |
|                                   | enough clock days available to    |
|                                   | waive the debt, they must pay any |
|                                   | outstanding balance.              |
