| 1                                 | NEF assessments + Read more ...   |
|                                   |                                   |
|                                   | Pre-birth and post-birth PPL      |
|                                   | claims can be made NEF.           |
|                                   |                                   |
|                                   | In most cases, the NEF will occur |
|                                   | automatically when processing the |
|                                   | claim. However, manual coding can |
|                                   | be completed in certain           |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Automatic NEF assessments         |
|                                   |                                   |
|                                   | For PPL pre-claims that have been |
|                                   | assessed and are awaiting proof   |
|                                   | of the child's birth or entry to  |
|                                   | care, a reminder letter will be   |
|                                   | sent 14 days after the nominated  |
|                                   | start date to provide the proof.  |
|                                   |                                   |
|                                   | If the proof is not provided      |
|                                   | within 56 days of the nominated   |
|                                   | start date, the claim will be     |
|                                   | automatically assessed as NEF.    |
|                                   |                                   |
|                                   | PPL already claimed by customer   |
|                                   |                                   |
|                                   | For duplicate PPL claims lodged   |
|                                   | for children born or adopted on   |
|                                   | or after 1 July 2020 where the    |
|                                   | full 18 weeks was not paid, any   |
|                                   | subsequent claims will            |
|                                   | automatically assess as Not       |
|                                   | Effective (NEF) when processed.   |
|                                   |                                   |
|                                   | Claims will be assessed as        |
|                                   | NEF-PAC (PPL already claimed by   |
|                                   | customer), if the customer’s full |
|                                   | PPL entitlement has not been      |
|                                   | paid.                             |
|                                   |                                   |
|                                   | For duplicate PPL claims lodged   |
|                                   | for children born or adopted on   |
|                                   | or after 1 July 2020 where the    |
|                                   | full 18 weeks was claimed, any    |
|                                   | subsequent claims will            |
|                                   | automatically assess as rejected  |
|                                   | REJ-PAS (PPL already paid to      |
|                                   | customer) when processed.         |
|                                   |                                   |
|                                   | This includes claims cancelled at |
|                                   | the customer’s request (CAN-CRQ). |
|                                   | Note: if a customer’s original    |
|                                   | claim was CAN-CRQ, the duplicate  |
|                                   | claim will automatically NEF.     |
|                                   |                                   |
|                                   | Note: if the customer has         |
|                                   | previously requested their        |
|                                   | initial PPL claim for the child   |
|                                   | be cancelled CRQ, this decision   |
|                                   | cannot be appealed. See           |
|                                   | Cancellation of Parental Leave    |
|                                   | Pay (PPL).                        |
|                                   |                                   |
|                                   | If the customer advises the CRQ   |
|                                   | cancellation is incorrect,        |
|                                   | consider if the payment should be |
|                                   | restored. For more information,   |
|                                   | see Restoration of Parental Leave |
|                                   | Pay (PPL).                        |
|                                   |                                   |
|                                   | For duplicate claims lodged for   |
|                                   | children born or adopted before 1 |
|                                   | July 2020, complete a manual NEF  |
|                                   | assessment. Go to Step 4.         |
|                                   |                                   |
|                                   | Customer contact                  |
|                                   |                                   |
|                                   | If the customer contacts after    |
|                                   | their claim was made NEF, go to   |
|                                   | Step 5.                           |
