Example of viewing withholdings from a CCS entitlement

Table 1

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Padma's child Ben attends after   |
|                                   | school care at Provider 1 with a  |
|                                   | weekly CCS entitlement of $85.60, |
|                                   | Padma's other child Byron attends |
|                                   | long day care at Provider 2 with  |
|                                   | a weekly CCS entitlement of       |
|                                   | $322.02.                          |
|                                   |                                   |
|                                   | The total CCS entitlement to the  |
|                                   | family is $407.62.                |
|                                   |                                   |
|                                   | Padma's CCS Recent Payments shows |
|                                   | 2 amounts for the relevant week:  |
|                                   |                                   |
|                                   | -   28 Oct 2019 - 03 Nov 2019     |
|                                   |     Child Care Subsidy Provider   |
|                                   |     05 Nov 2019 $330.17           |
|                                   | -   28 Oct 2019 - 03 Nov 2019     |
|                                   |     Child Care Subsidy Provider   |
|                                   |     05 Nov 2019 $77.45            |
|                                   |                                   |
|                                   | The total for the week was        |
|                                   | $407.62. ($330.17 + $77.45). The  |
|                                   | $77.45 is the 20% withholdings    |
|                                   | (the lower amount).               |
|                                   |                                   |
|                                   | This means:                       |
|                                   |                                   |
|                                   | -   5% ($20.38) is withheld to    |
|                                   |     offset any future potential   |
|                                   |     debt. ($407.62 - $20.38 =     |
|                                   |     $387.24)                      |
|                                   | -   20% of the remaining $387.24  |
|                                   |     is withheld to repay the      |
|                                   |     current debt, this is $77.45  |
|                                   |                                   |
|                                   | The amount of $77.45 will be      |
|                                   | deducted from the CCS payment to  |
|                                   | both providers.                   |
|                                   |                                   |
|                                   | Customers will need to discuss    |
|                                   | amounts paid with their provider. |
+-----------------------------------+-----------------------------------+
