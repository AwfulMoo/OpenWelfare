This page contains guidance on when Services Australia can accept
information from the parents of Youth Allowance (YA) customers.

Advice from parents of a YA customer

Accepting information from parents

This table describes the steps for determining whether information can
be accepted from parents of a YA customer.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Obtain identification from        |
|                                   | parent + Read more ...            |
|                                   |                                   |
|                                   | Ask the parent to identify        |
|                                   | themselves by supplying their     |
|                                   | name and contact number.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Check alert DOC or written        |
|                                   | request on file + Read more ...   |
|                                   |                                   |
|                                   | Has appropriate authority been    |
|                                   | given by the customer for their   |
|                                   | parent to act on their behalf?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Implied authority be applied to   |
|                                   | accept information from the       |
|                                   | customer's parents + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Can implied authority be applied  |
|                                   | to accept information from the    |
|                                   | customer's parents?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer present to provide       |
|                                   | authority + Read more ...         |
|                                   |                                   |
|                                   | Is the customer present to        |
|                                   | provide authority for parents to  |
|                                   | supply information?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 5                                 | Parent provide the customer's     |
|                                   | name, current address, date of    |
|                                   | birth and Customer Reference      |
|                                   | Number (CRN) + Read more ...      |
|                                   |                                   |
|                                   | Can the parent provide the        |
|                                   | customer's name, current address, |
|                                   | date of birth and Customer        |
|                                   | Reference Number (CRN)?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 6                                 | Information accurate + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the information being supplied |
|                                   | accurate and is the parent        |
|                                   | supplying the information         |
|                                   | reliable and acting in the best   |
|                                   | interests of the customer?        |
|                                   |                                   |
|                                   | -   Yes, update details on record |
|                                   |     and advise the customer of    |
|                                   |     the changes in writing. Go to |
|                                   |     Step 8                        |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Do not make any changes to        |
|                                   | customer's record + Read more ... |
|                                   |                                   |
|                                   | Note: parents receiving payments  |
|                                   | into their account on behalf of   |
|                                   | the customer are permitted to     |
|                                   | update bank account details.      |
+-----------------------------------+-----------------------------------+
| 8                                 | Record details and discussion on  |
|                                   | a DOC with parent's name, contact |
|                                   | number and details of changes +   |
|                                   | Read more ...                     |
