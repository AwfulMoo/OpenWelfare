Date of effect rule - Changes of marital status

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Scenario                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer separates + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Omar and Amal have been partnered |
|                                   | since the beginning of the CCS    |
|                                   | year. Amal is claiming CCS for    |
|                                   | their children, which has been    |
|                                   | calculated on a combined income   |
|                                   | of $175,000. Amal contacts 17     |
|                                   | September 2019 to advise Amal and |
|                                   | Omar separated 13 September 2019. |
|                                   | The new CCS percentage based on   |
|                                   | Amal's income only takes effect   |
|                                   | from 23 September - the first CCS |
|                                   | Monday following the change in    |
|                                   | circumstance.                     |
|                                   |                                   |
|                                   | Where the change in marital       |
|                                   | status is the first Monday of a   |
|                                   | CCS fortnight, the change will    |
|                                   | take effect from that date.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer becomes partnered + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Gene has been single since the    |
|                                   | beginning of the CCS year. Gene   |
|                                   | has been receiving CCS at 85% for |
|                                   | 72 hours per fortnight.           |
|                                   |                                   |
|                                   | Gene contacts on 3 January 2020   |
|                                   | to advise Gene has become         |
|                                   | partnered from 10 October 2019.   |
|                                   | Gene's partner's income of        |
|                                   | $80,000 is added using a Date of  |
|                                   | Event of 10 October 2019 (the     |
|                                   | date of becoming partnered) and   |
|                                   | Paid Work of 80 hours per         |
|                                   | fortnight with a ATD Date of      |
|                                   | Receipt of 3 January 2020 (the    |
|                                   | date the customer notified of the |
|                                   | change).                          |
|                                   |                                   |
|                                   | The change in marital status and  |
|                                   | the newly combined family income  |
|                                   | estimate for CCS will take effect |
|                                   | from 21 October 2019 - the first  |
|                                   | Monday of the CCS fortnight       |
|                                   | following the change in           |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Due to late notification, the     |
|                                   | partner's Activity Test Result    |
|                                   | will only take effect from 16     |
|                                   | December 2019 - the earliest CCS  |
|                                   | Monday up to 28 days prior to     |
|                                   | date of notification. Gene would  |
|                                   | have nil hours of entitlement     |
|                                   | from 21 October to 15 December    |
|                                   | 2019 as there would be no         |
|                                   | activity recorded for the         |
|                                   | partner.                          |
|                                   |                                   |
|                                   | As a result of the late           |
|                                   | notification, an undetermined     |
|                                   | debt due to both income and       |
|                                   | Activity Test would be generated  |
|                                   | in the case.                      |
|                                   |                                   |
|                                   | Where the change in marital       |
|                                   | status is the first Monday of a   |
|                                   | CCS fortnight, the change will    |
|                                   | take effect from that date.       |
