Information on where to look for job vacancies

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Workforce Australia + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Workforce Australia website   |
|                                   | may be accessed using the self    |
|                                   | service terminals available for   |
|                                   | job seekers at most Service       |
|                                   | Centres. It can also be found on  |
|                                   | the Department of Employment and  |
|                                   | Workplace Relations website.      |
|                                   |                                   |
|                                   | Vacancies are listed by location, |
|                                   | occupation or seasonal work. A    |
|                                   | full list of Employment Services  |
|                                   | Providers, Workforce Australia -  |
|                                   | Work for the Dole activities and  |
|                                   | links to employment related sites |
|                                   | are available on this website.    |
|                                   | Job seekers can post their resume |
|                                   | on the Internet and the Workforce |
|                                   | Australia website will give daily |
|                                   | updates of jobs matching their    |
|                                   | resume as well as giving          |
|                                   | employers access to the resume.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Private Recruitment and           |
|                                   | Employment Agencies + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Yellow Pages lists Private    |
|                                   | Recruitment and Employment        |
|                                   | Agencies with which job seekers   |
|                                   | can register. The Yellow Pages    |
|                                   | can also be used to find contact  |
|                                   | details of businesses,            |
|                                   | associations and industries which |
|                                   | job seekers can use to seek work. |
|                                   | These contact details can be      |
|                                   | useful when cold canvassing.      |
+-----------------------------------+-----------------------------------+
| 3                                 | Cold Canvassing + Read more ...   |
|                                   |                                   |
|                                   | Cold Canvassing can be done by    |
|                                   | writing to, telephoning or        |
|                                   | personally approaching employers. |
+-----------------------------------+-----------------------------------+
| 4                                 | Networking + Read more ...        |
|                                   |                                   |
|                                   | The job seeker's network of       |
|                                   | family members, friends,          |
|                                   | relatives, past work mates and    |
|                                   | past employers can be sources of  |
|                                   | job leads.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Union or Professional             |
|                                   | Associations + Read more ...      |
|                                   |                                   |
|                                   | Contacting Union or Professional  |
|                                   | Associations can assist a job     |
|                                   | seeker in finding out more about  |
|                                   | a particular occupation or        |
|                                   | industry. It may also be an       |
|                                   | opportunity for job seekers to    |
|                                   | establish networks within the     |
|                                   | industry as sources for job       |
|                                   | leads. The Yellow Pages can be    |
|                                   | used to find contact details of   |
|                                   | businesses, associations and      |
|                                   | industries which job seekers can  |
|                                   | use to seek work.                 |
+-----------------------------------+-----------------------------------+
