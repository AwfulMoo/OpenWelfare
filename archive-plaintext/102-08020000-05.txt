| 1                                 | Question: How do I identify       |
|                                   | changes on a Birth Certificate    |
|                                   | issued by the Victorian (VIC)     |
|                                   | Registry of Births Deaths and     |
|                                   | Marriages (RBDM)?                 |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   VIC RBDM advises that the     |
|                                   |     letter C is recorded after    |
|                                   |     the registration number on    |
|                                   |     the birth certificate         |
|                                   |     indicating a change of sex or |
|                                   |     a correction to the           |
|                                   |     certificate. This includes    |
|                                   |     the correction of any         |
|                                   |     spelling errors.              |
|                                   | -   VIC RBDM advises that the     |
|                                   |     letter N is recorded after    |
|                                   |     the registration number on    |
|                                   |     the birth certificate         |
|                                   |     indicating a name change. A   |
|                                   |     letter N is recorded for each |
|                                   |     name change so there can be   |
|                                   |     more than one letter N on any |
|                                   |     certificate. This has been in |
|                                   |     place since 1986.             |
|                                   |                                   |
|                                   | See Identity Documents A to Z for |
|                                   | Images of Birth, marriage and     |
|                                   | change of name certificates for   |
|                                   | VIC                               |
