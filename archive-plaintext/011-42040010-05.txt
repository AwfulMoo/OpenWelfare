View a patient's history screen for referral details

The LSDT column in the patient history screen in Mainframe shows the
expiry date of a referral. This is located next to the column where the
referral date is recorded.

The referral expiry date is recorded in the system as follows:

-   For a 3 month referral - 3 months from the date of the first
    appointment
-   For a 12 month referral - 12 months from the date of the first
    appointment
-   For an indefinite referral - 1259 or 311259

Common enquiries

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Can an attendance item  |
|                                   | be paid when the only purpose of  |
|                                   | the attendance is to issue a      |
|                                   | referral?                         |
|                                   |                                   |
|                                   | Answer: Yes. Where there is a     |
|                                   | personal attendance between the   |
|                                   | patient and a treating medical    |
|                                   | practitioner, the appropriate     |
|                                   | attendance item would apply.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: From when is a referral |
|                                   | valid?                            |
|                                   |                                   |
|                                   | Answer: A referral is valid from  |
|                                   | the date of the first service     |
|                                   | with the specialist or consultant |
|                                   | physician.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Can a patient see       |
|                                   | multiple health professionals in  |
|                                   | the same specialty with the same  |
|                                   | referral?                         |
|                                   |                                   |
|                                   | Answer: If a health professional  |
|                                   | has accepted a referral by        |
|                                   | attending to that patient, they   |
|                                   | are the only health professional  |
|                                   | who can see the patient under     |
|                                   | that referral unless the patient  |
|                                   | is seen under a locum             |
|                                   | arrangement.                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Why are claims          |
|                                   | rejecting with code 605 for       |
|                                   | ‘referral expired’ when the       |
|                                   | patient has an indefinite or      |
|                                   | other non-standard referral?      |
|                                   |                                   |
|                                   | Answer: If the wrong referral     |
|                                   | period is transmitted on the      |
|                                   | first claim using the referral,   |
|                                   | all future claims for any DOS     |
|                                   | after that initial end date will  |
|                                   | reject 605- referral expired.     |
|                                   |                                   |
|                                   | Contact a PSO if assistance is    |
|                                   | required to process a claim.      |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: What is the difference  |
|                                   | between a request and a referral? |
|                                   |                                   |
|                                   | Answer: A request is generally    |
|                                   | issued by a health professional   |
|                                   | to organise tests for diagnostic  |
|                                   | purposes. This can include        |
|                                   | ultrasounds, pathology services,  |
|                                   | or other tests.                   |
|                                   |                                   |
|                                   | A referral is issued by a health  |
|                                   | professional to organise          |
|                                   | specialist or consultant          |
|                                   | physician attention, specifically |
|                                   | regarding the treatment, care or  |
|                                   | ongoing management of the         |
|                                   | patient. Also, a referral may be  |
|                                   | issued in order for the patient   |
|                                   | to access allied health services  |
|                                   | under a specific program or       |
|                                   | initiative.                       |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: How many initial        |
|                                   | attendance items can a health     |
|                                   | professional claim using the same |
|                                   | referral?                         |
|                                   |                                   |
|                                   | Answer: One only. A referral is   |
|                                   | valid for the continuing          |
|                                   | management of a previously        |
|                                   | referred condition. Subsequent    |
|                                   | attendances would apply in this   |
|                                   | instance.                         |
+-----------------------------------+-----------------------------------+
| 7                                 | Question: Are the services        |
|                                   | provided by a locum acting on     |
|                                   | behalf of a specialist or         |
|                                   | consultant physician entitled to  |
|                                   | the same Medicare benefit rate?   |
|                                   |                                   |
|                                   | Answer: Yes, as long as the       |
|                                   | locum-tenen is recognised as a    |
|                                   | specialist or consultant          |
|                                   | physician within the same         |
|                                   | speciality.                       |
+-----------------------------------+-----------------------------------+
| 8                                 | Question: If a medical            |
|                                   | practitioner conducts an initial  |
|                                   | referred consultation with a      |
|                                   | patient but the locum-tenens      |
|                                   | performs subsequent services, is  |
|                                   | the locum eligible to claim an    |
|                                   | initial attendance item from the  |
|                                   | same referral?                    |
|                                   |                                   |
|                                   | Answer: No, if the initial        |
|                                   | attendance is performed by a      |
|                                   | treating specialist or consultant |
|                                   | physician, and their locum sees   |
|                                   | their patient for subsequent      |
|                                   | attendances, the follow up        |
|                                   | attendances would need to be      |
|                                   | claimed as subsequent attendances |
|                                   | only.                             |
+-----------------------------------+-----------------------------------+
| 9                                 | Question: How long do health      |
|                                   | professionals have to keep        |
|                                   | records of referrals?             |
|                                   |                                   |
|                                   | Answer: 2 years                   |

Referral types/situations and how to manage them

Table 2

+-----------------------------------+-----------------------------------+
| Referral type/situation           | Validity and claim/account        |
|                                   | requirements                      |
+-----------------------------------+-----------------------------------+
| Emergency referrals               | A service that has occurred in an |
|                                   | emergency is valid for the        |
|                                   | initial attendance only.          |
|                                   |                                   |
|                                   | The health professional needs to  |
|                                   | obtain a referral for subsequent  |
|                                   | attendances for Medicare benefits |
|                                   | to be paid at the referred rate.  |
|                                   |                                   |
|                                   | They need to endorse              |
|                                   | claims/accounts with ‘Emergency   |
|                                   | Referral’ in order for the        |
|                                   | attendance item to be paid at the |
|                                   | referred rate.                    |
+-----------------------------------+-----------------------------------+
| Public patients within a public   | State and territory governments   |
| hospital                          | are responsible for the provision |
|                                   | of public hospital services.      |
|                                   | Services performed must be in     |
|                                   | accordance with the National      |
|                                   | Healthcare Agreement.             |
+-----------------------------------+-----------------------------------+
| Private patients within a public  | Where a medical practitioner      |
| or private hospital               | issues a referral during an       |
|                                   | episode of hospital treatment for |
|                                   | services provided or arranged by  |
|                                   | that hospital, Medicare benefits  |
|                                   | are paid at the referred rate for |
|                                   | the attendance item.              |
|                                   |                                   |
|                                   | Claims/accounts must be endorsed  |
|                                   | with ‘Referral within [name of    |
|                                   | hospital]’ and the patient’s      |
|                                   | hospital records must also show   |
|                                   | evidence of the referral          |
|                                   | (including the referring          |
|                                   | practitioner's signature).        |
|                                   |                                   |
|                                   | Where a medical practitioner      |
|                                   | working within a hospital refers  |
|                                   | a patient (for example,           |
|                                   | specialist or consultant          |
|                                   | physician within their private    |
|                                   | rooms) normal referral            |
|                                   | arrangements apply. This includes |
|                                   | the requirement for the treating  |
|                                   | practitioner to retain the        |
|                                   | referral letter.                  |
+-----------------------------------+-----------------------------------+
| Lost, stolen or destroyed         | A written referral that has been  |
| referrals                         | lost, stolen or destroyed is      |
|                                   | valid for the first attendance    |
|                                   | only.                             |
|                                   |                                   |
|                                   | Subsequent attendances will only  |
|                                   | attract a benefit at the referred |
|                                   | rate, where the specialist or     |
|                                   | consultant physician is in        |
|                                   | possession of a referral at the   |
|                                   | time of the attendance.           |
|                                   |                                   |
|                                   | Claims/accounts must show the     |
|                                   | name of the referring medical     |
|                                   | practitioner, practice address or |
|                                   | provider number (if the treating  |
|                                   | medical practitioner knows these) |
|                                   | and endorsed with ‘Lost           |
|                                   | Referral’.                        |
+-----------------------------------+-----------------------------------+
| Self-referral                     | Eligible medical practitioners    |
|                                   | can refer themselves to a         |
|                                   | consultant physician and/or       |
|                                   | specialist.                       |
|                                   |                                   |
|                                   | Medicare benefits apply at the    |
|                                   | referred rate for the attendance. |
+-----------------------------------+-----------------------------------+
| Locum referral                    | Where a locum is acting on behalf |
|                                   | of a specialist or consultant     |
|                                   | physician within the same         |
|                                   | speciality a new referral is not  |
|                                   | required.                         |
|                                   |                                   |
|                                   | Where the principle specialist or |
|                                   | consultant physician has seen a   |
|                                   | patient for an initial            |
|                                   | attendance, the locum             |
|                                   | practitioner cannot claim an      |
|                                   | additional initial attendance     |
|                                   | item. In this instance, Medicare  |
|                                   | benefits at the referred rate     |
|                                   | would only be payable for         |
|                                   | subsequent attendance items.      |
