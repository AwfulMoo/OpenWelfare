This procedure contains instruction for WA Ex-nuptial child support
cases only.

This page contains information about the process of negotiating debt
repayments for payee overpayment of WA ex-nuptial child support cases.
It also contains information about reviewing payment arrangement
defaults and contacting WA ex-nuptial payees to discuss defaults.

Child Support debt repayment

Negotiating repayment of a Child Support debt resulting from an overpayment

This table describes the process steps that Service Officers should
follow when contacting a payee to negotiate a repayment of a Child
Support debt for WA ex-nuptial child support cases.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Contact payee + Read more ...     |
|                                   |                                   |
|                                   | Contact the payee to advise:      |
|                                   |                                   |
|                                   | -   how the overpayment           |
|                                   |     occurred - see WA Ex-nuptial  |
|                                   |     child support cases - Child   |
|                                   |     Support overpayments and      |
|                                   |     other payee debts             |
|                                   | -   the amount of the overpayment |
|                                   | -   they are required to repay    |
|                                   |     the amount                    |
|                                   | -   possible impacts on Family    |
|                                   |     Tax Benefit payments, offer   |
|                                   |     to transfer them to           |
|                                   |     Centrelink after the payment  |
|                                   |     arrangement has been          |
|                                   |     finalised                     |
|                                   |                                   |
|                                   | See Contact with Child Support    |
|                                   | customers.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Negotiate debt repayment + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Negotiate repayment of the debt   |
|                                   | in line with the Debt repayment.  |
|                                   |                                   |
|                                   | The payment arrangement must give |
|                                   | consideration to:                 |
|                                   |                                   |
|                                   | -   the payee’s financial         |
|                                   |     resources in relation to      |
|                                   |     their caring responsibilities |
|                                   | -   the reason for the            |
|                                   |     overpayment, e.g. were the    |
|                                   |     circumstances leading to the  |
|                                   |     overpayment outside the       |
|                                   |     payee’s control               |
|                                   | -   if the overpayment has been   |
|                                   |     refunded to the payer from    |
|                                   |     consolidated revenue (as      |
|                                   |     refunding payments to the     |
|                                   |     payer is no longer a factor)  |
|                                   | -   not disadvantaging children   |
|                                   |     in the payee’s care by        |
|                                   |     unreasonably minimising       |
|                                   |     ongoing child support         |
|                                   |                                   |
|                                   | Depending on the individual       |
|                                   | circumstances this may result in  |
|                                   | an arrangement below the          |
|                                   | acceptable range using the debt   |
|                                   | repayment capacity calculator in  |
|                                   | the Collection and debt macro.    |
|                                   | Follow the process in the Debt    |
|                                   | repayment to determine an         |
|                                   | acceptable arrangement.           |
|                                   |                                   |
|                                   | If a payee has multiple cases and |
|                                   | in one of those cases they have a |
|                                   | role of a payer with outstanding  |
|                                   | child support:                    |
|                                   |                                   |
|                                   | -   make a payment arrangement    |
|                                   |     for the total debt            |
|                                   |     (outstanding child support    |
|                                   |     plus the overpayment)         |
|                                   | -   child support will be paid    |
|                                   |     before the overpayment        |
|                                   |                                   |
|                                   | If a payee has an overpayment in  |
|                                   | more than one child support case: |
|                                   |                                   |
|                                   | -   make a payment arrangement    |
|                                   |     for the total overpayment     |
|                                   | -   payments received will be     |
|                                   |     apportioned by the cases      |
|                                   |                                   |
|                                   | Document the discussion in the    |
|                                   | Payment Arrangement - Client      |
|                                   | window. See Documenting Child     |
|                                   | Support information.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Contact payer + Read more ...     |
|                                   |                                   |
|                                   | If a payment arrangement has been |
|                                   | negotiated to recover the         |
|                                   | overpayment, contact the payer    |
|                                   | and explain:                      |
|                                   |                                   |
|                                   | -   if the payee has negotiated   |
|                                   |     to repay the debt at 100% of  |
|                                   |     the future entitlement, the   |
|                                   |     payer can cease making        |
|                                   |     payments for the same         |
|                                   |     duration. EW must be stopped, |
|                                   |     see Employer Withholding      |
|                                   |     (linking) for Child Support   |
|                                   |     customers.                    |
|                                   | -   if the payee has negotiated   |
|                                   |     to repay the debt at less     |
|                                   |     than 100% of their            |
|                                   |     entitlement, and there is an  |
|                                   |     ongoing liability, the payer  |
|                                   |     will have to continue to pay  |
|                                   |     their ongoing liability. In   |
|                                   |     this circumstance, it will be |
|                                   |     appropriate to refund the     |
|                                   |     overpayment to the payer and  |
|                                   |     create a consolidated revenue |
|                                   |     debt. Note: because a payee   |
|                                   |     payment arrangement is keyed  |
|                                   |     as a percentage of the        |
|                                   |     disbursement, for the payee   |
|                                   |     to receive the agreed portion |
|                                   |     of their entitlement, the     |
|                                   |     full amount must be available |
|                                   |     for disbursement. If the      |
|                                   |     overpayment is not refunded   |
|                                   |     to the payer, they will have  |
|                                   |     to pay the full liability     |
|                                   |     every month which is not a    |
|                                   |     desirable option              |
|                                   | -   if there is no ongoing        |
|                                   |     liability the overpayment     |
|                                   |     will be refunded at the same  |
|                                   |     rate as it is recovered from  |
|                                   |     the payee                     |
|                                   |                                   |
|                                   | Note: the refund to the payer is  |
|                                   | not an automatic Cuba process.    |
|                                   | See Excess cash on a Child        |
|                                   | Support customer's account and    |
|                                   | Refunds Cuba Process Help.        |
|                                   |                                   |
|                                   | If the payer has an ongoing child |
|                                   | support liability and refuses to  |
|                                   | make payments for child support   |
|                                   | until the overpayment has been    |
|                                   | recovered, consider using funds   |
|                                   | from consolidated revenue. Where  |
|                                   | there is a history of             |
|                                   | non-compliance Child Support may  |
|                                   | credit the refund to the payer’s  |
|                                   | account to be used for future     |
|                                   | child support liabilities. See    |
|                                   | Creating a consolidated revenue   |
|                                   | debt.                             |
|                                   |                                   |
|                                   | Document the discussion with the  |
|                                   | payer in the payer Communication  |
|                                   | window using the category         |
|                                   | ‘collection’ and the issue        |
|                                   | ‘overpay/con rev’.                |
+-----------------------------------+-----------------------------------+
| 4                                 | Unlock customers + Read more ...  |
|                                   |                                   |
|                                   | Delete the Payee Overpayment      |
|                                   | intray and unlock the customers   |
|                                   | after the payment arrangement has |
|                                   | been entered and both customers   |
|                                   | have been contacted.              |
+-----------------------------------+-----------------------------------+

Payee payment arrangement defaults

Managing payee payment arrangement defaults

This table describes the process steps that Service Officers should
follow when managing payee payment arrangement defaults for WA
Ex-nuptial child support cases.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Payee default indicated + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a payee defaults on their      |
|                                   | payment arrangement a Payment     |
|                                   | Arrangement Default (Payee)       |
|                                   | intray generates and is actioned  |
|                                   | by the customer manager or        |
|                                   | business line.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Review payment arrangement        |
|                                   | default + Read more ...           |
|                                   |                                   |
|                                   | Before contacting the payee:      |
|                                   |                                   |
|                                   | -   check the Payment Arrangement |
|                                   |     window and notepad to confirm |
|                                   |     the amount and period are     |
|                                   |     correctly recorded            |
|                                   | -   review the payment history on |
|                                   |     the Payee Account Summary     |
|                                   |     screen to determine if        |
|                                   |     payment(s) have previously    |
|                                   |     been received in line with    |
|                                   |     the payment arrangement       |
|                                   | -   check whether the default is  |
|                                   |     because the payee they made a |
|                                   |     partial payment or made the   |
|                                   |     payment later than expected   |
|                                   |                                   |
|                                   | Use this information to inform    |
|                                   | the discussion with the payee.    |
|                                   |                                   |
|                                   | Go to Step 3.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Contact payee to discuss          |
|                                   | default + Read more ...           |
|                                   |                                   |
|                                   | Contact the payee and discuss the |
|                                   | following:                        |
|                                   |                                   |
|                                   | -   the reason for the payment    |
|                                   |     arrangement default           |
|                                   | -   the importance of contacting  |
|                                   |     before their payment is due   |
|                                   |     if they can't make their      |
|                                   |     payment                       |
|                                   | -   arrangements to make the      |
|                                   |     outstanding payment           |
|                                   |                                   |
|                                   | See Contact with Child Support    |
|                                   | customers.                        |
|                                   |                                   |
|                                   | Attempt to resolve issues that    |
|                                   | contributed to the default.       |
|                                   |                                   |
|                                   | Based upon the discussion with    |
|                                   | the payee, undertake one of the   |
|                                   | following actions:                |
|                                   |                                   |
|                                   | -   extend the payment            |
|                                   |     arrangement - when the        |
|                                   |     arrangement continues to be   |
|                                   |     appropriate but the payee is  |
|                                   |     unable to make the missed     |
|                                   |     payment/s before the next     |
|                                   |     payment is due, see Payment   |
|                                   |     arrangements Cuba Process     |
|                                   |     Help                          |
|                                   | -   negotiate a new payment       |
|                                   |     arrangement - when the        |
|                                   |     current arrangement is no     |
|                                   |     longer appropriate, see       |
|                                   |     Negotiating debt repayment    |
|                                   |     table                         |
|                                   | -   no change to the payment      |
|                                   |     arrangement - when the payee  |
|                                   |     advises they will make the    |
|                                   |     missed payments               |
|                                   |                                   |
|                                   | Document all contact attempts and |
|                                   | details of the discussion with    |
|                                   | the payee in the Payment          |
|                                   | Arrangement window. See           |
|                                   | Documenting Child Support         |
|                                   | information.                      |
