Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Locate match data + Read more ... |
|                                   |                                   |
|                                   | Check if there are any anomalies  |
|                                   | or mismatches with the match data |
|                                   | information.                      |
|                                   |                                   |
|                                   | Check if EER customer             |
|                                   |                                   |
|                                   | Use details listed on the         |
|                                   | Employment Income Summary (EANS)  |
|                                   | to check if:                      |
|                                   |                                   |
|                                   | -   the customer is employed with |
|                                   |     an Australian Disability      |
|                                   |     Enterprise (ADE) employer,    |
|                                   |     and                           |
|                                   | -   their employment income is    |
|                                   |     reported through the Employer |
|                                   |     Electronic Reporting (EER)    |
|                                   |     process                       |
|                                   |                                   |
|                                   | If both of the above apply, refer |
|                                   | the intervention to the CADO      |
|                                   | Service Desk. Procedure ends here |
|                                   |                                   |
|                                   | Check match data                  |
|                                   |                                   |
|                                   | Locate the details received from  |
|                                   | the ATO.                          |
|                                   |                                   |
|                                   | Go to the Match Data (IRMD)       |
|                                   | screen while in the review. Key   |
|                                   | 'IRMD' in the Nxt: field or key   |
|                                   | 'M' next to the review on the     |
|                                   | Review Summary (IRRS) screen.     |
|                                   |                                   |
|                                   | Interventions will have a keyword |
|                                   | of Tier 1 or Tier 2 added to the  |
|                                   | IRS review. This keyword will     |
|                                   | help identify the tiered amount   |
|                                   | of JobKeeper Payment received.    |
|                                   |                                   |
|                                   | Discuss the match data details    |
|                                   | with the customer to confirm a    |
|                                   | link between the customer and the |
|                                   | employer details listed in the    |
|                                   | match data.                       |
|                                   |                                   |
|                                   | Check if the:                     |
|                                   |                                   |
|                                   | -   customer is or was receiving  |
|                                   |     income from the matched       |
|                                   |     employer, (wage or salary     |
|                                   |     earner), or                   |
|                                   | -   match data relates to         |
|                                   |     self-employment. This could   |
|                                   |     include sole trader, member   |
|                                   |     of a partnership or through a |
|                                   |     private trust or private      |
|                                   |     company                       |
|                                   |                                   |
|                                   | If the customer says they do not  |
|                                   | know the employer or they have    |
|                                   | never worked for the matched      |
|                                   | employer, check if the customer   |
|                                   | may know the matched employer     |
|                                   | under another name, e.g. a        |
|                                   | trading name:                     |
|                                   |                                   |
|                                   | -   Check the ABN Lookup guide    |
|                                   |     for other trading names the   |
|                                   |     matched employer may be       |
|                                   |     trading under. This may help  |
|                                   |     the customer recognise the    |
|                                   |     match data                    |
|                                   | -   Review the customer record to |
|                                   |     see if there is any           |
|                                   |     information to support the    |
|                                   |     match data                    |
|                                   |                                   |
|                                   | When considering if a customer    |
|                                   | may be self-employed:             |
|                                   |                                   |
|                                   | -   check if the Legal            |
|                                   |     Name/Trading Name on the IRMD |
|                                   |     screen matches the customer's |
|                                   |     name, or the business address |
|                                   |     is the same as the customer's |
|                                   | -   some sole traders do not      |
|                                   |     register under a Trading Name |
|                                   |     and will have the match data  |
|                                   |     showing the ABN only. If the  |
|                                   |     details on the IRMD screen    |
|                                   |     have an ABN but no            |
|                                   |     Legal/Trading name, the       |
|                                   |     customer is likely to be a    |
|                                   |     sole trader                   |
|                                   |                                   |
|                                   | Is there a link between the       |
|                                   | customer and the matched          |
|                                   | employer?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, there has been a possible |
|                                   |     identity mismatch. See Step 1 |
|                                   |     in Table 3                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine employment              |
|                                   | circumstances + Read more ...     |
|                                   |                                   |
|                                   | JobKeeper Payment is assessed     |
|                                   | differently depending on the      |
|                                   | customer's employment             |
|                                   | circumstances. Use questioning    |
|                                   | skills and investigate the        |
|                                   | customer's record to determine    |
|                                   | their employment circumstances.   |
|                                   |                                   |
|                                   | Determine if the customer is      |
|                                   | employed as a:                    |
|                                   |                                   |
|                                   | wage/salary earner (Employee), or |
|                                   |                                   |
|                                   | self-employed as a:               |
|                                   |                                   |
|                                   | -   sole trader                   |
|                                   | -   member of a partnership, or   |
|                                   | -   private trust or private      |
|                                   |     company                       |
|                                   |                                   |
|                                   | If the customer has said they are |
|                                   | self-employed, use the following  |
|                                   | information to help decide the    |
|                                   | type of self-employment.          |
|                                   |                                   |
|                                   | Check the Match data:             |
|                                   |                                   |
|                                   | -   If the Trading Name displays  |
|                                   |     2 names (e.g. Jane Citizen    |
|                                   |     and John Citizen) this can    |
|                                   |     indicate that the customer is |
|                                   |     involved in a partnership     |
|                                   | -   Involvement in a private      |
|                                   |     trust can be identified by    |
|                                   |     the Legal Name or Trading     |
|                                   |     Names including the words     |
|                                   |     Family Trust, The Trustee     |
|                                   |     for, Unit Trust or As Trustee |
|                                   |     for (ATF), or                 |
|                                   | -   For a private company, the    |
|                                   |     Legal Name or Trading Name    |
|                                   |     can include the words         |
|                                   |     Proprietary Limited, PTY LTD  |
|                                   |     or P/L at the end of the name |
|                                   |                                   |
|                                   | Check the following screens to    |
|                                   | see what has been coded:          |
|                                   |                                   |
|                                   | For an employee, check EAN/EANS   |
|                                   | screen                            |
|                                   |                                   |
|                                   | For self-employed as:             |
|                                   |                                   |
|                                   | -   a sole trade, check Real      |
|                                   |     Estate and Business Summary   |
|                                   |     (REBS) screen                 |
|                                   | -   member of a business          |
|                                   |     partnership, check REBS       |
|                                   |     screen                        |
|                                   | -   private trust or private      |
|                                   |     company, check both the Link  |
|                                   |     Summary (LS) and              |
|                                   |     Trust/Company/Income/Assets   |
|                                   |     Summary (TRCIAS) screens      |
|                                   |                                   |
|                                   | Although a customer may run a     |
|                                   | business as a sole trader, they   |
|                                   | may be working as an employee and |
|                                   | not self-employed. For example:   |
|                                   |                                   |
|                                   | -   a contractor can be           |
|                                   |     considered self-employed or   |
|                                   |     an employee depending on      |
|                                   |     their circumstances           |
|                                   | -   if the customer receives      |
|                                   |     group certificates, they are  |
|                                   |     generally considered an       |
|                                   |     employee                      |
|                                   |                                   |
|                                   | The Decision Support Tool (DST)   |
|                                   | helps to decide the customer's    |
|                                   | employment status. Only use it    |
|                                   | for customers who identify as a   |
|                                   | sole trader or member of a        |
|                                   | partnership. Do not use it for    |
|                                   | customers involved in private     |
|                                   | trusts or private companies. The  |
|                                   | DST uses questions to help staff  |
|                                   | weigh up the facts and decide     |
|                                   | each case on its own merit.       |
|                                   |                                   |
|                                   | Record the employment             |
|                                   | circumstances decision on the     |
|                                   | intervention DOC. See the         |
|                                   | Resources page for the            |
|                                   | 'Self-employed/employee decision  |
|                                   | support tool question' DOC        |
|                                   | template.                         |

| 3                                 | Questions to determine scope      |
|                                   | period + Read more ...            |
|                                   |                                   |
|                                   | Establish the period the customer |
|                                   | received the JobKeeper Payment:   |
|                                   |                                   |
|                                   | -   Use the match data as a guide |
|                                   |     to start the conversation     |
|                                   |     with the customer             |
|                                   | -   Use questioning skills to     |
|                                   |     gather more information from  |
|                                   |     the customer about their      |
|                                   |     circumstances.                |
|                                   |                                   |
|                                   | Questions may include:            |
|                                   |                                   |
|                                   | -   When did your JobKeeper       |
|                                   |     Payments start/cease?         |
|                                   | -   Were you physically working   |
|                                   |     with your employer or just    |
|                                   |     getting JobKeeper Payment     |
|                                   |     income from them?             |
|                                   | -   Were you still receiving your |
|                                   |     regular wage before getting   |
