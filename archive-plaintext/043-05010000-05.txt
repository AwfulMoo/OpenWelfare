Terms used for BSPVES

Definition of terms exclusive to BSPVES

This table provides clarification of terms relevant for BSPVES.

+-----------------------------------+-----------------------------------+
| Terms                             | Description relevant for BSPVES   |
+-----------------------------------+-----------------------------------+
| Driver                            | A driver is an adult passenger    |
|                                   | the service operator nominates as |
|                                   | the driver of the passenger       |
|                                   | vehicle.                          |
+-----------------------------------+-----------------------------------+
| Eligible passenger                | An eligible passenger is:         |
|                                   |                                   |
|                                   | -   a person in an eligible       |
|                                   |     passenger vehicle on a Bass   |
|                                   |     Strait service, and           |
|                                   | -   can be the driver of the      |
|                                   |     passenger vehicle and/or an   |
|                                   |     eligible person               |
+-----------------------------------+-----------------------------------+
| Eligible person for King Island   | An eligible person for King       |
|                                   | Island travel is:                 |
|                                   |                                   |
|                                   | a driver who travels on a direct  |
|                                   | commercial airline service        |
|                                   | between:                          |
|                                   |                                   |
|                                   | -   King Island and a place on    |
|                                   |     the Australian mainland (a    |
|                                   |     northbound crossing), or      |
|                                   | -   a place on the Australian     |
|                                   |     mainland and King Island (a   |
|                                   |     southbound crossing)          |
|                                   |                                   |
|                                   | While their passenger vehicle is  |
|                                   | carried in the same direction of  |
|                                   | travel on a:                      |
|                                   |                                   |
|                                   | -   King Island service (on or    |
|                                   |     about the same day of         |
|                                   |     travel), and                  |
|                                   | -   there are no Bass Strait      |
|                                   |     services between those places |
|                                   |     where it is possible for the  |
|                                   |     passenger vehicle to be       |
|                                   |     accompanied by a driver       |
+-----------------------------------+-----------------------------------+
| Eligible person for Furneaux      | An eligible person is for         |
| Group                             | Furneaux Group is:                |
|                                   |                                   |
|                                   | a driver who travels on a direct  |
|                                   | commercial airline service on or  |
|                                   | after 1 January 2010:             |
|                                   |                                   |
|                                   | -   between an island of the      |
|                                   |     Furneaux Group and a place on |
|                                   |     the Australian mainland (a    |
|                                   |     northbound crossing), or      |
|                                   | -   between a place on the        |
|                                   |     Australian mainland and an    |
|                                   |     island of the Furneaux Group  |
|                                   |     (a southbound crossing)       |
|                                   |                                   |
|                                   | While their passenger vehicle is  |
|                                   | carried in the same direction of  |
|                                   | travel on a:                      |
|                                   |                                   |
|                                   | -   Furneaux Group service (on or |
|                                   |     about the same day of         |
|                                   |     travel), and                  |
|                                   | -   there are no Bass Strait      |
|                                   |     services between those places |
|                                   |     where it is possible for the  |
|                                   |     passenger vehicle to be       |
|                                   |     accompanied by a driver       |
+-----------------------------------+-----------------------------------+
| Eligible person with medical      | An eligible person with a medical |
| condition                         | condition is:                     |
|                                   |                                   |
|                                   | a driver, who has a medical       |
|                                   | condition at the time of travel,  |
|                                   | and who travels on a commercial   |
|                                   | airline service between:          |
|                                   |                                   |
|                                   | -   Melbourne and Devonport (a    |
|                                   |     southbound crossing) or       |
|                                   |     between Devonport and         |
|                                   |     Melbourne (a northbound       |
|                                   |     crossing)                     |
|                                   | -   the Australian mainland and   |
|                                   |     King Island (a southbound     |
|                                   |     crossing) or between King     |
|                                   |     Island and the Australian     |
|                                   |     mainland (a northbound        |
|                                   |     crossing), or                 |
|                                   | -   on or after 1 January 2010,   |
|                                   |     between the Australian        |
|                                   |     mainland and an island of the |
|                                   |     Furneaux Group (a southbound  |
|                                   |     crossing) or between an       |
|                                   |     island of the Furneaux Group  |
|                                   |     and the Australian mainland   |
|                                   |     (a northbound crossing)       |
|                                   |                                   |
|                                   | While their passenger vehicle is  |
|                                   | carried in the same direction of  |
|                                   | travel on a Bass Strait service   |
|                                   | on or about the same day of       |
|                                   | travel.                           |
+-----------------------------------+-----------------------------------+
| Eligible passenger vehicle        | Definitions vary for an eligible  |
|                                   | passenger vehicle depending on    |
|                                   | the service:                      |
|                                   |                                   |
|                                   | -   Bass Strait service - a       |
|                                   |     passenger vehicle which       |
|                                   |     accompanies an eligible       |
|                                   |     passenger on a Bass Strait    |
|                                   |     service                       |
|                                   | -   King Island service - the     |
|                                   |     passenger vehicle booked to   |
|                                   |     travel by an eligible person  |
|                                   | -   Furneaux Group service - the  |
|                                   |     passenger vehicle booked to   |
|                                   |     travel by an eligible person  |
+-----------------------------------+-----------------------------------+
| Medical condition                 | A medical condition:              |
|                                   |                                   |
|                                   | -   is a condition certified by a |
|                                   |     medical practitioner, and     |
|                                   | -   the condition certified       |
|                                   |     prevents a driver from        |
|                                   |     travelling comfortably by sea |
|                                   |     for medical reasons           |
+-----------------------------------+-----------------------------------+
| Rebate                            | A rebate is:                      |
|                                   |                                   |
|                                   | the amount applicable to an       |
|                                   | eligible passenger vehicle, and   |
|                                   |                                   |
|                                   | -   deducted from the vehicle     |
|                                   |     fare of an eligible passenger |
|                                   |     by a Service Operator; or     |
|                                   | -   paid to an eligible passenger |
|                                   |     by a Service Operator, or     |
|                                   | -   reimbursed directly to an     |
|                                   |     eligible passenger            |
|                                   |                                   |
|                                   | See Ministerial Directions for    |
|                                   | further explanation in Schedule   |
|                                   | 1, paragraph 7.3(a), 7.3(b) and   |
|                                   | clause 8.                         |
+-----------------------------------+-----------------------------------+
