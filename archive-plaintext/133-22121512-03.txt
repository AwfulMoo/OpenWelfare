|                                   |                                   |

|                                   |     displays                      |
|                                   |                                   |
|                                   | Select GPTP on the left hand side |

|                                   |                                   |
|                                   | Select Add from under the Action  |
|                                   | column                            |

| 3                                 | Add Banking details + Read more   |
|                                   | ...                               |

|                                   | The add banking details screen    |
|                                   | displays.                         |

|                                   | Location and Location ID fields   |

|                                   |     complete Account Details,     |
|                                   |     BSB, Account Number and       |
|                                   |     Account Name,                 |
|                                   |                                   |
|                                   | The user must complete the        |
|                                   | Acknowledgement by ticking the ‘I |
|                                   | acknowledge that’ box.            |
|                                   |                                   |
|                                   | Select Submit and save update.    |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Reset will return to          |
|                                   |     pre-entry details and user    |
|                                   |     can recommence entering new   |
|                                   |     details, or                   |
|                                   | -   Cancel will return user to    |
|                                   |     the banking details main page |
+-----------------------------------+-----------------------------------+

|                                   | ...                               |
|                                   |                                   |
|                                   | User will be returned to the home |
|                                   | banking details page.             |

|                                   | action taken with banking         |
|                                   | details.                          |
|                                   |                                   |

|                                   | display for the Supervisor or     |
|                                   | Registrar on the following day.   |
