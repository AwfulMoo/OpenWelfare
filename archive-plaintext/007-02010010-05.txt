Surrogacy scenarios

+-----------------------------------+-----------------------------------+
| Item                              | Birth mother claims PPL and is    |
|                                   | giving child up as part of        |
|                                   | surrogacy, and…                   |
+-----------------------------------+-----------------------------------+
| 1                                 | Claims 18 week continuous payment |
|                                   |                                   |
|                                   | Jamie gave birth to Marcus on 1   |
|                                   | February and lodges an effective  |
|                                   | PPL claim within 28 days of       |
|                                   | giving birth. Jamie:              |
|                                   |                                   |
|                                   | -   nominates the date of birth   |
|                                   |     as the PPL period start date, |
|                                   |     and                           |
|                                   | -   connects the 30 Flexible days |
|                                   |     to the end of the PPL period  |
|                                   |     (18 weeks continuous payment) |
|                                   |                                   |
|                                   | Jamie is eligible for PPL from 1  |
|                                   | February to 6 June. Jamie ceases  |
|                                   | to be eligible for PPL on 6 June, |
|                                   | when baby Marcus is 18 weeks old. |
|                                   |                                   |
|                                   | -   PPL period 1 February to 25   |
|                                   |     April                         |
|                                   | -   Connected period 26 April to  |
|                                   |     6 June                        |
|                                   |                                   |
|                                   | Jamie is eligible to receive 18   |
|                                   | weeks PPL as one continuous       |
|                                   | period.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Claims 18 week continuous payment |
|                                   |                                   |
|                                   | Kayla gives birth to Kevin on 1   |
|                                   | February and lodges an effective  |
|                                   | PPL claim on 7 March. As this is  |
|                                   | more than 28 days after Kevin’s   |
|                                   | birth, Kayla’s PPL period starts  |
|                                   | from the date the effective claim |
|                                   | and supporting documents are      |
|                                   | lodged. Kayla cannot be paid past |
|                                   | the 6 June as this is when baby   |
|                                   | Kevin is 18 weeks old.            |
|                                   |                                   |
|                                   | Kayla elects to be paid the PPL   |
|                                   | as a continuous payment.          |
|                                   |                                   |
|                                   | -   PPL period 7 March to 29 May  |
|                                   | -   Connected period 30 May to 6  |
|                                   |     June (5 payable week days)    |
|                                   |                                   |
|                                   | Kayla is paid 12 weeks PPL period |
|                                   | and 5 connected PPL days.         |
+-----------------------------------+-----------------------------------+
| 3                                 | Claims 12 week PPL Period and     |
|                                   | Flexible PPL days                 |
|                                   |                                   |
|                                   | Lucy gives birth to Amy on 1      |
|                                   | February and lodges an effective  |
|                                   | PPL claim on 7 March 2022. As     |
|                                   | this is more than 28 days after   |
|                                   | Amy’s birth, Lucy’s PPL period    |
|                                   | starts from the date the          |
|                                   | effective claim and supporting    |
|                                   | documents were lodged. Lucy       |
|                                   | cannot be paid past the 6 June as |
|                                   | this is when baby Amy is 18 weeks |
|                                   | old.                              |
|                                   |                                   |
|                                   | Lucy: elects to be paid the PPL   |
|                                   | period and nominates consecutive  |
|                                   | Flexible PPL days to start after  |
|                                   | her PPL period.                   |
|                                   |                                   |
|                                   | -   PPL period 7 March to 29 May  |
|                                   | -   Flexible PPL days 30 May to 6 |
|                                   |     June (7 days)                 |
|                                   |                                   |
|                                   | Lucy is paid 12 weeks PPL period  |
|                                   | and 7 Flexible PPL days.          |
+-----------------------------------+-----------------------------------+
| 4                                 | Claims 18 week continuous and     |
|                                   | returns to work within PPL period |
|                                   |                                   |
|                                   | Rachelle gives birth to Emily on  |
|                                   | 1 February and lodges an          |
|                                   | effective PPL claim within 28     |
|                                   | days of giving birth. Rachelle:   |
|                                   |                                   |
|                                   | -   nominates the date of birth   |
|                                   |     as the PPL period start date, |
|                                   |     and                           |
|                                   | -   connects the 30 Flexible PPL  |
|                                   |     days to the end of the PPL    |
|                                   |     period (18 weeks continuous   |
|                                   |     payment)                      |
|                                   |                                   |
|                                   | Rachelle is eligible for PPL from |
|                                   | 1 February to 6 June. Rachelle    |
|                                   | ceases to be eligible for PPL on  |
|                                   | 6 June, when baby Emily is 18     |
|                                   | weeks old.                        |
|                                   |                                   |
|                                   | -   PPL period 1 February to 25   |
|                                   |     April                         |
|                                   | -   Connected period 26 April to  |
|                                   |     6 June                        |
|                                   |                                   |
|                                   | Rachelle contacts Services        |
|                                   | Australia to advise of returning  |
|                                   | to work on 18 April. As this is   |
|                                   | within the PPL period, Rachelle’s |
|                                   | PPL period will cease on 18       |
|                                   | April.                            |
|                                   |                                   |
|                                   | Rachelle was paid 11 weeks of the |
|                                   | PPL period. Rachelle may be able  |
|                                   | to claim Flexible PPL days on     |
|                                   | days Rachelle is not working      |
|                                   | between 26 April (after the PPL   |
|                                   | period) and 6 June.               |
+-----------------------------------+-----------------------------------+
| 5                                 | Claims 18 week continuous PPL     |
|                                   | payment and returns to work in    |
|                                   | future date                       |
|                                   |                                   |
|                                   | Monica gives birth to Demi on 1   |
|                                   | February and lodges an effective  |
|                                   | PPL claim within 28 days of       |
|                                   | giving birth. Monica:             |
|                                   |                                   |
|                                   | -   nominates the date of birth   |
|                                   |     as the PPL period start date, |
|                                   |     and                           |
|                                   | -   connects the 30 Flexible PPL  |
|                                   |     days to the end of the PPL    |
|                                   |     period (18 weeks continuous   |
|                                   |     payment)                      |
|                                   |                                   |
|                                   | Monica is eligible for PPL from 1 |
|                                   | February to 6 June. Monica ceases |
|                                   | to be eligible for PPL on 6 June  |
|                                   | as this is when baby Demi is 18   |
|                                   | weeks old.                        |
|                                   |                                   |
|                                   | -   PPL period 1 February to 25   |
|                                   |     April                         |
|                                   | -   Connected period 26 April to  |
|                                   |     6 June                        |
|                                   |                                   |
|                                   | Monica contacts the agency on 26  |
|                                   | April to advise Monica is         |
|                                   | planning to return to work on 25  |
|                                   | May. The Service Officer tells    |
|                                   | Monica that PPL will be cancelled |
|                                   | on 25 May due to returning to     |
|                                   | work                              |
|                                   |                                   |
|                                   | The Service Officer discusses     |
|                                   | Flexible PPL days. Monica:        |
|                                   |                                   |
|                                   | -   decides to withdraw the       |
|                                   |     existing connected Flexible   |
|                                   |     days, then                    |
|                                   | -   nominates new Flexible PPL    |
|                                   |     days (on days Monica is not   |
|                                   |     working) from 26 April to 25  |
|                                   |     May                           |
|                                   |                                   |
|                                   | This includes weekends, as Monica |
|                                   | does not work Saturdays or        |
|                                   | Sundays.                          |
|                                   |                                   |
|                                   | Monica is paid 12 weeks of PPL    |
|                                   | period and 30 Flexible PPL days.  |
