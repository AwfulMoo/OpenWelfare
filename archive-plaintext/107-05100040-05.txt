Recovery Fee codes in DOC

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Recovery Fee codes in DOCs        |
|                                   |                                   |
|                                   | The Debt Action script creates a  |
|                                   | DOC when application of the       |
|                                   | Recovery Fee is considered. The   |
|                                   | script places a single alpha code |
|                                   | before the debt amount in the DOC |
|                                   | Extra Details summary line. The   |
|                                   | full details are available in the |
|                                   | DOC text.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Recovery Fee is imposed           |
|                                   |                                   |
|                                   | I = Recovery Fee Imposed          |
+-----------------------------------+-----------------------------------+
| 3                                 | Recovery Fee is not imposed       |
|                                   |                                   |
|                                   | L = Q175 Warning letter           |
|                                   |                                   |
|                                   | V = Verbal Warning                |
|                                   |                                   |
|                                   | N = No Warning has been issued    |
|                                   |                                   |
|                                   | A DOC detailing the decision      |
|                                   | about the Recovery Fee is still   |
|                                   | required. These codes are a quick |
|                                   | reference.                        |
