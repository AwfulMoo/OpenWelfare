Independence categories

Details of independence categories that are subject to review if there
is a change in customer circumstances:

+-----------------------------------+-----------------------------------+
| Independence Category             | Reviewable on notification of:    |
+-----------------------------------+-----------------------------------+
| Living away from home 18-20 year  | -   young person moved back to    |
| old (inclusive)                   |     parental home permanently     |
+-----------------------------------+-----------------------------------+
| Unable to live at home for        | -   young person moved back to    |
| medical reasons 16-17 year old    |     parental home permanently     |
| (inclusive)                       |                                   |
+-----------------------------------+-----------------------------------+
| Orphan                            | -   young person becomes legally  |
|                                   |     adopted                       |
+-----------------------------------+-----------------------------------+
| Parents cannot exercise           | -   parent is no longer missing   |
| responsibilities                  | -   parent has completed prison   |
|                                   |     sentence                      |
|                                   | -   parent recovers from mental   |
|                                   |     incapacity, or                |
|                                   | -   parent returns home from      |
|                                   |     nursing home                  |
+-----------------------------------+-----------------------------------+
| Refugee                           | -   young person's parents come   |
|                                   |     to Australia, or              |
|                                   | -   young person becomes          |
|                                   |     dependent on someone else on  |
|                                   |     a long term basis             |
+-----------------------------------+-----------------------------------+
| State care                        | -   young person returns to       |
|                                   |     parental home, or             |
|                                   | -   young person is no longer in  |
|                                   |     state care (not because of    |
|                                   |     age)                          |
+-----------------------------------+-----------------------------------+
| Unable to live at home (UTLAH)    | -   young person moves            |
|                                   |     permanently back into the     |
|                                   |     parental home, or             |
|                                   | -   young person receives         |
|                                   |     continuous support            |
+-----------------------------------+-----------------------------------+
| People who are disadvantaged      | -   young person returns to the   |
|                                   |     parental home, or             |
|                                   | -   young person receives         |
|                                   |     financial support from the    |
|                                   |     parent or from another person |
|                                   |     who is acting as the person's |
|                                   |     guardian on a long term basis |
+-----------------------------------+-----------------------------------+
