This page contains a list of fields to be completed by the prescriber
and the fields located on the resident demographic section on the
standardised National Residential Medication Chart (NRMC) and links to
the Australian Commission on Safety and Quality in Health Care website
and the User guide for pharmacists.

On this Page:

Fields to be completed by prescriber on the Standardised National Medication Chart

This table contains the fields which are filled by the prescriber for a
medication entry for a PBS /Repatriation Pharmaceutical Benefits Scheme
(RPBS) item on the NRMC.

+-----------------------------------+-----------------------------------+
| Field                             | Description                       |
+-----------------------------------+-----------------------------------+
| Medicine                          | This field is used for the        |
|                                   | prescriber to indicate the        |
|                                   | medicine name, strength and       |
|                                   | dosage form.                      |
+-----------------------------------+-----------------------------------+
| Dose, frequency and route of      | This field will allow the         |
| administration                    | prescriber to indicate the dose,  |
|                                   | frequency and route of            |
|                                   | administration for the medicine.  |
+-----------------------------------+-----------------------------------+
| PBS/RPBS                          | The prescriber uses this field to |
|                                   | indicate if the prescription is   |
|                                   | to be supplied under the          |
|                                   | Pharmaceutical Benefits Scheme    |
|                                   | (PBS) or Repatriation             |
|                                   | Pharmaceutical Benefits Scheme    |
|                                   | (RPBS).                           |
|                                   |                                   |
|                                   | For private or non-PBS items, the |
|                                   | PBS/RPBS should be struck out or  |
|                                   | the prescription should be        |
|                                   | endorsed as 'non-PBS' or          |
|                                   | 'private'.                        |
+-----------------------------------+-----------------------------------+
| Brand substitution not permitted  | The prescriber uses this tick-box |
|                                   | to indicate that brand            |
|                                   | substitution is not permitted.    |
+-----------------------------------+-----------------------------------+
| Streamlined Authority Code        | For Authority required            |
|                                   | (STREAMLINED) medicines, the      |
|                                   | prescriber must write the         |
|                                   | four-digit streamlined authority  |
|                                   | code that corresponds to the      |
|                                   | appropriate restriction for the   |
|                                   | medicine.                         |
+-----------------------------------+-----------------------------------+
| Date of prescribing               | This field indicates the date the |
|                                   | prescriber wrote the order on the |
|                                   | medication chart for a particular |
|                                   | item.                             |
+-----------------------------------+-----------------------------------+
| Ongoing                           | This field enables the prescriber |
|                                   | to indicate their intention for   |
|                                   | the resident's regular medicine   |
|                                   | to be supplied on an ongoing      |
|                                   | basis until the chart expiry      |
|                                   | date.                             |
|                                   |                                   |
|                                   | Note: the prescriber should not   |
|                                   | use this field for short term     |
|                                   | medicine - they should use the    |
|                                   | stop date field instead.          |
+-----------------------------------+-----------------------------------+
| Start date                        | This field enables the prescriber |
|                                   | to indicate that the medicine     |
|                                   | administration is not to be       |
|                                   | commenced from the date of        |
|                                   | prescribing, but from the future  |
|                                   | date indicated.                   |
|                                   |                                   |
|                                   | Completion of the field by the    |
|                                   | prescriber is not mandatory for   |
|                                   | PBS/RPBS supply and claiming      |
|                                   | purposes.                         |
+-----------------------------------+-----------------------------------+
| Stop date                         | This field enables the prescriber |
|                                   | to indicate their intention for   |
|                                   | the resident's short term         |
|                                   | medicine to be supplied up to a   |
|                                   | particular stop date (stop date   |
|                                   | cannot exceed the chart expiry    |
|                                   | date).                            |
|                                   |                                   |
|                                   | The approved pharmacist must      |
|                                   | determine the quantity to be      |
|                                   | supplied such that treatment is   |
|                                   | provided up to and including the  |
|                                   | stop date, but not beyond the     |
|                                   | stop date.                        |
|                                   |                                   |
|                                   | Note: the prescriber should not   |
|                                   | use this field for the resident's |
|                                   | regular medicines - they should   |
|                                   | use the 'ongoing' field instead.  |
+-----------------------------------+-----------------------------------+
| Closing the Gap (CTG)             | From 1 July 2021, PBS prescribers |
|                                   | are no longer legally required to |
| (for eligible Aboriginal and      | write or electronically print     |
| Torres Strait Islander residents) | ‘CTG’ on eligible PBS             |
|                                   | prescriptions for registered      |
|                                   | Aboriginal and Torres Strait      |
|                                   | Islander people. However,         |
|                                   | annotation of a PBS prescription  |
|                                   | will help community pharmacists,  |
|                                   | when dispensing the medicines,    |
|                                   | know that the patient is          |
|                                   | registered for the program.       |
+-----------------------------------+-----------------------------------+
| Prescriber's signature and name   | This field contains the signature |
|                                   | and printed name of the           |
|                                   | prescriber who has written the    |
|                                   | medication chart entry.           |
+-----------------------------------+-----------------------------------+

Fields on resident demographic section on Standardised National Medication Chart

This table contains a list of fields that are located on the resident
demographic section on the standardised National Residential Medication
Chart (NRMC) and although resident demographics appear on a different
section of the chart, copies of these pages need to be provided to the
dispensing pharmacy for PBS supply and claiming purposes:

+-----------------------------------+-----------------------------------+
| No.                               | Field and description             |
+-----------------------------------+-----------------------------------+
| Resident's full Name              | This field indicates the name of  |
|                                   | the resident, which must match    |
|                                   | with the name registered on the   |
|                                   | Medicare card.                    |
+-----------------------------------+-----------------------------------+
| Resident's date of birth          | This field contains the           |
|                                   | resident's date of birth.         |
+-----------------------------------+-----------------------------------+
| Prescriber's Name                 | This field contains the main      |
|                                   | prescriber's name and any         |
|                                   | additional prescribers using the  |
|                                   | standardised national medication  |
|                                   | chart.                            |
|                                   |                                   |
|                                   | The medication chart entries      |
|                                   | completed by the main prescriber  |
|                                   | can be distinguished from those   |
|                                   | of additional prescribers based   |
|                                   | upon the signature provided       |
|                                   | against the medication chart      |
|                                   | entry.                            |
+-----------------------------------+-----------------------------------+
| Prescriber's Address              | This field contains the address   |
|                                   | of the main prescriber and any    |
|                                   | additional prescribers using the  |
|                                   | standardised national medication  |
|                                   | chart.                            |
+-----------------------------------+-----------------------------------+
| Prescriber Number                 | This field contains the           |
|                                   | prescriber's number.              |
+-----------------------------------+-----------------------------------+
| RACS ID                           | This field contains the           |
|                                   | Residential Aged Care Service     |
|                                   | Identification Number (RACS ID),  |
|                                   | which is a unique four character  |
|                                   | identification number allocated   |
|                                   | by the Department of Health and   |
|                                   | Aged Care to residential aged     |
|                                   | care facilities. The RACS ID will |
|                                   | act as a proxy for the resident's |
|                                   | address.                          |
|                                   |                                   |
|                                   | This number is also known as the  |
|                                   | RACF ID (Residential Aged Care    |
|                                   | Facility Identification Number).  |
+-----------------------------------+-----------------------------------+
| Medicare and Other Entitlement    | This field indicates the Medicare |
| Numbers                           | and other entitlement numbers for |
|                                   | the resident.                     |
+-----------------------------------+-----------------------------------+
| Chart expiry date                 | This field indicates the last     |
|                                   | date of validity for all items on |
|                                   | the medication chart. The period  |
|                                   | of validity of the medication     |
|                                   | chart is no greater than four     |
|                                   | months from the date of           |
|                                   | prescribing of the first          |
|                                   | medication chart entry, that is,  |
|                                   | if the first prescription is      |
|                                   | written on a residential          |
|                                   | medication chart on 11 June, the  |
|                                   | validity of the residential       |
|                                   | medication chart starts on 11     |
|                                   | June and ends on 31 October.      |
+-----------------------------------+-----------------------------------+
