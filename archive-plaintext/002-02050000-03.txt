| Step                              | Action                            |

| 1                                 | Is the asset a non-farm asset     |
|                                   | under section 35 of the Farm      |
|                                   | Household Support Act 2014? +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the asset being assessed:      |
|                                   |                                   |
|                                   | a right or interest in land used  |
|                                   | wholly or mainly for the purposes |
|                                   | of a farm enterprise, OR          |
|                                   |                                   |
|                                   | a right or interest in livestock, |
|                                   | crop, plant or equipment that is  |
|                                   | either used wholly or mainly for  |
|                                   | the purposes of the farm          |
|                                   | enterprise or is produce of a     |
|                                   | farm enterprise, OR               |
|                                   |                                   |
|                                   | a loan that meets the beneficiary |
|                                   | loan requirements?                |
|                                   |                                   |
|                                   | -   Yes, the asset is a farm      |
|                                   |     asset. Go to Step 8           |
|                                   | -   No, the asset is a non-farm   |
|                                   |     asset. Go to Step 2           |

| 2                                 | Is the asset held for the         |
|                                   | purposes of carrying out a farm   |
|                                   | enterprise? + Read more ...       |
|                                   |                                   |
|                                   | Consider:                         |
|                                   |                                   |
|                                   | -   what industry does the farm   |
|                                   |     enterprise operate in? Is     |
|                                   |     this the same industry the    |
|                                   |     asset relates to?             |
|                                   | -   is there a formal document    |
|                                   |     that requires a person to     |
|                                   |     hold the asset in order to    |
|                                   |     conduct their business? (For  |
|                                   |     example, a cooperative share  |
|                                   |     standard may require a dairy  |
|                                   |     farmer to hold a certain      |
|                                   |     number of shares in order to  |
|                                   |     supply their milk product to  |
|                                   |     the cooperative)              |
|                                   | -   if there is a formal          |
|                                   |     requirement, what value of    |
|                                   |     the asset must be held? (for  |
|                                   |     example, suppliers of the     |
|                                   |     previous Murray Goulburn      |
|                                   |     Cooperative (MG Co-op) were   |
|                                   |     required to hold a minimum of |
|                                   |     500 shares plus a number of   |
|                                   |     shares based on the amount of |
|                                   |     milk solids they supply       |
|                                   | -   compare the customer's        |
|                                   |     required holdings with their  |
|                                   |     actual holdings. Holdings in  |
|                                   |     excess of the required amount |
|                                   |     will not be exempt            |
|                                   |                                   |
|                                   | Note: search company websites for |
|                                   | corporate documents in order to   |
|                                   | gather this information - look    |
|                                   | for Constitution documents or     |
|                                   | Share Policy documents.           |
|                                   |                                   |
|                                   | Service Officers are not to       |
|                                   | contact organisations directly.   |
|                                   | For assistance to source          |
|                                   | particular information, refer the |
|                                   | query to the Technical Peer       |
|                                   | Support officer.                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 8              |

| 3                                 | Can the asset only be held for    |
|                                   | the purposes of carrying out a    |
|                                   | farm enterprise? + Read more ...  |
|                                   |                                   |
|                                   | Consider:                         |
|                                   |                                   |
|                                   | For shares, what type of entity   |
|                                   | are the shares for (private or    |
|                                   | public owned)?                    |
|                                   |                                   |
|                                   | For example, there were 2         |
|                                   | separate Murray Goulburn entities |
|                                   | in which a person may have an     |
|                                   | interest.                         |
|                                   |                                   |
|                                   | The Murray Goulburn Cooperative   |
|                                   | was privately owned by suppliers. |
|                                   | Only suppliers of Murray Goulburn |
|                                   | (dairy farmers) were able to      |
|                                   | purchase shares.                  |
|                                   |                                   |
|                                   | The Murray Goulburn Unit Trust    |
|                                   | was listed on the ASX, and as per |
|                                   | the Murray Goulburn website,      |
|                                   | members of the public were able   |
|                                   | to purchase units in order to     |
|                                   | benefit financially from the      |
|                                   | performance of Murray Goulburn.   |
|                                   |                                   |
|                                   | Can the asset be held only by     |
|                                   | farmers?                          |
|                                   |                                   |
|                                   | For example, some companies offer |
|                                   | employee share schemes. Such      |
|                                   | shares would not be exempt under  |
|                                   | the Rule as they can be held by a |
|                                   | person who is not a farmer in the |
|                                   | relevant industry.                |
|                                   |                                   |
|                                   | Are there any other benefits to   |
