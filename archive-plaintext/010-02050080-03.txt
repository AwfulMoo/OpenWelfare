| 3                                 | Customer qualifies for            |
|                                   | Incidentals Allowance for the new |
|                                   | course + Read more ...            |
|                                   |                                   |
|                                   | The customer's initial            |
|                                   | Incidentals Allowance entitlement |
|                                   | is automatically calculated and   |
|                                   | paid as part of their new claim   |
|                                   | for ABSTUDY or ABSTUDY Pensioner  |
|                                   | Education Supplement. The initial |
|                                   | Incidentals Allowance rate is     |
|                                   | determined by the period of       |
|                                   | enrolment in the course, or       |
|                                   | period as an Australian           |
|                                   | Apprentice, during the year of    |
|                                   | assistance. The rate paid depends |
|                                   | on whether the period of          |
|                                   | enrolment or training is:         |
|                                   |                                   |
|                                   | -   Less than 12 weeks            |
|                                   | -   12-16 weeks                   |
|                                   | -   17-23 weeks                   |
|                                   | -   24 weeks to 1 year            |
|                                   |                                   |
|                                   | Rate of Incidentals Allowance is  |
|                                   | determined by the system          |
|                                   |                                   |
|                                   | The system calculates the rate of |
|                                   | Incidental Allowance by counting  |
|                                   | the number of weeks between the   |
|                                   | Institution Course Start Date and |
|                                   | the Institution Course End date.  |
|                                   | The number of weeks of the course |
|                                   | length determines the rate of     |
|                                   | Incidental Allowance. When the    |
|                                   | Institution Course Start Date and |
|                                   | Institution Course End Date are   |
|                                   | in different calendar years, the  |
|                                   | system calculates the rate of     |
|                                   | Incidental Allowance by counting  |
|                                   | the number of weeks between the   |
|                                   | Institution Course Start Date and |
|                                   | the end of the calendar year, and |
|                                   | the start of the calendar year    |
|                                   | and the Institution Course End    |
|                                   | Date respectively. However, in    |
|                                   | this circumstance, the system     |
|                                   | counts the number of weeks as     |
|                                   | follows:                          |
|                                   |                                   |
|                                   | If the Institution Course Start   |
|                                   | Date is:                          |
|                                   |                                   |
|                                   | -   in any year prior to the year |
|                                   |     in which the Incidentals      |
|                                   |     Allowance is being made, then |
|                                   |     the rate of Incidentals       |
|                                   |     Allowance is calculated by    |
|                                   |     the system by commencing to   |
|                                   |     count weeks from 28 January   |
|                                   | -   between 1 July and 27 July    |
|                                   |     inclusive, in the year in     |
|                                   |     which the Incidentals         |
|                                   |     Allowance is being made, then |
|                                   |     the rate of Incidentals       |
|                                   |     Allowance is calculated by    |
|                                   |     the system by commencing to   |
|                                   |     count weeks from 28 July      |
|                                   |                                   |
|                                   | Note: some non-traditional        |
|                                   | courses run continuously over the |
|                                   | Christmas holiday break, for      |
|                                   | example correspondence courses.   |
|                                   | Ensure the rate of Incidentals    |
|                                   | Allowance calculated by the       |
|                                   | system is correct based on the    |
|                                   | length of the course.             |
