Types of letters that may be sent to DSP customers

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Letter Q055 + Read more ...       |
|                                   |                                   |
|                                   | Notification to a customer that   |
|                                   | that they may be eligible for an  |
|                                   | alternative payment. Also issued  |
|                                   | by JCA Processing Team as an      |
|                                   | invitation to claim DSP following |
|                                   | an assessment of a Job Capacity   |
|                                   | Assessment (JCA) report.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Letter Q114 + Read more ...       |
|                                   |                                   |
|                                   | Confirmation of continuation of   |
|                                   | DSP.                              |
+-----------------------------------+-----------------------------------+
| 3                                 | Letter Q134 + Read more ...       |
|                                   |                                   |
|                                   | Notification of rejection,        |
|                                   | cancellation, suspension or       |
|                                   | deferment of pension, allowance   |
|                                   | or benefit.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Letter Q263 + Read more ...       |
|                                   |                                   |
|                                   | Request for Parts A, B, or C of a |
|                                   | new claim or review from the      |
|                                   | customer.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Letter Q322 + Read more ...       |
|                                   |                                   |
|                                   | Notification that the customer’s  |
|                                   | incapacity for work is classified |
|                                   | as long term and the customer is  |
|                                   | invited to claim for DSP.         |
+-----------------------------------+-----------------------------------+
| 6                                 | Letter Q349 + Read more ...       |
|                                   |                                   |
|                                   | Notification of cancellation of   |
|                                   | DSP after their pension has been  |
|                                   | suspended for two years.          |
+-----------------------------------+-----------------------------------+
| 7                                 | Letter Q351 + Read more ...       |
|                                   |                                   |
|                                   | Advice to the customer from the   |
|                                   | Subject Matter Expert (SME) that: |
|                                   |                                   |
|                                   | -   the decision has changed      |
|                                   | -   the request for review has    |
|                                   |     been withdrawn, or            |
|                                   | -   the matter has been referred  |
|                                   |     to an Authorised Review       |
|                                   |     Officer (ARO)                 |
|                                   |                                   |
|                                   | This letter can also be used to   |
|                                   | request customer contact in       |
|                                   | regards to a review request.      |
|                                   |                                   |
|                                   | Only generates for reviews and    |
|                                   | quality checks registered before  |
|                                   | 15 May 2021.                      |
+-----------------------------------+-----------------------------------+
| 8                                 | Letter Q449 + Read more ...       |
|                                   |                                   |
|                                   | Advice to the customer:           |
|                                   |                                   |
|                                   | -   from the Subject Matter       |
|                                   |     Expert (SME) that they were   |
|                                   |     unable to contact the         |
|                                   |     customer (to provide an       |
|                                   |     explanation), and inviting    |
|                                   |     contact if the customer would |
|                                   |     still like to talk about the  |
|                                   |     decision                      |
|                                   | -   from any staff member that    |
|                                   |     the customer's application    |
|                                   |     for a formal review has been  |
|                                   |     referred to an Authorised     |
|                                   |     Review Officer                |
|                                   |                                   |
|                                   | Only generates for explanations   |
|                                   | and formal reviews registered on  |
|                                   | or after 15 May 2021.             |
+-----------------------------------+-----------------------------------+
| 9                                 | Letter Q426 + Read more ...       |
|                                   |                                   |
|                                   | Confirmation of continuation of   |
|                                   | DSP after the medical review is   |
|                                   | completed.                        |
+-----------------------------------+-----------------------------------+
| 10                                | Letter Q999 + Read more ...       |
|                                   |                                   |
|                                   | Free text letter.                 |
+-----------------------------------+-----------------------------------+
| 11                                | Letter Q888 + Read more ...       |
|                                   |                                   |
|                                   | Free text letter.                 |
+-----------------------------------+-----------------------------------+
