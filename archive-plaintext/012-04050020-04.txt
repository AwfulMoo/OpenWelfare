Service Level Agreement (SLA) compliance

This table outlines timeframes that apply to all My Health Record
applications under the SLA agreement with the Department of Health.

+-----------------------------------+-----------------------------------+
| Item                              | SLA timeframes                    |
+-----------------------------------+-----------------------------------+
| 1                                 | SLA for processing application    |
|                                   |                                   |
|                                   | Applications with all required    |
|                                   | information must be processed     |
|                                   | within 14 calendar days from date |
|                                   | of receiving complete             |
|                                   | application.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | SLA for requesting information to |
|                                   | complete application              |
|                                   |                                   |
|                                   | -   1st contact is made           |
|                                   | -   2nd contact to be made 14     |
|                                   |     calendar days after 1st       |
|                                   |     contact date                  |
|                                   | -   Rejected: 7 days after 2nd    |
|                                   |     contact                       |
|                                   | -   Total = 21 calendar days      |
