This document outlines how Compliance Officers (CO) generate customer
outcome letters using the Generate Outcome Letter workflow in Process
Direct.

Generate Outcome Letter workflow

This workflow supports COs in generating an outcome letter for online
compliance interventions. These include the Employment Income
Confirmation (EIC) and the check and update past income (CUPI) service.

Accessing the workflow

The workflow is an application in process Direct. It can be accessed via
the Integrated Review in Process Direct.
