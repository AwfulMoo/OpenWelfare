Healthcare Identifiers (HI) service

Description of the components of a pseudonymous IHI

+-----------------------------------+-----------------------------------+
| Component                         | Description                       |
+-----------------------------------+-----------------------------------+
| Name                              | Nominated by the requesting       |
|                                   | individual.                       |
+-----------------------------------+-----------------------------------+
| Date of birth                     | Selected by the individual,       |
|                                   | however, the date must be within  |
|                                   | 6 months of their real age.       |
|                                   |                                   |
|                                   | Note: this is needed as           |
|                                   | healthcare providers may make     |
|                                   | clinical decisions based on the   |
|                                   | age of the individual, therefore  |
|                                   | the date must be close to the     |
|                                   | individual's real age.            |
+-----------------------------------+-----------------------------------+
| Birth date accuracy field         | Recorded in the pseudonymous IHI  |
|                                   | as 'day month and year values     |
|                                   | estimated' (EEE).                 |
+-----------------------------------+-----------------------------------+
| Address                           | Any valid address declared by the |
|                                   | individual and should be          |
|                                   | different to that held on the     |
|                                   | original Verified IHI.            |
|                                   |                                   |
|                                   | Note: if the individual supplies  |
|                                   | their true address, it must be    |
|                                   | accepted.                         |
+-----------------------------------+-----------------------------------+
