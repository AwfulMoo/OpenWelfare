Seasonal employment additional information

This table describes additional information for job seekers seeking
seasonal employment.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Jobsearch Facilities              |
|                                   |                                   |
|                                   | The Jobsearch Facilities located  |
|                                   | in Service Centres can be used to |
|                                   | access information on Seasonal    |
|                                   | Work from the Department of       |
|                                   | Education, Skills and Employment  |
|                                   | website under Australian Job      |
|                                   | Search - Harvest Trail.           |
|                                   |                                   |
|                                   | Harvest Trail - Overview          |
+-----------------------------------+-----------------------------------+
| 2                                 | Local industries, businesses and  |
|                                   | focus points                      |
|                                   |                                   |
|                                   | Local industries, businesses and  |
|                                   | focus points can provide          |
|                                   | information about seasonal work.  |
|                                   | For example local weigh           |
|                                   | bridges/weigh stations in some    |
|                                   | States are a useful access point  |
|                                   | for information about demand for  |
|                                   | seasonal employment.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Small Area Labour Markets -       |
|                                   | Australia                         |
|                                   |                                   |
|                                   | Unemployment rates for individual |
|                                   | local areas are available on the  |
|                                   | Department of Education, Skills   |
|                                   | and Employment website.           |
|                                   |                                   |
|                                   | Small Area Labour Markets         |
+-----------------------------------+-----------------------------------+
| 4                                 | Workabout Australia               |
|                                   |                                   |
|                                   | The Workabout Australia website   |
|                                   | contains information for people   |
|                                   | interested in seasonal work in    |
|                                   | Australia. The website also       |
|                                   | contains an ordering facility to  |
|                                   | purchase the publication          |
|                                   | 'Workabout Australia' which       |
|                                   | provides information on casual    |
|                                   | and seasonal employment in        |
|                                   | multiple locations in all states  |
|                                   | of Australia.                     |
|                                   |                                   |
|                                   | Workabout Australia               |
