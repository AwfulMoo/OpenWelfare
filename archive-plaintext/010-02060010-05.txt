This page contains scenarios to assist with determining the start date
for the away from home rate.

Determining when it is reasonable to move out of home

This table describes scenarios that illustrate when it is reasonable for
the customer to move out of home and the date from which the away from
home rate will be paid.

+-----------------------------------+-----------------------------------+
| Scenario                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Reasonable need to move out of    |
|                                   | home before course starts + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Sally completes a Bachelor of     |
|                                   | Arts (Honours) at Melbourne       |
|                                   | University while receiving        |
|                                   | ABSTUDY. She is accepted into the |
|                                   | University of New South Wales in  |
|                                   | Sydney to undertake a Masters     |
|                                   | Degree in Indigenous Studies for  |
|                                   | the following year.               |
|                                   |                                   |
|                                   | Sally qualifies for the away from |
|                                   | home rate to move away from a     |
|                                   | permanent home in Melbourne to    |
|                                   | undertake the course.             |
|                                   |                                   |
|                                   | Sally moves to Sydney 4 weeks     |
|                                   | before the first day of the       |
|                                   | course to start looking for       |
|                                   | accommodation to rent for the     |
|                                   | study year. Sally stays with      |
|                                   | friends. It takes Sally 4 weeks   |
|                                   | to find an affordable place and   |
|                                   | sign the lease agreement. Sally   |
|                                   | moves in 3 days later.            |
|                                   |                                   |
|                                   | The away from home rate is        |
|                                   | approved from the date Sally      |
|                                   | moved to Sydney, 4 weeks before   |
|                                   | the course commencement, so she   |
|                                   | can look for accommodation. It is |
|                                   | reasonable that it would take 4   |
|                                   | weeks to find affordable          |
|                                   | accommodation accessible to the   |
|                                   | university, apply for the         |
|                                   | accommodation, sign a lease and   |
|                                   | move into the accommodation.      |

| 2                                 | Unreasonable need to move out of  |
|                                   | home before course starts + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Michael completes a Bachelor of   |
|                                   | Arts (Honours) at Adelaide        |
|                                   | University while receiving        |
|                                   | ABSTUDY. Michael is accepted into |
|                                   | the University of New South Wales |
|                                   | in Sydney to undertake a Masters  |
|                                   | Degree in Indigenous Studies in   |
|                                   | the following year.               |
|                                   |                                   |
|                                   | Michael qualifies for the away    |
|                                   | from home rate to move away from  |
|                                   | a permanent home in Adelaide to   |
|                                   | undertake the course.             |
|                                   |                                   |
|                                   | Michael has a brother in Sydney   |
|                                   | and arranges to board at his      |
|                                   | house for the year. Michael moves |
|                                   | to Sydney on January 2 of the new |
|                                   | year to watch the last few days   |
|                                   | of the test cricket and then buy  |
|                                   | books for the course. Michael     |
|                                   | spends the next few months        |
|                                   | hanging out at Bondi beach.       |
|                                   |                                   |
|                                   | Michael would be eligible for the |
|                                   | away from home rate from the      |
|                                   | first date of the course as the   |
|                                   | reasons for moving to Sydney      |
|                                   | early were not reasonable.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Moves out of home after course    |
|                                   | starts due to circumstances       |
|                                   | beyond control + Read more ...    |
|                                   |                                   |
|                                   | Leroy enrols in a Certificate in  |
|                                   | Dance at the Western Australian   |
|                                   | Academy of Performing Arts        |
|                                   | (WAAPA). Leroy has not studied    |
|                                   | for 3 years and the first day of  |
|                                   | eligibility for ABSTUDY Living    |
|                                   | Allowance is the first day of the |
|                                   | course.                           |
|                                   |                                   |
|                                   | Leroy intends to move from home   |
|                                   | in Geraldton to rent a room at    |

|                                   | On the day before the move, Leroy |

|                                   | another 3 days at home            |
|                                   | convalescing. Leroy moves to      |
|                                   | Perth 2 weeks late to commence    |
|                                   | the course. As Leroy had reserved |
|                                   | a room at the hostel, it was      |
|                                   | charged to him from the first day |
|                                   | of the course.                    |
|                                   |                                   |
|                                   | Leroy is entitled to the away     |
|                                   | from home rate from the first day |
|                                   | of the course because of          |
|                                   | circumstances beyond Leroy's      |
|                                   | control. These circumstances      |
|                                   | prevented Leroy from commencing   |
|                                   | to live away from home earlier,   |
|                                   | and the accommodation costs were  |
|                                   | accrued from the first day of the |
|                                   | course.                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Moves out of home on the day      |
|                                   | course starts + Read more ...     |
|                                   |                                   |
|                                   | Samantha's home is at Oombulgurri |

|                                   |                                   |

|                                   | Samantha is entitled to the away  |
|                                   | from home rate from the first     |
|                                   | date of the term entitlement      |
|                                   | period in Term 1. This is January |
|                                   | 1 of the year.                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Moves out of home during week 5   |
|                                   | of the course + Read more ...     |
|                                   |                                   |
|                                   | Lateisha's home is at Bamaga in   |
|                                   | the remote Cape York region.      |
|                                   | Lateisha is accepted to board at  |

|                                   | As Lateisha has not commenced by  |
