Creating and storing paper files for reviews and appeals

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Creating review and appeal paper  |
|                                   | files + Read more ...             |
|                                   |                                   |
|                                   | A new review and appeals paper    |
|                                   | file is not required for the      |
|                                   | following cases:                  |
|                                   |                                   |
|                                   | -   Authorised Review Officer     |
|                                   |     (ARO), or                     |
|                                   | -   Administrative Appeals        |
|                                   |     Tribunal (AAT) first review   |
|                                   |                                   |
|                                   | Documents relevant to the review  |
|                                   | may be stored on the customer     |
|                                   | paper file/s. However, in most    |
|                                   | cases the paper file will not be  |
|                                   | needed as relevant documents will |
|                                   | be scanned to the customer's      |
|                                   | electronic record.                |
|                                   |                                   |
|                                   | New paper files are created for:  |
|                                   |                                   |
|                                   | -   Administrative Appeals        |
|                                   |     Tribunal (AAT) second review, |
|                                   |     and                           |
|                                   | -   Court appeals                 |
|                                   |                                   |
|                                   | Requesting a new paper file       |
|                                   |                                   |
|                                   | When a file is to be created,     |
|                                   | request the new file via the      |
|                                   | Records Management service        |
|                                   | request webform. See the          |
|                                   | Resources page for a link to the  |
|                                   | webform.                          |
|                                   |                                   |
|                                   | The requesting officer must       |
|                                   | provide the following information |
|                                   | at the time of the request:       |
|                                   |                                   |
|                                   | -   type of file required - AAT   |
|                                   |     second review or Court appeal |
|                                   | -   customer's Customer Reference |
|                                   |     Number (CRN)                  |
|                                   | -   surname, first name, middle   |
|                                   |     name or middle initial        |
|                                   | -   date of birth                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Scanning documentation for a      |
|                                   | review of decision + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | All documents obtained or created |
|                                   | for a review of decision must be  |
|                                   | scanned to the customer's         |
|                                   | electronic record.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Documentation for Administrative  |
|                                   | Appeals Tribunal (AAT) first      |
|                                   | review + Read more ...            |
|                                   |                                   |
|                                   | All relevant evidence held by the |
|                                   | department is prepared and        |
|                                   | forwarded to the AAT              |
|                                   | electronically by the External    |
|                                   | Review of Decision Team.          |
|                                   |                                   |
|                                   | The AAT first review transfers    |
|                                   | decisions to Services Australia   |
|                                   | electronically. The AAT first     |
|                                   | review also transfers relevant    |
|                                   | documents received from the       |
|                                   | customer or third parties in      |
|                                   | relation to the review. The       |
|                                   | External Review of Decision Team  |
|                                   | uploads the decision and other    |
|                                   | documents to the customer record. |
|                                   | The Litigation and Information    |
|                                   | Release Branch note the receipt   |
|                                   | of the decision documents by      |
|                                   | running the Appeal Decision       |
|                                   | Implementation script, which will |
|                                   | record the AAT decision in the    |
|                                   | Appeals Management System (APL).  |
+-----------------------------------+-----------------------------------+
| 4                                 | Documentation for AAT second      |
|                                   | review + Read more ...            |
|                                   |                                   |
|                                   | The Litigation and Information    |
|                                   | Release Branch will request a new |
|                                   | paper file to be created for the  |
|                                   | AAT case by the Records           |
|                                   | Management Unit (RMU). Use the    |
|                                   | Records Management service        |
|                                   | request webform. See the          |
|                                   | Resources page for a link to the  |
|                                   | webform.                          |
|                                   |                                   |
|                                   | The AAT file should hold all      |
|                                   | documents created or obtained for |
|                                   | the review, including working     |
|                                   | notes and requests for            |
|                                   | instructions.                     |
|                                   |                                   |
|                                   | Copies of documents directly      |
|                                   | related to the customer's         |
|                                   | entitlement or the decision under |
|                                   | review should be added to the     |
|                                   | customer file. This includes:     |
|                                   |                                   |
|                                   | -   any new evidence obtained     |
|                                   |     during the review process     |
|                                   | -   the AAT decision              |
|                                   |                                   |
|                                   | There is likely to be some        |
|                                   | duplication in the documents held |
|                                   | on both files.                    |
|                                   |                                   |
|                                   | When the AAT second review        |
|                                   | decision is received, the         |
|                                   | Litigation and Information        |
|                                   | Release Branch will create the    |
|                                   | AAT2 Implement Decision Fast      |
|                                   | Note. Select Auto Text. Use       |
|                                   | Appeals > AAT Implementations >   |
|                                   | AAT2 APL-Implement Decision. This |
|                                   | will advise the most appropriate  |
|                                   | team of the outcome of the AAT    |
|                                   | second review and the action to   |
|                                   | be taken.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Structure of Court appeals        |
|                                   | files + Read more ...             |
|                                   |                                   |
|                                   | All Court appeals are handled by  |
|                                   | the Litigation and Information    |
|                                   | Release Branch.                   |
|                                   |                                   |
|                                   | The Court file should hold all    |
|                                   | documents created or obtained for |
|                                   | the review, including working     |
|                                   | notes, correspondence from        |
|                                   | solicitors/barristers and         |
|                                   | requests for, and clearance of    |
|                                   | instructions.                     |
|                                   |                                   |
|                                   | Copies of documents directly      |
|                                   | related to the customer's         |
|                                   | entitlement or the decision under |
|                                   | review should be added to the     |
|                                   | customer file. This includes:     |
|                                   |                                   |
|                                   | -   any new evidence obtained     |
|                                   |     during the review process     |
|                                   | -   the Court decision            |
|                                   |                                   |
|                                   | There is likely to be some        |
|                                   | duplication in the documents held |
|                                   | on both files.                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Storing and handling review and   |
|                                   | appeal paper files + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Storage                           |
|                                   |                                   |
|                                   | Depending on the stage of the     |
|                                   | review, paper files will usually  |
|                                   | reside in one of the following    |
|                                   | locations:                        |
|                                   |                                   |
|                                   | -   Litigation and Information    |
|                                   |     Release Branch                |
|                                   | -   Records Management Unit (RMU) |
|                                   |                                   |
|                                   | If a file is required to be kept  |
|                                   | by an officer, it must be kept    |
|                                   | out of public view and in a       |
|                                   | secure area, when not in use.     |
|                                   |                                   |
|                                   | File movement                     |
|                                   |                                   |
|                                   | If paper files are required,      |
|                                   | request them from the local RMU   |
|                                   | via the Records Management        |
|                                   | service request webform. All file |
|                                   | requests should be actioned by    |
|                                   | the RMU within 3 working days of  |
|                                   | receipt of the request.           |
|                                   |                                   |
|                                   | Advise the RMU of all movements   |
|                                   | of all files with the exception   |
|                                   | of their return to the RMU via    |
|                                   | the Records Management service    |
|                                   | request webform.                  |
|                                   |                                   |
|                                   | -   When a file is received from  |
|                                   |     the AAT, Litigation and       |
|                                   |     Information Release Branch    |
|                                   |     must record the file as being |
|                                   |     transferred to a specific     |
|                                   |     officer (either the office    |
|                                   |     coordinator or a lawyer in    |
|                                   |     the Branch)                   |
|                                   |                                   |
|                                   | Note: responsibility for the      |
|                                   | paper file will remain with the   |
|                                   | original assignee if this advice  |
|                                   | is not received.                  |
