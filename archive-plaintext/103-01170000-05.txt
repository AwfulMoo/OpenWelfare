Scenarios under LTWPR

Table 1: This table contains different scenarios for enhanced IM under
the Long Term Welfare Payment Recipient measure.

+-----------------------------------+-----------------------------------+
| Scenario                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer not eligible for         |
|                                   | enhanced IM under LTWPR measure   |
|                                   |                                   |
|                                   | John received Parenting Payment   |
|                                   | (Single) for the period 4         |
|                                   | September 2023 to 1 April 2024    |
|                                   | (30 weeks) and Jobseeker Payment  |
|                                   | from 20 August 2024 to 24 October |
|                                   | 2024 (9 weeks).                   |
|                                   |                                   |
|                                   | John would not be subject to      |
|                                   | enhanced IM under the Long Term   |
|                                   | Welfare Payment Recipient measure |
|                                   | as he has only been in receipt of |
|                                   | a relevant payment for a total of |
|                                   | 39 weeks in the previous 104      |
|                                   | weeks.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer in receipt of JobSeeker  |
|                                   | Payment (JSP) for intermittent    |
|                                   | periods                           |
|                                   |                                   |
|                                   | Theo has been in receipt of       |
|                                   | Jobseeker Payment several times   |
|                                   | over the past 104 weeks:          |
|                                   |                                   |
|                                   | -   7 May 2022 to 29 August 2022  |
|                                   |     (15 weeks)                    |
|                                   | -   11 February 2023 to 4 July    |
|                                   |     2023 (20 weeks)               |
|                                   | -   2 December 2023 to date (24   |
|                                   |     April 2024) (20 weeks)        |
|                                   |                                   |
|                                   | Theo would be eligible for        |
|                                   | enhanced IM under the Long Term   |
|                                   | Welfare Payment Recipient measure |
|                                   | as he has been on payments for    |
|                                   | more than 52 weeks during the     |
|                                   | past 104 week period.             |
+-----------------------------------+-----------------------------------+

Scenarios under DEY

Table 2: This table contains different scenarios for enhanced IM under
the Disengaged Youth measure.

+-----------------------------------+-----------------------------------+
| Scenario                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer not eligible for         |
|                                   | enhanced IM under the Disengaged  |
|                                   | Youth measure                     |
|                                   |                                   |
|                                   | John has received Youth Allowance |
|                                   | (YAL) for the period 7 December   |
|                                   | 2023 to 25 January 2024 (7 weeks) |
|                                   | and Jobseeker Payment (JSP) from  |
|                                   | 3 April 2024 to 24 April 2024 (3  |
|                                   | weeks). John would not be subject |
|                                   | to enhanced IM under the          |
|                                   | Disengaged Youth measure as he    |
|                                   | has only been in receipt of a     |
|                                   | relevant payment for a total of   |
|                                   | 10 weeks in the previous 26       |
|                                   | weeks.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer in receipt of YAL and    |
|                                   | JSP over the past 26 weeks        |
|                                   |                                   |
|                                   | Theo has been in receipt of Youth |
|                                   | Allowance and JobSeeker Payment   |
|                                   | over the past 26 weeks:           |
|                                   |                                   |
|                                   | -   YAL - 1 November 2023 to 14   |
|                                   |     December 2023 (6 weeks)       |
|                                   | -   JSP - 7 February 2024 to date |
|                                   |     (24 April 2024) (11 weeks)    |
|                                   |                                   |
|                                   | Theo would be eligible for        |
|                                   | enhanced IM under the Disengaged  |
|                                   | Youth measure as he has been on   |
|                                   | payments for more than 13 weeks   |
|                                   | during the past 26 week period.   |
+-----------------------------------+-----------------------------------+
