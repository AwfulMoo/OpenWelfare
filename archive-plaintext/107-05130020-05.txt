Reason codes for and additional information on Debt write offs

+-----------------------------------+-----------------------------------+
| Code                              | Description                       |
+-----------------------------------+-----------------------------------+
| DEC                               | Deceased Insufficient Estate +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | 'DEC' - this write off reason     |
|                                   | will permanently write off all    |
|                                   | outstanding debts.                |
|                                   |                                   |
|                                   | Family debt offsetting applies    |
|                                   | for:                              |
|                                   |                                   |
|                                   | -   Family Tax Benefit (FTB)      |
|                                   |     instalment arrears, and       |
|                                   | -   garnishee of Australian       |
|                                   |     Taxation Office (ATO) refund  |
|                                   |                                   |
|                                   | If funds become available from    |
|                                   | the deceased customer's estate,   |
|                                   | apply the funds to the debt.      |
|                                   |                                   |
|                                   | This code is not valid for PPL    |
|                                   | organisations.                    |
|                                   |                                   |
|                                   | -   Debt ID specific: No          |
|                                   | -   Automatic FTB offsetting: Yes |
|                                   | -   Automatic recovery of FTB     |
|                                   |     instalment arrears: Yes       |
|                                   | -   Automatic tax garnishee: Yes  |
|                                   | -   Displayed on Money You Owe:   |
|                                   |     No                            |
+-----------------------------------+-----------------------------------+
| BRD                               | Bankruptcy Discharged, No Further |
|                                   | Estate/No Fraud + Read more ...   |
|                                   |                                   |
|                                   | Use BRD when a debt is covered by |
|                                   | bankruptcy and:                   |
|                                   |                                   |
|                                   | -   no dividends are expected     |
|                                   | -   the debt is not due to fraud  |
|                                   |                                   |
|                                   | This code is valid for            |
|                                   | individuals and organisations.    |
|                                   |                                   |
|                                   | -   Debt ID specific: Yes         |
|                                   | -   Automatic FTB offsetting: No  |
|                                   | -   Automatic recovery of FTB     |
|                                   |     instalment arrears: No        |
|                                   | -   Automatic tax garnishee: No   |
|                                   | -   Displayed on Money You Owe:   |
|                                   |     No                            |
+-----------------------------------+-----------------------------------+
| IRD                               | Insolvency - Discharged + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use IRD when an organisation debt |
|                                   | is for a period prior to the date |
|                                   | of insolvency and:                |
|                                   |                                   |
|                                   | -   no dividends are expected     |
|                                   | -   the debt is not due to fraud  |
|                                   |                                   |
|                                   | This code is not valid for        |
|                                   | individuals.                      |
|                                   |                                   |
|                                   | -   Debt ID specific: Yes         |
|                                   | -   Automatic FTB offsetting: No  |
|                                   | -   Automatic recovery of FTB     |
|                                   |     instalment arrears: No        |
|                                   | -   Automatic tax garnishee: No   |
|                                   | -   Displayed on Money You Owe:   |
|                                   |     No                            |

|                                   |                                   |
|                                   | Use STB when 6 years has passed   |
|                                   | without any action on the debt.   |
|                                   |                                   |
|                                   | This code is only valid for       |
|                                   | debts, which reached the 6 years  |
|                                   | date prior to 1 January 2017.     |
|                                   | This write off reason will only   |
|                                   | permanently write off the         |
|                                   | selected debt, not all            |
|                                   | outstanding debts.                |
|                                   |                                   |
|                                   | This code is valid for            |
|                                   | individuals and organisations.    |
|                                   |                                   |
|                                   | For more information on statute   |
|                                   | barred debts, see Six Year        |
|                                   | Limitation Write off.             |
|                                   |                                   |
|                                   | -   Debt ID specific: Yes         |
|                                   | -   Automatic FTB offsetting: No  |
|                                   | -   Automatic recovery of FTB     |
|                                   |     instalment arrears: No        |
|                                   | -   Automatic tax garnishee: No   |
|                                   | -   Displayed on Money You Owe:   |

|                                   | more ...                          |
|                                   |                                   |
|                                   | Use IAL when:                     |
|                                   |                                   |

|                                   |     defends a Judgement Order,    |

|                                   | 'IAL'.                            |

|                                   |     means of fraud)               |
|                                   |                                   |
