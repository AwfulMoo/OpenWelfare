Accreditation of Quality Management Officers (QMO)

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check proficiency + Read more ... |
|                                   |                                   |
|                                   | To show proficiency in the work   |
|                                   | type they will be checking as a   |
|                                   | QMO, a staff member must have:    |
|                                   |                                   |
|                                   | -   achieved and maintained at    |
|                                   |     least 95% correctness in the  |
|                                   |     work types they will be       |
|                                   |     checking                      |
|                                   | -   a minimum of 20 activities    |
|                                   |     quality checked for the       |
|                                   |     relevant work type            |
|                                   | -   processed activities/work     |
|                                   |     items for a minimum of 2      |
|                                   |     weeks                         |
|                                   | -   activities/work items QOL/QMA |
|                                   |     checked for up to the last 14 |
|                                   |     months                        |
|                                   |                                   |
|                                   | For more information, see Staff   |
|                                   | QMA proficiency policy.           |
|                                   |                                   |
|                                   | Access the staff member’s QMA     |
|                                   | data using the QMA Dashboard -    |
|                                   | Team Leader tile in Process       |
|                                   | Direct to confirm correctness     |
|                                   | rate. See the Resources page for  |
|                                   | a link to the QMA Reporting       |
|                                   | Dashboard Guide. If there are     |
|                                   | discrepancies with the data,      |
|                                   | contact QMA QOL tools.            |
|                                   |                                   |
|                                   | Note: if the work item has        |
|                                   | transitioned to Process Direct    |
|                                   | and there is no history or not    |
|                                   | enough QMA data in the QMA        |
|                                   | Dashboard, QOL results may be     |
|                                   | used.                             |
|                                   |                                   |
|                                   | An alternative proficiency        |
|                                   | assessment can be applied where   |
|                                   | staff members do not have enough  |
|                                   | work items completed. The         |
|                                   | correctness rate of 95% is        |
|                                   | mandatory.                        |
|                                   |                                   |
|                                   | Has the staff member met          |
|                                   | proficiency requirements:         |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   If 95% correctness has been   |
|                                   |     met, but the staff member has |
|                                   |     not completed enough work     |
|                                   |     items. Alternative            |
|                                   |     proficiency may apply. Go to  |
|                                   |     Step 2                        |
|                                   | -   If neither measure has been   |
|                                   |     met. Continue to monitor the  |
|                                   |     staff member’s results and    |
|                                   |     consider QMO accreditation at |
|                                   |     a later stage. Procedure ends |
|                                   |     here.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Alternative proficiency           |
|                                   | Assessment + Read more ...        |
|                                   |                                   |
|                                   | These assessments can cover       |
|                                   | situations such as:               |
|                                   |                                   |
|                                   | -   start of new benefit, payment |
|                                   |     or business type              |
|                                   | -   technical programmes          |
|                                   |     undertaken by specialist      |
|                                   |     staff only                    |
|                                   |                                   |
|                                   | If an Alternative Proficiency     |
|                                   | assessment is required:           |
|                                   |                                   |
|                                   | -   complete the Alternate        |
|                                   |     Proficiency Accreditation     |
|                                   |     Application form. See the     |
|                                   |     Resources page for a link     |
|                                   | -   email the form to QMA QOL     |
|                                   |     tools                         |
|                                   |                                   |
|                                   | The QMA team will make an         |
|                                   | assessment and advise the         |
|                                   | outcome.                          |
|                                   |                                   |
|                                   | Has the QMA team approved the     |
|                                   | request?                          |
|                                   |                                   |
|                                   | -   Yes, go to step 3             |
|                                   | -   No, continue to monitor the   |
|                                   |     staff member’s results and    |
|                                   |     consider QMO accreditation at |
|                                   |     a later stage. Procedure ends |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Check training + Read more ...    |
|                                   |                                   |
|                                   | Does the staff member currently   |
|                                   | hold the relevant QMO             |
|                                   | accreditation, or did QMO         |
|                                   | accreditation lapse less than 12  |
|                                   | months ago?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, the staff member will     |
|                                   |     need to complete the          |
|                                   |     mandatory QMA training. See   |
|                                   |     Training & Support. Procedure |
|                                   |     ends here until the training  |
|                                   |     has been completed            |
+-----------------------------------+-----------------------------------+
| 4                                 | Check the online test + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Confirm the staff member has      |
|                                   | passed the CLK00875 - QMA Skills  |
|                                   | Verification Test.                |
|                                   |                                   |
|                                   | Have they achieved 100% accuracy  |
|                                   | within 3 attempts?                |
|                                   |                                   |
|                                   | Yes, go to Step 5                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Provide extra support as      |
|                                   |     required                      |
|                                   | -   The staff member can do the   |
|                                   |     test again                    |
|                                   | -   Procedure ends here until the |
|                                   |     staff member has achieved     |
|                                   |     100% accuracy                 |
+-----------------------------------+-----------------------------------+
| 5                                 | Complete QMO form + Read more ... |
|                                   |                                   |
|                                   | Complete the QMO                  |
|                                   | Accreditation/Reaccreditation     |
|                                   | form. No response will be given   |
|                                   | upon submission of the form. The  |
|                                   | QMA team uses this information as |
|                                   | a way to track/monitor/undertake  |
|                                   | analysis as well as for auditing  |
|                                   | purposes.                         |
|                                   |                                   |
|                                   | See QMO Prerequisites table.      |
|                                   |                                   |
|                                   | Is the staff member assessed as   |
|                                   | effective for each prerequisite?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, provide extra support if  |
|                                   |     appropriate. Procedure ends   |
|                                   |     here until requirements are   |
|                                   |     met                           |
+-----------------------------------+-----------------------------------+
| 6                                 | Code ESSentials + Read more ...   |
|                                   |                                   |
|                                   | Note: if alternative proficiency  |
|                                   | was requested, approval by QMA    |
|                                   | QOL tools is required before the  |
|                                   | QMO skill tag can be coded in     |
|                                   | ESSentials.                       |
|                                   |                                   |
|                                   | The QMO skill tag must only be    |
|                                   | coded for up to 12 months. A date |
|                                   | earlier than 12 months may be     |
|                                   | used to align with other          |
|                                   | resources if needed, for example, |
|                                   | QOL or skill tags.                |
|                                   |                                   |
|                                   | Team Leaders/Line Managers are to |
|                                   | code the QMO skill tag in         |
|                                   | ESSentials:                       |
|                                   |                                   |
|                                   | -   Select Customer First         |
|                                   | -   Select Foundation             |
|                                   | -   Select the Quality Management |
|                                   |     Officer skill tag             |
|                                   | -   From date defaults to system  |
|                                   |     date                          |
|                                   | -   To date: code an expiry date  |
|                                   |     of up to 12 months. This      |
|                                   |     ensures the QMO               |
|                                   |     reaccreditation process is    |
|                                   |     completed every 12 months as  |
|                                   |     required. Do not leave the    |
|                                   |     date as 31.12.9999            |
|                                   | -   Competency: select            |
|                                   |     Experienced                   |
|                                   | -   Select Next and Finish        |
|                                   |                                   |
|                                   | See Essentials for more details.  |
+-----------------------------------+-----------------------------------+
| 7                                 | Retain records + Read more ...    |
|                                   |                                   |
|                                   | Retain records of the QMO         |
|                                   | accreditation for potential       |
|                                   | audits.                           |

Reaccreditation of QMOs

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | QMOs must be reaccredited every   |
|                                   | 12 months + Read more ...         |
|                                   |                                   |
|                                   | To be reaccredited, the QMO must  |
|                                   | have:                             |
|                                   |                                   |
|                                   | -   performed QMA duties in the   |
|                                   |     last 12 months                |
|                                   | -   maintained at least 95%       |
|                                   |     correctness in the work items |
|                                   |     they will be checking         |
|                                   | -   completed at least 20 QMA     |
|                                   |     checks of the relevant work   |
|                                   |     items over the last 12 months |
|                                   | -   obtained 100% accuracy on the |
|                                   |     QMA Skills verification test  |
|                                   |                                   |
|                                   | Access the staff member’s QMA     |
|                                   | data using the QMA Dashboard -    |
|                                   | Team Leader tile in Process       |
|                                   | Direct to confirm correctness     |
|                                   | rate and QMA checks completed.    |
|                                   | See the Resources page for a link |
|                                   | to the QMA Reporting Dashboard    |
|                                   | Guide. If there are discrepancies |
|                                   | with the data, contact QMA QOL    |
|                                   | tools.                            |
|                                   |                                   |
|                                   | If the work type has transitioned |
|                                   | to Process Direct and there is no |
|                                   | history or not enough QMA data in |
|                                   | the QMA Dashboard, QOL results    |
|                                   | may be used.                      |
|                                   |                                   |
|                                   | Does the QMO meet the above       |
|                                   | criteria?                         |
|                                   |                                   |
|                                   | -   Yes, they can be              |
|                                   |     reaccredited, go to Step 2    |
|                                   | -   No, they must undertake the   |
|                                   |     full accreditation process.   |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+

|                                   | Complete the QMO                  |
|                                   | Accreditation/Reaccreditation     |
|                                   | form.                             |

|                                   | No response will be given once    |
|                                   | the form has been submitted. The  |
|                                   | QMA team uses this information as |
|                                   | a way to monitor and undertake    |
|                                   | analysis and for auditing         |
|                                   | purposes.                         |
+-----------------------------------+-----------------------------------+

|                                   | The QMO skill tag must only be    |
|                                   | coded for up to 12 months. A date |
|                                   | earlier than 12 months may be     |
|                                   | used to align with other          |
|                                   | resources if needed, for example, |
|                                   | QOL or skill tags.                |
|                                   |                                   |

|                                   | code the QMO skill tag in         |
|                                   | ESSentials:                       |
|                                   |                                   |
|                                   | -   Select Customer First         |
|                                   | -   Select Foundation             |
|                                   | -   Select the Quality Management |

|                                   | -   From date defaults to system  |

|                                   | access + Read more ...            |
|                                   |                                   |
|                                   | Access is limited to those who    |
|                                   | line manage staff. The access is  |
|                                   | not automatically granted.        |
|                                   |                                   |
|                                   | If the QMA Dashboard - Team       |
|                                   | Leader tile in Process Direct     |

|                                   | ZC.C4.CF.QMA.SAMPLING.TL          |
|                                   |                                   |
|                                   | Enter detailed notes outlining    |
|                                   | why this access is required.      |
|                                   | Requests without this information |
|                                   | will be rejected.                 |
|                                   |                                   |
|                                   | The QMA team will review the      |
|                                   | request and approve access as     |
|                                   | appropriate.                      |
|                                   |                                   |
|                                   | See SAP Role Request for more     |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Viewing QMA feedback for the      |
|                                   | team + Read more ...              |
|                                   |                                   |
|                                   | Access the data from:             |
|                                   |                                   |
|                                   | -   Smart Coach, or               |
|                                   | -   the QMA Dashboard - Team      |
|                                   |     Leader tile in Process Direct |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the QMA Reporting Dashboard    |
|                                   | Guide. Email QMA QOL tools if     |
|                                   | there are issues or concerns with |
|                                   | the data displayed.               |
+-----------------------------------+-----------------------------------+
| 3                                 | One-to-one checking for new       |
|                                   | starters + Read more ...          |
|                                   |                                   |
|                                   | The dedicated QMO should assign   |
|                                   | the QMA work to themselves. They  |
|                                   | do this in Work Optimiser         |
|                                   | Customer Search > Select the      |
|                                   | Claim > Assign to Me.             |
|                                   |                                   |
|                                   | This will prevent the claim/work  |
|                                   | item from allocating to another   |
|                                   | QMO.                              |
+-----------------------------------+-----------------------------------+
| 4                                 | How staff can learn without       |
|                                   | fixing their own errors + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Each day staff must check their   |
|                                   | QMA Feedback tile in Process      |
|                                   | Direct to review:                 |
|                                   |                                   |
|                                   | -   their QMA results             |
|                                   | -   the details of all the errors |
|                                   |     identified                    |
|                                   | -   review any Operational        |
|                                   |     Blueprint links provided      |
|                                   | -   replay work items where       |
|                                   |     error/s have been identified  |
|                                   |     (selected work items only)    |
|                                   |                                   |
|                                   | Staff can also view their own     |
|                                   | historical QMA data and run       |
|                                   | reports up to 12 months, using    |
|                                   | the QMA Dashboard – Service       |
|                                   | Officer tile in Process Direct.   |
|                                   | Staff should talk to their team   |
|                                   | leaders if there are issues or    |
|                                   | concerns.                         |
|                                   |                                   |
|                                   | Team leaders and Service Support  |
|                                   | Officers will be able to          |
|                                   | determine learning needs and      |
|                                   | tailor a learning plan based on   |
|                                   | feedback trends.                  |
+-----------------------------------+-----------------------------------+
| 5                                 | QMA Replay + Read more ...        |
|                                   |                                   |
|                                   | Staff can replay selected work    |
|                                   | items if an error was identified. |
|                                   | After completing the QMA check,   |
|                                   | staff can access the replay from  |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   their QMA Feedback tile, or   |
|                                   | -   the QMA Dashboard - Service   |
|                                   |     Officer tile, using the work  |
|                                   |     item link on the QMA Sampling |
|                                   |     Report page                   |
|                                   |                                   |
|                                   | See Comparison tables for a list  |
|                                   | of work items that have the       |
|                                   | replay function.                  |
|                                   |                                   |
|                                   | Team leaders can access the QMA   |
|                                   | Replay from the QMA Dashboard -   |
|                                   | Team Leader tile, using the work  |
|                                   | item link on the QMA Sampling     |
|                                   | Report page.                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Auto sampling override + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The auto sampler can be           |
|                                   | overridden for staff who receive  |
|                                   | targeted performance support,     |
|                                   | where 100% sampling for a         |
|                                   | specified timeframe needs to      |
|                                   | apply.                            |
|                                   |                                   |
|                                   | An override should only apply     |
|                                   | when the staff member:            |
|                                   |                                   |
|                                   | -   is subject to a performance   |
|                                   |     improvement plan              |
|                                   | -   has an auto sampling rate     |
|                                   |     which is not truly reflective |
|                                   |     of their performance          |
|                                   | -   has recommenced the work      |
|                                   |     after a period or other       |
|                                   |     tasks, or                     |
|                                   | -   has had a significant change  |
|                                   |     to the work they are          |
|                                   |     processing                    |
|                                   |                                   |
|                                   | Team Leader/Line Managers are     |
|                                   | responsible for managing and      |
|                                   | monitoring overrides for their    |
|                                   | staff (applying and removing).    |
|                                   |                                   |
|                                   | For more information, see QMA     |
|                                   | Sampling Override tool.           |
