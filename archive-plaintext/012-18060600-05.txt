| Registration details              | To be registered with Medicare,   |
|                                   | an Accredited Mental Health       |
|                                   | Social Worker (AMHSW) or          |
|                                   | Accredited Social Worker (ASS)    |
|                                   | must meet the eligibility         |
|                                   | criteria.                         |
|                                   |                                   |
|                                   | For the AMHSW to provide Focussed |
|                                   | Psychological Strategies (FPS)    |
|                                   | they must supply a certificate    |
|                                   | issued by the Australian          |
|                                   | Association of Social Workers     |
|                                   | (AAWS) confirming:                |
|                                   |                                   |
|                                   | -   Current member of the AASW,   |
|                                   |     and                           |
|                                   | -   Accreditation as an Allied    |
|                                   |     Health Professional (Mental   |
|                                   |     Health Social Worker), and    |
|                                   | -   Meeting the standards for     |
|                                   |     mental health set out in      |
|                                   |     Practice Standards for Mental |
|                                   |     Health Social Workers 2014    |
|                                   |                                   |
|                                   | For the AMHSW to provide          |
|                                   | Non-directive pregnancy support   |
|                                   | counselling (PSC) they must       |
|                                   | supply a certificate issued by    |
|                                   | the Australian Association of     |
|                                   | Social Workers (AAWS) confirming: |
|                                   |                                   |
|                                   | -   Current member of the AASW,   |
|                                   |     and                           |
|                                   | -   Accreditation as an Allied    |
|                                   |     Health Professional (Mental   |
|                                   |     Health Social Worker), and    |
|                                   | -   Meeting the standards for     |
|                                   |     mental health set out in      |
|                                   |     Practice Standards for Mental |
|                                   |     Health Social Workers 2014,   |
|                                   |     and                           |
|                                   | -   A certificate of completion   |
|                                   |     in Perinatal Non-Directive    |
|                                   |     Counselling Training: V2      |
|                                   |     issued by the Australian      |
|                                   |     Psychological Society         |
|                                   |                                   |
|                                   | For ASW to provide Non-directive  |
|                                   | pregnancy support counselling     |
|                                   | (PSC) they must supply:           |
|                                   |                                   |
|                                   | -   A membership of the           |
|                                   |     Australian Association of     |
|                                   |     Social Workers (AASW)         |
|                                   |     certificate, and              |
|                                   | -   A certificate as an ASW       |
|                                   |     issued by the AASW, and       |
|                                   | -   A certificate of completion   |
|                                   |     in Perinatal Non-Directive    |
|                                   |     Counselling Training: V2      |
|                                   |     issued by the Australian      |
|                                   |     Psychological Society         |
