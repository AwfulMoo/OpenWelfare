Troubleshooting CMS

If CMS does not respond when entering the system, some steps to navigate
through these errors are:

Press [CTRL + R]

Try to navigate back to home

It may lock the customer record. If this happens, on the home page in
Customer First:

-   select log-off
-   re-enter the Customer Reference Number (CRN)
-   if locked in the record, navigate to 'LOVER' and force the unlock

Start the process again
