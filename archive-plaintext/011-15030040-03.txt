Processing a notice of reimbursement arrangement

This table describes how to process a notice of reimbursement
arrangement in the Compensation Case Management System (CCMS) once a
case has been selected from the 'Process Reimbursement' queue.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine if case is to be        |
|                                   | referred to supervisor + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The case is to be referred to     |
|                                   | supervisor only:                  |
|                                   |                                   |
|                                   | -   if the injured person or      |
|                                   |     claimant’s name has changed   |
|                                   |     in the Consumer Directory     |
|                                   |     Maintenance System (CDMS), or |
|                                   | -   'other information' needs to  |
|                                   |     be updated, for example, if a |
|                                   |     minor has turned 14 years of  |
|                                   |     age and the parent is no      |
|                                   |     longer the claimant, to add   |
|                                   |     or change a case flag, or if  |
|                                   |     a valid notice of past        |
|                                   |     benefits exists and an        |
|                                   |     amendment is required so the  |
|                                   |     'dates' can be deleted        |
|                                   |                                   |
|                                   | Does the case need to be referred |
|                                   | to supervisor?                    |
|                                   |                                   |
|                                   | Yes, select 'Yes' in the          |
|                                   | workflow. Before releasing the    |
|                                   | work form, go to Step 11 to add a |
|                                   | case note and then go to the CCMS |
|                                   | supervisor queue:                 |
|                                   |                                   |
|                                   | -   select the case and update    |
|                                   |     the details                   |
|                                   | -   return to the 'Process        |
|                                   |     Reimbursement' workflow to    |
|                                   |     continue processing until the |
|                                   |     case is actioned or closed    |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Identify if the case is a class   |
|                                   | action suit + Read more ...       |
|                                   |                                   |
|                                   | Is this case a class action suit? |
|                                   |                                   |
|                                   | This question will only appear if |
|                                   | the case does not already have a  |
|                                   | class action suit set against it. |
|                                   | Therefore, there is no            |
|                                   | requirement to set a class action |
|                                   | suit in this workflow.            |
|                                   |                                   |
|                                   | Go to Step 3.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Claimant's address + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Verify claimant’s address?        |
|                                   |                                   |
|                                   | -   Yes, only select 'Yes' if the |
|                                   |     agency has been advised of a  |
|                                   |     change of address (over the   |
|                                   |     phone or in writing), or if   |
|                                   |     other agency records have     |
|                                   |     been updated with new address |
|                                   |     details. An Address field box |
|                                   |     will appear where the new     |
|                                   |     address information can be    |
|                                   |     entered, go to Step 4         |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Go to next transaction + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If there is No valid notice of    |
|                                   | past benefits - select 'Yes', go  |
|                                   | to Step 6                         |
|                                   |                                   |
|                                   | If there is a valid notice of     |
|                                   | past benefits - select 'No', go   |
|                                   | to Step 5                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Check if services need to be      |
|                                   | removed + Read more ...           |
|                                   |                                   |
|                                   | Are there services identified     |
|                                   | after the date reimbursement was  |
|                                   | made or outside the period of     |
|                                   | liability?                        |
|                                   |                                   |
|                                   | No, go to Step 6                  |
|                                   |                                   |
|                                   | Yes, re-route to statement queue  |
|                                   | to remove services. A dropdown    |
|                                   | box menu appears with a list of   |
|                                   | transactions:                     |
|                                   |                                   |
|                                   | -   Select ‘Process statement by  |
|                                   |     claimant’ and [Enter]         |
|                                   | -   Do not release the work form, |
|                                   |     go to Step 12 to add a case   |
|                                   |     note                          |
|                                   |                                   |
|                                   | CCMS will automatically re-route  |
|                                   | the case to the selected work     |
|                                   | queue. Follow through on the case |
|                                   | to remove services. See at Table  |
|                                   | 2, Step 6 ‘Processing statement   |
|                                   | by claimant’ in Statement by      |
|                                   | claimant for Medicare             |
|                                   | Compensation Recovery.            |
+-----------------------------------+-----------------------------------+
| 6                                 | Select next transaction + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A dropdown box menu appears with  |
|                                   | a list of transactions:           |
|                                   |                                   |
|                                   | -   Adhoc enquiry or data update  |
|                                   | -   Process extension requests    |
|                                   | -   Case termination              |
|                                   | -   Request for Ntc of            |
|                                   |     Pastbenefit                   |
|                                   | -   Process statement by claimant |
|                                   | -   Process nursing home details  |
|                                   | -   Process judgment or           |
|                                   |     settlement                    |
|                                   | -   Voluntary recoveries          |
|                                   | -   Process reimbursements        |
|                                   | -   Overdue statement by claimant |
|                                   |                                   |
|                                   | Choose the desired work queue     |
|                                   | depending on the progress of the  |
|                                   | case and Press [Enter].           |
|                                   | Note: on most occasions, the      |
|                                   | 'Request for Ntc of Pastbenefit'  |
|                                   | or 'Process statement by          |
|                                   | claimant' work queue will be the  |
|                                   | selection required. Do not        |
|                                   | release the work form, go to Step |
|                                   | 12 to add a case note             |
|                                   |                                   |
|                                   | CCMS will automatically re-route  |
|                                   | the case to the selected work     |
|                                   | queue, follow through on case     |
|                                   | until further actions are         |
|                                   | complete                          |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Case status + Read more ...       |
|                                   |                                   |
|                                   | Enter the case status?            |
|                                   |                                   |
|                                   | -   select 'CLOSED' from the      |
|                                   |     dropdown box menu             |
|                                   | -   go to Step 8                  |
+-----------------------------------+-----------------------------------+
| 8                                 | Closure date + Read more ...      |
|                                   |                                   |
|                                   | Enter the closure date?           |
|                                   |                                   |
|                                   | -   the date is automatically     |
|                                   |     populated as today's date -   |
|                                   |     Press [Enter]                 |
|                                   | -   go to Step 9                  |
+-----------------------------------+-----------------------------------+
| 9                                 | CICS registration + Read more ... |
|                                   |                                   |
|                                   | Updated CICS registration?        |
|                                   |                                   |
|                                   | -   press [Enter], a message is   |
|                                   |     returned confirming           |
|                                   |     'Completed'                   |
|                                   | -   go to Step 10                 |
+-----------------------------------+-----------------------------------+
| 10                                | Finalisation advice + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Send finalisation advice?         |
|                                   |                                   |
|                                   | -   always select 'No' from the   |
|                                   |     list of options - do not      |
|                                   |     release the work form         |
|                                   | -   go to Step 11                 |
+-----------------------------------+-----------------------------------+
| 11                                | Completing email template         |
|                                   | Referring a case to compensation  |
|                                   | Finance + Read more ...           |
|                                   |                                   |
|                                   | Is there an amount payable to the |
|                                   | agency in the notice of charge?   |
|                                   |                                   |
|                                   | -   Yes, complete sections A, C,  |
|                                   |     E, F and G in the email       |
|                                   |     template Referring a case to  |
|                                   |     compensation Finance. The     |
|                                   |     Resources page contains a     |
|                                   |     link to the template. Go to   |
|                                   |     Step 12                       |
|                                   | -   No, go to Step 12             |
+-----------------------------------+-----------------------------------+
| 12                                | Add a case note + Read more ...   |
|                                   |                                   |
|                                   | If the Case Notes screen has not  |
|                                   | opened automatically, select the  |
|                                   | Add notes button from the list on |
|                                   | the right hand side of the main   |
|                                   | CCMS screen.                      |
|                                   |                                   |
|                                   | The top box shows all notes       |
|                                   | previously recorded on this case  |
|                                   |                                   |
|                                   | In the blank bottom box, record   |
|                                   | the actions taken, information    |
|                                   | requested and from whom and       |
|                                   | reasons for a decision. Use       |
|                                   | standard case notes where         |
|                                   | possible                          |
|                                   |                                   |
|                                   | Select Add to save notes. A       |
|                                   | confirmation box will appear,     |
|                                   | select Yes to add case note       |
|                                   |                                   |
|                                   | Press [F4] to release the work    |
|                                   | item, and                         |
|                                   |                                   |
|                                   | -   If this is the end of the     |
|                                   |     process the case will close,  |
|                                   |     or                            |
|                                   | -   If a Medicare history         |
|                                   |     statement or notice of charge |
|                                   |     needs to be issued, CCMS will |
|                                   |     automatically re-route the    |
|                                   |     case to the selected work     |
|                                   |     queue, follow through on case |
|                                   |     until further actions are     |
|                                   |     complete                      |
+-----------------------------------+-----------------------------------+
