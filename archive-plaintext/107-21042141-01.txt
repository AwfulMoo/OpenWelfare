This document contains information on creating invoices and navigating
transactions in SAP Finance.

For Medicare Debt Management Officers (DMO) only

SAP

Medicare Recovery uses SAP Finance to:

-   create invoices, and
-   manage the recovery of Medicare debt

Journal upload

The journal upload template makes bulk adjustments to General Ledger
(GL) accounts.

The journal entry is the written record of a financial transaction in a
double entry accounting system. Every entry contains an equal debit and
credit, together with the:

-    account number
-    description of the transaction, and
-   date of event

The Resources page contains the names and a description of the SAP
Customer Line Item Display column names and General Ledger (GL) codes.
