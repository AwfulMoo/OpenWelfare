Locating Review Types and Review Reasons

Use field help (?) for the Review Type: and Review Reason: fields on the
following IRS search screens:

-   Review Enquiry (IRRE)
-   National Search (IRNS)
-   Review Performance (IRRP)
-   Review Totals (IRRT)

The Review Class Enquiry (IRRCE) screen lists Review Types and Review
Reasons codes.

The IRRCE screen can also provide information about:

-   the Service Updates and Reviews loaded to IRS
-   which cycles are applicable to Review Types and Reasons
-   when a Review Class is ended

To access this screen, key 'IRRCE' in the Nxt: field and request a
search using one or more of the following criteria:

-   Review Type
-   Review Reason
-   Review Category, or
-   Cluster

Note: Updating is only allowed on this screen to add a new Review Cycle
to a Review for which access to update is restricted.
