|                                   |                                   |
|                                   | Note: if the customer is also     |
|                                   | claiming or receiving Family Tax  |
|                                   | Benefit (FTB), see Immunisation   |
|                                   | requirements for FTB Part A from  |
|                                   | 1 July 2018.                      |
|                                   |                                   |
|                                   | To check the CCS-AIR link status: |
|                                   |                                   |
|                                   | Within the claim activity:        |
|                                   |                                   |
|                                   | -   From [] select the child      |
|                                   | -   From [My Child Care] go to    |
|                                   |     the Child Immunisation        |
|                                   |     Details (CID) table           |
|                                   |                                   |
|                                   | Standalone update (Change of      |
|                                   | circumstance):                    |
|                                   |                                   |
|                                   | -   From [] select the child      |

|                                   |                                   |
|                                   | If the child record has not       |

|                                   | following will display:           |
|                                   |                                   |
|                                   | -   Reason Code: 5 digits,        |
|                                   |     starting with a '9', ‘8’ or   |
|                                   |     ‘7’                           |
|                                   | -   AIR Response: of R (Reject)   |
|                                   |     or W (wait)                   |
|                                   | -   Status: no information will   |
|                                   |     exist                         |
|                                   |                                   |
|                                   | Has the child record linked with  |
|                                   | the AIR?                          |
|                                   |                                   |
|                                   | -   Yes, continue processing the  |
|                                   |     claim and assess CCS based on |
|                                   |     the immunisation status       |
|                                   |     provided (either a 'Yes' or   |
|                                   |     'No'). Note: if message SR210 |
|                                   |     MY CHILD CARE: Child DOB      |
|                                   |     mismatch between CLK and AIR  |
|                                   |     presents after linking,       |
|                                   |     continue processing claim and |

|                                   | AIR + Read more ...               |

|                                   | If the child's record is not      |

|                                   |                                   |

| 3                                 | Child/ren under 3 months of age + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Children aged 3 months and under  |
|                                   | meet immunisation requirements    |
|                                   | and CCS will grant without the    |
|                                   | AIR immunisation status. Once the |
|                                   | child is over 3 months of age, a  |
|                                   | link with the AIR is needed to    |
|                                   | get the child's immunisation      |
|                                   | status and determine ongoing      |
|                                   | eligibility for CCS.              |
|                                   |                                   |
|                                   | If a link cannot be made for a    |
|                                   | child that is 3 months of age or  |
|                                   | less, do not delete the ‘R’       |
|                                   | rejected link from the CID screen |
|                                   | within the claim. Continue        |
|                                   | processing.                       |
|                                   |                                   |
|                                   | The AIR link request may fail due |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   using the parent's Medicare   |
|                                   |     number (from FA081)           |
|                                   | -   guessing the Individual       |
|                                   |     Reference Number (IRN)        |
|                                   | -   the child's Medicare number   |
|                                   |     is recorded without the IRN   |
|                                   | -   the child's name or date of   |
|                                   |     birth is recorded differently |
|                                   |     in Medicare                   |
|                                   | -   no Medicare record exists     |
|                                   |     because the child is not      |
|                                   |     enrolled and no immunisation  |
|                                   |     information has been recorded |
|                                   |     by a vaccination provider     |
|                                   |                                   |
|                                   | If any of this information is     |
|                                   | available, an attempt can be made |
|                                   | to establish a link inside a      |
|                                   | claim, after initial attempt is   |
|                                   | unsuccessful. For details go to   |
|                                   | Step 7.                           |
|                                   |                                   |
|                                   | If still unsuccessful continue    |
|                                   | processing claim.                 |
|                                   |                                   |
|                                   | To request a link after CCS has   |
|                                   | been granted, go to Step 4.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Setting up a link after CCS       |
|                                   | granted + Read more ...           |
|                                   |                                   |
|                                   | If a customer is CCS current for  |
|                                   | a child 3 months of age or under, |
|                                   | and is providing the child’s      |
|                                   | Medicare details, record the      |
|                                   | details on the child’s Child      |
|                                   | Immunisation Details (CIM) screen |
|                                   | in Process Direct.                |
|                                   |                                   |
|                                   | There are separate AIR links for  |
|                                   | Family Tax Benefit (FTB) and CCS. |
|                                   | For CCS, a link to the AIR is     |
|                                   | established using the CID/CIM     |
|                                   | tables in Process Direct.         |
|                                   | Updating the FTB Medicare         |
|                                   | Information (MEI) screen in       |
|                                   | Customer First will not establish |
|                                   | a link to the AIR for CCS.        |
|                                   |                                   |
|                                   | In Process Direct:                |
|                                   |                                   |
|                                   | From [] select the child          |
|                                   |                                   |
|                                   | In the Super Key field, key 'CIM' |
|                                   | to go to the CIM screen           |
|                                   |                                   |
|                                   | Select [Adding a new line] and    |
|                                   | code the child’s Medicare number  |
|                                   | and IRN                           |
|                                   |                                   |
|                                   | Select the Get AIR Data button    |
|                                   |                                   |
|                                   | If the AIR link is successful, a  |
|                                   | reason code starting with '10'    |
|                                   | will display                      |
|                                   |                                   |
|                                   | -   Save the AIR immunisation     |
|                                   |     status details (this may be a |
|                                   |     status of 'Yes' or 'No') and  |
|                                   |     Finalise the transaction      |
|                                   | -   A confirmed line will display |
|                                   |     on the CIM table in the       |
|                                   |     child’s record                |
|                                   |                                   |
|                                   | if the AIR link is not            |
|                                   | successful, a reason code         |
|                                   | starting with a 9, 8 or 7 will    |
|                                   | display, and an AIR Response of   |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   W, go to Step 5, or           |
|                                   | -   R, go to Step 7               |
+-----------------------------------+-----------------------------------+
| 5                                 | Waiting for the AIR response +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the AIR system cannot respond  |
|                                   | straight away, the Status column  |
|                                   | will be blank and the AIR         |
|                                   | Response will display W (wait).   |
|                                   | When waiting on a response from   |
|                                   | the AIR, this message displays    |
|                                   | within a claim: 'SR212: MY CHILD  |
|                                   | CARE: WAIT for AIR manual         |
|                                   | matching to complete'.            |
|                                   |                                   |
|                                   | If the AIR gives a W response,    |
|                                   | place the claim or transaction on |
|                                   | hold for 2 business days. This    |
|                                   | lets the AIR attempt to manually  |
|                                   | complete the link                 |
|                                   |                                   |
|                                   | State the hold reason as          |
|                                   | ‘Referral to policy/helpdesk’     |
|                                   |                                   |
|                                   | Record progress in Notes          |
|                                   |                                   |
|                                   | After 2 business days, if:        |
|                                   |                                   |
|                                   | -   the warning still presents,   |
|                                   |     go to Step 6, or              |
|                                   | -   an AIR link has been          |
|                                   |     established, assess the claim |
|                                   |     based on the information      |
|                                   |     transmitted. See Processing   |
|                                   |     CCS claims                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Claim allocated after hold and W  |
|                                   | response is still present + Read  |
|                                   | more ...                          |
|                                   |                                   |

|                                   |                                   |
|                                   | -   the claim has come off hold   |

|                                   | -   the W response is still       |
|                                   |     present                       |
|                                   |                                   |
|                                   | Include the following details in  |
|                                   | any escalations to Centrelink-AIR |
|                                   | Liaison staff:                    |
|                                   |                                   |
|                                   | -   Parent CRN:                   |
|                                   | -   Affected Child:               |
|                                   | -   Claim ID: (if applicable)     |
|                                   | -   CCS Issues page checked:      |
|                                   |     YES/NO                        |
|                                   | -   Issue:                        |
|                                   |                                   |
|                                   | Centrelink AIR Liaison staff will |
|                                   | troubleshoot the linking request. |
|                                   |                                   |
|                                   | Hold the claim for a further 2    |
|                                   | business days and document the    |
|                                   | record with the action taken.     |
|                                   |                                   |
|                                   | Procedure ends here.              |

| 7                                 | Unsuccessful link with the AIR +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If an R (reject) Reason Code is   |
|                                   | returned when attempting a link   |
|                                   | to the AIR in Process Direct,     |
|                                   | hold the cursor over the reason   |
|                                   | code and hover help text will     |
|                                   | display. For more information,    |

|                                   | such as:                          |
|                                   |                                   |
|                                   | -   the child’s details may       |
|                                   |     differ between Centrelink and |
|                                   |     Medicare (spelling or         |
|                                   |     positioning of name, date of  |
|                                   |     birth (DOB) etc.)             |
|                                   | -   using the parent's Medicare   |
|                                   |     number (from FA081)           |
|                                   | -   guessing the Individual       |
|                                   |     Reference Number (IRN)        |
|                                   | -   recording the child's         |
|                                   |     Medicare number without the   |
|                                   |     IRN                           |
|                                   | -   the child's name or date of   |
|                                   |     birth is recorded differently |
|                                   |     in Medicare                   |
|                                   | -   the child's Medicare number   |
|                                   |     was not provided              |
|                                   | -   no Medicare record exists     |
|                                   |     because the child is not      |
|                                   |     enrolled and no immunisation  |
|                                   |     information has been recorded |
|                                   |     by a vaccination provider     |
|                                   | -   the child’s hyphenated name   |
|                                   |     recorded in the Centrelink    |
|                                   |     record may incorrectly have a |
|                                   |     space recorded before or      |
|                                   |     after the hyphen              |
|                                   | -   there may be a system issue,  |
|                                   |     check Network News Updates    |
|                                   |     (NNUs)                        |
|                                   |                                   |
|                                   | If the information is now         |
|                                   | available, an attempt can be made |
|                                   | to establish a link inside a      |
|                                   | claim, after initial attempt is   |
|                                   | unsuccessful.                     |
|                                   |                                   |
|                                   | If still unsuccessful, refer the  |
|                                   | record to a Centrelink AIR        |
|                                   | Liaison Officer who can confirm   |
|                                   | the child record exists in        |
|                                   | Medicare (CDMS) and check the     |
|                                   | Medicare number and personal      |
|                                   | details.                          |
|                                   |                                   |
|                                   | Include the following details in  |
|                                   | any escalations to Centrelink AIR |
|                                   | Liaison staff:                    |
|                                   |                                   |
|                                   | -   Parent CRN:                   |
|                                   | -   Affected Child:               |
|                                   | -   Claim ID: (if applicable)     |
|                                   | -   CCS Issues page checked:      |
|                                   |     YES/NO                        |
|                                   | -   Issue:                        |
|                                   | -   CC your team leader/Service   |
|                                   |     Support Officer/s in the      |
|                                   |     email to ensure response is   |
|                                   |     received/actioned             |
|                                   | -   Use Email Classification      |
|                                   |     Official: Sensitive           |
|                                   |                                   |
|                                   | Hold the claim for 14 days using  |
|                                   | reason ‘Customer to Provide       |
|                                   | Information’ to allow time for    |
|                                   | Medicare enrolment to be          |
|                                   | completed, if:                    |
|                                   |                                   |
|                                   | -   no Medicare record exists     |
|                                   |     because the child is not      |
|                                   |     enrolled and they recently    |
|                                   |     arrived in Australia, or      |
|                                   | -   a customer indicates on the   |
|                                   |     FA081 to enrol the child in   |
|                                   |     Medicare. Ensure there is an  |

|                                   |     proof of a child's birth for  |
|                                   |     the process and scenarios     |
|                                   |     where this may be appropriate |
|                                   |                                   |
|                                   | If child details (name or DOB)    |
|                                   | are corrected in Customer First   |
|                                   | due to a difference between       |
|                                   | Medicare and Centrelink, complete |
|                                   | a SAP refresh and select          |
|                                   | Regenerate Claim. A new AIR link  |
|                                   | request will automatically occur  |
|                                   | when the regenerated claim is     |
|                                   | processed.                        |
|                                   |                                   |
|                                   | If none of the above reasons      |
|                                   | apply, try again to link:         |
|                                   |                                   |
|                                   | -   From [] select the child      |
|                                   | -   Select My Child Care > Child  |
|                                   |     Immunisation Details (CID)    |
|                                   |     table                         |
|                                   | -   Select [Adding a new line]    |
|                                   | -   Add the child’s Medicare      |
|                                   |     number (10 digits) and        |
|                                   |     Individual Reference Number   |
|                                   |     (IRN) (1 digit) and select    |
|                                   |     Proceed                       |
|                                   |                                   |
|                                   | Note: the Proceed button can be   |
|                                   | selected to attempt a link        |
|                                   | without the Medicare details.     |
|                                   | Immunisation status will be       |
|                                   | provided if an AIR link for the   |
|                                   | child already exists.             |
|                                   |                                   |
|                                   | If unable to establish a link     |
|                                   | after following these steps,      |
|                                   | refer the record to a Centrelink  |
|                                   | AIR Liaison Officer. Centrelink   |
|                                   | AIR Liaison staff will            |
|                                   | troubleshoot the linking request. |

Immunisation grace period (IGP) for CCS current customers

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer contacts due to child    |
|                                   | being placed in an Immunisation   |
|                                   | grace period + Read more ...      |
|                                   |                                   |
|                                   | The customer has become aware     |
|                                   | that their child/ren have been    |
|                                   | placed in an Immunisation Grace   |
|                                   | Period. Where the customer        |
|                                   | advises the child meets           |
|                                   | immunisation requirements.        |
|                                   |                                   |
|                                   | To view child immunisation grace  |
|                                   | period details:                   |
|                                   |                                   |
|                                   | -   go to the customers Child     |
|                                   |     care Summary table, My Child  |
|                                   |     Care icon, and                |
|                                   | -   select the Child Grace Period |
|                                   |     Details table                 |
|                                   |                                   |
|                                   | If the customer advises the child |
|                                   | is meeting immunisation           |
|                                   | requirements, investigate the     |
|                                   | record. Go to Step 2.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Investigate the record + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   From [], select the child     |
|                                   | -   In the Super Key field, key   |
|                                   |     'CIM' to go to the CIM screen |
|                                   |     and the Create Immunisation   |
|                                   |     Details table will appear     |
|                                   | -   Select Get AIR Data           |
|                                   |                                   |
|                                   | Review the AIR Details presented  |
|                                   | and take action depending on the  |
|                                   | result. If the AIR response is:   |
|                                   |                                   |
|                                   | -   accepted and status Yes, go   |
|                                   |     to Step 3                     |
|                                   | -   accepted and status is still  |
|                                   |     No (warning message No change |
|                                   |     to existing immunisation      |
|                                   |     status may present), go to    |
|                                   |     Step 4                        |
|                                   | -   Wait or Rejected, go to Step  |
|                                   |     5                             |
+-----------------------------------+-----------------------------------+
| 3                                 | AIR response is Accepted and      |
|                                   | Immunisation status is Yes + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select Save and finalise change   |
|                                   | of circumstance activity, which   |
|                                   | should now present on the         |
|                                   | child/ren’s CIM table.            |
|                                   |                                   |
|                                   | Result will update:               |
|                                   |                                   |
|                                   | -   information on the Child      |
|                                   |     Grace Details in the          |
|                                   |     customers Child Care Subsidy  |
|                                   |     Summary page My Child Care,   |
|                                   |     and                           |
|                                   | -   Immunisation Grace Period     |
|                                   |     should now be end dated       |
|                                   |                                   |
|                                   | Advise customer a reassessment    |
|                                   | letter will be issued.            |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | AIR response is Accepted and      |
|                                   | immunisation status is No + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a customer has advised the     |
|                                   | child/ren meets immunisation      |
|                                   | requirements and should not be in |
|                                   | an Immunisation Grace Period      |
|                                   | (IGP):                            |
|                                   |                                   |
|                                   | -   refer the customer to their   |
|                                   |     myGov account to check the    |
|                                   |     child’s Immunisation History  |
|                                   |     Statement                     |
|                                   | -   The Resources page links to   |
|                                   |     help for customers on how to  |
|                                   |     access the statement          |
|                                   |                                   |
|                                   | If the statement confirms:        |
|                                   |                                   |
|                                   | the child is up to date go to     |
|                                   | Step 5                            |
|                                   |                                   |
|                                   | the child is not up to date:      |
|                                   |                                   |
|                                   | -   refer the customer back to    |
|                                   |     their vaccination provider to |
|                                   |     ensure the child is fully     |
|                                   |     immunised and all details     |
|                                   |     have been provided to the AIR |
|                                   | -   Do not refer the customer to  |
|                                   |     the Australian Immunisation   |
|                                   |     Register (AIR)                |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Referral to Centrelink AIR        |
|                                   | liaison + Read more ...           |
|                                   |                                   |
|                                   | The Centrelink AIR Liaison will   |
|                                   | need to investigate child         |
|                                   | immunisation linking or           |
|                                   | immunisation status.              |
|                                   |                                   |
|                                   | Before escalating, check Network  |
|                                   | News Update (NNU) and Operational |
|                                   | Messaging (OM) for any current    |
|                                   | issues or procedural changes.     |
|                                   |                                   |
|                                   | Include the following details in  |
|                                   | all escalations to Centrelink AIR |
|                                   | Liaison:                          |
|                                   |                                   |
