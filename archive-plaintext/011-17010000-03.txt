|                                   | Read more ...                     |

|                                   | the VNSC screen is displayed.     |

|                                   | Read more ...                     |

|                                   |     Step 5                        |

| 5                                 | Expand the search + Read more ... |
|                                   |                                   |
|                                   | Perform a nationwide search:      |
|                                   |                                   |
|                                   | -   Key '0' in the Postcode       |
|                                   |     field. Dash out all other     |
|                                   |     numbers (for example 0---     |
|                                   |     ---- ) and press [Enter]      |
|                                   |     Note: Can only be used when   |
|                                   |     complete surname and first    |
|                                   |     name are used                 |
|                                   | -   Change the Surname Match and  |
|                                   |     First Name Match fields to    |
|                                   |     'P' for possible match and    |
|                                   |     press [Enter]                 |
|                                   | -   Change the First Name field   |
|                                   |     to the veteran's first name   |
|                                   |     initial only. Ensure 'P' is   |
|                                   |     in the First Name Match field |
|                                   |     and search by postcode or     |
|                                   |     state. Press [Enter]          |
|                                   | -   If no first name provided,    |
|                                   |     search by surname. Filter     |
|                                   |     search further with exact     |
|                                   |     postcode or date of birth     |
|                                   |     (can search with just the     |
|                                   |     day, month or year of birth). |
|                                   |     Press [Enter]                 |
|                                   |                                   |
|                                   | Is the patient found?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 6                                 | Select the patient + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select the patient by keying:     |
|                                   |                                   |
|                                   | -   'S' in the SEL field          |
|                                   | -   'VEXI' in the TRAN field      |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Check that at least 2 of the  |
|                                   |     search criteria match the     |
|                                   |     search result. For example,   |
|                                   |     veteran's name and birth date |
|                                   |     or veteran's name and address |
|                                   | -   If there are more than 5      |
|                                   |     matched patients, press       |
|                                   |     [PF8]-Forward to display the  |
|                                   |     next 5 and so on              |
|                                   | -   If DVA file number found then |
|                                   |     use reason code 249 when      |
|                                   |     finalising the claim          |
+-----------------------------------+-----------------------------------+
| 7                                 | Unable to locate correct          |
|                                   | patient + Read more ...           |
|                                   |                                   |
|                                   | If unable to locate the correct   |
|                                   | patient reject the service with   |
|                                   | RSN code 376 - Patient cannot be  |
|                                   | identified from information       |
|                                   | supplied.                         |
|                                   |                                   |
|                                   | Note: For trainee Service         |
|                                   | Officers or during periods of     |
|                                   | high quality control errors, a    |
|                                   | second Service Officer must:      |
|                                   |                                   |
|                                   | -   perform a VNSI name search    |
|                                   | -   confirm the patient cannot be |
|                                   |     identified                    |
|                                   | -   for Manual Claims only - key  |
|                                   |     appropriate comments under    |
|                                   |     processing notes in PaNDA     |
|                                   |     with their details and reason |
|                                   |     code 376                      |

How to determine eligibility 2-NIL ELIG

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check file number + Read more ... |
|                                   |                                   |
|                                   | If another file number is         |
|                                   | displayed at the bottom of the    |
|                                   | pend processing screen, process   |
|                                   | the claim using this new file     |
|                                   | number.                           |
|                                   |                                   |
|                                   | -   Key P at the end of the       |
|                                   |     control line and press        |
|                                   |     [Enter] twice                 |
|                                   | -   Overkey the FILE No field     |
|                                   |     with the new file number and  |
|                                   |     press [Enter]                 |
|                                   | -   Place the cursor on the       |
|                                   |     correct patient and press PF5 |
|                                   | -   Apply Reason (RSN) code 249   |
|                                   |     and continue assessing        |
|                                   |                                   |
|                                   | Check VHOI                        |
|                                   |                                   |
|                                   | Press Page Up key to swap         |
|                                   | screens. On the second screen key |
|                                   | 'VHOI,Veterans File no' and press |
|                                   | [Enter]. Press PF10 to determine  |
|                                   | if a different, eligible card     |
|                                   | number has been used to process   |
|                                   | previous services. If an eligible |
|                                   | card is present, process the      |
|                                   | claim using this new file number. |
|                                   |                                   |
|                                   | -   Key P at the end of the       |
|                                   |     control line and press        |
|                                   |     [Enter] twice                 |
|                                   | -   Overkey the FILE No field     |
|                                   |     with the new file number and  |
|                                   |     press [Enter]                 |
|                                   | -   Place the cursor on the       |
|                                   |     correct patient and press PF5 |
|                                   | -   Apply Reason (RSN) code 249   |
|                                   |     and continue assessing        |
|                                   |                                   |
|                                   | If unable to locate an active     |
|                                   | file number, go to Step 2.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Check PARS + Read more ...        |
|                                   |                                   |
|                                   | Check Prior Approval and Referral |
|                                   | System (PARS) for an approval.    |
|                                   |                                   |
|                                   | If PARS is present and payment    |
|                                   | is:                               |
|                                   |                                   |
|                                   | -   approved for item claimed,    |
|                                   |     pay with PI 3L or 3X to force |
|                                   |     match to approval             |
|                                   | -   not approved for Item         |
|                                   |     claimed, reject the services  |
|                                   |     with RSN code 289             |
|                                   |                                   |
|                                   | If PARS is not present, go to     |
|                                   | Step 3.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Check VEXI + Read more ...        |
|                                   |                                   |
|                                   | Check VEXI for additional         |
|                                   | details.                          |
|                                   |                                   |
|                                   | Key 'VEXI,Veterans File no' and   |
|                                   | press [Enter].                    |
|                                   |                                   |
|                                   | If the card type is a Personal    |
|                                   | Treatment Entitlement Card (PTEC) |
|                                   | or Specified Treatment            |
|                                   | Entitlement Card (STEC) and:      |
|                                   |                                   |
|                                   | -   the veteran is under 16 years |
|                                   |     of age, pay with PI 3L        |
|                                   | -   the address is overseas, the  |
|                                   |     postcode is listed as 0099 or |
|                                   |     the patient is over 16 years  |
|                                   |     of age, complete an           |
|                                   |     Eligibility query form and    |
|                                   |     refer to the Programme        |
|                                   |     Support Officer (PSO)         |
|                                   | -   if no card type is present    |
|                                   |     and a date of death is        |
|                                   |     recorded, complete an         |
|                                   |     Eligibility query form and    |
|                                   |     refer to the VAP Programme    |
|                                   |     Support Officer (PSO)         |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the Eligibility query     |
|                                   | form                              |
|                                   |                                   |
|                                   | If the above criteria are not     |
|                                   | met, and PSO are unable to        |
|                                   | provide further information, go   |
|                                   | to Step 4.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Reject claim + Read more ...      |
|                                   |                                   |
|                                   | If the patient does not have any  |
|                                   | eligibility, reject all services  |
|                                   | with RSN code 211.                |

|                                   | press [Enter].                    |
|                                   |                                   |
|                                   | Is the card type a Personal       |
|                                   | Treatment Entitlement Card        |
|                                   | (PTEC)?                           |

|                                   |                                   |

|                                   |                                   |

|                                   | disability.                       |

|                                   | -   No, go to Step 6              |

|                                   |                                   |
|                                   | Do the services relate to the     |
|                                   | patient’s entitlements?           |
|                                   |                                   |
|                                   | Note: for dental, podiatry or     |
|                                   | optical check, go to Step 7 for   |
|                                   | further information.              |
|                                   |                                   |
|                                   | -   Yes, pay with PI 3L           |
|                                   | -   No, go to Step 8              |
|                                   |                                   |
|                                   | Note: condition must be specific  |
|                                   | to area being treated e.g. Left   |
|                                   | or Right Knee. The only exception |

|                                   | disabilities is sufficient for    |
|                                   | payment of services.              |

|                                   | -   Fracture                      |

|                                   | -   Joint Instability             |
|                                   | -   Labral Tear                   |
