Date of effect used when recognising a specialist general practitioner

Table 1: when adding specialist general practitioner recognition (540),
the date of effect to be used is whichever is most recent:

-   the Registration Date of the Registered General practice
    registration in the Ahpra window, or
-   16 June 2021

The date of effect cannot be prior to when the legislation came into
effect. The relevant date is bolded in the table below.

Registration Type: Specialist

Specialty Name: General practice

+-----------------------+-----------------------+-----------------------+
| Registration Date     | Status                | Effective From        |
+-----------------------+-----------------------+-----------------------+
| DD/MM/YYYY            | Unregistered –        | DD/MM/YYYY            |
|                       | Surrendered           |                       |
+-----------------------+-----------------------+-----------------------+
| DD/MM/YYYY            | Registered – Current  | DD/MM/YYYY            |
+-----------------------+-----------------------+-----------------------+

Date of effect used when ending recognition as a specialist general practitioner

Table 2: when ending recognition as a specialist general practitioner
(540), FRACGP (132) or FACRRM (532) the date of effect to be used is the
Effective From date of the most recent Unregistered General practice
registration in the Ahpra window. The relevant date is bolded in the
table below.

Registration Type: Specialist

Specialty Name: General practice

+-----------------------+-----------------------+-----------------------+
| Registration Date     | Status                | Effective From        |
+-----------------------+-----------------------+-----------------------+
| DD/MM/YYYY            | Unregistered –        | DD/MM/YYYY            |
|                       | Surrendered           |                       |
+-----------------------+-----------------------+-----------------------+

Date of effect used when ending recognition as a medical practitioner

Table 3: when ending recognition of a medical practitioner who was on
the Vocational Register (130) the date of effect to be used is the
Effective From date of the most recent Unregistered General registration
in the Ahpra window. The relevant date is bolded in the table below.

Note: if ending recognition of a medical practitioner as both a FRACGP
(132)/FRACRRM(532) and having been on the Vocational Register (130),
then the date of effect to apply to any additional specialty codes (e.g.
649 – MHST), if required, is the most recent of the 2 dates of effect
(if they are different). This makes sure that the medical practitioner
can continue to access GP items while they are still considered a GP for
Medicare purposes.

| Registration    | Registration    | Status          | Effective From  |
| Type            | Date            |                 |                 |
+-----------------+-----------------+-----------------+-----------------+
| General         | DD/MM/YYYY      | Unregistered –  | DD/MM/YYYY      |
|                 |                 | Surrendered     |                 |
+-----------------+-----------------+-----------------+-----------------+

Frequently asked questions

Table 6: This table contains frequently asked questions.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: If a GP application     |
|                                   | form was received by Services     |
|                                   | Australia before COB 15/06/2021,  |
|                                   | can it be processed?              |
|                                   |                                   |
|                                   | Answer: No, it is not possible to |
|                                   | process an application form for   |
|                                   | recognition or removal of a GP    |
|                                   | after COB 15/06/2021.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: If a medical            |
|                                   | practitioner has applied to Ahpra |
|                                   | to be registered as a Specialist  |
|                                   | in General practice, can they     |
|                                   | also send their application to    |
|                                   | Services Australia to be          |
|                                   | processed as a specialist general |
|                                   | practitioner?                     |
|                                   |                                   |
|                                   | Answer: No, this cannot be        |
|                                   | processed in advance. Once the    |
|                                   | medical practitioner is           |
|                                   | registered as a Specialist in     |
|                                   | General practice with Ahpra, we   |
|                                   | will then be notified of their    |
|                                   | eligibility through the daily     |
|                                   | data feed and their recognition   |
|                                   | will be automatically processed.  |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: If a medical            |
|                                   | practitioner has overseas         |
|                                   | qualifications, should their      |
|                                   | period of registration as a       |
|                                   | specialist general practitioner   |
|                                   | be limited (such as end dated)?   |
|                                   |                                   |
|                                   | Answer: No. A medical             |
|                                   | practitioner’s recognition as     |
|                                   | general practitioner for Medicare |
|                                   | purposes is only dependent on the |
|                                   | medical practitioner’s            |
|                                   | registration with Ahpra. It is    |
|                                   | not affected by the medical       |
|                                   | practitioner’s place of           |
|                                   | qualification.                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: If a medical            |
|                                   | practitioner’s recognition as a   |
|                                   | general practitioner (GP) has     |
|                                   | been removed because of their     |
|                                   | registration with Ahpra, is there |
|                                   | any way they can appeal to the    |
|                                   | agency to be recognised as a GP   |
|                                   | for Medicare purposes?            |
|                                   |                                   |
|                                   | Answer: No. Services Australia    |
|                                   | does not have any discretion to   |
|                                   | otherwise recognise a medical     |
|                                   | practitioner as a general         |
|                                   | practitioner for Medicare         |
|                                   | purposes. Where a medical         |
|                                   | practitioner does not hold the    |
|                                   | required registration to be       |
|                                   | recognised as a GP for Medicare   |
|                                   | purposes, they should be directed |
|                                   | to Ahpra.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: If the system has       |
|                                   | recognised or removed a medical   |
|                                   | practitioner’s GP status, is it   |
|                                   | possible to view and/or re-issue  |
|                                   | the letter sent by the system?    |
|                                   |                                   |
|                                   | Answer: Yes, where these letters  |
|                                   | have been sent through the        |
|                                   | Provider Directory System (PDS),  |
|                                   | these will be viewable in the PDS |
|                                   | Correspondence screen.            |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: How does a medical      |
|                                   | practitioner contact Ahpra to     |
|                                   | make an enquiry about             |
|                                   | recognition?                      |
|                                   |                                   |
|                                   | Answer: They can contact Ahpra    |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   completing a web form enquiry |
|                                   |     on the Ahpra website, or      |
|                                   | -   calling Ahpra                 |
+-----------------------------------+-----------------------------------+
