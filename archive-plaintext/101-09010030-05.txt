Standard wording – transfer of consent

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Standard wording- transfer of     |
|                                   | consent                           |
|                                   |                                   |
|                                   | From {DATE} {NEW BUSINESS} will   |
|                                   | be providing your {SERVICES EG/   |
|                                   | WATER, ELECTRICITY, MOBILE PHONE} |
|                                   | service/s. Any government         |
|                                   | concessions that currently apply  |
|                                   | to your {OLD BUSINESS} account    |
|                                   | will be carried over to {NEW      |
|                                   | BUSINESS}. {NEW BUSINESS} will    |
|                                   | use the consent you provided to   |
|                                   | {OLD BUSINESS} to use Centrelink  |
|                                   | Confirmation eServices to obtain  |
|                                   | information from Services         |
|                                   | Australia to confirm your         |
|                                   | eligibility for any               |
|                                   | concession/rebate or service.     |
|                                   |                                   |
|                                   | You don’t have to do anything for |
|                                   | this to happen. If you would      |
|                                   | prefer that Service Australia     |
|                                   | didn’t confirm your eligibility   |
|                                   | for concessions/rebates or        |
|                                   | service you can call {NEW         |
|                                   | BUSINESS} on {CONTACT NUMBER} to  |
|                                   | withdraw your consent. If you do  |
|                                   | withdraw your consent for {NEW    |
|                                   | BUSINESS} to use Centrelink       |
|                                   | Confirmation eServices, you will  |
|                                   | need to prove your eligibility    |
|                                   | for concessions/rebate/service    |
|                                   | another way.                      |
+-----------------------------------+-----------------------------------+
