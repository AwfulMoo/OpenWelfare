This page contains information on managing compulsory Quarterly
Participation Interviews (QTIs) for Partial Capacity to Work (PCW) and
Temporary Reduced Work Capacity (TRWC) (0-7 or 8-14 hours per week) job
seekers.

On this page:

Triggering and recording a QTI requirement for a job seeker with PCW or
TRWC

Completing a QTI for job seekers with a PCW or TRWC

Triggering and recording a QTI requirement for a job seeker with PCW or TRWC

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | QTI + Read more ...               |
|                                   |                                   |
|                                   | A QTI is a compulsory requirement |
|                                   | for job seekers with an assessed: |
|                                   |                                   |
|                                   | -   PCW of 0-7 or 8-14 hours per  |
|                                   |     week, or                      |
|                                   | -   TRWC of 0-7 or 8-14 hours per |
|                                   |     week that extends beyond 12   |
|                                   |     weeks                         |
|                                   |                                   |
|                                   | A job seeker will be identified   |
|                                   | as requiring QTIs when an         |
|                                   | Employment Services Assessment    |
|                                   | (ESAt) report has been accepted   |
|                                   | and recorded by the agency and    |
|                                   | the job seeker is assessed as     |
|                                   | having either a PCW or TRWC of    |
|                                   | 0-14 hours per week, including    |
|                                   | TRWC for less than 12 weeks in    |
|                                   | duration.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Triggering the QTI requirement +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   The ESAt report is accepted,  |
|                                   |     triggering the 12 week count  |
|                                   |     for the QTI. Note: the ESAt   |
|                                   |     report will have a            |
|                                   |     'submitted' status for 28     |
|                                   |     days but this does not affect |
|                                   |     the commencement of QTI       |
|                                   |     requirements. The job         |
|                                   |     seeker's work capacity and    |
|                                   |     QTI requirements are          |
|                                   |     effective from the date the   |
|                                   |     ESAt is accepted              |
|                                   | -   If the job seeker has already |
|                                   |     lodged a medical certificate, |
|                                   |     follow the process to assess  |
|                                   |     the medical certificate and   |
|                                   |     update the Medical            |
|                                   |     Certificate (MC) screen to    |
|                                   |     either grant or not grant a   |
|                                   |     Temporary Incapacity (INP)    |
|                                   |     exemption                     |
|                                   |                                   |
|                                   | An automatic letter is generated  |
|                                   | for the job seeker advising of    |
|                                   | their mutual obligation           |
|                                   | requirements, and the requirement |
|                                   | to negotiate a Job Plan with the  |
|                                   | agency.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | An appointment is automatically   |
|                                   | booked + Read more ...            |
|                                   |                                   |
|                                   | If the job seeker does not have   |
|                                   | an exemption from their mutual    |
|                                   | obligation requirements (for      |
|                                   | example, INP) with an end date    |
|                                   | more than 2 weeks in the future,  |

|                                   | automatically booked using the    |
|                                   | 'Quarterly Interview' appointment |
|                                   | type.                             |
|                                   |                                   |
|                                   | This appointment is to ensure     |
|                                   | that the QTI is coded on the      |
|                                   | Activity and Exemption Summary    |
|                                   | (AEX) screen and the Job Plan     |
|                                   | updated.                          |
|                                   |                                   |
|                                   | If job seeker has an exemption    |
|                                   | for at least 2 weeks, their Job   |
|                                   | Plan item will be included in     |
|                                   | their Job Plan when the exemption |
|                                   | ends.                             |
+-----------------------------------+-----------------------------------+
| 4                                 | Conducting the QTI requirement    |
|                                   | appointment + Read more ...       |
|                                   |                                   |
|                                   | Phone interview pre-call          |
|                                   | notification                      |
|                                   |                                   |
|                                   | Consideration should be given to  |
|                                   | sending a pre-call notification   |
|                                   | using Desktop Electronic          |
|                                   | Messaging Capability (DEMC) for   |
|                                   | job seekers with a phone QTI      |
|                                   | appointment. See Centrelink       |
|                                   | letters online and Electronic     |
|                                   | Messaging.                        |
|                                   |                                   |
|                                   | Does the job seeker attend the    |
|                                   | appointment?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 5                                 | A Job Plan needs to be negotiated |
|                                   | with the job seeker and the AEX   |
|                                   | screen coded + Read more ...      |
|                                   |                                   |
|                                   | In Customer First, launch the     |
|                                   | Service Profiling Update          |
|                                   | Workflow. The workflow can be     |
|                                   | launched directly from the booked |
|                                   | appointment by selecting the      |
|                                   | 'execute' button.                 |
|                                   |                                   |
|                                   | Discuss the new mutual obligation |
|                                   | requirements as a result of the   |
|                                   | ESAt assessment with the job      |
|                                   | seeker.                           |
|                                   |                                   |
|                                   | Check the customer's reporting    |
|                                   | requirement and process if        |
|                                   | required.                         |
|                                   |                                   |
|                                   | Is the job seeker fully meeting   |
|                                   | their requirements through paid   |
|                                   | work or self-employment within    |
|                                   | their assessed bandwidth hours?   |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 6                                 | Job seeker is fully meeting       |
|                                   | requirements through paid work +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the Job Plan item, 'I will     |
|                                   | meet my agreed mutual obligation  |
|                                   | requirements by undertaking       |
|                                   | approved activities and report my |
|                                   | participation in them to          |
|                                   | Centrelink' to reflect the job    |
|                                   | seeker's paid work or             |
|                                   | self-employment activities        |
|                                   | included in their Job Plan and    |
|                                   | correctly recorded on the AEX     |
|                                   | screen?                           |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   The job seeker does not have  |
|                                   |     QTI requirements while they   |
|                                   |     are fully meeting their       |
|                                   |     requirements through work or  |
|                                   |     self-employment               |
|                                   | -   Advise the job seeker they    |
|                                   |     must notify the agency of any |
|                                   |     reduction in their working    |
|                                   |     hours or if they cease work.  |
|                                   |     If this happens, they will be |

|                                   |     the agency for the duration   |
|                                   |     of their PCW or TRWC          |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Select Customer Summary from  |
|                                   |     the Process Direct Landing    |
|                                   |     page                          |
|                                   | -   Enter the customer's CRN,     |
|                                   |     select Go                     |
|                                   | -   In the customer's record, key |
|                                   |     'AEX' in the Super Key,       |
|                                   |     select [Enter]                |
|                                   | -   The Activity and Exemption    |
|                                   |     Summary (AEX) screen will     |
|                                   |     display                       |
|                                   | -   Code the AEX screen with the  |
|                                   |     appropriate requirement       |
|                                   |     details for paid work (PTW)   |
|                                   |     or self-employment (SEL)      |
|                                   | -   Advise the job seeker they    |
|                                   |     must notify the agency of any |
|                                   |     reduction in their working    |
|                                   |     hours or if they cease work.  |
|                                   |     At that time, they will be    |
|                                   |     required to attend QTIs with  |
|                                   |     Services Australia for the    |
|                                   |     duration of their PCW or TRWC |
|                                   | -   See Negotiating Job Plans for |

|                                   |                                   |

|                                   | Read more ...                     |

Completing a QTI for job seekers with a PCW or TRWC

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Job seeker attends QTI + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Conduct the QTI by launching the  |
|                                   | Service Profiling Update          |
|                                   | Workflow. The workflow can be     |
|                                   | launched directly from the booked |
|                                   | appointment by selecting the      |
|                                   | 'Execute' button, or via the      |
|                                   | Workspace page in Customer First. |
+-----------------------------------+-----------------------------------+
| 2                                 | Conduct QTI + Read more ...       |
|                                   |                                   |
|                                   | During the interview discuss:     |
|                                   |                                   |
|                                   | Whether the ESAt assessment is    |
|                                   | still current and valid, or if    |
|                                   | the customer's circumstances have |
|                                   | changed                           |
|                                   |                                   |
|                                   | If the job seeker advises a       |
|                                   | change in circumstances, re-run   |
|                                   | the Job Seeker Snaphot and refer  |
|                                   | for a new ESAt if appropriate,    |
|                                   | including where:                  |
|                                   |                                   |
|                                   | -   there has been a change in    |
|                                   |     circumstances                 |
|                                   | -   the QTI requirement has       |
|                                   |     expired for a job seeker with |
|                                   |     a TRWC and the job seeker is  |
|                                   |     likely to be eligible for     |
|                                   |     QTI, as it appears they still |
|                                   |     have a work capacity of less  |
|                                   |     than 15 hours per week        |
|                                   |                                   |
|                                   | Discuss assistance                |
|                                   | recommendations from their last   |
|                                   | ESAt/Job Capacity Assessment      |
|                                   | (JCA)                             |
|                                   |                                   |
|                                   | Look at social participation      |
|                                   | options where the job seeker has  |
|                                   | been assessed as having a TRWC    |
|                                   |                                   |
|                                   | Encourage voluntary job search,   |
|                                   | including voluntary participation |
|                                   | with a Workforce Australia,       |
|                                   | Disability Employment Services    |
|                                   | (DES) or Community Development    |
|                                   | Program (CDP) provider. Note:     |
|                                   | voluntary participation with a    |
|                                   | provider does not end the QTI     |
|                                   | requirement                       |
|                                   |                                   |
|                                   | Discuss and assess any employment |
|                                   | the job seeker may be doing as    |
|                                   | paid employment may meet the job  |
|                                   | seeker's mutual obligation        |
|                                   | requirements and they may no      |
|                                   | longer be required to attend a    |
|                                   | QTI                               |
|                                   |                                   |
|                                   | Discuss voluntary work and other  |
|                                   | community assistance options      |
|                                   |                                   |
|                                   | Provide information on the        |
|                                   | benefits of work (such as Working |
|                                   | Credit and Mobility Allowance)    |
|                                   |                                   |
|                                   | Outline the benefits of planning  |
|                                   | for the future and explain        |
|                                   | assistance services like          |
|                                   | Financial Information Services    |
|                                   |                                   |
|                                   | Outline the disadvantages of long |
|                                   | term income support dependence,   |
|                                   | and                               |
|                                   |                                   |
|                                   | Explore eligibility for other     |
|                                   | income support payments           |
|                                   |                                   |
|                                   | If the job seeker has a TRWC,     |
|                                   | check when this period ends as    |
|                                   | the job seeker may no longer be   |
|                                   | required to attend QTIs and will  |
|                                   | need to re-engage with a          |
|                                   | Workforce Australia, DES or CDP   |
|                                   | provider. A referral will be      |
|                                   | required when the TRWC period     |
|                                   | ends                              |
|                                   |                                   |
|                                   | Check the job seeker knows the    |
|                                   | date of their next quarterly      |
|                                   | participation interview and when  |
|                                   | other reviews or appointments     |
|                                   | will be                           |
|                                   |                                   |
|                                   | Select the appropriate radio      |
|                                   | button once a discussion topic is |
|                                   | addressed.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete QTI + Read more ...      |
|                                   |                                   |
|                                   | After the appointment is          |
|                                   | complete, finalise the Service    |
|                                   | Profiling Update Workflow. This   |
|                                   | will automatically update the     |
|                                   | status of the Service Component   |
|                                   | to 'COM' (completed).             |
