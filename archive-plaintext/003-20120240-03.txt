|                                   | Austrade + Read more ...          |

|                                   |                                   |
|                                   | By using data from the GST        |
|                                   | exclusive G1 figures in the BAS,  |
|                                   | the Government can use a          |
|                                   | pre-existing verifiable data set. |
|                                   | This can be validated by the      |
|                                   | Government.                       |
|                                   |                                   |
|                                   | The Government acknowledges there |
|                                   | are various ways that travel      |
|                                   | agents and tour arrangement       |
|                                   | service providers can report GST  |
|                                   | figures in the BAS at G1. This    |
|                                   | varies among businesses depending |
|                                   | on their tax arrangements and     |
|                                   | business structure.               |
|                                   |                                   |

|                                   | discrepancies in grants between   |
|                                   | businesses similar in size.       |
|                                   |                                   |
|                                   | While applicants can exhaust all  |
|                                   | steps in the review and appeal    |
|                                   | process, Services Australia:      |
|                                   |                                   |
|                                   | -   must apply the precedent set  |
|                                   |     by Austrade, and              |
|                                   | -   will not forward appeals of   |
|                                   |     this nature to Austrade to    |
|                                   |     consider exceptional          |
|                                   |     circumstances                 |
|                                   |                                   |
|                                   | Tell the applicant                |
|                                   |                                   |
|                                   | The Guidelines do not allow       |
|                                   | Services Australia to use a       |
|                                   | different turnover figure to that |
|                                   | on the ATO Certificate. The       |
|                                   | Guidelines allow Services         |
|                                   | Australia to refer a decision to  |
|                                   | Austrade to consider under        |
|                                   | exceptional circumstances.        |
|                                   | However, Austrade has already     |
|                                   | made precedent setting decisions  |
|                                   | in relation to this. They         |
|                                   | determined that:                  |
|                                   |                                   |
|                                   | -   no exceptional circumstances  |
|                                   |     exist just because a G1       |
|                                   |     reporting figure did not      |
|                                   |     represent the total           |
|                                   |     transaction value (TTV),      |
|                                   |     total gross sales or total    |
|                                   |     ticket sales                  |
|                                   | -   there is no other source of   |
|                                   |     GST turnover amount outside   |
|                                   |     the ATO certificate           |
|                                   |                                   |
|                                   | If the applicant still does not   |
|                                   | agree, tell them they can contact |
