Cancellation of FTB due to the Sunset Clause

| 1                                 | Check FTB customer's record +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If FTB has been cancelled due to  |
|                                   | the Sunset Clause, the status     |
|                                   | will be CAN-L3Y, CAN-N30 (manual  |
|                                   | override), CAN-N3P (partner       |
|                                   | non-lodger) or CAN-N3Y).          |
|                                   |                                   |

|                                   |                                   |

|                                   |     for the relevant financial    |
|                                   |     year/s. Note: if              |
|                                   |     reconciliation has been       |
|                                   |     completed advise outcome and  |
|                                   |     record details on a DOC       |
|                                   | -   Check Debt List (OPDL) screen |
|                                   |     for outstanding non-lodger    |
|                                   |     debts                         |
|                                   | -   Check DOCs detailing PIP      |
|                                   |     information and discuss with  |
|                                   |     the customer                  |
|                                   | -   Check letters issued on the   |
|                                   |     History Summary (HS) screen   |

| 2                                 | Tax return lodged or not required |

|                                   | years?                            |

| 3                                 | Sunset Clause + Read more ...     |
|                                   |                                   |
|                                   | FTB cannot be restored if it is   |
|                                   | cancelled due to the Sunset       |
|                                   | Clause.                           |
|                                   |                                   |
|                                   | The customer can re-claim FTB     |
|                                   | instalments only if adequate      |
|                                   | income (either ATO income or not  |
|                                   | required to lodge advise) details |
|                                   | are available.                    |
|                                   |                                   |
|                                   | FTB can be paid from:             |
|                                   |                                   |
|                                   | the date the last outstanding tax |
|                                   | return is lodged; this date is    |
|                                   | found in the Date of Lodgement    |
|                                   | field on the Families Income      |
|                                   | Previous Year (FIPY) screen, or   |
|                                   |                                   |
|                                   | the date the customer last        |
|                                   | advised they were not required to |
|                                   | lodge a tax return for an         |
|                                   | applicable year: this date is     |
|                                   | found on the AMR next to the      |
|                                   | applicable income update on the   |
|                                   | FAO Income for Previous Year      |
|                                   | (FIPY) screen (or the AMR on the  |
|                                   | FAO Non-Lodgement Exemption       |
|                                   | (FNLE) screen where non-lodgement |
|                                   | advice was lodged with the ATO),  |
|                                   | and                               |
|                                   |                                   |
|                                   | -   FTB reconciliation has        |
|                                   |     completed for all outstanding |
|                                   |     years, or                     |
|                                   | -   all outstanding non-lodger    |
|                                   |     debts are fully repaid        |
|                                   |                                   |
|                                   | If the customer advises:          |
|                                   |                                   |
|                                   | a tax return has been lodged, but |
|                                   | actual income details have not    |
|                                   | yet been transferred from the     |
|                                   | ATO, see Australian Taxation      |
|                                   | Office (ATO) information not      |
|                                   | received for family assistance    |
|                                   | reconciliation or lump sum claim  |
|                                   |                                   |
|                                   | they (or their partner) are not   |
|                                   | required to lodge a tax return    |
|                                   | for one of the outstanding years  |
|                                   | as per advice from the tax office |
|                                   | or an accountant:                 |
|                                   |                                   |
|                                   | -   Run Family Assistance Office  |
|                                   |     non lodgement exemption       |
|                                   |     workflow to record            |
|                                   |     non-lodgement exemption       |
|                                   | -   Check or update the income    |
|                                   |     estimate for the relevant     |
|                                   |     year. See Recording not       |
|                                   |     required to lodge advice and  |
|                                   |     previous year income          |
|                                   |                                   |
|                                   | Go to Step 4.                     |

| 9                                 | Email received from Families      |
|                                   | Reconciliation Team + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | When Families Reconciliation Team |
|                                   | responds to request:              |
|                                   |                                   |
|                                   | Reinstate FTB via a restoration   |
|                                   | activity on the Benefit Action    |
|                                   | (BA) screen (if FTB cancelled in  |
|                                   | the previous financial year, a    |
|                                   | dummy FTB NCL will be required to |
|                                   | commence FTB from the deferral    |
|                                   | start date)                       |
|                                   |                                   |
|                                   | Annotate the Sunset Special Circ. |
|                                   | Request DOC with the following    |
|                                   | information:                      |
|                                   |                                   |
|                                   | -   Type of deferral requested:   |
|                                   |     family and domestic           |
|                                   |     violence/severe               |
|                                   |     hardship/severe illness       |
|                                   |     (remove as required)          |
|                                   | -   Evidence provided             |
|                                   | -   Period of deferral (start and |
|                                   |     end dates)                    |
|                                   | -   Outcome of deferral           |
|                                   |     (restoration of FTB and       |
|                                   |     details of arrears paid)      |
|                                   |                                   |
|                                   | Customers are expected to resolve |
|                                   | their non-lodger circumstances by |
|                                   | the end of the deferral period    |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   lodging all outstanding tax   |
|                                   |     returns                       |
|                                   | -   advising they or their        |
|                                   |     partner are not required to   |
|                                   |     lodge, or                     |
|                                   | -   by repaying all outstanding   |
|                                   |     non-lodger debts              |
|                                   |                                   |
|                                   | If they fail to do so and their   |
|                                   | special circumstances no longer   |

| 11                                | Advise the customer + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | FTB cannot be restored if         |
|                                   | cancelled due to the Sunset       |
|                                   | Clause. FTB must be re-claimed.   |
|                                   | See Initial contact by customers  |
|                                   | claiming payments for families.   |
|                                   |                                   |
|                                   | Note: if the customer separates   |
|                                   | from their partner and FTB was    |
|                                   | cancelled due to their partner's  |
|                                   | outstanding tax returns, they can |
|                                   | re-claim FTB. FTB arrears will be |
|                                   | paid from the date of separation. |
|                                   |                                   |
|                                   | If the customer partners, or      |
|                                   | reconciles with their ex-partner, |
|                                   | who has had 3 or more outstanding |
|                                   | tax returns, and at least 1       |
|                                   | non-lodger debt is outstanding,   |
|                                   | FTB will automatically cancel due |
|                                   | to the Sunset Clause once their   |
|                                   | relationship details are updated. |
|                                   |                                   |
|                                   | FTB can be re-claimed as          |
|                                   | fortnightly payments if:          |
|                                   |                                   |
|                                   | -   all outstanding tax returns   |
|                                   |     are lodged and actual income  |
|                                   |     has been received from the    |
|                                   |     Australian Taxation Office    |
|                                   |     (ATO) and FTB reconciliation  |
|                                   |     has occurred for all relevant |
|                                   |     financial years. (check Act   |
|                                   |     Inc DOL: field on the FTB     |
|                                   |     Status Override (FIPY) screen |
|                                   |     for customer or partner       |
|                                   |     actual lodgement date) or     |
|                                   | -   not required to lodge details |
|                                   |     are provided and FTB          |
|                                   |     reconciliation has occurred   |
|                                   |     for relevant financial years, |
|                                   |     or                            |
|                                   | -   all debts with an outstanding |
|                                   |     non-lodger debt status are    |
|                                   |     fully repaid                  |
|                                   |                                   |
|                                   | The instalment claim must be      |
|                                   | granted no earlier than the date  |
|                                   | the income was confirmed (check   |
|                                   | Act Inc DOL: field on the FIPY    |
|                                   | screen) or all non-lodger debts   |
|                                   | were repaid. Any sunset arrears   |
|                                   | prior to this date in the current |
|                                   | and immediate previous financial  |
|                                   | year cannot be paid until:        |
|                                   |                                   |
|                                   | -   the customer has lodged an    |
|                                   |     FTB lump sum claim. See       |
|                                   |     Resources for an example and  |
|                                   |     coding requirements, and      |
|                                   | -   the customer's (and partner's |
|                                   |     if applicable) Actual         |
|                                   |     Adjusted Taxable Income       |
|                                   |     (AATI) is confirmed.          |
|                                   |     Note: customers must lodge a  |
|                                   |     lump sum claim and confirm    |
|                                   |     income by the end of the      |
|                                   |     lodgement year                |
|                                   |                                   |
|                                   | Any periods that are paying FTB   |
|                                   | arrears prior to the date AATI is |
|                                   | confirmed / non-lodger debts are  |
|                                   | repaid, must be overridden on the |
|                                   | FTB status override (FYSO) screen |
|                                   | as:                               |
|                                   |                                   |
|                                   | -   Status: CAN                   |
|                                   | -   Reason: OTH (other)           |
|                                   |                                   |
|                                   | Any previous financial year lump  |
|                                   | sum assessments that are          |
|                                   | generating prior to a FTB lump    |
|                                   | sum claim being lodged need to be |
|                                   | overridden on FYSO as a separate  |
|                                   | line to the current financial     |
|                                   | year:                             |
|                                   |                                   |
|                                   | -   Status: CAN                   |
|                                   | -   Reason: OTH (other)           |
|                                   |                                   |
|                                   | For information on coding         |
|                                   | estimate for when FTB is claimed  |
|                                   | after the Sunset Clause has been  |
|                                   | applied, see Date of event (DOV)  |
|                                   | for family assistance income      |
|                                   | estimates                         |
|                                   |                                   |
|                                   | For manual processing of FTB new  |
|                                   | claims and the treatment of       |
|                                   | incorrect arrears, see Manual     |
|                                   | processing of Family Tax Benefit  |
|                                   | (FTB) claims                      |
|                                   |                                   |
|                                   | Lump sum claim lodged             |
|                                   |                                   |
|                                   | -   If an FTB lump sum claim is   |
|                                   |     lodged after the end of the   |
|                                   |     financial year, coding which  |
|                                   |     prevents FTB assessment for   |
|                                   |     the applicable period on the  |
|                                   |     FYSO screen must be deleted   |
|                                   | -   Actual income details will be |
|                                   |     needed for the claim to be    |
|                                   |     determined and any arrears    |
|                                   |     paid, if entitled             |
