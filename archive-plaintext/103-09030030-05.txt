Processing a Centrepay deduction for a customer under the legacy weekly payment model

This table provides an example of processing a Centrepay deduction for a
customer under the legacy weekly payment model.

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| Customer is under the legacy      | A pension customer's pay period   |
| payment model                     | is 20 October 2017 to 2 November  |
|                                   | 2017, and is under the legacy     |
|                                   | payment model.                    |
|                                   |                                   |
|                                   | The customer makes a request on   |
|                                   | 31 October 2017 to start a        |
|                                   | deduction immediately.            |
|                                   |                                   |
|                                   | As the customer has made the      |
|                                   | request in the second week of     |
|                                   | their pay period (the first week  |
|                                   | was from 20 October 2017 to 26    |
|                                   | October 17), the deduction will   |
|                                   | now have to be coded in the first |
|                                   | week of their next pay period,    |
|                                   | that is, 3 November 2017 to 16    |
|                                   | December 2017.                    |
+-----------------------------------+-----------------------------------+
