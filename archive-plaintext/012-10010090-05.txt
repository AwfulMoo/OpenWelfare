Aged Care Access Incentive (ACAI) bank statement reference number

The bank statement related to an ACAI payment has a reference number.
This includes the:

-   practitioner’s provider number stem (first 6 digits), and
-   payment run number

For example, if the provider number is 123456AB and the payment run
number is 043, the reference number is 123456043.
