+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer requests information     |
|                                   | about the debt amount and/or      |
|                                   | reason for the debt + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the Debt Deferral pending      |
|                                   | reason:                           |
|                                   |                                   |
|                                   | -   is being applied with the     |
|                                   |     current activity go to step 2 |
|                                   | -   already applies go to step 3  |
+-----------------------------------+-----------------------------------+
| 2                                 | Debt Deferral pending reason is   |
|                                   | being applied with current        |
|                                   | activity. + Read more ...         |
|                                   |                                   |
|                                   | If the customer wants to know why |
|                                   | the amount of debt is being       |
|                                   | calculated and the Debt Deferral  |
|                                   | pending reason is being applied   |

|                                   | current activity (for example,    |
|                                   | when recording non-lodgement of a |
|                                   | tax return):                      |

|                                   |     screen                        |
