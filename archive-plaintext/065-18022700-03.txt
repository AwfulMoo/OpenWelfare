This page contains processes for the release of financial information to
service providers and information regarding signature requirements for
Aged Care Calculation of your cost of care (means assessment) forms.

On this page:

Financial disclosure to service providers

Signature requirements for the Aged Care Calculation of your cost of
care forms

Financial disclosure to service providers

Table 1: This table describes the processes of recording a care
recipient’s consent to release financial information to a service
provider, recording the withdrawal of previously provided consent and
actioning requests for the release of financial information to service
providers.

Note: portions of this process can only be completed by Aged Care
Programmes Branch staff.

Signature requirements for the Aged Care Calculation of your cost of care forms

Table 2: This table outlines the signature requirements for the Aged
Care Calculation of your cost of care forms.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Form is signed and the signatory  |
|                                   | is a person other than the care   |
|                                   | recipient + Read more ...         |
|                                   |                                   |
|                                   | Where a person other than the     |
|                                   | care recipient has signed the     |
|                                   | form, they must also provide a    |
|                                   | name and contact details to aged  |
|                                   | care skill tagged staff to        |
|                                   | determine whether that person is  |
|                                   | an authorised third party.        |
|                                   |                                   |
|                                   | Where the signatory is an         |
|                                   | authorised third party            |
|                                   |                                   |
|                                   | A means assessment can be         |
|                                   | accepted where a signatory is     |
|                                   | already recorded in:              |
|                                   |                                   |
|                                   | -   Customer First as a current   |
|                                   |     correspondence nominee        |
|                                   | -   ACMPS as an aged care         |
|                                   |     nominee, or                   |
|                                   | -   Customer First as a Fast Note |
|                                   |     showing documentation has     |
|                                   |     been provided authorising     |
|                                   |     them to act on the care       |
|                                   |     recipient’s behalf            |
|                                   |                                   |
|                                   | Ensure that all authorised        |
|                                   | parties have signed the form when |
|                                   | a document authorises more than   |
|                                   | one person jointly.               |
|                                   |                                   |
|                                   | Where the signatory is not an     |
|                                   | authorised third party            |
|                                   |                                   |
|                                   | Where a signatory has not         |
|                                   | previously provided documentation |
|                                   | that grants them authority to     |
|                                   | sign on behalf of the care        |
|                                   | recipient, they are requested to  |
|                                   | do so in the form.                |
|                                   |                                   |
|                                   | A person can apply for a          |
|                                   | calculation of your cost of care  |
|                                   | on behalf of someone else if:     |
|                                   |                                   |
|                                   | -   they hold a power of attorney |
|                                   |     or guardianship order         |
|                                   | -   a letter from a doctor, nurse |
|                                   |     or similar health             |
|                                   |     professional is provided      |
|                                   |     stating that the care         |
|                                   |     recipient is unable to sign   |
|                                   |     the form                      |
|                                   | -   the application is made by    |
|                                   |     the Director if Nursing at    |
|                                   |     the aged care home where the  |
|                                   |     care recipient is a resident  |
|                                   |                                   |
|                                   | Where the care recipient is       |
|                                   | deceased:                         |
|                                   |                                   |
|                                   | -   only the executor of the      |
|                                   |     estate or person holding a    |
|                                   |     letter of administration can  |
|                                   |     sign/act on their behalf      |
|                                   | -   Any previous power of         |
|                                   |     attorney or guardianship      |
|                                   |     orders are no longer valid    |
|                                   |                                   |
|                                   | Where acceptable documentation is |
|                                   | received with the form:           |
|                                   |                                   |
|                                   | -   the form can be accepted      |
|                                   | -   in circumstances where a      |
|                                   |     document authorises more than |
|                                   |     1 person jointly, all         |
|                                   |     authorised parties must have  |
|                                   |     signed the form               |
|                                   |                                   |
|                                   | Where acceptable documentation    |
|                                   | has not been provided with the    |
|                                   | form:                             |
|                                   |                                   |
|                                   | the form cannot be accepted       |
|                                   |                                   |
|                                   | there is no authority to contact  |
|                                   | the signatory as they are not an  |
|                                   | authorised third party:           |
|                                   |                                   |
|                                   | -   an Aged Care Request for      |
|                                   |     Further Information letter    |
|                                   |     must be sent to the care      |
|                                   |     recipient asking for either   |
|                                   |     their signature, or 1 of the  |
|                                   |     previously listed documents   |
|                                   |     that authorises the third     |
|                                   |     party to sign on their behalf |
|                                   | -   See the Resources page for    |
|                                   |     link to the Request for       |
|                                   |     Further Information letter    |
|                                   |     templates                     |
|                                   |                                   |
|                                   | complete a Progress of Claim DOC  |
|                                   | with action taken                 |
