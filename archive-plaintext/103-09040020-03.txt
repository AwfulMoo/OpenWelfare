Post Approval Check

Table 1:

This table describes how to complete a PAC after the Business
Application (BA) has been approved.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | PAC activity received + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | A Centrepay PAC is to be          |
|                                   | completed after a business have   |
|                                   | been approved for 30 business     |
|                                   | days. The PAC activity will be    |
|                                   | received via WLM.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Contacting the business + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Attempt to call the business.     |
|                                   | Arrange for a date and time to    |
|                                   | complete the PAC and request the  |
|                                   | Deduction Authority to be emailed |
|                                   | to Services Australia. If unable  |
|                                   | to contact by phone try again the |
|                                   | following day.                    |
|                                   |                                   |
|                                   | Is the business contactable?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Send email to the business + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Send an email to the business     |
|                                   | requesting that they make contact |
|                                   | by phone within 10 business days. |
|                                   |                                   |
|                                   | Email text:                       |
|                                   |                                   |
|                                   | I am writing to you about your    |
|                                   | recent approval to use Centrepay. |
|                                   |                                   |
|                                   | Following your approval, we would |
|                                   | like to discuss your obligations  |
|                                   | and responsibilities under the    |
|                                   | Centrepay Policy and Terms.       |
|                                   |                                   |
|                                   | I have tried to contact you by    |
|                                   | phone but have been unsuccessful. |
|                                   |                                   |
|                                   | Please contact me by (date + 10   |
|                                   | business days) on (phone number). |
|                                   |                                   |
|                                   | Please also send a copy of your   |
|                                   | Deduction Authority that you will |
|                                   | be using to obtain customer       |
|                                   | consent to (email address) within |
|                                   | the same timeframe.               |
|                                   |                                   |
|                                   | Has the business made contact?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, email the business        |
|                                   |     advising their approval to    |
|                                   |     use Centrepay will be         |
|                                   |     withdrawn and provide 10      |
|                                   |     business days for the         |
|                                   |     business to make contact      |
|                                   |                                   |
|                                   | Email text:                       |
|                                   |                                   |
|                                   | I am writing to you to advise     |
|                                   | that the agency is considering    |
|                                   | suspending/withdrawing your       |
|                                   | business’s approval to use        |
|                                   | Centrepay.                        |
|                                   |                                   |
|                                   | Your business was required to     |
|                                   | contact the agency and provide a  |
|                                   | copy of your Deduction Authority  |
|                                   | by (date).                        |
|                                   |                                   |
|                                   | I have tried to contact you       |
|                                   | numerous times without success.   |
|                                   |                                   |
|                                   | If your business does not make    |
|                                   | contact by (date + 10 business    |
|                                   | days), your approval to use       |
|                                   | Centrepay will be withdrawn.      |
|                                   |                                   |
|                                   | Note: If the business does not    |
|                                   | respond to the email, a referral  |
|                                   | should be made to the Program     |
|                                   | Support Manager to withdraw       |
|                                   | approval. After approval has been |
|                                   | withdrawn, use the referral       |
|                                   | template to have the PAC deleted. |
+-----------------------------------+-----------------------------------+
| 4                                 | Performing the PAC + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The PAC will be discussed with    |
|                                   | the business.                     |
|                                   |                                   |
|                                   | Ensure a copy of the Deduction    |
|                                   | Authority is obtained prior to    |
|                                   | the phone call. Check that it     |
|                                   | meets the standards as set out in |
|                                   | the Procedural Guide. If not,     |
|                                   | this needs to be discussed with   |
|                                   | the business during the PAC.      |
|                                   | Ensure an updated version of the  |
|                                   | Deduction Authority is obtained   |
|                                   | and uploaded before completing    |
|                                   | the PAC. The business may need    |
|                                   | time to update the wording. Set   |
|                                   | up a calendar reminder to ensure  |
|                                   | follow up.                        |
|                                   |                                   |
|                                   | If a copy of the Deduction        |
|                                   | Authority is not obtained within  |
|                                   | the period set, send an email to  |
|                                   | the Third Party Programmes Team – |
|                                   | Program Support Manager so they   |
|                                   | can apply a sanction.             |
|                                   |                                   |
|                                   | The PAC covers:                   |
|                                   |                                   |
|                                   | Business and contact details      |
|                                   |                                   |
|                                   | -   Centrepay support and         |
|                                   |     helpdesk information          |
|                                   | -   Obligations of an approved    |
|                                   |     business                      |
|                                   | -   Review of the business’s      |
|                                   |     deduction authority to ensure |
|                                   |     it meets requirements         |
|                                   | -   Where relevant, clarification |
|                                   |     of responsibilities under the |
|                                   |     Centrepay Policy or Terms     |
|                                   |     related to specific service   |
|                                   |     reasons                       |
|                                   |                                   |
|                                   | Refer to the question sets        |
|                                   | spreadsheet for a full list of    |
|                                   | discussion points, located in the |
|                                   | relevant shared drive, as well as |
|                                   | on the Resources Tab.             |
+-----------------------------------+-----------------------------------+
| 5                                 | PAC completed + Read more ...     |
|                                   |                                   |
|                                   | Once the PAC discussion has been  |
|                                   | completed:                        |
|                                   |                                   |
|                                   | -   DOC the record with the below |
|                                   |     text                          |
|                                   | -   upload the example of the     |
|                                   |     deduction authority to the    |
|                                   |     business record               |
|                                   | -   update the business record if |
|                                   |     any details are incorrect or  |
|                                   |     have changed                  |
|                                   | -   send a PAC follow up email to |
|                                   |     the business. Refer to the    |
|                                   |     wording located in the        |
|                                   |     relevant shared driver for an |
|                                   |     example of the email.         |
|                                   |                                   |
|                                   | PAC completed text:               |
|                                   |                                   |
|                                   | Post Approval Check conducted     |
|                                   | with (Name) on (date)             |
|                                   |                                   |
|                                   | Confirmed ABN/Business            |
|                                   | Name/Trading Name                 |
|                                   |                                   |
|                                   | Confirmed address Details/Contact |
|                                   | Officers - additional contact     |
|                                   | officer provided                  |
|                                   |                                   |
|                                   | Example Deduction Authority       |
|                                   | reviewed and uploaded to the      |
|                                   | record                            |
|                                   |                                   |
|                                   | Note: approval for Centrepay will |
|                                   | need to be withdrawn if the       |
|                                   | business no longer accepts the    |
|                                   | Policy and Terms. If withdrawal   |
|                                   | of approval is required, escalate |
|                                   | to a Program Support Manager.     |
+-----------------------------------+-----------------------------------+
