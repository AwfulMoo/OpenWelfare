This page contains information about the cancellation or rejection of
Single Income Family supplement (SIFS).

SIFS claims

SIFS cancellation or rejection

This table describes reasons SIFS may be cancelled or rejected.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | FTB customers + Read more ...     |
|                                   |                                   |
|                                   | SIFS is also paid automatically   |
|                                   | to Family Tax Benefit (FTB)       |
|                                   | customers during FTB              |
|                                   | reconciliation or lump sum claim  |
|                                   | assessment. See Cancellation,     |
|                                   | rejection and Current Zero Rate   |
|                                   | codes for Family Tax Benefit      |
|                                   | (FTB).                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Standalone claims + Read more ... |
|                                   |                                   |
|                                   | SIFS standalone claims are        |
|                                   | assessed as FTB LSC activities.   |
|                                   | Only SIFS entitlement is          |
|                                   | assessed.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Claim finalisation + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Claim finalisation can result in  |
|                                   | a status of cancelled or rejected |
|                                   | (which includes claims that are   |
|                                   | not effective):                   |
|                                   |                                   |
|                                   | -   Cancellation - SIFS can be    |
|                                   |     claimed annually, so the      |
|                                   |     entitlement period is         |
|                                   |     automatically ended for each  |
|                                   |     financial year, similar to    |
|                                   |     FTB lump sum claim            |
|                                   |     assessment.                   |
|                                   | -   Not effective (NEF) - the     |
|                                   |     claim cannot be assessed as   |
|                                   |     information or documentation  |
|                                   |     required to assess SIFS       |
|                                   |     entitlement has not been      |
|                                   |     provided as requested. The    |
|                                   |     customer will need to lodge a |
|                                   |     new claim to receive SIFS.    |
|                                   | -   Rejection - no eligibility    |
|                                   |     for the period claimed.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Claim rejection + Read more ...   |
|                                   |                                   |
|                                   | SIFS may be rejected (or made not |
|                                   | effective) automatically during   |
|                                   | claim assessment based on claim   |
|                                   | circumstance data, or manually    |
|                                   | via the Benefit Action (BA)       |
|                                   | screen before finalising the      |
|                                   | claim.                            |
+-----------------------------------+-----------------------------------+
| 5                                 | Restoration + Read more ...       |
|                                   |                                   |
|                                   | SIFS cannot be restored.          |
+-----------------------------------+-----------------------------------+
