This page contains steps on how to create an Unverified IHI in the
Healthcare identifiers (HI) service.

Creating an Unverified IHI

Creating an Unverified IHI

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Logon + Read more ...             |
|                                   |                                   |
|                                   | -   Logon to the Consumer         |
|                                   |     Directory maintenance System  |
|                                   | -   On the CDMS home page, select |
|                                   |     Register New Consumer         |
|                                   | -   The Registration Type screen  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Navigate to the Search for        |
|                                   | Consumer (Register) screen + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Healthcare Identifiers in the |
|                                   |     Program field                 |
|                                   | -   New Consumer                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Conduct a search + Read more ...  |
|                                   |                                   |
|                                   | Conduct a basic, detailed or      |
|                                   | address search to find the IHI    |
|                                   | for the individual.               |
|                                   |                                   |
|                                   | See Search for Individual         |
|                                   | Healthcare Identifier (IHI) in    |
|                                   | Healthcare Identifiers (HI)       |
|                                   | Service.                          |
+-----------------------------------+-----------------------------------+
| 4                                 | Check for matching IHI + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a matching IHI is:             |
|                                   |                                   |
|                                   | found to an existing:             |
|                                   |                                   |
|                                   | -   Verified IHI, let the         |
|                                   |     individual know the Verified  |
|                                   |     IHI. Procedure ends here      |
|                                   | -   Unverified IHI, the           |
|                                   |     individual can ask to         |
|                                   |     continue to verify the        |
|                                   |     existing IHI or update        |
|                                   |     details on the existing       |
|                                   |     unverified IHI                |
|                                   |                                   |
|                                   | not found:                        |
|                                   |                                   |
|                                   | -   Select New Consumer           |
|                                   | -   The New IHI Details screen    |
|                                   |     displays and defaults to the  |
|                                   |     Personal Details tab          |
|                                   |     Note: the personal details    |
|                                   |     are populated with any search |
|                                   |     details previously entered.   |
|                                   |     See Table 2                   |
+-----------------------------------+-----------------------------------+

Add demographic details to an Unverified IHI

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Key or select personal details    |
|                                   | for the individual + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Address Details tab    |
|                                   | -   Enter or select address       |
|                                   |     details for the individual    |
|                                   |                                   |
|                                   | See also: Rapid Addressing for    |
|                                   | Medicare.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Select Contact Details tab + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Enter or select contact details   |
|                                   | for the individual.               |
+-----------------------------------+-----------------------------------+
| 3                                 | Select Comments tab + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select the Comments tab and add   |
|                                   | comments in the free text box if  |
|                                   | necessary.                        |
|                                   |                                   |
|                                   | Note: the Status Details tab is   |
|                                   | not applicable for creating       |
|                                   | unverified IHIs. This tab is used |
|                                   | for recording Evidence of         |
|                                   | Identity.                         |
+-----------------------------------+-----------------------------------+

Finalise creation of Unverified IHI

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Select Validate + Read more ...   |
|                                   |                                   |
|                                   | Are there validation errors or    |
|                                   | missing mandatory information?    |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   An error message is returned. |
|                                   |     Change invalid or add missing |
|                                   |     details                       |
|                                   | -   Select Validate               |
|                                   |                                   |
|                                   | No, the Confirm New IHI Details   |
|                                   | screen displays                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Check the Confirm New IHI Details |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Check the details on the Confirm  |
|                                   | New IHI Details screen.           |
|                                   |                                   |
|                                   | Are the details correct?          |
|                                   |                                   |
|                                   | Yes, select Update                |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   select Change                 |
|                                   | -   correct errors and select     |
|                                   |     Update                        |
|                                   |                                   |
|                                   | Note: select Print to print the   |
|                                   | Confirm New IHI Details screen if |
|                                   | needed.                           |
|                                   |                                   |
|                                   | The Registration Type screen      |
|                                   | displays:                         |
|                                   |                                   |
|                                   | -   consumer ID                   |
|                                   | -   name                          |
|                                   | -   date of birth                 |
|                                   | -   registration start date       |
|                                   | -   IHI                           |
|                                   | -   IHI status                    |
