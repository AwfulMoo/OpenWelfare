| 5                                 | Determine if a further period of  |
|                                   | temporary incapacity exemption    |
|                                   | can be granted + Read more ...    |
|                                   |                                   |
|                                   | A further period of temporary     |
|                                   | incapacity exemption can be       |
|                                   | granted if the job seeker:        |
|                                   |                                   |
|                                   | -   is still unable to work or    |
|                                   |     otherwise meet their mutual   |
|                                   |     obligation requirements for 8 |
|                                   |     hours or more per week due to |
|                                   |     an incapacity, and            |
|                                   | -   the incapacity is temporary.  |
|                                   |     This includes temporary       |
|                                   |     exacerbations of permanent    |
|                                   |     conditions                    |
|                                   |                                   |
|                                   | Points to consider:               |
|                                   |                                   |
|                                   | All medical conditions listed on  |
|                                   | the medical certificate, except   |
|                                   | where:                            |
|                                   |                                   |
|                                   | -   The primary condition listed  |
|                                   |     is wholly or mainly due to    |
|                                   |     drug and/or alcohol           |
|                                   |     dependency or misuse and the  |
|                                   |     job seeker is not a Community |
|                                   |     Development Program (CDP)     |
|                                   |     participant. A temporary      |
|                                   |     incapacity exemption cannot   |
|                                   |     be granted for this medical   |
|                                   |     certificate. This is whether  |
|                                   |     or not there is a secondary   |
|                                   |     temporary condition not       |
|                                   |     related to drug or alcohol    |
|                                   |     for which an exemption may be |
|                                   |     applied. If the medical       |
|                                   |     certificate does not          |
|                                   |     differentiate between primary |
|                                   |     and secondary, determine if   |
|                                   |     the customer's incapacity is  |
|                                   |     wholly or mainly due to drug  |
|                                   |     or alcohol dependency/misuse. |
|                                   |     This can be based on previous |
|                                   |     medical details provided by   |
|                                   |     the customer and/or a current |
|                                   |     and valid ESAt or JCA report. |
|                                   |     If unsure, seek relevant      |
|                                   |     advice (e.g. from the         |
|                                   |     customer's treating doctor,   |
|                                   |     Service Support Officer,      |
|                                   |     Technical Peer Support)       |
|                                   |                                   |
|                                   | If all medical conditions,        |
|                                   | including temporary, have a       |
|                                   | prognosis of 24 months plus, a    |
|                                   | temporary incapacity exemption    |
|                                   | cannot be granted                 |
|                                   |                                   |
|                                   | If multiple medical conditions    |
|                                   | are listed, including temporary   |
|                                   | and permanent, a temporary        |
|                                   | incapacity exemption can be       |
|                                   | granted if the job seeker is      |
|                                   | unable to work or otherwise meet  |
|                                   | their mutual obligation           |
|                                   | requirements for 8 hours or more  |
|                                   | per week due to at least one of   |
|                                   | the temporary or                  |
|                                   | exacerbated/recurring conditions  |
|                                   |                                   |
|                                   | All medical condition(s) listed   |
|                                   | are already included in a current |
|                                   | and valid ESAt or JCA report.     |
|                                   | Information within the report can |
|                                   | be used unless the new medical    |
|                                   | evidence indicates the previously |
|                                   | assessed medical condition(s) has |
|                                   | changed (for example, existing    |
|                                   | spinal condition has worsened or  |
|                                   | treatment has changed). New       |
|                                   | conditions are to be assessed     |
|                                   | based on the latest medical       |
|                                   | evidence supplied                 |
|                                   |                                   |
|                                   | A temporary incapacity exemption  |
|                                   | is based on the job seeker having |
|                                   | a temporary incapacity that       |
|                                   | prevents them from undertaking    |
|                                   | work or otherwise meet their      |
|                                   | mutual obligation requirements    |
|                                   | for 8 hours or more per week. The |
|                                   | temporary incapacity exemption is |
|                                   | not based on a particular medical |
|                                   | condition. If consecutive medical |
|                                   | certificates list different       |
|                                   | conditions, a temporary           |
|                                   | incapacity exemption can still be |
|                                   | granted                           |

|                                   | Services Assessment (ESAt) is     |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | ESAt referrals are not applicable |
|                                   | for ParentsNext participants.     |
|                                   |                                   |
|                                   | Once a decision is made to grant  |
|                                   | or not grant a temporary          |
|                                   | incapacity exemption, consider if |
|                                   | an ESAt referral is appropriate   |
|                                   | to reassess the job seeker's work |
|                                   | capacity.                         |
|                                   |                                   |
|                                   | An ESAt may be considered if the  |
|                                   | medical certificate and/or other  |
|                                   | available information indicates   |
|                                   | the job seeker:                   |
|                                   |                                   |
|                                   | -   has permanent medical         |
|                                   |     condition(s) that impact      |
|                                   |     their ability to work         |
|                                   | -   has medical condition(s) that |
|                                   |     are medium or long term (24   |
|                                   |     months) in nature, including  |
|                                   |     episodic or chronic           |
|                                   |     conditions (for example,      |
|                                   |     schizophrenia, psychosis,     |
|                                   |     depression and anxiety)       |
|                                   | -   cannot be granted a temporary |
|                                   |     incapacity exemption and may  |
|                                   |     have a temporary reduced work |
|                                   |     capacity (TRWC) of less than  |
|                                   |     30 hours per week             |
|                                   | -   has medical condition(s) that |
|                                   |     appear to be no longer of a   |
|                                   |     temporary incapacity nature   |
|                                   |     based on multiple medical     |
|                                   |     certificates being lodged for |
|                                   |     the same condition            |
|                                   |                                   |
|                                   | A referral to an ESAt should not  |
|                                   | proceed where the job seeker:     |
|                                   |                                   |
|                                   | -   had medical conditions        |
|                                   |     assessed through a previous   |
|                                   |     ESAt that is still valid (for |
|                                   |     instance, no significant      |
|                                   |     change in medical conditions) |
|                                   | -   the conditions are temporary  |
|                                   |     (less than 13 weeks)          |
|                                   | -   is a compulsory ParentsNext   |
|                                   |     participant                   |
|                                   |                                   |
|                                   | Use the Online Diary to initiate  |
|                                   | an ESAt referral if required.     |
