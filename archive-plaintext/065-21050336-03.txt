Investigating means assessment data received from Services Australia or DVA

Table 2: This table contains the steps to investigate and confirm which
agency/department is responsible for the care recipient.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check Services Australia (CLK)    |
|                                   | means assessment information +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In Aged Care Management Payment   |
|                                   | System (ACMPS):                   |
|                                   |                                   |
|                                   | -   select Means Testing Data     |
|                                   | -   go to Means Tested            |
|                                   |     Assessments                   |
|                                   | -   select the Assessment ID of   |
|                                   |     the held means assessment     |
|                                   | -   select Circumstance Details   |
|                                   |     tab                           |
|                                   | -   go to ASSISTANCE TYPE         |
|                                   |                                   |
|                                   | If the ASSISTANCE TYPE is:        |
|                                   |                                   |
|                                   | -   SFR – self-funded retiree, go |
|                                   |     to Step 2                     |
|                                   | -   MND – means not disclosed, go |
|                                   |     to Step 3                     |
|                                   | -   DVA – Aged Care (RCA)         |
|                                   |     rejected-DVA, go to Step 4    |
|                                   | -   CAN – RCA cancelled, go to    |
|                                   |     Step 4                        |
|                                   | -   DEA – care recipient now      |
|                                   |     deceased, go to Step 5        |
+-----------------------------------+-----------------------------------+
| 2                                 | Services Australia (CLK) means    |
|                                   | assessment - SFR + Read more ...  |
|                                   |                                   |
|                                   | In Customer First it will show    |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   a means assessment is         |
|                                   |     completed by Services         |
|                                   |     Australia, and                |
|                                   | -   the care recipient is         |
|                                   |     assessed as a self-funded     |
|                                   |     retiree                       |
|                                   |                                   |
|                                   | As this activity has been held,   |
|                                   | ACMPS has identified that         |
|                                   | Department of Veterans’ Affairs   |
|                                   | (DVA) have previously completed a |
|                                   | means assessment for this care    |
|                                   | recipient.                        |
|                                   |                                   |
|                                   | Investigate to determine if       |
|                                   | Services Australia is now         |
|                                   | responsible.                      |
|                                   |                                   |
|                                   | In ACMPS:                         |
|                                   |                                   |
|                                   | select Means Testing Data         |
|                                   |                                   |
|                                   | check Means Tested Assessments    |
|                                   |                                   |
|                                   | -   look for the last activity    |
|                                   |     received from DVA             |
|                                   |                                   |
|                                   | check the Circumstance Details    |
|                                   | tab for Responsibility            |
|                                   |                                   |
|                                   | Does a line show DVA              |
|                                   | RESPONSIBILITY = N and is it      |
|                                   | dated after the most recent DVA   |
|                                   | means assessment?                 |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   DVA is no longer responsible  |
|                                   |     / Services Australia is now   |
|                                   |     responsible                   |
|                                   | -   see Step 1 in Table 3         |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   further investigation needed  |
|                                   | -   go to Step 6                  |

Finalising a means assessment from Services Australia or DVA

Table 3: This table contains the steps to approve or reject a means
assessment in Aged Care Management Payment System (ACMPS).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Approve the means assessment +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Complete held means assessments   |
|                                   | in date order – oldest first.     |
|                                   |                                   |
|                                   | Note: the message ‘Previous       |
|                                   | Activity ID: NNNNNNNN’ displays   |
|                                   | if the means assessments are not  |
|                                   | completed in date order.          |
|                                   |                                   |
|                                   | In ACMPS approve the means        |
|                                   | assessment:                       |
|                                   |                                   |
|                                   | -   go to Means Testing Data tab  |
|                                   | -   go to Means Tested            |
|                                   |     Assessments                   |
|                                   | -   select the earliest           |
|                                   |     Assessment ID with an MTA     |
|                                   |     Status of On Hold             |
|                                   | -   go to the Transaction History |
|                                   |     tab                           |
|                                   | -   select Means Test Data update |
|                                   |     by Diff. Source               |
|                                   | -   select Edit button next to    |
|                                   |     Task Details                  |
|                                   | -   change Active Status field to |
|                                   |     Approved                      |
|                                   | -   select Save and Back          |
|                                   | -   select care recipient name in |
|                                   |     Applicant field to return to  |
|                                   |     ACMPS ID                      |
|                                   | -   complete any other held means |
|                                   |     assessment activities         |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Rejecting a means assessment +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In ACMPS reject the means         |
|                                   | assessment:                       |
|                                   |                                   |
|                                   | -   go to Means Testing Data tab  |
|                                   | -   go to Means Tested            |
|                                   |     Assessments                   |
|                                   | -   select the earliest           |
|                                   |     Assessment ID with an MTA     |
|                                   |     Status of On Hold             |
|                                   | -   go to the Transaction History |
|                                   |     tab                           |
|                                   | -   select Means Test Data update |
|                                   |     by Diff. Source               |
|                                   | -   select Edit button next to    |
|                                   |     Task Details                  |
|                                   | -   change Active Status field to |
|                                   |     Rejected                      |
|                                   | -   select Save and Back          |
|                                   | -   select care recipient name in |
|                                   |     Applicant field to return to  |
|                                   |     ACMPS ID                      |
|                                   | -   complete any other held means |
|                                   |     assessment activities         |
+-----------------------------------+-----------------------------------+
