This page contains the claim deletion rules, information requirements
and contact details to handle claims that cannot be deleted.

Claim deletion rules

Claim deletion rules for each claim type

Table 1: This table outlines the status a claim must have before it can
be deleted.

+-----------------------------------+-----------------------------------+
| Claim type                        | Rule                              |
+-----------------------------------+-----------------------------------+
| Medicare Online                   | A claim can be deleted if either: |
|                                   |                                   |
|                                   | -   PAID (processed) on the same  |
|                                   |     date                          |
|                                   | -   Not fully processed (for      |
|                                   |     example, status WAITPEND)     |
|                                   |                                   |
|                                   | Note: for claims in WAITBKGD,     |
|                                   | advise the practice the claim     |
|                                   | cannot be deleted and they must   |
|                                   | call back later.                  |
+-----------------------------------+-----------------------------------+
| Medicare Easyclaim                | A claim can be deleted if either: |
|                                   |                                   |
|                                   | -   Payee code 2 - Unpaid account |
|                                   | -   Payee code 6 - Bulk bill.     |
|                                   |     Note: for claims in WAITBKGD, |
|                                   |     advise the practice the claim |
|                                   |     cannot be deleted and they    |
|                                   |     must call back later          |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Payee code 7 - PAID by EFT to |
|                                   |     Claimant, cannot be deleted   |
|                                   | -   Easyclaim patient claims must |
|                                   |     not be deleted after 5.30pm   |
|                                   |     (Eastern Standard Time).      |
|                                   |     Deleting claims after this    |
|                                   |     time impacts the system's     |
|                                   |     ability to process banking    |
|                                   |     reports appropriately. Advise |
|                                   |     practices to submit a request |
|                                   |     for a Latter Day Adjustment   |
|                                   |     in this instance              |
+-----------------------------------+-----------------------------------+
| Medicare Bulk Bill Webclaim       | A claim can be deleted if either: |
|                                   |                                   |
|                                   | -   PAID (processed) on the same  |
|                                   |     date before the claim is sent |
|                                   |     to the overnight payment run  |
|                                   | -   Not fully processed (for      |
|                                   |     example, status WAITPEND)     |
+-----------------------------------+-----------------------------------+
| Medicare Patient Claim Webclaim   | A claim can be deleted if either: |
|                                   |                                   |
|                                   | -   PAID (processed) on the same  |
|                                   |     date before the claim is sent |
|                                   |     to the overnight payment run  |
|                                   | -   Not fully processed (for      |
|                                   |     example, status WAITPEND)     |
+-----------------------------------+-----------------------------------+

Handle claim that cannot be deleted

If the claim has been processed on an earlier day or cannot be deleted
for other reasons, advise the caller an adjustment is required. Service
Officers can contact the relevant area for support or warm transfer the
caller.
