This page contains information about completing a Claim for Carer
Allowance following Carer Payment Review (SA341).

Additional information

Claiming CA (adult) following a CP review

This table describes the additional information required when a carer is
completing the Claim for Carer Allowance following a Carer Payment
review form.

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Additional information to claim   |
|                                   | Carer Allowance                   |
|                                   |                                   |
|                                   | -   This section asks the carer   |
|                                   |     whether they provide care on  |
|                                   |     a daily basis because of the  |
|                                   |     care receiver's medical       |
|                                   |     condition as well as the      |
|                                   |     carer and care receiver's     |
|                                   |     living arrangements           |
|                                   | -   Payment Details: the carer    |
|                                   |     has the option here of        |
|                                   |     receiving their payment into  |
|                                   |     the same account as other     |
|                                   |     Centrelink payments or to     |
|                                   |     nominate a new account. The   |
|                                   |     bank account must be in the   |
|                                   |     carers name or jointly with   |
|                                   |     their partner                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Statement                         |
|                                   |                                   |
|                                   | The carer must sign the claim     |
|                                   | before any payment can be made.   |
+-----------------------------------+-----------------------------------+
