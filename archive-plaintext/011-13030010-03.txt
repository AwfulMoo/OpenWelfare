How to claim for a reimbursement from EBPRP

This table describes how an individual can claim reimbursement for an
external breast prosthesis. Reimbursement must be paid directly to the
claimant. Payment cannot be made to a claimant's supplier or retailer.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complete claim form + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Complete the Claim for            |
|                                   | reimbursement External Breast     |
|                                   | Prostheses Program form.          |
|                                   |                                   |
|                                   | The claim form is available:      |
|                                   |                                   |
|                                   | -   Online, see the Resources     |
|                                   |     page for a link to the claim  |
|                                   |     form on the Services          |
|                                   |     Australia website             |
|                                   | -   From service centres          |
|                                   | -   By contacting Medicare        |
|                                   |     general enquiries             |
|                                   |     Note: Service Officers must   |
|                                   |     warm transfer to the EBPRP    |
|                                   |     Processing Team               |
+-----------------------------------+-----------------------------------+
| 2                                 | Supporting documents + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Supporting evidence is required   |
|                                   | to make a claim.                  |
|                                   |                                   |
|                                   | This includes an original receipt |
|                                   | that identifies the following:    |
|                                   |                                   |
|                                   | -   The supplier/retailer         |
|                                   | -   The date of purchase          |
|                                   | -   A description of the purchase |
|                                   | -   The purchase price            |
|                                   | -   Paid in full                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Lodge claim form + Read more ...  |
|                                   |                                   |
|                                   | The completed claim form and      |
|                                   | supporting documents can be       |
|                                   | lodged:                           |
|                                   |                                   |
|                                   | -   By email                      |
|                                   | -   By post                       |
|                                   | -   At any service centre         |
|                                   |                                   |
|                                   | All the details are on the form.  |
|                                   | See the Resources page for a      |
|                                   | link.                             |
|                                   |                                   |
|                                   | Note: lodgement by email is the   |
|                                   | quickest way for a claim to be    |
|                                   | processed.                        |
+-----------------------------------+-----------------------------------+
