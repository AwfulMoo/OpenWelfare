Process the RTS mail for SIP and GP ACAI

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Returned mail received + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Open the RTS mail.                |
|                                   |                                   |
|                                   | Identify the:                     |
|                                   |                                   |
|                                   | Provider’s details:               |
|                                   |                                   |
|                                   | -   Provider’s Name               |
|                                   | -   Provider number               |
|                                   |                                   |
|                                   | RTS mail type                     |
|                                   |                                   |
|                                   | If the RTS mail type is a:        |
|                                   |                                   |
|                                   | -   Request for bank account      |
|                                   |     details, go to Step 5         |
|                                   | -   Rejected Practice Incentives  |
|                                   |     Service Incentive Payments    |
|                                   |     banking details (IP011) form, |
|                                   |     go to Step 2                  |
|                                   | -   SIP or GP ACAI Statement, go  |
|                                   |     to Step 5                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirm postal address + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check if the address on the       |
|                                   | letter matches the Address        |
|                                   | (Question 3) and/or Postal        |
|                                   | address (Question 4) on the IP011 |
|                                   | form.                             |
|                                   |                                   |
|                                   | If the address on the letter      |
|                                   | matches:                          |
|                                   |                                   |
|                                   | -   Question 3 only, go to Step 3 |
|                                   | -   Question 4 only, re-send the  |
|                                   |     RTS mail to the Address       |
|                                   |     (Question 3). Go to Step 4    |
|                                   | -   Questions 3 and 4. Go to Step |
|                                   |     3                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Check provider number details +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check that the address on the     |
|                                   | letter/form match the provider    |
|                                   | number details in the Provider    |
|                                   | Directory System (PDS).           |
|                                   |                                   |
|                                   | Log on to the PDS.                |
|                                   |                                   |
|                                   | Search for the practitioner:      |
|                                   |                                   |
|                                   | -   Select Provider location from |
|                                   |     the Provider Directories menu |
|                                   | -   Select View from the drop     |
|                                   |     down                          |
|                                   | -   Key Provider stem (first 6    |
|                                   |     digits)                       |
|                                   | -   Key Provider location (7th    |
|                                   |     character, alpha or numeric)  |
|                                   | -   Select Search                 |
|                                   |                                   |
|                                   | Check if the below details match  |
|                                   | the letter/form?                  |
|                                   |                                   |
|                                   | -   Provider number, and          |
|                                   | -   Provider name                 |
|                                   |                                   |
|                                   | If not, go to Step 5              |
|                                   |                                   |
|                                   | Does the provider’s preferred     |
|                                   | mailing address match the         |
|                                   | letter/form (Questions 3 and/or   |
|                                   | 4)?                               |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, re-send the RTS mail to   |
|                                   |     the preferred mailing         |
|                                   |     address. Go to Step 4         |
+-----------------------------------+-----------------------------------+
| 4                                 | Re-send RTS mail + Read more ...  |
|                                   |                                   |
|                                   | -   Complete the return to sender |
|                                   |     cover page using the new      |
|                                   |     postal address                |
|                                   | -   Attach the RTS to the Return  |
|                                   |     to sender cover page          |
|                                   | -   Post the documents to the     |
|                                   |     provider in a new envelope    |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Register RTS mail + Read more ... |
|                                   |                                   |
|                                   | Open the Return mail register     |
|                                   | located at:                       |
|                                   |                                   |
|                                   | -   R:\SA\ADEL-209-Gr             |
|                                   | eenhill\INCENTIVESWORKFLOW\Return |
|                                   |     to Sender\Returned Mail       |
|                                   |                                   |
|                                   | Record the RTS mail details under |
|                                   | the:                              |
|                                   |                                   |
|                                   | -   SIP Bank Detail RTS tab, or   |
|                                   | -   SIP Statements tab            |
|                                   |                                   |
|                                   | Place the RTS mail in a secure    |
|                                   | bin                               |
