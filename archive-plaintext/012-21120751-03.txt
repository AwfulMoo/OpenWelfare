| 1                                 | Basic eligibility + Read more ... |
|                                   |                                   |
|                                   | When answering enquiries it is    |
|                                   | important customers know the      |
|                                   | eligibility requirements. The     |
|                                   | main requirements are:            |
|                                   |                                   |
|                                   | the claim is submitted in an      |
|                                   | approved form. An approved form   |
|                                   | is:                               |
|                                   |                                   |
|                                   | -   an application through the    |
|                                   |     claimant's Medicare online    |
|                                   |     account through myGov         |
|                                   | -   the Express Plus Medicare     |
|                                   |     mobile app, or                |
|                                   | -   Medicare online claiming or   |
|                                   |     using the COVID-19 Vaccine    |
|                                   |     Claims Scheme (CVCS)          |
|                                   |     Application form (MO062).     |
|                                   |     This is available on the      |
|                                   |     Services Australia website    |
|                                   |     Note: the first 2 options are |
|                                   |     only available where the      |
|                                   |     claimant is the patient or    |
|                                   |     for someone under 14 years of |
|                                   |     age who is on the claimant’s  |
|                                   |     Medicare card                 |
|                                   |                                   |
|                                   | the patient developed a clinical  |
|                                   | condition or an administrative    |
|                                   | related harm that has been        |
|                                   | diagnosed by a relevant medical   |
|                                   | practitioner. The CVCS medical    |
|                                   | report form (MO063) must be       |
|                                   | completed by a medical            |
|                                   | practitioner who either:          |
|                                   |                                   |
|                                   | -   made the diagnosis and is     |
|                                   |     qualified to make that        |
|                                   |     diagnosis, or                 |
|                                   | -   has all the reports from the  |
|                                   |     treating practitioner         |
|                                   |                                   |
|                                   | the patient received a COVID-19   |
|                                   | vaccine under the COVID-19        |
|                                   | vaccine rollout. Three COVID-19   |
|                                   | vaccines were included in the     |
|                                   | rollout:                          |
|                                   |                                   |
|                                   | -   AstraZeneca Vaxzevria         |
|                                   |     (COVAST)                      |
|                                   | -   Moderna Spikevax (MODERN)     |
|                                   | -   Pfizer Comirnaty (COMIRN)     |
|                                   |     Note: if the patient receive  |
|                                   |     a COVID-19 vaccine that is    |
|                                   |     not one of the 3 above the    |
|                                   |     claim will not be eligible    |
|                                   |                                   |
|                                   | the patient incurred out of       |
|                                   | pocket costs and/or lost earnings |
|                                   | and carer costs (both past and    |
|                                   | future) of at least $1,000        |
|                                   |                                   |
|                                   | Frequently asked questions and    |
|                                   | details about conditions covered  |
|                                   | by the scheme are on the          |
|                                   | Resources page.                   |

Customer at risk of suicide or self-harm – refer to Social Worker

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record information + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the customer threatens         |
|                                   | self-harm and there are serious   |
|                                   | concerns for their wellbeing and  |
|                                   | safety, record the following      |
|                                   | details:                          |
|                                   |                                   |
|                                   | -   Name (full name if possible)  |
|                                   | -   Phone number                  |
|                                   | -   If appropriate, ask for home  |
|                                   |     address                       |
|                                   |                                   |
|                                   | Advise the following:             |
|                                   |                                   |
|                                   | -   'With your consent, I will    |
|                                   |     refer you to a social worker  |
|                                   |     who may be able to assist     |
|                                   |     you'                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact Social Worker + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Make contact with a social worker |
|                                   | through normal referral           |
|                                   | protocols.                        |
|                                   |                                   |
|                                   | To transfer a call:               |
|                                   |                                   |
|                                   | select the Services Australia     |
|                                   | Workspace social work transfer    |
|                                   | option and choose ‘option 1’ from |
|                                   | the IVR list for priority access  |
|                                   | to a social worker                |
|                                   |                                   |
|                                   | advise the social worker that     |
|                                   | Services Australia is taking      |
|                                   | calls/managing enquiries on       |
|                                   | behalf of the COVID-19 Vaccine    |
|                                   | Claims Scheme (CVCS). Provide the |
|                                   | customer’s:                       |
|                                   |                                   |
|                                   | -   name                          |
|                                   | -   address                       |
|                                   | -   phone number and CRN, if they |
|                                   |     have one                      |
|                                   |                                   |
|                                   | provide a summary of the concerns |
|                                   | to the social worker and the      |
|                                   | caller’s contact details          |
|                                   |                                   |
|                                   | transfer the call                 |
|                                   |                                   |
|                                   | If an escalation to police:       |
|                                   |                                   |
|                                   | -   is required, including when   |
|                                   |     the call is disconnected      |
|                                   |     before transfer to a social   |
|                                   |     worker, go to Step 3          |
|                                   | -   was not required, advise a    |
|                                   |     Team Leader or supervisor and |
|                                   |     go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Further escalation + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If an escalation to police is     |
|                                   | required, follow usual Service    |
|                                   | Australia procedures.             |
|                                   |                                   |
|                                   | Escalate the incident using       |
|                                   | Services Australia templates and  |
|                                   | protocols. This includes          |
|                                   | escalation to the police when     |
|                                   | required via Security Incident    |
|                                   | Reporting > Police contact.       |
|                                   |                                   |
|                                   | For more information on security  |
|                                   | incident reporting see Customer   |
|                                   | aggression – Reporting and        |
|                                   | recording incidents.              |
|                                   |                                   |
|                                   | Advise a Team Leader or           |
|                                   | supervisor of the action taken.   |
