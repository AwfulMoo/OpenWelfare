Instructions and tips for using Customer Online View

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Accessing Customer Online View +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Launch Customer First         |
|                                   | -   Enter the Customer Reference  |
|                                   |     Number (CRN) to access the    |
|                                   |     customer’s record             |
|                                   | -   Select Customer Online View   |
|                                   |     in the Customer Online        |
|                                   |     account section. The Service  |
|                                   |     Officer will be automatically |
|                                   |     signed in to the customer's   |
|                                   |     Centrelink online account,    |
|                                   |     opening on the Centrelink     |
|                                   |     online account homepage       |
|                                   |                                   |
|                                   | Note: only services applicable to |
|                                   | the customer's payment type can   |
|                                   | be selected.                      |
|                                   |                                   |
|                                   | The same data displayed to the    |
|                                   | customer will be displayed to the |
|                                   | Service Officer in Customer       |
|                                   | Online View.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer's access locked or       |
|                                   | suspended + Read more ...         |
|                                   |                                   |
|                                   | If the customer's Centrelink      |
|                                   | account is currently locked or    |
|                                   | suspended, Customer Online View   |
|                                   | will still display the record.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Staff helping customers with the  |
|                                   | Report Employment Income (REI)    |
|                                   | service + Read more ...           |
|                                   |                                   |
|                                   | REI is the only service in        |
|                                   | Customer Online View that staff   |
|                                   | can enter and save information on |
|                                   | behalf of a customer or           |
|                                   | Individual Correspondence Nominee |
|                                   | (NOC).                            |
|                                   |                                   |
|                                   | Note: Service Officers cannot     |
|                                   | submit a report on behalf of a    |
|                                   | customer. The customer must be    |
|                                   | advised to sign in to their       |
|                                   | Centrelink online account through |
|                                   | myGov, review the saved           |
|                                   | information and then submit their |
|                                   | report.                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Signing out and closing Customer  |
|                                   | Online View + Read more ...       |
|                                   |                                   |
|                                   | Selecting the Back arrow at the   |
|                                   | top of the page in the Customer   |
|                                   | Online View will close the window |
|                                   | displaying the customer view, and |
|                                   | return the Service Officer to     |
|                                   | Customer First.                   |
+-----------------------------------+-----------------------------------+
