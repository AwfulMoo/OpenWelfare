How to vary a DPO

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Vary a DPO + Read more ...        |
|                                   |                                   |
|                                   | Any changes made to a DPO will be |
|                                   | to correct errors on the face of  |
|                                   | the order itself, on:             |
|                                   |                                   |
|                                   | -   the discretion of the         |
|                                   |     Secretary                     |
|                                   | -   application by the person     |
|                                   | -   incorrect issue date of the   |
|                                   |     DPO                           |
|                                   |                                   |
|                                   | It is not necessary to vary a DPO |
|                                   | due to a change in debt amount or |
|                                   | receipt of payment.               |
|                                   |                                   |
|                                   | A DPO cannot be varied to allow a |
|                                   | person to depart Australia. A DPO |
|                                   | can be revoked, or the person can |
|                                   | apply for a Departure             |
|                                   | Authorisation Certificate (DAC)   |
|                                   | where necessary                   |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   If a DPO is issued under the  |
|                                   |     wrong Act, revoke the DPO and |
|                                   |     reissue it under the correct  |
