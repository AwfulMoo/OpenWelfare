Eligibility for ACCS (Temporary Financial Hardship)

Table 1:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for Child Care        |
|                                   | Subsidy (CCS) + Read more ...     |
|                                   |                                   |
|                                   | To be eligible for ACCS           |
|                                   | (Temporary Financial Hardship)    |
|                                   | customers must be eligible for    |
|                                   | CCS.                              |
|                                   |                                   |
|                                   | Is the customer CCS current?      |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No, and the customer:             |
|                                   |                                   |
|                                   | -   has not claimed CCS -         |
|                                   |     encourage the customer to     |
|                                   |     lodge a claim for CCS         |
|                                   |     including ACCS (Temporary     |
|                                   |     Financial Hardship) go to     |
|                                   |     Step 2                        |
|                                   | -   has claimed CCS, and has been |
|                                   |     assessed as not eligible.     |
|                                   |     advise the customer they are  |
|                                   |     not eligible for ACCS         |
|                                   |     (Temporary Financial          |
|                                   |     Hardship). Procedure ends     |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Temporary Financial Hardship      |
|                                   | event(s) + Read more ...          |
|                                   |                                   |
|                                   | To be eligible for ACCS           |
|                                   | (Temporary Financial Hardship),   |
|                                   | the customer (or their partner)   |
|                                   | must have experienced a Temporary |
|                                   | Financial Hardship event(s),      |
|                                   | which has placed them in          |
|                                   | financial hardship, is            |
|                                   | unforeseen, and out of their      |
|                                   | control.                          |
|                                   |                                   |
|                                   | Note: an event may also apply on  |
|                                   | the Secretary's own initiative,   |
|                                   | such as in the case of a natural  |
|                                   | disaster. The Resources page      |
|                                   | contains an example of where the  |
|                                   | secretary may make a              |
|                                   | determination of ACCS (Temporary  |
|                                   | Financial Hardship).              |
|                                   |                                   |
|                                   | Is the customer experiencing an   |
|                                   | event (or multiple events) of     |
|                                   | Temporary Financial Hardship?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS (Temporary  |
|                                   |     Financial Hardship).          |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Event(s) occurred within the last |
|                                   | 6 months + Read more ...          |
|                                   |                                   |
|                                   | An event(s) can only be used for  |
|                                   | the purposes of ACCS (Temporary   |
|                                   | Financial Hardship) if it has     |
|                                   | occurred within 6 months of the   |
|                                   | application being lodged.         |
|                                   |                                   |
|                                   | Has the event(s) occurred within  |
|                                   | the last 6 months?                |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS (Temporary  |
|                                   |     Financial Hardship).          |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Reduction in ability to pay child |
|                                   | care fees + Read more ...         |
|                                   |                                   |
|                                   | A financial hardship              |
|                                   | determination is based on the     |
|                                   | objective and holistic            |
|                                   | circumstances that impact on the  |
|                                   | individual's ability to pay child |
|                                   | care fees. What might be a small  |
|                                   | impact on one family may be a     |
|                                   | considerable impact on another.   |
|                                   | Therefore 'hardship' can only be  |
|                                   | determined by considering all     |
|                                   | relevant factors (noting that     |
|                                   | this list is not exhaustive):     |
|                                   |                                   |
|                                   | 'out of pocket' expenses          |
|                                   |                                   |
|                                   | level of debt or savings and how  |
|                                   | these might impact                |
|                                   |                                   |
|                                   | possible changes (including       |
|                                   | variability) in income            |
|                                   |                                   |
|                                   | other financial matters relevant  |
|                                   | to their circumstances - such as: |
|                                   |                                   |
|                                   | -   when income might be          |
|                                   |     available, if known,          |
|                                   | -   the impact of insurance       |
|                                   |     settlements occurring while   |
|                                   |     the individual is in receipt  |
|                                   |     of ACCS (Temporary Financial  |
|                                   |     Hardship)                     |
|                                   |                                   |
|                                   | individual circumstances as       |
|                                   | disclosed by the individual and/  |
|                                   | or partner                        |
|                                   |                                   |
|                                   | Has the event(s) caused a         |
|                                   | significant reduction in the      |
|                                   | customer's ability to pay child   |
|                                   | care fees?                        |
|                                   |                                   |
|                                   | -   Yes, the customer meets the   |
|                                   |     eligibility criteria for ACCS |
|                                   |     (Temporary Financial          |
|                                   |     Hardship) subject to the      |
|                                   |     provision of supporting       |
|                                   |     evidence. See the Resources   |
|                                   |     page for evidence             |
|                                   |     requirements                  |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS (Temporary  |
|                                   |     Financial Hardship).          |
|                                   |     Procedure ends here           |

Applying for ACCS (Temporary Financial Hardship) via online services

Table 2: [] this table outlines the process for customers to apply for
ACCS (Temporary Financial Hardship) via online services. This
information will assist service officers in understanding the online
application process if customers contact or are applying via a
self-service terminal.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Online applications for ACCS      |
|                                   | (Temporary Financial Hardship) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The preferred option for          |
|                                   | customers to apply for ACCS       |
|                                   | (Temporary Financial Hardship) is |
|                                   | via their Centrelink online       |
|                                   | account which they can access via |
|                                   | their myGov account:              |
|                                   |                                   |
|                                   | -   For non-current CCS           |
|                                   |     customers, go to Step 2       |
|                                   | -   For CCS current customers, go |
|                                   |     to Step 3                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Making an online application -    |
|                                   | non-current CCS customers + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Once a customer has accessed      |
|                                   | their Centrelink online account   |
|                                   | via myGov, they will:             |
|                                   |                                   |
|                                   | -   Select the Online Claims tile |
|                                   |     or the Apply for a Payment >  |
|                                   |     Make a claim menu option and  |
|                                   |     then Start a new claim        |
|                                   | -   On the Make a Claim page,     |
|                                   |     select Get Started from the   |
|                                   |     Families category, then       |
|                                   |     select Apply for Child Care   |
|                                   |     Subsidy                       |
|                                   | -   Answer all questions          |
|                                   |     associated with the CCS       |
|                                   |     claim, such as income         |
|                                   |     estimate, activity test       |
|                                   |     details, child details, child |
|                                   |     care details                  |
|                                   | -   If the customer has indicated |
|                                   |     they are experiencing         |
|                                   |     financial hardship when       |
|                                   |     completing the CCS claim, the |
|                                   |     ACCS (Temporary Financial     |
|                                   |     Hardship) questions will be   |
|                                   |     presented as a part of that   |
|                                   |     claim.                        |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Making an online application -    |
|                                   | current CCS customers + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | When a current CCS customer       |
|                                   | accesses their Centrelink online  |
|                                   | account via myGov, they can start |
|                                   | a new application for ACCS        |
|                                   | (Temporary Financial Hardship)    |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   selecting Child Care Subsidy  |
|                                   |     menu, and                     |
|                                   | -   the Temporary Financial       |
|                                   |     Hardship Service              |
|                                   |                                   |
|                                   | A customer who has applied for    |
|                                   | ACCS (Temporary Financial         |
|                                   | Hardship) in the past can also:   |
|                                   |                                   |
|                                   | -   view their history of ACCS    |
|                                   |     (Temporary Financial          |
|                                   |     Hardship) applications and    |
|                                   |     decisions                     |
|                                   | -   view current and pending      |
|                                   |     applications                  |
|                                   | -   withdraw pending applications |
|                                   | -   end a current period of ACCS  |
|                                   |     (Temporary Financial          |
|                                   |     Hardship)                     |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | ACCS (Temporary Financial         |
|                                   | Hardship) specific questions +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Temporary Financial Hardship  |
|                                   | Application collects information  |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   the events which have caused  |
|                                   |     the temporary financial       |
|                                   |     hardship                      |
|                                   | -   the date of the event         |
|                                   | -   from what date assistance is  |
|                                   |     required                      |
|                                   | -   how long assistance is        |
|                                   |     required (maximum of 13       |
|                                   |     weeks)                        |
|                                   | -   the children for which the    |
|                                   |     additional assistance is to   |
|                                   |     be applied                    |
|                                   | -   details of loss of income     |
|                                   |                                   |
|                                   | CCS current customers applying    |
|                                   | via online services, go to Step   |
|                                   | 5.                                |
|                                   |                                   |
|                                   | Non-current CCS customers         |
|                                   | completing a CCS/ACCS (Temporary  |
|                                   | Financial Hardship) combined      |
|                                   | claim, go to Step 6.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Submitting a standalone ACCS      |
|                                   | (Temporary Financial Hardship)    |
|                                   | application + Read more ...       |
|                                   |                                   |
|                                   | When a customer submits their     |
|                                   | application, they will be         |
|                                   | presented with a Receipt          |
|                                   | confirming their application was  |
|                                   | successfully completed.           |
|                                   |                                   |
|                                   | Next Steps                        |
|                                   |                                   |
|                                   | Customers will be advised:        |
|                                   |                                   |
|                                   | -   what evidence they must       |
|                                   |     provide based on the event(s) |
|                                   |     they have advised has caused  |
|                                   |     their financial hardship      |
|                                   | -   to use the Upload documents   |
|                                   |     service to submit their       |
|                                   |     evidence, or submit it in     |
|                                   |     person at a Service Centre    |
|                                   |                                   |
|                                   | Note: if evidence is not provided |
|                                   | within 22 days (14 days + 8 days  |
|                                   | for postage) of submitting their  |
|                                   | application, the claim will be    |
|                                   | taken as not having been made     |
|                                   | (not effective). The customer     |
|                                   | will need to submit a new         |
|                                   | application if they wish to apply |
|                                   | for ACCS (Temporary Financial     |
|                                   | Hardship). Payment can only be    |
|                                   | backdated up to 28 days.          |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Submitting a CCS/ACCS (Temporary  |
|                                   | Financial Hardship) combined      |
|                                   | claim + Read more ...             |
|                                   |                                   |
|                                   | After completing the CCS/ACCS     |
|                                   | claim, the Next Steps page        |
|                                   | displays:                         |
|                                   |                                   |
|                                   | -   a list of Required tasks,     |
|                                   |     which will include evidence   |
|                                   |     requirements for ACCS         |
|                                   |     (Temporary Financial          |
|                                   |     Hardship)                     |
|                                   | -   the date by which the claim   |
|                                   |     must be submitted before it   |
|                                   |     will expire                   |
|                                   | -   a link to Upload documents    |
|                                   |     via an Upload button against  |
|                                   |     each Required task            |
|                                   |                                   |
|                                   | All Required tasks must be        |
|                                   | provided before the customer can  |
|                                   | Submit their CCS/ACCS combined    |
|                                   | claim.                            |
|                                   |                                   |
|                                   | Customers can access their online |
|                                   | claim to return to the Next steps |
|                                   | page at any time before it is     |
|                                   | submitted.                        |
|                                   |                                   |
|                                   | Customers have the option to:     |
|                                   |                                   |
|                                   | -   go Back to the Review your    |
|                                   |     claim page                    |
|                                   | -   Save the claim                |
|                                   |                                   |
|                                   | Until all required documentation  |
|                                   | has been provided, and the claim  |
|                                   | submitted, it is deemed no claim  |
|                                   | has been lodged or decision made. |
|                                   | This means a customer's CCS and   |
|                                   | ACCS (Temporary Financial         |
|                                   | Hardship) payment can only be     |
|                                   | backdated from the date the claim |
|                                   | is submitted. This is not         |
|                                   | appealable.                       |
|                                   |                                   |
|                                   | Note: CCS and ACCS (Temporary     |
|                                   | Financial Hardship) can only be   |
|                                   | backdated up to 28 days from the  |
|                                   | date of claim. While ACCS         |
|                                   | (Temporary Financial Hardship)    |
|                                   | can start on any Monday, it       |
|                                   | cannot have a start date before   |
|                                   | CCS.                              |
|                                   |                                   |
|                                   | Started claims will automatically |
|                                   | expire after 13 weeks.            |
|                                   |                                   |
|                                   | Once submitted, all CCS/ACCS      |
|                                   | (Temporary Financial Hardship)    |
|                                   | combined claims are allocated to  |
|                                   | the ACCS Processing Team for      |
|                                   | determination.                    |
|                                   |                                   |
|                                   | Procedure ends here.              |
