Requesting access to the Tax Adjustment Details (TXAJ) screen

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Request access to the TXAJ screen |
|                                   |                                   |
|                                   | To request access go to the ICT   |
|                                   | Security Portal (ISP):            |
|                                   |                                   |
|                                   | -   Select Request Resource       |
|                                   | -   In Search Criteria under the  |
|                                   |     Categories sub-heading,       |
|                                   |     select the magnifying glass   |
|                                   | -   Find Mainframe Application    |
|                                   |     and select '+' for more       |
|                                   |     sub-headings                  |
|                                   | -   Select Income Security,       |
|                                   |     select Search                 |
|                                   | -   Find TXAJ – ACCESS TO TXAJ    |
|                                   |     SCREEN IN ISIS in Search      |
|                                   |     Results and select '>' to     |
|                                   |     move it to Resources to       |
|                                   |     Request                       |
|                                   | -   Select Submit                 |
+-----------------------------------+-----------------------------------+
