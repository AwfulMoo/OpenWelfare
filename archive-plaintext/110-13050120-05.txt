Extension of temporary cessation of care (TCC) for Carer Payment (CP) and Carer Allowance (CA)

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| COVID-19 Extension calendar year  | If the extension is related to    |
| 2020                              | COVID19 in the calendar year      |
|                                   | 2020:                             |
|                                   |                                   |
|                                   | it is appropriate to allow an     |
|                                   | extension of respite days under   |
|                                   | 'special reasons' where:          |
|                                   |                                   |
|                                   | -   a carer or care receiver(s)   |
|                                   |     are separated, and            |
|                                   | -   the separation results in     |
|                                   |     their absence from care       |
|                                   |     exceeding the annual 63 day   |
|                                   |     respite provisions, and       |
|                                   | -   care is unable to be provided |
|                                   |     because of COVID-19 isolation |
|                                   |     or travel restrictions        |
|                                   |                                   |
|                                   | there is no requirement for the   |
|                                   | carer or care receiver to provide |
|                                   | verification of their requirement |
|                                   | to be isolated or travel          |
|                                   | restrictions                      |
+-----------------------------------+-----------------------------------+
