The CCS reconciliation process

Results of Child Care Subsidy (CCS) reconciliation or claims should not
be discussed with the child care service. Advise the service to ask the
customer to contact Services Australia.

| 6                                 | Does the reconciliation period    |
|                                   | cover all child care              |
|                                   | attendance? + Read more ...       |
|                                   |                                   |
|                                   | CCS reconciliation covers all     |
|                                   | customers who had at least one    |
|                                   | day of CCS entitlement (including |
|                                   | 0%) in the relevant year.         |
|                                   |                                   |
|                                   | If the customer lodged a CCS      |
|                                   | claim within 28 days of child     |
|                                   | care starting, all child care     |
|                                   | attendance should be included, as |
|                                   | the 28 day backdating provisions  |
|                                   | will apply.                       |
|                                   |                                   |
|                                   | CCS is based on Monday to Sunday  |
|                                   | childcare fortnights, the CCS     |
|                                   | year ends on the Sunday of the    |
|                                   | fortnight containing 30 June.     |
|                                   |                                   |
|                                   | For example: CCS for 2018-19      |
|                                   | covered the period 2 July 2018 to |
|                                   | 30 June 2019.                     |
|                                   |                                   |
|                                   | Service Officers must check:      |
|                                   |                                   |
|                                   | -   the CCS reconciliation        |
|                                   |     dashboard in Process Direct   |
|                                   |     and select the applicable     |
|                                   |     year                          |
|                                   | -   Check adequate income details |
|                                   |     have been received by         |
|                                   |     selecting > Reconciliation    |
|                                   |     History then > Tax return     |
|                                   |     Status                        |
|                                   | -   Compare those periods with    |
|                                   |     child care attendance details |
|                                   |     by selecting ‘Provider        |
|                                   |     Payments’ tab                 |
|                                   | -   The customer is liable for    |
|                                   |     any child care fees for       |
|                                   |     attendance prior to the 28    |
|                                   |     day CCS backdated start date  |
|                                   | -   If all periods are paid for   |
|                                   |     in the CCS claim but the      |
|                                   |     customer indicates the child  |
|                                   |     has attended care for periods |
|                                   |     child care attendance has not |
|                                   |     been received, advise the     |
|                                   |     customer to discuss the issue |
|                                   |     with their child care         |
|                                   |     provider                      |
|                                   | -   If a child care provider has  |
|                                   |     missed submitting child care  |
|                                   |     attendance for a period       |
|                                   |     outside of 28 days, they will |
|                                   |     need to seek approval from    |
|                                   |     the Department of Education,  |
|                                   |     Skills and Employment to      |

|                                   | -   If attendance is submitted    |

|                                   |     system will automatically     |
|                                   |     re-reconcile any previous CCS |
|                                   |     assessments for that relevant |
|                                   |     year                          |
|                                   | -   Record details on a DOC of    |
|                                   |     any advice given or action    |
|                                   |     taken                         |
|                                   |                                   |

| 7                                 | Dates of effect used for          |
|                                   | circumstance changes + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | CCS entitlement is calculated in  |
|                                   | a CCS fortnight, starting on the  |
|                                   | relevant Monday until the second  |
|                                   | Sunday after that day.            |
|                                   |                                   |
|                                   | With any change of circumstances  |
|                                   | 3 different dates apply:          |
|                                   |                                   |
|                                   | -   date of receipt (the date     |
|                                   |     Services Australia is         |
|                                   |     notified of the change)       |
|                                   | -   date of event (the date the   |
|                                   |     change of circumstance        |
|                                   |     actually occurred)            |
|                                   | -   date of effect (the date the  |
|                                   |     changes will be applied to    |
|                                   |     their payment)                |
|                                   |                                   |
|                                   | The date the change takes effect  |
|                                   | will be determined by:            |
|                                   |                                   |
|                                   | -   the date the customer         |
|                                   |     notified of the change, and   |
|                                   | -   if the impact of the change   |
|                                   |     has a favourable or           |
|                                   |     unfavourable outcome on the   |
|                                   |     customer’s entitlement        |
|                                   |                                   |
|                                   | See Changes in circumstances and  |
|                                   | date of effect for CCS and ACCS.  |
+-----------------------------------+-----------------------------------+
| 8                                 | Deadlines + Read more ...         |
|                                   |                                   |
|                                   | A late lodger process applies to  |
|                                   | CCS reconciliation. Any customer  |
|                                   | who does not meet CCS             |
|                                   | reconciliation requirements by    |
|                                   | certain deadlines will be         |
|                                   | penalised.                        |
|                                   |                                   |
|                                   | 1st Deadline                      |
|                                   |                                   |
|                                   | Customers must meet               |
|                                   | reconciliation requirements by    |
|                                   | midnight 30 June of the year      |
|                                   | following the one being           |
|                                   | reconciled.                       |
|                                   |                                   |
|                                   | If they do not meet the           |
|                                   | requirements by this time, their  |
|                                   | CCS and ACCS entitlement (except  |
|                                   | ACCS (Child Wellbeing) will be    |
|                                   | reduced to 0% from the first CCS  |
|                                   | Monday after the 1st deadline     |
|                                   | date.                             |
|                                   |                                   |
|                                   | Note: on 30 June 2020, the        |
|                                   | Department of Education, Skills   |
|                                   | and Employment (DESE) announced   |
|                                   | that the 1st deadline date for    |
|                                   | 2018-19 financial year would be   |
|                                   | extended until 31 March 2021.     |
|                                   | This is to allow customers who    |
|                                   | have not met their CCS            |
|                                   | reconciliation requirements for   |
|                                   | 2018-19 to retain their CCS       |
|                                   | entitlement until 5 April 2021,   |
|                                   | the first CCS Monday in the       |
|                                   | fortnight following 31 March      |
|                                   | 2021.                             |
|                                   |                                   |
|                                   | This means that for the 2018-19   |
|                                   | financial year, customers have    |
|                                   | until 31 March 2021 to meet their |
|                                   | reconciliation requirements for   |
|                                   | CCS. This extension only applies  |
|                                   | to CCS and does not apply to FTB  |
|                                   | or for ATO lodgement requirements |
|                                   | for the 2018-19 financial year.   |
|                                   |                                   |
|                                   | If income requirements have not   |
|                                   | been met, a non-lodger            |
|                                   | undetermined debt will be raised  |
|                                   | which totals all CCS amounts paid |
|                                   | during the relevant year. The     |
|                                   | customer will not have to pay the |
|                                   | debt at this point.               |
|                                   |                                   |
|                                   | For information on identifying    |
|                                   | whether a customer is affected by |
|                                   | the 1st deadline, see What to     |
|                                   | check when CCS rate has changed.  |
|                                   |                                   |
|                                   | 2nd Deadline                      |
|                                   |                                   |
|                                   | If customers do not meet the      |
|                                   | requirements by midnight 30 June  |
|                                   | of the year following the 1st     |
|                                   | deadline, or by the 2^(nd)        |
|                                   | deadline extension, CCS           |
|                                   | eligibility will be cancelled     |
|                                   | from the following CCS Monday.    |
|                                   | The undetermined non-lodger debt  |
|                                   | which was raised at the 1st       |
|                                   | deadline now becomes determined.  |
|                                   | The customer is required to repay |
|                                   | the non-lodger debt amount until  |
|                                   | they confirm their family income  |
|                                   | for the relevant outstanding      |
|                                   | year.                             |
|                                   |                                   |
|                                   | If relevant year income is        |
|                                   | confirmed after the 2nd deadline: |
|                                   |                                   |
|                                   | the determined non lodger debt    |
|                                   | status is changed to Finalised-No |
|                                   | Debt (FND) and recovery ceases    |
|                                   |                                   |
|                                   | Reconciliation is completed for   |
|                                   | the outstanding year which may    |
|                                   | result in:                        |
|                                   |                                   |
|                                   | -   A nil adjustment – the        |
|                                   |     customer has received correct |
|                                   |     entitlement, or               |
|                                   | -   A debt, or                    |
|                                   | -   A top up amount, which will   |
|                                   |     not be paid to the customer.  |
|                                   | -   Any withholding amounts will  |
|                                   |     also not be paid. However,    |
|                                   |     the withholding amounts may   |
|                                   |     be used to offset the current |
|                                   |     year reconciliation debt.     |
|                                   |                                   |
|                                   | With reconciliation completed,    |
|                                   | the customer can reclaim CCS but  |
|                                   | a claim will only be deemed       |
|                                   | effective if:                     |
|                                   |                                   |
|                                   | -   there are no outstanding CCS  |
|                                   |     debt amounts, or              |
|                                   | -   if the customer has an        |
|                                   |     adequate repayment plan in    |
|                                   |     place                         |
|                                   |                                   |
|                                   | Note: the customer will enter an  |
|                                   | automatic repayment arrangement   |
|                                   | through the granting of the new   |
|                                   | CCS claim which will meet this    |
|                                   | requirement.                      |

|                                   | of CCS payable + Read more ...    |
|                                   |                                   |
|                                   | The following three factors will  |
|                                   | determine the rate of CCS a       |
|                                   | family may be entitled to         |
|                                   | receive:                          |

+-----------------------------------+-----------------------------------+
| 2                                 | Income + Read more ...            |
|                                   |                                   |
|                                   | A CCS customer’s percentage (%)   |
|                                   | rate is based on their family's   |
|                                   | estimated Adjusted Taxable Income |
|                                   | (ATI). After the end of the       |
|                                   | financial year, the family's CCS  |
|                                   | rate is balanced (reconciled)     |
|                                   | using their actual ATI.           |
|                                   |                                   |
|                                   | See Combined family income for    |
|                                   | CCS.                              |
+-----------------------------------+-----------------------------------+
| 3                                 | Activity test + Read more ...     |
|                                   |                                   |
|                                   | The number of hours of subsidised |
|                                   | child care a family can access in |
|                                   | a fortnight is determined by an   |
|                                   | activity test. The customer (and  |
|                                   | their partner) need to            |
|                                   | participate in recognised         |
|                                   | activity/ies for a specific       |
|                                   | amount of hours, or have an       |
|                                   | exemption. See Activity Test for  |
|                                   | CCS.                              |
|                                   |                                   |
|                                   | Updating the activity test        |
|                                   | details for a customer will       |
|                                   | trigger a re-reconciliation.      |
|                                   |                                   |
|                                   | Exceptions:                       |
|                                   |                                   |
|                                   | -   ACCS (Grandparent) provides a |
|                                   |     subsidy equal to the actual   |
|                                   |     fee charged, up to 120% of    |
|                                   |     the hourly rate cap           |
|                                   |     (whichever is lower), for up  |
|                                   |     to 100 hours per fortnight    |
|                                   | -   ACCS (Child Wellbeing)        |
|                                   |     provides a subsidy equal to   |
|                                   |     100% of the fee charged, up   |
|                                   |     to 120% of the hourly rate    |
|                                   |     cap and up to 100 hours of    |
|                                   |     subsidised care per fortnight |
|                                   |     (regardless of the activity   |
|                                   |     level of the customer and/or  |
|                                   |     partner)                      |
|                                   | -   ACCS (Transition to Work)     |
|                                   |     provides a subsidy equal to   |
|                                   |     95% of the fee charged, up to |
|                                   |     95% of the hourly rate cap    |
|                                   |     and up to their eligible      |
|                                   |     subsidised hours of care      |
|                                   |                                   |
|                                   | The customer's eligible hour      |
|                                   | limit is applied across all       |
|                                   | services.                         |
|                                   |                                   |
|                                   | If the customer:                  |
|                                   |                                   |
|                                   | -   uses more than 1 service, CCS |
|                                   |     is paid during the year based |
|                                   |     on the hours the customer has |
|                                   |     nominated to use at each      |
|                                   |     service, or                   |
|                                   | -   has not nominated the hours   |
|                                   |     with the service, services    |
|                                   |     are paid on a first in first  |
|                                   |     paid basis during the year    |
|                                   |                                   |
|                                   | At reconciliation, the nominated  |
|                                   | CCS hours will not be reassessed  |
|                                   | up to the customer's eligible     |
|                                   | hour limit, across all services.  |

|                                   | -   In Home care                  |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Entitlement for Child Care    |
|                                   |     Subsidy (CCS)                 |

Child care attendance

Results of approved Child Care Subsidy (CCS) reconciliation or claims
should not be discussed with the child care service. Advise the service
to ask the customer to contact Services Australia.

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Child care session reports from   |
|                                   | services + Read more ...          |
|                                   |                                   |
|                                   | -   Services transfer attendance  |
|                                   |     details directly to the       |
|                                   |     Department of Education,      |
|                                   |     Skills and Employment who     |
|                                   |     transfer details to Services  |
|                                   |     Australia                     |
|                                   | -   Services Australia calculates |
|                                   |     the customer's CCS and passes |
|                                   |     the relevant amounts on to    |
|                                   |     the child care provider for   |
|                                   |     the CCS fortnight             |
|                                   | -   CCS reconciliation will not   |
|                                   |     wait for late submitted       |
|                                   |     attendance. Services are      |
|                                   |     expected to report attendance |
|                                   |     weekly. Reconciliation        |
|                                   |     commences from the 29th day   |
|                                   |     after the relevant CCS year.  |
|                                   |     It is assumed that services   |
|                                   |     will have already submitted   |
|                                   |     all attendance for the year   |
|                                   |                                   |
|                                   | Attendance received               |
|                                   | post-reconciliation               |
|                                   |                                   |
|                                   | If a service reports attendance   |
|                                   | for additional weeks after        |
|                                   | reconciliation, any top up of CCS |
|                                   | is paid to the customer.          |
|                                   |                                   |
|                                   | Note: if a childcare service      |
|                                   | reports a change to attendance    |
|                                   | that results in a debt, the debt  |
|                                   | will be raised with the provider. |
|                                   |                                   |
|                                   | Viewing attendance details        |
|                                   |                                   |
|                                   | Customers can use Online Services |
|                                   | such as myGov, or the Express     |
|                                   | Plus App to view their childcare  |
|                                   | attendance reported by their      |
|                                   | provider.                         |
