This page contains typical questions and appropriate responses to death
reason code questions for the Pharmaceutical Benefits Scheme (PBS).

Example responses to death reason code questions

Example responses to handle death reason code questions in the PBS

This table details typical enquiries and appropriate responses general
Service Officers may provide in response to questions relating to death
reason codes 619, 620, 621 and 622, for the PBS.

+-----------------------------------+-----------------------------------+
| Item                              | Questions and answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: I’ve received reason    |
|                                   | code 620 which is a rejection.    |
|                                   | Can you please explain what this  |
|                                   | means? + Read more ...            |
|                                   |                                   |
|                                   | Answer: Reason code 620 indicates |
|                                   | that the patient was not eligible |
|                                   | for PBS subsidised medication at  |
|                                   | the date of supply provided in    |
|                                   | your claim.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: There are other warning |
|                                   | reason codes with the same        |
|                                   | information, why is this a        |
|                                   | reject? + Read more ...           |
|                                   |                                   |
|                                   | Answer: Release of specific       |
|                                   | entitlement details for the       |
|                                   | patient is restricted under the   |
|                                   | secrecy provisions. I can only    |
|                                   | confirm that your claim was       |
|                                   | assessed against Services         |
|                                   | Australia's records and the       |
|                                   | rejection was based on the        |
|                                   | patient's ineligibility at the    |
|                                   | date of supply provided.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Is there any other      |
|                                   | information you can provide? +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Answer: I can confirm that the    |
|                                   | patient does not currently hold   |
|                                   | valid entitlement to PBS          |
|                                   | subsidised medications.           |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Why can't you provide   |
|                                   | any further information? + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Answer: Due to the secrecy        |
|                                   | provisions outlined in the        |
|                                   | National Health Act 1953 and the  |
|                                   | Health Insurance Act 1973, I do   |
|                                   | not have the authority to release |
|                                   | any further details.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: I have the patient in   |
|                                   | my pharmacy with me right now,    |
|                                   | what should I say to them? + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Answer: Advise the patient to     |
|                                   | visit a service centre or contact |
|                                   | Services Australia to confirm     |
|                                   | their Medicare eligibility and    |
|                                   | current details.                  |
+-----------------------------------+-----------------------------------+
