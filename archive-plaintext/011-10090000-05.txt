Frequently asked questions about AIR payment statements

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Question & Answer                 |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: I haven’t received my   |
|                                   | paper statement that we usually   |
|                                   | receive in the mail.              |
|                                   |                                   |
|                                   | Answer: The AIR no longer mails   |
|                                   | out paper payment statements      |
|                                   | anymore.                          |
|                                   |                                   |
|                                   | You can go online and log on to   |
|                                   | the AIR site to see your payment  |
|                                   | statements at any time.           |
|                                   |                                   |
|                                   | Once you’ve logged on you can     |
|                                   | view, print, save, download, or   |
|                                   | email your statements.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Why aren’t you sending  |
|                                   | paper statements anymore?         |
|                                   |                                   |
|                                   | Answer: We’re always looking to   |
|                                   | make sure that our services are   |
|                                   | accessible and efficient.         |
|                                   |                                   |
|                                   | Having your payment statements    |
|                                   | available on the AIR site means   |
|                                   | you can access them at any time.  |
|                                   | This makes it faster and easier   |
|                                   | for you to get the information    |
|                                   | you need, when you want it.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: I like getting paper    |
|                                   | statements. Why are you taking    |
|                                   | this away?                        |
|                                   |                                   |
|                                   | Answer: We’re always looking to   |
|                                   | make sure that our services are   |
|                                   | accessible and efficient.         |
|                                   |                                   |
|                                   | Going online to the AIR site for  |
|                                   | your statements means that you    |
|                                   | can get them at any time, without |
|                                   | delay.                            |
|                                   |                                   |
|                                   | You can access them when it’s     |
|                                   | convenient for you and make as    |
|                                   | many copies as you like, print or |
|                                   | email them, without having to     |
|                                   | call us.                          |
|                                   |                                   |
|                                   | This makes it faster and easier   |
|                                   | for you to get the information    |
|                                   | you need, when you want it.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Ok, can you send me     |
|                                   | copies of my statements from      |
|                                   | before the change?                |
|                                   |                                   |
|                                   | Answer: You can access your last  |
|                                   | 2 years of monthly payment        |
|                                   | statements and the last 5 years   |
|                                   | of your financial statements on   |
|                                   | the AIR site now.                 |
|                                   |                                   |
|                                   | Your statements are always        |
|                                   | available to you on the AIR site. |
|                                   | Once you’ve logged on, you can    |
|                                   | choose to view, print, save, or   |
|                                   | email your statements.            |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: How do I get my         |
|                                   | statements now?                   |
|                                   |                                   |
|                                   | Answer: By going online, you can  |
|                                   | get all of your AIR:              |
|                                   |                                   |
|                                   | -   payment                       |
|                                   | -   reconciliation, and           |
|                                   | -   financial statements          |
|                                   |                                   |
|                                   | Log on to the AIR site to access  |
|                                   | your statements at any time.      |
|                                   |                                   |
|                                   | From there you can view, save,    |
|                                   | print or email your statements.   |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Paper statements are    |
|                                   | just easier. Going online is too  |
|                                   | hard and takes too long.          |
|                                   |                                   |
|                                   | Answer: Going online to get your  |
|                                   | payment statements is quick and   |
|                                   | easy.                             |
|                                   |                                   |
|                                   | Just:                             |
|                                   |                                   |
|                                   | -   log on to the AIR site        |
|                                   | -   select Payment Statements     |
|                                   |     from the left hand main menu  |
|                                   | -   from the expanded menu,       |
|                                   |     choose whether you’d like to  |
|                                   |     view your Immunisation        |
|                                   |     Payment Statement or          |
|                                   |     Immunisation Financial        |
|                                   |     Statement. A list of all your |
|                                   |     statements will display on    |
|                                   |     the screen.                   |
|                                   |                                   |
|                                   | Once you’ve selected which        |
|                                   | statement you want, you can view, |
|                                   | print, save or email it. Your     |
|                                   | statements are always available   |
|                                   | online, at any time.              |

|                                   |                                   |
|                                   | Answer: I can check your AIR site |

|                                   | issues you might be having.       |

| 8                                 | Question: I can’t go to the AIR   |

|                                   |                                   |
|                                   | -   a tablet                      |
|                                   | -   your mobile phone             |
|                                   | -   the home computer, etc.       |
|                                   |                                   |
|                                   | If you don’t have internet access |
|                                   | on any device, I can take your    |
|                                   | details and forward to another    |
|                                   | area who will contact you to      |
|                                   | discuss how we can help.          |
+-----------------------------------+-----------------------------------+
