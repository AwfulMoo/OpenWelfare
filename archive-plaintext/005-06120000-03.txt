|                                   |                                   |
|                                   | -   suspend, go to Step 2         |
|                                   | -   restore, go to Step 3         |
|                                   | -   cancel, go to Step 5          |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer's SRSS payment to be     |
|                                   | suspended + Read more ...         |
|                                   |                                   |
|                                   | In the SRSS customer's record in  |
|                                   | Customer First:                   |
|                                   |                                   |
|                                   | navigate to the customers most    |
|                                   | recent approved Social Service    |
|                                   | Plan                              |
|                                   |                                   |
|                                   | -   run the Change Processes      |
|                                   |     guided procedure, selecting   |
|                                   |     'SSP Suspension' with the     |
|                                   |     correct date of effect        |
|                                   | -   approve the follow-up Payment |
|                                   |     Determined Social Service     |
|                                   |     Plan                          |
|                                   |                                   |
|                                   | suspension letter will be issued  |
|                                   | to the customer                   |
|                                   |                                   |
|                                   | record details on a DOC using the |
|                                   | SRSS SUS Fast Note                |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Determine SRSS Restoration + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Service officers must determine   |
|                                   | if the payment can be restored.   |
|                                   |                                   |
|                                   | Is it appropriate to restore SRSS |
|                                   | payment?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Restoring SRSS payment + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the customer was suspended due |
|                                   | to failure to report, all         |
|                                   | reporting requirements must be    |
|                                   | actioned first before the Change  |
|                                   | Processes guided procedure can be |
|                                   | run.                              |
|                                   |                                   |
|                                   | Note: Service officers must       |
|                                   | update the customer's record      |
|                                   | before restoring the payment.     |
|                                   |                                   |
|                                   | In the SRSS customer's record in  |
|                                   | Customer First:                   |
|                                   |                                   |
|                                   | -   navigate to the customers     |
|                                   |     most recent approved Social   |
|                                   |     Service Plan                  |
|                                   | -   run the Change Processes      |
|                                   |     guided procedure, select 'SSP |
|                                   |     Restoration' with the correct |
|                                   |     date of effect                |
|                                   |                                   |
|                                   | Is the customer entitled to any   |
|                                   | arrears generated?                |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   approve the follow-up Payment |
|                                   |     Determined Social Service     |
|                                   |     Plan                          |
|                                   | -   a restoration letter will be  |
|                                   |     sent to the customer          |
|                                   | -   record details on a DOC using |
|                                   |     the SRSS Restored Fast Note   |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   escalate to ICT               |

|                                   |     Service Plan                  |
|                                   | -   DOC the record using the SRSS |
|                                   |     ICT escalation Fast Note      |
|                                   | -   once resolved, approve the    |
|                                   |     Social Service Plan           |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer is to be manually        |
|                                   | cancelled off SRSS payment + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Services Australia made the       |
|                                   | decision to cancel the customer's |
|                                   | SRSS payment, or the SRSS         |
|                                   | customer has requested            |
|                                   | cancellation of SRSS payment.     |
|                                   |                                   |
|                                   | SRSS payments will usually cancel |
|                                   | automatically after 13 weeks of   |
|                                   | suspension, or a follow up        |
|                                   | Payment Determined Social Service |
|                                   | Plan is generated. Services       |
|                                   | Officers must check and approve   |
|                                   | the Social Service Plan.          |
|                                   |                                   |
|                                   | Where the customer's payment is   |
|                                   | current zero rate, the Service    |
|                                   | Officer will need to manually     |
|                                   | cancel the payment:               |
|                                   |                                   |
|                                   | -   after 6 fortnights with a     |
|                                   |     date of effect 6 fortnights   |
|                                   |     +1                            |
|                                   | -   with cancellation reason code |
|                                   |     E6F (6 consecutive nil        |
|                                   |     income), or                   |
|                                   | -   E6P (partners earning)        |
|                                   |                                   |
|                                   | To cancel the payment:            |
|                                   |                                   |
|                                   | -   run the Change Processes      |
|                                   |     guided procedure, selecting   |
|                                   |     Cancellation with the correct |
|                                   |     date of effect                |
|                                   | -   approve the follow-up payment |
|                                   |     Determined Social Services    |
|                                   |     Plan                          |
|                                   | -   record details on a DOC using |
|                                   |     the SRSS CAN Fast Note        |
|                                   | -   Email SRSS Helpdesk who will  |
|                                   |     advise the Department of Home |
|                                   |     Affairs                       |
|                                   |                                   |
|                                   | Note: if a customer is cancelled  |
|                                   | part through their Entitlement    |
|                                   | period, any money they are        |
|                                   | entitled to will not be paid      |
|                                   | immediately, but will be paid on  |
|                                   | their usual Entitlement Period    |
|                                   | End Date.                         |
|                                   |                                   |
|                                   | Procedure ends here.              |
