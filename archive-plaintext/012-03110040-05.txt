This page contains links to the My Health Record, HPOS and information
on how to check an Application to amend a healthcare organisation record
form (HW037) for completeness before processing.

Check HW037 before processing

This table describes how Service Officers check a submitted Application
to amend a healthcare organisation record form (HW037) for completeness
before processing.

Note: the whole form must be checked before processing to note all
questions that need confirmation or further information. The applicant
should be contacted once to obtain all the missing information.

+-----------------------------------+-----------------------------------+
| Question on form                  | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | RO/OMO number + Read more ...     |
|                                   |                                   |
|                                   | This question is not mandatory.   |
|                                   |                                   |
|                                   | The RO/OMO number may be used to  |
|                                   | determine if the correct          |
|                                   | individual has been located after |
|                                   | searching using the details the   |
|                                   | applicant has listed in the form. |
|                                   |                                   |
|                                   | Refer to Searching for records in |
|                                   | the Healthcare Identifiers        |
|                                   | Service using the eHealth Program |
|                                   | (EHP) for search techniques.      |
+-----------------------------------+-----------------------------------+
| 2-3                               | Applicant's personal details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | To ensure the correct individual  |
|                                   | is located and updated, the       |
|                                   | applicant should supply their:    |
|                                   |                                   |
|                                   | -   family name                   |
|                                   | -   first given name              |
|                                   | -   date of birth (DOB)           |
|                                   |                                   |
|                                   | Note: if the RO/OMO number and    |
|                                   | DOB are not supplied on the       |
|                                   | application form, call the        |
|                                   | applicant to obtain these         |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
| 4-6                               | Additional demographic details +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | These are not mandatory fields;   |
|                                   | however, the information may be   |
|                                   | used as search criteria to ensure |
|                                   | the correct record is selected    |
|                                   | for amending.                     |
+-----------------------------------+-----------------------------------+
| 8                                 | HPI-O and name of organisation to |
|                                   | be amended + Read more ...        |
|                                   |                                   |
|                                   | These fields must be completed.   |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   the Healthcare Provider       |
|                                   |     Identifier - Organisation     |
|                                   |     (HPI-O) is not necessary if   |
|                                   |     the organisation to amend can |
|                                   |     be identified using the       |
|                                   |     organisation name and the     |
|                                   |     RO/OMO confirmed by using the |
|                                   |     applicant's personal and      |
|                                   |     demographic details           |
|                                   | -   the organisation name is not  |
|                                   |     necessary if the organisation |
|                                   |     to amend can be identified    |
|                                   |     using the HPI-O and the OMO   |
|                                   |     confirmed by using the        |
|                                   |     applicant's personal and      |
|                                   |     demographic details           |
+-----------------------------------+-----------------------------------+
| 9                                 | Amend organisation details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The applicant must tick at least  |
|                                   | one box to indicate which details |
|                                   | require updating. More than one   |
|                                   | box can be ticked and multiple    |
|                                   | updates can be made using the one |
|                                   | 'Application to amend a           |
|                                   | healthcare organisation record'   |
|                                   | form (HW037).                     |
|                                   |                                   |
|                                   | Note: if no box is ticked, check  |
|                                   | the rest of the form. If the      |
|                                   | applicant has completed certain   |
|                                   | questions in the form, assume the |
|                                   | applicant wants to amend those    |
|                                   | organisation details.             |
|                                   |                                   |
|                                   | If the applicant would like to    |
|                                   | remove, add or amend the          |
|                                   | organisation's:                   |
|                                   |                                   |
|                                   | -   name details, see Question 10 |
|                                   | -   address details, see Question |
|                                   |     18                            |
|                                   | -   service details, see Question |
|                                   |     22                            |
|                                   | -   contact details, see Question |
|                                   |     25                            |
+-----------------------------------+-----------------------------------+
| 10-17                             | Organisation name details + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Certified evidence of identity    |
|                                   | must be provided to support a     |
|                                   | change of name or ownership.      |
|                                   |                                   |
|                                   | If no box is ticked at Question   |
|                                   | 10, refer to Questions 11 to 17   |
|                                   | and process according to the      |
|                                   | information supplied.             |
|                                   |                                   |
|                                   | In the eHealth Program (EHP),     |
|                                   | the:                              |
|                                   |                                   |
|                                   | -   name usage start date entered |
|                                   |     is the processing date unless |
|                                   |     otherwise supplied at         |
|                                   |     Question 16                   |
|                                   | -   end date of the previous      |
|                                   |     organisation name(s) is       |
|                                   |     either the processing date or |
|                                   |     the date supplied at Question |
|                                   |     16                            |
|                                   | -   start date accuracy is        |
|                                   |     recorded as date, month and   |
|                                   |     year are unknown if the       |
|                                   |     processing date is used or    |
|                                   |     date, month and year are      |
|                                   |     accurate if supplied          |
|                                   |                                   |
|                                   | Note: when amending existing      |
|                                   | details the existing details are  |
|                                   | not technically amended. End date |
|                                   | the existing record with the      |
|                                   | processing date, and enter in the |
|                                   | amended details as a new record   |
|                                   | with the processing date as the   |
|                                   | start date.                       |
+-----------------------------------+-----------------------------------+
| 18-21                             | Organisation address details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If no box is ticked at Question   |
|                                   | 18, refer to Questions 19 to 21   |
|                                   | and process according to the      |
|                                   | information supplied.             |
|                                   |                                   |
|                                   | In the eHealth Program (EHP),     |
|                                   | the:                              |
|                                   |                                   |
|                                   | -   end date of the previous      |
|                                   |     address(es) supplied is the   |
|                                   |     processing date               |
|                                   | -   start date of the new         |
|                                   |     address(es) is the processing |
|                                   |     date                          |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Ensure the RO and OMO address |
|                                   |     details are updated as above  |
|                                   |     if requested at Question 21.  |
|                                   | -   When amending existing        |
|                                   |     details, the existing details |
|                                   |     are not technically amended.  |
|                                   |     End date the existing record  |
|                                   |     with the processing date, and |
|                                   |     enter in the amended details  |
|                                   |     as a new record with the      |
|                                   |     processing date as the start  |
|                                   |     date.                         |
+-----------------------------------+-----------------------------------+
| 22-24                             | Organisation service details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If no box is ticked at Question   |
|                                   | 22, refer to Questions 23 to 24   |
|                                   | and process according to the      |
|                                   | information supplied.             |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   If the organisation service   |
|                                   |     start date is not provided,   |
|                                   |     use the processing date       |
|                                   | -   When amending existing        |
|                                   |     details, the existing details |
|                                   |     are not technically amended.  |
|                                   |     End date the existing record  |
|                                   |     with the processing date, and |
|                                   |     enter in the amended details  |
|                                   |     as a new record with the      |
|                                   |     processing date as the start  |
|                                   |     date                          |
+-----------------------------------+-----------------------------------+
| 25-28                             | Organisation contact details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If no box is ticked at Question   |
|                                   | 25, refer to Questions 26 to 28   |
|                                   | and process according to the      |
|                                   | information supplied.             |
+-----------------------------------+-----------------------------------+
| 30                                | Declaration + Read more ...       |
|                                   |                                   |
|                                   | The declaration must be signed by |
|                                   | the applicant.                    |
|                                   |                                   |
|                                   | Check the signature where         |
|                                   | possible to ensure it is signed   |
|                                   | by the applicant. If signature    |
|                                   | EOI has been supplied, this can   |
|                                   | be used to verify the applicant's |
|                                   | signature. Where possible, and    |
|                                   | the signature is readable, check  |
|                                   | that it is the correct name. If   |
|                                   | it is obvious the signature is    |
|                                   | from someone other than the       |
|                                   | applicant, contact the applicant  |
|                                   | to advise they must sign the form |
|                                   | and send again.                   |
|                                   |                                   |
|                                   | Note: if the date field is not    |
|                                   | completed, the date the email was |
|                                   | received may be used.             |
|                                   |                                   |
|                                   | When the form has been completed  |
|                                   | in the eHealth Program (EHP),     |
|                                   | attempt to contact the applicant  |
|                                   | and advise them that:             |
|                                   |                                   |
|                                   | -   all the requested amendments  |
|                                   |     to their organisation's       |
|                                   |     record have been made         |
|                                   | -   for future reference, some    |
|                                   |     amendments to their           |
|                                   |     contact/address details can   |
|                                   |     be made by going to Health    |
|                                   |     Professional Online Services  |
|                                   |     (HPOS)                        |
