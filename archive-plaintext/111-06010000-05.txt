Keyboard shortcuts

Table 1: Customer First general keyboard shortcuts. These are not
available for Medicare.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| Tab                               | Moves between fields              |
+-----------------------------------+-----------------------------------+
| Spacebar                          | Checks/ unchecks a checkbox or    |
|                                   | opens a dropdown list             |
+-----------------------------------+-----------------------------------+
| Enter                             | Selects a button, link or an      |
|                                   | option from a dropdown list       |
+-----------------------------------+-----------------------------------+
| Esc                               | Closes a dropdown list            |
+-----------------------------------+-----------------------------------+
| Arrow Keys                        | Moves between options in a        |
|                                   | dropdown list                     |
+-----------------------------------+-----------------------------------+
| Alt+1                             | Communication toolbar             |
+-----------------------------------+-----------------------------------+
| Alt+2                             | Navigation toolbar                |
+-----------------------------------+-----------------------------------+
| Alt+4                             | Work Area                         |
+-----------------------------------+-----------------------------------+
| CRL+M                             | From Workspace or Inbox,          |
|                                   | retrieves a new work item         |
|                                   | (Workload Management function)    |
+-----------------------------------+-----------------------------------+
