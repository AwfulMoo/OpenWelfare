Medicare entitlement types and MES eligibility

Table 5: MES eligibility based on a person's Medicare entitlement type,
including under a Reciprocal Health Care Agreement (RHCA).

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Australian citizens               |
|                                   |                                   |
|                                   | Australian citizens are eligible  |
|                                   | for a MES if they:                |
|                                   |                                   |
|                                   | -   have been absent from         |
|                                   |     Australia for 5 years or more |
|                                   |     (from initial departure), and |
|                                   | -   have not returned to reside   |
+-----------------------------------+-----------------------------------+
| 2                                 | Permanent residents (migrants)    |
|                                   |                                   |
|                                   | Permanent residents of Australia  |
|                                   | are eligible for a MES if they:   |
|                                   |                                   |
|                                   | -   have been absent from         |
|                                   |     Australia for 12 months or    |
|                                   |     more, or                      |
|                                   | -   do not live in Australia      |
|                                   |     (have never resided in        |
|                                   |     Australia), and/or            |
|                                   | -   have not returned to          |
|                                   |     Australia to reside           |
|                                   |                                   |
|                                   | Australian permanent resident     |
|                                   | holders or Australian citizens    |
|                                   | who reside in Cocos (Keeling),    |
|                                   | Christmas or Lord Howe Islands    |
|                                   | are eligible to enrol in          |
|                                   | Medicare.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | New Zealand citizens              |
|                                   |                                   |
|                                   | New Zealand citizens are eligible |
|                                   | for a MES if they travel          |
|                                   | frequently in and out of          |
|                                   | Australia and spend less than 6   |
|                                   | months in a 12 month period in    |
|                                   | Australia or does not intend to   |
|                                   | reside in Australia.              |
|                                   |                                   |
|                                   | From 1 July 2014, if an applicant |
|                                   | is a New Zealand citizen          |
|                                   | travelling frequently in and out  |
|                                   | of Australia and it has been      |
|                                   | assessed that they spend 6 or     |
|                                   | more months in a 12 month period  |
|                                   | in Australia, they are considered |
|                                   | to be residing in Australia and   |
|                                   | are eligible to enrol in          |
|                                   | Medicare. Any previously approved |
|                                   | applications for a MES are not    |
|                                   | relevant to the current           |
|                                   | application.                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Applicants for permanent          |
|                                   | residency                         |
|                                   |                                   |
|                                   | Applicants for permanent          |
|                                   | residency, including applicants   |
|                                   | for Permanent Protection, are     |
|                                   | eligible for a MES if any of the  |
|                                   | following apply:                  |
|                                   |                                   |
|                                   | -   they do not have permission   |
|                                   |     to work, and they do not have |
|                                   |     a spouse, de facto, parent or |
|                                   |     child who is an Australian    |
|                                   |     citizen, New Zealand citizen  |
|                                   |     or permanent resident holder  |
|                                   | -   the application for permanent |
|                                   |     residency has been refused    |
|                                   |     and they have not lodged an   |
|                                   |     appeal against the refused    |
|                                   |     decision relating to a        |
|                                   |     permanent resident visa       |
|                                   |     application                   |
|                                   | -   they have applied under an    |
|                                   |     Aged Parent visa (subclasses  |
|                                   |     804 or 103) and are not       |
|                                   |     eligible to be enrolled in    |
|                                   |     Medicare under a Reciprocal   |
|                                   |     Health Care Agreement (RHCA), |
|                                   |     or                            |
|                                   | -   they have applied for a       |
|                                   |     Contributory Parent visa      |
|                                   |     (subclass 143 or 864), but do |
|                                   |     not hold a Temporary          |
|                                   |     Contributory Parent visa      |
|                                   |     (subclass 173 or 884), and    |
|                                   |     are not eligible to be        |
|                                   |     enrolled in Medicare under an |
|                                   |     RHCA                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Retirees (visa subclasses 405 and |
|                                   | 410)                              |
|                                   |                                   |
|                                   | Retirees are eligible for a MES   |
|                                   | if they:                          |
|                                   |                                   |
|                                   | -   hold a visa subclass 405 and  |
|                                   |     are not enrolled in Medicare, |
|                                   |     or                            |
|                                   | -   hold a visa subclass 410      |
|                                   |     issued after 1 December 1998  |
|                                   |     and are not enrolled in       |
|                                   |     Medicare                      |
|                                   |                                   |
|                                   | Visa subclass 410 applied or      |
|                                   | issued before 1 December 1998     |
|                                   |                                   |
|                                   | People who were issued a visa     |
|                                   | subclass 410 before 1 December    |
|                                   | 1998 and are from a Reciprocal    |
|                                   | Health Care Agreement (RHCA)      |
|                                   | country had the option to:        |
|                                   |                                   |
|                                   | -   participate in the Medicare   |
|                                   |     program, or                   |
|                                   | -   apply to receive a Medicare   |
|                                   |     levy exemption annually, see  |
|                                   |     Resources for a link to ATO – |
|                                   |     Medicare levy exemption       |
|                                   |     information                   |
|                                   |                                   |
|                                   | This excludes visitors from       |
|                                   | Belgium, Norway and Slovenia as   |
|                                   | these countries did not have an   |
|                                   | RHCA with Australia before 1      |
|                                   | December 1998.                    |
|                                   |                                   |
|                                   | See RHCA eligibility for Medicare |
|                                   | and Pharmaceutical Benefits       |
|                                   | Scheme (PBS).                     |
|                                   |                                   |
|                                   | The holder of a visa subclass 410 |
|                                   | applied or issued before 1        |
|                                   | December 1998 is eligible for a   |
|                                   | MES if:                           |
|                                   |                                   |
|                                   | -   they have no record in CDMS   |
|                                   |     or their Medicare record      |
|                                   |     states Medicare levy          |
|                                   |     exemption in the Comments     |
|                                   |     tab, and                      |
|                                   | -   they are not from an RHCA     |
|                                   |     country                       |
+-----------------------------------+-----------------------------------+

|                                   |                                   |
