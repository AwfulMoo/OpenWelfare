|                                   |                                   |

|                                   | claim for CP and/or CA.           |
|                                   |                                   |
|                                   | Consultation with a Social Work   |
|                                   | Manager is mandatory before:      |
|                                   |                                   |
|                                   | -   contacting a young carer 13   |
|                                   |     years and under, as it is     |
|                                   |     expected that in most         |
|                                   |     instances alternative sources |
|                                   |     of verification will be used  |
|                                   |     to assess eligibility         |
|                                   | -   finalising a carer assessment |
|                                   |     for a young carer aged 15     |
|                                   |     years and under               |
|                                   |                                   |
|                                   | The main reasons for social       |
|                                   | worker involvement in these cases |
|                                   | are:                              |
|                                   |                                   |
|                                   | -   to make sure that the carer   |
|                                   |     has the physical and          |
|                                   |     emotional capacity to provide |
|                                   |     the level of care needed, and |
|                                   | -   a young carer may have        |
|                                   |     difficulty accessing support  |
|                                   |     services and resources. A     |
|                                   |     social worker can provide     |
|                                   |     information and help in this  |
|                                   |     regard                        |
|                                   |                                   |
|                                   | If unable to contact carer, see   |
|                                   | Item 7                            |
|                                   |                                   |
|                                   | For carers under the minimum      |
|                                   | school leaving age, the social    |
|                                   | worker should check that the      |
|                                   | appropriate education authority   |
|                                   | has given permission for the      |
|                                   | young person to leave school.     |
|                                   |                                   |
|                                   | Note: school leaving age differs  |
|                                   | depending on the state or         |
|                                   | territory in which the person     |
|                                   | resides (not the location of the  |
|                                   | school).                          |
|                                   |                                   |
|                                   | Social workers may contact        |
|                                   | relevant third parties for        |
|                                   | example, school, support workers, |
|                                   | counsellors, other family members |
|                                   | involved in the care, in order to |
|                                   | support their assessment.         |
|                                   |                                   |
|                                   | If the carer is under 18, it is   |
|                                   | mandatory for social workers to   |
|                                   | use the template to complete and  |
|                                   | upload the report. See the        |
|                                   | Resources page for a link to the  |
|                                   | template and instructions to      |
|                                   | attach a report to a customer's   |
|                                   | record.                           |
|                                   |                                   |
|                                   | Once the report is complete, the  |
|                                   | social worker summaries the       |
|                                   | information relating to the       |
|                                   | recommendation into CSAT          |

|                                   | This activity loops back to       |

|                                   | -   These situations may be       |
|                                   |     referred to a social worker   |
|                                   |     for a professional assessment |
|                                   |     where the initial Service     |
|                                   |     Officer investigation has not |
|                                   |     been able to ascertain the    |
|                                   |     level of care being provided  |
|                                   |                                   |
|                                   | In conducting Carer Assessments   |
|                                   | social workers help in providing  |
|                                   | information for:                  |
|                                   |                                   |
|                                   | -   the care receiver's personal  |
|                                   |     care or supervision needs in  |
|                                   |     the home                      |

|                                   |     with the care of the care     |

|                                   |     providing constant care       |
|                                   | -   if the carer has the physical |
|                                   |     and/or emotional capacity to  |
|                                   |     deal with the care receiver's |

|                                   | -   if there are concerns about   |
|                                   |     the capacity of the carer to  |
|                                   |     provide an appropriate level  |

|                                   | about the caring situation.       |
|                                   |                                   |
|                                   | If there is outstanding CA/CP     |
|                                   | claims, the social worker will    |
|                                   | incorporate information relating  |
|                                   | to the recommendation into CSAT   |
|                                   | Assessment Finalised Fast Note.   |
|                                   | This activity loops back to       |
|                                   | region CD1 for a delegate to make |
|                                   | a decision about qualification.   |

|                                   | carer/ treating health            |
