Delete duplicate donor only entitlement

Table 1: this table describes how to delete a person's duplicate Donor
Only entitlement.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Amendment search + Read more ...  |
|                                   |                                   |
|                                   | On the CDMS home page, complete a |
|                                   | detailed search for an existing   |
|                                   | organ donor registration or       |
|                                   | entitlement:                      |
|                                   |                                   |
|                                   | -   Select Amend Consumer Details |
|                                   | -   The Amendment Search page     |
|                                   |     displays                      |
|                                   |                                   |
|                                   | Note: the Delete Donor Record     |
|                                   | option will only appear by        |
|                                   | following the correct search      |
|                                   | method in Amend Consumer Details. |
+-----------------------------------+-----------------------------------+
| 2                                 | Select record + Read more ...     |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | from the list of matched donor    |
|                                   | records, select the radio button  |
|                                   | for the record to be deleted      |
|                                   |                                   |
|                                   | where only 1 result displays, the |
|                                   | record is a matched duplicate and |
|                                   | the View donor only entitlement   |
|                                   | screen will automatically open:   |
|                                   |                                   |
|                                   | -   go back to Search page        |
|                                   | -   select Entitlement again to   |
|                                   |     open up the Matched Donor     |
|                                   |     Records screen                |
|                                   | -   select the radio button for   |
|                                   |     the record                    |
|                                   |                                   |
|                                   | View AODR Details                 |
|                                   |                                   |
|                                   | the View Donor Only Entitlement   |
|                                   | screen displays                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Delete record + Read more ...     |
|                                   |                                   |
|                                   | -   Select Delete Donor Record    |
|                                   | -   The Confirm delete Donor Only |
|                                   |     record screen displays        |
+-----------------------------------+-----------------------------------+
| 4                                 | Confirm deletion + Read more ...  |
|                                   |                                   |
|                                   | -   Select Confirm Delete         |
|                                   | -   The message 'You have         |
|                                   |     successfully deleted the      |
|                                   |     selected donor' displays      |
+-----------------------------------+-----------------------------------+

Delete duplicate consumer ID with AODR entitlement

Table 2: This table describes how to delete a duplicate consumer ID in
CDMS with AODR entitlement only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine and identify + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Complete search for an existing   |
|                                   | organ donor registration or       |
|                                   | entitlement:                      |
|                                   |                                   |
|                                   | -   select Amend Consumer Details |
|                                   | -   complete a basic search via   |
|                                   |     identification number         |
|                                   | -   select All Programs from the  |
|                                   |     program drop down             |
|                                   | -   check both records to         |
|                                   |     identify a duplicate consumer |
|                                   |     ID with AODR entitlement      |
+-----------------------------------+-----------------------------------+
| 2                                 | End date record + Read more ...   |
|                                   |                                   |
|                                   | -   Change Program drop down back |
|                                   |     to Australian Organ Donor     |
|                                   |     Register                      |
|                                   | -   Select the consumer record to |
|                                   |     be deleted (record with AODR  |
|                                   |     entitlement only)             |
|                                   | -   Select the Program Details    |
|                                   |     tab in the Donor End Date     |
|                                   |     field, key:                   |
|                                   | -   Appropriate end date/todays   |
|                                   |     date                          |
|                                   | -   Select End date reason drop   |
|                                   |     down as ‘Duplicate’           |
+-----------------------------------+-----------------------------------+
| 3                                 | Update details + Read more ...    |
|                                   |                                   |
|                                   | Update relevant fields,           |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   Donor Media Type              |
|                                   | -   Donor Delivery Type           |
|                                   | -   Date of Last Contact          |
|                                   |                                   |
|                                   | These fields must be updated.     |
+-----------------------------------+-----------------------------------+
| 4                                 | Comments + Read more ...          |
|                                   |                                   |
|                                   | In the Comments tab enter:        |
|                                   |                                   |
|                                   | -   any ODR requested letter or   |
|                                   |     personal details comments,    |
|                                   |     where required                |
|                                   |                                   |
|                                   | See Record information in         |
|                                   | Comments field in Processing      |
|                                   | rules for Australian Organ Donor  |
|                                   | Register (AODR).                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Confirm updates + Read more ...   |
|                                   |                                   |
|                                   | Select Validate:                  |
|                                   |                                   |
|                                   | -   the Confirm Amended AODR      |
|                                   |     Entitlement Details screen    |
|                                   |     displays                      |
|                                   |                                   |
|                                   | Select Update:                    |
|                                   |                                   |
|                                   | -   a confirmation message        |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Action Corrections + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | On the existing located consumer  |
|                                   | ID with Medicare entitlement:     |
|                                   |                                   |
|                                   | -   create a new registration via |
|                                   |     Medicare number per form/call |
|                                   |     update                        |
|                                   |                                   |
|                                   | When creating a new registration, |
|                                   | the confirmation letter cannot be |
|                                   | used as the registration method.  |
|                                   | This is because a registration    |
|                                   | must already exist before a       |
|                                   | confirmation letter is sent. The  |
|                                   | system identifies this step as an |
|                                   | initial registration:             |
|                                   |                                   |
|                                   | -   create the registration with  |
|                                   |     another Donor Media Type such |
|                                   |     as ‘brochure’, and            |
|                                   | -   change the source to          |
|                                   |     confirmation letter once      |
|                                   |     created                       |
|                                   |                                   |
|                                   | See Register a new organ donor in |
|                                   | the Australian Organ Donor        |
|                                   | Register (AODR).                  |
|                                   |                                   |
|                                   |                                   |
+-----------------------------------+-----------------------------------+
| 7                                 | Escalate duplicate consumer ID +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Escalate to an AODR PSO           |
|                                   | providing:                        |
|                                   |                                   |
|                                   | Duplicate Consumer ID Number      |
|                                   |                                   |
|                                   | Correct Consumer ID Number        |
|                                   |                                   |
|                                   | Medicare Number, Medicare PIN     |
|                                   | Number, DOB, Post Code            |
|                                   |                                   |
|                                   | -   the AODR PSO team will        |
|                                   |     escalate to AODR Policy for   |
|                                   |     deletion                      |
|                                   |                                   |
|                                   | See Resources for AODR PSO        |
|                                   | contact details.                  |
+-----------------------------------+-----------------------------------+
