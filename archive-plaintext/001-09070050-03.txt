+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Self-employment as an approved    |
|                                   | activity + Read more ...          |
|                                   |                                   |
|                                   | A job seeker, who has mutual      |
|                                   | obligation requirements and is    |
|                                   | self-employed, may use their      |
|                                   | self-employment to fully meet     |
|                                   | their requirements.               |
|                                   |                                   |
|                                   | A job seeker can fully meet their |
|                                   | mutual obligation requirements    |
|                                   | through self-employment if:       |
|                                   |                                   |
|                                   | the net business income is equal  |
|                                   | to, or more than, the national    |
|                                   | minimum wage (for the number of   |
|                                   | hours required to meet the job    |
|                                   | seeker's mutual obligation        |
|                                   | requirements), and                |
|                                   |                                   |
|                                   | they are working the required     |
|                                   | number of hours in                |
|                                   | self-employment:                  |
|                                   |                                   |
|                                   | -   35 hours per week for job     |
|                                   |     seekers who have full time    |
|                                   |     mutual obligation             |
|                                   |     requirements                  |
|                                   | -   30 hours per fortnight for    |
|                                   |     principal carers, those with  |
|                                   |     a partial capacity to work    |
|                                   |     (PCW) of 15-29 hours per week |
|                                   |     and job seekers aged 55 and   |
|                                   |     over                          |
|                                   | -   0-14 hours per week (within   |
|                                   |     their assessed bandwidth) for |
|                                   |     job seekers with a PCW or     |
|                                   |     temporary reduced work        |
|                                   |     capacity (TRWC)               |
|                                   |                                   |
|                                   | Note: principal carers combining  |
|                                   | suitable self-employment and      |
|                                   | other paid work, or               |
|                                   | self-employment and approved      |
|                                   | study must complete a combined    |
|                                   | total of 30 hours per fortnight.  |
|                                   |                                   |
|                                   | Once the self-employment (SEL)    |
|                                   | approved activity is coded on the |
|                                   | job seeker's record, the details  |
|                                   | are sent to the Department of     |
|                                   | Education, Skills and Employment  |
|                                   | and the job seeker will be        |
|                                   | suspended from Employment         |
|                                   | Provider Services. This           |
|                                   | suspension is automatic. There is |
|                                   | no need for the job seeker to     |
|                                   | discuss this with their           |
|                                   | Employment Services Provider.     |
|                                   | However, the job seeker may       |
|                                   | choose to volunteer into          |
|                                   | Employment Provider Services. In  |
|                                   | these cases, the job seeker must  |
|                                   | discuss this with their provider. |
|                                   |                                   |
|                                   | Note: to view the business        |
|                                   | details, see the Real             |
|                                   | Estate/Business Summary (REBS)    |
|                                   | screen.                           |

| 6                                 | Job seeker's initial assessment   |
|                                   | satisfies mutual obligation       |
|                                   | requirements + Read more ...      |
|                                   |                                   |
|                                   | Self-employment must equal 30     |
|                                   | hours or more per fortnight for   |
|                                   | partial capacity to work (PCW)    |
|                                   | 15-29 hours per week, job seekers |
|                                   | aged 55 and over and principal    |
|                                   | carers. In addition, net business |
|                                   | income must also equal or be more |
|                                   | than the national minimum wage    |
|                                   | per fortnight for the number of   |
|                                   | required hours.                   |
|                                   |                                   |
|                                   | Job seekers assessed with a PCW   |
|                                   | or temporary reduced work         |
|                                   | capacity (TRWC) of 0-7 or 8-14    |
|                                   | hours per week may fully meet     |
|                                   | their mutual obligation           |
|                                   | requirements through              |
|                                   | self-employment or a combination  |
|                                   | of self-employment and paid work  |
|                                   | activities within their assessed  |
|                                   | bandwidth.                        |
|                                   |                                   |
|                                   | Principal carers can also combine |
|                                   | self-employment and other paid    |
|                                   | work, or self-employment and      |
|                                   | study totalling at least 30 hours |
|                                   | per fortnight to fully meet       |
|                                   | requirements, as long as the      |
|                                   | total remuneration is sufficient  |
|                                   | for the agreed hours of           |
|                                   | self-employment.                  |
|                                   |                                   |
|                                   | Record the self-employment as an  |
|                                   | approved activity on the AEX      |
|                                   | screen using the following codes: |
|                                   |                                   |
|                                   | -   For job seekers aged 55 and   |
|                                   |     over doing a combination of   |
|                                   |     voluntary work and            |
|                                   |     self-employment, use both     |
|                                   |     codes of 'SEL' and 'VWK'      |
|                                   | -   For principal carers doing a  |
|                                   |     combination of                |
|                                   |     self-employment and other     |
|                                   |     paid work, use the code 'PWS' |
|                                   | -   For principal carers doing a  |
|                                   |     combination of                |
|                                   |     self-employment and approved  |
|                                   |     study, use the code of 'SSE'  |
|                                   |                                   |
|                                   | The start date used for the AEX   |
|                                   | screen should be the date of      |
|                                   | coding, to ensure details are     |
|                                   | transferred to the Department of  |
|                                   | Education, Skills and Employment  |
|                                   | system                            |
|                                   |                                   |
|                                   | Negotiate a Job Plan with the job |
|                                   | seeker that includes the agreed   |
|                                   | activity/ies and ensure it        |
|                                   | includes corresponding dates to   |
|                                   | the activity/ies coded on the AEX |
|                                   | screen. In the Job Plan workflow, |
|                                   | the 'Self employment' activity is |
|                                   | located under the 'Employment'    |
|                                   | heading                           |
|                                   |                                   |
|                                   | Ensure any other non-current      |
|                                   | activities are removed from the   |
|                                   | Job Plan                          |
|                                   |                                   |
|                                   | Note: SEL should not be coded on  |
|                                   | the AEX screen for more than 12   |
|                                   | months (363 days) at a time and   |
|                                   | requires a new Job Plan every 363 |
|                                   | days if the work is continuing.   |
|                                   |                                   |
|                                   | Details must be recorded using    |
|                                   | the Self employment script.       |
|                                   |                                   |
|                                   | Advise the job seeker they:       |
|                                   |                                   |
|                                   | -   are fully meeting their       |
|                                   |     mutual obligation             |
|                                   |     requirements while they are   |
|                                   |     doing the agreed approved     |
|                                   |     activity                      |
|                                   | -   do not have job search        |
|                                   |     efforts however, they still   |
|                                   |     must declare their            |
|                                   |     participation in the approved |
|                                   |     activity to Services          |
|                                   |     Australia each fortnight      |
|                                   | -   will be automatically         |
|                                   |     suspended from Employment     |
|                                   |     Provider Services, however    |
|                                   |     they can voluntarily continue |
|                                   |     to participate with their     |
|                                   |     provider                      |
|                                   | -   must advise Services          |
|                                   |     Australia of changes about    |
|                                   |     their participation in the    |
|                                   |     agreed approved activity      |
|                                   |     (e.g. cessation of            |
|                                   |     self-employment or increase   |
|                                   |     in income) within the         |
|                                   |     required timeframe and how    |
|                                   |     the changes may affect their  |
|                                   |     mutual obligation             |
|                                   |     requirements                  |
|                                   |                                   |
|                                   | If they are not connected to a    |
|                                   | provider or they are suspended    |
|                                   | from Employment Provider          |
|                                   | Services, they do not need to     |
|                                   | attend ongoing provider           |
|                                   | appointments.                     |
|                                   |                                   |
|                                   | Job seekers aged 55 and over are  |
|                                   | suspended from Employment         |
|                                   | Provider Services, however, they  |
|                                   | must stay connected to their      |
|                                   | provider and still must accept    |
|                                   | full-time, suitable paid work or  |
|                                   | an increase in their work hours   |
|                                   | and must accept all referrals to  |
|                                   | suitable job interviews.          |
