Processing Relative Value Guide (RVG) items

This table contains the processing requirements for Group 10 – Relative
Value Guide (RVG) items.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Processing order + Read more ...  |
|                                   |                                   |
|                                   | RVG claims must be processed with |
|                                   | the services in the following     |
|                                   | order:                            |
|                                   |                                   |
|                                   | 1. Basic Unit Value/s (where more |
|                                   | than one appears on an            |
|                                   | invoice/receipt, they should all  |
|                                   | be listed above the time value    |
|                                   | unit – Medicare benefit will only |
|                                   | be payable for one Basic Unit     |
|                                   | Value)                            |
|                                   |                                   |
|                                   | 2. One Time Unit Value            |
|                                   |                                   |
|                                   | 3. Physical Status Modifier –     |
|                                   | where applicable                  |
|                                   |                                   |
|                                   | 4. Age Modifier – where           |
|                                   | applicable                        |
|                                   |                                   |
|                                   | 5. ‘In Hours’ Emergency Modifier  |
|                                   | – where applicable                |
|                                   |                                   |
|                                   | 6. Therapeutic/Diagnostic         |
|                                   | Service/s – where applicable      |
|                                   |                                   |
|                                   | 7. Emergency After Hours Modifier |
|                                   | – where applicable                |
|                                   |                                   |
|                                   | The Resources page contains an    |
|                                   | infographic with more details.    |
+-----------------------------------+-----------------------------------+
