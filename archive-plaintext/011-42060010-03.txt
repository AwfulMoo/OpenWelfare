Processing Relative Value Guide (RVG) items

This table contains the processing requirements for Group 10 – Relative
Value Guide (RVG) items.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Processing order                  |
|                                   |                                   |
|                                   | In order for RVG claims to be     |
|                                   | correctly processed they need to  |
|                                   | be processed in a certain order.  |
|                                   |                                   |
|                                   | This order is as follows:         |
|                                   |                                   |
|                                   | 1. Basic Unit Value/s (where more |
|                                   | than one appears on an account,   |
|                                   | they should all be listed above   |
|                                   | the time value unit – Medicare    |
|                                   | benefit will only be payable for  |
|                                   | one Basic Unit Value)             |
|                                   |                                   |
|                                   | 2. One Time Unit Value            |
|                                   |                                   |
|                                   | 3. Physical Status Modifier –     |
|                                   | where applicable                  |
|                                   |                                   |
|                                   | 4. Age Modifier – where           |
|                                   | applicable                        |
|                                   |                                   |
|                                   | 5. ‘In Hours’ Emergency Modifier  |
|                                   | – where applicable                |
|                                   |                                   |
|                                   | 6. Emergency After Hours Modifier |
|                                   | – where applicable                |
|                                   |                                   |
|                                   | 7. Therapeutic/Diagnostic         |
|                                   | Service/s – where applicable      |
+-----------------------------------+-----------------------------------+
