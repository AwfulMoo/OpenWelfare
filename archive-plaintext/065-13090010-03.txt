Check the manual registration form

Table 1: this table describes how Service Officers manually check the
aged care online claiming forms for completion.

Note: this process only applies for forms received via the manual
channel (e.g. email, mail or fax). For the online users' registration
process, no form checking is required. See the Aged Care Provider Portal
(ACPP) self service

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check form is complete + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Note: while the Register, amend   |
|                                   | or remove users for Aged Care     |
|                                   | Provider Portal form (AC004) is   |
|                                   | the most commonly used            |
|                                   | registration form, see the        |
|                                   | Resources page for the list of    |
|                                   | other aged care online claiming   |
|                                   | registration forms available to   |
|                                   | 'A users'.                        |
|                                   |                                   |
|                                   | Service Officers must check the   |
|                                   | below sections in the manual aged |
|                                   | care online claiming forms for    |
|                                   | completion:                       |
|                                   |                                   |
|                                   | -   Service details               |
|                                   | -   Nominated user details        |
|                                   | -   Declaration                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Missing information + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The aged care service must        |
|                                   | resubmit the form if any of the   |
|                                   | below information is missing or   |
|                                   | unclear:                          |
|                                   |                                   |
|                                   | Service ID                        |
|                                   |                                   |
|                                   | Name and phone number of the      |
|                                   | service’s contact person          |
|                                   |                                   |
|                                   | The first and family name of each |
|                                   | nominated user                    |
|                                   |                                   |
|                                   | -   Each user’s personal details  |
|                                   |     must exactly match the family |
|                                   |     name and first name listed on |
|                                   |     their PRODA account           |
|                                   | -   Service Officers can go to    |
|                                   |     Verifying identity for a      |
|                                   |     PRODA account table for       |
|                                   |     further information           |
|                                   |                                   |
|                                   | Aged Care User ID (if known)      |
|                                   |                                   |
|                                   | Email address for each nominated  |
|                                   | users                             |
|                                   |                                   |
|                                   | Authorisation levels for each     |
|                                   | nominated user (needed for a new  |
|                                   | user or amending the 'A user’s'   |
|                                   | access levels)                    |
|                                   |                                   |
|                                   | Transmission details. This is     |
|                                   | only needed for the Register or   |
|                                   | update for Aged Care web services |
|                                   | form (AC027)                      |
|                                   |                                   |
|                                   | Authorised person’s full name,    |
|                                   | signature and date in the         |
|                                   | Declaration section               |
+-----------------------------------+-----------------------------------+
| 3                                 | Returning an incomplete form to   |
|                                   | the aged care service + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | To return an incomplete form,     |
|                                   | Service Officers must contact the |
|                                   | aged care service outlining the   |
|                                   | reasons via:                      |
|                                   |                                   |
|                                   | -   phone                         |
|                                   | -   email, or                     |
|                                   | -   mail                          |
|                                   |                                   |
|                                   | If Service Officers contact the   |
|                                   | aged care service by phone, they  |
|                                   | must complete the necessary       |
|                                   | security checks. See Security     |
|                                   | check for service providers table |
|                                   | for further information.          |
|                                   |                                   |
|                                   | Record a note in PaNDA advising   |
|                                   | the action taken.                 |
|                                   |                                   |
|                                   | Procedure ends here until         |
|                                   | required details are provided.    |
|                                   |                                   |
|                                   | Note: when Services Australia     |
|                                   | receives the resubmitted form, it |
|                                   | is treated as a new application.  |
