Forms

Claim for Pensioner Education Supplement form (SY004)

Examples - One EdEP per calendar year or 12 month period

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| JobSeeker Payment (JSP) single    | The customer is a JSP single      |
| principal carer in receipt of     | principal carer getting PES due   |
| Pensioner Education Supplement    | to undertaking a part-time        |
| (PES)                             | Bachelor course at University.    |
|                                   | They start the course on 4 March  |
|                                   | and get PES and EdEP. If their    |
|                                   | study is continued, they will be  |
|                                   | entitled to another EdEP in 12    |
|                                   | months. The system will generate  |
|                                   | a Manual Follow Up (MFU) in 12    |
|                                   | months to check eligibility for   |
|                                   | EdEP.                             |
+-----------------------------------+-----------------------------------+
| Job seeker (short course)         | Customer is a job seeker          |
|                                   | undertaking a short course. They  |
|                                   | are provider managed with         |
|                                   | approved study included in their  |
|                                   | Job Plan. They are entitled to 1  |
|                                   | EdEP for their short course. If   |
|                                   | the customer starts another short |
|                                   | course within the same 12 month   |
|                                   | period, they will not get a       |
|                                   | second EdEP for this course.      |
+-----------------------------------+-----------------------------------+
| JSP transferring to Austudy       | Customer is currently a job       |
|                                   | seeker transferring to Austudy to |
|                                   | start a full-time Bachelor degree |
|                                   | at University. They are entitled  |
|                                   | to EdEP due to commencing a       |
|                                   | course and transferring to        |
|                                   | Austudy. This entitles them to    |
|                                   | only 1 EdEP as Austudy is not a   |
|                                   | qualifying payment for EdEP to be |
|                                   | paid in the future.               |
+-----------------------------------+-----------------------------------+
| Parenting Payment (PP) single in  | Customer is getting PP single and |
| receipt of PES                    | PES, undertaking a Bachelor       |
|                                   | course at University. The course  |
|                                   | started in second semester and    |
|                                   | they received a payment of PES    |
|                                   | and EdEP on 24 July. They are     |
|                                   | only eligible for 1 EdEP payment  |
|                                   | per calendar year. As a           |
|                                   | continuing student, they will be  |
|                                   | entitled to EdEP again in early   |
|                                   | January the following year. A MFU |
|                                   | is generated to check eligibility |
|                                   | to future entitlement.            |
+-----------------------------------+-----------------------------------+
| Continuous receipt of income      | Customer has been in receipt of   |
| support payment for the purpose   | JSP for over 12 months and        |
| of attracting EdEP                | undertaking an approved 'short    |
|                                   | course'. Their payments have been |
|                                   | suspended twice with a            |
|                                   | non-payment period of 6 weeks.    |
|                                   | Regardless, they are entitled to  |
|                                   | EdEP because they have been on    |
|                                   | ISP continuously for 12 months.   |
+-----------------------------------+-----------------------------------+
| Continuous receipt of income      | Customer was in receipt of JSP    |
| support payment for the purpose   | for 48 weeks when they cancelled  |
| of attracting EdEP, period of     | their payment due to gaining      |
| non-payment longer than 13 weeks  | employment. After 14 weeks, they  |
|                                   | became unemployed and restarted   |
|                                   | on JSP. They also undertake an    |
|                                   | approved short course. Therefore, |
|                                   | they are not entitled to get EdEP |
|                                   | due to not being in receipt of an |
|                                   | income support payment for 12     |
|                                   | months or more.                   |
|                                   |                                   |
|                                   | As they had a break in            |
|                                   | entitlement longer than 13 weeks, |
|                                   | a new period of continuous        |
|                                   | entitlement begins from the date  |
|                                   | their JSP became current.         |
+-----------------------------------+-----------------------------------+
| Continuous receipt of income      | Customer was in receipt of JSP    |
| support payment for the purpose   | for more than 60 weeks when they  |
| of attracting EdEP, period of     | cancelled their payment due to    |
| non-payment less than 13 weeks    | gaining employment. After 12      |
|                                   | weeks, they became unemployed and |
|                                   | restarted on JSP. They also       |
|                                   | undertake an approved short       |
|                                   | course. Therefore, they are       |
|                                   | entitled to EdEP as they are      |
|                                   | considered to have been in        |
|                                   | receipt of an income support      |
|                                   | payment for 12 months or more.    |
|                                   |                                   |
|                                   | As the customer was receiving     |
|                                   | income support payments for a     |
|                                   | long time (over 12 months) the    |
|                                   | non-payment period can be counted |
|                                   | as a period of income support     |
|                                   | provided it is not longer than 13 |
|                                   | weeks. The period of up to 13     |
|                                   | weeks can be made up of several   |
|                                   | such periods.                     |
+-----------------------------------+-----------------------------------+
| Continuous receipt of income      | Customer was in receipt of Youth  |
| support payment (which is not     | Allowance (student) for 46 weeks  |
| their current payment)            | and transferred to JSP when their |
| continuously for 12 months        | course finished. After 10 weeks   |
|                                   | of being on JSP, the customer     |
|                                   | decides to return to study for a  |
|                                   | year and claims for Austudy       |
|                                   | (student).                        |
|                                   |                                   |
|                                   | As the customer was in receipt of |
|                                   | JSP immediately before their new  |
|                                   | course commences and they were in |
|                                   | receipt of an income support      |
|                                   | payment continuously for at least |
|                                   | 12 months, the customer is        |
|                                   | entitled to EdEP.                 |
+-----------------------------------+-----------------------------------+
