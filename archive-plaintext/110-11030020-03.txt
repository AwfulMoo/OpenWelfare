+-----------------------------------+-----------------------------------+
| 1                                 | Recording inter agency request    |
|                                   | and approval + Read more ...      |
|                                   |                                   |
|                                   | Create an appropriate inter       |
|                                   | agency/third party request        |
|                                   | task/activity and code relevant   |
|                                   | details.                          |
|                                   |                                   |
|                                   | Request the required information  |
|                                   | from the Australian Government    |
|                                   | agency as outlined in this        |
|                                   | section. If appropriate, select   |
|                                   | the required template from the    |
|                                   | Investigation Toolkit, complete   |
|                                   | the template and issue the        |
|                                   | request.                          |
|                                   |                                   |
|                                   | If a request is issued, update    |
|                                   | status of task/activity to        |
|                                   | Waiting Response.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Recording response to inter       |
|                                   | agency request + Read more ...    |
|                                   |                                   |
|                                   | Select the appropriate inter      |
|                                   | agency/third party request        |
|                                   | task/activity and record relevant |
|                                   | details in the activity or case   |
|                                   | notes.                            |
|                                   |                                   |
|                                   | Upload any documentation received |
|                                   | as an attachment to the           |
|                                   | task/activity.                    |
|                                   |                                   |
|                                   | Update status of task/activity to |
|                                   | Completed (Closed).               |
+-----------------------------------+-----------------------------------+
| 3                                 | No response received + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | For Department of Foreign affairs |
|                                   | and Trade (DFAT) requests, see    |
|                                   | Step 6 in Table 2.                |
|                                   |                                   |
|                                   | If possible, contact the third    |
|                                   | party by phone to request the     |
|                                   | return of information request.    |
|                                   | Allow extra time if appropriate.  |
|                                   | Update the investigation record   |
|                                   | to reflect the extension of time  |
|                                   | and any other relevant            |
|                                   | information.                      |
|                                   |                                   |
|                                   | When no reply is expected or      |
|                                   | likely to be received, review the |
|                                   | progress of the case and          |
|                                   | determine the action to be taken. |
|                                   |                                   |
|                                   | Update investigation record:      |
|                                   |                                   |
|                                   | -   add notes, and                |
|                                   | -   update status of the inter    |
|                                   |     agency/third party request    |
|                                   |     task/activity to 'Cancelled'  |
|                                   |     (Closed)                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Voluntary request letter + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A Voluntary Request Letter        |
|                                   | (IMS002) must be used if there is |
|                                   | no legislative basis to exercise  |
|                                   | the coercive                      |
|                                   | information-gathering power to    |
|                                   | obtain information or documents   |
|                                   | from third parties as detailed in |
|                                   | Part 5 of the Social Security     |
|                                   | (Administration) Act 1999.        |
|                                   |                                   |
|                                   | It is also to be used to request  |
|                                   | additional information or         |
|                                   | documents to assist the           |
|                                   | Commonwealth Director of Public   |
|                                   | Prosecutions (CDPP) when the      |
|                                   | power cannot be exercised.        |
|                                   |                                   |
|                                   | Note: when requesting information |
|                                   | from Commonwealth, State or       |
|                                   | Territory Governments, it is      |
|                                   | preferable that any existing      |
|                                   | disclosure mechanism be used to   |
|                                   | gather information first, as a    |
|                                   | matter of courtesy.               |
|                                   |                                   |
|                                   | The Voluntary Request Letter      |
|                                   | (IMS002) is available from within |
|                                   | the Investigation Toolkit.        |
