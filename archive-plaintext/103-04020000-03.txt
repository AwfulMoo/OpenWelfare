| 1                                 | Available payment methods + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Payment Destination Overseas  |
|                                   | (PAPO) screen can only be updated |
|                                   | in Environment I.                 |
|                                   |                                   |
|                                   | Only staff in CIS are to record   |
|                                   | overseas bank account details on  |
|                                   | the PAPO screen. However, all     |
|                                   | staff can record Australian bank  |
|                                   | account details on the PAPO       |
|                                   | screen.                           |
|                                   |                                   |
|                                   | In general, a customer requesting |
|                                   | to be paid outside of Australia   |
|                                   | may be paid by:                   |
|                                   |                                   |
|                                   | -   direct deposit (Australian    |
|                                   |     bank, Automated Clearing      |
|                                   |     House (ACH) and Worldlink     |
|                                   |     Wire), or                     |
|                                   | -   cheque                        |
|                                   |                                   |
|                                   | The payment method needed by the  |
|                                   | customer (within the limits       |
|                                   | allowed) must be obtained from    |
|                                   | the customer. If the information  |
|                                   | is not recorded for portability   |
|                                   | cases or on the claim form (for   |
|                                   | Agreement claims), contact the    |
|                                   | customer and request overseas     |
|                                   | bank details are provided to      |
|                                   | Centrelink within 28 days.        |
|                                   |                                   |
|                                   | If the customer does not want to  |
|                                   | receive their payment into an     |
|                                   | overseas bank account, they can   |
|                                   | choose to be paid into an         |
|                                   | Australian bank account. The      |
|                                   | customer will continue to receive |
|                                   | payments every 4 weeks (if in     |
|                                   | environment I). However, they     |
|                                   | will receive it in Australian     |
|                                   | dollars. Australian bank account  |
|                                   | details can be obtained from the  |
|                                   | system (if they are unchanged) or |
|                                   | over the phone.                   |
|                                   |                                   |
|                                   | -   Direct deposit is the         |
|                                   |     preferred method for paying   |
|                                   |     the customer their Australian |
|                                   |     payment. Go to Step 3         |
|                                   | -   Cheques should only be        |
|                                   |     offered as an option for      |
|                                   |     payment if direct deposit is  |
|                                   |     not possible or suitable. Go  |
|                                   |     to Step 2                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Cheque payment method + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Cheques are not the preferred     |

|                                   |                                   |

|                                   |     causes a delay in the         |

|                                   | -   stale cheque timeframes exist |

|                                   |                                   |

|                                   |                                   |

| 3                                 | Direct deposit method + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The benefits of overseas direct   |
|                                   | deposits:                         |
|                                   |                                   |
|                                   | -   quickest method for the       |
|                                   |     customer to receive their     |
|                                   |     payment                       |
|                                   | -   most secure delivery as       |
|                                   |     payments can be traced from   |
|                                   |     the agency to the customer's  |
|                                   |     bank                          |
|                                   | -   cost effective for the        |
|                                   |     customer                      |
|                                   |                                   |
|                                   | There are 3 direct deposit        |
|                                   | methods:                          |
|                                   |                                   |
|                                   | -   Automated Clearing House      |
|                                   |     (ACH) is an electronic        |
|                                   |     funds-transfer system that    |
|                                   |     facilitates transactions      |
|                                   |     between Citibank and the      |
|                                   |     customer's bank               |
|                                   | -   Legacy ACH is a funds         |
|                                   |     transfer system between an    |
|                                   |     account held by Services      |
|                                   |     Australia and the customer's  |
|                                   |     bank in the country of        |
|                                   |     payment                       |
|                                   |     A customer in an ACH country  |
|                                   |     or Legacy ACH country must    |
|                                   |     have payment into an          |
|                                   |     Australian or overseas bank   |
|                                   |     account, unless an exemption  |
|                                   |     reason exists. Make sure the  |
|                                   |     account is held in the name   |
|                                   |     of the customer or their      |
|                                   |     payment nominee only and is   |
|                                   |     recorded to match the details |
|                                   |     on the bank account           |
|                                   |     Note: the customer may choose |
|                                   |     to receive payment by cheque  |
|                                   |     to allow them time to open an |
|                                   |     account after initial         |
|                                   |     departure. (Except for        |
|                                   |     customers in Austria,         |
|                                   |     Belgium, Denmark, Finland,    |
|                                   |     Ireland, Kenya, Malta,        |
|                                   |     Netherlands, New Zealand,     |
|                                   |     South Africa and Tunisia)     |
|                                   | -   Worldlink Wire (WWR) are      |
|                                   |     electronic fund transfers     |
|                                   |     sent through a correspondent  |
|                                   |     bank or intermediary bank. It |
|                                   |     is processed manually by the  |
|                                   |     receiving bank rather than    |
|                                   |     sent automatically to the     |
|                                   |     customer's bank account like  |
|                                   |     the ACH payment method        |
|                                   |     A customer in a WWR country   |
|                                   |     may choose to have their      |
|                                   |     payment deposited into an     |
|                                   |     Australian or overseas bank   |
|                                   |     account, or made by cheque to |
|                                   |     their home address outside    |
|                                   |     Australia                     |
|                                   |                                   |
|                                   | If paying by overseas direct      |
|                                   | deposit, check the country on     |
|                                   | OBAD to see the direct deposit    |
|                                   | payment method and currency used. |
|                                   |                                   |
|                                   | If the country is not on OBAD,    |
|                                   | contact the CIS VSSO hub and      |
|                                   | request that country banking      |
|                                   | requirements are added to enable  |
|                                   | the recording of overseas bank    |
|                                   | details for that country, based   |
|                                   | on the:                           |
|                                   |                                   |
|                                   | -   international Bank Account    |
|                                   |     Details (AUS178) form, or     |
|                                   | -   bank statement the customer   |
|                                   |     has given on the PAPO screen, |
|                                   |     or                            |
|                                   | -   verbal bank details (the      |
|                                   |     customer must still provide   |
|                                   |     written evidence of their     |
|                                   |     overseas bank details)        |
|                                   |                                   |
|                                   | International bank details can be |
|                                   | accepted if the customer has:     |
|                                   |                                   |
|                                   | completed and returned the AUS178 |
|                                   | form relevant for their country,  |
|                                   | or                                |
|                                   |                                   |
|                                   | provided a bank statement or      |
|                                   | document which shows their bank   |
|                                   | details, including bank account   |
|                                   | number or International Bank      |
|                                   | Account Number (IBAN), and the    |
|                                   | name(s) of the account holder(s), |
|                                   | with a request for payment to be  |
|                                   | made to that account, or          |
|                                   |                                   |
|                                   | provided their bank details       |
|                                   | verbally. The customer must also  |
|                                   | give documents (e.g. an official  |
|                                   | bank statement) of their new      |
|                                   | payment details as soon as        |
|                                   | possible to confirm the details   |
|                                   | given verbally. Make sure that:   |
|                                   |                                   |
|                                   | -   the customer is aware of the  |
|                                   |     currency paid and can confirm |
|                                   |     their bank account can accept |
|                                   |     payments                      |
|                                   | -   the IBAN is validated by      |
|                                   |     checking it in the IBAN       |
|                                   |     Validation Tool               |
|                                   |                                   |
|                                   | If the customer is paid in a      |
|                                   | currency other than their local   |
|                                   | currency (such as USD or EUR),    |
|                                   | and intermediary details have not |

|                                   | intermediary bank details from    |
|                                   | the RBA.                          |
