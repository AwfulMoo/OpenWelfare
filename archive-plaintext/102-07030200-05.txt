Q134 suspension letter

Note: a manual suspension letter must include the following information:

-   reason for suspension
-   date of effect
-   payment type
-   reference to the relevant act, and
-   review and appeal rights

Example text for the Q134 suspension letter

This table contains text to be used when creating a Q134 letter to
inform the customer the reason that their payment has been suspended.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer paid Austudy or YA as a  |
|                                   | full-time student                 |
|                                   |                                   |
|                                   | Under section 80 of the Social    |
|                                   | Security Administration Act, your |
|                                   | [payment type] has been suspended |
|                                   | from [date of suspension] because |
|                                   | you are not studying full time or |
|                                   | you have stopped study.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer paid YA as a Jobseeker   |
|                                   |                                   |
|                                   | Under Section 80 of the Social    |
|                                   | Security Administration Act, your |
|                                   | Youth Allowance has been          |
|                                   | suspended from [date of           |
|                                   | suspension] because you are no    |
|                                   | longer unemployed.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Customer paid ABSTUDY             |
|                                   |                                   |
|                                   | Your ABSTUDY has been suspended   |
|                                   | from [date of suspension] because |
|                                   | you are not studying full time,   |
|                                   | or you have stopped study. This   |
|                                   | suspension has been made under    |
|                                   | the provisions of Chapter 1.5 and |
|                                   | Chapter 2.4 of the ABSTUDY Policy |
|                                   | Manual.                           |
