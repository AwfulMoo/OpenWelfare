Process claims using service text

When a customer claims an item that is restrictive with another item, or
has been paid previously, an error message will display.

DVA Webclaim using service text

Table 3: This table describes how a health professional or delegate
makes a DVA Webclaim using service text and how a Service Officer
processes claims containing service text.

+-----------------------+-----------------------+-----------------------+
| Stage                 | Action                | Who                   |
+-----------------------+-----------------------+-----------------------+
| 1                     | Select Yes in         | Health professional   |
|                       | Separate Service      | or delegate           |
|                       | Indicator field +     |                       |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | Select Yes in the     |                       |
|                       | Separate Service      |                       |
|                       | Indicator field to    |                       |
|                       | indicate that the     |                       |
|                       | item is a separate    |                       |
|                       | attendance or site.   |                       |
+-----------------------+-----------------------+-----------------------+
| 2                     | Key information in    | Health professional   |
|                       | the Service Text      | or delegate           |
|                       | field + Read more ... |                       |
|                       |                       |                       |
|                       | The claim is pended   |                       |
|                       | for further           |                       |
|                       | assessment by VAP.    |                       |
+-----------------------+-----------------------+-----------------------+
| 3                     | Access information in | Service Officer       |
|                       | the VAPC screen +     |                       |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | In the VAPC screen,   |                       |
|                       | place cursor on T in  |                       |
|                       | the A column and      |                       |
|                       | presses [PF1].        |                       |
+-----------------------+-----------------------+-----------------------+
| 4                     | Assess the            | Service Officer       |
|                       | information           |                       |
|                       | displayed + Read more |                       |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | Assesses information  |                       |
|                       | that displays against |                       |
|                       | the relevant business |                       |
|                       | rules relating to the |                       |
|                       | restrictive           |                       |
|                       | situation.            |                       |
+-----------------------+-----------------------+-----------------------+
| 5                     | Pay the claim or      | Service Officer       |
|                       | reject the service +  |                       |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | Either:               |                       |
|                       |                       |                       |
|                       | -   key PI 42 to      |                       |
|                       |     override the      |                       |
|                       |     restriction and   |                       |
|                       |     pay the claim     |                       |
|                       | -   reject the        |                       |
|                       |     service with the  |                       |
|                       |     appropriate       |                       |
|                       |     reason code       |                       |
