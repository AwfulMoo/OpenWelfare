Adjust an underpaid AMD

Table 1: This table describes the process to adjust an underpaid
amendment (AMD) for Veterans' Affairs Processing (VAP).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check item number + Read more ... |
|                                   |                                   |
|                                   | Check the item number involved is |
|                                   | not listed under Category         |
|                                   | specific AMD procedures.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine the correct fee + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Consult Medicare's payment        |
|                                   | enquiry system, Operational       |
|                                   | Blueprint or the Medical Benefits |
|                                   | Schedule (MBS) to determine the   |
|                                   | correct fee.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Check patient history for amount  |
|                                   | already paid + Read more ...      |
|                                   |                                   |
|                                   | Check patient history to          |
|                                   | determine the amount already paid |
|                                   | and subtract this from the amount |
|                                   | required.                         |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   When Goods and Services Tax   |
|                                   |     (GST) is applicable,          |
|                                   |     calculate using pre-GST       |
|                                   |     amounts.                      |
|                                   | -   Changing items from GST to    |
|                                   |     non-GST or vice versa cannot  |
|                                   |     be performed through the      |
|                                   |     system - a file fix is        |
|                                   |     required to change these      |
|                                   |     items. See also: Request a    |
|                                   |     file fix for claims in        |
|                                   |     Veterans' Affairs Processing  |
|                                   |     (VAP) for file fix            |
|                                   |     procedures.                   |
+-----------------------------------+-----------------------------------+
| 4                                 | Key the claim + Read more ...     |
|                                   |                                   |
|                                   | Data enter the claim; key the     |
|                                   | adjusted amount in the Amount     |
|                                   | field and press [Enter]           |
+-----------------------------------+-----------------------------------+
| 5                                 | Does the adjustment item restrict |
|                                   | with the service already paid +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The adjusted item may restrict    |
|                                   | with the service already paid on  |
|                                   | history.                          |
|                                   |                                   |
|                                   | Override the restriction with the |
|                                   | appropriate PI code 43, 45, 58 or |
|                                   | 61.                               |
|                                   |                                   |
|                                   | When the restriction cannot be    |
|                                   | overridden, request a restriction |
|                                   | lift from the VAP Helpdesk.       |
|                                   |                                   |
|                                   | When a restriction lift cannot be |
|                                   | performed, a manual cheque needs  |
|                                   | to be issued to pay the adjusted  |
|                                   | amount.                           |
|                                   |                                   |
|                                   | See also: Veterans' Affairs       |
|                                   | Processing (VAP) for contact      |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Ensure the amount is correct +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Ensure the correct amount appears |
|                                   | in the PAID field.                |
|                                   |                                   |
|                                   | When the correct amount does not  |
|                                   | display in the PAID field, refer  |
|                                   | to a Programme Support Officer    |
|                                   | (PSO).                            |
+-----------------------------------+-----------------------------------+
| 7                                 | Finalise the claim + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key AMD in the RSN field and      |
|                                   | press [Enter].                    |

Adjust a travel not paid AMD

Table 2: This table describes the process for amendment (AMD)
adjustments involving travel not paid KM (an allowance by DVA for
kilometres travelled) or OT80 (a Remote Area Allowance paid to
Occupational Therapists) for veterans.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check patient history for item +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check patient history to          |
|                                   | determine the consultation item   |
|                                   | the travel is required for. The   |
|                                   | consult item will need to be      |
|                                   | keyed to enable payment of        |
|                                   | travel.                           |
|                                   |                                   |
|                                   | Note: when travel has been        |
|                                   | partially paid a file fix is      |
|                                   | required and a subsequent manual  |
|                                   | cheque is to be issued.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Key the item and travel details + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | At data entry, key:               |
|                                   |                                   |
|                                   | -   the consultation item and     |
|                                   |     travel                        |
|                                   | -   key ? in the Amount field for |
|                                   |     the consult item and press    |
|                                   |     [Enter]                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Key the amount + Read more ...    |
|                                   |                                   |
|                                   | In pend, 1-INV CHG return message |
|                                   | appears, key 01 (1c) in the       |
|                                   | Amount field for the consult item |
|                                   | and press [Enter].                |
|                                   |                                   |
|                                   | Note: for those items that        |
|                                   | attract GST key 5 (5c) in the     |
|                                   | Amount field.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Does the item restrict with the   |
|                                   | service already paid? + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The consultation item may         |
|                                   | restrict with the service already |
|                                   | paid on history. Override the     |
|                                   | restriction with the appropriate  |
|                                   | PI code.                          |
|                                   |                                   |
|                                   | When pend return message is:      |
|                                   |                                   |
|                                   | -   6-SUS DUP, key PI 43 and      |
|                                   |     press [Enter]                 |
|                                   | -   6-RST TD, key PI 45 and press |
|                                   |     [Enter]                       |
|                                   |                                   |
|                                   | The error message 8-BEN INCR is   |
|                                   | displayed.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Override the restriction + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Key PI 58 and press [Enter].      |
|                                   |                                   |
|                                   | 1- INV FEE is displayed.          |
+-----------------------------------+-----------------------------------+
| 6                                 | Finalise the claim + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | AMD in the RSN field, 01 (1c) in  |
|                                   | the Fee field for the consult     |
|                                   | item and press [Enter]            |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   for those items that attract  |
|                                   |     GST, key 5 (5c) in the Amount |
|                                   |     field                         |
|                                   | -   the system will default the   |
|                                   |     fee for the consultation item |
|                                   |     to 5c. The travel will pay as |
|                                   |     normal.                       |
+-----------------------------------+-----------------------------------+

Adjust Dental items

Table 3: This table describes the procedure for amendment (AMD)
adjustments involving dental items 771, 743, 744 to be paid, restrictive
item with lower fee already paid on history.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm the items would have been |
|                                   | rejected + Read more ...          |
|                                   |                                   |
|                                   | Confirm the item paid on history  |
|                                   | would have been rejected had the  |
|                                   | items been processed together.    |
|                                   |                                   |
|                                   | Notes:                            |
|                                   |                                   |
|                                   | -   Adjustment claims should be   |
|                                   |     keyed using the original      |
|                                   |     claim ID with an alpha        |
|                                   |     suffix.                       |
|                                   | -   For items 771, 743 and 744    |
|                                   |     reason code AMD cannot be     |
|                                   |     entered in the RSN field as   |
|                                   |     the system must display UPR   |
|                                   |     or LWR for these items. The   |
|                                   |     presence of PI 58 is the only |
|                                   |     indication of why an amended  |
|                                   |     fee has been paid.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Check patient history for amount  |
|                                   | paid + Read more ...              |
|                                   |                                   |
|                                   | Check patient history to          |
|                                   | determine the amount already paid |
|                                   | and subtract this from the amount |
|                                   | required.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Key adjusted amount + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | In pend, key the correct adjusted |
|                                   | amount in the Amount field and    |
|                                   | press [Enter]                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Override the restriction + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The item will restrict with the   |
|                                   | service already paid on history.  |
|                                   | Override the restriction with PI  |
|                                   | 58.                               |
+-----------------------------------+-----------------------------------+
| 5                                 | Check amount to be paid in the    |
|                                   | system + Read more ...            |
|                                   |                                   |
|                                   | Ensure that the correct amount to |
|                                   | be paid appears in the PAID       |
|                                   | field. When the correct amount    |
|                                   | does not display in the PAID      |
|                                   | field, refer to a Programme       |
|                                   | Support Officer (PSO).            |
+-----------------------------------+-----------------------------------+
| 6                                 | Finalise the claim + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key AMD in the RSN field and      |
|                                   | press [Enter].                    |

Reissue single and multiple statements

Table 10: This table describes how to access the VQRF screen which is

screen can be used to reissue multiple statements to the same provider.

months.

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Key provider number + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key VQRF, Provider Number, P and  |
|                                   | press [Enter]                     |
|                                   |                                   |
|                                   | Note: P displays all paid claims  |
+-----------------------------------+-----------------------------------+
| 2                                 | Reissue a single statement + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Key D in the S field for          |
|                                   | statements you want to reissue,   |
|                                   | press [Enter].                    |
|                                   |                                   |
|                                   | The message STMT COMP is          |
|                                   | displayed. Statement for the      |
|                                   | first claim has been reissued.    |
|                                   |                                   |
|                                   | Note: to use this transaction the |
|                                   | provider's number and claim ID    |
|                                   | need to be known.                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Reissue multiple statements +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Continue for each of the claims   |
|                                   | selected, press [Enter].          |
|                                   |                                   |
|                                   | Note: as the statement is         |
|                                   | reissued, the D will change to a  |
|                                   | B in the S field.                 |
|                                   |                                   |
|                                   | The message STMT COMP is          |
|                                   | displayed. Statement has been     |
|                                   | reissued.                         |
