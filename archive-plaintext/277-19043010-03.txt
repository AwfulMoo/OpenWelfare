|                                   | -   Select Create Account from    |
|                                   |     the Perform Action dropdown   |
|                                   |     menu                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Password Confirmation + Read more |
|                                   | ...                               |

|                                   |                                   |
|                                   | If they choose to receive their   |
|                                   | password:                         |
|                                   |                                   |
|                                   | over the phone:                   |
|                                   |                                   |
|                                   | -   select Over the phone         |
|                                   | -   once the customer has passed  |
|                                   |     authentication POI and save   |
|                                   |     is selected, the Account      |
|                                   |     Action screen displays the    |
|                                   |     password details              |
|                                   | -   a letter will not issue if    |
|                                   |     this option is chosen         |
|                                   |                                   |
|                                   | via a letter:                     |
|                                   |                                   |
|                                   | -   select Letter only            |
|                                   | -   a letter containing the       |
|                                   |     password will issue to the    |
|                                   |     customer                      |
|                                   | -   the Account Action screen     |
|                                   |     containing the temporary      |
|                                   |     password will not display     |
+-----------------------------------+-----------------------------------+
| 4                                 | Authentication POI + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Use the slider in the Outcome     |
|                                   | column to record the customer’s   |
|                                   | responses to the POI questions.   |
|                                   | For Authentication POI to be      |
|                                   | accepted, the customer’s          |
|                                   | responses need to total 100       |
|                                   | points or more.                   |
|                                   |                                   |
|                                   | Select the Accept or Reject radio |
|                                   | buttons to record the decision on |
|                                   | Authentication POI                |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | If the decision is to accept, go  |
|                                   | to Step 5                         |
|                                   |                                   |
|                                   | If the decision is to reject, no  |

|                                   | Customer Summary screen will      |

|                                   | new online account, an error      |
|                                   | message will display:             |

|                                   | -   select OK                     |
|                                   | -   select the most appropriate,  |
|                                   |     either Cancel or Reject       |
|                                   | -   select Save, until the        |
|                                   |     customer’s ineligible reason  |
|                                   |     is addressed                  |
|                                   |                                   |
|                                   | For more information, see Access  |
|                                   | to Child Support self service -   |
|                                   | set up, checking and updates.     |
+-----------------------------------+-----------------------------------+

|                                   | password + Read more ...          |

|                                   | If the customer has chosen to     |
|                                   | receive their password:           |

|                                   | over the phone:                   |
|                                   |                                   |
|                                   | -   the Account Action screen     |
|                                   |     displays the password         |
|                                   |     details - give the customer   |
|                                   |     the key information and the   |
|                                   |     temporary password            |
|                                   | -   a letter will not have issued |
|                                   |     if this option is chosen      |
|                                   | -   select Close                  |
|                                   |                                   |
|                                   | via a letter:                     |
|                                   |                                   |
|                                   | -   a pop up, Letter sent         |
|                                   |     successfully displays         |
|                                   | -   a letter containing the       |
|                                   |     password will issue to the    |
|                                   |     customer                      |
|                                   | -   if the customer wants to      |
|                                   |     their password given to them  |
|                                   |     verbally, find the password   |
|                                   |     on the letter in the letters  |
|                                   |     list. See Viewing and         |
|                                   |     searching correspondence in   |
|                                   |     Pluto                         |
|                                   |                                   |
|                                   | The customer summary Online Acc.  |
|                                   | screen will display with the      |
|                                   | recent account action.            |
