Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |

| 1                                 | Payment status is SUS-DDI + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a student fails to complete    |
|                                   | the DSLD online review within 14  |
|                                   | days their payment will be        |
|                                   | suspended with reason code 'DDI'. |
|                                   | This suspension reason cannot be  |
|                                   | manually restored via the Benefit |
|                                   | Action (BA) screen.               |
|                                   |                                   |
|                                   | If a student’s payment status is  |
|                                   | SUS-DDI, the student must         |
|                                   | complete the DSLD online review   |
|                                   | within the following 14 days      |
|                                   | (days 15 to 28 inclusive) to have |
|                                   | their payment restored            |
|                                   | automatically.                    |
|                                   |                                   |
|                                   | Service Officers who have         |
|                                   | submitted the online review via   |
|                                   | Customer First Workspace >        |
|                                   | Customer Online Account > Student |
|                                   | Digital Study Load Dec should     |
|                                   | clear out of the student’s record |
|                                   | immediately to allow restoration  |
|                                   | triggers to run.                  |
|                                   |                                   |
|                                   | If the DSLD online review has     |
|                                   | been submitted but the payment    |
|                                   | status is still SUS-DDI:          |
|                                   |                                   |
|                                   | -   check the Network News Update |
|                                   |     (NNU) for known issues        |
|                                   | -   do not update the Customer    |
|                                   |     Study Details (EDC) screen in |
|                                   |     Process Direct or Education   |
|                                   |     Course Details (EDC) screen   |
|                                   |     in Customer First             |
|                                   | -   do not attempt manual         |
|                                   |     workarounds                   |
|                                   | -   do not escalate to ICT        |
|                                   |     Service Desk                  |
|                                   | -   do not change the status of   |
