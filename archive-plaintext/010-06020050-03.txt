Lodgement of UTLAH statements

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Under 18 + Read more ...          |
|                                   |                                   |
|                                   | Is the customer under 18?         |
|                                   |                                   |
|                                   | -   Yes, there is no requirement  |
|                                   |     for the customer to complete  |
|                                   |     any forms. A social worker    |
|                                   |     completes the assessment. See |
|                                   |     Unreasonable to live at home  |
|                                   |     (UTLAH) initial contact for   |
|                                   |     Youth Allowance (YA),         |
|                                   |     Disability Support Pension    |
|                                   |     (DSP), Special Benefit (SpB)  |
|                                   |     or Tertiary Access Payment    |
|                                   |     (TAP). Procedure ends here    |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Currently receiving/claiming YA   |
|                                   | or TAP + Read more ...            |
|                                   |                                   |
|                                   | Has the customer claimed:         |
|                                   |                                   |
|                                   | and currently receiving YA        |
|                                   |                                   |
|                                   | YA in the previous 13 weeks, or   |
|                                   |                                   |
|                                   | TAP since 1 January of the        |
|                                   | current calendar year?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, see Contact in relation   |
|                                   |     to an intended claim          |
+-----------------------------------+-----------------------------------+
| 3                                 | At risk assessment + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Does the young person aged 18 or  |
|                                   | over appear to be at risk?        |
|                                   |                                   |
|                                   | Yes, refer the customer to a      |
|                                   | social worker, who will undertake |
|                                   | a risk assessment and discuss     |
|                                   | community support options         |
|                                   |                                   |
|                                   | -   Note: do not book a UTLAH     |
|                                   |     appointment                   |
|                                   | -   Service Officers must action  |
|                                   |     any lodged UTLAH              |
|                                   |     statement(s), regardless of   |
|                                   |     whether a social work risk or |
|                                   |     support referral is made. Go  |
|                                   |     to Step 4                     |
|                                   |                                   |
|                                   | No, go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Lodgement of UTLAH statements +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers who are 18 or over      |
|                                   | must, in most cases, provide 2    |
|                                   | statements that have been         |
|                                   | completed and signed:             |
|                                   |                                   |
|                                   | -   Youth Allowance and ABSTUDY   |
|                                   |     'Unreasonable to live at      |
|                                   |     home' Statement by Young      |
|                                   |     Person (SY015)                |
|                                   | -   Youth Allowance and ABSTUDY   |
|                                   |     'Unreasonable to live at      |
|                                   |     home' Statement for           |
|                                   |     Parent(s)/Guardian(s) (SY016) |
|                                   |     Note: if parents are          |
|                                   |     separated both parents need   |
|                                   |     to complete an SY016 form     |
|                                   |     each                          |
|                                   |                                   |
|                                   | Has the customer lodged the       |
|                                   | statements?                       |
|                                   |                                   |
|                                   | Yes, and the statements have been |
|                                   | completed and signed, go to Step  |
|                                   | 6                                 |
|                                   |                                   |
|                                   | No, if:                           |
|                                   |                                   |
|                                   | -   the customer has not lodged   |
|                                   |     any statements, go to Step 5  |
|                                   | -   the only form that has been   |
|                                   |     completed and signed is the   |
|                                   |     Statement by Young Person     |
|                                   |     (SY015). Check the young      |
|                                   |     person has provided details   |
|                                   |     of the parent/s and           |
|                                   |     independent third party for   |
|                                   |     assessment of a UTLAH claim   |
|                                   |     information in the form and   |
|                                   |     has given permission to       |
|                                   |     contact. Go to Step 6         |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer has not lodged any       |
|                                   | statements + Read more ...        |
|                                   |                                   |
|                                   | Issue the 2 statements:           |
|                                   |                                   |
|                                   | -   Statement by Young Person     |
|                                   |     (SY015)                       |
|                                   | -   Statement by                  |
|                                   |     Parent(s)/Guardian(s)         |
|                                   |     (SY016). If the parents do    |
|                                   |     not live together, issue an   |
|                                   |     SY016 for each parent to      |
|                                   |     complete, unless the young    |
|                                   |     person can provide            |
|                                   |     verification that there has   |
|                                   |     been no recent contact with   |
|                                   |     the 'other' parent. O18 can   |
|                                   |     assess if 2 SY016 statements  |
|                                   |     are necessary                 |
|                                   |                                   |
|                                   | The Statement by an Independent   |
|                                   | Third Party form (SY017) will     |
|                                   | only be requested by the Over 18  |
|                                   | UTLAH Assessment team if:         |
|                                   |                                   |
|                                   | -   the Statement by young person |
|                                   |     form (SY015) and Statement by |
|                                   |     Parent(s)/Guardian(s) form    |
|                                   |     (SY016) do not align          |
|                                   | -   a parent is refusing or       |
|                                   |     failing to provide a          |
|                                   |     statement or the young person |
|                                   |     has advised it would put them |
|                                   |     at risk for the parent(s) to  |
|                                   |     be contacted, or              |
|                                   | -   there is doubt about the      |
|                                   |     reason the young person is    |
|                                   |     unable to live at home        |
|                                   |                                   |
|                                   | Record details on a DOC.          |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Update customer's record + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Using the information provided    |
|                                   | verbally by the customer and/or   |
|                                   | completed in the Statement by     |
|                                   | Young Person (SY015) form, update |
|                                   | the customer's:                   |
|                                   |                                   |
|                                   | -   Address details               |
|                                   | -   Telephone details. If the     |
|                                   |     customer does not have a      |
|                                   |     telephone number, obtain an   |
|                                   |     alternative contact number so |
|                                   |     that the Over 18 UTLAH        |
|                                   |     Assessment Team can contact   |
|                                   |     the customer for the          |
|                                   |     assessment to proceed         |
|                                   |                                   |
|                                   | If the customer has previously    |
|                                   | indicated that their parent(s) or |
|                                   | third party has permission to     |
|                                   | enquire, and this arrangement is  |
|                                   | active on their record, confirm   |
|                                   | with the customer whether this    |
|                                   | should continue. See Person       |
|                                   | Permitted to Enquire (PPE) or     |
|                                   | Update (PPU) authority.           |
|                                   |                                   |
|                                   | If the customer has a nominee,    |
|                                   | consider whether this arrangement |
|                                   | should continue. See Cancelling a |
|                                   | nominee arrangement.              |
|                                   |                                   |
|                                   | If the customer is currently      |
|                                   | receiving YA and their payment is |
|                                   | directed to their parents,        |
|                                   | consider updating the payment     |
|                                   | destination.                      |
+-----------------------------------+-----------------------------------+
| 7                                 | Priority assessment check + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A young person may be considered  |
|                                   | for a priority UTLAH assessment   |
|                                   | if they do not receive the basic  |
|                                   | rate of a Centrelink income       |
|                                   | support payment and meet one of   |
|                                   | the below criteria:               |
|                                   |                                   |
|                                   | -   has no form of accommodation, |
|                                   |     that is, they need to access  |
|                                   |     refuge or emergency           |
|                                   |     accommodation                 |
|                                   | -   has significant health issues |
|                                   |     (physical or psychological)   |
|                                   | -   are at risk of suicide or     |
|                                   |     self-harm                     |
|                                   | -   has left a family and         |
|                                   |     domestic violence situation   |
|                                   |     and has no support            |
|                                   |                                   |
|                                   | Note: a priority assessment check |
|                                   | is not applicable for TAP claims. |
|                                   | Eligibility is determined 42 days |
|                                   | after the customer's student      |
|                                   | start date. Encourage customers   |
|                                   | to claim Youth Allowance or       |
|                                   | ABSTUDY as a UTLAH customer to    |
|                                   | facilitate a priority assessment  |
|                                   | check.                            |
|                                   |                                   |
|                                   | Is the customer in one of the     |
|                                   | above situations and needs a      |
|                                   | priority assessment?              |
|                                   |                                   |
|                                   | Yes, record reasons for priority  |
|                                   | assessment by creating an open    |
|                                   | Fast Note, using Auto Text        |
|                                   | function:                         |
|                                   |                                   |
|                                   | -   Social Worker > Update > O18  |
|                                   |     UTLAH ACTDOC Enquiry          |
|                                   | -   Go to Step 8                  |
|                                   |                                   |
|                                   | No, go to Step 8                  |
+-----------------------------------+-----------------------------------+
| 8                                 | Scan UTLAH statements + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Prepare and scan the available    |
|                                   | UTLAH statements onto the         |
|                                   | customer's record. See            |
|                                   | Circumstance Change Monitor (CCM) |
|                                   | for scanning a document.          |
|                                   |                                   |
|                                   | When scanning:                    |
|                                   |                                   |
|                                   | -   make sure the first page to   |
|                                   |     be scanned includes the page  |
|                                   |     with the barcode              |
|                                   | -   do not tick 'store only' to   |
|                                   |     ensure an activity is created |
|                                   |     on the customer's record.     |
|                                   |     This activity is              |
|                                   |     automatically allocated to    |
|                                   |     the Student Network - Over 18 |
|                                   |     UTLAH (O18) Processing Team   |
|                                   | -   make sure the documents are   |
|                                   |     batch stored appropriately    |
|                                   | -   statements provided for a new |
|                                   |     claim activity, once scanned, |
|                                   |     place the claim on hold,      |
|                                   |     Awaiting UTLAH, in Process    |
|                                   |     Direct for 14 days            |
