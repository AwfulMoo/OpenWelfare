Note:

-   addressee can be 1 or 2 parents, a guardian or an organisation or a
    combination of all
-   Centrelink can send Medicare a maximum of 2 addressees per teenager
-   DVA can send Medicare a maximum of 1 addressee per teenager
-   Addressee details are not displayed for a child/teenager who is
    eligible for Youth Allowance or ABSTUDY. In this case, the fields in
    this tab are blank and the letter of notification (when sent) is
    addressed to the child/teenager.

Entitlement Details tab

Table 3: This table describes the field descriptions for the Entitlement
Details tab in CDMS.

+-----------------------------------+-----------------------------------+
| Field                             | Description                       |
+-----------------------------------+-----------------------------------+
| Currently Eligible                | Yes/No                            |
+-----------------------------------+-----------------------------------+
| Date of Inquiry                   | Displays the current date of      |

+-----------------------------------+-----------------------------------+
| Current Entitlement Period        | Displays the 2 consecutive        |

|                                   | The first calendar year is        |

|                                   | first CDBS service.               |
|                                   |                                   |
|                                   | Example: A child/teenager         |
|                                   | receives and claims their first   |
|                                   | dental service in 2021.           |
|                                   | Therefore, their current          |
|                                   | entitlement period is 2021-2022   |
|                                   |                                   |
|                                   | If a child/teenager is eligible   |
|                                   | for CDBS in 2021 but does not     |
|                                   | have any dental services in 2021, |

|                                   | is 2022-2023                      |

|                                   | Note: A child/teenager must meet  |
|                                   | eligibility criteria for each     |
|                                   | calendar year of entitlement      |

| CAP Amount                        | Displays the CDBS cap amount.     |
|                                   |                                   |
|                                   | The cap amount is indexed yearly  |
|                                   | from 1 January. The increase in   |
|                                   | cap amount will only apply to a   |
|                                   | child/teenager who received their |
|                                   | first eligible service in that    |
|                                   | year.                             |
|                                   |                                   |

|                                   | their first service in 2022       |

|                                   | year eligibility is 2022).        |

|                                   | 2022 is $1,026 this amount will   |

|                                   |                                   |

|                                   | year 1 of their 2 year            |
|                                   | eligibility in 2021) they will    |
|                                   | receive the 2021 benefit cap      |
|                                   | amount of $1,013 which will apply |
