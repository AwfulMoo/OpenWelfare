Checking the Dementia and Cognition Supplement form

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if the form is complete +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check the AC014 form has the      |
|                                   | following information:            |
|                                   |                                   |
|                                   | Eligibility assessment            |
|                                   |                                   |
|                                   | -   One (1) eligibility           |
|                                   |     assessment criteria score     |
|                                   |     must be provided              |
|                                   |                                   |
|                                   | Service details                   |
|                                   |                                   |
|                                   | -   Service name                  |
|                                   | -   Service ID                    |
|                                   |                                   |
|                                   | Care recipient details            |
|                                   |                                   |
|                                   | -   Care recipient ID             |
|                                   | -   Title                         |
|                                   | -   Family name                   |
|                                   | -   First given name              |
|                                   | -   Second given name (if         |
|                                   |     applicable)                   |
|                                   | -   Date of birth                 |
|                                   |                                   |
|                                   | Declaration                       |
|                                   |                                   |
|                                   | -   Authorised person’s full name |
|                                   | -   Authorised person’s           |
|                                   |     signature, and                |
|                                   | -   Date                          |
|                                   |                                   |
|                                   | The form is incomplete if any of  |
|                                   | the above is missing or           |
|                                   | incomplete.                       |
|                                   |                                   |
|                                   | Is the form complete?             |
|                                   |                                   |
|                                   | -   Yes, See Table 2 to key the   |
|                                   |     care recipient’s details into |
|                                   |     ACMPS. Procedure ends here    |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Returning an incomplete form +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Contact the aged care service to  |
|                                   | advise the missing information    |
|                                   | and that the form must be         |
|                                   | resubmitted via:                  |
|                                   |                                   |
|                                   | -   phone                         |
|                                   | -   email, or                     |
|                                   | -   mail. See Aged Care letter    |
|                                   |     templates                     |
|                                   |                                   |
|                                   | Record a note in PaNDA advising   |
|                                   | the action taken.                 |
|                                   |                                   |
|                                   | Procedure ends here.              |
|                                   |                                   |
|                                   | Note: when resubmitted it is      |
|                                   | treated as a new application.     |

View and code Dementia Supplement details for care recipient in ACMPS

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Search care recipient + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Log on to ACMPS                   |
|                                   |                                   |
|                                   | -   On the Home screen - select   |
|                                   |     Manage Partner on the left    |
|                                   |     hand side of the menu         |
|                                   | -   Select Manage Care Recipient  |
|                                   |     option                        |
|                                   | -   The Search Care Recipient     |
|                                   |     screen displays               |
|                                   | -   Key Care Recipient ID or      |
|                                   |     search by surname, first      |
|                                   |     name, DOB                     |
|                                   | -   Death indicator as not set    |
|                                   |     for current care recipient or |
|                                   |     set for deceased care         |
|                                   |     recipient                     |
|                                   | -   Select Search                 |
|                                   | -   Select the relevant care      |
|                                   |     recipient. The Care Recipient |
|                                   |     Profile screen will display   |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   the search fields displayed   |
|                                   |     are not mandatory             |
|                                   | -   the search operation can be   |
|                                   |     performed using any single    |
|                                   |     parameter or multiple         |
|                                   |     parameters                    |
|                                   | -   the search parameters can be  |
|                                   |     selected from the drop down   |
|                                   |     box                           |
|                                   | -   search rows can be            |
|                                   |     copied/removed by clicking    |
|                                   |     the plus (+) or minus (-)     |
|                                   |     icon                          |
|                                   | -   to clear the search values,   |
|                                   |     select Clear. This clears the |
|                                   |     values in the search fields   |
|                                   |     but retains any new search    |
|                                   |     criteria added                |
+-----------------------------------+-----------------------------------+
| 2                                 | View Dementia details + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   In the Care Recipient Profile |
|                                   |     screen select Dementia        |
|                                   |     Details tab at the top of the |
|                                   |     screen                        |
|                                   | -   The Dementia Supplement       |
|                                   |     details will display          |
|                                   |                                   |
|                                   | Note: if a particular tab is not  |
|                                   | visible on the screen, select     |
|                                   | Display tabs icon and select the  |
|                                   | relevant tab from the list.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Code Dementia details + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | In the Dementia Supplement tab,   |
|                                   | select the Edit List icon         |
|                                   |                                   |
|                                   | Key the required information in   |
|                                   | the fields provided               |
|                                   |                                   |
|                                   | -   Date of Diagnosis - select    |
|                                   |     Date icon to display the      |
|                                   |     calendar                      |
|                                   | -   Diagnosis Date of Receipt -   |
|                                   |     select Date icon to display   |
|                                   |     the calendar                  |
|                                   |                                   |
|                                   | Select Save to save the updated   |
|                                   | information                       |
|                                   |                                   |
|                                   | Is the updated information saved? |
|                                   |                                   |
|                                   | -   Yes, the care recipient’s     |
|                                   |     dementia details are saved    |
|                                   |     and a message will display as |
|                                   |     confirmation. Procedure ends  |
|                                   |     here                          |
|                                   | -   No, Service Officers are      |
|                                   |     unable to save the dementia   |
|                                   |     details due to an error. Go   |
|                                   |     to Step 4                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Unable to save due to error       |
|                                   | message + Read more ...           |
|                                   |                                   |
|                                   | If the error message - Duplicate  |
|                                   | record with similar values for    |
|                                   | RCT Data. Please Check is         |
|                                   | encountered, Service Officers     |
|                                   | must ensure the escalation        |
|                                   | process is actioned               |
|                                   | appropriately:                    |
|                                   |                                   |
|                                   | -   Open mySupport                |
|                                   | -   In the Search field, key in   |
|                                   |     ‘ACMPS issues’. The search    |
|                                   |     results will display          |
|                                   | -   Select Aged Care Management   |
|                                   |     Payment System (ACMPS) issues |
|                                   |     mySupport form                |
|                                   | -   The Aged Care Management      |
|                                   |     Payment System (ACMPS) issues |
|                                   |     mySupport form will display   |
|                                   | -   Complete all mandatory fields |
|                                   | -   Select Submit                 |
|                                   | -   Once the error has been fixed |
|                                   |     and advised by IT, repeat     |
|                                   |     Step 3                        |
