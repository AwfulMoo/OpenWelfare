Nominee IM and enhance IM scenarios and suspension rules - hierarchy

This table describes several scenarios relating to nominee Income
Management and enhanced IM, and nominee Income Management and enhanced
IM suspension rules.

+-----------------------------------+-----------------------------------+
| Item                              | Scenario                          |
+-----------------------------------+-----------------------------------+
| 1                                 | John's scenario                   |
|                                   |                                   |
|                                   | John is not currently welfare     |
|                                   | quarantined, but his payment      |
|                                   | nominee, Suzie, has just started  |
|                                   | enhanced IM under the Long Term   |
|                                   | Welfare Payment Recipient (LTWPR) |
|                                   | measure. John is switched onto    |
|                                   | NeIM-CeIM. John contacts two      |
|                                   | weeks after being placed on       |
|                                   | NIM-CIM to transfer to Voluntary  |
|                                   | enhanced IM (VeIM). John is not   |
|                                   | eligible for VeIM as he is        |
|                                   | already being welfare quarantined |
|                                   | under another measure.            |
+-----------------------------------+-----------------------------------+
| 2                                 | James' scenario                   |
|                                   |                                   |
|                                   | James is currently on VeIM and    |
|                                   | decides to appoint a payment      |
|                                   | nominee, as he will be in         |
|                                   | hospital for eight weeks. He      |
|                                   | appoints his sister Julie to be   |
|                                   | his payment nominee and she is    |
|                                   | welfare quarantined under the     |
|                                   | Disengaged Youth (DEY) measure.   |
|                                   | NeIM-CeIM is manually switched on |
|                                   | in the background. James will     |
|                                   | continue to be welfare            |
|                                   | quarantined under VeIM as the     |
|                                   | agreement was in place before     |
|                                   | NeIM-CeIM being switched on.      |
+-----------------------------------+-----------------------------------+
| 3                                 | June's scenario                   |
|                                   |                                   |
|                                   | June is currently welfare         |
|                                   | quarantined under the Vulnerable  |
|                                   | Welfare Payment Recipient (VWPR)  |
|                                   | measure, and her payment nominee  |
|                                   | (partner) is welfare quarantined  |
|                                   | under the Child Protection (CP)   |
|                                   | measure. NeIM-CP is manually      |
|                                   | switched on in the background.    |
|                                   | When the Vulnerable notice        |
|                                   | expires, VWPR ends, June will     |
|                                   | continue to be welfare            |
|                                   | quarantined under NeIM-CP,        |
|                                   | meaning the percentage of her     |
|                                   | payments being welfare            |
|                                   | quarantined will be 70%.          |
+-----------------------------------+-----------------------------------+
| 4                                 | Nominee IM suspension rules       |
|                                   |                                   |
|                                   | -   If a payment nominee's        |
|                                   |     payments are SUS-INM for      |
|                                   |     failing to attend an Initial  |
|                                   |     Assessment interview,         |
|                                   |     principals welfare            |
|                                   |     quarantined in their own      |
|                                   |     right will not have their     |
|                                   |     payments suspended            |
|                                   | -   A principal who is eligible   |
|                                   |     for Income Management in      |
|                                   |     their own right and whose     |
|                                   |     payment nominee does not      |
|                                   |     attend the Initial Assessment |
|                                   |     interview within the relevant |
|                                   |     engagement period for that    |
|                                   |     measure will have their       |
|                                   |     payments suspended            |
|                                   | -   If a principal's payments are |
|                                   |     welfare quarantined under     |
|                                   |     Nominee Income Management and |
|                                   |     their payment nominee's       |
|                                   |     payments have been SUS-INM    |
|                                   |     for failing to attend their   |
|                                   |     (payment nominee's) Initial   |
|                                   |     Assessment interview, the     |
|                                   |     principal's payments will     |
|                                   |     also be suspended             |
+-----------------------------------+-----------------------------------+
