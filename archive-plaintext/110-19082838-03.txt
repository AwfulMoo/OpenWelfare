| 11                                | Assess superannuation income +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Following retirement from the     |
|                                   | workforce, a person can access    |
|                                   | their superannuation, receiving   |
|                                   | either lump sum withdrawals from  |
|                                   | their superannuation fund or      |
|                                   | regular ongoing payments from an  |
|                                   | income stream product such as an  |
|                                   | annuity or a superannuation       |
|                                   | pension.                          |
|                                   |                                   |
|                                   | Go to the relevant screen in      |
|                                   | Customer Summary by keying the    |
|                                   | screen name in the Super Key      |
|                                   | field.                            |
|                                   |                                   |
|                                   | To enter a new line:              |
|                                   |                                   |
|                                   | -   select [Adding a new line]    |
|                                   | -   enter the required updates    |
|                                   | -   select Save                   |
|                                   |                                   |
|                                   | To edit details already coded:    |
|                                   |                                   |
|                                   | -   select Edit to edit the data  |
|                                   | -   edit the information as       |
|                                   |     required                      |
|                                   | -   select Save                   |
|                                   |                                   |
|                                   | If superannuation has:            |
|                                   |                                   |
|                                   | -   Been coded before, update the |
|                                   |     balance if the customer has   |
|                                   |     made a withdrawal.            |
|                                   |     Withdrawals from a            |
|                                   |     superannuation fund are not   |
|                                   |     an income stream, see         |
|                                   |     Assessing withdrawals of      |
|                                   |     superannuation                |
|                                   | -   Not been coded before, see    |
|                                   |     Assessing superannuation      |
