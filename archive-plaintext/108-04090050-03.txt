|                                   |                                   |
|                                   | Offer the customer a trial        |
|                                   | assessment of their managed       |
|                                   | investments and/or shares.        |

|                                   |                                   |
|                                   | If the automatic bulk update for  |
|                                   | 20 March or 20 September has been |
|                                   | loaded onto the customer's        |
|                                   | record, it will show an Event     |
|                                   | Date in the future.               |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   today's date is 15 March, and |
|                                   | -   the Event Date for the        |
|                                   |     investment shows 20 March     |
|                                   |                                   |
|                                   | In these cases, the workaround    |
|                                   | for MIS/SIS updates immediately   |
|                                   | prior to 20 March or 20 September |
|                                   | must be followed.                 |
|                                   |                                   |
|                                   | If the customer is:               |
|                                   |                                   |
|                                   | asking for a revaluation only, go |
|                                   | to Step 2                         |
|                                   |                                   |
|                                   | advising details have changed,    |
|                                   | for example some units or shares  |
|                                   | have been sold, see:              |
|                                   |                                   |
|                                   | -   Updating shares and           |
|                                   |     securities, or                |
|                                   | -   Managed investments -         |
|                                   |     updating existing investments |
+-----------------------------------+-----------------------------------+
| 2                                 | Bulk update of all listed         |
|                                   | shares + Read more ...            |
|                                   |                                   |
|                                   | Tell the customer there is a bulk |
|                                   | update of the value of all listed |
|                                   | shares and unitised managed       |
|                                   | investments each March and        |
|                                   | September.                        |
|                                   |                                   |
|                                   | Does the customer want a          |
|                                   | revaluation of their shares now?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 3                                 | Change in value of one            |
|                                   | investment + Read more ...        |
|                                   |                                   |
|                                   | Tell the customer a revaluation   |
|                                   | will include all their listed     |
|                                   | shares and unitised managed       |
|                                   | investments. This may increase    |
|                                   | the asset value of their total    |
|                                   | investment portfolio.             |
|                                   |                                   |
|                                   | Offer the customer a trial        |
|                                   | assessment of their managed       |
|                                   | investments and/or shares.        |
|                                   |                                   |
|                                   | Does the customer want a trial    |
|                                   | assessment?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 4                                 | Trial assessment + Read more ...  |
|                                   |                                   |
|                                   | In Customer First or Customer     |
|                                   | Record go to either the:          |
|                                   |                                   |
|                                   | Managed Investments Summary (MIS) |
|                                   | or Shares and Investments Summary |
|                                   | (SIS) screens.                    |
|                                   |                                   |
|                                   | Make a note of the value of the   |
|                                   | managed investments or shares.    |
|                                   | Enter:                            |
|                                   |                                   |
|                                   | -   'Y' in the Calculate field    |
|                                   | -   the date of event in the      |
|                                   |     Event Date field              |
|                                   | -   'Y' in the Trial field        |
|                                   | -   the appropriate code in the   |
|                                   |     Source field                  |
|                                   | -   the date of receipt in the    |
|                                   |     DOR field                     |
|                                   | -   select Continue               |
|                                   |                                   |
|                                   | Compare the updated values to the |
|                                   | previous values. Tell the         |
|                                   | customer the outcome.             |
|                                   |                                   |
|                                   | Does the customer want a          |
|                                   | revaluation of their shares and   |
|                                   | managed investments?              |
|                                   |                                   |
|                                   | Yes, go to Step 5                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   go to the Activity List (AL)  |
|                                   |     screen                        |
|                                   | -   delete (D) the trial activity |
|                                   | -   select Continue               |
|                                   | -   confirm cancel activity       |
|                                   | -   go to Step 8                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Cancel the trial activity + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | On the MIS/SIS screens, enter 'N' |
|                                   | in the Trial field.               |
|                                   |                                   |
|                                   | This converts the trial activity  |
|                                   | into a normal activity. Finalise  |
|                                   | the activity on the Assessment    |
|                                   | Results (AR) screen.              |
|                                   |                                   |
|                                   | Complete the recalculation.       |
+-----------------------------------+-----------------------------------+
| 6                                 | Recalculation + Read more ...     |
|                                   |                                   |
|                                   | Go to either the MIS or SIS       |
|                                   | screens in the customer's record. |
|                                   |                                   |
|                                   | -   Key 'Y' in the Calculation    |
|                                   |     field                         |
|                                   | -   Complete the Source and DOR   |
|                                   |     fields                        |
|                                   | -   Select Continue               |
|                                   | -   Complete the activity on the  |
|                                   |     AR screen                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Recalculation of asset value +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The system recalculates the asset |
|                                   | value on all the customer's       |
|                                   | listed shares and unitised        |
|                                   | managed funds. The current values |
|                                   | from the Managed Investments      |
|                                   | (MIN) and Securities and          |
|                                   | Investments (SIN) databases are   |
|                                   | used. The deemed income also      |
|                                   | updates.                          |
+-----------------------------------+-----------------------------------+
| 8                                 | Final step + Read more ...        |
|                                   |                                   |
|                                   | Record details on a DOC.          |
+-----------------------------------+-----------------------------------+
