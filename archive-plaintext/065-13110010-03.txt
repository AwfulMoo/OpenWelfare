Cancelling SPARC access

Table 5: this table describes how to cancel access to SPARC.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Reasons for SPARC access          |
|                                   | cancellation requests + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | It is the Line Manager’s          |
|                                   | responsibility to ensure system   |
|                                   | access is ceased when it is no    |
|                                   | longer appropriate.               |
|                                   |                                   |
|                                   | A user’s SPARC access requires    |
|                                   | cancellation when they:           |
|                                   |                                   |
|                                   | -   cease employment with         |
|                                   |     Services Australia            |
|                                   | -   move to a role that does not  |
|                                   |     require SPARC access, or      |
|                                   | -   take long term leave          |
+-----------------------------------+-----------------------------------+
| 2                                 | Request SPARC access              |
|                                   | cancellation + Read more ...      |
|                                   |                                   |
|                                   | The user’s line manager must send |
|                                   | a SPARC account cancellation      |
|                                   | request to the Operational Change |
|                                   | and Correspondence team. The      |
|                                   | email must include:               |
|                                   |                                   |
|                                   | the user’s SPARC Logon ID, and    |
|                                   |                                   |
|                                   | the cancellation reason – for     |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   left agency                   |
|                                   | -   left role and no longer       |
|                                   |     requires SPARC access         |
|                                   | -   long term leave               |
|                                   |                                   |
|                                   | See the Resources page for        |
|                                   | contact details.                  |
|                                   |                                   |
|                                   | Procedure ends here.              |

| Step                              | Action                            |

|                                   | Read more ...                     |
|                                   |                                   |

|                                   | after keying in their SPARC user  |
