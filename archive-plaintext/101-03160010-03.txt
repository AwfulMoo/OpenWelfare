Customer fails to comply with their ParentsNext participation requirements

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | ParentsNext action required +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a customer:                    |
|                                   |                                   |
|                                   | []has failed to attend their      |
|                                   | scheduled Services Australia      |
|                                   | ParentsNext appointment, go to    |
|                                   | Step 2                            |
|                                   |                                   |
|                                   | [] contacts because:              |
|                                   |                                   |
|                                   | -   they want to reschedule their |
|                                   |     Services Australia            |
|                                   |     ParentsNext appointment,      |
|                                   |     advise them if appropriate    |
|                                   |     that they can reschedule      |
|                                   |     their appointments online.    |
|                                   |     See ParentsNext eligibility   |
|                                   |     and servicing                 |
|                                   | -   their payment has been        |
|                                   |     suspended FDH (failed to      |
|                                   |     attend Services Australia     |
|                                   |     ParentsNext appointment), go  |
|                                   |     to Step 4                     |
|                                   | -   their payment has been        |
|                                   |     cancelled FDH, advise them    |
|                                   |     their payment cannot be       |
|                                   |     restored. They will need to   |
|                                   |     lodge a new claim if they     |
|                                   |     still need income support     |
|                                   |     assistance. For more          |
|                                   |     information, see Claiming     |
|                                   |     Parenting Payment (PP).       |
|                                   |     Procedure ends here           |
|                                   | -   their payment has been        |
|                                   |     suspended or cancelled due to |
|                                   |     compliance action from their  |
|                                   |     ParentsNext provider, see     |
|                                   |     Targeted Compliance Framework |
|                                   | -   their payment has been        |
|                                   |     cancelled FRP (failed to      |
|                                   |     report), see Cancellation,    |
|                                   |     suspension and rejection      |
|                                   |     codes for Parenting Payment   |
|                                   |     (PP). Procedure ends here     |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer has failed to attend a   |
|                                   | ParentsNext appointment with      |
|                                   | Services Australia + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | For ParentsNext processing        |
|                                   | Service Officers only.            |
|                                   |                                   |
|                                   | If a customer fails to attend     |
|                                   | either a ParentsNext phone        |
|                                   | (including where their contact    |
|                                   | number/s are disconnected) or     |
|                                   | face-to-face appointment, no      |
|                                   | further outbound contact is       |
|                                   | required. However, as             |
|                                   | circumstances can change which    |
|                                   | may result in an exit from the    |
|                                   | program, it must be confirmed the |
|                                   | customer is still eligible for    |
|                                   | ParentsNext before applying a     |
|                                   | suspension.                       |
|                                   |                                   |
|                                   | Navigate to the ParentsNext       |
|                                   | Summary page.                     |
|                                   |                                   |
|                                   | Is the customer still eligible as |
|                                   | a compulsory participant?         |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, the customer is no longer     |
|                                   | required to participate. To       |
|                                   | finalise the appointment:         |
|                                   |                                   |
|                                   | -   in Customer First, from Quick |
|                                   |     Links, select                 |
|                                   |     Appointments>Book and Manage  |
|                                   |     Appointments                  |
|                                   | -   finalise the appointment with |
|                                   |     an outcome of Completed       |
|                                   |     (COM). For more information,  |
|                                   |     see Complete and finalise     |
|                                   |     appointment in the Centrelink |
|                                   |     Appointment System. Procedure |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Suspend the customer's Parenting  |
|                                   | Payment (PP) + Read more ...      |
|                                   |                                   |
|                                   | For ParentsNext processing        |
|                                   | Service Officers only.            |
|                                   |                                   |
|                                   | As the customer has not attended  |
|                                   | a compulsory ParentsNext          |
|                                   | appointment (either phone or face |
|                                   | to face) their PP must be         |
|                                   | suspended.                        |
|                                   |                                   |
|                                   | Finalise the appointment          |
|                                   |                                   |
|                                   | -   In Customer First, from Quick |
|                                   |     Links, select                 |
|                                   |     Appointments>Book and Manage  |
|                                   |     Appointments                  |
|                                   | -   Finalise the appointment with |
|                                   |     an outcome of Did Not Attend  |
|                                   |     (DNA). For more information,  |
|                                   |     see Complete and finalise     |
|                                   |     appointment in the Centrelink |
|                                   |     Appointment System            |
|                                   |                                   |
|                                   | Suspend Parenting Payment (PP)    |
|                                   |                                   |
|                                   | -   In Customer First, go to the  |
|                                   |     Benefit Action (BA) screen    |
|                                   | -   Svc Rsn: PPP or PPS. Note:    |
|                                   |     for PPP ensure system is set  |
|                                   |     to PGA. For PPS, ensure       |
|                                   |     system is set to PEN          |
|                                   | -   Action: suspend               |
|                                   | -   Reason: FDH - Failed to       |
|                                   |     attend Services Australia     |
|                                   |     ParentsNext appointment - for |
|                                   |     all ParentsNext participants  |
|                                   | -   Effect Date: enter the date   |
|                                   |     the customer failed to attend |
|                                   |     the appointment               |
|                                   | -   Source: select relevant       |
|                                   |     source of information         |
|                                   | -   DOR: date the information was |
|                                   |     recorded                      |
|                                   | -   Select Submit Change and go   |
|                                   |     to the Assessment Results     |
|                                   |     (AR) screen and finalise the  |
|                                   |     activity                      |
|                                   |                                   |
|                                   | Create a closed DOC               |
|                                   |                                   |
|                                   | -   Use Fast Note – select Auto   |
|                                   |     text, use Parents > Update >  |
|                                   |     ParentsNext Suspension        |
|                                   | -   Once the text has been        |
|                                   |     confirmed, update all the     |
|                                   |     fields in the text to reflect |
|                                   |     circumstances                 |
|                                   | -   Complete the DOC including    |
|                                   |     how many ParentsNext          |
|                                   |     appointments have been missed |
|                                   |     in this ParentsNext           |
|                                   |     eligibility cycle             |
|                                   |                                   |
|                                   | For further details on creating a |
|                                   | Fast Note, see Creating,          |
|                                   | reviewing and deleting documents  |
|                                   | (including Fast Notes and DOA     |
|                                   | DOCs).                            |
|                                   |                                   |
|                                   | For more information, see Online  |
|                                   | Document Recording (ODR) and      |
|                                   | Recording reasons for decisions   |
|                                   | under Getting it Right (GIR).     |
|                                   |                                   |
|                                   | If a customer does not comply     |
|                                   | with their requirements within 13 |
|                                   | weeks their PP will automatically |
|                                   | cancel with the reason FDH. When  |
|                                   | PP cancels for this reason it     |
|                                   | cannot be restored and the        |
|                                   | customer must lodge a new claim.  |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Suspended due to failure to       |
|                                   | attend a ParentsNext              |
|                                   | appointment + Read more ...       |
|                                   |                                   |
|                                   | If the customer has missed 3 or   |
|                                   | more ParentsNext appointments     |
|                                   | with Services Australia within    |
|                                   | this ParentsNext eligibility      |
|                                   | cycle (since the most recent time |
|                                   | they were selected as a           |
|                                   | compulsory participant), payment  |
|                                   | is not to be restored until the   |
|                                   | appointment has been conducted.   |
|                                   | Check the suspension DOC or the   |
|                                   | customer's appointment history to |
|                                   | see if they have missed previous  |
|                                   | ParentsNext appointments.         |
|                                   |                                   |
|                                   | Go to the ParentsNext Summary     |
|                                   | page for information about the    |
|                                   | customer's eligibility.           |
|                                   |                                   |
|                                   | Go to the customer details page   |
|                                   | in Customer First by clicking on  |
|                                   | the CRN/BP hyperlink and then     |
|                                   | select the appointments tab.      |
|                                   |                                   |
|                                   | Has the customer missed fewer     |
|                                   | than 3 ParentsNext appointments   |
|                                   | with Services Australia within    |
|                                   | this ParentsNext eligibility      |
|                                   | cycle?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer has missed fewer than 3  |
|                                   | ParentsNext appointments + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A future ParentsNext appointment  |
|                                   | must be booked and Parenting      |
|                                   | Payment (PP) must be restored at  |
|                                   | the first point of contact.       |
|                                   |                                   |
|                                   | Do not book ParentsNext           |
|                                   | appointments before 9.00am in the |
|                                   | customer’s local time zone unless |
|                                   | they ask to do so. Phone          |
|                                   | appointments will not generally   |
|                                   | be available before 9.00am.       |
|                                   |                                   |
|                                   | To book a ParentsNext             |
|                                   | appointment:                      |
|                                   |                                   |
|                                   | -   Use the correct Centrelink    |
|                                   |     Appointment System time       |
|                                   |     display. See the Appointment  |
|                                   |     time display attachment in    |
|                                   |     Booking appointments in the   |
|                                   |     Centrelink Appointment System |
|                                   | -   Book a ParentsNext            |
|                                   |     appointment. For more help,   |
|                                   |     see Booking appointments in   |
|                                   |     the Centrelink Appointment    |
|                                   |     System                        |
|                                   | -   Advise the customer of the    |
|                                   |     appointment details, their    |
|                                   |     obligation to attend, and     |
|                                   |     that if they miss another     |
|                                   |     ParentsNext appointment their |
|                                   |     PP may again be suspended.    |
|                                   |     Clearly document this on the  |
|                                   |     customer’s record             |
+-----------------------------------+-----------------------------------+
| 6                                 | Restore Parenting Payment + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the customer's Family Tax      |
|                                   | Benefit (FTB) was cancelled       |
|                                   | because of the PP being           |
|                                   | suspended/cancelled, restore FTB  |
|                                   | first if the customer is eligible |
|                                   | for Rent Assistance (RA).         |
|                                   |                                   |
|                                   | To restore Parenting Payment:     |
|                                   |                                   |
|                                   | In Customer First, go to the      |
|                                   | Benefit Action (BA) screen, code: |
|                                   |                                   |
|                                   | -   Svc Rsn: PPP or PPS. Note:    |
|                                   |     for PPP ensure system is set  |
|                                   |     to PGA. For PPS ensure system |
|                                   |     is set to PEN                 |
|                                   | -   Action: Restore               |
|                                   | -   Reason: leave blank           |
|                                   | -   Effect Date: select [Enter]   |
|                                   |     for the date of effect to     |
|                                   |     default to the date of        |
|                                   |     suspension                    |
|                                   | -   Source: select relevant       |
|                                   |     source                        |
|                                   | -   DOR: today                    |
|                                   |                                   |
|                                   | Select Submit Change and go to    |
|                                   | the Assessment Results (AR)       |
|                                   | screen and finalise the activity  |
|                                   |                                   |
|                                   | Record the details on a DOC. In   |
|                                   | Customer First, a Fast Note can   |
|                                   | be recorded by selecting          |
|                                   | Parents > SUS/CAN/RES > PP        |
|                                   | restored and add the restoration  |
|                                   | is due to the customer complying  |
|                                   | with their ParentsNext            |
|                                   | requirements                      |
|                                   |                                   |
|                                   | Note: if the activity is selected |
|                                   | for Quality Online Checking (QOL) |
|                                   | it must not be returned because   |
|                                   | of no DOC on the record. Once the |
|                                   | activity is released, the correct |
|                                   | DOC (meeting the Getting it Right |
|                                   | standards) will be created.       |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Customer has missed 3 or more     |
|                                   | ParentsNext appointments + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A new appointment must be booked  |
|                                   | and the customer must be advised  |
|                                   | that their PP will remain         |
|                                   | suspended until they attend the   |
|                                   | new ParentsNext appointment. This |
|                                   | is to ensure the customer does    |
|                                   | not continually miss appointments |
|                                   | to avoid commencing in the        |
|                                   | program.                          |
|                                   |                                   |
|                                   | Do not book ParentsNext           |
|                                   | appointments before 9.00am in the |
|                                   | customer’s local time zone unless |
|                                   | they ask. Phone appointments will |
|                                   | not generally be available before |
|                                   | 9.00am.                           |
|                                   |                                   |
|                                   | To book a ParentsNext             |
|                                   | appointment:                      |
|                                   |                                   |
|                                   | -   Use the correct Centrelink    |
|                                   |     Appointment System time       |
|                                   |     display. See the Appointment  |
|                                   |     time display attachment in    |
|                                   |     Booking appointments in the   |
|                                   |     Centrelink Appointment System |
|                                   | -   Book a ParentsNext            |
|                                   |     appointment. Appointment type |
|                                   |     details. For more help, see   |
|                                   |     Booking appointments in the   |
|                                   |     Centrelink Appointment System |
|                                   |                                   |
|                                   | Can the next available            |
|                                   | appointment be booked before the  |
|                                   | customer's next Entitlement       |
|                                   | Period End Day (EPED)?            |
|                                   |                                   |
|                                   | -   Yes, advise the customer      |
|                                   |     their payment will be         |
|                                   |     restored once they have       |
|                                   |     attended their initial        |
|                                   |     ParentsNext appointment with  |
|                                   |     Services Australia. Procedure |
|                                   |     ends here                     |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 8                                 | No appointments available before  |
|                                   | the customer's next EPED + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the appointment booked was a   |
|                                   | phone appointment, also check the |
|                                   | availability of face-to-face      |
|                                   | appointments in the local service |
|                                   | centre.                           |
|                                   |                                   |
|                                   | Is there a face-to-face           |
|                                   | appointment available before the  |
|                                   | customer's next EPED?             |
|                                   |                                   |
|                                   | -   Yes, book the ParentsNext     |
|                                   |     appointment and advise the    |
|                                   |     customer their payments will  |
|                                   |     be restored once they attend  |
|                                   |     this appointment. Clearly     |
|                                   |     document the customer's       |
|                                   |     record                        |
|                                   | -   No, book the next available   |
|                                   |     appointment (phone or         |
|                                   |     face-to-face). Advise the     |
|                                   |     customer that if they need an |
|                                   |     earlier appointment they will |
|                                   |     need to attend their local    |
|                                   |     service centre to have this   |
|                                   |     appointment completed.        |
|                                   |     Clearly document this on the  |
|                                   |     customer's record             |
