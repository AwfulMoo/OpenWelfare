Updating or reviewing the FTB/CCS income estimate

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check the date the FEG or GEERS   |
|                                   | payment was received and the date |
|                                   | the FTB/CCS income estimate was   |
|                                   | last revised + Read more ...      |
|                                   |                                   |
|                                   | Does the customer need to provide |
|                                   | a new estimate?                   |
|                                   |                                   |
|                                   | -   Yes, contact the customer     |
|                                   |     again (via phone or letter)   |
|                                   |     requesting further            |
|                                   |     information then go to Step 2 |
|                                   | -   No, annotate the existing DOC |
|                                   |     to record the current FTB/CCS |
|                                   |     estimate to include the FEG   |
|                                   |     or GEERS payments. Procedure  |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Set up a review if required +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the information is not         |
|                                   | available immediately, set up a   |
|                                   | review to check the customer has  |
|                                   | returned the required             |
|                                   | information.                      |
|                                   |                                   |
|                                   | When details are provided, go to  |
|                                   | Step 3.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | New FTB/CCS estimate provided +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When the customer provides a new  |
|                                   | FTB/CCS estimate, run the Family  |
|                                   | Income and Choices workflow to    |
|                                   | update the customer's estimate.   |
|                                   |                                   |
|                                   | Note: FEG or GEERS payments are   |
|                                   | usually taxable income for the    |
|                                   | financial year in which the       |
|                                   | payments were received, however   |
|                                   | the taxable amount must be        |
|                                   | confirmed with the customer.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Department of Education, Skills   |
|                                   | and Employment Overpayment + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If GEERS income has previously    |
|                                   | been coded on the customer's      |
|                                   | record, reverse the income and if |
|                                   | appropriate pay the customer any  |
|                                   | arrears owed.                     |
