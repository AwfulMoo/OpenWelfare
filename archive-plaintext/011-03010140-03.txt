This page contains information about how to reinstate an IHI to its
former status where the IHI has been retired in error.

Reinstate IHI

This table describes the process Service Officers follow when restoring
an IHI to its former status where the IHI has been retired in error.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | When to reinstate an IHI + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Reinstate an IHI where either of  |
|                                   | the below have occurred for a     |
|                                   | Verified IHI:                     |
|                                   |                                   |
|                                   | -   a date of death has been      |
|                                   |     recorded, the death has been  |
|                                   |     confirmed by fact of death    |
|                                   |     data and no activity has been |
|                                   |     recorded against the IHI in   |
|                                   |     the previous 90 days          |
|                                   | -   the individual's age reaches  |
|                                   |     130 years                     |
|                                   |                                   |
|                                   | If 1 of the above has occurred,   |
|                                   | go to Step 2.                     |
|                                   |                                   |
|                                   | Otherwise procedure ends here.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Logon + Read more ...             |
|                                   |                                   |
|                                   | Logon to CDMS.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Amend consumer details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | From the CDMS home page select    |
|                                   | Amend Consumer Details            |
|                                   |                                   |
|                                   | The Amendment Search screen       |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Perform search + Read more ...    |
|                                   |                                   |
|                                   | Perform a basic or detailed       |
|                                   | search to locate the individual's |
|                                   | IHI.                              |
|                                   |                                   |
|                                   | See Search for records in         |
|                                   | Healthcare Identifiers (HI)       |
|                                   | Service and select Personal.      |
|                                   |                                   |
|                                   | The 'Amend IHI Details' screen    |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Check details + Read more ...     |
|                                   |                                   |
|                                   | Where the:                        |
|                                   |                                   |
|                                   | date of death is incorrectly      |
|                                   | recorded for this individual:     |
|                                   |                                   |
|                                   | -   delete the date of death      |
|                                   | -   set the date of death         |
|                                   |     accuracy indicator to blank   |
|                                   | -   set the source of death       |
|                                   |     notification to blank         |
|                                   |                                   |
|                                   | date of birth is incorrectly      |
|                                   | recorded, making the individual   |
|                                   | older than 130 years:             |
|                                   |                                   |
|                                   | -   correct the recorded date of  |
|                                   |     birth                         |
|                                   |                                   |
|                                   | select Validate                   |
|                                   |                                   |
|                                   | An information message displays   |
|                                   | informing if the amendments are   |
|                                   | acceptable or not.                |
+-----------------------------------+-----------------------------------+
| 6                                 | Check amendments + Read more ...  |
|                                   |                                   |
|                                   | Are the amendments acceptable?    |
|                                   |                                   |
|                                   | -   Yes, select OK. Go to Step 6  |
|                                   | -   No, change the amendments as  |
|                                   |     needed. Go to Step 6          |
+-----------------------------------+-----------------------------------+
| 7                                 | Validate result + Read more ...   |
|                                   |                                   |
|                                   | Scroll down and select Validate.  |
|                                   |                                   |
|                                   | The Confirm Amended IHI Details   |
|                                   | screen displays the updated       |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
| 8                                 | Confirm or correct details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the details are:               |
|                                   |                                   |
|                                   | -   not correct, select Change    |
|                                   |     and go back to Step 4         |
|                                   | -   correct, select Update        |
|                                   |                                   |
|                                   | A message appears to confirm that |
|                                   | the individual's details have     |
|                                   | been updated.                     |
+-----------------------------------+-----------------------------------+
