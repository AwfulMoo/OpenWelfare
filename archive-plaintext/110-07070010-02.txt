Flowchart

Reissuing an (AIC) End of Year review form

This flowchart summarises what to do when an applicant asks for a
replacement End of Year (EOY) review form. Portions of this flowchart
can only be completed by Assistance for Isolated Children (AIC) Smart
Centre Processing staff. Look up local Processing Service details for:
Assistance for Isolated Children (AIC).

Note: a text based version of the following process summary image is
available.

[When an applicant requests a replacement End of Year (EOY) review form,
check if it's been 26 weeks since the review was sent. If no, reprint
the form and provide advice. If yes, create a Fast Note - select Auto
Text, then Students, ABSTUDY and Apprentices > AIC > AIC update action
required. Some parts of this flowchart can only be completed by
Assistance for Isolated Children (AIC) Smart Centre Processing staff.] 
