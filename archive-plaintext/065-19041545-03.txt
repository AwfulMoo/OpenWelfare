Residential care manual means assessment - post 1 July 2014

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Request for a manual means        |
|                                   | assessment received after         |
|                                   | departure from care + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Does a correct and valid means    |
|                                   | assessment already exist for the  |
|                                   | date of entry to care? Note: this |
|                                   | includes means not disclosed      |
|                                   | (MND).                            |
|                                   |                                   |
|                                   | Yes                               |
|                                   |                                   |
|                                   | -   Complete a manual             |
|                                   |     reassessment                  |
|                                   | -   See Aged care reviews -       |
|                                   |     discharge and departure       |
|                                   |     reviews to complete this      |
|                                   |     process                       |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check if the care recipient has   |
|                                   | departed care and the reason for  |
|                                   | departure + Read more ...         |
|                                   |                                   |
|                                   | Access the Aged Care Staff Portal |
|                                   | (ACSP)                            |
|                                   |                                   |
|                                   | Locate the care recipient record  |
|                                   |                                   |
|                                   | Go to Care recipient > Care       |
|                                   | recipient summary screen to check |
|                                   | if a departure date is coded      |
|                                   |                                   |
|                                   | Check Customer First for any      |
|                                   | notification or coding showing    |
|                                   | the care recipient is deceased.   |
|                                   | This may include:                 |
|                                   |                                   |
|                                   | -   a person (including third     |
|                                   |     parties) contacting Services  |
|                                   |     Australia directly to advise  |
|                                   |     of the death of the           |
|                                   |     recipient, or                 |
|                                   | -   notification by a service     |
|                                   |     provider                      |
|                                   |                                   |
|                                   | Is there a departure from care in |
|                                   | ACSP or deceased coding (or       |
|                                   | notification) in Customer First?  |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Refer the manual assessment   |
|                                   |     request for Aged Care New     |
|                                   |     Claim - Experienced skill     |
|                                   |     tagged staff to complete      |
|                                   | -   Unassign the work item from   |
|                                   |     the Customer First inbox      |
|                                   | -   Recategorise the work item as |
|                                   |     ZIAM_RCA_DEA                  |
|                                   | -   Aged care staff without New   |
|                                   |     Claim - Experienced skill     |
|                                   |     tag, procedure ends here      |
|                                   | -   Aged care staff with New      |
|                                   |     Claim - Experienced skill     |
|                                   |     tag, go to Step 3             |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Complete a means assessment,  |
|                                   |     see Aged care means           |
|                                   |     assessment - preliminary      |
|                                   |     checks                        |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Check if there is enough          |
|                                   | information to complete the means |
|                                   | assessment + Read more ...        |
|                                   |                                   |
|                                   | Review the calculation of your    |
|                                   | cost of care answers against the  |
|                                   | Checklist to confirm:             |
|                                   |                                   |
|                                   | -   all supporting documentation  |
|                                   |     needed has been given         |
|                                   | -   the care recipient or         |
|                                   |     authorised third party has    |
|                                   |     signed all forms lodged - see |
|                                   |     Aged care - consent and       |
|                                   |     disclosure for more           |
|                                   |     information                   |
|                                   |                                   |
|                                   | Note: if the care recipient is    |
|                                   | deceased, confirm that evidence   |
|                                   | has been provided that the person |
|                                   | requesting the means assessment   |
|                                   | is the executor or administrator. |
|                                   | If not, this must be requested    |
|                                   | before any assessment is          |
|                                   | completed.                        |
|                                   |                                   |
|                                   | Care recipients in receipt of a   |
|                                   | means tested income support       |
|                                   | payment may be eligible for a     |
|                                   | verbal claim. See Verifying       |
|                                   | income and assets to determine    |
|                                   | when supporting documentation is  |
|                                   | needed.                           |
|                                   |                                   |
|                                   | Is more information needed to     |
|                                   | complete this means assessment?   |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | More information is needed to     |
|                                   | complete a means assessment +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Attempt to get the information    |
|                                   | verbally (from the care recipient |
|                                   | or authorised third party).       |
|                                   |                                   |
|                                   | If this is not possible:          |
|                                   |                                   |
|                                   | -   Request the outstanding       |
|                                   |     information using the 'Aged   |
|                                   |     Care Request for Further      |
|                                   |     Information' letter. See Aged |
|                                   |     care letters - Request for    |
|                                   |     information (RFI)             |
|                                   | -   Upload the completed letter   |
|                                   |     to Customer First in PDF      |
|                                   |     format                        |
|                                   | -   Record a progress of claim    |
|                                   |     (POC) document                |
|                                   | -   Hold for 21 days plus allow   |
|                                   |     extra time for mail delivery  |
|                                   |                                   |
|                                   | Did the care recipient or         |
|                                   | authorised third party give the   |
|                                   | information needed?               |
|                                   |                                   |
|                                   | Yes, go to Step 5                 |
|                                   |                                   |
|                                   | No, the care recipient is deemed  |
|                                   | not low means                     |
|                                   |                                   |
|                                   | -   Update the Progress of Claim  |
|                                   |     document with the outcome,    |
|                                   |     then go to Step 8             |
+-----------------------------------+-----------------------------------+
| 5                                 | Complete the means assessment +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Calculate the assessable income   |
|                                   | and assets.                       |
|                                   |                                   |
|                                   | If the care recipient is:         |
|                                   |                                   |
|                                   | Deceased (in Customer First)      |
|                                   |                                   |
|                                   | -   Complete the RCA/NCL activity |
|                                   |     in the Customer First         |
|                                   |     Training Environment          |
|                                   | -   See Customer First training   |
|                                   |     environment for assistance if |
|                                   |     required                      |
|                                   |                                   |
|                                   | Not deceased (in Customer First)  |
|                                   |                                   |
|                                   | -   Complete the RCA/NCL activity |
|                                   |     in Customer First             |
|                                   |                                   |
|                                   | In Customer First (or Training    |
|                                   | Environment):                     |
|                                   |                                   |
|                                   | Use the RCA Task Selector (RCTS)  |
|                                   | screen to code income and asset   |
|                                   | information                       |
|                                   |                                   |
|                                   | -   (Deceased care recipients     |
|                                   |     only) Code bank balances      |
|                                   |     together as a lump sum on the |
|                                   |     Savings screen                |
|                                   |                                   |
|                                   | Confirm the results on screens    |
|                                   |                                   |
|                                   | -   RCA Assessment Determination  |
|                                   |     (RCAD), and                   |
|                                   | -   RCA Income Assessment Summary |
|                                   |     (RIAS)                        |
|                                   |                                   |
|                                   | (Deceased care recipients only)   |
|                                   | Copy a screenshot of RCAD and     |
|                                   | RIAS into a Word document         |
|                                   |                                   |
|                                   | In the Aged Care Fee Calculator   |
|                                   | (available on the Resources       |
|                                   | page):                            |
|                                   |                                   |
|                                   | Use the Aged Care Fee Calculator  |
|                                   | tab to calculate the correct fees |
|                                   | for each period, based on the     |
|                                   | assessable income and assets from |
|                                   | RCAD and RIAS                     |
|                                   |                                   |
|                                   | A new calculation is needed for   |
|                                   | each change in income and assets, |
|                                   | including Consumer Price Index    |
|                                   | (CPI) and threshold changes       |
|                                   | (occurring on 20 March, 1 July    |
|                                   | and 20 September)                 |
|                                   |                                   |
|                                   | Use the Manual Adjustment         |
|                                   | Calculator tab                    |
|                                   |                                   |
|                                   | -   Key the correct fees (result  |
|                                   |     from the Aged Care Fee        |
|                                   |     Calculator tab)               |
|                                   | -   Complete all Care Recipient   |
|                                   |     Profile details               |
|                                   |                                   |
|                                   | In Customer First:                |
|                                   |                                   |
|                                   | Upload a copy of the calculations |

|                                   | -   Using Document Tools and Form |
