Policy FAQs

Service Officers and Social Workers need to use available reference
material to research a query before escalating to Service Support
Officers or a Level 1 contact. (Unless Operational Blueprint instructs a
direct referral to a Level 2 Policy Helpdesk).

The Level 1 contact in Social Work/F2F/Smart Centres will then
review/assess and if cannot answer they will escalate as needed (to
Level 2 Helpdesk).

| Anticipation of violence          | Can a customer qualify for Crisis |
|                                   | Payment due to an expectation of  |
|                                   | violence? + Read more ...         |
|                                   |                                   |
|                                   | For example, a customer believes  |
|                                   | there is a reasonable threat from |
|                                   | a partner being released from     |
|                                   | prison, tracking down and harming |
|                                   | the claimant. No actual threat    |
|                                   | has yet been made: no emails, no  |
|                                   | SMS, just an expectation of the   |
|                                   | customer/warning from the police  |
|                                   | that the customer is likely to be |
|                                   | harmed.                           |
|                                   |                                   |
|                                   | Social workers believe that if    |
|                                   | the authorities believe there is  |
|                                   | a valid future threat, the        |
|                                   | customer ought to qualify.        |
|                                   |                                   |
|                                   | No.                               |
|                                   |                                   |
|                                   | CrP cannot be paid pre-emptively  |
|                                   | and the customer must have left   |
|                                   | the home as outlined in           |
|                                   | Operational Blueprint.            |
|                                   |                                   |
|                                   | There is some misunderstanding    |
|                                   | that ‘intention to leave home’ is |
|                                   | somehow a qualifying condition.   |
|                                   | It is an incorrect merging of the |
|                                   | ‘left home’ and ‘established or   |
|                                   | intends to establish a new home’  |
|                                   | criteria.                         |
|                                   |                                   |
|                                   | Part 2.23A, 1061JH (1) of the     |
|                                   | Social Security Act 1991:         |
|                                   |                                   |
|                                   | -   a) the person has left, or    |
|                                   |     cannot return, to his or her  |
|                                   |     home because of an extreme    |
|                                   |     circumstance, and             |
|                                   | -   e) the person makes a claim   |
|                                   |     for CrP within 7 days after   |
|                                   |     the extreme circumstance      |
|                                   |     occurred                      |
|                                   |     Note: examples of extreme     |
|                                   |     circumstances that would      |
|                                   |     qualify are….. a person being |
|                                   |     subjected to domestic or      |
|                                   |     family violence               |
|                                   |                                   |
|                                   | A customer can test their         |
|                                   | eligibility for any payment by    |
|                                   | lodging a claim and being         |
|                                   | assessed for that payment.        |

| Financial hardship                | Can the customer’s joint bank     |
|                                   | account balances be excluded from |
|                                   | financial hardship assessment for |
|                                   | Crisis Payment? + Read more ...   |
|                                   |                                   |
|                                   | Generally No. However, if         |
|                                   | considered 'unrealisable' these   |
|                                   | funds may be excluded.            |
|                                   |                                   |
|                                   | Social Security Guide, 3.7.4.10,  |
|                                   | Qualification for CrP - general   |
|                                   | provisions has details on the     |
|                                   | severe financial hardship test.   |
|                                   | The policy states:                |
|                                   | 'When assessing hardship all cash |
|                                   | and readily realisable assets     |
|                                   | should be taken into account,     |
|                                   | including money in the bank and   |
|                                   | earnings due.'                    |
|                                   |                                   |
|                                   | Eligibility for Crisis Payment    |
|                                   | (CrP) explains that a 'readily    |
|                                   | realisable asset’ is any asset    |
|                                   | that is not deemed an             |
|                                   | unrealisable asset. In the case   |
|                                   | of Trust or Company income, this  |
|                                   | must be determined by a Complex   |
|                                   | Assessment Officer as per         |
|                                   | Unrealisable assets under the     |
|                                   | Assets Test hardship provisions.  |
|                                   |                                   |
|                                   | For example, the joint account    |
|                                   | has been frozen as part of        |
|                                   | separation or divorce proceedings |
|                                   | by either party, and the customer |
|                                   | provides proof of this. Social    |
|                                   | Security Guide, 4.6.7.60,         |
|                                   | Determining severe financial      |
|                                   | hardship - readily available      |
|                                   | funds.                            |

|                                   | for Crisis Payment? + Read more   |
|                                   | ...                               |
|                                   |                                   |

|                                   |                                   |
|                                   | New claim for Crisis Payment      |
|                                   | Other Extreme Circumstances       |

|                                   | people.                           |

|                                   | homeless.                         |
+-----------------------------------+-----------------------------------+
| Returning home                    | Does a customer qualify for EXV   |
|                                   | if they left the home due to      |
|                                   | family and domestic violence but  |
|                                   | advise at the time of the social  |
|                                   | worker assessment for the EXV     |
|                                   | claim that they have now returned |
|                                   | home? + Read more ...             |
|                                   |                                   |
|                                   | No.                               |
|                                   |                                   |
|                                   | The person who at the time of     |
|                                   | assessment has subsequently       |
|                                   | returned to their home after      |
|                                   | submitting a Crisis Payment claim |
|                                   | no longer qualifies under the     |
|                                   | extreme circumstances forcing     |
|                                   | departure from home category.     |
|                                   |                                   |
|                                   | A person who has returned home    |
|                                   | would generally not be able to    |
|                                   | satisfy sections (1)(b) and (c)   |
|                                   | of section 1061JH of the Social   |
|                                   | Security Act 1991.                |
|                                   |                                   |
|                                   | In cases of family and domestic   |
|                                   | violence, it is up to the social  |
|                                   | worker to be satisfied that it is |
|                                   | unreasonable to expect the person |
|                                   | to return home and that the       |
|                                   | person has established or intends |
|                                   | to establish a new home. If a     |
|                                   | person chooses to return home     |
|                                   | before their Crisis Payment claim |
|                                   | is assessed, the social worker    |
|                                   | would not be satisfied that these |
|                                   | provisions have been met.         |
|                                   |                                   |
|                                   | People returning to a violent     |
|                                   | relationship, particularly women, |
|                                   | must be made aware of the support |
|                                   | services available to them and    |
|                                   | offered appropriate referrals.    |
+-----------------------------------+-----------------------------------+
| Arrears payments                  | Does a customer with an arrears   |
|                                   | payment of an ISP or FTB have     |
|                                   | this money taken into account     |
|                                   | when assessing financial          |
|                                   | hardship? + Read more ...         |
|                                   |                                   |
|                                   | The financial hardship assessment |
|                                   | is taken on the date of claim,    |
|                                   | based on the information provided |
|                                   | in the claim by the customer:     |
|                                   | cash on hand, savings and         |
|                                   | investment balances including     |
|                                   | shares.                           |
|                                   |                                   |
|                                   | The system in Process Direct when |
|                                   | processing EXV does not look at   |
|                                   | the Payment Summary (PS) screen   |
|                                   | for upcoming payments/recent      |
|                                   | arrears to determine financial    |
|                                   | hardship.                         |

|                                   |                                   |
|                                   | Special Benefit is a qualifying   |
|                                   | payment for Crisis Payment        |
|                                   | purposes. The child is to be      |
