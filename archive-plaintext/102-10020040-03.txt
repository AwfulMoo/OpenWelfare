|                                   | concession + Read more ...        |

|                                   |     change of circumstances, and  |
|                                   | -   continues to attend the same  |
|                                   |     school or is enrolled at the  |
|                                   |     same distance education       |
|                                   |     institution                   |
|                                   |                                   |
|                                   | Does the applicant meet the above |
|                                   | criteria?                         |
|                                   |                                   |
|                                   | -   Yes, this procedure does not  |
|                                   |     apply. Go to Change of        |
|                                   |     Address. Procedure ends here  |
|                                   | -   No, go to Step 3              |

Actioning AIC debts

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check for Applicant overpayment   |
|                                   | or incorrect Third Party          |
|                                   | Organisation (TPO) payment + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Applicants are responsible for    |
|                                   | repaying any AIC payments         |
|                                   | received in excess of their       |
|                                   | entitlement.                      |
|                                   |                                   |
|                                   | In certain circumstances AIC      |
|                                   | payments may be recovered from a  |
|                                   | TPO due to either:                |
|                                   |                                   |
|                                   | -   an administration error       |
|                                   | -   a change in circumstances     |
|                                   |     advised by the applicant      |
|                                   |     within 14 days, and the funds |
|                                   |     still issue incorrectly, or   |
|                                   | -   the TPO contacts and wishes   |
|                                   |     to repay AIC funds they       |
|                                   |     incorrectly received          |
|                                   |                                   |
|                                   | Is recovery of AIC funds from a   |
|                                   | TPO to be actioned?               |
|                                   |                                   |
|                                   | -   Yes, Go to Table 3            |
|                                   |                                   |
|                                   | Did the applicant receive         |
|                                   | payments in excess of their       |
|                                   | entitlement and third party       |
|                                   | recovery does not apply?          |
|                                   |                                   |
|                                   | -   Yes, follow up and raise      |
|                                   |     overpayment. See Actioning an |
|                                   |     undetermined debt on the Debt |
|                                   |     Management and Information    |
|                                   |     System (DMIS)                 |
|                                   |                                   |
|                                   | No, Go to step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Manual follow up + Read more ...  |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Check the Activity List (AL)      |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Is there a Manual Follow Up (MFU) |
|                                   | on the AL screen?                 |
|                                   |                                   |
|                                   | -   Yes, create a manual letter.  |
|                                   |     Go to Step 3                  |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Send manual letter + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Send a manual letter (Q999) to    |
|                                   | the applicant.                    |
|                                   |                                   |
|                                   | -   Give the reason(s) for        |
|                                   |     cancellation of payment       |
|                                   | -   Include their rights of       |
|                                   |     review and appeal             |
+-----------------------------------+-----------------------------------+
| 4                                 | Document decision + Read more ... |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Record details on a DOC. Include  |

|                                   | Guidelines references.            |
|                                   |                                   |
|                                   | If the applicant has other AIC    |
|                                   | students, complete the above      |
|                                   | procedures for each student.      |

Third Party Organisation (TPO) payment recovery

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | TPO contacts for funds received + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a TPO contacts and wishes to   |
|                                   | repay AIC funds they incorrectly  |
|                                   | received, AIC Processing staff    |
|                                   | can issue a debt recovery notice. |
|                                   |                                   |
|                                   | Telephony Service Officers:       |
|                                   |                                   |
|                                   | Send a Fast Note to the AIC Smart |
|                                   | Centre. Select Auto text, use     |
|                                   | Students, ABSTUDY & Apprentices > |
|                                   | AIC update action required        |
|                                   |                                   |
|                                   | Include details of:               |
|                                   |                                   |
|                                   | -   TPO name and Customer         |
|                                   |     Reference number (CRN) of who |
|                                   |     received the funds            |
|                                   | -   date TPO contacted            |
|                                   | -   person who contacted          |
|                                   | -   who the incorrect entitlement |
|                                   |     was received for              |
|                                   | -   student’s date of birth (DOB) |
|                                   |     and CRN (if known)            |
|                                   | -   payment type, date received,  |
|                                   |     period it was for and the     |
|                                   |     amount                        |
|                                   | -   reason for raising debt (that |
|                                   |     is, administration error/TPO  |
|                                   |     wishes to pay)                |
|                                   |                                   |
|                                   | Tell the TPO this will be         |
|                                   | allocated to a Service Officer to |
|                                   | action. The Service Officer may   |
|                                   | contact them for more             |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Actioning TPO debt recovery +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Staff must:                       |
|                                   |                                   |
|                                   | confirm reason for the debt       |
|                                   |                                   |
|                                   | determine who is entitled to the  |
|                                   | funds (applicant, correct TPO)    |
|                                   |                                   |
|                                   | manually raise a debt on the      |
|                                   | TPO’s record for the amount they  |
|                                   | were incorrectly paid. Use        |
|                                   | standard debt raising processes   |
|                                   |                                   |
|                                   | contact the TPO and tell them:    |
|                                   |                                   |
|                                   | -   the agency will send them an  |
|                                   |     Accounts Payable letter to    |
|                                   |     recover the funds, and        |
|                                   | -   they must advise the agency   |
|                                   |     when they have returned the   |
|                                   |     funds                         |
|                                   |                                   |
|                                   | In addition to the TPO DOC,       |
|                                   | update the details on the AIC     |
|                                   | student’s record and include the  |
|                                   | Debt ID.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | TPO contacts to advise funds      |
|                                   | returned + Read more ...          |
|                                   |                                   |
|                                   | Go to Organisation Record in      |
|                                   | Customer First:                   |
|                                   |                                   |
|                                   | swap to Environment B             |
|                                   |                                   |
|                                   | view the debt details and confirm |
|                                   | the TPO has repaid the debt:      |
|                                   |                                   |
|                                   | -   go to Customer First >        |
|                                   |     Workspace                     |
|                                   | -   go to the Debts assignment    |
|                                   |     block                         |
|                                   | -   select Debt Information       |
|                                   |                                   |
|                                   | If the status of debt is:         |
|                                   |                                   |
|                                   | recovered:                        |
|                                   |                                   |
|                                   | -   send a Fast Note from the AIC |
|                                   |     student’s record              |
|                                   | -   use Fast Note - select Auto   |
|                                   |     text, use Students, ABSTUDY & |
|                                   |     Apprentices > AIC update      |
|                                   |     action required, for          |
|                                   |     processing staff to determine |
|                                   |     who is entitled to the funds  |
|                                   |     and reissue the funds if      |
|                                   |     required                      |
|                                   |                                   |
|                                   | showing determined, the recovery  |
|                                   | of funds has not been processed.  |
|                                   | Code a manual review:             |
|                                   |                                   |
|                                   | -   Service Reason: EIA           |
|                                   | -   Review Reason: Returned       |
|                                   |     Cheque                        |
|                                   | -   Due Date: 7 days              |
|                                   | -   Source: PHO                   |
|                                   | -   Date of Receipt               |
|                                   | -   DOC organisation contact      |
|                                   |                                   |
|                                   | When the funds have been          |
|                                   | recovered, reissue to the correct |
|                                   | applicant if required.            |
