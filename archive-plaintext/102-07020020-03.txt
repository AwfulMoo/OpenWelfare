Changing study load from full-time to part-time or concessional study load

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Study load to be reduced + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The YA, Austudy or PES student    |
|                                   | advises they are reducing their   |
|                                   | study load.                       |
|                                   |                                   |
|                                   | If their study load will be:      |
|                                   |                                   |
|                                   | -   75% or more of the normal     |
|                                   |     amount of full-time study, go |
|                                   |     to Step 3                     |
|                                   | -   less than 75% of the normal   |
|                                   |     amount of full time study, go |
|                                   |     to Step 2                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Aggregation of study load + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Aggregation is adding the study   |
|                                   | load of multiple study periods    |
|                                   | together to determine whether     |
|                                   | someone is undertaking full-time  |
|                                   | study. Therefore, if the student  |
|                                   | has dropped to a part-time study  |
|                                   | load in the current study period, |
|                                   | they may still be considered a    |
|                                   | full-time student if their study  |
|                                   | for the academic year can be      |
|                                   | added together to reach an        |
|                                   | aggregated 75% study load.        |
|                                   |                                   |
|                                   | For customer to take advantage of |
|                                   | the aggregation rules, they must  |
|                                   | be enrolled in all units/subjects |
|                                   | being included in the aggregation |
|                                   | assessment at the start of the    |
|                                   | study period. For further         |
|                                   | information regarding aggregation |
|                                   | of study load refer to Assessing  |
|                                   | study load requirements.          |
|                                   |                                   |
|                                   | Could the student still be        |
|                                   | considered full-time if their     |
|                                   | study load is aggregated over the |
|                                   | full academic year?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Full-time student + Read more ... |
|                                   |                                   |
|                                   | The customer is still considered  |
|                                   | to be a full-time student; there  |
|                                   | is no need to reassess payments.  |
|                                   |                                   |
|                                   | Tell the customer to notify       |
|                                   | Services Australia within 14 days |
|                                   | if their study load reduces to    |
|                                   | below 75%.                        |
|                                   |                                   |
|                                   | Record details of the assessment  |
|                                   | in a Note/DOC, listing the study  |
|                                   | periods included in the           |
|                                   | assessment and the study load for |
|                                   | each.                             |
|                                   |                                   |
|                                   | If the customer has been assessed |
|                                   | as a full-time student due to     |
|                                   | aggregation of study load:        |
|                                   |                                   |
|                                   | -   create a Fast Note. Select    |
|                                   |     Auto Text option > Students,  |
|                                   |     ABSTUDY & Apprentices >       |
|                                   |     TCSI > TCSI review exemption  |
|                                   | -   record study load aggregation |
|                                   |     as the reason for review      |
|                                   |     exemption                     |
|                                   | -   add the start and end dates   |
|                                   |     of all part-time study        |
|                                   |     periods included in the       |
|                                   |     aggregation assessment        |
|                                   |                                   |
|                                   | This will be referred to the TCSI |
|                                   | processing team to update.        |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Percentage range of normal study  |
|                                   | load + Read more ...              |
|                                   |                                   |
|                                   | When a PES customerâ€™s study load  |
|                                   | is reduced, allowable time and    |
|                                   | satisfactory progress may need to |
|                                   | be reassessed. See, Assessing     |
|                                   | satisfactory progress and         |
|                                   | allowable time for tertiary       |
|                                   | students for Pensioner Education  |
|                                   | Supplement (PES)                  |
|                                   |                                   |
|                                   | Study load is between 66% and     |
|                                   | 74%:                              |
|                                   |                                   |
|                                   | For customers undertaking between |
|                                   | 66% and 74% of the normal         |
|                                   | full-time study load, determine   |
|                                   | eligibility before returning to   |
|                                   | this process:                     |
|                                   |                                   |
|                                   | -   Assessing concessional study  |
|                                   |     loads for YA                  |
|                                   | -   Assessing eligibility for the |
|                                   |     66% concessional study load   |
|                                   |     for Austudy                   |
|                                   | -   Assessing eligibility for the |
|                                   |     66% concessional study load   |
|                                   |     for PES                       |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
|                                   |                                   |
|                                   | Study load is between 25% and     |
|                                   | 65%:                              |
|                                   |                                   |
|                                   | Customers receiving YA (student)  |
|                                   | cease to be eligible if their     |
|                                   | study load is less than 66%,      |
|                                   | unless they have a temporary      |
|                                   | incapacity exemption. If the YA   |
|                                   | (student) has reduced their study |
|                                   | load:                             |
|                                   |                                   |
|                                   | -   Due to a temporary            |
|                                   |     incapacity, see YA Student or |
|                                   |     Australian Apprentice becomes |
|                                   |     incapacitated                 |
|                                   | -   Due to the death of their     |
|                                   |     partner and has a current a   |
|                                   |     Partner Bereavement (BPP or   |
|                                   |     BPE) activity test exemption  |
|                                   |     on the AEX screen. See        |
|                                   |     Activity test exemptions for  |
|                                   |     YA (student) customers when   |
|                                   |     their partner dies. Procedure |
|                                   |     ends here                     |
|                                   | -   Voluntarily. This is the same |
|                                   |     as ceasing study. The study   |
|                                   |     details must be updated to    |
|                                   |     cancel the student payment.   |
|                                   |     See Table 2 in Payment of YA  |
|                                   |     and Austudy when a student is |
|                                   |     deferring or discontinuing    |
|                                   |     study before the end of the   |
|                                   |     course                        |
|                                   |                                   |
|                                   | For Austudy or PES customers      |
|                                   | undertaking between 25% and 65%   |
|                                   | of the normal full-time study     |
|                                   | load, determine eligibility       |
|                                   | before returning to this process: |
|                                   |                                   |
|                                   | -   Assessing eligibility for 25% |
|                                   |     concessional study load for   |
|                                   |     Austudy                       |
|                                   | -   Assessing eligibility for the |
|                                   |     25% concessional study load   |
|                                   |     for PES                       |
|                                   |                                   |
|                                   | See Step 3 in Table 2.            |
|                                   |                                   |
|                                   | Study load is less than 25%:      |
|                                   |                                   |
|                                   | Customers undertaking less than   |
|                                   | 25% of a normal full-time study   |
|                                   | load no longer qualify for        |
|                                   | payments as a student. The        |
|                                   | customer may test their           |
|                                   | eligibility for another income    |
|                                   | support payment, for:             |
|                                   |                                   |
|                                   | -   YA and Austudy, see           |
|                                   |     Cancellation of Austudy or YA |
|                                   |     students and Australian       |
|                                   |     Apprentices                   |
|                                   | -   PES, see Cancellation of PES, |
|                                   |     ABSTUDY PES and Assistance    |
|                                   |     for Isolated Children (AIC)   |
|                                   |     PES payments                  |
|                                   |                                   |
|                                   | Procedure ends here.              |
