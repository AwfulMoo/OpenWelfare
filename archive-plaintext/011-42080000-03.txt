This page has information about calculating and processing the
Diagnostic Imaging Multiple Services Rule (DIMSR).

Multiple diagnostic imaging services rules processing

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | More than one R-type and/or       |
|                                   | NR-type item only + Read more ... |
|                                   |                                   |
|                                   | The system applies Rule A of the  |
|                                   | DIMSR when more than 1 diagnostic |
|                                   | imaging item is claimed by the:   |
|                                   |                                   |
|                                   | -   same health professional      |
|                                   | -   to the same patient           |
|                                   | -   on the same day               |
|                                   |                                   |
|                                   | The system automatically          |
|                                   | calculates the benefit payable.   |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | A health professional has claimed |
|                                   | items 57715, 57521 and 57712 for  |
|                                   | the same date of service.         |
|                                   |                                   |
|                                   | -   Medicare benefit for item     |
|                                   |     57715 (the diagnostic imaging |
|                                   |     item with the highest         |
|                                   |     schedule fee) has been        |
|                                   |     calculated using the full     |
|                                   |     schedule fee                  |
|                                   | -   the schedule fee for each     |
|                                   |     remaining diagnostic imaging  |
|                                   |     item (57521 and 57712) has    |
|                                   |     been reduced by $5.00         |
|                                   | -   Medicare benefit for items    |
|                                   |     57521 and 57712 have been     |
|                                   |     calculated using the reduced  |
|                                   |     schedule fee                  |
|                                   |                                   |
|                                   | When the payment process is       |
|                                   | completed, the system             |
|                                   | automatically inserts reason code |
|                                   | 154 (DIMSR applied to service)    |
|                                   | against each item that has had    |
|                                   | its schedule fee reduced.         |
|                                   |                                   |
|                                   | The reason code(s) and the        |
|                                   | reduced schedule fee(s) is:       |
|                                   |                                   |
|                                   | -   added to the patient’s claims |
|                                   |     history                       |
|                                   | -   printed on the Statement of   |
|                                   |     Benefit                       |
