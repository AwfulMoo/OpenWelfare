Adjustments claims for an original split claim

When an adjustment is received for a split claim, it is possible not all
of the service lines to be adjusted are contained in the same part of
the split claim.

The adjustment must be divided into two or more adjustments so the
service lines correspond with the original claim.

Note: the system treats each of these claims as a separate adjustment,
the DHSI control line which was entered to initiate processing is unique
(each has a different claim ID suffix)
