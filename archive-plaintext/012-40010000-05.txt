Pay Group Link Quick Reference

This table describes who can sign as the Requested payee

+-----------------------------------+-----------------------------------+
| Business type                     | Requested payee signatory         |
+-----------------------------------+-----------------------------------+
| Associateship                     | Authorised signatory for          |
|                                   | associateship                     |
+-----------------------------------+-----------------------------------+
| Company                           | Authorised signatory for company  |
+-----------------------------------+-----------------------------------+
| Government                        | Authorised signatory for          |
|                                   | government                        |
+-----------------------------------+-----------------------------------+
| Hospital                          | Authorised signatory for hospital |
+-----------------------------------+-----------------------------------+
| Indigenous                        | Authorised signatory for clinic   |
+-----------------------------------+-----------------------------------+
| Joint venture                     | Authorised signatory for joint    |
|                                   | venture                           |
+-----------------------------------+-----------------------------------+
| Natural Person                    | N/A - provider signs under        |
|                                   | declaration                       |
+-----------------------------------+-----------------------------------+
| Partnership                       | Authorised signatory for          |
|                                   | partnership                       |
+-----------------------------------+-----------------------------------+
| Sole trader                       | N/A - provider signs under        |
|                                   | declaration                       |
+-----------------------------------+-----------------------------------+

The provider may sign as the requested payee where they are an
authorised signatory.
