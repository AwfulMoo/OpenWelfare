28 day notification rule

This table describes the 28 day notification rule for International
Services (CIS) customers.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | A customer notifies Services      |
|                                   | Australia of an event + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Has the customer notified within  |
|                                   | 14 days of the event?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Bereavement event or CIS          |
|                                   | customers + Read more ...         |
|                                   |                                   |
|                                   | Is the customer notifying of a    |
|                                   | bereavement event or are they CIS |
|                                   | customers and notified within 28  |
|                                   | days?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Apply the date of effect rules    |
|                                   | for customers who have notified   |
|                                   | of an event within the            |
|                                   | notification period + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | See the References page for more  |
|                                   | information on the date of effect |
|                                   | rules.                            |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Apply the date of effect rules    |
|                                   | for customers who have not        |
|                                   | notified of an event within the   |
|                                   | notification period + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | See the References page for more  |
|                                   | information on the date of effect |
|                                   | rules.                            |
+-----------------------------------+-----------------------------------+
