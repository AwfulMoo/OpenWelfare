Accept or revoke a third party authority

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check MCRS + Read more ...        |
|                                   |                                   |
|                                   | View documents and case notes to  |
|                                   | determine if there is an existing |
|                                   | authorised third party. To        |
|                                   | release information to a third    |
|                                   | party, a completed MO021 form     |
|                                   | must be provided by the injured   |
|                                   | person or claimant.               |
|                                   |                                   |
|                                   | Is there an existing valid Third  |
|                                   | party authority already on file?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 5              |
|                                   |                                   |
|                                   | Note: all existing third party    |
|                                   | authorities are valid even after  |
|                                   | the claim is closed, unless       |
|                                   | otherwise revoked verbally or in  |
|                                   | writing by the injured person,    |
|                                   | claimant or authorised third      |
|                                   | party, or unless the claimant has |
|                                   | passed away.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Revoking an existing third party  |
|                                   | authority + Read more ...         |
|                                   |                                   |
|                                   | Is the existing third party       |
|                                   | authority form being revoked      |
|                                   | because:                          |
|                                   |                                   |
|                                   | the injured person, claimant has  |
|                                   | advised the agency that they:     |
|                                   |                                   |
|                                   | -   no longer want the existing   |
|                                   |     third party, and/or           |
|                                   | -   want to authorise a different |
|                                   |     third party                   |
|                                   |                                   |
|                                   | or                                |
|                                   |                                   |
|                                   | -   the authorised third party    |
|                                   |     has advised that they no      |
|                                   |     longer represent the injured  |
|                                   |     person or claimant            |
|                                   |                                   |
|                                   | or                                |
|                                   |                                   |
|                                   | -   the injured person has passed |
|                                   |     away                          |
|                                   |                                   |
|                                   | Do any of the above apply?        |
|                                   |                                   |
|                                   | -   Yes, see Search for or update |
|                                   |     an existing claim or register |
|                                   |     a new claim for process steps |
|                                   |     on updating claimant details  |
|                                   | -   No, a TPA form has been       |
|                                   |     received, go to Step 3        |
+-----------------------------------+-----------------------------------+
| 3                                 | Changes to the existing Third     |
|                                   | party authority + Read more ...   |
|                                   |                                   |
|                                   | Are there any changes to the      |
|                                   | existing authorised third party   |
|                                   | (change of solicitor or change of |
|                                   | solicitor address)?               |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   attach the document to the    |
|                                   |     claim, see Search for or      |
|                                   |     update an existing claim or   |
|                                   |     register a new claim          |
|                                   | -   procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Valid Third party authority form  |
|                                   | (MO021) + Read more ...           |
|                                   |                                   |
|                                   | Has all the mandatory information |
|                                   | been included in the MO021 form?  |
|                                   |                                   |
|                                   | Yes, see Search for or update an  |
|                                   | existing claim or register a new  |
|                                   | claim for process steps on        |
|                                   | updating claimant details         |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   complete except for claimant  |
|                                   |     signature, seek verbal        |
|                                   |     consent, go to Step 5         |
|                                   | -   incomplete, go to Step 6      |
+-----------------------------------+-----------------------------------+
| 5                                 | Verbal consent for an unsigned    |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | Note: the form must have already  |
|                                   | been signed by the third party to |
|                                   | be authorised.                    |
|                                   |                                   |
|                                   | Has the injured person or         |
|                                   | claimant provided verbal consent  |
|                                   | for the otherwise complete and    |
|                                   | current MO021 form?               |
|                                   |                                   |
|                                   | -   Yes, see Search for or update |
|                                   |     an existing claim or register |
|                                   |     a new claim for process steps |
|                                   |     on updating claimant details  |
|                                   | -   No, the authority is invalid, |
|                                   |     go to Step 8                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Date of injury or illness + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Confirm if the third party        |
|                                   | authority form is for this claim. |
|                                   |                                   |
|                                   | Is the date of injury correct?    |
|                                   |                                   |
|                                   | Yes, go to Step 7                 |
|                                   |                                   |
|                                   | No, date is missing, or a         |
|                                   | different date to what is         |
|                                   | recorded on this compensation     |
|                                   | claim                             |
|                                   |                                   |
|                                   | -   contact the injured person or |
|                                   |     claimant, and                 |
|                                   | -   verbally get the correct date |
|                                   |     of injury or illness over the |
|                                   |     phone                         |
|                                   | -   if contact is successful and  |

|                                   |     confirmed, go to Step 7       |

|                                   |     go to Step 8                  |
|                                   |                                   |

|                                   | a different claim for the same    |
|                                   | person. If so, get correct claim  |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
| 7                                 | Missing information + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is there other missing            |
|                                   | information?                      |
|                                   |                                   |
|                                   | -   Yes, form is invalid, go to   |
|                                   |     Step 8                        |
|                                   | -   No, form is valid, see Search |

|                                   |     authorised third party        |
|                                   |     details                       |

|                                   | Read more ...                     |
