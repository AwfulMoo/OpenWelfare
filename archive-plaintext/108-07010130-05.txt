+-----------------------------------+-----------------------------------+
| 1                                 | Work Bonus calculations from 1    |
|                                   | July 2019                         |
|                                   |                                   |
|                                   | The legislated calculation is     |
|                                   | dependent on the amount of        |
|                                   | eligible income earned and is     |
|                                   | determined as follows (for a 14   |
|                                   | day entitlement period):          |
|                                   |                                   |
|                                   | Step 1: $300 Work Bonus is given  |
|                                   |                                   |
|                                   | Step 2:                           |
|                                   |                                   |
|                                   | If eligible income is greater     |
|                                   | than or equal to $300, it is      |
|                                   | reduced by $300                   |
|                                   |                                   |
|                                   | -   if balance is greater than or |
|                                   |     equal to remaining eligible   |
|                                   |     income, eligible income is    |
|                                   |     reduced to nil and any unused |
|                                   |     Work Bonus is banked          |
|                                   | -   if balance is greater than    |
|                                   |     nil but less than remaining   |
|                                   |     eligible income, eligible     |
|                                   |     income is reduced by an       |
|                                   |     amount equal to the balance   |
|                                   |                                   |
|                                   | If eligible income is less than   |
|                                   | $300, it is reduced to nil        |
|                                   |                                   |
|                                   | -   any unused balance is banked  |
|                                   |                                   |
|                                   | If there is no eligible income    |
|                                   |                                   |
|                                   | -   $300 is banked                |
|                                   |                                   |
|                                   | However, it is simpler when       |
|                                   | expressed as:                     |
|                                   |                                   |
|                                   | -   available balance for the     |
|                                   |     latest period = Work Bonus    |
|                                   |     for the period plus previous  |
|                                   |     balance                       |
|                                   | -   assessable eligible income =  |
|                                   |     eligible income less          |
|                                   |     available Work Bonus balance  |
|                                   | -   new balance (not less than    |
|                                   |     nil or more than $7,800) =    |
|                                   |     available balance less        |
|                                   |     eligible income (not less     |
|                                   |     than nil or more than $7,800) |
|                                   |                                   |
|                                   | This is also how the Employment   |
|                                   | Income Assessment (EIA) screen    |
|                                   | will display.                     |

| 5                                 | Both customer and partner have    |
|                                   | employment income and entitled to |
|                                   | Work Bonus from 1 July 2019       |
|                                   |                                   |
|                                   | Scenario: both members of a       |
|                                   | couple had Work Bonus eligible    |
|                                   | income in the last fortnight. Lee |
|                                   | has earned $400 from employment.  |
|                                   | Sam has earned $180 from active   |
|                                   | participation in self-employment  |
|                                   | that met the personal exertion    |
|                                   | test. Both are entitled to the    |
|                                   | Work Bonus.                       |
|                                   |                                   |
|                                   | Sam's Work Bonus calculation:     |
|                                   |                                   |
|                                   | -   $180 - $300 = - $120          |
|                                   | -   The assessable                |
|                                   |     self-employment income amount |
|                                   |     will be $0. $120 is banked to |
|                                   |     Sam's Work Bonus balance      |
|                                   |                                   |
|                                   | Lee's Work Bonus calculation:     |
|                                   |                                   |
|                                   | -   $400 - $300 = $100            |
|                                   | -   If Lee has no current Work    |
|                                   |     Bonus balance, Lee's          |
|                                   |     employment income assessment  |
|                                   |     amount would be $100          |
|                                   |                                   |
|                                   | Combined eligible income          |
|                                   | assessment amount for Sam and Lee |
|                                   | for the fortnight is $100.        |

|                                   | Current Zero Rate (CZR) from 1    |
|                                   | July 2019                         |
|                                   |                                   |
|                                   | Work Bonus balance prior to       |
|                                   | period 1: $1,200                  |
|                                   |                                   |
|                                   | Work Bonus period 1: $300         |
|                                   |                                   |
|                                   | Available balance period 1:       |
|                                   | $1,500                            |
|                                   |                                   |
|                                   | Eligible income in period 1:      |
|                                   | $3,500                            |
|                                   |                                   |
|                                   | $3,500 minus $1,500 = $2,000      |
|                                   | assessable income                 |
|                                   |                                   |
|                                   | Balance at end of period one      |
|                                   | should be nil, but due to         |
|                                   | $2,000eligible income plus $1,000 |

|                                   | Balance prior to period 2: $1,200 |
|                                   |                                   |
|                                   | Work Bonus period 2: $300         |

|                                   |                                   |
|                                   | Nil assessable income             |
|                                   |                                   |

| 7                                 | One member of couple paid under   |
|                                   | transitional rules                |
|                                   |                                   |
|                                   | If the customer's pension is paid |
|                                   | under the transitional rules the  |
|                                   | Work Bonus is not used to         |
|                                   | determine their rate entitlement  |
|                                   | under the pre 20 September 2009   |
|                                   | income test but may still be used |
|                                   | to calculate their partner's      |
|                                   | income support payment            |
|                                   | entitlement as long as the        |
|                                   | partner is not also paid under    |
|                                   | the transitional rules.           |
|                                   |                                   |
|                                   | Note: the Work Bonus is used to   |
|                                   | determine if the customer is      |
|                                   | entitled to a higher rate of      |
|                                   | payment under the transitional    |
|                                   | rules or the post 20 September    |
|                                   | 2009 income test. See             |
|                                   | Transitional rules for pension    |
|                                   | customers who were on payment at  |
|                                   | 19 September 2009.                |
|                                   |                                   |
|                                   | When both members of the couple   |
|                                   | are entitled to the Work Bonus,   |
|                                   | as all income is combined and     |
|                                   | then halved (except for any       |
|                                   | direct deduction income amount),  |
|                                   | both parties are likely to be on  |
|                                   | a rate subject to the             |
|                                   | transitional rules.               |
|                                   |                                   |
|                                   | If only one member is entitled to |
|                                   | the Work Bonus and the other is   |
|                                   | not, for example, one is on Age   |
|                                   | Pension under the transitional    |
|                                   | rules and the other is on         |
|                                   | JobSeeker Payment (JSP), the Work |
|                                   | Bonus may still be used when      |
|                                   | calculating the rate entitled for |
|                                   | the customer on JSP. See Item 8.  |
+-----------------------------------+-----------------------------------+
| 8                                 | One member of a couple entitled   |
|                                   | to Work Bonus, other on payment   |
|                                   | not entitled to Work Bonus from 1 |
|                                   | July 2019                         |
|                                   |                                   |
|                                   | When the partner of a couple is   |
|                                   | on a payment not entitled to the  |
|                                   | Work Bonus, they may still get    |
|                                   | the benefit of the Work Bonus     |
|                                   | reduction that their partner is   |
|                                   | entitled to use, as all income is |
|                                   | combined and then halved (except  |
|                                   | for direct deductions) before the |
|                                   | income test is applied.           |
|                                   |                                   |
|                                   | This also applies to a partner of |
|                                   | a customer who is not entitled to |
|                                   | the Work Bonus due to payment     |
|                                   | under the transitional rules. In  |
|                                   | this case, although the customer  |
|                                   | is not entitled to use the Work   |
|                                   | Bonus their partner may still     |
|                                   | benefit from the Work Bonus.      |
|                                   |                                   |
|                                   | Scenario: Lou is on Age Pension   |
|                                   | and paid under the transitional   |
|                                   | rules, therefore not entitled to  |
|                                   | use the Work Bonus. Chris is on   |
|                                   | JobSeeker Payment (JSP). Lou has  |
|                                   | self-employment income of $500    |
|                                   | for the fortnight.                |
|                                   |                                   |
|                                   | JSP customers have no entitlement |
|                                   | to the Work Bonus for their own   |
|                                   | employment income but Lou's       |
|                                   | self-employment income assessment |
|                                   | amount will come under the post   |
|                                   | 20 September 2009 income test     |
|                                   | rules when calculating Chris's    |
|                                   | assessable income against the JSP |
|                                   | entitlement. This applies even    |
|                                   | though Lou is on payment subject  |
|                                   | to the transitional rules and is  |
|                                   | not entitled to use the Work      |
|                                   | Bonus for the Age Pension rate    |
|                                   | under the transitional rules.     |
|                                   |                                   |
|                                   | In this scenario, although Lou    |
|                                   | does not have access to the Work  |
|                                   | Bonus, Chris can still benefit    |
|                                   | from its use when their combined  |
|                                   | income is used to determine the   |
|                                   | amount of Lou's self-employment   |
|                                   | income that is assessable.        |
|                                   |                                   |
|                                   | Lou's self-employment income is   |
|                                   | above the Work Bonus fortnightly  |
|                                   | amount, therefore $500 - $300 =   |
|                                   | $200. In this case Lou does not   |
|                                   | have access to a Work Bonus       |
|                                   | balance.                          |
|                                   |                                   |
|                                   | When all of the couple's income   |
|                                   | is combined and then halved       |
|                                   | before applying the income test   |
|                                   | only the reduced amount of Lou's  |
|                                   | self-employment income ($200) is  |
|                                   | used to determine the rate of     |
|                                   | JSP.                              |
|                                   |                                   |
|                                   | If the customer is partnered and  |
|                                   | the partner is not entitled to    |
|                                   | the Work Bonus, only the          |
|                                   | customer's reduction of eligible  |
|                                   | income can be used before         |
|                                   | applying the income test. It      |
|                                   | cannot be used for the partner's  |
|                                   | own income.                       |

| 9                                 | Pensioner with varying income     |
|                                   | from 1 July 2019                  |
|                                   |                                   |
|                                   | Melissa is 69 and receives Carer  |
|                                   | Payment (CP) due to caring for    |
|                                   | their partner Paul who receives   |
|                                   | Disability Support Pension (DSP). |
|                                   | As at 1 July 2019, Melissa has a  |
|                                   | current Work Bonus balance of     |
|                                   | $350.                             |
|                                   |                                   |
|                                   | In the first fortnight after      |
|                                   | this, Melissa earns $200.         |
|                                   | Centrelink disregards all of it   |
|                                   | as it is less than $300, and      |
|                                   | because there is $100 of unused   |
|                                   | Work Bonus ($300 - $200) the      |
|                                   | balance is increased to $450      |
|                                   | ($350 + $100).                    |
|                                   |                                   |
|                                   | In the second fortnight, Melissa  |
|                                   | earns $550. Centrelink first      |
|                                   | reduces this to $250 by deducting |
|                                   | that fortnight's Work Bonus       |
|                                   | ($300). Centrelink then further   |
|                                   | discounts the eligible income,    |
|                                   | this time to nil, by reducing the |
|                                   | Work Bonus balance to $200 ($450  |
|                                   | less $250).                       |
|                                   |                                   |
|                                   | In the third fortnight, Melissa   |
|                                   | earns $600. Centrelink reduces    |
|                                   | this to $300 by deducting that    |
|                                   | fortnight's Work Bonus.           |
|                                   | Centrelink then further discounts |
|                                   | the eligible income, this time to |
|                                   | $100 ($300 - $200) which reduces  |
|                                   | the Work Bonus balance to nil.    |

| 10                                | Pensioner who works sporadically  |
|                                   | from 1 July 2019                  |
|                                   |                                   |
|                                   | Joe is on Age Pension and has not |
|                                   | worked since December 2010.       |
|                                   |                                   |
|                                   | From 1 July 2011 Joe's Work Bonus |
|                                   | balance increased by $250 each    |
|                                   | fortnight and then increased by   |
|                                   | $300 each fortnight from 1 July   |
|                                   | 2019. By 3 November 2019 the Work |
|                                   | Bonus balance is $7,800.          |
|                                   |                                   |
|                                   | Joe then starts work earning $700 |
|                                   | per fortnight. Each fortnight,    |
|                                   | the Work Bonus reduces this       |
|                                   | eligible income by $300 to $400.  |
|                                   | Then Joe's Work Bonus balance is  |
|                                   | reduced by the remaining $400 per |
|                                   | fortnight so that no income is    |
|                                   | assessed.                         |
|                                   |                                   |
|                                   | Joe's opening balance was such    |
|                                   | that it would be at least 19      |
|                                   | fortnights ($7,800 / $400) before |
|                                   | the balance is fully depleted and |
|                                   | any eligible income is assessed.  |
|                                   | Once the work stops, the balance  |
|                                   | starts to increase again.         |

|                                   |                                   |

|                                   | employment income since that      |
|                                   | time.                             |
|                                   |                                   |

|                                   | self-employment income is         |
