Enquiries about age qualifications for LIC or FST

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Child under 16 + Read more ...    |
|                                   |                                   |
|                                   | Is the child under 16 years of    |
|                                   | age?                              |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Child in the care of an           |
|                                   | organisation + Read more ...      |
|                                   |                                   |
|                                   | Is the child in the care of an    |
|                                   | organisation (including Approved  |
|                                   | Care Organisations (ACOs) and     |
|                                   | State or Territory Government     |
|                                   | departments)?                     |
|                                   |                                   |
|                                   | -   Yes, the organisation will    |
|                                   |     need to lodge a Low Income    |
|                                   |     Health Care Card (LIC) claim  |
|                                   |     for the child. A Claim for a  |
|                                   |     Health Care Card (SS050)      |
|                                   |     needs to be completed in the  |
|                                   |     name of the child and         |
|                                   |     Identity Confirmation         |
|                                   |     provided for the child. A     |
|                                   |     person (PER) record will need |
|                                   |     to be created for the child.  |
|                                   |     See Searching for a customer  |
|                                   |     on the system                 |
|                                   |                                   |
|                                   | For more information on the claim |
|                                   | process, documents required and   |
|                                   | coding, see Claiming Low Income   |
|                                   | Health Care Card (LIC) and        |
|                                   | Processing Low Income Health Care |
|                                   | Card (LIC) claims                 |
|                                   |                                   |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Foster care child + Read more ... |
|                                   |                                   |
|                                   | Is the child a foster care child? |
|                                   |                                   |
|                                   | Yes, the foster carer can []      |
|                                   | claim online or will need to      |
|                                   | complete:                         |
|                                   |                                   |
|                                   | -   a customised Claim for Foster |
|                                   |     Child Health Care Card        |
|                                   |     (SS489). This is issued by    |
|                                   |     running the First Contact     |
|                                   |     Service Offer (FCSO)          |
|                                   |     workflow, or                  |
|                                   | -   complete a Claim for a Health |
|                                   |     Care Card (SS050) on behalf   |
|                                   |     of the child. See Health Care |
|                                   |     Cards (HCC) for foster        |
|                                   |     children                      |
|                                   |                                   |
|                                   | No, go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Check if customer was on Carer    |
|                                   | Allowance (CA) prior to their     |
|                                   | 16^(th) birthday + Read more ...  |
|                                   |                                   |
|                                   | Was the customer a CA care        |
|                                   | recipient the day prior to their  |
|                                   | 16th birthday?                    |
|                                   |                                   |
|                                   | -   Yes, see Ex-Carer Allowance   |
|                                   |     (child) Health Care Card      |
|                                   |     (EHC). Procedure ends here    |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer undertaking secondary    |
|                                   | studies + Read more ...           |
|                                   |                                   |
|                                   | Is the child under 19 years of    |
|                                   | age and undertaking secondary     |
|                                   | studies?                          |
|                                   |                                   |
|                                   | -   Yes, the child is not         |
|                                   |     entitled to a LIC unless they |
|                                   |     are receiving prescribed      |
|                                   |     education scheme payment, or  |
|                                   |     if they are not a FTB child   |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Advise the customer of Low Income |
|                                   | Health Care Card (LIC)            |
|                                   | qualifications + Read more ...    |
|                                   |                                   |
|                                   | Qualifications include that the   |
|                                   | customer must:                    |
|                                   |                                   |
|                                   | -   be in Australia in order to   |
|                                   |     lodge a valid claim and be    |
|                                   |     either                        |
|                                   | -   an Australian resident, or    |
|                                   | -   the holder of a non-protected |
|                                   |     Special Category Visa (SCV)   |
|                                   |     who is residing in Australia, |
|                                   |     or                            |
|                                   | -   a child living in Australia   |
|                                   |     with an Australian resident   |
|                                   |     or SCV holder who is residing |
|                                   |     in Australia, or              |
|                                   | -   the certain temporary visas   |
|                                   |     that qualify customers for    |
|                                   |     some payments under section   |
|                                   |     729(2)(f)(v)                  |
|                                   |                                   |
|                                   | and                               |
|                                   |                                   |
|                                   | -   either be in Australia or     |
|                                   |     have been temporarily outside |
|                                   |     Australia for less than 13    |
|                                   |     weeks                         |
|                                   |                                   |
|                                   | and                               |
|                                   |                                   |
|                                   | -   meet the LIC income test      |
|                                   |     under section 1071A of the    |
|                                   |     Social Security Act 1991. See |
|                                   |     the Resources page for a link |
|                                   |     to the LIC income test and    |
|                                   |     further information regarding |
|                                   |     income limits                 |
|                                   |                                   |
|                                   | and                               |
|                                   |                                   |
|                                   | -   have served the newly arrived |
|                                   |     resident's waiting period     |
|                                   |     (NARWP), where applicable.    |
|                                   |     See Residence assessment for  |
|                                   |     adult customers claiming a    |
|                                   |     Low Income Health Care Card   |
|                                   |     (LIC)                         |
|                                   |                                   |
|                                   | Dependants must meet certain age  |
|                                   | qualifications to be included on  |
|                                   | their parent's LIC.               |
|                                   |                                   |
|                                   | Note: a young person who is under |
|                                   | 19, is an FTB child and is a      |
|                                   | secondary student, is not         |
|                                   | qualified for a LIC of their own. |
|                                   | They may be included on their     |
|                                   | parent's card instead.            |
|                                   |                                   |
|                                   | See the References page to        |
|                                   | determine which dependants are    |
|                                   | not qualified for a Health Care   |
|                                   | Card (HCC).                       |
+-----------------------------------+-----------------------------------+
| 7                                 | Invite the customer to claim +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For more information on the claim |
|                                   | process, documents required and   |
|                                   | coding, see Claiming Low Income   |
|                                   | Health Care Card (LIC) and        |
|                                   | Processing Low Income Health Care |
|                                   | Card (LIC) claims.                |
