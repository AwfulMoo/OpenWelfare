|                                   | Contact Interpreter Connect       |

|                                   | Workspace:                        |
|                                   |                                   |

|                                   | -   ‘Interpreter Services’ will   |

|                                   |                                   |

|                                   | -   enter the 4 digit Office ID   |
|                                   |     and press #                   |
|                                   | -   enter the 3 digit Language    |
|                                   |     code corresponding to the     |
|                                   |     language required and press # |

|                                   |     selected. Press 1 if correct. |
|                                   |     Press 2 if incorrect to       |
|                                   |     return to the previous        |
|                                   |     selection                     |
|                                   |                                   |
|                                   | Enter the required duration of    |
|                                   | the call:                         |
|                                   |                                   |
|                                   | -   for up to 15 minutes, press 1 |

|                                   | -   for up to 45 minutes, press 3 |
|                                   | -   for more than 45 minutes,     |
|                                   |     press 4                       |
|                                   |                                   |
|                                   | Wait while the system connects to |
|                                   | an interpreter                    |
|                                   |                                   |
|                                   | If the system:                    |
|                                   |                                   |
|                                   | -   cannot connect to an          |
|                                   |     interpreter, do not hang up,  |
|                                   |     go to Step 6                  |
|                                   | -   connects to an interpreter,   |
|                                   |     go to Step 7                  |

|                                   | -   If the language does not      |

|                                   |     select Yes in the Interpreter |

|                                   |     Contact Detail the customer's |
|                                   |     preferred language            |

| 6                                 | Interpreter cannot be found +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the system cannot connect to   |
|                                   | an interpreter, the call          |
|                                   | transfers to a Language Services  |
|                                   | Booking Officer. The call will be |
|                                   | placed on hold, while the Booking |
|                                   | Officer manually attempts to      |
|                                   | locate an interpreter.            |
|                                   |                                   |
|                                   | If a Booking Officer locates an   |
|                                   | interpreter, go to Step 5         |
|                                   |                                   |
|                                   | If the Booking Officer cannot     |
|                                   | locate an interpreter, they may:  |
|                                   |                                   |
|                                   | -   ask if the customer speaks    |
|                                   |     another language              |
|                                   | -   advise to call back at a      |
|                                   |     later time, or                |
|                                   | -   request an interpreter        |
|                                   |     booking for a future          |
|                                   |     appointment. See Booking an   |
|                                   |     interpreter for an            |
|                                   |     appointment.                  |
|                                   |                                   |
|                                   | Procedure ends here.              |
