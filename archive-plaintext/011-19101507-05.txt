Common enquiries quick reference table

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: How many treatment      |
|                                   | services can a patient access     |
|                                   | under their plan?                 |
|                                   |                                   |
|                                   | Answer: An initial referral can   |
|                                   | only be made for up to:           |
|                                   |                                   |
|                                   | -   10 eating disorder            |
|                                   |     psychological treatment       |
|                                   |     services                      |
|                                   | -   20 eating disorder dietetic   |
|                                   |     services (items 10954,        |
|                                   |     82350-82351, 93074 and 93108) |
|                                   |                                   |
|                                   | Reviews of the treatment and      |
|                                   | management plan must be done      |
|                                   | every 10 psychological treatment  |
|                                   | services before the patient can   |
|                                   | access the next 10 services. A    |
|                                   | new referral is also required for |
|                                   | every 10 eating disorder          |
|                                   | psychological treatment services. |
|                                   | See also on this page the         |
|                                   | infographic that outlines the     |
|                                   | review and referral process.      |
|                                   |                                   |
|                                   | Patients can access up to the     |
|                                   | following amounts if reviews are  |
|                                   | undertaken:                       |
|                                   |                                   |
|                                   | -   40 eating disorder            |
|                                   |     psychological treatment       |
|                                   |     services, and                 |
|                                   | -   20 eating disorder dietetics  |
|                                   |     services                      |
|                                   |                                   |
|                                   | See Question 4 below for more     |
|                                   | information about the review      |
|                                   | process.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Do patients need review |
|                                   | items on their history before     |
|                                   | they can access more than 10      |
|                                   | psychological treatment services? |
|                                   |                                   |
|                                   | Note: for internal use only       |
|                                   |                                   |
|                                   | Answer: No, patients do not need  |
|                                   | to have review items 90264-90269  |
|                                   | on their Medicare claims history  |
|                                   | to access more than 10 treatment  |
|                                   | services. It is a requirement     |
|                                   | that health professionals         |
|                                   | undertake a review of an eating   |
|                                   | disorder treatment and management |
|                                   | plan before the patient is        |
|                                   | referred for more than 10         |
|                                   | psychological treatment services. |
|                                   |                                   |
|                                   | See also the link below           |
|                                   | Infographic – treatment and       |
|                                   | management plan review and        |
|                                   | renewal process.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Do patients need a      |
|                                   | referral to receive services      |
|                                   | under an eating disorder          |
|                                   | treatment and management plan?    |
|                                   |                                   |
|                                   | Answer: Yes, patients need a      |
|                                   | referral for a consultant         |
|                                   | psychiatrist or consultant        |
|                                   | paediatrician to create a plan    |
|                                   | and a referral for allied health  |
|                                   | treatment services recommended in |
|                                   | their treatment and management    |
|                                   | plan.                             |
|                                   |                                   |
|                                   | Referrals for allied health       |
|                                   | treatment services can be made    |
|                                   | for up to:                        |
|                                   |                                   |
|                                   | -   10 eating disorder            |
|                                   |     psychological treatment       |
|                                   |     services                      |
|                                   | -   20 eating disorder dietetics  |
|                                   |     services                      |
|                                   |                                   |
|                                   | Psychological treatment services  |
|                                   | provided under a Mental Health    |
|                                   | Treatment plan or Chronic Disease |
|                                   | Management plan may have          |
|                                   | different referral requirements.  |
|                                   |                                   |
|                                   | See also:                         |
|                                   |                                   |
|                                   | -   Infographic - Eligibility for |
|                                   |     treatment services            |
|                                   | -   Referrals for assessing in    |
|                                   |     Medicare.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Can patients have an    |
|                                   | eating disorder treatment and     |
|                                   | management plan and a GP mental   |
|                                   | health treatment plan?            |
|                                   |                                   |
|                                   | Answer: Patients should generally |
|                                   | only have 1 treatment and         |
|                                   | management plan in place at any   |
|                                   | given time.                       |
|                                   |                                   |
|                                   | There are no restrictions on      |
|                                   | having a GP mental health         |
|                                   | treatment plan item and an eating |
|                                   | disorder treatment and management |
|                                   | plan item for the same period.    |
|                                   |                                   |
|                                   | Health professionals may be       |
|                                   | subject to post-payment audits by |
|                                   | the Department of Health and      |
|                                   | should ensure they are claiming   |
|                                   | Medicare benefits appropriately.  |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Can patients have an    |
|                                   | eating disorder treatment and     |
|                                   | management plan and a chronic     |
|                                   | disease management plan?          |
|                                   |                                   |
|                                   | Answer: Yes, provided that the    |
|                                   | chronic disease management plan   |
|                                   | is clinically relevant for the    |
|                                   | patient.                          |
|                                   |                                   |
|                                   | Health professionals may be       |
|                                   | subject to post-payment audits by |
|                                   | the Department of Health and      |
|                                   | should ensure they are claiming   |
|                                   | Medicare benefits appropriately.  |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Do patients have to     |
|                                   | have a new treatment and          |
|                                   | management plan created after 12  |
|                                   | months?                           |
|                                   |                                   |
|                                   | Answer: Yes, eating disorder      |
|                                   | treatment and management plans    |
|                                   | expire after 12 months. Patients  |
|                                   | must have a valid treatment and   |
|                                   | management plan in place to       |
|                                   | continue to access eating         |
|                                   | disorder psychological and        |
|                                   | dietetics treatment services.     |
+-----------------------------------+-----------------------------------+
