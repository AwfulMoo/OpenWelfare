|                                   | grandparent of the child          |

|                                   | Nick and Bonnie are the           |
|                                   | biological parents of their child |
|                                   | Milton. Nick and Bonnie separate  |
|                                   | and Nick marries Marika. Marika   |
|                                   | is Milton’s step parent. In later |
|                                   | years Milton marries Sarah and    |
|                                   | they have a child Nathaniel.      |
|                                   |                                   |
|                                   | Marika is the step grandparent of |
|                                   | Nathaniel.                        |
|                                   |                                   |
|                                   | Milton and Sarah eventually       |
|                                   | separate and Milton marries       |
|                                   | Helga.                            |
|                                   |                                   |
|                                   | Helga is Nathaniel’s step parent  |
|                                   | and Helga’s parents are           |
|                                   | Nathaniel’s step grandparents.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Natural, adoptive or step         |
|                                   | great-grandparent of the child    |
|                                   |                                   |
|                                   | Bill and Ping are the biological  |
|                                   | parents of Jessica. Jessica and   |
|                                   | partner Bjorn are the biological  |
|                                   | parents of Theo. Bill and Ping    |
|                                   | are the natural grandparents of   |
|                                   | Theo. In later years Theo adopts  |
|                                   | a child Phoebe.                   |
|                                   |                                   |
|                                   | Bill and Ping are the adoptive    |
|                                   | great-grandparents of Phoebe.     |

|                                   | relationship parent of the child  |

|                                   |                                   |
|                                   | Neville and Connie are the        |
|                                   | grandparents of Nicolas.          |
+-----------------------------------+-----------------------------------+
| 4                                 | Former partner of natural,        |
|                                   | adoptive or relationship          |
|                                   | grandparent or great-grandparent  |
|                                   |                                   |
|                                   | Riccardo and Jean are the         |
|                                   | biological parents of Ben. Ben is |
|                                   | the biological parent of Alison.  |
|                                   | Riccardo and Jean are the         |
|                                   | grandparents of Alison. Riccardo  |
|                                   | and Jean separate and Jean has a  |
|                                   | new partner Brian. Jean and Brian |
|                                   | are the grandparents of Alison    |
|                                   | (along with Riccardo). In later   |
|                                   | years Jean and Brian separate.    |
|                                   |                                   |
|                                   | Brian is the former partner of    |

| 10                                | Income Support Payment            |
|                                   |                                   |
|                                   | Padma is caring for Padma's       |
|                                   | adopted grandchild Sarita and     |
|                                   | claims CCS. Padma does not        |
|                                   | receive an ACCS (Grandparent)     |
|                                   | eligible income support payment   |
|                                   | and was ineligible for ACCS       |
|                                   | Grandparent. Padma has instead    |
|                                   | been assessed for an exemption    |
|                                   | from the CCS activity test and is |
|                                   | eligible for 100 hours approved   |
|                                   | care.                             |
|                                   |                                   |
|                                   | Padma becomes partnered to Santos |
|                                   | who is receiving a Disability     |
|                                   | Support Payment. As Padma's       |
|                                   | partner is in receipt of an ACCS  |

|                                   | rate cap where the fees charged   |
|                                   | are above the hourly rate cap.    |
