Check Medicare claims history

Table 2: process for helping a customer understand entries in their Care
plan search history screen. This should only be done after completing
the process in Table 1.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complete public security check +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Perform security check to         |
|                                   | authenticate Medicare record.     |
|                                   |                                   |
|                                   | Proceed only if the caller meets  |
|                                   | the security check                |
|                                   |                                   |
|                                   | Advise the caller that the        |
|                                   | information provided is:          |
|                                   |                                   |
|                                   | -   based on the patientâ€™s        |
|                                   |     Medicare claiming history at  |
|                                   |     the time of the enquiry       |
|                                   | -   subject to change if there    |
|                                   |     are claims pending            |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for plan items on the       |
|                                   | patient's history + Read more ... |
|                                   |                                   |
|                                   | Check for GP management plan      |
|                                   | items.                            |
|                                   |                                   |
|                                   | Check for Mental health plan      |
|                                   | items.                            |
|                                   |                                   |
|                                   | The care plan history (used by    |
|                                   | health professionals in HPOS)     |
|                                   | does not support eating disorder  |
|                                   | treatment and management plan     |
|                                   | items.                            |
|                                   |                                   |
|                                   | Are there any services on or      |
|                                   | around the date/s listed on the   |
|                                   | Care plan search history screen?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Provide customer information      |
|                                   | about items claimed + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Provide the customer a brief      |
|                                   | description of the item claimed.  |
|                                   | Advise them it is likely their    |
|                                   | doctor checked their eligibility  |
|                                   | for that service before sending   |
|                                   | the claim to Medicare.            |
|                                   |                                   |
|                                   | -   If the customer understands   |
|                                   |     and has no further concerns,  |
|                                   |     procedure ends here           |
|                                   | -   If the customer is concerned  |
|                                   |     their information was         |
|                                   |     accessed incorrectly, go to   |
|                                   |     Step 5                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Explain care plan history to      |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | Advise the customer that entries  |
|                                   | on the Care plan search history   |
|                                   | screen appear when a health       |
|                                   | professional has checked their    |
|                                   | eligibility for a care plan. This |
|                                   | does not necessarily mean they    |
|                                   | have a care plan in place or that |
|                                   | they have paid for a care plan    |
|                                   | service.                          |
|                                   |                                   |
|                                   | If the customer is concerned      |
|                                   | their information was accessed    |
|                                   | incorrectly, go to Step 5.        |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer concerned about          |
|                                   | inappropriate access + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the customer is concerned      |
|                                   | their information has been        |
|                                   | accessed inappropriately, they    |
|                                   | can report suspected fraud via    |
|                                   | the Services Australia website.   |
|                                   |                                   |
|                                   | See the Resources page for a      |
|                                   | link.                             |
