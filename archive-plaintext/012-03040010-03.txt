Conduct a basic search for an IHI in CDMS

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Inquirer security check + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Before an IHI can be given to an  |
|                                   | individual or third party conduct |
|                                   | a security check.                 |
|                                   |                                   |
|                                   | For:                              |
|                                   |                                   |
|                                   | -   an individual for their own   |
|                                   |     IHI or a parent calling for   |
|                                   |     their childrenâ€™s IHI, see     |
|                                   |     Security checks and release   |
|                                   |     of Individual Healthcare      |
|                                   |     Identifier (IHI) information  |
|                                   | -   an authorised employee, or    |
|                                   |     Healthcare Provider, see      |
|                                   |     Handle call from provider or  |
|                                   |     authorised employee about an  |
|                                   |     Individual Healthcare         |
|                                   |     Identifier (IHI)              |
+-----------------------------------+-----------------------------------+
| 2                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to CDMS                |
|                                   | -   On the Home Page, select      |
|                                   |     Inquire on Consumer Details   |
|                                   | -   The Inquiry Search screen     |
|                                   |     displays defaulting to the    |
|                                   |     Basic Search tab              |
+-----------------------------------+-----------------------------------+
| 3                                 | ID Number Details field + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In the ID Number Details field:   |
|                                   |                                   |
|                                   | -   Select Healthcare Identifiers |
|                                   |     in the Program field          |
|                                   | -   Enter the ID number in ID     |
|                                   |     Number field                  |
|                                   | -   Select the ID number type     |
|                                   |     corresponding to the number   |
|                                   |     keyed in the ID Number Type   |
|                                   |     field                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Display IHI number + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select the Personal button    |
|                                   | -   The IHI displays on the       |
|                                   |     Personal Details screen       |
+-----------------------------------+-----------------------------------+
| 5                                 | Return to Inquiry Search screen + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Search Page to return to   |
|                                   | the Inquiry Search screen.        |
+-----------------------------------+-----------------------------------+
| 6                                 | Provide number + Read more ...    |
|                                   |                                   |
|                                   | Provide IHI to requesting         |
|                                   | individual.                       |

Conduct a detailed search for an IHI in CDMS

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to CDMS                |
|                                   | -   On the Home Page, select      |
|                                   |     Inquire on Consumer Details   |
|                                   | -   The Inquiry Search screen     |
|                                   |     displays defaulting to the    |
|                                   |     Basic Search tab              |
+-----------------------------------+-----------------------------------+
| 2                                 | Detailed Search tab + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Detailed Search tab    |
|                                   | -   The detailed search fields    |
|                                   |     display                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Select All Programs + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select All Programs in the    |
|                                   |     Program field                 |
|                                   |                                   |
|                                   | Note: keep the Exact Name Match   |
|                                   | and Historical Search check boxes |
|                                   | unchecked.                        |
|                                   |                                   |
|                                   | Note: the below fields are        |
|                                   | mandatory:                        |
|                                   |                                   |
|                                   | -   family name                   |
|                                   | -   gender                        |
|                                   | -   either date of birth, age, or |
|                                   |     age range                     |
|                                   |                                   |
|                                   | If too many possible matches are  |
|                                   | returned, change the Program to   |
|                                   | Healthcare Identifiers and search |
|                                   | again.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Attempt to find a match + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For the first attempt to find a   |
|                                   | match, use:                       |
|                                   |                                   |
|                                   | -   Family Name                   |
|                                   | -   Gender, select All            |
|                                   | -   Date of birth                 |
|                                   |                                   |
|                                   | Formatting of a previously        |
|                                   | entered name may make it more     |
|                                   | difficult to find a match. This   |
|                                   | broader search criteria gives the |
|                                   | best chance of finding a possible |
|                                   | match.                            |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Try variations of the         |
|                                   |     surname. For example the      |
|                                   |     spelling or if it is          |
|                                   |     hyphenated, try with and      |
|                                   |     without the hyphen and try    |
|                                   |     each of the hyphenated        |
|                                   |     surnames separately           |
|                                   | -   If there are too many         |
|                                   |     results, refine the search by |
|                                   |     adding the first initial of   |
|                                   |     the customerâ€™s first name     |
+-----------------------------------+-----------------------------------+
| 5                                 | Select Personal + Read more ...   |
|                                   |                                   |
|                                   | When there is:                    |
|                                   |                                   |
|                                   | -   Only 1 matched consumer, the  |
|                                   |     Personal Details Inquiry      |
|                                   |     screen displays the personal  |
|                                   |     details of the individual     |
|                                   | -   More than 1 matched consumer, |
|                                   |     scroll to Matched Consumers   |
|                                   |     (or select Results link) to   |
|                                   |     display a list of consumer    |
|                                   |     matches                       |
|                                   |     Note: when there are more     |
|                                   |     than 10 results found, select |
|                                   |     List 2 link to display the    |
|                                   |     next 10 matches, and so on    |
|                                   |                                   |
|                                   | Ensure the correct consumer is    |
|                                   | selected, confirm using other     |
|                                   | details, such as address, to      |
|                                   | ensure the correct consumer has   |
|                                   | been selected, then:              |
|                                   |                                   |
|                                   | -   Select Personal               |
|                                   | -   The Personal Details Inquiry  |
|                                   |     screen displays the IHI of    |
|                                   |     the individual                |
|                                   |                                   |
|                                   | Note: select Search Page to       |
|                                   | return to the Inquiry Search      |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 6                                 | Provide number + Read more ...    |
|                                   |                                   |
|                                   | Provide IHI to requesting         |
|                                   | individual.                       |
+-----------------------------------+-----------------------------------+

Conduct an address search for an IHI in CDMS

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to CDMS                |
|                                   | -   On the Home Page, select      |
|                                   |     Inquire on Consumer Details   |
|                                   | -   The Inquiry Search screen     |
|                                   |     displays defaulting to the    |
|                                   |     Basic Search tab              |
+-----------------------------------+-----------------------------------+
| 2                                 | Address Search button + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Address Search button. |
|                                   | -   The Address Search screen     |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Select Healthcare Identifiers +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Healthcare Identifiers in  |
|                                   | the Program field                 |
|                                   |                                   |
|                                   | Enter address details. Note: the  |
|                                   | below fields are mandatory when   |
|                                   | the Country field is Australia:   |
|                                   |                                   |
|                                   | -   address line 1                |
|                                   | -   locality                      |
|                                   | -   postcode                      |
|                                   | -   state                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Select Consumers + Read more ...  |
|                                   |                                   |
|                                   | A list of individuals' details at |
|                                   | the bottom of the screen.         |
|                                   |                                   |
|                                   | Note: when the address search is  |
|                                   | unsuccessful, select Continue to  |
|                                   | change or add information and     |
|                                   | repeat Steps 3 and 4.             |
+-----------------------------------+-----------------------------------+
| 5                                 | Select the correct individual +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   the radio button next to the  |
|                                   |     correct individual            |
|                                   | -   Personal                      |
|                                   |                                   |
|                                   | Note: select Search Page to       |
|                                   | return to the Address Search      |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 6                                 | Display IHI number for a Medicare |
|                                   | or DVA eligible individual + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select the Personal button    |
|                                   | -   From the Personal Details     |
|                                   |     screen, select the            |
|                                   |     Entitlement button            |
|                                   | -   The IHI number displays on    |
|                                   |     the IHI Details screen        |
|                                   |                                   |
|                                   | Note: select Search Page to       |
|                                   | return to the Address Search      |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 7                                 | Display IHI number for a person   |
|                                   | NOT eligible for Medicare or      |
|                                   | DVA + Read more ...               |
|                                   |                                   |
|                                   | -   Select the Personal button    |
|                                   | -   The IHI number displays on    |
|                                   |     the Personal Details screen   |
|                                   |                                   |
|                                   | Note: select Search Page to       |
|                                   | return to the Address Search      |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 8                                 | Provide number + Read more ...    |
|                                   |                                   |
|                                   | Provide IHI to requesting         |
|                                   | individual.                       |
+-----------------------------------+-----------------------------------+
