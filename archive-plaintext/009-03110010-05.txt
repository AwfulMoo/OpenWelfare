Determining the Date of Effect for cancellation of CP and/or CA

This table describes a determining the correct Date of Effect (DOE) for
cancellations of CP and/or CA.

+-----------------------------------+-----------------------------------+
| Scenario                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | CLR: Withdrawn/Voluntary          |
|                                   | Surrender                         |
|                                   |                                   |
|                                   | The DOE for cancellation of CP    |
|                                   | and/or CA where the carer has     |
|                                   | requested to be cancelled,        |
|                                   | without providing additional      |
|                                   | information will be determined if |
|                                   | there is a known notification     |
|                                   | event which effected the carer’s  |
|                                   | entitlement to payment.           |
|                                   |                                   |
|                                   | If there is:                      |
|                                   |                                   |
|                                   | no known notification event       |
|                                   |                                   |
|                                   | -   DOE: date the carer is        |
|                                   |     requesting payment to be      |
|                                   |     cancelled                     |
|                                   |                                   |
|                                   | a notification event and the      |
|                                   | carer does not provide the date   |
|                                   | the change occurred               |
|                                   |                                   |
|                                   | -   DOE: date paid to + 1         |
+-----------------------------------+-----------------------------------+
