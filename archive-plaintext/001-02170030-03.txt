| 3                                 | Verify customer's enrolment in    |
|                                   | AMEP + Read more ...              |

|                                   | type/Non-applied exemptions >     |
|                                   | Referral summary                  |
|                                   |                                   |
|                                   | Check the Adult Migrant English   |
|                                   | Program Referral on the Referral  |
|                                   | summary, or                       |

|                                   | the Referral summary screen, seek |
|                                   | permission from the customer to   |

|                                   | Recording System (ARMS) helpdesk  |

|                                   | assist ARMS to identify the       |
|                                   | customer and respond in a timely  |
|                                   | manner:                           |
|                                   |                                   |
|                                   | -   your contact details          |
|                                   | -   customer's first name, last   |
|                                   |     name and date of birth        |
|                                   | -   Customer's visa information   |
|                                   |     (e.g. visa grant number,      |
|                                   |     passport number or            |

|                                   | If the customer has not given     |

|                                   | to all AMEP customers at          |

|                                   | personal details, AMEP timetable, |
|                                   | previous activities including     |
|                                   | start and end dates, achievements |
|                                   | and remaining entitlement.        |
|                                   | Participating in SPP, AMEP Extend |

| 4                                 | Check hours customer is           |
|                                   | participating in AMEP + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Participation of 20 hours or more |
|                                   | per week is regarded as full-time |
|                                   | participation. The 20 hours may   |
|                                   | consist of up to 15 hours face to |
|                                   | face and five hours of            |
|                                   | independent learning per week.    |
|                                   |                                   |
|                                   | Anything less than 20 hours per   |
|                                   | week is regarded as part-time     |
|                                   | participation.                    |
|                                   |                                   |
|                                   | -   Ask customer to provide       |
|                                   |     letter from AMEP provider     |
|                                   |     advising the amount of hours  |
|                                   |     they are studying, or         |
|                                   | -   Phone AMEP provider to verify |
|                                   |     details over the phone.       |
|                                   |     Customer will need to be      |
|                                   |     present in order to overcome  |
|                                   |     any privacy requirements      |
|                                   | -   Record the details in a DOC   |
|                                   |     on the customer's record      |
|                                   |                                   |
|                                   | Next steps:                       |
|                                   |                                   |
|                                   | -   If the customer is a newly    |
|                                   |     arrived refugee post their    |
|                                   |     refugee exemption and is      |
|                                   |     eligible to remain Centrelink |
|                                   |     managed. See Refugee and      |
|                                   |     humanitarian entrant service  |
|                                   |     offer for newly arrived       |
|                                   |     refugee Centrelink managed    |
|                                   |     eligibility, go to Step 5     |
|                                   | -   For all other customers, go   |
|                                   |     to Step 6                     |

|                                   | Read more ...                     |

Exits and recommencements from AMEP

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Exits from AMEP + Read more ...   |
|                                   |                                   |
|                                   | A manual follow up (MFU) review   |
|                                   | (I002RR) will be created when the |
|                                   | placement in AMEP has been        |
|                                   | recorded, and will mature one     |
|                                   | month prior to the placement end  |
|                                   | date. Service Officers should     |
|                                   | check the MFUs when the end date  |
|                                   | has been reached to confirm the   |
|                                   | customer has been exited from     |
|                                   | AMEP. The MFU can be              |
|                                   | completed/finalised once the exit |
|                                   | has been recorded.                |
|                                   |                                   |
|                                   | A participant may withdraw from   |
|                                   | AMEP for a period of time. The    |
|                                   | AMEP referral will be             |
|                                   | automatically finalised with the  |
|                                   | reason 'Withdrawn'. The date of   |
|                                   | effect of this finalisation may   |
|                                   | be up to three months in the      |
|                                   | past.                             |
|                                   |                                   |
|                                   | When AMEP is finalised on the     |
|                                   | Referral Summary screen it will   |
|                                   | also finalise the AMEP activity   |
|                                   | on the Activity and Exemption     |
|                                   | Summary (AEX) screen. This will   |
|                                   | allow the job seeker to be able   |
|                                   | to recommence AMEP at a later     |
|                                   | date if they have unused AMEP     |
|                                   | hours left.                       |
|                                   |                                   |
|                                   | Participants will exit AMEP on    |
|                                   | achievement of Vocational         |
|                                   | English.                          |
|                                   |                                   |
|                                   | When the job seeker is no longer  |
|                                   | eligible and not participating,   |
|                                   | the referral will be updated      |
|                                   | automatically to 'Finalised'      |
|                                   | (FIN) and the level of            |
|                                   | achievement in an applicable      |
|                                   | Curriculum will also be displayed |
|                                   | on the Referral Summary screen.   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Recommencement in AMEP + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a customer wishes to           |
|                                   | recommence in AMEP and they are:  |
|                                   |                                   |
|                                   | -   not participating in another  |
|                                   |     English language program, go  |
|                                   |     to Step 3                     |
|                                   | -   participating in another      |
|                                   |     program, go to Step 4         |
+-----------------------------------+-----------------------------------+
| 3                                 | Recommencement in AMEP and not    |
|                                   | participating in another          |
|                                   | program + Read more ...           |
|                                   |                                   |
|                                   | When an AMEP placement is         |
|                                   | finalised for the reason          |
|                                   | 'withdrawn', a participant may    |
|                                   | recommence in training.           |
|                                   |                                   |
|                                   | When the participant resumes      |
|                                   | training, their record will be    |
|                                   | automatically updated to 'placed' |
|                                   | in the Referral Summary screen.   |
|                                   |                                   |
|                                   | The AEX screen will be updated    |
|                                   | with the information from the     |
|                                   | Referral Summary screen. The      |
|                                   | start date of the activity on the |
|                                   | AEX screen will be the date of    |
|                                   | the recommencement and the        |
|                                   | activity will have an end date 12 |
|                                   | months in the future.             |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Recommencement in AMEP and        |
|                                   | participant is in another         |
|                                   | program + Read more ...           |
|                                   |                                   |
|                                   | If the job seeker is placed in    |
|                                   | SEE, the SEE referral must be     |
|                                   | finalised before recommencing in  |
|                                   | AMEP.                             |
|                                   |                                   |
|                                   | The AMEP recommencement will not  |
|                                   | be automatically updated in the   |
|                                   | Referral Summary screen if the    |
|                                   | participant is participating in   |
|                                   | another program that is not an    |
|                                   | allowable combination with AMEP.  |
|                                   |                                   |
|                                   | A RFL/AUT MFU with keywords       |
|                                   | I08IRR and AMEREC will require    |
|                                   | the Service Officer to contact    |
|                                   | the providers and confirm the     |
|                                   | program the participant is        |
|                                   | attending. If the job seeker is   |
|                                   | not attending the other program,  |
|                                   | the provider is to exit them from |
|                                   | the program. AMEP recommencement  |
|                                   | is then to be manually coded by   |
|                                   | staff with REFUPD access. If the  |
|                                   | customer is attending the other   |
|                                   | program then the MFU can be       |
|                                   | cancelled. A DOC is created with  |
|                                   | help text explaining the course   |
|                                   | of action that needs to be taken. |
+-----------------------------------+-----------------------------------+
| 5                                 | SEE participation withdrawn by    |
|                                   | the Department of Home Affairs +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For job seekers who have their    |
|                                   | SEE placement withdrawn by the    |
|                                   | Department of Home Affairs        |
|                                   | (indicating that the customer did |
|                                   | not start in SEE) subsequent to   |
|                                   | AMEP finalisation with reason     |
|                                   | 'Participating in Skills for      |
|                                   | Education and Employment', a      |
|                                   | RFL/AUT MFU with keywords I077RR  |
|                                   | and AMEFIN is created.            |
|                                   |                                   |
|                                   | This requires the Service Officer |
|                                   | to contact the SEE and AMEP       |
|                                   | providers and confirm in which    |
|                                   | program the job seeker is         |
|                                   | participating. If the job seeker  |
|                                   | continues AMEP participation then |
|                                   | a Service Officer with REFUPD     |
|                                   | access will need to delete the    |
|                                   | AMEP finalisation with reason     |
|                                   | 'Participating in Skills for      |
|                                   | Education and Employment'.        |
+-----------------------------------+-----------------------------------+
| 6                                 | AMEP Identifier (ARM Id)          |
|                                   | Unlinked + Read more ...          |
|                                   |                                   |
|                                   | For job seekers who have their    |
|                                   | ARM Id unlinked by the Department |
|                                   | of Home Affairs, a RFL/AUT MFU    |
|                                   | with keywords ARMDEL and I078RR   |
|                                   | will be created. This required    |
|                                   | investigation on the AMEP         |
|                                   | referral that was associated with |
|                                   | incorrect ARM Id and deleting the |
|                                   | AMEP referral if necessary. A     |
|                                   | Service Officer needs REFUPD      |
|                                   | access to delete an AMEP          |
|                                   | referral.                         |
