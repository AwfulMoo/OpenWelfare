|                                   | as a bundle.                      |
|                                   |                                   |
|                                   | Bundles are allocated to staff    |
|                                   | with the assigned skill tag for   |

|                                   | claim.                            |
|                                   |                                   |

|                                   | records.                          |

|                                   | Program business owners decide    |

|                                   | Business Rules include the        |
|                                   | bundling rules.                   |
|                                   |                                   |
|                                   | Work allocated via Quick Ready    |
|                                   | will not be bundled.              |
|                                   |                                   |
|                                   | When staff are allocated a bundle |
|                                   |                                   |
|                                   | An Allocate the following bundle  |
|                                   | pop up box displays for them. It  |
|                                   | shows the work items in the       |
|                                   | bundle before they get assigned   |
|                                   | to their inbox:                   |
|                                   |                                   |
|                                   | -   Staff are automatically       |
|                                   |     assigned all work in the      |
|                                   |     bundle they are skilled for   |
|                                   | -   Staff can accept extra work   |
|                                   |     items they may not have a     |
|                                   |     skill tag for, but have the   |

|                                   | confirms the total number of      |
|                                   | bundled work items on the         |
|                                   | customer and partner records      |
|                                   |                                   |

|                                   | bundled work item                 |
|                                   |                                   |
|                                   | Allocate: a checkbox indicates    |
|                                   | items that will be assigned from  |
|                                   | the bundle to their inbox:        |
|                                   |                                   |
|                                   | -   Select the checkbox next to a |
|                                   |     work item to accept it        |
|                                   | -   Select the checkbox in the    |
|                                   |     table header to accept all    |
|                                   |     items in the bundle           |

|                                   |     item they have the skill tag  |
|                                   |     for                           |
|                                   |                                   |

|                                   | -   Customer Contact is Required  |
|                                   | -   Service Officer has Incorrect |
|                                   |     skills                        |
|                                   | -   Work Item not Actionable      |
|                                   | -   To improve processing         |
|                                   |     efficiency. This auto         |
|                                   |     displays when a staff member  |
|                                   |     accepts a work item not       |
|                                   |     assigned to them              |
|                                   |                                   |
|                                   | Work item ID for each item in the |
|                                   | bundle                            |
|                                   |                                   |
|                                   | Submit to assign all selected     |
|                                   | work items to inbox               |
|                                   |                                   |
|                                   | Staff should complete any         |
|                                   | outstanding work on a customer's  |
|                                   | record if they can. This gives a  |
|                                   | better outcome for customers.     |
