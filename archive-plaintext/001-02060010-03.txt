Selection and allocation of a Workforce Australia Employment Services Provider

Table 2: This table describes the process of selecting and allocating a
provider. Note: this process does not apply to job seeker selected for
Workforce Australia Online for Individuals. These job seekers have the
referral completed by the Department of Employment and Workplace
Relations.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | The List Providers screen + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A list of all providers mapped to |
|                                   | the job seeker's postcode appears |
|                                   | on this screen. More than one     |
|                                   | page may be available and there   |
|                                   | is an ability to scroll through   |
|                                   | these selections. Providers which |
|                                   | are full or not yet open for      |
|                                   | business will display even though |
|                                   | they may not be selected.         |
|                                   |                                   |
|                                   | This screen provides different    |
|                                   | options to display Workforce      |
|                                   | Australia providers that are      |
|                                   | mapped to the job seeker's        |
|                                   | postcode. There are 2 radio       |
|                                   | buttons displayed on top of the   |
|                                   | screen: Show providers with       |
|                                   | Immediate Appointments, and Show  |
|                                   | all providers. The Show Providers |
|                                   | with Immediate Appointments       |
|                                   | button will be defaulted on this  |
|                                   | screen.                           |
|                                   |                                   |
|                                   | -   For RapidConnect customers:   |
|                                   |     when Show Providers with      |
|                                   |     Immediate Appointments button |
|                                   |     is selected, the list will    |
|                                   |     display only those providers  |
|                                   |     that have appointments within |
|                                   |     the system date plus 2        |
|                                   |     working days. Select the Show |
|                                   |     All Providers radio button,   |
|                                   |     the list will be amended to   |
|                                   |     display those providers that  |
|                                   |     have available appointments   |
|                                   |     within 14 days                |
|                                   | -   For non-RapidConnect          |
|                                   |     customers: when Show          |
|                                   |     Providers with Immediate      |
|                                   |     Appointments is selected, the |
|                                   |     list will display only those  |
|                                   |     providers that have           |
|                                   |     appointments within the       |
|                                   |     system date plus 2 working    |
|                                   |     days. By selecting the Show   |
|                                   |     All Providers radio button,   |
|                                   |     the list will be amended to   |
|                                   |     display those providers that  |
|                                   |     have available appointments   |
|                                   |     within 28 days                |
|                                   |                                   |
|                                   | Providers who do not have         |
|                                   | available appointments on the day |
|                                   | or within the next 2 working days |
|                                   | will have a status of having 'no  |
|                                   | immediate appointments sessions   |
|                                   | within 2 business days'. Job      |
|                                   | seekers should be encouraged to   |
|                                   | choose a provider with immediate  |
|                                   | appointments available on the day |
|                                   | or within 2 working days.         |
|                                   |                                   |
|                                   | Ask the job seeker to select a    |
|                                   | Workforce Australia provider from |
|                                   | the list.                         |
|                                   |                                   |
|                                   | -   only an 'Available', 'No      |
|                                   |     sessions within 2 business    |
|                                   |     days', and 'No immediate      |
|                                   |     sessions within 14 business   |
|                                   |     days' provider with available |
|                                   |     capacity can be selected      |
|                                   | -   a job seeker can have only    |
|                                   |     one Workforce Australia       |
|                                   |     provider                      |
|                                   | -   if the job seeker's preferred |
|                                   |     Workforce Australia provider  |
|                                   |     is full, they will need to    |
|                                   |     select another provider       |
|                                   |                                   |
|                                   | Has the job seeker selected a     |
|                                   | Workforce Australia provider?     |
|                                   |                                   |
|                                   | -   Yes, place the mouse on the   |
|                                   |     provider details and select   |
|                                   |     once. Go to Step 6            |
|                                   | -   No, go to Step 2              |

+-----------------------------------+-----------------------------------+

|                                   | Australia provider + Read more    |
|                                   | ...                               |

|                                   | provider more suitable to their   |
|                                   | needs.                            |
|                                   |                                   |
|                                   | Refine the search by selecting    |

|                                   | -   Provider Name: field          |

|                                   |                                   |

|                                   | service call.                     |
|                                   |                                   |
|                                   | Ask the job seeker to select a    |

|                                   | Only an open provider with        |

|                                   | selected.                         |
|                                   |                                   |

|                                   | Workforce Australia provider from |
|                                   | the list resulting from the       |

|                                   |     select once or press [Enter]. |

| 5                                 | Random allocation + Read more ... |
|                                   |                                   |
|                                   | Random allocation uses the        |
|                                   | criteria of the job seeker's      |

|                                   | available capacity.               |
|                                   |                                   |
|                                   | Press the 'random allocation'     |
