into SAP notes.

|                                   | decision made to write off        |
|                                   | <amount>, <invoice number>        |
|                                   | overpayment. All attempts made to |
|                                   | contact this customer via mail,   |
|                                   | phone and/or email. Numbers are   |

|                                   | sender and/or no response to      |

|                                   |                                   |
|                                   | All searches for contact details  |
|                                   | completed through Google, White   |
|                                   | Pages, CDMS and program area. No  |
|                                   | new/updated details available.    |
|                                   |                                   |
|                                   | Overpayment written off           |
|                                   | uneconomical to pursue under      |
|                                   | section 11 of the PGPA Rule 2014. |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer deceased with no estate  |
|                                   | / Legally unrecoverable           |
|                                   |                                   |
|                                   | Further investigation confirms    |
|                                   | customer passed away on <date>.   |
|                                   | [Advice received that there is no |
|                                   | estate and customer has           |
|                                   | insufficient funds available/or   |
|                                   | no other details in relation to   |
|                                   | the executor of the will]. All    |
|                                   | avenues are exhausted.            |
|                                   | Overpayment of <amount> written   |
|                                   | off not legally recoverable under |
|                                   | section 11 of the PGPA Rule 2014. |
