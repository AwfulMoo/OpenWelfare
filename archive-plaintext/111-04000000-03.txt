| 3                                 | Scam impersonating Services       |
|                                   | Australia Brand + Read more ...   |
|                                   |                                   |
|                                   | The Services Australia brand is   |
|                                   | often impersonated by scammers    |
|                                   | who use a range of tactics to get |
|                                   | what they want, which may be      |
|                                   | money, personal information or    |
|                                   | login information. Scammers may   |
|                                   | contact over the telephone and    |
|                                   | disguise (spoof) their number/s,  |
|                                   | and use automated calling         |
|                                   | techniques or recorded messages.  |
|                                   | The Resources page contains a     |
|                                   | link to the Brand intranet page.  |
|                                   |                                   |
|                                   | Scammers will often develop fake  |
|                                   | websites that look genuine,       |
|                                   | encouraging customers to click on |
|                                   | a link from an email or SMS,      |
|                                   | which will redirect the customer  |
|                                   | to the fake website.              |
|                                   |                                   |
|                                   | Check to see if the Helpdesk is   |
|                                   | already aware of the scam, see:   |
|                                   |                                   |
|                                   | -   Phishing scam records -       |
|                                   |     Helpdesk Resources            |
|                                   | -   Scams Awareness Hub -         |
|                                   |     Intranet                      |
|                                   | -   Scamwatch                     |
|                                   |                                   |
|                                   | If the helpdesk requires further  |
|                                   | details about the scam:           |
|                                   |                                   |
|                                   | -   Ask the customer to forward   |
|                                   |     the phishing email or a       |
|                                   |     screen shot of the phishing   |
|                                   |     SMS, social media post to the |
|                                   |     Helpdesk email address        |
|                                   | -   Acknowledge receipt and move  |
|                                   |     the email to the 'Scams'      |
|                                   |     folder                        |
|                                   | -   The Identity Security, Scams  |
|                                   |     and Response Team will        |
|                                   |     complete further analysis and |
|                                   |     update support material (if   |
|                                   |     required). Note: If the scam  |
|                                   |     is via phone, take note of    |
|                                   |     the scammers methodology e.g. |
|                                   |     are they using a robo dialler |
|                                   |     and what does this message    |
|                                   |     say, and include this         |
|                                   |     information in the email to   |
|                                   |     Identity Security, Scams and  |
|                                   |     Response Team                 |
|                                   |                                   |
|                                   | Has the customer clicked on a     |
|                                   | hyperlink and/or provided         |
|                                   | personal information to the       |
|                                   | scammer?                          |
|                                   |                                   |
|                                   | Yes, the customer has clicked on  |
|                                   | a hyperlink and/or disclosed      |
|                                   | their personal information, go to |
|                                   | Item 4                            |
|                                   |                                   |
|                                   | No, tell the customer the         |
|                                   | following (as appropriate):       |
|                                   |                                   |
|                                   | -   the contact was a scam and    |
|                                   |     they are unlikely to be at    |
|                                   |     risk                          |
|                                   | -   scams target everyone.        |
|                                   |     Scammers use stealth,         |
|                                   |     surprise, intimidation, and   |
|                                   |     clever tactics to get what    |
|                                   |     they want                     |
|                                   | -   scammers use technology to    |
|                                   |     disguise their phone numbers  |
|                                   |     and email address to make the |
|                                   |     scam more believable          |
|                                   | -   scammers often pretend to be  |
|                                   |     from trusted organisations,   |
|                                   |     including Services Australia  |
|                                   | -   recommend the customer visit  |
|                                   |     the Scamwatch website to      |
|                                   |     report the scam and read the  |
|                                   |     scam educational resources    |
|                                   |     available. This includes      |
|                                   |     visiting the 'Get Help'       |
|                                   |     section of their website      |
|                                   | -   if the customer received a    |
|                                   |     scam telephone call, tell     |
|                                   |     them to report the incident   |
|                                   |     to their telecommunication    |
|                                   |     provider                      |
|                                   | -   Procedure ends here           |
