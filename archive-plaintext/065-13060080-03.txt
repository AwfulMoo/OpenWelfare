This page contains various processes about returning or redirecting aged
care services forms.

On this page:

Returning a form to an aged care provider

Redirect Centrelink forms to the Smart Centres Aged Care team

Returning a form to an aged care provider

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Receiving a form + Read more ...  |
|                                   |                                   |
|                                   | Service Officers in the Aged Care |
|                                   | and Allied Health (ACAH) Branch   |
|                                   | can receive (via PaNDA):          |
|                                   |                                   |
|                                   | -   an incomplete aged care form  |
|                                   | -   a form for either the         |
|                                   |     Department of Health and Aged |
|                                   |     Care or the Department of     |
|                                   |     Veterans’ Affairs (DVA)       |
|                                   | -   a form for Centrelink         |
|                                   |                                   |
|                                   | When Service Officers receive:    |
|                                   |                                   |
|                                   | -   an incomplete aged care form, |
|                                   |     go to Step 2                  |
|                                   | -   a form for the Department of  |
|                                   |     Health and Aged Care or DVA,  |
|                                   |     go to Step 3                  |
|                                   | -   a form for Centrelink, go to  |
|                                   |     Table 2                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Returning an incomplete form +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When Service Officers receive an  |
|                                   | incomplete form via PaNDA, the    |
|                                   | form must be returned to the aged |
|                                   | care service with a letter.       |
|                                   |                                   |
|                                   | Service Officers can access       |
|                                   | letter templates from Operational |
|                                   | Blueprint letters and electronic  |
|                                   | messaging                         |
|                                   |                                   |
|                                   | Under Medicare:                   |
|                                   |                                   |
|                                   | -   select Provider and Third     |
|                                   |     Party letters                 |
|                                   | -   select Aged Care              |
|                                   |                                   |
|                                   | Select the relevant template:     |
|                                   |                                   |
|                                   | -   Z0264 - Standard Letter       |
|                                   |     Template More information     |
|                                   |     required to process           |
|                                   |     application (excluding ACFI)  |
|                                   |                                   |
|                                   | Select the Enable Macros button   |
|                                   | at the top of the letter template |
|                                   |                                   |
|                                   | Complete the Standard Letter      |
|                                   | Template with relevant details    |
|                                   | from the incomplete form          |
|                                   |                                   |
|                                   | Add a note in PaNDA with all      |
|                                   | actions taken                     |
|                                   |                                   |
|                                   | -   see View, add and edit        |
|                                   |     processing notes to a work    |
|                                   |     item – Medicare only table    |
|                                   |                                   |
|                                   | Upload a PDF copy of the letter   |
|                                   | in PaNDA                          |
|                                   |                                   |
|                                   | -   see Add a document to an      |
|                                   |     existing work item table      |
|                                   |                                   |
|                                   | Post the letter to the aged care  |
|                                   | service and include a copy of the |
|                                   | incomplete form                   |
|                                   |                                   |
|                                   | Service Officers must also        |
|                                   | contact the aged care service via |
|                                   | email or telephony:               |
|                                   |                                   |
|                                   | Advise the form is incomplete and |
|                                   | to submit a completed form        |
|                                   |                                   |
|                                   | Add a note in PaNDA with all      |
|                                   | actions taken                     |
|                                   |                                   |
|                                   | -   see View, add and edit        |
|                                   |     processing notes to a work    |
|                                   |     item – Medicare only table    |
|                                   |                                   |
|                                   | For all email correspondence with |
|                                   | the aged care service, a PDF copy |
|                                   | of the email and a note must also |
|                                   | be added in PaNDA                 |
|                                   |                                   |
|                                   | -   upload a PDF copy of a        |
|                                   |     letter, form or email         |
|                                   |     correspondence, see Add a     |
|                                   |     document to an existing work  |
|                                   |     item table                    |
|                                   |                                   |
|                                   | Procedure ends here until the     |
|                                   | details are provided.             |
|                                   |                                   |
|                                   | Note: once the resubmitted form   |
|                                   | is received it will be treated as |
|                                   | a new work item.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Returning a form sent to the ACAH |
|                                   | Branch incorrectly + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | When Service Officers receive     |
|                                   | forms via PaNDA that another      |
|                                   | policy department is responsible  |
|                                   | for, the form must be returned to |
|                                   | the aged care service.            |
|                                   |                                   |
|                                   | Service Officers must contact the |
|                                   | aged care service via email or    |
|                                   | telephony advising:               |
|                                   |                                   |
|                                   | -   the form has been sent to the |
|                                   |     incorrect department          |
|                                   | -   the aged care service must    |
|                                   |     contact the relevant policy   |
|                                   |     department to confirm the     |
|                                   |     correct contact details on    |
|                                   |     where the form needs to be    |
|                                   |     sent to                       |
|                                   |                                   |
|                                   | After contacting the aged care    |
|                                   | service, Service Officers must:   |
|                                   |                                   |
|                                   | add a note in PaNDA with all      |
|                                   | actions taken                     |
|                                   |                                   |
|                                   | -   see View, add and edit        |
|                                   |     processing notes to a work    |
|                                   |     item – Medicare only table    |
|                                   |                                   |
|                                   | upload a PDF copy of the email in |
|                                   | PaNDA for all email               |
|                                   | correspondence with the aged care |
|                                   | services)                         |
|                                   |                                   |
|                                   | -   see Add a document to an      |
|                                   |     existing work item table      |
|                                   |                                   |
|                                   | The Resources page contains links |
|                                   | to contact information for the    |
|                                   | Department of Health and Aged     |
|                                   | Care and the Department of        |
|                                   | Veterans’ Affairs.                |

Redirect Centrelink forms to the Smart Centres Aged Care team

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Receiving forms incorrectly sent  |
|                                   | to the ACAH Branch + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Note: Service Officers must       |
|                                   | attach the form and the work item |
|                                   | ID in an email when escalating    |
|                                   | the request to the Team Leader.   |
|                                   |                                   |
|                                   | When Service Officers receive a   |
|                                   | form via PaNDA directed to        |
|                                   | Centrelink, the form must be      |
|                                   | redirected to the Smart Centres   |
|                                   | Aged Care team for processing.    |
|                                   |                                   |
|                                   | Service Officers must:            |
|                                   |                                   |
|                                   | advise the aged care service via  |
|                                   | email or telephony that the form  |
|                                   | has been sent to the incorrect    |
|                                   | department                        |
|                                   |                                   |
|                                   | advise that the ACAH Branch will  |
|                                   | redirect the form to the Smart    |
|                                   | Centres Aged Care team for        |
|                                   | processing                        |
|                                   |                                   |
|                                   | print out the form (the document) |
|                                   | and post the document to Smart    |
|                                   | Centres Aged Care team via        |
|                                   | internal mail                     |
|                                   |                                   |
|                                   | -   address the mail to RCA       |
|                                   |     Centrelink                    |
|                                   |                                   |
|                                   | add a note in PaNDA stating that  |
|                                   | the hard copy form has been sent  |
|                                   | to Smart Centres Aged Care team   |
|                                   |                                   |
|                                   | mark the work item as a Completed |
|                                   | status in PaNDA:                  |
|                                   |                                   |
|                                   | -   to add a note in PaNDA, see   |
|                                   |     View, add and edit processing |
|                                   |     notes to a work item –        |
|                                   |     Medicare only table           |
|                                   |                                   |
|                                   | The form sent via internal mail   |
|                                   | will be directed to the Smart     |
|                                   | Centre Aged Care team and         |
|                                   | allocated as a workflow item via  |
|                                   | the Work Optimiser – Centrelink   |
|                                   | system.                           |
|                                   |                                   |
|                                   | The Smart Centres Aged Care team  |
|                                   | are able to access the Work       |
|                                   | Optimiser – Centrelink system and |
|                                   | retrieve the form for processing. |
