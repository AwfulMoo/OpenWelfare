Granting an Identity Review Period, and recommence cancelled payment

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessing an Identity Review      |
|                                   | Period review + Read more ...     |
|                                   |                                   |
|                                   | Code and verify acceptable        |
|                                   | identity documents lodged by the  |
|                                   | customer using the Identity       |
|                                   | Confirmation Dashboard in Process |
|                                   | Direct                            |
|                                   |                                   |
|                                   | Complete the assessment           |
|                                   |                                   |
|                                   | The Identity Review Period of 28  |
|                                   | or 56 days automatically applies  |
|                                   | based on:                         |
|                                   |                                   |
|                                   | -   assessment results and        |
|                                   | -   the customer's circumstances  |
|                                   |                                   |
|                                   | An online task will request more  |
|                                   | identity documents                |
|                                   |                                   |
|                                   | Customers can upload identity     |
|                                   | documents or lodge over the phone |
+-----------------------------------+-----------------------------------+
| 2                                 | Return of documents for Identity  |
|                                   | Review Period + Read more ...     |
|                                   |                                   |
|                                   | Has the customer lodged the       |
|                                   | required identity documents       |
|                                   | within the Identity Review        |
|                                   | Period?                           |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Update the customer's record  |
|                                   |     using the Identity            |
|                                   |     Confirmation Dashboard in     |
|                                   |     Process Direct                |
|                                   | -   See Coding identity documents |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Recommencement of payment if      |
|                                   | cancelled insufficient identity,  |
|                                   | Identity Review Period expires +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Only restore the customer's       |
|                                   | payment or service:               |
|                                   |                                   |
|                                   | if it was cancelled due to expiry |
|                                   | of an Identity Review Period, and |
|                                   |                                   |
|                                   | after using the Identity          |
|                                   | Confirmation Dashboard in Process |
|                                   | Direct to:                        |
|                                   |                                   |
|                                   | -   code and verify the required  |
|                                   |     identity documents, or        |
|                                   | -   complete the Alternative      |
|                                   |     Identity assessment           |
|                                   |                                   |
|                                   | []Before restoring a customer's   |
|                                   | payment, they must:               |
|                                   |                                   |
|                                   | -   complete photo linkage in     |
|                                   |     person, or                    |
|                                   | -   be assessed for an            |
|                                   |     Alternative Identity (if not  |
|                                   |     already completed)            |
|                                   |                                   |
|                                   | Do not restore customer's         |
|                                   | payments for:                     |
|                                   |                                   |
|                                   | -   Carer Payment (CP) customers  |
|                                   |     cancelled due to insufficient |
|                                   |     identity documents for their  |
|                                   |     care receiver, or             |
|                                   | -   Mobility Allowance (MOB)      |
|                                   |     customers                     |
|                                   |                                   |
|                                   | Consider Alternative Identity for |
|                                   | these customers.                  |
|                                   |                                   |
|                                   | See Alternative Identity or       |
|                                   | Commencement of Identity for      |
|                                   | customers who have not lodged     |
|                                   | identity documents within the     |
|                                   | review period because:            |
|                                   |                                   |
|                                   | -   they are genuinely finding    |
|                                   |     them difficult to get, or     |
|                                   | -   they are unable to get/lodge  |
|                                   |     the identity documents        |
