This page contains information on how to finalise an assessment,
complete a Quality Assurance (QA) check of the assessment process and
make a recommendation to the delegate for initial, subsequent, S38
response and refund applications, submitted via the MIOC system.

|                                   |     38 letter                     |

|                                   | to deem the claim not payable, go |
|                                   | to Step 8                         |
|                                   |                                   |
|                                   | For subsequent payment            |
|                                   | applications                      |

|                                   | -   update the existing filing    |

| 9                                 | Quality checks and obtaining      |
|                                   | delegate approval + Read more ... |
|                                   |                                   |
|                                   | Once the AO has assessed the      |
|                                   | claim, a Quality Assurance (QA)   |
|                                   | check is required from another    |
|                                   | AO.                               |
|                                   |                                   |
|                                   | The second AO is required to:     |
|                                   |                                   |
|                                   | -   undertake a QA of the MI      |
|                                   |     payment application to        |
|                                   |     determine if they agree with  |
|                                   |     the AO recommendation         |
|                                   | -   do a final proofread to       |
|                                   |     ensure formatting, spelling,  |
|                                   |     grammar and financials are    |
|                                   |     correct. If unsure, refer to  |
|                                   |     the Services Australia        |
|                                   |     Writing Style Guide on the    |
|                                   |     Resources page                |
|                                   |                                   |
|                                   | If during the QA the second AO    |
|                                   | finds errors or issues in         |
|                                   | relation to the MI payment        |
|                                   | application assessment, discuss   |
|                                   | them with the first AO and if     |
|                                   | necessary the EL1 Assistant       |
|                                   | Director. Once agreement has been |
|                                   | reached, all relevant             |
|                                   | documentation must be amended as  |
|                                   | required.                         |
|                                   |                                   |
|                                   | Once the second AO is sure that   |
|                                   | the MI payment application has    |
|                                   | been assessed correctly and the   |
|                                   | documentation is correct:         |
|                                   |                                   |
|                                   | -   complete the Assessment       |
|                                   |     Checklist                     |
|                                   | -   select and initial the QA box |
|                                   |     on the covering sheet         |
|                                   | -   update the Claims on Hand     |
|                                   | -   place the file in the EL1’s   |
|                                   |     cupboard for EL1 Assistant    |
|                                   |     Director to conduct a third   |
|                                   |     and final Quality Check (QC)  |
|                                   |     before the claim is handed to |
|                                   |     the delegate for their        |
|                                   |     consideration                 |
|                                   |                                   |
|                                   | The EL1 checks the entire claim   |
|                                   | including the payment             |
|                                   | application, the eligibility for  |
|                                   | payment, HP Records Manager, the  |
|                                   | registration and assessment       |
|                                   | worksheets, the file note and the |
|                                   | recommendation.                   |
|                                   |                                   |
|                                   | If any errors are identified at   |
|                                   | this point the claim is then      |
|                                   | handed back to the first AO. Once |
|                                   | the first AO has amended any      |
|                                   | errors, the claim undergoes the   |
|                                   | third QC by the EL1 again.        |
|                                   |                                   |
|                                   | Once the EL1 endorses and agrees  |
|                                   | with the recommendation, the EL1  |
|                                   | will sign the file note and       |
|                                   | coversheet, update Claims on Hand |
|                                   | to note that the MI payment       |

|                                   | Director of the MI team for       |
|                                   | approval.                         |
|                                   |                                   |
|                                   | If the Delegate agrees with the   |
|                                   | recommendation, they will sign    |
|                                   | the assessment worksheet on the   |
|                                   | file and coversheet. If the       |
|                                   | delegate does not agree with the  |
|                                   | recommendation, the delegate will |
|                                   | discuss with the EL1 for          |
|                                   | resolution of the payment         |
|                                   | application.                      |

|                                   |     process                       |
