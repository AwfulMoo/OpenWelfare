FAQs - ACFI questions and answers

Table 2: this table contains questions and answers associated with the
Aged Care Funding Instrument (ACFI).

+-----------------------------------+-----------------------------------+
| Item                              | Question and answers              |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: What is a CNR?          |
|                                   |                                   |
|                                   | Answer: CNR equates to the term   |
|                                   | 'care needs reassessed'.          |
|                                   |                                   |
|                                   | A service may submit an ACFI with |
|                                   | appraisal type 'CNR' when a care  |
|                                   | recipient’s care needs have been  |
|                                   | reassessed:                       |
|                                   |                                   |
|                                   | -   six months after discharge    |
|                                   |     from hospital leave           |
|                                   | -   when Resident Classification  |
|                                   |     Scale (RCS) has expired       |
|                                   | -   more than 12 months after the |
|                                   |     last appraisal was performed  |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: What is TRA and how do  |
|                                   | I use it?                         |
|                                   |                                   |
|                                   | Answer: TRA equates to the term   |
|                                   | 'transfer from another facility'. |
|                                   |                                   |
|                                   | A service may submit an ACFI with |
|                                   | appraisal type 'TRA' when a       |
|                                   | recipient's care needs have been  |
|                                   | reassessed following transfer of  |
|                                   | a care recipient within 28 days.  |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How do I convert the    |
|                                   | 'Reason for this application'     |
|                                   | section into an appropriate       |
|                                   | appraisal code type?              |
|                                   |                                   |
|                                   | Answer: The service should use    |
|                                   | the ACFI Appraisal Pack when      |
|                                   | completing the ACFI web form as   |
|                                   | this information is the same.     |
|                                   |                                   |
|                                   | See the Department of Health and  |
|                                   | Aged Care website, Aged Care      |
|                                   | Funding Instrument User Guide.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: How can I look up to    |
|                                   | see whether a new resident to my  |
|                                   | facility has had an ACFI          |
|                                   | appraisal completed by another    |
|                                   | service?                          |
|                                   |                                   |
|                                   | Answer: Aged care services are    |
|                                   | unable to view ACFIs submitted by |
|                                   | other services due to privacy     |
|                                   | reasons. The service may call the |
|                                   | previous service for the          |
|                                   | information.                      |

PDF version of payment statements – residential aged care services

A PDF version of a payment statement for a residential aged care service
is not currently available through the Aged Care Provider Portal (ACPP).

If a residential aged care service requires a PDF version of a payment
statement, the aged care service provider:

-   must open the relevant page from the payment statement/summary for
    the appropriate claim month from the service landing page
-   press the control and P buttons on the keyboard (Ctrl + P)
-   under the ‘printer’ dropdown (for Microsoft Edge) or ‘destination’
    dropdown (for Google Chrome, select ‘save as PDF’
-   select the ‘save’ button at the bottom of the popup screen
-   save the relevant page of the payment statement on the desktop or in
    a folder that is easy to access

Repeat the above steps for any other pages of the payment statement.

PDF version of Aged Care Funding Instrument (ACFI) – residential aged care services

A PDF version of an Aged Care Funding Instrument (ACFI) for a
residential care recipient is not currently available through the Aged
Care Provider Portal (ACPP).

If a residential aged care service requires a PDF version of a care
recipient’s ACFI, the aged care service provider:

-   must open the relevant page from the ACFI summary screen for the
    relevant care recipient in ACPP
-   press the control and P buttons on the keyboard (Ctrl + P)
-   under the ‘printer’ dropdown (for Microsoft Edge) or ‘destination’
    dropdown (for Google Chrome, select ‘save as PDF’
-   select the ‘save’ button at the bottom of the popup screen
-   save the relevant page of the ACFI on the desktop or in a folder
    that is easy to access

Repeat the above steps for any other pages of the ACFI and/or for other
care recipients.
