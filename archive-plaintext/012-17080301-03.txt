This page contains information on how an Assessing Officer (AO)
completes an assessment of a MIOC ROCS or ROCS/HCCS initial or
subsequent payment application, or a Section 38 response, to determine
if the payment application meets the relevant eligibility requirements.

Assessing MIOC ROCS or ROCS/HCCS S38 Responses

|                                   |                                   |
|                                   | Supporting Evidence:              |
|                                   |                                   |
|                                   | -   The evidence of the insurer   |
|                                   |     approving indemnity of the    |
|                                   |     insured person can be in the  |
|                                   |     form of a Claims Verification |
|                                   |     Form (CVF), policy schedule   |
|                                   |     or a letter or email from the |
|                                   |     insurer                       |
|                                   | -   The date of incident (DoI)    |
|                                   |     can be identified from court  |
|                                   |     documents, such as the        |
|                                   |     Statement of Claim (SoC),     |
|                                   |     Notice of Claim (NoC), or a   |
|                                   |     letter from the member        |
|                                   |     stating details of the claim  |
|                                   | -   The Australian state,         |
|                                   |     territory or external         |
|                                   |     territory in which the        |
|                                   |     incident took place can be    |
|                                   |     identified from court         |
|                                   |     documents or a letter from    |
|                                   |     the member stating details of |
|                                   |     the claim                     |
|                                   | -   The DoN can be evidenced by   |
|                                   |     date stamped correspondence   |
|                                   |     from the insured person to    |
|                                   |     the insurer, or by a          |
|                                   |     telephone activity log        |
|                                   |     evidencing verbal             |
|                                   |     notification                  |
|                                   | -   The evidence that the insured |
|                                   |     person was a practitioner of  |
|                                   |     an eligible medical           |
|                                   |     practitioner, confirming the  |
|                                   |     member’s specialisation, may  |
|                                   |     be identified from court      |
|                                   |     documents, or other           |
|                                   |     supporting documents, such as |
|                                   |     the CVF or policy schedule    |
|                                   | -   The insurer is required to    |
|                                   |     confirm the insured’s ROCS    |
|                                   |     eligibility and ROCS          |
|                                   |     eligibility class at DoN and  |
|                                   |     must provide a copy of the    |
|                                   |     relevant ROCS contract and a  |
|                                   |     ROCS Statutory Declaration    |
|                                   |     (or other compulsory          |
|                                   |     supporting evidence, for      |
|                                   |     example a death certificate   |
|                                   |     where the insured is          |
|                                   |     deceased)                     |
|                                   | -   The insurer is required to    |
|                                   |     provide the pre-ROCS policy   |
|                                   |     to confirm the details in the |
|                                   |     ROCS policy provided, as ROCS |
|                                   |     policies are based on the     |
|                                   |     last policy in place prior to |
|                                   |     becoming ROCS eligible        |
|                                   |                                   |
|                                   | Questions 14 to 22: Check the     |

|                                   | evidence on the worksheet and     |

|                                   |     for the claim                 |
|                                   | -   there are no special          |
|                                   |     conditions or sublimits       |
|                                   |     applicable to the claim       |
|                                   | -   any policy deductibles have   |
|                                   |     been deducted from the claim, |
|                                   |     or evidence has been provided |

|                                   | the supporting evidence on the    |
