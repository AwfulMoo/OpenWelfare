This page contains a description of the process to retire a Healthcare
Provider Identifier - Organisation (HPI-O) organisation in the
Healthcare Identifiers (HI) Service.

Retire Healthcare Provider Identifier - Organisation

Retire HPI-O in the HI Service

This table describes how to retire a HPI-O organisation in the HI
Service.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Acting on behalf of another       |
|                                   | person + Read more ...            |
|                                   |                                   |
|                                   | Is the Service Officer acting on  |
|                                   | behalf of another person known to |
|                                   | the HI Service (Responsible       |
|                                   | Officer (RO) or Organisation      |
|                                   | Maintenance Officer (OMO)), that  |
|                                   | is, in response to a telephone    |
|                                   | request?                          |
|                                   |                                   |
|                                   | -   Yes, complete Handle 'Act     |
|                                   |     for' call in Healthcare       |
|                                   |     Identifiers (HI) Service      |
|                                   |     procedure, and Go to Step 2   |
|                                   | -   No, Go to Step 2              |
|                                   |                                   |
|                                   | Note: for the purpose of this     |
|                                   | procedure, Service Officers are   |
|                                   | not acting on behalf of another   |
|                                   | individual and will search        |
|                                   | directly for the organisation.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for organisation + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | From eHealth program (EHP),       |
|                                   | perform a search for the          |
|                                   | organisation being retired.       |
|                                   |                                   |
|                                   | See also: Searching for records   |
|                                   | in the Healthcare Identifiers     |
|                                   | Service using the eHealth Program |
|                                   | (EHP).                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Amend action + Read more ...      |
|                                   |                                   |
|                                   | Select Amend in the Action column |
|                                   | of the relevant record.           |
|                                   |                                   |
|                                   | The Amend HPI-O Seed/Network      |
|                                   | Organisation screen is displayed. |
+-----------------------------------+-----------------------------------+
| 4                                 | Select and retire organisation +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Organisation Details tab      |
|                                   | -   Retire                        |
|                                   |                                   |
|                                   | The Organisation Activity Status  |
|                                   | changes to 'Retired'.             |
+-----------------------------------+-----------------------------------+
| 5                                 | Record comments and submit + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Key a comment into the Reason |
|                                   |     for Activity Status change    |
|                                   |     field                         |
|                                   | -   Select Submit                 |
|                                   |                                   |
|                                   | Are there validation errors?      |
|                                   |                                   |
|                                   | -   Yes, correct errors as        |
|                                   |     prompted                      |
|                                   | -   No, the Confirm Amend HPI-O   |
|                                   |     Seed/Network Details screen   |
|                                   |     is displayed                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Save when correct + Read more ... |
|                                   |                                   |
|                                   | If the details are:               |
|                                   |                                   |
|                                   | -   Incorrect, select Edit to     |
|                                   |     return to the Amend HPI-O     |
|                                   |     Seed/Network Organisation     |
|                                   |     screen                        |
|                                   | -   Correct, select Save          |
|                                   |                                   |
|                                   | Note: contact Tier 3 support to   |
|                                   | arrange for a letter of           |
|                                   | notification to be mailed to the  |
|                                   | organisation.                     |
+-----------------------------------+-----------------------------------+
