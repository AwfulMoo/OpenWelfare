This page contains the steps for Assessing and processing changes in
residency status for medical practitioners whose immigration status has
progressed from temporary resident to permanent resident or citizen.

On this page:

Assessing a change in residency status notification

Processing a change in residency status for a medical practitioner
without access to Medicare benefits

Processing a change in residency status for a medical practitioner who
satisfies 19AA with post graduate medical qualification

Processing a change in residency status for a medical practitioner who
satisfies 19AA with 3GA placement

Processing a change in residency status for a medical practitioner who
has access to Medicare benefits but does not satisfy 19AA

Processing a change in residency status for a medical practitioner with
access to Medicare benefits for assisting at operations only approved by
Department of Health and Aged Care

Processing a change in residency status notification from Australian
permanent resident to Australian Citizen

Assessing a change in residency status notification

Table 1: this table describes how to assess a change in residency status
notification from temporary resident to Australian permanent resident or
citizen for a medical practitioner in the Provider Directory System
(PDS).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Ensure correct evidence is        |
|                                   | supplied + Read more ...          |
|                                   |                                   |
|                                   | Once it has been identified on    |
|                                   | the Application for a Medicare    |
|                                   | provider number and, or           |
|                                   | prescriber number for a medical   |
|                                   | practitioner form (HW019) there   |
|                                   | has been a change in the          |
|                                   | residency status of the medical   |
|                                   | practitioner, the following       |
|                                   | evidence is required to amend the |
|                                   | residency status in PDS.          |
|                                   |                                   |
|                                   | Australian Permanent Resident     |
|                                   |                                   |
|                                   | Australian permanent resident     |
|                                   | visa, or                          |
|                                   |                                   |
|                                   | visa email confirmation, or       |
|                                   |                                   |
|                                   | -   VEVO confirmation (print      |
|                                   |     out/screen shot)              |
|                                   | -   visa label in passport and    |
|                                   |     the bio-data (picture) page   |
|                                   |     of their passport             |
|                                   |                                   |
|                                   | Australian Citizenship            |
|                                   |                                   |
|                                   | -   Australian citizenship        |
|                                   |     certificate, or               |
|                                   | -   Bio-data (picture) page of    |
|                                   |     their Australian passport     |
|                                   |                                   |
|                                   | Where the medical practitioner    |
|                                   | has supplied an Australian        |
|                                   | citizenship document or an        |
|                                   | Australian passport and there is  |

|                                   | applicant and request they supply |
|                                   | their permanent residency visa so |
|                                   | PDS can be updated accordingly.   |
|                                   |                                   |
|                                   | PDS must reflect the medical      |
|                                   | practitioner is no longer a       |
|                                   | temporary resident by adding      |
|                                   | comments on the registration to   |
|                                   | reflect the date Australian       |
|                                   | citizenship was obtained.         |
|                                   |                                   |
|                                   | Do not amend the residency status |
|                                   | field on PDS to permanent         |
|                                   | resident until evidence has been  |
|                                   | submitted.                        |
|                                   |                                   |
|                                   | Note: New Zealand passport        |
|                                   | holders who become Australian     |

|                                   | certificate) is used as the       |
|                                   | commencement date of Australian   |
|                                   | citizenship.                      |
|                                   |                                   |
|                                   | If the application is incomplete  |
|                                   | or documents not supplied go to   |
|                                   | Step 2.                           |
|                                   |                                   |
|                                   | If correct documents supplied go  |
|                                   | to Step 3.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Incomplete application + Read     |

Processing a change in residency status for a medical practitioner without access to Medicare benefits

Table 2: this table describes how to process a change in residency
status notification from temporary resident to Australian permanent
resident or citizen without Medicare benefits for a medical practitioner
in PDS.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Update/process change in          |
|                                   | residency on PDS + Read more ...  |
|                                   |                                   |
|                                   | If permanent Australian residency |
|                                   | details are already recorded and  |
|                                   | only citizen details need to be   |
|                                   | recorded see Table 7.             |
|                                   |                                   |
|                                   | If permanent Australian residency |
|                                   | details need to be recorded       |
|                                   | select Edit to update residency   |
|                                   | details:                          |
|                                   |                                   |
|                                   | Select Personal tab               |
|                                   |                                   |
|                                   | Under Residency Details           |
|                                   |                                   |
|                                   | -   Select Australian Permanent   |
|                                   |     Resident from the Residency   |
|                                   |     Status dropdown menu          |
|                                   | -   Key date of permanent         |
|                                   |     residency according to        |
|                                   |     documents supplied            |
|                                   | -   Update Passport Number and    |
|                                   |     Country of Issue if details   |
|                                   |     are provided                  |
|                                   | -   Key Visa Type according to    |
|                                   |     documents supplied            |
|                                   |                                   |
|                                   | Select Specialty codes tab to     |
|                                   | update the Specialty Code         |
|                                   |                                   |
|                                   | -   End date the 181 specialty    |
|                                   |     code the day before permanent |
|                                   |     residency                     |
|                                   | -   Key the 110 specialty code    |
|                                   |     from the date permanent       |
|                                   |     residency was granted         |
|                                   | -   If the 500 specialty code end |
|                                   |     date is in the future, end    |
|                                   |     date 519 specialty code in    |
|                                   |     line with 500 specialty code  |
|                                   | -   If the 500 specialty code is  |
|                                   |     in the past, end the 519      |
|                                   |     specialty code the day before |
|                                   |     permanent residency was       |
|                                   |     granted                       |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | Note: if the update is only to    |
|                                   | residency status, go to Step 2.   |
|                                   | Otherwise, continue processing    |
|                                   | the application.                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Create confirmation letter + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Complete the 19AB-Visa Letter |
|                                   | -   Save letter as a PDF          |
|                                   | -   Upload confirmation letter to |
|                                   |     file in PaNDA                 |
|                                   | -   Ensure the Surname, First     |
|                                   |     Name, Provider Number and     |
|                                   |     Case ID fields are complete   |
|                                   |     and correct                   |
|                                   | -   Print the letter and send to  |
|                                   |     the medical practitioner      |
|                                   | -   Add any appropriate comments  |
|                                   |     to the PaNDA file             |
|                                   | -   Select Complete               |
|                                   | -   Select Request Approved       |
|                                   | -   Select Save or Save and Next, |
|                                   |     as appropriate                |
