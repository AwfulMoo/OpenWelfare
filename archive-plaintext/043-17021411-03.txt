Requesting a waiver/write off and non-recovery of TFES debt

Waiver/write off and non-recovery

Table 4: This table provides the processes for requesting a waiver/write
off from the Department of Infrastructure and Regional Development and
potential non recovery.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Debt waiver + Read more ...       |
|                                   |                                   |
|                                   | []Waiving a debt means            |
|                                   | relinquishing the right to        |
|                                   | recover the overpaid amount. A    |
|                                   | debt waiver can be considered     |
|                                   | where:                            |
|                                   |                                   |
|                                   | -   there has been extraordinary  |
|                                   |     administrative error          |
|                                   | -   the debt is small, unable to  |
|                                   |     be recovered by withholdings  |
|                                   | -   the claimant is bankrupt or   |
|                                   |     under administration          |
|                                   | -   the claimant no longer exists |
|                                   |                                   |
|                                   | Note: debt waivers will rarely be |
|                                   | approved as no future recovery    |
|                                   | action can be taken if the        |
|                                   | claimant’s situation changes.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Debt write off + Read more ...    |
|                                   |                                   |
|                                   | []A debt write off is where       |
|                                   | recovery action for the debt is   |
|                                   | suspended. A written off debt may |
|                                   | be reactivated at a later time if |
|                                   | the claimant’s circumstances      |
|                                   | change.                           |
|                                   |                                   |
|                                   | Examples may include trading out  |
|                                   | of administration or the claimant |
|                                   | begins claiming again after a     |
|                                   | period of time.                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Requesting a waiver or write      |
|                                   | off + Read more ...               |
|                                   |                                   |
|                                   | []Waiver and write off requests   |
|                                   | must be referred to the           |
|                                   | Department of Infrastructure and  |
|                                   | Regional Development.             |
|                                   |                                   |
|                                   | -   the request can be initiated  |
|                                   |     either by the claimant or by  |
|                                   |     the Tasmanian Transport       |
|                                   |     Program                       |
|                                   | -   requests from claimants must  |
|                                   |     be in writing detailing the   |
|                                   |     reasons for waiver/write off, |
|                                   |     together with any supporting  |
|                                   |     evidence                      |
|                                   | -   refer requests for            |
|                                   |     waiver/write off to the SSO   |
|                                   |     or Team Leader for action     |
+-----------------------------------+-----------------------------------+
| 4                                 | Processing a request for a        |
|                                   | waiver/write off + Read more ...  |
|                                   |                                   |
|                                   | []Document the request in         |
|                                   | claimant notes, including:        |
|                                   |                                   |
|                                   | -   Debt ID                       |
|                                   | -   details of any conversations  |
|                                   |     or documentation provided by  |
|                                   |     the claimant, and             |
|                                   | -   any supporting evidence       |
|                                   |     provided or discovered        |
|                                   |                                   |
|                                   | SSO or Team Leader will complete  |
|                                   | a Request to write-off a debt     |
|                                   | form, found in the Financial      |
|                                   | forms index, and forward to the   |
|                                   | Programme Team via e-mail for     |
|                                   | consideration. The Programme Team |
|                                   | will then forward to Department   |
|                                   | of Infrastructure and Regional    |
|                                   | Development for a decision. The   |
|                                   | Resources page contains a link to |
|                                   | the Financial forms index.        |
|                                   |                                   |
|                                   | If approval is received, the Debt |
|                                   | Portfolio Holder will action:     |
|                                   |                                   |
|                                   | -   If an invoice has already     |
|                                   |     been raised, send write off   |
|                                   |     approval to Accounts          |
|                                   |     Receivable by email. They     |
|                                   |     will finalise the account and |
|                                   |     confirm nil recovery. The     |
|                                   |     Debt Portfolio Holder will    |
|                                   |     document the outcome on the   |
|                                   |     claimant record               |
|                                   | -   If no invoice was raised or   |
|                                   |     confirmation received from    |
|                                   |     Accounts Receivable, the Debt |
|                                   |     Portfolio Holder will         |
|                                   |     complete the recovery action  |
|                                   |     on the claimant record and    |
|                                   |     document any action           |
|                                   |     undertaken on the claimant    |
|                                   |     record in the debt comments   |
|                                   |     section                       |
|                                   |                                   |
|                                   | Note: if waiver or write off is   |
|                                   | not approved, see the Raising an  |
|                                   | invoice, checking an account      |
|                                   | status or invoice repayment       |
|                                   | table.                            |
+-----------------------------------+-----------------------------------+
| 5                                 | Non-recovery + Read more ...      |
|                                   |                                   |
|                                   | []If an invoice has been raised,  |
|                                   | Accounts Receivable will monitor  |
|                                   | repayment and issue written       |
|                                   | reminders at appropriate          |
|                                   | intervals. If not recovered they  |
|                                   | will contact Tasmanian Transport  |
|                                   | Program Team to discuss options.  |
|                                   |                                   |
|                                   | If we are awaiting recovery by    |
|                                   | offsetting debt against future    |
|                                   | claims but the claimant fails to  |
|                                   | lodge within 30 days, the Debt    |
|                                   | Portfolio Holder will initiate    |
|                                   | the invoice procedure. See the    |
|                                   | Raising an invoice, checking an   |
|                                   | account status or invoice         |
|                                   | repayment table.                  |
