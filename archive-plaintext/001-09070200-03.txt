This page contains a summary table of the types of activities that can
be considered for the Annual Activity Requirement. A job seeker should
discuss suitable activities with their Employment Services Provider.

List of programs and activities suitable for meeting the Annual Activity Requirement

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Work for the Dole (WFD) + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For job seekers aged 18 to 49     |
|                                   | years with full-time mutual       |
|                                   | obligation requirements, WFD is   |
|                                   | the principal activity job        |
|                                   | seekers may do to meet their      |
|                                   | Annual Activity Requirement       |
|                                   | unless they have:                 |
|                                   |                                   |
|                                   | -   arranged with their provider  |
|                                   |     to meet their Annual Activity |
|                                   |     Requirement through another   |
|                                   |     approved activity that will   |
|                                   |     start at the time they become |
|                                   |     subject to the Annual         |
|                                   |     Activity Requirement, or      |
|                                   | -   undertaken other approved     |
|                                   |     activities at the time they   |
|                                   |     enter the WFD Phase           |
+-----------------------------------+-----------------------------------+
| 2                                 | Defence Force Reserves + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Defence Force Reserves can be     |
|                                   | regarded as an approved activity  |
|                                   | and may include annual training   |
|                                   | camps, programmed courses,        |
|                                   | full-time or part-time work in    |
|                                   | the defence force.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Employment, including part-time   |
|                                   | work and self-employment + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Job seekers are encouraged to |
|                                   |     engage in part-time work as a |
|                                   |     way to maintain skills and    |
|                                   |     increase opportunities to     |
|                                   |     move to full-time employment. |
|                                   |     For some job seekers, for     |
|                                   |     example principal carers,     |
|                                   |     suitable part-time work can   |
|                                   |     fully meet their mutual       |
|                                   |     obligation requirements       |
|                                   | -   Self-employment is an         |
|                                   |     approved activity, however    |
|                                   |     the job seeker must meet the  |
|                                   |     sufficient work test or       |
|                                   |     undertake additional job      |
|                                   |     search in order to fulfil     |
|                                   |     their mutual obligation       |
|                                   |     requirements. For some job    |
|                                   |     seekers, self-employment can  |
|                                   |     meet all mutual obligation    |
|                                   |     requirements                  |
|                                   | -   New Enterprise Incentive      |
|                                   |     Scheme (NEIS) is an approved  |
|                                   |     activity that assists         |
|                                   |     eligible job seekers with     |
|                                   |     training and support to       |
|                                   |     establish a viable business.  |
|                                   |     Participation in NEIS will    |
|                                   |     fully satisfy a job seeker's  |
|                                   |     mutual obligation             |
|                                   |     requirements, if they are     |
|                                   |     still eligible for some       |
|                                   |     income support once the NEIS  |
|                                   |     program commences, and they   |
|                                   |     are paid NEIS Allowance by    |
|                                   |     their provider                |
+-----------------------------------+-----------------------------------+
| 4                                 | Language and literacy             |
|                                   | assistance + Read more ...        |
|                                   |                                   |
|                                   | Job seekers identified as         |
|                                   | requiring assistance with         |
|                                   | language or literacy skills can   |
|                                   | be referred to programmes such as |
|                                   | Skills for Education and          |
|                                   | Employment (SEE) or the Adult     |
|                                   | Migrant English Program (AMEP).   |
|                                   |                                   |
|                                   | If the number of hours            |
|                                   | participating in the language or  |
|                                   | literacy programme are not        |
|                                   | sufficient to meet the Annual     |
|                                   | Activity Requirement, the job     |
|                                   | seeker will need to make up the   |
|                                   | balance of hours they are         |
|                                   | required to undertake by          |
|                                   | participating in WFD or another   |
|                                   | approved activity.                |
|                                   |                                   |
|                                   | While participating in a SEE      |
|                                   | Programme, job seekers will still |
|                                   | generally be required to complete |
|                                   | 20 job searches per month.        |
+-----------------------------------+-----------------------------------+
| 5                                 | Study + Read more ...             |
|                                   |                                   |
|                                   | Job seekers can undertake study   |
|                                   | as an approved activity,          |
|                                   | depending on their circumstances  |
|                                   | and the type of payment they      |
|                                   | receive. For further details, see |
|                                   | relevant links below.             |
|                                   |                                   |
|                                   | -   Part-time or full-time study  |
|                                   |     as an approved activity for   |
|                                   |     JobSeeker Payment (JSP)       |
|                                   | -   JobSeeker Payment (JSP) job   |
|                                   |     seekers returning to (or      |
|                                   |     continuing in) part-time      |
|                                   |     study                         |
|                                   |                                   |
|                                   | Part-time study can be an         |
|                                   | approved activity for a job       |
|                                   | seeker to meet their Annual       |
|                                   | Activity Requirement if the       |
|                                   | course is at a Certificate III    |
|                                   | level or higher. Non-accredited   |
|                                   | education and training and        |
|                                   | Certificate I or II courses will  |
|                                   | not count towards meeting a job   |
|                                   | seeker’s Annual Activity          |
|                                   | Requirement.                      |
|                                   |                                   |
|                                   | Providers can approve short       |
|                                   | courses for a job seeker on the   |
|                                   | condition that the course is:     |
|                                   |                                   |
|                                   | -   vocationally orientated       |
|                                   | -   less than 12 months in        |
|                                   |     duration                      |
|                                   | -   highly likely to provide a    |
|                                   |     job seeker with a             |
|                                   |     qualification that will       |
|                                   |     result in an employment       |
|                                   |     outcome, improve their        |
|                                   |     employability or lead to      |
|                                   |     qualifications in an          |
|                                   |     identified area of skills     |
|                                   |     shortage                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Voluntary work + Read more ...    |
|                                   |                                   |
|                                   | Voluntary work is available as a  |
|                                   | work experience activity for      |
|                                   | certain job seekers to work with  |
|                                   | approved volunteer organisations. |
|                                   | Job seekers aged 60 and over can  |
|                                   | use voluntary work to meet their  |
|                                   | Annual Activity Requirement of 10 |
|                                   | hours per fortnight.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Unpaid work experience + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Job seekers may be able to        |
|                                   | undertake unpaid work experience  |
|                                   | to improve their job prospects    |
|                                   | and get valuable work experience. |
|                                   | This activity is monitored by the |
|                                   | job seeker's Employment Services  |
|                                   | Provider.                         |
+-----------------------------------+-----------------------------------+
| 8                                 | Non-vocational activities + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For stream C job seekers,         |
|                                   | participation for the relevant    |
|                                   | number of hours in non-vocational |
|                                   | assistance and interventions will |
|                                   | meet their Annual Activity        |
|                                   | Requirement. Stream C job seekers |
|                                   | may benefit from 1 of the         |
|                                   | following activities:             |
|                                   |                                   |
|                                   | -   undertaking an assessment     |
|                                   | -   drug and alcohol              |
|                                   |     rehabilitation                |
|                                   | -   intervention                  |
|                                   | -   motivational training         |
|                                   | -   personal development training |
|                                   | -   parenting skills program      |
|                                   | -   self-help and support group   |
|                                   | -   health maintenance program    |
|                                   | -   counselling                   |
|                                   |                                   |
|                                   | Stream A and B job seekers can    |
|                                   | also volunteer to undertake drug  |
|                                   | dependency treatments or          |
|                                   | interventions, which may count    |
|                                   | towards meeting their Annual      |
|                                   | Activity Requirements.            |
+-----------------------------------+-----------------------------------+
