Further information to determine Payee for Living Allowance

More information to assist in determining the Payee for Living Allowance

Table 2: This table describes additional information regarding the Payee
for Living Allowance.

+-----------------------------------+-----------------------------------+
| Item                              | Payee                             |
+-----------------------------------+-----------------------------------+
| 1                                 | Hostel is not signatory to the    |
|                                   | standard hostels agreement + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a student is boarding at a     |
|                                   | hostel that is not signatory to   |
|                                   | the standard hostels agreement,   |
|                                   | and the payee is the hostel, then |
|                                   | payments must only be made on a   |
|                                   | fortnightly basis.                |
|                                   |                                   |
|                                   | See ABSTUDY Standard Hostels      |
|                                   | Agreement for more information.   |
+-----------------------------------+-----------------------------------+
| 2                                 | ABSTUDY boarding instalments of   |
|                                   | Living Allowance + Read more ...  |
|                                   |                                   |
|                                   | If a student is boarding at a     |
|                                   | boarding school or hostel,        |
|                                   | payments must be made direct to   |
|                                   | the boarding school or hostel via |
|                                   | ABSTUDY boarding instalment.      |
|                                   |                                   |
|                                   | See Payment on Advice for ABSTUDY |
|                                   | away from home rate for more      |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Residential Cost Option + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where a student has opted for the |
|                                   | residential cost option, payments |
|                                   | of residential costs are to be    |
|                                   | made direct to the residential    |
|                                   | college or hostel.                |
|                                   |                                   |
|                                   | See Payment of Residential Costs  |
|                                   | Option (RCO) for ABSTUDY and      |
|                                   | ABSTUDY Pensioner Education       |
|                                   | Supplement (PES) for more         |
|                                   | information.                      |
