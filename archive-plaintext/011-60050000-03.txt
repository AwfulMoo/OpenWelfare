Pharmacists dispensing and claiming CTG prescriptions

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Pharmacist dispenses the PBS      |
|                                   | item + Read more ...              |
|                                   |                                   |
|                                   | The pharmacist must submit a CTG  |
|                                   | 6-character value, either         |
|                                   | manually or generated by their    |
|                                   | dispensing software, into the     |
|                                   | software system for CTG patients. |
|                                   |                                   |
|                                   | The pharmacist dispenses the PBS  |
|                                   | item as per normal dispensing     |
|                                   | procedures.                       |
|                                   |                                   |
|                                   | The dispensing software will      |
|                                   | verify a patient’s CTG status in  |
|                                   | real time and calculates the:     |
|                                   |                                   |
|                                   | -   Co-payment to be charged to   |
|                                   |     the patient                   |
|                                   | -   PBS Safety Net amount to      |
|                                   |     count towards the family's    |
|                                   |     Safety Net records            |
|                                   |                                   |
|                                   | Note: eligible pharmacies can     |
|                                   | check a patient’s CTG status      |
|                                   | within HPOS. See Delegate and     |
|                                   | Pharmacy Organisation Steps to    |
|                                   | Access HPOS for Closing the Gap   |
|                                   | on the Resources page.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Non-concessional                  |
|                                   | patient/Concessional patient +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the prescription is for a:     |
|                                   |                                   |
|                                   | -   Non-concessional patient,     |
|                                   |     ensure that the prescription  |
|                                   |     is included in the General    |
|                                   |     bundle for the next PBS claim |
|                                   |     once it has been receipted by |
|                                   |     the patient or agent          |
|                                   | -   Concessional patient, ensure  |
|                                   |     the prescription is included  |
|                                   |     in the Concessional bundle    |
|                                   |     for the next PBS claim once   |
|                                   |     it has been receipted by the  |
|                                   |     patient or agent              |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   RPBS prescriptions must be    |
|                                   |     included in the Repat bundle  |
|                                   |     as per normal                 |
|                                   | -   Do not include under          |
|                                   |     Co-payment prescriptions in   |
|                                   |     the PBS claim bundles         |
|                                   |     (unless, as per normal        |
|                                   |     practice, they are on the     |
|                                   |     same prescription form as     |
|                                   |     claimable items)              |
|                                   | -   Under Co-payment              |
|                                   |     prescriptions are those       |
|                                   |     written for non-concessional  |
|                                   |     patients for PBS items priced |
|                                   |     at the time of supply below   |
|                                   |     the general patient           |
|                                   |     Co-payment, or concessional   |
|                                   |     patients for PBS items priced |
|                                   |     at the time of supply below   |
|                                   |     the concessional patient      |
|                                   |     Co-payment                    |
|                                   | -   The Under Co-payment          |
|                                   |     Additional Surcharge (UCPAS)  |
|                                   |     cannot be charged to patients |
|                                   |     under this program            |
