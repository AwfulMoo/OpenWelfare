This page contains details of Pharmaceutical Benefits Scheme (PBS)
eligibility for residents of Australian dependencies.

PBS eligibility for Australian dependencies

PBS eligibility for residents of Australian dependencies

This table outlines the Pharmaceutical Benefits Scheme (PBS) eligibility
for residents of Australian dependencies.

+-----------------------------------+-----------------------------------+
| Australian dependency             | PBS eligibility status            |
+-----------------------------------+-----------------------------------+
| Cocos (Keeling) Islands           | Residents of Cocos (Keeling)      |
|                                   | Islands are eligible for PBS      |
|                                   | prescriptions supplied in         |
|                                   | Australia if their citizenship or |
|                                   | permanent residency status is     |
|                                   | established.                      |
+-----------------------------------+-----------------------------------+
| Christmas Island                  | Residents of Christmas Island are |
|                                   | eligible for PBS prescriptions    |
|                                   | supplied in Australia if their    |
|                                   | citizenship or permanent          |
|                                   | residency status is established.  |
+-----------------------------------+-----------------------------------+
| Lord Howe Island                  | For the purposes of PBS, Lord     |
|                                   | Howe Island is regarded as part   |
|                                   | of New South Wales.               |
+-----------------------------------+-----------------------------------+
| Norfolk Island                    | For the purposes of PBS, Norfolk  |
|                                   | Island is regarded as part of New |
|                                   | South Wales from 1 July 2016.     |
+-----------------------------------+-----------------------------------+
