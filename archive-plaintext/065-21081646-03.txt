|                                   | Aged Care. They also provide      |
|                                   | information and support customers |
|                                   | to understand their financial     |
|                                   | situation and their options in    |

|                                   |     financial options related to  |
|                                   |     aged care                     |
|                                   |                                   |
|                                   | ACSOs are not financial planners, |
|                                   | advisors or counsellors. They do  |
|                                   | not:                              |
|                                   |                                   |
|                                   | -   advise people about how to    |
|                                   |     invest their money            |
|                                   | -   deal with other agencies or   |
|                                   |     businesses on behalf of       |
|                                   |     people                        |
|                                   | -   make decisions about          |
|                                   |     eligibility for payments, or  |
|                                   |     aged care or update customer  |
|                                   |     income and asset information  |
|                                   |                                   |
|                                   | See Identifying a customer in     |
|                                   | crisis or needing special help if |
|                                   | the customer is in immediate      |
|                                   | financial crisis. It may be       |
|                                   | appropriate to consult with an    |
|                                   | ACSO onsite.                      |

|                                   | Note: the customer must agree to  |
|                                   | the referral to an ACSO.          |
|                                   |                                   |
|                                   | See Identifying a customer in     |
|                                   | crisis or needing special help if |
|                                   | the customer is in immediate      |
|                                   | financial crisis. It may be       |
|                                   | appropriate to consult with an    |
|                                   | ACSO onsite.                      |

Appointment with an ACSO

Table 2

Note: this process is for Aged Care Specialist Officers only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Greet customer + Read more ...    |
|                                   |                                   |
|                                   | Greet the customer, assess and    |
|                                   | discuss interview requirements    |
|                                   | using the ACSO tool.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Make updates + Read more ...      |
|                                   |                                   |
|                                   | Complete any required updates in  |
|                                   | the appropriate system:           |
|                                   |                                   |
|                                   | Siebel (Department of Health)     |
|                                   |                                   |
|                                   | Topics include but not limited    |
|                                   | to:                               |
|                                   |                                   |
|                                   | In depth information on Aged      |
|                                   | Care:                             |
|                                   |                                   |
|                                   | -   Help at home                  |
|                                   | -   Short term care               |
|                                   | -   Aged Care homes               |
|                                   |                                   |
|                                   | My Aged Care online account       |
|                                   | registration                      |
|                                   |                                   |
|                                   | My Aged Care system updates       |
|                                   |                                   |
|                                   | Appoint a My Aged Care            |
|                                   | representative (nominee)          |
|                                   |                                   |
|                                   | Connect customer to aged care     |
|                                   | services (external referral)      |
|                                   |                                   |
|                                   | Aged Care Assessment referral     |
|                                   | (ACAT)                            |
|                                   |                                   |
|                                   | Regional Assessment (RAS)         |
|                                   |                                   |
|                                   | Customer First                    |
|                                   |                                   |
|                                   | Topics include but not limited    |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   Financial Information         |
|                                   | -   Income and Assets updates     |
|                                   | -   Appointing a nominee          |
|                                   | -   Cost of care                  |
|                                   | -   Internal referrals e.g.       |
|                                   |     Social worker or a Complex    |
|                                   |     Assessment Officer            |
+-----------------------------------+-----------------------------------+
| 3                                 | Record details + Read more ...    |
|                                   |                                   |
|                                   | Record details of discussion in   |
|                                   | the ACSO tool and print a copy    |
|                                   | for the customer.                 |
|                                   |                                   |
|                                   | Save record of interview in       |
|                                   | secured folder on shared drive.   |
|                                   |                                   |
|                                   | Finalise any updates required in  |
|                                   | Customer First and Siebel.        |
|                                   |                                   |
|                                   | Document the customer’s Services  |
|                                   | Australia and Seibel records.     |

|                                   |                                   |

|                                   |                                   |
|                                   | (customer/nominee/representative) |
|                                   | -   what office the ACSO is       |
|                                   |     located at, and               |
|                                   | -   reason for contact            |
