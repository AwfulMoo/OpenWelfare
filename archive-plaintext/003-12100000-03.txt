For Services Australia social workers only.

Social work customer contact activities

This table provides information on the requirements for recording social
worker contact activities.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Social worker contact with a      |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | Record the customer contact in    |
|                                   | the Social Work Information       |
|                                   | System.                           |
|                                   |                                   |
|                                   | If the contact results in:        |
|                                   |                                   |
|                                   | -   some action being taken or    |
|                                   |     information regarding the     |
|                                   |     customer's entitlement was    |
|                                   |     discussed/received, record    |
|                                   |     details of the contact on a   |
|                                   |     DOC. Go to Step 2             |
|                                   | -   the social worker making a    |
|                                   |     delegated decision, record    |
|                                   |     the findings of the case in a |
|                                   |     report. See Assessment and    |
|                                   |     report writing                |
+-----------------------------------+-----------------------------------+
| 2                                 | Record contact on a DOC + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Access the DOC field in SWIS.     |
|                                   | This will allow a DOC to be       |
|                                   | written in SWIS and be added      |
|                                   | directly to the customer's record |
|                                   | in Centrelink system.             |
|                                   |                                   |
|                                   | In the DOC title field, the       |
|                                   | default entry is 'SW Contact'. A  |
|                                   | more detailed entry can be        |
|                                   | written to alert Service Officers |
|                                   | to the action taken.              |
|                                   |                                   |
|                                   | Note: if sensitive information    |
|                                   | needs to be recorded and          |
|                                   | retained; the information should  |
|                                   | be stored on SWIS Case notes.     |
|                                   | However, a DOC should also be     |
|                                   | completed to record that further  |
|                                   | information is available on SWIS  |
|                                   | Case notes.                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Record the date and type of       |
|                                   | contact made + Read more ...      |
|                                   |                                   |
|                                   | Examples of contact types:        |
|                                   | telephone contact, office         |
|                                   | interview.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Record the action taken by the    |
|                                   | social worker + Read more ...     |
|                                   |                                   |
|                                   | For example, referral to a        |
|                                   | service, referral to other        |
|                                   | Services Australia staff,         |
|                                   | appointment made to interview     |
|                                   | customer.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Record any action requested of    |
|                                   | the customer + Read more ...      |
|                                   |                                   |
|                                   | For example, to provide           |
|                                   | supporting documentation for a    |
|                                   | claim/review.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Record any follow up action       |
|                                   | required by a Service Officer +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Provide details of action         |
|                                   | required, for example finalising  |
|                                   | an activity on the customer's     |
|                                   | record.                           |
+-----------------------------------+-----------------------------------+
| 7                                 | Record social worker details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Record the:                       |
|                                   |                                   |
|                                   | -   name                          |
|                                   | -   designation                   |
|                                   | -   location (service             |
|                                   |     centre/Smart Centre) of the   |
|                                   |     social worker                 |
+-----------------------------------+-----------------------------------+
| 8                                 | Finalise the DOC + Read more ...  |
|                                   |                                   |
|                                   | The DOC Completion field default  |
|                                   | option is 'Complete the DOC'.     |
|                                   | This will close the DOC.          |
|                                   |                                   |
|                                   | However if the customer issue is  |
|                                   | not resolved at first contact a   |
|                                   | Fast Note needs to be created and |
|                                   | placed on hold and assigned to    |
|                                   | the social worker's inbox.        |
+-----------------------------------+-----------------------------------+
