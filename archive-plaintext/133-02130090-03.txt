How to cancel an account in PRODA

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Help with self service + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The first option is to help the   |
|                                   | user cancel their own account     |
|                                   | online.                           |
|                                   |                                   |
|                                   | Has the user been able to cancel  |
|                                   | their account?                    |
|                                   |                                   |
|                                   | -   Yes, procedure ends here      |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Access user's PRODA account +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Access the PRODA system.          |
|                                   |                                   |
|                                   | Select Inquiry type related to    |
|                                   | the inquiry.                      |
|                                   |                                   |
|                                   | Search for PRODA account.         |
|                                   |                                   |
|                                   | [                                 |
|                                   | \\INTERNAL.DEPT.LOCAL\Shared\NAT\ |
|                                   | SERDELEXCEL\WORKPRODIMP\Operation |
|                                   | Blueprint Migration\RDT Release   |
|                                   | Icons\32w\icon-phone.png]For      |
|                                   | Inquiry type select Professional. |
|                                   |                                   |
|                                   | Search for PRODA account and      |
|                                   | conduct security check.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Cancel account + Read more ...    |
|                                   |                                   |
|                                   | PRODA accounts can be cancelled   |
|                                   | from today or a future date.      |
|                                   |                                   |
|                                   | Is the account to be cancelled    |
|                                   | from today?                       |
|                                   |                                   |
|                                   | -   Yes, on the Manage account    |
|                                   |     screen under Account status,  |
|                                   |     select Cancel account button  |
|                                   | -   No, on the Manage account     |
|                                   |     screen under Account status,  |
|                                   |     enter the future date and     |
|                                   |     then select Cancel account    |
|                                   |     button                        |
|                                   |                                   |
|                                   | Confirmation box is displayed.    |
|                                   |                                   |
|                                   | If a future date needs to be      |
|                                   | adjusted, select the Cancel       |
|                                   | account button and select No in   |
|                                   | the conformation box. This will   |
|                                   | clear the date, enabling a new    |
|                                   | date to be entered.               |
+-----------------------------------+-----------------------------------+
| 4                                 | Reason for cancelling account +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Enter reason for cancelling       |
|                                   | account in the text box:          |
|                                   |                                   |
|                                   | -   at provider's request         |
|                                   | -   include reason why the        |
|                                   |     provider wanted to cancel     |
|                                   |     their account                 |
|                                   |                                   |
|                                   | Select Yes button                 |
|                                   |                                   |
|                                   | Manage account screen will be     |
|                                   | displayed.                        |
|                                   |                                   |
|                                   | Current status will display       |
|                                   | Cancelled.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Advice to account holder + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Advise the account holder that 90 |
|                                   | days after the date of            |
|                                   | cancellation they will not be     |
|                                   | able to reactivate the cancelled  |
|                                   | account and they will need to     |
|                                   | create a new account if they want |
|                                   | to access any PRODA services.     |
