Agency roles

Roles of staff in Centrelink, Child Support and Medicare sites

This category table contains information about child support and family
assistance and the roles of staff in Centrelink, Child Support and
Medicare.

|                                   | their FTB Part A rate             |

|                                   | Interview/advise customers about  |
|                                   | applying for a child support      |

|                                   |     support assessment, you       |
|                                   |     decide whether you want to    |

|                                   |     Support to collect the child  |
