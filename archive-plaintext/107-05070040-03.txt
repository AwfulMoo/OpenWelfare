For Debt Management Officers (DMO) only.

This page contains information on what to do with Bankruptcy Act Part IX
Debt Agreements and Part X Personal Insolvency Agreements (PIA).

New agreements

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Agreement proposal or outcome +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Most agreements will offer a      |
|                                   | 'cents in the dollar' proposal    |
|                                   | paid by instalments.              |
|                                   |                                   |
|                                   | -   If a proposal is received, go |
|                                   |     to Step 2                     |
|                                   | -   If a proposal has been        |
|                                   |     accepted, go to Step 6        |
|                                   | -   If a proposal has been        |
|                                   |     rejected, go to Step 7        |
+-----------------------------------+-----------------------------------+
| 2                                 | Consider the offer + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Received for a:                   |
|                                   |                                   |
|                                   | -   Part IX Debt Agreement, a     |
|                                   |     Statement of Claim and Voting |
|                                   |     Form                          |
|                                   | -   Part X PIA, a Statement of    |
|                                   |     Claim and Proxy Form and a    |
|                                   |     Proof of debt form            |
|                                   |                                   |
|                                   | Does Services Australia need to   |
|                                   | vote?                             |
|                                   |                                   |
|                                   | Creditors will usually get an     |
|                                   | equal share of dividends and the  |
|                                   | agency does not need to vote.     |
|                                   |                                   |
|                                   | The agency will need to vote if   |
|                                   | the debt amount in the proposal   |
|                                   | is a lot different to the debt    |
|                                   | balance.                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Voting + Read more ...            |
|                                   |                                   |
|                                   | -   Complete the voting form.     |
|                                   |     Make sure it confirms the     |
|                                   |     balance of the debt when the  |
|                                   |     form was accepted. This       |
|                                   |     confirms the provable amount  |
|                                   |     is correct on the National    |
|                                   |     Personal Insolvency Index     |
|                                   |     (NPII)                        |
|                                   | -   The APS5 Bankruptcy Officer   |
|                                   |     must authorise the form       |
|                                   | -   Return the form by fax or     |
|                                   |     email to the administrator or |
|                                   |     trustee. See Resources page   |
|                                   |     for contact details           |
+-----------------------------------+-----------------------------------+
| 4                                 | Cease recovery of the debt + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Before write off, record details  |
|                                   | on a separate DOC titled 'Debt    |
|                                   | Agreement or PIA Received'.       |
|                                   |                                   |
|                                   | Include on the DOC:               |
|                                   |                                   |
|                                   | -   administration number         |
|                                   | -   date accepted for processing  |
|                                   | -   administrator and their       |
|                                   |     contact details               |
|                                   | -   voting closing date           |
|                                   | -   amount listed on the          |
|                                   |     statement of claim            |
|                                   | -   amount listed on voting form  |
|                                   |     if returned                   |
|                                   | -   if documents were faxed or    |
|                                   |     emailed to Australian         |
|                                   |     Financial Security Authority  |
|                                   |     (AFSA)                        |
|                                   | -   confirmation documents were   |
|                                   |     scanned to customer file and  |
|                                   |     on which at date              |
|                                   |                                   |
|                                   | For Part X (PIAs) include:        |
|                                   |                                   |
|                                   | -   estate number                 |
|                                   | -   trustee and trustee contact   |
|                                   |     details                       |
|                                   | -   date of appointment           |
|                                   | -   Services Australia included   |
|                                   |     in PIA for $                  |
|                                   | -   date of creditors meeting     |
|                                   | -   the amount listed on          |
|                                   |     Statement of claim            |
|                                   | -   amount listed on voting form  |
|                                   |     if returned                   |
|                                   | -   if documents were faxed back  |
|                                   |     to Trustee                    |
|                                   | -   confirmation documents were   |
|                                   |     scanned to customer file and  |
|                                   |     on which at date              |
+-----------------------------------+-----------------------------------+
| 5                                 | Code a temporary write off + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the Debt recovery script to:  |
|                                   |                                   |
|                                   | Code a temporary write off. This  |
|                                   | allows time for the vote and      |
|                                   | notification of the outcome       |
|                                   |                                   |
|                                   | -   use reason code 'BRP'         |
|                                   |     (bankruptcy pending)          |
|                                   | -   write off for 2 months from   |
|                                   |     when the administrator        |
|                                   |     accepts the proposal or the   |
|                                   |     trustee is appointed          |
|                                   |                                   |
|                                   | Do not issue Q217 letter at this  |
|                                   | stage                             |
|                                   |                                   |
|                                   | On the Add Write Off (OPWA)       |
|                                   | screen, add these details in the  |
|                                   | Comments field:                   |
|                                   |                                   |
|                                   | -   Reason for write off, e.g.    |
|                                   |     Debt Agreement or PIA         |
|                                   |     including number              |
|                                   | -   Debt Agreement dated dd/mm/yy |
|                                   |     awaiting outcome of vote      |
|                                   |                                   |
|                                   | Scan and action all documents     |
|                                   | related to the bankruptcy.        |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Proposal accepted + Read more ... |
|                                   |                                   |
|                                   | Check for any withholding or      |
|                                   | payment after the date the        |
|                                   | administrator received the        |
|                                   | proposal.                         |
|                                   |                                   |
|                                   | Does the customer have more than  |
|                                   | 1 debt?                           |
|                                   |                                   |
|                                   | Yes, do not refund. Transfer the  |
|                                   | payments to a debt in the period  |
|                                   | after bankruptcy                  |
|                                   |                                   |
|                                   | No, refund any amounts received   |
|                                   | after the date accepted for       |
|                                   | processing. For help, see:        |
|                                   |                                   |
|                                   | -   Transfers and refunds         |
|                                   | -   Is a refund of repayments     |
|                                   |     payable to a bankrupt         |
|                                   |     customer?                     |
|                                   |                                   |
|                                   | See Step 1 in Table 3.            |
+-----------------------------------+-----------------------------------+
| 7                                 | Proposal rejected + Read more ... |
|                                   |                                   |
|                                   | Remove any temporary write off    |
|                                   | still in place.                   |
|                                   |                                   |
|                                   | Record details on a DOC. Include  |
|                                   | 'Debt Agreement Terminated or     |
|                                   | Rejected' in the summary line.    |
|                                   |                                   |
|                                   | Send a Q217 letter to the         |
|                                   | customer. This confirms the       |
|                                   | outstanding balance they need to  |
|                                   | pay.                              |
|                                   |                                   |
|                                   | -   Restart recovery. For help,   |
|                                   |     see Negotiation of debt       |
|                                   |     repayments with customers     |
|                                   | -   If the customer has gone into |
|                                   |     bankruptcy, see What to do    |
|                                   |     when a customer is bankrupt   |
|                                   |                                   |
|                                   | Procedure ends here.              |

Current agreements - variation or termination

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Part IX Debt agreements + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A request for variation or        |
|                                   | termination may be lodged with    |
|                                   | the Official Receiver. The        |
|                                   | administrator may request a       |
|                                   | variation on behalf of a customer |
|                                   | or a new creditor. A creditor may |
|                                   | request termination after getting |
|                                   | a default notice.                 |
|                                   |                                   |
|                                   | The agency will not normally make |
|                                   | these requests and does not need  |
|                                   | to vote in these cases. The       |
|                                   | agency will act on the outcome of |
|                                   | the vote.                         |
|                                   |                                   |
|                                   | Debt adjustments                  |
|                                   |                                   |
|                                   | Advise the administrator if the   |
|                                   | debt amount changes. Any change   |
|                                   | in the amount owed will affect    |
|                                   | how much all creditors receive.   |
|                                   |                                   |
|                                   | Agreements accepted for           |
|                                   | processing after 1 July 2007      |
|                                   |                                   |
|                                   | -   Any debt raised after the     |
|                                   |     agreement was accepted, but   |
|                                   |     for a period covered by the   |
|                                   |     agreement, is included in the |
|                                   |     agreement. Voting is not      |
|                                   |     needed. It may trigger a      |
|                                   |     variation                     |
|                                   | -   When a new debt is included,  |
|                                   |     the agency is entitled to a   |
|                                   |     one-off equalisation dividend |
|                                   | -   If a customer falls 3 months  |
|                                   |     in arrears, the administrator |
|                                   |     informs all creditors         |
|                                   | -   If a customer falls 6 months  |
|                                   |     in arrears, the agreement     |
|                                   |     terminates. AFSA advises      |
|                                   |                                   |
|                                   | Agreements accepted for           |
|                                   | processing before 1 July 2007     |
|                                   |                                   |
|                                   | A debt raised after the agreement |
|                                   | was accepted, but for a period    |
|                                   | before that date, may be included |
|                                   | in the agreement. A variation     |
|                                   | request and vote may be needed. A |
|                                   | 'No' vote will mean the agency    |
|                                   | cannot participate in the         |
|                                   | agreement                         |
|                                   |                                   |
|                                   | The agency can make a private     |
|                                   | arrangement with the customer     |
|                                   | outside the agreement. So be      |
|                                   | agreeable when negotiating        |
|                                   |                                   |
|                                   | There is no automatic             |
|                                   | termination. If a customer is in  |
|                                   | significant arrears and an        |
|                                   | administrator advises they are    |
|                                   | not making contributions:         |
|                                   |                                   |
|                                   | -   request a Status Report from  |
|                                   |     the administrator, and        |
|                                   | -   request that a proposal to    |
|                                   |     terminate be put to creditors |
|                                   |     to vote. AFSA will send forms |
+-----------------------------------+-----------------------------------+
| 2                                 | Part X PIAs + Read more ...       |
|                                   |                                   |
|                                   | Creditors, with the debtor's      |
|                                   | written consent, can vary the     |
|                                   | agreement by passing a special    |
|                                   | resolution.                       |
|                                   |                                   |
|                                   | A debtor can ask their trustee to |
|                                   | vary their agreement. The trustee |
|                                   | will send a notice to the         |
|                                   | creditors. If no creditor         |
|                                   | objects, the terms will be        |
|                                   | varied. If any creditor objects   |
|                                   | in writing, a creditors meeting   |
|                                   | can be called to consider the     |
|                                   | proposal.                         |
|                                   |                                   |
|                                   | A court can set aside or          |
|                                   | terminate an agreement in some    |
|                                   | situations.                       |
|                                   |                                   |
|                                   | Creditors can terminate an        |
|                                   | agreement if the trustee is       |
|                                   | satisfied the debtor is not       |
|                                   | complying.                        |
|                                   |                                   |
|                                   | An event can terminate an         |
|                                   | agreement if the agreement says   |
|                                   | it will.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Variation or termination proposal |
|                                   | received + Read more ...          |
|                                   |                                   |
|                                   | Record details on a DOC. Include: |
|                                   |                                   |
|                                   | -   'Debt agreement variation/    |
|                                   |     termination' in the summary   |
|                                   |     line                          |
|                                   | -   debt agreement proposal       |
|                                   |     number                        |
|                                   | -   name of the administrator and |
|                                   |     their contact details         |
|                                   | -   full details of the variation |
|                                   |     or termination                |
|                                   |                                   |
|                                   | The agency does not need to vote  |
|                                   | if creditors are receiving an     |
|                                   | equal share.                      |
|                                   |                                   |
|                                   | Code a manual review (RVR) with   |
|                                   | keywords 'BRT' for 2 months after |
|                                   | the date accepted for processing. |
|                                   |                                   |
|                                   | Ensure documents are scanned to   |
|                                   | the customer's record.            |
+-----------------------------------+-----------------------------------+
| 4                                 | Variation accepted + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Record details on a DOC.          |
|                                   |                                   |
|                                   | Also update the original          |
|                                   | acceptance DOC with new variation |
|                                   | details.                          |
|                                   |                                   |
|                                   | If required:                      |
|                                   |                                   |
|                                   | -   update any changes to the     |
|                                   |     original write off            |
|                                   | -   code relevant reviews         |
+-----------------------------------+-----------------------------------+
| 5                                 | Termination accepted + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Remove any temporary write off    |
|                                   | still in place.                   |
|                                   |                                   |
|                                   | Record details on a DOC. Include  |
|                                   | 'Debt Agreement Terminated or     |
|                                   | Rejected' in the summary line.    |
|                                   |                                   |
|                                   | Send a Q217 letter to the         |
|                                   | customer. This confirms the       |
|                                   | outstanding balance they need to  |
|                                   | pay                               |
|                                   |                                   |
|                                   | -   Restart recovery. For help,   |
|                                   |     see Negotiation of debt       |
|                                   |     repayments with customers     |
|                                   | -   If the customer has gone into |
|                                   |     bankruptcy, see What to do    |
|                                   |     when a customer is bankrupt   |
+-----------------------------------+-----------------------------------+
| 6                                 | Variation or termination          |
|                                   | rejected + Read more ...          |
|                                   |                                   |
|                                   | Record details on a DOC.          |
|                                   |                                   |
|                                   | Await further reports from AFSA   |
|                                   | or the administrator.             |

Finalising debt agreement activities

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Reparation or judgement orders +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Special instructions exist for    |
|                                   | customers with these orders.      |
|                                   |                                   |
|                                   | To check if an order is in place  |
|                                   | or pending:                       |
|                                   |                                   |
|                                   | For debts under $10,000, if there |
|                                   | is no indication that prosecution |
|                                   | is pending or proceeding, go to   |
|                                   | Step 2                            |
|                                   |                                   |
|                                   | For debts $10,000 and over, check |
|                                   | for DOCs about prosecution. If    |
|                                   | DOCs indicate case pending or     |
|                                   | proceeding. or no DOC exists:     |
|                                   |                                   |
|                                   | -   Send an email to the Fraud    |
|                                   |     Policy team                   |
|                                   | -   Include the details of the    |
|                                   |     case to check if reparation   |
|                                   |     order is in place or pending, |
|                                   |     and for the debts covered by  |
|                                   |     it. Make note of the date the |
|                                   |     order was placed              |
|                                   |                                   |
|                                   | Is a reparation order or a        |
|                                   | judgement order in place?         |
|                                   |                                   |
|                                   | Yes, and the order was obtained:  |
|                                   |                                   |
|                                   | -   before the date of            |
|                                   |     bankruptcy, the debt is not   |
|                                   |     recoverable after bankruptcy. |
|                                   |     After the customer is         |
|                                   |     discharged from bankruptcy,   |
|                                   |     record a permanent write-off. |
|                                   |     Recovery will not restart. Go |
|                                   |     to Step 3                     |
|                                   | -   on or after the date of       |
|                                   |     bankruptcy, the order has no  |
|                                   |     bearing on the bankruptcy. Go |
|                                   |     to Step 2                     |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Is the debt due to fraud? + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For help, see Deciding whether a  |
|                                   | debt was due to fraud for a       |
|                                   | bankrupt customer.                |
|                                   |                                   |
|                                   | -   Yes, recovery of the debt     |
|                                   |     will restart after the        |
|                                   |     agreement ends                |
|                                   | -   No, recovery of the debt will |
|                                   |     not restart after the         |
|                                   |     agreement ends                |
+-----------------------------------+-----------------------------------+
| 3                                 | Process the agreement + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | When processing the agreement,    |
|                                   | record details on a DOC titled    |
|                                   | 'Debt Agreement or PIA Accepted'. |
|                                   |                                   |
|                                   | Include all of the below details  |
|                                   | on a DOC:                         |
|                                   |                                   |
|                                   | For Part IX Debt Agreements:      |
|                                   |                                   |
|                                   | -   Debt Agreement Proposal       |
|                                   |     number                        |
|                                   | -   administrator and their       |
|                                   |     contact details               |
|                                   | -   total amount to be            |
|                                   |     distributed to unsecured      |
|                                   |     creditors                     |
|                                   | -   Services Australia included   |
|                                   |     in agreement for $            |
|                                   | -   Services Australia included   |
|                                   |     in voting when returned       |
|                                   | -   date first dividend will be   |
|                                   |     paid to creditors             |
|                                   | -   estimated dividend %          |
|                                   | -   details of any lump sum       |
|                                   |     payments                      |
|                                   | -   frequency of dividends        |
|                                   | -   date customer will complete   |
|                                   |     debt agreement                |
|                                   | -   estimated balance at          |
|                                   |     completion                    |
|                                   | -   all debt IDs covered by the   |
|                                   |     Debt Agreement and their      |
|                                   |     current debt balances         |
|                                   | -   whether or not recovery is to |
|                                   |     recommence after completion   |
|                                   |     of the debt agreement for     |
|                                   |     each debt ID                  |
|                                   | -   determinations for these      |
|                                   |     decisions.                    |
|                                   |                                   |
|                                   | For Part X PIAs                   |
|                                   |                                   |
|                                   | -   estate number                 |
|                                   | -   trustee and trustee contact   |
|                                   |     details                       |
|                                   | -   date of appointment           |
|                                   | -   total creditors $             |
|                                   | -   Services Australia included   |
|                                   |     in agreement for $            |
|                                   | -   proposed dividend to          |
|                                   |     creditors                     |
|                                   | -   date of first dividend        |
|                                   | -   frequency of dividends        |
|                                   | -   term of agreement             |
|                                   | -   details of any lump sum       |
|                                   |     payments                      |
|                                   | -   all debt IDs covered by the   |
|                                   |     PIA and their current debt    |
|                                   |     balances                      |
|                                   | -   whether or not recovery is to |
|                                   |     recommence after completion   |
|                                   |     of the PIA for each debt ID   |
|                                   |     and determinations for these  |
|                                   |     decisions.                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Temporary write off + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Use the Debt recovery script to   |
|                                   | record the temporary write off:   |
|                                   |                                   |
|                                   | -   use the code 'DAC' (debt      |
|                                   |     agreement current)            |
|                                   | -   the write-off is for the      |
|                                   |     period of the agreement       |
|                                   | -   on the Add Write Off (OPWOA)  |
|                                   |     screen, include in the        |
|                                   |     Comments field the reason,    |
|                                   |     e.g. Debt Agreement or PIA    |
|                                   |     including number, dated       |
|                                   |     dd/mm/yy. Do this whether or  |
|                                   |     not the debt can be recovered |
|                                   |     when the agreement ends       |
+-----------------------------------+-----------------------------------+
| 5                                 | Advise the customer of the        |
|                                   | outcome + Read more ...           |
|                                   |                                   |
|                                   | Send the customer a Q217 letter   |
|                                   | to advise them if or when         |
|                                   | recovery will restart.            |
|                                   |                                   |
|                                   | If recovery will restart after    |
|                                   | the agreement ends, in the        |
|                                   | ${AFTER BANKRUPTCY AMT} field     |
|                                   | insert this text:                 |
|                                   |                                   |
|                                   | '$amount less dividends received  |
|                                   | from the administrator'           |
+-----------------------------------+-----------------------------------+
| 6                                 | Record a manual review + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Code a manual review with the     |
|                                   | keyword DMTDIV for 2 weeks after  |
|                                   | the first dividend is due. This   |
|                                   | is to ensure the dividend is      |
|                                   | received.                         |
|                                   |                                   |
|                                   | After each review, code another   |
|                                   | review for the next expected      |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Record a final manual review with |
|                                   | keyword FINALDIV for 2 weeks      |
|                                   | after a the final dividend is     |
|                                   | due.                              |
|                                   |                                   |
|                                   | If the dividend is not received,  |
|                                   | contact the debt agreement        |
|                                   | administrator. If obligations are |
|                                   | not met, the agreement may be     |
|                                   | terminated.                       |
|                                   |                                   |
|                                   | If the agreement is terminated,   |
|                                   | see Negotiation of debt           |
|                                   | repayments with customers.        |
+-----------------------------------+-----------------------------------+
| 7                                 | Receipt of final dividends + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check that all dividends have     |
|                                   | been received against the debt    |
|                                   | agreement.                        |
|                                   |                                   |
|                                   | Record details on a DOC.          |
+-----------------------------------+-----------------------------------+
| 8                                 | After the agreement ends receipt  |
|                                   | of the final dividend + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Was the debt due to fraud?        |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   restart recovery and advise   |
|                                   |     the customer. For help, see   |
|                                   |     Negotiation of debt           |
|                                   |     repayments with customers     |
|                                   | -   send a final Q217 letter to   |
|                                   |     confirm the debt is           |
|                                   |     recoverable                   |
|                                   |                                   |
|                                   | No, use the Debt recovery script  |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   permanently write off the     |
|                                   |     balance of the debt/s covered |
|                                   |     by the agreement. Use the     |
|                                   |     code 'BRD' (bankruptcy        |
|                                   |     discharged - no fraud)        |
|                                   | -   record details on a DOC       |
