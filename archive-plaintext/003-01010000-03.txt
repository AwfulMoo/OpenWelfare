customer is with another person, on speaker phone, or already identified

business conversation. Otherwise, go to the Family and domestic violence
procedure to conduct the risk identification and referral process.

| 5                                 | Principal place of residence +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A principal place of residence is |
|                                   | where a person:                   |
|                                   |                                   |
|                                   | -   regularly lives with a degree |
|                                   |     of settled purpose, and       |
|                                   | -   has a lawful right to live    |
|                                   |                                   |
|                                   | Degree of settled purpose in that |
|                                   | place means:                      |
|                                   |                                   |
|                                   | -   a regular or usual pattern of |
|                                   |     living there, and             |
|                                   | -   settlement with a degree of   |
|                                   |     permanence                    |
|                                   |                                   |
|                                   | Settlement could include, but is  |
|                                   | not limited to, living in a place |
|                                   | for:                              |
|                                   |                                   |
|                                   | -   caring,                       |
|                                   | -   education,                    |
|                                   | -   work or                       |
|                                   | -   medical treatment or          |
|                                   |     condition.                    |
|                                   |                                   |
|                                   | This is not a complete list.      |
|                                   |                                   |
|                                   | A principal place of residence    |
|                                   | can include any of the following, |
|                                   | if the customer regularly lives   |
|                                   | there with a degree of settled    |
|                                   | purpose and has a lawful right to |
|                                   | live there:                       |
|                                   |                                   |
|                                   | -   home that a customer owns     |
|                                   | -   home that a customer rents    |
|                                   |                                   |
|                                   | A caravan or tent can be a        |
|                                   | principal place of residence if a |
|                                   | customer has a right or licence   |
|                                   | to be on the land where the       |
|                                   | caravan is parked or the tent is  |
|                                   | pitched. If the customer has no   |
|                                   | right or licence to park their    |
|                                   | caravan or pitch their tent on    |
|                                   | the land, this does not meet the  |
|                                   | definition of ‘principal place of |
|                                   | residence’.                       |
|                                   |                                   |
|                                   | A person can have more than one   |
|                                   | principal place of residence but  |
|                                   | can only make one claim for the   |
|                                   | same major disaster. For example, |
|                                   | a customer may have a second      |
|                                   | principal place of residence that |
|                                   | they live in during the week as   |
|                                   | it is close to a hospital where   |
|                                   | they receive on-going medical     |
|                                   | treatment.                        |
|                                   |                                   |
|                                   | A person may not have a principal |
|                                   | place of residence if they do not |
|                                   | have a residence they normally    |
|                                   | live in and have a legal right or |
|                                   | license to live in.               |
|                                   |                                   |
|                                   | Holiday homes, second residences  |
|                                   | and investment properties         |
|                                   | generally do not meet the         |
|                                   | definition of a person's          |
|                                   | principal place of residence.     |
|                                   |                                   |
|                                   | The principal place of residence: |
|                                   |                                   |
|                                   | includes all rooms that are fully |
|                                   | enclosed and under a common roof  |
|                                   | space                             |
|                                   |                                   |
|                                   | in most free-standing homes       |
|                                   | includes a garage or a similar    |
|                                   | structure (such as a laundry) if  |
|                                   | it was                            |
|                                   |                                   |
|                                   | -   under the common roof space,  |
|                                   |     or                            |
|                                   | -   attached to the main          |
|                                   |     residence                     |
|                                   |                                   |
|                                   | can include structures apart from |
|                                   | the main residence but on the     |
|                                   | same property if the person       |
|                                   |                                   |
|                                   | -   regularly lives in the place  |
|                                   |     with a degree of settled      |
|                                   |     purpose, and                  |
|                                   | -   has a lawful right to live in |
|                                   |     the place                     |
|                                   |                                   |
|                                   | for high density housing          |
|                                   | (multi-storey unit, apartment     |
|                                   | block or townhouse) is the        |
|                                   | self-contained residence only. It |
|                                   | excludes all common areas the     |
|                                   | person may access (for example,   |
|                                   | staircases, hallways and common   |
|                                   | car parks or basements). However, |
|                                   | it can include common areas that  |
|                                   | form part of the internal make-up |
|                                   | of the self-contained residence   |
|                                   | (for example, common walls and    |
|                                   | ceilings)                         |
|                                   |                                   |
|                                   | Evidence of a customer’s          |
|                                   | principal place of residence is   |
|                                   | not required, unless there are    |
|                                   | concerns about the validity of an |
|                                   | address a customer has provided   |
|                                   | (for example advising they have   |
|                                   | moved into a declared LGA).       |
|                                   |                                   |
|                                   | If there are concerns:            |
|                                   |                                   |
|                                   | Ask for evidence such as:         |
|                                   |                                   |
|                                   | -   utility bills                 |
|                                   | -   current rental agreement      |
|                                   | -   bank statements               |
|                                   | -   driver’s licence              |
|                                   |                                   |
|                                   | See Item 12                       |
|                                   |                                   |
|                                   | A prison or place of detention    |
|                                   | (AGDRP only)                      |
|                                   |                                   |
|                                   | A prison or place of detention is |
|                                   | not a person's principal place of |
|                                   | residence. Although they normally |
|                                   | live there, they do not have a    |
|                                   | license or right to live there.   |
|                                   | This may mean that some prisoners |
|                                   | have no principal place of        |
|                                   | residence.                        |
|                                   |                                   |
|                                   | However, they may be eligible for |
|                                   | AGDRP under other adversely       |
|                                   | affected criteria. For example,   |
|                                   | the prisoner:                     |
|                                   |                                   |
|                                   | -   is the immediate family       |
|                                   |     member of an Australian       |
|                                   |     resident or citizen killed,   |
|                                   |     or                            |
|                                   | -   has been seriously injured    |
|                                   |                                   |
|                                   | This is not relevant for NZ DRP   |
|                                   | due to the nature of the visa     |

|                                   | See the Resources page for        |
|                                   | scenarios and questions to ask    |

|                                   | Evidence                          |
|                                   |                                   |
|                                   | Evidence is not needed, unless    |

| 9                                 | Major damage - flood and rain     |

|                                   |                                   |
|                                   | Floodwater                        |
|                                   |                                   |
|                                   | Floodwater is water flowing along |
|                                   | the ground at floor level.        |
|                                   |                                   |
|                                   | As long as floodwaters have       |
|                                   | entered a person’s principal      |
|                                   | place of residence, at floor      |
|                                   | level through any means, as a     |
|                                   | direct result of the major        |
|                                   | disaster, they meet the criteria  |
|                                   | for major damage.                 |
|                                   |                                   |
|                                   | Rainwater                         |
|                                   |                                   |
|                                   | Rainwater is water that falls as  |
|                                   | rain from the sky.                |
|                                   |                                   |
|                                   | For heavy rainfall to meet the    |
|                                   | major damage criteria, rainwater  |
|                                   | needs to have entered a person’s  |
|                                   | principal place of residence      |
|                                   | through a breach, as a direct     |
|                                   | result of the major disaster. For |
|                                   | example, there must have been     |
|                                   | damage to the residence that      |
|                                   | exposes the interior of the       |
|                                   | residence to the elements.        |
|                                   |                                   |
|                                   | Rainwater that comes through a    |
|                                   | breach:                           |
|                                   |                                   |
|                                   | -   that is supposed to be there, |
|                                   |     for example doorway or window |
|                                   |     (open or closed) - this does  |
|                                   |     not meet the major damage     |
|                                   |     criteria                      |
|                                   | -   that is not supposed to be    |
|                                   |     there, for example, broken    |
|                                   |     tiles, lifted tin,            |
|                                   |     overflowing gutters, hole in  |
|                                   |     the wall – this does meet the |
|                                   |     major damage criteria         |
|                                   |                                   |
|                                   | Note: a breach is a gap in, for   |
|                                   | example, a wall, window or roof   |
|                                   |                                   |
|                                   | Mould                             |
|                                   |                                   |
|                                   | If a customer advises they have   |
|                                   | mould in their principal place of |
|                                   | residence, this does not mean     |
|                                   | they meet the major damage        |
|                                   | criteria. Flood or rainwater must |
|                                   | have entered the customer’s       |
|                                   | principal place of residence, as  |
|                                   | a direct result of the major      |
|                                   | disaster as per Floodwater and    |
|                                   | Rainwater above, for them to meet |
|                                   | this criteria.                    |
|                                   |                                   |
|                                   | Mould growth because of           |
|                                   | atmospheric moisture (high        |
|                                   | humidity) for the period of a     |
|                                   | disaster event does not meet the  |
|                                   | major damage criteria.            |
|                                   |                                   |
