This page contains the procedures on how to identify potential replica
Individual Healthcare Identifier (IHI) and how to resolve replica IHI.

Identify replica non-Medicare enrolled IHIs

|                                   | Log on to the Consumer Directory  |
|                                   | Maintenance System (CDMS).        |
+-----------------------------------+-----------------------------------+
| 2                                 | Select Amend Consumer Details +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   On the CDMS home page, select |

|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Select Healthcare Identifiers +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Healthcare Identifiers     |
|                                   | from the Program field.           |
+-----------------------------------+-----------------------------------+
| 4                                 | Search for the IHI + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Conduct a basic, detailed or      |
|                                   | address search to find the IHI    |
|                                   | for the individual.               |
+-----------------------------------+-----------------------------------+
| 5                                 | Check if the exact match is       |
|                                   | found + Read more ...             |
|                                   |                                   |
|                                   | If an exact match is:             |
|                                   |                                   |
|                                   | -   Found, the Amend Personal     |
|                                   |     Details screen displays with  |
|                                   |     1 or more records displayed   |
|                                   |     in the Matched IHIs section   |
|                                   |     of the screen                 |
|                                   | -   Not found, refine search.     |
|                                   |     Note: to refine search, add   |
|                                   |     or remove search criteria to  |
|                                   |     obtain a match for the        |
|                                   |     correct individual            |
+-----------------------------------+-----------------------------------+
| 6                                 | Select the consumer's record +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Select the required           |
|                                   |     consumer's record             |
|                                   | -   Select Personal               |
|                                   | -   The Amend Personal Details    |
|                                   |     screen displays               |
+-----------------------------------+-----------------------------------+
| 7                                 | Select Report/Resolve and Report  |
|                                   | Replica + Read more ...           |
|                                   |                                   |
|                                   | -   Select Report/Resolve         |
|                                   | -   Select Report Replica         |
|                                   | -   The Report Replica IHI screen |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 8                                 | Add comments and replica flag +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Enter comments into the       |
|                                   |     Recorded Comments field.      |
|                                   |     Example: the demographics of  |
|                                   |     the other individuals         |
|                                   |     incorrectly assigned to the   |
|                                   |     IHI                           |
|                                   | -   Select Add Comment            |
|                                   |                                   |
|                                   | Note: to return to the previous   |
|                                   | page without adding a replica     |
|                                   | flag, select Return.              |
|                                   |                                   |
|                                   | -   To add a replica flag select  |
|                                   |     Validate                      |
|                                   | -   The Confirm Report Replica    |
|                                   |     IHI screen displays           |
+-----------------------------------+-----------------------------------+
| 9                                 | Check details are correct and     |
|                                   | then update + Read more ...       |
|                                   |                                   |
|                                   | Check the details on the Confirm  |
|                                   | Report Replica IHI screen.        |
|                                   |                                   |
|                                   | Are the details correct?          |
|                                   |                                   |
|                                   | -   Yes, select Update to flag    |
|                                   |     the record as a potential     |
|                                   |     replica                       |
|                                   | -   No, select Change to return   |
|                                   |     to the Report Replica IHI     |
|                                   |     screen to make changes        |
