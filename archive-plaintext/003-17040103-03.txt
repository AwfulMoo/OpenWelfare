| 9                                 | Principal place of resident       |
|                                   | sustained major damage as a       |
|                                   | direct result of the disaster +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The following elements establish  |
|                                   | whether a principal place of      |
|                                   | residence has suffered 'major     |
|                                   | damage' when, as a direct result  |
|                                   | of the disaster:                  |
|                                   |                                   |
|                                   | -   at least one-quarter of the   |
|                                   |     interior of the residence has |
|                                   |     been affected by the          |
|                                   |     disaster, or                  |
|                                   | -   the residence has been        |
|                                   |     declared structurally unsound |
|                                   |     as supported by evidence, ◦   |
|                                   |     evidence could include, but   |
|                                   |     is not limited to a builder's |
|                                   |     or engineer's report, or      |
|                                   | -   the residence has been        |
|                                   |     breached, exposing at least   |
|                                   |     one-quarter or more of the    |
|                                   |     rooms and/or surface floor    |
|                                   |     area of the residence of the  |
|                                   |     interior of the residence to  |
|                                   |     the elements, or              |
|                                   | -   the interior of the residence |
|                                   |     has experienced sewerage      |
|                                   |     contamination                 |
|                                   |                                   |
|                                   | 'One-quarter' requires at least   |
|                                   | 25 per cent or more of the        |
|                                   | interior of the residence to be   |
|                                   | affected by the disaster.         |
|                                   |                                   |
|                                   | 'One-quarter' can be calculated   |
|                                   | by assessing either the           |
|                                   | proportion of the number of rooms |
|                                   | affected or the proportion of the |
|                                   | surface floor area of the         |
|                                   | residence.                        |

|                                   | The size of the area affected can |
|                                   | be calculated by assessing either |
|                                   | the proportion of the number of   |
|                                   | rooms affected or surface floor   |
|                                   | area of the residence.            |

|                                   | the floor area that could         |
|                                   | possibly be affected. For         |
|                                   | example, in some circumstances,   |
|                                   | the ground floor of a             |
|                                   | multi-storey home can be          |
|                                   | considered affected.              |
|                                   |                                   |

|                                   | contain a full description of     |
