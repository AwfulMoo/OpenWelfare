Common enquiries

Table 11: this table is to assist Service Officers respond to enquiries
from health professionals.

+-----------------------------------+-----------------------------------+
| Item                              | Common enquiry and response       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: can I claim TCAs and    |
|                                   | GPMP items for the same date of   |
|                                   | service?                          |
|                                   |                                   |
|                                   | Answer: yes, the 2 items do not   |
|                                   | restrict with each other and can  |
|                                   | be claimed for the same day.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: can I claim 2 review    |
|                                   | items (e.g. 2x732) for the same   |
|                                   | date of service?                  |
|                                   |                                   |
|                                   | Answer: yes, you must include     |
|                                   | text with their claim indicating  |
|                                   | that one item 732 was reviewing   |
|                                   | TCAs and the other was to review  |
|                                   | the GPMP                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: if I visit my patients  |
|                                   | who are care recipients in        |
|                                   | residential aged care facilities  |
|                                   | and bill chronic disease          |
|                                   | management services, is there     |
|                                   | anything else I can bill for      |
|                                   | having to travel from my          |
|                                   | consulting rooms?                 |
|                                   |                                   |
|                                   | Answer: there are 3 flag fall     |
|                                   | items. See the factsheets on MBS  |
|                                   | Online for the items that are     |
|                                   | available to your profession and  |
|                                   | for more information.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: I visit my patients who |
|                                   | are care recipients in            |
|                                   | residential aged care facilities  |
|                                   | and bill chronic disease          |
|                                   | management services, can I bill a |
|                                   | flag fall item for each patient I |
|                                   | see?                              |
|                                   |                                   |
|                                   | Answer: no, the flag fall items   |
|                                   | are only valid with associated    |
|                                   | items and are only available for  |
|                                   | the initial attendance at one     |
|                                   | residential aged care facility on |
|                                   | one occasion.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: if I see my patients in |
|                                   | multiple residential aged care    |
|                                   | facilities and perform chronic    |
|                                   | disease management services, can  |
|                                   | I only bill a flag fall item      |
|                                   | once?                             |
|                                   |                                   |
|                                   | Answer: you can actually bill a   |
|                                   | flag fall item for the initial or |
|                                   | first patient you provide         |
|                                   | services to, for each of the      |
|                                   | residential aged care facilities, |
|                                   | per occasion.                     |
