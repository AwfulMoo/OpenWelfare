When to request an updated care costs check

This table outlines what needs to be considered by Service Officers in
processing the notice of past benefits queue before requesting an
updated care costs check.

+-----------------------------------+-----------------------------------+
| Situation                         | Whether or not a care cost check  |
|                                   | can be requested                  |
+-----------------------------------+-----------------------------------+
| A new statement by claimant has   | If 'Yes' has been ticked to care  |
| been completed                    | costs:                            |
|                                   |                                   |
|                                   | -   Yes - an updated care costs   |
|                                   |     check can be requested        |
+-----------------------------------+-----------------------------------+
| A new statement by claimant has   | If 'No' has been ticked to care   |
| been completed                    | costs:                            |
|                                   |                                   |
|                                   | -   No - only the previous care   |
|                                   |     costs amount will be carried  |
|                                   |     forward                       |
+-----------------------------------+-----------------------------------+
| No statement by claimant has been | If a notice of past benefits -    |
| received                          | all services taken is to be       |
|                                   | issued:                           |
|                                   |                                   |
|                                   | -   No - only the previous care   |
|                                   |     costs amount will be carried  |
|                                   |     forward                       |
+-----------------------------------+-----------------------------------+
| A 23A statement has been received | -   No - only the previous care   |
|                                   |     costs amount will be carried  |
|                                   |     forward                       |
+-----------------------------------+-----------------------------------+
