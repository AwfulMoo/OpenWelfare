Check eligibility for treatment services

Table 2: this table describes the process to check a patient’s
eligibility for mental health treatment services if affected by
bushfires (91000 – 91731).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complete Security check + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Note: proceed only if the caller  |
|                                   | meets the security check.         |
|                                   |                                   |
|                                   | Advise the caller that the        |
|                                   | eligibility information provided  |
|                                   | is based on the patient’s         |
|                                   | Medicare claiming history and is  |
|                                   | subject to change if there are    |
|                                   | claims pending.                   |
|                                   |                                   |
|                                   | See Perform security check when a |
|                                   | health professional or their      |
|                                   | staff call about a claim table.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Retrieve the patient’s history +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Key NHSI,CARDNUMBER,PATIENT   |
|                                   |     and press [Enter]             |
|                                   | -   Press [PF9]. The patient PIN  |
|                                   |     is displayed                  |
|                                   |                                   |
|                                   | Note: check history by PIN in all |
|                                   | instances to ensure the complete  |
|                                   | claiming history is returned, not |
|                                   | just the history associated with  |
|                                   | 1 card.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Check for mental health treatment |
|                                   | items on the patient’s history +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Count the number of mental health |
|                                   | treatment services on the         |
|                                   | patient’s claims history since 1  |
|                                   | January.                          |
|                                   |                                   |
|                                   | -   Key                           |
|                                   |     N                             |
|                                   | HSI,PIN,ITEM/91000-91731,SORT/DOS |
|                                   |     and press [Enter]             |
|                                   |                                   |
|                                   | Have less than 10 services been   |
|                                   | provided since 1 January (current |
|                                   | year)?                            |
|                                   |                                   |
|                                   | -   Yes, advise the caller that   |
|                                   |     the patient is eligible for a |
|                                   |     mental health treatment       |
|                                   |     service. The process ends     |
|                                   |     here.                         |
|                                   | -   No, advise the caller that    |
|                                   |     the patient is not eligible   |
|                                   |     for these mental health       |
|                                   |     treatment service items. The  |
|                                   |     process ends here.            |
+-----------------------------------+-----------------------------------+
