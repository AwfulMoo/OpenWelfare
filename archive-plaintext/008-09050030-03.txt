Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Issuing MA005(E)M + Read more ... |
|                                   |                                   |
|                                   | The MA005(E)M is automatically    |
|                                   | issued when the 'Details          |
|                                   | Verified?' field is coded as 'N'  |
|                                   | on the Employment/Training        |
|                                   | Details (ETD) screen.             |
|                                   |                                   |
|                                   | This can occur:                   |
|                                   |                                   |
|                                   | -   at new claim when the         |
|                                   |     customer has not provided     |
|                                   |     proof of employment,          |
|                                   |     voluntary work or job search  |
|                                   |     activities                    |
|                                   | -   for current Mobility          |
|                                   |     Allowance (MOB) customers, if |
|                                   |     changes relating to           |
|                                   |     employment, voluntary work or |
|                                   |     job search activities, and    |
|                                   |     details need to be verified   |
|                                   |                                   |
|                                   | This form is in a letter style,   |
|                                   | and includes a due date for       |
|                                   | return. The details of the        |
|                                   | employer or provider are printed  |
|                                   | on the form.                      |
|                                   |                                   |
|                                   | -   Part A: shows the details of  |
|                                   |     the customer's                |
|                                   |     employer/service provider we  |
|                                   |     have recorded. These details  |
|                                   |     have not been verified        |
|                                   | -   Part B: is to be completed    |
|                                   |     and signed by the             |
|                                   |     employer/service provider -   |
|                                   |     provides details of the       |
|                                   |     employer/service provider and |
|                                   |     the activities undertaken by  |
|                                   |     the customer                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Verifying employment, voluntary   |
|                                   | work or job search activities +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Work details must be provided to  |
|                                   | determine if the customer         |
|                                   | qualifies for the standard or     |
|                                   | higher rate of MOB.               |
|                                   |                                   |
|                                   | Proof of employment/voluntary     |
|                                   | work or job search activities can |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   recent payslips showing the   |
|                                   |     name of the employer and how  |
|                                   |     many hours worked             |
|                                   | -   a letter from the employer    |
|                                   |     stating how many hours worked |
|                                   | -   a copy of a recent attendance |
|                                   |     sheet                         |
|                                   | -   a letter from a welfare,      |
|                                   |     charitable or community       |
|                                   |     organisation stating hours    |
|                                   |     worked                        |
|                                   | -   if self-employed, a letter    |
|                                   |     from an accountant stating    |
|                                   |     hours worked                  |
|                                   | -   if verification of Job Plan   |
|                                   |     activities is found in the    |
|                                   |     customer's income support     |
|                                   |     record, including Disability  |
|                                   |     Employment Services (DES) -   |
|                                   |     Disability Management Service |
|                                   | -   evidence provided by the      |
|                                   |     customer of recent job search |
|                                   |     activities where they do not  |
|                                   |     have a Job Plan. For example, |
|                                   |     volunteer job seekers not     |
|                                   |     receiving an income support   |
|                                   |     payment and accessing         |
|                                   |     Workforce Australia Online.   |
|                                   |     This may include evidence of  |
|                                   |     recent regular fortnightly    |
|                                   |     job search through any        |
|                                   |     appropriate channel uploaded  |
|                                   |     by the customer with their    |
|                                   |     claim for MOB payment.        |
|                                   |     Customers also need to show   |
|                                   |     that they are meeting the     |
|                                   |     travel test                   |
|                                   |                                   |
|                                   | If the customer cannot provide    |
|                                   | proof via any of the above, the   |
|                                   | employer, voluntary work          |
|                                   | organisation or service provider  |
|                                   | may complete Part B of the        |
|                                   | MA005(E)M to verify the details.  |
|                                   |                                   |
|                                   | If the customer is using          |
|                                   | Workforce Australia Online and    |
|                                   | does not have a Job Plan          |
|                                   | negotiated with an employment     |
|                                   | services provider, the onus is    |
|                                   | generally on the customer to      |
|                                   | provide evidence of recent        |
|                                   | regular job search activities to  |
|                                   | support their MOB claim, as well  |
|                                   | as meeting the travel test. Staff |
|                                   | will also need to confirm that    |
|                                   | the customer is registered for    |
|                                   | Workforce Australia Online.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Reissuing MA005(E)M + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Access to the MOB system by       |
|                                   | coding 'SVMOB' in the Nxt: field  |
|                                   | in the ISS system. The ACT: field |
|                                   | will then display (MOB) in        |
|                                   | parentheses.                      |
|                                   |                                   |
|                                   | Go to the Activity List (AL)      |
|                                   | screen in the customer's record.  |
|                                   | 'S'elect the ETR activity.        |
|                                   |                                   |
|                                   | -   'S'elect the required         |
|                                   |     employer on the               |
|                                   |     Employment/Training (ETS)     |
|                                   |     screen                        |
|                                   | -   On the Employment/Training    |
|                                   |     Details (ETD) screen, code    |
|                                   |     'T' in the 'Details           |
|                                   |     Verified?' field              |
|                                   | -   Do not finalise the activity. |
|                                   |     (Do not go to the Assessment  |
|                                   |     Results (AR) screen)          |
|                                   | -   Record details on a DOC       |

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Issuing MA005(T)M + Read more ... |
|                                   |                                   |
|                                   | The MA005(T)M is automatically    |
|                                   | issued when the Details Verified? |
|                                   | field is coded as 'N' on the      |
|                                   | Employment/Training Details (ETD) |
|                                   | screen.                           |
|                                   |                                   |
|                                   | This can occur:                   |
|                                   |                                   |
|                                   | -   at new claim when the         |
|                                   |     customer has not provided     |
|                                   |     proof of training             |
|                                   | -   for current Mobility          |
|                                   |     Allowance (MOB) customers, if |
|                                   |     changes relating to training  |
|                                   |     and details need to be        |
|                                   |     verified                      |
|                                   |                                   |
|                                   | The form is in a letter style and |
|                                   | includes a due date for return.   |
|                                   | The details of the training       |
|                                   | organisation are printed on the   |
|                                   | form.                             |
|                                   |                                   |
|                                   | -   Part A: shows the customer's  |
|                                   |     training provider we have     |
|                                   |     recorded. These details have  |
|                                   |     not been verified             |
|                                   | -   Part B: is to be completed    |
|                                   |     and signed by the training    |
|                                   |     organisation - provides       |
|                                   |     details of the training       |
|                                   |     organisation and the          |
|                                   |     activities undertaken by the  |
|                                   |     customer                      |
|                                   |                                   |
|                                   | The form must be scanned to the   |
|                                   | customer's record on the same day |
|                                   | of receipt.                       |
|                                   |                                   |
|                                   | For current MOB customers, when   |
|                                   | the review is uploaded or scanned |
|                                   | to the record, the MOB/VER        |
|                                   | activity will be taken off the    |
|                                   | Future Activity List (FAL)        |
|                                   | screen. This ensures reminder     |
|                                   | letters are not issued.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Verifying training details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Proof of training can include:    |
|                                   |                                   |
|                                   | -   an enrolment record           |
|                                   | -   a letter from the training    |
|                                   |     organisation showing the      |
|                                   |     number of hours the person is |
|                                   |     undertaking training          |
|                                   |                                   |
|                                   | If the customer cannot provide    |
|                                   | proof, the training organisation  |
|                                   | is to complete Part B of the      |
|                                   | MA005(T)M to verify the training  |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Reissuing MA005(T)M + Read more   |

|                                   |                                   |
|                                   | Go to the Activity List (AL)      |

|                                   | 'S'elect the ETR activity         |

|                                   | -   'S'elect the required         |

|                                   | -   Do not finalise the activity. |
|                                   |     (Do not go to the Assessment  |
|                                   |     Results (AR) screen)          |
|                                   | -   Record details on a DOC       |
