| 1                                 | Evidence requirements for ACCS    |
|                                   | (Temporary Financial Hardship) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers need to submit evidence |
|                                   | before they can submit a combined |
|                                   | CCS/ACCS (Temporary Financial     |
|                                   | Hardship) claim (online or via    |
|                                   | the ACC). Excludes CCS claims     |
|                                   | combined with Family Tax Benefit  |
|                                   | (FTB), Paid Parental Leave (PPL)  |
|                                   | or Dad and Partner pay (DAP).     |
|                                   |                                   |
|                                   | Customers can submit some         |
|                                   | CCS/ACCS claims and ACCS          |
|                                   | (Temporary Financial Hardship)    |
|                                   | applications without uploading    |
|                                   | evidence for:                     |
|                                   |                                   |
|                                   | -   standalone ACCS applications  |
|                                   |     (considered a Change of       |
|                                   |     Circumstances), or            |
|                                   | -   CCS/ACCS claims combined with |
|                                   |     FTB, PPL or DAP               |
|                                   |                                   |
|                                   | For these applications, customers |
|                                   | are advised what evidence is      |
|                                   | required and the due date on      |
|                                   | their receipt page. This is       |
|                                   | deemed a request for information, |
|                                   | so a further request is not       |
|                                   | needed unless additional          |
|                                   | information is required.          |
|                                   |                                   |
|                                   | Although customers are advised    |
|                                   | they have 14 days to provide      |
|                                   | evidence, an extra 8 days is      |
|                                   | provided for postage. Therefore:  |
|                                   |                                   |
|                                   | -   if evidence is not provided   |
|                                   |     within 22 days (14 days + 8   |
|                                   |     days for postage) of          |
|                                   |     submitting their application, |
|                                   | -   the claim will be taken as    |
|                                   |     not having been made (not     |
|                                   |     effective)                    |
|                                   |                                   |
|                                   | The customer will need to submit  |
|                                   | a new application if they wish to |
|                                   | apply for ACCS (Temporary         |
|                                   | Financial Hardship). Payment can  |
|                                   | only be backdated up to 28 days.  |
|                                   |                                   |
|                                   | If a period of reduced evidence   |
|                                   | is approved, where the customer   |
|                                   | or their partner receive an       |
|                                   | applicable government disaster    |
|                                   | payment, they will only need to   |
|                                   | show proof of this payment). No   |
|                                   | further evidence will be required |
|                                   | to demonstrate financial          |
|                                   | hardship. See the Resources page  |
|                                   | for a list of approved events and |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Evidence must be provided:        |
|                                   |                                   |
|                                   | -   for an ACCS (Temporary        |
|                                   |     Financial Hardship)           |
|                                   |     application to be effective,  |
|                                   |     and                           |
|                                   | -   to enable eligibility for     |
|                                   |     ACCS (Temporary Financial     |
|                                   |     Hardship) to be assessed      |
|                                   |                                   |
|                                   | If the customer has experienced   |
|                                   | more than 1 Temporary Financial   |
|                                   | Hardship event, evidence is       |
|                                   | required for all events included  |
|                                   | in the application. If not        |
|                                   | provided for all events, the      |
|                                   | application will be Not           |
|                                   | Effective. This is because the    |
|                                   | application is assessed as a      |
|                                   | whole, and not separated          |
|                                   | depending on evidence provided.   |
|                                   |                                   |
|                                   | Customers may lodge an ACCS       |
|                                   | (Temporary Financial Hardship)    |
|                                   | claim in anticipation of an event |
|                                   | occurring.                        |
|                                   |                                   |
|                                   | If deemed effective, claims       |
|                                   | cannot be finalised prior to the  |
|                                   | event occurring.                  |
|                                   |                                   |
|                                   | Select Documents to display all   |
|                                   | documentation attached to the     |
|                                   | record.                           |
|                                   |                                   |
|                                   | Note: before requesting           |
|                                   | documents, review all evidence    |
|                                   | provided and on the customer's    |
|                                   | record that supports the          |
|                                   | application. This will ensure     |
|                                   | only 1 letter requesting          |
|                                   | documents is sent to the          |
|                                   | customer.                         |
|                                   |                                   |
|                                   | If evidence has been provided or  |
|                                   | is on the customer's record,      |
|                                   | confirm it meets the evidence     |
|                                   | requirements. The Resources page  |
|                                   | lists suitable evidence for each  |
|                                   | event.                            |
|                                   |                                   |
|                                   | As well as evidence of the        |
|                                   | event(s), customers must provide  |
|                                   | a causal link between the event   |
|                                   | and their temporary financial     |
|                                   | hardship. This may include a      |
|                                   | Commonwealth statutory            |
|                                   | declaration and supporting        |
|                                   | evidence of expenses incurred     |
