Start date for lump sum preclusion periods

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | A lump sum payment includes       |
|                                   | arrears of periodic compensation  |
|                                   | for a specific period, plus other |
|                                   | amounts such as an impairment     |
|                                   | payment + Read more ...           |
|                                   |                                   |
|                                   | The start date of the preclusion  |
|                                   | period is:                        |
|                                   |                                   |
|                                   | -   The date of incident,         |
|                                   |     diagnosis or economic loss if |
|                                   |     no periodic payments have     |
|                                   |     been made                     |
|                                   | -   The day after periodic        |
|                                   |     payments ceased if periodic   |
|                                   |     payments have been made       |
+-----------------------------------+-----------------------------------+
| 2                                 | A lump sum payment for redemption |
|                                   | of part of future periodic        |
|                                   | payments is received and ongoing  |
|                                   | periodic compensation continues   |
|                                   | at a reduced rate + Read more ... |
|                                   |                                   |
|                                   | The lump sum preclusion period    |
|                                   | starts on the day the periodic    |
|                                   | compensation payments first       |
|                                   | reduce.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | A customer received a lump sum    |
|                                   | payment for more than one         |
|                                   | compensable event and the         |
|                                   | arbitrator has awarded a          |
|                                   | component for past lost earning   |
|                                   | capacity + Read more ...          |
|                                   |                                   |
|                                   | The preclusion period starts from |
|                                   | the date of the incident          |
|                                   | diagnosis or economic loss if:    |
|                                   |                                   |
|                                   | -   no periodic payments have     |
|                                   |     been made, and                |
|                                   | -   there was no specific date    |
|                                   |     for claim for loss of earning |
|                                   |     capacity                      |
+-----------------------------------+-----------------------------------+
| 4                                 | A customer is receiving ongoing   |
|                                   | periodic compensation payments    |
|                                   | and is awarded a lump sum payment |
|                                   | before the periodic payments      |
|                                   | cease + Read more ...             |
|                                   |                                   |
|                                   | If the customer receives a lump   |
|                                   | sum compensation payment, the     |
|                                   | start date of the preclusion      |
|                                   | period can only be determined     |
|                                   | when the periodic payments cease  |
|                                   | or the rate reduces as a result   |
|                                   | of the lump sum payment.          |
+-----------------------------------+-----------------------------------+
