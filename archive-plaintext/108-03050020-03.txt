| 2                                 | Select the claim activity + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Navigate to Veterans' Affairs     |
|                                   | payments (DVAS) screen in         |
|                                   | Customer Record (CR) or in        |

|                                   | the customer's record in DVA      |
|                                   | View.                             |

|                                   | If not currently on DVA payment   |
|                                   | confirm, correct or delete        |

|                                   | inserting 'Y' and navigate to     |
|                                   | Assessment Result (AR) screen.    |
|                                   |                                   |
|                                   | In Process Direct: In Super Key   |
|                                   | box enter DVA.                    |
|                                   |                                   |
|                                   | Confirm, correct or delete        |
|                                   | incorrect coding of DVA data on   |
|                                   | DVA Payment screen, and code any  |
|                                   | payments that have not been       |
|                                   | recorded for customer or partner. |
|                                   |                                   |
|                                   | To clear the SR247 and/or SR307   |
|                                   | edits in Process Direct:          |
|                                   |                                   |
|                                   | -   Locate the provisional entry  |
|                                   |     on the DVA screen with a      |

|                                   |     next to that entry            |
|                                   | -   Delete provisional entry from |
|                                   |     DVA screen                    |
|                                   |                                   |

|                                   |                                   |
|                                   | To clear the DVA adjustment TDI:  |

|                                   | -   Select the Action box on the  |

| 3                                 | Customer current on DVA payment.  |
|                                   | Reverse Clearance process is      |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | Complete Request for a DVA        |
|                                   | Clearance pro-forma Part A and    |
|                                   | email to DVA Income Support Team. |
|                                   |                                   |
|                                   | The pro-forma is used to advise   |
|                                   | DVA about the Centrelink new      |
|                                   | claim, to request cancellation of |
|                                   | a current DVA payment and obtain  |
|                                   | the following details:            |
|                                   |                                   |
|                                   | -   DVA debt details (if any)     |
|                                   | -   Work Bonus Balance at date    |
|                                   |     prior to Centrelink grant     |
|                                   |     date                          |
|                                   | -   Any grandfathered account     |
|                                   |     based income stream under pre |
|                                   |     1/1/15 rules                  |
|                                   | -   Whether the customer is paid  |
|                                   |     under Transitional rules      |
|                                   |                                   |
|                                   | The date of cancellation for the  |
|                                   | DVA payment is to be negotiated   |
|                                   | with DVA, depending on the        |
|                                   | customer's individual             |
|                                   | circumstances, keeping the        |
|                                   | following principles in mind:     |
|                                   |                                   |
|                                   | If the customer is receiving      |
|                                   | Veteran Payment from DVA:         |
|                                   |                                   |
|                                   | -   Request DVA to cancel the     |
|                                   |     Customer's payment at the end |
|                                   |     of the current DVA            |
|                                   |     entitlement period, and grant |
|                                   |     the Centrelink payment from   |
|                                   |     the following day (i.e. Date  |
|                                   |     Paid To + 1), as the customer |
|                                   |     will receive a lower          |
|                                   |     fortnightly rate with         |
|                                   |     Centrelink than they would    |
|                                   |     with DVA                      |
|                                   | -   It is a Government priority   |
|                                   |     that Veteran Payment          |
|                                   |     customers are not             |
|                                   |     disadvantaged when they move  |
|                                   |     onto a Centrelink income      |
|                                   |     support payment               |
|                                   |                                   |
|                                   | If the customer is receiving      |
|                                   | Service Pension from DVA and will |
|                                   | be granted a Pension by           |
|                                   | Centrelink, the date of DVA       |
|                                   | cancellation would generally be   |
|                                   | backdated to the date that the    |
|                                   | customer claimed the Centrelink   |
|                                   | payment, however the start date   |
|                                   | is usually negotiated with DVA    |
|                                   |                                   |
|                                   | If the customer is receiving      |
|                                   | Service Pension from DVA and will |
|                                   | be granted an Allowance by        |
|                                   | Centrelink, or if Pension is not  |
|                                   | maximum rate, it may be more      |
|                                   | beneficial to the customer to     |
|                                   | request DVA to cancel their       |
|                                   | payment at the end of the current |
|                                   | DVA entitlement period, and grant |
|                                   | the Centrelink payment from the   |
|                                   | following day (i.e. Date Paid     |
|                                   | To + 1) because the customer      |
|                                   | would generally receive a lower   |
|                                   | fortnightly rate with Centrelink  |
|                                   | than they would with DVA          |
