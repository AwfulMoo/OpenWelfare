(CIS), scenarios of portability of Widow Allowance (WA), and links to
the Services Australia Website for information about portability of
payments and travelling overseas with Pharmaceutical Benefit Scheme
(PBS) medicine.

Scenarios - portability of Widows Allowance

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer goes overseas for 5      |
|                                   | weeks                             |
|                                   |                                   |
|                                   | Nicole receives Widow Allowance   |
|                                   | (WA) and is going overseas to     |
|                                   | Canada for 5 weeks.               |
|                                   |                                   |
|                                   | Nicole is paid Pharmaceutical     |
|                                   | Allowance (PhA), Telephone        |
|                                   | Allowance (TAL) (or Pension       |
|                                   | Supplement if at Age Pension age) |
|                                   | and Rent Assistance (RA). Nicole  |
|                                   | will continue to pay rent while   |
|                                   | overseas.                         |
|                                   |                                   |
|                                   | Nicole's WA is portable and       |
|                                   | normal payments will be received  |
|                                   | during this period, including the |
|                                   | add-ons.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer goes overseas for 9      |
|                                   | months                            |
|                                   |                                   |
|                                   | On 23 February, Fiona is going    |
|                                   | overseas to Italy for 9 months.   |
|                                   | Fiona is paid WA, PhA and TAL.    |
|                                   |                                   |
|                                   | Fiona's WA, PhA and TAL (or       |
|                                   | Pension Supplement if at Age      |
|                                   | Pension age) will be paid for up  |
|                                   | to the maximum portability period |
|                                   | of 6 weeks after which the        |
|                                   | payment will be suspended).       |
|                                   |                                   |
|                                   | As Fiona will return to Australia |
|                                   | more than 13 weeks after their    |
|                                   | payment has stopped, payment will |
|                                   | cancel. Fiona must claim a        |
|                                   | different income support payment  |
|                                   | such as JobSeeker Payment (JSP).  |
+-----------------------------------+-----------------------------------+
| 3                                 | Customer leaves Australia to live |
|                                   | in another country                |
|                                   |                                   |
|                                   | Beryl is leaving Australia to     |
|                                   | live in another country. Beryl's  |
|                                   | WA will cancel on departure.      |
|                                   |                                   |
|                                   | If Beryl returns to live in       |
|                                   | Australia, a claim for a          |
|                                   | different income support payment  |
|                                   | and proof of residence will be    |
|                                   | required.                         |
+-----------------------------------+-----------------------------------+
