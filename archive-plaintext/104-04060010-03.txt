|                                   | nominee, PPE or PPU. This         |
|                                   | procedure is for POA only         |
|                                   | arrangements.                     |
|                                   |                                   |
|                                   | Check the Nominee (NOLS) screen   |
|                                   | and Document List (DL) screens    |
|                                   | for a Display on Access (DOA) DOC |
|                                   | for the Power of Attorney. For    |
|                                   | example, POA DOA DOC, PPE DOA DOC |
|                                   | or PPU DOA DOC including level of |
|                                   | authority and time frame.         |
|                                   |                                   |
|                                   | If the attorney is a:             |
|                                   |                                   |
|                                   | -   Nominee, see Disclosing       |
|                                   |     information to a nominee      |
|                                   | -   PPE or PPU, see Person        |
|                                   |     Permitted to Enquire (PPE) or |
|                                   |     Update (PPU)                  |
|                                   | -   POA only, go to Step 2        |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirm proof of identity has     |
|                                   | been provided + Read more ...     |
|                                   |                                   |
|                                   | A photo identification document   |
|                                   | must be provided except if the    |
|                                   | POA has a confirmed identity      |
|                                   | status on record or is a business |
|                                   |                                   |
|                                   | Check the Identity Summary on the |
|                                   | POA's record:                     |
|                                   |                                   |
|                                   | -   If photo identification is    |
|                                   |     sighted, record details in    |
|                                   |     the Identity Confirmation     |
|                                   |     Dashboard in Process Direct   |
|                                   |     and verification for DVS      |
|                                   |                                   |
|                                   | If a POA does not own an approved |
|                                   | photographic identity document,   |
|                                   | contact the Identity Policy       |
|                                   | Helpdesk.                         |
|                                   |                                   |
|                                   | Once provided, record the         |
|                                   | identification in Process Direct  |
|                                   | and annotate the POA DOA DOC to   |
|                                   | state this has now been done.     |
|                                   |                                   |
|                                   | Note: photo identification is not |
|                                   | required if the POA is a business |
|                                   | or an authorised employee.        |
|                                   |                                   |
|                                   | Has photo identification been     |
|                                   | provided?                         |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   If POA is a                   |
|                                   |     business/authorised employee, |
|                                   |     photo identification is not   |
|                                   |     required, go to Step 3        |
|                                   | -   Otherwise, tell the POA they  |
|                                   |     must provide approved photo   |
|                                   |     identification and a copy of  |
|                                   |     the legal document to their   |
|                                   |     local service centre before   |
|                                   |     any information can be        |
|                                   |     accepted or provided.         |
|                                   |     Procedure ends here           |

|                                   | No, go to Step 7                  |

|                                   | decisions need to be made.        |

|                                   |     Tell the attorney they        |

|                                   |     from the other attorneys or   |
|                                   |     they will need the customer   |
|                                   |     to complete and lodge an      |
|                                   |     Authorising a person or       |
|                                   |     organisation to enquire or    |
|                                   |     act on your behalf (SS313)    |
|                                   |     form. DOC the contact.        |

|                                   | Read more ...                     |

|                                   | -   sign and lodge agency forms   |
