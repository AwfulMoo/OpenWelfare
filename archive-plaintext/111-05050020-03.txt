|                                   |                                   |
|                                   | Information in Notes must only be |
|                                   | deleted for the following         |
|                                   | reasons:                          |
|                                   |                                   |
|                                   | -   judgemental statements or     |
|                                   |     personal opinion about the    |
|                                   |     customer or their situation   |
|                                   | -   information not mindful of    |
|                                   |     Freedom of Information (FOI)  |

|                                   | -   abusive or offensive language |

|                                   | Street.                           |
|                                   |                                   |
|                                   | Privacy incident: if there has    |
|                                   | been a breach of privacy, record  |
|                                   | a privacy incident. For more      |
|                                   | information, see the Resources    |
|                                   | page for a link to the Privacy    |
|                                   | and Secrecy intranet page.        |
|                                   |                                   |
|                                   | To request deletion               |
|                                   |                                   |
|                                   | This process is for deleting      |
|                                   | Notes generated and viewable in   |
|                                   | Process Direct only.              |
|                                   |                                   |
|                                   | Complete and send the email       |
|                                   | template on the Resources page.   |
|                                   |                                   |
|                                   | The Operations Design and         |
|                                   | Transformation team will action   |
|                                   | the deletion request within 1     |
|                                   | business day.                     |
|                                   |                                   |
|                                   | If the deletion request is:       |
|                                   |                                   |
|                                   | -   accepted, information will be |
|                                   |     deleted and text left to      |
|                                   |     indicate information has been |
|                                   |     removed                       |
|                                   | -   not accepted, an email will   |
|                                   |     be sent to the staff member   |
|                                   |     for follow-up action.         |
