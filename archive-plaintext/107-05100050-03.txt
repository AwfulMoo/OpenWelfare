Process for authorised Debt Management users to update the IC

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Calculation of an Interest        |
|                                   | Charge + Read more ...            |
|                                   |                                   |
|                                   | The rate of the Interest Charge   |
|                                   | is the 90 day Bank Accepted Bill  |
|                                   | rate published by the Reserve     |
|                                   | Bank of Australia (Interest Rates |
|                                   | and Yields - Money Market -       |
|                                   | Monthly - F1.1), plus an          |
|                                   | additional 7%, as is applied by   |
|                                   | the ATO for tax debts (General    |
|                                   | interest charge (GIC) rates).     |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the RBA and ATO websites. The  |
|                                   | Interest Charge is updated        |
|                                   | quarterly.                        |
|                                   |                                   |
|                                   | The Interest Charge is compounded |
|                                   | on a daily basis and is applied   |
|                                   | to the debt monthly, during the   |
|                                   | period:                           |
|                                   |                                   |
|                                   | -   the debt remains unpaid or    |
|                                   | -   until an acceptable payment   |
|                                   |     arrangement is entered into   |
|                                   |     and                           |
|                                   | -   a payment is made under that  |
|                                   |     arrangement and the           |
|                                   |     arrangement continues to be   |
|                                   |     complied with.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Rate of Interest Charge + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The system calculates and updates |
|                                   | the IC rate based on the 90 day   |
|                                   | Bank Accepted Bill rate, plus     |
|                                   | adding an additional 7% to the    |
|                                   | base interest rate for that day.  |
|                                   | That total is divided by the      |
|                                   | number of days in the calendar    |
|                                   | year.                             |
|                                   |                                   |
|                                   | The base interest rate used       |
|                                   | depends on which quarter of the   |
|                                   | year the day is in. For each day  |
|                                   | in the quarter:                   |
|                                   |                                   |
|                                   | -   1 January to 31 March - use   |
|                                   |     the preceding November base   |
|                                   |     interest rate                 |
|                                   | -   1 April to 30 June - use the  |
|                                   |     preceding February base       |
|                                   |     interest rate                 |
|                                   | -   1 July to 30 September - use  |
|                                   |     the preceding May base        |
|                                   |     interest rate                 |
|                                   | -   1 October to 31 December -    |
|                                   |     use the preceding August base |
|                                   |     interest rate                 |
|                                   |                                   |
|                                   | If the monthly average yield of   |
|                                   | 90 day Bank Accepted Bills for a  |
|                                   | particular month is not published |
|                                   | by the Reserve Bank of Australia  |
|                                   | (RBA) before the beginning of the |
|                                   | relevant quarter, the system      |
|                                   | assumes it is the same as the     |
|                                   | last monthly average yield of 90  |
|                                   | day Bank Accepted Bills published |
|                                   | by the RBA before that month.     |
|                                   |                                   |
|                                   | The base interest rate must be    |
|                                   | rounded to the second decimal     |
|                                   | place (rounding .005 upwards).    |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the current General       |
|                                   | interest charge (GIC) rates.      |
+-----------------------------------+-----------------------------------+
| 3                                 | Process for authorised users to   |
|                                   | update the IC rate + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Debt Management Branch is     |
|                                   | responsible for updating the rate |
|                                   | of interest when it is changed.   |
|                                   | Within a month before each new    |
|                                   | quarter starts, authorised users  |
|                                   | extract the updates that are      |
|                                   | posted on the Reserve Bank of     |
|                                   | Australia (RBA) website.          |
|                                   |                                   |
|                                   | If the rates have changed,        |
|                                   | authorised users update the       |
|                                   | interest rate in:                 |
|                                   |                                   |
|                                   | -   ISIS Reference Data Facility  |
|                                   |     (RDF) tables, for debts       |
|                                   |     stored in DMIS                |
|                                   | -   SAP for debts stored there,   |
|                                   |     such as Child Care Subsidy    |
+-----------------------------------+-----------------------------------+
| 4                                 | Security Access for updating      |
|                                   | Reference Data Facility (RDF)     |
|                                   | tables + Read more ...            |
|                                   |                                   |
|                                   | To update an RDF table, the       |
|                                   | resources RDFTAB and RDFENQ in    |
|                                   | the ICT Security Portal (ISP) are |
|                                   | required. See Resources for a     |
|                                   | link to ISP.                      |
|                                   |                                   |
|                                   | In ISP, select Mainframe          |
|                                   | Applications>REFERENCE DATA       |
|                                   | FACILITY>RDFTAB and RDFENQ. In    |
|                                   | the Reason field, enter:          |
|                                   | 'Required to update DMN tables,   |
|                                   | including interest charge rates   |
|                                   | for DMIS debts'.                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Updating the interest charge      |
|                                   | rates RDF table + Read more ...   |
|                                   |                                   |
|                                   | Open Customer Record, and:        |
|                                   |                                   |
|                                   | -   in the Sys field, key 'RDF'   |
|                                   | -   in the Env field, key 'C'     |
|                                   | -   at the next prompt, key       |
|                                   |     'RDTL'                        |
|                                   |                                   |
|                                   | The Table Listing screen appears. |
|                                   |                                   |
|                                   | -   Search for the required       |
|                                   |     table,                        |
|                                   |     DMN.INTEREST.CHARGE.RATES     |
|                                   | -   Select the table              |
|                                   | -   Read each of definition       |
|                                   |     screens and press [Enter]     |
|                                   |                                   |
|                                   | The row listing for the interest  |
|                                   | rate table displays.              |
|                                   |                                   |
|                                   | -   To add a new record, key 'A'  |
|                                   |     against the line and press    |
|                                   |     [Enter]                       |
|                                   | -   Add a date the new rate is    |
|                                   |     effective from. Format is     |
|                                   |     YYYYMMDD                      |
|                                   | -   Add the base rate. For        |
|                                   |     example, 1.76% is recorded as |
|                                   |     0.0176                        |
|                                   | -   Add the Premium interest      |
|                                   |     rate. For example, 7% is      |
|                                   |     recorded as 0.07. Press       |
|                                   |     [Enter]                       |
|                                   |                                   |
|                                   | The new Interest Charge rates are |
|                                   | updated from the date entered. If |
|                                   | this date is in the past, the     |
|                                   | system can read these changes     |
|                                   | immediately, if an interest       |
|                                   | recalculation is triggered.       |
