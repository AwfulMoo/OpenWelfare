Preventing self service failures

Table 8: this table provides information to staff on preventing self
service failures.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | What can be done to limit failed  |
|                                   | self service reports? + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Self service reports can be       |
|                                   | unsuccessful for various reasons. |
|                                   | On sign up for self service       |
|                                   | reporting, Service Officers       |
|                                   | should attempt to limit the       |
|                                   | amount of unsuccessful reports by |
|                                   | checking the following on the     |
|                                   | customer's record:                |
|                                   |                                   |
|                                   | -   Discuss with the customer at  |
|                                   |     initial registration contact, |
|                                   |     the types of questions asked  |
|                                   |     based on the items outlined   |
|                                   |     in the Job Plan and           |
|                                   |     requirements listed on the    |
|                                   |     Activity and Exemption (AEX)  |
|                                   |     screen. The customer needs to |
|                                   |     be aware of their mutual      |
|                                   |     obligation or participation   |
|                                   |     requirements to make sure     |
|                                   |     they understand the questions |
|                                   |     they may be asked             |
|                                   | -   Encourage customers with an   |
|                                   |     active phone account to       |
|                                   |     register for SMS messaging.   |
|                                   |     This makes sure customers     |
|                                   |     have the correct access to    |
|                                   |     report a change of            |
|                                   |     circumstances                 |
|                                   | -   Encourage customers to        |
|                                   |     register for Online Letters   |
|                                   |     (OLL). This enables customers |
|                                   |     to get their mail within 2    |
|                                   |     days and have quick access to |
|                                   |     their reporting dates         |
|                                   | -   Check customer's Partner      |
|                                   |     Permitted to Enquire (PPE)    |
|                                   |     status. To make sure maximum  |
|                                   |     benefit from self service,    |
|                                   |     reporting customers should be |
|                                   |     encouraged to have PPE set to |
|                                   |     'yes' if they are partnered   |
|                                   | -   Use minimal abbreviations     |
|                                   |     when coding an employer's     |
|                                   |     trading name. When coding     |
|                                   |     employment income for         |
|                                   |     customers, make sure minimal  |
|                                   |     abbreviations are used in     |
|                                   |     employer's trading name etc.  |
|                                   |     Make sure the employer's      |
|                                   |     Australian Business Number    |
|                                   |     (ABN) or Withholding Payer    |
|                                   |     Number (WPN) is correct       |
|                                   | -   Do not use 'dummy' held       |
|                                   |     stimulus activities. The use  |

|                                   |     customers from completing     |
|                                   |     their report via self service |

|                                   |     government departments coming |
