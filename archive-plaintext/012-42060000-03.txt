Table 2: this table describes how to assess the eligibility for medical
practitioners to access the level to Medicare they have requested.
Examples include Medicare benefits, assisting at operations or refer and
request.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identifying speciality codes and  |
|                                   | eligibility to access Medicare    |
|                                   | benefits + Read more ...          |
|                                   |                                   |
|                                   | Check for the speciality codes    |
|                                   | located at the end of the bottom  |
|                                   | of the providerâ€™s stem page.      |
|                                   |                                   |
|                                   | Providers who have an open ended  |
|                                   | speciality code of 110 do not     |
|                                   | have access to Medicare benefits  |
|                                   | other than assisting at           |
|                                   | operations if they do not have    |
|                                   | one of the following open post    |
|                                   | graduate speciality codes:        |
|                                   |                                   |
|                                   | -   between 001-099               |
|                                   | -   132, 130, 530, 532, 540. or   |
|                                   | -   between 801-808, 898 or 899   |
|                                   |                                   |
|                                   | Providers who have an open ended  |
|                                   | specialty 104 and 198 codes have  |
|                                   | access to Medicare benefits when  |
|                                   | they have one of the following    |
|                                   | open post graduate speciality     |
|                                   | codes:                            |
|                                   |                                   |

|                                   |                                   |
|                                   | The Department of Health and Aged |

|                                   | on locations that the health      |
|                                   | practitioner is eligible to work  |
