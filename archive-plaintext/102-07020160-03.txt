Coding study load changes

Coding increased study load for ABSTUDY students

Table 2: This table describes how to update an ABSTUDY student record
where study load has increased.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | New or Existing Course + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the student has increased      |
|                                   | their study load due to:          |
|                                   |                                   |
|                                   | -   commencing a new course, see  |
|                                   |     Coding new study details,     |
|                                   |     procedure ends here           |
|                                   | -   a change to their existing    |
|                                   |     course, go to Step 2          |
+-----------------------------------+-----------------------------------+
| 2                                 | Assessment of when student        |
|                                   | changed study load + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Did the ABSTUDY student change to |
|                                   | a full-time or concessional study |
|                                   | load by Friday of the third week  |
|                                   | in the study period in which the  |
|                                   | study load increased?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the student is only       |
|                                   |     entitled to Living Allowance  |
|                                   |     or ABSTUDY PES from the date  |
|                                   |     they changed to a full-time   |
|                                   |     or concessional student. Go   |
|                                   |     to Step 5                     |
|                                   |                                   |
|                                   | Note: if the change of study was  |
|                                   | outside the first three weeks of  |
|                                   | classes because of circumstances  |
|                                   | beyond the control of the         |
|                                   | student, they are treated the     |
|                                   | same as those who did commence    |
|                                   | full-time or concessional study   |
|                                   | load within the first three       |
|                                   | weeks.                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Eligibility date for Living       |
|                                   | Allowance or ABSTUDY PES payment  |
|                                   | when student changed to a         |
|                                   | full-time or concessional study   |
|                                   | load within the first three weeks |
|                                   | of classes + Read more ...        |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Has the student previously        |
|                                   | studied a full-time or            |
|                                   | concessional study load, and is   |
|                                   | resuming a full-time or           |
|                                   | concessional study load after a   |
|                                   | break?                            |
|                                   |                                   |
|                                   | Yes, and                          |
|                                   |                                   |
|                                   | -   the break was less than or    |
|                                   |     equal to one semester, go to  |
|                                   |     Step 4                        |
|                                   | -   the break was greater than    |
|                                   |     one semester. The student is  |
|                                   |     eligible to Living Allowance  |
|                                   |     or ABSTUDY PES from the first |
|                                   |     day of the study period. Go   |
|                                   |     to Step 5                     |
|                                   |                                   |
|                                   | No, the student is only eligible  |
|                                   | to Living Allowance or ABSTUDY    |
|                                   | PES from the first day of the     |
|                                   | study period. Go to Step 5        |
+-----------------------------------+-----------------------------------+
| 4                                 | Date ABSTUDY student eligible to  |
|                                   | receive Living Allowance or       |
|                                   | ABSTUDY PES + Read more ...       |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The student is eligible to Living |
|                                   | Allowance or ABSTUDY PES from:    |
|                                   |                                   |
|                                   | -   1 January if they commenced   |
|                                   |     the study period between 1    |
|                                   |     January and 31 March, or      |
|                                   | -   1 July if they commenced the  |
|                                   |     study period between 1 July   |
|                                   |     and 31 July, or               |
|                                   | -   the first day of the study    |
|                                   |     period if they commenced the  |
|                                   |     study period outside the      |
|                                   |     above dates                   |
|                                   |                                   |
|                                   | Note: If the student has been in  |
|                                   | receipt of a Social Security      |
|                                   | payment past the date they are    |
|                                   | eligible to receive Living        |
|                                   | Allowance, then Living Allowance  |
|                                   | can only be paid from the date    |
|                                   | the Social Security payment       |
|                                   | ceased.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Update course details for ABSTUDY |
|                                   | student who changed to full-time  |
|                                   | or concessional study load + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The course that is currently      |
|                                   | coded as part-time on the         |
|                                   | Education Course History (EDCH)   |
|                                   | screen needs to be changed to     |
|                                   | full-time. To change existing     |
|                                   | details:                          |
|                                   |                                   |
|                                   | -   On the EDCH screen, 'S'elect  |
|                                   |     the relevant course to be     |
|                                   |     updated                       |
|                                   | -   Update the Participation      |
|                                   |     Status Event Date: field to   |
|                                   |     the date Living Allowance or  |
|                                   |     ABSTUDY PES entitlement is to |
|                                   |     commence                      |
|                                   | -   Code the Participation        |
|                                   |     Status: field as 'FTS' for    |
|                                   |     full-time or the relevant     |
|                                   |     code for concessional study   |
|                                   |     load                          |
|                                   | -   Code 'I' for insert in the    |
|                                   |     Action: field and Press       |
|                                   |     [Enter]. This will now        |
|                                   |     display two entries on the    |
|                                   |     EDCH screen for the current   |
|                                   |     course, one as part-time and  |
|                                   |     one with the updated status   |
|                                   | -   'S'elect the 'FTS' entry that |
|                                   |     was just created and correct  |
|                                   |     the Student Start Date: to    |
|                                   |     the date Living Allowance or  |
|                                   |     ABSTUDY PES entitlement is to |
|                                   |     commence                      |
|                                   | -   If proof of enrolment is      |
|                                   |     required, update the          |
|                                   |     Enrolment Verification: field |
|                                   |     with 'VRR' for verification   |
|                                   |     required. This will be        |
|                                   |     changed to 'VER' when         |
|                                   |     enrolment details are         |
|                                   |     provided. See Proof of        |
|                                   |     enrolment in a course of      |
|                                   |     study for details             |
|                                   | -   Code 'C' for Correct in the   |
|                                   |     Action: field and Press       |
|                                   |     [Enter]                       |
|                                   | -   Select the original 'PTS'     |
|                                   |     entry from the EDCH screen    |
|                                   | -   Correct the Student End Date  |
|                                   |     to the day before the student |
|                                   |     became eligible for Living    |
|                                   |     Allowance or ABSTUDY PES      |
|                                   | -   Code 'C' for Correct in the   |
|                                   |     Action: field and Press       |
|                                   |     [Enter]                       |
|                                   | -   Finalise the activity on the  |
|                                   |     AR screen and record details  |
|                                   |     in a DOC                      |
|                                   |                                   |
|                                   | Note: if the student is eligible  |
|                                   | for Incidentals Allowance and it  |
|                                   | has already been paid to the      |
|                                   | student for the current study     |
|                                   | year as part of the previously    |
|                                   | coded part-time course, the       |
|                                   | system will auto-limit the        |
|                                   | payment of this allowance again.  |
|                                   | However, if the student has not   |
|                                   | yet received the allowance for    |
|                                   | the current study year, an        |
|                                   | Incidentals Allowance payment     |
|                                   | will be generated.                |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | ABSTUDY recipient not eligible    |
|                                   | for Schooling B Award, Tertiary   |
|                                   | Award or Master and Doctorate     |
|                                   | Award + Read more ...             |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The recipient may be eligible for |
|                                   | ABSTUDY assistance under:         |
|                                   |                                   |
|                                   | -   ABSTUDY eligibility criteria  |
|                                   |     for Testing and Assessment    |
|                                   |     Award                         |
|                                   | -   ABSTUDY eligibility criteria  |
|                                   |     for Lawful Custody Award      |
|                                   |                                   |
|                                   | or remain eligible for the        |
|                                   | Part-time Award                   |
|                                   |                                   |
|                                   | Record details of any advice      |
|                                   | given to recipient on a DOC.      |
