CrP scenarios

Income Management extension for VWPR (Crisis Payment) customers

This table describes scenarios of when Income Management may be extended
for VWPR (Crisis Payment) customers.

+-----------------------------------+-----------------------------------+
| Item                              | Scenario                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer moves out of Income      |
|                                   | Management area                   |
|                                   |                                   |
|                                   | Where a customer moves out of an  |
|                                   | Income Management area within     |
|                                   | their 12 month VWPR Income        |
|                                   | Management period, they will      |
|                                   | remain on VWPR Income Management  |
|                                   | as per existing portability       |
|                                   | rules.                            |
|                                   |                                   |
|                                   | If, within the first 13 weeks of  |
|                                   | the customer's VWPR Income        |
|                                   | Management period, the customer   |
|                                   | receives a second or subsequent   |
|                                   | CrP due to release from prison or |
|                                   | psychiatric confinement, the      |
|                                   | customer's end date will be       |
|                                   | extended to 12 months from the    |
|                                   | date of the most recent CrP.      |
|                                   |                                   |
|                                   | Note: this rule can also apply to |
|                                   | customers who no longer reside in |
|                                   | an Income Management area.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Automatically reinstated Income   |
|                                   | Management                        |
|                                   |                                   |
|                                   | When a customer's Income          |
|                                   | Management eligible payment is    |
|                                   | cancelled, and subsequently       |
|                                   | restored within 13 weeks, VWPR    |
|                                   | Youth Income Management is        |
|                                   | automatically reinstated.         |
|                                   |                                   |
|                                   | If at the time Income Management  |
|                                   | is reinstated, the customer had   |
|                                   | been granted a second or          |
|                                   | subsequent CrP due to release     |
|                                   | from prison or psychiatric        |
|                                   | confinement within the last 13    |
|                                   | weeks, their VWPR Income          |
|                                   | Management will be extended for   |
|                                   | 12 months from the date of their  |
|                                   | most recent CrP.                  |
+-----------------------------------+-----------------------------------+
