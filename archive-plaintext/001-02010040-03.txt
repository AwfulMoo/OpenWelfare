|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Government sector agencies are    |
|                                   | not-for-profit and do not need to |
|                                   | supply proof of their status.     |
|                                   |                                   |
|                                   | Community based organisations     |
|                                   | must provide proof they are       |
|                                   | not-for-profit. Funds raised by   |
|                                   | the organisation must not be paid |
|                                   | to any members, even if it stops  |
|                                   | operating.                        |
|                                   |                                   |
|                                   | Treat Australian public schools   |
|                                   | as government sector              |
|                                   | organisations for voluntary work  |
|                                   | organisation assessments.         |
|                                   | However, independent Parent and   |
|                                   | Citizen Associations and private  |
|                                   | schools are not government        |
|                                   | organisations and must provide    |
|                                   | proof.                            |
|                                   |                                   |
|                                   | Acceptable proof includes:        |
|                                   |                                   |
|                                   | -   Proof of membership of a      |
|                                   |     state, territory or regional  |
|                                   |     volunteer centre              |
|                                   | -   Articles of incorporation     |
|                                   | -   Certificate of incorporation  |
|                                   | -   Notification of a name change |
|                                   |     of an incorporated entity     |
|                                   | -   A copy of the organisation's  |
|                                   |     constitution showing how      |

|                                   | -   Proof the organisation is a   |

|                                   |                                   |

|                                   |     the Local Activities Database |

|                                   | more ...                          |
|                                   |                                   |
|                                   | Government sector agencies will   |
|                                   | have appropriate insurance cover  |
|                                   | and do not need to supply         |
|                                   | verification of insurance.        |
|                                   |                                   |
|                                   | Community based voluntary work    |
|                                   | organisations must provide        |
|                                   | verification they have both:      |
|                                   |                                   |

|                                   |     at least $5 million, and      |

|                                   |     workers insurance             |
|                                   |                                   |
|                                   | Public liability insurance covers |
|                                   | the cost of claims made against   |
|                                   | organisations by third parties,   |
|                                   | if their services cause injury or |
|                                   | property damage.                  |
|                                   |                                   |
|                                   | Personal accident/voluntary       |
|                                   | workers insurance covers the cost |
|                                   | of claims against an organisation |
|                                   | by volunteers, due to a work      |
|                                   | accident.                         |
|                                   |                                   |
|                                   | The level of personal             |
|                                   | accident/voluntary workers        |
|                                   | insurance may differ for each     |
|                                   | organisation. Services Australia  |
|                                   | cannot advise what level of cover |
|                                   | is appropriate. If organisations  |
|                                   | ask Services Australia about what |
|                                   | is an appropriate level of cover, |
|                                   | tell them to get professional     |
|                                   | advice.                           |
|                                   |                                   |
|                                   | Documents issued by the insurance |
|                                   | provider are acceptable           |
|                                   | verification if they confirm the: |
|                                   |                                   |
|                                   | -   organisation name             |
|                                   | -   type of insurance(s)          |
|                                   |     purchased                     |
|                                   | -   coverage period, and          |
|                                   | -   coverage amount               |
|                                   |                                   |
|                                   | Acceptable verification includes: |
|                                   |                                   |
|                                   | -   Certificate of Currency       |
|                                   | -   Certificate of Insurance      |
|                                   | -   written or verbal statements  |
|                                   |     from the insurance provider   |
|                                   |                                   |
|                                   | The following do not confirm the  |
|                                   | organisation has paid for or has  |
|                                   | active insurance policies:        |
|                                   |                                   |
|                                   | -   Tax invoices                  |
|                                   | -   Renewal notices               |
|                                   | -   Policy quotes                 |
|                                   | -   Policy schedules, and         |
|                                   | -   Product Disclosure Statements |
|                                   |     (PDS)                         |
|                                   |                                   |
|                                   | They are not acceptable           |
|                                   | verification.                     |
|                                   |                                   |
|                                   | Has the organisation provided all |
|                                   | required supporting verification? |
|                                   |                                   |
