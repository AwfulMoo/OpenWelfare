Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Contact requirements + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The contact requirements differ   |
|                                   | between:                          |
|                                   |                                   |
|                                   | -   new claims, where an employer |
|                                   |     has only submitted the claim  |
|                                   |     once                          |
|                                   | -   claims resubmitted once,      |
|                                   |     where an employer has         |
|                                   |     resubmitted the claim once,   |
|                                   |     after a request for more      |
|                                   |     information, and              |
|                                   | -   claims resubmitted more than  |
|                                   |     once, where an employer has   |
|                                   |     resubmitted the claim 2 or    |
|                                   |     more times after requests for |
|                                   |     more information              |
|                                   |                                   |
|                                   | Determine if a claim has been     |
|                                   | resubmitted by reviewing          |
|                                   | Comments, Service Notes in ADMS   |
|                                   | and Actions taken field of the    |
|                                   | Work allocation list:             |
|                                   |                                   |
|                                   | Comments in ADMS display requests |
|                                   | made to employers for more        |
|                                   | information. A claim has been     |
|                                   | resubmitted if the comments show: |
|                                   |                                   |
|                                   | -   a request for more            |
|                                   |     information has been made and |
|                                   |     the claim status is currently |
|                                   |     Assessment, or                |
|                                   | -   multiple requests for more    |
|                                   |     information                   |
|                                   |                                   |
|                                   | the Actions taken field in the    |
|                                   | Work allocation list will         |
|                                   | indicate if a claim has           |
|                                   | previously been Returned, and     |
|                                   | Resubmitted                       |
|                                   |                                   |
|                                   | Is this a new claim?              |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No, and the claim has been        |
|                                   | resubmitted:                      |
|                                   |                                   |
|                                   | -   once, go to Step 3            |
|                                   | -   more than once, employer      |
|                                   |     contact is not required, for  |
|                                   |     claims resubmitted more than  |
|                                   |     once. Go to Step 4 in Table 3 |
+-----------------------------------+-----------------------------------+
| 2                                 | New claim + Read more ...         |
|                                   |                                   |
|                                   | For new claims, contact is        |
|                                   | required:                         |
|                                   |                                   |
|                                   | -   when requesting further       |
|                                   |     information in ADMS, and      |
|                                   | -   17 business days after the    |
|                                   |     request for further           |
|                                   |     information unless the claim  |
|                                   |     is finalised before           |
|                                   |                                   |
|                                   | Attempt to contact the employer   |
|                                   | on Day 0 and Day 17. If this      |
|                                   | attempt is unsuccessful, a second |
|                                   | attempt is required the next      |
|                                   | business day.                     |
|                                   |                                   |
