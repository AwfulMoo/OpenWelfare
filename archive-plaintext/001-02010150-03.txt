Providing labour market information to customers

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine whether the customer    |
|                                   | requires labour market            |
|                                   | information + Read more ...       |
|                                   |                                   |
|                                   | This can be determined if the     |
|                                   | customer has the following        |
|                                   | questions:                        |
|                                   |                                   |
|                                   | -   What are the employment       |
|                                   |     opportunities/prospects for a |
|                                   |     particular occupation?        |
|                                   | -   What is the future for a      |
|                                   |     career in a particular        |
|                                   |     industry? For example,        |
|                                   |     nursing, architecture,        |
|                                   |     building                      |
|                                   | -   A friend/family member works  |
|                                   |     in hospitality and says there |
|                                   |     are lots of jobs in the       |
|                                   |     hospitality industry. Is this |
|                                   |     true?                         |
|                                   | -   What industry has             |
|                                   |     skill/occupational shortages  |
|                                   |     at the moment?                |
+-----------------------------------+-----------------------------------+
| 2                                 | Advise customer of the relevant   |
|                                   | information and referral          |
|                                   | options + Read more ...           |
|                                   |                                   |
|                                   | -   Advise customer that the      |
|                                   |     information in this area is   |
|                                   |     volatile and varies           |
|                                   | -   Refer customers directly to   |
|                                   |     expert sources. See the       |
|                                   |     Resources page                |
+-----------------------------------+-----------------------------------+
| 3                                 | Record details on a DOC + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Record details of the discussion  |
|                                   | with the customer about labour    |
|                                   | market information on a DOC.      |
+-----------------------------------+-----------------------------------+
