Deceased is Group contact with other people on the card

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Deceased is the Group contact     |
|                                   | with other people on the Medicare |
|                                   | card + Read more ...              |
|                                   |                                   |
|                                   | Where the Group contact is        |
|                                   | deceased, change the group        |
|                                   | contact to another person on the  |
|                                   | Medicare card who is over the age |
|                                   | of 15 years.                      |
|                                   |                                   |
|                                   | On the CDMS Home page:            |
|                                   |                                   |
|                                   | -   select Amend Consumer Details |
|                                   | -   key the Medicare card number  |
|                                   |     in the ID Number field        |
|                                   | -   select Group                  |
|                                   |                                   |
|                                   | The Amend Medicare Card Details   |
|                                   | screen will display.              |
|                                   |                                   |
|                                   | Select the Name/Address tab:      |
|                                   |                                   |
|                                   | -   check the Group Association   |
|                                   |     check box                     |
|                                   | -   key the reference number of   |
|                                   |     the person who will be the    |
|                                   |     new Group contact             |
|                                   | -   select Get Details            |
|                                   |                                   |
|                                   | The Contact name field will       |
|                                   | populate with the person’s name.  |
|                                   |                                   |
|                                   | -   select Validate, Validate,    |
|                                   |     Update                        |
|                                   |                                   |
|                                   | The Amend Medicare Card Details   |
|                                   | screen will display               |
|                                   |                                   |
|                                   | Note: if the people who remain on |
|                                   | the Medicare card are under the   |
|                                   | age of 15 years of age, you must  |
|                                   | check if they are on another      |
|                                   | Medicare card.                    |
|                                   |                                   |
|                                   | -   If they are not enrolled on   |
|                                   |     another card they will become |
|                                   |     the group contact             |
|                                   | -   If they are on another card,  |
|                                   |     end the association for the   |
|                                   |     child on this enrolment       |
+-----------------------------------+-----------------------------------+
| 2                                 | Add date of death + Read more ... |
|                                   |                                   |
|                                   | On the Amend Medicare Card        |
|                                   | details screen:                   |
|                                   |                                   |
|                                   | -   select Personal tab           |
|                                   |                                   |
|                                   | The Amend Personal Details screen |
|                                   | displays.                         |
|                                   |                                   |
|                                   | select the radio button to select |
|                                   | the customer to be amended        |
|                                   |                                   |
|                                   | key the date of death in the Date |
|                                   | of Death field                    |
|                                   |                                   |
|                                   | select the source in the Death    |
|                                   | Notification Source field         |
|                                   |                                   |
|                                   | select Comments tab and record    |
|                                   | the notifying person’s:           |
|                                   |                                   |
|                                   | -   name                          |
|                                   | -   relationship                  |
|                                   | -   Medicare number (if card      |
|                                   |     details are known)            |
|                                   |                                   |
|                                   | Select Validate, Validate, Update |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalise changes + Read more ...  |
|                                   |                                   |
|                                   | The Family Safety Net             |
|                                   | registration must be updated      |
|                                   | where the deceased is registered  |
|                                   | as the Family Contact (FC).       |
|                                   |                                   |
|                                   | The Service Officer is to change: |
|                                   |                                   |
|                                   | -   FC to the Spouse (SP), and    |
|                                   | -   SP to FC                      |
|                                   |                                   |
|                                   | Note: do not remove the deceased  |
|                                   | person from the Family Safety Net |
|                                   | registration. They are            |
|                                   | automatically removed at the end  |
|                                   | of the calendar year.             |
|                                   |                                   |
|                                   | For more information see Family   |
|                                   | registration and confirmation for |
|                                   | Medicare Safety Net.              |
+-----------------------------------+-----------------------------------+

Deceased is the only person on the card

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Amend consumer details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where the deceased is the only    |
|                                   | person on the Medicare card the   |
|                                   | Group contact must be updated     |
|                                   | before a date of death can be     |
|                                   | entered in Personal. The Group    |
|                                   | contact’s name is changed to EST  |
|                                   | OF followed by the person’s name. |
|                                   |                                   |
|                                   | On the CDMS Home page:            |
|                                   |                                   |
|                                   | -   select Amend Consumer Details |
|                                   | -   key the Medicare card number  |
|                                   |     in the ID Number field        |
|                                   | -   select Group                  |
|                                   |                                   |
|                                   | The Amend Personal Details screen |
|                                   | displays.                         |
|                                   |                                   |
|                                   | -   select Name/Address tab       |
|                                   | -   untick the Group Association  |
|                                   |     button                        |
|                                   | -   remove the Contact Reference  |
|                                   |     Number                        |
|                                   | -   select Est of                 |
|                                   |                                   |
|                                   | This will change the Group        |
|                                   | contact to Est Of and an          |
|                                   | Information Message will appear.  |
|                                   |                                   |
|                                   | -   OK the Message and Validate,  |
|                                   |     Validate, Update              |
|                                   |                                   |
|                                   | The Amend Medicare Card Details   |
|                                   | screen will display.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Add date of death in Personal     |
|                                   | Details tab + Read more ...       |
|                                   |                                   |
|                                   | On the Amend Medicare Card        |
|                                   | details screen:                   |
|                                   |                                   |
|                                   | -   select Personal tab           |
|                                   |                                   |
|                                   | The Amend Personal Details screen |
|                                   | displays.                         |
|                                   |                                   |
|                                   | -   key the date of death in the  |
|                                   |     Date of Death field           |
|                                   | -   select the source in the      |
|                                   |     Death Notification Source     |
|                                   |     field                         |
|                                   | -   select Comments tab and       |
|                                   |     record the notifying person’s |
|                                   |     name and Medicare number (if  |
|                                   |     known)                        |
|                                   | -   select Validate, Validate,    |
|                                   |     Update                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalise changes + Read more ...  |
|                                   |                                   |
|                                   | The Family Safety Net             |
|                                   | registration must be updated      |
|                                   | where the deceased is registered  |
|                                   | as the Family Contact (FC).       |
|                                   |                                   |
|                                   | The Service Officer is to change: |
|                                   |                                   |
|                                   | -   FC to the Spouse (SP), and    |
|                                   | -   SP to FC                      |
|                                   |                                   |
|                                   | Note: do not remove the deceased  |
|                                   | person from the Family Safety Net |
|                                   | registration as they are          |
|                                   | automatically removed at the end  |
|                                   | of the calendar year.             |
|                                   |                                   |
|                                   | For more information see Family   |
|                                   | registration and confirmation for |
|                                   | Medicare Safety Net.              |
