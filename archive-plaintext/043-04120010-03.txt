|                                   | Read more ...                     |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Note: an organisation record can  |
|                                   | only be added by first conducting |
|                                   | a search. Nominee organisation    |
|                                   | records can only be added in the  |
|                                   | 'B' environment.                  |
|                                   |                                   |
|                                   | A nominee organisation must have  |
|                                   | a unique nominee record. Where an |
|                                   | organisation has an existing      |
|                                   | record, such as a Trust and       |
|                                   | Company (TAC) or Centrepay record |

|                                   |                                   |

|                                   | Details of the nominee            |
|                                   | organisation will be found in the |
|                                   | Authorising a person or           |
|                                   | organisation to act or enquire on |
|                                   | your behalf (SS313) form.         |
|                                   |                                   |
|                                   | Go to Step 4.                     |

|                                   | organisation record + Read more   |

|                                   |     the Name field.               |

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | OL screen + Read more ...         |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The OL screen displays any        |
|                                   | matches found.                    |
|                                   |                                   |
|                                   | Have any matches been found?      |
|                                   |                                   |
|                                   | Note: for nominee organisations   |
|                                   | ensure any potential match has a  |
|                                   | 'NOM' stakeholder code before     |
|                                   | selecting it.                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | From the OL screen + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Select the organisation by typing |
|                                   | 'S' on the line next to the       |

|                                   | [Enter].                          |
|                                   |                                   |
|                                   | Is the organisation selected a    |
|                                   | correct match?                    |
|                                   |                                   |
|                                   | -   Yes, a new organisation       |
|                                   |     record is not required.       |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 3              |

| 3                                 | Input the full name of the        |
|                                   | organisation in the Org Name      |
|                                   | field + Read more ...             |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Search again. If:                 |
|                                   |                                   |
|                                   | the correct record is found, no   |
|                                   | further action is required.       |
|                                   | Procedure ends here               |
|                                   |                                   |
|                                   | no record exists, add the         |
|                                   | organisation to the system:       |
|                                   |                                   |
|                                   | -   in the Nxt field key 'AO'     |
|                                   | -   press [Enter]                 |
|                                   | -   go to Step 4                  |

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | []                                |

|                                   |                                   |

|                                   | nominee organisations.            |
|                                   |                                   |
