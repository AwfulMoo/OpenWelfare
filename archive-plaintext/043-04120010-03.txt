|                                   |                                   |

|                                   | -   Yes, go to Step 2             |

+-----------------------------------+-----------------------------------+
| 2                                 | From the OL screen + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Select the organisation by typing |
|                                   | 'S' on the line next to the       |
|                                   | appropriate match and press       |
|                                   | [Enter].                          |
|                                   |                                   |
|                                   | Is the organisation selected a    |
|                                   | correct match?                    |
|                                   |                                   |
|                                   | -   Yes, a new organisation       |

|                                   | -   No, go to Step 3              |

| 3                                 | Input the full name of the        |

|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Search again. If:                 |
|                                   |                                   |
|                                   | the correct record is found, no   |
|                                   | further action is required.       |
|                                   | Procedure ends here               |
|                                   |                                   |
|                                   | no record exists, add the         |
|                                   | organisation to the system:       |
|                                   |                                   |
|                                   | -   in the Nxt field key 'AO'     |
|                                   | -   press [Enter]                 |
|                                   | -   go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Add Organisation (AO) screen +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The legal name of the company or  |
|                                   | trust (as keyed into the ON       |
|                                   | screen) is displayed at the top   |
|                                   | of the screen.                    |
|                                   |                                   |
|                                   | Check name meets requirements for |
|                                   | nominee organisations.            |
|                                   |                                   |
|                                   | Key the external reference number |

|                                   | following fields:                 |
|                                   |                                   |
|                                   | -   ABN field                     |
|                                   | -   ACN field                     |
