Table 2

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Switching to the partner's        |
|                                   | record + Read more ...            |
|                                   |                                   |
|                                   | If the surviving partner was a    |
|                                   | joint owner of any assets with    |
|                                   | their late partner, full          |
|                                   | ownership of the assets legally   |
|                                   | transfers to the surviving        |
|                                   | partner following the death of    |
|                                   | their partner. However, for       |
|                                   | social security purposes, if      |
|                                   | assets are owned in joint names   |
|                                   | with the deceased partner, the    |
|                                   | surviving partner is only         |
|                                   | assessed as owning 50%. The       |
|                                   | system will continue to asses     |
|                                   | that the surviving partner owns   |
|                                   | 50% of the asset until the        |
|                                   | surviving partner advises of the  |
|                                   | transfer of assets. Where the     |
|                                   | transfer of the deceased's assets |
|                                   | and/or income to the survivor is  |
|                                   | likely to affect the survivor's   |
|                                   | rate, code a manual review for 16 |
|                                   | weeks following the date of       |
|                                   | death.                            |
|                                   |                                   |
|                                   | Check whether assets owned by the |
|                                   | deceased or superannuation,       |
|                                   | compensation or other payments    |
|                                   | made to the deceased will be      |
|                                   | transferred to the surviving      |
|                                   | partner. They should not be       |
|                                   | assessed until they become        |
|                                   | available to the surviving        |
|                                   | partner.                          |
|                                   |                                   |
|                                   | If the surviving partner is       |
|                                   | granted a reversionary income     |
|                                   | stream, it is to be assessed from |
|                                   | the date that it is first         |
|                                   | entitled to be received. A lump   |
|                                   | sum arrears payment received by   |
|                                   | the reversionary beneficiary,     |
|                                   | accrued between the death of the  |
|                                   | primary beneficiary and when the  |
|                                   | reversionary beneficiary          |
|                                   | commences to get regular          |
|                                   | payments, is assessed as income   |
|                                   | from the income stream.           |
|                                   | This means the income is assessed |
|                                   | over the period to which it       |
|                                   | relates and may result in a debt  |
|                                   | for that period.                  |
|                                   | Effective from 9 May 2018, where  |
|                                   | bereavement provisions apply and  |
|                                   | the assessment of lump sum        |
|                                   | arrears result in a debt during   |
|                                   | the bereavement period for the    |
|                                   | surviving spouse, it can be       |
|                                   | waived under the legislative      |
|                                   | instrument 'Social Security       |
|                                   | (Waiver of Debts - Bereavement    |
|                                   | Period) Specification 2017' made  |
|                                   | under subsection 1237AB(1) of the |
|                                   | Social Security Act 1991.         |
|                                   | Note: any debt that is incurred   |
|                                   | after the Act's 14 week           |
|                                   | bereavement period has ended      |
|                                   | would not be waived under the     |
|                                   | provisions of this legislative    |
|                                   | instrument. The legislative       |
|                                   | instrument is not retrospective.  |
