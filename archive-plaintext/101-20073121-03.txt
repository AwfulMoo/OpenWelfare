For Income Management Contact Officers (IMCOs) only.

This page contains information to assist the Income Management Contact
Officer (IMCO), when a request is received from the Family
responsibilities Commission via the Business Hub, to commence a customer
on the Cashless Debit Card program.

On this page:

Processing request for the Cashless Debit Card from Family
Responsibilities Commission

Personal Information Disclosure via Business Hub

Notice to Commence Cashless Debit Card

Extending period of Cashless Debit Card

Notice to Revoke the Cashless Debit Card

View a Cape York Cashless Debit Card customer's expense allocations and
a Personal Information Disclosure (PID)

|                                   | rejection will be displayed       |
|                                   | advising why an 'inactive' notice |
|                                   | could not be created.             |
|                                   |                                   |

|                                   | Processing fails, manual          |
|                                   | processing must be completed:     |

|                                   |     meets eligibility under the   |
|                                   |     Cape York Initiative. If the  |
|                                   |     customer is not eligible for  |
|                                   |     Cape York Cashless Debit      |
