This page contains information to assist the Income Management Contact
Officer (IMCO), when a request is received from the Family
responsibilities Commission via the Business Hub, to commence a customer
on the enhanced IM program.

Notice to Commence enhanced Income Management

Table 3: This table describes how an Income Management Contact Officer
(IMCO) is to action a notice to commence enhanced IM

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Actioning a Notice to Commence    |
|                                   | enhanced IM + Read more ...       |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | On receipt of a Notice to         |
|                                   | Commence Income Management or     |
|                                   | enhanced Income Management the    |
|                                   | system will automatically create  |
|                                   | an 'inactive' notice on the       |
|                                   | customer record and place an      |
|                                   | automatic DOC on the record       |
|                                   | stating a notice has been         |
|                                   | received. When this occurs, the   |
|                                   | Status of the notice in the       |
|                                   | Business Hub will set to 'auto    |
|                                   | processing'.                      |
|                                   |                                   |
|                                   | If Auto Processing fails and an   |
|                                   | 'inactive' notice cannot be       |
|                                   | created, the Status of the notice |
|                                   | in the Business Hub will be set   |
|                                   | to 'unprocessed' and a reason for |
|                                   | rejection will be displayed       |
|                                   | advising why an 'inactive' notice |
|                                   | could not be created.             |
|                                   |                                   |
|                                   | In circumstances where Auto       |
|                                   | Processing fails, manual          |
|                                   | processing must be completed:     |
|                                   |                                   |
|                                   | -   search the TPISO screen for   |
|                                   |     an 'unprocessed' Notice to    |
|                                   |     Commence Income Management    |
|                                   | -   launch the Enhanced Income    |
|                                   |     Management (Cape York) Notice |
|                                   |     Details Screen and record an  |
|                                   |     'inactive' notice on the Cape |
|                                   |     York Notice Details screen    |
|                                   | -   Update the status of the      |
|                                   |     Notice to Commence Income     |
|                                   |     Management or enhanced Income |
|                                   |     Management via the TPISO      |
|                                   |     screen from 'unprocessed' to  |
|                                   |     ‘pending'                     |
|                                   | -   Identify if the customer      |
|                                   |     meets eligibility for         |
|                                   |     enhanced Income Management    |
|                                   |     under the Cape York           |
|                                   |     Initiative. If the customer   |
|                                   |     is not eligible for Cape York |
|                                   |     enhanced Income Management,   |
|                                   |     update the status of the      |
|                                   |     Notice to Commence Income     |
|                                   |     Management via the TPISO      |
|                                   |     screen from 'unprocessed' to  |
|                                   |     'processed'                   |
|                                   |                                   |
|                                   | Go to Step 2.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Record outcome of the enhanced IM |
|                                   | decision + Read more ...          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Do not record outcomes until a    |
|                                   | formal decision about enhanced    |
|                                   | Income Management has been made   |
|                                   | (i.e. customer has been switched  |
|                                   | on or a decision has been made    |
|                                   | not to apply enhanced IM for      |
|                                   | customer, for example, not        |
|                                   | eligible).                        |
|                                   |                                   |
|                                   | Once the decision has been made,  |
|                                   | select the 'Yes' or 'No' option   |
|                                   | for the question 'Switch on?'.    |
|                                   |                                   |
|                                   | Access the TPISO screen again and |
|                                   | select the relevant notice from   |
|                                   | the list of search results. This  |
|                                   | will display the Commence IM for  |
|                                   | Cape York screen where the Record |
|                                   | Outcomes section of the notice is |
|                                   | to be completed.                  |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   the Third Party Information   |
|                                   |     Summary (TPIS) screen can be  |
|                                   |     accessed from the customer's  |
|                                   |     record. Any activity that     |
|                                   |     appears on the TPISO screen   |
|                                   |     that is relevant to a         |
|                                   |     specific customer will be     |
|                                   |     displayed on that customer's  |
|                                   |     TPIS screen                   |
|                                   | -   If the customer has been on   |
|                                   |     CDC or eIM in the NT          |
|                                   |     previously and TCU was their  |
|                                   |     card issuer of choice, TCU    |
|                                   |     will be defaulted as the card |
|                                   |     issuer when the customer is   |
|                                   |     switched on to enhanced IM in |
|                                   |     Cape York                     |
|                                   | -   The following message will    |
|                                   |     displayed on the CY Notice    |
|                                   |     Screen: ‘The customer’s       |
|                                   |     current card issuer is        |
|                                   |     Traditional Credit Union’ to  |
|                                   |     indicate the customer is with |
|                                   |     TCU and will remain with TCU  |
|                                   |     unless they elect to switch   |
|                                   |     card issuers                  |
|                                   |                                   |
|                                   | Has the customer been placed on   |
|                                   | to enhanced IM?                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 3                                 | Customer has been placed on to    |
|                                   | enhanced IM + Read more ...       |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | The customer's enhanced IM status |
|                                   | will be automatically updated on  |
|                                   | the TPISO screen, and the notice  |
|                                   | changed from 'unprocessed' to     |
|                                   | 'pending' or 'processed'.         |
|                                   |                                   |
|                                   | This automated processing will    |
|                                   | send an advice outcome to the     |
|                                   | referring authority via the daily |
|                                   | report advising of the change in  |
|                                   | enhanced IM status.               |
|                                   |                                   |

|                                   | successful the 'unprocessed'      |
|                                   | notice will need to be manually   |
|                                   | updated on the TPISO screen.      |
|                                   |                                   |
|                                   | -   Scroll to the bottom of the   |
|                                   |     Notice to the Record outcome  |
|                                   |     section                       |
|                                   | -   Select 'Yes' for question     |
|                                   |     'Has the Client been switched |
|                                   |     on to Income Management'?     |
|                                   | -   Go to Step 4                  |
|                                   |                                   |
|                                   | Note: this sends a commencement   |
|                                   | advice outcome to the referring   |
|                                   | authority advising of the         |
|                                   | decision to switch the customer   |
|                                   | on to the enhanced IM program.    |
+-----------------------------------+-----------------------------------+
| 4                                 | enhanced IM start date + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where a customer has a New Cape   |
|                                   | York enhanced IM Notice applied   |

|                                   | applied. If the customer still    |
|                                   | has access to an active           |
|                                   | SmartCard, their start date can   |
|                                   | be brought forward.               |
|                                   |                                   |
|                                   | The FRC will ask the customer if  |
|                                   | they have an active card in their |
|                                   | possession when they are          |
|                                   | notifying them of their New       |
|                                   | notice and will relay this        |
|                                   | information to Services Australia |
|                                   | to support bringing forward their |
|                                   | start date.                       |
|                                   |                                   |
|                                   | If the customer does not have an  |
|                                   | active card in their possession,  |
|                                   | the FRC will advise the customer  |
|                                   | to attend their preferred Agent   |
|                                   | site or service centre to get a   |
|                                   | temporary card. The customer can  |
|                                   | then request to bring forward     |
|                                   | their start date by contacting a  |
|                                   | service centre or the SmartCard   |
|                                   | eIM hotline.                      |
|                                   |                                   |
|                                   | For further information on        |
|                                   | changing an enhanced IM start     |
|                                   | date, see enhanced Income         |
|                                   | Management - changing initial     |
|                                   | waiting periods.                  |
|                                   |                                   |
|                                   | For further information on        |
|                                   | temporary SmartCards, see         |
|                                   | Temporary SmartCards.             |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer has not been placed on   |
|                                   | to the enhanced IM program + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Select the 'unprocessed' notice   |
|                                   | from the TPISO screen.            |
|                                   |                                   |
|                                   | Scroll to the bottom of the       |
|                                   | notice to the Record outcome      |
|                                   | section.                          |
|                                   |                                   |
|                                   | Answer the question to 'Has the   |
|                                   | Client been switched on to Income |
|                                   | Management under the Cape York    |
|                                   | Initiative'?                      |

| 1                                 | A Continuation Notice + Read more |
|                                   | ...                               |
