Add person - Error Response reasons

Table 2

+-----------------------------------+-----------------------------------+
| Response Reason                   | Short Description                 |
+-----------------------------------+-----------------------------------+
| No match                          | The ATO was unable to match the   |
|                                   | Centrelink customer to an ATO     |
|                                   | client                            |
+-----------------------------------+-----------------------------------+
| Mismatch                          | The ATO finds the Centrelink CRN  |
|                                   | has already been matched to an    |
|                                   | ATO client with different         |
|                                   | personal details                  |
+-----------------------------------+-----------------------------------+
| Multi-ID person                   | The ATO finds that the Centrelink |
|                                   | customer has already been matched |
|                                   | to an ATO client for a different  |
|                                   | CRN                               |
+-----------------------------------+-----------------------------------+
| Multi-match                       | The ATO finds multiple possible   |
|                                   | ATO client matches for the        |
|                                   | Centrelink customer               |
+-----------------------------------+-----------------------------------+
| Invalid value                     | The ATO has found a personal      |
|                                   | detail supplied in the request is |
|                                   | not in a valid format             |
+-----------------------------------+-----------------------------------+
