For staff conducting Earned Income Interventions - Mail (EIIM) only

This document outlines how the Circumstance Review Action script is used
to action Earned Income Interventions (Mail based) (EIIM). The script
continues and finalises an EIIM including sending out the required
notices to third parties to request verification. All intervention
actions should be processed with the Circumstance Review Action script
where available.

The Resources page contains a link to DOC templates for Earned Income
Interventions (Mail-based) (EIIM).
