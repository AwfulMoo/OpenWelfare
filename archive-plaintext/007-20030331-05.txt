Flexible PPL declaration

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Flexible PPL declaration          |
|                                   |                                   |
|                                   | The Flexible PPL declaration must |
|                                   | be made by the customer each time |
|                                   | they claim a Flexible PPL day. It |
|                                   | is available in the online        |
|                                   | service and guided procedure.     |
|                                   |                                   |
|                                   | 'I declare:                       |
|                                   |                                   |
|                                   | -   The information I have given  |
|                                   |     is correct                    |
|                                   | -   I need to advise Services     |
|                                   |     Australia of any changes to   |
|                                   |     my circumstances as soon as   |
|                                   |     the change occurs             |
|                                   | -   I have read and accept the    |
|                                   |     privacy statement             |
|                                   |                                   |
|                                   | For each day I have claimed       |
|                                   | Flexible Paid Parental Leave      |
|                                   | days, for each of these days:     |
|                                   |                                   |
|                                   | -   I did not/will not work       |
|                                   | -   I had/will have primary care  |
|                                   |     of the child                  |
|                                   | -   I expect to meet Parental     |
|                                   |     Leave Pay residence           |
|                                   |     requirements                  |
|                                   |                                   |
|                                   | I understand:                     |
|                                   |                                   |
|                                   | -   I need to advise Centrelink   |
|                                   |     of any changes to my          |
|                                   |     circumstances as soon as the  |
|                                   |     change occurs                 |
|                                   | -   Centrelink can make relevant  |
|                                   |     enquiries to make sure I      |
|                                   |     receive my correct            |
|                                   |     entitlement                   |
|                                   | -   Any money I owe will need to  |
|                                   |     be paid back                  |
|                                   | -   Giving false or misleading    |
|                                   |     information is a serious      |
|                                   |     offence                       |
|                                   | -   If another person claims or   |
|                                   |     has claimed Parental Leave    |
|                                   |     Pay for the same child,       |
|                                   |     Centrelink may advise them of |
|                                   |     Flexible Paid Parental Leave  |
|                                   |     entitlement information       |
|                                   |     relating to the child         |
|                                   | -   For each new Flexible Paid    |
|                                   |     Parental Leave day I have     |
|                                   |     claimed, Centrelink will use  |
|                                   |     my Tax File Number I provided |
|                                   |     in my initial claim for       |
|                                   |     Parental Leave Pay'           |
+-----------------------------------+-----------------------------------+

Examples – Recording transfer arrangements

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Full PPL transfer                 |
|                                   |                                   |
|                                   | Frances lodged a claim for PPL    |
|                                   | before 12 September 2020. Frances |
|                                   | nominated a PPL period start date |
|                                   | of 27 July and requested to       |
|                                   | transfer the full amount of PPL   |
|                                   | to Ravi.                          |
|                                   |                                   |
|                                   | The claim is initially assessed   |
|                                   | as:                               |
|                                   |                                   |
|                                   | -   Ravi’s PPL period (18 weeks): |
|                                   |     27 July to 27 November        |
|                                   |                                   |
|                                   | On 12 September, the claim is     |
|                                   | reassessed under the new PPL      |
|                                   | rules.                            |
|                                   |                                   |
|                                   | Frances is contacted and advised  |
|                                   | that they would like to give Ravi |
|                                   | permission to claim all of their  |
|                                   | Flexible PPL days.                |
|                                   |                                   |
|                                   | The Service Officer records the   |
|                                   | permission and Ravi’s claim       |
|                                   | updated to:                       |
|                                   |                                   |
|                                   | -   Ravi’s PPL period (12 weeks): |
|                                   |     27 July to 16 October         |
|                                   | -   Ravi’s connected Flexible PPL |
|                                   |     days (30 weekdays): 17        |
|                                   |     October to 27 November        |
+-----------------------------------+-----------------------------------+
| 2                                 | Partial transfer – transfer date  |
|                                   | within first 12 weeks             |
|                                   |                                   |
|                                   | Blake lodged a claim for PPL      |
|                                   | before 12 September 2020. Blake   |
|                                   | nominated a PPL period start date |
|                                   | of 7 August and requested a       |
|                                   | partial transfer of PPL to        |
|                                   | DeAndre. The transfer date is 18  |
|                                   | September.                        |
|                                   |                                   |
|                                   | The claim is initially assessed   |
|                                   | as:                               |
|                                   |                                   |
|                                   | -   Blake’s PPL period (6 weeks): |
|                                   |     7 August to 17 September      |
|                                   | -   DeAndre’s PPL period (12      |
|                                   |     weeks): 18 September to 10    |
|                                   |     December                      |
|                                   |                                   |
|                                   | On 12 September, the claim is     |
|                                   | reassessed under the new PPL      |
|                                   | rules. This converts the PPL      |
|                                   | period to 12 weeks plus 30        |
|                                   | unclaimed Flexible PPL days.      |
|                                   |                                   |
|                                   | -   Blake’s PPL period (6 weeks): |
|                                   |     7 August to 17 September      |
|                                   | -   DeAndre’s PPL period (6       |
|                                   |     weeks): 18 September to 29    |
|                                   |     October                       |
|                                   |                                   |
|                                   | Blake is contacted to check if    |
|                                   | permission is given for another   |
|                                   | person to claim the 30 Flexible   |
|                                   | PPL days or if Blake will keep    |
|                                   | them to claim later.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Partial transfer – transfer date  |
|                                   | more than 12 weeks after PPL      |
|                                   | period start date                 |
|                                   |                                   |
|                                   | Maree lodged a claim for PPL      |
|                                   | before 12 September 2020. Maree   |
|                                   | nominated a PPL period start date |
|                                   | of 1 September and requested a    |
|                                   | partial transfer of PPL to        |
|                                   | Martin. The transfer date is 1    |
|                                   | December.                         |
|                                   |                                   |
|                                   | The claim is initially assessed   |
|                                   | as:                               |
|                                   |                                   |
|                                   | -   Maree’s PPL period (13        |
|                                   |     weeks): 1 September to 30     |
|                                   |     November                      |
|                                   | -   Martin’s PPL period (5        |
|                                   |     weeks): 1 December to 4       |
|                                   |     January                       |
|                                   |                                   |
|                                   | On 12 September, the claim is     |
|                                   | reassessed under the new PPL      |
|                                   | rules. This converts the PPL      |
|                                   | period to 12 weeks plus 30        |
|                                   | unclaimed Flexible PPL days.      |
|                                   |                                   |
|                                   | -   Maree’s PPL period (12        |
|                                   |     weeks): 1 September to 23     |
|                                   |     November                      |
|                                   | -   Maree’s connected Flexible    |
|                                   |     PPL days (5 weekdays): 24     |
|                                   |     November to 30 November       |
|                                   |                                   |
|                                   | As Maree nominated a partial      |
|                                   | transfer, Maree is contacted to   |
|                                   | check if permission is given for  |
|                                   | another person to claim the       |
|                                   | remaining 25 Flexible PPL days or |
|                                   | if Maree will keep them to claim  |
|                                   | later.                            |
+-----------------------------------+-----------------------------------+
