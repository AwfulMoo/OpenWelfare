This page contains the mandatory procedure on how to complete a security
check when a provider calls about a Healthcare Provider Identifier -
Individual (HPI-I), before providing any information.

Provider calls about a HPI-I number

Security check when a provider calls about a HPI-I number

This table describes the steps Tier 2 Service Officers take when a
provider calls about their HPI-I number.

Note: all calls should be transferred to Tier 2 HI Operations Team.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Advise caller of required         |
|                                   | security check + Read more ...    |
|                                   |                                   |
|                                   | Advise the provider that a        |
|                                   | security check needs to be        |
|                                   | conducted before any information  |
|                                   | can be released.                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Request information from          |
|                                   | provider + Read more ...          |
|                                   |                                   |
|                                   | Request HPI-I number from the     |
|                                   | provider.                         |
|                                   |                                   |
|                                   | If unable to supply this          |
|                                   | information, request the          |
|                                   | provider's:                       |
|                                   |                                   |
|                                   | -   full name                     |
|                                   | -   date of birth                 |
|                                   | -   business, mailing or postal   |
|                                   |     address                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Check details against eHealth     |
|                                   | Program (EHP) system + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check this information on the     |
|                                   | Healthcare Identifiers EHP        |
|                                   | system.                           |
|                                   |                                   |
|                                   | See Searching for records in the  |
|                                   | Healthcare Identifiers Service    |
|                                   | using the eHealth Program (EHP).  |
|                                   |                                   |
|                                   | If the provider's HPI-I record    |
|                                   | is:                               |
|                                   |                                   |
|                                   | -   Found, go to Step 4           |
|                                   | -   Not found, advise the         |
|                                   |     provider that no information  |
|                                   |     can be provided as they did   |
|                                   |     not meet the requirements of  |
|                                   |     the security check. If the    |
|                                   |     provider disputes this,       |
|                                   |     transfer the call to the team |
|                                   |     coach                         |
|                                   |                                   |
|                                   | Note: if the provider believes    |
|                                   | that the information in the       |
|                                   | record is incorrect, the call     |
|                                   | must be transferred to HI Service |
|                                   | Tier 3.                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Ask caller what enquiry is        |
|                                   | about + Read more ...             |
|                                   |                                   |
|                                   | Ask the caller what the enquiry   |
|                                   | is about.                         |
|                                   |                                   |
|                                   | See Resources for how to resolve  |
|                                   | various scenarios.                |
+-----------------------------------+-----------------------------------+
