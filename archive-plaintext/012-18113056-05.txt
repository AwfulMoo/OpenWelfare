|                                   | Access (used for Nurse            |

|                                   |                                   |
|                                   | 651 - Nurse Practitioner working  |
|                                   | in a location listed on a 19(2)   |
|                                   | or 19(5) Direction (including     |
|                                   | COAG) (full access to Medicare)   |
|                                   |                                   |
|                                   | 651 – Nurse Practitioner Refer    |
|                                   | and Request used for Nurse        |
|                                   | Practitioners in a public         |
|                                   | facility (at stem and location)   |
|                                   |                                   |
|                                   | -   Add 618 provider service rule |
|                                   |     to location to prevent        |
|                                   |     billing                       |
|                                   |                                   |
|                                   | 651 Prescribing only (at stem and |
|                                   | location) (silent location        |
|                                   | created)                          |
|                                   |                                   |
|                                   | -   Add 618 provider service rule |
|                                   |     to location to prevent        |
|                                   |     claiming                      |
|                                   |                                   |
|                                   | 698 - Nurse Practitioner          |
|                                   | Prescribing only (specialty code  |
|                                   | no longer added for prescribing   |
|                                   | only)                             |
|                                   |                                   |
|                                   | Given the additional assessment   |
|                                   | required for Nurse Practitioner   |
|                                   | locations, they are unable to     |
|                                   | create a new location in HPOS.    |
|                                   | All stems should have the         |
|                                   | following spec code added:        |
|                                   |                                   |
|                                   | -   510 - New location creation   |
|                                   |     restricted in HPOS at stem    |
|                                   |     level                         |
|                                   |                                   |
|                                   | Note: all Nurse Practitioner      |
|                                   | applications must be              |
|                                   | re-categorised to National Office |
|                                   | folder in PANDA for approval.     |
|                                   | Ensure Ahpra certificate is       |
|                                   | attached.                         |

| Eligible for Prescriber number    | Yes - Nurse Practitioners are     |
|                                   | eligible to prescribe a limited   |
|                                   | range of medication.              |
|                                   |                                   |
|                                   | A location needs to be opened in  |
|                                   | order to activate a prescriber    |
|                                   | number.                           |
|                                   |                                   |
|                                   | See also the Prescriber number    |
|                                   | eligibility table in Provider     |
|                                   | Directory System (PDS) field      |
|                                   | codes for allied health and       |
|                                   | non-medical health professionals. |
|                                   |                                   |
|                                   | Where the Nurse Practitioner is   |
|                                   | only applying for a prescriber    |
|                                   | number (and not a provider        |
|                                   | number), a silent (for official   |
|                                   | use only) location needs to be    |
|                                   | opened to activate a prescriber   |
|                                   | number. Do not release this       |
|                                   | provider number to the applicant. |
|                                   | It is for Pharmaceutical Benefits |
|                                   | Scheme (PBS) purposes only and    |
|                                   | should be entered as 'Office Use  |
|                                   | Only'.                            |
|                                   |                                   |
|                                   | Enter on the location:            |
|                                   |                                   |
|                                   | Address tab:                      |
|                                   |                                   |
|                                   | -   'Office Use Only' in Building |
|                                   |     or Department                 |
|                                   | -   The hospital or practice name |
|                                   |     in Street Name                |
|                                   | -   Key suburb and post code as   |
|                                   |     per provider number           |
|                                   |     application                   |
|                                   |                                   |
|                                   | Contact tab:                      |
|                                   |                                   |
|                                   | -   Location Start Date to be     |
|                                   |     date of processing            |
|                                   |                                   |
|                                   | Loc-Spec Codes tab:               |
|                                   |                                   |
|                                   | -   Key 651 in Speciality Code    |
|                                   |     Description                   |
|                                   | -   Key date of processing in     |
|                                   |     Start Date                    |
|                                   | -   Select Add                    |
|                                   |                                   |
|                                   | Loc-Service Rules tab:            |
|                                   |                                   |
|                                   | -   Add Service Rule              |
|                                   | -   Key 618 in Service Rule       |
|                                   |     Number                        |
|                                   | -   Select Yes from Activity Code |
|                                   |     drop down menu                |
|                                   | -   Key date of processing in     |
|                                   |     Start Date                    |
|                                   |                                   |
|                                   | Select Add New                    |

| Request diagnostic imaging and    | Yes - Nurse Practitioners with a  |
| pathology tests                   | specialty code 651 are eligible   |
|                                   | to request a limited range of     |
|                                   | diagnostic imaging or pathology   |
|                                   | tests under Medicare.             |
+-----------------------------------+-----------------------------------+
| Refer to Specialist or Consultant | Yes - Nurse Practitioners with a  |
| physician                         | specialty code 651 are eligible   |
|                                   | to write referrals to Specialists |
|                                   | and Consultant Physicians.        |
|                                   |                                   |
|                                   | The referral is valid for 12      |
|                                   | months.                           |
+-----------------------------------+-----------------------------------+
| Eligible for 90 Day Cheque scheme | No                                |
+-----------------------------------+-----------------------------------+
| Registration requirements/details | A Nurse Practitioner applicant    |
|                                   | must have:                        |
|                                   |                                   |
|                                   | -   current registration to       |
|                                   |     practise with the Nursing and |
|                                   |     Midwifery Board of Australia  |
|                                   |     which is supported by the     |
|                                   |     Australian Health             |
|                                   |     Practitioner Regulation       |
|                                   |     Agency (Ahpra), and           |
|                                   | -   the registration record must  |
|                                   |     state in the ‘Endorsements’   |
|                                   |     section ‘Nurse Practitioner’  |
|                                   |                                   |
|                                   | Failure to renew or meet          |
|                                   | registration requirements:        |
|                                   |                                   |
|                                   | -   Nurse Practitioners are       |
|                                   |     required to renew their       |
|                                   |     registration annually on 31   |
|                                   |     May                           |
|                                   | -   Medicare benefits cannot be   |
|                                   |     paid for services provided or |
|                                   |     requested when they are not   |
|                                   |     registered with Ahpra         |
|                                   | -   Gaps in registration result   |
|                                   |     in gaps in eligibility to     |
|                                   |     access Medicare               |
