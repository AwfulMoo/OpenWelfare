|                                   | the notifying customer to ensure  |
|                                   | all relevant information is       |

|                                   | both customers what a change in   |

|                                   | in care decision is Services      |
|                                   | Australia's decision about the    |

|                                   | assessment. The agency does not   |
|                                   | determine where the child should  |

|                                   | percentage of care of the child   |

|                                   | the percentage of care            |
|                                   | established by a written care     |
|                                   | arrangement. A care event is a    |
|                                   | night of care or several          |
|                                   | consecutive nights of care that   |
|                                   | follow a recurring pattern.       |
|                                   | Confirm the date of event -       |
|                                   | future changes in care cannot be  |
|                                   | taken into account and a decision |
|                                   | cannot be revoked unless a new    |
|                                   | care event has occurred.          |
|                                   |                                   |
|                                   | The circumstances around care     |
|                                   | changes can be distressing,       |
|                                   | especially where the care is      |
|                                   | disputed or not occurring in line |
|                                   | with a written care arrangement.  |
|                                   | Be considerate and sensitive to   |
|                                   | the customer’s experience when    |
|                                   | discussing and explaining care    |
|                                   | changes.                          |
|                                   |                                   |
|                                   | Confirm with the customer:        |
|                                   |                                   |
|                                   | -   the details of the change in  |
|                                   |     care                          |
|                                   | -   the date the change applies   |
|                                   |     from                          |
|                                   |                                   |
|                                   | Discuss with the customer:        |
|                                   |                                   |
|                                   | -   any impacts on the assessment |
|                                   |     and the amount payable        |
|                                   | -   facts the decision will be    |
|                                   |     based on                      |
|                                   | -   options if they disagree with |
|                                   |     the decision                  |
|                                   |                                   |
|                                   | Registrar’s discretionary power   |
|                                   |                                   |
|                                   | Advise the customer that unless   |
|                                   | the reported care change results  |
|                                   | in a change in the cost           |
|                                   | percentage, there will be no      |
|                                   | effect on the assessment and we   |
|                                   | may not revoke the existing care  |
|                                   | or make a new care determination. |
|                                   | This is because under Section 54H |
|                                   | of the Child Support (Assessment) |
|                                   | Act 1989 (see the References page |
|                                   | for a link to legislation), we    |
|                                   | have discretion to revoke care    |
|                                   | and make a new determination that |
|                                   | would not make a material         |
|                                   | difference to the assessment. In  |
|                                   | the event a change in care is     |
|                                   | contested. A care percentage      |
|                                   | decision will only be made if it  |
|                                   | would result in a change to the   |
|                                   | child support assessment. The     |
|                                   | exception is where a written care |
|                                   | arrangement exists that has not   |
|                                   | yet been recorded or applied by   |
|                                   | Child Support. In this situation  |
|                                   | the written care arrangement      |
|                                   | needs to be recorded in Cuba as:  |
|                                   |                                   |
|                                   | -   a new care determination with |
|                                   |     the appropriate written care  |
|                                   |     type if being adhered to, or  |
|                                   | -   for information only if the   |
|                                   |     actual care is different to   |
|                                   |     that described in the written |
|                                   |     care arrangement              |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Investigation of change in child  |
|                                   | support care from 1 July 2018 (15 |
|                                   | May 2019 for WA Ex-nuptial        |
|                                   | cases).                           |
|                                   |                                   |
|                                   | Note: if it is evident the        |
|                                   | reported care change will not     |
|                                   | vary the cost percentage, the WA  |
|                                   | Ex-nuptial and pre 23/05/2018     |
|                                   | Change in care details macro must |
|                                   | still be used to guide the        |
|                                   | customer conversation and         |
|                                   | accurately calculate the reported |
|                                   | care percentage.                  |

+-----------------------------------+-----------------------------------+
| 6                                 | Calculate care period + Read more |
|                                   |                                   |
|                                   | A care period is the 12 month     |
|                                   | period starting from the date of  |
|                                   | event. In some circumstances, a   |
|                                   | shorter or longer care period may |
|                                   | be appropriate, however judge     |
|                                   | each case on its own merits. If   |
|                                   | determining a care period other   |
|                                   | than 12 months, seek technical    |
|                                   | support. See Technical support in |
|                                   | Child Support.                    |
|                                   |                                   |
|                                   | Care periods longer than 12       |
|                                   | months                            |
|                                   |                                   |
|                                   | A care period of more than 12     |
|                                   | months may be appropriate where   |
|                                   | the care arrangement follows a    |
|                                   | cycle over a period greater than  |
|                                   | 12 months, for example every 2    |
|                                   | years.                            |
|                                   |                                   |
|                                   | See Resources page for examples   |
|                                   | of care periods longer than 12    |
|                                   | months.                           |
|                                   |                                   |
|                                   | Care periods shorter than 12      |
|                                   | months                            |
|                                   |                                   |
|                                   | A care period of less than 12     |
|                                   | months might be appropriate where |
|                                   | the pattern of care will          |
|                                   | gradually change over definable   |
|                                   | periods in a specific and         |
|                                   | measurable way, for example one   |
|                                   | parent is to increase their care  |
|                                   | by one day per week every 3       |
|                                   | months over the upcoming period.  |
|                                   |                                   |
|                                   | A shorter care period may also be |
|                                   | appropriate where a carer has     |
|                                   | unexpectedly gained 100% care of  |
|                                   | a child for a temporary period    |
|                                   | (generally 4 weeks minimum). In   |
|                                   | these situations, the Registrar   |
|                                   | will determine the care over a    |
|                                   | short care period related to the  |
|                                   | unexpected circumstance and make  |
|                                   | a one-off block of 100% care      |
|                                   | determination.                    |
|                                   |                                   |
|                                   | See Resources page for examples   |
|                                   | of care periods shorter than 12   |
|                                   | months, and References page in    |
|                                   | Change in child support care      |
|                                   | levels before 1 July 2018 (15 May |
|                                   | 2019 for WA Ex-nuptial cases) for |
|                                   | further information.              |
|                                   |                                   |
|                                   | Note: Child Support will not give |
|                                   | effect to the future events until |
|                                   | the customers advise they have    |
|                                   | actually occurred. If neither     |
|                                   | customer advises of the           |
|                                   | subsequent change, the existing   |
|                                   | percentage of care will continue. |

|                                   | The term 'likely' means           |
|                                   | information gathered from the     |
|                                   | initial customer only. Care is    |

| 4                                 | Percentage of care (special       |
|                                   | rounding rules) + Read more       |
|                                   |                                   |
|                                   | Cuba will automatically calculate |
|                                   | the percentage of care used in    |
|                                   | the assessment when the number of |
|                                   | nights care has been entered into |
|                                   | the Care window.                  |
|                                   |                                   |
|                                   | There may be occasions when       |
|                                   | Service Officers need to manually |
|                                   | calculate the percentage of care, |
|                                   | or explain how the figures were   |
|                                   | calculated, as percentages of     |
|                                   | care calculations have special    |
|                                   | rounding rules.                   |
|                                   |                                   |
|                                   | Percentage of care = (number of   |
|                                   | nights / 365) x 100               |
|                                   |                                   |
|                                   | To calculate a percentage of care |
|                                   | the following special rounding    |
|                                   | rules must be applied (to ensure  |
|                                   | the total equals 100%). If the    |
|                                   | percentage of care is:            |
|                                   |                                   |
|                                   | -   below 50%, round the number   |
|                                   |     down to the next whole        |
|                                   |     percentage                    |
|                                   | -   50% or more, round the number |
|                                   |     up to the next whole          |
|                                   |     percentage                    |
|                                   |                                   |
|                                   | Where each customer has a care    |
|                                   | percentage of 50%, record the     |
|                                   | payee as having 50% care, see the |
|                                   | Care Cuba Process Help. Do not    |
|                                   | record this as a number of        |
|                                   | nights. Cuba will automatically   |
|                                   | change this to show the payee has |
|                                   | 51% care and the other party has  |
|                                   | 49% care. The Care Reported as    |
|                                   | 50% checkbox will be              |
|                                   | automatically set by Cuba and a   |
|                                   | care percentage of 50%            |
|                                   | transferred to Centrelink.        |

on the type of care, care percentages and date of event details provided
by the notifying customer.

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Care types + Read more            |

|                                   | different types of care decisions |

|                                   |     agreement                     |

|                                   |     determination is made         |
|                                   | -   there is a court order,       |
|                                   |     parenting plan or written     |
|                                   |     agreement, and                |
|                                   | -   care is not occurring in line |
