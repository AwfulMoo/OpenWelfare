Example of child attending Family Day Care (FDC) and customer is an FDC educator

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Rachel is a customer who has 3    |
|                                   | children under school age - Zac,  |
|                                   | Callum and Jane. They attend ABC  |
|                                   | FDC every Tuesday, Wednesday and  |
|                                   | Thursday.                         |
|                                   |                                   |
|                                   | Rachel is also self-employed as a |
|                                   | FDC educator with ZZZ FDC Service |
|                                   | and cares for other children in   |
|                                   | the family home on Tuesdays and   |
|                                   | Thursdays.                        |
|                                   |                                   |
|                                   | Rachel is not eligible to receive |
|                                   | CCS or ACCS for Zac, Callum and   |
|                                   | Jane who attend ABC FDC on        |
|                                   | Tuesdays and Thursdays due to     |
|                                   | working on these days as a FDC    |
|                                   | educator.                         |
|                                   |                                   |
|                                   | Rachel is eligible to receive CCS |
|                                   | or ACCS for Zac, Callum and Jane: |
|                                   |                                   |
|                                   | -   if they were to attend MMM    |
|                                   |     Long Day Care service on      |
|                                   |     Tuesdays and Thursdays, or    |
|                                   | -   to attend ABC FDC on          |
|                                   |     Wednesdays when Rachel is not |
|                                   |     providing care as a FDC       |
|                                   |     educator herself              |
|                                   |                                   |
|                                   | There are specified circumstances |
|                                   | where Rachel may continue to be   |
|                                   | eligible for 1 or more biological |
|                                   | children on Tuesdays and          |
|                                   | Thursdays, such as a child having |
|                                   | a particular disability.          |
|                                   |                                   |
|                                   | See Family Day Care Legislation   |
|                                   | Changes – Child Swapping FAQ’s.   |
+-----------------------------------+-----------------------------------+
