Scenarios for net rent rules assessment

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| Rent with lodgers                 | Norma pays rent of $190 per week  |
|                                   | for a private house, as stated on |
|                                   | a written tenancy agreement       |
|                                   | (lease). Norma’s 41 year old son  |
|                                   | and 24 year old grandson also     |
|                                   | live in the house and each pay    |
|                                   | $70 per week lodgings.            |
|                                   |                                   |
|                                   | Assessment                        |
|                                   |                                   |
|                                   | Norma’s net rent is $50 per week  |
|                                   | ($190 - $140 for lodgings).       |
+-----------------------------------+-----------------------------------+
| Rent and two thirds rule          | Robert pays rent of $340 per week |
|                                   | for a house, as claimed on a rent |
|                                   | certificate. He has 2 exchange    |
|                                   | students living in the house with |
|                                   | him, each paying Robert $90 per   |
|                                   | week for board and lodging. As it |
|                                   | could not be determined how much  |
|                                   | was for lodgings, the two-thirds  |
|                                   | rule was applied and $60 per week |
|                                   | is for lodging.                   |
|                                   |                                   |
|                                   | Assessment                        |
|                                   |                                   |
|                                   | Robert’s net rent is $220 per     |
|                                   | week ($340 - $120 for lodgings).  |
+-----------------------------------+-----------------------------------+
| Rent and sharing                  | Joy receives an income support    |
|                                   | payment and shares her home with  |
|                                   | Maddie who does not receive a     |
|                                   | payment. Joy is the only name on  |
|                                   | the lease. The total rent for the |
|                                   | address is $280 per week. Maddie  |
|                                   | pays Joy $150 per week rent.      |
|                                   |                                   |
|                                   | Assessment                        |
|                                   |                                   |
|                                   | Joy’s net rent is $130 per week   |
|                                   | ($280 - $150).                    |
+-----------------------------------+-----------------------------------+
