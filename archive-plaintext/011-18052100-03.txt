Assess Pharmaceutical claims

Table 2: how processing staff assess a Pharmaceutical claim.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check pharmaceutical item(s) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the pharmaceutical item(s) on  |
|                                   | the claimant’s approved list and  |
|                                   | has it been notated as approved   |
|                                   | by their health professional in   |
|                                   | the last 12 months?               |
|                                   |                                   |
|                                   | The health professional must      |
|                                   | state that the prescribed         |
|                                   | pharmaceutical(s) is or are       |
|                                   | required and is directly related  |
|                                   | to the treatment of an            |
|                                   | injury(ies) sustained in the      |
|                                   | adverse event?                    |
|                                   |                                   |
|                                   | The claimant’s approved list is   |
|                                   | in the claimant’s file.           |
|                                   |                                   |
|                                   | Approved lists are to be verified |
|                                   | every 12 months. Claims should    |
|                                   | not be paid until verification    |
|                                   | has been received, approved and   |
|                                   | confirmed. For those categories   |
|                                   | only eligible for pharmaceuticals |
|                                   | as part of psychiatric treatment, |
|                                   | the prescribing health            |
|                                   | professional must be their        |
|                                   | treating psychiatrist.            |
|                                   |                                   |
|                                   | Yes, progress with processing the |
|                                   | claim. Go to the claims screen to |
|                                   | process the claim, see Step 5 in  |
|                                   | Table 1                           |
|                                   |                                   |
|                                   | No, it is not on the approval     |
|                                   | list or is over 12 months old:    |
|                                   |                                   |
|                                   | -   as the claim needs more       |
|                                   |     information, complete the     |
|                                   |     DHCAS cannot be processed     |
|                                   |     letter                        |
|                                   | -   send the letter to the        |
|                                   |     claimant. Attach a copy of    |
|                                   |     the letter to the claim       |
|                                   |     documentation                 |
|                                   | -   notate on the claim form      |
|                                   |     steps taken to seek           |
|                                   |     confirmation of the goods     |
|                                   |     supplied                      |
|                                   | -   pend the claim ‘awaiting      |
|                                   |     response from the health      |
|                                   |     professional’ confirming that |
|                                   |     the pharmaceutical item was   |
|                                   |     prescribed by them            |
|                                   | -   once the health professional  |
|                                   |     has confirmed the             |
|                                   |     information, go to the claims |
|                                   |     screen to process the claim,  |
|                                   |     see Step 5 in Table1          |

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   |                                   |
|                                   | The costs of allied health        |
|                                   | services include but are not      |
|                                   | limited to physiotherapy,         |
|                                   | remedial massage, mental health   |
|                                   | (psychology, psychiatric and      |
|                                   | counselling), prosthetic claims,  |
|                                   | travel and accommodation, less    |
|                                   | any amounts covered by Medicare,  |
|                                   | other State or Territory          |
|                                   | Government schemes and/or private |
|                                   | health insurance.                 |
|                                   |                                   |
|                                   | Refer to Team Leader if a Program |
|                                   | Management determination is       |
|                                   | required for high cost claims,    |
|                                   | pre-payment and claims outside of |
|                                   | standard guidelines.              |
|                                   |                                   |
|                                   | Note 1: High cost claims are      |
|                                   | $10,000.00 and above.             |
|                                   |                                   |
|                                   | Note 2: Mental Health as well as  |
|                                   | travel and accommodation claims   |
|                                   | require additional approval       |
|                                   | processes for payment.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for supporting              |
|                                   | documentation in last 12 months + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Does the claimant have supporting |
|                                   | documentation (either attached or |
|                                   | on file) in the last 12 months    |
|                                   | from their health professional?   |
|                                   |                                   |

|                                   | claim in DHCAS for when a report  |
|                                   | is required.                      |
|                                   |                                   |
|                                   | The supporting letter from the    |

|                                   | should state the following:       |

|                                   | any other information they        |

|                                   | to the agency from their          |
|                                   | provider.                         |
|                                   |                                   |
|                                   | -   Yes, go back to the claims    |
|                                   |     screen to continue processing |
|                                   |     the claim, see Step 5 in      |
|                                   |     Table 1                       |

+-----------------------------------+-----------------------------------+
| 3                                 | Check if claimant has previously  |

|                                   | file + Read more ...              |
|                                   |                                   |
|                                   | Has the claimant previously had   |
|                                   | supporting documentation on file? |

|                                   |                                   |
|                                   | -   complete the ‘Request for     |

|                                   | -   notate on the document steps  |
|                                   |     taken to seek confirmation of |

Assess and process a travel and accommodation claim

Table 4: how processing staff assess and process a new claim for travel
and/or accommodation.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assess travel claim is related to |
|                                   | treatment + Read more ...         |
|                                   |                                   |
|                                   | It must be confirmed that travel  |
|                                   | costs are directly related to the |
|                                   | claimant attending an appointment |
|                                   | with the treating health care     |
|                                   | provider. This can be confirmed   |
|                                   | by checking:                      |
|                                   |                                   |
|                                   | -   that the claim documentation  |
|                                   |     submitted at the same time as |
|                                   |     the Disaster Health Care      |
|                                   |     Assistance Scheme Motor       |
|                                   |     Vehicle and/or Accommodation  |
|                                   |     Out-of-Pocket Travel          |
|                                   |     Application form has occurred |
|                                   |     on the same date, or within a |
|                                   |     day of the appointment with   |
|                                   |     the health professional       |
|                                   | -   that the treating health      |
|                                   |     professional has made a       |
|                                   |     notation on their receipt     |
|                                   |     that the goods and/or         |
|                                   |     services provided are related |
|                                   |     to the adverse event          |
|                                   |                                   |
|                                   | If the date(s) for the            |
|                                   | goods/services being claimed on   |
|                                   | the Disaster Health Care          |
|                                   | Assistance Scheme claim form and  |
|                                   | the above Application form do not |
|                                   | correlate, return all             |
|                                   | documentation to the customer     |
|                                   | with a DHCAS claim needs more     |
|                                   | information letter requesting the |
|                                   | claimant verify the dates.        |
|                                   |                                   |
|                                   | Accommodation can only be claimed |
|                                   | when the claimant lives in a      |
|                                   | different town or region, 50km+   |
|                                   | from the health provider.         |
|                                   | Accommodation required for more   |
|                                   | than 10 consecutive days must be  |
|                                   | pre-approved.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Process a travel claim form for   |
|                                   | DHCAS + Read more ...             |
|                                   |                                   |
|                                   | Complete the DHCAS checklist.     |
|                                   |                                   |
|                                   | Costs incurred for travel using a |
|                                   | private vehicle will be paid at   |
|                                   | the same rate as an employee of   |
|                                   | Services Australia. Air travel    |
|                                   | costs are to be paid at economy   |
|                                   | rates.                            |
|                                   |                                   |
|                                   | Does the claim form contain the   |
|                                   | following information?            |
|                                   |                                   |
|                                   | -   customer details              |
|                                   | -   mode of transportation        |
|                                   |     details                       |
|                                   | -   journey details (from start   |
|                                   |     to finish)                    |
|                                   | -   receipts for travel if        |
|                                   |     transport is by air, rail,    |
|                                   |     taxi or public transport      |
|                                   | -   health care provider name and |
|                                   |     full address                  |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No, if the claimant has not       |
|                                   | supplied a completed Disaster     |
|                                   | Health Care Assistance Scheme     |
|                                   | Motor Vehicle and/or              |
|                                   | Accommodation Out-of-Pocket       |
|                                   | Application form:                 |
|                                   |                                   |
|                                   | -   complete the DHCAS claim      |
|                                   |     needs more information        |
|                                   |     letter, attaching the         |
|                                   |     original claim to the letter  |
|                                   |     by merging the files          |
|                                   | -   send the letter to the        |
|                                   |     claimant                      |
|                                   | -   add a notation on the claim   |
|                                   |     outlining the steps taken to  |
|                                   |     request the completed         |
|                                   |     Disaster Health Care          |
|                                   |     Assistance Scheme Motor       |
|                                   |     Vehicle and/or Accommodation  |
|                                   |     Out-of-Pocket Application     |
|                                   |     form                          |
|                                   | -   pend the claim awaiting a     |
|                                   |     response                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Calculate travel distance and     |
|                                   | amount payable for transport by a |
|                                   | private motor vehicle + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | In calculating the total distance |
|                                   | travelled by the claimant, a      |
|                                   | specific calculation must be used |
|                                   | incorporating the standard        |
|                                   | corporate rate per kilometre.     |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   claimant total distance       |
|                                   |     travelled is 50km (25km each  |
|                                   |     way)                          |
|                                   | -   the corporate rate per km is  |
|                                   |     0.62c                         |
|                                   |                                   |
|                                   | The calculation will be:          |
|                                   |                                   |
|                                   | -   50km x 0.62c per km = $31.00  |
|                                   |                                   |
|                                   | The kilometres must be confirmed  |
|                                   | as being correct before the claim |
|                                   | can be paid.                      |
|                                   |                                   |
|                                   | To do this:                       |
|                                   |                                   |
|                                   | -   go to online maps             |
|                                   | -   type in the address of the    |
|                                   |     claimant                      |
|                                   | -   type in the address of the    |
|                                   |     health care provider          |
|                                   | -   select Get Directions         |
|                                   | -   directions page will display  |
|                                   | -   the directions page will      |
|                                   |     calculate the fastest route   |
|                                   |     that can be taken from the    |
|                                   |     claimant’s residential        |
|                                   |     address to the health care    |
|                                   |     provider’s address            |
|                                   | -   the online map gives the most |
|                                   |     direct route with the amount  |
|                                   |     of kilometres it takes to get |
|                                   |     there                         |
|                                   |                                   |
|                                   | If the claimant has provided a    |
|                                   | different distance to the one in  |
|                                   | the online map, use shorter       |
|                                   | distance.                         |
|                                   |                                   |
|                                   | For example, the claimant is      |
|                                   | claiming a distance of 100        |
|                                   | kilometres, however the online    |
|                                   | map states it is 75 kilometres    |
|                                   | round trip from the claimant’s    |
|                                   | residential address to their      |
|                                   | health care provider’s location   |
|                                   | and return to their residential   |
|                                   | address. In this circumstance     |
|                                   | always use the shorter distance.  |
|                                   |                                   |
|                                   | Do not calculate any distances of |
|                                   | travel that the claimant has not  |
|                                   | supplied on the claim form.       |
|                                   |                                   |
|                                   | Payments will not be paid for     |
|                                   | non-related detours made whilst   |
|                                   | travelling.                       |
|                                   |                                   |
|                                   | Amounts payable for air, taxi or  |
|                                   | public transport with a fixed     |
|                                   | out-of-pocket amount are          |
|                                   | processed based on the invoice    |
|                                   | provided. Distances and           |
|                                   | allowances do not need to be      |
|                                   | calculated. Air travel costs are  |
|                                   | to be paid at economy rates.      |
+-----------------------------------+-----------------------------------+

|                                   | Sydney to Brisbane and return     |
|                                   | travel on the same day over 4     |
|                                   | weeks:                            |
|                                   |                                   |
|                                   | -   6:30 am Sydney to Brisbane -  |
|                                   |     $170.00                       |
|                                   | -   5:30 pm Brisbane to Sydney -  |
|                                   |     $170.00                       |
|                                   | -   6:30 am Sydney to Brisbane -  |
|                                   |     $135.00                       |
|                                   | -   5:30 pm Brisbane to Sydney -  |
|                                   |     $170.00                       |
|                                   | -   6:30 am Sydney to Brisbane -  |
|                                   |     $135.00                       |
