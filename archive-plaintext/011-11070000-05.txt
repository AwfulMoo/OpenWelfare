Organ Donor Registrant (ODR) registration details check

This table lists the registration details to confirm an ODR which must
be checked to ensure correct information is provided to the Authorised
Medical Personnel (AMP) for AODR.

+-----------------------------------+-----------------------------------+
| Check                             | Comments                          |
+-----------------------------------+-----------------------------------+
| Personal Details                  | To verify name, date of birth,    |
|                                   | gender (and address for donor     |
|                                   | only records).                    |
+-----------------------------------+-----------------------------------+
| Donor Decision Details            | The Donor Status type for consent |
|                                   | or intent and the Donor Decision  |
|                                   | key 'Yes/No' must be checked.     |
|                                   |                                   |
|                                   | Most Recent Donation Decision     |
|                                   | Date (do not advise other dates). |
|                                   |                                   |
|                                   | Where required, organs and/or     |
|                                   | tissues for donation.             |
+-----------------------------------+-----------------------------------+
| End date                          | For:                              |
|                                   |                                   |
|                                   | -   Australian Organ Donor        |
|                                   |     Register (AODR) entitlement - |
|                                   |     this is found under Program   |
|                                   |     Details tab                   |
|                                   | -   Donor only entitlement - this |
|                                   |     is located under the Donation |
|                                   |     decisions tab                 |
|                                   |                                   |
|                                   | Note: if an end date is recorded  |
|                                   | against the organ donor           |
|                                   | registration, Service Officers    |
|                                   | must not provide any information  |
|                                   | about the registration.           |
|                                   |                                   |
|                                   | If the record has a recent end    |
|                                   | date and reason of 'deceased or   |
|                                   | fact of death' the record could   |
|                                   | belong to the person being        |
|                                   | searched for. In this instance    |
|                                   | the Service Officer can advise    |
|                                   | the AMP of a possible match with  |
|                                   | a recently deceased person.       |
+-----------------------------------+-----------------------------------+
| Comments                          | For any relevant additional       |
|                                   | details (such as medical          |
|                                   | information or next of kin        |
|                                   | details and wishes).              |
|                                   |                                   |
|                                   | Note: only disclose information   |
|                                   | recorded at the request of the    |
|                                   | registrant to the AMP, for        |
|                                   | example 'Estranged from family'   |
|                                   | or 'objection due to faith'. Do   |
|                                   | not disclose any comments not     |
|                                   | added at the ODR's request such   |
|                                   | as QC or processing comments.     |
+-----------------------------------+-----------------------------------+
