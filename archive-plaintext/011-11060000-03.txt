Search for existing organ donor registration

Table 1: This table describes how to search for and determine whether an
organ donor registration exists.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | CDMS home page + Read more ...    |
|                                   |                                   |
|                                   | From the CDMS home page select    |
|                                   | either:                           |
|                                   |                                   |
|                                   | Inquire on consumer details       |
|                                   |                                   |
|                                   | Amend consumer details            |
|                                   |                                   |
|                                   | -   The search page displays      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search type + Read more ...       |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Detailed Search tab           |
|                                   | -   Australian Organ Donor        |
|                                   |     Register from the Program     |
|                                   |     drop down list                |
+-----------------------------------+-----------------------------------+
| 3                                 | Search criteria + Read more ...   |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | -   Family Name                   |
|                                   | -   Gender or All                 |
|                                   | -   Date of Birth or Age Range    |
|                                   |                                   |
|                                   | Select Entitlement:               |
|                                   |                                   |
|                                   | -   The system searches CDMS for  |
|                                   |     existing registration based   |
|                                   |     on the search criteria        |
|                                   |                                   |
|                                   | The results display either:       |
|                                   |                                   |
|                                   | matched consumers with an AODR    |
|                                   | entitlement                       |
|                                   |                                   |
|                                   | matched Donor only entitlements   |
|                                   |                                   |
|                                   | both of the above                 |
|                                   |                                   |
|                                   | Information Message 01138 'View   |
|                                   | End Dated Entitlement Details':   |
|                                   |                                   |
|                                   | -   This result displays when     |
|                                   |     conducting a search via the   |
|                                   |     Amend screen and the          |
|                                   |     entitlement has already been  |
|                                   |     end-dated                     |
|                                   | -   Go to Step 1 and search via   |
|                                   |     Inquire                       |
|                                   |                                   |
|                                   | Information Message 00461 'No     |
|                                   | records found':                   |
|                                   |                                   |
|                                   | -   This result returns if the    |
|                                   |     search cannot find any        |
|                                   |     matching data                 |
|                                   | -   Amend search criteria to      |
|                                   |     attempt again                 |
|                                   | -   Search by any previous names  |
|                                   |     the customer/registrant may   |
|                                   |     have been known by            |
|                                   | -   If no matches are returned    |
|                                   |     and the Medicare number is    |
|                                   |     not authorised, change        |
|                                   |     Program drop down to ‘All     |
|                                   |     Programs’ to check for an     |
|                                   |     existing Consumer ID.         |
|                                   |     Duplicate Consumer ID’s must  |
|                                   |     not be created                |
+-----------------------------------+-----------------------------------+
| 4                                 | Open record + Read more ...       |
|                                   |                                   |
|                                   | Where only one record matches,    |
|                                   | the record will automatically     |
|                                   | open.                             |
|                                   |                                   |
|                                   | For multiple records results,     |
|                                   | select:                           |
|                                   |                                   |
|                                   | -   Radio button for the correct  |
|                                   |     donor registrant from the     |
|                                   |     displayed list                |
|                                   | -   If more than 50 records       |
|                                   |     matching the search details   |
|                                   |     have been found, the 'More    |
|                                   |     Records Found' message        |
|                                   |     displays                      |
|                                   | -   Service Officers must refine  |
|                                   |     the search details to display |
|                                   |     a shorter list                |
|                                   |                                   |
|                                   | The following fields must be      |
|                                   | updated when contact/enquiries    |
|                                   | are made and the record is        |
|                                   | accessed. This update must occur  |
|                                   | even if no amendments to the      |
|                                   | record are required:              |
|                                   |                                   |
|                                   | -   Donor Media Type              |
|                                   | -   Donor Delivery Type           |
|                                   | -   Date of Last Contact          |
|                                   |                                   |
|                                   | See also Security check for organ |
|                                   | donor registration.               |
+-----------------------------------+-----------------------------------+

Inquire on and view existing AODR and Donor Only entitlements

Table 2: This table describes how to search and view an existing AODR
entitlement.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Perform search + Read more ...    |
|                                   |                                   |
|                                   | Perform an Inquiry search for an  |
|                                   | existing registration:            |
|                                   |                                   |
|                                   | -   Matched Consumers list        |
|                                   |     displays                      |
|                                   |                                   |
|                                   | See also: Search for an existing  |
|                                   | registration.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Record selection + Read more ...  |
|                                   |                                   |
|                                   | Where only one record matches,    |
|                                   | the record will automatically     |
|                                   | open.                             |
|                                   |                                   |
|                                   | -   Select the required organ     |
|                                   |     donor record from the Matched |
|                                   |     Consumers or Matched Donor    |
|                                   |     Records lists.                |
|                                   |                                   |
|                                   | Note: Message 00461 ‘No records   |
|                                   | found’ is returned if the search  |
|                                   | cannot find any matching data     |
|                                   |                                   |
|                                   | -   Amend search criteria to      |
|                                   |     attempt again                 |
|                                   | -   If more than 50 records       |
|                                   |     matching the search details   |
|                                   |     have been found, the 'More    |
|                                   |     Records Found' message        |
|                                   |     displays                      |
|                                   | -   Service Officers must refine  |
|                                   |     the search details to display |
|                                   |     a shorter list                |
+-----------------------------------+-----------------------------------+
| 3                                 | Entitlement selection + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Entitlement:               |
|                                   |                                   |
|                                   | -   the Inquire AODR Entitlement  |
|                                   |     Details screen displays       |
+-----------------------------------+-----------------------------------+
| 4                                 | Registration information + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select tabs at the top of the     |
|                                   | screen to access different        |
|                                   | aspects of the registration       |
|                                   | information. For example:         |
|                                   |                                   |
|                                   | -   Personal Details              |
|                                   | -   Donor Decision Details        |
|                                   | -   Program Details               |
|                                   | -   Comments                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Search again + Read more ...      |
|                                   |                                   |
|                                   | Select Search Page to return to   |
|                                   | the Inquiry Search screen to      |
|                                   | start another search.             |
+-----------------------------------+-----------------------------------+

Basic Search

Table 5: This table describes how to search for an AODR registration by
identification number using Basic Search. This method is only used for
telephony purposes where the individual has provided a unique identifier
(such as a Medicare Card Number) or after a detailed search has been
conducted using the Identification Number located.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | CDMS home page + Read more ...    |
|                                   |                                   |
|                                   | From the CDMS home page select    |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   Inquire on consumer details   |
|                                   | -   Amend consumer details        |
|                                   | -   The search page displays      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search type + Read more ...       |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Basic Search tab              |
|                                   | -   Australian Organ Donor        |
|                                   |     Register from the Program     |
|                                   |     drop down list                |
+-----------------------------------+-----------------------------------+
| 3                                 | Search criteria + Read more ...   |
|                                   |                                   |
|                                   | In the ID Number field, enter     |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   Donor Register registration   |
|                                   |     ID number                     |
|                                   | -   Medicare Card Number          |
|                                   |                                   |
|                                   | Note: Medicare PIN and Consumer   |
|                                   | ID will also return results for   |
|                                   | the Program.                      |
|                                   |                                   |
|                                   | In the ID Number Type drop down,  |
|                                   | select either:                    |
|                                   |                                   |
|                                   | -   Donor ID                      |
|                                   | -   Medicare Card Number          |
|                                   | -   Medicare PIN                  |
|                                   | -   Consumer ID                   |
|                                   |                                   |
|                                   | Select Entitlement:               |
|                                   |                                   |
|                                   | -   The system searches CDMS for  |
|                                   |     existing registration based   |
|                                   |     on the search criteria        |
+-----------------------------------+-----------------------------------+
| 4                                 | Record selection + Read more ...  |
|                                   |                                   |
|                                   | The record will automatically     |
|                                   | open.                             |
|                                   |                                   |
|                                   | Message 00461 ‘No records found’  |
|                                   | is returned if the search cannot  |
|                                   | find any matching data:           |
|                                   |                                   |
|                                   | Check the Identification Number   |
|                                   | and ID Number Type entered        |
|                                   |                                   |
|                                   | -   If a record was a Duplicate   |
|                                   |     Donor Only, it may have been  |
|                                   |     deleted                       |
|                                   |                                   |
|                                   | Conduct Search for existing organ |
|                                   | donor registration                |
|                                   |                                   |
|                                   | Note: conduct a Detailed Search   |
|                                   | to check for further/duplicate    |
|                                   | records.                          |
+-----------------------------------+-----------------------------------+
