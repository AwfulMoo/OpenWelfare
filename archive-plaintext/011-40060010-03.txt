Basic search in CDMS

Table 1: searching for a consumer where an existing or previous ID
number is available.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Basic search + Read more ...      |
|                                   |                                   |
|                                   | On the CDMS Home page, select     |
|                                   | Inquire on Consumer Details.      |
|                                   |                                   |
|                                   | In the Basic Search tab:          |
|                                   |                                   |
|                                   | -   in the Program field –        |
|                                   |     ‘Medicare’ is the default,    |
|                                   |     from the drop down box select |
|                                   |     All Programs                  |
|                                   | -   key the Medicare card number  |
|                                   |     in ID Number field            |
|                                   |                                   |
|                                   | Note: if there is no ID number,   |
|                                   | see Table 2 to perform a detailed |
|                                   | search in CDMS.                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Display customer details + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select 1 of the following:        |
|                                   |                                   |
|                                   | -   Personal - Personal details   |
|                                   |     of the customer               |
|                                   | -   Entitlement - Medicare        |
|                                   |     entitlement details of a      |
|                                   |     customer                      |
|                                   | -   Group - Group or card details |
|                                   |     of a customer                 |
|                                   |                                   |
|                                   | This will display the details of  |
|                                   | the customer, if found.           |
|                                   |                                   |
|                                   | Possible Matches                  |
|                                   |                                   |
|                                   | If possible matches are found,    |
|                                   | they will display under Matched   |
|                                   | Consumers. Select the consumer    |
|                                   | using the radio button.           |
|                                   |                                   |
|                                   | If no records are found:          |
|                                   |                                   |
|                                   | -   A message displays indicating |
|                                   |     that no records were found    |
|                                   | -   Select OK to acknowledge the  |
|                                   |     message                       |
|                                   |                                   |
|                                   | Confirm details such as Medicare  |
|                                   | number, name, address and date of |
|                                   | birth and proceed with enquiry or |
|                                   | amendment.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Swap to Amend screen + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Personal                   |
|                                   |                                   |
|                                   | Scroll to bottom and select Edit  |
|                                   |                                   |
|                                   | A message will display asking if  |
|                                   | the user wants to swap from       |
|                                   | Inquiry to Amend:                 |
|                                   |                                   |
|                                   | -   select Yes                    |
|                                   | -   amend Personal Details will   |
|                                   |     display                       |
|                                   | -   proceed with any amendments   |

Detailed search in CDMS

Table 2: searching for a customer where no existing or previous ID
number is available, required before creating a new registration.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | CDMS home + Read more ...         |
|                                   |                                   |
|                                   | On the CDMS Home page, select     |
|                                   | Inquire on Consumer Details.      |
|                                   |                                   |
|                                   | Select the Detailed Search tab.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter name search details + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Key search details as follows:    |
|                                   |                                   |
|                                   | -   Program field – 'Medicare' is |
|                                   |     the default, from the drop    |
|                                   |     down box select All Programs  |
|                                   | -   Family Name (mandatory)       |
|                                   | -   Gender (mandatory) – select   |
|                                   |     All                           |
|                                   | -   Age or Age Range (mandatory)  |
|                                   | -   Postcode or Postcode range    |
|                                   |     (leave field as default, for  |
|                                   |     example, 0001-9999)           |
|                                   |                                   |
|                                   | Select the following:             |
|                                   |                                   |
|                                   | -   Personal - personal details   |
|                                   |     of the customer               |
|                                   |                                   |
|                                   | This will display the details of  |
|                                   | the customer if found.            |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   try variations of the         |
|                                   |     surname, for example the      |
|                                   |     spelling or if it is          |
|                                   |     hyphenated, try with and      |
|                                   |     without the hyphen and try    |
|                                   |     each of the hyphenated        |
|                                   |     surnames separately           |
|                                   | -   if there are too many         |
|                                   |     results, refine the search by |
|                                   |     adding the first initial of   |
|                                   |     the customer’s first name     |
|                                   |                                   |
|                                   | Are there any possible matches    |
|                                   | found?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 3                                 | Possible Matches found + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If possible matches are found,    |
|                                   | they will display under Matched   |
|                                   | Consumers. Select the consumer    |
|                                   | using the radio button.           |
|                                   |                                   |
|                                   | Confirm details such as name,     |
|                                   | address and date of birth and     |
|                                   | proceed with the enquiry or       |
|                                   | amendment.                        |
|                                   |                                   |
|                                   | If the customer details require   |
|                                   | updating, for example adding a    |
|                                   | middle name, go to Step 4.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Swap to Amend screen + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Personal                   |
|                                   |                                   |
|                                   | Scroll to bottom and select Edit  |
|                                   |                                   |
|                                   | A message will display asking if  |
|                                   | the user wants to swap from       |
|                                   | Inquiry to Amend:                 |
|                                   |                                   |
|                                   | -   select Yes                    |
|                                   | -   amend Personal Details will   |
|                                   |     display                       |
|                                   | -   proceed with any amendments   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | No results + Read more ...        |
|                                   |                                   |
|                                   | Where CDMS returns no results, go |
|                                   | to Step 1 in Table 6 to conduct   |
|                                   | an address search.                |
