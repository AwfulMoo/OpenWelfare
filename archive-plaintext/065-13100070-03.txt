This page contains information for Service Officers about the Aged Care
debt recovery process.

Debt recovery

Aged Care debt recovery process

This table describes the debt recovery process for Service Officers.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Closure notification + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Notification of closure is        |
|                                   | received from the Department of   |
|                                   | Health and Aged Care to the Aged  |
|                                   | Care processing team.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact + Read more ...           |
|                                   |                                   |
|                                   | The provider is contacted and     |
|                                   | relevant contact details sourced  |
|                                   | for discussion of closure         |
|                                   | procedures.                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Calculate overpayment + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Calculate total overpayment       |
|                                   | amount requiring recovery.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Advise provider of overpayment +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A debt recovery letter with       |
|                                   | attached payment statements are   |
|                                   | supplied to the service provider  |
|                                   | for review. The provider must     |
|                                   | agree to the debt amount in       |
|                                   | writing and select an option for  |
|                                   | repayment (either an invoice or   |
|                                   | offset against ongoing service    |
|                                   | ID).                              |
|                                   |                                   |
|                                   | Repayment options:                |
|                                   |                                   |
|                                   | -   invoice go to Step 5          |
|                                   | -   offset against ongoing        |
|                                   |     service go to Step 6          |
+-----------------------------------+-----------------------------------+
| 5                                 | Invoice for overpayment           |
|                                   | recovery + Read more ...          |
|                                   |                                   |
|                                   | Complete a Debt Advice Notice     |
|                                   | (DAN) and send to Medicare debt   |
|                                   | recovery with:                    |
|                                   |                                   |
|                                   | -   provider’s confirmation in    |
|                                   |     writing, and                  |
|                                   | -   initial debt recovery letter  |
|                                   |     and payment statements        |
|                                   |                                   |
|                                   | Medicare debt recovery will       |
|                                   | commence the recovery action. The |
|                                   | Aged Care processing team take no |
|                                   | further action.                   |
|                                   |                                   |
|                                   | The Resources page has contact    |
|                                   | details for Medicare debt         |
|                                   | recovery.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Offset overpayment against        |
|                                   | ongoing service ID + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The provider opts to adjust the   |
|                                   | debt against an ongoing service   |
|                                   | ID – a manual negative adjustment |
|                                   | is processed against the          |
|                                   | nominated service in the          |
|                                   | nominated month.                  |
|                                   |                                   |
|                                   | See Process manual adjustments    |
|                                   | for aged care services.           |
+-----------------------------------+-----------------------------------+
| 7                                 | Outstanding debt + Read more ...  |
|                                   |                                   |
|                                   | If the debt is still outstanding  |
|                                   | after failing to establish        |
|                                   | recovery agreement with provider, |
|                                   | consider possible actions:        |
|                                   |                                   |
|                                   | -   refer to Legals               |
|                                   | -   debt Waiver/Write off         |
|                                   |                                   |
|                                   | Note: only specific Aged Care     |
|                                   | Liaison officers have the         |
|                                   | Delegation under the Aged Care    |
|                                   | Act 1997 to waiver/Write Off      |
|                                   | amounts. The References page      |
|                                   | contains a link to legislation.   |
+-----------------------------------+-----------------------------------+
