Customer contact

Table 1: This table describes the action required when a customer
responds to the Income Confirmation letter.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Initiation + Read more ...        |
|                                   |                                   |
|                                   | Income Confirmation letter is     |
|                                   | issued when an Intervention has   |
|                                   | been initiated.                   |
|                                   |                                   |
|                                   | The letter requires the customer  |
|                                   | to contact to confirm or update   |
|                                   | their income details.             |
|                                   |                                   |
|                                   | The letter will include the       |
|                                   | following information:            |
|                                   |                                   |
|                                   | -   The request for the customer  |
|                                   |     to contact within 28 days     |
|                                   | -   What will happen if they do   |
|                                   |     not contact, we may apply the |
|                                   |     income information to the     |
|                                   |     customer's record which may   |
|                                   |     result in a debt that will    |
|                                   |     need to be repaid             |
|                                   |                                   |
|                                   | Has contact been made?            |
|                                   |                                   |
|                                   | -   Yes, follow the phone contact |
|                                   |     requirements then go to Step  |
|                                   |     2                             |
|                                   | -   No, refer to the Customer     |
|                                   |     fails to respond table        |
+-----------------------------------+-----------------------------------+
| 2                                 | View intervention in Customer     |
|                                   | First + Read more ...             |
|                                   |                                   |
|                                   | In Customer First:                |
|                                   |                                   |
|                                   | -   select the CRN/BP hyperlink   |
|                                   |     to display the Customer       |
|                                   |     Details screen                |
|                                   | -   scroll down and select        |
|                                   |     Interaction History           |
|                                   |                                   |
|                                   | The Integrated Review (Manual     |
|                                   | Income Confirmation) will have a  |
|                                   | status of Open.                   |
|                                   |                                   |
|                                   | Select the hyperlink then the     |
|                                   | Note tab to display any           |
|                                   | information regarding previous    |
|                                   | contact.                          |
|                                   |                                   |
|                                   | Upon all contact, the CO must     |
|                                   | check the Dates tab within the    |
|                                   | Integrated Review (Manual Income  |
|                                   | Confirmation).                    |
|                                   |                                   |
|                                   | Has the Initial Notification      |
|                                   | Receipt Date populated?           |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, the CO will need to confirm   |
|                                   | the letter has been delivered in  |
|                                   | order to populate the Initial     |
|                                   | Notification Receipt Date and     |
|                                   | commence the countdown. Go to     |
|                                   | Record letter as delivered table  |
|                                   | in Registered Mail Processing     |
|                                   | then go to Step 3                 |
|                                   |                                   |
|                                   | -   The CO will need to take the  |
|                                   |     same action where the         |
|                                   |     Integrated Review (Manual     |
|                                   |     Income Confirmation) has a    |
|                                   |     status of For Manual Action   |
|                                   |     with the status reason        |
|                                   |     Returned to Sender or Aus     |
|                                   |     Post Tracking Expired         |
+-----------------------------------+-----------------------------------+
| 3                                 | Customer conversation + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Advise the customer of the        |
|                                   | purpose of the intervention:      |
|                                   |                                   |
|                                   | 'Thank you for contacting us      |
|                                   | today. As the letter states, we   |
|                                   | have received information about   |
|                                   | your income that we need your     |
|                                   | help to confirm or update. The    |
|                                   | income information is different   |
|                                   | to what you told us when getting  |
|                                   | a payment. We use your income     |
|                                   | information so you get paid the   |
|                                   | right amount.'                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Locate match data + Read more ... |
|                                   |                                   |
|                                   | The CO will be required to locate |
|                                   | the match data in order to        |
|                                   | address the risk.                 |
|                                   |                                   |
|                                   | Information regarding the         |
|                                   | compliance trigger is available   |
|                                   | via the Match Data (IRMD) screen  |
|                                   | in the Integrated Review System   |
|                                   | (IRS).                            |

Customer conversation

Table 2: This table describes what information must be discussed to
establish the customer's correct income.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Discuss match data + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | A match has occurred because the  |
|                                   | information on the customer's     |
|                                   | Centrelink record differs from    |
|                                   | the information received from a   |
|                                   | third party for example,          |
|                                   | Australian Taxation Office (ATO), |
|                                   | Department of Education, Skills   |
|                                   | and Employment.                   |
|                                   |                                   |
|                                   | Check the income details on the   |
|                                   | customer's Centrelink record and  |
|                                   | discuss the discrepancy with the  |
|                                   | customer. For further             |
|                                   | information, refer to the         |
|                                   | relevant 'screen' column in       |
|                                   | Assessing and Coding Income and   |
|                                   | Assets. The Resources page        |
|                                   | contains a link to this file.     |
|                                   |                                   |
|                                   | The scope start date will either  |
|                                   | be:                               |
|                                   |                                   |
|                                   | -   the commencement date of the  |
|                                   |     income, or                    |
|                                   | -   if the commencement date is   |
|                                   |     greater than 7 years, the     |
|                                   |     scope start date will be 1    |
|                                   |     July of the financial year, 7 |
|                                   |     years prior to the current    |
|                                   |     financial year. For example,  |
|                                   |     for the 2019/2020 financial   |
|                                   |     year, the scope start date is |
|                                   |     1 July 2012                   |
|                                   |                                   |
|                                   | The scope end date will be        |
|                                   | determined by whether the         |
|                                   | customer is still receiving       |
|                                   | income or not. If the customer:   |
|                                   |                                   |
|                                   | -   is still receiving income     |
|                                   |     from self-employment or a     |
|                                   |     trust, the scope end date     |
|                                   |     will be Date Paid To          |
|                                   | -   has ceased receiving income,  |
|                                   |     the scope end date will be    |
|                                   |     the cessation date            |
|                                   |                                   |
|                                   | It is important the Compliance    |
|                                   | Officer (CO) discusses the        |
|                                   | amounts listed in any of the      |
|                                   | income fields on the Match Data   |
|                                   | (IRMD) screen.                    |
|                                   |                                   |
|                                   | COs should use their questioning  |
|                                   | skills to obtain as much          |
|                                   | information as possible from the  |
|                                   | customer over the phone.          |
|                                   |                                   |
|                                   | If the customer indicates the     |
|                                   | information is incorrect, the CO  |
|                                   | is to thoroughly review the       |
|                                   | customer record to determine an   |
|                                   | anomaly or mismatch.              |
|                                   |                                   |
|                                   | Inform the customer:              |
|                                   |                                   |
|                                   | 'Would you mind holding the line  |
|                                   | while I check your record'        |
|                                   |                                   |
|                                   | Preliminary assessment            |
|                                   |                                   |
|                                   | COs must ensure that they use all |
|                                   | information available to          |
|                                   | determine if an anomaly or        |
|                                   | mismatch exists between the       |
|                                   | customer and the match data.      |
|                                   |                                   |
|                                   | If at least 2 commonalities can   |
|                                   | be identified (for example, name  |
|                                   | and date of birth matches, or     |
|                                   | name and address matches) between |
|                                   | the customer and the match data,  |
|                                   | it can be determined that the     |
|                                   | match data is not a mismatch and  |
|                                   | should be continued. Extra care   |
|                                   | must be taken in the case of      |
|                                   | twins or intertwined records.     |
|                                   |                                   |
|                                   | Alternative spelling or           |
|                                   | misspelling of a customer's name  |
|                                   | does not automatically constitute |
|                                   | a mismatch.                       |
|                                   |                                   |
|                                   | Can an anomaly or mismatch be     |
|                                   | identified from the customer      |
|                                   | record?                           |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   Advise the customer that      |
|                                   |     further investigation will    |
|                                   |     have to occur and that        |
|                                   |     someone will contact them to  |
|                                   |     advise the outcome and next   |
|                                   |     steps                         |
|                                   | -   Advise the customer that if   |
|                                   |     they cannot be contacted,     |
|                                   |     they will need to contact     |
|                                   |     back to complete their        |
|                                   |     intervention within 14 days.  |
|                                   |     If they fail to contact       |
|                                   |     within 14 days, we may apply  |
|                                   |     the match data to their       |
|                                   |     record and this may result in |
|                                   |     a debt. Go to Step 2          |
|                                   |                                   |
|                                   | No, the intervention is to be     |
|                                   | continued. The CO must refer to   |
|                                   | the relevant table in order to    |
|                                   | assess the income:                |
|                                   |                                   |
|                                   | -   Family Day Care (FDC) income  |
|                                   | -   Trust income                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Anomaly or mismatch identified +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a discrepancy is identified    |
|                                   | between the customer's details    |
|                                   | and the match data, referral to a |
|                                   | Quality Development Officer (QDO) |
|                                   | is required.                      |
|                                   |                                   |
|                                   | Prior to referral, the CO is to   |
|                                   | cancel the Integrated Review      |
|                                   | (Manual Income Confirmation) in   |
|                                   | Customer First to enable Manual   |
|                                   | IRS Processing:                   |
|                                   |                                   |
|                                   | -   select Edit from the          |
|                                   |     Integrated Review (ZIRV) in   |
|                                   |     Customer First                |
|                                   | -   select Cancelled from the     |
|                                   |     Activity Status drop down     |
|                                   |     menu                          |
|                                   | -   select Manual IRS Processing  |
|                                   |     Required button               |
|                                   | -   select Save                   |
|                                   | -   add the keyword MICAMM on     |
|                                   |     IRMR                          |
|                                   | -   put intervention on HLD for   |
|                                   |     14 days on the Review Contact |
|                                   |     Details (IRCD) screen         |
|                                   |                                   |
|                                   | The QDO is to investigate the     |
|                                   | mismatch to validate the          |
|                                   | referral. These can be located by |
|                                   | undertaking an IRNS search using  |
|                                   | the keyword MICAMM.               |
|                                   |                                   |
|                                   | Where the mismatch referral is    |
|                                   | not supported the QDO is required |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   delete the keyword MICAMM     |
|                                   | -   return the referral to the CO |
|                                   |     with feedback explaining the  |
|                                   |     referral error                |
|                                   |                                   |
|                                   | Where the mismatch referral is    |
|                                   | supported the QDO is required to: |
|                                   |                                   |
|                                   | delete the keyword MICAMM         |
|                                   |                                   |
|                                   | add the keyword AMMYES            |
|                                   |                                   |
|                                   | ensure the intervention Notes     |
|                                   | reflect validation of the         |
|                                   | anomaly/mismatch                  |
|                                   |                                   |
|                                   | forward the following information |
|                                   | to the Customer Compliance        |
|                                   | Service Desk via the Customer     |
|                                   | Compliance Service Desk Online    |
|                                   | Form:                             |
|                                   |                                   |
|                                   | -   the Customer Reference Number |
|                                   |     (CRN)                         |
|                                   | -   details of the mismatch       |
|                                   | -   any additional comments that  |
|                                   |     may assist the service desk   |
|                                   |     in their assessment of the    |
|                                   |     review                        |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the Customer Compliance   |
|                                   | Service Desk Online Form.         |
|                                   |                                   |
|                                   | Additional risk                   |
|                                   |                                   |
|                                   | If during the intervention        |
|                                   | process it is identified that the |
|                                   | customer has an additional risk,  |
|                                   | this must be investigated and     |
|                                   | updated as part of the current    |
|                                   | intervention.                     |
|                                   |                                   |
|                                   | For further information on how to |
|                                   | action the additional risk(s),    |
|                                   | see the Assessing Additional Risk |
|                                   | Guide table on the Resources      |
|                                   | page, which will help with the    |
|                                   | assessment.                       |

Family Day Care (FDC) income

Table 3: This table describes the steps involved in assessing Family Day
Care (FDC) income.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm Family Day Care (FDC)     |
|                                   | self-employment and FDC income +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Compliance Officers (CO) need to  |
|                                   | confirm if the customer is still  |
|                                   | a Family Day Care (FDC) Educator. |
|                                   |                                   |
|                                   | CO is to inform the customer:     |
|                                   |                                   |
|                                   | 'We have received information     |
|                                   | from the Department of Education, |
|                                   | Skills and Employment, that you   |
|                                   | have been working as an educator  |
|                                   | for a Family Day Care Provider.   |
|                                   |                                   |
|                                   | The information we have received  |
|                                   | indicates that for the financial  |
|                                   | year/s XXXX you worked for XXXX   |
|                                   | and charged fees whilst providing |
|                                   | care.'                            |
|                                   |                                   |
|                                   | CO is to ask the customer:        |
|                                   |                                   |
|                                   | -   When did you start working as |
|                                   |     a Family Day Care educator?   |
|                                   | -   Which service providers, past |
|                                   |     and present do you have       |
|                                   |     contracts with?               |
|                                   | -   Have you told Centrelink      |
|                                   |     about this business? (Check   |
|                                   |     REBS, EANS, OINS to make sure |
|                                   |     business is coded)            |
|                                   | -   Are you still working as a    |
|                                   |     Family Day Care Educator? If  |
|                                   |     not, when did you cease?      |
|                                   | -   How are /were you paid?       |
|                                   |     (Discuss, for example,        |
|                                   |     Provider, parents, bank       |
|                                   |     account, cash)                |
|                                   |                                   |
|                                   | CO is to ensure that the          |
|                                   | following questions are asked for |
|                                   | each relevant financial year.     |
|                                   |                                   |
|                                   | -   What services do you provide? |
|                                   |     (Before/after school care or  |
|                                   |     all day care?)                |
|                                   | -   How many hours a day do you   |
|                                   |     provide care?                 |
|                                   | -   How many days per week do you |
|                                   |     provide care?                 |
|                                   | -   Do you provide additional     |
|                                   |     care during school holidays?  |
|                                   |     If yes how many weeks in a    |
|                                   |     year do you provide care for? |
|                                   |     Does this number vary?        |
|                                   | -   How many children do you      |
|                                   |     provide care for? Does this   |
|                                   |     number vary?                  |
|                                   | -   What are the ages of the      |
|                                   |     children?                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Assets and Expenses + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | CO needs to establish what assets |
|                                   | the customer has purchased to     |
|                                   | operate the business and what     |
|                                   | expenses are for the business.    |
|                                   |                                   |
|                                   | Note: for all types of expenses   |
|                                   | it is important to ensure that    |
|                                   | the amount claimed is the         |
|                                   | business portion and not the full |
|                                   | cost of the expense.              |
|                                   |                                   |
|                                   | CO is to ask the customer:        |
|                                   |                                   |
|                                   | Assets                            |
|                                   |                                   |
|                                   | -   Are there assets that you     |
|                                   |     have purchased in order to    |
|                                   |     operate your business that    |
|                                   |     you have not previously       |
|                                   |     advised us?                   |
|                                   |                                   |
|                                   | Depreciation                      |
|                                   |                                   |
|                                   | -   What was the cost of the      |
|                                   |     item?                         |
|                                   | -   What percentage do you use    |
|                                   |     the item for your business?   |
|                                   |                                   |
|                                   | Food                              |
|                                   |                                   |
|                                   | -   Do you provide food? Is it    |
|                                   |     meals or just snacks?         |
|                                   |                                   |
|                                   | Household expenses                |
|                                   |                                   |
|                                   | -   How many rooms are in your    |
|                                   |     home? How many rooms are used |
|                                   |     for child care compared to    |
|                                   |     the total number of rooms in  |
|                                   |     the home?                     |
|                                   | -   Do you use any room           |
|                                   |     exclusively for your          |
|                                   |     business?                     |
|                                   | -   Do you own or rent your home? |
|                                   |     How much rent do you pay each |
|                                   |     week? How much mortgage do    |
|                                   |     you pay per month?            |
|                                   | -   What are your total annual    |
|                                   |     household costs? (Any items   |
|                                   |     that relate to home expenses  |
|                                   |     should be included, for       |
|                                   |     example, utilities, phone,    |
|                                   |     internet, insurance etc.)     |
|                                   |                                   |
|                                   | Motor vehicle                     |
|                                   |                                   |
|                                   | -   Do you require a car for your |
|                                   |     business?                     |
|                                   | -   Do you own or lease the car?  |
|                                   | -   Do you drop off/pick up       |
|                                   |     children?                     |
|                                   | -   What calculation method was   |
|                                   |     used when claiming motor      |
|                                   |     vehicle expenses? For         |
|                                   |     example: log book, cents per  |
|                                   |     kilometre, 12% of original    |
|                                   |     value, 1/3 of actual expenses |
|                                   | -   Is the portion claimed the    |
|                                   |     full cost or the business     |
|                                   |     portion only?                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Undeclared or previously declared |
|                                   | income + Read more ...            |
|                                   |                                   |
|                                   | CO is to check the following to   |
|                                   | see if already coded:             |
|                                   |                                   |
|                                   | -   Investment Update: Current    |
|                                   |     bank balances                 |
|                                   | -   Check SVS/MIS/SIS and update  |
|                                   | -   Check REBS for                |
|                                   |     self-employment               |
|                                   |                                   |
|                                   | If the customer is self-employed, |
|                                   | they must provide a MOD F and a   |
|                                   | Profit and Loss Statement (SU580) |
|                                   | at the time of disclosure of this |
|                                   | circumstance for each sole trader |
|                                   | and partnership business.         |
|                                   |                                   |
|                                   | Each MOD F should be returned     |
|                                   | with the required documentation   |
|                                   | (as outlined within the MOD F).   |
|                                   | For new businesses not yet        |
|                                   | recorded on the customer’s record |
|                                   | Interim statements may be         |
|                                   | requested if the business has     |
|                                   | been operating for less than      |
|                                   | twelve months or the last         |
|                                   | statement does not accurately     |
|                                   | reflect the current business      |
|                                   | situation. Customers can obtain   |
|                                   | these forms from                  |
|                                   | service                           |
|                                   | saustralia.gov.au/customer/forms. |
|                                   |                                   |
|                                   | If the customer has a company or  |
|                                   | trust set up for their Family Day |
|                                   | Care business, they must provide  |
|                                   | a MOD PC or MOD PT. The modules   |
|                                   | should be returned with the       |
|                                   | required documentation (as        |
|                                   | outlined within the forms).       |
|                                   | Customers can obtain these forms  |
|                                   | from                              |
|                                   | service                           |
|                                   | saustralia.gov.au/customer/forms. |
|                                   |                                   |
|                                   | If the customer/partner is paying |
|                                   | themselves a wage from their own  |
|                                   | business the amount of wages      |
|                                   | should be added back to the       |
|                                   | business income rather than       |
|                                   | recorded on EANS (unless it is a  |
|                                   | company rather than a sole trader |
|                                   | or partnership).                  |
|                                   |                                   |
|                                   | Previously declared by customer   |
|                                   |                                   |
|                                   | Where customers have provided up  |
|                                   | to date information on REBS we    |
|                                   | need to advise them that we will  |
|                                   | review their expenses. If there   |
|                                   | are any discrepancies we will     |
|                                   | contact them, and they may be     |
|                                   | required to provide further       |
|                                   | verification.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Supporting documents + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | If documented verification is     |
|                                   | required, the customer must be    |
|                                   | given the opportunity to provide  |
|                                   | suitable verification documents   |
|                                   | before contacting any third       |
|                                   | parties.                          |
|                                   |                                   |
|                                   | Mandatory evidence                |
|                                   |                                   |
|                                   | In addition to all verbal updates |
|                                   | obtained policy dictates that     |
|                                   | mandatory documented evidence is  |
|                                   | also required for FDC cases as    |
|                                   | income needs to be verified.      |
|                                   |                                   |
|                                   | Where the customer has advised    |
|                                   | that they have ceased being a FDC |
|                                   | Educator, the CO is to request    |
|                                   | verification from the customer,   |
|                                   | remembering that the match year   |
|                                   | may still require verification.   |
|                                   | For further information, see FDC  |
|                                   | Cessation in Source Document      |
|                                   | Options. A link to this file can  |
|                                   | be found on the Resources page.   |
|                                   |                                   |
|                                   | CO is to ask customer:            |
|                                   |                                   |
|                                   | 'Have you lodged your tax returns |
|                                   | for this business with the ATO?'  |
|                                   |                                   |
|                                   | If yes, request full Income Tax   |
|                                   | Returns (ITR) including balance   |
|                                   | sheets, profit and loss statement |
|                                   | and a depreciation schedule       |
|                                   | detailing the assets purchased.   |
|                                   |                                   |
|                                   | -   Please provide year ………… and  |
|                                   |     …………..                        |
|                                   | -   Please provide your Education |
|                                   |     Payment History Report from   |
|                                   |     ……… to ………                    |
|                                   | -   Are you registered for Self   |
|                                   |     Service?                      |
|                                   |                                   |
|                                   | 'Has your income changed since    |
|                                   | your last ITR?'                   |
|                                   |                                   |
|                                   | If yes, request a Profit and Loss |
|                                   | statement and Education Payment   |
|                                   | History Report.                   |
|                                   |                                   |
|                                   | If the customer requires further  |
|                                   | information or they ask for the   |
|                                   | request in writing, they should   |
|                                   | be referred to                    |
|                                   | servicesaustralia.gov.au/individ  |
|                                   | uals/subjects/compliance-program. |
|                                   |                                   |
|                                   | If the customer informs they have |
|                                   | not lodged their tax returns the  |
|                                   | CO is to:                         |
|                                   |                                   |
|                                   | -   ask the customer when they    |
|                                   |     will be lodging their tax     |
|                                   |     returns for the business and  |
|                                   |     reason for not lodging tax    |
|                                   |     returns. If the customer will |
|                                   |     be lodging their tax return,  |
|                                   |     the CO is to advise the CO    |
|                                   |     that they have 14 days from   |
|                                   |     receiving the tax notice      |
|                                   |     assessment to provide their   |
|                                   |     full ITR to the agency        |
|                                   | -   request a Profit and Loss     |
|                                   |     statement and Education       |
