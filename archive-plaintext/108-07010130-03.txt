Determining eligibility for the Work Bonus and balance checking

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for Work Bonus + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Work Bonus applies to customers   |
|                                   | with eligible income who are over |
|                                   | Age Pension age and are claiming  |
|                                   | or in receipt of one of the       |
|                                   | following payments:               |
|                                   |                                   |
|                                   | -   Age Pension                   |
|                                   | -   Carer Payment                 |
|                                   | -   Disability Support Pension    |
|                                   |     (DSP)                         |
|                                   | -   Income Support Supplement     |
|                                   |     (ISS)                         |
|                                   | -   Service Pension (DVA)         |
|                                   | -   Age Pension (AGC)             |
|                                   |                                   |
|                                   | Blind pensioners: Customers       |
|                                   | receiving Age Pension (Blind) or  |
|                                   | DSP (Blind) may be exempt from    |
|                                   | the Means Test unless applying    |
|                                   | for Rent Assistance, but income   |
|                                   | and assets details are required   |
|                                   | to determine their partner's      |
|                                   | entitlement. However, all Blind   |
|                                   | pensioners over Age Pension age   |
|                                   | will accrue a Work Bonus balance  |
|                                   | (and deplete it if advising       |
|                                   | enough employment income). When   |
|                                   | determining the amount of the     |
|                                   | balance, it is assumed that Blind |
|                                   | pensioners not required to report |
|                                   | are earning nil.                  |
|                                   |                                   |
|                                   | Is the customer eligible for Work |
|                                   | Bonus?                            |
|                                   |                                   |
|                                   | Yes, and the customer:            |
|                                   |                                   |
|                                   | -   has self-employment, go to    |
|                                   |     Step 2                        |
|                                   | -   does not have self-employment |
|                                   |     income, go to Step 5          |
|                                   |                                   |
|                                   | No, Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Self-employment income from       |
|                                   | personal exertion + Read more ... |
|                                   |                                   |
|                                   | From 1 July 2019 Work Bonus       |
|                                   | applies to income earned from     |
|                                   | personal exertion in              |
|                                   | self-employment.                  |
|                                   |                                   |
|                                   | This includes attribution and     |
|                                   | distribution income from private  |
|                                   | trusts and companies generated by |
|                                   | the customer's personal exertion. |
|                                   | For example truck driving,        |
|                                   | bookkeeping, or sales.            |
|                                   |                                   |
|                                   | It does not include management or |
|                                   | administration of:                |
|                                   |                                   |
|                                   | -   financial investments         |
|                                   | -   real estate                   |
|                                   | -   the customer's home property, |
|                                   |     or                            |
|                                   | -   domestic or household         |
|                                   |     maintenance tasks             |
|                                   |                                   |
|                                   | Is the self-employment income     |
|                                   | derived from the customer's       |
|                                   | personal exertion?                |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, the Work Bonus cannot be      |
|                                   | applied to the self-employment    |
|                                   | income. If the customer:          |
|                                   |                                   |
|                                   | -   also has employment income,   |
|                                   |     go to Step 5                  |
|                                   | -   does not have employment      |
|                                   |     income, procedure ends here   |
+-----------------------------------+-----------------------------------+
| 3                                 | Work Bonus income percentage +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer will be eligible  |
|                                   | for the Work Bonus for            |
|                                   | self-employment the percentage of |
|                                   | their income that has been earned |
|                                   | from personal exertion will need  |
|                                   | to be recorded.                   |
|                                   |                                   |
|                                   | Work Bonus income % is            |
|                                   | displayed/updated/recorded on the |
|                                   | following screens:                |
|                                   |                                   |
|                                   | Sole Traders and partnerships-    |
|                                   | Business Details Summary (BUS)    |
|                                   |                                   |
|                                   | Overseas farm or business income- |
|                                   | Foreign Income Details (FID)      |
|                                   |                                   |
|                                   | Royalty or commission income-     |
|                                   | Other Income (OIN)                |
|                                   |                                   |
|                                   | Income from a farm (FRM)- Real    |
|                                   | Estate (RE)                       |
|                                   |                                   |
|                                   | Private trust and private company |
|                                   | income- Trust/Company             |
|                                   | Income/Assets (TRCIA)             |
|                                   |                                   |
|                                   | -   The Work Bonus Event Date is  |
|                                   |     the date from which the       |
|                                   |     income percentage is to be    |
|                                   |     applied                       |
|                                   | -   If directed to update TRCIA   |
|                                   |     as a result of a CAO          |
|                                   |     assessment, go to Step 4      |
|                                   |                                   |
|                                   | Note: both members of a           |
|                                   | partnership can have 100% of      |
|                                   | their income eligible for Work    |
|                                   | Bonus. For example, a customer    |
|                                   | and partner have a business       |
|                                   | partnership. They each receive    |
|                                   | 50% of the partnership income. If |
|                                   | all of each person's share of the |
|                                   | business income is from personal  |
|                                   | exertion, they will each have a   |
|                                   | Work Bonus income percent of 100. |
|                                   |                                   |
|                                   | Go to Step 5                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Work Bonus income percentage to   |
|                                   | be updated + Read more ...        |
|                                   |                                   |
|                                   | An update to the Work Bonus       |
|                                   | income percentage is needed when: |
|                                   |                                   |
|                                   | -   the trust or company is       |
|                                   |     assessed for the first time   |
|                                   | -   a percentage has not been     |
|                                   |     previously recorded           |
|                                   | -   a change in income earned     |
|                                   |     from the customer's personal  |
|                                   |     exertion has been identified. |
|                                   |     For example, the trust or     |
|                                   |     company now has an employee   |
|                                   |     who is undertaking work       |
|                                   |     previously done by the        |
|                                   |     customer                      |
|                                   |                                   |
|                                   | Is an update required?            |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   TRCIA cannot be updated in    |
|                                   |     Process Direct, to complete   |
|                                   |     the update, use Customer      |
|                                   |     First or Customer Record      |
|                                   | -   Go to the Trust/Company       |
|                                   |     Income/Assets Summary         |
|                                   |     (TRCIAS) screen               |
|                                   | -   Select the entity by entering |
|                                   |     'S' in the field next to the  |
|                                   |     entity name and press [Enter] |
|                                   | -   Enter the customer and        |
|                                   |     partner's (if applicable)     |
|                                   |     Work Bonus Income % as        |
|                                   |     advised by the customer       |
|                                   | -   Enter the Work Bonus Event    |
|                                   |     Date - this is the date the   |
|                                   |     Work Bonus Income % applies   |
|                                   |     from.                         |
|                                   |     The date cannot be before the |
|                                   |     first instance of income from |
|                                   |     the trust or company for the  |
|                                   |     customer                      |
|                                   | -   Enter the source and date of  |
|                                   |     receipt                       |
|                                   | -   Note: the TRCIAS screen is    |
|                                   |     able to be updated outside of |
|                                   |     any other outstanding         |
|                                   |     activities on the customer's  |
|                                   |     record (for example new       |
|                                   |     claims)                       |
|                                   |                                   |
|                                   | No, go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | How the Work Bonus is             |
|                                   | automatically applied + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Eligible customers will be        |
|                                   | credited with a Work Bonus amount |
|                                   | of $300 each fortnight.           |
|                                   |                                   |
|                                   | For short entitlement periods,    |
|                                   | the Work Bonus will be $300/14 x  |
|                                   | number of days in the period.     |
|                                   |                                   |
|                                   | If eligible income, variable or   |
|                                   | continuous is recorded, the       |
|                                   | system will automatically apply   |
|                                   | the Work Bonus and any balance to |
|                                   | discount the customer's eligible  |
|                                   | income.                           |
|                                   |                                   |
|                                   | Eligible income will be any       |
|                                   | income from employment or active  |
|                                   | participation in self-employment  |
|                                   | where the self-employment meets   |
|                                   | the personal exertion test.       |
|                                   |                                   |
|                                   | If the customer's gross           |
|                                   | fortnightly eligible income is:   |
|                                   |                                   |
|                                   | less than $300:                   |
|                                   |                                   |
|                                   | -   nil eligible income will be   |
|                                   |     assessed                      |
|                                   | -   unused Work Bonus ($300 less  |
|                                   |     eligible income) will be      |
|                                   |     banked                        |
|                                   |                                   |
|                                   | $300 or more, eligible income     |
|                                   | will be discounted by the Work    |
|                                   | Bonus ($300). If remaining        |
|                                   | eligible income is:               |
|                                   |                                   |
|                                   | -   less than or equal to their   |
|                                   |     Work Bonus balance, nil       |
|                                   |     income will be assessed and   |
|                                   |     the balance reduced by the    |
|                                   |     eligible income amount over   |
|                                   |     $300                          |
|                                   | -   more than their Work Bonus    |
|                                   |     balance, eligible income will |
|                                   |     be further discounted by      |
|                                   |     their available balance, and  |
|                                   |     their balance reduced to zero |
|                                   |                                   |
|                                   | A customer's Work Bonus balance   |
|                                   | cannot drop below zero or exceed  |
|                                   | $7,800 at any time.               |
|                                   |                                   |
|                                   | Note: between 1 December 2022 and |
|                                   | 31 December 2023, the maximum     |
|                                   | Work Bonus balance amount will be |
|                                   | increased from $7,800 to $11,800. |
|                                   | Eligible pensioners will also     |
|                                   | receive a one-off $4,000 increase |
|                                   | to their Work Bonus balance.      |
|                                   |                                   |
|                                   | The $7,800 accrual maximum may    |
|                                   | still display on the EIA screen   |
|                                   | and the customer’s online account |
|                                   | due to not having a Work Bonus    |
|                                   | affecting event since 1 December  |
|                                   | 2022. The system will apply the   |
|                                   | top up, and increase to the       |
|                                   | maximum Work Bonus balance if     |
|                                   | they declare Work Bonus eligible  |
|                                   | income.                           |
|                                   |                                   |
|                                   | Staff should complete a           |
|                                   | reassessment of the entitlement   |
|                                   | on the BA screen with a date of   |
|                                   | effect of today if a customer     |
|                                   | contacts, and they ask to see the |
|                                   | full amount,                      |
|                                   |                                   |
|                                   | See the Resources page for        |
|                                   | examples.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Employment Income Assessment -    |
|                                   | Work Bonus (EIA) screen + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The EIA screen shows the Work     |
|                                   | Bonus and balance calculations.   |
|                                   | See Resources for more            |
|                                   | information.                      |
|                                   |                                   |
|                                   | Manual updates                    |
|                                   |                                   |
|                                   | The EIA screen allows staff in    |
|                                   | the Department of Veterans'       |
|                                   | Affairs (DVA) Clearance Team      |
|                                   | staff to update a balance based   |
|                                   | on information received for a     |
|                                   | customer or their partner who has |
|                                   | previously accrued a balance on a |
|                                   | DVA pension, for example, when a  |
|                                   | DVA customer becomes partnered to |
|                                   | a Centrelink customer or when     |
|                                   | transferring agencies.            |
|                                   |                                   |
|                                   | Other appropriate network staff   |
|                                   | may make necessary changes to the |
|                                   | available balance amount if there |
|                                   | is a system requirement but this  |
|                                   | should only occur after approval  |
|                                   | from the relevant helpdesk.       |
|                                   |                                   |
|                                   | All manual adjustments to the EIA |
|                                   | screen must be clearly documented |
|                                   | on the customer's file, including |
|                                   | confirmation of approval from the |
|                                   | applicable helpdesk.              |
|                                   |                                   |
|                                   | Is the customer partnered?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 7                                 | Partnered customers + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The fortnightly Work Bonus and    |
|                                   | any carryover balance are applied |
|                                   | to the customer's eligible income |
|                                   | before any other calculations.    |
|                                   |                                   |
|                                   | Only eligible income in excess of |
|                                   | the Work Bonus is combined with   |
|                                   | any other income of the customer  |
|                                   | and their partner.                |
|                                   |                                   |
|                                   | This total is then assessed as if |
|                                   | they earned half each. The effect |
|                                   | of this is that the partner       |
|                                   | receives half the Work Bonus so   |
|                                   | they benefit if they are on any   |
|                                   | income support payment (even if   |
|                                   | one member of the couple is paid  |
|                                   | by DVA and the other by           |
|                                   | Centrelink).                      |
|                                   |                                   |
|                                   | If both are entitled to the Work  |
|                                   | Bonus, each individual's Work     |
|                                   | Bonus is applied to their own     |
|                                   | eligible income before the net    |
|                                   | amounts are added to other joint  |
|                                   | income and the income test is     |
|                                   | applied                           |
|                                   |                                   |
|                                   | If only one person is entitled to |
|                                   | the Work Bonus:                   |
|                                   |                                   |
|                                   | -   the Work Bonus is applied to  |
|                                   |     that person's eligible income |
|                                   | -   the person not entitled to    |
|                                   |     the Work Bonus will still     |
|                                   |     have the benefit of their     |
|                                   |     partner's reduced eligible    |
|                                   |     income being assessed         |
|                                   |                                   |
|                                   | See the Resources page for        |
|                                   | examples.                         |
+-----------------------------------+-----------------------------------+
| 8                                 | Current income test is applied +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When the eligible income          |
|                                   | assessment amount is calculated:  |
|                                   |                                   |
|                                   | -   Any other income (apart from  |
|                                   |     any direct deduction income   |
|                                   |     amount) is added to the       |
|                                   |     eligible income assessment    |
|                                   |     amount before the income test |
|                                   |     is applied for example,       |
|                                   |     income from investments,      |
|                                   |     income streams, rental        |
|                                   |     property income etc..         |
|                                   |                                   |
|                                   | For information on the income     |
|                                   | test, see:                        |
|                                   |                                   |
|                                   | -   Income Test for single        |
|                                   |     pension customers             |
|                                   | -   Income Test for partnered     |
|                                   |     pension customers             |
|                                   | -   Income Test for pension       |
|                                   |     customers with partners on    |
|                                   |     allowance                     |
|                                   | -   Income Test for illness       |
|                                   |     separated or partner in       |
|                                   |     prison customers              |
|                                   | -   Income and assets tests for   |
|                                   |     blind customers               |

| 9                                 | Receiving payment under           |
|                                   | transitional rules? + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer or their partner  |
|                                   | receiving payment under the       |
|                                   | transitional rules for pension    |
|                                   | customers?                        |
|                                   |                                   |
|                                   | If unsure, see the Grandfathering |
|                                   | Summary (GFS) screen for each     |
|                                   | person.                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 10            |
|                                   | -   No, procedure ends here       |

| 10                                | Rate comparison - transitional    |
|                                   | rate customers + Read more ...    |
|                                   |                                   |
|                                   | The system calculates and         |
|                                   | compares the following rates:     |
|                                   |                                   |
|                                   | -   the customer's transitional   |
|                                   |     rate using the old income     |
|                                   |     test that does not include    |
|                                   |     the Work Bonus                |
|                                   | -   the customer's rate using the |
|                                   |     current income test that      |
|                                   |     includes the Work Bonus       |
|                                   |                                   |
|                                   | Note: if the customer is          |
|                                   | partnered and both parties are    |
|                                   | entitled to the transitional      |
|                                   | rules, a separate rate            |
|                                   | calculation is required for each  |
|                                   | person.                           |
|                                   |                                   |
|                                   | For help, see Transitional rules  |
|                                   | for pension customers who were on |
|                                   | payment at 19 September 2009.     |
|                                   |                                   |
|                                   | Does the current income test      |
|                                   | including the Work Bonus result   |
|                                   | in a lower rate?                  |
|                                   |                                   |
|                                   | -   Yes, the customer's pension   |
|                                   |     is paid under the             |
|                                   |     transitional rules. The Work  |
|                                   |     Bonus is not used to          |
|                                   |     determine their rate. A       |
|                                   |     notional balance accrues and  |
|                                   |     depletes so that the          |
|                                   |     customer's rate under the     |
|                                   |     current income test can be    |
|                                   |     calculated and compared to    |
|                                   |     the transitional rate, and    |
|                                   |     for use in their partner's    |
|                                   |     income test.                  |
|                                   | -   No, the customer's pension is |
|                                   |     paid using the current income |
|                                   |     test (post 20 September 2009  |
|                                   |     income test), which includes  |
|                                   |     the application of the Work   |
|                                   |     Bonus.                        |
|                                   |                                   |
|                                   | The Resources page contains       |
|                                   | examples.                         |
