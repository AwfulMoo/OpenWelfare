Allowable Time Reached example

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Rejected Allowable Time Reached   |
|                                   | (REJ/ATR)                         |
|                                   |                                   |
|                                   | Eli is 27 and previously          |
|                                   | completed a 4-year Bachelor of    |
|                                   | Social Work. Eli has lodged an    |
|                                   | Austudy claim because he is now   |
|                                   | studying a 3-year Bachelor of     |
|                                   | Science starting on 26/02/2021.   |
|                                   |                                   |
|                                   | Taking into account Eli's         |
|                                   | previous completed 4 year TGB     |
|                                   | level course, he has already      |
|                                   | exceeded his Allowable Time.      |
|                                   |                                   |
|                                   | To reject the claim Allowable     |
|                                   | Time Reached (REJ/ATR), code the  |
|                                   | Inst Course Start Date: and       |
|                                   | Allowable Time End Date: fields   |
|                                   | with the day before the Student   |
|                                   | Start Date:                       |
|                                   |                                   |
|                                   | -   Inst Course Start Date:       |
|                                   |     25/02/2021                    |
|                                   | -   Student Start Date:           |
|                                   |     26/02/2021                    |
|                                   | -   Allowable Time End Date:      |
|                                   |     25/02/2021                    |
+-----------------------------------+-----------------------------------+

Student Programme Resource Tool

The Student Programme Resource Tool contains useful reference
information and links. It includes:

-   Allowable Time Calculator
-   Distance Education & Private Provider Date Calculator
-   Long Term Income Support Calculator

This tool is used to assist in the calculation of Allowable Time. Please
ensure you check the University EFTSL prior to using the calculator.
This tool is a guide only and used to assist in the calculation of
Allowable Time.
