|                                   | customer’s payday. The customer’s |
|                                   | EPED will either be 1 or 2 days   |
|                                   | prior to the payday, depending on |
|                                   | the payment type.                 |

|                                   | couple and Person Permitted to    |

|                                   |     provided to them about their  |
|                                   |     partner's payments            |
