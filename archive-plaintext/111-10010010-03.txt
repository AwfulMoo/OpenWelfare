+-----------------------------------+-----------------------------------+
| 1                                 | Viewing and saving the digital    |
|                                   | images for classification + Read  |
|                                   | more ...                          |

|                                   |                                   |
|                                   | Note: to receive Restricted       |
|                                   | Access work items, Support        |
|                                   | Officers must have the            |

|                                   | -   No, save the documents, then  |
|                                   |     go to Step 3                  |
|                                   |                                   |
|                                   | Staff must save each of the       |
|                                   | document/s to their desktop       |
|                                   | before continuing.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Complete all required             |
|                                   | classification fields + Read more |
|                                   | ...                               |
|                                   |                                   |

|                                   | -   Launch Customer First and     |
|                                   |     search for a customer         |
|                                   | -   Select Workspace or Quick     |
