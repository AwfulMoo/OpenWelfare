|                                   |                                   |
|                                   | -   favourable, and results in a  |
|                                   |     rate increase or resumption   |
|                                   |     after                         |
|                                   |     cancellation/suspension, or   |
|                                   | -   unfavourable, and results in  |
|                                   |     a rate reduction or           |
|                                   |     cancellation/suspension       |
|                                   |                                   |
|                                   | whether the determination is:     |
|                                   |                                   |
|                                   | -   automatic, or                 |
|                                   | -   non-automatic                 |

|                                   |                                   |
|                                   | -   the date of event (DOV) of    |
|                                   |     the change                    |

|                                   |     the change, and               |
|                                   | -   the date the change was       |
|                                   |     actioned                      |

|                                   | incorrectly have a debt raised    |
|                                   | with the automatic reason code    |
|                                   | IEA from 6 December 2020. See the |

|                                   |                                   |

| 8                                 | Notification Handler (NOHL) +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A generic NOHL operates across    |
|                                   | all payment systems. The NOHL:    |
|                                   |                                   |
|                                   | applies the date of event (DOV)   |
|                                   | rules for a statement reporter or |
|                                   | notification reporter             |
|                                   |                                   |
|                                   | evaluates:                        |
|                                   |                                   |
|                                   | -   when an event occurred        |
|                                   | -   when an event was reported    |
|                                   | -   when the event was actioned,  |
|                                   |     and                           |
|                                   | -   whether an assessment is a    |
|                                   |     review of decision            |
|                                   |                                   |
|                                   | The Review of Assessment (ROA)    |
|                                   | screen provides information to    |
|                                   | the NOHL to enable it to apply    |
|                                   | the correct DOE to favourable or  |
|                                   | unfavourable determinations of    |
|                                   | entitlement for most changes in   |
|                                   | circumstance.                     |
|                                   |                                   |
|                                   | NOHL:                             |
|                                   |                                   |
|                                   | -   offsets underpayments against |
|                                   |     overpayments within the debt  |
|                                   |     period                        |
|                                   | -   calculates a debt if          |
|                                   |     overpayments exceed           |
|                                   |     underpayments                 |
|                                   | -   sets the debt to zero for a   |
|                                   |     net underpayment              |
|                                   | -   will not pay arrears unless   |
|                                   |     the customer would be         |
|                                   |     normally entitled to those    |
|                                   |     arrears (for example,         |
|                                   |     separately to the debt        |
|                                   |     calculation)                  |
|                                   | -   offsets any arrears that are  |
|                                   |     payable against any           |
|                                   |     calculated debt amount, and   |
|                                   |     pays the balance to the       |
|                                   |     customer                      |
|                                   | -   calculates the DOE where the  |
|                                   |     correct DOV and DOR are used  |
|                                   |                                   |
|                                   | When calculating a debt, NOHL:    |
|                                   |                                   |
|                                   | -   cannot apply a DOE before a   |
|                                   |     limiting date                 |
|                                   | -   may not include payments      |
|                                   |     issued through a Payment      |
|                                   |     Adjustment (PYAJ) screen      |
|                                   |                                   |
|                                   | Once the debt shell has been      |
|                                   | created, any further updates made |
|                                   | for the same period will not      |
|                                   | alter the amount in the potential |
|                                   | debt. However, other actions that |
|                                   | are offset against potential      |
|                                   | debts such as restoration may     |
|                                   | affect the debt shell amount. To  |
|                                   | identify any additional amounts,  |
|                                   | select Related activities on the  |
|                                   | ADEX screen.                      |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Notification Handler (NOHL).      |

| 10                                | Underpayments outside the debt    |
|                                   | period + Read more ...            |
|                                   |                                   |
|                                   | Determine the debt period:        |
|                                   |                                   |
|                                   | -   start date using the date of  |
|                                   |     effect (DOE) provisions       |
|                                   |     contained in the legislation  |
|                                   |     that applies to that payment  |
|                                   | -   end date as the last day the  |
|                                   |     customer was overpaid         |
|                                   |                                   |
|                                   | An underpayment means that a      |
|                                   | customer would have been entitled |
|                                   | to receive a higher rate for a    |
|                                   | period, if they had notified      |
|                                   | correctly.                        |
|                                   |                                   |
|                                   | Do not use underpayments          |
|                                   | occurring outside of the debt to  |
|                                   | offset the debt amount unless the |
|                                   | arrears are legally payable.      |
|                                   | Apply favourable determinations   |
|                                   | for an event that is not          |
|                                   | correctly notified from the date  |
|                                   | of receipt (DOR) not the date of  |
|                                   | event (DOV).                      |
|                                   |                                   |
|                                   | See Payment of arrears after      |
|                                   | reassessment.                     |
|                                   |                                   |
|                                   | Underpayments that fall within    |
|                                   | the debt period form part of the  |
|                                   | debt calculation, and are used to |
|                                   | offset the debt amount.           |

| 1                                 | Family Assistance Law customers + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Family Tax Benefit (FTB) and      |
|                                   | Child Care Subsidy (CCS)          |
|                                   | customers must notify of any      |

|                                   | or affect eligibility as soon as  |

|                                   | related.                          |
|                                   |                                   |
|                                   | The family assistance and CCS     |
|                                   | reconciliation process determines |
|                                   | income related debts.             |
|                                   |                                   |
|                                   | Raise any FTB or CCS              |
|                                   | qualification debt that does not  |
|                                   | require verification of income    |
|                                   | immediately. Do not wait for the  |
|                                   | reconciliation process.           |
|                                   |                                   |
