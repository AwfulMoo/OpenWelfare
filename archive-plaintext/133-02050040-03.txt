How to create new mail in the Program mailbox

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Create new mail + Read more ...   |
|                                   |                                   |
|                                   | From the Mail Centre- My mailbox: |
|                                   |                                   |
|                                   | Select Compose new mail button    |
+-----------------------------------+-----------------------------------+
| 2                                 | Key new mail details + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key the following details:        |
|                                   |                                   |
|                                   | To field, key the provider stem   |
|                                   | into the box provided             |
|                                   |                                   |
|                                   | -   Tab out of the box and the    |
|                                   |     provider's name will be       |
|                                   |     displayed next to the field   |
|                                   | -   Check the displayed name to   |
|                                   |     ensure that the mail is going |
|                                   |     to the correct provider. Mail |
|                                   |     can only be sent to one       |
|                                   |     provider                      |
|                                   |                                   |
|                                   | Subject field, key the subject of |
|                                   | the mail being sent. Note: there  |
|                                   | are 250 characters to make the    |
|                                   | subject meaningful                |
|                                   |                                   |
|                                   | Text field, enter the text of the |
|                                   | mail                              |
|                                   |                                   |
|                                   | Attachments field:                |
|                                   |                                   |
|                                   | -   up to five attachments can be |
|                                   |     sent with the mail            |
|                                   | -   use the Browse button to      |
|                                   |     select the file to attach and |
|                                   |     select Add File to attach     |
|                                   |     them to the message           |
|                                   |                                   |
|                                   | to remove an attached file,       |
|                                   | select Remove (next to the        |
|                                   | attachment)                       |
|                                   |                                   |
|                                   | Select Send                       |
+-----------------------------------+-----------------------------------+

How to reply to mail in the Form upload mailbox

Table 6

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access HPOS Support Home Screen + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select                            |
|                                   |                                   |
|                                   | -   HPOS Support from the list    |
|                                   | -   Choose any source office code |
|                                   | -   Submit                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Form Upload + Read more ...       |
|                                   |                                   |
|                                   | Select Form Upload                |
+-----------------------------------+-----------------------------------+
| 3                                 | Reference number + Read more ...  |
|                                   |                                   |
|                                   | Use the HPOS reference number     |
|                                   | from the HPOS Reference Number    |
|                                   | field in PaNDA. For example,      |
|                                   | 123456789                         |
|                                   |                                   |
|                                   | -   Key reference number in Quick |
|                                   |     Search field                  |
|                                   | -   Select Search                 |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   If unable to locate the email |
|                                   |     that corresponds to the       |
|                                   |     reference number, contact     |
|                                   |     your Programme Support        |
|                                   |     Officer (PSO)                 |
|                                   | -   Reference numbers are made up |
|                                   |     of two parts, the first part  |
|                                   |     is the reference number and   |
|                                   |     the second part is the        |
|                                   |     sequence number. The sequence |
|                                   |     number will change when the   |
|                                   |     email has been sent again     |
|                                   |     with additional information   |
|                                   |     or when a Service Officer     |
|                                   |     replies to the message for    |
|                                   |     example 12345678-01,          |
|                                   |     1234567-02                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Open the message + Read more ...  |
|                                   |                                   |
|                                   | Select Reply                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Key mail details + Read more ...  |
|                                   |                                   |
|                                   | Key the following details:        |
|                                   |                                   |
|                                   | Text field (mandatory), enter the |
|                                   | text of the mail that is being    |
|                                   | attached to the letter. Note:     |
|                                   | there is a limit of 4800          |
|                                   | characters                        |
|                                   |                                   |
|                                   | Attachments field:                |
|                                   |                                   |
|                                   | -   up to five attachments can be |
|                                   |     sent with the mail            |
|                                   | -   use the Browse button to      |
|                                   |     select the file to attach and |
|                                   |     select Add File to attach     |
|                                   |     them to the message           |
|                                   | -   to remove an attached file,   |
|                                   |     select Remove (next to the    |
|                                   |     attachment)                   |
|                                   |                                   |
|                                   | Select Send                       |
+-----------------------------------+-----------------------------------+
| 6                                 | Return to Form Upload screen +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Form Upload from the Main  |
|                                   | menu                              |
