Referring notification of DVA income received to the Clearance Team

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Mutual customer contacts to       |
|                                   | advise they are in receipt of     |
|                                   | income or lump sum payments from  |
|                                   | DVA, including Wholly Dependent   |
|                                   | Partner payment + Read more ...   |
|                                   |                                   |
|                                   | To ensure payments have been      |
|                                   | recorded correctly, refer the     |
|                                   | details to DVA Clearance Team for |
|                                   | follow up.                        |
|                                   |                                   |
|                                   | Create an open work item using    |
|                                   | Fast Note:                        |
|                                   |                                   |
|                                   | -   Select Auto Text use          |
|                                   |     Generic > Enquiry > DVA       |
|                                   |     Clearance/Advice Request      |
|                                   | -   Text: Remove the pre-filled   |
|                                   |     text and replace with details |
|                                   |     of payment type, frequency,   |
|                                   |     amounts and the dates         |
|                                   |     payments have been received   |
|                                   |     from DVA.                     |
|                                   | -   Select Confirm                |
|                                   |                                   |
|                                   | Advise the customer referral has  |
|                                   | been made to the Clearance Team.  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
