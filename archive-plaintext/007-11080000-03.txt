| 4                                 | Family Tax Benefit (FTB) Part B   |
|                                   | Quarantine multiple partner       |
|                                   | scenario + Read more ...          |
|                                   |                                   |
|                                   | This scenario is where:           |
|                                   |                                   |
|                                   | -   the customer has more than    |
|                                   |     one partner in the financial  |
|                                   |     year and is the lower income  |
|                                   |     earner for one partner but    |
|                                   |     not for the other(s); the     |
|                                   |     other partner(s) was also the |
|                                   |     lower income earner           |
|                                   | -   a return to work is recorded  |
|                                   |     in relation to an eligible    |
|                                   |     Part B quarantine child for   |
|                                   |     more than one lower income    |
|                                   |     earner (for example, the      |
|                                   |     customer and a partner, both  |
|                                   |     partners, or all 3)           |
|                                   |                                   |
|                                   | The system will apply a potential |
|                                   | quarantine period for the return  |
|                                   | to work recorded for each lower   |
|                                   | income earner.                    |
|                                   |                                   |
|                                   | Only one person's return to work  |
|                                   | in relation to a child attracts   |
|                                   | the FTB Part B quarantine. This   |
|                                   | is the return to work of the      |
|                                   | person for whom the potential FTB |
|                                   | Part B quarantine period begins   |
|                                   | first.                            |
|                                   |                                   |
|                                   | Eligibility for the quarantine    |
|                                   | measure is also subject to the    |
|                                   | Part B primary earner income      |
|                                   | limit, except for periods the     |
|                                   | customer or their partner         |
|                                   | received an income support        |
|                                   | payment. If the primary earner's  |
|                                   | income exceeds the limit, the     |
|                                   | customer is not eligible for      |
|                                   | quarantining.                     |
|                                   |                                   |
|                                   | Manual intervention is required   |
|                                   | to override those periods that    |
|                                   | are not the first return to work  |
|                                   | using the ineligibility code      |
|                                   | 'N1R'.                            |
|                                   |                                   |
|                                   | The manual intervention activity  |
|                                   | is identified by:                 |
|                                   |                                   |
|                                   | -   Keyword: #QMP                 |
|                                   | -   Reason: QMP                   |

|                                   | Note: the School Enrolment and    |

|                                   |                                   |
|                                   | From the December 2010 major      |
|                                   | systems release, there have been  |
|                                   | limited cases of continued SEAM   |
|                                   | suspension periods over 13 weeks  |
|                                   | (payments automatically cancelled |
|                                   | after 13 consecutive weeks and    |
|                                   | were subject to review and manual |
|                                   | determinations after 13           |
|                                   | cumulative weeks of suspension).  |
|                                   | There were also limited           |
|                                   | reconciliation periods prior to   |
|                                   | the December 2010 release that    |
|                                   | included SEAM suspension periods  |
|                                   | exceeding 13 continuous weeks. In |
|                                   | these limited cases, the          |
|                                   | following process applied:        |
|                                   |                                   |
|                                   | For reconciliation purposes,      |
|                                   | where a customer's income support |
|                                   | payment had a SEAM suspension of  |
|                                   | 13 or more continuous weeks, the  |
|                                   | customerâ€™s Family Tax Benefit     |
|                                   | (FTB) Part A and/or Child Care    |
|                                   | Benefit (CCB) were subject to the |
|                                   | income test for the entire        |
|                                   | suspension period. This included  |
|                                   | the initial 13 week suspension if |
|                                   | the entire period fell within the |
|                                   | same financial year.              |
|                                   |                                   |
|                                   | For suspension prior to the       |
|                                   | December 2010 release, the        |
|                                   | customer's record will have been  |
|                                   | suspended with one of the         |
|                                   | following codes:                  |
|                                   |                                   |
|                                   | -   'ATF' (Attendance not         |
|                                   |     satisfactory)                 |
|                                   | -   'ENF' (Enrolment details not  |
|                                   |     provided)                     |
|                                   |                                   |
|                                   | For suspension post the December  |
|                                   | 2010 release and up to 31         |
|                                   | December 2017, the customer's     |
|                                   | record will have been suspended   |
|                                   | with one of the following codes:  |
|                                   |                                   |
|                                   | -   'ATN' (Attendance not         |
|                                   |     satisfactory)                 |
|                                   | -   'ENN' (Enrolment details not  |
|                                   |     provided)                     |
|                                   |                                   |
|                                   | If the suspension spans 2         |
|                                   | financial years, only the         |
|                                   | suspension period in the second   |
|                                   | financial year is to be income    |
|                                   | tested. The income test will not  |
|                                   | apply to FTB Part A and/or CCB    |
|                                   | for the suspension period in the  |
|                                   | first financial year, regardless  |
|                                   | of the length of the suspension.  |
|                                   | If the income support payment was |
|                                   | suspended for less than 13        |
|                                   | continuous weeks, the customerâ€™s  |
|                                   | FTB Part A and/or CCB is free of  |
|                                   | the income test for the           |
|                                   | suspension period.                |
|                                   |                                   |
|                                   | If a customer's payment was       |
|                                   | 'SUS/ATF' or 'SUS/ENF' anytime    |
|                                   | during the financial year and is  |
|                                   | suspended as at 30 June, manual   |
|                                   | intervention is required if the   |
|                                   | payment was not being             |
|                                   | automatically assessed. The MIV   |
|                                   | team will override the suspension |
|                                   | period which has been assessed as |
|                                   | 'off income support'.             |
|                                   |                                   |
|                                   | Manual intervention is required   |
|                                   | to ensure that the above rules    |
|                                   | are applied correctly at          |
|                                   | reconciliation. Where required,   |
|                                   | the FAO ISP Override (FISPO)      |
|                                   | screen may be coded to correct    |
|                                   | the assessment.                   |
