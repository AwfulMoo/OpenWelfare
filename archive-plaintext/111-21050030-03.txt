Process Direct

Using Earnings and reporting workflow tool in Process Direct

This table describes the steps to enter Carer Payment (CP) details to
the Earnings and reporting workflow tool in Process Direct.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Messages screen in the Earnings   |
|                                   | and Reporting workflow in Process |
|                                   | Direct + Read more ...            |
|                                   |                                   |
|                                   | During the Earnings and Reporting |
|                                   | workflow in Process Direct, the   |
|                                   | following prompt will display on  |
|                                   | the Messages screen if the        |
|                                   | customer is currently receiving   |
|                                   | CP:                               |
|                                   |                                   |
|                                   | ‘Carer Payment customers can      |
|                                   | work, study, train or volunteer   |
|                                   | for 25 hours per week (including  |
|                                   | travel time). If the customer     |
|                                   | wishes to relinquish payment, do  |
|                                   | not continue in this workflow.’   |
|                                   |                                   |
|                                   | This message is provided to       |
|                                   | ensure staff are aware of the     |
|                                   | special processing that will be   |
|                                   | involved with CP customers. It    |
|                                   | also flags that if the customer   |
|                                   | is starting work and has arranged |
|                                   | ongoing alternate care that this  |
|                                   | should be dealt with as a         |
|                                   | cancellation of payment outside   |
|                                   | the Earnings and Reporting        |
|                                   | workflow.                         |
|                                   |                                   |
|                                   | -   For customers who want to     |
|                                   |     relinquish their Carer        |
|                                   |     Payment, go to Cancellation   |
|                                   |     of Carer Payment (CP).        |
|                                   |     Procedure ends here           |
|                                   | -   For customers who have        |
|                                   |     ongoing employment, where the |
|                                   |     total of all carer activities |
|                                   |     will exceed 25 hours per      |
|                                   |     week, on an ongoing basis,    |
|                                   |     see Carer Payment (CP) and    |
|                                   |     Carer Allowance (CA) carer    |
|                                   |     undertakes paid employment,   |
|                                   |     self-employment, voluntary    |
|                                   |     work, education or training   |
|                                   |     activities. Procedure ends    |
|                                   |     here                          |
|                                   |                                   |
|                                   | Staff cannot update               |
|                                   | self-employment or hours for      |
|                                   | carers using the Earnings and     |
|                                   | Reporting workflow. Advise CP     |
|                                   | customers who start or change     |
|                                   | their self-employment income to   |
|                                   | use the Manage Income and Assets  |
|                                   | online service or issue a Profit  |
|                                   | and Loss Statement (SU580). If    |
|                                   | their self-employment income is   |
|                                   | already recorded, staff can       |
|                                   | update their activity hours       |
|                                   | directly via the Carer Payment    |
|                                   | Activity Summary (CPAS) / Carer   |
|                                   | Payment Activity Details (CPAD)   |
|                                   | screens. For more information,    |
|                                   | see Carer Payment (CP) and Carer  |
|                                   | Allowance (CA) carer undertakes   |
|                                   | paid employment, self-employment, |
|                                   | voluntary work, education or      |
|                                   | training activities. Procedure    |
|                                   | ends here.                        |
|                                   |                                   |
|                                   | * Employment income paid in a     |
|                                   | foreign currency must be coded on |
|                                   | the Foreign Income Paid Details   |
|                                   | (FEIP) screen outside the         |
|                                   | workflow. FEIP coding does not    |
|                                   | create a Paid employment activity |
|                                   | and a workaround is required.     |
|                                   | Activity details for foreign      |
|                                   | employment income should be       |
|                                   | recorded using another activity   |
|                                   | type on CPAS, such as Voluntary   |
|                                   | work. Procedure ends here.        |
|                                   |                                   |
|                                   | All CP recipients with employment |
|                                   | income must have their income     |
|                                   | recorded as detailed in Recording |
|                                   | and correcting employment income  |
|                                   | details.                          |
|                                   |                                   |
|                                   | Note: for CP (XWP) customers,     |
|                                   | this message is not displayed as  |
|                                   | they are exempt from the 25 hour  |
|                                   | rule and CP activity reporting    |
|                                   | requirements There is no          |
|                                   | restriction on the number of      |
|                                   | hours CP (XWP) carers can be away |
|                                   | from their care receiver.         |
|                                   | Activity hours are not collected  |
|                                   | for these customers in the        |
|                                   | workflow or via self service      |
|                                   | channels. Procedure ends here for |
|                                   | CP (XWP) customers.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Updating Carer Payment Activity   |
|                                   | Details in Process Direct + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For customers receiving CP, the   |
|                                   | Carer Payment Activity Summary    |
|                                   | (CPAS) table will automatically   |
|                                   | appear.                           |
|                                   |                                   |
|                                   | If the customer is working or     |
|                                   | undertaking activities:           |
|                                   |                                   |
|                                   | select the pencil for the         |
|                                   | relevant activity type the        |
|                                   | customer is undertaking. The Edit |
|                                   | Carer Payment Activity Summary    |
|                                   | pop-up will appear:               |
|                                   |                                   |
|                                   | -   enter the name of the         |
|                                   |                                   |
|                                   | Employer/Organisation/Institution |
|                                   | -   select how many weeks need to |
|                                   |     be recorded (1 or 2)          |
|                                   | -   enter the hours and minutes   |
|                                   |     undertaken. If hours vary     |
|                                   |     each week, code each week     |
|                                   |     separately                    |
|                                   | -   Event Date: Only one event    |
|                                   |     date is required if the carer |
|                                   |     is working or undertaking an  |
|                                   |     activity involving regular    |
|                                   |     hours, and should generally   |
|                                   |     be the Entitlement Period     |
|                                   |     Start Day (EPSD). A second    |
|                                   |     event date can be recorded    |
|                                   |     and will default to the start |
|                                   |     date of the next week in the  |
|                                   |     entitlement period (EPSD +    |
|                                   |     7). Date can also be up to 30 |
|                                   |     days in the future to cease   |
|                                   |     the activity hours if not     |
|                                   |     continuing                    |
|                                   |     Note: for customers reporting |
|                                   |     late, where there is a need   |
|                                   |     to update the number of hours |
|                                   |     of their carer activity as a  |
|                                   |     result of updated employment  |
|                                   |     income, the system defaults   |
|                                   |     the event date the current    |
|                                   |     EPSD. Ensure the event dates  |
|                                   |     in the prompt are amended to  |
|                                   |     the EPSD for the outstanding  |
|                                   |     period                        |
|                                   | -   select Yes or No as to        |
|                                   |     whether the carer still       |
|                                   |     provided care while working.  |
|                                   |     If Yes, record how care was   |
|                                   |     met. Select Save and Next     |
|                                   |                                   |
|                                   | the Carer Payment Activity table  |
|                                   | will display. To view the updated |
|                                   | activity, select the relevant     |
|                                   | twisty for the Carer Payment      |
|                                   | Activity Details (CPAD) to        |
|                                   | display. Ensure details are       |
|                                   | correct and update using the      |
|                                   | pencil if required. Select Next   |
|                                   |                                   |
|                                   | if total activity hours exceed 25 |
|                                   | in a week and ‘care is not met’,  |
|                                   | a message will display about      |
|                                   | ongoing entitlement. See Carer    |
|                                   | Payment (CP) and Carer Allowance  |
|                                   | (CA) carer undertakes paid        |
|                                   | employment, self-employment,      |
|                                   | voluntary work, education or      |
|                                   | training activities. If customer  |
|                                   | is eligible to use respite:       |
|                                   |                                   |
|                                   | select Yes or No as to whether    |
|                                   | the customer would like to use    |
|                                   | respite or if the Carer Payment   |
|                                   | Activity Details (CPAD) need      |
|                                   | updating. Select Next             |
|                                   |                                   |
|                                   | if Yes to updating CPAD, the      |
|                                   | Carer Payment Activity table will |
|                                   | present. Select the pencil to     |
|                                   | make updates and Save. Select     |
|                                   | Next                              |
|                                   |                                   |
|                                   | if activity hours are still more  |
|                                   | than 25 hours where ‘care is not  |
|                                   | met’, the Temporary Cessation of  |
|                                   | care summary (TCC) table will     |
|                                   | display. To view absences taken,  |
|                                   | select the twisty for the care    |
|                                   | receiver. Select Next             |
|                                   |                                   |
|                                   | -   the Temporary Absence Details |
|                                   |     (ABSN) table will display.    |
|                                   |     Select [Adding a new line] to |
|                                   |     code the temporary absence if |
|                                   |     eligible and Save. See Carer  |
|                                   |     Payment (CP) and Carer        |
|                                   |     Allowance (CA) carer          |
|                                   |     undertakes paid employment,   |
|                                   |     self-employment, voluntary    |
|                                   |     work, education or training   |
|                                   |     activities. Select Next and   |
|                                   |     finalise the activity         |
+-----------------------------------+-----------------------------------+
| 3                                 | Care is met + Read more ...       |
|                                   |                                   |
|                                   | If care continued to be provided  |
|                                   | and the employment income         |
|                                   | activity is the carer's only      |
|                                   | activity, regardless of the hours |
|                                   | worked the customer can continue  |
|                                   | to receive CP subject to the      |
|                                   | income test.                      |
|                                   |                                   |
|                                   | For help, see Carer Payment (CP)  |
|                                   | and Carer Allowance (CA) carer    |
|                                   | undertakes paid employment,       |
|                                   | self-employment, voluntary work,  |
|                                   | education or training activities. |
|                                   |                                   |
|                                   | A prompt will display to advise   |
|                                   | the customer and ensure they are  |
|                                   | aware of the need to advise any   |
|                                   | change in care arrangements.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Carer's activity does not exceed  |
|                                   | 25 hours per week + Read more ... |
|                                   |                                   |
|                                   | If care is not met for any        |
|                                   | activities, but the total of all  |
|                                   | current activities is 25 hours or |
|                                   | less per week, the CP can         |
|                                   | continue, as total activities do  |
|                                   | not exceed 25 hours per week.     |
|                                   |                                   |
|                                   | For help, see Carer Payment (CP)  |
|                                   | and Carer Allowance (CA) carer    |
|                                   | undertakes paid employment,       |
|                                   | self-employment, voluntary work,  |
|                                   | education or training activities. |
+-----------------------------------+-----------------------------------+
| 5                                 | Carer's activity is more than 25  |
|                                   | hours per week + Read more ...    |
|                                   |                                   |
|                                   | If the total Carer activities are |
|                                   | more than 25 hours per week, an   |
|                                   | assessment of ongoing eligibility |
|                                   | will need to be undertaken.       |
|                                   |                                   |
|                                   | Finalise the activity once the    |
|                                   | Earnings (EAPP) and CPAD/CPAS     |
|                                   | screens have been updated.        |
|                                   |                                   |
|                                   | If more action is needed, a       |
|                                   | manual follow-up (MFU) activity   |
|                                   | will be created. This is actioned |
|                                   | by a Service Officer with         |
|                                   | appropriate Carers' knowledge.    |
|                                   |                                   |
|                                   | The MFU activity will appear on   |
|                                   | the Activity List (AL) screen. It |
|                                   | will contain the text 'Review     |
|                                   | Carer Payment eligibility         |
|                                   | immediately as total hours of     |
|                                   | employment/training/study plus    |
|                                   | travel time exceeds 25 hrs per    |
|                                   | week'.                            |
|                                   |                                   |
|                                   | More information is included in   |
|                                   | the activity as follows:          |
|                                   |                                   |
|                                   | -   cancel Carer Payment (CP) if  |
|                                   |     the customer has exceeded 25  |
|                                   |     hours per week and respite    |
|                                   |     has not been used; and        |
|                                   |     customer has no remaining     |
|                                   |     Working Credit balance        |
|                                   | -   if CP is to be cancelled, the |
|                                   |     customer’s Working Credit     |
|                                   |     balance has not yet reached   |
|                                   |     zero and customer has a       |
|                                   |     monthly (MTE) exception that  |
|                                   |     is not met, place the         |
|                                   |     customer on statement         |
|                                   |     reporting                     |
|                                   | -   a future cancellation may be  |
|                                   |     required after Working Credit |
|                                   |     balance has depleted to       |
|                                   |     zero - for customers under    |
|                                   |     Age Pension age               |
|                                   | -   use the Temporary Cessation   |
|                                   |     of Care workflow to code      |
|                                   |     respite where activities are  |
|                                   |     were going to exceed 25 hours |
|                                   |     on a one-off basis            |
|                                   |                                   |
|                                   | Respite cannot be used if the     |
|                                   | carer is involved in activities   |
|                                   | greater than 25 hours per week on |
|                                   | an ongoing basis. See Carer       |
|                                   | Payment (CP) and Carer Allowance  |
|                                   | (CA) carer undertakes paid        |
|                                   | employment, self-employment,      |
|                                   | voluntary work, education or      |
|                                   | training activities. Procedure    |
|                                   | ends here.                        |
|                                   |                                   |
|                                   | If more than one activity exists, |
|                                   | the system totals all the         |
|                                   | activities based on the event     |
|                                   | dates. For example, training or   |
|                                   | study in addition to the work.    |
|                                   |                                   |
|                                   | Carers are allowed to use respite |
|                                   | options on a one-off basis if:    |
|                                   |                                   |
|                                   | -   work, study, voluntary work   |
|                                   |     and/or training (including    |
|                                   |     total travel time) exceeds 25 |
|                                   |     hours per week on a temporary |
|                                   |     basis, and                    |
|                                   | -   care is continuing            |
|                                   |                                   |
|                                   | If the customer chooses:          |
|                                   |                                   |
|                                   | -   not to use respite, the       |
|                                   |     Service Officer will be       |
|                                   |     prompted to check details     |
|                                   |     with the carer to see if they |
|                                   |     are still eligible for        |
|                                   |     payment                       |
|                                   | -   to use respite, the Service   |
|                                   |     Officer should leave the      |
|                                   |     activity hours as a record    |
|                                   |     and Temporary Cessation of    |
|                                   |     Care screens will be          |
|                                   |     displayed to code this and    |
|                                   |     then finalise the Earnings    |
|                                   |     and Reporting workflow        |
|                                   |                                   |
|                                   | For help, see Carer Payment (CP)  |
|                                   | and Carer Allowance (CA) carer    |
|                                   | undertakes paid employment,       |
|                                   | self-employment, voluntary work,  |
|                                   | education or training activities. |

Customer First

Using Earnings and reporting workflow tool in Customer First

This table describes the steps to enter Carer Payment (CP) details to
the Earnings and reporting workflow tool in Customer First.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Employment income Summary         |
|                                   | Messaging + Read more ...         |
|                                   |                                   |
|                                   | During the Earnings and Reporting |
|                                   | workflow in Customer First, the   |
|                                   | following prompt will display on  |
|                                   | the Employment income Summary     |
|                                   | screen if the customer is         |
|                                   | currently receiving CP.           |
|                                   |                                   |
|                                   | 'CP customers can work, study,    |
|                                   | train or volunteer for 25 hours   |
|                                   | per week (including total         |
|                                   | travelling time). If the customer |
|                                   | wishes to relinquish payment, the |
|                                   | Earnings and Reporting workflow   |
|                                   | will create an activity, which    |
|                                   | will launch the Carer             |
|                                   | cancellation and suspension       |
|                                   | workflow.                         |
|                                   |                                   |
|                                   | This message is provided to       |
|                                   | ensure staff are aware of the     |
|                                   | special processing that will be   |
|                                   | involved with CP customers. It    |
|                                   | also flags that if the customer   |
|                                   | is starting work and has arranged |
|                                   | ongoing alternate care that this  |
|                                   | should be dealt with as a         |
|                                   | cancellation of payment outside   |
|                                   | the Earnings and Reporting        |
|                                   | workflow.                         |
|                                   |                                   |
|                                   | Note: staff cannot update         |
|                                   | self-employment income or hours   |
|                                   | for carers using the Earnings and |
|                                   | Reporting workflow. Advise CP     |
|                                   | customers who start or change     |
|                                   | their self-employment income to   |
|                                   | use the Manage Income and Assets  |
|                                   | online service or issue a Profit  |
|                                   | and Loss Statement (SU580). If    |
|                                   | their self-employment income is   |
|                                   | already recorded, staff can       |
|                                   | update their activity hours       |
|                                   | directly via the Carer Payment    |
|                                   | Activity Summary (CPAS) / Carer   |
|                                   | Payment Activity Details (CPAD)   |
|                                   | screens. For more information,    |
|                                   | see Carer Payment (CP) and Carer  |
|                                   | Allowance (CA) carer undertakes   |
|                                   | paid employment, self-employment, |
|                                   | voluntary work, education or      |
|                                   | training activities.              |
|                                   |                                   |
|                                   | If a Service Officer indicates    |
|                                   | that respite was used, a link is  |
|                                   | provided to the Temporary         |
|                                   | Cessation of Care workflow.       |
|                                   |                                   |
|                                   | Note: for CP (XWP) customers,     |
|                                   | this message is not displayed as  |
|                                   | they are exempt from the 25 hour  |
|                                   | rule and CP activity reporting    |
|                                   | requirements There is no          |
|                                   | restriction on the number of      |
|                                   | hours CP (XWP) carers can be away |
|                                   | from their care receiver.         |
|                                   | Activity hours are not collected  |
|                                   | for these customers in the        |
|                                   | workflow or via self service      |
|                                   | channels.                         |
|                                   |                                   |
|                                   | Employment income received by all |
|                                   | CP customers must still be        |
|                                   | recorded. See Recording and       |
|                                   | correcting employment income      |
|                                   | details.                          |
|                                   |                                   |
|                                   | Procedure ends here for CP (XWP)  |
|                                   | customers.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Employment income Update          |
|                                   | Messaging in Customer First +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A mandatory Carer Payment Details |
|                                   | question set will be displayed    |
|                                   | for each employer on the          |
|                                   | Employment Income Update screen   |
|                                   | in the Earnings and Reporting     |
|                                   | workflow when a new occurrence of |
|                                   | employment income for a CP        |
|                                   | customer has been inserted. Note: |
|                                   | the Carer Payment Details         |
|                                   | question set is not displayed for |
|                                   | CP (XWP) customers. Their         |
|                                   | activity hours are not assessed.  |
|                                   |                                   |
|                                   | Have the employment activity      |
|                                   | hours changed since 'DDMONYYYY'   |
|                                   | (currently 'hh/mins' per week)?   |
|                                   |                                   |
|                                   | This question is to determine if  |
|                                   | the carer's employment activity   |
|                                   | hours have changed since last     |
|                                   | reported. The answer is defaulted |
|                                   | to 'Yes'. Changing to a 'No'      |
|                                   | response will inhibit the rest of |
|                                   | the question set. Where a         |
|                                   | customer is continuing to work    |
|                                   | regular hours, there is no need   |
|                                   | to update the hours.              |
|                                   |                                   |
|                                   | For customers reporting late,     |
|                                   | where there is a need to update   |
|                                   | the number of hours of their      |
|                                   | carer activity because of updated |
|                                   | employment income, the system     |
|                                   | defaults the event date to the    |
|                                   | current Entitlement Period Start  |
|                                   | Day (EPSD). Ensure the event      |
|                                   | dates in the prompt are amended   |
|                                   | to the EPSD for the outstanding   |
|                                   | period.                           |
|                                   |                                   |
|                                   | How many weeks do you need to     |
|                                   | record (if the hours worked each  |
|                                   | week are different during the     |
|                                   | fortnight, select 2 weeks)?       |
|                                   |                                   |
|                                   | This question allows a Service    |
|                                   | Officer to record more than one   |
|                                   | set of hours for each week that   |
|                                   | is worked. Where the hours worked |
|                                   | are different for each week of    |
|                                   | the fortnight, the Service        |
|                                   | Officer should select '2' and two |
|                                   | separate event dates and details  |
|                                   | can be recorded.                  |
|                                   |                                   |
|                                   | How many hours were worked per    |
|                                   | week (include total travel time)? |
|                                   |                                   |
|                                   | The details will be recorded as   |
|                                   | hours and minutes through the     |
|                                   | Carer Payment Details question    |
|                                   | set.                              |
|                                   |                                   |
|                                   | To display the Carer Payment      |
|                                   | Details question set when         |
|                                   | recording employment income:      |
|                                   |                                   |
|                                   | -   save the response for the IOP |
|                                   |     or LOP details                |
|                                   | -   select Yes at ‘Are there more |
|                                   |     employment details to declare |
|                                   |     for this employer?’           |
|                                   | -   at Type of Employment Details |
|                                   |     Update, select Carer Activity |
|                                   |     (CPAD) only                   |
|                                   |                                   |
|                                   | The Carer Payment Details         |
|                                   | question set will appear          |
|                                   | automatically for                 |
|                                   | customers/partners receiving CP   |
|                                   | who are eligible for a continuous |
|                                   | income frequency of MTE or where  |
|                                   | Service Officers specifically     |
|                                   | select the ‘Carer Activity (CPAD) |
|                                   | only’ option.                     |
|                                   |                                   |
|                                   | If the carer has started working  |
|                                   | but has not yet been paid, after  |
|                                   | adding the new employer details,  |
|                                   | select Carer Activity (CPAD) only |
|                                   | as the Type of Employment Details |
|                                   | Update.                           |
|                                   |                                   |
|                                   | What is the Event Date?           |
|                                   |                                   |
|                                   | -   Carer Payment Details (CPAS): |
|                                   |     only one event date is        |
|                                   |     required if the carer is      |
|                                   |     working regular hours, and    |
|                                   |     should generally be the       |
|                                   |     Entitlement Period Start Day  |
|                                   |     (EPSD). A second event date   |
|                                   |     can be recorded and will      |
|                                   |     default to the start date of  |
|                                   |     the next week in the          |
|                                   |     entitlement period (EPSD+7).  |
|                                   |     Date can also be up to 30     |
|                                   |     days in the future to cease   |
|                                   |     the hours worked if not       |
|                                   |     continuing their employment   |
|                                   |                                   |
|                                   | If the customer works different   |
|                                   | hours each week of the fortnight, |
|                                   | record the hours separately for   |
|                                   | each week.                        |
|                                   |                                   |
|                                   | The hours and minutes should      |
|                                   | include total travel time to and  |
|                                   | from work each week. Details of   |
|                                   | this may need to be recorded in   |
|                                   | the Notes facility if an adverse  |
|                                   | decision on eligibility for       |
|                                   | payment is to be made, i.e. when  |
|                                   | time spent in employment and      |
|                                   | travelling to and from work (by   |
|                                   | itself or in combination with     |
|                                   | other activities) exceeds 25      |
|                                   | hours per week.                   |
|                                   |                                   |
|                                   | Was the carer still providing     |
|                                   | care while working? For example,  |
|                                   | working from home and still able  |
|                                   | to care, Yes/No/Uncertain?        |
|                                   |                                   |
|                                   | -   Yes, Care is met. This is     |
|                                   |     where the care receiver is    |
|                                   |     with the customer during      |
|                                   |     employment and the care is    |
|                                   |     still provided by the carer,  |
|                                   |     consistent with the care      |
|                                   |     receiver's care needs.        |
|                                   |     For example, a carer who is   |
|                                   |     working from home. A          |
|                                   |     mandatory question 'Please    |
|                                   |     describe how care is          |
|                                   |     continuing to be able to be   |
|                                   |     undertaken.' will be          |
