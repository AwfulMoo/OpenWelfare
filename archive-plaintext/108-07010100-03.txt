|                                   | non-teaching support staff        |
|                                   | employed by the NSW Department of |
|                                   | Education + Read more ...         |
|                                   |                                   |
|                                   | In New South Wales (NSW),         |
|                                   | non-teaching support staff        |
|                                   | (teacher’s aides, library         |
|                                   | assistants or other non-teaching  |
|                                   | staff), are employees of the NSW  |
|                                   | Department of Education. As they  |
|                                   | are unable to work during term    |
|                                   | breaks, they receive a stand down |
|                                   | payment, based on them meeting    |
|                                   | certain criteria.                 |
|                                   |                                   |
|                                   | Stand down payments have never    |
|                                   | been considered leave payment for |
|                                   | Social Security purposes:         |
|                                   |                                   |
|                                   | -   Before 20 April 2010, stand   |
|                                   |     down payments were assessed   |
|                                   |     as a remunerative lump sum    |
|                                   |     and held for a 52-week period |
|                                   |     from the date the customer    |
|                                   |     received it                   |
|                                   | -   From 20 April 2010, these     |
|                                   |     payments are assessed as      |
|                                   |     ordinary employment income    |
|                                   |     earned in the customers       |
|                                   |     entitlement period            |
|                                   |                                   |

|                                   | or termination payment, assets    |

|                                   | income and assets for Centrelink  |
|                                   | payments and services.            |
