Suggested text for Q999 manual letters

This table describes some suggested text to be used for manual letters.
The Appeal paragraph should be included in both examples.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Grant letter for a deceased carer |
|                                   |                                   |
|                                   | Estate of the Late <carer's name> |
|                                   |                                   |
|                                   | Address                           |
|                                   |                                   |
|                                   | Date                              |
|                                   |                                   |
|                                   | Dear xxxxxxxx,                    |
|                                   |                                   |
|                                   | Please accept our sympathy on the |
|                                   | recent loss of <carer's name>. We |
|                                   | realise that this is not a good   |
|                                   | time to discuss payments but it   |
|                                   | is important that we advise you   |
|                                   | of the outcome of <carer's name>  |
|                                   | claim for <Carer Payment/Carer    |
|                                   | Allowance>.                       |
|                                   |                                   |
|                                   | The claim for <Carer              |
|                                   | Payment/Carer Allowance> has been |
|                                   | granted from <date>.              |
|                                   |                                   |
|                                   | A payment of $XXXX will be        |
|                                   | delivered to <carer's name>       |
|                                   | nominated bank account on <date>. |
|                                   |                                   |
|                                   | This is an information notice     |
|                                   | given under social security law.  |
+-----------------------------------+-----------------------------------+
| 2                                 | Rejection letter for a deceased   |
|                                   | carer                             |
|                                   |                                   |
|                                   | Estate of the Late <carer's name> |
|                                   |                                   |
|                                   | Address                           |
|                                   |                                   |
|                                   | Date                              |
|                                   |                                   |
|                                   | Dear xxxxxx,                      |
|                                   |                                   |
|                                   | Please accept our sympathy on the |
|                                   | recent loss of <carer's name>. We |
|                                   | realise that this is not a good   |
|                                   | time to discuss payments but it   |
|                                   | is important that we advise you   |
|                                   | of the outcome of <carer's name>  |
|                                   | claim for <Carer Payment/Carer    |
|                                   | Allowance>.                       |
|                                   |                                   |
|                                   | The claim for <Carer              |
|                                   | Payment/Carer Allowance> has not  |
|                                   | been successful because <state    |
|                                   | reason/s>.                        |
|                                   |                                   |
|                                   | This is an information notice     |
|                                   | given under social security law.  |
+-----------------------------------+-----------------------------------+
