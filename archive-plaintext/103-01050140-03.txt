| 5                                 | Customer's payment is suspended + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a customer's income support    |
|                                   | payment is suspended, Income      |
|                                   | Management will continue until    |
|                                   | the payment is restored. If the   |
|                                   | income support payment is         |
|                                   | subsequently cancelled, Income    |
|                                   | Management will automatically     |
|                                   | end.                              |
|                                   |                                   |
|                                   | The following action should be    |
|                                   | taken, when the customer enters   |
|                                   | prison and their payments are or  |
|                                   | will be suspended:                |
|                                   |                                   |
|                                   | -   Priority needs - if the       |
|                                   |     customer's Income Management  |
|                                   |     account has a credit balance, |
|                                   |     these funds will be available |
|                                   |     to them for their priority    |
|                                   |     needs. It is highly likely    |
|                                   |     imprisonment will have        |
|                                   |     changed the customer's        |
|                                   |     priority needs, however       |
|                                   |     consideration should be given |
|                                   |     to customers who may be       |
|                                   |     supporting family members     |
|                                   |     using their income managed    |
|                                   |     funds whilst in prison        |
|                                   | -   Income Management expenses -  |
|                                   |     until the customer can be     |
|                                   |     contacted, a customer's       |
|                                   |     Income Management expenses    |
|                                   |     can be stopped when they are  |
|                                   |     admitted to prison. This      |
|                                   |     should occur immediately      |
|                                   |     where the priority need is no |
|                                   |     longer relevant (for example, |
|                                   |     regular allocation to         |
|                                   |     BasicsCard), until            |
|                                   |     discussions can be held with  |
|                                   |     the customer. If no action is |
|                                   |     taken to end Income           |
|                                   |     Management expenses, they     |
|                                   |     will continue to be paid      |
|                                   |     until income managed funds    |
|                                   |     are exhausted or Income       |
|                                   |     Management ends. Where Income |
|                                   |     Management expenses continue  |
|                                   |     to be paid, recovery of funds |
|                                   |     may be required following     |
|                                   |     discussions with the          |
|                                   |     customer. For more            |
|                                   |     information see Recall of     |
|                                   |     Income Management funds       |
|                                   | -   BasicsCard - if the customer  |
|                                   |     will be imprisoned for less   |
|                                   |     than 13 weeks, the BasicsCard |
|                                   |     should be suspended upon      |
|                                   |     entry to prison to avoid      |
|                                   |     unauthorised use. If the      |
|                                   |     customer's payment is         |
|                                   |     subsequently restored, the    |
|                                   |     BasicsCard can be reactivated |
|                                   |     for immediate use by the      |
|                                   |     customer                      |
|                                   | -   Arrears payments - if a       |
|                                   |     customer is paid an arrears   |
|                                   |     payment, it will be 100%      |
|                                   |     income managed. An            |
|                                   |     Unrestricted Cash payment     |
|                                   |     (UCP) may be payable if the   |
|                                   |     customer requests             |
|                                   | -   Payment Nominee: customers    |
|                                   |     may choose to appoint a       |
|                                   |     payment nominee during their  |
|                                   |     period of imprisonment. If    |
|                                   |     appointed, the payment        |
|                                   |     nominee will be responsible   |
|                                   |     for managing a customer's     |
|                                   |     Income Management funds. For  |
|                                   |     more information see Nominee  |
|                                   |     arrangements under Income     |
|                                   |     Management                    |
|                                   |                                   |
|                                   | If required, Service Officers can |
|                                   | seek support from a Prison        |
|                                   | Liaison Officer to confirm        |
|                                   | admission or release details. See |
|                                   | Resources for a link to the       |
|                                   | Prison Liaison Officer contacts   |
|                                   | intranet page.                    |
|                                   |                                   |
|                                   | For further information which may |
|                                   | assist, see:                      |
|                                   |                                   |
|                                   | -   Prison admission process and  |
|                                   |     claims during incarceration   |
|                                   | -   Effect on Income Management   |
|                                   |     when payment is restored or   |
|                                   |     re-granted                    |
|                                   | -   Coding Income Management      |
|                                   |     expenses                      |
|                                   | -   Restricted Direct Payment     |
|                                   |     (RDP)                         |
|                                   | -   Unrestricted Cash payment     |
|                                   |     (UCP)                         |
|                                   |                                   |
|                                   | Procedure ends here.              |

| 6                                 | Customer's income support payment |
|                                   | is cancelled + Read more ...      |
|                                   |                                   |
|                                   | If a customer's income support    |
|                                   | payment is cancelled, Income      |
|                                   | Management will automatically     |
|                                   | end. Customers are notified in    |
|                                   | writing of Income Management      |
|                                   | ending and invited to contact     |
|                                   | Services Australia to discuss how |
|                                   | they would like any remaining     |
|                                   | Income Management funds paid.     |
|                                   |                                   |
|                                   | The following will occur when a   |
|                                   | customer's Income Management ends |
|                                   | when entering prison:             |
|                                   |                                   |
|                                   | -   Income Management expenses -  |
|                                   |     Income Management expenses    |
|                                   |     will continue to pay for 14   |
|                                   |     days. If the customer's       |
|                                   |     payment is not likely to be   |
|                                   |     restored within 14 days,      |
|                                   |     regular expenses should be    |
|                                   |     ended when notification is    |
|                                   |     received of the customer      |
|                                   |     entering prison               |
|                                   | -   BasicsCard - as the customer  |
|                                   |     will not be able to use their |
|                                   |     BasicsCard while in prison,   |
|                                   |     the card should be closed     |
|                                   |     upon entry to prison to avoid |
|                                   |     any unauthorised use          |
|                                   | -   Disbursement of residual      |
|                                   |     funds - incarcerated          |
|                                   |     customers are unable to       |
|                                   |     attend Disbursement           |
|                                   |     interviews. When Income       |
|                                   |     Management automatically      |
|                                   |     ends, any balance in their    |
|                                   |     Income Management Account     |
|                                   |     will be disbursed under the   |
|                                   |     Auto Disbursement rules.      |
|                                   |     Funds are paid to the         |
|                                   |     customer's last known payment |
|                                   |     destination. It is therefore  |
|                                   |     important payment destination |
|                                   |     details are correct for       |
|                                   |     Service Reason 'GPY'          |
|                                   | -   Payment Nominee - customers   |
|                                   |     may choose to appoint a       |
|                                   |     payment nominee during their  |
|                                   |     period of imprisonment. If    |
|                                   |     appointed, any residual funds |
|                                   |     will be paid to the payment   |
|                                   |     nominee's nominated bank      |
|                                   |     account. For more information |
|                                   |     see Nominee arrangements      |
|                                   |     under Income Management       |
|                                   |                                   |
|                                   | If required, Service Officers can |
|                                   | seek support from a Prison        |
|                                   | Liaison Officer to confirm        |
|                                   | admission or release details. See |
|                                   | Resources for a link to the       |
|                                   | Prison Liaison Officer contacts   |
|                                   | intranet page.                    |

| 2                                 | Income Management customer        |
|                                   | leaving prison + Read more ...    |
|                                   |                                   |
|                                   | When a customer leaves prison,    |
|                                   | and their payment is restored or  |

|                                   | support payment, Income           |
|                                   | Management may automatically be   |
|                                   | reinstated, or they may be        |
|                                   | automatically identified for      |

|                                   | circumstances.                    |
|                                   |                                   |
|                                   | -   Customers whose payments were |

|                                   |     automatically have Income     |
|                                   |     Management reinstated. Even   |
|                                   |     where a customer moves out of |
|                                   |     a declared Income Management  |
|                                   |     area, Income Management may   |
|                                   |     continue depending on the     |
|                                   |     measure. For more information |
|                                   |     see Customer moves out of an  |
|                                   |     Income Management area        |
|                                   | -   Customers granted an income   |
|                                   |     support payment, who move to  |
|                                   |     a declared Income Management  |
|                                   |     area when leaving prison, may |
|                                   |     be identified for Income      |
|                                   |     Management due to their       |
|                                   |     circumstances. For more       |
|                                   |     information see               |
|                                   |     Identification and            |
|                                   |     eligibility for Income        |
|                                   |     Management                    |

| 3                                 | Income Management reinstated +    |
|                                   | Read more ...                     |
|                                   |                                   |

|                                   | on leaving prison, Service        |
|                                   | Officers should conduct an        |
|                                   | interview with the customer to:   |
|                                   |                                   |
|                                   | -   remind them how Income        |
|                                   |     Management will affect their  |
|                                   |     payments                      |
|                                   | -   review their priority needs   |
|                                   |     and set up any new Income     |
|                                   |     Management deductions         |
|                                   | -   reactivate a previously       |
|                                   |     suspended BasicsCard or issue |
|                                   |     a new or replacement          |

|                                   | -   remind customers of the       |
|                                   |     Income Management and         |
|                                   |     BasicsCard self service       |

|                                   | -   where applicable, discuss an  |
|                                   |     exemption (Disengaged Youth   |
|                                   |     and Long Term Welfare Payment |

|                                   |     appropriate specialist as per |
|                                   |     current business process      |

|                                   | (CrP) + Read more ...             |
