Accessing the Immunisation Main Menu (YIMM)

This table describes how Service Officers access the Immunisation Main
menu (YIMM) in AIR and the screens displayed for each option.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access YIMM - Immunisation Main   |
|                                   | menu + Read more ...              |
|                                   |                                   |
|                                   | In AIR Mainframe, key 'YIMM'      |
|                                   | anywhere on a clear screen and    |
|                                   | press Enter.                      |
|                                   |                                   |
|                                   | AIR processing options display.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Select relevant function + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select from the following         |
|                                   | options:                          |
|                                   |                                   |
|                                   | -   1 Enquiries Menu (YEMM)       |
|                                   | -   2 Registration Menu (YCRM)    |
|                                   | -   3 Claims Processing Menu      |
|                                   |     (YIEM)                        |
|                                   | -   4 Not available               |
|                                   | -   5 Not available               |
|                                   | -   6 Immunisation Payment Menu   |
|                                   |     (YFPM)                        |
|                                   | -   7 Bank Reconciliation Menu    |
|                                   |     (BRIM)                        |
|                                   | -   8 Centrelink Menu (YCLK).     |
|                                   |     Note: this option is          |
|                                   |     restricted to AIR National    |
|                                   |     Office staff                  |
+-----------------------------------+-----------------------------------+
