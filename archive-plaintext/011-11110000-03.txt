Rework transaction process

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Australian Organ Donor Register   |
|                                   | (AODR) Policy + Read more ...     |
|                                   |                                   |
|                                   | Provides the Quality Management   |
|                                   | Team with a monthly report        |
|                                   | extracted from EDW identifying    |
|                                   | Donor Only records which have the |
|                                   | Medicare number keyed in the      |
|                                   | registration, but have not been   |
|                                   | 'linked'.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Quality Management Team + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Separates the report and provides |
|                                   | individual monthly reports via    |
|                                   | email to source code area.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Source code area + Read more ...  |
|                                   |                                   |
|                                   | Actions the rework report. The    |
|                                   | rework email from Quality         |
|                                   | Management will contain a link to |
|                                   | an Operational Blueprint          |
|                                   | procedure, with instructions on   |
|                                   | how to link a Donor Only          |
|                                   | Entitlement.                      |
|                                   |                                   |
|                                   | Replies by email to Quality       |
|                                   | Management Team within 3 working  |
|                                   | days, advising that the           |
|                                   | transaction requiring rework      |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   has been amended, see Link a  |
|                                   |     Donor Only entitlement for    |
|                                   |     rework transactions           |
|                                   | -   requires no action            |
|                                   | -   has been escalated for        |
|                                   |     further investigation (and to |
|                                   |     whom)                         |
+-----------------------------------+-----------------------------------+

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Donor ID search + Read more ...   |
|                                   |                                   |
|                                   | go to CDMS home page              |
|                                   |                                   |
|                                   | select Amend Consumer Details     |
|                                   |                                   |
|                                   | select Basic Search tab           |
|                                   |                                   |
|                                   | select Australian Organ Donor     |
|                                   | Register from drop down list      |
|                                   |                                   |
|                                   | key the Donor ID number from      |
|                                   | rework report in ID Number field  |
|                                   |                                   |
|                                   | select Donor ID from drop down    |
|                                   | list in the ID Number Type field  |
|                                   |                                   |
|                                   | select Entitlement                |
|                                   |                                   |
|                                   | -   the 'Amend Donor Only         |
|                                   |     Entitlement' screen displays  |
+-----------------------------------+-----------------------------------+
| 2                                 | Link consumer + Read more ...     |
|                                   |                                   |
|                                   | Select Link to Consumer           |
|                                   |                                   |
|                                   | -   the New AODR Entitlement      |
|                                   |     Details screen displays       |
+-----------------------------------+-----------------------------------+
| 3                                 | Postcode differs + Read more ...  |
|                                   |                                   |
|                                   | Did you receive a Postcode        |
|                                   | differs error message?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Donor Details + Read more ...     |
|                                   |                                   |
|                                   | select Donor Decisions Details    |
|                                   | tab and re-enter:                 |
|                                   |                                   |
|                                   | -   Most Recent Donation Decision |
|                                   |     Date                          |
|                                   | -   Date of Last Contact          |
|                                   | -   Key Medicare number in the    |
|                                   |     Medicare Card Number field,   |
|                                   |     if field is blank             |
+-----------------------------------+-----------------------------------+
| 5                                 | Program Details + Read more ...   |
|                                   |                                   |
|                                   | -   Select Program Details tab    |
|                                   |                                   |
|                                   | Update the following drop down    |
|                                   | boxes:                            |
|                                   |                                   |
|                                   | -   Donor Media type              |
|                                   | -   Donor Delivery type           |
|                                   |                                   |
|                                   | Note: the media type and delivery |
|                                   | type appear on the record but     |
|                                   | must be re-entered                |
|                                   |                                   |
|                                   | -   Select Validate               |
+-----------------------------------+-----------------------------------+
| 6                                 | Validation errors + Read more ... |
|                                   |                                   |
|                                   | Are there validation error        |
|                                   | messages?                         |
|                                   |                                   |
|                                   | Yes, go to Step 7                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   the Confirm Amended AODR      |
|                                   |     Entitlement Details screen    |
|                                   |     displays                      |
|                                   | -   go to Step 11                 |
+-----------------------------------+-----------------------------------+
| 7                                 | Error messages + Read more ...    |
|                                   |                                   |
|                                   | -   Note the Medicare number from |
|                                   |     the Medicare Card Number      |
|                                   |     field                         |
|                                   | -   Remove the Medicare number    |
|                                   |     from the Medicare Card Number |
|                                   |     field                         |
|                                   |                                   |
|                                   | Determine the error message:      |
|                                   |                                   |
|                                   | -   Postcode differs error        |
|                                   |     message. Check history on     |
|                                   |     both the Donor Only           |
|                                   |     entitlement and the Medicare  |
|                                   |     record to determine most      |
|                                   |     current address.              |
|                                   |                                   |
|                                   | If the most current address is:   |
|                                   |                                   |
|                                   | -   On the Medicare record -      |
|                                   |     update Donor Only entitlement |
|                                   |     with the new address, go to   |
|                                   |     Step 2                        |
|                                   | -   On the Donor Only             |
|                                   |     entitlement - update the      |
|                                   |     address on Medicare, go to    |
|                                   |     Step 2                        |
|                                   | -   If the address is a Personal  |
|                                   |     vs Group address error -      |
|                                   |     escalate to a PSO via a PSO   |
|                                   |     Referral form                 |
|                                   |                                   |
|                                   | -   Name Differs error message,   |
|                                   |     go to Step 8                  |
|                                   | -   DOB Differs error message, go |
|                                   |     to Step 9                     |
|                                   | -   Gender Differs, go to Step 10 |
|                                   |                                   |
|                                   | See Change address details via    |
|                                   | Donor Registration form on the    |
|                                   | Resources page of Australian      |
|                                   | Organ Donor Register (AODR).      |
+-----------------------------------+-----------------------------------+
| 8                                 | Name Differs + Read more ...      |
|                                   |                                   |
|                                   | Where the discrepancy is the      |
|                                   | surname:                          |
|                                   |                                   |
|                                   | -   Refer to Confirm previous     |
|                                   |     name, see Link Donor Only     |
|                                   |     entitlement to consumer       |
|                                   |     record                        |
|                                   |                                   |
|                                   | Where the discrepancy is a major  |
|                                   | discrepancy, the Donor Only       |
|                                   | entitlement cannot be linked:     |
|                                   |                                   |
|                                   | -   In the Comments tab enter a   |
|                                   |     comment e.g. ‘AODR-Not linked |
|                                   |     due to name discrepancy', go  |
|                                   |     to Step 4                     |
|                                   |                                   |
|                                   | Where the discrepancy is a Group  |
|                                   | vs personal name difference       |
|                                   |                                   |
|                                   | -   Update the registration to    |
|                                   |     match the name recorded in    |
|                                   |     Medicare group, go to Step 2  |
|                                   |                                   |
|                                   | See Processing rules for          |
|                                   | Australian Organ Donor Register   |
|                                   | (AODR).                           |
+-----------------------------------+-----------------------------------+
| 9                                 | DOB Differs + Read more ...       |
|                                   |                                   |
|                                   | Where the discrepancy is a minor  |
|                                   | discrepancy                       |
|                                   |                                   |
|                                   | -   Update the DOB to match the   |
|                                   |     Medicare Record, go to Step 2 |
|                                   |                                   |
|                                   | Where the discrepancy is a major  |
|                                   | discrepancy the Donor Only        |
|                                   | entitlement cannot be linked:     |
|                                   |                                   |
|                                   | -   In the Comments tab enter a   |
|                                   |     comment, e.g. ‘AODR-Not       |
|                                   |     linked due to name            |
|                                   |     discrepancy’, go to Step 4    |
|                                   |                                   |
|                                   | See Processing rules for          |
|                                   | Australian Organ Donor Register   |
|                                   | (AODR).                           |
+-----------------------------------+-----------------------------------+
| 10                                | Gender Differs + Read more ...    |
|                                   |                                   |
|                                   | Any gender variation should be    |
|                                   | processed using the gender        |
|                                   | recorded on Consumer Directory    |
|                                   | Maintenance System (CDMS), where  |
|                                   | possible.                         |
|                                   |                                   |
|                                   | Determine if the gender variation |
|                                   | is a minor or major discrepancy.  |
|                                   |                                   |
|                                   | Where the discrepancy is a minor  |
|                                   | discrepancy:                      |
|                                   |                                   |
|                                   | -   After processing, forward the |
|                                   |     form with the gender          |
|                                   |     variation to Medicare Public  |
|                                   |     Eligibility via the AODR PSO  |
|                                   |     Team                          |
|                                   |                                   |
|                                   | Where the discrepancy is a major  |
|                                   | discrepancy:                      |
|                                   |                                   |
|                                   | -   Process as normal using the   |
|                                   |     gender recorded               |
|                                   | -   Enter AODR comments, e.g.:    |
|                                   |     ‘AODR form rcvd, major gender |
|                                   |     discrepancy not altered per   |
|                                   |     <comments/Medicare/phone      |
|                                   |     call/PSO>                     |
|                                   |                                   |
|                                   | See Change of sex or gender for   |
|                                   | Medicare enrolments.              |
+-----------------------------------+-----------------------------------+
| 11                                | Confirm updates + Read more ...   |
|                                   |                                   |
|                                   | select Update                     |
|                                   |                                   |
|                                   | -   a confirmation message        |
|                                   |     displays                      |
