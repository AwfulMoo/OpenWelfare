Example

Multiple child percentage

This table contains an example of a multiple child percentage
calculation, based on the standard hourly rate for the 2017-18 financial
year.

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Multiple child percentage         |
|                                   | calculation                       |
|                                   |                                   |
|                                   | Tony and Mary both work full-time |
|                                   | and have 3 children. Their actual |
|                                   | adjusted taxable income is below  |
|                                   | $41,026 per year. They use long   |
|                                   | day care for all their children.  |
|                                   | Their multiple child percentage   |
|                                   | is calculated as follows:         |
|                                   |                                   |
|                                   | 100 x (maximum weekly benefit for |
|                                   | all children in care / (maximum   |
|                                   | weekly benefit for one child in   |
|                                   | the type of care x all children   |
|                                   | in that type of care))            |
|                                   |                                   |
|                                   | -   = 100 x {701.14 / (215.00 x   |
|                                   |     3)}                           |
|                                   | -   = 100 x {701.14 / 645.00}     |
|                                   | -   = 100 x 1.087                 |
|                                   | -   = 108.7%                      |
+-----------------------------------+-----------------------------------+
