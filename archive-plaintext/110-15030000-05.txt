Handle customer enquiries - BI flag lifted

Table 1: how Service Officers respond to questions when claim details
have been verified and the BI flag hotline team has lifted the BI flag.
Do not disclose to the claimant that the card has a BI flag.

+-----------------------------------+-----------------------------------+
| Item                              | Questions and answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Why does it take so     |
|                                   | long to process my claim?         |
|                                   |                                   |
|                                   | Answer: Services Australia as     |
|                                   | part of the benefit payment       |
|                                   | process undertakes verification   |
|                                   | of claim details to ensure the    |
|                                   | correct entitlement to Medicare   |
|                                   | benefit payments. The claim you   |
|                                   | have submitted will require a     |
|                                   | pre-payment check and this        |
|                                   | process may take slightly longer  |
|                                   | than usual.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Is there something      |
|                                   | wrong with my Medicare card?      |
|                                   |                                   |
|                                   | Answer: No. As part of the        |
|                                   | benefit payment process Services  |
|                                   | Australia may undertake           |
|                                   | verification of claim details to  |
|                                   | ensure the correct entitlement to |
|                                   | Medicare benefit payments. The    |
|                                   | claim you have submitted will     |
|                                   | require a pre-payment check and   |
|                                   | this process may take slightly    |
|                                   | longer than usual.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Is there a problem with |
|                                   | the receipt from my doctor?       |
|                                   |                                   |
|                                   | Answer: As part of the benefit    |
|                                   | payment process Services          |
|                                   | Australia may undertake           |
|                                   | verification of claim details to  |
|                                   | ensure the correct entitlement to |
|                                   | Medicare benefit payments. Part   |
|                                   | of this process is verifying the  |
|                                   | details of the service/s on the   |
|                                   | account with the provider.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: How can I submit a      |
|                                   | complaint?                        |
|                                   |                                   |
|                                   | Answer: In the first instance you |
|                                   | can talk to the person you have   |
|                                   | been dealing with, and if you are |
|                                   | not satisfied with the response   |
|                                   | you can ask to speak to their     |
|                                   | manager.                          |
|                                   |                                   |
|                                   | You can also:                     |
|                                   |                                   |
|                                   | -   call our feedback and         |
|                                   |     complaints line               |
|                                   | -   send us a secure online       |
|                                   |     message, or                   |
|                                   | -   write to us                   |
|                                   |                                   |
|                                   | See Complaints contact            |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Will this keep          |
|                                   | happening every time I make a     |
|                                   | claim?                            |
|                                   |                                   |
|                                   | Answer: To mitigate any risk to   |
|                                   | you and Services Australia,       |
|                                   | pre-payment verification checks   |
|                                   | will continue until the risk has  |
|                                   | been evaluated and treated        |
|                                   | accordingly.                      |
+-----------------------------------+-----------------------------------+

Handle customer enquiries - BI flag cannot be lifted

Table 2: how Service Officers respond to questions when the BI flag
cannot be lifted, such as when claim is submitted outside Medicare
Compliance business hours. Do not disclose to the claimant that the card
has a BI flag.

+-----------------------------------+-----------------------------------+
| Item                              | Questions and answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Why are you unable to   |
|                                   | process my claim?                 |
|                                   |                                   |
|                                   | Answer: Services Australia is     |
|                                   | unable to provide immediate       |
|                                   | payment at this time. The claim   |
|                                   | you have submitted will require a |
|                                   | pre-payment check and this        |
|                                   | process may take slightly longer  |
|                                   | than usual. We will be able to    |
|                                   | process the transaction during    |
|                                   | normal week day business hours.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: When will I be able to  |
|                                   | get my claim processed?           |
|                                   |                                   |
|                                   | Answer: As I am unable to process |
|                                   | the claim at this time you may    |
|                                   | return during normal week day     |
|                                   | business hours.                   |
|                                   |                                   |
|                                   | Alternatively, you may wish to    |
|                                   | complete a Medicare Claim form    |
|                                   | (MS014) and leave your claim with |
|                                   | me for processing during normal   |
|                                   | week day business hours.          |
+-----------------------------------+-----------------------------------+
