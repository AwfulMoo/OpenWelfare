|                                   | -   change the indicator to ‘N’   |
|                                   |                                   |

|                                   | the Child Profound Disability     |
|                                   | Details (CPDD) screen, if         |

|                                   | coding the review date of effect  |
|                                   | in the Event Date field and       |

|                                   | condition and short life          |
|                                   | expectancy to ‘N’ if the child    |
|                                   | meets savings provisions, update  |
|                                   | the remaining fields.             |
|                                   |                                   |
|                                   | Finalise the review by completing |
|                                   | the CPRV screen. Payment will be  |
|                                   | assessed based on the DCLAD       |
|                                   | (2010) and the carer informed of  |

|                                   | outcome of the terminal illness   |
