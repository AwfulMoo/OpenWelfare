Process when transferring or referring calls to specialist staff

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Transferring or referring a call  |
|                                   | to a specialist officer + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When transferring or referring a  |
|                                   | call to a specialist officer,     |
|                                   | staff will:                       |
|                                   |                                   |
|                                   | -   Advise the caller why the     |
|                                   |     referral is necessary         |
|                                   | -   If transferring the call,     |
|                                   |     brief the receiving officer   |
|                                   |     before releasing the call to  |
|                                   |     them                          |
|                                   | -   If using Online Document      |
|                                   |     Recording (ODR) for the       |
|                                   |     referral, tell the customer   |
|                                   |     that their call will be       |
|                                   |     returned by the specialist    |
|                                   |     officer within 24 hours       |
|                                   | -   Record enough information on  |
|                                   |     the DOC to allow the          |
|                                   |     specialist officer to         |
|                                   |     adequately prepare            |
|                                   |     themselves. Do this while the |
|                                   |     customer is on the phone or   |
|                                   |     immediately after they hang   |
|                                   |     up. Include any times when    |
|                                   |     the customer will not be      |
|                                   |     available to take the return  |
|                                   |     call                          |
+-----------------------------------+-----------------------------------+
| 2                                 | What not to do when               |
|                                   | transferring/referring a call to  |
|                                   | a specialist officer + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | When transferring a call to a     |
|                                   | specialist officer, staff will    |
|                                   | not:                              |
|                                   |                                   |
|                                   | -   become an intermediary        |
|                                   |     between the customer and the  |
|                                   |     specialist officer            |
|                                   | -   release a call to the         |
|                                   |     specialist officer without    |
|                                   |     adequate introduction or      |
|                                   |     before the officer is ready   |
|                                   |     to accept it                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Things for a specialist officer   |
|                                   | to do + Read more ...             |
|                                   |                                   |
|                                   | When receiving a transferred or   |
|                                   | referred call, specialist         |
|                                   | officers will:                    |
|                                   |                                   |
|                                   | -   monitor their work daily and  |
|                                   |     return referrals within 24    |
|                                   |     hours of the enquiry          |
|                                   | -   make arrangements for someone |
|                                   |     to answer their phone or      |
|                                   |     check AYS and action          |
|                                   |     referrals if they are unable  |
|                                   |     to do so themselves           |
|                                   | -   annotate all penalty period   |
|                                   |     DOCs with the time the        |
|                                   |     specialist officer/Service    |
|                                   |     Centre Manager contacted or   |
|                                   |     attempted to contact the      |
|                                   |     customer and any action that  |
|                                   |     was taken                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Things for a specialist officer   |
|                                   | not to do + Read more ...         |
|                                   |                                   |
|                                   | When receiving a transferred or   |
|                                   | referred call, specialist         |
|                                   | officers will not:                |
|                                   |                                   |
|                                   | -   Use the staff member          |
|                                   |     transferring the call as an   |
|                                   |     intermediary                  |
|                                   | -   Leave a staff member on hold  |
|                                   |     under any circumstances       |
|                                   | -   Leave a customer on hold for  |
|                                   |     long periods without advising |
|                                   |     them of the progress of the   |
|                                   |     investigation or offering to  |
|                                   |     call them back                |
|                                   | -   Refuse a call because they    |
|                                   |     consider it inappropriate.    |
|                                   |     Note: a failure in procedure  |
|                                   |     can be addressed via the      |
|                                   |     Staff Feedback Tool after the |
|                                   |     customer's needs are met      |
+-----------------------------------+-----------------------------------+
