Example

Calculating a partnered customer's pension rate

This table describes an example of using the Income Test to work out a
customer's member of a couple pension rate.

These amounts may not be current. They are examples only. For current
payments, see Rates and thresholds.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Pensioner couple                  |
|                                   |                                   |
|                                   | Tom and Alex have combined        |
|                                   | ordinary income of $7,980 pa.     |
|                                   |                                   |
|                                   | Combined income / 2 = $3,990      |
|                                   |                                   |
|                                   | Half of the pension couple income |
|                                   | free area = $3,328 pa             |
|                                   |                                   |
|                                   | Excess income = $3,990 - $3,328 = |
|                                   | $662 each                         |
|                                   |                                   |
|                                   | Income reduction = $662 x 0.5 =   |
|                                   | $331.00 each                      |
|                                   |                                   |
|                                   | Maximum annual pension rate less  |
|                                   | reduction = $14,034.80 - $331.00  |
|                                   | = $13,703.80 pa each              |
|                                   |                                   |
|                                   | Fortnightly pension rate =        |
|                                   | $13,703.80/ 26 = $527.10 each     |
|                                   | (rounded up)                      |
+-----------------------------------+-----------------------------------+
