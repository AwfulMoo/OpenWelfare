Manual processing of concession card

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Manual process to refresh         |
|                                   | concession entitlement + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Manual processing can only be     |
|                                   | done in Customer First when the   |
|                                   | Confirm Concession Entitlement    |
|                                   | guided procedure is not           |
|                                   | available.                        |
|                                   |                                   |
|                                   | To manually refresh concession    |
|                                   | entitlements:                     |
|                                   |                                   |
|                                   | Go to the Concession Benefit      |
|                                   | Action (MCBA) screen and key:     |
|                                   |                                   |
|                                   | -   Action: RSH                   |
|                                   | -   Reason: HIC (Health Insurance |
|                                   |     Commission)                   |
|                                   | -   Effect date: today's date     |
|                                   |                                   |
|                                   | Go to Assessment Results (AR)     |
|                                   | screen and finalise               |
|                                   |                                   |
|                                   | If a concession entitlement       |
|                                   | refresh has recently been         |
|                                   | completed and the concession      |
|                                   | entitled customer is still        |
|                                   | experiencing difficulties         |
|                                   | accessing PBS pharmaceuticals,    |
|                                   | refer the case to the             |
|                                   | Applications Service Desk via     |
|                                   | normal protocols. See Resources   |
|                                   | tab.                              |
|                                   |                                   |
|                                   | Note: customers should not be     |
|                                   | provided with a screen dump of    |
|                                   | the Card Details (CCID) screen    |
|                                   | when requesting proof of their    |
|                                   | eligibility for concessions.      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Manual process to issue interim   |
|                                   | voucher and/or replacement card   |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | If the Concession Card Reissue    |
|                                   | guided procedure is unavailable,  |
|                                   | a manual Confirmation of          |
|                                   | Concession Card Entitlement       |
|                                   | (SS055) can be issued.            |
|                                   |                                   |
|                                   | -   The SS055 is an accountable   |
|                                   |     form completed by hand by a   |
|                                   |     Service Officer               |
|                                   | -   This option should only be    |
|                                   |     used in instances of system   |
|                                   |     failure or declared emergency |
|                                   |     situations where there is no  |
|                                   |     print facility available to   |
|                                   |     staff                         |
|                                   | -   A Service Officer should only |
|                                   |     issue a SS055 manually if the |
|                                   |     customer's current concession |
|                                   |     entitlement has been verified |
|                                   |     via the Concession            |
|                                   |     Entitlement Summary (MCCS)    |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Manually record details + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To manually record the issue of   |
|                                   | Confirmation of Concession Card   |
|                                   | Entitlement (SS055):              |
|                                   |                                   |
|                                   | Go to the Concession Card Issue   |
|                                   | Summary (CCIS) screen             |
|                                   |                                   |
|                                   | Key 'I' next to the concession    |
|                                   | card required                     |
|                                   |                                   |
|                                   | The Concession Card Interim       |
|                                   | Voucher (CCIV) screen will        |
|                                   | display. Complete the following   |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Serial no: use the serial     |
|                                   |     number on the SS055 form      |
|                                   | -   Valid To: default is 14 days. |
|                                   |     Can be extended for a maximum |
|                                   |     period of 3 months from the   |
|                                   |     date of issue (subject to the |
|                                   |     expiry date)                  |
|                                   | -   Reason for Issue: Use field   |
|                                   |     help ('?') for a list of      |
|                                   |     valid values                  |
|                                   |                                   |
|                                   | Complete the Source and DOR       |
|                                   | fields                            |
|                                   |                                   |
|                                   | Print the completed CCIV screen.  |
|                                   | The printout is to be given to    |
|                                   | the Accountable Forms Officer as  |
|                                   | proof of the issue                |
|                                   |                                   |
|                                   | Press [Enter]. The system updates |
|                                   | the customer’s record to show the |
|                                   | issue of an interim voucher on    |
|                                   | the CCIS screen:                  |
|                                   |                                   |
|                                   | -   Ensure only the appropriate   |
|                                   |     card illustration is date     |
|                                   |     stamped and all applicable    |
|                                   |     boxes on the form are filled  |
|                                   | -   The Date of Grant of          |
|                                   |     concession card can be found  |
|                                   |     on the MCCS screen            |
|                                   | -   The customer should be made   |
|                                   |     aware of the Please note      |
|                                   |     provisions at the bottom of   |
|                                   |     the SS055 and must sign the   |
|                                   |     SS055 immediately             |
|                                   | -   The Service Officer must then |
|                                   |     sign the SS055 before issuing |
|                                   |     the completed form to the     |
|                                   |     customer. There are special   |
|                                   |     requirements for what fields  |
|                                   |     are to be completed for       |
|                                   |     interim vouchers to be issued |
|                                   |     to custodial parents for use  |
|                                   |     by non-custodial parents. See |
|                                   |     Concession cards for eligible |
|                                   |     children temporarily in       |
|                                   |     non-custodial parent's care   |
|                                   |                                   |
|                                   | Finalise the activity and record  |
|                                   | the details on a DOC              |
|                                   |                                   |
|                                   | Note: if this process is applied  |
|                                   | a requested replacement card may  |
|                                   | arrive at the same time, or even  |
|                                   | earlier than the interim voucher. |
|                                   |                                   |
|                                   | If necessary, issue an auto       |
|                                   | replacement card via the CCIS     |
|                                   | screen. See How to reissue        |
|                                   | concession cards.                 |
