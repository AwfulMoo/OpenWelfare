ACER FAQs

+-----------------------------------+-----------------------------------+
| Item                              | Questions and Answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Does the aged care service need   |
|                                   | to notify Services Australia when |
|                                   | a person commences any type of    |
|                                   | services with them?               |
|                                   |                                   |
|                                   | The Aged Care Entry Record (ACER) |
|                                   | form is a mandatory requirement   |
|                                   | for any care recipients           |
|                                   | commencing a homecare package or  |
|                                   | entering permanent residential    |
|                                   | care from 1 July 2014.            |
|                                   |                                   |
|                                   | An ACER form containing any other |
|                                   | care type is returned to the aged |
|                                   | care service and deemed as        |
|                                   | inaccurate. See Reject an ACER    |
|                                   | form table.                       |
|                                   |                                   |
|                                   | Any entry related to Respite      |
|                                   | care, Transition care and Short   |
|                                   | Term Restorative care can be      |
|                                   | notified via the service’s        |
|                                   | monthly claim forms.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Is there a timeframe for the ACER |
|                                   | to be completed?                  |
|                                   |                                   |
|                                   | Yes. The ACER is required to be   |
|                                   | completed within 28 days of care  |
|                                   | recipients commencing their home  |
|                                   | care package or entering          |
|                                   | residential permanent care.       |
+-----------------------------------+-----------------------------------+
| 3                                 | How does an aged care service get |
|                                   | a copy of the ACER form?          |
|                                   |                                   |
|                                   | The ACER form can be downloaded   |
|                                   | from the agency’s website.        |
+-----------------------------------+-----------------------------------+
| 4                                 | How can an aged care service      |
|                                   | submit the ACER?                  |
|                                   |                                   |
|                                   | Aged care services can submit the |
|                                   | ACER form to Services Australia   |
|                                   | by:                               |
|                                   |                                   |
|                                   | the Aged Care Provider Portal     |
|                                   | (ACCP) as an entry event          |
|                                   |                                   |
|                                   | email                             |
|                                   |                                   |
|                                   | -   the paper form is to be       |
|                                   |     scanned and emailed to the    |
|                                   |     Aged Care liaison team        |
|                                   |     (external contacts)           |
|                                   |                                   |
|                                   | Mail                              |
|                                   |                                   |
|                                   | -   Services Australia            |
|                                   | -   Aged Care Payments            |
|                                   | -   GPO Box 9923                  |
|                                   | -   Sydney NSW 2001               |
+-----------------------------------+-----------------------------------+
