This page contains a summary of the types of activities available for
mature age job seekers to meet their participation requirements. A job
seeker should discuss suitable activities with Services Australia or, if
appropriate, their Employment Services Provider. Note: this list is not
exhaustive.

On this Page:

Approved activities E-W

Approved activities A-D

Table 1: This table describes the types of approved activities that can
be undertaken by a mature age job seeker. The approved activities are
listed in alphabetical order from Australian Disability Enterprises to
Disability Employment Services.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Australian Disability Enterprises |
|                                   | (ADE) + Read more ...             |
|                                   |                                   |
|                                   | Australian Disability Enterprises |
|                                   | (ADE) is an approved activity     |
|                                   | that employs people with a        |
|                                   | disability, illness or injury     |
|                                   | often in a specialist work        |
|                                   | environment.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Community Services Order + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A Community Service Order can be  |
|                                   | imposed by the Court (with the    |
|                                   | offender's consent) on suitable   |
|                                   | offenders, sentencing them to     |
|                                   | perform community work for a      |
|                                   | certain number of hours instead   |
|                                   | of serving time in prison or in   |
|                                   | lieu of paying a fine.            |
|                                   |                                   |
|                                   | The offenders perform unpaid work |
|                                   | at approved organisations and are |
|                                   | supervised during their community |
|                                   | service. These customers are not  |
|                                   | considered to be in gaol. A       |
|                                   | community service order is a type |
|                                   | of Community Based Order that can |
|                                   | be imposed by a Court.            |
+-----------------------------------+-----------------------------------+
| 3                                 | Defence Force Reserves + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Defence Force Reserves can be     |
|                                   | regarded as an approved activity  |
|                                   | and may include annual training   |
|                                   | camps, programmed courses, full   |
|                                   | time or part-time work in the     |
|                                   | Defence Force.                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Disability Employment Services    |
|                                   | (DES) + Read more ...             |
|                                   |                                   |
|                                   | Disability Employment Services    |
|                                   | (DES) work with job seekers who   |
|                                   | have disabilities to help them    |
|                                   | gain and maintain employment.     |
+-----------------------------------+-----------------------------------+

Approved activities E-W

Table 2: This table describes the types of approved activities that can
be undertaken by a mature age job seeker. The approved activities are
listed in alphabetical order from employment to Work for the Dole (WFD).

+-----------------------------------+-----------------------------------+
| Category                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Employment, including part-time   |
|                                   | work and self-employment + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Job seekers are encouraged to |
|                                   |     engage in part-time work as a |
|                                   |     way to maintain skills and    |
|                                   |     increase opportunities to     |
|                                   |     move to full-time employment. |
|                                   |     For some job seekers,         |
|                                   |     part-time work may meet all   |
|                                   |     mutual obligation             |
|                                   |     requirements where the hours  |
|                                   |     match work capacity           |
|                                   | -   The Restart wage subsidy may  |
|                                   |     be paid to employers that     |
|                                   |     employ a mature age job       |
|                                   |     seeker                        |
|                                   | -   Self-employment is a          |
|                                   |     recognised approved activity  |
|                                   |     and can be included in the    |
|                                   |     Job Plan as a voluntary or    |
|                                   |     compulsory activity           |
|                                   | -   New Enterprise Incentive      |
|                                   |     Scheme (NEIS) is an approved  |
|                                   |     activity that assists         |
|                                   |     eligible job seekers with     |
|                                   |     training and support to       |
|                                   |     establish a viable business   |
+-----------------------------------+-----------------------------------+
| 2                                 | Job search + Read more ...        |
|                                   |                                   |
|                                   | Employment Provider Services      |
|                                   | providers will determine the      |
|                                   | number of job search contacts     |
|                                   | required for a job seeker and     |
|                                   | include these in the Job Plan.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Language and literacy             |
|                                   | assistance + Read more ...        |
|                                   |                                   |
|                                   | Customers identified as requiring |
|                                   | assistance with language or       |
|                                   | literacy skills can be referred   |
|                                   | to the following programs:        |
|                                   |                                   |
|                                   | -   Adult Migrant Education       |
|                                   |     Program (AMEP)                |
|                                   | -   Skills for Education and      |
|                                   |     Employment (SEE)              |
+-----------------------------------+-----------------------------------+
| 4                                 | Non-vocational activities + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Job seekers may benefit from 1 of |
|                                   | the following activities:         |
|                                   |                                   |
|                                   | -   Undertaking an assessment of  |
|                                   |     their present capacity or     |
|                                   |     needs by a provider or        |
|                                   |     organisation                  |
|                                   | -   Drug and alcohol              |
|                                   |     rehabilitation                |
|                                   | -   Intervention - non specific   |
|                                   | -   Parenting skills program      |
|                                   | -   Health maintenance program    |
|                                   | -   Child care                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Restart wage subsidy + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Restart wage subsidy is paid  |
|                                   | to employers that employ a mature |
|                                   | age job seeker, who has been      |
|                                   | unemployed and on income support  |
|                                   | for 6 months or more, in a        |
|                                   | position:                         |
|                                   |                                   |
|                                   | -   that commenced on or after 1  |
|                                   |     July 2014                     |
|                                   | -   in which the employee works   |
|                                   |     for a minimum of 15 hours per |
|                                   |     week                          |
|                                   | -   that is a sustainable,        |
|                                   |     ongoing position, where the   |
|                                   |     employer knows of no reasons  |
|                                   |     why the job will not continue |
|                                   |     indefinitely (the employment  |
|                                   |     should not be expected to end |
|                                   |     when the wage subsidy ceases) |
|                                   | -   in which the employer is not  |
|                                   |     receiving any other           |
|                                   |     government wage subsidies,    |
|                                   |     with the exception of a       |
|                                   |     Tasmanian Jobs Programme wage |
|                                   |     subsidy                       |
|                                   | -   that complies with minimum    |
|                                   |     standards for employment as   |
|                                   |     established under relevant    |
|                                   |     Commonwealth, State or        |
|                                   |     Territory law (including any  |
|                                   |     award conditions applicable   |
|                                   |     to the position)              |
|                                   | -   that is not a                 |
|                                   |     commission-based,             |
|                                   |     self-employment or a          |
|                                   |     subcontracted position        |
|                                   | -   that does not displace an     |
|                                   |     existing employee and         |
|                                   | -   that will not bring the       |
|                                   |     Commonwealth into disrepute   |
|                                   |                                   |
|                                   | The subsidy will be delivered     |
|                                   | through jobactive providers. The  |
|                                   | Resources page contains a link to |
|                                   | information on Restart.           |
+-----------------------------------+-----------------------------------+
| 6                                 | Study + Read more ...             |
|                                   |                                   |
|                                   | Part-time or full-time study may  |
|                                   | be undertaken as an approved      |
|                                   | activity, depending on the job    |
|                                   | seeker's circumstances. Part-time |
|                                   | study may be approved with        |
|                                   | additional job search.            |
+-----------------------------------+-----------------------------------+
| 7                                 | Voluntary work + Read more ...    |
|                                   |                                   |
|                                   | Voluntary work is available as a  |
|                                   | voluntary activity for customers  |
|                                   | who are 55 years of age or over.  |
|                                   | Voluntary work must be with an    |
|                                   | approved volunteer organisation.  |
|                                   |                                   |
|                                   | Note: if the job seeker is        |
|                                   | required to attend a 'base' or    |
|                                   | like place, and is participating  |
|                                   | in a work-like activity for the   |
|                                   | duration of their time on call,   |
|                                   | these hours may count towards     |
|                                   | their voluntary work hours.       |
|                                   |                                   |
|                                   | If the job seeker is going about  |
|                                   | their daily life, without         |
|                                   | participating in any work-like    |
|                                   | activity while on call, these     |
|                                   | hours cannot be counted towards   |
|                                   | their voluntary work hours.       |
+-----------------------------------+-----------------------------------+
| 8                                 | Work for the Dole (WFD) + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Under jobactive, job seekers can  |
|                                   | participate in Work for the Dole  |
|                                   | (WFD) to improve their job        |
|                                   | prospects, get valuable work      |
|                                   | experience and help their local   |
|                                   | community. Participants are       |
|                                   | entitled to a supplement when     |
|                                   | participating in WFD.             |
