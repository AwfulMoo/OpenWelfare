FAQ’s - non-payment of FTB instalments

This table consists of Frequently Asked Questions about non-payment of
FTB instalments to zero entitlement customers.

+-----------------------------------+-----------------------------------+
| Item                              | Questions and Answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Who is affected by this           |
|                                   | measure? + Read more ...          |
|                                   |                                   |
|                                   | This measure only affects         |
|                                   | families who have had 2           |
|                                   | consecutive years of confirmed    |
|                                   | zero entitlement to FTB Part A    |
|                                   | and/or FTB Part B (referred to as |
|                                   | zero entitlement years) due to    |
|                                   | their actual taxable income (ATI) |
|                                   | being too high. These families    |
|                                   | will be prevented from receiving  |
|                                   | fortnightly instalments until     |
|                                   | they can demonstrate that they    |
|                                   | are again eligible. Where a       |
|                                   | significant change in             |
|                                   | circumstances occurs, families    |
|                                   | will not be precluded from        |
|                                   | fortnightly payments.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Who is not affected by this       |
|                                   | measure? + Read more ...          |
|                                   |                                   |
|                                   | Families who claim FTB via lump   |
|                                   | sum claim, families on income     |
|                                   | support payments and those that   |
|                                   | have a change in circumstances    |
|                                   | which meet the special            |
|                                   | circumstances for a non-payment   |
|                                   | of FTB instalments preclusion     |
|                                   | criteria are not affected.        |
+-----------------------------------+-----------------------------------+
| 3                                 | When does this measure            |
|                                   | commence? + Read more ...         |
|                                   |                                   |
|                                   | These changes are in place from 1 |
|                                   | July 2012.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | When will the changes end? + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This is an ongoing change to      |
|                                   | arrangements that apply to FTB.   |
+-----------------------------------+-----------------------------------+
