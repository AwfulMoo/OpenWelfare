IBNR overview

+-----------------------------------+-----------------------------------+
| Category                          | Description                       |
+-----------------------------------+-----------------------------------+
| Eligibility                       | Must be an active participating   |
|                                   | member of an MDO means a person   |
| (members)                         | who was a member of the MDO on 30 |
|                                   | June 2002, (whether or not the    |
|                                   | person ceased to be a member of   |
|                                   | the MDO after that date). Members |
|                                   | must have had incident occurring  |
|                                   | based cover on 30 June 2002       |
+-----------------------------------+-----------------------------------+
| Eligibility                       | Applications for claims under the |
|                                   | IBNR can only be made by United   |
| (Applicant)                       | Medical Protection Limited. They  |
|                                   | are the 1 eligible participating  |
|                                   | MDO for the scheme.               |
+-----------------------------------+-----------------------------------+
| Notification Date                 | Incident/Claim must be notified   |
|                                   | on or after 1 July 2002           |
+-----------------------------------+-----------------------------------+
| Criteria                          | -   The incident must have        |
|                                   |     occurred on or before 30 June |
|                                   |     2002                          |
|                                   | -   Have occurred in the course   |
|                                   |     of, or in connection with,    |
|                                   |     the person’s practice of      |
|                                   |     their medical profession      |
|                                   | -   The MDO must not have been    |
|                                   |     notified of the occurrence of |
|                                   |     the incident, nor claim       |
|                                   |     against or by the person in   |
|                                   |     relation to the incident,     |
|                                   |     before 1 July 2002            |
|                                   | -   The MDO must have made a      |
|                                   |     payment in relation to the    |
|                                   |     claim                         |
|                                   | -   The payment must be made in   |
|                                   |     line with the insurers        |
|                                   |     indemnity policy issued and   |
|                                   |     within the ordinary course of |
|                                   |     business                      |
|                                   | -   The eligibility of the        |
|                                   |     applicant and the member is   |
|                                   |     asserted                      |
+-----------------------------------+-----------------------------------+
| What is not payable?              | Applicants cannot claim an amount |
|                                   | if:                               |
|                                   |                                   |
|                                   | -   another party has             |
|                                   |     responsibility to pay it      |
|                                   | -   it is an insurer-to-insurer   |
|                                   |     payment                       |
|                                   | -   the cost wasn’t incurred in   |
|                                   |     relation to the incident      |
|                                   | -   the judgment or settlement    |
|                                   |     amounts are still within the  |
|                                   |     appeal period or have been    |
|                                   |     appealed or stayed            |
|                                   | -   any portion relates to the    |
|                                   |     incidents that occur partly   |
|                                   |     or fully within the course of |
|                                   |     treating a public patient in  |
|                                   |     a public hospital. If it      |
|                                   |     occurs fully, the application |
|                                   |     is rejected. If partly, the   |
|                                   |     total application payment     |
|                                   |     must be reduced               |
|                                   |     proportionately               |
|                                   | -   the claim is for members who  |
|                                   |     have received an exemption 5, |
|                                   |     8 or 9 from the UMP SP        |
|                                   |     arrangements are not eligible |
|                                   |     for payment                   |
+-----------------------------------+-----------------------------------+
| Clarification                     | A payment that is eligible for    |
|                                   | ROCS may not be submitted under   |
|                                   | the IBNR scheme. ROCS should be   |
|                                   | applied for in the first          |
|                                   | instance.                         |
+-----------------------------------+-----------------------------------+
