Updating titles in CDMS

This table describes when a title must be updated on existing enrolment
records as indicated on a form.

+-----------------------+-----------------------+-----------------------+
| Existing record       | Form details reflect: | Action required:      |
| displays:             |                       |                       |
+-----------------------+-----------------------+-----------------------+
| Title recorded        | No title              | Do not update         |
|                       |                       | existing title.       |
+-----------------------+-----------------------+-----------------------+
| Title recorded        | A different title     | Update record with    |
|                       |                       | new title. Note: if   |
|                       |                       | person is the group   |
|                       |                       | contact, update group |
|                       |                       | contact details.      |
+-----------------------+-----------------------+-----------------------+
| No title recorded     | A title               | Update record with    |
|                       |                       | new title.            |
+-----------------------+-----------------------+-----------------------+

Note: Update a title on an existing enrolment record at the customer’s
request where a customer attends a Service Centre or calls the general
enquiries line and passes a public security check.
