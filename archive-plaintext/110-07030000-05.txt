Reminder notification will issue 18 calendar days after the initial request

Table 1

+-----------------------------------+-----------------------------------+
| Day 18 occurs on a:               | Reminder notification issued      |
+-----------------------------------+-----------------------------------+
| Weekday                           | On the same weekday               |
+-----------------------------------+-----------------------------------+
| Saturday                          | Friday before the Saturday        |
+-----------------------------------+-----------------------------------+
| Sunday                            | Monday after Sunday               |
+-----------------------------------+-----------------------------------+
| Public holiday not immediately    | First weekday after public        |
| before or after weekend           | holiday                           |
|                                   |                                   |
| (not part of a long weekend)      |                                   |
+-----------------------------------+-----------------------------------+
| Public holiday immediately before | Earliest available weekday before |
| or after weekend                  | long weekend                      |
|                                   |                                   |
| (part of a long weekend)          |                                   |
