Participants who were in the CDC temporarily or address was incorrect

Table 3: How to record details to end a person's participation in the
CDC who were only in the CDC area temporarily, or had incorrect address
coding making them ineligible for the CDC. Portions of this process are
completed by the Cashless Debit Card Processing Team only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Participant in CDC area           |
|                                   | temporarily + Read more ...       |
|                                   |                                   |
|                                   | The CDC Processing Team will      |
|                                   | assess whether the person's       |
|                                   | participation in the CDC can be   |
|                                   | ended, taking into account        |
|                                   | address history and ties to the   |
|                                   | area, and apply End Participation |
|                                   | coding to the record.             |
|                                   |                                   |
|                                   | Recent contacts with the agency,  |
|                                   | Rent or Centrepay deductions are  |
|                                   | also taken into account.          |
|                                   |                                   |
|                                   | Supporting evidence from the      |
|                                   | participant may be required to    |
|                                   | determine the correct address     |
|                                   | details before a change or end of |
|                                   | participation in the CDC can      |
|                                   | occur. Evidence may include       |
|                                   | letters, utility bills, invoices  |
|                                   | or statements from a reputable    |
|                                   | third party. Statutory            |
|                                   | Declarations alone will not       |
|                                   | suffice as evidence.              |
|                                   |                                   |
|                                   | If it cannot be satisfied that    |
|                                   | the move genuinely occurred, the  |
|                                   | participant will remain on the    |
|                                   | CDC, even if they have backdated  |
|                                   | their address, until evidence can |
|                                   | be provided.                      |
|                                   |                                   |
|                                   | Note: updating or deleting home   |
|                                   | address details for the purposes  |
|                                   | of circumventing the CDC will not |
|                                   | automatically end participation.  |
|                                   | Address updates done incorrectly  |
|                                   | in an effort to avoid the CDC     |
|                                   | place the participant at risk of  |
|                                   | overpayment for Rent Assistance   |
|                                   | or Remote Area Allowance. Service |
|                                   | Officers must not update          |
|                                   | addresses for the purposes of     |
|                                   | ending a person's participation   |
|                                   | in the CDC without discussing     |
|                                   | with the CDC Processing Team      |
|                                   | initially.                        |
|                                   |                                   |
|                                   | Is it confirmed that the          |
|                                   | participant was only in the CDC   |
|                                   | area temporarily, or they moved   |
|                                   | before the CDC start date?        |
|                                   |                                   |
|                                   | -   Yes, see Resources for        |
|                                   |     contact details for technical |
|                                   |     support to allow them to      |
|                                   |     assess the request. If the    |
|                                   |     request is approved, update   |
|                                   |     the address                   |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 2                                 | End participation + Read more ... |
|                                   |                                   |
|                                   | For Cashless Debit Card           |
|                                   | Processing Team only              |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | -   Go to Workspace > Cashless    |
|                                   |     Debit Card > Decision Details |
|                                   |     and select End Participation  |
|                                   | -   Select Decision Type of IIE   |
|                                   |     (Customer was incorrectly     |
|                                   |     identified as eligible) from  |
|                                   |     the drop down menu            |
|                                   | -   Under Additional Information  |
|                                   |     select Add mandatory notes,   |
|                                   |     add notes and select Continue |
|                                   |     to complete the activity      |
|                                   |                                   |
|                                   | This will end the CDC             |
|                                   | participation and issue an advice |
|                                   | letter.                           |
|                                   |                                   |
|                                   | Check the PQSS screen to confirm  |
|                                   | that the participant is no longer |
|                                   | in CDC service component 716.     |
|                                   |                                   |
|                                   | The participant will now remain   |
|                                   | off the CDC, unless their home    |
|                                   | address is subsequently changed   |
|                                   | to a CDC area.                    |
