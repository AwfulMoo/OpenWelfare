This page has examples of common challenges faced by parents and
provides some guidance on addressing these challenges. Service Officers
should examine the customer's circumstances to identify the position
they are currently in, before making broad statements and advising that
participation will help in improving their current circumstance.

| 3                                 | Partner's attitude + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Most parents with children do |
|                                   |     participate in part-time work |

|                                   |     the number and availability   |
