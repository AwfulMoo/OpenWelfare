This page contains information relating to an ABSTUDY student or
Australian Apprentice leaving the parental home.

ABSTUDY customer leaving parental home

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Leaving the parental home         |
|                                   |                                   |
|                                   | If an ABSTUDY student or          |
|                                   | Australian Apprentice leaves      |
|                                   | their parental home, the reasons  |
|                                   | for their departure should be     |
|                                   | investigated as they may need a   |
|                                   | reassessment of their rate of     |
|                                   | payment. A change in the          |
|                                   | customer's circumstances may also |
|                                   | affect the entitlements of other  |
|                                   | siblings in the family unit.      |
|                                   |                                   |
|                                   | If the customer is:               |
|                                   |                                   |
|                                   | -   in receipt of the dependent   |
|                                   |     rate of ABSTUDY, and is now   |
|                                   |     living with a different       |
|                                   |     Parent, see Changes which may |
|                                   |     affect the Parental Income    |
|                                   |     Test(PIT)                     |
|                                   | -   wishing to apply for an away  |
|                                   |     from home rate of ABSTUDY,    |
|                                   |     see Away from home rate for   |
|                                   |     ABSTUDY                       |
|                                   | -   wishing to applying for an    |
|                                   |     Independent rate of ABSTUDY,  |
|                                   |     see Independence for ABSTUDY  |
|                                   |     customers                     |
|                                   | -   in receipt of Schooling A     |
|                                   |     Allowances (under 16 and      |
|                                   |     student is living at home),   |
|                                   |     see Processing ABSTUDY School |
|                                   |     Term Allowance payments       |
+-----------------------------------+-----------------------------------+
