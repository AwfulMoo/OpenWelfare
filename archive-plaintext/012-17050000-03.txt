Registering a new hospital claim

Table 1: This table describes the steps required to register a new
hospital claim.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log in CICS Production + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Key ‘VRDI’ and press [Enter].     |
|                                   |                                   |
|                                   | The claim registration screen     |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter registration information +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Enter the required information in |
|                                   | the following fields:             |
|                                   |                                   |
|                                   | -   Payee provider (only when     |
|                                   |     shown on the claim form)      |
|                                   | -   Service provider              |
|                                   | -   Claim ID                      |
|                                   | -   DOL                           |
|                                   | -   Claim date                    |
|                                   | -   Vouchers quoted               |
|                                   | -   Claim type (H – for           |
|                                   |     in-patient private hospital   |
|                                   |     claim: R – for public         |
|                                   |     hospital claim for example,   |
|                                   |     prosthesis and private        |
|                                   |     non-contracted hospital)      |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | See the Resources page for more   |
|                                   | information about the field       |
|                                   | descriptions.                     |
|                                   |                                   |
|                                   | Details of the servicing          |
|                                   | provider, and payee provider if   |
|                                   | applicable, display.              |
|                                   |                                   |
|                                   | ‘U TO UPDTE’ message will also    |
|                                   | display.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Check for errors + Read more ...  |
|                                   |                                   |
|                                   | Check the details displayed to    |
|                                   | ensure the correct provider       |
|                                   | numbers have been keyed. If       |
|                                   | errors exist:                     |
|                                   |                                   |
|                                   | -   correct the errors            |
|                                   | -   press [Enter]                 |
|                                   | -   over-key VRDC with VRDU and   |
|                                   |     press [Enter] to update and   |
|                                   |     start entering the claim      |
|                                   |     details                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Access hospital data entry        |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Over-key VRDC with ‘VRDU’ and     |
|                                   | press [Enter].                    |
|                                   |                                   |
|                                   | The Hospital data entry screen is |
|                                   | displayed. The type of data entry |
|                                   | screen displayed depends on the   |
|                                   | type of hospital claim being      |
|                                   | processed.                        |

Identifying, accessing and resolving claims

Table 3: This table describes how to identify, access and resolve claims
using the Pend Resolution Basic screen.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identify claim + Read more ...    |
|                                   |                                   |
|                                   | Identify a claim with the status  |
|                                   | of ASS/ERR.                       |
|                                   |                                   |
|                                   | For more information see Pend     |
|                                   | Resolution in Claims for          |
|                                   | Veterans' Affairs Processing      |
|                                   | (VAP)                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter details + Read more ...     |
|                                   |                                   |
|                                   | Key either:                       |
|                                   |                                   |
|                                   | -   ‘VAPI’, PROVIDER NUMBER,CLAIM |
|                                   |     ID                            |
|                                   | -   ‘VQOI’ CLAIM ID               |
|                                   | -   ‘VQRF’, CLAIM ID              |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | Note: where a payee provider is   |
|                                   | involved, the provider number     |
|                                   | keyed in the control line is that |
|                                   | of the payee and not the service  |
|                                   | provider.                         |
|                                   |                                   |
|                                   | Where there is more than one      |
|                                   | claim with the same Claim ID, the |
|                                   | system returns the message ‘More  |
|                                   | lines’. Select the correct claim, |
|                                   | by keying ‘S’ in the S field and  |
|                                   | select [Enter].                   |
|                                   |                                   |
|                                   | The Pend processing screen is     |
|                                   | displayed.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Check for errors + Read more ...  |
|                                   |                                   |
|                                   | The system highlights any errors  |
|                                   | in the claim and displays return  |
|                                   | messages that can assist in       |
|                                   | resolving these errors. Place the |
|                                   | cursor on the message and select  |
|                                   | [PF1] for instructions on         |
|                                   | resolving the error.              |
|                                   |                                   |
|                                   | When no errors can be found the   |
|                                   | system returns message ‘NO MORE   |
|                                   | ERRORS OK TO PAY’.                |
+-----------------------------------+-----------------------------------+
| 4                                 | Pay the claim + Read more ...     |
|                                   |                                   |
|                                   | Over-key VAPC with ‘VPPI’ and     |
|                                   | press [Enter] then when ‘No       |
|                                   | Error’ message is displayed key   |
|                                   | ‘P’ and press [Enter] to pay the  |
|                                   | claim.                            |

Adding a CERT (3B Certificate) line

Table 6: This table describes the procedure to add a CERT (3B
Certificate) line. A 3B certificate is required when the veteran claims
for hospitalisation as an acute treatment exceeds 35 continuous days.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Action Pend Resolution Insert     |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | On the Pend Resolution Insert     |
|                                   | screen, key ‘I’ in the A field    |
|                                   | and select PF6-INS/DEL.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter details + Read more ...     |
|                                   |                                   |
|                                   | Enter the following details:      |
|                                   |                                   |
|                                   | -   ‘CERT’ in the ITEM field      |
|                                   | -   commencement date of acute    |
|                                   |     care period in the DOS field  |
|                                   |                                   |
|                                   | Press [Enter].                    |
+-----------------------------------+-----------------------------------+
| 3                                 | System message displayed + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | The system displays the message   |
|                                   | ‘1-LST DT’ and highlights the TO  |
|                                   | field on the CERT line.           |
+-----------------------------------+-----------------------------------+
| 4                                 | Enter end date details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Enter the end date of the acute   |
|                                   | care period in the TO field of    |
|                                   | the CERT line and press [Enter].  |
|                                   |                                   |
|                                   | The message ‘NO ERRORS OK TO PAY’ |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
