Multi-Functional Device (MFD) scanning errors

Table 1

+-----------------------------------+-----------------------------------+
| Error                             | Resolution                        |
+-----------------------------------+-----------------------------------+
| Paper jam                         | The Clear the Paper Jam in the    |
|                                   | Document Feeder error message     |
|                                   | will show.                        |
|                                   |                                   |
|                                   | Clear the paper jam:              |
|                                   |                                   |
|                                   | -   open the document feeder      |
|                                   |     cover                         |
|                                   | -   remove the paper mail         |
|                                   | -   close the document feeder     |
|                                   |     cover                         |
|                                   | -   select Confirm on the MFD     |
|                                   |     screen                        |
|                                   | -   select Delete on the MFD      |
|                                   |     screen                        |
|                                   | -   rescan the paper mail. Start  |
|                                   |     from the jammed paper mail    |
|                                   |                                   |
|                                   | All paper mail scanned prior to   |
|                                   | the paper jam is sent to KOFAX    |
|                                   | Validation application.           |
|                                   |                                   |
|                                   | Do not rescan paper mail scanned  |
|                                   | before the paper jam.             |
+-----------------------------------+-----------------------------------+
| Undeliverable transmission        | The Multi-Functional Device (MFD) |
|                                   | may print an undelivered          |
|                                   | transmission receipt.             |
|                                   |                                   |
|                                   | The transmission receipt will     |
|                                   | show the pages that did not scan  |
|                                   | successfully.                     |
|                                   |                                   |
|                                   | If the transmission receipt shows |
|                                   | a:                                |
|                                   |                                   |
|                                   | -   work item, refer to the Team  |
|                                   |     Leader                        |
|                                   | -   document separator page,      |
|                                   |     ignore                        |
+-----------------------------------+-----------------------------------+
