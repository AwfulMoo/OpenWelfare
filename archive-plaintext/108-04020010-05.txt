Examples of determining if a valuation is required

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Claiming Age Pension              |
|                                   |                                   |
|                                   | Trevor is claiming Age Pension    |
|                                   | and will be paid under the Assets |
|                                   | Test. Trevor's assets include     |
|                                   | earth moving plant and equipment. |
|                                   | If there is insufficient evidence |
|                                   | to accept Trevor's estimate, the  |
|                                   | Service Officer should request a  |
|                                   | valuation of Trevor's plant and   |
|                                   | equipment.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Taxi licence value                |
|                                   |                                   |
|                                   | Anne owns and operates a taxi in  |
|                                   | Sydney. Due to an injury Anne is  |
|                                   | unable to operate her business at |
|                                   | present. Anne will be paid under  |
|                                   | the Assets Test. Anne's assets    |
|                                   | include a taxi licence. The       |
|                                   | Service Officer should check      |
|                                   | information on Valuation of Real  |
|                                   | Estate and Other Assets for       |
|                                   | current details regarding taxi    |
|                                   | licence values. Market Value of a |
|                                   | taxi licence can also be          |
|                                   | requested via email to the        |
|                                   | Valuation Helpdesk.               |
+-----------------------------------+-----------------------------------+
