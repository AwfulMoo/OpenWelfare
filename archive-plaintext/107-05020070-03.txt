For Debt Staff only

Discounting arrangements

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customers capacity to repay +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Use the Debt recovery script to   |
|                                   | action a discount arrangement.    |
|                                   |                                   |
|                                   | The customer must ask for a       |
|                                   | discount. Do not offer or suggest |
|                                   | a discount. If the customer has   |
|                                   | the capacity to pay the debt in   |
|                                   | full, do not accept a discount.   |
|                                   |                                   |
|                                   | Check the Income and Assets (ITS) |
|                                   | screen for any savings or         |
|                                   | investments. This information is  |
|                                   | an indication only. Discuss any   |
|                                   | savings or investments with the   |
|                                   | customer to find out if they can  |
|                                   | repay the debt in full. Update    |
|                                   | any outdated information.         |
|                                   |                                   |
|                                   | Can the customer repay the debt   |
|                                   | in full?                          |
|                                   |                                   |
|                                   | -   Yes, do not accept a          |
|                                   |     discount. Record details on a |
|                                   |     DOC. Procedure ends here      |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Previous discount + Read more ... |
|                                   |                                   |
|                                   | If a previous discount            |
|                                   | arrangement was broken, do not    |
|                                   | consider another discount         |
|                                   | arrangement for the same debt     |
|                                   | unless exceptional circumstances  |
|                                   | exist.                            |
|                                   |                                   |
|                                   | If the customer wants to          |
|                                   | renegotiate a pending discount    |
|                                   | arrangement, consider their       |
|                                   | circumstances. A customer cannot  |
|                                   | continue to renegotiate a         |
|                                   | discount arrangement as the       |
|                                   | present value may change.         |
|                                   |                                   |
|                                   | Is a discount appropriate?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, tell the customer the     |
|                                   |     reason/s. DOC the details why |
|                                   |     the discount is not suitable. |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | When discounts cannot be          |
|                                   | considered + Read more ...        |
|                                   |                                   |
|                                   | Do not consider discounts for     |
|                                   | debts raised under:               |
|                                   |                                   |
|                                   | -   the Public Governance,        |
|                                   |     Performance and               |
|                                   |     Accountability Act 2013       |
|                                   |     (PGPA), or                    |
|                                   | -   prior to 1 July 2014, the     |
|                                   |     Financial Management and      |
|                                   |     Accountability (FMA) Act 1997 |
|                                   |     or                            |
|                                   | -   other Government Department   |
|                                   |     debts (OGD)                   |
|                                   |                                   |
|                                   | To check if a debt is a PGPA/FMA  |
|                                   | Act debt:                         |
|                                   |                                   |
|                                   | -   go to the Debt List (OPDL)    |
|                                   |     screen                        |
|                                   | -   select the debt/s             |
|                                   | -   view the Authority field      |
|                                   |                                   |
|                                   | To check if a debt is an OGD      |
|                                   | debt:                             |
|                                   |                                   |
|                                   | -   go to the Debt List (OPDL)    |
|                                   |     screen                        |
|                                   | -   view the Benefit (Ben) field. |
|                                   |     (DVA, AFFA, NZ, DEWR-NEIS,    |
|                                   |     CSS are OGD benefit codes)    |
|                                   |                                   |
|                                   | Does the customer have at least   |
|                                   | one debt raised under the         |
|                                   | PGPA/FMA Act or an OGD debt?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer has at least one debt    |
|                                   | raised under the PGPA/FMA Act or  |
|                                   | an OGD debt + Read more ...       |
|                                   |                                   |
|                                   | If the customer has only          |
|                                   | outstanding debt/s raised under   |
|                                   | the PGPA/FMA Act or an OGD:       |
|                                   |                                   |
|                                   | -   do not consider a discount    |
|                                   | -   record details on a DOC       |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | If the customer has multiple      |
|                                   | debts, including at least one     |
|                                   | debt raised under the PGPA/FMA    |
|                                   | Act or an OGD debt:               |
|                                   |                                   |
|                                   | -   Consider a discount on the    |
|                                   |     balance of the debts which do |
|                                   |     not include the PGPA/FMA Act  |
|                                   |     and/or OGD debts              |
|                                   | -   Go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Delegation + Read more ...        |
|                                   |                                   |
|                                   | Does the staff member have the    |
|                                   | delegation to discount a debt?    |
|                                   | See the Resources page for a link |
|                                   | to the delegation instrument.     |
|                                   |                                   |
|                                   | Yes, go to Step 6                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Refer to a staff member with  |
|                                   |     the correct delegation        |
|                                   | -   Record the details of the     |
|                                   |     referral on a DOC             |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 6                                 | Discussions + Read more ...       |
|                                   |                                   |
|                                   | The customer needs to:            |
|                                   |                                   |
|                                   | -   know how much they can afford |
|                                   |     to pay                        |
|                                   | -   know where the money will     |
|                                   |     come from, for example        |
|                                   |     savings, loan, sale of asset  |
|                                   | -   agree to the discount on the  |
|                                   |     same day as the negotiation.  |
|                                   |     If negotiations fail, or the  |
|                                   |     customer cannot agree to the  |
|                                   |     discount, the discount is     |
|                                   |     void. It cannot form part of  |
|                                   |     future discussions as the     |
|                                   |     present value of the debt may |
|                                   |     change between discussions    |
|                                   |                                   |
|                                   | Is the customer ready to discuss  |
|                                   | a discount?                       |
|                                   |                                   |
|                                   | Yes, go to Step 7                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Advise the reasons why        |
|                                   |     discussions cannot proceed    |
|                                   | -   Record details on a DOC       |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 7                                 | Update the Interest Charge + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Update the Interest Charge for    |
|                                   | the debt(s) before proceeding.    |
|                                   |                                   |
|                                   | In Customer First:                |
|                                   |                                   |
|                                   | -   In Workspace, select Debt     |
|                                   |     List workflow (located under  |
|                                   |     Financial Customer Care)      |
|                                   | -   Select Update Interest button |
|                                   |     (in the header) to request an |
|                                   |     update on the current         |
|                                   |     interest values. This will    |
|                                   |     recalculate and show the      |
|                                   |     amount of interest up to the  |
|                                   |     previous day                  |
|                                   |                                   |
|                                   | In Process Direct                 |
|                                   |                                   |
|                                   | Select the Debt icon to launch    |
|                                   | Debt Management                   |
|                                   |                                   |
|                                   | The Debt List screen will display |
|                                   |                                   |
|                                   | Select the Debt Case ID           |
|                                   |                                   |
|                                   | Select [more options] > Update    |
|                                   | Interest. To prompt a             |
|                                   | Confirmation message to appear    |
|                                   |                                   |
|                                   | Select: Yes, to update the        |
|                                   | balance                           |
|                                   |                                   |
|                                   | The update will prompt one of the |
|                                   | following messages:               |
|                                   |                                   |
|                                   | -   Successful update - 'Interest |
|                                   |     has been re-calculated for    |
|                                   |     the debt'                     |
|                                   | -   Interest already updated -    |
|                                   |     'E250: Interest is already up |
|                                   |     to date'                      |
|                                   | -   Not due - 'E250: No interest  |
|                                   |     to be changed'                |
|                                   |                                   |
|                                   | Include a note about needing to   |
|                                   | update in DMIS if customer has    |
|                                   | debts in both systems to receive  |
|                                   | final payout figure               |
|                                   |                                   |
|                                   | Go to Step 8.                     |
+-----------------------------------+-----------------------------------+
| 8                                 | Present value of debt + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the Present Value           |
|                                   | Calculator (PVC) to find out the  |
|                                   | present value of the debt. To     |
|                                   | access the PVC, see Resources     |
|                                   | page.                             |
|                                   |                                   |
|                                   | The present value shown on OPDI   |
|                                   | is different to the present value |
|                                   | shown on the PVC. The PVC is the  |
|                                   | correct amount. Do not use the    |
|                                   | OPDI present value for your       |
|                                   | negotiations.                     |
|                                   |                                   |
|                                   | Enter the Outstanding debt        |
|                                   | amount:                           |
|                                   |                                   |
|                                   | -   Go to OPDL and use the total  |
|                                   |     outstanding balance           |
|                                   | -   This amount includes any      |
|                                   |     Recovery fee and interest     |
|                                   |     charges                       |
|                                   | -   Do not include any PGPA/FMA   |
|                                   |     or OGD debts in this amount   |
|                                   |                                   |
|                                   | Enter the Instalment amount and   |
|                                   | Frequency:                        |
|                                   |                                   |
|                                   | -   Go to OPAL and check for a    |
|                                   |     current payment arrangement   |
|                                   | -   If the customer has an        |
|                                   |     existing arrangement use that |
|                                   |     amount and frequency          |
|                                   | -   If the customer does not have |
|                                   |     an existing arrangement, use  |
|                                   |     the best arrangement the      |
|                                   |     customer could afford, for    |
|                                   |     example, what they would pay  |
|                                   |     if a discount was not         |
|                                   |     available                     |
|                                   |                                   |
|                                   | Press Calculate Value             |
|                                   |                                   |
|                                   | Check the Present Value field in  |
|                                   | the results section               |
|                                   |                                   |
|                                   | -   This is the maximum           |
|                                   |     discounted amount             |
|                                   | -   Do not disclose this amount   |
|                                   |     to the customer               |
|                                   | -   The discounted amount must    |
|                                   |     not be below the present      |
|                                   |     value                         |
|                                   | -   Start negotiations at the     |
|                                   |     smallest discount amount (at  |
|                                   |     least 50%). See Resources for |
|                                   |     an example                    |
|                                   |                                   |
|                                   | Did the customer agree to a       |
|                                   | discounted amount above the       |
|                                   | present value of the debt?        |
|                                   |                                   |
|                                   | Yes, go to Step 9                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Tell the customer the agency  |
|                                   |     cannot accept their offer     |
|                                   | -   Record details on a DOC       |
|                                   | -   Negotiate a cash arrangement. |
|                                   |     See Arranging cash payments   |
|                                   |     to recover debts              |
+-----------------------------------+-----------------------------------+
| 9                                 | Method of payment + Read more ... |
|                                   |                                   |
|                                   | Ask the customer which payment    |
|                                   | method they would like. Give them |
|                                   | the payment details.              |
|                                   |                                   |
|                                   | Internet - The customer can make  |
|                                   | payments via the Money You Owe    |
|                                   | option online within their:       |
|                                   |                                   |
|                                   | -   Centrelink online account, or |
|                                   | -   Express Plus Centrelink       |
|                                   |     mobile app                    |
|                                   |                                   |
|                                   | BPAY® - The customer can make     |
|                                   | payments with BPAY® through their |
|                                   | financial institution.            |
|                                   |                                   |
|                                   | -   Use Biller Code 21915         |
|                                   | -   If the customer's financial   |
|                                   |     institution allows a BPAY®    |
|                                   |     credit card payment, there    |
|                                   |     may be additional fees or     |
|                                   |     interest charges. They should |
|                                   |     check this with their         |
|                                   |     financial institution before  |
|                                   |     they make the payment         |
|                                   |                                   |
|                                   | Australia Post - The customer can |
|                                   | make payments at any Australian   |
|                                   | post office or postal outlet      |
|                                   |                                   |
|                                   | -   They can pay via EFTPOS or    |
|                                   |     cash, using the bar-code on   |
|                                   |     their letter or payment card  |
|                                   | -   Do not issue a payment card   |
|                                   |     for a discount payment. The   |
|                                   |     customer can use the account  |
|                                   |     payable or arrangement        |
|                                   |     confirmation letter           |
|                                   |                                   |
|                                   | POSTbillpay - This is a phone or  |
|                                   | online payment made via Australia |
|                                   | POSTbillpay                       |
|                                   |                                   |
|                                   | -   Use POSTbillpay code: 0802    |
|                                   |                                   |
|                                   | Mail - Make cheques payable to    |
|                                   | the 'Collector of Public Money'.  |
|                                   | The Customer Reference Number     |
|                                   | (CRN) must be written on the back |
|                                   | of the cheque. Post the cheque to |
|                                   | the address on the top left of    |
|                                   | their Account Payable, see        |
|                                   | Collector of Public Money.        |
|                                   | Cheques are not a preferred       |
|                                   | payment option                    |
|                                   |                                   |
|                                   | A customer cannot pay a discount  |
|                                   | by direct debit.                  |
|                                   |                                   |
|                                   | Go to Step 10.                    |
+-----------------------------------+-----------------------------------+
| 10                                | Payment date + Read more ...      |
|                                   |                                   |
|                                   | Ask the customer when they can    |
|                                   | make the payment. The start date  |
|                                   | must:                             |
|                                   |                                   |
|                                   | be within 1 month - Do not make   |
|                                   | an arrangement that starts more   |
|                                   | than 1 month in the future. The   |
|                                   | customer will continue to accrue  |
|                                   | Interest Charge (IC) until they   |
|                                   | make the first payment. They can  |
|                                   | also use this as a method to      |
|                                   | avoid other recovery action such  |
|                                   | as a tax garnishee or forget      |
|                                   | about the payment if it is too    |
|                                   | far into the future               |
|                                   |                                   |
|                                   | start after the due date of a     |
|                                   | debt – if an arrangement start    |
|                                   | date is before the due date of a  |
|                                   | debt, and a payment that is due   |
|                                   | before the due date is            |
|                                   | insufficient or not paid:         |
|                                   |                                   |
|                                   | -   a 'Payment Not Received'      |
|                                   |     letter does not issue         |
|                                   | -   a broken arrangement activity |
|                                   |     does not generate             |
|                                   | -   Interest Charge (IC) applies  |
|                                   |     from the due date for         |
|                                   |     non-current customers         |
|                                   |                                   |
|                                   | Go to Step 11.                    |
+-----------------------------------+-----------------------------------+
| 11                                | Commitment to pay + Read more ... |
|                                   |                                   |
|                                   | Repeat the details of the         |
|                                   | arrangement, including amount,    |
|                                   | date and method.                  |
|                                   |                                   |
|                                   | Tell the customer that only 1     |
|                                   | discount can apply to these       |
|                                   | debts. If they do not pay the     |
|                                   | discounted amount as agreed, no   |
|                                   | further discounts will be         |
|                                   | discussed.                        |
|                                   |                                   |
|                                   | Confirm the terms of their        |
|                                   | payment arrangement. For example: |
|                                   |                                   |
|                                   | 'Today you have entered into a    |
|                                   | verbal agreement to repay this    |
|                                   | debt. It is important you keep to |
|                                   | the terms of the agreement you    |
|                                   | have made today. If you are       |
|                                   | unable to make a payment at any   |
|                                   | stage please contact us straight  |
|                                   | away so can we help you.'         |
|                                   |                                   |
|                                   | Explain the consequences of not   |
|                                   | keeping to their payment          |
|                                   | arrangement. This includes        |
|                                   | recovery actions that may take    |
|                                   | place under the cost              |
|                                   | effectiveness guidelines.         |
|                                   |                                   |
|                                   | Do not threaten the customer with |
|                                   | recovery action. This can         |
|                                   | escalate the situation.           |
|                                   |                                   |
|                                   | Go to Step 12.                    |
+-----------------------------------+-----------------------------------+
| 12                                | Stop existing arrangement + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Stop any current cash             |
|                                   | arrangements before coding a      |
|                                   | discount.                         |
|                                   |                                   |
|                                   | Go to the Payment Arrangement     |
|                                   | List (OPAL) screen                |
|                                   |                                   |
|                                   | -   'S'elect the current          |
|                                   |     arrangement                   |
|                                   | -   press [Enter]                 |
|                                   | -   Type 'T' in the Cease Date    |
|                                   |     field                         |
|                                   | -   press [Enter]                 |
|                                   |                                   |
|                                   | Do not stop withholdings as they  |
|                                   | will automatically reapply        |
|                                   | overnight.                        |
|                                   |                                   |
|                                   | Go to Step 13.                    |
+-----------------------------------+-----------------------------------+
| 13                                | The Discount (OPDI) screen + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the Debt recovery script.     |
|                                   |                                   |
|                                   | Follow the manual steps when the  |
|                                   | script is not available.          |
|                                   |                                   |
|                                   | On the Activity List (AL) screen: |
|                                   |                                   |
|                                   | -   'S'elect the DMN/RCV No       |
|                                   |     Arrangement activity          |
|                                   | -   Press [Enter]                 |
|                                   |                                   |
|                                   | If there is no DMN/RCV No         |
|                                   | Arrangement activity:             |
|                                   |                                   |
|                                   | -   Key 'OPAL' in the Nxt field   |
|                                   | -   Press [Enter]                 |
|                                   | -   In the Create new arrangement |
|                                   |     type field, type 'DSC'        |
|                                   | -   Press [Enter]                 |
|                                   |                                   |
|                                   | On the Discount (OPDI) screen,    |
|                                   | complete the following fields:    |
|                                   |                                   |
|                                   | Instalment Amount $ this is the   |
|                                   | same amount used in the PVC       |
|                                   |                                   |
|                                   | Arrangement Start Date this       |
|                                   | should be the same as the Payment |
|                                   | Due Date                          |
|                                   |                                   |
|                                   | Frequency this is the same        |
|                                   | frequency used in the PVC         |
|                                   |                                   |
|                                   | The Amount to be Discounted $     |
|                                   | field is prefilled. This should   |
|                                   | be the total outstanding debt     |
|                                   | balance and should be the same    |
|                                   | amount as used in the PVC. Update |
|                                   | this field if required            |
|                                   |                                   |
|                                   | Press [Enter], the Present Value  |
|                                   | amount will automatically         |
|                                   | calculate                         |
|                                   |                                   |
|                                   | Amount to be Paid $ this amount   |
|                                   | must not be below the present     |
|                                   | value amount                      |
|                                   |                                   |
|                                   | Payment Due Date this is the date |
|                                   | the customer has agreed to pay    |
|                                   | the discounted amount of debt as  |
|                                   | negotiated                        |
|                                   |                                   |
|                                   | Press [Enter], the Amount to be   |
|                                   | Waived $ field will automatically |
|                                   | update                            |
|                                   |                                   |
|                                   | -   The present value shown on    |
|                                   |     OPDI is different to the      |
|                                   |     present value shown on the    |
|                                   |     PVC. The PVC is the correct   |
|                                   |     amount. Do not use the OPDI   |
|                                   |     present value for             |
|                                   |     negotiations                  |
|                                   |                                   |
|                                   | Comments include any relevant     |
|                                   | notes of the arrangement          |
|                                   |                                   |
|                                   | Go to the Assessment Finalisation |
|                                   | (OPAF) screen:                    |
|                                   |                                   |
|                                   | -   Type 'Y' in the Finished?     |
|                                   |     Field                         |
|                                   | -   Press [Enter]                 |
|                                   | -   If the activity is to be      |
|                                   |     Quality On Line (QOL)         |
|                                   |     checked, type 'Q' in the      |
|                                   |     Finished? field               |
|                                   |                                   |
|                                   | A Confirmation of Discount letter |
|                                   | will send to the customer. This   |
|                                   | confirms the details of the       |
|                                   | arrangements.                     |
|                                   |                                   |
|                                   | Record the details on a DOC and   |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   the Section of the Act that   |
|                                   |     covers the decision           |
|                                   | -   the balance of the            |
|                                   |     outstanding debt              |
|                                   | -   the agreed discounted amount  |
|                                   | -   the amount to be waived       |
|                                   | -   the date and method of        |
|                                   |     payment                       |
|                                   | -   the customer's employment     |
|                                   |     details (name and location).  |
|                                   |     Record if the customer is     |
|                                   |     unwilling to provide          |
|                                   |     employment details            |
|                                   | -   details of any warnings given |
|                                   |                                   |
|                                   | The Debt Management and           |
|                                   | Information System (DMIS) will    |
|                                   | waive the discount amount after   |
|                                   | the due date when the customer    |
|                                   | pays the agreed amount.           |
|                                   |                                   |
|                                   | If prosecution action is pending  |
|                                   | for the customer, tell the        |
|                                   | Intelligence and Investigations   |
|                                   | Branch about the discounted       |
|                                   | arrangement.                      |
|                                   |                                   |
|                                   | If discount arrangement payments  |
|                                   | are attributed to debts raised    |
|                                   | under the PGPA/FMA Act or an OGD  |
|                                   | debt adjust the payments. See     |
|                                   | Transfers and refunds.            |
