|                                   | ...                               |
|                                   |                                   |
|                                   | WLM allocates work items to staff |
|                                   | using business rules. Work items  |
|                                   | contain information about         |
|                                   | something that needs action on a  |
|                                   | customer’s record.                |
|                                   |                                   |
|                                   | To receive a new work item:       |
|                                   |                                   |
|                                   | -   Select Inbox from the left    |

|                                   | item or a 'bundle' of work items. |
|                                   |                                   |
|                                   | See Workload Management.          |

|                                   | work + Read more ...              |

|                                   | reallocation from the Customer    |
|                                   | First Inbox:                      |
|                                   |                                   |
|                                   | Select the table row containing   |
|                                   | the work item. The row will       |
|                                   | highlight yellow                  |
|                                   |                                   |
|                                   | Select Unassign under Result List |
|                                   | heading                           |
|                                   |                                   |

|                                   | Capability Adjusted or            |
|                                   | Proficiency Raised. If staff      |
|                                   | members believe these work items  |
