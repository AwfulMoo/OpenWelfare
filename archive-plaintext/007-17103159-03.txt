Viewing ACCS (Child Wellbeing) information on a customer's record

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Viewing ACCS (Child Wellbeing)    |
|                                   | information on customer CRNs +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers are notified when a     |
|                                   | certificate or determination has  |
|                                   | been made in relation to ACCS     |
|                                   | (Child Wellbeing), some           |
|                                   | information can be viewed on the  |
|                                   | customer's record.                |
|                                   |                                   |
|                                   | To view ACCS (Child Wellbeing)    |
|                                   | information on the customer's     |
|                                   | record:                           |
|                                   |                                   |
|                                   | -   Select the Child Care icon    |
|                                   | -   Open the ACCS Child Wellbeing |
|                                   |     twisty to display certificate |
|                                   |     or determination information  |
|                                   |     for relevant children         |
|                                   |                                   |
|                                   | Note: the Show History can be     |
|                                   | selected to view a history of     |
|                                   | previous certificates and         |
|                                   | determinations.                   |
|                                   |                                   |
|                                   | Child Wellbeing Certificates only |
|                                   | apply to the child care service   |
|                                   | who issued the certificate. Child |
|                                   | Wellbeing determinations apply to |
|                                   | all child care centres the child  |
|                                   | attends for the relevant period.  |
|                                   | In some instances the name of the |
|                                   | child care centre may be replaced |
|                                   | with DO NOT DISCLOSE DETAILS TO   |
|                                   | CUSTOMER. This will display in    |
|                                   | the event that the determination  |
|                                   | has been approved for a child     |
|                                   | care enrolment ID that is linked  |
|                                   | to another customer. In these     |
|                                   | circumstances it is important not |
|                                   | to disclose the details of the    |
|                                   | child care service to the         |
|                                   | customer.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | CCS eligible individual + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If an individual enquiries about  |
|                                   | ACCS (Child Wellbeing) for a      |
|                                   | child that is in their care,      |
|                                   | staff are required to check they  |
|                                   | have claimed and are eligible for |
|                                   | CCS for the child/ren:            |
|                                   |                                   |
|                                   | -   select the Child Care icon    |
|                                   | -   open the Entitlement twisty   |
|                                   | -   Entitlement will display the  |
|                                   |     Status for relevant children  |
|                                   | -   If a child in their care is   |
|                                   |     not displaying or has a       |
|                                   |     Status of Not Eligible, tell  |
|                                   |     the individual to claim CCS   |
|                                   |     for the relevant children.    |
|                                   |     See Claiming Child Care       |
|                                   |     Subsidy and Additional Child  |
|                                   |     Care Subsidy (ACCS)           |
|                                   | -   If appropriate, advise the    |
|                                   |     individual to discuss         |
|                                   |     claiming ACCS (Child          |
|                                   |     Wellbeing) with their         |
|                                   |     Approved Service Provider     |
|                                   |                                   |
|                                   | Note: if the Status for the       |
|                                   | child/ren is Not Eligible, staff  |
|                                   | should perform normal             |
|                                   | troubleshooting to assist the     |
|                                   | customer resolve their CCS        |
|                                   | eligibility for the child/ren.    |
|                                   | See Eligibility for Child Care    |
|                                   | Subsidy (CCS).                    |
+-----------------------------------+-----------------------------------+

Viewing ACCS (Child Wellbeing) certificates on a child care service's record

Table 2: This table provides information for ACCS Processing Team or
Specialist staff to view certificates on a child care service's record.

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Accessing the Child Care Service  |
|                                   | Organisation record + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the Process Direct landing   |
|                                   | page select Organisation Summary. |
|                                   |                                   |
|                                   | When prompted at the Search       |
|                                   | Criteria screen:                  |
|                                   |                                   |
|                                   | -   Search for: Organisation from |
|                                   |     the drop down menu            |
|                                   | -   ID Type: Select Organisation  |
|                                   |     CRN from the drop down menu   |
|                                   |     and enter the organisation's  |
|                                   |     CRN                           |
|                                   | -   Select Go                     |
|                                   |                                   |
|                                   | Select the relevant               |
|                                   | Organisation's record. (BUS or    |
|                                   | LGL type) from the Search         |
|                                   | Results.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Viewing ACCS (Child Wellbeing)    |
|                                   | information + Read more ...       |
|                                   |                                   |
|                                   | Key 'CWB' in the Super Key field  |
|                                   | to go to the Child Wellbeing      |
|                                   | table                             |
|                                   |                                   |
|                                   | Expand the ACCS Child Wellbeing   |
|                                   | twisty to view a list of all      |
|                                   | children who have:                |
|                                   |                                   |
|                                   | -   been issued a certificate or  |
|                                   | -   applied for a determination   |
|                                   |     in the last 12 months         |
|                                   |                                   |
|                                   | This section shows a list of      |
|                                   | children's Customer Reference     |
|                                   | Numbers (CRNs) who attend the     |
|                                   | particular service and have been  |
|                                   | assessed for a certificate or     |
|                                   | determination. Information        |
|                                   | viewable incudes:                 |
|                                   |                                   |
|                                   | -   Document ID, which is the     |
|                                   |     number assigned to the        |
|                                   |     certificate or determination  |
|                                   |     and any supporting documents. |
|                                   |     Go to Step 5 for more details |
|                                   |     on viewing supporting         |
|                                   |     documents                     |
|                                   | -   Child CRN: The child to which |
|                                   |     the certificate or            |
|                                   |     determination applies         |
|                                   | -   Approving Agency: Services    |
|                                   |     Australia or DESE (Department |
|                                   |     of Education, Skills and      |
|                                   |     Employment)                   |
|                                   | -   Request Action: for example,  |
|                                   |     new, cancel                   |
|                                   | -   Request Type:                 |
|                                   |     certificate/determination     |
|                                   | -   Start and end dates for       |
|                                   |     certificates and              |
|                                   |     determinations                |
|                                   | -   Risk Weeks: the number of     |
|                                   |     weeks a child has been        |
|                                   |     granted ACCS (Child           |
|                                   |     Wellbeing)                    |
|                                   | -   Record Status: will display   |
|                                   |     as confirmed, provisional or  |
|                                   |     QMA                           |
|                                   | -   Status: will display as       |
|                                   |     approved, rejected, revoked   |
|                                   |     or cancelled                  |
|                                   |                                   |
|                                   | Select the More Info icon to view |
|                                   | more information associated with  |
|                                   | a specific certificate or         |
|                                   | determination.                    |
|                                   |                                   |
|                                   | -   Risk reasons - (this          |
|                                   |     information is not mandatory) |
|                                   | -   Documents - evidence          |
|                                   |     submitted by the provider to  |
|                                   |     support a certificate or      |
|                                   |     determination                 |
|                                   |                                   |
|                                   | Select the CWB History icon, to   |
|                                   | view further information about    |
|                                   | the claim                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Exceptional Circumstance + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This section shows any increase   |
|                                   | in subsidised hours or increased  |
|                                   | ACCS percentage for ACCS (Child   |
|                                   | Wellbeing) - provider eligible    |
|                                   | cases.                            |
|                                   |                                   |
|                                   | Coding is only completed on the   |
|                                   | organisation record for ACCS      |
|                                   | (Child Wellbeing) provider        |
|                                   | eligible exceptional circumstance |
|                                   | requests for increased hours or   |
|                                   | ACCS percentage increases.        |
|                                   | Exceptional circumstances can     |
|                                   | only be coded after a decision    |
|                                   | has been made by the Department   |
|                                   | of Education, Skills and          |
|                                   | Employment.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Child Wellbeing cap percentage +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | This section shows the ACCS       |
|                                   | (Child Wellbeing) cap percentage  |
|                                   | the Department of Education,      |
|                                   | Skills and Employment have placed |
|                                   | on the service.                   |
|                                   |                                   |
|                                   | This is not to be updated by      |
|                                   | Services Australia staff. Any     |
|                                   | changes to the cap percentage are |
|                                   | completed by the Department of    |
|                                   | Education, Skills and Employment. |
+-----------------------------------+-----------------------------------+
| 5                                 | Viewing supporting documents +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Evidence provided to Services     |
|                                   | Australia is linked to the        |
|                                   | Document ID of the certificate or |
|                                   | determination when it is uploaded |
|                                   | by the child care                 |
|                                   | service/provider. Evidence can be |
|                                   | stored against certificates or    |
|                                   | determinations. This can be       |
|                                   | viewed within the                 |
|                                   | determination/certificate from    |
|                                   | the 'CWB' Super Key.              |
|                                   |                                   |
|                                   | Alternatively, select the         |
|                                   | Documents icon to display         |
|                                   | uploaded documents.               |
|                                   |                                   |
|                                   | -   Expand the Supporting         |
|                                   |     Documents twisty to be        |
|                                   |     presented with a list of      |
|                                   |     documents available to view   |
|                                   |     on the service/provider       |
|                                   |     record                        |
|                                   | -   Select the Document ID field  |
|                                   |     in the heading of the         |
|                                   |     Supporting Documents table    |
|                                   | -   Paste or type the number to   |
|                                   |     view associated supporting    |
|                                   |     document(s)                   |
|                                   | -   Select View to display the    |
|                                   |     document                      |

Change or cancel a Certificate

For ACCS Processing Team (ACCPT) staff only.

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Updating a Certificate + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | It may be necessary to update a   |
|                                   | certificate due to a change in    |
|                                   | circumstances. For example:       |
|                                   |                                   |
|                                   | -   the service/provider advises  |
|                                   |     the child is no longer at     |
|                                   |     risk                          |
|                                   | -   the service/provider request  |
|                                   |     to cancel the certificate     |
|                                   | -   Services Australia is         |
|                                   |     provided with other evidence  |
|                                   |     to indicate the child is no   |
|                                   |     longer at risk                |
|                                   |                                   |
|                                   | Do not delete a Child Wellbeing   |
|                                   | certificate.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Child no longer at risk, service  |
|                                   | requests cancellation of          |
|                                   | certificate + Read more ...       |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Where the provider has lodged     |
|                                   | Child No Longer at Risk           |
|                                   | notification, or Services         |
|                                   | Australia otherwise becomes aware |
|                                   | that the child is no longer at    |
|                                   | risk:                             |
|                                   |                                   |
|                                   | -   select the Pencil icon on the |
|                                   |     Determination activity. This  |
|                                   |     will present the Change       |
|                                   |     Certificate/Determination pop |
|                                   |     up box                        |
|                                   | -   edit and end date any active  |
|                                   |     Certificates or               |
|                                   |     Determinations from the date  |
|                                   |     the provider has advised the  |
|                                   |     child is no longer at risk    |
|                                   |     Do not delete a Child         |
|                                   |     Wellbeing certificate or      |
|                                   |     determination                 |
|                                   | -   select APPROVED from the      |
|                                   |     Status drop down              |
|                                   | -   select Save to finalise the   |
|                                   |     reassessment                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Create case notes + Read more ... |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Details of any change made to a   |
|                                   | certificate must be recorded in   |
|                                   | Case Notes.                       |
|                                   |                                   |
|                                   | Select the Notes icon to record   |
|                                   | details of the request for        |
|                                   | information                       |
|                                   |                                   |
|                                   | Select Add Case Note. Note: where |
|                                   | previous contact was              |
|                                   | unsuccessful, annotate the        |
|                                   | previous Case Note                |
|                                   |                                   |
|                                   | In the Text field enter the       |
|                                   | following information:            |
|                                   |                                   |
|                                   | -   Document ID:                  |
|                                   | -   Child CRN and Name:           |
|                                   | -   Change of circumstances:      |
|                                   |     [insert details]              |
|                                   | -   Risk Weeks: [Changed from xxx |
|                                   |     to xxxx / No change]          |
|                                   | -   Level 3 Policy approval       |
|                                   |     granted: YES/NO               |
|                                   | -   Evidence to support updates:  |
|                                   |     YES/NO. If NO, include        |
|                                   |     details of why not required   |
|                                   | -   Other information relevant to |
|                                   |     reassessment: Assessment made |
|                                   |     in accordance with FA Act     |
|                                   |     [insert relevant reference    |
|                                   |     from the list below]          |
|                                   |     - S85CF (Variation and        |
|                                   |     revocation of determinations) |
|                                   |     - section 85CA(4) (Minister's |
|                                   |     rules relating to when a      |
|                                   |     child is or is not taken to   |
|                                   |     be at risk of serious abuse   |
|                                   |     or neglect) (to the extent    |
|                                   |     that these rules affect what  |
|                                   |     needs to be captured for a    |
|                                   |     certificate or application)   |
|                                   |     - section 204K (notices to    |
|                                   |     state/territory body of child |
|                                   |     at risk of serious abuse or   |
|                                   |     neglect) (including what      |
|                                   |     needs to be captured and what |
|                                   |     is required for an            |
|                                   |     application to be made)       |
|                                   |     - section 67FC (notice if     |
|                                   |     child not at risk of serious  |
|                                   |     abuse or neglect)             |
|                                   |                                   |
|                                   | Select Save to finalise the Case  |
|                                   | Note                              |
