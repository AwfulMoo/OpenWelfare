Your right to privacy

Information that can be provided to a PPE

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Verify the PPE arrangement before |
|                                   | disclosing the SRSS customer’s    |
|                                   | information to a third party      |
|                                   |                                   |
|                                   | Before responding to an enquiry   |
|                                   | from a PPE about an SRSS          |
|                                   | customer’s record, the Service    |
|                                   | Officers must take care to:       |
|                                   |                                   |
|                                   | Check the PPE currently recorded  |
|                                   | on the SRSS customer’s record     |
|                                   |                                   |
|                                   | Ask sufficient questions and      |
|                                   | collect sufficient evidence from  |
|                                   | the person making the enquiry, to |
|                                   | be reasonably satisfied they are  |
|                                   | permitted to enquire. This may    |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   asking specific information   |
|                                   |     about the SRSS customer, the  |
|                                   |     organisation and the          |
|                                   |     organisation’s representative |
|                                   |     to verify the                 |
|                                   |     representative’s identity,    |
|                                   |     and                           |
|                                   | -   seeking the representative’s  |
|                                   |     confirmation that they have   |
|                                   |     authority to obtain           |
|                                   |     information about the SRSS    |
|                                   |     customer on behalf of the     |
|                                   |     organisation                  |
|                                   |                                   |
|                                   | If the Service Officer is         |
|                                   | concerned about the identity of   |
|                                   | the person making the enquiry,    |
|                                   | they should not provide any       |
|                                   | personal information about the    |
|                                   | SRSS customer, and terminate the  |
|                                   | enquiry.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Information a Service Officer can |
|                                   | confirm to a PPE                  |
|                                   |                                   |
|                                   | The extent of information a       |
|                                   | Service Officer can confirm to a  |
|                                   | PPE, depends on the enquiry, and  |
|                                   | includes, but is not limited to:  |
|                                   |                                   |
|                                   | -   rate of SRSS payment          |
|                                   | -   cancellation or rejection     |
|                                   |     reasons applied to the SRSS   |
|                                   |     payment                       |
|                                   | -   overpayment and arrears       |
|                                   |     information                   |
|                                   | -   the basis for assessment of   |
|                                   |     the customer's payments       |
|                                   | -   factors affecting the rate of |
|                                   |     payment, such as income       |
|                                   |     and/or assets as advised by   |
|                                   |     the customer                  |
|                                   |                                   |
|                                   | A Service Officer can confirm     |
|                                   | current information held, but     |
|                                   | must not disclose it freely. It   |
|                                   | is essential that when disclosing |
|                                   | information, provide only the     |
|                                   | minimum information necessary to  |
|                                   | fulfil the needs of the           |
|                                   | situation.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Examples                          |
|                                   |                                   |
|                                   | -   A Service Officer would not   |
|                                   |     discuss the customer's rate   |
|                                   |     of payment unless that was    |
|                                   |     the reason for the contact    |
|                                   | -   A Service Officer would not   |
|                                   |     discuss the customer's debt   |
|                                   |     unless that was the reason    |
|                                   |     for the PPE contacting        |
+-----------------------------------+-----------------------------------+

Privacy Statement

Privacy and your personal information

The privacy and security of your personal information is important to
us, and is protected by law. We need to collect this information so we
can process and manage your applications and payments, and provide
services to you. We only share your information with other parties where
you have agreed, or where the law allows or requires it.

For more information, visit the Services Australia website.
