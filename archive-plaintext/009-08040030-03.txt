This page contains information to help Service Officers action
hospitalisation reviews, Manual Follow-ups (MFU), reissue forms and
register returned reviews.

|                                   | Carer Payment and Carer Allowance |

|                                   | carer.                            |

|                                   | occur even if the 12 continuous   |
|                                   | weeks spans a calendar year.      |
|                                   |                                   |
|                                   | Note: the 12 week review will be  |
|                                   | carried out a total of 7 times if |
|                                   | the customer remains in hospital  |
|                                   | for over 21 months.               |
