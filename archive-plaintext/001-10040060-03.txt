Proper notification and reasonable notice

Table 1: This table describes how to assess if a job seeker or
ParentsNext participant has received appropriate and reasonable notice.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Proper notification of the        |
|                                   | requirement to attend an          |
|                                   | appointment + Read more ...       |
|                                   |                                   |
|                                   | Services Australia and Employment |
|                                   | Services Providers, including     |
|                                   | ParentsNext providers, are        |
|                                   | delegated to issue formal         |
|                                   | notification in relation to       |
|                                   | compulsory requirements.          |
|                                   |                                   |
|                                   | A notification must include the:  |
|                                   |                                   |
|                                   | -   date, time and location of    |
|                                   |     the appointment, and          |
|                                   | -   consequences for failing to   |
|                                   |     attend or failing to advise   |
|                                   |     of an inability to attend     |
|                                   |                                   |
|                                   | Note: if possible, appointment    |
|                                   | notifications should also include |
|                                   | the contact person, purpose       |
|                                   | and/or reason for the             |
|                                   | appointment, however not doing so |
|                                   | does not invalidate the           |
|                                   | requirement. This is because all  |
|                                   | Job Plans and Participation Plans |
|                                   | include a requirement to contact  |
|                                   | the provider with an acceptable   |
|                                   | reason if unable to meet a        |
|                                   | requirement.                      |
|                                   |                                   |
|                                   | The requirement to enter into a   |
|                                   | Job Plan or Participation Plan    |
|                                   | can be included within the        |
|                                   | original notice or delivered      |
|                                   | verbally at or before the         |
|                                   | appointment.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Proper notification of the        |
|                                   | requirement to attend, undertake  |
|                                   | or provide evidence of a          |
|                                   | compulsory activity + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | For job seekers in the Community  |
|                                   | Development Program (CDP),        |
|                                   | Disability Employment Services    |
|                                   | (DES) or ParentsNext              |
|                                   | participants, the requirement     |
|                                   | must be recorded in the Job Plan  |
|                                   | or Participation Plan and must    |
|                                   | include the following:            |
|                                   |                                   |
|                                   | -   a description of the          |
|                                   |     requirement                   |
|                                   | -   start and end date the job    |
|                                   |     seeker is required to         |
|                                   |     participate, and              |
|                                   | -   the number of hours per       |
|                                   |     fortnight of required         |
|                                   |     participation                 |
|                                   |                                   |
|                                   | CDP and DES job seekers receive   |
|                                   | formal notification of their Job  |
|                                   | Search Requirement through their  |
|                                   | Job Plan. This includes the       |
|                                   | number of Job Search efforts      |
|                                   | required, the date the efforts    |
|                                   | are due to be reported, and how   |
|                                   | they can meet this requirement    |
|                                   | and report their Job Search       |
|                                   | efforts.                          |
|                                   |                                   |
|                                   | For job seekers in Workforce      |
|                                   | Australia and Workforce Australia |
|                                   | Online, the Job Plan is           |
|                                   | simplified and must only include  |
|                                   | mandatory requirements like:      |
|                                   |                                   |
|                                   | -   achieving a monthly points    |
|                                   |     target which includes a       |
|                                   |     minimum job search            |
|                                   |     requirement                   |
|                                   | -   attending compulsory          |
|                                   |     appointments, and             |
|                                   | -   participation in Mandatory    |
|                                   |     Activation Point tasks as     |
|                                   |     required                      |
|                                   |                                   |
|                                   | Workforce Australia job seekers   |
|                                   | receive formal notification of    |
|                                   | their Points Requirement          |
|                                   | (including minimum job search)    |
|                                   | via their Workforce Australia     |
|                                   | homepage.                         |
|                                   |                                   |
|                                   | All job seekers are required to   |
|                                   | make prior contact with the       |
|                                   | provider and provide an           |
|                                   | acceptable reason if unable to    |
|                                   | comply with a requirement.        |
|                                   |                                   |
|                                   | Under the Targeted Compliance     |
|                                   | Framework, job seekers are        |
|                                   | responsible for recording their   |
|                                   | own attendance at activities      |
|                                   | using the Workforce Australia     |
|                                   | App, Homepage, or by contacting   |
|                                   | their provider by close of        |
|                                   | business on the day of the        |
|                                   | requirement.                      |
|                                   |                                   |
|                                   | Providers are required to issue   |
|                                   | formal notification any time they |
|                                   | set or change a compulsory        |
|                                   | requirement. This includes when   |
|                                   | they:                             |
|                                   |                                   |
|                                   | -   set the requirement for the   |
|                                   |     first time in a job seeker's  |
|                                   |     Calendar/Activity Diary       |
|                                   | -   reschedule a requirement      |
|                                   |     because the job seeker has an |
|                                   |     acceptable reason for not     |
|                                   |     being able to meet a          |
|                                   |     requirement                   |
|                                   | -   adjust requirements to        |
|                                   |     reflect they no longer need   |

|                                   | -   the organisation the          |
|                                   |     information should be         |
|                                   |     provided to                   |

| 3                                 | Methods of notification + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Formal (or primary) notification  |
|                                   |                                   |
|                                   | Formal notification is a          |
|                                   | notification that meets the       |
|                                   | requirements under Social         |
|                                   | Security Law. It must include the |
|                                   | following details:                |
|                                   |                                   |
|                                   | -   What the requirement is, for  |
|                                   |     example, an appointment or an |
|                                   |     activity                      |
|                                   | -   The purpose or reason for the |
|                                   |     requirement (if relevant)     |
|                                   | -   The date and time of the      |
|                                   |     requirement, or the date      |
|                                   |     and/or time the requirement   |
|                                   |     needs to be completed by      |
|                                   | -   The location or address of    |
|                                   |     the requirement (if relevant) |
|                                   | -   That the job seeker must      |
|                                   |     contact their Provider before |
|                                   |     the requirement if they have  |
|                                   |     an acceptable reason for not  |
|                                   |     being able to meet it         |
|                                   | -   The consequences of not       |
|                                   |     meeting the requirement, and  |
|                                   | -   A statement that the          |
|                                   |     Notification is being issued  |
|                                   |     under Social Security Law,    |
|                                   |     explaining the job seeker     |
|                                   |     must meet their requirements  |
|                                   |     in order to receive income    |
|                                   |     support payments              |
|                                   |                                   |
|                                   | Formal (primary) notification     |
|                                   | methods used by providers can     |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   Letter-mailed - a letter is   |
|                                   |     mailed to the job seeker's    |
|                                   |     residential address on the    |
|                                   |     day the requirement is        |
|                                   |     scheduled                     |
|                                   | -   Letter-handed - a letter is   |
|                                   |     handed to the job seeker in   |
|                                   |     person as they are currently  |
|                                   |     with the Provider             |
|                                   | -   Phone (script) - a script is  |
|                                   |     read to the job seeker by     |
|                                   |     their provider as the         |
|                                   |     requirement is being          |
|                                   |     scheduled. Note: leaving a    |
|                                   |     message on an answering       |
|                                   |     machine or with another       |
|                                   |     person in the job seeker's    |
|                                   |     household does not count as   |
|                                   |     formal notification           |
|                                   | -   Face to face (script) - a     |
|                                   |     script is read to the job     |
|                                   |     seeker as they are in         |
|                                   |     attendance at the provider's  |
|                                   |     office as the requirement is  |
|                                   |     being scheduled               |
|                                   | -   Email -sent to the job seeker |
|                                   |     with a read receipt sent back |
|                                   |     to the Provider. This is only |
|                                   |     available if it is recorded   |
|                                   |     as the job seeker's preferred |
|                                   |     method of Notification        |
|                                   |                                   |
|                                   | Secondary notification            |
|                                   | (reminders)                       |
|                                   |                                   |
|                                   | Depending on the requirement, job |
|                                   | seekers may also receive          |
|                                   | reminders.                        |
|                                   |                                   |
|                                   | Job seekers may receive reminders |
|                                   | through:                          |
|                                   |                                   |
|                                   | -   SMS                           |
|                                   | -   Email                         |
|                                   | -   A Push Notification on their  |
|                                   |     Job Seeker App                |
|                                   | -   An Inbox message through      |
|                                   |     their homepage on the         |
|                                   |     Workforce Australia website   |

| 4                                 | Keeping notification records +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Both Services Australia and       |
|                                   | providers are required to keep    |
|                                   | accurate records of all           |
|                                   | notification and contact          |
|                                   | attempts, whether successful or   |
|                                   | not.                              |
|                                   |                                   |
|                                   | These records are used when       |
|                                   | determining if a financial        |
|                                   | penalty is appropriate when       |
|                                   | non-compliance occurs.            |
|                                   |                                   |
|                                   | Records should include:           |
|                                   |                                   |
|                                   | -   the date and method of any    |
|                                   |     notification or reminder      |
|                                   | -   the nature or purpose of the  |
|                                   |     requirement                   |
|                                   | -   that a compliance warning was |
|                                   |     issued                        |
|                                   | -   any explanations given by a   |
|                                   |     job seeker for requesting to  |
|                                   |     reschedule a notified         |
|                                   |     appointment                   |
|                                   |                                   |
|                                   | When notification of a            |
|                                   | requirement was provided in       |
|                                   | writing, a copy should be kept as |
|                                   | evidence of notification.         |
+-----------------------------------+-----------------------------------+
| 5                                 | Services Australia notification   |
|                                   | records + Read more ...           |
|                                   |                                   |
|                                   | Evidence that proper notification |
|                                   | has occurred is shown by one or   |
|                                   | more of the following:            |
|                                   |                                   |
|                                   | -   written notification of the   |
|                                   |     appointment or the            |
|                                   |     requirement on the History    |
|                                   |     Summary (HS) screen           |
|                                   | -   the job seeker's Job Plan or  |
|                                   |     Participation Plan            |
|                                   | -   details recorded in a DOC     |
|                                   |     when the job seeker was       |
|                                   |     verbally notified of the      |
|                                   |     appointment and no workflow   |
|                                   |     was used to facilitate the    |
|                                   |     appointment                   |
|                                   | -   information recorded in the   |
|                                   |     Online Diary                  |
|                                   | -   any other appointment,        |
|                                   |     notification and reminder     |
|                                   |     details recorded              |

| 6                                 | Guidelines for reasonable notice  |
|                                   | timeframes + Read more ...        |
|                                   |                                   |
|                                   | The job seeker must be given      |
|                                   | enough time between formal        |
|                                   | notification being issued and a   |
|                                   | requirement occurring to receive  |
|                                   | the notification and prepare for  |
|                                   | the requirement.                  |
|                                   |                                   |
|                                   | Reasonable notice will vary       |
|                                   | according to the circumstances of |
|                                   | the notification and whether or   |
|                                   | not the job seeker was directly   |
|                                   | consulted prior to notice being   |
|                                   | given.                            |
|                                   |                                   |
|                                   | Direct consultation:              |
|                                   |                                   |
|                                   | When an appointment has been made |
|                                   | in direct consultation with a job |
|                                   | seeker and the job seeker has     |
|                                   | agreed to attend, reasonable      |
|                                   | notice is considered to have been |
|                                   | given, even if the appointment is |
|                                   | on the same day as the            |
|                                   | notification.                     |
|                                   |                                   |
|                                   | Reasonable notice time frames for |
|                                   | other formal notification         |
|                                   | methods:                          |
|                                   |                                   |
|                                   | The Employment Services System    |
|                                   | (ESS) ensures that reasonable     |
|                                   | notice timeframes are met when    |
|                                   | the provider is setting           |
|                                   | requirements and creating formal  |
|                                   | notification for job seekers,     |
|                                   | preventing a requirement on a     |
|                                   | specific day or use of a          |
|                                   | particular type of notification   |
|                                   | if these will not meet the        |
|                                   | following policy guidelines:      |
|                                   |                                   |
|                                   | -   Letter mailed to a job        |
|                                   |     seeker - 6 business days,     |
|                                   |     Community Development Program |
|                                   |     (CDP) providers are required  |
|                                   |     to post their letters 8       |
|                                   |     business days ahead of the    |
|                                   |     scheduled appointment         |
|                                   | -   Letter handed to a job        |
|                                   |     seeker - 3 calendar days      |
|                                   | -   Verbal script read face to    |
|                                   |     face or over the phone - 3    |
|                                   |     calendar days                 |
|                                   | -   Email (can only be used if    |
|                                   |     this is the job seeker's      |
|                                   |     preferred notification        |
|                                   |     method) - 2 business days     |
|                                   |     with a read receipt sent back |
|                                   |     to confirm the job seeker has |
|                                   |     received, read and            |
|                                   |     acknowledged the notification |
|                                   |                                   |
|                                   | When direct consultation has not  |
|                                   | occurred and the job seeker       |
|                                   | disputes notification of the      |
|                                   | requirement, Service Officers     |
|                                   | should consider the following:    |
|                                   |                                   |
|                                   | -   the address where the letter  |
|                                   |     was sent                      |
|                                   | -   if the notification was made  |
|                                   |     by email, whether the         |
|                                   |     requirement for a read        |
|                                   |     receipt was met               |
|                                   | -   if there was sufficient time  |
|                                   |     for the job seeker to receive |
|                                   |     the notification, and         |
|                                   | -   if there was sufficient time  |
|                                   |     for the job seeker to make    |
|                                   |     the necessary arrangements    |
|                                   |     required to attend. Examples  |
|                                   |     include arranging transport   |
|                                   |     to the appointment or making  |
|                                   |     child care arrangements       |
|                                   |                                   |
|                                   | Note: in rural and remote areas,  |
|                                   | local mail delivery conditions    |
|                                   | and the job seeker's arrangements |
|                                   | for collecting mail should also   |
|                                   | be taken into consideration.      |

Provider notification requirements

| 1                                 | Provider notification records +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Employment Services System    |
|                                   | (ESS) retains a history of any    |
|                                   | formal notification issued on the |
|                                   | Notification List.                |
|                                   |                                   |
|                                   | The notification record must be   |
|                                   | provided if a non-compliance      |
|                                   | event is reported to the agency.  |
|                                   | The Services Australia IT system  |
|                                   | automatically picks up the        |
|                                   | notifications for any             |

|                                   | seeker's Calendar.                |
|                                   |                                   |
|                                   | Linked notification record/s can  |
|                                   | be viewed in the Participation    |
|                                   | Compliance workflow when          |
|                                   | conducting a compliance           |
|                                   | investigation. Linked             |
|                                   | notifications are available       |
|                                   | within the relevant provider      |
|                                   | report on the Participation       |
|                                   | Compliance Workflow Summary       |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Job Seeker Compliance Framework:  |
|                                   |                                   |
|                                   | -   The notification linked to    |
|                                   |     the Non-Attendance Report     |
|                                   |     (NAR) can be found            |
|                                   |     immediately above the         |
|                                   |     'Evidence' heading within the |
|                                   |     report                        |
|                                   | -   Notifications linked to       |
|                                   |     Participation Reports and     |
|                                   |     Provider Appointment Reports  |
|                                   |     (PAR) appear under the        |
|                                   |     'Notification' heading        |
|                                   |                                   |
|                                   | Targeted Compliance Framework:    |
|                                   |                                   |
|                                   | -   The type of notification used |
|                                   |     by the provider is recorded   |
|                                   |     on the other related          |
|                                   |     information page under the    |
|                                   |     recent non-compliance link    |
|                                   |     available within the          |
|                                   |     Compliance Investigation      |
|                                   |     workflow, or within the NCE   |
|                                   |     on the Provider Report        |
|                                   |     Summary screen                |
|                                   |                                   |
|                                   | Details in the linked             |
|                                   | notification records should be    |
|                                   | consistent with other details     |
|                                   | recorded by the provider.         |
|                                   |                                   |
|                                   | Details of the notification given |
|                                   | may not be displayed in these     |
|                                   | links, for example when:          |
|                                   |                                   |
|                                   | -   a verbal notification has     |
|                                   |     been issued                   |
|                                   | -   the notification was          |

|                                   |                                   |

|                                   | reporting the non-compliance      |
|                                   | event will include evidence to    |
|                                   | satisfy appropriate notification. |

|                                   | notification and/or reasonable    |

|                                   |                                   |

|                                   | the Activity Diary/Calendar, the  |

|                                   | also be able to see the notice    |

|                                   | on the Workforce Australia        |
|                                   | website or in the Workforce       |
|                                   | Australia App.                    |

|                                   | -   the date a locally printed    |
|                                   |     letter was handed or posted   |
|                                   |     to the job seeker, or         |
|                                   | -   any follow up contacts to     |
|                                   |     confirm appointment           |
|                                   |                                   |
|                                   | If a job seeker challenges some   |
|                                   | aspect of the notification, and   |
|                                   | that aspect is not clearly        |
|                                   | documented, confirmation may need |
|                                   | to be sought from the provider.   |

|                                   | endorsed scripts to assist        |
|                                   | providers to meet the             |
|                                   | notification requirements under   |

|                                   |                                   |
|                                   | A hard copy of the letter should  |

|                                   |                                   |
|                                   | The Department of Employment and  |
