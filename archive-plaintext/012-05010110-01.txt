This document outlines the Major and Minor customer identifiers (ID) for
Medicare Online, Medicare Easyclaim and manual claiming.

Criteria to update a Minor ID via an inbound call

eBusiness staff can update a Minor ID listed against a provider number
if the below criteria is met:

-   The caller can pass the eBusiness security check, see Verify caller
    identity for eBusiness Service Centre enquiries for information
-   The Minor ID has been previously registered as an organisation in
    the Provider Directory System (PDS), see Check a Minor ID has been
    registered in the PDS table on the Process page
-   The caller can confirm the correct banking details registered
    against the provider number in PDS

Note: no more than 5 provider number locations can be updated per
inbound phone call.
