|                                   | application is rejected ‘too      |

|                                   | -   Use the patient history to    |

|                                   |     supply will last to’ and tell |
|                                   |     the prescriber of the         |
|                                   |     rejection                     |
|                                   |                                   |
|                                   | The assessment results of the     |
|                                   | patient’s previous authorities    |
|                                   | are abbreviated in the Patient    |
|                                   | History pop up. Assessment        |
|                                   | Results in bold should be in the  |
|                                   | manual history check:             |
|                                   |                                   |
|                                   | -   1 = Approved                  |
|                                   | -   2 = Approved with changes     |
|                                   | -   3 = Rejected                  |
|                                   | -   4 = Pended/Saved              |
|                                   | -   5 = Previously rejected/now   |
|                                   |     approved                      |
|                                   | -   6 = Authority not required    |
|                                   | -   D = Duplicate                 |
|                                   | -   C = Cancelled                 |
|                                   | -   E = Re-Issued                 |
|                                   |                                   |
|                                   | Note: when assessing ‘Too Soon’,  |
|                                   | OPA:                              |
|                                   |                                   |
|                                   | -   performs a consistent         |
|                                   |     approach to all ‘Too Soon’    |
|                                   |     checking                      |
|                                   | -   starts the assessment from    |
|                                   |     the most recent approval and  |
|                                   |     works back                    |
|                                   | -   stops assessing once the      |
|                                   |     first instance of ‘Too Soon’  |
|                                   |     is found                      |
|                                   | -   uses up to 12 months of       |
|                                   |     history to calculate ‘Too     |
|                                   |     Soon’                         |
|                                   | -   considers if any previous     |
|                                   |     ‘Too Soon’ rejections have    |
|                                   |     been overridden               |
|                                   | -   considers whether the         |
|                                   |     original has been supplied    |
|                                   |     from previous approvals       |
|                                   | -   does not take into            |
|                                   |     consideration any prior       |
|                                   |     approvals that may compound   |
|                                   |     the calculation (if the       |
|                                   |     doctor is disputing a         |
|                                   |     rejection, a manual           |
|                                   |     calculation could extend the  |
|                                   |     next approval date further    |
|                                   |     into the future)              |
|                                   |                                   |
|                                   | OPA does not include              |
|                                   | prescriptions for non-authority   |
|                                   | or Streamlined quantities and/or  |
|                                   | repeats that may appear on the    |
|                                   | patient’s dispensing history.     |
|                                   | These supplies may be included    |
|                                   | when doing a manual calculation   |
|                                   |                                   |
|                                   | For Service Officer information   |
|                                   | only (not for release): remember  |
|                                   | that the way the OPA calculates   |
|                                   | ‘too soon’ may not return the     |
|                                   | same result as a manual           |
|                                   | calculation.                      |
|                                   |                                   |
|                                   | When to perform a manual          |
|                                   | calculation to determine ‘too     |
|                                   | soon’                             |
|                                   |                                   |
|                                   | In some cases, a prescriber may   |
|                                   | dispute the too soon result       |
|                                   | returned from OPA. Perform a      |
|                                   | manual calculation using the      |
|                                   | Display Patient History function  |
|                                   | before changing the OPA           |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | -   When assessing manually,      |
|                                   |     calculate the past 6 months   |
|                                   |     of authority approval history |
|                                   | -   Once investigated, if there   |
|                                   |     is a valid reason for an      |
|                                   |     early authority request, the  |
|                                   |     rejection can be overridden   |
|                                   | -   Select Approved and an        |
|                                   |     appropriate reason from the   |
|                                   |     Override Reason drop down     |
|                                   |     box. Then OVERRIDE the        |
|                                   |     application request           |
|                                   | -   If the application is not     |
|                                   |     approvable, advise the        |
|                                   |     prescriber the history has    |
|                                   |     been double checked and the   |
|                                   |     application is not approvable |
|                                   |     unless there was an           |
|                                   |     extenuating circumstance or   |
|                                   |     explanation                   |
|                                   |                                   |
|                                   | Some authorities may return a too |
|                                   | soon date within the next 24 to   |
|                                   | 48 hours. Do not automatically    |
|                                   | override these applications       |
|                                   | because ‘it's only 1 or 2 days    |
|                                   | away’. Advise the prescriber the  |
|                                   | patient has access to 1 or 2      |
|                                   | months (depending on the drug) of |
|                                   | medication. An application can be |
|                                   | made on the displayed date        |
|                                   | through the prescribers HPOS      |
|                                   | system or via the PBS authority   |
|                                   | line.                             |
