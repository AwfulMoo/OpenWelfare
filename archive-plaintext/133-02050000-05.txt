Information and services available via HPOS

Table 1

+-----------------------------------+-----------------------------------+
| Services                          | Information available             |
+-----------------------------------+-----------------------------------+
| Australian Immunisation Register  | Record vaccination information,   |
| (AIR)                             | view individual’s vaccination     |
|                                   | history, record a vaccination     |
|                                   | exemption, access AIR Reports,    |
|                                   | view AIR payments, update         |
|                                   | provider’s contact details.       |
|                                   | Program is only available to GPs  |
|                                   | and allied health professionals   |
|                                   | who have registered for access.   |
+-----------------------------------+-----------------------------------+
| Centrelink forms                  | GPs can access, complete and      |
|                                   | transmit the Medical Report       |
|                                   | Disability Support Pension and    |
|                                   | Medical Certificate forms to      |
|                                   | Centrelink.                       |
+-----------------------------------+-----------------------------------+
| Delegate facility                 | Allows a health professional to   |
|                                   | nominate a delegate to act on     |
|                                   | their behalf (including practice  |
|                                   | staff with their own PRODA        |
|                                   | account to undertake              |
|                                   | administrative tasks.             |
+-----------------------------------+-----------------------------------+
| Department of Veterans’ Affairs   | Receive Patient Treatment Reports |
| (DVA) Patient Treatment reports   | for veterans in their care who    |
|                                   | are enrolled in the Coordinated   |
|                                   | Veterans' Care (CVC) Program.     |
+-----------------------------------+-----------------------------------+
| DVA Webclaim                      | Health professionals can use DVA  |
|                                   | Webclaim to submit Medical and    |
|                                   | allied health services for the    |
|                                   | treatment of eligible DVA         |
|                                   | customers. Private Hospitals and  |
|                                   | Community Nursing facilities      |
|                                   | cannot lodge services through DVA |
|                                   | Webclaim.                         |
+-----------------------------------+-----------------------------------+
| My Health Record system –         | Register organisations for the My |
| organisation registration         | Health Record system and maintain |
|                                   | details. Establish a list of      |
|                                   | individual healthcare providers   |
|                                   | who are authorised to access the  |
|                                   | My Health Record Provider Portal. |
|                                   | Authorise contracted service      |
|                                   | providers to interact with the My |
|                                   | Health Record system on the       |
|                                   | organisations behalf.             |
+-----------------------------------+-----------------------------------+
| Find a patient                    | Confirm and search patient        |
|                                   | Medicare card details. (includes  |
|                                   | Concessional entitlement          |
|                                   | verification)                     |
+-----------------------------------+-----------------------------------+
| Healthcare Identifiers (HI)       | The Healthcare Identifiers (HI)   |
| Service                           | Service in HPOS can be used to:   |
|                                   |                                   |
|                                   | -   maintain healthcare           |
|                                   |     identifier details            |
|                                   | -   search for healthcare         |
|                                   |     providers and organisations   |
|                                   |     in the HI Service, and        |
|                                   | -   maintain links to providers   |
|                                   |     working at the organisation,  |
|                                   |     to display in the Healthcare  |
|                                   |     Provider Directory            |
|                                   |                                   |
|                                   | What can be performed in the HI   |
|                                   | Service through HPOS is tailored  |
|                                   | for the individual roles of       |
|                                   | either a Responsible Officer,     |
|                                   | Organisation Maintenance Officer  |
|                                   | or Individual Healthcare          |
|                                   | Provider.                         |
+-----------------------------------+-----------------------------------+
| MBS items online checker          | The MBS items online checker      |
|                                   | enables the:                      |
|                                   |                                   |
|                                   | -   viewing and checking of       |
|                                   |     patient eligibility based on  |
|                                   |     their MBS history             |
|                                   | -   checking of providers         |
|                                   |     eligibility for claiming MBS  |
|                                   |     items, and                    |
|                                   | -   checking of the claiming      |
|                                   |     conditions for MBS items      |
|                                   |                                   |
|                                   | This service helps to make        |
|                                   | informed treatment decisions and  |
|                                   | avoid rejected claims.            |
+-----------------------------------+-----------------------------------+
| Medicare Bulk Bill Webclaim       | Medicare Bulk Bill Webclaim is an |
|                                   | internet based claim channel that |
|                                   | is accessed via the Health        |
|                                   | Professional Online Services      |
|                                   | (HPOS) portal on the Services     |
|                                   | Australia website.                |
+-----------------------------------+-----------------------------------+
| Medicare Patient Claim Webclaim   | Medicare Patient Claim Webclaim   |
|                                   | is an online claiming channel     |
|                                   | through HPOS where Medicare       |
|                                   | claims are submitted on behalf of |
|                                   | patients at the point of service. |
|                                   | This provides real time           |
|                                   | assessment and reduces the need   |
|                                   | for patients to obtain additional |
|                                   | information from the provider to  |
|                                   | receive their Medicare benefit.   |
|                                   | For example, as part of           |
|                                   | submitting the claim, the         |
|                                   | provider can inform Services      |
|                                   | Australia that a service was not  |
|                                   | aftercare.                        |
+-----------------------------------+-----------------------------------+
| My details                        | View provider personal details    |
|                                   | including provider number         |
|                                   | details, banking details and add  |
|                                   | a new or reopen a closed provider |
|                                   | number (location).                |
+-----------------------------------+-----------------------------------+
| Practice Incentives Program (PIP) | Practice owners and authorised    |
|                                   | contacts with a PRODA account can |
|                                   | now:                              |
|                                   |                                   |
|                                   | -   register for the PIP          |
|                                   | -   update their practice details |
|                                   |     and apply for additional      |
|                                   |     incentives under the PIP      |
|                                   | -   access information about the  |
|                                   |     PIP                           |
|                                   | -   view their PIP statements     |
|                                   | -   receive broadcasts            |
+-----------------------------------+-----------------------------------+
| Workforce Incentive Program       | Apply for the WIP – Practice      |
| (WIP) - Practice Stream           | Stream, view statements, letters  |
|                                   | and broadcasts.                   |
+-----------------------------------+-----------------------------------+
| Prescription Shopping Information | Information, patient summary      |
| Program                           | reports and FAQs.                 |
|                                   |                                   |
|                                   | After contacting the Prescription |
|                                   | Shopping Information Service, if  |
|                                   | the patient met the programme’s   |
|                                   | criteria, providers can access a  |
|                                   | report listing medicines supplied |
|                                   | to the patient in the last 3      |
|                                   | months.                           |
+-----------------------------------+-----------------------------------+
| Rural Incentives Programs         | General Practice Rural Incentives |
|                                   | Program (GPRIP) including the GP  |
|                                   | Component, Registrars Component   |
|                                   | and the Rural Relocation          |
|                                   | Incentive Grant (RRIG), the Rural |
|                                   | Procedural Grants Program (RPGP)  |
|                                   | and the HECS Reimbursement Scheme |
|                                   | information and services. Apply   |
|                                   | for the GPRIP, RRIG and HECS and  |
|                                   | view statements.                  |
+-----------------------------------+-----------------------------------+
| Workforce Incentive Program       | Provider can view status, service |
| (WIP) - Doctor Stream             | and payment history and update    |
|                                   | bank account details.             |
+-----------------------------------+-----------------------------------+
| HPOS Messages                     | Secure email facility to receive  |
|                                   | statements, letters and           |
|                                   | notifications online for:         |
|                                   |                                   |
|                                   | -   90 day pay doctor cheque      |
|                                   |     scheme EFT payments           |
|                                   | -   Australian Immunisation       |
|                                   |     Register                      |
|                                   | -   DVA Coordinated Veterans'     |
|                                   |     Care Program Patient          |
|                                   |     Treatment Report              |
|                                   | -   DVA Webclaim EFT statement,   |
|                                   |     recipient credited tax        |
|                                   |     invoice statement and zero    |
|                                   |     benefit statement             |
|                                   | -   Medicare bulk bill – Medicare |
|                                   |     Bulk Bill Payment Report      |
|                                   |     (Statement)                   |
|                                   | -   Midwives Professional         |
|                                   |     Indemnity (MPIS)              |
|                                   | -   National Bowel Cancer         |
|                                   |     Screening Program – Payment   |
|                                   |     advice and tax invoice        |
|                                   | -   Rural incentives programs     |
|                                   | -   Track and Scale - scaling     |
|                                   |     details and location s19AB    |
|                                   |     exemption details             |
+-----------------------------------+-----------------------------------+
| HPOS Messages - Form upload       | Health professionals and their    |
|                                   | delegates will be able to         |
|                                   | securely upload select forms via  |
|                                   | a new 'form upload' facility in   |
|                                   | HPOS. The service will be for the |
|                                   | submission of scanned and/or      |
|                                   | photographed Health Professional  |
|                                   | forms and accompanying documents. |
|                                   | The service will also provide a   |
|                                   | receipt and the capability for    |
|                                   | Services Australia Service        |
|                                   | Officers to securely reply to the |
|                                   | health professional via email.    |
+-----------------------------------+-----------------------------------+
| Closing the Gap PBS Co-payment    | PBS Prescribers, Aboriginal       |
| Register                          | Health Practitioners and their    |
|                                   | delegates are able to register an |
|                                   | Aboriginal or Torres Strait       |
|                                   | Islander patient for the Closing  |
|                                   | the Gap (PBS) co-payment program. |
|                                   | Pharmacies can search to see if a |
|                                   | patient is registered for the     |
|                                   | Closing the Gap (PBS) co-payment  |
|                                   | program for PBS claiming          |
|                                   | purposes.                         |
