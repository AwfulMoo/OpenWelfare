Review system stages

Table 2

+-----------------------------------+-----------------------------------+
| Stage                             | Description                       |
+-----------------------------------+-----------------------------------+
| Cancellation Stage                | -   This is the date that the     |
|                                   |     system will invoke an         |
|                                   |     automatic cancellation of the |
|                                   |     Online Intervention           |
|                                   | -   Once the payment has been     |
|                                   |     cancelled, the review will    |
|                                   |     also complete                 |
|                                   | -   The system will only perform  |
|                                   |     the automatic cancellation in |
|                                   |     cases where no parties have   |
|                                   |     responded when the            |
|                                   |     Cancellation date is reached  |
+-----------------------------------+-----------------------------------+
| Review Completion Due Date        | -   This is the date that serves  |
|                                   |     as a benchmark date that will |
|                                   |     decide when the review will   |
|                                   |     be workflowed to a user       |
|                                   |     (Online Intervention (OI)     |
|                                   |     status set to For Manual      |
|                                   |     Action)                       |
|                                   | -   Before this date, if all      |
|                                   |     parties in the AoS responds   |
|                                   |     to their questionnaires the   |
|                                   |     OI is workflowed immediately  |
|                                   | -   On or after this date, if at  |
|                                   |     least one party in the AoS    |
|                                   |     responds to the               |
|                                   |     questionnaire, the OI will be |
|                                   |     workflowed                    |
+-----------------------------------+-----------------------------------+
| Extensions Date                   | This is the start date of this    |
|                                   | particular review cycle only      |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   In a scenario where the AoE   |
|                                   |     started receiving a payment   |
|                                   |     on 1/1/17                     |
|                                   | -   The first quarterly review    |
|                                   |     will have an extension date   |
|                                   |     of 1/1/17                     |
|                                   | -   The second quarterly review   |
|                                   |     will have an extension date   |
|                                   |     of 1/4/17 and so on           |
+-----------------------------------+-----------------------------------+
| Form Issue Stage                  | -   This is date the forms is     |
|                                   |     issued to the customer, this  |
|                                   |     includes the paper and/or     |
|                                   |     online form whichever is      |
|                                   |     applicable                    |
|                                   | -   Once this date is reached,    |
|                                   |     the forms are issued and the  |
|                                   |     status of the OI is set to    |
|                                   |     ‘IN PROCESS’                  |
+-----------------------------------+-----------------------------------+
| Initial Notification Receipt Date | This is the start date of the     |
|                                   | review cycle for this year only   |
|                                   |                                   |
|                                   | The yearly review only calculates |
|                                   | recoverable amounts paid to the   |
|                                   | AoE from this date                |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Where AoE started receiving a |
|                                   |     payment on 1/1/17             |
|                                   | -   All quarterly reviews until   |
|                                   |     the yearly review, have an    |
|                                   |     initial notification receipt  |
|                                   |     date of 1/1/17                |
|                                   | -   However, after the first      |
|                                   |     yearly review, the next       |
|                                   |     year’s quarterly reviews has  |
|                                   |     an internal notification      |
|                                   |     receipt date of 1/1/18        |
+-----------------------------------+-----------------------------------+
| Reminder Stage (RM1)              | -   When this date is reached,    |
|                                   |     the reminder letter for       |
|                                   |     parties to respond is         |
|                                   |     generated                     |
|                                   | -   This is usually the same date |
|                                   |     as the review completion due  |
|                                   |     date                          |
+-----------------------------------+-----------------------------------+
| Suspension Stage                  | -   This is the date the system   |
|                                   |     invokes an automatic          |
|                                   |     cancellation of the Online    |
|                                   |     Intervention                  |
|                                   | -   Once the payment is           |
|                                   |     cancelled, the review remains |
|                                   |     in the ‘In process’ status    |
|                                   | -   The system only performs the  |
|                                   |     automatic suspension in cases |
|                                   |     where no parties have         |
|                                   |     responded when the            |
|                                   |     Cancellation date is reached  |
+-----------------------------------+-----------------------------------+

System process map

Table 3

  ----------------------------------- -------------------------------- -------------------------------------------------------------------------------------------
  Stages                              Date                             Description
  Extension Date                      Review period start fate (EXT)   Date work item us created (to be processed at a future date).
  Form Issue Stage                    Start of review date (FMI)       The date the quarterly and yearly review are opened for response by AOR and AOE.
  Initial Notification Receipt Date   First Review period date         Same as extension date.
  Reminder Stage (RM1)                Due date (DUE)                   21 days from ‘From Issue Stage Date’.
  Suspension Stage (SUS)              Suspension date (SUS)            35 days from ‘From Issue Stage Date’.
  Cancelation Stage (CAN)             Cancelation date (CAN)           56 days from ‘From Issue Stage Date’.
  Future Action date (FUT)            Future Action date (FUT)         Used for commencement and yearly reviews to set up work item. Used when date rescheduled.
