Add customer to opt-out list

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Removing Customer Research        |
|                                   | Participation Code                |
|                                   |                                   |
|                                   | A customer can consent to be      |
|                                   | removed from taking part in       |
|                                   | customer satisfaction surveys and |
|                                   | other market research undertaken  |
|                                   | by Services Australia. This will  |
|                                   | show on the Other Contact Details |
|                                   | (OCD) screen.                     |
|                                   |                                   |
|                                   | To exclude a customer, update the |
|                                   | field with a value of 'N' for no. |
|                                   |                                   |
|                                   | The valid values are:             |
|                                   |                                   |
|                                   | -   Y = Yes                       |
|                                   | -   N = No                        |
|                                   | -   Blank = Not Recorded (passive |
|                                   |     consent)                      |
+-----------------------------------+-----------------------------------+
