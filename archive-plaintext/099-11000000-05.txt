Bereavement review timeline

+-----------------------------------+-----------------------------------+
| Date                              | Event                             |
+-----------------------------------+-----------------------------------+
| Day zero                          | Date of death. Death notification |
|                                   | is expected within 28 days.       |
+-----------------------------------+-----------------------------------+
| Day 56 (8 weeks after death)      | Centrelink Bereavement Statement  |
|                                   | is issued.                        |
+-----------------------------------+-----------------------------------+
| Days 56 to 112                    | If the customer updates income    |
|                                   | and assets during this time, the  |
|                                   | review will auto complete.        |
+-----------------------------------+-----------------------------------+
| Day 112 (16 weeks after death)    | System issues Income and Asset    |
|                                   | Update (SA220) form. The          |
|                                   | bereavement review starts.        |
+-----------------------------------+-----------------------------------+
| After day 112                     | If the customer updates income    |
|                                   | and assets online, the            |
|                                   | bereavement review will auto      |
|                                   | complete.                         |
|                                   |                                   |
|                                   | If the customer updates income    |
|                                   | and assets by phone or in         |
|                                   | writing, the bereavement review   |
|                                   | is manually completed by a        |
|                                   | Service Officer.                  |
+-----------------------------------+-----------------------------------+
| Day 117 (5 days after SA220       | Update your income and assets     |
| issued)                           | task displays on the customer’s   |
|                                   | Centrelink online account. A      |
|                                   | delay of 5 days is input to allow |
|                                   | for the SA220 to be received in   |
|                                   | the mail.                         |
+-----------------------------------+-----------------------------------+
| Day 154 (6 weeks after SA220      | A reminder (second request) SA220 |
| issued)                           | is issued if there has been no    |
|                                   | reply or request for extension.   |
+-----------------------------------+-----------------------------------+
| Day 182 (4 weeks after SA220      | The bereavement review falls due  |
| reminder issued)                  | for customer contact if no reply  |
|                                   | has been received.                |
+-----------------------------------+-----------------------------------+
| Up to day 365 (1 year after       | Maximum time customers can extend |
| death)                            | their review using their          |
|                                   | Centrelink online account.        |
+-----------------------------------+-----------------------------------+
| Up to day 720 (2 years after      | Maximum time staff can extend the |
| death)                            | bereavement review using Process  |
|                                   | Direct.                           |
+-----------------------------------+-----------------------------------+
| After day 720 (more than 2 years) | Manual review completed based on  |
|                                   | current income and assets.        |
|                                   |                                   |
|                                   | When the deceased partner's       |
|                                   | estate has been finalised the     |
|                                   | surviving partner notifies        |
|                                   | Services Australia. If needed,    |
|                                   | more changes to income or assets  |
|                                   | are made on the surviving         |
|                                   | partner's customer record.        |
+-----------------------------------+-----------------------------------+
