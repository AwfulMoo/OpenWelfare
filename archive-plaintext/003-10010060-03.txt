This page contains the process for urgent payment requests due to a
delay in the delivery of a customer's payment because of a national
public holiday.

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Late lodging due to public        |
|                                   | holiday + Read more ...           |
|                                   |                                   |
|                                   | Does the customer need to report  |
|                                   | their employment income and is    |
|                                   | late lodging or due to report on  |
|                                   | the day before the public         |
|                                   | holiday?                          |
|                                   |                                   |
|                                   | -   Yes, process the late         |
|                                   |     lodgement and stimulate the   |
|                                   |     payment. Go to Step 3         |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Has the customer's payment been   |
|                                   | delayed for any other reason? +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Yes, this procedure cannot be     |
|                                   | used:                             |
|                                   |                                   |
|                                   | -   Advise the customer that the  |
|                                   |     national public holiday will  |
|                                   |     not affect the day the        |
|                                   |     payment would normally be     |
|                                   |     delivered                     |
|                                   | -   If the customer still         |
|                                   |     requests an urgent payment,   |
|                                   |     assess under usual urgent     |
|                                   |     payment guidelines. See       |
|                                   |     Urgent payments due to        |
|                                   |     exceptional and unforeseen or |
|                                   |     extraordinary circumstances.  |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Checking the payment summary for  |
|                                   | payment status + Read more ...    |
|                                   |                                   |
|                                   | Check the Payment Summary (PS)    |
|                                   | screen for status of payment.     |
|                                   |                                   |
|                                   | If a date displays on the PS      |
|                                   | screen, the payment was issued    |
|                                   | via direct credit. It cannot be   |
|                                   | reissued.                         |
|                                   |                                   |
|                                   | Does Immediate or late stm        |
|                                   | display on the PS screen?         |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No, this procedure cannot be      |
|                                   | used:                             |
|                                   |                                   |
|                                   | -   Advise the customer that the  |
|                                   |     national public holiday will  |
|                                   |     not affect the day the        |
|                                   |     payment would normally be     |
|                                   |     delivered                     |
|                                   | -   If the customer still         |
|                                   |     requests an urgent payment,   |
|                                   |     assess under usual urgent     |
|                                   |     payment guidelines. See       |
|                                   |     Urgent payments due to        |
|                                   |     exceptional and unforeseen or |

| 4                                 | Processing urgent payment         |
|                                   | request + Read more ...           |
|                                   |                                   |
|                                   | Due to the public holiday periods |
|                                   | delaying payment, especially for  |
|                                   | late lodgers, the customer is     |
|                                   | eligible for an urgent payment.   |
|                                   |                                   |
|                                   | Use the reason code 'DEP'         |
|                                   | (Centrelink/Departmental delay)   |
|                                   | with an explanation of 'public    |
|                                   | holiday processing'.              |
|                                   |                                   |
|                                   | An urgent payment due to          |
|                                   | Centrelink delay should be issued |
|                                   | for the lesser of:                |
|                                   |                                   |
|                                   | -   the amount required to meet   |
|                                   |     the customer's immediate      |
|                                   |     need, or                      |
|                                   | -   $1,000                        |

|                                   | Note: urgent payments for FHA     |
|                                   | customers cannot be processed     |
|                                   | using the Immediate Payment       |
|                                   | Request script.                   |

|                                   | requests.                         |
