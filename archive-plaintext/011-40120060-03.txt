This page contains the processes for Service Officers and Programme
Support Officers (PSO) to transfer and receive calls via a PSO line and
what to do when the PSO line is busy. It also contains processes to
follow when a PSO refuses to accept a call, or when a Service Officer
inappropriately transfers a call.

Transferring and receiving PSO calls

Table 1: This table describes the process for Service Officers and
Programme Support Officers (PSO) when transferring and receiving a call
via the appropriate PSO line.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Call received by Service          |
|                                   | Officer + Read more ...           |
|                                   |                                   |
|                                   | Service Officer to answer call    |
|                                   | and perform a security check.     |
|                                   |                                   |
|                                   | -   If the enquiry cannot be      |
|                                   |     resolved using the available  |
|                                   |     reference material, the       |
|                                   |     Service Officer can contact   |
|                                   |     the PSO line for assistance.  |
|                                   | -   If a customer requests to     |
|                                   |     speak to a supervisor for     |
|                                   |     further clarification of      |
|                                   |     technical information, advise |
|                                   |     the customer technical        |
|                                   |     assistance from a Programme   |
|                                   |     Support Officer will be       |
|                                   |     sought. The call may be       |
|                                   |     transferred if it is          |
|                                   |     required.                     |
|                                   |                                   |
|                                   | Note: Customer calls regarding    |
|                                   | telephony wait times or operator  |
|                                   | behaviour should be referred to   |
|                                   | the Service Officer’s team leader |
|                                   | and not the PSO line.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Obtain relevant information +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Note: It is a requirement for the |
|                                   | Service Officer to check:         |
|                                   |                                   |
|                                   | -   OB prior to contacting a PSO  |
|                                   |     and                           |
|                                   | -   PSO availability on the PSO   |
|                                   |     Share Point Page              |
|                                   |                                   |
|                                   | Ensure the following information  |
|                                   | is available prior to contacting  |
|                                   | a PSO:                            |
|                                   |                                   |
|                                   | -   Customer Reference Number     |
|                                   |     (e.g. Medicare Card Number,   |
|                                   |     Provider Number, Prescriber   |
|                                   |     Number etc.)                  |
|                                   | -   Customer’s full name          |
|                                   | -   Details of enquiry            |
|                                   | -   Actions taken prior to        |
|                                   |     contacting the PSO line, for  |
|                                   |     example, checking Operational |
|                                   |     Blueprint (OB)                |
+-----------------------------------+-----------------------------------+
| 3                                 | Place on hold + Read more ...     |
|                                   |                                   |
|                                   | Advise caller they will be put on |
|                                   | hold while technical assistance   |
|                                   | is sought from a Programme        |
|                                   | Support Officer.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Contact a PSO + Read more ...     |
|                                   |                                   |
|                                   | Contact the appropriate PSO line  |
|                                   | (The Resources page contains a    |
|                                   | link to the Medicare Health       |
|                                   | Programme Support Officers (PSO)  |
|                                   | full list of contact numbers).    |
|                                   | The Service Officer should        |
|                                   | introduce themselves with their   |
|                                   | name and universal log on or P    |
|                                   | number.                           |
|                                   |                                   |
|                                   | PSO (example only): “Welcome to   |
|                                   | Health Programme Support, this is |
|                                   | (name)”.                          |
|                                   |                                   |
|                                   | Service Officer (example only):   |
|                                   | “Hello (PSO name), this is        |
|                                   | (Service Officer name),           |
|                                   | (universal log on or P number)”.  |
|                                   |                                   |
|                                   | Note: if line is busy, see PSO    |
|                                   | line is busy table.               |
+-----------------------------------+-----------------------------------+
| 5                                 | Provide relevant information +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Service Officer to advise PSO,    |
|                                   | details of the:                   |
|                                   |                                   |
|                                   | -   caller (for example, Medicare |
|                                   |     card number and full name)    |
|                                   | -   enquiry                       |
|                                   | -   reference material used       |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Do not conference the call at |
|                                   |     the initial point of          |
|                                   |     transfer. The Service Officer |
|                                   |     must discuss the enquiry with |
|                                   |     the PSO                       |
|                                   | -   Conference calls should only  |
|                                   |     occur with the agreement of   |
|                                   |     all parties                   |
|                                   | -   The Service Officer should    |
|                                   |     inform the PSO and customer   |
|                                   |     if they want to remain on the |
|                                   |     phone and listen to the call  |
|                                   |     for development purposes. In  |
|                                   |     this case, Service Officers   |
|                                   |     should activate the mute      |
|                                   |     button                        |
+-----------------------------------+-----------------------------------+
| 6                                 | PSO assistance + Read more ...    |
|                                   |                                   |
|                                   | The PSO will attempt to resolve   |
|                                   | the enquiry with the Service      |
|                                   | Officer by reviewing the          |
|                                   | Operational Blueprint material    |
|                                   | you have looked at and/or         |
|                                   | providing guidance to the         |
|                                   | relevant information.             |
|                                   |                                   |
|                                   | The PSO can remain on hold while  |
|                                   | additional information is sought  |
|                                   | from the caller or when           |
|                                   | additional information is being   |
|                                   | provided to the caller to resolve |
|                                   | the enquiry.                      |
|                                   |                                   |
|                                   | If the PSO cannot resolve the     |
|                                   | enquiry and a return call is      |
|                                   | required, it is the               |
|                                   | responsibility of the PSO to      |
|                                   | contact the person with           |
|                                   | resolution of the enquiry. In     |
|                                   | this instance, the PSO may        |
|                                   | request the Service Officer to    |
|                                   | transfer the call to them.        |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   If the call needs to be       |
|                                   |     escalated to the PSO, they    |
|                                   |     may ask the Service Officer   |
|                                   |     to conference the call and    |
|                                   |     stay on the line to listen to |
|                                   |     the outcome for their         |
|                                   |     development. The Service      |
|                                   |     Officer can also request to   |
|                                   |     remain on the phone by        |
|                                   |     speaking with the PSO before  |
|                                   |     they conference the call.     |
|                                   |     Seek consent from the         |
|                                   |     customer to remain on the     |
|                                   |     call. If the customer does    |
|                                   |     not agree, the Service        |
|                                   |     Officer should end the call   |
|                                   |     after they’ve transferred it  |
|                                   |     to the PSO                    |
|                                   | -   The PSO and Service Officer   |
|                                   |     must be conscious to avoid    |
|                                   |     lengthy hold times and        |
|                                   |     consider conferencing the     |
|                                   |     call to resolve complex       |
|                                   |     issues or to mentor the       |
|                                   |     Service Officer. PSOs should  |
|                                   |     include this action in the    |
|                                   |     PSO Support Tool comments     |
|                                   |                                   |
|                                   | See also: PSO refuses to accept   |
|                                   | call table and Call transferred   |
|                                   | to PSO inappropriately table.     |
+-----------------------------------+-----------------------------------+
| 7                                 | Determine if call to be           |
|                                   | transferred + Read more ...       |
|                                   |                                   |
|                                   | PSO to determine if they should   |
|                                   | take over the call or if the      |
|                                   | Service Officer can assist the    |
|                                   | caller following PSO advice.      |
+-----------------------------------+-----------------------------------+
| 8                                 | Log call + Read more ...          |
|                                   |                                   |
|                                   | PSO completes the PSO Support     |
|                                   | Tool details.                     |
|                                   |                                   |
|                                   | PSO must complete the PSO Support |
|                                   | Tool for every PSO contact, every |
|                                   | time from every contact source.   |
|                                   |                                   |
|                                   | Note: access to PSO Support Tool  |
|                                   | is located on SharePoint via the  |
|                                   | Quality and Support Tool.         |
+-----------------------------------+-----------------------------------+
| 9                                 | PSO tool received by Service      |
|                                   | Officer + Read more ...           |
|                                   |                                   |
|                                   | Service Officers are to review    |
|                                   | PSO Support Tool emails including |
|                                   | any reference links during        |
|                                   | Learning and Development time to  |
|                                   | develop their knowledge and       |
|                                   | understanding of the programme.   |

The PSO line is busy

Table 2: This table describes the procedure for Service Officers to
follow when the Programme Support Officers (PSO) line is busy.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Consider queue wait time + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Service Officers should use their |
|                                   | judgement as to the length of     |
|                                   | time to wait for a PSO.           |
|                                   |                                   |
|                                   | Service Officers should always    |
|                                   | seek confirmation from the        |
|                                   | customer that they are happy to   |
|                                   | hold and regularly check in with  |
|                                   | the customer while on hold.       |
|                                   |                                   |
|                                   | If the PSO line is busy and the   |
|                                   | customer has been on hold for a   |
|                                   | considerable amount of time or no |
|                                   | PSOs are available go to Step 2.  |
|                                   |                                   |
|                                   | Note: a PSO referral form should  |
|                                   | be submitted when a SO is unable  |
|                                   | to reach a PSO within 10 minutes  |
|                                   | unless the query is urgent, see   |
|                                   | Resources page.                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Return to call + Read more ...    |
|                                   |                                   |
|                                   | Advise the caller that technical  |
|                                   | assistance is unavailable and     |
|                                   | their enquiry will be referred to |
|                                   | a PSO who will contact them.      |
|                                   |                                   |
|                                   | Confirm the customer’s contact    |
|                                   | details and complete a PSO        |
|                                   | Referral Form including:          |
|                                   |                                   |
|                                   | -   as much relevant information  |
|                                   |     as possible                   |
|                                   | -   the contact information for   |
|                                   |     the customer                  |
|                                   |                                   |
|                                   | Note: PSO referral forms with     |
|                                   | insufficient information for the  |
|                                   | PSO to address the enquiry, will  |
|                                   | be returned to the Service        |
|                                   | Officer.                          |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the PSO Referral Form.    |
+-----------------------------------+-----------------------------------+

PSO enquiries outside business hours

Table 3: this table describes the process for Service Officers to follow
when a PSO is unavailable outside business hours.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine if the call needs to be |
|                                   | escalated + Read more ...         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, complete the call as per  |
|                                   |     standard procedures           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact a Health Service Delivery |
|                                   | Division PSO. + Read more ...     |
|                                   |                                   |
|                                   | Is a PSO available?               |
|                                   |                                   |
|                                   | -   Yes, the call will be managed |
|                                   |     by the PSO                    |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | PSO not available + Read more ... |
|                                   |                                   |
|                                   | If the PSO team cannot be         |
|                                   | contacted, complete and submit a  |
|                                   | PSO Referral form. See the        |
|                                   | Resources page for a link to the  |
|                                   | form.                             |
|                                   |                                   |
|                                   | Assure the caller their enquiry   |
|                                   | will be attended to as soon as    |
|                                   | possible.                         |
|                                   |                                   |
|                                   | Service Officers must complete    |
|                                   | the PSO Referral form in full,    |
|                                   | providing as much details as      |
|                                   | possible including customer       |
|                                   | contact numbers.                  |
+-----------------------------------+-----------------------------------+

PSO refuses to accept a call

Table 4: This table describes the process for Service Officers to follow
when a Programme Support Officers (PSO) refuses to accept a call.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record details + Read more ...    |
|                                   |                                   |
|                                   | Service Officer to record:        |
|                                   |                                   |
|                                   | -   PSO’s name                    |
|                                   | -   logon                         |
|                                   | -   date and time                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Forward details to Team Leader +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Service Officer to forward        |
|                                   | details and a copy of any         |
|                                   | response email to the Service     |
|                                   | Officer's Team Leader.            |
+-----------------------------------+-----------------------------------+
| 3                                 | Team Leader to investigate + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Investigate the incident and      |
|                                   | consider:                         |
|                                   |                                   |
|                                   | -   What the enquiry was and the  |
|                                   |     reason for escalation         |
|                                   | -   If the call should normally   |
|                                   |     be handled by a Service       |
|                                   |     Officer                       |
|                                   | -   If the call should have been  |
|                                   |     directed to the Team Leader   |
|                                   | -   If there is a knowledge or    |
|                                   |     training gap                  |
|                                   | -   If refusal was appropriate    |
|                                   |                                   |
|                                   | If refusal was appropriate,       |
|                                   | discuss the incident with the     |
|                                   | Service Officer and how they      |
|                                   | could handle it next time.        |
|                                   |                                   |
|                                   | If the PSO should have taken over |
|                                   | the call, submit feedback using   |
|                                   | the Staff Feedback Tool.          |
+-----------------------------------+-----------------------------------+

Inappropriate transfer to a PSO or Tier 2

Table 5: This table describes the process for Programme Support Officers
(PSO) to follow when a Service Officer inappropriately transfers a call.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record details + Read more ...    |
|                                   |                                   |
|                                   | PSO to record:                    |
|                                   |                                   |
|                                   | -   Service Officer’s name        |
|                                   | -   logon                         |
|                                   | -   date and time                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Forward details to Team Leader +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | PSO to forward details, including |
|                                   | any PSO tool emails sent to the   |
|                                   | Service Officer to the PSO's Team |
|                                   | Leader.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Team Leader to provide feedback + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Team Leader to investigate the    |
|                                   | incident and if appropriate, send |
|                                   | feedback to the Service Officer’s |
|                                   | Team Leader using the Staff       |
|                                   | Feedback Tool.                    |
+-----------------------------------+-----------------------------------+
