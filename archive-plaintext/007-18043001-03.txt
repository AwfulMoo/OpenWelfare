Assessing and recording extensions

Table 3: this table describes the process to manage extensions to
complete HSFS requirements for financial years from 2018-19 onwards.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Extension + Read more ...         |
|                                   |                                   |
|                                   | An extension of time to meet      |
|                                   | health check requirements can be  |
|                                   | applied when special              |
|                                   | circumstances prevented the       |
|                                   | customer from meeting HSFS        |
|                                   | requirements prior to the child's |
|                                   | fifth birthday. These generally   |
|                                   | relate to the customer's          |
|                                   | circumstances rather than the     |
|                                   | child's. Requests must be         |
|                                   | considered on a case by case      |
|                                   | basis.                            |
|                                   |                                   |
|                                   | Special circumstance extension    |
|                                   | criteria are satisfied if the     |
|                                   | customer has unusual and/or       |
|                                   | unexpected circumstances as       |
|                                   | outlined on the Resources page.   |
|                                   | If assistance is required in      |
|                                   | determining eligibility, call     |
|                                   | Access support (service centre    |
|                                   | staff) or contact Technical Peer  |
|                                   | Support (TPS) (Smart Centre       |
|                                   | Call).                            |
|                                   |                                   |
|                                   | The Resources page contains more  |
|                                   | information about assessing       |
|                                   | special circumstances and checks  |
|                                   | to be made before assessing       |
|                                   | eligibility for a special         |
|                                   | circumstances exemption.          |
|                                   |                                   |
|                                   | Is the customer eligible for an   |
|                                   | extension of time to complete the |
|                                   | HSFS requirements?                |
|                                   |                                   |
|                                   | Note: an extension cannot be      |
|                                   | applied for until the child's     |
|                                   | fifth birthday.                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Refer the record for coding the   |
|                                   | extension + Read more ...         |
|                                   |                                   |
|                                   | Access to code the extension is   |
|                                   | restricted. Service Officers must |
|                                   | refer the record directly using   |
|                                   | the HSFS exemption request        |
|                                   | webform. The Resources page       |
|                                   | contains a link to the National   |
|                                   | Referral process. To access the   |
|                                   | webform:                          |
|                                   |                                   |
|                                   | -   from the Resources page link  |
|                                   |     go to the Business Process    |
|                                   |     and Change Branch homepage,   |
|                                   |     and                           |
|                                   | -   select the link to the        |
|                                   |     National ICT Referral Process |
|                                   |     within the Featured content   |
|                                   |     section, then                 |
|                                   | -   select the HSFS extension     |
|                                   |     request button                |
