Interaction Workspace (iWS) Softphone Skills - announced, unannounced and cold call transfer procedures

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Unannounced Transfers (Cold) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In some instances it may be       |
|                                   | appropriate to instantly transfer |
|                                   | a caller to another business line |
|                                   | without introducing the caller    |
|                                   | first. This is referred to as an  |
|                                   | unannounced or cold transfer.     |
|                                   |                                   |
|                                   | Does the call need to instantly   |
|                                   | transfer to another line?         |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   with the caller on the line   |
|                                   |     (not on hold) select instant  |
|                                   |     call transfer                 |
|                                   | -   search for and select the     |
|                                   |     required number or programme  |
|                                   | -   select instant call transfer  |
|                                   | -   call will transfer and        |
|                                   |     softphone will drop into 10   |
|                                   |     secs of After Call Work (ACW) |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Announced Transfers (Warm) + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | An announced transfer (warm) is   |
|                                   | when the call is introduced by    |
|                                   | the first Service Officer to      |
|                                   | another Service Officer or other  |
|                                   | person receiving the call. All    |
|                                   | call escalations should be warm   |
|                                   | transferred.                      |
|                                   |                                   |
|                                   | To warm transfer a call, Service  |
|                                   | Officers must:                    |
|                                   |                                   |
|                                   | -   with the caller on the line   |
|                                   |     (not on hold) select start a  |
|                                   |     consultation                  |
|                                   | -   search for and select the     |
|                                   |     required number or programme  |
|                                   | -   select initialise a voice     |
|                                   |     consultation. The first       |
|                                   |     caller will be placed on hold |
|                                   |     and a new call will be        |
|                                   |     started                       |
|                                   | -   to connect the two calls      |
|                                   |     select instant call           |
|                                   |     conference                    |
|                                   | -   introduce the caller to the   |
|                                   |     Service Officer or the other  |
|                                   |     person taking over the        |
|                                   |     enquiry                       |
|                                   | -   end the call                  |
|                                   | -   call will transfer and        |
|                                   |     softphone will drop into 10   |
|                                   |     secs of After Call work (ACW) |
|                                   |                                   |
|                                   | Procedure ends here.              |
