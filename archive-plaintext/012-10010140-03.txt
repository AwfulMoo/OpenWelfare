Managing audit/compliance enquiries

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Audit / compliance enquiry + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Did the customer receive          |
|                                   | correspondence about an audit or  |
|                                   | compliance activity?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Correspondence + Read more ...    |
|                                   |                                   |
|                                   | Ask the caller if there are any   |
|                                   | contact details on the            |
|                                   | correspondence.                   |
|                                   |                                   |
|                                   | Is there contact details          |
|                                   | available?                        |
|                                   |                                   |
|                                   | -   Yes, the customer must use    |
|                                   |     those contact details for     |
|                                   |     their enquiry. Procedure ends |
|                                   |     here                          |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Security check + Read more ...    |
|                                   |                                   |
|                                   | Only provide practice information |
|                                   | to an authorised person. Conduct  |
|                                   | a security check.                 |
|                                   |                                   |
|                                   | Is the caller authorised?         |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No, do not discuss any            |
|                                   | information with the customer.    |
|                                   | Advise the customer:              |
|                                   |                                   |
|                                   | -   they did not meet the         |
|                                   |     requirements of the security  |
|                                   |     check                         |
|                                   | -   an owner or authorised        |
|                                   |     contact person must call back |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Check comments + Read more ...    |
|                                   |                                   |
|                                   | Check the comments on the         |
|                                   | practice profile. Look for any    |
|                                   | comments about an audit or        |
|                                   | compliance activity.              |
|                                   |                                   |
|                                   | Is a relevant comment available:  |
|                                   |                                   |
|                                   | -   Yes, tell the customer how to |
|                                   |     proceed per the comment.      |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Check associated documents + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the associated documents.   |
|                                   | Look for any correspondence about |
|                                   | an audit or compliance activity.  |
|                                   |                                   |
|                                   | Is a relevant document available? |
|                                   |                                   |
|                                   | -   Yes, tell the customer how to |
|                                   |     proceed as per the associated |
|                                   |     document. Procedure ends here |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Escalation database + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Search the Incentives Escalation  |
|                                   | Database for any information      |
|                                   | about an audit or compliance      |
|                                   | activity for this practice.       |
|                                   |                                   |
|                                   | Is there information?             |
|                                   |                                   |
|                                   | -   Yes, tell the customer how to |
|                                   |     proceed as per the escalation |
|                                   |     response. Procedure ends here |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Contact a PSO + Read more ...     |
|                                   |                                   |
|                                   | Contact a Program Support Officer |
|                                   | (PSO) for assistance.             |
|                                   |                                   |
|                                   | Advise the caller of the advice   |
|                                   | from the PSO.                     |

Telephone security checks

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Advise security check + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Tell the customer they must       |
|                                   | complete a security check before  |
|                                   | releasing any information.        |
|                                   |                                   |
|                                   | Suggested wording: ‘Before I can  |
|                                   | discuss details with you, do you  |
|                                   | mind if I go through some quick   |
|                                   | security questions to establish I |
|                                   | am talking to the correct         |
|                                   | person?’                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Program + Read more ...           |
|                                   |                                   |
|                                   | If the request relates to the:    |
|                                   |                                   |
|                                   | -   PIP/WIP – Practice Stream, go |
|                                   |     to Step 3                     |
|                                   | -   ACAI, go to Step 5            |
+-----------------------------------+-----------------------------------+
| 3                                 | PIP/WIP – Practice Stream         |
|                                   | security check + Read more ...    |
|                                   |                                   |
|                                   | Only provide practice information |
|                                   | to an owner or authorised contact |
|                                   | person.                           |
|                                   |                                   |
|                                   | Is the enquiry from an owner or   |
|                                   | authorised contact person?        |
|                                   |                                   |
|                                   | -   Yes, continue with the        |
|                                   |     enquiry. Procedure ends here. |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | PIP/WIP – Practice Stream verbal  |
|                                   | permission + Read more ...        |
|                                   |                                   |
|                                   | Request verbal permission         |
|                                   | directly from an owner or         |
|                                   | authorised contact person.        |
|                                   |                                   |
|                                   | Is there verbal permission?       |
|                                   |                                   |
|                                   | -   Yes, continue with the        |
|                                   |     enquiry. Procedure ends here. |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 5                                 | ACAI security check + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | PIP GP ACAI payments are made to  |
|                                   | practitioners. Only provide       |
|                                   | details directly to the           |
|                                   | practitioner related to the       |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Ask the practitioner for the      |
|                                   | following details?                |
|                                   |                                   |
|                                   | -   Provider number               |
|                                   | -   Name                          |
|                                   | -   Date of birth                 |
+-----------------------------------+-----------------------------------+
| 6                                 | ACAI security check – search      |
|                                   | practitioner + Read more ...      |
|                                   |                                   |
|                                   | To check the practitioner:        |
|                                   |                                   |
|                                   | -   Log on to the Provider        |
|                                   |     Directory System (PDS)        |
|                                   | -   Select Provider               |
|                                   | -   Select View                   |
|                                   | -   Key Provider Stem (first 6    |
|                                   |     digits) in the Search ID      |
|                                   |     field                         |
|                                   | -   Select Provider Stem from the |
|                                   |     ID Type drop down             |
|                                   | -   Select Search                 |
|                                   |                                   |
|                                   | Does the practitioner’s name and  |
|                                   | date of birth match the details   |
|                                   | provided?                         |
|                                   |                                   |
|                                   | -   Yes, continue with the        |
|                                   |     enquiry, Procedure ends here  |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | ACAI verbal permission + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Request verbal permission         |
|                                   | directly from the practitioner    |
|                                   | related to the payment.           |
|                                   |                                   |
|                                   | Is there verbal permission?       |
|                                   |                                   |
|                                   | -   Yes, continue with the        |
|                                   |     enquiry. Procedure ends here. |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 8                                 | Caller not identified + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Advise the customer they did not  |
|                                   | meet the requirements of the      |
|                                   | security check. For the:          |
|                                   |                                   |
|                                   | -   PIP/WIP – Practice Stream, an |
|                                   |     owner or authorised contact   |
|                                   |     person must call back         |
|                                   | -   ACAI, the service provider    |
|                                   |     must call bank                |
|                                   |                                   |
|                                   | Do not discuss any information    |
|                                   | with the customer.                |

Escalate a processing enquiry

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check reference material + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Can the Service Officer resolve   |
|                                   | the processing enquiry?           |
|                                   |                                   |
|                                   | -   Yes, continue processing the  |
|                                   |     work item. Procedure ends     |
|                                   |     here                          |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact a PSO + Read more ...     |
|                                   |                                   |
|                                   | Contact a Program Support Officer |
|                                   | (PSO). Select Incentive PSO from  |
|                                   | the SoftPhone Transfer Options.   |
|                                   |                                   |
|                                   | Is a PSO available?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Discuss enquiry with PSO + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Discuss the processing enquiry    |
|                                   | with the PSO. Provide the         |
|                                   | following details:                |
|                                   |                                   |
|                                   | PaNDA work item details           |
|                                   |                                   |
|                                   | -   Work type                     |
|                                   | -   Case ID                       |
|                                   |                                   |
|                                   | The enquiry                       |
|                                   |                                   |
|                                   | Investigation and action taken    |
|                                   |                                   |
|                                   | The PSO will attempt to resolve   |
|                                   | the enquiry by reviewing the      |
|                                   | Operational Blueprint and other   |
|                                   | reference material.               |
|                                   |                                   |
|                                   | Did the PSO resolve the enquiry?  |
|                                   |                                   |
|                                   | -   Yes, continue processing the  |
|                                   |     work item as per the PSO      |
|                                   |     advice. Procedure ends here   |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Open the Incentives Access        |
|                                   | Database + Read more ...          |
|                                   |                                   |
|                                   | Open the Incentives Access        |
|                                   | Database from the below file      |
|                                   | path:                             |
|                                   |                                   |
|                                   | G:\INCENTIVESWORKFLOW\Access      |
|                                   | Database\Incentives               |
|                                   |                                   |
|                                   | -   Select Submit New Escalation  |
|                                   |     from the Main page            |
|                                   | -   The Submit New Escalation     |
|                                   |     form will show                |
+-----------------------------------+-----------------------------------+
| 5                                 | Reference details + Read more ... |
|                                   |                                   |
|                                   | Key the following reference       |
|                                   | details:                          |
|                                   |                                   |
|                                   | -   Operator ID                   |
|                                   | -   PaNDA ID (if applicable)      |
|                                   | -   PIP ID and/or WIP – Practice  |
|                                   |     Stream ID                     |
|                                   | -   Organisation name (if         |
|                                   |     applicable)                   |
|                                   | -   Practitioner name (if         |
|                                   |     applicable)                   |
|                                   | -   Provider number (if           |
|                                   |     applicable)                   |
+-----------------------------------+-----------------------------------+
| 6                                 | Work item details + Read more ... |
|                                   |                                   |
|                                   | Select/key the following details: |
|                                   |                                   |
|                                   | PaNDA work type, if processing a  |
|                                   | work item in PaNDA                |
|                                   |                                   |
|                                   | Non PaNDA work type, if           |
|                                   | processing a work item not in     |
|                                   | PaNDA                             |
|                                   |                                   |
|                                   | Notes, including:                 |
|                                   |                                   |
|                                   | -   the query                     |
|                                   | -   a description of the scenario |
|                                   | -   investigation and action      |
|                                   |     taken                         |
+-----------------------------------+-----------------------------------+
| 7                                 | Submit escalation + Read more ... |
|                                   |                                   |
|                                   | -   Select Save Record            |
|                                   | -   Select Close Form             |
|                                   |                                   |
|                                   | The work item is sent to the      |
|                                   | Program Support Officer (PSO) for |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | Check the Incentives Access       |
|                                   | Database regularly for a returned |
|                                   | escalation.                       |

Retrieve an escalation

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Open the Incentives Access        |
|                                   | Database + Read more ...          |
|                                   |                                   |
|                                   | Open the Incentives Access        |
|                                   | Database from the below file      |
|                                   | path:                             |
|                                   |                                   |
|                                   | G:\INCENTIVESWORKFLOW\Access      |
|                                   | Database\Incentives               |
|                                   |                                   |
|                                   | -   Select Returned Escalations   |
|                                   |     from the Main page            |
|                                   | -   The Returned Escalations form |
|                                   |     will show                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for escalation + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Search for the previously         |
|                                   | escalated work item:              |
|                                   |                                   |
|                                   | Select the Operator field         |
|                                   |                                   |
|                                   | Press [CTRL] + [F]. A Find and    |
|                                   | Replace window will show          |
|                                   |                                   |
|                                   | Key the operator number that      |
|                                   | relates to the escalation         |
|                                   |                                   |
|                                   | Select Find Next                  |
|                                   |                                   |
|                                   | -   The escalation with the       |
|                                   |     earliest escalation date will |
|                                   |     show                          |
|                                   | -   Select Find Next until the    |
|                                   |     relevant escalation shows     |
|                                   |                                   |
|                                   | Does the relevant escalation      |
|                                   | show?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the escalation is not     |
|                                   |     finalised. Procedure ends     |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Process work item and close       |
|                                   | escalation + Read more ...        |
|                                   |                                   |
|                                   | Process the work item as per the  |
|                                   | Programme Officer notes.          |
|                                   |                                   |
|                                   | -   Processing a PaNDA work item, |
|                                   |     return to the relevant        |
|                                   |     Operational Blueprint         |
|                                   |     procedure and continue        |
|                                   |     processing                    |
|                                   | -   Triaging an email, respond to |
|                                   |     the email                     |
|                                   |                                   |
|                                   | Once action complete, close the   |
|                                   | escalation:                       |
|                                   |                                   |
|                                   | -   Key today’s day in the Case   |
|                                   |     Closed field                  |
|                                   | -   Select Save Record            |
|                                   | -   Select Close Form             |
