|                                   | Details (NMIS) screen.            |

|                                   |                                   |

|                                   |     date the death was notified   |
|                                   | -   Notified by - for PBV to be   |
|                                   |     paid, the notification must   |
|                                   |     be recorded as the customer   |
|                                   |     or their correspondence       |
|                                   |     nominee, even if advised by a |
|                                   |     third party                   |
|                                   | -   select PBV as the Payment     |
|                                   |     Choice                        |
|                                   | -   select Save                   |
|                                   |                                   |
|                                   | The Bereavement Period – Partner  |
|                                   | exemption (BPP) is recorded from  |
|                                   | the date of death of the partner  |
|                                   | for 14 weeks.                     |
|                                   |                                   |
|                                   | If the bereaved person is         |
|                                   | pregnant at the time of the       |
|                                   | partner’s death, once             |
|                                   | verification of pregnancy is      |
|                                   | provided they will have a         |
|                                   | Bereavement Period - Pregnant     |
|                                   | Partner (BPE) exemption:          |
|                                   |                                   |
|                                   | -   for 14 weeks from the date of |
|                                   |     the partner’s death, or       |
|                                   | -   to the end of the pregnancy,  |
|                                   |     whichever is later            |
|                                   |                                   |
|                                   | Go to Step 3.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Unemployment or Unemployment      |
|                                   | Non-Payment Period Failure + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Has an unemployment or            |
|                                   | unemployment non-payment period   |
|                                   | failure been generated for        |

|                                   | -   Yes, record a compliance      |
|                                   |     action rejection using        |

|                                   |     (FRB) code. Go to Step 4      |

+-----------------------------------+-----------------------------------+
