Identifying enrolment reviews

Table 1:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Enrolment check + Read more ...   |
|                                   |                                   |
|                                   | How enrolment checks are received |
|                                   | depends on the education provider |
|                                   | recorded on the Customer Study    |
|                                   | Details (EDC) screen in Process   |
|                                   | Direct or Education Course        |
|                                   | Details (EDC) screen in Customer  |
|                                   | First.                            |
|                                   |                                   |
|                                   | Paper enrolment checks            |
|                                   |                                   |
|                                   | -   the Study Load Compliance     |
|                                   |     processing team scans the     |
|                                   |     review onto the education     |
|                                   |     provider's organisation CRN   |
|                                   |     (Org CRN) as a Q790 letter    |
|                                   | -   Workload Management (WLM)     |
|                                   |     creates a ZDMS_Q790 work item |
|                                   |     for allocation                |
|                                   |                                   |
|                                   | If a ZDMS_Q790 work item has been |
|                                   | allocated or an applicant is      |
|                                   | asking about a paper enrolment    |
|                                   | check, go to Step 2.              |
|                                   |                                   |
|                                   | Electronic enrolment checks in    |
|                                   | the Integrated Review System      |
|                                   | (IRS)                             |
|                                   |                                   |
|                                   | -   an AIC Review Received DOC is |
|                                   |     recorded, and                 |
|                                   | -   a ZIAM_AIC_RVW work item is   |
|                                   |     created in WLM for allocation |
|                                   |                                   |
|                                   | If a ZIAM_AIC_RVW work item has   |
|                                   | been allocated or an applicant is |
|                                   | enquiring about an enrolment      |
|                                   | check, go to Step 3.              |
|                                   |                                   |
|                                   | Applicant responds or activity    |
|                                   | falls due                         |
|                                   |                                   |
|                                   | If the applicant has provided     |
|                                   | documents in response to a        |
|                                   | request for information (RFI) for |
|                                   | an enrolment check, or an open    |
|                                   | DOC has fallen due, see Step 1 in |
|                                   | Table 3.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Paper enrolment checks + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Enrolment checks that contain     |
|                                   | details for multiple students,    |
|                                   | are scanned onto the education    |
|                                   | provider's Organisation CRN (Org  |
|                                   | CRN).                             |
|                                   |                                   |
|                                   | To find the Org CRN in Customer   |
|                                   | First:                            |
|                                   |                                   |
|                                   | -   note the student's            |
|                                   |     Institution ID code, from the |
|                                   |     EDC screen                    |
|                                   | -   enter Codes in the Next field |
|                                   | -   select Enter                  |
|                                   | -   select the Education          |
|                                   |     Institution Summary line by   |
|                                   |     placing an 'S' in the field   |
|                                   | -   select Continue, to show the  |
|                                   |     Education Institution Summary |
|                                   |     (EIS) screen                  |
|                                   | -   Code: field, enter the        |
|                                   |     Institution ID from the       |
|                                   |     student's EDC screen          |
|                                   | -   select Continue               |
|                                   | -   copy the Org CRN from the     |
|                                   |     Educational Inst Details      |
|                                   |     (EID) screen                  |
|                                   | -   select Home                   |
|                                   | -   enter the Org CRN into the    |
|                                   |     CRN/BP field                  |
|                                   | -   select Enter                  |
|                                   | -   select Swap to B Environment  |
|                                   |                                   |
|                                   | Find the scanned paper enrolment  |
|                                   | check via Document Tools, from    |
|                                   | Customer First workspace.         |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Electronic enrolment checks +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Reviews returned electronically   |
|                                   | create an individual IRS review   |
|                                   | on each student record.           |
|                                   |                                   |
|                                   | There are 4 review types in IRS:  |
|                                   |                                   |
|                                   | -   WSE - Withdrawn Secondary     |
|                                   | -   NSE - No Record - Secondary   |
|                                   | -   PSE - Part Time - Secondary   |
|                                   | -   LSE - Late Starter Secondary  |
|                                   |                                   |
|                                   | To find the electronic enrolment  |
|                                   | check in the student's record:    |
|                                   |                                   |
|                                   | -   enter IRRS in the Next field  |
|                                   | -   select Enter                  |
|                                   | -   code 'S' against the REG or   |
|                                   |     STA review to view the Review |
|                                   |     Details (IRRD) screen, which  |
|                                   |     includes details about the    |
|                                   |     dates relevant to the review  |
|                                   | -   code 'S' in the Review        |
|                                   |     Actions field, to view the    |
|                                   |     Review Action History (IRRA)  |
|                                   |     screen, which includes a      |
|                                   |     history of the review actions |
|                                   | -   select Continue to go back to |
|                                   |     the IRRD screen               |
|                                   | -   code 'S' in the Match Data    |
|                                   |     field, to view the Match Data |
|                                   |     (IRMD) screen, which includes |
|                                   |     details about the term in     |
|                                   |     review, the education         |
|                                   |     provider, and the review type |
|                                   | -   select Continue to go back to |
|                                   |     the IRRD screen               |
+-----------------------------------+-----------------------------------+
| 4                                 | Review enrolment check details +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Compare the details in the        |
|                                   | enrolment check against the       |
|                                   | details on the student's record,  |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   EDC screen                    |
|                                   | -   Document List (DL), for       |
|                                   |     details in a DOC              |
|                                   | -   scanned documents via         |
|                                   |     Document Tools                |
|                                   |                                   |
|                                   | Is there a discrepancy between    |
|                                   | the enrolment check and the       |
|                                   | student's record, or are any      |
|                                   | updates needed?                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, see Step 1 in Table 4     |
+-----------------------------------+-----------------------------------+
| 5                                 | Outbound contact + Read more ...  |
|                                   |                                   |
|                                   | Where the details in the          |
|                                   | enrolment check show a change in  |
|                                   | circumstances, attempt to contact |
|                                   | the applicant to verbally confirm |
|                                   | the details.                      |
|                                   |                                   |
|                                   | Where the contact attempt is      |
|                                   | successful, see Step 1 in Table 2 |
|                                   |                                   |
|                                   | If no contact details are         |
|                                   | available, or contact is          |
|                                   | unsuccessful:                     |
|                                   |                                   |
|                                   | issue a Q781 letter asking the    |
|                                   | applicant to verify enrolment     |
|                                   | and/or study load                 |
|                                   |                                   |
|                                   | Record details in an open Fast    |
|                                   | Note. Select Auto Text, use       |
|                                   | Students, ABSTUDY & Apprentices > |
|                                   | AIC > AIC Enrolment Check         |
|                                   |                                   |
|                                   | Hold the Fast Note till the due   |
|                                   | date from the Q781 letter. Note:  |
|                                   | the open DOC acts as the review   |
|                                   | for return of the documents       |
|                                   |                                   |
|                                   | Where the enrolment check is:     |
|                                   |                                   |
|                                   | -   a paper review, go to Step 6  |
|                                   | -   an electronic review, go to   |
|                                   |     Step 7                        |
+-----------------------------------+-----------------------------------+
| 6                                 | Unsuccessful contact for paper    |
|                                   | review + Read more ...            |
|                                   |                                   |
|                                   | Have all student records          |
|                                   | identified on the paper enrolment |
|                                   | check been actioned?              |
|                                   |                                   |
|                                   | -   Yes, mark the scanned         |
|                                   |     document as Completed         |
|                                   | -   No, repeat steps for all      |
|                                   |     relevant student records      |
|                                   |     until all records are         |
|                                   |     actioned                      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Update IRS with unsuccessful      |
|                                   | contact + Read more ...           |
|                                   |                                   |
|                                   | Update the IRS review to show     |
|                                   | unsuccessful contact with the     |
|                                   | applicant.                        |
|                                   |                                   |
|                                   | In the student's record:          |
|                                   |                                   |
|                                   | -   enter IRRS in the Next field  |
|                                   | -   'S'elect the REG or STA       |
|                                   |     review                        |
|                                   | -   enter IRCS in the Next field  |
|                                   | -   select Enter, to go to the    |
|                                   |     Review Contact Summary (IRCS) |
|                                   |     screen                        |
|                                   | -   Meth: field, PHO              |
|                                   | -   Date Contacted: field, the    |
|                                   |     date the attempt to contact   |
|                                   |     the applicant was made        |
|                                   | -   select Continue               |
|                                   | -   Confirm: field, Y             |
|                                   | -   select Continue               |
|                                   | -   record all relevant fields on |
|                                   |     the Review Contact Details    |
|                                   |     (IRCD) screen                 |
|                                   |                                   |
|                                   | Note: once a contact starts, the  |
|                                   | review status changes from        |
|                                   | Registered (REG) to started       |
|                                   | (STA).                            |
|                                   |                                   |
|                                   | Procedure ends here.              |

Assessment and coding of an enrolment check

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Decide impact of the enrolment    |
|                                   | check + Read more ...             |
|                                   |                                   |
|                                   | Has the applicant confirmed the   |
|                                   | details in the enrolment check,   |
|                                   | or can confirm ongoing AIC        |
|                                   | entitlement without the need for  |
|                                   | further evidence?                 |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Issue a Q781 letter asking    |
|                                   |     the applicant to verify       |
|                                   |     enrolment and/or study load   |
|                                   | -   Suspend the AIC payment       |
|                                   |     'OTH' - Other, to prevent     |
|                                   |     overpayment                   |
|                                   | -   Record details in an open     |
|                                   |     Fast Note. Select Auto Text,  |
|                                   |     use Students, ABSTUDY &       |
|                                   |     Apprentices > AIC > AIC       |
|                                   |     Enrolment Check               |
|                                   | -   Hold the Fast Note till the   |
|                                   |     due date from the Q781 letter |
|                                   | -   Go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Update student record + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Update the student's record:      |
|                                   |                                   |
|                                   | Complete any changes to relevant  |
|                                   | screens. E.g., EDC                |
|                                   |                                   |
|                                   | If payments are suspended and     |
|                                   | ongoing eligibility has been      |
|                                   | confirmed:                        |
|                                   |                                   |
|                                   | -   restore the student's payment |
|                                   |     via the Benefit Action (BA)   |
|                                   |     screen                        |
|                                   | -   update and Finalise the AIC   |
|                                   |     Enrolment Check DOC           |
|                                   | -   See Step 1 in Table 4         |
+-----------------------------------+-----------------------------------+
| 3                                 | Letter action + Read more ...     |
|                                   |                                   |
|                                   | Once the AIC payment is           |
|                                   | suspended, issue a suspension     |
|                                   | letter to the applicant:          |
|                                   |                                   |
|                                   | -   go to the Letter Display (LD) |
|                                   |     screen                        |
|                                   | -   Letter Code: field, Q999      |
|                                   | -   select Enter                  |
|                                   | -   Service Reason: field, EIC    |
|                                   | -   select Teleserv No 2 in       |
|                                   |     contact field                 |
|                                   | -   include the appeal paragraph  |
|                                   | -   complete all other relevant   |
|                                   |     fields                        |
|                                   | -   Command: field, FIN -         |
|                                   |     Finalise                      |
|                                   |                                   |
|                                   | Resources contains text to use in |
|                                   | each free text paragraph.         |
|                                   |                                   |
|                                   | Procedure ends here.              |

Applicant responds or enrolment check falls due

Table 3:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Enrolment check activity falls    |
|                                   | due + Read more ...               |
|                                   |                                   |
|                                   | If the applicant has:             |
|                                   |                                   |
|                                   | -   supplied all necessary        |
|                                   |     evidence/details, go to Step  |
|                                   |     2                             |
|                                   | -   not supplied all necessary    |
|                                   |     evidence/details, go to Step  |
|                                   |     4                             |
|                                   | -   not responded to the letter   |
|                                   |     by the due date, go to Step 4 |
+-----------------------------------+-----------------------------------+
| 2                                 | Applicant responds + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Do the details /evidence supplied |
|                                   | confirm ongoing AIC entitlement?  |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   Make any necessary updates to |
|                                   |     the student's circumstances   |
|                                   | -   Record details on a DOC. See  |
|                                   |     Step 1 in Table 4             |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Eligibility not confirmed + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where AIC eligibility stops:      |
|                                   |                                   |
|                                   | -   Make any necessary updates to |
|                                   |     the student's circumstances   |
|                                   | -   Cancel the AIC payment via    |
|                                   |     the Benefit Action (BA)       |
|                                   |     screen (if a manual           |
|                                   |     cancellation is necessary)    |
|                                   | -   Record details on a DOC. See  |
|                                   |     Step 1 in Table 4             |
+-----------------------------------+-----------------------------------+
| 4                                 | Cancellation of payment + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where the applicant does not      |
|                                   | respond to the request for        |
|                                   | information, or does not verify   |
|                                   | ongoing entitlement:              |
|                                   |                                   |
|                                   | Update the relevant screens with  |
|                                   | the entitlement end date from the |
|                                   | enrolment check, where there has  |
|                                   | been a change to:                 |
|                                   |                                   |
|                                   | -   study details update EDC      |
|                                   | -   living arrangements update    |
|                                   |     Living Arrangement Details    |
|                                   |     (EILA)                        |
|                                   |                                   |
|                                   | Cancel the AIC payment via the    |
|                                   | Benefit Action (BA) screen from   |
|                                   | the date eligibility stopped. Use |
|                                   | the Reason Code: FRC - Failure to |
|                                   | reply to correspondence           |
|                                   |                                   |
|                                   | Where cancellation is not         |
|                                   | possible via the BA screen        |
|                                   | because of error E514EP - Effect  |
|                                   | Date must be later than Date of   |
|                                   | Commencement, go to Step 5        |
|                                   |                                   |
|                                   | Record details on a DOC. See Step |
|                                   | 1 in Table 4                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Cancellation where error          |
|                                   | present + Read more ...           |
|                                   |                                   |
|                                   | Complete cancellation in 2        |
|                                   | separate activities.              |
|                                   |                                   |
|                                   | -   Use Reason Code: FRC Failure  |
|                                   |     to reply to correspondence:   |
|                                   |     on the Benefit Action (BA)    |
|                                   | -   Cancel the current year       |
|                                   |     payments from 01 January and  |
|                                   |     finalise                      |
|                                   | -   Cancel previous year payments |
|                                   |     from the date AIC eligibility |
|                                   |     stopped and finalise          |
|                                   | -   Record details on a DOC. See  |
|                                   |     Step 1 in Table 4             |
+-----------------------------------+-----------------------------------+

Finalise enrolment check

Table 4:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identify debt, rate reduction or  |
|                                   | cancellation has occurred + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Has the enrolment check or        |
|                                   | assessment resulted in:           |
|                                   |                                   |
|                                   | a debt,                           |
|                                   |                                   |
|                                   | rate reduction, or                |
|                                   |                                   |
|                                   | cancellation of payment?          |
|                                   |                                   |
|                                   | -   Yes, note the debt period and |
|                                   |     amount. Raise the debt for    |
|                                   |     the student. Verbally advise  |
|                                   |     the applicant of the          |
|                                   |     unfavourable decision. Go to  |
|                                   |     Step 2                        |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Finalise the enrolment check +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Written advice must be given to   |
|                                   | the applicant when contact has    |
|                                   | been made, but the applicant has  |
|                                   | not:                              |
|                                   |                                   |
|                                   | -   been told of the intervention |
|                                   |     finalisation, or              |
|                                   | -   had any debt advice issued,   |
|                                   |     where appropriate             |
|                                   |                                   |
|                                   | Issue a Q999 letter to the        |
|                                   | applicant and, if relevant, the   |
|                                   | nominee, to say the enrolment     |
|                                   | check is now complete.            |
|                                   |                                   |
|                                   | Finalise or cancel the review, as |
|                                   | appropriate. For:                 |
|                                   |                                   |
|                                   | electronic enrolment checks, see  |
|                                   | Finalising and attributing        |
|                                   | actions to a Service Update or    |
|                                   | Review in the Integrated Review   |
|                                   | System (IRS)                      |
|                                   |                                   |
|                                   | paper enrolment checks, have all  |
|                                   | student records identified on the |
|                                   | paper enrolment check been        |
|                                   | actioned?                         |
|                                   |                                   |
|                                   | -   Yes, mark the scanned         |
|                                   |     document as Completed         |
|                                   | -   No, repeat steps for all      |
|                                   |     relevant student records      |
|                                   |     until all records are         |
|                                   |     actioned                      |
