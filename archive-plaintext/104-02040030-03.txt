This page contains supplementary information for Complaints Officers in
Centrelink and Medicare managing Level 2 complaints received directly
from internal and external sources.

Investigate complaint from the Ombudsman, the Minister's Office, Members of Parliament and received via executive correspondence

The information provided in the section is supplementary to the
complaints management processes in Level 2 complaints (CLK)(MED). Use
the drop-down menu below to locate additional instructions for managing
complaints from specific internal and external sources before returning
to Level 2 complaints (CLK)(MED).

Customer Liaison Officer (CLO)

This table provides supplementary information for Complaints Officers
when investigating and responding to complaints received via a Customer
Liaison Officer (CLO).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer Liaison Officer (CLO) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The CLO is responsible for        |
|                                   | receiving and managing enquiries  |
|                                   | and feedback on behalf of         |
|                                   | Services Australia from the       |
|                                   | Minister's Office and other       |
|                                   | related business areas. If the    |
|                                   | CLO determines the issue is best  |
|                                   | placed with another business area |
|                                   | for response, the issue will be   |
|                                   | forwarded and responses are       |
|                                   | required on the day of receipt or |
|                                   | within 24 hours if received in    |
|                                   | the afternoon.                    |
|                                   |                                   |
|                                   | The CLO identifies and forwards   |
|                                   | complaints requiring response     |
|                                   | from the Level 2 team to the      |
|                                   | appropriate positional mailbox.   |
|                                   | The complaint is allocated to a   |
|                                   | Complaints Officer.               |
|                                   |                                   |
|                                   | To manage the complaint:          |
|                                   |                                   |
|                                   | -   immediately discuss any       |
|                                   |     concerns about the            |
|                                   |     vulnerability of the customer |
|                                   |     with the Team Leader or       |
|                                   |     Manager                       |
|                                   | -   create a Customer Feedback    |
|                                   |     Tool entry and include all    |
|                                   |     details of the feedback.      |
|                                   |     Update and maintain this      |
|                                   |     complaint record during the   |
|                                   |     management of the complaint   |
|                                   | -   investigate and finalise the  |
|                                   |     complaint                     |
|                                   | -   notify the customer of the    |
|                                   |     outcome. If unable to contact |
|                                   |     the customer by phone, issue  |
|                                   |     a finalisation letter         |
|                                   | -   finalise the Customer         |
|                                   |     Feedback Tool entry,          |
|                                   |     recording all details of the  |
|                                   |     actions taken, information    |
|                                   |     provided and complaint remedy |
|                                   | -   update the CLO note on the    |
|                                   |     customer's record with the    |
|                                   |     contact and action taken      |
|                                   |                                   |
|                                   | Return to Level 2 complaints      |
|                                   | (CLK)(MED) to continue managing   |
|                                   | the complaint.                    |

Departmental Liaison Officer (DLO)

This table provides supplementary information for Complaints Officers
when investigating and responding to complaints received via the
Departmental Liaison Officer (DLO).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Departmental Liaison Officer      |
|                                   | (DLO) + Read more ...             |
|                                   |                                   |
|                                   | Note: this procedure relates only |
|                                   | to direct customer contact to the |
|                                   | Minister's office.                |
|                                   |                                   |
|                                   | When a customer contacts the      |
|                                   | Minister's Office, the DLO can    |
|                                   | email a request for follow up     |
|                                   | action to the Level 2 team or via |
|                                   | a warm telephony transfer. As the |
|                                   | DLO has had direct contact from   |
|                                   | the customer, they have implied   |
|                                   | authority to follow up on behalf  |
|                                   | of the customer.                  |
|                                   |                                   |
|                                   | Once received by the Level 2      |
|                                   | Team, the request is allocated to |
|                                   | a Complaints Officer. These       |
|                                   | requests require urgent action    |
|                                   | and the customer must be          |
|                                   | contacted on the day of receipt   |
|                                   | or within 24 hours if received in |
|                                   | the afternoon.                    |
|                                   |                                   |
|                                   | To manage the complaint:          |
|                                   |                                   |
|                                   | -   create a Customer Feedback    |
|                                   |     Tool entry and include all    |
|                                   |     details of the feedback.      |
|                                   |     Update and maintain this      |
|                                   |     complaint record during the   |
|                                   |     management of the complaint   |
|                                   | -   fully investigate the         |
|                                   |     customer's record and resolve |
|                                   |     issues as required            |
|                                   | -   contact the customer by phone |
|                                   |     on day of receipt to          |
|                                   |     acknowledge the               |
|                                   |     correspondence and respond to |
|                                   |     the concerns raised           |
|                                   | -   provide confirmation by email |
|                                   |     to the DLO that the customer  |
|                                   |     issue has been addressed      |
|                                   |                                   |
|                                   | Return to Level 2 complaints      |
|                                   | (CLK)(MED) to continue managing   |
|                                   | the complaint.                    |

|                                   | Media feedback is received by the |

|                                   | Communication Division or Digital |
|                                   | Media Section and is allocated to |
|                                   | a Complaints Officer for          |
|                                   | management.                       |

|                                   | Media protocols.                  |
|                                   |                                   |
|                                   | When the Communication Division   |
|                                   | or Digital Media Section request  |
|                                   | an immediate response:            |
|                                   |                                   |
|                                   | -   the Complaints Officer        |
|                                   |     contacts the Complaints       |
|                                   |     Officer's manager and         |
|                                   |     provides a verbal response    |
|                                   | -   the manager escalates the     |
|                                   |     response                      |
|                                   | -   the Complaints Officer        |
|                                   |     follows the processes below   |
|                                   |     to manage the complaint,      |

|                                   | profile or significant service    |

|                                   |                                   |

|                                   | example co-location, emergency    |
|                                   | management                        |
|                                   |                                   |
|                                   | Sensitives issues including:      |
|                                   |                                   |
|                                   | -   customer vulnerability        |
|                                   |     (family violence, critical    |
|                                   |     health issues, homelessness)  |
|                                   | -   current media attention       |
|                                   | -   poor management that may      |
|                                   |     affect the agency's           |
|                                   |     reputation, and               |
|                                   | -   significant privacy or        |
|                                   |     security incidents, such as   |
|                                   |     complaints about a Senior     |
|                                   |     Executive staff member or a   |
|                                   |     significant staff code of     |
|                                   |     conduct matter                |
|                                   |                                   |
|                                   | an allegation of staff fraud      |
|                                   |                                   |
|                                   | To manage the complaint:          |
|                                   |                                   |
|                                   | immediately discuss any concerns  |
|                                   | about the vulnerability of the    |
|                                   | customer with the team leader or  |
|                                   | manager                           |
|                                   |                                   |
|                                   | create a Customer Feedback Tool   |
|                                   | entry and include all details of  |
|                                   | the feedback. Update and maintain |
|                                   | this complaint record during the  |
|                                   | management of the complaint       |
|                                   |                                   |
|                                   | contact the customer:             |
|                                   |                                   |
|                                   | -   on the same day the media     |
|                                   |     feedback email is received in |
|                                   |     cases of immediate or swift   |
|                                   |     response, or                  |
|                                   | -   within 24 hours of the media  |
|                                   |     feedback email being          |
|                                   |     allocated to acknowledge the  |
|                                   |     feedback                      |
|                                   |                                   |
|                                   | investigate and finalise the      |
|                                   | complaint                         |
|                                   |                                   |
|                                   | notify the customer of the        |
|                                   | outcome of the investigation. If  |
|                                   | unable to contact the customer by |
|                                   | phone, issue a finalisation       |
|                                   | letter                            |
|                                   |                                   |
|                                   | finalise the Customer Feedback    |
|                                   | Tool recording all details of the |
|                                   | actions taken, information        |
|                                   | provided and complaint remedy     |
|                                   |                                   |
|                                   | record all information in the     |
|                                   | response email template and email |
|                                   | to either the Communication       |
|                                   | Division or Digital Media Section |
|                                   | (whichever allocated the          |
|                                   | complaint)                        |
|                                   |                                   |
|                                   | Go to Step 3                      |

Ministerial feedback

This table provides supplementary information for Complaints Officers
when investigating and responding to complaints received via Ministerial
feedback.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Ministerial feedback + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Ministerial correspondence refers |
|                                   | to letters or emails sent to the  |
|                                   | Minister about Services           |
|                                   | Australia. The correspondence can |
|                                   | come from a variety of sources,   |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   Members of Parliament -       |
|                                   |     Federal and State             |
|                                   | -   organisations representing    |
|                                   |     customers or the community    |
|                                   | -   the general public, including |
|                                   |     constituents of Ministers     |
|                                   |                                   |
|                                   | The Minister's Office may receive |
|                                   | contact from a customer (response |
|                                   | is via a Departmental             |
|                                   | Correspondence Brief) or from a   |
|                                   | local Member of Parliament or     |
|                                   | Senator (response is via a        |
|                                   | Ministerial Brief). The           |
|                                   | Ministerial Correspondence Team   |
|                                   | sends a package of documents to   |
|                                   | be completed, including a         |
|                                   | clearance sheet, brief etc.       |
|                                   |                                   |
|                                   | Ministerial correspondence brief  |
|                                   | is received by the Level 2 team   |
|                                   | via email from the Ministerial    |
|                                   | Correspondence Team. Ministerial  |
|                                   | correspondence is allocated to a  |
|                                   | Complaints Officer.               |
|                                   |                                   |
|                                   | The Complaints Officer:           |
|                                   |                                   |
|                                   | creates a Customer Feedback Tool  |
|                                   | entry                             |
|                                   |                                   |
|                                   | contacts the customer where       |
|                                   | possible and provides a full      |
|                                   | explanation of the situation and  |
|                                   | apology if appropriate            |
|                                   |                                   |
|                                   | prepares a Draft Correspondence   |
|                                   | Brief and Correspondence Summary  |
|                                   | Sheet for the Ministerial         |
|                                   | Correspondence Team, making sure: |
|                                   |                                   |
|                                   | -   the Draft Correspondence      |
|                                   |     Brief and Correspondence      |
|                                   |     Summary Sheet addresses all   |
|                                   |     issues raised in the          |
|                                   |     correspondence                |
|                                   | -   the clearance sheet is        |
|                                   |     completed, if required,       |
|                                   |     including relevant background |
|                                   |     to issues without repeating   |
|                                   |     history outlined in the       |
|                                   |     letter                        |
|                                   | -   there is no assumed prior     |
|                                   |     knowledge (avoid use of       |
|                                   |     jargon and acronyms)          |
|                                   | -   correct names, phone numbers, |
|                                   |     dates etc. are provided       |
|                                   | -   references to relevant        |
|                                   |     legislation are made (if      |
|                                   |     applicable), and              |
|                                   | -   copies of relevant letters to |
|                                   |     the customer are provided (if |
|                                   |     appropriate)                  |
|                                   |                                   |
|                                   | returns the Draft Correspondence  |
|                                   | Brief and Correspondence Summary  |
|                                   | Sheet to the Ministerial          |
|                                   | Correspondence Team on completion |
|                                   | of a quality assurance check      |
|                                   |                                   |
|                                   | The Ministerial Correspondence    |
|                                   | Team are responsible for          |
|                                   | completing the Correspondence     |
|                                   | Brief, preparing a letter for the |
|                                   | Minister's signature and          |
|                                   | obtaining clearance and approvals |
|                                   | from appropriate clearance        |
|                                   | officers.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Departmental Brief (customer      |
|                                   | complaint) + Read more ...        |
|                                   |                                   |
|                                   | In all cases attempt to resolve   |
|                                   | the customer's concerns by phone  |
|                                   | contact. If phone contact is      |
|                                   | successful and the customer's     |
|                                   | issues are resolved, a formal     |
|                                   | written response is not required  |
|                                   | unless the customer requests      |
|                                   | this.                             |
|                                   |                                   |
|                                   | Where the customer's concerns     |
|                                   | have been resolved through phone  |
|                                   | contact respond to the            |
|                                   | departmental request by email     |
|                                   | using the approved template.      |
|                                   |                                   |
|                                   | Where the customer requests a     |
|                                   | written response or phone contact |
|                                   | was not successful, complete the  |
|                                   | Draft Correspondence Brief and    |
|                                   | Correspondence Summary Sheet,     |
|                                   | including enough detail to        |
|                                   | support the ministerial           |
|                                   | correspondence team in preparing  |
|                                   | the written response.             |
|                                   |                                   |
|                                   | The proposed response should      |
|                                   | provide enough details to explain |
|                                   | the circumstances surrounding the |
|                                   | complaint, identify any issues    |
|                                   | and concerns and actions taken.   |
|                                   |                                   |
|                                   | Go to Step 4                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Ministerial Correspondence        |
|                                   | Briefing (MP or Senator           |

Secretarial and executive correspondence

This table provides supplementary information for Complaints Officers
when investigating and responding to complaints received via Secretarial
or executive correspondence.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Secretarial and executive         |
|                                   | response + Read more ...          |
|                                   |                                   |
|                                   | A secretarial brief request       |
|                                   | results from direct contact with  |
|                                   | the Secretary of the agency or    |
|                                   | another senior executive          |
|                                   | correspondence. These are         |
|                                   | allocated to a Complaints         |
|                                   | Officer. The Secretary's office   |
|                                   | sends a package of documents to   |
|                                   | be completed, including a         |
|                                   | clearance sheet, brief etc.       |
|                                   |                                   |
|                                   | In all cases a formal written     |
|                                   | response to the correspondent is  |
|                                   | required.                         |
|                                   |                                   |
|                                   | To manage the complaint:          |
|                                   |                                   |
|                                   | -   immediately discuss any       |
|                                   |     concerns about the            |
|                                   |     vulnerability of the customer |
|                                   |     with the team leader or       |
|                                   |     manager                       |
|                                   | -   create a record in the        |
|                                   |     Customer Feedback Tool        |
|                                   | -   fully investigate the         |
|                                   |     customer's record and resolve |
|                                   |     issues as required            |
|                                   | -   phone the customer to         |
|                                   |     acknowledge the               |
|                                   |     correspondence and respond to |
|                                   |     the concerns raised           |
|                                   | -   draft a letter of response to |
|                                   |     the customer addressing the   |
|                                   |     concerns raised and action    |
|                                   |     taken                         |
|                                   | -   complete the clearance sheet  |
|                                   |     with a summary of the issues  |
|                                   |     raised, detail of the         |
|                                   |     customer's circumstances,     |
|                                   |     action taken, result of       |
|                                   |     contact with the customer and |
|                                   |     a summary of the points       |
|                                   |     addressed in the written      |
|                                   |     response                      |
|                                   |                                   |
|                                   | Return to Level 2 complaints      |
|                                   | (CLK)(MED) to continue managing   |
|                                   | the complaint.                    |

Complaints relating to ATO issues

This table provides supplementary information for Complaints Officers
when investigating and responding to complaints received via Ministerial
feedback.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complaints relating to ATO        |
|                                   | issues + Read more ...            |
|                                   |                                   |
|                                   | Complaints are escalated to Level |
|                                   | 2 when they are about Australian  |
|                                   | Taxation Office (ATO) issues or   |
|                                   | there is a need to collaborate    |
|                                   | with the ATO about a complaint    |
|                                   | for the benefit of mutual         |
|                                   | customers. When the ATO issue     |
|                                   | cannot be resolved                |
|                                   | administratively the complaint is |
|                                   | escalated by:                     |
|                                   |                                   |
|                                   | -   Level 1 staff, under          |
|                                   |     escalation reason 5 -         |
|                                   |     Complex, cross program or     |
|                                   |     multiple issues               |
|                                   |                                   |
|                                   | Liaison with the ATO is managed   |
|                                   | by the escalated complaints team. |
|                                   | This ensures customers are given  |
|                                   | correct and consistent advice,    |
|                                   | administrative options are        |
|                                   | considered and customers are only |
|                                   | required to liaise with one       |
|                                   | agency to have their issues       |
|                                   | resolved.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Explore administrative options +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Complaint Officers must attempt   |
|                                   | to resolve an ATO complaint using |
|                                   | administrative options, current   |
|                                   | complaint handling procedures and |
|                                   | protocols and by seeking          |
|                                   | technical advice where            |
|                                   | appropriate.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | How to refer a complaint for ATO  |
|                                   | liaison + Read more ...           |
|                                   |                                   |
|                                   | Level 2 Officers request contact  |
|                                   | with the ATO by forwarding the    |
|                                   | DHS/ATO Complaint Support         |
|                                   | Template to ATO liaison           |
|                                   | complaints.                       |
|                                   |                                   |
|                                   | The Memorandum of Understanding   |
|                                   | (MOU) between Services Australia  |
|                                   | and the ATO makes provision for   |
|                                   | urgent enquiries, including       |
|                                   | sensitive complaints from         |
|                                   | vulnerable customers. If a        |
|                                   | request is urgent the Level 2     |
|                                   | Officer will forward the template |
|                                   | and phone the Complaints Program  |
|                                   | Manager to request a more         |
|                                   | immediate response. The           |
|                                   | Complaints Program Manager will   |
|                                   | liaise directly with the ATO      |
|                                   | liaison officer if required.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Approval required before ATO      |
|                                   | contact + Read more ...           |
|                                   |                                   |
|                                   | If unable to resolve the          |
|                                   | complaint administratively, the   |
|                                   | Level 2 Officer must obtain       |
|                                   | written approval from their       |
|                                   | Service Support Manager (SSM) and |
|                                   | approval from the customer before |
|                                   | requesting information from the   |
|                                   | ATO.                              |
|                                   |                                   |
|                                   | To seek approval from the         |
|                                   | customer for Services Australia   |
|                                   | to contact the ATO:               |
|                                   |                                   |
|                                   | -   Advise the customer Services  |
|                                   |     Australia must contact the    |
|                                   |     ATO to try and resolve the    |
|                                   |     complaint                     |
|                                   | -   Ask the customer for their    |
|                                   |     permission for Services       |
|                                   |     Australia to liaise with the  |
|                                   |     ATO about the customer's      |
|                                   |     complaint                     |
|                                   | -   If the customer does not give |
|                                   |     permission for Services       |
|                                   |     Australia to liaise with the  |
|                                   |     ATO, advise the customer they |
|                                   |     will need to contact the ATO  |
|                                   |     themselves. The customer can  |
|                                   |     visit the ATO website for     |
|                                   |     more information              |
|                                   |                                   |
|                                   | If the customer gives permission  |
|                                   | for Services Australia to liaise  |
|                                   | with the ATO:                     |
|                                   |                                   |
|                                   | -   advise the customer that      |
|                                   |     liaison with the ATO does not |
|                                   |     guarantee their complaint     |
|                                   |     will be resolved              |
|                                   | -   advise the customer they do   |
|                                   |     not need to contact the ATO   |
|                                   |     and that the ATO will respond |
|                                   |     to the Services Australia     |
|                                   |     request for information       |
|                                   |     within 5 business days of     |
|                                   |     receiving the request         |
|                                   | -   advise the customer that      |
|                                   |     Services Australia intends to |
|                                   |     resolve complaints in 10      |
|                                   |     working days and will keep    |
|                                   |     them informed of any delays   |
|                                   | -   document in the CFT that the  |
|                                   |     customer has agreed for       |
|                                   |     Services Australia to contact |
|                                   |     the ATO                       |
