This page contains questions and answers about the homeless supplement,
a link to the Viability expansion component and homeless supplement
assessment form (AC024),a link to contact details for Aged Care, and a
link to the Department of Health website.

On this Page:

Homeless supplement frequently asked questions and answers

This table will help staff respond to questions about the homeless
supplement from aged care services.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | When did the homeless supplement  |
|                                   | come into effect?                 |
|                                   |                                   |
|                                   | From 1 October 2013, and was paid |
|                                   | from this date.                   |
|                                   |                                   |
|                                   | Note: system changes were         |
|                                   | implemented on 15 December 2013.  |
|                                   | The system back paid to 1 October |
|                                   | 2013, if the aged care home was   |
|                                   | eligible.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Who is eligible for the homeless  |
|                                   | supplement?                       |
|                                   |                                   |
|                                   | Aged care homes when all of the   |
|                                   | following criteria are met:       |
|                                   |                                   |
|                                   | -   They have registered with the |
|                                   |     Department of Health for the  |
|                                   |     homeless component of the     |
|                                   |     Viability Supplement          |
|                                   | -   They have greater than 50% of |
|                                   |     all residents meeting the low |
|                                   |     care needs, homeless and/or   |
|                                   |     Indigenous Australians        |
|                                   |     criterion, and                |
|                                   | -   They send the AC024 form to   |
|                                   |     Services Australia            |
+-----------------------------------+-----------------------------------+
| 3                                 | Is there an application form for  |
|                                   | the homeless supplement?          |
|                                   |                                   |
|                                   | Yes. Aged care homes registered   |
|                                   | with the Department of Health for |
|                                   | the homeless component of the     |
|                                   | Viability Supplement will need to |
|                                   | send the AC024 form to Services   |
|                                   | Australia. Once pages 3 and 4 of  |
|                                   | the AC024 form has been processed |
|                                   | by Services Australia, the aged   |
|                                   | care homes will automatically     |
|                                   | receive the homeless supplement   |
|                                   | in addition to the funding        |
|                                   | provided by the Viability         |
|                                   | Supplement.                       |
|                                   |                                   |
|                                   | See the Department of Health      |
|                                   | website for information about how |
|                                   | to apply for the viability        |
|                                   | supplement.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | How much is the homeless          |
|                                   | supplement?                       |
|                                   |                                   |
|                                   | The Department of Health’s        |
|                                   | website publishes current and     |
|                                   | historical rates of the homeless  |
|                                   | supplement.                       |
+-----------------------------------+-----------------------------------+
| 5                                 | Where to seek answers for system  |
|                                   | related queries                   |
|                                   |                                   |
|                                   | For any system related questions  |
|                                   | that require further              |
|                                   | investigation, Service Officers   |
|                                   | will need to ensure the           |
|                                   | escalation process is actioned    |
|                                   | appropriately prior to forwarding |
|                                   | queries to the Aged Care Support  |
|                                   | positional mailbox.               |
+-----------------------------------+-----------------------------------+
| 6                                 | Where can services find more      |
|                                   | information on the homeless       |
|                                   | supplement?                       |
|                                   |                                   |
|                                   | See the Department of Health      |
|                                   | website for more policy           |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
