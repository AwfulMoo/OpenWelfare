Business rules

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Rule                              |
+-----------------------------------+-----------------------------------+

|                                   | one or more accepted mental       |
|                                   | health condition/s are eligible   |
|                                   | for the CVC program               |

Frequently asked questions about CVC

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Question and Answer               |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: What is the CVC         |
|                                   | Program?                          |
|                                   |                                   |
|                                   | Answer: The Coordinated Veterans' |
|                                   | Care Program (CVC) is a health    |
|                                   | program designed to reduce        |
|                                   | unplanned hospitalisation. It is  |
|                                   | available to eligible Department  |
|                                   | of Veterans' Affairs (DVA) gold   |
|                                   | card holders who have chronic     |
|                                   | disease and complex care needs.   |
|                                   | From 1 July 2021, White Card      |
|                                   | holders with one or more accepted |
|                                   | mental health condition/s are     |
|                                   | eligible to participate.          |
|                                   |                                   |
|                                   | Provider enquiries about the      |
|                                   | Coordinated Veterans' Care        |
|                                   | Program, including patient        |
|                                   | eligibility, should be directed   |
|                                   | to the CVC Helpline (External     |
|                                   | contacts - VAP).                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Who provides the        |
|                                   | services in the CVC Program?      |
|                                   |                                   |
|                                   | Answer: General Practitioners     |
|                                   | (GPs), along with Practice Nurses |
|                                   | or DVA Contracted Community       |
|                                   | Nurses, work closely with         |
|                                   | participants to manage their      |
|                                   | health conditions, and to plan    |
|                                   | and coordinate their ongoing      |
|                                   | care.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How long can a          |
|                                   | participant remain in the         |
|                                   | program?                          |
|                                   |                                   |
|                                   | Answer: It is expected that most  |
|                                   | participants who are enrolled in  |
|                                   | the CVC program will stay on the  |
|                                   | program indefinitely.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: What if the participant |
|                                   | passes away during the program?   |
|                                   |                                   |
|                                   | Answer: If a participant passes   |
|                                   | away partway through a period of  |
|                                   | care, the provider is entitled to |
|                                   | claim the full amount for the     |
|                                   | quarterly care payment.           |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Can a participant       |
|                                   | transfer care during the program? |
|                                   |                                   |
|                                   | Answer: Although a participant in |
|                                   | the program can change providers  |
|                                   | there are rules that apply to the |
|                                   | transfer of care of the patient:  |
|                                   |                                   |
|                                   | GP to GP - Where a participant    |
|                                   | changes GPs after starting the    |
|                                   | program, the new GP:              |
|                                   |                                   |
|                                   | -   cannot claim the initial      |
|                                   |     payment (UP01 or UP02)        |
|                                   | -   can only claim for a          |
|                                   |     quarterly care payment (UP03  |
|                                   |     or UP04) where the quarter    |
|                                   |     commenced after the expiry of |
|                                   |     the previous GP’s quarterly   |
|                                   |     care period                   |
|                                   |                                   |
|                                   | Practice Nurse to Community Nurse |
|                                   | or Community Nurse to Practice    |
|                                   | Nurse:                            |
|                                   |                                   |
|                                   | -   a change from a Practice      |
|                                   |     Nurse to a Community Nurse,   |
|                                   |     or vice versa, can occur      |
|                                   |     after discussions with the GP |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Can UP items be claimed |
|                                   | at the same time as               |
|                                   | consultations, care plan items,   |
|                                   | health assessments or other MBS   |
|                                   | items?                            |
|                                   |                                   |
|                                   | Answer: Yes. The UP item benefits |
|                                   | are paid in addition to all       |
|                                   | existing MBS items such as        |
|                                   | consultations or chronic disease  |
|                                   | management items. Care provided   |
|                                   | under the CVC Program is not a    |
|                                   | replacement of the continued      |
|                                   | interaction between the GP and    |
|                                   | the patient.                      |
+-----------------------------------+-----------------------------------+
| 7                                 | Question: Is a VAP incentive item |
|                                   | payable in association with UP    |
|                                   | items?                            |
|                                   |                                   |
|                                   | Answer: No. VAP incentive items   |
|                                   | are not payable for any of the    |
|                                   | new CVC UP items. However, VAP    |
|                                   | incentive items will continue to  |
|                                   | be paid in association with LMO   |
|                                   | items that are claimed by a GP    |
|                                   | under the existing rules.         |
+-----------------------------------+-----------------------------------+
| 8                                 | Question: Is the provider         |
|                                   | required to indicate the accepted |
|                                   | mental health condition when      |
|                                   | submitting claims for a CVC item? |
|                                   |                                   |
|                                   | Answer: No. The claim can         |
|                                   | continue to be assessed based the |
|                                   | conditions listed on the veterans |
|                                   | file. If ‘all mental health       |
|                                   | conditions’ ‘treatment only’ or   |
|                                   | ‘provisional’ is noted on the     |
|                                   | clients file or next to the       |
|                                   | mental health condition this      |
|                                   | relates to treatment covered      |
|                                   | under NHLC and not CVC.           |
|                                   |                                   |
|                                   | Note: Refer to the list of        |
|                                   | accepted mental health conditions |
|                                   | under the CVC programme in        |
|                                   | section Eligibility to receive    |
|                                   | CVC services.                     |
+-----------------------------------+-----------------------------------+

Frequently asked questions about CVC Quarterly Care Payments

Table 4

+-----------------------------------+-----------------------------------+
| Item                              | Question and Answer               |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: What date of service    |
|                                   | should be used when claiming item |
|                                   | CVC Quarterly Care payments UP03? |
|                                   |                                   |
|                                   | Answer: The first day of the      |
|                                   | period of care must be used as    |
|                                   | the date of service for item      |
|                                   | UP03.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Why was item UP03       |
|                                   | rejected with RSN 759?            |
|                                   |                                   |
|                                   | Answer: UP03 must be submitted at |
|                                   | the end of the quarterly care     |
|                                   | period. If the claim is lodged    |
|                                   | before the end of this period,    |
|                                   | the item will be automatically    |
|                                   | rejected with RSN 759 'Item       |
|                                   | cannot be claimed until the last  |
|                                   | day of period of care'.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Do I need to claim      |
|                                   | health assessment or a care plan  |
|                                   | item/s prior to commencing the    |
|                                   | CVC program?                      |
|                                   |                                   |
|                                   | Answer: No, an MBS health         |
|                                   | assessment or a care plan item/s  |
|                                   | is not required to be present on  |
|                                   | the veteran’s history prior to    |
|                                   | claiming for CVC items. MBS and   |
|                                   | CVC items are not restrictive.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: What is the procedure   |
|                                   | if I have incorrectly claimed CVC |
|                                   | items UP01 and UP03 (with         |
|                                   | Practice Nurse) instead of items  |
|                                   | UP02 and UP04 (without Practice   |
|                                   | Nurse) respectively?              |
|                                   |                                   |
|                                   | Answer: If an overpayment has     |
|                                   | occurred, you are required to     |
|                                   | refund any overpaid benefit. The  |
|                                   | veteran's history will be         |
|                                   | adjusted to display the correct   |
|                                   | item/s, allowing correct          |
|                                   | processing of future claims.      |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Why was item UP06 -     |
|                                   | Community Nursing Subsequent Care |
|                                   | rejected with reason code 160     |
|                                   | 'Maximum number of services for   |
|                                   | this item already paid'?          |
|                                   |                                   |
|                                   | Answer: UP06 cannot be paid       |
|                                   | within 3 months of both items     |
|                                   | UP01 GP Initial payment with      |
|                                   | Practice Nurse or UP03 GP         |
|                                   | Quarterly Care payment with       |
|                                   | Practice Nurse. In addition, only |
|                                   | 1 item UP06 is payable every 28   |
|                                   | days.                             |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Can a GP/LMO provider   |
|                                   | claim a standard consultation     |
|                                   | item for each consult with a      |
|                                   | patient in relation to the CVC    |
|                                   | program?                          |
|                                   |                                   |
|                                   | Answer: The UP item benefits are  |
|                                   | paid in addition to all existing  |
|                                   | MBS items such as consultations.  |
