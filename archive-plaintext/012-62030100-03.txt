Manual claim for payment where prescription(s) or claim-related paperwork has been declared lost/missing by a Statutory Declaration

Table 1: This table describes how Service Officers process these manual
claims.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Review the claim + Read more ...  |
|                                   |                                   |
|                                   | Claims will be allocated to a     |
|                                   | Team Leader (APS6) who will       |
|                                   | assign the claim to a Service     |
|                                   | Officer for review.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine the action required +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | See the Resources page for        |
|                                   | example circumstances and         |
|                                   | suggested outcome.                |
|                                   |                                   |
|                                   | For complex scenarios, consult a  |
|                                   | Pharmaceutical Adviser            |
|                                   |                                   |
|                                   | If the claim for payment is:      |
|                                   |                                   |
|                                   | -   accepted, go to Step 3        |
|                                   | -   rejected, go to Step 5        |
|                                   | -   needs more information to be  |
|                                   |     processed, go to Step 6       |
+-----------------------------------+-----------------------------------+
| 3                                 | Claim accepted for payment + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Seek approval from a Team     |
|                                   |     Leader (APS6) to accept the   |
|                                   |     claim                         |
|                                   | -   Complete the ‘Your claim for  |
|                                   |     a prescription(S) is          |
|                                   |     accepted’ letter (Z2640). See |
|                                   |     the Resources page for a link |
|                                   |     to this letter                |
|                                   |                                   |
|                                   | Note: refer to a Pharmaceutical   |
|                                   | Adviser where free text has been  |
|                                   | used in the letter. A processing  |
|                                   | note must be added in PaNDA.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Process payment + Read more ...   |
|                                   |                                   |
|                                   | Process payment for the claim and |
|                                   | then go to Step 7.                |
+-----------------------------------+-----------------------------------+
| 5                                 | Claim is rejected for payment +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Seek approval from a Team     |
|                                   |     Leader (APS6) to reject the   |
|                                   |     claim                         |
|                                   | -   Complete the ‘Your claim for  |
|                                   |     a lost prescription/s cannot  |
|                                   |     be processed’ letter (Z2641). |
|                                   |     See the Resources page for a  |
|                                   |     link to this letter           |
|                                   |                                   |
|                                   | Note: refer to a Pharmaceutical   |
|                                   | Adviser where free text has been  |
|                                   | used in the letter. A processing  |
|                                   | note must be added in PaNDA.      |
+-----------------------------------+-----------------------------------+
| 6                                 | Pend claim for payment where more |
|                                   | information is needed + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Complete the ‘Your claim for  |
|                                   |     a lost prescription/s – more  |
|                                   |     information needed’ letter    |
|                                   |     (Z2642). See the Resources    |
|                                   |     page for a link to this       |
|                                   |     letter                        |
|                                   | -   Go to Step 7                  |
+-----------------------------------+-----------------------------------+
| 7                                 | Record in PaNDA + Read more ...   |
|                                   |                                   |
|                                   | In PaNDA:                         |
|                                   |                                   |
|                                   | -   record the outcome            |
|                                   | -   complete the work item        |

Perform a quality assurance (QA) check on free text used in a letter – Pharmaceutical Advisor

Table 2: This table describes how Pharmaceutical Advisors (as the agreed
delegate) perform a QA check for Service Officers on certain free text
letters. These outline the outcome for a manual claim for payment where
prescription(s) or claim-related paperwork has been declared
lost/missing by a Statutory Declaration.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Review the outcome of the         |
|                                   | assessment + Read more ...        |
|                                   |                                   |
|                                   | Review the outcome in             |
|                                   | consultation with the Service     |
|                                   | Officer.                          |
|                                   |                                   |
|                                   | -   If there is agreement on the  |
|                                   |     outcome of the assessment, go |
|                                   |     to Step 2                     |
|                                   | -   If there is disagreement on   |
|                                   |     the outcome of assessment,    |
|                                   |     refer to another              |
|                                   |     Pharmaceutical Adviser for    |
|                                   |     another opinion. Agree to an  |
|                                   |     outcome before progressing    |
|                                   |     any further                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Review the letter + Read more ... |
|                                   |                                   |
|                                   | Review the free text used in the  |
|                                   | letter for:                       |
|                                   |                                   |
|                                   | -   spelling and grammar,         |
|                                   | -   use of plain English, and     |
|                                   | -   accuracy of the information   |
|                                   |     provided                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Add a processing note/finalise +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Once satisfied that the letter is |
|                                   | error free, add a processing note |
|                                   | in PaNDA                          |
|                                   |                                   |
|                                   | Advise the Service Officer they   |
|                                   | can finalise the letter and       |
|                                   | complete the work item in PaNDA.  |
+-----------------------------------+-----------------------------------+
