| 1                                 | Determine if a terminating event  |

|                                   |                                   |
|                                   | A customer or a third party may   |
|                                   | advise of a terminating event     |
|                                   | verbally, electronically or in    |
|                                   | writing. The following            |
|                                   | terminating events will cause a   |
|                                   | case to end:                      |
|                                   |                                   |
|                                   | -   A child is deceased, see      |
|                                   |     Deceased Child Support child  |
|                                   | -   A child marries or becomes a  |
|                                   |     member of a couple            |
|                                   | -   A child is adopted            |
|                                   | -   A child turns 18 (or if the   |
|                                   |     assessment has been extended  |
|                                   |     beyond age 18 to the end of a |
|                                   |     school year, when the school  |
|                                   |     year or the child’s studies   |
|                                   |     finish)                       |
|                                   | -   The payee or payer is         |
|                                   |     deceased, see Deceased Child  |
|                                   |     Support customer management   |
|                                   | -   Both parents of a child cease |
|                                   |     to be eligible carers and     |
|                                   |     there are no non-parent       |
|                                   |     carers entitled to be paid    |
|                                   |     child support in relation to  |
|                                   |     the child. See Change in      |
|                                   |     child support care levels     |
|                                   |     (post 1 July 2010) for        |
|                                   |     examples of a change in care  |
|                                   |     that could be classed as a    |
|                                   |     terminating event             |
|                                   | -   All persons who were eligible |
|                                   |     carers of the child cease to  |
|                                   |     be eligible carers and a      |
|                                   |     parent who would have becomes |
|                                   |     an eligible carer does not    |
|                                   |     because Child Support was not |
|                                   |     advised of the change in care |
|                                   |     within 26 weeks               |
|                                   | -   A child becomes covered by    |
|                                   |     child welfare law in SA or WA |
|                                   | -   A child is not present in     |
|                                   |     Australia, is not an          |
|                                   |     Australian citizen and is not |
|                                   |     ordinarily a resident of      |
|                                   |     Australia and the payee is    |
|                                   |     not a resident of a           |
|                                   |     reciprocating jurisdiction    |
|                                   | -   A child’s parents become      |
|                                   |     members of the same couple    |

|                                   |     pay child support would cease |

|                                   |     resident of Australia (and is |

|                                   |     international maintenance     |

|                                   |     jurisdiction and does not     |

|                                   |     child is a member of a couple |
|                                   |                                   |

|                                   | adoption order made by a court    |

Review and process terminating event

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Review case + Read more ...       |
|                                   |                                   |
|                                   | Check the case for any issues     |
|                                   | that are not finalised, e.g. Non  |
|                                   | Agency Payments (NAPS), estimates |
|                                   | and agreements and then process   |
|                                   | accordingly. If any of the        |
|                                   | assessments are based either on a |
|                                   | provisional income or an estimate |
|                                   | of income, update the provisional |
|                                   | incomes and review any estimates  |
|                                   | of income where possible.         |
|                                   |                                   |
|                                   | Advise the customer that          |
|                                   | regardless of the fact that the   |
|                                   | case is ending, the provisional   |
|                                   | income will be replaced when a    |
|                                   | taxable income becomes available  |
|                                   | or Child Support may              |
|                                   | review/reconcile an estimate.     |
|                                   | This may result in arrears or an  |
|                                   | overpayment. See Income           |
|                                   | Processing for Child Support      |
|                                   | customers.                        |
|                                   |                                   |
|                                   | Note: estimates are automatically |
|                                   | reconciled when customer’s income |
|                                   | tax assessment is issued for the  |
|                                   | relevant year. Encourage          |
|                                   | customers to lodge tax returns on |
|                                   | time each year because it means   |
|                                   | that their estimates will be      |
|                                   | reconciled promptly.              |
|                                   |                                   |
|                                   | Change of Assessment (COA)        |
|                                   | application progress              |
|                                   |                                   |
|                                   | If there is an application for a  |
|                                   | Change of Assessment (COA) in     |
|                                   | progress, check the date to see   |
|                                   | when the COA application was      |
|                                   | made.                             |
|                                   |                                   |
|                                   | If the COA was lodged before the  |
|                                   | date of notification of the       |
|                                   | terminating event, the COA        |
|                                   | application may still be valid    |
|                                   |                                   |
|                                   | The applicant will need to be     |
|                                   | asked if they wish to proceed     |
|                                   | with the COA, and explain they    |
|                                   | should discuss this with the      |
|                                   | Decision Maker (DM). Advise them  |
|                                   | the DM will need to make a        |
|                                   | decision on whether the COA       |
|                                   | application is valid/invalid, due |
|                                   | to the terminating event.         |
|                                   |                                   |
|                                   | Manage customer expectations      |
|                                   | around the timeframe for a        |
|                                   | decision on whether the COA can   |
|                                   | still proceed, including the      |
|                                   | response timeframe of 7 days for  |
|                                   | a Contracted DM and 2 days for an |
|                                   | Internal DM.                      |
|                                   |                                   |
|                                   | If the COA was lodged after the   |
|                                   | date of notification of the       |
|                                   | terminating event, the COA        |
|                                   | application may not be valid      |
|                                   |                                   |
|                                   | Explain to the customer, the DM   |
|                                   | will need to make a decision on   |
|                                   | whether the COA application is    |
|                                   | valid/invalid, due to the         |
|                                   | terminating event.                |
|                                   |                                   |
|                                   | Manage customer expectations      |
|                                   | around timeframe for a decision   |
|                                   | on whether the COA can still      |
|                                   | proceed, including the response   |
|                                   | timeframe of 7 days for a         |
|                                   | Contracted Decision Maker and 2   |
|                                   | days for an Internal Decision     |
|                                   | Maker.                            |
|                                   |                                   |
|                                   | Raise a COA Call Back Required    |
|                                   | intray (2 days) and route as      |
|                                   | appropriate to COA team. The      |
|                                   | intray alert the COA DM the       |
|                                   | terminating event decision has    |
|                                   | been finalised.                   |
|                                   |                                   |
|                                   | Note: this intray must be sent    |
|                                   | regardless of whether the         |
|                                   | terminating event was notified    |
|                                   | before or after the COA           |
|                                   | application was made. See Step 8  |
|                                   | in the Incomplete or ineligible   |
|                                   | table in Manage a change of       |
|                                   | assessment application (CS).      |
|                                   |                                   |
|                                   | Assessment is based on a child    |
|                                   | support agreement                 |
|                                   |                                   |
|                                   | If the assessment is based on a   |
|                                   | child support agreement, explain  |
|                                   | to the customer that if they      |
|                                   | re-apply for a child support      |
|                                   | assessment in the future, their   |
|                                   | agreement may still apply. This   |
|                                   | will depend on the type of        |
|                                   | terminating event, the terms of   |
|                                   | the agreement and the types of    |
|                                   | provisions in the agreement.      |
|                                   | Transfer the customer to the      |
|                                   | Agreements and Court Orders team  |
|                                   | to discuss the impact of the      |
|                                   | terminating event on the          |
|                                   | agreement.                        |
|                                   |                                   |
|                                   | Proceed to the appropriate        |
|                                   | processing step:                  |
|                                   |                                   |
|                                   | -   Go to Step 2 for child        |
|                                   |     terminating event             |
|                                   | -   Go to Step 3 for eligible     |
|                                   |     carer terminating event       |
|                                   | -   Go to Step 4 for residency    |
|                                   |     terminating event             |
|                                   | -   Go to Step 5 for dual         |
|                                   |     liability terminating event   |
|                                   | -   Go to Step 6 for suspension   |
|                                   |     of liability on               |
|                                   |     reconciliation terminating    |
|                                   |     event                         |
|                                   | -   Go to Step 7 for section 151  |
|                                   |     election                      |
|                                   | -   Go to Step 8 for agreement to |
|                                   |     end liability                 |
|                                   | -   Go to Step 9 for non-parent   |
|                                   |     carer cases                   |
|                                   |                                   |
|                                   | Document the reasons for any      |
|                                   | decision and reference the        |
|                                   | legislation used to make that     |
|                                   | decision.                         |
