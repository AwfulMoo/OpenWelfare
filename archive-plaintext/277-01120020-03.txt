|                                   | that is still their intention.    |
|                                   |                                   |
|                                   | If no-one is going to apply for   |
|                                   | letters of administration:        |
|                                   |                                   |
|                                   | -   opt out and discharge, go to  |
|                                   |     Step 3                        |
|                                   | -   non-pursuit, go to Step 6     |
|                                   |                                   |
|                                   | The arrears must be addressed     |
|                                   | before unlocking the customers.   |
|                                   |                                   |
|                                   | Note: administrators perform a    |
|                                   | similar role to that of an        |
|                                   | executor but distribute the       |
|                                   | assets of the deceased according  |
|                                   | to intestacy laws.                |
+-----------------------------------+-----------------------------------+
| 5                                 | Public trustee + Read more ...    |
|                                   |                                   |
|                                   | If the payer has significant      |
|                                   | assets their family may be able   |
|                                   | to approach the public trustee to |
|                                   | administer the estate. The        |
|                                   | requirements of the public        |
|                                   | trustee will vary from state to   |
|                                   | state, however generally the      |
|                                   | value of the assets must be       |
|                                   | reasonably significant and the    |
|                                   | public trustee charge according   |
|                                   | to the value of the estate.       |
|                                   |                                   |
|                                   | If the estate is taken on by the  |
|                                   | public trustee treat them in the  |
|                                   | same manner as an executor and    |
|                                   | follow the process detailed in    |
|                                   | Step 3 in Table 3.                |
|                                   |                                   |
|                                   | Note: Child Support cannot refer  |
|                                   | a deceased customer’s estate to   |
|                                   | the public trustee.               |
|                                   |                                   |
|                                   | Different legislation applies to  |
|                                   | Indigenous estates. If the        |
|                                   | customer is Indigenous contact    |
|                                   | the relevant public trustee to    |
|                                   | determine their requirements for  |
|                                   | accepting a referral to           |
|                                   | administer the estate.            |
|                                   |                                   |
|                                   | If the payer’s estate is not      |

|                                   | Support (Registration and         |
|                                   | Collection) Act 1988 if:          |

|                                   |                                   |
|                                   | See Obsolete Child Support        |

| 8                                 | No arrears on case - other        |
|                                   | issues + Read more ...            |
|                                   |                                   |

|                                   | Overpayment                       |

|                                   | overpayment in the system using   |
