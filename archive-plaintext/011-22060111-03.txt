Adding or amending Care of a child Indicators

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access the record in CDMS + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | On the CDMS home page:            |
|                                   |                                   |
|                                   | -   select Amend Consumer Details |
|                                   | -   add the Medicare card number  |
|                                   |     in ID Number                  |
|                                   | -   select Personal               |
+-----------------------------------+-----------------------------------+
| 2                                 | Care of a child category + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In Consumers, select the radio    |
|                                   | button against the child or young |
|                                   | person.                           |
|                                   |                                   |
|                                   | Select the Sensitive Information  |
|                                   | tab.                              |
|                                   |                                   |
|                                   | To add an indicator, select:      |
|                                   |                                   |
|                                   | -   Care of a child from the      |
|                                   |     Category dropdown menu        |
|                                   | -   Care of a child from the      |
|                                   |     Indicator dropdown menu       |
|                                   |                                   |
|                                   | To amend an indicator:            |
|                                   |                                   |
|                                   | -   select the radio button       |
|                                   |     against the indicator         |
|                                   | -   review the details that       |
|                                   |     populate in the Add / Amend   |
|                                   |     Sensitive Information         |
|                                   |     Indicators fields             |
+-----------------------------------+-----------------------------------+
| 3                                 | Details field + Read more ...     |
|                                   |                                   |
|                                   | The Details field describes the   |
|                                   | care arrangements for the child   |
|                                   | or young person.                  |
|                                   |                                   |
|                                   | Select from the Details dropdown  |
|                                   | menu:                             |
|                                   |                                   |
|                                   | -   Formal care, if the child or  |
|                                   |     young person is placed under  |
|                                   |     the parental responsibility   |
|                                   |     of the state or territory     |
|                                   |     Minister for Child Protection |
|                                   | -   Informal care, if there is a  |
|                                   |     private arrangement between a |
|                                   |     parent and third party,       |
|                                   |     without a court order         |
|                                   | -   Statutory kinship care, if a  |
|                                   |     child protection intervention |
|                                   |     has occurred and the child or |
|                                   |     young person is in the care   |
|                                   |     of a family or community      |
|                                   |     member                        |
|                                   | -   Pre-adoption, if the child or |
|                                   |     young person is enrolled in   |
|                                   |     Medicare prior to the         |
|                                   |     finalisation of their         |
|                                   |     adoption                      |
|                                   | -   Surrogacy, if the birth       |
|                                   |     certificate or parentage      |
|                                   |     order has not been finalised  |
|                                   | -   Voluntary care, if the child  |
|                                   |     or young person is placed in  |
|                                   |     temporary care under a        |
|                                   |     ‘voluntary care agreement’    |
|                                   | -   Removed from card by National |
|                                   |     Office (for National Office   |
|                                   |     use only)                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Notification Channel + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select the option that best       |
|                                   | describes how the information was |
|                                   | received.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Specialist Referral + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Specialist Referral field     |
|                                   | helps alert other staff to any    |
|                                   | referrals offered, or made at the |
|                                   | point of service.                 |
|                                   |                                   |
|                                   | Select from the Specialist        |
|                                   | Referral dropdown menu:           |
|                                   |                                   |
|                                   | -   Already supported             |
|                                   | -   Declined offer                |
|                                   | -   External, for example Kids    |
|                                   |     Helpline or 1800RESPECT       |
|                                   | -   Internal, for example an      |
|                                   |     Indigenous Service Officer    |
|                                   |     (ISO) or a social worker      |
|                                   | -   Internal and external         |
|                                   | -   Not applicable                |
+-----------------------------------+-----------------------------------+
| 6                                 | Start Date + Read more ...        |
|                                   |                                   |
|                                   | The Start Date field is optional. |
|                                   |                                   |
|                                   | If the start date of the care     |
|                                   | arrangement is known, add this    |
|                                   | date in the Start Date field.     |
+-----------------------------------+-----------------------------------+
| 7                                 | Document field + Read more ...    |
|                                   |                                   |
|                                   | The Document field describes      |
|                                   | supporting documents provided by  |
|                                   | the customer, if any.             |
|                                   |                                   |
|                                   | Select from the Document dropdown |
|                                   | menu:                             |
|                                   |                                   |
|                                   | -   Centrelink document           |
|                                   | -   Court Order                   |
|                                   | -   Family Violence or Protection |
|                                   |     Order                         |
|                                   | -   Letter                        |
|                                   | -   Specialist advice             |
|                                   | -   Statutory declaration         |
|                                   | -   Not applicable                |
+-----------------------------------+-----------------------------------+
| 8                                 | Comment field + Read more ...     |
|                                   |                                   |
|                                   | Only use the Comment field to:    |
|                                   |                                   |
|                                   | -   record the Child Protection   |
|                                   |     Agency, if the child is in    |
|                                   |     formal care                   |
|                                   | -   record a document reference   |
|                                   |     number, if provided           |
|                                   | -   refer Service Officers to     |
|                                   |     additional information in the |
|                                   |     Comments tab                  |
|                                   |                                   |
|                                   | For example, DHHS VIC File:       |
|                                   | AB0000001234.                     |
|                                   |                                   |
|                                   | Text added to this field:         |
|                                   |                                   |
|                                   | -   is limited to 80 characters   |
|                                   | -   appears in the Sensitive      |
|                                   |     Information pop up table      |
|                                   | -   can be released under a       |
|                                   |     Freedom of Information (FOI)  |
|                                   |     request                       |
+-----------------------------------+-----------------------------------+
| 9                                 | End Date + Read more ...          |
|                                   |                                   |
|                                   | In the End Date field, add        |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   the care arrangement review   |
|                                   |     date or end date, if known    |
|                                   | -   the date the child or young   |
|                                   |     person turns 18 years         |
|                                   |                                   |
|                                   | Once the end date is reached, the |
|                                   | indicator:                        |
|                                   |                                   |
|                                   | -   will no longer trigger the    |
|                                   |     Sensitive Information alert   |
|                                   | -   will not be displayed in the  |
|                                   |     Sensitive Information pop up  |
|                                   |     table                         |
|                                   | -   can only be viewed in the     |
|                                   |     Sensitive Information tab in  |
|                                   |     Inquire                       |
+-----------------------------------+-----------------------------------+
| 10                                | End Date reason + Read more ...   |
|                                   |                                   |
|                                   | This field is mandatory if a date |
|                                   | is keyed in the End Date field.   |
|                                   |                                   |
|                                   | If documents have been provided,  |
|                                   | select:                           |
|                                   |                                   |
|                                   | -   Court adjournment date, if    |
|                                   |     provided on an interim Court  |
|                                   |     Order                         |
|                                   | -   Court order end date, if      |
|                                   |     provided on a Court Order     |
|                                   | -   Document end date, if         |
|                                   |     provided on a document other  |
|                                   |     than a Court Order            |
|                                   |                                   |
|                                   | If no documents have been         |
|                                   | provided, select:                 |
|                                   |                                   |
|                                   | -   End of arrangement, if the    |
|                                   |     arrangement end date is known |
|                                   | -   No longer required, for any   |
|                                   |     other reason                  |
+-----------------------------------+-----------------------------------+
| 11                                | Confirm the details + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Validate, Validate, then   |
|                                   | Update to confirm the details.    |
|                                   |                                   |
|                                   | Note: if the child or young       |
|                                   | person has safety concerns,       |
|                                   | consider adding a Safety          |
|                                   | Indicator for Family and domestic |
|                                   | violence. See Step 3 in Table 6.  |
