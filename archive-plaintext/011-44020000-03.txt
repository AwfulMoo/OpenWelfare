Appeals to a visa refusal

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Appeals to a visa refusal + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A customer may have their visa    |
|                                   | application refused by the        |
|                                   | Department of Home Affairs. They  |
|                                   | can appeal this decision with the |
|                                   | Administrative Appeals Tribunal   |
|                                   | (AAT).                            |
|                                   |                                   |
|                                   | They are eligible for Medicare    |
|                                   | while appealing the decision.     |
|                                   |                                   |
|                                   | Has the customer lodged an appeal |
|                                   | with the AAT for their visa       |
|                                   | refusal?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the customer is           |
|                                   |     ineligible for Medicare       |
|                                   |     entitlements. Procedure ends  |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Documents required + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | To re-enrol, customers will need  |
|                                   | to provide:                       |
|                                   |                                   |
|                                   | proof of an appeal. They must     |
|                                   | show one of the below:            |
|                                   |                                   |
|                                   | -   Administrative Appeals        |
|                                   |     Tribunal (AAT) document       |
|                                   | -   Federal Court or High Court   |
|                                   |     document                      |
|                                   | -   Ministerial Intervention      |
|                                   |     document or Visa Entitlement  |
|                                   |     Verification Online (VEVO) to |
|                                   |     confirm Ministerial           |
|                                   |     Intervention has begun        |
|                                   |     Note: Step 3 has the proof of |
|                                   |     an appeal requirements        |
|                                   |                                   |
|                                   | a valid visa with permission to   |
|                                   | work via one of the below:        |
|                                   |                                   |
|                                   | -   in their passport             |
|                                   | -   in the form of a travel       |
|                                   |     document                      |
|                                   | -   an email or a letter from the |
|                                   |     Department of Home Affairs    |
|                                   |                                   |
|                                   | an identification document via    |
|                                   | one of the below:                 |
|                                   |                                   |
|                                   | -   passport                      |
|                                   | -   travel document               |
|                                   | -   Immicard                      |
|                                   | -   Current Australian driver’s   |
|                                   |     licence (only if the          |
|                                   |     passport, travel document or  |
|                                   |     Immicard was previously       |
|                                   |     recorded in CDMS as the       |
|                                   |     initial enrolment             |
|                                   |     identification)               |
|                                   |                                   |
|                                   | Service Officers must do a VEVO   |
|                                   | check. Customers must hold a      |
|                                   | bridging visa with work rights.   |
|                                   | If the customer does not have     |
|                                   | work rights, they must supply     |
|                                   | relationship documents to be      |
|                                   | eligible for Medicare.            |
|                                   |                                   |
|                                   | Expired foreign passport          |
|                                   |                                   |
|                                   | To extend a Medicare card when a  |
|                                   | visa is refused, the agency can   |
|                                   | accept an expired foreign         |
|                                   | passport up to a maximum of 2     |
|                                   | years from the expiry date        |
|                                   | providing this passport number:   |
|                                   |                                   |
|                                   | -   was recorded in CDMS at the   |
|                                   |     initial enrolment, and        |
|                                   | -   can be verified in VEVO when  |
|                                   |     they present                  |
|                                   |                                   |
|                                   | Note: an expired Immi Card is     |
|                                   | still a valid identity document.  |
|                                   | Do not accept an expired Immi     |
|                                   | Card as evidence of identity for  |
|                                   | initial enrolments.               |
