| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Appeals to a visa refusal + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When the Department of Home       |
|                                   | Affairs refuse an application for |
|                                   | permanent residency, the customer |
|                                   | will get a notice telling them of |
|                                   | the decision.                     |
|                                   |                                   |
|                                   | Medicare will cease their         |
|                                   | entitlement. The system will send |
|                                   | a letter telling them about the   |
|                                   | refusal.                          |
|                                   |                                   |
|                                   | Customers can appeal this         |
|                                   | decision and they have 28 days to |
|                                   | do this. The customer is eligible |
|                                   | for Medicare while appealing the  |
|                                   | decision.                         |
|                                   |                                   |
|                                   | Note: where a child is born in    |
|                                   | Australia to parent or parents    |
|                                   | who have lodged an appeal against |
|                                   | a visa refusal, the child may be  |
|                                   | eligible to enrol. The child must |
|                                   | hold a bridging visa, and this    |
|                                   | must be on the Department of Home |
|                                   | Affairs letter and the            |
|                                   | Administrative Appeals Tribunal   |
|                                   | (AAT) letter. See Child born in   |
|                                   | Australia enrolling in Medicare.  |
|                                   |                                   |
|                                   | To re-enrol a customer who has an |
|                                   | adverse outcome for Permanent     |
|                                   | Residence (Skilled Regional) visa |
|                                   | (subclass 191), see Ministerial   |
|                                   | Orders and Medicare eligibility   |
|                                   | for details. Customers have       |
|                                   | lodged an appeal against the      |
|                                   | refused decision, go to Step 2.   |

| 2                                 | Documents required + Read more    |
|                                   | ...                               |

|                                   | -   Federal Court or High Court   |
|                                   |     document                      |
|                                   |     Note: go to Step 3 to see     |
|                                   |     proof of an appeal            |
|                                   |     requirements                  |
|                                   |                                   |
|                                   | a valid visa with permission to   |
|                                   | work via one of the below:        |
|                                   |                                   |
|                                   | -   in their passport             |
|                                   | -   in the form of a travel       |
|                                   |     document                      |
|                                   | -   an email or a letter from the |
|                                   |     Department of Home Affairs    |
|                                   |                                   |
