Initiation and assessment process for child death matching

Table 2: This table describes the steps to be taken to initiate and
assess child death matches.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Initiate the child intervention + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Ensure the initial compliance     |
|                                   | checks have been completed.       |
|                                   |                                   |
|                                   | Initiate the intervention using   |
|                                   | the manual process.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Actioning a Child Death Match +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | All actions on the death of a     |
|                                   | child for which Family Tax        |
|                                   | Benefit, Child Care Subsidy,      |
|                                   | Parental Leave Pay (PPL) and Dad  |
|                                   | and Partner Pay (DAP) is being    |
|                                   | paid, are to be referred to the   |
|                                   | Families Bereavement Processing   |
|                                   | Team for action.                  |
|                                   |                                   |
|                                   | All action for non-current child  |
|                                   | records that is, child records    |
|                                   | that no one is receiving a        |
|                                   | payment for are to be processed   |
|                                   | by the RGO Customer Compliance    |
|                                   | Team.                             |
|                                   |                                   |
|                                   | Is the action for a non-current   |
|                                   | child record?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Referring to Families Bereavement |
|                                   | Processing Team + Read more ...   |
|                                   |                                   |
|                                   | Send a Fast Note – Auto Text –    |
|                                   | FAMILIES > UPDATES > RGO          |
|                                   | Bereavement to the Families       |
|                                   | Bereavement Processing team to    |
|                                   | action the customer's record.     |
|                                   |                                   |
|                                   | Integrated Review System (IRS)    |
|                                   | action.                           |
|                                   |                                   |
|                                   | -   Place the intervention on HLD |
|                                   |     for 14 days pending outcomes  |
|                                   |     from the Families Bereavement |
|                                   |     Processing team               |
|                                   | -   When action is completed by   |
|                                   |     the Families Bereavement      |
|                                   |     Processing Team, go to        |
|                                   |     Finalising RGOs               |
+-----------------------------------+-----------------------------------+
| 4                                 | Action for non-current child      |
|                                   | records + Read more ...           |
|                                   |                                   |
|                                   | The record will need to be        |
|                                   | updated to ensure the record is   |
|                                   | ended. Update the customer record |
|                                   | to reflect the correct date of    |
|                                   | death.                            |
|                                   |                                   |
|                                   | Run the RGO review script to code |
|                                   | death action. If the script is    |
|                                   | not available, use the guided     |
|                                   | procedure. If the guided          |
|                                   | procedure is not available        |
|                                   | manually code the death action.   |
|                                   | As the record is for a            |
|                                   | non-current child record no       |
|                                   | bereavement action is required.   |
|                                   |                                   |
|                                   | Once the record has been actioned |
|                                   | the intervention will need to be  |
|                                   | finalised, see Finalising RGOs.   |
