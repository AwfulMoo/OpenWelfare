Job Seeker registration for students

+-----------------------------------+-----------------------------------+
| Scenario                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Full time student who has not     |
|                                   | completed the normal requirements |
|                                   | of the course                     |
|                                   |                                   |
|                                   | Pam has finished all the lectures |
|                                   | in the final semester of the      |
|                                   | course but still has one exam to  |
|                                   | do and wants to register as a job |
|                                   | seeker.                           |
|                                   |                                   |
|                                   | -   Pam has not yet completed the |
|                                   |     normal requirements of the    |
|                                   |     course and is still           |
|                                   |     considered to be a full-time  |
|                                   |     student. Pam is ineligible to |
|                                   |     register as a job seeker      |
+-----------------------------------+-----------------------------------+
| 2                                 | Full time student who has         |
|                                   | completed the normal requirements |
|                                   | of the course                     |
|                                   |                                   |
|                                   | Andrew's course end date is 30    |
|                                   | November but an extension has     |
|                                   | been granted to lodge the final   |
|                                   | assignment for one subject. On 1  |
|                                   | December Andrew wishes to         |
|                                   | register as a job seeker.         |
|                                   |                                   |
|                                   | -   Andrew has completed all the  |
|                                   |     normal requirements of the    |
|                                   |     course and is eligible to     |
|                                   |     register as a job seeker.     |
|                                   |     Andrew cannot be considered   |
|                                   |     to be a full-time student as  |
|                                   |     the course end date has       |
|                                   |     passed and only one subject   |
|                                   |     is being undertaken           |
