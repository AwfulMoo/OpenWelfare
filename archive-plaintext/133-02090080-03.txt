Using Medicare online accounts

Table 1: How customers can change how they get their Medicare letters.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access service + Read more ...    |
|                                   |                                   |
|                                   | Sign in to myGov and select       |
|                                   | Medicare services.                |
|                                   |                                   |
|                                   | If the customer has a return to   |
|                                   | sender flag on their record, they |
|                                   | will need to complete a task      |
|                                   | first.                            |
|                                   |                                   |
|                                   | The task will ask them to confirm |
|                                   | their residential and/or mailing  |
|                                   | addresses.                        |
|                                   |                                   |
|                                   | The customer cannot submit any    |
|                                   | claims online until the task is   |
|                                   | done.                             |
|                                   |                                   |
|                                   | From the Main Menu tab, select My |
|                                   | details > letter delivery         |
|                                   | preference. The customer will     |
|                                   | only have this option if:         |
|                                   |                                   |
|                                   | -   they have set up myGov Inbox, |
|                                   |     and                           |
|                                   | -   their electronic mail         |
|                                   |     preference is set to YES in   |
|                                   |     CDMS                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Letter delivery preference + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A customer's letter delivery      |
|                                   | preference can be either:         |
|                                   |                                   |
|                                   | -   your postal address           |
|                                   | -   your myGov Inbox              |
|                                   |                                   |
|                                   | To change their preference,       |
|                                   | select the ‘Change delivery       |
|                                   | preference’ button.               |
|                                   |                                   |
|                                   | If they change from myGov inbox   |
|                                   | to postal address, they will be   |
|                                   | prompted to confirm their postal  |
|                                   | and residential addresses.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Submit + Read more ...            |
|                                   |                                   |
|                                   | Select Submit to confirm their    |
|                                   | choice.                           |
|                                   |                                   |
|                                   | The customer can only submit if   |
|                                   | they have changed their choice.   |
|                                   |                                   |
|                                   | Their new choice saves to CDMS    |
+-----------------------------------+-----------------------------------+
| 4                                 | myGov notification + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Customers can choose how myGov    |
|                                   | lets them know they have a new    |
|                                   | letter in their myGov Inbox. They |
|                                   | can choose SMS or email.          |
|                                   |                                   |
|                                   | They can do this in Account       |
|                                   | settings in myGov.                |
+-----------------------------------+-----------------------------------+

Using the Consumer Directory Management System (CDMS)

Table 2: How a Service Officer can view and update how a customer gets
their Medicare letters. Staff can also view a history of updates.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | View or update the customer's     |
|                                   | choice + Read more ...            |
|                                   |                                   |
|                                   | Select option in CDMS:            |
|                                   |                                   |
|                                   | -   Select Amend Consumer Details |
|                                   | -   Key the customer’s Medicare   |
|                                   |     card number and select        |
|                                   |     Entitlement                   |
|                                   |                                   |
|                                   | On the Amend Medicare entitlement |
|                                   | Details screen, select:           |
|                                   |                                   |
|                                   | -   consumer ID to be amended     |
|                                   | -   Entitlement from the Summary  |
|                                   |     tab                           |
|                                   | -   Online Services via MCA       |
|                                   | -   Entitlement                   |
|                                   |                                   |
|                                   | On the Entitlement Details        |
|                                   | screen:                           |
|                                   |                                   |
|                                   | -   select the check box Tick if  |
|                                   |     electronic mail through the   |
|                                   |     myGov Inbox is preferred      |
|                                   | -   unselect the check box if the |
|                                   |     customer wants all their      |
|                                   |     letters posted                |
|                                   |                                   |
|                                   | Select Validate to confirm the    |
|                                   | customer's choice                 |
+-----------------------------------+-----------------------------------+
| 2                                 | View history of updates + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select option in CDMS:            |
|                                   |                                   |
|                                   | Key the customer's Medicare card  |
|                                   | number in the *ID Number field    |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   consumer ID to be amended     |
|                                   | -   Entitlement                   |
|                                   | -   the relevant customer to view |
|                                   |     their history                 |
|                                   | -   History from the Summary      |
|                                   |     button at the bottom of the   |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Select the radio button on the    |
|                                   | left of the history to view more  |
|                                   | details                           |
|                                   |                                   |
|                                   | Details include:                  |
|                                   |                                   |
|                                   | the customer's choice             |
|                                   |                                   |
|                                   | -   ELECTRONIC MAIL PREFERENCE    |
|                                   |     HAS BEEN SET TO YES, or       |
|                                   | -   ELECTRONIC MAIL PREFERENCE    |
|                                   |     HAS BEEN SET TO NO            |
|                                   |                                   |
|                                   | the date and time of the update   |
|                                   |                                   |
|                                   | User ID                           |
|                                   |                                   |
|                                   | Source office code                |
|                                   |                                   |
|                                   | .                                 |
