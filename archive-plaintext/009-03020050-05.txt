Examples of co-resident and non-co-resident carers

This table describes examples of co-resident and non-co-resident carers
for adult care receivers when care is shared on a fortnightly basis.

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Example 1                         |
|                                   |                                   |
|                                   | If the care receiver does not     |
|                                   | have a principle home and the     |
|                                   | care is provided in the homes of  |
|                                   | the carers - then the carers are  |
|                                   | co-residents.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Example 2                         |
|                                   |                                   |
|                                   | If the carers provide the care in |
|                                   | the home of the care receiver,    |
|                                   | but reside elsewhere, - then the  |
|                                   | carers are not co-residents.      |
|                                   | (That is, they are a              |
|                                   | non-co-resident carer.)           |
+-----------------------------------+-----------------------------------+
| 3                                 | Example 3                         |
|                                   |                                   |
|                                   | If the carers both reside with    |
|                                   | the care receiver, then they are  |
|                                   | co-residents.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Example 4                         |
|                                   |                                   |
|                                   | If 1 carer resides with the care  |
|                                   | receiver in the care receiver's   |
|                                   | principle home and the care       |
|                                   | receiver spends alternative weeks |
|                                   | with the second carer, the first  |
|                                   | carer is a co-resident and the    |
|                                   | second carer is a non-co-resident |
|                                   | and is not qualified.             |
+-----------------------------------+-----------------------------------+
