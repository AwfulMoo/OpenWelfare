12 week period of grace

Table 1: this table describes when the 12 week POG rules apply.

+-----------------------+-----------------------+-----------------------+
| Income Support        | POG payable at        | POG payable at higher |
| Payment               | standard rate?        | rate?                 |
+-----------------------+-----------------------+-----------------------+
| No income support     | Yes, if standard 12   | No, there is no POG.  |
| payment               | week POG rules apply  | However, if the       |
|                       |                       | customer was          |
|                       |                       | previously receiving  |
|                       |                       | higher rate MOB and   |
|                       |                       | ceased to qualify due |
|                       |                       | solely to increased   |
|                       |                       | income, the higher    |
|                       |                       | rate continues to be  |
|                       |                       | payable while         |
|                       |                       | customer not on an    |
|                       |                       | income support        |
|                       |                       | payment.              |
+-----------------------+-----------------------+-----------------------+
| JobSeeker Payment     | Yes, if standard 12   | Yes, if higher 12     |
| (JSP)                 | week POG rules apply  | week POG rules apply  |
+-----------------------+-----------------------+-----------------------+
| Austudy               | Yes, if standard 12   | No                    |
|                       | week POG rules apply  |                       |
+-----------------------+-----------------------+-----------------------+
| ABSTUDY               | Yes, if standard 12   | No                    |
|                       | week POG rules apply  |                       |
+-----------------------+-----------------------+-----------------------+
| Youth Allowance (job  | Yes, if standard 12   | Yes, if higher 12     |
| seeker)               | week POG rules apply  | week POG rules apply  |
|                       |                       | and customer was      |
|                       |                       | receiving higher rate |
|                       |                       | of MOB                |
+-----------------------+-----------------------+-----------------------+
| Disability Support    | Yes, if standard 12   | Yes, if higher 12     |
| pension (DSP)         | week POG rules apply  | week POG rules apply  |
+-----------------------+-----------------------+-----------------------+
| Parenting Payment     | Yes, if standard 12   | Yes, if higher 12     |
| (Single or Partnered) | week POG rules apply  | week POG rules apply  |
+-----------------------+-----------------------+-----------------------+
