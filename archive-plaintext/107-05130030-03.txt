For staff who have responsibility for debt raising and/or recovery

Determining and actioning statute-barred debts

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for statute of        |
|                                   | limitations + Read more ...       |
|                                   |                                   |
|                                   | Is the debt affected by the       |
|                                   | statute of limitations?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, take normal debt recovery |
|                                   |     actions. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check the customer's              |
|                                   | record/file + Read more ...       |
|                                   |                                   |
|                                   | Check between the start date of   |
|                                   | the first 6 year limitation and 1 |
|                                   | January 2017. Confirm if any:     |
|                                   |                                   |
|                                   | -   recovery actions have taken   |
|                                   |     place, or                     |
|                                   | -   actions extended the          |
|                                   |     limitation period             |
|                                   |                                   |
|                                   | Do any of the above actions       |
|                                   | apply?                            |
|                                   |                                   |
|                                   | -   Yes, take normal debt         |
|                                   |     recovery actions. Procedure   |
|                                   |     ends here                     |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Voluntary payments + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Legislation prevents recovery by  |
|                                   | involuntary methods.              |
|                                   |                                   |
|                                   | Has the debtor offered voluntary  |
|                                   | repayments?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Recovering the debt by voluntary  |
|                                   | repayment/s + Read more ...       |
|                                   |                                   |
|                                   | Is the debtor likely to make this |
|                                   | payment?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Permanently writing off the       |
|                                   | debt + Read more ...              |
|                                   |                                   |
|                                   | Write off the debt with permanent |
|                                   | write off code 'STB'              |
|                                   | (statute-barred).                 |
|                                   |                                   |
|                                   | Record details of action taken on |
|                                   | a DOC.                            |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Restart recovery + Read more ...  |
|                                   |                                   |
|                                   | -   Navigate to the Debt List     |
|                                   |     (OPDL) screen                 |
|                                   | -   'S'elect the Debt ID: and go  |
|                                   |     to the Write Off (OPWO)       |
|                                   |     screen                        |
|                                   | -   Code 'Y' in the Restart       |
|                                   |     Recovery: field and press     |
|                                   |     [Enter]                       |
|                                   |                                   |
|                                   | Record details of action taken on |
|                                   | a DOC.                            |
+-----------------------------------+-----------------------------------+
