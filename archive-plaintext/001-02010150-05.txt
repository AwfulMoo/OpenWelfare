Information on the labour market

This table describes information on how to locate labour market reports
and publications.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Labour Market Information         |
|                                   |                                   |
|                                   | On the Department of Employment   |
|                                   | and Workplace Relations website   |
|                                   | in the search facility field,     |
|                                   | type 'Labour Market Information   |
|                                   | Portal' for a wide range of       |
|                                   | reports and publications          |
|                                   | including the following:          |
|                                   |                                   |
|                                   | -   Employment Service Area       |
|                                   | -   Employment Projections        |
|                                   | -   Vacancy Reports.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Private Counsellors               |
|                                   |                                   |
|                                   | Private Counsellors can provide   |
|                                   | specialist career guidance and    |
|                                   | labour market information for a   |
|                                   | fee. They are listed in the       |
|                                   | Yellow Pages under Vocational     |
|                                   | Guidance.                         |
+-----------------------------------+-----------------------------------+
