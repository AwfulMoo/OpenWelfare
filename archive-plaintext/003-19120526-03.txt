This page contains information on assessing claims for Emergency
payments from customers affected by the Victorian Bushfires, February
2019. Emergency payments include Australian Government Disaster Recovery
Payment (AGDRP), Disaster Recovery Allowance (DRA) and their associated
ex-gratia payments.

AGDRP and ex-gratia DRP claims

For staff processing Australian Government Disaster Recovery Payment
(AGDRP) and ex-gratia Disaster Recovery Payment (DRP) claims only.

Table 3: This table describes the steps to assess eligibility for
AGDRP/ex-gratia DRP for customers affected by the Victorian Bushfires,
February 2019. Service Officers should not process a claim unless they
have completed formal training.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Disaster determined and basic     |
|                                   | eligibility + Read more ...       |
|                                   |                                   |
|                                   | A determination has been made     |
|                                   | that a major disaster has         |
|                                   | occurred for the purposes of      |
|                                   | AGDRP/ex-gratia DRP for the       |
|                                   | Victorian Bushfires, February     |
|                                   | 2019.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Assess eligibility for            |
|                                   | AGDRP/ex-gratia DRP + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | See Eligibility for AGDRP and NZ  |
|                                   | DRP                               |
|                                   |                                   |
|                                   | For the Victorian Bushfires,      |
|                                   | February 2019 the assessment of   |
|                                   | major damage from smoke and/or    |
|                                   | ash has been updated. The         |
|                                   | following questions may be useful |
|                                   | to determine if major damage from |
|                                   | smoke and/or ash has occurred:    |
|                                   |                                   |
|                                   | -   Please describe the damage to |
|                                   |     your home.                    |
|                                   | -   Is the interior or exterior   |
|                                   |     damaged? Note: exterior       |
|                                   |     damage is not eligible        |
|                                   | -   Did you have to replace       |
|                                   |     contents?                     |
|                                   | -   Did you have to repair        |
|                                   |     interior damage?              |
|                                   | -   Could the smoke/ash damage be |
|                                   |     cleaned? Note: Soft           |
|                                   |     furnishings or electrical     |
|                                   |     appliances must need          |
|                                   |     repairing or replacing. These |
|                                   |     do not need to have already   |
|                                   |     been replaced or repaired for |
|                                   |     the customer to meet the      |
|                                   |     major damage criteria.        |
|                                   | -   How could you demonstrate the |
|                                   |     damage occurred? E.g. could   |
|                                   |     you provide a photo or        |
|                                   |     invoice? Note: []this         |
|                                   |     information can be forwarded  |
|                                   |     to the agency through a       |
|                                   |     customer’s online account via |
|                                   |     the upload document service   |
|                                   |                                   |
|                                   | Major damage does not include     |
|                                   | cleaning an item or surface from  |
|                                   | smoke, soot, char or ash.         |
|                                   |                                   |
|                                   | For elements to consider when     |
|                                   | assessing major damage from smoke |
|                                   | and/or ash, see Eligibility for   |
|                                   | AGDRP and NZ DRP. It also has     |
|                                   | examples of repair work to        |
|                                   | consider when assessing claims.   |
|                                   |                                   |
|                                   | Note: for ex-gratia DRP a         |
|                                   | customer must have participated   |
|                                   | in the Australian taxation system |
|                                   | for at least 1 year in the        |
|                                   | following past 3 financial years  |
|                                   | 2016-17, 2017-18, 2018-19 or will |
|                                   | do so before 5 December 2020.     |
+-----------------------------------+-----------------------------------+
| 3                                 | Code claim + Read more ...        |
|                                   |                                   |
|                                   | See Processing AGDRP and NZ DRP   |
|                                   | claims in Customer First.         |
|                                   |                                   |
|                                   | If a customer is not eligible for |
|                                   | AGDRP/ex-gratia DRP, claims are   |
|                                   | still to be recorded and the      |
|                                   | outcome noted as rejected.        |
|                                   |                                   |
|                                   | Claims cannot be rejected until   |
|                                   | formal approval has been          |
|                                   | provided. Staff will be advised   |
|                                   | once they are able to reject      |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Claim should be held with the     |
|                                   | relevant keyword  if formal       |
|                                   | approval  has not  been provided. |
|                                   |                                   |
|                                   | For Minimum Activity Management   |
|                                   | Standards, see the Resources      |
