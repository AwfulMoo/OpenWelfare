AHHCCS overview

+-----------------------------------+-----------------------------------+
| Category                          | Description                       |
+-----------------------------------+-----------------------------------+
| Eligibility                       | A claim may be eligible if it is  |
|                                   | made against a person who is 1 of |
| (members)                         | the following:                    |
|                                   |                                   |
|                                   | -   an allied health professional |
|                                   | -   a midwife engaged as an       |
|                                   |     employee to practise as a     |
|                                   |     midwife                       |
|                                   |                                   |
|                                   | They must be:                     |
|                                   |                                   |
|                                   | -   Registered with the           |
|                                   |     Australian Health             |
|                                   |     Practitioner Regulation       |
|                                   |     Agency (AHPRA)                |
|                                   |                                   |
|                                   | Indemnified under an insurance    |
|                                   | policy with an eligible Medical   |
|                                   | Indemnity Insurer (MII) or        |
|                                   | Medical Defence Organisation      |
|                                   | (MDO)                             |
+-----------------------------------+-----------------------------------+
| Eligibility                       | All participating MIIs and MDOs   |
|                                   | are eligible                      |
| (Applicant)                       |                                   |
+-----------------------------------+-----------------------------------+
| Notification Date                 | The Date of Notification (DoN) of |
|                                   | the incident/claim must be on or  |
|                                   | after 1 July 2020.                |
+-----------------------------------+-----------------------------------+
| Criteria                          | The Date of Incident (DoI), or    |
|                                   | series of incidents, must have    |
|                                   | occurred in the course of, or in  |
|                                   | connection with:                  |
|                                   |                                   |
|                                   | -   practice by the practitioner  |
|                                   |     of their allied health        |
|                                   |     profession                    |
|                                   | -   practice by the midwife       |
|                                   |     engaged as an employee        |
|                                   | -   that occurred on or after 1   |
|                                   |     July 2020                     |
|                                   |                                   |
|                                   | (incidents that occurred before I |
|                                   | July 2020 may be eligible under   |
|                                   | the HCCS)                         |
|                                   |                                   |
|                                   | -   the incident must have        |
|                                   |     occurred within Australia or  |
|                                   |     one of its external           |
|                                   |     territories                   |
|                                   | -   the applicant (insurer) must  |
|                                   |     have made a payment, or be    |
|                                   |     liable to pay a payment in    |
|                                   |     relation to the claim         |
|                                   | -   the payment must be made in   |
|                                   |     line with the insurer’s       |
|                                   |     indemnity policy issued and   |
|                                   |     within the ordinary course of |
|                                   |     business                      |
|                                   | -   the insurer must not have     |
|                                   |     been notified of the          |
|                                   |     occurrence of the incident,   |
|                                   |     nor claim against or by the   |
|                                   |     person in relation to the     |
|                                   |     incident before 1 July 2020   |
|                                   |                                   |
|                                   | the claim amount must exceed the  |
|                                   | threshold of $500,000.00          |
+-----------------------------------+-----------------------------------+
| What is not payable?              | Applicants cannot claim an amount |
|                                   | if:                               |
|                                   |                                   |
|                                   | -   the incident occurred before  |
|                                   |     1 July 2020                   |
|                                   |                                   |
|                                   | -   another party has             |
|                                   |     responsibility to pay it      |
|                                   |                                   |
|                                   | -   it is an insurer to insurer   |
|                                   |     payment                       |
|                                   |                                   |
|                                   | -   the cost was not incurred in  |
|                                   |     relation to the incident      |
|                                   |                                   |
|                                   | -   the settlement or judgment    |
|                                   |     amounts are still within the  |
|                                   |     appeal period or have been    |
|                                   |     appealed or stayed            |
|                                   |                                   |
|                                   | -   it’s a GST components of a    |
|                                   |     defence costs that is not     |
|                                   |     exempt                        |
|                                   |                                   |
|                                   | -   it relates to incidents that  |
|                                   |     occur partly or fully within  |
|                                   |     the course of treating a      |
|                                   |     public patient in a public    |
|                                   |     hospital. If it occurs fully, |
|                                   |     the application is rejected,  |
|                                   |     If partly, the total          |
|                                   |     application payment must be   |
|                                   |     reduced proportionately       |
|                                   |                                   |
|                                   | -   the incident/s occurred       |
|                                   |     outside of Australia and      |
|                                   |     external territories          |
+-----------------------------------+-----------------------------------+
| Clarification                     | An allied health professional is  |
|                                   | a person who practices an allied  |
|                                   | health profession, other than the |
|                                   | medical profession – see the      |
|                                   | AHHCCS Speciality Codes table’.   |
+-----------------------------------+-----------------------------------+

Specialist codes for AHHCCS

Medical practitioner specialist codes are received from insurers and are
used in reports sent to the Australian Government Actuary.

These Insurance Statistics Australia (ISA) specialty codes can be used
for an AHHCCS payment application.

For incidents that happened on or after 1 July 2020

  ---------------- -----------------------------------------------------------
  Category title   Description
  CDENTI           Dentist
  CPSYCO           Psychologist
  CRADOG           Radiographer
  CRADTH           Radiation therapist
  CCMPRA           Chinese medicine practitioner
  CCHIRO           Chiropractor
  COCTHE           Occupational therapist
  COPTOM           Optometrist
  COSTEO           Osteopath
  CPHYSI           Physiotherapist
  CPODTS           Podiatrist
  CMIDWI           Midwife
  CATSIH           Aboriginal and Torres Strait Islander health practitioner
  CALLHE           Medical allied health professional - not otherwise stated
  ---------------- -----------------------------------------------------------
