This page contains information to assist with the processing of Single
Income Family Supplement (SIFS) standalone claims for families who are
not eligible for Family Tax Benefit (FTB) or do not want to claim FTB.

|                                   |                                   |
|                                   | SIFS cannot be paid for a period  |

|                                   | available for the customer, and   |
|                                   | their partner/ex-partner for that |
|                                   | period.                           |
|                                   |                                   |
|                                   | For SIFS, adequate income means:  |
|                                   |                                   |
|                                   | -   Accepted not required to      |
|                                   |     lodge advice and provision of |
|                                   |     an income estimate.           |
|                                   | -   Taxable income details        |
|                                   |     electronically transferred    |
|                                   |     from the Australian Taxation  |
|                                   |     Office (ATO) if the person is |
|                                   |     required to lodge a tax       |
|                                   |     return                        |
|                                   |                                   |
|                                   | Tax returns must be lodged (or    |
|                                   | not required to lodge advice      |
|                                   | provided) by 30 June of the       |
|                                   | lodgement year for SIFS to be     |
|                                   | paid. Claims will be              |
|                                   | automatically rejected (late      |
|                                   | lodgement) if the ATO advise      |
|                                   | either the customer or their      |
|                                   | partner lodged tax returns after  |
|                                   | the end of the lodgement year,    |
|                                   | unless a special circumstances    |
|                                   | income confirmation extension has |
|                                   | been granted.                     |
|                                   |                                   |
|                                   | Note: only taxable income (TI) is |
|                                   | used to assess SIFS, not adjusted |
|                                   | taxable income (ATI). The         |
|                                   | customer is asked to estimate     |
|                                   | their ATI for SIFS only claims to |
