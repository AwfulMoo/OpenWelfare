This page contains information about the process of requesting from
Inland Revenue, New Zealand (NZIR) an Amended Certificate on a case that
NZIR has transmitted to Child Support.

Request for Amended Certificate

This table describes how Service Officers request an Amended
Certificate.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Request an Amended certificate +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If there has been a change or new |
|                                   | information provided for a case   |
|                                   | that has been transmitted to      |
|                                   | Child Support (NZ In Case) an     |
|                                   | amended certificate request needs |
|                                   | to be sent to NZIR.               |
|                                   |                                   |
|                                   | Create intray, go to Step 2       |
+-----------------------------------+-----------------------------------+
| 2                                 | Create intray (Service Officer) + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Create a new intray: NZ in Req    |
|                                   | Acert from NZIR                   |
|                                   |                                   |
|                                   | Route the intray to the following |
|                                   | location:                         |
|                                   |                                   |
|                                   | -   Region: INTERNATIONAL         |
|                                   | -   Office: INTERNATIONAL HOBART  |
|                                   | -   Steam/Team: MC S005 NZPROJECT |
|                                   |     TM 1                          |

|                                   |     NZ                            |
|                                   |                                   |

|                                   | report to be sent to NZIR as part |
