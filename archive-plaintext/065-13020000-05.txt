This page contains residential care funding changes, removal of high/low
categories, leave condition enquiries, links to contact details for Aged
Care, useful external website links, and a link to the Residential Care
Manual (historical).

Funding changes for extra service in residential care FAQs

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Response                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: When did the changes to |
|                                   | extra service funding come into   |
|                                   | effect?                           |
|                                   |                                   |
|                                   | Answer: These changes came into   |
|                                   | effect from 1 July 2014.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Who was affected by the |
|                                   | changes to extra service          |
|                                   | payments?                         |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Residential care services are |
|                                   |     no longer limited to applying |
|                                   |     for extra service status for  |
|                                   |     an entire service or a        |
|                                   |     distinct part of a service.   |
|                                   |     They can apply for extra      |
|                                   |     service status for one or     |
|                                   |     more rooms in the service     |
|                                   | -   All services can offer        |
|                                   |     additional amenities (extra   |
|                                   |     service) on an opt-in opt-out |
|                                   |     basis. The fees for these     |
|                                   |     services is an agreement made |
|                                   |     between the resident and the  |
|                                   |     service                       |
|                                   | -   Care recipients potentially   |
|                                   |     have a greater range of       |
|                                   |     services available to them    |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How were extra service  |
|                                   | places affected?                  |
|                                   |                                   |
|                                   | Answer: Extra service places were |
|                                   | granted before 1 July 2014 and    |
|                                   | continued to have extra service   |
|                                   | status after 1 July 2014.         |
|                                   |                                   |
|                                   | Care recipients who entered an    |
|                                   | extra service place before 1 July |
|                                   | 2014 have not been affected by    |
|                                   | the changes. After 1 July 2014,   |
|                                   | they continued to have their fees |
|                                   | calculated in accordance with the |
|                                   | same rules that were in place up  |
|                                   | until 30 June 2014.               |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: How did this impact     |
|                                   | services?                         |
|                                   |                                   |
|                                   | Answer: Services providing extra  |
|                                   | services no longer have the       |
|                                   | additional 25% charge recovered   |
|                                   | for care recipients who enter     |
|                                   | care on or after 1 July 2014.     |
+-----------------------------------+-----------------------------------+

Removal of high/low categories from residential aged care approvals FAQs

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Response                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: When did the removal of |
|                                   | the residential high/low          |
|                                   | categories come into effect?      |
|                                   |                                   |
|                                   | Answer: the removal of            |
|                                   | residential high/low categories   |
|                                   | came into effect from 1 July      |
|                                   | 2014.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: What are the changes?   |
|                                   |                                   |
|                                   | Answer: from 1 July 2014,         |
|                                   | residential care approvals        |
|                                   | replaced residential high and     |
|                                   | residential low approvals.        |
|                                   | Existing care recipients with a   |
|                                   | residential category of high or   |
|                                   | low are now known as having a     |
|                                   | 'residential care approval'.      |
+-----------------------------------+-----------------------------------+
| 3                                 | How did this will this affect     |
|                                   | existing National Screening and   |
|                                   | Assessment Forms (NSAF)?          |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   From 1 July 2014, care        |
|                                   |     recipients with approvals     |
|                                   |     dated before 1 July 2014 are  |
|                                   |     no longer limited to          |
|                                   |     residential low care          |
|                                   | -   Residential care approvals in |
|                                   |     place on or before 30 June    |
|                                   |     2014 will not lapse, but may  |
|                                   |     expire if they were           |
|                                   |     explicitly time limited       |
+-----------------------------------+-----------------------------------+
| 4                                 | How does it affect payments?      |
|                                   |                                   |
|                                   | Answer: from 1 July 2014, for     |
|                                   | care recipients limited to a      |
|                                   | default low rate where they have  |
|                                   | a low National Screening and      |
|                                   | Assessment Form (NSAF) and a high |
|                                   | Aged Care Funding Instrument      |
|                                   | (ACFI), the payment system        |
|                                   | applies the rate applicable to    |
|                                   | ACFI                              |
+-----------------------------------+-----------------------------------+
| 5                                 | How did this affect the system?   |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   System changes utilised the   |
|                                   |     functionality used for        |
|                                   |     loading care recipient NSAF   |
|                                   |     information, service          |
|                                   |     eligibility and payment       |
|                                   |     calculation                   |
|                                   | -   From 1 July 2014, operators   |
|                                   |     can only enter the one type   |
|                                   |     of residential care approval. |
|                                   |     There is no option to enter a |
|                                   |     high or low care approval     |
+-----------------------------------+-----------------------------------+
| 6                                 | How did this impact services?     |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   The system for residential    |
|                                   |     care services is more         |
|                                   |     streamlined. There are no     |
|                                   |     different rules for services  |
|                                   |     providing high care versus    |
|                                   |     low care                      |
|                                   | -   NSAF updates are still        |
|                                   |     available through the same    |
|                                   |     process                       |
|                                   | -   The system applies a default  |
|                                   |     payment rate to new entry     |
|                                   |     until an ACFI has been        |
|                                   |     provided and then apply the   |
|                                   |     ACFI daily rate from the date |
|                                   |     of entry                      |
|                                   | -   Unless specifically time      |
|                                   |     limited, approvals do not     |
|                                   |     lapse                         |
+-----------------------------------+-----------------------------------+
