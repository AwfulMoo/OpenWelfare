Actioning the death of the partner of a PA customer

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notification of death received +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When a person contacts to advise  |
|                                   | about the death of a customer,    |
|                                   | staff must offer:                 |
|                                   |                                   |
|                                   | -   A social worker referral in   |
|                                   |     the first instance or at any  |
|                                   |     time the notifier becomes or  |
|                                   |     shows signs of distress       |
|                                   |     during the contact. See       |
|                                   |     Referring customers to a      |
|                                   |     social worker for more        |
|                                   |     information                   |
|                                   | -   The 'Needing help after       |
|                                   |     someone has died? (HD001)'    |
|                                   |     pamphlet. Issue if accepted   |
|                                   |                                   |
|                                   | Consider and respect the          |
|                                   | following cultural issues when    |
|                                   | being advised of the death of an  |
|                                   | Indigenous customer.              |
|                                   |                                   |
|                                   | -   The name of the deceased      |
|                                   |     person cannot be mentioned by |
|                                   |     immediate family members.     |
|                                   |     This means immediate family   |
|                                   |     members may have to ask other |
|                                   |     relatives/friends to call on  |
|                                   |     their behalf or attend the    |
|                                   |     interview with the notifier,  |
|                                   |     because they cannot say the   |
|                                   |     name of the deceased          |
|                                   | -   In this situation, it may     |
|                                   |     also be offensive to the      |
|                                   |     person for the Service        |
|                                   |     Officer to say the name of    |
|                                   |     the deceased customer, if     |
|                                   |     unsure, Service Officers      |
|                                   |     should sensitively ask the    |
|                                   |     person if it is all right for |
|                                   |     them to use the deceased      |
|                                   |     customer's name. If the       |
|                                   |     person is not comfortable     |
|                                   |     with this, avoid using the    |
|                                   |     deceased customer's name      |
|                                   |     wherever possible. See        |
|                                   |     Cultural Considerations when  |
|                                   |     someone dies                  |
|                                   |                                   |
|                                   | Does the customer's record        |
|                                   | indicate that they were income    |
|                                   | managed, as shown by INM on the   |
|                                   | benefit status line?              |
|                                   |                                   |
|                                   | -   Yes, follow process to update |
|                                   |     record for death of an income |
|                                   |     managed customer. Return to   |
|                                   |     this procedure and go to Step |
|                                   |     2                             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirm the death of the deceased |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | Run the Notification of death     |
|                                   | guided procedure in the deceased  |
|                                   | customer's record to determine if |
|                                   | death can be confirmed.           |
|                                   |                                   |
|                                   | Confirmation of death can be      |
|                                   | accepted where the customer's     |
|                                   | full name is provided together    |
|                                   | with 2 of the following items:    |
|                                   |                                   |
|                                   | -   deceased customer's last      |
|                                   |     address in full               |
|                                   | -   date of birth and age at      |
|                                   |     death                         |
|                                   | -   Customer Reference Number     |
|                                   |     (CRN)                         |
|                                   | -   name of partner               |
|                                   |                                   |
|                                   | Confirm if the person notifying   |
|                                   | of the death is the executor of   |
|                                   | the estate. If the informant is   |
|                                   | not the executor, but knows the   |
|                                   | executor details, obtain the:     |
|                                   |                                   |
|                                   | -   name and address of the       |
|                                   |     executor, and                 |
|                                   | -   name of the funeral director  |
|                                   |     or hospital                   |
|                                   |                                   |
|                                   | See Confirmation of the death of  |
|                                   | an adult or child.                |
|                                   |                                   |
|                                   | Nominee arrangements and Power of |
|                                   | Attorney (POA) stop at the time   |
|                                   | of death. Any correspondence,     |
|                                   | including the payment summary and |
|                                   | possible debt recovery action     |
|                                   | from a deceased estate, will be   |
|                                   | sent to the:                      |
|                                   |                                   |
|                                   | -   executor, or                  |
|                                   | -   administrator of an estate    |
|                                   |                                   |
|                                   | Services Australia's ability to   |
|                                   | finalise any outstanding matters  |
|                                   | will be impacted if executor or   |
|                                   | administrator details are:        |
|                                   |                                   |
|                                   | -   incorrectly recorded, or      |
|                                   | -   not recorded when details     |
|                                   |     have been provided            |
|                                   |                                   |
|                                   | Where executor details are not    |
|                                   | known at the time of death        |
|                                   | notification, leave the executor  |
|                                   | details fields in the             |
|                                   | Notification of death guided      |
|                                   | procedure blank.                  |
|                                   |                                   |
|                                   | Note: executor details can be     |
|                                   | recorded manually on the          |
|                                   | Death/Executor (DEA) screen if    |
|                                   | details become known later.       |
|                                   |                                   |
|                                   | Can informant supply enough       |
|                                   | information to confirm death?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Death cannot be confirmed + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the death cannot be confirmed: |
|                                   |                                   |
|                                   | -   use the Notification of death |
|                                   |     guided procedure to obtain    |
|                                   |     and record all relevant       |
|                                   |     information from the          |
|                                   |     informant including their own |
|                                   |     name, address, contact number |
|                                   |     and relationship to the       |
|                                   |     deceased, and if known,       |
|                                   |     details of the executor, the  |
|                                   |     name of the funeral director, |
|                                   |     and the name of the hospital  |
|                                   | -   the guided procedure will not |
|                                   |     record the death, but will    |
|                                   |     record details on a DOC which |
|                                   |     creates an open work item to  |
|                                   |     be allocated to a Service     |
|                                   |     Officer with the required     |
|                                   |     skill set to confirm the      |
|                                   |     death                         |
|                                   | -   if the guided procedure is    |
|                                   |     not available, create an open |
|                                   |     work item. Use Fast note -    |
|                                   |     select Auto text, use         |
|                                   |     Generic > Update > Death      |
|                                   |     Notification (Adult) - ACTION |
|                                   |     REQUIRED                      |
|                                   |                                   |
|                                   | Where multiple notifications are  |
|                                   | received in relation to the death |
|                                   | of the same person do not re-run  |
|                                   | the guided procedure. Check the   |
|                                   | record and annotate the original  |
|                                   | DOC to state details are correct  |
|                                   | and who advised.                  |
|                                   |                                   |
|                                   | If there is doubt about the       |
|                                   | identity of the deceased, contact |
|                                   | the funeral director or the       |
|                                   | hospital where the deceased       |
|                                   | passed away. If this is not an    |
|                                   | option and contact with relatives |
|                                   | is proposed, social worker        |
|                                   | involvement may be necessary.     |
|                                   | Where appropriate, send Request   |
|                                   | for Verification (SS257) form.    |
|                                   |                                   |
|                                   | Where death cannot be confirmed   |
|                                   | and the customer's whereabouts    |
|                                   | cannot be established, payment is |
|                                   | to be suspended until the death   |
|                                   | can be confirmed. The suspension  |
|                                   | date of event should be the       |
|                                   | unconfirmed date of death or the  |
|                                   | date paid to plus one (DPT + 1),  |
|                                   | whichever is the later:           |
|                                   |                                   |
|                                   | -   select the suspension option  |
|                                   |     in the Notification of death  |
|                                   |     guided procedure, or          |
|                                   | -   manually code the suspension  |
|                                   |     for reason 'OTH' (Other)      |
|                                   |                                   |
|                                   | Record a DOC on the customer's    |
|                                   | record, ensuring that it meets    |
|                                   | GIR DOC Minimum Standards for     |
|                                   | significant decisions and include |
|                                   | what action is required to        |
|                                   | confirm the death.                |
|                                   |                                   |
|                                   | Do not issue a suspension letter. |
|                                   | Cancel the advice activity on the |
|                                   | AL screen triggered by the 'OTH'  |
|                                   | suspension.                       |
|                                   |                                   |
|                                   | Note: confirm and action the      |
|                                   | death as a matter of urgency.     |
|                                   |                                   |
|                                   | When death is confirmed, go to    |
|                                   | Step 4.                           |

| 4                                 | Death is confirmed + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | A death action cannot be          |
|                                   | processed with activities on the  |
|                                   | Activity List (AL) screen. The    |
|                                   | Notification of death guided      |
|                                   | procedure will provide a prompt   |
|                                   | if there is an activity on the AL |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Note: if a deceased customer's    |
|                                   | record was suspended while the    |
|                                   | death was being confirmed and     |
|                                   | their death has now been          |
|                                   | confirmed, manually check for     |
|                                   | started activities on the AL      |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Where possible ensure any         |
|                                   | activities are completed and run  |
|                                   | the daily driver (DDR):           |
|                                   |                                   |
|                                   | -   if there is a 'STA' debt      |
|                                   |     activity on the AL screen,    |
|                                   |     create an open work item for  |
|                                   |     region XWA advising them of   |
|                                   |     the death and requesting the  |
|                                   |     debt activity be completed.   |
|                                   |     Use Fast Note - select Auto   |
|                                   |     text, use > Debts > DMB       |
|                                   |     Raising > Debt Raising ACTion |
|                                   |     DOC Request. See Requesting   |
|                                   |     refunds and raising debts for |
|                                   |     deceased customer             |
|                                   | -   for other outstanding         |
|                                   |     activities, immediately       |
|                                   |     contact the team or Service   |
|                                   |     Officer who needs to complete |
|                                   |     the outstanding activity. If  |
|                                   |     unsure who to contact, refer  |
|                                   |     to existing escalation        |
|                                   |     channels                      |
|                                   |                                   |
|                                   | Have started activities been      |
|                                   | completed?                        |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 5              |

| 5                                 | When it is not possible to        |
|                                   | complete an activity + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | If there is an outstanding        |
|                                   | activity that the Service Officer |
|                                   | is unable to complete, suspend    |
|                                   | the deceased customer's payment   |
|                                   | (if not already suspended).       |
|                                   |                                   |
|                                   | The suspension option is not      |
|                                   | available in the Notification of  |
|                                   | death guided procedure if there   |
|                                   | is an activity preventing the     |
|                                   | death action:                     |
|                                   |                                   |
|                                   | -   manually suspend using the    |
|                                   |     reason 'OTH' (other) pending  |
|                                   |     completion of the outstanding |
|                                   |     activity                      |
|                                   | -   the suspension date of event  |
|                                   |     is the date of death or date  |
|                                   |     paid to plus 1 (DPT + 1),     |
|                                   |     whichever is the later        |
|                                   | -   record a DOC on the           |
|                                   |     customer's record, ensuring   |
|                                   |     that it meets Online Document |
|                                   |     Recording (ODR) standards for |
|                                   |     significant decisions and     |
|                                   |     include what action must      |
|                                   |     occur so the death can be     |
|                                   |     actioned                      |
|                                   | -   as the customer's death has   |
|                                   |     been confirmed, no letter     |
|                                   |     must be sent. It is not       |
|                                   |     appropriate to send a letter  |
|                                   |     to a deceased customer.       |
|                                   |     Cancel the advice activity on |
|                                   |     the AL screen triggered by    |
|                                   |     the 'OTH' suspension          |
|                                   |                                   |
|                                   | If an open DOC/work item for the  |
|                                   | death notification has not        |
|                                   | already been created, either:     |
|                                   |                                   |
|                                   | -   run the Notification of death |
|                                   |     guided procedure to generate  |
|                                   |     an open work item for the     |
|                                   |     relevant staff to complete    |
|                                   |     the death action once the     |
|                                   |     outstanding activities have   |
|                                   |     been completed, or            |
|                                   | -   if the guided procedure is    |
|                                   |     not available create an open  |
|                                   |     work item. Use Fast note -    |
|                                   |     select Auto text, use         |
|                                   |     Generic > Update > Death      |
|                                   |     Notification (Adult) - ACTION |
|                                   |     REQUIRED                      |
|                                   |                                   |
|                                   | When activity is completed, go to |
|                                   | Step 6.                           |

|                                   |     the restoration advice, as it |

|                                   | Different processes apply when    |

|                                   |                                   |
|                                   | -   a pensioner                   |
|                                   | -   a long-term Social Security   |
|                                   |     recipient, or                 |
|                                   | -   in receipt of Service         |
|                                   |     Pension, Veteran Payment or   |
|                                   |     Income Support Supplement     |

| 13                                | PA qualification will continue    |
|                                   | for the surviving partner during  |
|                                   | the bereavement period + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The survivor's rate should be     |
|                                   | immediately adjusted to the       |
|                                   | single rate from the later of     |
|                                   | date of death or date paid to +1. |
|                                   |                                   |
|                                   | Tell the customer:                |
|                                   |                                   |
|                                   | -   they will need to apply for a |
|                                   |     payment in their own right,   |
|                                   |     before the end of the         |
|                                   |     bereavement period            |
|                                   | -   if the customer starts work,  |
|                                   |     or is currently working,      |
|                                   |     working credits may enable    |
|                                   |     the customer to keep some of  |
|                                   |     their income support payment  |
|                                   |     while they are working        |
|                                   | -   if the customer's income      |
|                                   |     reduces their fortnightly     |
|                                   |     rate to nil and some of that  |
|                                   |     income is employment income,  |
|                                   |     the customer may remain       |
|                                   |     current at nil rate for up to |
|                                   |     6 fortnights and retain       |
|                                   |     certain benefits. Note: if    |
|                                   |     the customer's deceased       |
|                                   |     partner had any working       |
|                                   |     credits, these credits cannot |
|                                   |     be transferred to the         |
|                                   |     customer                      |
|                                   | -   if the customer was a joint   |
|                                   |     owner of any assets with      |
|                                   |     their late partner, full      |
|                                   |     ownership of the assets       |
|                                   |     legally transfers to the      |
|                                   |     surviving partner following   |
|                                   |     the death of their partner.   |
|                                   |     However, for Social Security  |
|                                   |     purposes, if assets are owned |
|                                   |     in joint names with the       |
|                                   |     deceased partner, the         |
|                                   |     surviving partner is only     |
|                                   |     assessed as owning 50%,       |
|                                   |     unless the surviving partner  |
|                                   |     advises that the assets have  |
|                                   |     been transferred to them      |
|                                   | -   if the surviving partner is   |
|                                   |     granted a reversionary income |
|                                   |     stream, then it is to be      |
|                                   |     assessed from the date that   |
|                                   |     it is first entitled to be    |
|                                   |     received. A lump sum arrears  |
|                                   |     payment received by the       |
|                                   |     reversionary beneficiary,     |
|                                   |     accrued between the death of  |
|                                   |     the primary beneficiary and   |
|                                   |     when the reversionary         |
|                                   |     beneficiary commences to      |
|                                   |     receive regular payments, is  |
|                                   |     assessed as income from the   |
|                                   |     income stream. This means the |
|                                   |     income is assessed over the   |
|                                   |     period to which it relates    |
|                                   |     and may result in a debt for  |
|                                   |     that period if the customer   |
|                                   |     has been in receipt of an     |
|                                   |     income support payment.       |
|                                   |     Effective from 9 May 2018,    |
|                                   |     where the assessment of the   |
|                                   |     lump sum arrears result in a  |
|                                   |     debt during the bereavement   |
|                                   |     period for the surviving      |
|                                   |     spouse, consideration should  |
|                                   |     be given to waiver provisions |
|                                   |     contained under Class of      |
|                                   |     Debt, Section 1237AB(1) of    |
|                                   |     the Social Security Act 1991. |
|                                   |     The legislative instrument    |
|                                   |     "Social Security (Waiver of   |
|                                   |     Debts - Bereavement Period)   |
|                                   |     Specification 2017" permits   |
|                                   |     this action if all provisions |
|                                   |     are satisfied. Note: any debt |
|                                   |     outside the 14 week           |
|                                   |     bereavement period cannot be  |
|                                   |     waived under the legislative  |
|                                   |     instrument. The legislative   |
|                                   |     instrument is not             |
|                                   |     retrospective. This means     |
|                                   |     that any of these debts that  |
|                                   |     occur before 9 May 2018       |
|                                   |     cannot be waived under class  |
|                                   |     of debt waiver provisions     |
|                                   |                                   |
|                                   | Offer referral to a social worker |
|                                   | if appropriate.                   |

| 14                                | PA customer may also be eligible  |
|                                   | for a Lump Sum Bereavement        |
|                                   | Payment (LBP) + Read more ...     |
|                                   |                                   |
|                                   | A PA customer must both be        |
|                                   | qualified for a LBP and an LBP    |
|                                   | must be payable.                  |
|                                   |                                   |
|                                   | The PA customer is qualified for  |
|                                   | an LBP if the deceased partner    |
|                                   | was a long-term social security   |
|                                   | recipient. This means that if the |
|                                   | deceased partner was a pensioner, |
|                                   | in receipt of an income support   |
|                                   | benefit or allowance, or in       |
|                                   | receipt of Service Pension,       |
|                                   | Veteran Payment or Income Support |
|                                   | Supplement (ISS), they must also  |
|                                   | have been a long-term social      |
|                                   | security recipient.               |
|                                   |                                   |
|                                   | Once death action has been        |
|                                   | recorded, switch to the surviving |
|                                   | partner's record.                 |
|                                   |                                   |
|                                   | The LBP is payable to the         |
|                                   | eligible surviving partner where  |
|                                   | the new single rate is less than  |
|                                   | the previous combined member of a |
|                                   | couple rate.                      |
|                                   |                                   |
|                                   | Once the death action has been    |
|                                   | recorded, switch to the surviving |
