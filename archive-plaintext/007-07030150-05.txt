Scenarios and suggested text for manual Q999 letters

This table contains suggested manual Q999 letter text to use for Family
Tax Benefit (FTB) rejection reasons if an automatic letter has not been
sent. Include the appeal rights and relevant Act reference.

+-----------------------------------+-----------------------------------+
| Scenario                          | Suggested text                    |
+-----------------------------------+-----------------------------------+
| 1                                 | FTB rejects - Income too high +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | 'Your claim for Family Tax        |
|                                   | Benefit for the XXXX/XXXX         |
|                                   | financial year has been rejected, |
|                                   | as your family income is too      |
|                                   | high. You should check your       |
|                                   | entitlement with us at the end of |
|                                   | the income year when your         |
|                                   | family’s actual income is known’. |
|                                   |                                   |
|                                   | -   include relevant Act          |
|                                   |     Reference, and review and     |
|                                   |     appeal rights                 |
+-----------------------------------+-----------------------------------+
| 2                                 | FTB rejects - Child not in care + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | 'We cannot pay you Family Tax     |
|                                   | Benefit for {child's name}        |
|                                   | because they are not considered   |
|                                   | to be in your care.'              |
+-----------------------------------+-----------------------------------+
| 3                                 | FTB rejects - Estimated income    |
|                                   | not accepted + Read more ...      |
|                                   |                                   |
|                                   | 'We cannot pay you Family Tax     |
|                                   | Benefit because we are not        |
|                                   | satisfied that your estimated     |
|                                   | family income is reasonable.'     |
+-----------------------------------+-----------------------------------+
