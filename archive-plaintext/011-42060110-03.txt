ART treatment cycles

Table 2: this table describes treatment cycles for the purposes of ART
services.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | ART treatment cycles + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | A treatment cycle is a series of  |
|                                   | treatments for the purposes of    |
|                                   | ART services and is defined as:   |
|                                   |                                   |
|                                   | beginning on either               |
|                                   |                                   |
|                                   | -   the day the recipient starts  |
|                                   |     treatment with super          |
|                                   |     ovulatory drugs, or           |
|                                   | -   the first day of the          |
|                                   |     recipient’s menstrual cycle   |
|                                   |                                   |
|                                   | ending on either                  |
|                                   |                                   |
|                                   | not more than 30 days after the   |
|                                   | treatment cycle begins, or        |
|                                   |                                   |
|                                   | the day after the last of the     |
|                                   | following services is provided:   |
|                                   |                                   |
|                                   | -   Oocyte retrieval (item 13212) |
|                                   | -   Transfer of embryos or both   |
|                                   |     ova and sperm (item 13215),   |
|                                   |     or                            |
|                                   | -   Preparation of sperm for the  |
|                                   |     purpose of artificial         |
|                                   |     insemination (item 13221)     |
|                                   |                                   |
|                                   | A treatment cycle has a maximum   |
|                                   | of 31 days, including the first   |
|                                   | day.                              |
+-----------------------------------+-----------------------------------+
| 2                                 | Short treatment cycle + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | A treatment cycle is considered a |
|                                   | ‘short cycle’ when it is 26 days  |
|                                   | or less.                          |
|                                   |                                   |
|                                   | Claims received for ART services  |
|                                   | within the 26 day restrictive     |
|                                   | period of the previous cycle must |
|                                   | be verified by a notation in the  |
|                                   | text stating 'short cycle' (or    |
|                                   | wording to that effect).          |
|                                   |                                   |
|                                   | These items then can be paid with |
|                                   | PI 45.                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Long treatment cycle + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | A treatment cycles is considered  |
|                                   | a ‘long cycle’ when it exceeds 31 |
|                                   | days.                             |
|                                   |                                   |
|                                   | Treatment cycles generally do not |
|                                   | exceed 31 days. However, if items |
|                                   | 13212, 13215 or 13221 are claimed |
|                                   | more than 30 days after           |
|                                   | applicable global items, the      |
|                                   | account must be annotated ‘long   |
|                                   | cycle’ or wording to that effect  |
|                                   | in order to attract a benefit.    |
|                                   |                                   |
|                                   | These items can then be paid with |
|                                   | PI 48.                            |
|                                   |                                   |
|                                   | The Medicare assessing system     |
|                                   | will apply the 26 day restriction |
|                                   | automatically when 2 cycles       |
|                                   | overlap due to a long cycle.      |
|                                   |                                   |
|                                   | Note: if the date of service for  |
|                                   | items 13212, 13215 or 13221 is    |
|                                   | before 1 November 2014 and 6-ASC  |
|                                   | NOT is returned, reject the       |
|                                   | service with reason code (RSN)    |
|                                   | 550.                              |
