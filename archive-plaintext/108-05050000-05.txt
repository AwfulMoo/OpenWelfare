Financial assets to which deeming applies

Financial assets to which deeming applies include:

-   bank, building society and credit union cheque and savings accounts
-   cash
-   term deposits
-   cash management accounts
-   money held in solicitors' trust accounts
-   managed investments
-   listed shares and securities
-   bonds, debentures, unsecured notes, bank bills
-   loans made to individuals, private companies and trusts
-   shares in unlisted public companies
-   gold and other bullion
-   investments in superannuation and roll-over funds held by recipients
    who are over Age Pension age
-   asset-tested income stream (short term)
-   from 1 January 2015, asset-tested income stream (long term) that is
    an account-based pension
-   from 1 January 2015, asset-tested income stream (long term) that is
    an account-based annuity
-   deprived assets

Examples of calculating income using the deeming threshold and rates

+-----------------------------------+-----------------------------------+
| Item                              | Examples                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Single JobSeeker Payment (JSP)    |
|                                   | customer                          |
|                                   |                                   |
|                                   | Jane is a single JSP customer     |
|                                   | with a total of $56,600 in        |
|                                   | financial investments: $5,000 is  |
|                                   | in a working account, $15,000 in  |
|                                   | a cash management account and     |
|                                   | $36,600 in a term deposit.        |
|                                   |                                   |
|                                   | Determine the value of the        |
|                                   | customer's financial assets:      |
|                                   |                                   |
|                                   | -   add financial investments and |
|                                   |     deprived assets               |
|                                   | -   Result: $56,600               |
|                                   |                                   |
|                                   | The value of the financial assets |
|                                   | ($56,600) is more than the        |
|                                   | current threshold                 |
|                                   |                                   |
|                                   | Determine the unused value        |
|                                   | amount:                           |
|                                   |                                   |
|                                   | -   total value of total          |
|                                   |     financial assets ($56,600)    |
|                                   | -   less threshold ($53,600)      |
|                                   | -   Result: $3,000                |
|                                   |                                   |
|                                   | Multiply the threshold by lower   |
|                                   | deeming percentage                |
|                                   |                                   |
|                                   | -   Result: $53,600 x 0.25% =     |
|                                   |     $134.00 deemed income per     |
|                                   |     year                          |
|                                   |                                   |
|                                   | Multiply the unused value by the  |
|                                   | higher deeming percentage         |
|                                   |                                   |
|                                   | -   Result: $3,000 x 2.25% =      |
|                                   |     $67.50 deemed income per year |
|                                   |                                   |
|                                   | Determine total deemed income:    |
|                                   |                                   |
|                                   | -   Result: $134.00 + $67.50 =    |
|                                   |     $201.50                       |
|                                   |                                   |
|                                   | If the total value of financial   |
|                                   | assets was less than the          |
|                                   | threshold amount, only the 0.25%  |
|                                   | deeming rate would apply          |

| 2                                 | Non-pensioner couple (neither     |
|                                   | receiving a pension)              |
|                                   |                                   |
|                                   | Neither member of the couple      |
|                                   | receives a social security        |
|                                   | pension, service pension or       |
|                                   | rehabilitation allowance.         |
|                                   |                                   |
|                                   | Ben receives JSP and his partner  |
|                                   | Chrissy receives Parenting        |
|                                   | Payment (PP). Ben has $5,000 in a |
|                                   | working account and a further     |
|                                   | $20,000 in a joint account.       |
|                                   | Chrissy has $7,500 in another     |
|                                   | account, shares valued at $30,000 |
|                                   | and her share of the joint        |
|                                   | account. Deemed income for Ben    |
|                                   | and Chrissy is calculated         |
|                                   | separately because the separate   |
|                                   | allowance Income Tests apply.     |
|                                   |                                   |
|                                   | Determine the value of each       |
|                                   | customer's total financial        |
|                                   | assets:                           |
|                                   |                                   |
|                                   | -   add financial investments and |
|                                   |     deprived assets               |
|                                   | -   Result for Ben: $15,000. This |
|                                   |     is less than the threshold of |
|                                   |     $44,500                       |
|                                   | -   Result for Chrissy: $47,500.  |
|                                   |     This is more than the         |
|                                   |     threshold of $44,500          |
|                                   |                                   |
|                                   | Determine the amount in excess of |
|                                   | threshold:                        |
|                                   |                                   |
|                                   | -   total value of financial      |
|                                   |     assets                        |
|                                   | -   less threshold                |
|                                   | -   Result for Ben: Not           |
|                                   |     Applicable                    |
|                                   | -   Result for Chrissy: $47,500 - |
|                                   |     $44,500 = $3,000              |
|                                   |                                   |
|                                   | For Ben:                          |
|                                   |                                   |
|                                   | -   multiply the total value of   |
|                                   |     his financial assets by 0.25% |
|                                   | -   $15,000 x 0.25% = $37.50      |
|                                   |     deemed income per year        |
|                                   |                                   |
|                                   | For Chrissy:                      |
|                                   |                                   |
|                                   | -   multiply the threshold by     |
|                                   |     0.25%                         |
|                                   | -   $44,500 x 0.25% = $111.25     |
|                                   |     deemed income per year        |
|                                   | -   multiply the balance in       |
|                                   |     excess of the threshold by    |
|                                   |     2.25%                         |
|                                   | -   $3,000 x 2.25% = $67.50       |
|                                   |                                   |
|                                   | Determine total deemed income:    |
|                                   |                                   |
|                                   | -   add total below threshold and |
|                                   |     above threshold amounts       |
|                                   | -   Result for Ben: $37.50        |
|                                   | -   Result for Chrissy: total     |
|                                   |     deemed income $111.25 +       |
|                                   |     $67.50 = $178.75              |
