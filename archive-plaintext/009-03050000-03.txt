This is a Family and Domestic Violence Interaction Point. If the
customer is with another person, on speaker phone, or already identified
with family and domestic violence concerns, continue with the current
business conversation. Otherwise, go to the Family and domestic violence
procedure to conduct the risk identification and referral process.

Confirming change in care and disputes between carers

Table 1: this table outlines the actions to take when a carer has lodged
a claim for a care receiver already being cared for.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check records for caring          |
|                                   | circumstances information + Read  |
|                                   | more ...                          |

|                                   | care receiver or nominees it is   |

|                                   | to contact the claiming carer,    |
|                                   | current carer, care receiver or   |
|                                   | any nominees if there is any      |
|                                   | indication of family and domestic |
|                                   | violence (FDV) on any of the      |

|                                   | information about the caring      |
|                                   | circumstances.                    |

|                                   |                                   |
|                                   | -   Address details - do the      |
|                                   |     carer and care receiver       |
|                                   |     reside together?              |

|                                   |     carer/s and care receiver     |
|                                   | -   Care arrangements (for child  |
|                                   |     care receiver)                |
|                                   | -   Care receiver's medical       |
|                                   |     conditions and care load      |
|                                   | -   Relationship/s between        |
|                                   |     carer/s and care receiver     |
|                                   | -   Nominee and PPE authorities - |
|                                   |     for privacy reasons, when     |
|                                   |     contacting either carer, do   |
|                                   |     not refer to the other carer  |
|                                   |     by their name unless the      |
|                                   |     carer being spoken with has   |
|                                   |     provided this information.    |
|                                   |     Until this occurs, refer to   |
|                                   |     them as 'another carer' or    |
|                                   |     'the other carer'             |
|                                   | -   If the care receiver is       |
|                                   |     partnered with the current    |
|                                   |     carer, and care has ceased    |
|                                   |     due to separation, see Change |
|                                   |     in relationship status from   |
|                                   |     partnered to single and       |
|                                   |     Separating safely -           |
|                                   |     protecting personal details   |
|                                   |     before returning here         |
|                                   |                                   |
|                                   | During contact with any of the    |
|                                   | parties, they may advise (or      |
|                                   | staff may suspect) they are       |
|                                   | affected by family and domestic   |
|                                   | violence (FDV). Follow the FDV    |
|                                   | Support Model to determine an     |
|                                   | appropriate referral and further  |
|                                   | agency support.                   |
|                                   |                                   |
|                                   | Note: if either carer, the care   |
|                                   | receiver or nominee advises       |
|                                   | someone other than themselves is  |
|                                   | affected by FDV, staff can        |
|                                   | provide them with referral        |
|                                   | options for the person affected   |
|                                   | by FDV, from the Family and       |
|                                   | domestic violence - Support       |
|                                   | services intranet page or Payment |
|                                   | and Service Finder.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact the carer who lodged a    |
|                                   | claim for CP and/or CA for the    |
|                                   | care receiver + Read more ...     |
|                                   |                                   |
|                                   | Make 2 genuine attempts to        |
|                                   | contact the claiming carer by     |
|                                   | phone to clarify information      |
|                                   | provided in their claim.          |

|                                   | information in Change of address  |
|                                   | for Carer Payment (CP) and Carer  |
|                                   | Allowance (CA)                    |
|                                   |                                   |
|                                   | Has any recent change to care     |
|                                   | occurred?                         |
|                                   |                                   |
|                                   | Is daily care still provided 7    |
|                                   | days per week?                    |
|                                   |                                   |
|                                   | Does anyone else provide care?    |
|                                   |                                   |
|                                   | -   If so, what care do they      |
|                                   |     provide?                      |
|                                   |                                   |
|                                   | Follow alternative Operational    |
|                                   | Blueprints if the carer and care  |
|                                   | receiver circumstances meet:      |
|                                   |                                   |
|                                   | -   Multiple care                 |
|                                   | -   Combined care                 |
|                                   | -   Shared care for CA, or        |
|                                   | -   Exchanged care (child)        |
|                                   |                                   |
|                                   | Record any discussion with the    |
|                                   | claiming carer in the claim       |
|                                   | progress Note and DOC. Do not     |
|                                   | include the name of the current   |
|                                   | carer, unless provided by the     |
|                                   | claiming carer.                   |
|                                   |                                   |
|                                   | Is there enough information to    |
|                                   | determine both carers'            |
|                                   | entitlements?                     |
|                                   |                                   |
|                                   | -   Yes, proceed with the         |
|                                   |     claiming carer's claim and    |
|                                   |     reassess the current carer’s  |
|                                   |     entitlement if required.      |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Contact the current carer in      |
|                                   | receipt of CP and/or CA for the   |
|                                   | care receiver + Read more ...     |
|                                   |                                   |
|                                   | Make 2 genuine attempts to        |
|                                   | contact the current carer by      |
|                                   | phone to clarify information      |
|                                   | about the claims by the claiming  |
|                                   | carer.                            |
|                                   |                                   |
|                                   | Note: for privacy reasons, when   |

|                                   | When contact is successful,       |
|                                   | discuss:                          |
|                                   |                                   |
|                                   | Has a recent change of address    |
|                                   | occurred?                         |
|                                   |                                   |
|                                   | Changes to living arrangements -  |
|                                   | is the claiming carer still       |
|                                   | residing with the care receiver?  |
|                                   |                                   |
|                                   | If the claiming carer and care    |
|                                   | receiver do not co-reside, what   |
|                                   | distance do they live apart? See  |
|                                   | 'Consideration of distance -      |
|                                   | assessment of constant care for   |
|                                   | CP' information in Change of      |
|                                   | address for Carer Payment (CP)    |
|                                   | and Carer Allowance (CA)          |
|                                   |                                   |
|                                   | Has any recent change to care     |
|                                   | occurred?                         |
|                                   |                                   |
|                                   | Is daily care still provided 7    |
|                                   | days per week?                    |
|                                   |                                   |
|                                   | Does anyone else provide care?    |
|                                   |                                   |
|                                   | -   If so, what care do they      |
|                                   |     provide?                      |
|                                   |                                   |
|                                   | Follow alternative Operational    |
|                                   | Blueprints if the carer and care  |
|                                   | receiver circumstances meet:      |
|                                   |                                   |
|                                   | -   Multiple care                 |
|                                   | -   Combined care                 |
|                                   | -   Shared care for CA, and/or    |
|                                   | -   Exchanged care (child)        |
|                                   |                                   |
|                                   | Record any discussion on the      |
|                                   | current carer’s record and the    |
|                                   | claiming carer’s record in the    |

|                                   |                                   |
|                                   | Is there enough information to    |
|                                   | determine both carers'            |

|                                   | No:                               |

|                                   |     unsuccessful for either care  |

|                                   |     but change in care cannot be  |
|                                   |     determined or there is a      |
|                                   |     dispute between the 2 carers  |
|                                   |     over the care being provided, |

+-----------------------------------+-----------------------------------+

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Attempts to contact the claiming  |

Resolving disputes between carers

Table 2: this table outlines the actions to take when change in care
cannot be determined or there is a dispute between 2 carers over the
care being provided.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Change in care cannot be          |
|                                   | determined + Read more ...        |
|                                   |                                   |
|                                   | If it is reasonable for the       |
|                                   | current carer to have supporting  |
|                                   | evidence, make 2 genuine attempts |
|                                   | to contact the current carer to   |
|                                   | verbally request this evidence be |
|                                   | provided.                         |
|                                   |                                   |
|                                   | While care schedules cannot be    |
|                                   | asked for, the carer could have   |
|                                   | other documents supporting their  |
|                                   | claims. For example, a letter     |
|                                   | from a hospital stating they      |
|                                   | attend to the care receiver on a  |
|                                   | regular basis.                    |
|                                   |                                   |
|                                   | Go to Step 2                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine if it reasonable to     |
|                                   | clarify the situation with the    |
|                                   | care receiver + Read more ...     |
|                                   |                                   |
|                                   | Check details on the carer's and  |
|                                   | care receiver's record - Notes,   |
|                                   | Document List (DL), medical       |
|                                   | reports, scanned documents, new   |
|                                   | carer's claim and current         |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Consider the care receiver's:     |
|                                   |                                   |
|                                   | -   Nominee arrangements          |
|                                   | -   Medical conditions that may   |
|                                   |     affect communication (such as |
|                                   |     cognitive abilities, hearing  |
|                                   |     loss)                         |
|                                   | -   Relationships with current    |
|                                   |     and claiming carer            |
|                                   | -   Location in relation to       |
|                                   |     current and claiming carer    |
|                                   |                                   |
|                                   | If any risk is identified or any  |
|                                   | other reason the care receiver    |
|                                   | should be contacted by a social   |
|                                   | worker, go to Step 3.             |
|                                   |                                   |
|                                   | Only if reasonable, make 2        |
|                                   | genuine attempts to contact the   |
|                                   | care receiver. If contact is      |
|                                   | successful, ask:                  |
|                                   |                                   |
|                                   | A domestic violence risk          |
|                                   | identification question such as   |
|                                   | “are you experiencing family and  |
|                                   | domestic violence?” For other     |
|                                   | questions, see Family and         |
|                                   | Domestic Violence, process table  |
|                                   | 2                                 |
|                                   |                                   |
|                                   | Have any recent changes to care   |
|                                   | occurred?                         |
|                                   |                                   |
|                                   | Any changes to address or living  |
|                                   | arrangements?                     |
|                                   |                                   |
|                                   | Is care receiver residing with    |
|                                   | any carer?                        |
|                                   |                                   |
|                                   | If carer and care receiver do not |
|                                   | co-reside, what distance do they  |
|                                   | live apart?                       |
|                                   |                                   |
|                                   | Is daily care provided 7 days per |
|                                   | week?                             |
|                                   |                                   |
|                                   | Is anyone else providing care?    |
|                                   |                                   |
|                                   | -   If so, what care is provided  |
|                                   |     and by whom?                  |
|                                   |                                   |
|                                   | Record discussion with care       |
|                                   | receiver in the claiming carer's  |
|                                   | progress DOC. Go to Step 4        |
|                                   |                                   |
|                                   | If contact is unsuccessful, go to |
|                                   | Step 5.                           |
|                                   |                                   |
|                                   | If it is unreasonable to contact  |
|                                   | the care receiver and dispute     |
|                                   | cannot be clarified, record       |
|                                   | reasons in the claim progress     |
|                                   | Note and DOC. Go to Step 5.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Social worker referral + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Complete a Multiple and Complex   |
|                                   | Social Work Referral on the care  |
|                                   | receiver's record. The referral   |
|                                   | to the social worker should       |
|                                   | detail all attempted phone        |
|                                   | contacts and discussions with     |
|                                   | carer and care receiver, if       |
|                                   | applicable. Details of the care   |
|                                   | receiver’s medical condition are  |
|                                   | to be included.                   |
|                                   |                                   |
|                                   | -   Record details of the social  |
|                                   |     worker referral in the claim  |
|                                   |     progress Note and/or DOC      |
|                                   | -   Hold the claim until the      |
|                                   |     current carer's review due    |
|                                   |     date                          |
|                                   | -   See Hold to User process in   |
|                                   |     Work Optimiser for staff      |
|                                   |                                   |
|                                   | When the SWO assessment is        |
|                                   | completed, go to Step 4           |
+-----------------------------------+-----------------------------------+
| 4                                 | Social worker assessment          |
|                                   | completed or contact with care    |
|                                   | receiver successful + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is there enough information to    |
|                                   | determine both carers'            |
|                                   | entitlements?                     |
|                                   |                                   |
|                                   | -   Yes, proceed with the         |
|                                   |     claiming carer's claim and    |
|                                   |     reassess the current carer’s  |
|                                   |     entitlement if required.      |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Dispute cannot be resolved + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Submit an enquiry to Level 2      |
|                                   | Policy Helpdesk.                  |
|                                   |                                   |
|                                   | Outline claim information,        |
|                                   | attempted contacts, and any       |
|                                   | additional information supplied   |
|                                   | by the carers and/or care         |
|                                   | receiver.                         |
|                                   |                                   |
|                                   | Record details of the request in  |
|                                   | the claiming carer's claim        |
|                                   | progress Note and/or DOC.         |
|                                   |                                   |
|                                   | Hold to User via Work Optimiser,  |
|                                   | waiting for Level 2 Policy        |
|                                   | Helpdesk instruction.             |
|                                   |                                   |
|                                   |                                   |
