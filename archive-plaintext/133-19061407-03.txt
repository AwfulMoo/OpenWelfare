Deactivate online account

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check reason + Read more ...      |
|                                   |                                   |
|                                   | Common reasons:                   |
|                                   |                                   |
|                                   | -   The customer no longer has    |
|                                   |     access to the internet        |
|                                   | -   The customer does not like    |
|                                   |     the online service            |
|                                   | -   Inappropriate use of their    |
|                                   |     online account                |
|                                   |                                   |
|                                   | If a customer has a temporary     |
|                                   | password, their online account    |
|                                   | remains pended until they         |
|                                   | activate it using the password.   |
|                                   | If they do not activate it within |
|                                   | 14 days, the account becomes      |
|                                   | inactive.                         |
|                                   |                                   |
|                                   | There is no need to deactivate an |
|                                   | online account so the customer    |
|                                   | can receive their letters or      |
|                                   | statements by post. They can keep |
|                                   | their online account active and   |
|                                   | change the way they receive       |
|                                   | letters or statements. Advise the |
|                                   | customer about changing their     |
|                                   | channel preference.               |
|                                   |                                   |
|                                   | Note: if a customer is reporting  |
|                                   | that the temporary password has   |
|                                   | failed to link their Child        |
|                                   | Support service in myGov.         |
|                                   | Deactivating and reactivating the |
|                                   | Child Support online account can  |
|                                   | be attempted for troubleshooting  |
|                                   | a successful link between Child   |
|                                   | support online service and myGov. |
