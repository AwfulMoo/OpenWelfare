This document outlines how to personalise, access, receive and manage
work items in Customer First. It shows how to navigate generic Income
and Asset summary screens and the Customer ICT Interaction viewer.

Access

Once the appropriate Customer First security role has been approved in
ESSentials, staff can log on to Customer First via the desktop icon.

Personalising Customer First

Staff can personalise Customer First to suit their role and preferences.

This includes the ability to:

-   generate receipt numbers for non-telephony staff
-   change Customer First preferences including keyboard shortcuts and
    error messages preview
-   add, remove or change the order of assignment blocks
-   change the format of assignment blocks
-   update user preferences, including default environment (Centrelink
    only)
-   adjust cursor size

The Resources page contains user guides, keyboard shortcuts and profiles
for receipt settings.
