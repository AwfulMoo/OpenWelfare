Eligibility for ACCS (Grandparent)

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for Child Care        |
|                                   | Subsidy (CCS) + Read more ...     |
|                                   |                                   |
|                                   | To be eligible for ACCS           |
|                                   | (Grandparent) customers must be   |
|                                   | eligible for CCS.                 |
|                                   |                                   |
|                                   | Is the customer current on CCS or |
|                                   | eligible for CCS?                 |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS             |
|                                   |     (Grandparent). Procedure ends |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Definition of grandparent + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A grandparent or a great          |
|                                   | grandparent is defined as someone |
|                                   | who is:                           |
|                                   |                                   |
|                                   | the biological                    |
|                                   | grandparent/great-grandparent of  |
|                                   | the child, or                     |
|                                   |                                   |
|                                   | a grandparent/great-grandparent   |
|                                   | as a result of:                   |
|                                   |                                   |
|                                   | -   adoption                      |
|                                   | -   step-parent relationships     |
|                                   |     (including former partners)   |
|                                   | -   artificial conception         |
|                                   |     procedures                    |
|                                   | -   de facto relationship, or     |
|                                   | -   surrogacy arrangements        |
|                                   |                                   |
|                                   | Is the customer a grandparent?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS             |
|                                   |     (Grandparent). Procedure ends |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | All or at least 65% ongoing       |
|                                   | care + Read more ...              |
|                                   |                                   |
|                                   | Grandparents must have all or at  |
|                                   | least 65% of the ongoing daily    |
|                                   | care for the child.               |
|                                   |                                   |
|                                   | The percentage of care is         |
|                                   | determined using the same rules   |
|                                   | as applies for Family Tax Benefit |
|                                   | (FTB). For more information, see  |
|                                   | Notification and assessment of    |
|                                   | shared care arrangements for FTB. |
|                                   |                                   |
|                                   | In most cases, a grandparent will |
|                                   | not be considered as the          |
|                                   | principal carer of the child if   |
|                                   | they reside in the same home as   |
|                                   | the biological parent. The        |
|                                   | Resources page has scenarios when |
|                                   | a grandparent may or may not be   |
|                                   | considered the principal carer    |
|                                   | for ACCS (Grandparent).           |
|                                   |                                   |
|                                   | Does the customer have 65% or     |
|                                   | more care of the child?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS             |
|                                   |     (Grandparent). Procedure ends |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 4                                 | Substantial autonomy of           |
|                                   | day-to-day decisions about the    |
|                                   | child’s care + Read more ...      |
|                                   |                                   |
|                                   | In addition to providing 65% or   |
|                                   | more daily ongoing daily care, a  |
|                                   | grandparent must have substantial |
|                                   | autonomy for the day-to-day       |
|                                   | decisions about the child’s care, |
|                                   | welfare and development. This     |
|                                   | means they generally have the     |
|                                   | power to make decisions without   |
|                                   | consulting with the child’s       |
|                                   | parents, when the child is in the |
|                                   | grandparent’s care.               |
|                                   |                                   |
|                                   | Does the customer have            |
|                                   | substantial autonomy of           |
|                                   | day-to-day decisions?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS             |
|                                   |     (Grandparent) Procedure ends  |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Eligible income support           |
|                                   | payments + Read more ...          |
|                                   |                                   |
|                                   | Is the customer, or their         |
|                                   | partner, in receipt of an ACCS    |
|                                   | (Grandparent) eligible income     |
|                                   | support payment?                  |
|                                   |                                   |
|                                   | -   Yes, the customer meets the   |
|                                   |     eligibility criteria for ACCS |
|                                   |     (Grandparent), subject to the |
|                                   |     provision of supporting       |
|                                   |     evidence. See Table 2         |
|                                   | -   No, the customer is not       |
|                                   |     eligible for ACCS             |
|                                   |     (Grandparent). Procedure ends |
|                                   |     here                          |

Evidence requirements for ACCS (Grandparent) applications

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Proof of relationship + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If a grandparent relationship has |
|                                   | already been established with     |
|                                   | Services Australia the customer   |
|                                   | is not required to provide        |
|                                   | evidence again as part of an      |
|                                   | application for ACCS              |
|                                   | (Grandparent).                    |
|                                   |                                   |
|                                   | If the relationship has not       |
|                                   | previously been determined,       |
|                                   | evidence required includes, but   |
|                                   | is not limited to:                |
|                                   |                                   |
|                                   | -   checking records and scanned  |
|                                   |     documents (including on the   |
|                                   |     grandchild’s parent's record) |
|                                   |     to determine the relationship |
|                                   | -   Family law order or custody   |
|                                   |     order                         |
|                                   | -   Parenting plan or order       |
|                                   | -   Birth certificates of the     |
|                                   |     grandparent’s own children    |
|                                   |     and their children            |
|                                   | -   A combination of birth        |
|                                   |     certificates and marriage     |
|                                   |     certificates for              |
|                                   |     step-grandparents             |
|                                   | -   Adoption papers to prove      |
|                                   |     relationship                  |
|                                   | -   Artificial conception or      |
|                                   |     surrogacy documentation,      |
|                                   |     where the child has not been  |
|                                   |     adopted                       |
|                                   | -   Commonwealth statutory        |
|                                   |     declaration, only accepted    |
|                                   |     where formal evidence cannot  |
|                                   |     be obtained                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Percentage of care + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If conclusive evidence has        |
|                                   | previously been provided          |
|                                   | confirming the level of care,     |
|                                   | there is no need for customers to |
|                                   | provide evidence again, as long   |
|                                   | as the evidence is accessible to  |
|                                   | process the application.          |
|                                   |                                   |
|                                   | However, evidence of care is      |
|                                   | required if:                      |
|                                   |                                   |
|                                   | -   the Service Officer is not    |
|                                   |     satisfied with information on |
|                                   |     hand                          |
|                                   | -   the evidence previously       |
|                                   |     provided is not accessible    |
|                                   |     from the customer’s record    |
|                                   | -   the evidence has expired      |
|                                   |     (e.g. court order has         |
|                                   |     expired) or is no longer      |
|                                   |     current (e.g. percentage of   |
|                                   |     care has changed from the     |
|                                   |     original court order), or     |
|                                   | -   care arrangements have        |
|                                   |     changed since care was        |
|                                   |     previously determined         |
|                                   |                                   |
|                                   | For further information about     |
|                                   | evidence of care, see             |
|                                   | Notification and assessment of    |
|                                   | shared care arrangements for      |
|                                   | Family Tax Benefit (FTB).         |
+-----------------------------------+-----------------------------------+
| 3                                 | Substantial autonomy to make      |
|                                   | decisions about the child + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Evidence to support substantial   |
|                                   | autonomy of decision-making is    |
|                                   | required if this cannot be        |
|                                   | established by other evidence     |
|                                   | already on hand and which can be  |
|                                   | accessed to process the ACCS      |
|                                   | (Grandparent) application.        |
|                                   |                                   |
|                                   | -   Evidence can include: Family  |
|                                   |     law order                     |
|                                   | -   Custody order                 |
|                                   | -   Parenting plan or order       |
|                                   | -   Foster care placement         |
|                                   | -   Court order for graduated     |
|                                   |     return to care where child is |
|                                   |     being integrated back into    |
|                                   |     the family                    |
|                                   | -   Where formal care             |
|                                   |     arrangements do not exist, a  |
|                                   |     letter from child’s school,   |
|                                   |     General Practitioner or other |
|                                   |     third party (who has regular  |
|                                   |     dealing with the family and   |
|                                   |     can independently verify the  |
|                                   |     circumstances) stating that   |
|                                   |     the grandparent has           |
|                                   |     responsibility for the day to |
|                                   |     day decisions about the       |
|                                   |     child’s care, welfare and     |
|                                   |     development                   |
|                                   | -   Commonwealth statutory        |
|                                   |     declaration, only accepted    |
|                                   |     where formal evidence cannot  |
|                                   |     be obtained                   |
|                                   |                                   |
|                                   | Where formal documentation does   |
|                                   | not confirm autonomy of care, the |
|                                   | following types of third party    |
|                                   | evidence can be considered on a   |
|                                   | case by case basis:               |
|                                   |                                   |
|                                   | -   letter from the child’s       |
|                                   |     General Practitioner, or      |
|                                   |     other allied health provider, |
|                                   |     stating the grandparent is    |
|                                   |     responsible for the child’s   |
|                                   |     medical care and expenses     |
|                                   | -   letter from a pharmacist      |
|                                   |     stating the grandparent pays  |
|                                   |     for prescriptions for their   |
|                                   |     grandchild and/or seek advice |
|                                   |     for managing their            |
|                                   |     grandchild’s health           |
|                                   | -   letter from the child’s       |
|                                   |     school stating that the       |
|                                   |     grandparent is the person to  |
|                                   |     call regarding decisions      |
|                                   |     about their grandchild’s      |
|                                   |     schooling, including the      |
|                                   |     child’s wellbeing while at    |
|                                   |     school                        |
|                                   | -   receipts from medical         |
|                                   |     expenses, travel, school      |
|                                   |     uniforms etc.                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Commonwealth statutory            |
|                                   | declarations as evidence + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Commonwealth statutory            |
|                                   | declarations should only be       |
|                                   | considered acceptable evidence    |
|                                   | for a 12 month period. This is to |
|                                   | ensure reasonable time is given   |
|                                   | for grandparents to obtain        |
|                                   | additional evidence. It is a      |
|                                   | requirement that reasonable steps |
|                                   | to obtain other evidence are made |
|                                   | within the timeframe.             |
|                                   |                                   |
|                                   | If a Service Officer determines   |
|                                   | it is still unfeasible for formal |
|                                   | evidence to be obtained, the      |
|                                   | Commonwealth statutory            |
|                                   | declaration can continue to be    |
|                                   | used as supporting evidence. Such |
|                                   | situations should be assessed on  |
|                                   | a case by case basis.             |
+-----------------------------------+-----------------------------------+
| 5                                 | Single forms of evidence + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In some cases, a single piece of  |
|                                   | evidence is sufficient to confirm |
|                                   | grandparent relationship, care    |
|                                   | arrangements and substantial      |
|                                   | autonomy of decision-making. For  |
|                                   | example a child custody and       |
|                                   | parenting arrangement order which |
|                                   | outlines the customer’s care      |
|                                   | arrangements and relationship to  |
|                                   | the child.                        |
|                                   |                                   |
|                                   | In such cases, separate evidence  |
|                                   | is not required to confirm        |
|                                   | relationship and principal care   |
|                                   | arrangements.                     |
