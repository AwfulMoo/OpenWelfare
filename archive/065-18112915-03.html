Add, update and delete contacts for provider in ACMPS

Table 1: This table describes how to add, update and delete the contacts for a provider in Aged Care Management Payment System (ACMPS).

Step

Action

1

Search provider + Read more ...

On the:

  • Home screen, select Manage Partner option from the side menu. Select Manage Provider
  • Search Provider screen, enter information into the relevant fields

Note: the search fields displayed are not mandatory. The search operation can be performed using any single parameter or multiple parameters. The search parameters can be selected from the drop down box. Search rows can be copied/removed by clicking the plus (+) or minus (-) icon.

Select the relevant operation for the search criteria. Enter the value for the search criterion.

Select Search.

Note: to clear the search values, select Clear. This operation clears the values in the search fields, but retains any new field that might have been added.

2

Select provider + Read more ...

Select the:

  • relevant provider by selecting it in the search results screen
  • Contact Person tab

Note: if the tab is not visible on the screen, select the Display Tabs button on the right hand side of the series of tabs and select Contact Person from the list displayed.

To:

  • create a new contact, go to Step 3
  • update the details of an existing contact, go to Step 4
  • delete a contact, go to Step 5

3

Create a new contact + Read more ...

Select New.

Enter the following details for the contact:

  • first name
  • surname
  • phone number - must be a 10-digit number
  • email address

Select Back and Save.

Procedure ends here.

4

Update details of an existing contact + Read more ...

Select the Edit button next to the contact to be updated.

Update the required details and then select Back and Save.

Procedure ends here.

5

Delete contact + Read more ...

Select the Delete button next to the contact to be deleted.

Select Save.

Table 4: This table describes how to search and view a provider in Aged Care Management Payment System (ACMPS).

Search and view provider + Read more ...

On the Home screen, select the Manage Partner option from the side menu and then select Manage Provider.

Key the following fields with information:

  • Provider ID
  • Provider Name
  • State

Note: the search fields displayed are not mandatory. The search operation can be performed using any single parameter or multiple parameters. The search parameters can be selected from the drop down box. Search rows can be copied/removed by clicking the plus (+) or minus (-) icon.

Note: if the tab is not visible on the screen, select the Display Tabs button on the right hand side of the series of tabs and select the relevant tab from the list.

2

Save search + Read more ...

To save a performed search:

  • specify the relevant file name is the provided text box
  • select Save

The saved search is located under My Saved Searches section in the Home menu. Selecting it will display the search results of the saved search operation.

Note:

  • For administrators only - the search data can be exported to into a Microsoft Excel spreadsheet by clicking the export icon
  • The number of searches that can be saved is dependent on the size of database used
