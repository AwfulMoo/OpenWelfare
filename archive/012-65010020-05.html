AHS search query fields

Table 2 Note: ‘*’ denotes at least 1 of these fields is required.

Field

Action

AHS registration number *

Key the AHS registration number for an AHS centre or community

Approved supplier number *

Key the approved supplier number

Date of supply

These fields can be used to enter either a specific supply date (in the first date field) or a date range

Claim reference number

Key the Claim reference number if available. Approved suppliers provide a claim reference number to Services Australia with each claim

AHS community name
  • Select from the drop down list the AHS community or outreach station that is affiliated with the Aboriginal Health Service centre holding the AHS registration number
  • If required, also select the main AHS centre by choosing the ‘Main AHS centre – no community’ option. By doing this the search is narrowed to a particular community or Main centre
  • The drop down list can remain on ‘Please Select’. This will ensure claims for all communities/outreach stations which meet the search criteria will display

AHS claim number *

Key the AHS claim number

Status

Select from the drop down list of possible claim status values. The possible values are Registered, In Progress, Completed and Abandoned

AHS reason codes

Table 3

Reason code

Reason text

Instructions

053

PBS item code invalid for AHS - does not exist

Check the claim form and screen details match. Amend if necessary and resubmit.

If the correct item code can be identified based on the description provided on the claim form, key the correct item code and resubmit.

If the correct item code cannot be identified, confirm the rejection.

138

PBS item code or item/brand combination has been translated

No operator action required.

141

The item provided was not a PBS benefit at the date of supply

Check the claim form and screen details match. Amend if necessary and resubmit.

If the incorrect item code has been provided, identify the correct item code based on the description provided on the claim form, key the correct item code and resubmit.

If the item provided was not a benefit at date of supply, confirm the rejection.

535

Invalid brand supplied for item

Check the claim form and screen details match. Amend if necessary and resubmit.

If the brand cannot be identified/is invalid by the system, select Amend Details and remove Brand details and resubmit.

640

Item cannot be claimed under section 100 AHS arrangements

Check the claim form and screen details match. Amend if necessary and resubmit.

If the item type is not permitted, confirm the rejection.

641

The number of total supplies appears excessive

Any amounts over 100 will come up with an automatic rejection code R641. Double check Quantity Supplied, amend if necessary.

If the total supplies is correct, select Amend Details, tick the Override Reject and resubmit.

642

Duplicate item supplied in claim

Check the claim form and screen details match.

Amend if necessary and resubmit.

If the duplicate item is to be paid, override the rejection.

643

Item not supplied, claimed in error

Add reason code, confirm rejection and continue processing.

644

Item not claimed, keyed in error

Add reason code, confirm rejection and continue processing.

645

Item description provided on AHS claim form does not match schedule-listed item code/brand details

Add reason code, confirm rejection and continue processing.
