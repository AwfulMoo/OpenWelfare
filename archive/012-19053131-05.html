Optometrist quick reference

Details

Description

Specialty Code
  • 550 Optometrist (base specialty code)
  • 551 prescribing Optometrist (where prescriber number has been requested, and has prescribing endorsement as noted on Ahpra and delegate approval authorised)

Note: 555, 101, 151 and 161 are old specialty codes, these are no longer in use.

Location Type

If the provider has indicated the practice is an Aboriginal Health Service check the details against the current Health Insurance Act 1973, Direction under sub-section 19(2).

If location is on 1 of these directions:

  • select ATSIHS and AMS in Location Type
  • select Add

Eligible for Prescriber number
  • Yes - for optometrists whose registration is endorsed eligible to prescribe. Endorsement must read:
    • Endorsed as qualified to administer, obtain, possess, prescribe, supply or use topical schedule 2, 3 or 4 medicines for the purpose of the practice of optometry

Note: Delegate approval is required prior to issuing. See Prescriber number for health professionals in Medicare for further information.

Request diagnostic imaging and pathology tests

No

Refer to Specialist or Consultant physician
  • Yes - Optometrists are eligible to write referrals to:
    • Ophthalmologists (specialist), and
    • other Optometrists based on the clinical needs of the patient
  • Referrals are valid for twelve months unless otherwise stated

Eligible for 90 Day Cheque scheme

No

Registration details
  • Current registration to practise optometry with the Optometry Board of Australia, which is a board supported by the Australian Health Practitioners Regulation Agency (Ahpra)
  • Ahpra registration certificate required with initial application
  • Registrations are renewed annual on 30 November
