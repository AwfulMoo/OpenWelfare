Business structure details

In PDS, ABN details come from the ABN Government Register.

Trading name details come from existing data in PDS that has been entered from previous applications under the relevant ABN.

Business structure details not completed in full

Request to not claim Medicare benefits

  • Enter the details supplied, issue provider number for no access to Medicare benefits and service rule the location

Request to claim a Medicare benefit (including assist at private operations only):

Make 1 attempt to call the health professional to get the required details over the telephone.

Contact successful:

  • Perform a security check
  • Confirm the business structure and record the details in PDS
  • Add note to PANDA file

Contact unsuccessful:

  • For allied health/non-medical health professional, other medical practitioners (104 specialty code) and medical practitioners subject to Section 19AA only:
    • Add note to PANDA file indicating call unsuccessful and contact not made
    • Continue processing the application according to current business processes
    • Send appropriate letter, tick the letter option for missing/incomplete business structure
  • For medical practitioners subject to section 19AB:
    • Add note to PANDA file that call unsuccessful and contact not made
    • Return application as incomplete with a Provider Registration - we need more information (Z2306)

Commonwealth-approved COVID-19 respiratory clinics

These clinics are refer and request only. Business structure and banking details are not recorded, even when supplied.

A 618 service rule is to be added to the location.

Extending an open provider number

Provider number can be extended if the provider number is open at the time the request to extend the provider number is received.

More than one selection in the employment status, business type or premises type fields

Select the most suitable option based on the name of the practice location.

See Employment status fields or Premises status fields tables.

Multiple provider number requests for no access to Medicare benefits

Acceptable formats:

  • Application with attached list of locations

Multiple results are returned when ABN is entered in PDS

Select the option that matches the details on the application form.

If no match, create a new record for the trading name supplied.

Older versions of provider number application form where applicant ticks ‘no’ to claiming a Medicare benefit/refer and request only

Form can be accepted if it states optional business structure application or application without business structure details.

Note: older versions of the allied health professional form do not ask if the provider is claiming Medicare benefits and are not accepted. Return the form asking for the current application form to be completed.

Provider number created before 9 September 2018, request to backdate received and business structure and EFT details are not recorded

Process backdate request and send letter appropriate to level of access.

Note: assess against backdating guidelines.

Provider number issued for no claiming, now wants to claim a Medicare benefit

Allied health/non-medical health professional, other medical practitioners (104 specialty code) and medical practitioners subject to Section 19AA only:

  • If only business structure details are not present, this information may be taken over the telephone
  • If banking details, or banking details and business structure are not supplied, tell the health professional to submit an application for a provider number or complete these details in HPOS then contact Services Australia
  • Delete 618 location service rule when business structure and banking details have been supplied

Medical practitioners subject to Section 19AB:

  • Submit an application for a provider number with supporting documents

Registered Business Name is not on the provider number application

Organisation details not saving when amending or creating a provider number with a 3GA placement
  • For existing 3GA placements, take a screenshot of provider number location page then:
