2

Review related queries + Read more ...

ACETI applicants who were assessed as ineligible for the ACETI completion payments are advised in writing of this outcome. Services Australia was responsible for the eligibility assessment and notification process.

For ACETI applicants who want to apply for a review of decision, advise them:

  • the Department of Health and Aged Care is responsible for all requests to review of a decision
  • applicants must write to Services Australia (the Aged Care Payments Team) seeking the review of a decision
  • all review requests are sent to Services Australia via the Aged Care Payments Team’s external positional mailbox
  • Services Australia will forward all appeals information to the Department of Health and Aged Care
  • the Department of Health and Aged Care conducts the review
  • there are 2 outcomes of the review
    • to uphold the initial decision to reject the completion application OR
    • to overturn the initial decision (that is, process the completion application)
  • the Department of Health and Aged Care advises Services Australia of all review outcomes
  • Services Australia processes the ACETI completion application for review outcomes that results in an overturn of the initial decision

For ACETI applicants asking about the progress of their request to review a decision, advise them:

  • their request is being investigated
  • Services Australia will advise the applicant of the outcome

Procedure ends here.

3

Payment related queries + Read more ...

For ACETI applicants who can no longer submit the completion application form, advise them:

  • the ACETI program closed on 31 December 2020
  • ACETI applicants had until 30 March 2021 to submit completion payment applications to services Australia, and
  • Services Australia no longer pays completion payments

For ACETI applicants asking about the outcome of their completion application form:

  • search the ACETI system for the applicant:
    • log on to the ACETI system
    • search for the applicant by keying in the following details (surname, first name and date of birth)
  • advise the applicant of the applications outcome

Procedure ends here.

Claim processing procedure

Table 2: This table describes how to process an Aged Care Education and Training Incentive (ACETI) application following a review outcome.

Step

Action

1

Department of Health and Aged Care's email notification + Read more ...

Services Australia receives email notification from the Department of Health and Aged Care’s advice to overturn the initial decision of rejecting the Aged Care Education and Training Incentive (ACETI) completion application.

Advice is received via Aged Care programme Branch's ACETI positional mailbox.

Service Officers can only process an ACETI completion application receiving the Department of Health and Aged Care's advice.

2

Search for the applicant in the ACETI system + Read more ...

  • Log on to the Aged Care Education and Training Incentive (ACETI) system
  • Search for the claim via the applicant’s personal details:
    • surname
    • first name
    • date of birth

3

Updating applicant’s personal details in the ACETI system + Read more ...

Key all claim information into the ACETI system. On the applicant’s profile, cross check the claim details against the details recorded.

On the Personal details tab:

  • Mandatory fields
    • Surname
    • Given name
    • Date of birth
    • Sex
    • Residency status
  • Select the Validate option
  • Select the Save option
  • Optional fields
    • Title
    • Second name
    • Medicare care number
    • Culturally and linguistic diverse background,
    • Aboriginal and/or Torres Strait Islander
  • Select the Validate option
  • Select the Save option

On the Contact details tab:

  • Mandatory fields
    • Address 1 - Service Officer must record the postal address if the applicant has provided both the residential and postal addresses
    • Suburb
    • State
    • Postcode
    • At least 1 phone number
  • Select the Validate option
  • Select the Save option
  • Optional fields
    • Address 2
    • Email address
    • Other phone numbers
    • Return mail (use only when a payment advice has been returned to sender)
  • select the Validate option
  • select the Save option

4

Checking the applicant’s bank details in the ACETI system + Read more ...

Note: contact the applicant if their bank details have been incorrectly recorded on the form. Advise the applicant to resubmit the bank details in writing and also attach the original form application.

On the applicant’s profile, cross check the claim’s bank details against the details recorded on the Banking details tab.

Is there a change to the applicant’s bank details?

  • Yes, go to Step 5
  • No, go to Step 6

5

Updating the applicant’s bank details in the ACETI system + Read more ...

Update the bank details. Another Service Officer is to verify these bank details. This verification process must be done before completing the claim.

In the Banking details tab:

  • Update the fields as required
  • Select the Validate option
  • Select the Save option

6

Entering claim details on the applicant’s profile in the ACETI system + Read more ...

Select the Claim Summary tab:

  • Select the relevant claim ID
  • Select the Add completion application option
  • Update the fields as required
    • Date completion application received
    • Completion application status
    • Date information requested
  • Select the Validate option
  • Select the Save option

Select the Course details tab:

  • Update the fields underneath the Course and Organisational heading as required
  • Select the Validate option
  • Select the Save option

Select the Employers details tab:

  • Select the Update option if the employer remains the same but the supervisor’s contact details have changed
    • Select the Validate option
    • Select the Save option
  • Select the Add previous employer option if the employer’s details have changed
  • Select the Validate option
  • Select the Save option
  • Select the Comments option and add the following
    • Date of lodgement of application
    • Details of certificate of completion
    • Details of Australian Health Practitioner Regulation Agency (AHPRA) registration (if required)
  • Select the Save option

An automated letter is sent to applicants advising their application has been approved.

Procedure ends here.

In the Claims in the ACETI folder, check:
