Actioning Income Tested Care Fee (ITCF) amount mismatch queries

Table 1: This table describes steps to take when a query relating to an ITCF amount on a letter does not match the amount in the Fees and Charges tab in the Aged Care Management Payment System (ACMPS).

This process applies to Income Tested Care Fee (ITCF) queries for the pre-delimiter period (before 1 January 2020) and/or deceased care recipients.

Service Officers from the Aged Care Payments Team (Aged Care and Allied Health Branch) action.

Step

Action

1

Log on to ACMPS + Read more ...

Check the ITCF amount on the letter against the ITCF amount in the Fees and Charges tab. Aged care services should be setting the ITCF amount based on the ITCF amount on the letter.

Note: if a service is closed, Service Officers will need to contact the aged care provider to confirm which service they want the manual adjustment to be made against.

  • Log on to ACMPS
  • Select Care Recipient Search option from the top menu
  • The Search: Care Recipient screen will display
  • Key Care Recipient ID or search by:
    • Last Name
    • First Name, and
    • DOB
  • Select the Death Indicator as:
    • Not set for a current care recipient
    • Set for a deceased care recipient
  • Select Search to display the result list will display
  • Select the relevant care recipient to display the Care Recipient Profile screen

2

Check the ITCF amount on the letter against the Fees and Charges tab + Read more ...

From the Care Recipient Profile screen:

  • Select the Fees and Charges tab. Check for the ITCF amount and the effective start date the ITCF was applied
  • Select the Communication History tab. Check date letter was issued to the aged care service
  • Select the care recipient Notes tab. Check if a manual letter has been issued to the aged care service

Does the ITCF amount on the letter match the amount in the Fees and Charges tab?

  • Yes:
    • Advise the aged care service the letter is correct and the ITCF has been applied correctly
    • Procedure ends here
  • No:
    • Advise the aged care service the ITCF has not been applied correctly
    • Apply The ITCF amount on the letter to the care recipient. Go to Step 3

3

Calculating the ITCF amount to be applied + Read more ...

  • Calculate the correct ITCF amount that should be applied
  • Determine the type of manual adjustment that needs to be created
  • Escalate the manual adjustment calculations to the SME or Team Leader via email. Include evidence, for example:
  • system screenshots
  • copy of the letter
  • payment statements

The Subject Matter Experts (SME) or Team Leader create a manual adjustment and a different SME or Team Leader must verify the manual adjustment. See Table 5.

Procedure ends here.

Remediating issues when the Income Tested Care Fee (ITCF) amount has not been applied to the payment statement

Table 2: This table describes the steps to take when the ITCF amount has not been applied to the payment statement in the Aged Care Management Payment System (ACMPS).

This process applies to:

  • ITCF queries for the pre-delimiter period (before 1 January 2020) and/or deceased care recipients, and
  • cases when the ITCF amount on the letter issued matches with the ITCF amount in the Fees and Charges tab

Service Officers from the Aged Care Payments Team (Aged Care and Allied Health Branch) action.

Step

Action

1

Log on to ACMPS + Read more ...

Check the ITCF amount on the letter against the ITCF amount in the Fees and Charges tab. Aged care services should be setting the ITCF amount based on the ITCF amount on the letter.

Note: if a service is closed, contact the aged care provider to confirm which service they want the manual adjustment to be made against.

  • Log on to ACMPS
  • Select Care Recipient Search option from the top menu
  • The Search: Care Recipient screen will display
  • Key Care Recipient ID or search by:
    • Last Name
    • First Name, and
    • DOB
  • Select the Death Indicator as:
    • Not set for a current care recipient, or
    • Set for a deceased care recipient
  • Select Search. The result list will display
  • Select the relevant care recipient to display the Care Recipient Profile screen

2

Check the ITCF amount on the letter against the Fees and Charges tab + Read more ...

From the Care Recipient Profile screen:

  • Select the Fees and Charges tab. Check the ITCF amount and the effective start date the ITCF was applied
  • Select the Communication History tab. Check when the letter was issued to the aged care service
  • Select the care recipient Notes tab. Check if a manual letter has been issued to the aged care service

3

Check the ITCF amount on the payment statement + Read more ...

  • Select Home at the top right corner, to display the Home screen
  • In the Printing title on the right side, select Payment Statement Reprint
  • The Search for Documents screen will display
  • Key the service ID number in the Service BP ID field
  • The list of payment statements will display in the Documents screen
  • Select the relevant monthly payment statement. Select Print Payment Statement to view the monthly payment statement
  • Check the care recipient’s date of entry to the latest payment statement to see if the ITCF amount has been applied to the payment statement

Has the ITCF amount on the letter been applied to the payment statement?

  • Yes:
    • Advise the aged care service the ITCF amount on the letter has been applied to the payment statement
    • Advise the aged care service of the relevant payment statement the ITCF amount was applied to
    • Procedure ends here
  • No, go to Step 4

4

Identify if the care recipient is still in care + Read more ...

Determine if the care recipient is still in care.

Is the care recipient still in care?

  • Yes, this is a system issue. Escalate to ICT via mySupport. Go to Step 5
  • No:
    • The care recipient is either deceased or have departed from the service
    • Do a manual adjustment calculation to ensure the correct ITCF amount has been applied for the relevant period. Go to Step 8

5

Escalating ACMPS issues with ICT via mySupport + Read more ...

Escalate issues with ACMPS not applying the ITCF amount to the payment statement to ICT via mySupport.

To escalate the issue with ICT:

  • Open mySupport
  • In the Search field, key in ‘ACMPS issues’ to display search result
  • Select Aged Care Management Payment System (ACMPS) issues to display the mySupport form
  • Complete the form. Note: include the ACMPS ID in the What is the ACMPS ID field
  • Take relevant screen shots (pdf or word document format)
  • Select Add Attachment to attach relevant screen shots
  • Select the Open button to display the attached screen shots
  • Select Submit
  • SD number will be sent via email

6

Add care recipient notes + Read more ...

Add care recipient notes in ACMPS detailing reasons for escalating the issues See Manage aged care recipient details to add care recipient notes.

7

Advise the aged care service + Read more ...

Advise the aged care service that the query has been escalated to ICT.

On receipt of the ICT response and resolution:

  • advise the aged care service that:
    • the issue has been rectified, and
    • the ITCF amount has been applied to the payment statement
  • update the care recipient note stating that the ICT escalation has been resolved

Procedure ends here.

8

Calculating the ITCF amount to be applied + Read more ...

  • Calculate the correct ITCF amount to apply
  • Determine the type of manual adjustment to create
  • Escalate the manual adjustment calculations to the SME or Team Leader via email. Include evidence, for example:
    • system screenshots
    • copy of the letter
    • payment statements

The Subject Matter Experts (SME) or Team Leader will create a manual adjustment. A different SME or Team Leader must verify the manual adjustment. See Table 5.

Procedure ends here.

 

Remediating issues when the letter has not been issued between the Quarterly Review (QR) period

Table 3: This table describes the steps to take when the Income Tested Care Fee (ITCF) amount has been correctly applied in the Aged Care Management Payment System (ACMPS) between the QR periods, but no letter has been issued to the aged care service.

This process applies to:

  • ITCF queries for the pre-delimiter period (before 1 January 2020) and/or deceased care recipients, and
  • cases where the ITCF has been applied correctly in the Fees and Charges tab and the payment statement

Services Officers from the Aged Care Payments Team (Aged Care and Allied Health Branch) can action.

Step

Action

1

Check the ITCF between the quarterly review dates + Read more ...

Check if the ITCF letter has been issued to the aged care service between the QR dates. If the ITCF letter has not been issued, apply the ITCF amount from a previous letter.

In ACMS, check for the ITCF amount and the effective start date in the Fees and Charges tab for the specific period (between QR effective dates).

Note: if a service is closed, contact the aged care provider to confirm which service they want the manual adjustment to be made against.

  • Log on to ACMPS
  • Select Care Recipient Search option from the top menu
  • The Search: Care Recipient screen will display
  • Key Care Recipient ID or search by:
    • Last Name
    • First Name, and
    • DOB
  • Select the Death Indicator as:
    • Not set for a current care recipient, or
    • Set for a deceased care recipient
  • Select Search to display the result list.
  • Select the relevant care recipient. The Care Recipient Profile screen will display
  • From the Care Recipient Profile screen, select the Fees and Charges tab to check:
    • ITCF amount, and
    • the effective start date the ITCF that was applied

2

Check if the ITCF letter has been issued to the aged care service + Read more ...

  • Select the Communication History tab. Check if the letter was issued to the aged care service
  • Select the care recipient Notes tab. Check if a manual letter was issued to the aged care service

Has the letter been issued to the aged care service when the ITCF was applied between the QR effective dates?

  • Yes:
    • The ITCF letter has been issued at the time the ITCF calculation was made
    • No manual adjustment required as ACMPS has applied the ITCF correctly
    • Procedure ends here
  • No, apply the ITCF amount on the previous letter up to when the next letter will be issued to match the amount in the Fees and Charges tab. Go to Step 3

3

Check the payment statement + Read more ...

Check the payment statement from the start of QR effective date up to the latest payment statement to see what ITCF as applied.

In ACMPS:

  • Select Home at the top right corner, to display the Home screen
  • In the Printing title on the right side, select Payment Statement Reprint
  • The Search for Documents screen will display
  • Key the service ID number in the Service BP ID field
  • The list of payment statements will display in the Documents screen
  • Select the relevant monthly payment statement. Select Print Payment Statement to view the monthly payment statement
  • Check the provider notes to see if any manual adjustments may apply to the care recipient

4

Calculating the ITCF amount to be applied + Read more ...

  • Calculate the correct ITCF amount to apply
  • Determine the type of manual adjustment to create
  • Escalate the manual adjustment calculations to the SME or Team Leader via email. Include evidence for example:
    • system screenshots
    • copy of the letter
    • payment statements

The Subject Matter Experts (SME) or Team Leader will create a manual adjustment. A different SME or Team Leader must verify the manual adjustment. See Table 5.

Procedure ends here.

Remediating issues when a letter has not been issued or issued after 6 months after the care recipient's date of entry

Table 4: This table describes steps to take when the ITCF has been applied in the Aged Care Payment System (ACMPS), but the initial letter was not issued, or was issued after 6 months from the care recipient's date of entry.

This process applies to:

  • ITCF queries for the pre-delimiter period (before 1 January 2020) and/or deceased care recipients, and
  • cases where the ITCF amount on the letter matches the amount in the Fees and Charges tab

Services Officers from the Aged Care Payments Team (Aged Care and Allied Health Branch) action.

Step

Action

1

Check the ITCF amount on the initial letter issued to the aged care + Read more ...

  • Check if the initial ITCF letter was issued advising the aged care service of the ITCF amount within 6 months from the care recipient's date of entry

Log onto ACMPS:

  • Select Care Recipient Search option from the top menu
  • The Care Recipient Search screen will display
  • Key Care Recipient ID or search by:
    • Last Name
    • First Name, and
    • DOB
  • Select the Death Indicator as:
    • Not set for a current care recipient, or
    • Set for a deceased care recipient
  • Select Search to display the result list
  • Select the relevant care recipient to display the Care Recipient Profile screen
  • Select the Fees and Charges tab. Check the ITCF amount and the effective start date ITCF was applied
  • Select the Communication History tab. Check if a letter was issued to the aged care service
  • Select the care recipient’s Notes tab for further information if a manual letter issued to the aged care service

Has the initial letter been issued to the aged care service advising them of the ITCF amount within 6 months from the care recipient’s date of entry?

  • Yes, go to Step 2
  • No, go to Step 4

2

Check the ITCF amount on the payment statement in ACMP + Read more ...

Check the payment statement to ensure the ITCF has been applied to the payment statement.

In ACMPS:

  • Select Home at the top right corner to display the Home
  • In the Printing title on the right side, select Payment Statement Reprint
  • The Search for Documents screen will display
  • Key the service ID number in the Service BP ID field
  • The list of payment statements will display in the Documents screen
  • Select the relevant monthly payment statement. Select on Print Payment Statement to view the monthly payment statement
  • Check the care recipient’s date of entry to the latest payment statement to see if the ITCF amount has been applied to the payment statement

Has the ITCF amount been applied to the payment statement?

  • Yes, go to Step 3
  • No:
    • To fix the issue See Table 2
    • When the ITCF amount has been applied to the payment statement, procedure ends here.

3

Advise the aged care service + Read more ...

Advise the aged care service:

  • letter issued at the time of the ITCF calculation, and
  • the ITCF amount has been applied to the payment statement and matches the letter issued

Procedure ends here.

4

Refunding the ITCF amount back, CSR to zero is applied + Read more ...

If the letter has not been issued or has been issued advising of the ITCF after 6 months from the care recipient’s date of entry:

  • Care Subsidy Reduction (CSR) to zero applies
  • ITCF will not be applied to the care recipient
  • the aged care service is entitled to a refund

Calculate a positive manual adjustment of the ITCF amount to be refunded back to the aged care service. The aged care service must apply the refund back to the care recipient.

  • Calculate the correct ITCF amount to be applied
  • Determine the type of manual adjustment that needs to be created
  • Escalate the manual adjustment calculations to the SME or Team Leader via email. Include evidence, for example:
    • system screenshots
    • copy of the initial letter (if relevant)
    • payment statements

The Subject Matter Experts (SME) or Team Leader create a manual adjustment. A different SME or Team Leader must verify the manual adjustment. See Table 5.

Procedure ends here.

Creating and verifying a manual adjustment for income Tested Care Fee (ITCF)

Table 5: For Subject Matter Experts (SME) and Team Leaders only. This table describes what steps to take when a manual adjustment is required to ensure the ITCF is applied correctly in the Aged Care Management Payment System (ACMPS).

This process applies to ITCF queries for the pre-delimiter period (before 1 January 2020) and/or deceased care recipients.

Step

Action

1

Checking manual adjustment calculations + Read more ...

Check if the manual adjustment calculation is correct.

Is the manual adjustment calculation correct?

  • Yes, go to Step 2
  • No, advise the Service Officer to recalculate the manual adjustment.

When the Service Officer has provided the correct manual adjustment calculation, go to Step 2.

2

Checking evidence provided + Read more ...

Check if the relevant evidence provided is correct, for example:

  • system screenshots
  • copy of the letter
  • payment statements

Is the evidence correct?

  • Yes, go to Step 3
  • No, advise the Service Officer to provide the correct evidence

When correct evidence has been provided, go to Step 3.

3

Creating a manual adjustment - actioned by SME or Team Leader + Read more ...

To create a positive manual adjustment, see Maintain positive manual adjustments in ACMPS

To create a negative manual adjustment, see Maintain negative manual adjustments in ACMPS

In the manual adjustment note, add:

  • care recipient name
  • care recipient ID number, and
  • the period of the manual adjustment

Add a care recipient note stating what the payment is for and the period of the manual adjustment.

4

Verify the manual adjustment + Read more ...

Once the manual adjustment has been created, a different SME or Team Leader must verify the process. See Verify a manual adjustment in ACMPS to verify or approve a manual adjustment.

5

Advise the Service Officer - verifying SME or Team Leader + Read more ...

Advise the Service Officer to contact the aged care service to advise:

  • the manual adjustment has been actioned, and
  • the ITCF has been applied correctly

6

Annual and lifetime cap adjustments + Read more ...

Verifying SME or Team Leader:

  • email the Age Care Programmes Branch via the Care Recipients section positional mailbox requesting they update the care recipient’s annual and lifetime cap record. See the Resources page for contact details

The SME or Team Leader provides the:

  • care recipient details
  • aged care service details
  • amount and the breakdown of the manual adjustment

Procedure ends here.
