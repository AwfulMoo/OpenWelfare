Using the NECC portal

Item

Description

1

Access the portal + Read more ...

Access the portal from the NECC website.

On entry, the Please Read screen will display. This screen has important messages that all Service Officers must be aware of before recording details of a call.

Read all information on the Please Read screen before selecting the Continue button.

2

Procedures window + Read more ...

The Procedures window gives access to instructions for answering calls.

Access this window by selecting the Procedures link on the right-hand side of each page. A new window will open to show current procedures.

As procedures may change or update at any time during an emergency, Service Officers must check this window often for the current situation for each emergency.

Procedures may be:

  • generic
  • apply to all events within the portal, or
  • emergency specific

Select Close Window to exit Procedures.

3

Information link + Read more ...

The Information window gives access to scripts for answering calls.

Access this window by selecting the Information link on the right-hand side of each page. A new window will open to show current details.

The Information link:

  • is available when recording a call
  • displays current information for all emergencies within the portal, and
  • must be referred to regularly between calls, as information update often during an emergency

Select Close Window to exit Information.

4

Information tab + Read more ...

If a caller asks for information about an emergency, Service Officers must indicate this on the Call Reasons tab.

To activate the Information tab and the Provide Information button:

  • select the call reason - 'caller is requesting information'
  • select the Information tab and all information about the specific emergency will show to discuss with the caller

Note: the 'caller is requesting information' option may not be available for all activations.

5

Alerts + Read more ...

Alerts are brief messages relating to urgent and important information. These messages display above the Record Call header.

Full details of an issue show on the Information or Procedures pages.

6

Recording a call + Read more ...

The NECC portal prompts the Service Officer to select the relevant emergency. The Call Reasons tab will then show.

The information needed will tailor to the emergency and may include the:

  • reason why a caller is calling
  • location of the caller

To record the details of a call, select the emergency that the call relates to and fill in the fields.

7

Call reasons tab + Read more ...

Use the Call Reasons tab to work out if there is a need to collect and/or give details based on the caller's needs.

If a task needs details:

  • collected, select the Record Information tab and collect all relevant details from the caller
  • given, select the Information tab and give all relevant details to the caller

8

Call summary screen + Read more ...

When all details have been collected and given, select the Show Summary button. This will show all the details from the call.

Select:

  • Edit Call Record, to edit the record, or
  • Save and Finalise the Record button to finalise

9

Cancelling the recording of a call + Read more ...

At any time during the recording of a call, Service Officers can cancel the record.

Select the Cancel Call button on the lower right-hand side of the record.

Note: all details entered, will be lost.

10

Log out from the NECC portal + Read more ...

Select the Logout button in the top right of any screen in the portal.

Do not log out while in the process of recording a call, as any call data entered, will be lost. Complete or cancel the call.
