Functions available in the HPOS messages and Support Mail Centre

Table 1

System

Users

Functions available

HPOS messages

External - health professional and their delegates
  • Compose new mail to a predefined list of internal mailboxes (ACIR, DVA, eBusiness Helpdesk, Fraud, GPII, Healthcare Identifiers, IVF/GIFT program, Medicare, MPIS, PBS, PIP, WIP – Practice Stream, Track and Scale)

  • Search for mail within the File upload Mail Centre

Fields displayed in the Mail Centre listing screen

Table 3

Field

Description

Selection column

The selection column displays a check box that can be selected if moving the mail message within the Mail Centre.

Attachments

If a paperclip image is displayed in this column it indicates that the message has an attachment.

Message Direction

The message direction displays the last action on the message.

Outgoing message:

Incoming message:

Practice/Provider

Displays the name of the person that has sent the message.

If the message has been sent by a provider, the provider stem number is also displayed.

Subject

The subject of the mail thread.

Update Date/time

The date and time of the last update to the mail thread.

Ref. No.

The HPOS Reference number of the mail message.

Each mail thread is allocated a unique HPOS Reference number and each subsequent reply to that mail thread is allocated a sequence number.

Example: 12458-02 is mail message 12458, with sequence number 02.
