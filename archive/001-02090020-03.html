Making a referral to SEE

Action

1

Determine if job seeker is Centrelink or provider managed + Read more ...

The Customer Information field on the Activity and Exemption Summary (AEX) screen shows if a customer is Centrelink or provider managed. Note: volunteers must be registered as volunteer job seekers before a referral can be made.

In Process Direct:

  • select Customer Summary from the Process Direct Landing page

2

Centrelink managed job seekers + Read more ...

The Service Officer must check the following screens in Process Direct before starting the referral process. This assists with determining eligibility:

Citizenship and Visa sub class

  • key 'RSLEG' in the Super Key, select Enter
  • the Legal Residency Details screen will display

Permanent visa details

  • key 'RSRD' in the Super Key, select Enter
  • the Residence Results Display screen will display

End date of the Newly Arrived Resident's Waiting Period (NARWP)

  • key 'RSPAP' in the Super Key, select Enter
  • the Period of Australian Presence screen will display

Job seekers with mutual obligation requirements:

  • advise these job seekers of their obligation to participate in SEE, if assessed as requiring training at the Pre-Training Assessment
  • use the Create SEE Referral workflow in Process Direct to refer them to SEE and arrange a Pre-Training Assessment appointment with the SEE provider while the job seeker is present

See Step 1 in Table 2.

3

Provider managed job seekers + Read more ...

Customer is eligible for SEE.

  • Tell the job seeker they are eligible to be referred to SEE. If the job seeker agrees to the referral, contact the job seeker's Employment Services Provider
  • Discuss any impact on the requirements already in the Job Plan, if a referral is made to SEE. This discussion can be with any staff member of the provider, not just the job seeker's Case Manager
  • If the provider has reasonable grounds to oppose a referral, tell the job seeker to contact their Case Manager. Do not proceed with the referral

If the provider agrees, continue with the referral:

  • Obtain the jobseeker's consent to pass on their personal details to the SEE provider
  • Use the Create SEE Referral workflow in Process Direct to refer the job seeker to SEE

See Step 1 in Table 2.

Action

1

Start the Create SEE Referral + Read more ...

In Process Direct:

  • key START in the Super Key
  • the START screen will display
  • from the Task Selector, select Create SEE Referral
    • Note: Create SEE Referral appears on the Referral Summary (RRSUM) screen but is not a functional referral option - only refer using the Create SEE Referral action from the Task Selector

  • A message will display advising if the customer is eligible or not eligible for SEE

If the job seeker is:

  • not eligible, go to Step 3

  • Yes, go to Step 5
  • No, go to Step 4

3

Job seeker is not eligible for SEE + Read more ...

A job seeker is not eligible when they:

  • are in New Enterprise Incentive Scheme (NEIS) or Self Employment Assistance
  • are in Adult Migrant Education Program (AMEP)
  • have recently participated in SEE and the last program exit date is within 12 weeks of the next referral date:
    • When it's a new provider, the workflow completes and creates a DOC advising the date when the job seeker can be referred back to SEE
    • Job seekers wanting to go back to their previous SEE provider can contact the provider themselves to re-start SEE
    • When a Service Officer attempts to code another referral to the same provider within the 12 week period will receive the following error ''E409RR - Not Eligible, can be referred to SEE after <Due Date>'. These job seekers are not eligible for a new referral by Services Australia before the date recorded against the previous finalisation
  • have recently successfully completed SEE. This is shown by the latest SEE status reason of Lang/Lit/Num Successful on the Referral Summary screen. Customers who have completed the program will be eligible again after a 12 week break
  • for other reasons, the workflow completes and creates a DOC with the text Customer is not eligible for SEE

Job seeker has changed address:

The job seeker can:

  • contact their previous provider to advise they have relocated. They should request their record be updated in the provider's system with Transfer to new provider. Once updated, a referral to the new provider is possible
  • wait until 12 weeks from their last exit date, then seek a new referral, or
  • resume with the previous SEE provider by contacting the provider directly

Job seeker wants to change SEE providers:

  • wait until 12 weeks from their last exit date, then seek a new referral

Finalise the transaction:

  • Select Next
  • The Referral Summary (RRSUM) screen will display a provisional line and list the referral ineligibility reason under the Status reason column
  • Select Next
  • The Errors (SWE) screen will display
  • Finalise the transaction by selecting Assess again, then select Finish. Record details of the update on the Finalise screen
  • Select Finalise to complete the transaction

Procedure ends here.

4

No suitable providers are available + Read more ...

Ask the job seeker if they can attend a SEE provider at a nearby location. A job seeker may choose to undertake the program at another location due to:

  • transportation difficulties
  • a disability
  • caring responsibilities
  • cultural constraints
  • no accessible SEE provider

To search for other SEE providers in nearby locations, select Yes in Include nearby regions. The list expands to include SEE providers in surrounding regions.

If the job seeker wishes to choose a provider outside their surrounding region, select Provider Search to manually search by region code, postcode or town/suburb.

Is a referral to another suitable SEE provider an option for the job seeker?

  • Yes, go to Step 5
  • No, when there are no suitable service providers available for the job seeker, a service gap is to be recorded:
    • From the Record service gap field select Yes
    • Select a service gap reason from the drop down list options. When this option is selected, the list of service providers is disabled
    • Select Next to continue the workflow
    • Select Yes to Are you sure you want to record the service gap?
    • Select Next and address any errors/warnings
    • Finalise the transaction by selecting Assess, then select Finish. Record details of the update on the Finalise screen
    • Select Finalise to complete the transaction

Procedure ends here.

5

Suitable provider is available + Read more ...

Ask the job seeker if they are interested in undertaking SEE by distance education, if the suitable provider list includes distance education providers. If not, only select local providers.

To search for a provider:

  • complete any of Search criteria fields and select Search Providers
  • select the appropriate Service provider from the Action column in the Service providers table

Arrange an appointment with the SEE provider for the Pre-Training Assessment. This is mandatory for all SEE referrals. Ensure the job seeker knows of the compliance consequences if they do not attend their Pre-Training Assessment appointment.

Service Officers should not discuss the job seeker's circumstances with the provider. Only disclose sufficient information to book the appointment.

If referring to a distance education provider, go to Step 6.

If referring to local providers:

  • check if the job seeker has a silent phone number – if so, seek their consent to pass the number to the provider over the phone as the referral system will exclude it from the online referral
  • phone the provider and make a Pre-Training Assessment appointment
  • advise the job seeker of the date, time and location of the appointment

Go to Step 7.

6

Referral to distance education provider + Read more ...

Job seekers may choose to undertake SEE by distance learning.

For all SEE courses through a distance education provider, job seekers require:

  • a referral from Services Australia or an Employment Services Provider, and
  • an assessment and phone interview with the distance education provider to assess language or literacy and numeracy needs

Services Australia initiated referral to a distance education provider:

  • Contact TAFE SA (Distance) to give the job seeker's details
  • Ensure the job seeker is present to obtain consent to disclose their relevant personal details
  • Advise the job seeker that the provider will contact them to arrange a Pre-Training Assessment appointment either by direct visit, online, telephone interview and/or mailed response

Job seeker initiated referral to a distance education provider

If the job seeker has already contacted TAFE SA (Distance) and requires a referral from Services Australia:

  • confirm the job seeker is eligible for SEE
  • contact the distance education provider, and
  • record the referral via the Create SEE Referral

For further details about SEE distance education providers, including site codes, see the link on the Resources page.

Go to Step 7.

7

Code referral to SEE + Read more ...

Complete the following fields on the Create Referral screen.

The Create referral screen will display.

Complete the following fields using the drop down menu:

  • Channel Code - the method the job seeker used to contact Services Australia
  • Requested by - the person/organisation requesting the referral (for example, Services Australia, Employment Services Provider)
  • Reason for referral - the reason why the job seeker has been referred to this Service Type
  • Due Date - this automatically populates

The Referral Status table shows:

  • Status - Provisional is displayed, as the initial referral status until referral is completed
  • Action date of effect - the date the referral was created
  • Status type code - the reasons for a provisional referral status will be displayed
  • Elig check for the eligibility check
  • Referred for the pending referral
  • Status reason:
    • Awaiting contact, this is for the service provider to contact the customer
    • Eligibility check for skills for education and employment displays until the service provider has completed assessment of eligibility of SEE course
  • If a referral is being made to a SEE provider outside the job seeker's region, use Edit under Action and choose the appropriate Status reason from the following options:
    • Client Preference
    • Family Reasons
    • No Service Available
    • Significant Disability
    • Travel Distance
  • Select Save

Only the service provider can be changed by selecting Previous and selecting a new provider before confirming the referral. Status type code and Status reasons are only able to be altered by staff with specific access.

Select Next. The workflow returns to the Referral Summary showing the provisional record just created.

Select Next and address any issues on the SWE screen.

Finalise the transaction by selecting Assess, then select Finish. Record details of the update on the Finalise screen.

Select Finalise to complete the transaction.
