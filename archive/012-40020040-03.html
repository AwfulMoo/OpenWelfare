Table 1: this table describes how to resolve error message 1-SCP#. This displays when a specimen collection point (SCP) number has been entered incorrectly or in the wrong format.

Step

Action

1

Examine account documents + Read more ...

2

Locate the SCP + Read more ...

4-PATH RQ error message

Table 4: this table describes how to resolve 4-PATH RQ error message. This displays when a pathology request (PA) line or processing indicator (PI) 11 has not been keyed for a pathology item from Group P1 to Group P8.

Step

Action

1

Examine account documents + Read more ...

Examine the account documents for a request indication.

2

Action request details + Read more ...

  • If request details are found, key a pathology request (PA) line containing details of the request and press [Enter]
  • If the service was self-determined, key processing indicator (PI) 11 (pathology service has been self-determined) in the PI field against the items involved and press [Enter]
  • If request details are not on the account and the services are not self-determined, reject the item involved with RSN 137 (details of requesting provider not shown on account or receipt) and press [Enter]

4-CHK RQST error message

Table 5: this table describes how to resolve 4-CHK RQST error message. This displays when a pathology request (PA) line is keyed without an associated pathology item or does not require a PA line.

Step

Action

1

Determine if there are associated pathology items + Read more ...

Examine the details keyed on the screen and those shown on the account documents to determine whether there are associated pathology items.

2

Action pathology items + Read more ...

  • If the associated pathology items from Group P1 to Group P8 are on the screen:
    • rekey details of the pathology request (PA) line immediately after the associated pathology items involved
    • delete the original PA line
  • If the associated pathology items from Group P1 to Group P8 are on the account and have not been keyed, insert item lines containing the pathology items immediately before the PA line involved
  • If there are no associated pathology items from Group P1 to Group P8 on the screen or on the account, delete the PA line involved
  • If the associated pathology item lines contains processing indicator 11 (pathology service has been self-determined):
    • erase processing indicator (PI) 11 from the associated item lines, or
    • delete the PA line involved
  • If processing an online claim, follow the error message and reject the claim

Press [Enter].

6-CMP error message

Table 6: this table describes how to resolve 6-CMP error message. This displays when a composite situation is detected during processing. The composite item that must be paid shows in the message and the individual items are highlighted. This table also describes circumstances where the pathology account contains the notation Rule 3 Exemption or S4B(3).

Step

Action

1

Claiming channel + Read more ...

  • Online claims, see Medicare online claiming – 6 CMP
  • Manual claims, go to Step 2

2

Examine account documents + Read more ...

Examine the account documents for any indication that the items involved can be paid individually.

3

Determine pathology service type + Read more ...

Determine if the services are:

  • Composite services, go to Step 4. Note: pathology services are considered to be composite when the account documents:
    • do not contain the notation ‘Rule 3 Exemption’ or ‘S4B(3)’ which allows the items to be paid individually
    • contain the notation ‘Rule 3 Exemption’ but the items claimed and circumstances involved do not allow an exemption to be granted
    • contain the notation ‘S4B(3)’ but the delegate determines that the items are not to be paid individually
  • Non-composite services, go to Step 5. Note: pathology services are considered to be non-composite when there is a notation on the account documents which allows the items to be paid individually (that is, ‘Rule 3 Exemption’ or ‘S4B(3)’)

4

Action composite services + Read more ...

  • Write down the composite item number displayed in the message
  • Identify the items that are to be rejected (highlighted by the system)
  • Reject these items with RSN 370 (benefit paid on item number other than that claimed)
  • Insert an item line before the associated pathology request (PA) line involved
  • Key details of the composite item on the blank item line with the charge being the total of all the amounts claimed for the items being rejected

Press [Enter].

5

Action non-composite services + Read more ...

For items with notation ‘Rule 3 Exemption’ or ‘R3EX’ key:

  • processing indicator (PI) 41 in the PI field
  • RSN 366 (benefit paid for Rule 3 Exemption item) (in the RSN field)

Press [Enter].

Forward ‘S4B(3)’ claims to a Team leader.

  • After the S4B(3) delegate determination process, determine whether the items are payable. If payable key:
    • processing indicator (PI) 41 in the PI field
    • RSN 365 (benefit paid for S4B(3))

Press [Enter]

  • Out-patient claim with tests endorsed ‘R3EX’, reject with RSN 162

6-ASC NOT

Action + Read more ...

The item can only be paid when another associated item has also been paid

  • Reject the item with RSN 550

6-RST PEI

Action + Read more ...

A patient episode initiation (PEI) has already been paid for the same PA date

Only one PEI is payable patient episode (per PA date), unless:

  • associated ‘R3EX’/’S4B(3)’ endorsed items with either PEI, or
  • either PEI is associated with a Group P5 or P6 test performed by a different approved pathology authority (APA), and the other APA has not performed a service from that relevant Group - P5 or P6

Check the PEI claimed or on history is by a different APA with tests:

  • 72813-72860 and the other APA has not claimed tests 72813-72860, or
  • 73043-73076 and the other APA has not claimed tests 73043-73076
    • Yes, pay the claim using PI 42
    • No, reject the PEI with RSN 160
  • If the PEI has no associated payable pathology tests, reject with RSN 353

6-RST TD

Action + Read more ...

The item has reached the limit claimable

Does the item 65120 has a text/notation R3EX?

  • Yes, pay the claim using PI 45
  • No, reject RSN 160

Is the PEI claimed is by a different approved pathology authority (APA) with tests 72813-72860 or 73043-73076

  • Yes, pay the claim using PI 45
  • No, reject RSN 160

Pathology and pend message assessment – restrictive items

Table 8: this table describes pathology pend message assessment for restrictive items.

Pend message

Business rule or principle

6-RST-ADJ

Action + Read more ...

These items are restrictive when performed in the same episode or under the same request.

The item is restrictive with another item that has a lower fee. Normally, the item with the highest fee is payable.

  • Pay using PI 42 for:
    • PEIs and/or incentives with payable pathology tests, or
    • either restrictive item when endorsed ‘R3EX’
  • For pathology tests, process an adjustment to pay the balance owing: subtract the low fee from the high and pay that amended fee with PI 58
  • For pathology items in digital self-service claims, redirect online claims to Complex Self Service (CS) queue. When redirecting a claim, include message text detailing the reason for the redirection
  • If a claim is incorrectly redirected, the claim is returned to the Service Officer with operator text explaining why
  • If there is no notation or notation doesn't support payment, reject with RSN 179 or other reason code as applicable

6-RST-REJ

Action + Read more ...

These items are restrictive when performed in the same episode or under the same request.

  • Pay using PI 42 for:
    • PEIs and/or incentives with payable pathology tests, or
    • either restrictive item when endorsed ‘R3EX’

Reject pathology items with RSN 159 or other reason code as applicable and continue to process the claim

8-AUTO-CON

Action + Read more ...

The system has automatically adjusted one or more of the fees in the claim due to coning

RSN 174 will be automatically allocated to the services that have been adjusted for Pathology coning rule

8-CHK-R3EX

Action + Read more ...

Is ‘Rule 3 Exempt’ or ‘R3EX’ notated in the claim?

  • Yes, key RSN 366
  • No, reject using RSN 173

8 PATH CON

Action + Read more ...

Maximum number of services paid

  • Three pathology test items with the highest schedule fee have been paid and,
  • The other items have been rejected by the system using RSN 173

8-CHK PEI

Action + Read more ...

Only one PEI is payable per patient episode.

Exemptions to this are:

  • the items are associated with a PEI on the patient history that has been paid with RSN 365 or 366, or
  • one of the items on the patient history or in this claim is from Group P5 or P6, and
    • the item(s) in the current claim are not in the same group as those on history

8-RST CLM

Action + Read more ...

These items are restrictive when performed in the same episode or under the same request.

  • If the restrictive item/s are:
  • in the same episode, even if endorsed R3EX, press [PF9] to accept rejection by the system with RSN 179
  • not in the same episode and either item is endorsed R3EX, apply PI 42
  • For item 73940, refer to the specimen referred rules
  • Otherwise, press [PF9] to accept rejection by the system with RSN 179

8-RST-HIST

Action + Read more ...

These items are restrictive when performed in the same episode or under the same request.

  • If either item is endorsed R3EX, apply PI 42
  • For item 73940, refer to the specimen referred rules
  • Otherwise, press [PF9] to accept rejection by the system with RSN 179
  • If all pathology items are rejected, PEI items are not paid on their own. Reject the PEI and any other incentive item with RSN 500 or other reason code as applicable

See the Resources page for more information on common reason codes and quick reference information.
