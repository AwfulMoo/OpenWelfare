Registration forms

Table 1: this table describes the aged care online claiming registration forms.

The form…

is used when …

Register, amend or remove users for Aged Care Provider Portal form (AC004)

an aged care service wants to register for aged care online claiming using web forms so they can:

  • view submitted web forms
  • view care recipient information
  • view a care recipient profile including associated web forms
  • view current and historical claims
  • view latest and historical payment statements
  • register (new), update (correct) and delete (reverse) web forms
  • finalise current claims

Register or update for Aged Care web services form (AC027)

an aged care service wants to register or amend details for Aged Care Web Services for providers using developer software so they can:

  • transmit aged care claims and events electronically using web services software
  • view data that has been transmitted by the service and its associated processing status
  • browse payment statements
  • browse and finalise monthly claims
  • view care recipient information

  • The Dev Support team in Services Australia is responsible for this registration process.

Information required in the aged care online claiming registration forms

Table 2: this table explains the information required in each section of the AC004 form.

Section

Description

Service Officer actions

Important information/when to use this form

Provides details about how to complete the form as well as:

  • information about authorisation levels for nominated users
  • Services Australia’s contact details for assistance completing the form
  • form lodgement details

Reason for application

For:

  • registration for online claiming form the service must indicate if the application is to:
    • amend an existing registration (to an application previously lodged)
    • create a new registration
  • change of user's details form the service must indicate if the application is to:
    • register a new user
    • update an existing user access level
    • deactivate an existing user

Note: more than one option can be selected. If the service is only deactivating ‘A users’, an email can be sent with the deactivation details to Aged Care liaison.

If an amendment is indicated, look for the changes in the form. A phone call to the service might be required to clarify.

Service details

Note: all fields in this section must be completed. A form is completed for one service only. If there are additional services for a provider, additional forms are required.

Service ID/Organisation ID is:

  • Residential services: up to 4 numbers, for example 3021
  • Home Care services: up to 5 numbers, for example 10246
  • Flexible services: up to 5 numbers, for example 13568

For registration for online claiming forms, a contact person must be nominated for the service. They can be registered as a nominated user in this section.

Nominated user's details

At least 2 staff must be registered by the service as nominated users. It is wise to nominate back up staff in case somebody is absent. All nominated users registered will be issued a user number (also known as the Aged Care user ID).

Note: there is room to nominate up to 4 people at a service. If more people need to be nominated, photocopy this page and include as extra pages.

For aged care online claiming registration, the authorisation levels required by the user must be indicated in the Authorisation levels section. Refer to page 1 of the form for definitions of access levels.

Date of effect should be completed with the date the user's authority will commence. End Date with service is only required if the nominated user has access for a specific period.

Service contact person(s) details

Only one person from the service can be nominated as the contact person.

Update the contact persons in the aged care system as the contacts for notification of payment certification.

Declaration

This section must be signed and completed by an authorised person of the approved provider to whom the service belongs.

The form cannot be accepted without a correct and valid signature and all relevant fields on the page completed.

Terms and conditions

The aged care online claiming terms and conditions must be read and understood by the service before completing and signing the form.
