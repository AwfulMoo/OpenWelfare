HP Records Manager access

Request access to HP Records Manager

Table 1: This table outlines how to request access to HP Records Manager.

Step

Action

1

Resource request + Read more ...

Throughout this procedure, it is assumed that Programme Support Officers already have access to HP Records Manager. For access to HP Record Manager, create a Resource Request through the ICT Security Portal.

The Resources page contains Intranet links to assist with the Resource Request.

If further information is required, guidance in the use of HP Records Manager can be found on the Intranet as well as e-Learning modules in ESS. The Training and Support page contains links to these training resources.

2

Request access to eHealth folders within TRIM + Read more ...

After requesting access to HP Records Manager, email Tier 3 provider support requesting access to the relevant eHealth folders within TRIM (these folders are restricted to those staff that require access to them).

Once access is granted, continue archiving process. See Step 1 in the Save completed My Health Record provider application in HP Records Manager table.

Save completed applications in HP Records Manager

Save completed My Health Record provider applications in HP Records Manager

Table 2: This table describes the process Service Officers follow when saving completed My Health Record provider applications in HP Records Manager.

Step

Action

1

Navigate to network folder + Read more ...

Open the network folder containing the My Health Record provider which needs to be stored in HP Records Manager.

For example: G:\Business Outreach Services\HI ehealth\My Health Record\Processed

Note: if require access to HP Records Manager, see the Request access to HP Records Manager table.

2

Select appropriate application + Read more ...

  • Right click on the My Health Record provider application which requires electronic filing
  • Select Send to and then select HP Records Manager
  • Ensure the Document option is selected and select the OK button

3

Amend name + Read more ...

In the New Record Document screen, amend the title name in the Title (Free Text Part) field as required.

Ensure the naming is in line with the naming convention located on the Resources page.

4

Complete HP Records Manager container details + Read more ...

In the Put On What File? field, type the HP Records Manager container that the electronic document needs to be filed into (as per the naming convention located on the Resources page).

5

Finalise + Read more ...

Select OK.

The My Health Record provider is saved in HP Records Manager with a HP Record Manager Record Number (for example, 13/199999)

To continue process, see Step 1 in the Record a TRIM HP Records Manager number in the My Health Record Tracking Database table.

Record TRIM HP Records Manager number in the Tracker

Record a TRIM HP Records Manager number in the Combined Tracker

Table 3: This table describes how Programme Support Officers record the HP Records Manager Number in the My Health Record Tracking Database (the Tracker). The Tracker is maintained by Programme Support/Service Officers and is used to record all applications (both Healthcare Identifiers (HI) and My Health Record) received for processing.

Step

Action

1

Search for Provider application + Read more ...

  • Open the Tracker from your desktop
  • Search for and select the required My Health Record application record

Note: for access to the Tracker – Managed by Provider Operations (Tier 3), email Tier 3 provider support.

2

Enter the HP Records Manager Record Number + Read more ...

To update or complete database entries: Programme Support Officer must use the steps in Section 4 Service Officer Role of the eHealth Tracking Database procedure located in the HI/eHealth folders starting at 'Search for Existing Application'.

The procedure is located at \Shared\VIC\MELB-595-Colli-MC\Business Outreach Services\HI ehealth\Procedures\eHealth Joint Applications Tracker Process -without reporting - Final.docx

In the Edit Application From Details screen, enter the HP Records Manager Record Number into the HP Records Manager Reference field in the HI Service Applications Tracker (e.g. 13/199999).

Select Close Form button.

3

Finalise + Read more ...

In the Close Form screen, select Yes. The HP Records Manager Number is now recorded and saved in the Tracker for the My Health Record provider application.

Note: a provider application may have more than 1 document requiring storage. Where an application has multiple documents, enter the My Health Record TRIM Reference field numbers: ‘13/199999, 13/199920, 13/1999921’ etc.

Once documents are saved in TRIM, all copies stored in network folders are to be deleted. Multiple copies must not be kept.

See:

  • Step 1 in the Handling and storing My Health Record provider documentation received via email or hard copy table, or
  • Step 1 in the Handling and storing My Health Record provider documentation stored in other locations table

Search for a stored My Health Record provider application

Search for a stored My Health Record provider application in HP Records Manager

Table 4: This table describes how Service Officers search for a My Health Record provider application once it has been stored in HP Records Manager using either a record number or the applicant's name.

Step

Action

1

Open application and begin search + Read more ...

  • Open the HP Records Manager Desktop application
  • Select the Search by field and select Record Number or 'Title word' (for searching by applicant's name)

2

Conduct search + Read more ...

In the Matching criteria field, enter either the record number (for example, 13/199999) or the applicant's name as appropriate.

Press [Enter].

3

Open document + Read more ...

When the stored record displays in a new window, double click on the document to open it.

Ensure the window is closed once no longer needed.

4

Check status + Read more ...

Refer back to the HP Records Manager window where the document was found and where the Edit Status in the Properties preview pane below says 'Checked In'.

Note: when a document is opened in HP Records Manager the status reads 'Checked out' and the name of the person who opened the document is displayed. When this occurs, no other staff member can open the same document until it is check back in.

Procedure ends here.

Handling and storing My Health Record documentation

Handling and storing My Health Record provider documentation received via email or hard copy

Table 5: This table describes how Programme Support Officers handle and archive provider documentation that has been received via email or hard copy (such as fax or mail).

Step

Action

1

Scan hard copies if required + Read more ...

Scan any hard copy documentation.

Save all the documentation to work allocation application tool (PaNDA).

For example: R:\NAT\HLTHSERDEL\MCAProvide\Digitised workflow\General\Updates\HI ehealth\1.UNPROCESSED\MHR

2

Process and save application + Read more ...

  • Process the application and update the Tracker as required
  • Save the completed application, including all additional documentation and screen prints, to PaNDA as appropriate

3

Pre-check - Quality checking for notification letters automated by PPR + Read more ...

This check must be completed before close of business on a daily basis.

4

On clearance at Step 3 - Prepare document for archiving + Read more ...

  • For combined application (HW018 Part A & B,):
    • remove all reference material pertaining to Part A including screen shots
    • save Part B and any additional information ready for trimming
  • For standalone:
    • save Part B and any additional information ready for trimming
  • All other documents:
    • save document including additional information ready for trimming

5

Save documentation in HP Records Manager container + Read more ...

Save the documentation in accordance with naming convention (see Resources page) in the relevant HP Records Manager and update the Tracker entry with the HP Records Manager record number.

Note: the HP Records Manager record number is allocated by HP Records Manager and visible once documentation is saved.

If documentation stored:

  • in email, go to Step 6
  • as soft/hard copies, delete all soft copies of the documentation on the network drive and securely destroy any hard copies that have been printed for processing purposes. Procedure ends here

6

Finalise - remove attachment and preserve original email + Read more ...

Remove attachment from email, preserving the original email that arrived in case it is needed in the future.

  • Open the email containing the documentation and right click on the attachment(s)
  • Select remove attachment
  • Include the HP Records Manager number from the Tracker in the subject line by placing the cursor at the beginning of the text and typing it in
  • Highlight the rest of the subject line and include the applicant's name if it isn't already there
  • Close the email by selecting the X at the top right hand corner of the email
  • Select Yes to save the changes
  • Procedure ends here

Applications currently stored in other locations

Handling and storing My Health Record provider documentation stored in other locations

Table 6: This table describes how Service Officers handle and archive My Health Record provider applications/documentation that have been stored elsewhere (for example, on the network drive) in HP Records Manager. It is a requirement that applications and documents are stored securely. HP Records Manager is the electronic storage system used Services to securely store documentation.

Step

Action

1

Navigate to relevant folder/location + Read more ...

Open the relevant folder/location containing the My Health Record provider application/documentation that needs to be stored in HP Records Manager.

For example: G:\Business Outreach Services\HI ehealth\Registered\eHealth

2

Select Appropriate Application /documentation + Read more ...

  • Right click on the My Health Record provider application/documentation that requires electronic filing
  • Select Sent to and then select HP Records Manager
  • Ensure the document is selected and select the OK button

3

Amend Name + Read more ...

In the New Record Document screen, amend the title name in the Title (free text part) field as required.

Ensure the naming is in line with the naming convention located on the Resources page.

4

Complete HP Records Manager container details + Read more ...

In the Put On What File? field, type the HP Records Manager Container that the electronic document needs to be filed into (as per the naming convention).

5

Finalise + Read more ...

Select OK.

The My Health Record provider application/documentation is saved in HP Records Manager with a HP Record Manager Record number (for example, 13/199999).

If the application/documentation was previously stored:

  • via email, go to Step 6
  • in a folder on the network drive/desktop, right click on document and select delete (once saved in HP Records Manager). Procedure ends here
  • hard copy, securely destroy any hard copies that have been printed for processing purposes. Procedure ends here

6

Finalise - remove attachment in email: + Read more ...

  • Open the email containing the documentation and right click on the attachment(s)
  • Select remove attachment
  • Include the HP Records Manager number from the Combined Tracker in the subject line by placing the cursor at the beginning of the text and typing it in
  • Highlight the rest of the subject line and include the applicant’s name if it isn't already there
  • Close the email by selecting the X at the top right hand corner of the email
  • Select Yes to save the changes
