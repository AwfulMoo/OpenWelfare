3

Viewing the Emergency situation summary (service) screen + Read more ...

The Emergency situation summary (service) screen:

  • displays all the current and historical declared emergency situations for an individual residential care service
  • is a view only screen

This information flows through to ACMPS via the NAPS gateway.

The Emergency situation summary (service) screen displays the following information in a table format:

  • Current emergency situations
    • Emergency situation type - displays either Disaster, Epidemic, Pandemic or Other
    • Care type
    • Start date
    • End date
    • Created date
    • Last updated
    • Status - the Declared status will display on or before the end date of the emergency situation
  • Historical emergency situations
  • Emergency situation type - displays either Disaster, Epidemic, Pandemic or Other
    • Care type
    • Start date
    • End date
    • Created date
    • Last updated
    • Status - displays either Ended or Deleted

Service Officers can:

  • select the number of records to display from the Show records drop down list (bottom left corner of the screen)
  • type in a keyword in the Search bar to search for all records that contain the keyword. Search results display where the keyword is used anywhere on the record, even if it appears in the middle of another word
