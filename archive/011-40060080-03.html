This page contains information on how to access, match, mark and bypass a Fact of Death Data (FODD) record in the Consumer Directory Maintenance System (CDMS).

Access FODD records

How to access Fact of Death Data records

Table 1: This table describes how to access FODD records in the CDMS.

Step

Action

1

Consumer Directory Maintenance System (CDMS) home page + Read more ...

On the CDMS home page, select Fact of Death Processing link. The Fact of Death Summary screen displays.

2

Select State or Territory + Read more ...

Select the State or Territory for processing.

Result:

  • the State Summary page displays, with the name of the State or Territory in the screen name, for example, New South Wales Summary screen
  • the screen displays alphabetical groupings by surname, (for example, O-S for Smith) showing the number of records for previously unresolved and currently unresolved records

3

Select the alphabetical surname group + Read more ...

Select the alphabetical surname group that contains the records that require processing.

4

Make a selection + Read more ...

  • Select Current for records from the current month
  • Select Previous for records from previous months

Result: the Fact of Death Data (FODD) Record screen displays with the first available FODD record for processing and a list of possible matches from the Consumer Directory.

5

Choose a record + Read more ...

Match a FODD record to a customer

Mark a FODD record as unmatchable

Bypass a FODD record

Match a FODD record to a customer

Match a Fact FODD record with a specific customer record

Table 2: This table describes how to match a FODD record with a specific customer record so the date of death is updated for that customer.

Step

Action

1

Access FODD records + Read more ...

Access the FODD Record screen. See: Access FODD records table.

2

Display details + Read more ...

To display details of the customer, select Consumer Details.

The Consumer Details screen displays.

3

Select match then return to record screen + Read more ...

  • Match to link the customer with the Fact of Death record, then go to Step 4
  • Return to return to the FODD Record screen

4

Matched consumers + Read more ...

In Matched Consumers:

  • select the customer that matches with the Fact of Death record
  • select Match

The Confirm FODD Record - Match screen displays.

5

Amend group contact + Read more ...

  • to amend the group contact name, overtype the name in the Contact Name field
  • if the group contact is deceased, select Est of. The contact name is amended with EST OF

An information message displays advising of the EST OF amendment, and reminding the Service Officer to enter a date of death for the person. Select OK to acknowledge the message.

6

Select change or update + Read more ...

  • Change to change the details, go to Step 2
  • Update to confirm the details

7

System update + Read more ...

The system updates the record with the amended details and displays either:

  • Fact of Death Record screen, if more records are to be processed
  • State Summary screen, when no records are to be processed

Mark a FODD record as unmatchable

How to mark a FODD record as unable to be matched

Table 3: This table describes how to mark a FODD record as unable to be matched to a customer record.

Step

Action

1

Access the FODD Record screen + Read more ...

Access the FODD Record screen, see Access FODD records table.

2

Matched consumers + Read more ...

In Matched Consumers:

  • select No Match to indicate none of the consumer records match the Fact of Death record

The Confirm FODD Record - No Match screen displays.

3

Select change or update + Read more ...

  • Change to change the details, go to Step 2
  • Update to confirm the details

4

System updates record + Read more ...

The system updates the record with the amended details and displays either:

  • Fact of Death Record screen, if more records are to be processed
  • State Summary screen, when no records are to be processed

Bypass a FODD record

Note: when a Fact of Death record is bypassed for later processing, the record remains available on the system until a Match or No Match decision is made.

Bypass a FODD record for later processing

Table 4: This table describes how to bypass a FODD record for later processing.

Step

Action

1

Access Fact of Death Data (FODD) Record + Read more ...

Access the FODD Record screen, see Access FODD records table.

2

Matched consumers + Read more ...

In Matched Consumers, select Next Record to skip the Fact of Death record.

The system displays the next Fact of Death record with possible matches for processing.
