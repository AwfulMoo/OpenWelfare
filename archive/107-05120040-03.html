Actioning a reversal of a debt waiver

To reverse a waiver for Process Direct managed debts, see Raising Child Care Subsidy (CCS) debts in Process Direct.

Step

Action

1

Reversing a debt waiver + Read more ...

A debt waiver has been recorded incorrectly on a customer's record. A decision has been made to reverse this action in order to:

  • adjust the overpayment amount, and/or
  • recommence recovery action

If family assistance waiver is to be actioned, go to Step 7, otherwise go to Step 2.

Note: any reversal of a waiver is subject to delegation limitations.

2

Recalculating a debt for a period + Read more ...

When recalculating a debt for a period, and:

  • the debt period includes a debt that was waived at the same time it was raised (for example a small debt waiver (less than $50 LTF)), and
  • the waiver requires reversing because the basis of the original decision was incorrect, or no longer exists, the correct procedure is to:
    • reverse the waiver
    • record it as Finalised No Debt (FND)
    • raise a new debt for the correct amount, and
    • finalise the previously waived debt as a no debt

When reversing a debt waiver, including an automatic small debt waiver (LTF) to recalculate a debt period:

  • all or part of the waived amount may be included within the new debt decision where the period of the waived amount falls within the new debt recalculation period
  • the reason the debt was generated (and subsequently waived) and the reason for the new debt (recalculation period) do not have to be for the same reason

3

Debt List (OPDL) screen + Read more ...

From the OPDL screen in the customer's record:

  • 'S'elect the Debt ID with a waiver action present and navigate to the Waiver (OPWA) screen
  • in the Reason Reversed: field, code one of the appropriate codes:
    • 'WIE' - Waived In Error
    • 'DAR' - Debt amount adjustment required
  • press [Enter]
  • press [Enter] again to proceed to the Assessment Finalisation (OPAF) screen (enter any comments to support the decision)
  • code 'Y' to finalise the action

4

Finalise the debt 'no debt' + Read more ...

From the OPDL screen in the customer's record:

  • 'S'elect the Debt ID with a waiver action present and navigate to the Debt Amount (OPDA) screen
  • in the Reason changed: field, enter the reason 'MUL' Multiple Debt Activities
  • in the Calculated Amount: field, enter the amount of zero. Press [Enter]
  • press [Enter] again to proceed to the Assessment Finalisation (OPAF) screen (enter any comments to support the decision)
  • code 'Y' to finalise the action

5

Cancel the Q417 advice activity on the Activity List screen + Read more ...

Issuing a Q417 is not required as the waived component is already in another debt. The activity to send the Q417 can be cancelled.

From the Activity List (AL) screen:

  • enter a 'C' to cancel the DMN/ADV activity. Press [Enter]
  • enter a 'C' in the Cancel all provisional changes?: field and press [Enter]

6

Record details of action and decision on a DOC + Read more ...

Procedure ends here.

7

Family assistance transitional waiver + Read more ...

Where a family assistance transitional waiver (reason codes of TWF or TWC) was applied to reconciliation debts raised for the 2000-01 year:

  • any re-conciliations on the customer debt automatically reversed the waiver on the original reconciliation overpayment result, and
  • applied it to the reconciliation results

This waiver reason is shown as 'IFR' - Reversed and included in reco (Auto Only).

Any other FAO waiver that has been automatically reversed as a result of re-conciliation is not automatically re-applied. If re-application of the waiver is appropriate, contact the FAO manual intervention team.
