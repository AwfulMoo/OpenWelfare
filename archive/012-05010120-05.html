This page contains the forms and email templates required for electronic Medicare claiming registration and Public Key Infrastructure (PKI).

On this Page:

Forms - Online claiming registration

Forms required for Online claiming registration

This table lists the forms used by customers for Online claiming registration. These forms are available on the Services Australia website.

Form link

Description

Online Claiming Banking Details form (HW052)

One form for each transmitting location.

Online claiming Provider Agreement form (HW027)

Completed and signed by each provider transmitting claims at the location.

Forms - PKI

Forms for PKI certificates

This table lists the forms used by customers for PKI certificates. These forms are available on the Services Australia website.

Form link

Description

Application for a Public Key Infrastructure Site Certificate (HW001)

Application form for sites that have:

  • not previously registered with Medicare
  • previously registered with Medicare

Application for a Public Key Infrastructure Individual Certificate (HW002)

Application form for healthcare professional individuals that have:

  • not previously registered with Medicare
  • previously registered with Medicare

Request to Revoke or Reissue a Public Key Infrastructure Certificate form (HW003)

Revocation/ suspension / reinstatement / renewal requests.

Acceptable Referee Declaration for Public Key Infrastructure Certificate applications form (HW004)

Acceptable referee identification form for:

  • individual certificates
  • site certificates held by sole traders
  • site certificates held by non-individuals, such as companies, trusts, partnerships

Hospital or Day Procedure Centre Registration for In Hospital Claiming form (HW025)

Application for hospitals and day procedure centres to register for in-hospital claiming via ECLIPSE.
