This code should only be used where the carer has requested to cancel CP and/or CA without providing other details. If the carer is advising information about a change in care/circumstances or is advising they are no longer providing care, this must be updated appropriately to ensure payment accuracy.

If the customer has requested cancellation and cannot/will not provide further information, cancel CP and/or CA using the Carer cancellation and suspension workflow.

Note: staff can cancel both CP and CA in the same activity, if applicable.
