Troubleshooting AIR site user issues, causes and resolution

Table 2

Issue

Possible reason and suggestions

Unable to open page

Vaccination provider to check they are running the latest version of their internet browser. Internet Explorer is the preferred and supported browser.

If Terms and Conditions have been rejected, contact an AIR Programme Support Officer (PSO) to resolve with AIR Program Support.

Page displays incorrectly

Vaccination provider to check they are running the latest version of their internet browser. Internet Explorer is the preferred and supported browser.

Provider number not recognised

Check that the provider number has been entered correctly. The provider number should be 8 characters, so zeros may need to be inserted at the start of the number, e.g. 001234AA.

Ancillary AIR provider has not accessed for more than 18 months and is end dated

Contact a PSO for referral to Tier 3 Helpdesk

Unable to identify individual

Request that vaccination provider clear their cache by deleting all AIR bookmarks, favourites and cookies in their internet browser.

Unable to print page

Vaccination providers to check that they are running the latest version of their internet browser.

Unable to print AIR site reports

To print AIR site reports from the HPOS page providers:

  • select Reports
  • select View tab
  • select Options
  • select Download
  • open the downloaded file to print
