This page contains a list of Simplified Billing enquiry FHEI screen functions and a list of modifiers including shortened modifiers used to display information on the FHEI screen.

FHEI screen and modifiers

Modifiers assist in refining the information displayed in the FHEI screen. For example, if the adjusted claim Id is included within the modifier, the system will only display items that belong to the adjusted claim.

Screen functions

FHEI screen functions

This table provides a list of keys that perform different functions on the FHEI screen.

Key

Description

PF1

Enables PF1 function (Help) to display description/definition (as per Patient claim screens) by cursor location: Reason Code.

Example: When the cursor is placed under a reason code and the F1 Help key is pressed, the explanation/description that applies for the reason code is to be displayed in the information area at the bottom of the screen.

PPF2

When selected will display the claim id and date of lodgement of claim for each item. For:

  • unadjusted items, selecting PF2 key, will display the original claim id and original date of lodgement
  • adjusted items, selecting this key, will display the respective adjusted claim id and adjusted date of lodgement

The CLAIM ID and DOL columns will be displayed in place of BENEFIT and SER PROV columns

Then press PF2 again to return back to basic screen.

Service Lines Limit

If the number of service lines is greater than 1,500, the message 1-MAXLINES is displayed with the FHEI control line. No other details are to be displayed on the screen at this stage.

When this message appears add the modifier to include the ‘CARD NUMBER,PROVIDER NUMBER’ to filter the retrieved segments.

Note: the 1-MAXLINES message will remain on the screen until the modifier is entered.

PF12-CANCEL

Cancels out the enquiry transaction.

PF8-FORWARD and PF7-BACKWARD

If all items retrieved for the claim cannot fit on one page, the message 'MORE' is displayed. Use these function keys to scroll through the claim details.
