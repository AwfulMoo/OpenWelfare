Card type display and veteran entitlement

In the DVA authority system, the Repatriation Benefits card (RPBC) card type is treated the same as for PTEC (gold card) veterans who have eligibility to all items, including authority items, within PBS and Repatriation Pharmaceutical Benefits Scheme (RPBS).

The card type displayed on all DVA authority screens reflects the highest level of entitlement at the current date.

Entitlement display on RPQE screen

The Patient Enquiries - Eligibility Details screen (RPQE) displays RPBC in the ENT DESC field to denote entitlement under the orange card.

Note: queries about more information of DVA entitlements are to be referred to VAPAC.

Entitlement display on RPQE screen in DVA authority system

This table describes how the orange card entitlement displays in the RPQE screen of the DVA authority system.

Screen fields

Explanation

Start Date

For veterans who became eligible for the orange card from the date of implementation, the Start Date will be 01 01 02. Any other current entitlement will be the correct entitlement for the veteran until this date.

For subsequently approved orange card veterans, the Start Date will be a date in the past.

Level Code

Displays RPBS for the orange card veterans in line with the current display for PTEC and STEC veterans.

Source Code

Displays DVA for the orange card veterans in line with the current display for PTEC and STEC veterans.
