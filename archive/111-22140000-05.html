Error status codes for Incentive Programs quality checking, error numbers below INC25

Table 2

Error status codes

Definition

Processing tip

INC02

Critical

Application form incomplete

It is an error when the incorrect response to a question has been selected and source documents do not match.

INC03

Critical

Insufficient documentation to support application.

It is an error if there is insufficient documentation to support the application, and there is no notation on PaNDA to support the action taken.

INC04

Critical

Declaration incomplete or incorrect signatory.

It is an error when the Declaration is:

  • missing the following information, and there is no notation on PIP/WIP – Practice Stream Online:
    • Signature
    • Name
    • Date (where applicable)
  • not signed by an owner or authorised contact person registered with the practice
  • not signed by the General Practitioner or Nurse Practitioner indicated on the form

Note: This error code includes when the date of the patient consent is after the date of lodgement.

INC05

Critical

University Declaration on Teaching Claim Form

It is an error when the University declaration does not have the university authorised signature.

INC06

Critical

Practice Name incorrect

It is an error when the Practice Name is keyed incorrectly.

INC07

Critical

Contact Person Details incorrect

It is an error when any of the following Primary/Secondary Contact Details are keyed/selected incorrectly, or not keyed/selected:

  • Family Name
  • First Given Name
  • RA Number (if applicable)

INC08

Critical

Practice Contact Details incorrect

It is an error when any of the following Practice Contact Details have been keyed incorrectly or not keyed:

  • Phone number
  • Fax number
  • Email address

INC09

Critical

Practice Address details incorrect

It is an error when any of the following Address details have been keyed/selected incorrectly or not updated:

  • postal address
  • main practice location
  • additional location

INC10

Critical

Practice Accreditation details incorrect

or

The address on the certificate does not match the practice profile

It is an error when any of the following have been keyed/selected incorrectly:

  • Accreditation Status
  • Start and End Dates
  • Address
  • Accreditation Body
  • Accreditation Number

INC11

Critical

Additional provider locations incorrect

It is an error when the additional provider location has been keyed or selected incorrectly.

INC12

Critical

Incorrect Incentive selected

It is an error when one or more of the following incentives in the Incentive field have not been selected or selected incorrectly for:

  • Cervical Screening
  • Asthma
  • Diabetes
  • Indigenous Health
  • After Hours

INC13

Critical

Bank details incorrect

It is an error when any of the following Bank details have been keyed incorrectly or not updated:

  • Account name
  • BSB
  • Account number

INC14

Critical

Details of Practice Ownership incorrect

It is an error when any of the following Practice Ownership details have been keyed/selected incorrectly, or not keyed/selected:

  • Arrangement Type
  • Company Name
  • Trading Name
  • Address
  • Declaration

INC15

Critical

Practitioner Start and/or end dates are incorrect

It is an error when the start and/or end date for the following have been keyed incorrectly or not keyed:

  • Provider
  • Nurse Practitioner

INC16

Non-critical

Professional Indemnity insurance details incomplete/incorrect

It is an error when the Indemnity insurance has been:

  • selected incorrectly, or
  • not selected

INC17

Critical

Procedural GP details incorrect

It is an error when the Procedural GP tier level has been updated incorrectly.

INC18

Critical

Program status field incorrect

It is an error when the a date in the Practice Program status field has not been keyed/selected when a practice has:

  • ceased operating as a practice
  • withdrawn from PIP
  • ceased operating at a practice branch location

INC19

Critical

Date practice withdraws from PIP/WIP – Practice Stream is incorrect

It is an error when the date a practice withdraws from PIP/WIP – Practice Stream has not been keyed, or keyed incorrectly.

INC20

Critical

Relocation or Amalgamation date incorrect

It is an error when the Relocation or Amalgamation date has been keyed incorrectly.

INC21

Critical

SIP and SIP/ACAI Program incorrectly selected

It is an error when the SIP and SIP/ACAI are:

  • not selected
  • selected incorrectly

INC22

Critical

Service Incentives Payments (SIPS) failed

It is an error when a SIPS payment has failed because:

  • SIPS comments have not been removed from the GDES spreadsheet
  • new bank details have not been updated on GDES spreadsheet

INC23

Critical

Provider number does not match the physical location of the practice

It is an error when the Provider Number keyed does not match the physical location of the practice where the service was provided.

INC24

Critical

Professional Indemnity insurance expired

It is an error when the Individual GP or nurse practitioner details (IP003) form has been processed and the date of the Professional Indemnity has expired.

 

Error status codes for Incentive Programs quality checking, error numbers INC25 and above

Table 3

Error status codes

Definition

Processing tip

INC25

Critical

Provider has not been registered or registered incorrectly

It is an error when the provider has:

  • been registered to the incorrect practice location
  • not been registered to a practice location

INC26

Critical

RA number has not been added to correct profile

It is an error when the Remote Access (RA) number has not been added to the correct profile.

INC27

Critical

Authorised Contact not removed from Practice Profile

It is an error when the Authorised contact has not been removed from the Practice Profile as requested.

INC28

Critical

Annual/Quarterly statement not updated/incorrect

It is an error when an Annual/Quarterly Confirmation Statement has been updated incorrectly or not been updated.

INC29

Critical

Average weekly hours not updated

It is an error when the Average weekly hours have been updated incorrectly, or not updated.

INC30

Non-critical

Work classification incorrect

It is an error when the Work classification has been keyed incorrectly or not updated.

INC31

Critical

IHI Patient incorrect

It is an error if the incorrect patient has been linked to the practice based on the information supplied.

INC32

Non-critical

Aboriginal or Torres Strait Islander indicator incorrect

It is an error when the Aboriginal or Torres Strait Islander indicator is not selected correctly.

INC33

Critical

IHI patient information incorrect (doesn't match source documents)

It is an error when the incorrect information has been entered and does not match source documents:

  • IHI chronic disease
  • IHI health check
  • IHI patient consent
  • Teaching Payment student attendance

INC34

Critical

Practice Profile - incorrect documents uploaded

It is an error when a form or document has been uploaded to the incorrect Practice Profile.

INC35

Critical

Document not uploaded

It is an error when a form or document has not been uploaded to:

  • Practice Profile
  • PaNDA

INC36

Critical

Letter incorrect

It is an error when information or content of a letter has not been keyed correctly.

INC37

Critical

Outdated form

It is an error when an outdated version of a form has been processed after the appropriate grace period.

INC38

Critical

Bank details incorrect (GPRIP)

It is an error when any of the following Bank details have been keyed incorrectly or not updated:

  • Account name
  • BSB
  • Account number

INC39

Critical

Documents uploaded to practice profile when practice was not identified

It is an error to upload information when the practice has not been clearly identified

INC40

Critical

Incorrect action taken on payment hold

It is an error to not remove a held payment where relevant forms have been supplied for change of ownership, relocation, accreditation

INC41

Critical

IHI patient has not been added to the practice profile/IHI form missed

It is an error when information provided has not been keyed

INC43

Critical

Work item not processed or processed incorrectly

It is an error when:

  • a work item has not been processed in part or in full
  • the form details do not match the profile
  • not all required questions have been answered/form is incomplete
  • Teaching Payment student attendance not entered correctly
  • Operational Blueprint process not followed
