Assess a request to recognise additional training in FPS

Table 1

Step

Action

1

Notice of additional training in FPS received + Read more ...

The notification to register a health professional as having completed additional training in FPS is received directly from the GPMHSC.

Notifications of registration are received from the Royal Australian College of General Practitioners (RACGP). The Resources page contains an example of this document.

If the request comes from any other source then re-categorise the application in PaNDA to National Office Escalations.

2

Confirm Provider details on Provider Directory System (PDS) + Read more ...

Open PDS:

  • Select Provider
  • Select View
  • Select Detailed Search
  • In the Family name field, key the first 3 letters of the last name, followed by a %
  • In the Given name field, key the first 3 letters of the first name, followed by a %
  • Enter date of birth in Date of Birth field
  • Select Professions
  • Select Search to display a list of health professionals with a matching name

Does the medical practitioner’s name and provider number (if provided) match the request?

  • Yes, go to Step 3
  • No, the information does not match a record then re-categorise the application in PaNDA to National Office Escalations

Note: if an exact match is found and multiple stems exist then select one of the provider numbers. All unsynchronised linked stems must be updated individually.

3

Check Specialty Codes – General Practitioner + Read more ...

Scroll to the bottom of the Provider Stem record to view the Specialty Codes for the medical practitioner.

  • Check if the health professional is a general practitioner (Specialty Codes 130, 132, 530, 532 or 540 are present and current). If they are, go to Process a health professional’s additional training in FPS - Vocationally Registered (650)
  • If the medical practitioner is recorded in PDS as a:
    • specialist or consultant physician (Specialty Codes 1-99, 802-808 or 898-899), and
    • is not also a general practitioner, no further action is required. File the request, as per current workflow procedures

5

Assess which code(s) to add + Read more ...

  • If the medical practitioner is a general practitioner (130, 132, 530, 532 or 540) add specialty code 650 to the stem. Go to Process a medical practitioner’s additional training in FPS - Vocationally Registered (650)
  • If the medical practitioner is eligible to receive non-VR benefits (see Step 4) add specialty code 665 to the stem. Go to Process a medical practitioner’s additional training in FPS - Non Vocationally Registered (665)
  • If the medical practitioner is eligible to receive VR benefits (see Step 5) add specialty code 650 to the stem. Go to Process a medical practitioner’s additional training in FPS - Vocationally Registered (650)
  • If the medical practitioner is eligible to receive both non-VR and VR benefits (at different locations) add specialty codes 665 and 650 to the stem. Go to Process a medical practitioner’s additional training in FPS - Non Vocationally Registered (665). Once the steps in this table are complete, go to Process a medical practitioner’s additional training in FPS - Vocationally Registered (650)

Note: medical practitioners who have multiple placement programs are often eligible for non-VR and VR benefits depending on their location and 3GA program. It is important to correctly assess eligibility and complete the correct processing table(s).

Process a medical practitioner’s additional training in FPS - Non Vocationally Registered (665)

Table 2

Step

Action

1

Update Specialty Code + Read more ...

Select:

  • Edit
  • Specialty tab
  • Add Specialty Code and key in:
    • Specialty Code - 665 (Mental Health Focussed Psychological Strategies)
    • Start Date: Key in the date as advised on the report from RACGP
    • End Date: no end date is required even if present on the report
  • Add New
  • Save

2

Preparation of letter + Read more ...

  • Select the Z2557 Allied Health and GPS Registering Additional Psychological Qualifications letter template and complete the required details

Note: check if the medical practitioner is eligible for specialty codes 665 and 650. If they are, go to Process a medical practitioner’s additional training in FPS - Vocationally Registered (650) before preparing a confirmation letter.

  • Upload the document to the application in PaNDA

3

Finalise + Read more ...

Once the application has been updated then:

  • Print the letter and send via post to the medical practitioner
  • File the request as per current workflow procedures
