The following conditions entitle a DVA health card holder to dental treatment.

Processing 300-399 range item numbers

Description and resolution

311, 314, 322, 323, 324

Extraction items + Read more ...

Rules

2 tier pricing structure applies

  • When 1 item is claimed the item is automatically paid the base amount
  • When 2 or more of the same item, date of service (DOS) and provider are claimed, and the tooth numbers are in the same quadrant, 1 item is paid the base amount and any subsequent items will automatically be paid the second tier fee
  • When a tooth number is not present for any of the items claimed, the system assumes that all items are for the same quadrant
  • When tooth numbers are present for only some of the dental step down items claimed (e.g. 2 items from different quadrants and 2 items without a tooth number), the system assumes that items without a tooth number are from the same quadrant and the second tier fee applies
  • When an item is claimed and the same item, DOS, quadrant, and provider has already been paid on history, the item claimed is paid the step down fee
  • The step down fee rules only apply when there is more than one of the same item claimed for the same provider, DOS and quadrant

Processing rules

If 2 extractions are claimed on separate days, the highest fee is payable for both items.

The following rules apply to same extraction items on the same date of service:

  • The first extraction from each quadrant is paid at the highest fee.
    The second and subsequent extraction items from the same quadrant are paid at the step-down fee.
    • Examples: Extractions 23 and 26 are in the same quadrant, so pay higher fee for first and step down fee for second. Extractions 13 and 26 are in different quadrants, so pay the higher fee for both.
  • If the teeth are not identified on the service voucher, assume all extractions are in the same quadrant and pay per price structure in the DVA fee schedule
  • Teeth are numbered using a 2 digit system (11-18, 21-28, 31-38 and 41-48). The first digit (1-4) refers to the quadrant, the second digit (1-8) refers to the tooth number
  • Sometimes teeth are numbered using the system ULx, URx, LRx, LLx.
    UL = Upper Left quadrant, UR = Upper Right, LR = Lower Right, LL = Lower Left.
    Pay the higher fee for each extraction in each separate quadrant, and the step-down fee for any additional extractions in the same quadrant
  • Tooth numbers are entered in:
    • Data Entry: On the Referral data entry screen (VKRC) press [PF2] to expand the fields. Enter tooth number in the NO field
    • Pend: NO field

Restrictions

Itself, check for tooth number:

  • If items are for the same quadrant or the tooth number is not indicated:
    • Pay the first item at the highest fee with PI 18
    • Pay subsequent items by amending the fee in the amount field to the step down fee with RSN code 288
    • If the correct fee structure has been applied, pay the claim with PI 18
  • If items are for different quadrants:
    • Pay the first item in each quadrant at the highest fee with PI 18
    • Key in the tooth numbers and the system will assess if the teeth are in the same quadrant. The system will automatically apply the step-down fee (in same quadrant) or pay the full fee (different quadrant) where appropriate
    • If the correct fee structure has been applied pay the claim with PI 18

Note: if different extraction items are claimed on the same day, assess each item separately.
