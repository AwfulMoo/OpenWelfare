Pend return messages

DVA optical consultation pend return messages

This table contains a list of optical consultation pend return messages, including the description of the message and resolution procedure.

Message

Description and resolution

3-AGE REST1

Description: DVA health card holder may not meet age requirements for the item claimed.

Applicable items: 10910, 10911, 10943.

Resolution procedure: check VEXI for age of card holder.

Are they within the age span stipulated in the item description?

  • Yes, pay the claim with PI 25
  • No, eject with RSN code S 585

6-OPT10907

Description: a different provider has performed an initial optical consultation for the DVA health card holder within a 24 month period.

Applicable items: 10905-10948.

Resolution procedure: reject with RSN code S 160.

Note: if a domiciliary visit 10931-10933 has been claimed, reject with RSN code S 737.

6-OPT10918

Description: the maximum number of initial optical consultations has already been performed by this provider, for this DVA health card holder in a 24 month period.

Applicable items: 10905-10948.

Resolution procedure: reject with RSN code S 160.

Note: if a domiciliary visit 10931-10933 has been claimed, reject with RSN code S 737.

8-MIX HIST

Description: a conflict exists between the service being processed and a service which has previously been paid by Services Australia.

Applicable items: 10905-10948.

Resolution procedure:

If the message 8-MIX HIST appears on its own, press [PF9] and continue processing.

If the message 8-MIX HIST appears with 1 of the following pend return messages, check the restricting item:

  • 6-SUS DUP
  • 6- RST/REJ
  • 6- RST/TD
  • 6- RST/ADJ

If the restriction is for:

  • the same item on the same date of service and has been paid by Services Australia, reject with RSN code S 258
  • a different consultation item that has been paid by Services Australia, reject the consultation item claimed with RSN code S 160

8-RST HIST

Description: there is an item on history which is restrictive with an item in the claim and the item on history has a higher schedule fee than the item claimed.

Applicable items: 10905-10948.

Resolution procedure: check the restricting item.

If the restriction is for:

  • the same item on the same date of service and has been paid by Services Australia, reject with RSN code S 258
  • a different consultation item that has been paid by Services Australia, reject the consultation item claimed with RSN code S 160

8-IT 10929

Description: the claim contains contact lens items which can only be paid if the DVA health card holder has a specific medical or optical condition.

Applicable items: 10905-10948.

Resolution procedure: check voucher/online text to determine the condition treated.

  • If one of the following conditions is present, reject with RSN code S 211:
    • myopia
    • hyperopia
    • astigmatism
    • anisometropia
  • If any other condition is present, press [PF9] and continue processing.

Eligibility query form

This form should be used to refer any claims that require additional assessment to establish a DVA health card holder's eligibility for DVA optical services. The form should be completed and returned to the VAP Policy Section (Tier 3) helpdesk. For contact details, see Veterans' Affairs Processing (VAP).
