myGov Code Generator app

Item

Description

1

Setting up myGov Code Generator app + Read more ...

Customers must be made aware:

  • before downloading the app, they should set up security codes by SMS as a sign in backup
  • not having SMS codes set up means if they lose access to their mobile device:
    • they will not be able to sign in to their myGov account, and
    • they will have to create a new myGov account

The myGov Code Generator app will prompt the customer to set up security codes by SMS as their sign in backup and warn them of the potential loss of access to their myGov account.

To set up the myGov Code Generator app, a customer needs to:

  • Download myGov Code Generator app from the App Store or Google Play. The Apple iOS or Android smartphones guides made available can provide help
  • Open the app to begin the registration process. Customers are advised about what the app does and if they want to change their myGov sign in option to the myGov Code Generator app
  • Enter their myGov sign in details (username and password), and answer a secret question or enter a security code. This will connect the mobile device to their myGov account
  • Customers will be advised to set up security codes by SMS in their myGov account as a sign in backup. The backup option will stop the need to create a new myGov account if they delete the app or have lost their device. This is not applicable if the customer's new device has a different mobile number to the number they have registered in their myGov account
  • Read and accept the Terms of Use to complete the registration process

A message notifies the customer that by continuing, their sign in option will change before registration is finalised. This changes the second factor authentication option to the myGov Code Generator app.

Customers uses the 6-digit one-time code generated by the app each time they sign in to their myGov account. The app creates a new code every 30 seconds. This code is valid for a maximum of 60 seconds.

2

Changing sign in options + Read more ...

If a customer wants to stop using the myGov Code Generator app, they can:

  • Sign in to myGov
  • Select Account settings
  • Select Sign-in options
  • Enter Password
  • Select myGov Code Generator app
  • Select Remove or Turn off

3

Removing myGov Code Generator app + Read more ...

If the myGov Code Generator app is removed from a customer's mobile device, or they want to use the app on another device, advise them to:

  • Go to my.gov.au to sign in
  • Enter username and password or Digital Identity details
  • Select I cannot access my app (if access app is removed)
  • Select Send me a code by SMS
    • if this step is not presented as an option to the customer then they didn’t enable security codes by SMS and will not be able to proceed. They will need to create a new myGov account. Service Officers can offer to release their email address. For more information, see Troubleshooting myGov
  • Enter the code, then select Next
  • Go to Account Settings
  • Select Sign-in options
  • Enter Password
  • Select myGov Code Generator app
  • Select Remove under the listed device

Once removed, the customer can:

  • download the app, on the same mobile device or a new one, and
  • set up their app registration again

4

Lost, stolen or new device + Read more ...

A customer can still sign in to their myGov account:

  • if they lose their mobile device with myGov Code Generator app installed on it, and
  • they have SMS codes set up as their sign in backup

In this scenario the customer needs to:

  • Go to my.gov.au to sign in
  • Enter username and password
  • Select I cannot access my app
  • Select Send me a code by SMS
    • if this step is not presented as an option to the customer then they didn’t enable security codes by SMS and will not be able to proceed. They will need to create a new myGov account. Service Officers can offer to release their email address. For more information, see Troubleshooting myGov
  • An SMS code is sent to the mobile number registered to the myGov account
  • Enter the code on the Enter code page to complete sign in

This function will only work, if the customer keeps the same phone number that is registered with their myGov account.

Once they have signed in, advise the customer to remove the myGov Code Generator app from the myGov account as follows:

  • Go to Account settings
  • Select Sign-in options
  • Enter Password when prompted
  • Select myGov Code Generator app
  • Select Remove under the listed device

Once removed, the sign in option will automatically default to SMS code. If the customer wants to reuse the myGov Code Generator app, they can:

  • download the app on their new mobile device, and
  • complete the app registration process using their myGov sign in details

This will connect the new mobile device to their myGov account.

If the customer does not have SMS codes set up as their sign in backup:

  • they will not be able to sign in to their myGov account, and
  • will need to create a new myGov account
    • to use the same email address for a new account, they need to contact the myGov helpdesk and ask for an email release

5

System requirements + Read more ...

myGov Code Generator app requires a minimum operating system of:

  • iPhone (4S and above), with a minimum operating system of iOS 9.0
  • Android devices with a minimum operating system of 4.4
