Income, assets and family circumstances assessment for CP claims

Table 1: this table describes how to assess the carer and care
receiver's income, assets and family circumstances when determining
eligibility to CP when care is provided for a disabled adult and their
dependent child.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Does the adult care receiver      |
|                                   | provide care for a dependent      |
|                                   | child? + Read more ...            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the carer (of the adult   |
|                                   |     care receiver) cannot qualify |
|                                   |     for CP under this provision.  |
|                                   |     To see if the adult care      |
|                                   |     receiver can qualify under    |
|                                   |     the 'full' qualification      |
|                                   |     provisions, see Eligibility   |
|                                   |     for Carer Payment (CP) when a |
|                                   |     care receiver is aged 16      |
|                                   |     years or over. At least 2     |
|                                   |     genuine attempts to contact   |
|                                   |     must be made to contact the   |
|                                   |     carer to advise of the        |
|                                   |     unfavourable decision. Record |
|                                   |     details on a DOC in the       |
|                                   |     carer's record. For more      |
|                                   |     details, see Advising         |
|                                   |     verbally of an unfavourable   |
|                                   |     decision (CLK). Procedure     |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Is the dependent child aged 6 or  |
|                                   | over? + Read more ...             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 3                                 | Does the dependent child attract  |
|                                   | payment of Carer Allowance (CA)   |
|                                   | (child)? + Read more ...          |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, the carer (of the adult   |
|                                   |     care receiver) cannot qualify |
|                                   |     for CP under this provision.  |
|                                   |     Check Eligibility for Carer   |
|                                   |     Payment (CP) when a care      |
|                                   |     receiver is aged 16 years or  |
|                                   |     over. At least 2 genuine      |
|                                   |     attempts to contact must be   |
|                                   |     made to contact the carer to  |
|                                   |     advise of the unfavourable    |
|                                   |     decision. Record details on a |
|                                   |     DOC in the carer's record.    |
|                                   |     For more information, see     |
|                                   |     Advising verbally of an       |
|                                   |     unfavourable decision (CLK).  |

| 4                                 | Care receiver has a dependent     |
|                                   | child + Read more ...             |
|                                   |                                   |
|                                   | To reach this step, the care      |
|                                   | receiver will have a dependent    |
|                                   | child who is either:              |
|                                   |                                   |
|                                   | -   aged under 6, or              |
|                                   | -   aged 6 to less than 16 and    |
|                                   |     who is attracting a payment   |
|                                   |     of CA (child)                 |
|                                   |                                   |
|                                   | Note: from this step onwards, the |
|                                   | care receiver referred to is the  |
|                                   | adult care receiver, and the      |
|                                   | carer is the carer of the adult   |
|                                   | care receiver.                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Is the care provided in the       |
|                                   | residence of the care receiver? + |
|                                   | Read more ...                     |

|                                   | -   No, go to Step 6              |

|                                   |                                   |

|                                   | care receiver + Read more ...     |
|                                   |                                   |
|                                   | Is the care receiver either:      |
|                                   |                                   |
|                                   | -   in receipt of a Social        |
|                                   |     Security pension or benefit,  |
|                                   |     or a Service Pension or       |
|                                   |     Income Support Supplement     |
|                                   |     (ISS) or Veteran Payment, or  |

Assessment of the medical report for CP claims

Table 2: this table describes how to assess a Carer Payment and/or Carer
Allowance - Medical Report (for a person 16 years or over) form
(SA332a).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Medical report + Read more ...    |
|                                   |                                   |
|                                   | Has the SA332(a) form been        |
|                                   | returned?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, processing of the new     |
|                                   |     claim cannot continue until   |
|                                   |     the Medical Report is         |
|                                   |     returned. Inform the carer of |
|                                   |     this fact, and issue the      |
|                                   |     SA332(a). When the SA332(a)   |
|                                   |     is returned, go to Step 2     |
+-----------------------------------+-----------------------------------+
| 2                                 | Do the Treating Health            |
|                                   | Professional's (THP) responses on |
|                                   | the form indicate that care is    |
|                                   | required for at least 6 months? + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Yes, calculate the ADAT score |
|                                   |     based solely on the health    |
|                                   |     professional's responses. Go  |
|                                   |     to Step 3                     |
|                                   | -   No, then the CP claim should  |
|                                   |     be manually rejected on the   |
|                                   |     Benefit Action (BA) screen,   |
|                                   |     using the reason code 'NMQ'   |
|                                   |     (not medically qualified). At |
|                                   |     least 2 genuine attempts to   |
|                                   |     contact must be made to       |
|                                   |     contact the carer to advise   |
|                                   |     of the unfavourable decision. |
|                                   |     Record details on a DOC in    |
|                                   |     the carer's record. For more  |
|                                   |     details, see Advising         |
|                                   |     verbally of an unfavourable   |
|                                   |     decision (CLK). Procedure     |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | What is the THP''s score? + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Less than 8, go to Step 4     |
|                                   | -   8 or greater, go to Step 5    |
+-----------------------------------+-----------------------------------+
| 4                                 | ADAT score based on THP's score   |
|                                   | does not attract CP for carer +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The CP claim should be manually   |
|                                   | rejected.                         |
|                                   |                                   |
|                                   | At least 2 genuine attempts to    |
|                                   | contact must be made to contact   |
|                                   | the carer to advise of the        |
|                                   | unfavourable decision. Record     |
|                                   | details on a DOC in the carer's   |
|                                   | record. For more details, see     |
|                                   | Advising verbally of an           |
|                                   | unfavourable decision (CLK).      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Calculate the total ADAT score +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Calculate the total ADAT score,   |
|                                   | which is the total achieved when  |
|                                   | the scores from the THP and carer |
|                                   | questionnaires are combined.      |
|                                   |                                   |
|                                   | -   Less than 20, go to Step 6    |
|                                   | -   20 or greater, go to Step 7   |
+-----------------------------------+-----------------------------------+
| 6                                 | Total ADAT score not sufficient   |
|                                   | to attract CP + Read more ...     |
|                                   |                                   |
|                                   | The total ADAT score is not       |
|                                   | sufficient to attract CP for the  |
|                                   | carer. The CP claim should be     |
|                                   | manually rejected. At least 2     |
|                                   | genuine attempts to contact must  |
|                                   | be made to contact the carer to   |
|                                   | tell of the unfavourable          |
|                                   | decision. Record details on a DOC |
|                                   | in the carer's record. For more   |
|                                   | details, see Advising verbally of |
|                                   | an unfavourable decision (CLK).   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Check THP's responses on          |
|                                   | SA332(a) + Read more ...          |
|                                   |                                   |
|                                   | Has the THP responded No or       |
|                                   | Unsure on the SA332(a) to either  |
|                                   | of the following questions:       |
|                                   |                                   |
|                                   | 'Does the person require help on  |
|                                   | a daily basis because of their    |
|                                   | disability and/or medical         |
|                                   | condition(s) to carry out routine |
|                                   | personal activities OR because    |
|                                   | they may be at risk to themselves |
|                                   | or to others?'                    |
|                                   |                                   |
|                                   | 'Is this care required for a      |
|                                   | significant period each day (at   |
|                                   | least the equivalent of a normal  |
|                                   | working day)?'                    |
|                                   |                                   |
|                                   | -   Yes, go to Step 8             |
|                                   | -   No, go to Step 9              |
+-----------------------------------+-----------------------------------+
| 8                                 | Making an assessment + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   The THP's responses should    |
|                                   |     not be an automatic           |
|                                   |     assessment that constant care |
|                                   |     is not being provided         |
|                                   | -   The decision maker (DM) will  |
|                                   |     need to make an assessment    |
|                                   |     based on the total level of   |
|                                   |     care provided to the care     |
|                                   |     receiver, that is, a carer    |
|                                   |     should reasonably be expected |
|                                   |     to provide at least the       |
|                                   |     equivalent of a normal        |
|                                   |     working day in personal care  |
|                                   | -   When coding the CP claim the  |
|                                   |     review officer's              |
|                                   |     determination if the constant |
|                                   |     care qualifications are met   |
|                                   |     should be recorded on the     |
|                                   |     Care Receiver Care &          |
|                                   |     Institution Details (CRCI)    |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| 9                                 | Code the claim for CP + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | For more details, see Coding a    |
|                                   | paper claim for Carer Payment     |
|                                   | (CP) where the care receiver is   |
|                                   | 16 years of age or over.          |
|                                   |                                   |
|                                   | Record details on a DOC in the    |
|                                   | carer's record.                   |
+-----------------------------------+-----------------------------------+
