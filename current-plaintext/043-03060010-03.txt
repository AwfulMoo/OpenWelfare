This page contains an explanation of the more complex questions on the
Business Details form (MOD F). Questions such as the name and address,
on Q1 to 10 have not been included.

Completing the MOD F

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| Information - About your business | A separate MOD F is required for  |
|                                   | each business                     |
+-----------------------------------+-----------------------------------+
| Q6-7                              | Are you self-employed operating a |
|                                   | business as a: + Read more ...    |
|                                   |                                   |
|                                   | -   Sole trader, go to Q12        |
|                                   | -   Partnership, the recipient    |
|                                   |     needs to give details of      |
|                                   |     business partners at Q11      |
+-----------------------------------+-----------------------------------+
| Q8-10                             | Work Bonus questions + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | These questions are for           |
|                                   | customers:                        |
|                                   |                                   |
|                                   | -   Of Age Pension age, or        |
|                                   | -   Claiming Age Pension and      |
|                                   |     involved in a partnership     |
|                                   |                                   |
|                                   | The customer needs to advise if   |
|                                   | their role (and their partner's   |
|                                   | role) in the business involves    |
|                                   | personal exertion. If so, they    |
|                                   | will need to state what           |
|                                   | percentage of their income is     |
|                                   | from personal exertion.           |
|                                   |                                   |
|                                   | If the customer has:              |
|                                   |                                   |
|                                   | -   not indicated at Q9 or Q10    |
|                                   |     any Work Bonus eligibility,   |
|                                   |     set the percentage as nil     |
|                                   |     (zero)                        |
|                                   | -   ticked 'Yes' at Q9 about      |
|                                   |     physical exertion but has not |
|                                   |     completed the percentage at   |
|                                   |     Q10, the percentage must be   |
|                                   |     obtained. See Requesting      |
|                                   |     information (CLK)             |
+-----------------------------------+-----------------------------------+
| Q12                               | Recipient needs to provide: +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Name and address of the       |
|                                   |     business                      |
|                                   | -   Date business commenced       |
|                                   |     operating from                |
+-----------------------------------+-----------------------------------+
| Q14                               | Documentation for business + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The customer must provide:        |
|                                   |                                   |
|                                   | For businesses:                   |
|                                   |                                   |
|                                   | -   Their latest personal income  |
|                                   |     tax return                    |
|                                   |     Note: this could be a copy of |
|                                   |     documents lodged through      |
|                                   |     MyTax                         |
|                                   | -   The latest profit and loss    |
|                                   |     statement for the business    |
|                                   | -   The latest balance sheet for  |
|                                   |     the business                  |
|                                   | -   The latest depreciation       |
|                                   |     schedule                      |
|                                   |                                   |
|                                   | For business partnerships:        |
|                                   |                                   |
|                                   | -   A copy of the partnership     |
|                                   |     agreement                     |
|                                   | -   The latest partnership Income |
|                                   |     Tax Return                    |
|                                   | -   If the partner is also a      |
|                                   |     business partner, the         |
|                                   |     partner's latest personal     |
|                                   |     Income Tax Return             |
|                                   | -   Current and capital account   |
|                                   |     details for each business     |
|                                   |     partner                       |
+-----------------------------------+-----------------------------------+
| Q15                               | Does the business hold any Real   |
|                                   | Estate? + Read more ...           |
|                                   |                                   |
|                                   | A MOD R needs to be completed for |
|                                   | each property held by the         |
|                                   | business.                         |
+-----------------------------------+-----------------------------------+
| Q16                               | Is the current business income    |
|                                   | different to that shown on the    |
|                                   | latest Profit and Loss Statement  |
|                                   | or Income Tax Returns? + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the current income from the    |
|                                   | business differs to the latest    |
|                                   | Profit and Loss Statement or      |
|                                   | Income Tax Returns, it is in the  |
|                                   | best interest for the             |
|                                   | customer/partner to provide:      |
|                                   |                                   |
|                                   | -   An estimate and gross amount  |
|                                   |     of the income for the current |
|                                   |     financial year                |
|                                   | -   Reason(s) why the income      |
|                                   |     differs, including supporting |
|                                   |     documentation                 |
+-----------------------------------+-----------------------------------+
| Q17                               | If the current value of the       |
|                                   | business assets and liabilities   |
|                                   | are different + Read more ...     |
|                                   |                                   |
|                                   | The recipient will need to        |
|                                   | provide:                          |
|                                   |                                   |
|                                   | -   A separate sheet with the     |
|                                   |     current value of the assets   |
|                                   |     and any liabilities           |
|                                   | -   Reason(s) why the values are  |
|                                   |     different                     |
|                                   | -   Documents to support their    |
|                                   |     estimate                      |
+-----------------------------------+-----------------------------------+
| Q18-20                            | Is the business still             |
|                                   | operating? + Read more ...        |
|                                   |                                   |
|                                   | Yes, go to question 22            |
|                                   |                                   |
|                                   | If business is no longer          |
|                                   | operating, the recipient will     |
|                                   | need to advise:                   |
|                                   |                                   |
|                                   | -   Reasons why the business is   |
|                                   |     no longer operating           |
|                                   | -   When the business stopped     |
|                                   |     operating                     |
|                                   | -   If they intend to resume in   |
|                                   |     the business, including date  |
|                                   |     they intend to resume         |
|                                   | -   If the business is generating |
|                                   |     any income                    |
+-----------------------------------+-----------------------------------+
| Q22-23                            | Working in the business + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers/applicants will need to |
|                                   | advise:                           |
|                                   |                                   |
|                                   | -   If they are still working in  |
|                                   |     the business, and if yes,     |
|                                   | -   The type of work they are     |
|                                   |     doing                         |
|                                   | -   The number of hours spent     |
|                                   |     working                       |
+-----------------------------------+-----------------------------------+
| Q24-25                            | Not working in the business +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer/applicant is not  |
|                                   | working in the business they will |
|                                   | need to advise:                   |
|                                   |                                   |
|                                   | -   Do they intend to resume the  |
|                                   |     business, and                 |
|                                   | -   Is anyone helping them to run |
|                                   |     it                            |
+-----------------------------------+-----------------------------------+
