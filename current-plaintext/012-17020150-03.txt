This table describes the steps to follow to verify a DVA health card
holders eligibility before paying or rejecting podiatry services when
the system returns '2-CHK-ELIG'

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | White card for podiatry           |
|                                   | treatment + Read more ...         |
|                                   |                                   |
|                                   | As podiatry services are specific |
|                                   | to a particular area of the body, |
|                                   | providers do not need to specify  |
|                                   | the condition being treated, as   |
|                                   | long as the DVA card holder’s     |
|                                   | accepted disabilities match a     |
|                                   | condition listed in step 3 of     |
|                                   | this table.                       |
|                                   |                                   |
|                                   | Provisional Access to Medical     |
|                                   | Treatment (PAMT)                  |
|                                   |                                   |
|                                   | For Veterans with PAMT            |
|                                   | eligibility, provider must        |
|                                   | specify 1 or more of the 20 PAMT  |
|                                   | conditions listed. For more       |
|                                   | information see Patient           |
|                                   | eligibility for Veterans’ Affairs |
|                                   | Processing (VAP).                 |
|                                   |                                   |
|                                   | See Resources page for links to   |
|                                   | the DVA website for more          |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | View DVA card holder's listed     |
|                                   | entitlement + Read more ...       |
|                                   |                                   |
|                                   | Press [PF10] to view their listed |
|                                   | entitlements.                     |
|                                   |                                   |

|                                   | they are eligible to claim.       |

| 3                                 | Review card holder's conditions + |
|                                   | Read more ...                     |
|                                   |                                   |

Footwear modifications and repairs F611-12 and F615

Process footwear modifications and repairs

This table describes how Service Officers process Medical Grade Footwear
(MGF) items F611, F612 and F615 for VAP.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check and process amount claimed  |
|                                   | for MGF + Read more ...           |
|                                   |                                   |
|                                   | Is the amount claimed lower than  |
|                                   | or equal to $100, excluding GST?  |
|                                   |                                   |
|                                   | -   Yes, override with PI 18 and  |
|                                   |     select [PF9] to confirm the   |
|                                   |     fee                           |
|                                   | -   No, go to Step 2              |
|                                   |                                   |
|                                   | See also: Footwear Modifications  |
|                                   | and Repairs fee schedule (DVA     |
|                                   | website). The Resources page      |
|                                   | contains a link to the DVA        |
|                                   | website.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for prior approval + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check for prior financial         |
|                                   | authorisation where the amount    |
|                                   | claimed is greater than $100.     |
|                                   |                                   |
|                                   | Requires Prior Financial          |
|                                   | Authorisation. Check PARS for an  |
|                                   | approval.                         |
|                                   |                                   |
|                                   | Is there a current PARS approval  |
|                                   | present or the item claimed?      |
|                                   |                                   |
|                                   | -   Yes, override with PI 18 and  |
|                                   |     select [PF9] to confirm the   |
|                                   |     fee                           |
|                                   | -   No, reject the item with      |
|                                   |     reason RSN code S 279         |
|                                   |                                   |
|                                   | Note: select [PF1] within the     |
|                                   | claim, to display the maximum     |
|                                   | amount claimable both with and    |
|                                   | without prior financial approval. |
