Social Security Agreement between Australia and Austria

This table describes general information about the Agreement with
Austria including the social security system in Austria, history of the
Agreement, exchange of information and contact details.

| Austrian Social Security System   | Austrian Social Security System + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In Austria, the pension insurance |
|                                   | legislation requires the          |
|                                   | compulsory payment of social      |
|                                   | security contributions on behalf  |
|                                   | of employees, usually through     |
|                                   | deductions from paid employment.  |
|                                   |                                   |
|                                   | Contributions insure employees    |
|                                   | against disability and            |
|                                   | unemployment and provides for     |
|                                   | income in the form of a pension   |
|                                   | once the person reaches           |
|                                   | retirement age.                   |
|                                   |                                   |
|                                   | The legislation is the            |
|                                   | responsibility of the Austrian    |
|                                   | federal government but the        |
|                                   | collection of contributions and   |
|                                   | payment of pensions is the        |
|                                   | responsibility of the various     |
|                                   | pension funds. These are broadly  |
|                                   | arranged to cover the different   |
|                                   | industries. There is also a       |
|                                   | central institution, which        |
|                                   | oversees all Austrian funds.      |
|                                   |                                   |
|                                   | There are secondary, voluntary    |
|                                   | occupational schemes and a        |
|                                   | universal social protection       |
|                                   | scheme for those with low         |
|                                   | incomes.                          |

|                                   |                                   |
|                                   | Tax treaty                        |
|                                   |                                   |
|                                   | Australia has a double tax        |
|                                   | agreement with Austria, which     |
|                                   | avoids the need to pay tax in     |
|                                   | both countries. A person who is   |
|                                   | 'resident' in one country         |
|                                   | (according to the definition in   |
|                                   | the tax agreement) generally only |
|                                   | pays tax on pensions in that      |
|                                   | country.                          |
|                                   |                                   |
|                                   | Tax deductions from income from   |
|                                   | one country may be allowed as a   |
|                                   | credit against tax payable for    |
|                                   | residents of the other country.   |
|                                   |                                   |
|                                   | Tax deductions                    |
|                                   |                                   |
|                                   | In Austria, Austrian pensions     |
|                                   | over a certain level are taxed at |
|                                   | source (Lohnsteuer). Austrian     |
|                                   | pensions are not taxed in         |
|                                   | Australia.                        |
|                                   |                                   |
|                                   | The gross rate of Austrian        |
|                                   | pension (before deductions) is    |
|                                   | maintained.                       |
|                                   |                                   |
|                                   | Austrian tax year                 |
|                                   |                                   |
|                                   | Austria generally uses the        |
|                                   | calendar year (1 Jan - 31 Dec) as |
|                                   | the tax year. Individuals must    |
|                                   | lodge a return by 31 July each    |
|                                   | year.                             |

|                                   |     applies, and                  |

|                                   |     included in the proportion    |
|                                   |     rate. These amounts are paid  |
|                                   |     automatically as a component  |
|                                   |     of the Australian pension if  |
|                                   |     the customer is qualified     |
|                                   |     (S14A, SS(IA)A 1999)          |

|                                   | includes the proportionalisation  |
|                                   | of the Austrian pension (Randisi  |
|                                   | concession) for use in the income |
|                                   | test (Article 12.1). The rate of  |
|                                   | foreign pension used in the       |
|                                   | income test is the customer's     |

|                                   | determines the Randisi proportion |

|                                   | to the legislation that is used   |
|                                   | to determine the proportional     |
|                                   | rate (420 months in most cases).  |
|                                   |                                   |

|                                   | Austrian pension (Article 12.2).  |
|                                   |                                   |

|                                   | qualified for an Australian       |

|                                   | Australia will be nil.            |
|                                   |                                   |

|                                   | partner's WLR may be used. See    |
|                                   | Working Life Residence (WLR).     |

|                                   | either they:                      |
