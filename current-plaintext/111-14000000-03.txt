Processing incomplete letters

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | ADV letter incomplete + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Access the Activity List (AL) |
|                                   |     screen                        |
|                                   | -   Select the ADV Letter         |
|                                   |     Incomplete activity           |
|                                   | -   Select Continue               |
|                                   |                                   |
|                                   | The Letter Display (LDC or LDT)   |
|                                   | screen is displayed.              |
|                                   |                                   |
|                                   | Complete any blank fields with    |
|                                   | the correct information.          |
|                                   |                                   |
|                                   | For Customer First:               |
|                                   |                                   |
|                                   | -   Select Continue to confirm    |
|                                   |     the text                      |
|                                   | -   Select Next Page to scroll to |
|                                   |     the next page                 |
|                                   |                                   |
|                                   | For Customer Record:              |
|                                   |                                   |
|                                   | -   Press [Enter] to confirm the  |
|                                   |     text                          |
|                                   | -   Select [F8] to scroll to the  |
|                                   |     next page                     |
|                                   |                                   |
|                                   | Once all blank fields have been   |
|                                   | completed, key FIN in the Command |
|                                   | field to complete the letter.     |
|                                   |                                   |
|                                   | The completed letter will be      |
|                                   | queued for central print, or      |
|                                   | delivered to the customer's myGov |
|                                   | Inbox in 2 working days.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Tell Help Desk, TEXT not found +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | These letters are incomplete      |
|                                   | because 1 or more of the          |
|                                   | paragraphs used to create the     |
|                                   | letter is missing.                |
|                                   |                                   |
|                                   | Report this to the ICT helpdesk   |
|                                   | by submitting the Enterprise      |
|                                   | Outbound Correspondence (EOC)     |
|                                   | issue webform:                    |
|                                   |                                   |
|                                   | -   Go to mySupport               |
|                                   | -   Key the name of the webform   |
|                                   |     or search using key words,    |
|                                   |     (e.g. SMS, email, messages,   |
|                                   |     letters, communications)      |
|                                   | -   Complete the form and include |
|                                   |     the missing paragraph code in |
|                                   |     the issue notes               |
|                                   | -   Submit the webform and take   |
|                                   |     note of the receipt number    |
