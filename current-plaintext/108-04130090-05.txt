Services Australia Valuation Request (SA470)

The SA470 is used for situations such as gifting, historical valuations
(for example, undisclosed properties), properties owned by private
trusts & private companies, and Residential Care Assessments.

Information on taxi licences

+-----------------------------------+-----------------------------------+
| State/territory                   | Description                       |
+-----------------------------------+-----------------------------------+
| Australian Capital Territory      | Road Transport Authority (RTA)    |
|                                   | deals with transfer of taxi       |
|                                   | licences and is able to provide   |
|                                   | value of a licence.               |
|                                   |                                   |
|                                   | If a value cannot be obtained     |
|                                   | from the RTA, Market Value of a   |
|                                   | taxi licence can also be          |
|                                   | requested via email to the        |
|                                   | Valuation Helpdesk.               |
+-----------------------------------+-----------------------------------+
| New South Wales                   | NSW taxi plate valuations for     |
|                                   | Sydney can now be viewed via the  |
|                                   | 'Point to Point' Transport        |
|                                   | Commissioners website.            |
|                                   |                                   |
|                                   | It may still be necessary to      |
|                                   | contact Point to Point regarding  |
|                                   | current market value of taxi      |
|                                   | plates outside the metro area of  |
|                                   | NSW.                              |
|                                   |                                   |
|                                   | If the sales/transfer data is     |
|                                   | older than 6 months, request a    |
|                                   | valuation from the Valuation      |
|                                   | Helpdesk.                         |
|                                   |                                   |
|                                   | Once received by an ISP           |
|                                   | recipient, a payment under the    |
|                                   | Additional Assistance Payment     |
|                                   | Scheme is assessable. See Step 4  |
|                                   | of the Assessing lump sums table  |
|                                   | in the process tab of Treatment   |
|                                   | of lump sums for coding           |
|                                   | instructions.                     |
+-----------------------------------+-----------------------------------+
| Northern Territory                | In the NT, taxi licences cannot   |
|                                   | be bought or sold, therefore they |
|                                   | have no intrinsic value. The      |
|                                   | number of taxis has been capped   |
|                                   | in the Darwin and Alice Springs   |
|                                   | areas and all licences are        |
|                                   | allocated by ballot.              |
|                                   |                                   |
|                                   | Licences are not purchased, but   |
|                                   | incur an annual fee (the same way |
|                                   | car registration is an annual     |
|                                   | fee). If the licence holder no    |
|                                   | longer wishes to retain the       |
|                                   | licence, they must return it to   |
|                                   | the relevant government           |
|                                   | department. The licence holder    |
|                                   | cannot transfer the licence to    |
|                                   | another person in any             |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Depending on when it is returned, |
|                                   | the licence holder may be         |
|                                   | entitled to a refund of some of   |
|                                   | the current year's licence fee.   |
|                                   | The fees vary depending on the    |
|                                   | area and type of taxi, that is,   |
|                                   | ordinary, multi-purpose or peak   |
|                                   | period taxi (PPT). For            |
|                                   | information regarding current     |
|                                   | taxi and commercial passenger     |
|                                   | vehicle licence fees, visit the   |
|                                   | transport industry fees webpage   |
|                                   | of the NT Government.             |
+-----------------------------------+-----------------------------------+
| Queensland                        | Qld Taxi/Limousine Licences       |
|                                   |                                   |
|                                   | To check the most recent sales    |

|                                   |                                   |

| Victoria                          | Victorian Taxi Licence -          |
|                                   | Transitional Assistance payment   |
|                                   |                                   |
|                                   | Legislation for the Victorian     |
|                                   | taxi industry support package     |
|                                   | passed on 10 August 2017 and      |
|                                   | payments commenced being made     |
|                                   | during September 2017.            |
|                                   |                                   |
|                                   | Income and Assets Impact          |
|                                   |                                   |
|                                   | Once received by an ISP           |
|                                   | recipient, Transition Assistance  |
|                                   | payments will be exempt lump sums |
|                                   | and their Victorian taxi licence  |
|                                   | value will be $0.                 |
|                                   |                                   |
|                                   | Service Officers must ask the     |
|                                   | recipient what they have done     |
|                                   | with the payments and update      |
|                                   | their ISP records accordingly.    |
|                                   |                                   |
|                                   | Assistance payments               |
|                                   |                                   |
|                                   | Transition assistance payments    |
|                                   | will be paid to eligible licence  |
|                                   | holders up to a total of 4        |
|                                   | licences for each licence holder. |
|                                   |                                   |
|                                   | Transition assistance payments    |
|                                   | will be provided to the entity    |
|                                   | registered with the Taxi Services |
|                                   | Commission as the licence holder. |
|                                   | If an entity owns more than 4     |
|                                   | licences in total, and the        |
|                                   | licences are different types, the |
|                                   | total payment will be calculated  |
|                                   | on the 4 highest value licences.  |
|                                   |                                   |
|                                   | To be eligible for the payments,  |
|                                   | the licence holder must:          |
|                                   |                                   |
|                                   | -   have held the licence on 15   |
|                                   |     August 2016, and              |
|                                   | -   continue to hold the licence  |
|                                   |     until 14 July 2017            |
|                                   |                                   |
|                                   | Possible exceptions to this may   |
|                                   | include where:                    |
|                                   |                                   |
|                                   | -   licence ownership has changed |
|                                   |     due to a bequest or court     |
|                                   |     order                         |
|                                   | -   a hire car or special purpose |
|                                   |     vehicle licence was applied   |
|                                   |     and paid for prior to 15      |
|                                   |     August 2016                   |
|                                   | -   a licence transfer            |
|                                   |     application was received by   |
|                                   |     the Taxi Services Commission  |
|                                   |     prior to 15 August 2016, or   |
|                                   | -   a Greater Metropolitan Taxi   |
|                                   |     Licence Release (GMTLR)       |
|                                   |     licence has been converted    |
|                                   |                                   |
|                                   | The Taxi Services Commission      |
|                                   | contacted licence holders in 2017 |
|                                   | with further details about        |
|                                   | transition assistance payments.   |
