| SH01-SH99                         |                                   |
|                                   | -   Maximum time payable = 150    |
|                                   |     minutes                       |
|                                   | -   Times need to be rounded to   |
|                                   |     the nearest 15 minutes. E.g.  |
|                                   |     48 mins would be keyed as     |
|                                   |     0:45hr; 78 mins would be      |
|                                   |     keyed as 1:15HR               |
|                                   | -   Key the time in the FEE field |
|                                   |     as H:MMhr and press [ENTER]   |
|                                   | -   If the provider has claimed   |
|                                   |     an amount without times,      |
|                                   |     reject with RSN code S 333    |
|                                   | -   If time claimed exceeds max   |
|                                   |     limit, reduce time to max     |
|                                   |     limit and insert RSN code 362 |
|                                   |                                   |
|                                   | Note: services that are rendered  |
|                                   | away from the provider's rooms    |
|                                   | attract a loading in addition to  |
|                                   | the normal time-based fee.        |
|                                   |                                   |
|                                   | This loading is equal to one      |
|                                   | quarter of the normal hourly      |
|                                   | rate.                             |
|                                   |                                   |
|                                   | The system will automatically     |
|                                   | apply the loading for applicable  |
|                                   | items.                            |
