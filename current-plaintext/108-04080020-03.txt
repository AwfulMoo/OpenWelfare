This page contains information relating to exempt assets and the
assessment process. There is also information if the customer vacates
their home.

Exempt assets

Assessment of potential exempt assets

This table describes information to assist when assessing potential
exempt assets.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Exempt asset + Read more ...      |
|                                   |                                   |
|                                   | Does the customer have an exempt  |
|                                   | asset?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, record any assessable     |
|                                   |     asset on relevant screen(s).  |
|                                   |     For assistance see, Coding    |
|                                   |     income and assets. Record     |
|                                   |     details on a DOC              |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Exempt monies + Read more ...     |
|                                   |                                   |
|                                   | Is the exempt asset monies        |
|                                   | received from the National        |
|                                   | Disability Insurance Scheme       |
|                                   | (NDIS)?                           |
|                                   |                                   |
|                                   | -   Yes, DOC the receipt of the   |
|                                   |     funds only. Do not code the   |
|                                   |     funds as an asset on the      |
|                                   |     customer's record             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Special Disability Trust + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the exempt asset in a Special  |
|                                   | Disability Trust (SDT)?           |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   Any changes or updates to an  |
|                                   |     SDT are actioned by the SDT   |
|                                   |     Processing Team. Send an      |
|                                   |     ACTDOC with details using the |
|                                   |     region and keyword of 'SDT'   |
|                                   | -   For SDT claims, see Special   |
|                                   |     Disability Trust (SDT) -      |
|                                   |     initial contact               |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | ACT and NSW Government Buybacks + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Has the customer had their home   |
|                                   | purchased by the ACT or NSW       |
|                                   | Governments under the scheme for  |
|                                   | homes that may be contaminated by |
|                                   | loose fill asbestos installed as  |
|                                   | insulation?                       |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   Advise the customer the sum   |
|                                   |     received, less the amount     |
|                                   |     they are intending to spend   |
|                                   |     on a replacement, will not be |
|                                   |     assessed as an asset or       |
|                                   |     deemed to be earning income.  |
|                                   |     This applies until they have  |
|                                   |     replaced the property subject |
|                                   |     to a 12 month limit (unless   |
|                                   |     there is a valid reason for   |
|                                   |     taking longer)                |
|                                   | -   Advise the customer they must |
|                                   |     continue to notify the        |
|                                   |     balance or value of all their |
|                                   |     assets, less the net amount   |
|                                   |     they received from the        |
|                                   |     buyback. For example, the     |
|                                   |     customer received $500,000    |
|                                   |     for their property and paid   |
|                                   |     off the mortgage of $200,000. |
|                                   |     For 12 months, or until the   |
|                                   |     money is used to buy another  |
|                                   |     property, they must advise    |
|                                   |     the actual bank balance(s)    |
|                                   |     less $300,000                 |
|                                   | -   Verification it is a buyback  |
|                                   |     home or the amount received   |
|                                   |     should not be required        |
|                                   | -   Set a Manual Follow Up for 12 |
|                                   |     months                        |
|                                   | -   Document the information      |
|                                   |     provided by the customer and  |
|                                   |     the notification obligations  |
|                                   |     explained to them             |
|                                   | -   If the funds have been placed |
|                                   |     anywhere other than a bank    |
|                                   |     account, contact the Income   |
|                                   |     Support Means Test team       |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Type of exempt asset + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer's exempt asset    |
|                                   | one of the following:             |
|                                   |                                   |
|                                   | -   compensation and insurance    |
|                                   |     payments for damage to        |
|                                   |     buildings, plant or personal  |
|                                   |     effects                       |
|                                   | -   principal home                |
|                                   | -   an inherited life,            |
|                                   |     reversionary, remainder or    |
|                                   |     contingent interest in an     |
|                                   |     asset                         |
|                                   | -   pre-paid funeral expense(s)   |
|                                   |     or cemetery plot(s)           |
|                                   | -   disability aids               |
|                                   | -   car(s) gifted from the        |
|                                   |     Commonwealth                  |
|                                   | -   medals and/or decorations for |
|                                   |     valour that are not part of a |
|                                   |     collection or hobby           |
|                                   | -   native title right(s) or      |
|                                   |     interest?                     |
|                                   |                                   |
|                                   | For more information including    |
|                                   | documents required and coding,    |
|                                   | see Resources page.               |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 6                                 | Verification required + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If verification is required       |
|                                   | (depending on asset type), record |
|                                   | details on a DOC and advise       |
|                                   | customer to provide               |
|                                   | documentation, see Coding income  |
|                                   | and assets for Centrelink         |
|                                   | payments and services             |
|                                   |                                   |
|                                   | -   If the customer's asset       |
|                                   |     circumstances have changed,   |
|                                   |     see Verifying and recording   |
|                                   |     assets changes                |
|                                   | -   Code manual review for follow |
|                                   |     up of receipt of verification |
|                                   | -   Advise customer of the        |
|                                   |     outcome                       |
|                                   |                                   |
|                                   | If verification is not required,  |
|                                   | record details on a DOC and       |
|                                   | advise customer of the outcome.   |
|                                   | The Resources page contains       |
|                                   | examples of exempt assets         |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Income stream + Read more ...     |
|                                   |                                   |
|                                   | Is the customer's exempt asset an |
|                                   | income stream?                    |
|                                   |                                   |
|                                   | -   Yes, see Income streams.      |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 8                                 | Superannuation + Read more ...    |
|                                   |                                   |
|                                   | Is the customer's and or their    |
|                                   | partner's superannuation an       |
|                                   | exempt asset?                     |
|                                   |                                   |
|                                   | -   Yes, record details on a DOC  |
|                                   |     that customer has a           |
|                                   |     superannuation fund. As the   |
|                                   |     superannuation is exempt      |
|                                   |     there is no need to provide   |
|                                   |     documentation, this will be   |
|                                   |     needed when the customer      |
|                                   |     turns Age Pension age.        |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 9              |
+-----------------------------------+-----------------------------------+
| 9                                 | Vacation or Sale of home + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the customer vacating the      |
|                                   | principal residence and wants to  |
|                                   | see if it can be considered to be |
|                                   | an exempt asset?                  |
|                                   |                                   |
|                                   | Select the link relevant to the   |
|                                   | customers circumstance:           |
|                                   |                                   |
|                                   | -   Vacation of principal home    |
|                                   |     due to illness                |
|                                   | -   Change of address to an aged  |
|                                   |     care home                     |
|                                   | -   Selling a home                |
|                                   | -   Temporary vacation of         |
|                                   |     principal home                |
|                                   | -   Permanent vacation of         |
|                                   |     principal home                |
