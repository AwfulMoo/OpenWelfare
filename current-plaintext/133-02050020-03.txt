Accessing HPOS from Services Australia website

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access Services Australia         |
|                                   | website + Read more ...           |
|                                   |                                   |
|                                   | Go to Services Australia website. |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Navigate to the myGov Sign in     |

|                                   | Read more ...                     |
|                                   |                                   |

|                                   | professionals.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Logon to HPOS + Read more ...     |
|                                   |                                   |
|                                   | Select Log on using PRODA button. |

Accessing HPOS with PKI Site Certificate

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access Services Australia         |
|                                   | website + Read more ...           |
|                                   |                                   |
|                                   | Go to Services Australia website. |
|                                   | The Resources page contains a     |
|                                   | link.                             |
|                                   |                                   |
|                                   | Navigate to the myGov Sign in     |
|                                   | button on top right of screen.    |
|                                   |                                   |
|                                   | Select HPOS under the Health      |
|                                   | professionals.                    |
|                                   |                                   |
|                                   | The HPOS landing page will        |
|                                   | display.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Select PKI Site Certificate log   |
|                                   | on + Read more ...                |
|                                   |                                   |
|                                   | Select Log on using a PKI         |
|                                   | Certificate button.               |
|                                   |                                   |
|                                   | The 'Choose a digital             |
|                                   | certificate' pop up window will   |
|                                   | display.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Select digital certificate + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the HPOS digital           |
|                                   | certificate to be used, from the  |
|                                   | list displayed.                   |
+-----------------------------------+-----------------------------------+
| 4                                 | Return to HPOS log on page + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select OK to proceed.             |
|                                   |                                   |
|                                   | The ‘private key’ pop up window   |
|                                   | may display.                      |
|                                   |                                   |
|                                   | Note: the CryptoAPI Private Key   |
|                                   | (password) may not display with   |
|                                   | each log on depending on the      |
|                                   | security level chosen at the time |
|                                   | of registration.                  |
|                                   |                                   |
|                                   | Select Cancel to return to HPOS   |
|                                   | log on page.                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Enter CryptoAPI Private Key       |
|                                   | password + Read more ...          |
|                                   |                                   |
|                                   | If the ‘private key’ window       |
|                                   | appears, enter the CryptoAPI      |
|                                   | Private Key password.             |
+-----------------------------------+-----------------------------------+
| 6                                 | Display Terms and Conditions of   |
|                                   | Use + Read more ...               |
|                                   |                                   |
|                                   | Select OK to proceed.             |
|                                   |                                   |
|                                   | The 'HPOS Terms and Conditions of |
|                                   | Use and Access' page will         |
|                                   | display.                          |
|                                   |                                   |
|                                   | Note: select Cancel to return to  |
|                                   | HPOS log on page.                 |
+-----------------------------------+-----------------------------------+
| 7                                 | Agree to Terms and Conditions of  |
|                                   | Use + Read more ...               |
|                                   |                                   |
|                                   | Select I Agree to proceed.        |
|                                   |                                   |
|                                   | The 'Patient Verification' page   |
|                                   | will display.                     |
+-----------------------------------+-----------------------------------+

How to check if a site has a valid certificate

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check certificate + Read more ... |
|                                   |                                   |
|                                   | To check if a site has a valid    |
|                                   | certificate, go to the Healthcare |
|                                   | Public Directory Search homepage. |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to this website.             |
|                                   |                                   |
|                                   | Note: if the site is currently    |
|                                   | transmitting claims via Medicare  |
|                                   | Online it is likely that it does  |
|                                   | have a current certificate.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Key details + Read more ...       |
|                                   |                                   |
|                                   | Key the relevant information for  |
|                                   | the person or site into the       |
|                                   | applicable fields:                |
|                                   |                                   |
|                                   | -   first name                    |
|                                   | -   surname or RA (certificate)   |
|                                   |     number                        |
|                                   | -   email address                 |
|                                   | -   organisation name             |
|                                   | -   state                         |
|                                   |                                   |
|                                   | Select Search.                    |
|                                   |                                   |
|                                   | A list of all valid certificates  |
|                                   | registered that match the         |
|                                   | criteria details keyed.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Revoke/suspend/reinstate/renew    |
|                                   | certificate + Read more ...       |
|                                   |                                   |
|                                   | If there is an existing           |
|                                   | certificate which has expired,    |
|                                   | complete the Request for          |
|                                   | revocation                        |
|                                   | /suspension/reinstatement/renewal |
|                                   | of location and individual        |
|                                   | certificate form.                 |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to this website.             |
+-----------------------------------+-----------------------------------+
