|                                   |                                   |
|                                   | SWIS case notes are not           |
|                                   | mandatory. They may be required   |
|                                   | to record a complex customer      |
|                                   | situation or sensitive            |
|                                   | information which should not      |
|                                   | recorded in a DOC.                |
|                                   |                                   |
|                                   | Reference to the existence of     |
|                                   | SWIS case notes should be made in |
|                                   | the social work DOC. Case notes   |
|                                   | should be succinct using the      |
|                                   | following social work assessment  |
|                                   | pro forma:                        |
|                                   |                                   |
|                                   | -   General customer              |
|                                   |     circumstances: circumstances  |
|                                   |     identified by file review     |
|                                   | -   Customer circumstances:       |
|                                   |     presenting circumstances and  |
|                                   |     vulnerabilities               |
|                                   | -   Customer circumstances        |
|                                   |     impacting on capacity to      |
|                                   |     comply: the social worker     |
|                                   |     identifies how personal       |
|                                   |     circumstances impact on the   |
|                                   |     customer's capacity to comply |
|                                   |     with their compulsory         |
|                                   |     requirements                  |
|                                   | -   Social worker                 |
|                                   |     intervention/referrals        |
|                                   |     completed: the social worker  |
|                                   |     records their intervention    |
|                                   |     and referrals made. For       |
|                                   |     example, crisis intervention, |
|                                   |     information provision, case   |
|                                   |     discussion with provider etc. |
|                                   |                                   |
|                                   | The Resources page contains an    |
|                                   | example of social work case notes |
|                                   | recorded for participation and    |
|                                   | compliance related issues.        |
