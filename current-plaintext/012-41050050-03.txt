This page contains the process for patient signatures for Telehealth.

Assignment of benefit via email

Email agreement

This process applies under Section 127 of the Health Insurance Act 1973.
The assignment of benefit form must be signed by the patient after the
service has been provided and the form completed. A copy of the
completed assignment form must be given to the patient.

Email agreement process for patient signature

Table 1: This table describes the email agreement process for a patient
to assign their right to a Medicare benefit to a specialist via email.

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Informs patient of actions        |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | The specialist or consultant at   |
|                                   | the time of the telehealth        |
|                                   | consultation tells patient:       |
|                                   |                                   |
|                                   | -   they need to verbally agree   |
|                                   |     to assign the benefit to bulk |
|                                   |     bill Medicare for the         |
|                                   |     service, and                  |
|                                   | -   an email will be sent to      |
|                                   |     their nominated email address |
|                                   |     confirming the details of the |
|                                   |     service and agreement to      |
|                                   |     assign the benefit, which the |
|                                   |     patient will need to respond  |
|                                   |     to                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Sends email with details of       |
|                                   | service and privacy note + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Before lodging the claim, the     |
|                                   | specialist sends an email to the  |
|                                   | patient listing:                  |
|                                   |                                   |
|                                   | details of the service:           |
|                                   |                                   |
|                                   | -   item number or description of |
|                                   |     service                       |
|                                   | -   benefit amount                |
|                                   | -   date and time of service      |
|                                   | -   patient's name (Medicare card |
|                                   |     number not to be included)    |
|                                   | -   practitioner's name (Provider |
|                                   |     number not to be included)    |
|                                   |                                   |
|                                   | a privacy note                    |
|                                   |                                   |
|                                   | The Resources page includes a     |
|                                   | sample of the privacy note.       |
|                                   |                                   |
|                                   | Note: due to privacy reasons, the |
|                                   | Medicare card number and the      |
|                                   | provider number must not be       |
|                                   | included in the email.            |
+-----------------------------------+-----------------------------------+
| 3                                 | Completes statement and sends     |
|                                   | email + Read more ...             |
|                                   |                                   |
|                                   | The patient completes statement   |
|                                   | and sends email to specialist     |
|                                   | agreeing to the assignment of     |
|                                   | benefit.                          |
+-----------------------------------+-----------------------------------+
| 4                                 | Completes form, retains email and |
|                                   | provides copy of form to          |
|                                   | patient + Read more ...           |
|                                   |                                   |
|                                   | After email from patient          |
|                                   | received, the specialist:         |
|                                   |                                   |
|                                   | -   completes a Bulk Bill         |
|                                   |     Assignment form, noting       |
|                                   |     'email agreement' (for        |
|                                   |     manually lodged claims only)  |
|                                   |     in the signature block and    |
|                                   |     submits it to Services        |
|                                   |     Australia as acknowledgement  |
|                                   |     that this process has been    |
|                                   |     completed                     |
|                                   | -   retains the email containing  |
|                                   |     the patient's consent and     |
|                                   |     email signature in hard copy  |
|                                   |     or electronic form for 2      |
|                                   |     years                         |
|                                   | -   provides a completed copy of  |
|                                   |     the Bulk Bill Assignment form |
|                                   |     to the patient                |
|                                   |                                   |
|                                   | Exception: for electronically     |
|                                   | lodged claims, the email          |
|                                   | signature does not need to be     |
|                                   | noted in the transmission but the |
|                                   | patient's email consent should be |
|                                   | kept on file.                     |
+-----------------------------------+-----------------------------------+

Assignment of benefit via email

Note: in order to meet the requirement of a patient's signature for a
Telehealth consultation under section 10 of the Electronic Transactions
Act 1999, the process below must be completed.

Process for assignment of benefit via email

Table 2: This table describes the process for a patient to assign their
right to a Medicare benefit to a specialist via email.

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Informs patient of actions        |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | The specialist will advise the    |
|                                   | patient that in order to bulk     |
|                                   | bill Medicare for the service     |
|                                   | they need to:                     |
|                                   |                                   |
|                                   | -   seek verbal agreement from    |
|                                   |     the patient to assign the     |
|                                   |     benefit                       |
|                                   | -   advise that an email will be  |
|                                   |     sent to the patient's         |
|                                   |     nominated email address       |
|                                   |     confirming the details of the |
|                                   |     service and agreement to      |
|                                   |     assign the benefit, which the |
|                                   |     patient must respond to       |
+-----------------------------------+-----------------------------------+
| 2                                 | Emails patient + Read more ...    |
|                                   |                                   |
|                                   | Prior to lodging the claim, the   |
|                                   | specialist will email the         |
|                                   | patient:                          |
|                                   |                                   |
|                                   | List the following details of the |
|                                   | service:                          |
|                                   |                                   |
|                                   | -   item numbers or description   |
|                                   |     of services                   |
|                                   | -   benefit amount for both the   |
|                                   |     base and derived fee items    |
|                                   | -   date and time of service      |
|                                   | -   patient's name (Medicare card |
|                                   |     number must not be included)  |
|                                   | -   practitioner's name (provider |
|                                   |     number must not be included)  |
|                                   |                                   |
|                                   | Include the following statement:  |
|                                   | "If you (the patient) agree to    |
|                                   | the assignment of the Medicare    |
|                                   | benefit directly to the           |
|                                   | specialist (bulk bill) reply to   |
|                                   | this email including:             |
|                                   |                                   |
|                                   | -   the word 'YES' in the body of |
|                                   |     the email                     |
|                                   | -   your (the patient's) name     |
|                                   |                                   |
|                                   | A privacy and personal            |
|                                   | information note                  |
|                                   |                                   |
|                                   | The Resources page includes a     |
|                                   | sample of the privacy and         |
|                                   | personal information note.        |
|                                   |                                   |
|                                   | Note: due to privacy reasons, the |
|                                   | Medicare card number and the      |
|                                   | provider number must not be       |
|                                   | included in the email.            |
+-----------------------------------+-----------------------------------+
| 3                                 | Sends reply email to the          |
|                                   | specialist + Read more ...        |
|                                   |                                   |
|                                   | The patient sends reply email to  |
|                                   | the specialist agreeing to the    |
|                                   | Assignment of benefit.            |
+-----------------------------------+-----------------------------------+
| 4                                 | Retains email, completes form,    |
|                                   | submits claim and provides copy   |
|                                   | of form to patient + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The specialist must:              |
|                                   |                                   |
|                                   | -   Retain the email containing   |
|                                   |     the patient's consent and     |
|                                   |     email signature (in hard copy |
|                                   |     or electronic form) for audit |
|                                   |     purposes for a period of 2    |
|                                   |     years from date of service    |
|                                   | -   Complete the Assignment of    |
|                                   |     benefit form notating in the  |
|                                   |     signature block ' Email       |
|                                   |     agreement'. Note: this        |
|                                   |     notation is only required for |
|                                   |     manual bulk bill claims. For  |
|                                   |     electronically lodged claims, |
|                                   |     the email signature does not  |
|                                   |     need to be noted in the       |
|                                   |     transmission however,         |
|                                   |     providers must keep the       |
|                                   |     patient's email consent on    |
|                                   |     file. Submit the claim to     |
|                                   |     Services Australia in         |
|                                   |     accordance with the Medicare  |
|                                   |     Benefit Schedule (MBS)        |
|                                   | -   Submit the claim to Services  |
|                                   |     Australia in accordance with  |
|                                   |     the MBS                       |
|                                   | -   Provide a completed copy of   |
|                                   |     the Assignment of benefit     |
|                                   |     form to the patient           |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Where the provider completes  |
|                                   |     the Assignment of benefit     |
|                                   |     form by notating 'Email       |
|                                   |     agreement', they are          |
|                                   |     acknowledging that they have  |
|                                   |     completed the related         |
|                                   |     requirements as detailed in   |
|                                   |     this process                  |
|                                   | -   There is no requirement for   |
|                                   |     the email to be sent to       |
|                                   |     Medicare unless requested     |
+-----------------------------------+-----------------------------------+
