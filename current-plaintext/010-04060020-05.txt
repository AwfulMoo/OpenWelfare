Approved free text - Public versus private transport

Table 6: This table provides text for use in an Online Advice (OLA) when
requesting evidence for external students undertaking a compulsory
component of their course at their institution. Note: each free text
paragraph can have 9 lines, has a limit of 74 characters per line and
does not line wrap.

+-----------------------------------+-----------------------------------+
| Paragraph                         | Text                              |
+-----------------------------------+-----------------------------------+
| 1                                 | You have travelled via private    |
|                                   | transport.                        |
|                                   |                                   |
|                                   | As per SSA 1991 - 1061ZAAK, if a  |
|                                   | person chooses to travel by       |
|                                   |                                   |
|                                   | private vehicle when public       |
|                                   | transport is available, they can  |
|                                   | only                              |
|                                   |                                   |
|                                   | be reimbursed the cost of the     |
|                                   | least expensive public transport  |
|                                   |                                   |
|                                   | fare, using a taxi or private car |
|                                   | which is paid based on a per km   |
|                                   |                                   |
|                                   | rate. This type of vehicle will   |
|                                   | only be reimbursed if there is no |
|                                   |                                   |
|                                   | public transport available        |
+-----------------------------------+-----------------------------------+
| 2                                 | Fares Allowance is primarily      |
|                                   | about providing reimbursement     |
|                                   |                                   |
|                                   | for the travel of the student,    |
|                                   | not their belongings. Therefore   |
|                                   |                                   |
|                                   | you are being reimbursed the      |
|                                   | cheapest form of practicable      |
|                                   |                                   |
|                                   | public transport.                 |

Approved free text - Airfare reimbursement - reasonable fare

Table 7: This table provides text for use in an Online Advice (OLA) when
requesting evidence for external students undertaking a compulsory
component of their course at their institution. Note: each free text
paragraph can have 9 lines and has a limit of 74 characters per line and
does not line wrap.

+-----------------------------------+-----------------------------------+
| Paragraph                         | Text                              |
+-----------------------------------+-----------------------------------+
| 1                                 | As per SSA 1991, 1061ZAAJ, the    |
|                                   | amount of fares allowance         |
|                                   |                                   |
|                                   | reimbursed is the cost of making  |
|                                   | the journey using a               |
|                                   |                                   |
|                                   | reasonable route and the least    |
|                                   | expensive form of public          |
|                                   |                                   |
|                                   | transport. You have booked your   |
|                                   | flights one day prior to          |
|                                   |                                   |
|                                   | departure, this is not deemed     |
|                                   | reasonable. Therefore the         |
|                                   |                                   |
|                                   | reimbursement has been calculated |
|                                   | using the average of the          |
|                                   |                                   |
|                                   | lowest economy flight, including  |
|                                   | one piece of luggage, over 5      |
|                                   |                                   |
|                                   | days. This is the amount          |
|                                   | reimbursed.                       |

Examples

Examples of assessing Fares Allowance (FAA) eligibility

Table 11: This table contains examples on how to assess eligibility for
FAA.

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| Online student                    | Aaron is receiving Austudy and is |
|                                   | an online student at the          |
|                                   | University of Queensland in       |
|                                   | Brisbane.                         |
|                                   |                                   |
|                                   | He lives with his partner in      |
|                                   | Melbourne. However, he has to     |
|                                   | attend the University in Brisbane |
|                                   | for a compulsory residential      |
|                                   | unit.                             |
|                                   |                                   |
|                                   | Aaron should submit a claim for   |
|                                   | Fares Allowance, with evidence of |
|                                   | the residential component         |
|                                   | requirement. Aaron can either     |
|                                   | submit a claim asking Services    |
|                                   | Australia to pre-book his travel  |
|                                   | arrangements or he can submit a   |
|                                   | claim to have his travel expenses |
|                                   | reimbursed after he has           |
|                                   | travelled.                        |
+-----------------------------------+-----------------------------------+
| Dependent student using private   | Maikeli is about to move from     |
| transport due to excess luggage   | Dubbo to Sydney to commence the   |
|                                   | first year of university. Maikeli |
|                                   | needs to take 3 large suitcases,  |
|                                   | bedding, kitchenware and study    |
|                                   | materials. There is a bus route   |
|                                   | between Dubbo and Sydney, however |
|                                   | the bus company restricts luggage |
|                                   | to one large suitcase per         |
|                                   | passenger.                        |
|                                   |                                   |
|                                   | It is not practicable for Maikeli |
|                                   | to travel via bus on this         |
|                                   | occasion, and Maikeli will be     |
|                                   | entitled to the private vehicle   |
|                                   | rate of FAA if he decides to      |
|                                   | travel from Dubbo to Sydney using |
|                                   | his car.                          |
+-----------------------------------+-----------------------------------+
| Dependent student, living away    | Sonya is studying at Charles      |
| from home                         | Sturt University in Bathurst,     |
|                                   | NSW. She is receiving Youth       |
|                                   | Allowance as a dependent student  |
|                                   | at the away from home rate        |
|                                   | because her parent's home is in   |
|                                   | Townsville, Queensland.           |
|                                   |                                   |
|                                   | Sonya paid for an economy flight  |
|                                   | home to see her parents at the    |
|                                   | end of the study year and had to  |
|                                   | take a bus to the airport.        |
|                                   |                                   |
|                                   | Sonya should submit a claim for   |
|                                   | Fares Allowance for both the cost |
|                                   | of the flight and the bus ticket. |
|                                   | If Sonya chose to travel by       |
|                                   | private vehicle instead of the    |
|                                   | bus, she would only be paid the   |
|                                   | amount of a bus fare as this is   |
|                                   | the cheapest form of public       |
|                                   | transport available.              |
+-----------------------------------+-----------------------------------+
| Combined pre-booked flights and   | Elizabeth is receiving Youth      |
| reimbursement of ancillary costs  | Allowance as a dependent student  |
| after journey made                | and lives in Sydney to attend     |
|                                   | university. Her permanent home is |
|                                   | in Perth.                         |
|                                   |                                   |
|                                   | Elizabeth wants to travel home to |
|                                   | see her parents at the            |
|                                   | end-of-year break, returning to   |
|                                   | university in the new year.       |
|                                   | Elizabeth can ask Services        |
|                                   | Australia to book and pay for the |
|                                   | most practical mode of transport  |
|                                   | to travel, for example, return    |
|                                   | flights from Sydney to Perth.     |
|                                   |                                   |
|                                   | Elizabeth catches the local bus   |
|                                   | to and from the airport, which    |
|                                   | cannot be pre-booked. She will    |
|                                   | need to submit her bus travel     |
|                                   | expenses for reimbursement after  |
|                                   | she has travelled.                |
+-----------------------------------+-----------------------------------+
| 1st year Uni student              | Kim has completed year 12 in 2021 |
|                                   | and will commence a 3 year        |
|                                   | bachelor of education at Gold     |
|                                   | Coast University in 2022.         |
|                                   |                                   |
|                                   | Kim needs to travel from her home |
|                                   | in Victoria to the Gold Coast to  |
|                                   | commence studies in February      |
|                                   | 2022.                             |
|                                   |                                   |
|                                   | Kim's Youth Allowance is granted  |
|                                   | from 1 January 2022 as she is     |
|                                   | deemed a continuing student due   |
|                                   | to completing year 12 in 2021.    |
|                                   |                                   |
|                                   | Kim is eligible for travel from   |
|                                   | her family home to the university |
|                                   | at the start of the study year    |
|                                   | and return back to her family     |
|                                   | home at the end of the study      |
|                                   | year.                             |
|                                   |                                   |
|                                   | Kim will also qualify for a mid   |
|                                   | year return trip back to the      |
|                                   | family home once she has been     |
|                                   | receipt of Youth Allowance for 3  |
|                                   | months.                           |
+-----------------------------------+-----------------------------------+
| Mid year semester start           | Jodie is granted Youth Allowance  |
|                                   | from 17 July 2022 as she has been |
|                                   | accepted in the midyear intake at |
|                                   | Uni to commence her nursing       |
|                                   | degree.                           |
|                                   |                                   |
|                                   | Jodie is entitled to the          |
|                                   | following travel while studying:  |
|                                   |                                   |
|                                   | In the first year of study        |
|                                   | (2022):                           |
|                                   |                                   |
|                                   | -   travel from the family home   |
|                                   |     to the Uni to commence        |
|                                   |     studies and return at the end |
|                                   |     of the study year             |
|                                   | -   one mid year return trip to   |
|                                   |     the family home once she has  |
|                                   |     been in receipt of Youth      |
|                                   |     Allowance for 3 months        |
|                                   |                                   |
|                                   | subsequent years of study:        |
|                                   |                                   |
|                                   | -   travel from the family home   |
|                                   |     to the Uni to commence        |
|                                   |     studies and return to the     |
|                                   |     family home at the end of the |
|                                   |     study year                    |
|                                   | -   one mid year travel from the  |
|                                   |     Uni to the family home and    |
|                                   |     return to the Uni once she    |
|                                   |     has been receiving Youth      |
|                                   |     Allowance for 3 months. As    |
|                                   |     Jodie is a continuing         |
|                                   |     student, the mid year trip    |
|                                   |     can be completed on or after  |
|                                   |     1 April 2023 as she has been  |
|                                   |     receiving Youth Allowance for |
|                                   |     3 months from 1 January       |
