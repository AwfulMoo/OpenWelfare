|                                   | CSO.DIRECT business role          |
|                                   | beforehand.                       |

|                                   | ZC.XX.REST.ACC.OFF.L1 within      |

Changing the RA Manager within a service centre

Changing the RA Manager within the same service centre

Table 2: this table describes the process to replace a current
Restricted Access Manager with another Restricted Access Manager in the
same service centre, where a customer is managed by their local service
centre.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access the customer's record +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In the customer's record in       |
|                                   | Customer First, navigate to the   |
|                                   | Sensitive Customer Information    |
|                                   | assignment block in Workspace.    |
|                                   |                                   |
|                                   | Select the Restricted Access      |
|                                   | Application menu item.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Locate the current Restricted     |
|                                   | Access Manager in the record +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | On the Restricted Access          |
|                                   | Application screen, select Edit   |
|                                   | to enable edit mode.              |
|                                   |                                   |
|                                   | Select Employee Responsible Open  |
|                                   | Input Help icon.                  |
|                                   |                                   |
|                                   | Search for the required employee  |
|                                   | by Last Name and First Name, or   |
|                                   | User Logon ID and select the      |
|                                   | employee from Search: Employees   |
|                                   | screen                            |
|                                   |                                   |
|                                   | Select Save to save the changes   |
|                                   | prior to navigating away from the |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Select Home to navigate to the    |
|                                   | Workspace Home Page               |
+-----------------------------------+-----------------------------------+
| 3                                 | Remove the current Restricted     |
|                                   | Access Manager from the           |
|                                   | customer's record + Read more ... |
|                                   |                                   |
|                                   | Select the CRN/BP link from the   |
|                                   | header in the Centrelink system,  |
|                                   | to navigate to the Customer       |
|                                   | Details screen                    |
|                                   |                                   |
|                                   | Select the Relationships tab      |
|                                   |                                   |
|                                   | On the Relationships tab, select  |
|                                   | the delete Action icon (trash     |
|                                   | can) of Has RA Manager            |
|                                   | relationship to remove the        |
|                                   | current Restricted Access Manager |
+-----------------------------------+-----------------------------------+
| 4                                 | Add the new Restricted Access     |
|                                   | Manager to the customer's         |
|                                   | record + Read more ...            |
|                                   |                                   |
|                                   | Select New on the Relationships   |
|                                   | tab                               |
|                                   |                                   |
|                                   | Select Relationship Has RA        |
|                                   | Manager from drop down menu       |
|                                   |                                   |
|                                   | Select Partner Open Input Help    |
|                                   | icon                              |
|                                   |                                   |
|                                   | Search for the required employee  |
|                                   | by Last Name and First Name, or   |
|                                   | User Logon ID and select an       |
|                                   | employee from Search: Employees   |
|                                   | screen                            |
|                                   |                                   |
|                                   | Select Done                       |
|                                   |                                   |
|                                   | Select Save to save the changes   |
|                                   | before navigating away from the   |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Finalise record + Read more ...   |
|                                   |                                   |
|                                   | Clear out of the customer’s       |
|                                   | record and wait a couple of       |
|                                   | minutes before accessing the      |
|                                   | customer’s record again.          |
|                                   |                                   |
|                                   | This is required to ensure a      |
|                                   | record locking conflict does not  |
|                                   | prevent the Centrelink system     |
|                                   | record from being updated.        |

Requesting temporary access to a Restricted Access customer's record

Non-Restricted Access staff requires temporary access to a Restricted Access customer's record

Table 3: this table describes the process for non-Restricted Access
staff to request temporary access to a Restricted Access customer's
record.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Staff require temporary access to |
|                                   | a Restricted Access customer's    |
|                                   | record + Read more ...            |
|                                   |                                   |
|                                   | Only specialist staff may be      |
|                                   | granted access to a Restricted    |
|                                   | Access customer's record.         |
|                                   | Specialist staff include:         |
|                                   |                                   |
|                                   | -   Authorised Review Officers    |
|                                   |     (ARO)                         |
|                                   | -   Compensation Service Officers |
|                                   | -   Debt Recovery Service         |
|                                   |     Officers                      |
|                                   | -   Business Integrity Service    |
|                                   |     Officers                      |
|                                   | -   Claims staff                  |
|                                   |                                   |
|                                   | Does the staff member have a      |
|                                   | specialist role?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, contact the National      |
|                                   |     Restricted Access Team (NRAT) |
|                                   |     or Restricted Access Officers |
|                                   |     listed on the Security Data   |
|                                   |     Warning (SDW) screen of the   |
|                                   |     customer's record. These      |
|                                   |     officers will be able to      |
|                                   |     process the particular        |
|                                   |     enquiry. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 2                                 | Specialist staff require          |
|                                   | temporary access + Read more ...  |
|                                   |                                   |
|                                   | Staff requiring access must be at |
|                                   | the APS3 level or above and have  |
|                                   | certain required Security Roles   |
|                                   | and Skill Tags. Note: temporary   |
|                                   | access may not be granted to      |
|                                   | Irregular and Intermittent (IIE)  |
|                                   | and Contractor staff for          |
|                                   | Restricted Access records. Email  |
|                                   | national.restricted.access for    |
|                                   | further instructions.             |
|                                   |                                   |
|                                   | If the staff member does not have |
|                                   | these resources, refer to the     |
|                                   | National Restricted Access page   |
|                                   | for information on what roles and |
|                                   | resources are required.           |
|                                   |                                   |
|                                   | Is temporary access to the        |
|                                   | Restricted Access customer's      |
|                                   | record still required?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, contact the Restricted    |
|                                   |     Access Officer(s) listed on   |
|                                   |     the Security Data Warning     |
|                                   |     (SDW) screen of the           |
|                                   |     customer's record and ask for |
|                                   |     assistance. Procedure ends    |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Contact the National Restricted   |
|                                   | Access Team (NRAT) or relevant    |
|                                   | Restricted Access Manager + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the Security Data Warning   |
|                                   | (SDW) screen of the customer's    |
|                                   | record.                           |
|                                   |                                   |
|                                   | If the Restricted Access customer |
|                                   | is managed by their local service |
|                                   | centre, contact the Restricted    |
|                                   | Access Manager listed on the SDW  |
|                                   | screen for temporary access. If   |
|                                   | the Restricted Access customer is |
|                                   | managed by NRAT, email            |
|                                   | national.restricted.access to     |
|                                   | request temporary access. Go to   |
|                                   | Step 4.                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Cases of extreme emergency + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If temporary access is related to |
|                                   | an extreme emergency, such as a   |
|                                   | processing error or urgent        |
|                                   | payment, contact the Centrelink   |
|                                   | Restricted Access phone number    |
|                                   | for assistance.                   |

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Restricted Access customer        |
|                                   | changes address + Read more ...   |
|                                   |                                   |
|                                   | When a Restricted Access customer |
|                                   | moves to another address that is  |
|                                   | not within the current service    |
|                                   | centre boundary and the customer  |

|                                   | centre, the losing Restricted     |
|                                   | Access Manager must firstly       |
|                                   | action the record.                |
|                                   |                                   |
|                                   | Is the customer moving            |
|                                   | temporarily?                      |
|                                   |                                   |
|                                   | -   Yes, advise the customer that |
|                                   |     their current Restricted      |
|                                   |     Access arrangements will      |
|                                   |     remain in place. Update the   |
|                                   |     customer's address details.   |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer moving permanently +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the customer moving            |
|                                   | interstate?                       |
|                                   |                                   |
|                                   | -   Yes, an interstate            |
|                                   |     environment transfer is       |
|                                   |     required first. See           |
|                                   |     Inter-environment change of   |
|                                   |     address (ICoA) transfer of a  |
|                                   |     customer record. Once the     |
|                                   |     transfer has been actioned,   |
|                                   |     return to this procedure. Go  |
|                                   |     to Step 3                     |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Contact the new Service Centre    |
|                                   | Manager and update the customer's |
|                                   | record + Read more ...            |
|                                   |                                   |

|                                   | centre responsible for the        |
|                                   | customer's new location by phone. |
|                                   |                                   |
|                                   | Complete the steps below while on |
|                                   | the phone with the gaining        |
|                                   | Service Centre Manager.           |
|                                   |                                   |
|                                   | In the customer's record in       |
|                                   | Customer First, navigate to the   |
|                                   | Sensitive Customer Information    |
|                                   | assignment block in Workspace.    |
|                                   |                                   |
|                                   | Select Restricted Access          |
|                                   | Application menu item.            |
|                                   |                                   |
|                                   | Select the current Restricted     |
|                                   | Access Application.               |
|                                   |                                   |
|                                   | Select Edit.                      |
|                                   |                                   |
|                                   | Select the Status field dropdown  |
|                                   | box and select RA Removed then    |
|                                   | select Save.                      |
|                                   |                                   |
|                                   | Clear out of the customer’s       |
|                                   | record and wait a couple of       |
|                                   | minutes before accessing the      |
|                                   | customer’s record again, to       |
|                                   | ensure a record locking conflict  |
|                                   | does not prevent the Centrelink   |
|                                   | system record from being updated. |
|                                   |                                   |
|                                   | Select Copy to create a new       |
|                                   | Restricted Access Application.    |
|                                   |                                   |
|                                   | Select Service Centre and select  |

|                                   |                                   |

|                                   | Once the gaining Service Centre   |
|                                   | Manager has completed the steps,  |
|                                   | go to Step 8.                     |
+-----------------------------------+-----------------------------------+

|                                   | action + Read more ...            |
|                                   |                                   |
|                                   | Remain on the phone with the      |
|                                   | losing Restricted Access Manager  |
