|                                   | Absences Details (ABSN) screen.   |
|                                   | To do so, continue with this      |
|                                   | procedure.                        |
|                                   |                                   |
|                                   | The absence may potentially be a  |
|                                   | Regular absence or a One-Off      |
|                                   | absence.                          |
|                                   |                                   |
|                                   | -   If Regular, go to Step 2      |
|                                   | -   If One-Off, go to Step 3      |
+-----------------------------------+-----------------------------------+
| 2                                 | Code the Regular absence          |
|                                   | manually: + Read more ...         |
|                                   |                                   |
|                                   | 'S'elect the care receiver from   |
|                                   | the Care Receiver Summary (CRS)   |
|                                   | screen. Press [Enter]             |
|                                   |                                   |
|                                   | On the Care Receiver Task         |
|                                   | Selector (CETS) screen, 's'elect: |
|                                   |                                   |
|                                   | -   Absence Enquiry for the       |
|                                   |     Absences Enquiry (ABCE)       |
|                                   |     screen                        |
|                                   | -   Absence Detail for the        |
|                                   |     Absences Details (ABSN)       |
|                                   |     screen                        |
|                                   | -   Assessment Results (Carer     |
|                                   |     Allowance only) for the Care  |
|                                   |     Receiver Benefit Action       |
|                                   |     (CJBA) screen                 |
|                                   |                                   |
|                                   | At the ABCE screen, code 'Y' to   |
|                                   | the question 'go to Absences      |
|                                   | Details Screen?' Press [Enter]    |
|                                   |                                   |
|                                   | Code the following fields:        |
|                                   |                                   |
|                                   | -   Start Date field. This is the |
|                                   |     date of the first night of    |
|                                   |     the absence                   |
|                                   | -   End Date field can be coded   |
|                                   |     if it is known. Regular       |
|                                   |     Absences do not require an    |
|                                   |     end date to be coded for the  |
|                                   |     activity to accept            |
|                                   | -   Reason field. The reason must |
|                                   |     be coded as 'HSP'             |
|                                   |     (Hospitalisation)             |
|                                   | -   Regular Nights PFN field.     |
|                                   |     This is the number of days    |
|                                   |     the care receiver is          |
|                                   |     regularly absent each         |
|                                   |     fortnight                     |
|                                   |                                   |
|                                   | As an absence for Hospitalisation |
|                                   | cannot be extended, the Absence   |
|                                   | Extended field can be left blank. |
|                                   | It will default to 'N'.           |
|                                   |                                   |
|                                   | Press [Enter]. The ABSN screen    |
|                                   | will redisplay with the absence   |
|                                   | accepted and a new input line.    |
|                                   |                                   |
|                                   | -   Code 'AR' in the Nxt: field   |
|                                   |     to go the Assessment Results  |
|                                   |     (AR) screen - do not finalise |
|                                   |     activity                      |
|                                   | -   Go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Code the One-Off absence          |
|                                   | manually: + Read more ...         |
|                                   |                                   |
|                                   | 'S'elect the care receiver from   |
|                                   | the CRS screen. Press [Enter].    |
|                                   |                                   |
|                                   | On the Care Receiver Task         |
|                                   | Selector (CETS) screen, 'S'elect  |
|                                   |                                   |
|                                   | -   Absence Enquiry for the       |
|                                   |     Absences Enquiry (ABCE)       |
|                                   |     screen                        |
|                                   | -   Absence Detail for the        |
|                                   |     Absences Details (ABSN)       |
|                                   |     screen                        |
|                                   | -   Assessment (Carer Allowance   |
|                                   |     only) for the Assessment      |
|                                   |     Results (AR) screen           |
|                                   |                                   |
|                                   | At the ABCE screen, code 'Y' to   |
|                                   | the question 'Go to Absences      |
|                                   | Details Screen?' Press [Enter]    |
|                                   |                                   |
|                                   | The following must be coded:      |
|                                   |                                   |
|                                   | -   Start Date field. This is the |
|                                   |     first full day (midnight to   |
|                                   |     midnight) that the care       |
|                                   |     receiver has been in hospital |
|                                   | -   End Date field. This is the   |
|                                   |     date of the last full day     |
|                                   |     (midnight to midnight) that   |
|                                   |     the care receiver is away     |
|                                   |     from home. For example, if    |
|                                   |     the care receiver returned    |
|                                   |     home on 29 November, code 28  |
|                                   |     November as this was the last |
|                                   |     evening the care receiver was |
|                                   |     absent. This will enable the  |
|                                   |     system to correctly calculate |
|                                   |     the absence                   |
|                                   | -   The right (that is,           |
|                                   |     unadjusted) dates must be     |
|                                   |     listed in the DOC attached to |
|                                   |     the activity in the carer's   |
|                                   |     record                        |
|                                   | -   An end date must be coded for |
|                                   |     a One-Off absence             |
|                                   | -   If the carer does not provide |
|                                   |     an end date, they must be     |
|                                   |     contacted to provide an       |
|                                   |     estimate. Assure the carer    |
|                                   |     that the situation will be    |
|                                   |     reviewed as the estimated end |
|                                   |     date draws near. This will be |
|                                   |     via a Manual Follow up (MFU). |
|                                   |     See What to do with an        |
|                                   |     absence Manual Follow-up      |
|                                   |     (MFU)                         |
|                                   | -   Reason field. The reason must |
|                                   |     be coded as 'HSP'             |
|                                   |     (Hospitalisation)             |
|                                   |                                   |
|                                   | As an absence for Hospitalisation |
|                                   | cannot be extended, the Absence   |
|                                   | Extended field can be left blank. |
|                                   | It will default to 'N'            |
|                                   |                                   |
|                                   | Press [Enter]. The ABSN screen    |
|                                   | will redisplay with the absence   |
|                                   | accepted and a new input line.    |
|                                   |                                   |
|                                   | -   Code 'AR' in the Nxt: field   |
|                                   |     to go to the Assessment       |
|                                   |     Results (AR) screen - do not  |
|                                   |     finalise activity             |
+-----------------------------------+-----------------------------------+

|                                   | ...                               |

|                                   | -   Go to the ABCE screen to      |

|                                   |                                   |
|                                   | Note: the ABCE screen will only   |
|                                   | show updated data after accessing |
|                                   | the AR screen.                    |

|                                   | record all details on a DOC in    |
