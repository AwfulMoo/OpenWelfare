FAQs

DVA podiatrist service common enquiries

This table contains Frequently Asked Questions (FAQs) from podiatry
service providers.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: The provider hasn’t     |
|                                   | notated a condition on the claim  |
|                                   | for a white card holder, can this |
|                                   | be accepted?                      |
|                                   |                                   |
|                                   | Answer: There is no requirement   |
|                                   | for the providers to add the      |
|                                   | condition. When processing in the |
|                                   | pend screen [PF10] will display   |
|                                   | the veteran's conditions, one of  |
|                                   | which must match the accepted     |
|                                   | podiatry conditions.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: There has been an item  |
|                                   | F998 claimed on a manual voucher  |
|                                   | which hasn’t been signed, can I   |
|                                   | accept this?                      |
|                                   |                                   |
|                                   | Answer: The item can be keyed and |
|                                   | paid as normal. There is no       |
|                                   | requirement for the veteran to    |
|                                   | sign the voucher for a postage    |
|                                   | item.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Can a veteran have an   |
|                                   | initial footwear assessment and a |
|                                   | subsequent podiatry consultation  |
|                                   | on the same day?                  |
|                                   |                                   |
|                                   | Answer: Yes, the patient can be   |
|                                   | paid for both, however if both    |
|                                   | items are initial consults then   |
|                                   | only one service is payable.      |
+-----------------------------------+-----------------------------------+
