|                                   | select the Right Slider icon.     |
|                                   |                                   |
|                                   | The Task Selection menu will      |
|                                   | display.                          |
|                                   |                                   |
|                                   | To access the Customer            |

|                                   |     Select the appropriate option |
|                                   |     from the drop down menu. This |

|                                   |     radio button next to the      |
|                                   |     customer correspondence –     |
|                                   |     manual letters task. If the   |
|                                   |     incorrect task is selected in |
|                                   |     error, select the Cancel icon |

|                                   |     field, or                     |

|                                   |     all icon to close all         |

|                                   | -   the individual section or     |
|                                   |     select the Form menu icon and |
|                                   |     select the section to view    |

|                                   | activities:                       |
|                                   |                                   |

|                                   | To search on different criteria   |
|                                   | use the:                          |

Update letter

Table 3:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Select letter + Read more ...     |
|                                   |                                   |
|                                   | Locate the letter for updating.   |
|                                   | Select the radio button next to   |
|                                   | the letter being updated in the   |
|                                   | Print Preview column.             |
|                                   |                                   |
|                                   | Select Open link next to the      |
|                                   | Letter View heading to view       |
|                                   | letter. This will open a new      |
|                                   | window to view the letter         |
|                                   | details.                          |
|                                   |                                   |
|                                   | To update the:                    |
|                                   |                                   |
|                                   | -   letter status, go to Step 2   |
|                                   | -   letter content, go to Step 3  |
+-----------------------------------+-----------------------------------+
| 2                                 | Update letter status + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select the Input drop down        |
|                                   | heading to display the letter     |
|                                   | details.                          |
|                                   |                                   |
|                                   | Confirm the letter details are    |
|                                   | correct.                          |
|                                   |                                   |
|                                   | Use the Letter Status drop down   |
|                                   | menu to select the appropriate    |
|                                   | status to update the letter.      |
|                                   |                                   |
|                                   | Select Submit. A notification     |
|                                   | Form Submitted Successfully will  |
|                                   | pop up.                           |
|                                   |                                   |
|                                   | If a mandatory field is not       |
|                                   | completed, an error message will  |
|                                   | display after selecting submit.   |
|                                   | Select the error message to       |
|                                   | navigate to the relevant field.   |
|                                   |                                   |
|                                   | After submitting form             |
|                                   | successfully select Assess.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Update letter content + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Some letters without the status   |
|                                   | of Printed will allow for the     |
|                                   | content to be updated.            |
|                                   |                                   |
|                                   | To update the letter content,     |
|                                   | select the letter name heading.   |
|                                   |                                   |
|                                   | Fields that cannot be updated are |
|                                   | disabled. Update any relevant     |
|                                   | fields that require updates using |
|                                   | the relevant radio buttons, drop  |
|                                   | down menus and free text fields.  |
|                                   |                                   |
|                                   | Select Submit. A notification     |
|                                   | Form Submitted Successfully will  |
|                                   | pop up.                           |
|                                   |                                   |
|                                   | If a mandatory field has not been |
|                                   | completed, an error message will  |
|                                   | display after selecting submit.   |
|                                   | Select the error message to       |
|                                   | navigate to the relevant field.   |
|                                   |                                   |
|                                   | After submitting form             |
|                                   | successfully, select Assess.      |
