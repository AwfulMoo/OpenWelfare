Amending POI documents on the Legacy POIA screen

Table 1: Face-to-face services only

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | A recorded POI document needs     |
|                                   | amending + Read more ...          |
|                                   |                                   |
|                                   | Go to the legacy POI Amendment    |
|                                   | (POIA) screen.                    |
|                                   |                                   |
|                                   | Update access is only possible on |
|                                   | a current customer's record when  |
|                                   | the POI indicator is Y.           |
|                                   |                                   |
|                                   | Access to the Legacy POIA screen  |
|                                   | is restricted in each site.       |
|                                   | Service Officers need to contact  |
|                                   | their Identity Subject Matter     |
|                                   | Expert with details of documents  |
|                                   | that need to be amended, inserted |
|                                   | or deleted.                       |
|                                   |                                   |
|                                   | Only update a document when the   |
|                                   | previous coding is clearly        |
|                                   | incorrect. If the document        |
|                                   | details have changed over of      |
|                                   | time, do not update details. For  |
|                                   | example, there is now a different |
|                                   | expiry date on a passport or      |
|                                   | licence. This includes when a     |
|                                   | document is provided as part of   |
|                                   | the Legacy Abridged POI process.  |
|                                   |                                   |
|                                   | If the POI indicator is E, the    |
|                                   | Legacy POIA screen is protected.  |
|                                   | A message will say that updates   |
|                                   | are not allowed. Any updates must |
|                                   | be made via the Legacy POI        |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Legacy POIA screen displays +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Documents will display as         |
|                                   | recorded on the Legacy POI        |
|                                   | screen.                           |
|                                   |                                   |
|                                   | The following fields are          |
|                                   | protected. They display as on the |
|                                   | Legacy Proof of Identity Summary  |
|                                   | (POIS) screen:                    |
|                                   |                                   |
|                                   | -   Tier Level Achieved           |
|                                   | -   POA/POB Supplied              |
|                                   | -   Total Points Achieved         |
|                                   |                                   |
|                                   | The Code: field is protected only |
|                                   | if the documents have been        |
|                                   | recorded on the Legacy POI        |
|                                   | screen. If the Code: field needs  |
|                                   | to be amended, go to Step 3.      |
|                                   |                                   |
|                                   | To insert, delete or correct POI, |
|                                   | go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | To change the Code: field + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the Code: field for a document |
|                                   | is incorrectly recorded and needs |
|                                   | to be amended:                    |
|                                   |                                   |
|                                   | -   delete the line, and          |
|                                   | -   insert a new line with the    |
|                                   |     correct document code         |
+-----------------------------------+-----------------------------------+
| 4                                 | Correct, insert or delete         |
|                                   | document details + Read more ...  |
|                                   |                                   |
|                                   | To avoid dropping a Legacy Tier   |
|                                   | Level, insert a new document      |
|                                   | before deleting one.              |
|                                   |                                   |
|                                   | However, if the customer has more |
|                                   | points than required for a Legacy |
|                                   | Tier Level, deleting will be      |
|                                   | allowed without inserting a new   |
|                                   | document.                         |
|                                   |                                   |
|                                   | If the legacy POIA screen is full |
|                                   | (all 11 lines have document       |
|                                   | details), delete one line before  |
|                                   | inserting a line.                 |
|                                   |                                   |
|                                   | -   To correct information, code  |
|                                   |     'C' in the Select: field.     |
|                                   |     Overstrike the existing       |
|                                   |     information with the correct  |
|                                   |     details. Not allowed on a     |
|                                   |     blank line                    |
|                                   | -   To insert new information,    |
|                                   |     code 'I' in the Select: field |
|                                   |     and record document details.  |
|                                   |     Not allowed on a recorded     |
|                                   |     document line                 |
|                                   | -   To delete information, code   |
|                                   |     'D' in the Select: field. Not |
|                                   |     allowed a blank line. No      |
|                                   |     other changes on that line    |
|                                   |     are allowed                   |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+

Recalculating the Legacy POI tier level

Table 2: Face-to-face services only

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Recalculate the Tier Level + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | After updates on the Legacy POIA  |
|                                   | screen, press [Enter] to          |
|                                   | recalculate the Legacy Tier       |
|                                   | Level.                            |
|                                   |                                   |
|                                   | If the outcome is:                |
|                                   |                                   |
|                                   | -   no change, a message displays |
|                                   |     that the updates were         |
|                                   |     completed successfully. The   |
|                                   |     legacy POIA screen will       |
|                                   |     display with updated details. |
|                                   |     Go to Step 3                  |
|                                   | -   increase, a message displays  |
|                                   |     the Legacy Tier Level         |
|                                   |     achieved. Go to Step 3        |
|                                   | -   decrease, a message displays  |
|                                   |     'Reduction of Tier Level not  |
|                                   |     allowed'. Go to Step 2        |
+-----------------------------------+-----------------------------------+
| 2                                 | Decrease in Legacy Tier Level +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A reduction in Legacy Tier level  |
|                                   | is not allowed.                   |
|                                   |                                   |
|                                   | A reduction may result from the   |
|                                   | change, e.g. due to incorrect     |
|                                   | coding of documents. If this      |
|                                   | occurs, refer the case back to    |
|                                   | the decision maker to obtain more |
|                                   | POI from the customer. Once       |
|                                   | provided, record the document     |
|                                   | details on the Legacy POIA screen |
|                                   | to meet the Legacy Tier Level     |
|                                   | requirement.                      |
|                                   |                                   |
|                                   | For more information, see Coding  |
|                                   | the legacy Proof of Identity      |
|                                   | (POI) screen                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalise activity + Read more ... |
|                                   |                                   |
|                                   | When action on the Legacy POIA    |
|                                   | screen is complete, finalise the  |
|                                   | activity on the Assessment        |
|                                   | Results (AR) screen.              |
|                                   |                                   |
|                                   | Record details on a DOC.          |
+-----------------------------------+-----------------------------------+
