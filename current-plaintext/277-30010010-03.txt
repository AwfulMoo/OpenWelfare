Delete an act of grace application

Table 4: This table describes how to delete an act of grace application.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - Delete an act of    |
|                                   | grace application + Read more ... |
|                                   |                                   |
|                                   | Start point: Compensation/Waiver  |
|                                   | window                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Delete the application + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Only delete an act of grace claim |
|                                   | if it has been recorded on the    |
|                                   | incorrect customer record.        |
|                                   |                                   |
|                                   | -   Highlight the relevant act of |
|                                   |     grace application in the      |
|                                   |     Compensation/Waiver List      |
|                                   |     group box                     |
|                                   | -   Select Delete from the File   |
|                                   |     menu                          |
|                                   |                                   |
|                                   | Note: an act of grace application |
|                                   | can only be deleted while it is   |
|                                   | active. The Delete option is not  |
|                                   | enabled if the act of grace       |
|                                   | application has been finalised or |
|                                   | withdrawn.                        |
|                                   |                                   |
|                                   | In the confirmation dialogue box: |
|                                   |                                   |
|                                   | -   click Yes to delete the act   |
|                                   |     of grace record, or           |
|                                   | -   click No to return to the     |
|                                   |     Compensation/Waiver window    |
|                                   |     without deleting the act of   |
|                                   |     grace record                  |

Table 5: This table describes how to finalise an act of grace

| Step                              | Action                            |

| 1                                 | Start point - Finalise an act of  |
|                                   | grace application + Read more ... |

|                                   | window                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Finalise the act of grace         |
|                                   | application + Read more ...       |

|                                   |     grace application in the      |

|                                   |     from the Status drop down     |

|                                   | Note: the list options will only  |
|                                   | be available if the status has    |
|                                   | been updated to REFERD TO DOFA.   |
|                                   |                                   |
|                                   | -   The Date field will           |
|                                   |     automatically populate with   |
|                                   |     today’s date. Delete and      |
|                                   |     enter the correct date if     |
|                                   |     necessary                     |

|                                   |     Department of Finance (DoF)   |
|                                   |     have approved to be paid in   |
