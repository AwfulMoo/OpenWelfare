| 3                                 | How urgent payment rules work for |
|                                   | income managed customers + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Explain to the customer:          |
|                                   |                                   |
|                                   | -   The minimum amount required   |
|                                   |     to satisfy the customer's     |
|                                   |     immediate hardship need is    |
|                                   |     payable as an urgent payment  |
|                                   | -   The urgent payment will be    |
|                                   |     income managed at the         |
|                                   |     percentage applicable to      |
|                                   |     their measure of Income       |
|                                   |     Management                    |
|                                   | -   The income managed portion of |
|                                   |     the urgent payment will be    |
|                                   |     credited to their Income      |
|                                   |     Management account and will   |
|                                   |     be available for immediate    |
|                                   |     use (on finalisation of the   |
|                                   |     urgent payment activity)      |
|                                   | -   The amount of the urgent      |
|                                   |     payment (total of income      |
|                                   |     managed and discretionary     |
|                                   |     funds) will be deducted from  |
|                                   |     their next payment            |
|                                   | -   As their next income support  |
|                                   |     payment will be reduced,      |
|                                   |     there may be insufficient     |
|                                   |     income managed funds          |
|                                   |     available to cover future     |
|                                   |     regular expenses. Service     |
|                                   |     Officers are to view the      |
|                                   |     Expense Management Summary    |
|                                   |     screen and view the           |
|                                   |     percentage of fortnightly     |
|                                   |     unallocated funds and         |
|                                   |     fortnight (14-days) balance   |
|                                   |     details to determine how the  |
|                                   |     urgent payment is likely to   |
|                                   |     impact future allocations     |
|                                   |                                   |
|                                   | The Service Officer should        |
|                                   | consider the above when           |
|                                   | determining the customer's        |
|                                   | request for an urgent payment.    |
|                                   |                                   |

|                                   | People at Risk measures + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Only 30% of the urgent payment    |

|                                   |                                   |

|                                   |                                   |

|                                   |     urgent payment if the         |
|                                   |     customer is in severe         |

|                                   |     Procedure ends here           |

+-----------------------------------+-----------------------------------+

|                                   | to determine if they are in       |
|                                   | severe financial hardship.        |
|                                   |                                   |
|                                   | Is the customer experiencing      |
|                                   | severe financial hardship?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for an urgent        |
|                                   |     payment. Deny the urgent      |
|                                   |     payment request. See          |
|                                   |     Processing urgent payment     |
|                                   |     requests                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Exceptional and unforeseen        |
|                                   | circumstances + Read more ...     |
|                                   |                                   |
|                                   | An urgent payment is not to be    |
|                                   | made for regular expenses where   |
|                                   | the circumstances are not         |
|                                   | exceptional and/or could be       |
|                                   | foreseen (for example, the        |
|                                   | customer has been evicted from    |
|                                   | their rental accommodation due to |
|                                   | not paying their rent) or the     |
|                                   | customer has solely contributed   |
|                                   | to the delay in payment.          |
|                                   |                                   |
|                                   | Is the customer's financial       |
|                                   | hardship due to exceptional and   |
|                                   | unforeseen circumstances?         |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, the customer is not       |
|                                   |     eligible for an urgent        |
|                                   |     payment. Deny the urgent      |
|                                   |     payment request. See          |
|                                   |     Processing urgent payment     |
|                                   |     requests                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Accrued entitlement meets         |
|                                   | immediate need + Read more ...    |
|                                   |                                   |
|                                   | The Immediate Payment requests    |
|                                   | script will calculate the         |
|                                   | available funds (the accrued      |
|                                   | entitlement).                     |
|                                   |                                   |
|                                   | An urgent payment is paid in      |
|                                   | arrears and is limited to the     |
|                                   | number of days between the 'date  |
|                                   | paid to' and the date on which    |
|                                   | the payment is being processed,   |
|                                   | less deductions for debt          |
|                                   | repayments, tax, Centrepay,       |
|                                   | immediate payment deduction etc.  |
|                                   | to ensure that enough funds are   |
|                                   | left to meet these commitments.   |
|                                   | The available amount, determined  |
|                                   | by the system, cannot be          |
|                                   | overridden.                       |
|                                   |                                   |
|                                   | If the customer's accrued         |
|                                   | entitlement does not meet their   |
|                                   | immediate need, the urgent        |
|                                   | payment request will not be       |
|                                   | approved.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Negotiate the amount to be paid + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The amount payable is the         |
|                                   | smallest of:                      |
|                                   |                                   |
|                                   | -   available funds calculated by |
|                                   |     the system                    |
|                                   | -   the amount the customer       |
|                                   |     requested                     |
|                                   | -   the minimum amount it will    |
|                                   |     cost to meet the customer's   |
|                                   |     immediate hardship need, or   |
|                                   | -   the maximum amount of $200    |
|                                   |                                   |
|                                   | Before offering an urgent payment |
|                                   | consider:                         |
|                                   |                                   |
|                                   | -   Is the amount being offered   |
|                                   |     the minimum required to meet  |
|                                   |     the customer's immediate      |
|                                   |     need?                         |
|                                   | -   If paid, how will the         |
|                                   |     customer meet their regular   |
|                                   |     ongoing expenses such as      |
|                                   |     rent, food, utilities on      |
|                                   |     their next payday? Reduce the |
|                                   |     amount offered or deny the    |
|                                   |     request if the customer is    |
|                                   |     not able to meet their        |
|                                   |     regular ongoing expenses      |
|                                   | -   Does the customer history     |
|                                   |     indicate they are struggling  |
|                                   |     to meet regular ongoing       |
|                                   |     expenses or appear to be in a |
|                                   |     crisis cycle? Consider a      |
|                                   |     referral to Financial         |
|                                   |     Wellbeing and Capability      |
|                                   |     Activities or weekly payment  |
|                                   |     assessment with an Authorised |
|                                   |     Weekly Payment Officer (AWPO) |
|                                   |                                   |
|                                   | Discuss with the customer and     |
|                                   | confirm they understand:          |
|                                   |                                   |
|                                   | -   any money the customer        |
|                                   |     receives for urgent payment   |
|                                   |     will reduce their next        |
|                                   |     fortnightly payment, and      |
|                                   | -   how they intend to meet their |
|                                   |     regular expenses with the     |
|                                   |     reduced funds received on     |
|                                   |     their regular payday          |
|                                   |                                   |
|                                   | While the system will allow as    |
|                                   | little as $1 to be paid, when     |
|                                   | determining the amount of the     |
|                                   | urgent payment consider:          |
|                                   |                                   |
|                                   | -   most stores require a minimum |
|                                   |     purchase of between $10 to    |
|                                   |     $15 to be made before EFTPOS  |
|                                   |     can be                        |
|                                   | -   ATMs issue a minimum of $20   |
|                                   | -   Electronic Benefits Transfer  |
|                                   |     (EBT) cards can only be       |
|                                   |     issued in denominations of    |
|                                   |     $20 or $50 as these are the   |
|                                   |     amounts available at ATMs     |
|                                   |                                   |
|                                   | Will receiving the urgent payment |
|                                   | extend or make the customer's     |
|                                   | financial hardship worse?         |
|                                   |                                   |
|                                   | -   Yes, the customer is not      |
|                                   |     eligible for an urgent        |
|                                   |     payment. Deny the payment.    |
|                                   |     See Processing urgent payment |
|                                   |     requests                      |
|                                   | -   No, see Processing urgent     |
|                                   |     payment requests              |
