Family member types for Medicare Safety Net

Table 1: this table lists the member types and validation rules for
Medicare Safety Net families.

+-----------------------+-----------------------+-----------------------+
| Type                  | Definition            | Validation rules      |
+-----------------------+-----------------------+-----------------------+
| FC                    | Family Contact        | -   Each family must  |
|                       |                       |     have a family     |
|                       |                       |     contact.          |
|                       |                       | -   Only 1 family     |
|                       |                       |     contact is        |
|                       |                       |     allowed per       |
|                       |                       |     family            |
|                       |                       |     registration      |
|                       |                       | -   The family        |
|                       |                       |     contact must be   |
|                       |                       |     present with      |
|                       |                       |     either spouse/de  |
|                       |                       |     facto partner,    |
|                       |                       |     student dependant |
|                       |                       |     or child          |
|                       |                       |     dependant         |
|                       |                       | -   The member        |
|                       |                       |     allocated as a    |
|                       |                       |     family contact    |
|                       |                       |     cannot belong to  |
|                       |                       |     any other family  |
|                       |                       |     registration      |
+-----------------------+-----------------------+-----------------------+
| SP                    | Spouse/de facto       | -   Optional member   |
|                       | partner               |     type              |
|                       |                       | -   Only 1 spouse/de  |
|                       |                       |     facto partner is  |
|                       |                       |     allowed per       |
|                       |                       |     family            |
|                       |                       |     registration      |
|                       |                       | -   The member        |
|                       |                       |     allocated as a    |
|                       |                       |     spouse/de facto   |
|                       |                       |     partner cannot    |
|                       |                       |     belong to any     |
|                       |                       |     other family      |
|                       |                       |     registration      |
+-----------------------+-----------------------+-----------------------+
| SD                    | Student Dependant     | Optional member type, |
|                       |                       | depending on presence |
|                       |                       | of other member types |
|                       |                       |                       |
|                       |                       | If there is no        |
|                       |                       | spouse/de facto       |
|                       |                       | partner or child      |
|                       |                       | dependant in the      |
|                       |                       | family, the family    |
|                       |                       | registration must     |
|                       |                       | contain a student     |
|                       |                       | dependant to be       |
|                       |                       | eligible              |
|                       |                       |                       |
|                       |                       | A family can have     |
|                       |                       | multiple student      |
|                       |                       | dependants            |
|                       |                       |                       |
|                       |                       | The member allocated  |
|                       |                       | as a student          |
|                       |                       | dependant must:       |
|                       |                       |                       |
|                       |                       | -   be a full time    |
|                       |                       |     student           |
|                       |                       | -   not belong to     |
|                       |                       |     more than 1 other |
|                       |                       |     family            |
|                       |                       |     registration      |
|                       |                       | -   be under 25 years |
|                       |                       |     of age            |
+-----------------------+-----------------------+-----------------------+
| CD                    | Child Dependant       | Optional member type, |
|                       |                       | depending on presence |
|                       |                       | of other member types |
|                       |                       |                       |
|                       |                       | If there is no        |
|                       |                       | spouse/de facto       |
|                       |                       | partner or student    |
|                       |                       | dependant in the      |
|                       |                       | family, the family    |
|                       |                       | registration must     |
|                       |                       | contain a child       |
|                       |                       | dependant to be       |
|                       |                       | eligible              |
|                       |                       |                       |
|                       |                       | A family can have     |
|                       |                       | multiple child        |
|                       |                       | dependants            |
|                       |                       |                       |
|                       |                       | The member allocated  |
|                       |                       | as a child dependant  |
|                       |                       | must:                 |
|                       |                       |                       |
|                       |                       | -   not belong to     |
|                       |                       |     more than 1 other |
|                       |                       |     family            |
|                       |                       |     registration      |
|                       |                       | -   be under 16 years |
|                       |                       |     of age            |
+-----------------------+-----------------------+-----------------------+

Amend Medicare Safety Net composition

Table 2: this table lists situations where the Family Contact (FC) is
deceased and what action is to follow.

+-----------------------------------+-----------------------------------+
| Situation                         | Action                            |
+-----------------------------------+-----------------------------------+
| Where the deceased person is the  |                                   |
| Family Contact (FC) and:          |                                   |
|                                   |                                   |
| -   there is a spouse/defacto     |                                   |
|     partner registered (SP)       |                                   |
|                                   |                                   |
| Service Officers must change the: |                                   |
|                                   |                                   |
| -   Family Contact to the Spouse  |                                   |
| -   Spouse to the Family Contact  |                                   |
|                                   |                                   |
| See: Amend a Medicare Safety Net  |                                   |
| Family Registration               |                                   |
+-----------------------------------+-----------------------------------+
| Where the deceased person is the  |                                   |
| Family Contact (FC) and:          |                                   |
|                                   |                                   |
| -   there is no spouse / de facto |                                   |
|     (SP)                          |                                   |
| -   no student dependant (SD) or  |                                   |
|     child 14 years or older       |                                   |
|     registered                    |                                   |
| -   the eldest CD is under 14     |                                   |
|     years of age                  |                                   |
|                                   |                                   |
| Service Officers must not change  |                                   |
| the Safety Net registration where |                                   |
| they have determined the:         |                                   |
|                                   |                                   |
| -   deceased or end-dated person  |                                   |
|     is the family contact         |                                   |
| -   Safety Net registration does  |                                   |
|     not contain a spouse/de facto |                                   |
|     partner or student dependant, |                                   |
|     and                           |                                   |
| -   eldest child dependant is     |                                   |
|     under 14 years of age         |                                   |
|                                   |                                   |
| See: Amend a Medicare Safety Net  |                                   |
| Family Registration               |                                   |
+-----------------------------------+-----------------------------------+
| Where the deceased person is the  |                                   |
| Family Contact (FC) and:          |                                   |
|                                   |                                   |
| -   there is no spouse / de facto |                                   |
|     (SP) and                      |                                   |
| -   there is a student dependant  |                                   |
|     (SD) or child 14 years of age |                                   |
|     or older registered (CD)      |                                   |
|                                   |                                   |
| Service Officers must change the: |                                   |
|                                   |                                   |
| -   oldest registered member to   |                                   |
|     the Family Contact            |                                   |
| -   Deceased Family Contact to    |                                   |
|     Spouse                        |                                   |
|                                   |                                   |
| See: Amend a Medicare Safety Net  |                                   |
| Family Registration               |                                   |
+-----------------------------------+-----------------------------------+
