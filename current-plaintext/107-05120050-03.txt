|                                   | person must, amongst other        |
|                                   | things, have been adversely       |
|                                   | affected by the natural disaster. |
|                                   | A person is adversely affected by |
|                                   | the natural disaster if, as a     |
|                                   | direct result:                    |
|                                   |                                   |
|                                   | -   the person was seriously      |
|                                   |     injured                       |
|                                   | -   an affected family member of  |

|                                   |     of residence was destroyed    |
|                                   | -   the person's principal place  |
|                                   |     of residence sustained major  |
|                                   |     damage                        |
|                                   | -   the person was unable to      |
|                                   |     return to his or her          |
|                                   |     principal place of residence  |
|                                   |     for at least 24 hours         |

Finalising the debt waiver

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check the status of the debt on   |
|                                   | the Overpayment Debt List (OPDL)  |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Is the debt finalised fully       |
|                                   | recovered (FRC)?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Finalised fully recovered + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For DMIS to allow a waiver, the   |
|                                   | debt status must show as DET      |
|                                   | (determined). This is done by     |
|                                   | adjusting the repayment history.  |
|                                   |                                   |
|                                   | Over-recovered debts              |
|                                   |                                   |
|                                   | Do not tell the customer if a     |
|                                   | refund will be paid. Debt staff   |
|                                   | will decide this. They may pay a  |
|                                   | refund or transfer the amount to  |
|                                   | other outstanding debts.          |
|                                   |                                   |
|                                   | Non-Debt staff                    |
|                                   |                                   |
|                                   | DOC the record using              |
|                                   |                                   |
|                                   | Fast Note. The Fast Note will     |
|                                   | create a DOC and automatically    |
|                                   | refer to Compliance and Debt      |
|                                   | Operations (NS7), requesting the  |
|                                   | repayment history be adjusted to  |
|                                   | allow the waiver to be applied.   |
|                                   |                                   |
|                                   | -   Select Auto Text option> Debt |
|                                   |     Recovery > Debt Recovery      |
|                                   |     referrals > OPRH Adj Req for  |
|                                   |     Debt Waiver                   |
|                                   |                                   |
|                                   | Select Confirm                    |
|                                   |                                   |
|                                   | Record relevant information as    |
|                                   | required                          |
|                                   |                                   |
|                                   | Debt staff                        |
|                                   |                                   |
|                                   | -   Select the debt to waive from |
|                                   |     the OPDL screen               |
|                                   | -   Go to the Repayment History   |
|                                   |     (OPRH) screen. Adjust the     |
|                                   |     last payment by $0.01 cent.   |
|                                   |     Note: reverse this before     |
|                                   |     finalising the debt           |
|                                   |                                   |
|                                   | Once a waiver is complete, DMIS   |
|                                   | generates an over-recovered       |
|                                   | activity for the repaid amount.   |
|                                   | These activities automatically    |
|                                   | map to Debt staff to determine    |
|                                   | whether a refund is due to the    |
|                                   | customer.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Recovery Activity on AL screen +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Locate the DMN/RCV activity   |
|                                   |     on the Activity List (AL)     |
|                                   |     screen                        |
|                                   | -   'S'elect the activity         |
|                                   | -   'S'elect the Debt ID to be    |
|                                   |     waived from the Debt List     |
|                                   |     (OPDL) screen                 |
|                                   | -   Press [Enter]                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Coding the waiver + Read more ... |
|                                   |                                   |
|                                   | The debt meets all the criteria   |
|                                   | for waiver.                       |
|                                   |                                   |
|                                   | Go to the Add Waiver (OPWAA)      |
|                                   | screen and code:                  |
|                                   |                                   |
|                                   | -   in the Amount Waived $:       |
|                                   |     field, the amount of the debt |
|                                   |     to be waived                  |
|                                   | -   in the Reason Waived: field,  |
|                                   |     COD                           |
|                                   | -   in the Comments: field, a     |
|                                   |     brief explanation of why the  |
|                                   |     debt is being waived          |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | Finalise activity on the          |
|                                   | Assessment Finalisation (OPAF)    |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Advise decision if necessary +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer has already been  |
|                                   | advised of the debt:              |
|                                   |                                   |
|                                   | -   contact them to advise the    |
|                                   |     decision to waive the debt    |
|                                   | -   make 2 genuine attempts to    |
|                                   |     contact                       |
|                                   | -   send a Q053 letter if unable  |
|                                   |     to contact, see approved text |
|                                   |                                   |
|                                   | Use                               |
|                                   |                                   |
|                                   | Fast Note to record details in a  |
|                                   | DOC.                              |
|                                   |                                   |
|                                   | Record all details on a DOC as    |
|                                   | per the minimum DOC standards for |
|                                   | debt waivers.                     |
