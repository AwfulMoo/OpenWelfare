|                                   |     bereavement period            |
|                                   | -   NR is the surviving partner's |
|                                   |     new fortnightly single rate   |

|                                   |     on the payday after their     |
|                                   |     death. If the deceased also   |
|                                   |     received Family Tax Benefit   |
|                                   |     (FTB) and the survivor is     |

|                                   |     Coronavirus Supplement, where |
|                                   |     the survivor received it      |
|                                   |     prior to the bereavement      |
|                                   |     period                        |
|                                   | -   NEPED is the number of        |
|                                   |     entitlement period end days   |
|                                   |     from and including the day of |

Care receiver LBP calculation

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Death of care receiver + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the carer was getting:         |
|                                   |                                   |
|                                   | -   Carer Payment (CP), go to     |
|                                   |     Step 2                        |
|                                   | -   Carer Allowance (CA) for an   |
|                                   |     adult care receiver, go to    |
|                                   |     Step 3                        |
|                                   | -   CA for a child care receiver, |
|                                   |     go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 2                                 | CP customer + Read more ...       |
|                                   |                                   |
|                                   | First check if the care receiver  |
|                                   | was partnered. If so, check if    |
|                                   | the surviving partner is entitled |
|                                   | to an LBP. The LBP is worked out  |
|                                   | differently for a surviving       |
|                                   | partner. If the non-carer partner |
|                                   | is entitled to an LBP, the carer  |
|                                   | will not be entitled to an LBP as |
|                                   | well, but will continue to        |
|                                   | receive CP for the bereavement    |
|                                   | period.                           |
|                                   |                                   |
|                                   | If the care receiver was a child  |
|                                   | with a severe disability or       |
|                                   | medical condition, or the care    |
|                                   | receiver child was in a           |
|                                   | short-term or episodic care       |
|                                   | situation, the carer may be       |
|                                   | entitled to an LBP.               |
|                                   |                                   |
|                                   | Complete the death action in the  |
|                                   | care receiver's record for the    |
|                                   | 'ripple' to initiate any lump sum |
|                                   | payment to be paid on the carer's |
|                                   | record.                           |
|                                   |                                   |
|                                   | If the Carer is entitled to an    |
|                                   | LBP, the LBP payable to the CP    |
|                                   | customer is the lesser of:        |
|                                   |                                   |
|                                   | -   7 times the partnered maximum |
|                                   |     basic pension rate            |
