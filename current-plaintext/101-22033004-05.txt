Examples of customers eligible to receive the Cost of Living Payment

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | On a nil rate of payment because  |
|                                   | of earnings                       |
|                                   |                                   |
|                                   | Tony is receiving JobSeeker       |
|                                   | Payment, but received a nil rate  |
|                                   | (CZR) of payment on the 29 March  |
|                                   | 2022 because of income received.  |
|                                   |                                   |
|                                   | Although Tony is on a nil rate of |
|                                   | payment, they are still taken to  |
|                                   | be receiving the JobSeeker        |
|                                   | Payment.                          |
|                                   |                                   |
|                                   | Tony is eligible for the Cost of  |
|                                   | Living Payment.                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Backdated claim granted to a day  |
|                                   | before or on the test date        |
|                                   |                                   |
|                                   | Freya’s Age Pension claim was     |
|                                   | assessed on 3 April 2022 (after   |
|                                   | the test date) and was            |
|                                   | subsequently granted from 28      |
|                                   | March 2022.                       |
|                                   |                                   |
|                                   | As Freya was entitled to the Age  |
|                                   | Pension on the 29 March 2022, she |
|                                   | is also considered eligible for   |
|                                   | the Cost of Living Payment and    |
|                                   | will receive the payment          |
|                                   | automatically when her new claim  |
|                                   | is granted.                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Payment suspended before test     |
|                                   | date. Payment then subsequently   |
|                                   | restored from date of suspension  |
|                                   |                                   |
|                                   | Marco’s Carer Payment was         |
|                                   | suspended from 27 March 2022.     |
|                                   |                                   |
|                                   | On 2 April 2022 Marco’s Carer     |
|                                   | Payment was restored from the     |
|                                   | date of suspension (27 March      |
|                                   | 2022) and continued to remain     |
|                                   | current thereafter.               |
|                                   |                                   |
|                                   | Although Marco’s payment was      |
|                                   | suspended during the test date,   |
|                                   | the subsequent restoration of his |
|                                   | Carer Payment means that he is    |
|                                   | considered eligible for the Cost  |
|                                   | of Living Payment and will        |
|                                   | receive the payment automatically |
|                                   | when restored.                    |
+-----------------------------------+-----------------------------------+

Examples of customers not eligible to receive the Cost of Living Payment

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer serving a waiting or     |
|                                   | preclusion period during the test |
|                                   | date. Waiting period has not been |
|                                   | reduced or waived                 |
|                                   |                                   |
|                                   | Liliana has been assessed for     |
|                                   | JobSeeker Payment.                |
|                                   |                                   |
|                                   | She needs to serve an Ordinary    |
|                                   | Waiting Period from 25 March 2022 |
|                                   | until 1 April 2022.               |
|                                   |                                   |
|                                   | As Liliana was not taken to be    |
|                                   | receiving JobSeeker Payment on 29 |
|                                   | March 2022, she is not considered |
|                                   | eligible for the Cost of Living   |
|                                   | Payment.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Receiving an ineligible payment   |
|                                   | that does not attract the Cost of |
|                                   | Living Payment                    |
|                                   |                                   |
|                                   | Francis was a Low Income Card     |
|                                   | (LIC) holder on 29 March 2022.    |
|                                   |                                   |
|                                   | As he was not receiving any       |
|                                   | eligible payment types or         |
|                                   | concession cards on the test      |
|                                   | date, he is not eligible to       |
|                                   | receive the Cost of Living        |
|                                   | Payment.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Payment cancelled during the test |
|                                   | date                              |
|                                   |                                   |
|                                   | Miriam’s Carer Allowance (CA) was |
|                                   | cancelled from 20 March 2022.     |
|                                   |                                   |
|                                   | The payment has not been          |
|                                   | restored.                         |
|                                   |                                   |
|                                   | As Miriam was not receiving a     |
|                                   | current eligible payment on the   |
|                                   | test date, she is not eligible    |
|                                   | for the Cost of Living Payment.   |
+-----------------------------------+-----------------------------------+
| 4                                 | Receiving ABSTUDY (Living         |
|                                   | Allowance) nil rate period (CZR)  |
|                                   |                                   |
|                                   | Jonah is receiving ABSTUDY        |
|                                   | (Living Allowance), but received  |
|                                   | a nil rate (CZR) of payment on    |
|                                   | the 29 March 2022.                |
|                                   |                                   |
|                                   | Jonah is not eligible to receive  |
|                                   | the Cost of Living Payment.       |
+-----------------------------------+-----------------------------------+
| 5                                 | Receiving 100% Home Equity Access |
|                                   | Scheme                            |
|                                   |                                   |
|                                   | Mary is receiving 100% Home       |
|                                   | Equity Access Scheme.             |
|                                   |                                   |
|                                   | On 29 March 2022, Mary was        |
|                                   | receiving a nil rate period (CZR) |
|                                   | because of the Home Equity Access |
|                                   | Scheme.                           |
|                                   |                                   |
|                                   | Mary is not eligible for the Cost |
|                                   | of Living Payment.                |
+-----------------------------------+-----------------------------------+
| 6                                 | Current RapidConnect failure      |
|                                   |                                   |
|                                   | Jamal was granted JobSeeker       |
|                                   | Payment before 29 March 2022.     |
|                                   |                                   |
|                                   | Jamal has not yet complied with   |
|                                   | his RapidConnect requirements and |
|                                   | as a result received a nil rate   |
|                                   | of payment (CZR) on 29 March      |
|                                   | 2022.                             |
|                                   |                                   |
|                                   | Jamal is not eligible to receive  |
|                                   | the Cost of Living Payment.       |
