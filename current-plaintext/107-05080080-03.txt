|                                   | Zealand debts + Read more ...     |
|                                   |                                   |
|                                   | For details about:                |
|                                   |                                   |
|                                   | -   New requests for recovery     |
|                                   |     action, go to Step 2          |
|                                   | -   Customer contacts, go to Step |
|                                   |     6                             |
|                                   |                                   |
|                                   | See the Resources page for an     |
|                                   | attachment with Resources for NZ  |
|                                   | debt recovery.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | New requests for recovery of New  |
|                                   | Zealand debts + Read more ...     |
|                                   |                                   |
|                                   | See Resources for the supporting  |
|                                   | information that will be required |
|                                   | throughout this procedure.        |
|                                   |                                   |
|                                   | Requests for recovery of a debt   |
|                                   | from New Zealand are sent to CIS  |
|                                   | via secure electronic liaison.    |
|                                   |                                   |
|                                   | -   Access the NZ electronic      |
|                                   |     liaison system                |
|                                   | -   Open the 'NZ Debt             |
|                                   |     Certificate' liaison          |
|                                   |                                   |
|                                   | To be able to be recovered, the   |
|                                   | debt must be:                     |
|                                   |                                   |
|                                   | -   for a New Zealand social      |
|                                   |     security payment, and         |
|                                   | -   more than AUD$400, and        |
|                                   | -   first found to be owing       |
|                                   |     within the last five years    |
|                                   |                                   |
|                                   | Does the debt meet the agreed     |
|                                   | terms and conditions for          |
|                                   | recovery?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, reply to the liaison      |
|                                   |     indicating that the debt does |
|                                   |     not meet the terms and        |
|                                   |     conditions for recovery.      |
|                                   |     Record and finalise the       |
|                                   |     liaison. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 3                                 | Search for customer record + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the details provided in the   |
|                                   | liaison to search for a record,   |
|                                   | see Searching for a customer on   |
|                                   | the system.                       |
|                                   |                                   |
|                                   | Is a customer record found?       |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, reply to the liaison      |
|                                   |     indicating that no customer   |
|                                   |     record was found. Record and  |
|                                   |     finalise the liaison.         |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Check whether recovery is         |
|                                   | possible + Read more ...          |
|                                   |                                   |
|                                   | To be recoverable, the customer   |
|                                   | must be currently in receipt of   |
|                                   | an income support payment from    |
|                                   | Australia (not family assistance) |
|                                   | with no Australian debt or        |
|                                   | advance payment being recovered.  |
|                                   |                                   |
|                                   | -   Go to the OPDL screen and     |
|                                   |     check the outstanding CRN     |
|                                   |     balance. If there is an       |
|                                   |     outstanding balance, debt     |
|                                   |     recovery for New Zealand      |
|                                   |     cannot proceed                |
|                                   | -   Go to the AVCS screen         |
|                                   |     (Customer Record) or run the  |
|                                   |     Manage Advances process in    |
|                                   |     Process Direct. If the        |
|                                   |     customer has an advance that  |
|                                   |     has not been fully recovered, |
|                                   |     debt recovery for New Zealand |
|                                   |     cannot proceed                |
|                                   |                                   |
|                                   | Is the New Zealand debt           |
|                                   | recoverable?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, reply to the liaison      |
|                                   |     indicating that debt recovery |
|                                   |     is not possible. Include last |
|                                   |     known contact details,        |
|                                   |     including address, telephone  |
|                                   |     and email. Record and         |
|                                   |     finalise the liaison.         |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Record the NZ debt + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | A new debt must be added to the   |
|                                   | customer record.                  |
|                                   |                                   |
|                                   | See Step 3 in the Manually        |
|                                   | creating a new debt record on the |
|                                   | DMIS table of Create a new debt   |
|                                   | record manually on the Debt       |
|                                   | Management and Information System |
|                                   | (DMIS). Use the following         |
|                                   | details:                          |
|                                   |                                   |
|                                   | -   Benefit Type: field is as     |
|                                   |     advised in the liaison form,  |
|                                   |     e.g. ‘UBZ’. Use field help if |
|                                   |     unsure                        |
|                                   | -   Reason: field is 'NZL'        |
|                                   | -   Date of Event: is the DOR of  |
|                                   |     the liaison                   |
|                                   | -   Start Date: is the debt start |
|                                   |     date according to the liaison |
|                                   | -   End Date: is the debt end     |
|                                   |     date according to the liaison |
|                                   | -   Calculated amount $: is the   |
|                                   |     amount according to the       |
|                                   |     liaison                       |
|                                   | -   Addtnl Reference No: is the   |
|                                   |     customer’s SWN                |
|                                   | -   NZ Office: field is as        |
|                                   |     advised in the liaison form   |
|                                   |                                   |
|                                   | Reply to the liaison indicating   |
|                                   | that debt recovery will commence. |
|                                   | Include updated contact details,  |
|                                   | including address, telephone and  |
|                                   | email                             |
|                                   |                                   |
|                                   | Procedure ends here.              |
