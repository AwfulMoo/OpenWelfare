| 1                                 | DAP claim submitted on another    |
|                                   | person's online account           |
|                                   |                                   |
|                                   | 'We cannot pay you Dad and        |
|                                   | Partner Pay because you have used |
|                                   | another person's online account   |
|                                   | to lodge a claim.                 |
|                                   |                                   |
|                                   | If you wish to claim Dad and      |
|                                   | Partner Pay using the online      |
|                                   | claim, you will need to lodge a   |
|                                   | claim using your own online       |
|                                   | account. If you don’t have an     |
|                                   | existing online account, you can  |
|                                   | register for a Centrelink online  |
|                                   | account to lodge a claim in your  |
|                                   | own name. Information about       |
|                                   | lodging a claim can be found at   |
|                                   | www.servicesaustralia.gov.au.'    |

| 2                                 | Not effective NEF/N97             |
|                                   |                                   |
|                                   | 'We are writing to you about your |

|                                   | your child is expected to <be     |
|                                   | born/come into your care>.        |
|                                   |                                   |

|                                   | expected <to be born/come into    |
|                                   | your care>.'                      |

|                                   | the same child on <initial claim  |
|                                   | date>.                            |
|                                   |                                   |
|                                   | You do not need to do anything    |
|                                   | regarding this letter. After we   |
