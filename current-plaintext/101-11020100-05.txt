This page contains a table detailing notification and status codes for
the Third Party Notification Summary (TPNS) screen.

Notification and status codes

Notification and status codes for the TPNS screen

This table details the status code combinations available for each
notification type.

+-----------------------+-----------------------+-----------------------+
| Notification type     | Status                | Description           |
+-----------------------+-----------------------+-----------------------+
| Trading status        | Trading               | Identifies that the   |
|                       |                       | organisation is       |
|                       |                       | currently trading.    |
+-----------------------+-----------------------+-----------------------+
| Trading status        | Not Trading           | Identifies that the   |
|                       |                       | organisation has      |
|                       |                       | ceased trading.       |
+-----------------------+-----------------------+-----------------------+
| System Use status     | In Use                |                       |
+-----------------------+-----------------------+-----------------------+
| System Use status     | User Alert            | Applies an            |
|                       |                       | information notice to |
|                       |                       | an active record.     |
+-----------------------+-----------------------+-----------------------+
| System Use status     | Do Not Use            | Record becomes        |
|                       |                       | inactive with view    |
|                       |                       | only access. Will     |
|                       |                       | still be displayed in |
|                       |                       | searches.             |
+-----------------------+-----------------------+-----------------------+
| System Use status     | Logical Delete        | Record becomes        |
|                       |                       | inactive with view    |
|                       |                       | only access. Will not |
|                       |                       | be displayed in       |
|                       |                       | searches.             |
+-----------------------+-----------------------+-----------------------+
