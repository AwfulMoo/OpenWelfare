Insert and delete lines in FAPI

Use the A field on the FAPC screen to insert and delete lines in FAPC.

Note: actions can be keyed for many service lines but they must be of
the same type, that is, all ‘D’ or all ‘I’.

Insert and delete line values, descriptions and explanations

Table 3: This table lists the valid values in the A field. Key one of
the following values in the A field and press [PF6].

+-----------------------+-----------------------+-----------------------+
| Value                 | Description           | Explanation           |
+-----------------------+-----------------------+-----------------------+
| I                     | Insert a service      | Insert a blank        |
|                       | line.                 | service line under    |
|                       |                       | this line and move    |
|                       |                       | all subsequent        |
|                       |                       | service lines down    |
|                       |                       | one line.             |
+-----------------------+-----------------------+-----------------------+
| D                     | Delete a service      | Delete this service   |
|                       | line.                 | line and move all     |
|                       |                       | subsequent service    |
|                       |                       | lines up one line.    |
+-----------------------+-----------------------+-----------------------+

FAPI PF Key functions

FAPI PF (function key) descriptions

be used on the FAPI suite of screens.

+-----------------------------------+-----------------------------------+
