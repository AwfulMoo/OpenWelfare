Income estimates for Paid Parental Leave scheme claims

Table 1: Portions of this process can be completed by Families and Child
Care Processing Services staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Paid Parental Leave (PPL) /Dad    |
|                                   | and Partner Pay (DAP) income      |
|                                   | test + Read more ...              |
|                                   |                                   |
|                                   | A PPL/DAP claimant’s adjusted     |
|                                   | taxable income (ATI) must be      |
|                                   | below the relevant financial year |
|                                   | income limit to be eligible for   |
|                                   | payment. Evidence of a claimant’s |
|                                   | ATI is needed when their estimate |
|                                   | is between $135,000 and the       |
|                                   | income limit for the relevant     |
|                                   | financial year.                   |
|                                   |                                   |
|                                   | Commencing from 1 July 2021, the  |
|                                   | PPL and DAP Income limits are     |
|                                   | subject to indexation each        |
|                                   | financial year.                   |
|                                   |                                   |
|                                   | The PPL/DAP Income guided         |
|                                   | procedure will display both the   |
|                                   | income estimate provided and any  |
|                                   | verified income on the claimant's |
|                                   | record.                           |
|                                   |                                   |
|                                   | Check the PPL/DAP Income guided   |
|                                   | procedure screen for the relevant |
|                                   | financial year. The screen will   |
|                                   | display the verified income for   |
|                                   | the applicable financial year, or |
|                                   | if this is not available, the     |
|                                   | last available estimate provided  |
|                                   | by the claimant for the           |
|                                   | applicable financial year.        |
|                                   |                                   |
|                                   | Note: a customer assessed as      |
|                                   | eligible for DAP (including       |
|                                   | meeting the income test), who     |
|                                   | later claims PPL for the same     |
|                                   | child, will automatically meet    |
|                                   | the income test for PPL. This     |
|                                   | also applies if a secondary       |
|                                   | claimant assessed as eligible for |
|                                   | PPL later claims DAP for the same |
|                                   | child (the customer will also     |
|                                   | meet the income test for DAP).    |
|                                   |                                   |
|                                   | Is the claimant’s estimated ATI   |
|                                   | between $135,000 and the relevant |
|                                   | financial year income limit?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Evidence is needed for estimates  |
|                                   | between $135,000 and the relevant |
|                                   | financial year income limit +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the total estimate of income   |
|                                   | is between $135,000 and the       |
|                                   | relevant financial year income    |
|                                   | limit, evidence is needed to      |
|                                   | support the estimate.             |
|                                   |                                   |
|                                   | For online claims, claimants are  |
|                                   | advised on the Next steps page to |
|                                   | provide evidence if it is needed. |
|                                   |                                   |
|                                   | Check if actual income details    |
|                                   | are available                     |
|                                   |                                   |
|                                   | If the claimant’s income is       |
|                                   | between $135,000 and the relevant |
|                                   | financial year income limit and   |
|                                   | the claimant’s actual adjusted    |
|                                   | taxable income (AATI) details     |
|                                   | have been transferred from the    |
|                                   | Australian Taxation Office (ATO)  |
|                                   | for Family Tax Benefit (FTB)      |
|                                   | further evidence should not be    |
|                                   | requested.                        |
|                                   |                                   |
|                                   | If there is a significant         |
|                                   | discrepancy between the provided  |
|                                   | income estimate and the verified  |
|                                   | income, consider if contact with  |
|                                   | the claimant is needed to         |
|                                   | establish eligibility. Contact    |
|                                   | with the customer should only be  |
|                                   | made if the verified income on    |
|                                   | record could change the payment   |
|                                   | outcome or the requirement for    |
|                                   | evidence.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Reasonable estimate needed for    |
|                                   | PPL and DAP claims + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | To be accepted, estimates must be |
|                                   | determined as reasonable.         |
|                                   |                                   |
|                                   | When there is a significant       |
|                                   | discrepancy between the income    |
|                                   | estimate provided in the claim    |
|                                   | and the verified income on the    |
|                                   | record, consider contacting the   |
|                                   | claimant. Contact must be made if |
|                                   |                                   |
|                                   | -   using the verified income     |
|                                   |     instead of the estimate       |
|                                   |     provided could change their   |
|                                   |     eligibility or                |
|                                   | -   evidence is required to       |
|                                   |     substantiate their income     |
|                                   |     estimate                      |
|                                   |                                   |
|                                   | Note: The PPL income test and DAP |
|                                   | income test apply to the claimant |
|                                   | only. Their partner's income is   |
|                                   | not included in determining if    |
|                                   | the claimant meets income         |
|                                   | eligibility.                      |
|                                   |                                   |
|                                   | PPL:                              |
|                                   |                                   |
|                                   | Individuals claiming PPL,         |
|                                   | including secondary/tertiary      |
|                                   | claimants, must provide a         |
|                                   | reasonable estimate of their      |
|                                   | adjusted taxable income (ATI):    |
|                                   |                                   |
|                                   | -   for a pre-birth/adoption      |
|                                   |     claim, an estimate is needed  |
|                                   |     for the financial year before |
|                                   |     the date of claim             |
|                                   | -   for a post-birth/adoption     |
|                                   |     claim, an estimate is needed  |
|                                   |     for the financial year before |
|                                   |     the date the child was born   |
|                                   |     or entered their care         |
|                                   |                                   |
|                                   | Pre and post birth PPL claims     |
|                                   | spanning financial years          |
|                                   |                                   |
|                                   | If a claimant has lodged a        |
|                                   | pre-birth claim in a previous     |
|                                   | financial year (for example,      |
|                                   | 2016/2017) and later lodged proof |
|                                   | of birth in the following         |
|                                   | financial year (2017/2018),       |
|                                   | additional income details are not |
|                                   | needed. The claim must be         |
|                                   | finalised using the income        |
|                                   | details provided with the         |
|                                   | pre-birth claim.                  |
|                                   |                                   |
|                                   | DAP:                              |
|                                   |                                   |
|                                   | Individuals claiming DAP must     |
|                                   | provide a reasonable estimate of  |
|                                   | their ATI.                        |
|                                   |                                   |
|                                   | -   for a pre-birth/adoption      |
|                                   |     claim, an estimate is needed  |
|                                   |     for the financial year before |
|                                   |     the date of the claim         |
|                                   | -   for a post-birth/adoption     |
|                                   |     claim, an estimate is needed  |
|                                   |     for the financial year before |
|                                   |     the date the date of claim or |
|                                   |     the nominated start date,     |
|                                   |     whichever is earlier          |
+-----------------------------------+-----------------------------------+
| 4                                 | Providing a reasonable estimate + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The claimant must provide an      |
|                                   | estimate in the PPL or DAP claim. |
|                                   |                                   |
|                                   | If they have lodged a claim that  |
|                                   | has not yet been processed, they  |
|                                   | may verbally provide a revised    |
|                                   | reasonable estimate. See Customer |
|                                   | notifies a change of              |
|                                   | circumstances for Parental Leave  |
|                                   | Pay (PPL).                        |
|                                   |                                   |
|                                   | If the claimant has lodged their  |
|                                   | tax return for the relevant       |
|                                   | financial year, their tax return  |
|                                   | and Notice of Assessment (NOA)    |
|                                   | will help them provide an         |
|                                   | accurate income estimate.         |
|                                   |                                   |
|                                   | The estimate must include an      |
|                                   | amount (or zero) for all ATI      |
|                                   | components:                       |
|                                   |                                   |
|                                   | -   Taxable income. If the        |
|                                   |     claimant received taxable     |
|                                   |     Centrelink payments, these    |
|                                   |     must be included as reported  |
|                                   |     on their payment summary.     |
|                                   |     Check details on the Tax      |
|                                   |     Payment Summary (TXGS) screen |
|                                   | -   Reportable fringe benefits    |
|                                   |     including exempt reportable   |
|                                   |     fringe benefits, as on the    |
|                                   |     payment summary from the      |
|                                   |     employer. From 1 January      |
|                                   |     2017, exempt reportable       |
|                                   |     fringe benefits and all other |
|                                   |     reportable fringe benefits    |
|                                   |     will be assessed differently. |
|                                   |     Claimants must provide any    |
|                                   |     exempt reportable fringe      |
|                                   |     benefits as a separate amount |
|                                   |     to any non-exempt reportable  |
|                                   |     fringe benefits they receive. |
|                                   |     This includes reportable      |
|                                   |     superannuation contributions  |
|                                   | -   Foreign income and tax exempt |
|                                   |     foreign income                |
|                                   | -   Total net investment losses   |
|                                   |     which includes net rental     |
|                                   |     property loss                 |
|                                   | -   Tax free pensions and         |
|                                   |     benefits per their payment    |
|                                   |     summary. Check details on     |
|                                   |     TXGS screen                   |
|                                   | -   Less child maintenance        |
|                                   |     expenditure                   |
|                                   |                                   |
|                                   | Note: when a customer provides an |
|                                   | estimate, a superannuation        |
|                                   | withdrawal under the First Home   |
|                                   | Super Saver (FHSS) Scheme is not  |
|                                   | counted towards their taxable     |
|                                   | income.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Has the claimant provided an      |
|                                   | estimate of all ATI components    |
|                                   | for their claim? + Read more ...  |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, the claimant must be      |
|                                   |     contacted to obtain the       |
|                                   |     details. If necessary, run    |
|                                   |     the Request and Manage        |
|                                   |     Customer Tasks guided         |
|                                   |     procedure to request the      |
|                                   |     information (and evidence if  |
|                                   |     it will be needed) be         |
|                                   |     provided within 14 days (21   |
|                                   |     days for remote or overseas   |
|                                   |     claimants). If not provided   |
|                                   |     as requested, finalise claim  |
|                                   |     as not effective (NEF-INF).   |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 6                                 | Has it been determined the income |
|                                   | estimate is reasonable? + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | No, go to Step 8                  |
|                                   |                                   |
|                                   | Yes, is the estimate between      |
|                                   | $135,000 and the relevant         |
|                                   | financial year income limit?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, go to Step 9              |
+-----------------------------------+-----------------------------------+
| 7                                 | Request evidence + Read more ...  |
|                                   |                                   |
|                                   | Issue Q777 via the Request        |
|                                   | Documents Q777 guided procedure   |
|                                   | to request evidence be provided   |
|                                   | within 14 days (21 days for       |
|                                   | remote or overseas claimants).    |
|                                   |                                   |
|                                   | Claimants can post, fax or        |
|                                   | document lodge evidence to        |
|                                   | support their estimate. Originals |
|                                   | do not have to be provided. The   |
|                                   | Resources page lists examples.    |
|                                   |                                   |
|                                   | Is evidence provided by the       |
|                                   | resubmit date?                    |
|                                   |                                   |
|                                   | -   Yes, scan documents and       |
|                                   |     return any originals to the   |
|                                   |     claimant. Go to Step 8        |
|                                   | -   No, finalise the claim as not |
|                                   |     effective, reason NPI (proof  |
|                                   |     of income not provided).      |
|                                   |     Process claim to              |
|                                   |     upload/record estimate and to |
|                                   |     indicate requested evidence   |
|                                   |     has not been provided.        |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 8                                 | Estimate not considered           |
|                                   | reasonable + Read more ...        |
|                                   |                                   |
|                                   | []For Families and Child Care     |
|                                   | Processing Services staff only.   |
|                                   |                                   |
|                                   | Discuss with the claimant why the |
|                                   | estimate is not considered        |
|                                   | reasonable and request a revised  |
|                                   | estimate.                         |
|                                   |                                   |
|                                   | If the claimant does not wish to  |
|                                   | provide an acceptable revised     |
|                                   | estimate or cannot be contacted:  |
|                                   |                                   |
|                                   | -   DOC the estimate and indicate |
|                                   |     it is not considered          |
|                                   |     reasonable. Unreasonable      |
|                                   |     estimates will not be applied |
|                                   |     for the PPL assessment        |
|                                   | -   Issue a Q777 using the        |
|                                   |     Request Documents Q777 guided |
|                                   |     procedure advising evidence   |
|                                   |     must be supplied within 19    |
|                                   |     days (21 days for remote or   |
|                                   |     overseas claimants) or claim  |
|                                   |     will be rejected              |
|                                   |                                   |
|                                   | If a reasonable estimate is not   |
|                                   | provided as requested:            |
|                                   |                                   |
|                                   | -   process the PPL claim or      |
|                                   | -   process the DAP claim         |
|                                   |                                   |
|                                   | Use the ’PPL Circumstances’ or    |
|                                   | ‘DAP Circumstances’ workflow in   |
|                                   | the PPL/DAP claim to              |
|                                   | upload/record the estimate        |
|                                   | provided is not reasonable. The   |
|                                   | estimate will not be applied and  |
|                                   | PPL or DAP will reject, reason    |
|                                   | ENA (Estimate not accepted). Go   |
|                                   | to Customer notifies a change in  |
|                                   | circumstances for PPL.            |

| 9                                 | Record reasonable income estimate |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | []For Families and Child Care     |
|                                   | Processing Services staff only.   |

|                                   | Once the claim is ready to        |
|                                   | process, use the Search Online    |
|                                   | Claims under the Claims Online    |
|                                   | heading in Customer First to      |
|                                   | upload/record income details and  |
|                                   | to indicate the estimate is       |
|                                   | reasonable.                       |

|                                   | Note: for information on how to   |

|                                   |                                   |
|                                   | []For Families and Child Care     |

|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   the claimant may be eligible  |
|                                   |     for PPL if other eligibility  |
|                                   |     criteria are met, or          |
|                                   | -   the claimant may be eligible  |
|                                   |     for DAP if other eligibility  |
|                                   |     criteria are met              |
|                                   |                                   |

+-----------------------------------+-----------------------------------+
| 2                                 | If claimant or partner claiming   |
|                                   | or receiving family assistance +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | []For Families and Child Care     |
|                                   | Processing Services staff only.   |
|                                   |                                   |
|                                   | Any PPL or DAP received by the    |
|                                   | claimant must be included in      |
|                                   | their estimated taxable income    |
|                                   | for the relevant period.          |
|                                   |                                   |
|                                   | Check the claimant's current      |
|                                   | financial year estimate is        |
|                                   | reasonable for Family Tax Benefit |
|                                   | (FTB).                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Claim rejected due to income      |
|                                   | (REJ-INC) + Read more ...         |
|                                   |                                   |
|                                   | PPL                               |
|                                   |                                   |
|                                   | If the claimant's pre-birth claim |
|                                   | was rejected due to income and    |
|                                   | the child is born or enters their |
|                                   | primary care in the new financial |
|                                   | year, they may wish to reclaim    |
|                                   | PPL and test their eligibility    |
|                                   | against income for the financial  |
|                                   | year before the child's birth or  |
|                                   | entry into care                   |
|                                   |                                   |
|                                   | If a claimant wants to revise     |
|                                   | their estimate for the same       |
|                                   | financial year, they should apply |
|                                   | for a formal review of the        |
|                                   | decision to reject their PPL      |
|                                   | claim.                            |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Claiming family assistance    |
|                                   |     and PPL for a newborn child   |
|                                   | -   Initial contact about a       |
|                                   |     decision and the review of    |
|                                   |     decision process              |
|                                   |                                   |

|                                   |                                   |

|                                   | financial year, they should apply |

|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Change in circumstance for    |
|                                   |     DAP                           |
|                                   | -   Initial contact about a       |
|                                   |     decision and the review of    |
