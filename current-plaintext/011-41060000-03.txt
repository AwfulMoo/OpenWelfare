This page contains the escalation processes for Cleft Lip and Cleft
Palate (CLaCP) Scheme claims.

CLaCP claims in the general queue are payable by Service Officers.
Claims should only be escalated if an error message presents or the
claim requires a restriction lift.

On this page:

Assessing time and history restrictions (RST TD and RST HIST)

Assessing age restrictions – initial item for permanent dentition
treatment (AGE-RST)

Assessing age restrictions – ongoing treatment (AGE-RST)

Assessing age restrictions – consultations, scans and miscellaneous
(AGE- RST)

Assessing time and history restrictions (RST TD and RST HIST)

Table 1: This process should only be performed by staff who have been
trained in the Claims Interpreter queue, Complex MOA queue or hold an
established patient claim skill tag. This table contains information for
processing claims for items 75045 and 75048.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Retrieve the patient’s history +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Key ‘NHSI,CARDNUMBER,PATIENT’ and |
|                                   | press [Enter]                     |
|                                   |                                   |
|                                   | Press [PF9].                      |
|                                   |                                   |
|                                   | The patient PIN displays.         |
|                                   |                                   |
|                                   | Note: check history by PIN in all |
|                                   | instances to ensure the complete  |
|                                   | claiming history is returned, not |
|                                   | just the history associated with  |
|                                   | 1 card.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for CLaCP items on the      |
|                                   | patient’s history + Read more ... |
|                                   |                                   |
|                                   | Key                               |
|                                   | ‘NH                               |
|                                   | SI,PIN,ITEM/75039-75048,SORT/DOS’ |
|                                   | and press [Enter].                |
|                                   |                                   |
|                                   | Identify the DOS of the initial 3 |
|                                   | months of treatment (item 75039   |
|                                   | or 75045).                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Determine whether the claim is    |
|                                   | payable + Read more ...           |
|                                   |                                   |
|                                   | Is the subsequent treatment DOS 3 |

|                                   |                                   |
|                                   | -   Yes, follow the error message |
|                                   |     to override the restriction   |

Assessing age restrictions – initial item for permanent dentition treatment (AGE-RST)

Table 2: this process should only be performed by staff who have been

established patient claim skill tag. This table is for claims for
initial 3 months of treatment service (item 75039 or 75045) where there
is an age restriction.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Has claimant attached letter of   |
|                                   | approval + Read more ...          |
|                                   |                                   |

|                                   | -   No, reject the claim          |
|                                   |     following the return message. |

|                                   |                                   |
|                                   | Request a restriction lift        |

|                                   | Helpdesk lift the restriction and |

|                                   | charge is correct and P to pay    |

Assessing age restrictions – ongoing treatment (AGE-RST)

Table 3: this process should only be performed by staff who have been
trained in the Claims Interpreter queue, Complex MOA queue or hold an
established patient claim skill tag. This table is for claims for
subsequent 3 months of treatment service (item 75042 or 75048) where
there is an age restriction.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Retrieve the patient’s history +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Key ‘NHSI,CARDNUMBER,PATIENT’ and |
|                                   | press [Enter].                    |
|                                   |                                   |
|                                   | Press [PF9].                      |
|                                   |                                   |
|                                   | The patient PIN displays.         |
|                                   |                                   |
|                                   | Note: check history by PIN in all |
|                                   | instances to ensure the complete  |
|                                   | claiming history is returned, not |
|                                   | just the history associated with  |
|                                   | 1 card.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for CLaCP items on the      |
|                                   | patient’s history + Read more ... |
|                                   |                                   |
|                                   | Key                               |
|                                   | ‘NH                               |
|                                   | SI,PIN,ITEM/75039-75048,SORT/DOS’ |
|                                   | and press [Enter]                 |
|                                   |                                   |
|                                   | Are there services on the         |
|                                   | patient’s history?                |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Reject he claim with RSN 101  |
|                                   |     and leave appropriate notes   |
|                                   |     in CDMS                       |
|                                   | -   Note: if no services are on   |
|                                   |     history the patient will need |
|                                   |     to submit a patient claim     |
|                                   |     with a letter from the health |
|                                   |     professional explaining why a |
|                                   |     course of treatment is being  |
|                                   |     started over the age of 28.   |
|                                   |     This will then need to be     |
|                                   |     assessed by Medicare          |
|                                   |     Assessing helpdesk            |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Check claim date of service       |
|                                   | (DOS) + Read more ...             |
|                                   |                                   |
|                                   | Is the DOS at least 3 months      |
|                                   | after the latest CLaCP treatment  |
|                                   | service (item 75039, 75042, 75045 |
|                                   | or 75048) on patient history?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, reject the claim using    |
|                                   |     the instruction in the return |
|                                   |     message. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 4                                 | Request a restriction lift + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Request a restriction lift        |
|                                   |                                   |
|                                   | When the Medicare Assessing       |
|                                   | Helpdesk lift the restriction and |
|                                   | notify the Service Officer, go to |
|                                   | Step 5.                           |
|                                   |                                   |
|                                   | Note: restriction lifts for CLaCP |
|                                   | claims must be only requested for |
|                                   | patients over 28 years old.       |
+-----------------------------------+-----------------------------------+
| 5                                 | Process the claim + Read more ... |
|                                   |                                   |
|                                   | Enter the claim ensuring the      |
|                                   | charge is correct and P to pay    |
|                                   | [Enter]. Procedure ends here.     |
+-----------------------------------+-----------------------------------+
