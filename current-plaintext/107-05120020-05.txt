Examples of circumstance changes that may create a debt for FTB customers

+-----------------------------------+-----------------------------------+
| Circumstance                      | Debt type                         |
+-----------------------------------+-----------------------------------+
| Child leaves care                 | Qualification debt:               |
|                                   |                                   |
|                                   | -   The customer is no longer     |
|                                   |     qualified for Family Tax      |
|                                   |     Benefit (FTB) for this child  |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| Change in shared care             | Qualification debt:               |
|                                   |                                   |
|                                   | -   The customer is no longer     |
|                                   |     qualifies for the previously  |
|                                   |     paid rate of FTB for this     |
|                                   |     child                         |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| Customer is no longer renting     | Qualification debt:               |
|                                   |                                   |
|                                   | -   The customer is no longer     |
|                                   |     qualified for Rent Assistance |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| Customer's accommodation amount   | Qualification debt:               |
| has decreased                     |                                   |
|                                   | -   The customer is no longer     |
|                                   |     qualified for the previously  |
|                                   |     paid rate of Rent Assistance  |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| Customer becomes partnered and    | Qualification debt:               |
| provides a partner income         |                                   |
| estimate                          | -   The customer is no longer     |
|                                   |     qualified for the previously  |
|                                   |     paid rate of FTB              |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| FTB school-aged child is no       | Qualification debt:               |
| longer studying                   |                                   |
|                                   | -   The customer is not entitled  |
|                                   |     to FTB for this child         |
|                                   | -   Raise the debt immediately    |
|                                   | -   Waive debts below $50 on the  |
|                                   |     Assessment Consequences (ASC) |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| Maintenance income has increased  | Income debt:                      |
|                                   |                                   |
|                                   | Debts resulting from maintenance  |
|                                   | income changes are automatically  |
|                                   | included in:                      |
|                                   |                                   |
|                                   | -   Mandatory Continuous          |
|                                   |     Adjustment (MCA)              |
|                                   |     calculations, and             |
|                                   | -   reconciliation                |
|                                   |                                   |
|                                   | Where a maintenance change is not |
|                                   | automatically included in MCA, a  |
|                                   | Service Officer is to key 'INV'   |
|                                   | at the ASC screen.                |
+-----------------------------------+-----------------------------------+
| Income changes for Customer or    | Income debt: Debts that result    |
| Partner                           | from a change of customer or      |
|                                   | partner income are included in:   |
|                                   |                                   |
|                                   | -   Mandatory Continuous          |
|                                   |     Adjustment (MCA)              |
|                                   |     calculations, and             |
|                                   | -   reconciliation                |
|                                   |                                   |
|                                   | Where an MCA does not include an  |
|                                   | income change, key 'INV' on the   |
|                                   | ASC screen.                       |
+-----------------------------------+-----------------------------------+
