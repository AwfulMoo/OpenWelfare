How to reverse an overpayment for VAP

-   An overpayment is reversed by SAPGUI when it has been raised
    incorrectly or documentation is outside the National Overpayments
    Procedures Manual agreed to by Medicare and Department of Veterans'
    Affairs (DVA).
-   Delegate approval from VAP Helpdesk must be obtained to reverse an
    overpayment

Table 3: This table contains information for VAP to reverse an
overpayment to a health practitioner or facility.

+-----------------------------------+-----------------------------------+

+-----------------------------------+-----------------------------------+

Requesting an EFT to replace a stale cheque for a health practitioner or facility

Table 4: This table contains the process VAP Service Officers and VAP
Overpayments Officers follow to request a manual cheque is issued to a
health practitioner or facility when an unpresented cheque has been
stopped.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Health practitioner or facility   |
|                                   | contacts VAP about an unpresented |
|                                   | or stopped Services Australia     |
|                                   | cheque + Read more ...            |
|                                   |                                   |
|                                   | VAP Service Officer will          |
|                                   | determine the status of the       |
|                                   | cheque.                           |
|                                   |                                   |
|                                   | In mainframe key the control      |
|                                   | line:                             |
|                                   |                                   |
|                                   | -   ‘BVEI’, followed by the       |
|                                   |     provider number (excluding    |
|                                   |     last letter)                  |
|                                   | -   select [Enter]                |
|                                   |                                   |
|                                   | Health practitioner or facility’s |
|                                   | cheque history will display       |
+-----------------------------------+-----------------------------------+
| 2                                 | What is the status of the         |
|                                   | cheque? + Read more ...           |
|                                   |                                   |
|                                   | Note: the Finance team stops an   |
|                                   | unpresented cheque more than 18   |
|                                   | months old and notifies           |
|                                   | eBusiness. Staff in eBusiness     |
|                                   | amend the cheque status in        |
|                                   | Mainframe to CANCEL and key STALE |
|                                   | STOPPED in the text field.        |
|                                   |                                   |
|                                   | A stale or cancelled cheque will  |
|                                   | display CANCEL in the status      |
|                                   | field.                            |
|                                   |                                   |
|                                   | -   In mainframe key ‘T’ in the D |
|                                   |     field                         |
|                                   | -   STALE STOPPED will display    |
+-----------------------------------+-----------------------------------+
| 3                                 | VAP Service Officer explains the  |
|                                   | process to the provider + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Confirm the health practitioner   |
|                                   | or facility’s current bank        |
|                                   | account details.                  |
|                                   |                                   |
|                                   | Key ‘REMM’ into mainframe         |
|                                   |                                   |
|                                   | Select option 6 Provider payment  |
|                                   | details (EFT enquiry)             |
|                                   |                                   |
|                                   | Enter the provider number and     |
|                                   | select Enter                      |
|                                   |                                   |
|                                   | -   Note: if bank details are     |
|                                   |     incorrect the health          |
|                                   |     practitioner or facility      |
|                                   |     needs to complete a           |

|                                   |     Funds Transfer payments form  |

|                                   |                                   |

|                                   | facility that a request for an    |

|                                   | days.                             |
+-----------------------------------+-----------------------------------+
| 4                                 | Create a manual statement + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In mainframe key ‘N‘ in the D     |
|                                   | field                             |
|                                   |                                   |
|                                   | The claim ID/s will display.      |
|                                   |                                   |
|                                   | Key the control line              |
|                                   | ‘BVCI,VTAR,Cheque No‘ and select  |
|                                   | [Enter]                           |
|                                   |                                   |
|                                   | Create a VPSI manual statement    |
|                                   | (see Step 4 in the How to         |
|                                   | initiate an overpayment for VAP   |

|                                   |                                   |

|                                   | Electronic Funds Transfer (EFT)   |
|                                   | to the appropriate VAP            |
|                                   | Overpayments Officer in the       |
|                                   | appropriate site. See Source      |
|                                   | office codes for Veterans'        |
|                                   | Affairs Processing (VAP) for      |
|                                   | source office codes. Contacts for |
|                                   | Veterans’ Affairs Processing      |
|                                   | (VAP) contains email addresses    |
|                                   | for VAP Overpayments Officers in  |
|                                   | each state.                       |
|                                   |                                   |
|                                   | VAP Service Officers, procedure   |
|                                   | ends here.                        |
|                                   |                                   |
|                                   | VAP Overpayments Officers, go to  |
|                                   | Step 5                            |
+-----------------------------------+-----------------------------------+
| 5                                 | VAP Overpayments Officer forwards |
|                                   | the request to Treasury + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Retrieve manual cheque request    |
|                                   | and statement (if available) from |
|                                   | digital workflow or site          |
|                                   | overpayments tray.                |
|                                   |                                   |
|                                   | Verify that all the information   |
|                                   | included on the form is correct.  |
|                                   |                                   |
|                                   | Forward all documentation to the  |
|                                   | VAP Helpdesk.                     |

Quality control for VAP recovery of overpayments

Table 5: This table describes the process used by staff undertaking
quality control checks of VAP overpayment recovery actions.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Decide the volume of quality      |
|                                   | control required + Read more ...  |
|                                   |                                   |
|                                   | To decide the volume of quality   |
|                                   | control required:                 |
|                                   |                                   |
|                                   | -   Determine the total volume of |
|                                   |     overpayments actioned for the |
|                                   |     preceding month               |
|                                   | -   Calculate 5% of the total     |
|                                   |     volume of overpayments        |
|                                   |     actioned. Note: fractions of  |
|                                   |     a number should be rounded up |
|                                   |     to the nearest whole number   |
+-----------------------------------+-----------------------------------+
| 2                                 | Select overpayments for quality   |
|                                   | checking + Read more ...          |
|                                   |                                   |
|                                   | Select overpayments for quality   |
|                                   | checking from:                    |
|                                   |                                   |
|                                   | -   completed overpayments        |
|                                   | -   overpayments referred to the  |
|                                   |     Department of Veterans'       |
|                                   |     Affairs (DVA)                 |
|                                   | -   overpayments that have been   |
|                                   |     reversed, and                 |
|                                   | -   outstanding overpayments      |
+-----------------------------------+-----------------------------------+
| 3                                 | Allocate quality checking + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Quality Controller must be an |
|                                   | independent Overpayments Officer  |
|                                   | who has not been involved in the  |
|                                   | recovery process.                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Conduct and record quality        |
|                                   | control checks + Read more ...    |
|                                   |                                   |
|                                   | Open the Quality Control          |
|                                   | Checklist on the Resources page.  |
|                                   |                                   |
|                                   | Using the checklist, record the   |
|                                   | action taken on the recovery of   |
|                                   | the overpayment. Ensure:          |
|                                   |                                   |
|                                   | -   all documentation is correct  |
|                                   |     and attached to the           |
|                                   |     overpayment                   |
|                                   | -   all KPIs as stipulated in the |
|                                   |     Recovery of VAP Overpayments  |
|                                   |     Policy have been adhered to   |
|                                   | -   health practitioner or        |
|                                   |     facility details are verified |
|                                   |     against the DVA Provider File |
|                                   |     (VPUI), and                   |
|                                   | -   all details have been         |
|                                   |     correctly entered on SAP GUI  |
+-----------------------------------+-----------------------------------+
| 5                                 | Notify actioning officer of       |
|                                   | inaccuracies or errors + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Any inaccuracies or errors are to |
|                                   | be bought to the attention of the |
|                                   | actioning officer and training is |
|                                   | to be provided on the correct     |
|                                   | procedure.                        |
+-----------------------------------+-----------------------------------+
| 6                                 | Store quality control findings +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Findings are kept within each     |
|                                   | site for auditing purposes. Store |
|                                   | the Quality Control Checklist     |
|                                   | used to conduct the quality check |
|                                   | in the site shared drive.         |
