| 1                                 | Flexible PPL declaration          |
|                                   |                                   |
|                                   | The Flexible PPL declaration must |
|                                   | be made by the customer each time |
|                                   | they claim or withdraw a Flexible |
|                                   | PPL day. It is available in the   |
|                                   | online service and guided         |
|                                   | procedure.                        |

|                                   |                                   |
|                                   | -   The information I have given  |
|                                   |     is correct                    |
|                                   | -   I need to advise Services     |
|                                   |     Australia of any changes to   |
|                                   |     my circumstances as soon as   |
|                                   |     the change occurs             |
|                                   | -   I have read and accept the    |
|                                   |     privacy statement             |
|                                   |                                   |
|                                   | For each day I have claimed       |
|                                   | Flexible Paid Parental Leave      |
|                                   | days, for each of these days:     |
|                                   |                                   |
|                                   | -   I did not/will not work       |
|                                   |     unless for an allowable       |
|                                   |     reason                        |
|                                   | -   I had/will have primary care  |
|                                   |     of the child                  |
|                                   | -   I expect to meet Parental     |
|                                   |     Leave Pay residence           |
|                                   |     requirements                  |
|                                   |                                   |
|                                   | I understand:                     |
|                                   |                                   |
|                                   | -   I need to advise Centrelink   |
|                                   |     of any changes to my          |
|                                   |     circumstances as soon as the  |

|                                   |     enquiries to make sure I      |
|                                   |     receive my correct            |
|                                   |     entitlement                   |

|                                   |     be paid back                  |

Claiming Flexible PPL days outside of claim limits (due to review of decision or system issue)

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Flexible PPL override -           |
|                                   | increasing connected Flexible PPL |
|                                   | days and claiming more than 2     |
|                                   | years after child’s               |
|                                   | birth/adoption                    |
|                                   |                                   |
|                                   | If a customer’s Flexible PPL      |
|                                   | needs to be reassessed following  |
|                                   | a review of decision, a Flexible  |
|                                   | PPL override can be recorded to   |
|                                   | allow:                            |
|                                   |                                   |
|                                   | -   an increase to the number of  |
|                                   |     connected Flexible PPL days   |
|                                   |     after the PPL period has      |
|                                   |     commenced                     |
|                                   | -   claim Flexible PPL days more  |
|                                   |     than 2 years after the        |
|                                   |     child’s birth or adoption.    |
|                                   |     Note: the claimed date/s must |
|                                   |     still be within 2 years of    |
|                                   |     the child’s birth or adoption |
|                                   |                                   |
|                                   | To request an override, refer to  |
|                                   | the Level 2 Policy Helpdesk.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Claiming Flexible PPL days more   |
|                                   | than 42 days in the past          |
|                                   |                                   |
|                                   | Customers can claim Flexible PPL  |
|                                   | for a date that is more than 42   |
|                                   | days in the past if:              |
|                                   |                                   |
|                                   | it can be seen (using DOCs on the |
|                                   | record and/or activity dates)     |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   the customer attempted to     |
|                                   |     claim or contacted about      |
|                                   |     claiming the Flexible PPL day |
|                                   |     within the 42 day allowable   |
|                                   |     claim timeframe, however      |
|                                   | -   the activity was not          |
|                                   |     finalised, due to system      |
|                                   |     issue or other reason, or     |
|                                   |                                   |
|                                   | the customer:                     |
|                                   |                                   |
|                                   | -   was granted PPL due to        |
|                                   |     meeting the extended work     |
|                                   |     test provision due to         |
|                                   |     coronavirus (COVID-19)        |
|                                   | -   received a COVID-19 Disaster  |
|                                   |     Payment (paid by Services     |
|                                   |     Australia on or after 3       |
|                                   |     June 2021) during their 10    |
|                                   |     month qualifying period       |
|                                   | -   Note: this shows on the Paid  |
|                                   |     Parental Leave Exemption      |
|                                   |     (P1EX) screen as ‘28D -       |
|                                   |     Backdating the start date'    |
|                                   |                                   |
|                                   | Note: before finalising a claim   |
|                                   | for a date more than 42 days in   |
|                                   | the past, tell the customer the   |
|                                   | risk of adverse impact on other   |
|                                   | payments/entitlements that were   |
|                                   | paid for the same period.         |
|                                   |                                   |
|                                   | To finalise a claim for a date    |
|                                   | that is outside this claiming     |
|                                   | limit:                            |
|                                   |                                   |
|                                   | -   when using the PPL            |
|                                   |     Circumstances Guided          |
|                                   |     Procedure to enter a date in  |
|                                   |     the Flexible PPL days (not    |
|                                   |     connected) section,           |
|                                   | -   the Reason: must be Date is   |
|                                   |     greater than 42 days in the   |
|                                   |     past                          |
|                                   |                                   |
|                                   | Once the activity is finalised,   |
|                                   | use Fast Note > Families >        |
|                                   | Updates > Change of Circs >       |
|                                   | DAP/PPL to record all details of  |
|                                   | the update, including how the     |
|                                   | customer was eligible for a date  |
|                                   | more than 42 days in the past.    |
