FAQ - COVID-19 General Practice Incentive payment

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Is it important which   |
|                                   | order the first and second-dose   |
|                                   | assessment services are billed?   |
|                                   |                                   |
|                                   | Answer: Yes. The first-dose       |
|                                   | assessment service must be billed |
|                                   | first.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Is there a different    |
|                                   | assessment service item number if |
|                                   | the patient does not get the      |
|                                   | vaccine?                          |
|                                   |                                   |
|                                   | Answer: No. The GP will bill      |
|                                   | either a first-dose or            |
|                                   | second-dose assessment service    |
|                                   | whether or not the vaccine dose   |
|                                   | is given.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Will the practice get a |
|                                   | payment if a GP is registered     |
|                                   | with the practice after the       |
|                                   | assessment services were billed?  |
|                                   |                                   |
|                                   | Answer: Yes, if the GP            |
|                                   | registration start date is before |
|                                   | the date of the first-dose        |
|                                   | assessment service.               |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Does the same GP have   |
|                                   | to bill the first and second-dose |
|                                   | assessment services for the       |
|                                   | practice to get a payment?        |
|                                   |                                   |
|                                   | Answer: No, but the following     |
|                                   | criteria apply:                   |
|                                   |                                   |
|                                   | -   the assessment services must  |
|                                   |     be billed at the same         |
|                                   |     practice                      |
|                                   | -   the GPs must be registered on |
|                                   |     the PIP profile with a start  |
|                                   |     date before the date of the   |
|                                   |     first service                 |
|                                   | -   the services must be billed   |
|                                   |     in the correct order          |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Will the practice get a |
|                                   | payment if the GP is now          |
|                                   | non-current on the PIP profile?   |
|                                   |                                   |
|                                   | Answer: Yes, if the GP was        |
|                                   | current at the date of the first  |
|                                   | and second-dose assessment        |
|                                   | services.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Will the practice get a |
|                                   | payment if they are approved for  |
|                                   | the PIP after the first-dose      |
|                                   | assessment service is billed?     |
|                                   |                                   |
|                                   | Answer: No. The practice must be  |
|                                   | approved for the PIP before the   |
|                                   | first-dose assessment service is  |
|                                   | billed.                           |
+-----------------------------------+-----------------------------------+
| 7                                 | Question: My practice did not get |
|                                   | a payment for some of the second  |
|                                   | dose assessment items provided    |
|                                   | this quarter. Can you check if    |
|                                   | the service/s was processed by    |
|                                   | the point-in-time date?           |
|                                   |                                   |
|                                   | Answer: In accordance with        |
|                                   | Services Australia’s privacy      |
|                                   | policy, we cannot investigate     |
|                                   | services provided to individuals  |
|                                   | to determine the eligibility or   |
|                                   | timing of payments. Where all the |
|                                   | requirements for payment have     |
|                                   | been met, payments will be made   |
|                                   | with the next PIP quarterly       |
|                                   | payment after successful          |
|                                   | processing of the second-dose     |
|                                   | assessment service.               |
+-----------------------------------+-----------------------------------+
| 8                                 | Question: My practice billed both |
|                                   | MBS items correctly. Why have I   |
|                                   | not received a payment for some   |
|                                   | patients?                         |
|                                   |                                   |
|                                   | Answer: Payments may be impacted  |
|                                   | by a patient’s decision to visit  |
|                                   | more than one general practice    |
|                                   | for vaccination services. In      |
|                                   | accordance with Services          |
|                                   | Australia’s privacy policy, we    |
|                                   | are unable to provide advice on   |
|                                   | the payment recipient of          |
|                                   | individual payments, or on how    |
|                                   | services provided to the patient  |
|                                   | by other practices may have       |
|                                   | impacted a practice meeting the   |
|                                   | requirements to receive the $10   |
|                                   | payment for that patient. Where   |
|                                   | all the requirements for payment  |
|                                   | have been met, payments will be   |
|                                   | made with the next PIP quarterly  |
|                                   | payment after successful          |
|                                   | processing of the second-dose     |
|                                   | assessment service.               |
+-----------------------------------+-----------------------------------+
