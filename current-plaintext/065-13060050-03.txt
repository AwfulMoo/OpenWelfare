Check that an ACER form is complete

Table 1: how Service Officers check that an Aged Care Entry Record
(ACER) form is complete. Note: do not make any mark on the original ACER
form unless it is indicated in the procedure.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check form is complete + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check if the AC021 form is        |
|                                   | complete:                         |
|                                   |                                   |
|                                   | -   Q1. Type of care              |
|                                   | -   Q2. Service name              |
|                                   | -   Q3. Service ID                |
|                                   | -   Q4. Care recipient details    |
|                                   | -   Q11. Date of entry            |
|                                   | -   Q17. Home Care Package level  |
|                                   | -   Declaration - Printed name    |
|                                   |     (person authorised by the     |
|                                   |     aged care service), position  |
|                                   |     held, phone number, signature |
|                                   |     and date                      |
|                                   |                                   |
|                                   | If any of the above information   |
|                                   | is missing, the AC021 form is     |
|                                   | incomplete.                       |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   An aged care service cannot   |
|                                   |     submit an ACER form unless    |
|                                   |     the care recipient has        |
|                                   |     entered into care and cannot  |
|                                   |     include a future commencement |
|                                   |     date                          |
|                                   | -   The form must be rejected and |
|                                   |     the aged care service is      |
|                                   |     notified (via email or phone) |
|                                   |     that future entry dates       |
|                                   |     cannot be entered on the      |
|                                   |     form, see Table 2             |
|                                   |                                   |
|                                   | Is the AC021 form complete?       |
|                                   |                                   |
|                                   | -   Yes, go to Table 3            |
|                                   | -   No, go to Table 2             |
+-----------------------------------+-----------------------------------+

Reject an ACER form

Table 2: how Service Officers reject an Aged Care Entry Record (ACER)
form in PaNDA.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Incomplete section + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Do not process an incomplete      |
|                                   | manual ACER form.                 |
|                                   |                                   |
|                                   | If the ACER form is incomplete:   |
|                                   |                                   |
|                                   | -   Enter a PaNDA note, include   |
|                                   |     the reason why the ACER form  |
|                                   |     cannot be processed           |
|                                   | -    Enter a care recipient note  |
|                                   |     in ACSP to reflect the        |
|                                   |     incomplete form               |
|                                   |                                   |
|                                   | Contact the service provider by:  |
|                                   |                                   |
|                                   | -   telephone to advise that the  |
|                                   |     ACER cannot be processed, or  |
|                                   | -   email, go to Step 2           |
+-----------------------------------+-----------------------------------+
| 2                                 | Send email to the aged care       |
|                                   | service + Read more ...           |
|                                   |                                   |
|                                   | Use the below template:           |
|                                   |                                   |
|                                   | Dear Care Manager,                |
|                                   |                                   |
|                                   | Thank you for the Aged Care Entry |
|                                   | Record (ACER) form received on    |
|                                   | <insert date of receipt> in       |
|                                   | regards to <insert care recipient |
|                                   | name and ID>.                     |
|                                   |                                   |
|                                   | We are unable to process your     |
|                                   | ACER form due to the following    |
|                                   | reason/s:                         |
|                                   |                                   |
|                                   | -   the form is incomplete as     |
|                                   |     missing information <insert   |
|                                   |     what information is missing>  |
|                                   |     or                            |
|                                   | -   the form has a future entry   |
|                                   |     date                          |
|                                   |                                   |
|                                   | As a matter of priority, please   |
|                                   | resubmit a completed form to      |
|                                   | aged.care                         |
|                                   | .liaison@servicesaustralia.gov.au |
|                                   |                                   |
|                                   | With thanks                       |
|                                   |                                   |
|                                   | Aged Care Payments                |

Process an ACER form for residential care

Table 3: how Service Officers process an Aged Care Entry Record (ACER)
form in the Aged Care Staff Portal (ACSP) for residential care. Note: do
not make any mark on the original ACER form unless it is indicated in
the procedure.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check completion and leave        |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Check the ACER form is complete,  |
|                                   | see Table 1.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for a care recipient       |
|                                   | record + Read more ...            |
|                                   |                                   |
|                                   | Search for a care recipient       |
|                                   | record on the ACSP before         |
|                                   | entering the ACER to ensure that: |
|                                   |                                   |
|                                   | -   the care recipient details    |
|                                   |     are recorded, and             |
|                                   | -   there is a valid National     |
|                                   |     Screening and Assessment Form |
|                                   |     (NSAF) with approval for      |
|                                   |     permanent residential care    |
|                                   |                                   |
|                                   | To search for:                    |
|                                   |                                   |
|                                   | -    a care recipient record, see |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal to search for a care   |
|                                   |     recipient – (Care Recipient   |
|                                   |     in Context (CRiC)) table      |
|                                   | -   a valid NSAF with approval    |
|                                   |     for permanent residential     |
|                                   |     care, see Step 2 in the View  |
|                                   |     and process NSAFs in the Aged |
|                                   |     Care Staff Portal (Care       |
|                                   |     Recipient in Context (CRiC))  |
|                                   |     table                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Locate care recipient record +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Has the care recipient record     |
|                                   | been located?                     |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   the care recipient record has |
|                                   |     been located with the         |
|                                   |     appropriate care approval     |
|                                   | -   go to Step 4                  |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   contact the aged care service |
|                                   |     by telephone or email to      |
|                                   |     advise there is no care       |
|                                   |     recipient record or the care  |
|                                   |     recipient does not have a     |
|                                   |     valid NSAF in the system and  |
|                                   |     reject the form, see Table 2  |
|                                   | -   Note: if the ACER form        |
|                                   |     includes inaccurate           |
|                                   |     information, reject the ACER  |
|                                   |     form and ask the aged care    |
|                                   |     service to resend the form,   |
|                                   |     see Table 2                   |
|                                   | -   mark the ACER form as         |
|                                   |     complete in PaNDA and add an  |
|                                   |     appropriate PaNDA note, see   |
|                                   |     Work Optimiser for staff for  |
|                                   |     details                       |
|                                   | -   procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Check for existing entry event +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Check the care recipient’s    |
|                                   |     record for an existing entry  |
|                                   |     event. See Step 9 in the      |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal to search for a care   |
|                                   |     recipient – (Care Recipient   |
|                                   |     in Context (CRiC)) table      |
|                                   |                                   |
|                                   | If the care recipient has an      |
|                                   | existing entry event with another |
|                                   | aged care service, the ACSP will  |
|                                   | automatically depart the care     |
|                                   | recipient from the existing aged  |
|                                   | care service.                     |
|                                   |                                   |
|                                   | The ACSP uses the date of entry   |
|                                   | and applies it to the new aged    |
|                                   | care service as recorded on the   |
|                                   | ACER form after the form has been |
|                                   | registered.                       |
+-----------------------------------+-----------------------------------+
| 5                                 | Enter new entry event + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Enter the new entry event in  |
|                                   |     ACSP for the new aged care    |
|                                   |     service. For more             |
|                                   |     information, see Registering  |
|                                   |     events in the Aged Care Staff |
|                                   |     Portal (Service in Context)   |
|                                   |     table                         |
+-----------------------------------+-----------------------------------+

Action award or settlement indicator on ACER form

Table 4: actions when a care recipient is declared as compensable on the
Aged Care Entry Record (ACER) form. Service Australia’s role is to
notify the Department of Health and Aged Care who will assess the
compensation details for the care recipient and make a decision to apply
a compensation payment reduction.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Process ACER form + Read more ... |
|                                   |                                   |
|                                   | Contact the aged care service to  |
|                                   | confirm that Question 12 Award or |
|                                   | Settlement ticked on the ACER     |
|                                   | form is:                          |
|                                   |                                   |
|                                   | -   correct, or                   |
|                                   | -   to avoid unnecessary stoppage |
|                                   |     of eligible subsidies         |
|                                   |                                   |
|                                   | Once confirmed the tick on the    |
|                                   | form is correct:                  |
|                                   |                                   |
|                                   | -   process the ACER form, and    |
|                                   | -   escalate this to the Aged     |
|                                   |     Care Programmes Branch        |
|                                   |                                   |
|                                   | The Aged Care Programmes Branch   |
|                                   | will:                             |
|                                   |                                   |
|                                   | -   code in the compensation from |
|                                   |     the ACER form into the Aged   |
|                                   |     Care Staff Portal (ACSP), see |
|                                   |     Table 3                       |
|                                   | -   notify the Department of      |
|                                   |     Health and Aged Care by email |
|                                   |     that the care recipient has   |
|                                   |     declared as compensable on    |
|                                   |     the ACER form (Question 12    |
|                                   |     Award or Settlement)          |
+-----------------------------------+-----------------------------------+
| 2                                 | Email and add care recipient      |
|                                   | notes + Read more ...             |
|                                   |                                   |
|                                   | Send an email to Department of    |
|                                   | Health and Aged Care Compensation |
|                                   | enquiries team                    |
|                                   |                                   |
|                                   | Include the below information in  |
|                                   | the email:                        |
|                                   |                                   |
|                                   | service name and ID               |
|                                   |                                   |
|                                   | care recipient full name and ID   |
|                                   |                                   |
|                                   | date of entry                     |
|                                   |                                   |
|                                   | liability type:                   |
|                                   |                                   |
|                                   | -   Workers' Compensation         |
|                                   | -   Third Party Insurance         |
|                                   | -   Common Law Settlement         |
|                                   |                                   |
|                                   | Add a care recipient note         |
|                                   | indicating compensation award or  |
|                                   | settlement claim and date         |
|                                   | referred to the Compensation      |
|                                   | Recovery Team. See Using the Aged |
|                                   | Care Staff Portal to manage care  |
|                                   | recipient notes (Care Recipient   |
|                                   | in Context (CRiC)) table.         |
+-----------------------------------+-----------------------------------+
| 3                                 | Assessment and advice + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Once the Department of Health and |
|                                   | Aged Care has received advice     |
|                                   | from Services Australia, the      |
|                                   | Department of Health and Aged     |
|                                   | Care will assess compensation     |
|                                   | details for the care recipient    |
|                                   | and make a decision to apply a    |
|                                   | compensation payment reduction.   |
|                                   |                                   |
|                                   | The Department of Health and Aged |
|                                   | Care Compensation Recovery Team   |
|                                   | will:                             |
|                                   |                                   |
|                                   | -   apply the compensation        |
|                                   |     payment reduction when        |
|                                   |     determined the compensation   |
|                                   |     payment includes an aged care |
|                                   |     component                     |
|                                   | -   manage the compensation in    |
|                                   |     the Aged Care Staff Portal    |
|                                   |     (ACSP) as appropriate. See    |
|                                   |     Step 3 in the Using the Aged  |
|                                   |     Care Staff Portal for         |
|                                   |     Supplments and subsidies menu |
|                                   |     tab – (Care Recipient in      |
|                                   |     Context (CRiC)) table         |
|                                   | -   advise the aged care service  |
|                                   |     and care recipient of the     |
|                                   |     outcome in writing            |
+-----------------------------------+-----------------------------------+
