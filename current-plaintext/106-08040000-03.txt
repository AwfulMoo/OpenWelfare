Table 1: This table describes specialist information relating to the
international data exchange program and auto-indexation of foreign
pensions.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Identifying records updated via   |
|                                   | the data exchange program or      |
|                                   | auto-indexation + Read more ...   |
|                                   |                                   |
|                                   | Customers who have had their      |
|                                   | foreign pensions updated through  |
|                                   | the bulk data exchange process or |
|                                   | auto-indexation will have Data    |
|                                   | Exchange (DEX) or Consumer Price  |
|                                   | Index (CPI) appear in the         |
|                                   | Channel: field on the Foreign     |
|                                   | Pension Details (FPD) screen.     |
|                                   |                                   |
|                                   | Note: channel code 'RVL' will     |
|                                   | appear after each monthly         |
|                                   | exchange rate revaluation; it may |
|                                   | be necessary to check older dates |
|                                   | of event to determine whether DEX |
|                                   | or CPI updates have been made in  |
|                                   | the past.                         |
|                                   |                                   |
|                                   | These codes were introduced to    |
|                                   | protect and identify foreign      |
|                                   | pension rate information obtained |
|                                   | direct from the paying authority. |
|                                   | Only authorised users (for        |
|                                   | example, International Services   |
|                                   | (CIS)) can change amounts after   |
|                                   | these codes have been applied.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Date of Effect for                |
|                                   | reassessments + Read more ...     |
|                                   |                                   |
|                                   | Auto-indexation increases (CPI)   |
|                                   | are applied from the 1st of the   |
|                                   | applicable month. These updates   |
|                                   | are applied prior to the monthly  |
|                                   | exchange rate revaluation         |
|                                   | process. The exception is United  |
|                                   | Kingdom (UK) increases which      |
|                                   | apply from early April each year. |
|                                   |                                   |
|                                   | The DEX update processes reassess |
|                                   | entitlements using a Date of      |
|                                   | Receipt of 'T' for today. This    |
|                                   | means no arrears or debts will be |
|                                   | automatically generated for DEX   |
|                                   | updates.                          |
|                                   |                                   |
|                                   | The 'favourable determination'    |
|                                   | rules under social security law   |
|                                   | apply in respect of Australian    |
|                                   | payments that increase whilst     |
|                                   | 'unfavourable determination'      |
|                                   | rules will apply when potential   |
|                                   | debts are investigated following  |
|                                   | DEX updates.                      |
|                                   |                                   |
|                                   | International Services (CIS) is   |
|                                   | responsible for investigating and |
|                                   | raising any debts that arise from |
|                                   | DEX updates.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Reviews and Appeals + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The normal review and appeals     |
|                                   | system applies.                   |
|                                   |                                   |
|                                   | Customer initiated reviews        |
|                                   | relating to DEX updates are the   |
|                                   | responsibility of the customer's  |
|                                   | local service centre.             |
|                                   |                                   |
|                                   | Information received via data     |
|                                   | exchange or a rate updated via    |
|                                   | auto-indexation is considered to  |
|                                   | be accurate. Customers who advise |
|                                   | that rates of foreign pension     |
|                                   | that are being maintained are     |
|                                   | incorrect should provide suitable |
|                                   | proof from the paying authority   |
|                                   | such as a pension statement,      |
|                                   | letter or certificate. A bank     |
|                                   | statement is not sufficient       |
|                                   | evidence of the gross rate of     |
|                                   | pension in the original currency. |
|                                   |                                   |
|                                   | However, only authorised users,   |
|                                   | for example International         |
|                                   | Services (CIS), can change        |
|                                   | amounts after DEX or CPI updates  |
|                                   | have been applied. If a customer  |
|                                   | supplies new information from the |
|                                   | paying authority about their rate |
|                                   | of foreign pension, details must  |
|                                   | be forwarded to CIS.              |
|                                   |                                   |
|                                   | CIS is responsible for appeals    |
|                                   | relating to any debts that are    |
|                                   | raised following DEX updates.     |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to CIS contact details.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Debt recovery + Read more ...     |
|                                   |                                   |
|                                   | Standard withholdings for debts   |
|                                   | raised because of DEX updates are |
|                                   | set at $50 per fortnight.         |
|                                   |                                   |
|                                   | Customers may choose to negotiate |
|                                   | repayments in accordance with     |
|                                   | normal debt recovery procedures.  |
