Resolve excess residential care recipient social leave

Table 2: this table describes how Service Officers resolve the excess
use of residential care recipient's social leave.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Non claimable social leave -      |
|                                   | check claim channel + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | A care recipient is entitled for  |
|                                   | 52 days claimable social leave in |
|                                   | a financial year. If a care       |
|                                   | recipient exceeds this amount     |
|                                   | within a financial year, these    |
|                                   | days are known as non-claimable   |
|                                   | social leave days.                |
|                                   |                                   |
|                                   | Exceeding claimable social leave  |
|                                   | days will prevent the monthly     |
|                                   | claim from being finalised. The   |
|                                   | process to resolve non-claimable  |
|                                   | social leave days is different    |
|                                   | depending on the claim channel    |
|                                   | (online or manual).               |
|                                   |                                   |
|                                   | Is this a manual claim via PaNDA  |
|                                   | (email, fax or post)?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 2                                 | Manual claims - identify affected |
|                                   | care recipients + Read more ...   |
|                                   |                                   |
|                                   | A manual claim will not finalise  |
|                                   | in the ACSP where the claim       |
|                                   | contains care recipients with     |
|                                   | non-claimable social leave days.  |
|                                   | An error message is displayed     |
|                                   | advising this issue.              |
|                                   |                                   |
|                                   | Service Officers will need to     |
|                                   | identify the affected care        |
|                                   | recipients and resolve the issue  |
|                                   | prior to finalising the claim     |
|                                   | again.                            |
|                                   |                                   |
|                                   | To identify affected care         |
|                                   | recipients with the non-claimable |
|                                   | social leave days, Service        |
|                                   | Officers will need to cross check |
|                                   | all care recipient’s social leave |
|                                   | balance in the current claim      |
|                                   | against a care recipient’s leave  |
|                                   | balance.                          |
|                                   |                                   |
|                                   | See the Using the Aged Care Staff |
|                                   | Portal to view care recipients’   |
|                                   | leave balances (Service in        |
|                                   | Context) table for the process to |
|                                   | view a care recipient’s leave     |
|                                   | balance.                          |
|                                   |                                   |
|                                   | Has the Service Officer           |
|                                   | identified the affected care      |
|                                   | recipients with social leave      |
|                                   | days?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, Service Officers must     |
|                                   |     repeat this process before    |
|                                   |     continuing                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Register the social leave + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Service Officers must record the  |
|                                   | leave event in the ACSP. See Step |
|                                   | 7 in the Registering events in    |
|                                   | the Aged Care Staff Portal        |
|                                   | (Service in Context) table.       |
|                                   |                                   |
|                                   | Has the Service Officer completed |
|                                   | this process for all affected     |
|                                   | care recipients?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, Service Officers must     |
|                                   |     repeat this step before       |
|                                   |     continuing                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Add a care recipient note + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Add a care recipient note to the  |
|                                   | claim notifying the approved aged |
|                                   | care service that the relevant    |
|                                   | care recipient has exceeded       |
|                                   | claimable social leave of 'nn     |
|                                   | days' and the balance of the      |
|                                   | social leave has been processed   |
|                                   | as non-claimable.                 |
|                                   |                                   |
|                                   | To add a care recipient note, see |
|                                   | Step 3 in the Using the Aged Care |
|                                   | Staff Portal to manage care       |
|                                   | recipient notes (Care Recipient   |
|                                   | in Context (CRiC)) table          |
|                                   |                                   |
|                                   | Has the Service Officer completed |
|                                   | this process for all affected     |
|                                   | care recipients?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, Service Officers must     |
|                                   |     repeat this step before       |
|                                   |     continuing                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Add a payment statement note +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Add a payment statement note to   |
|                                   | the claim. The details are as per |
|                                   | the care recipient note, that the |
|                                   | relevant care recipient has       |
|                                   | exceeded claimable social leave   |
|                                   | of 'nn days' and the balance of   |
|                                   | the social leave has been         |
|                                   | processed as non-claimable.       |
|                                   |                                   |
|                                   | To add a payment statement note   |
|                                   | see table Maintain payment        |
|                                   | statement notes in ACMPS.         |
|                                   |                                   |
|                                   | Has the Service Officer completed |
|                                   | this process for all affected     |
|                                   | care recipients?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, Service Officers must     |
|                                   |     repeat this step before       |
|                                   |     continuing                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Submit + Read more ...            |
|                                   |                                   |
|                                   | Submit the monthly claim for      |
|                                   | finalisation and approval.        |
|                                   |                                   |
|                                   | To finalise the monthly claim,    |
|                                   | see Step 5 of the Using the Aged  |
|                                   | Care Staff Portal at the service  |
|                                   | level (Service in Context) table. |
|                                   |                                   |
|                                   | To approve the monthly claim, see |
|                                   | Step 8 of the Using the Aged Care |
|                                   | Staff Portal home page table      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Online claims + Read more ...     |
|                                   |                                   |
|                                   | Aged care services are unable to  |
|                                   | finalise their online claims via  |
|                                   | the Aged Care Provider Portal if  |
|                                   | the claim contains care           |
|                                   | recipients with non-claimable     |
|                                   | social leave days. An error       |
|                                   | message will display advising     |
|                                   | this issue.                       |
|                                   |                                   |
|                                   | Aged care services must identify  |
|                                   | the affected care recipients and  |
|                                   | resolve the issue through their   |
|                                   | current process. The aged care    |
|                                   | service can resubmit the claim    |
|                                   | for finalisation after they have  |
|                                   | resolved the social leave issue.  |

|                                   |     left hand side menu           |
|                                   | -   The Claim & ACCR Search       |

|                                   | -   Select ACCR Search. The ACCR  |
