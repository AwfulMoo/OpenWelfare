Assessing the Claim for ABSTUDY Schooling A

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Initial checking + Read more ...  |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Ensure that the ABSTUDY Schooling |
|                                   | A claim has been checked.         |
|                                   |                                   |
|                                   | It is important to note that, in  |
|                                   | the event of a change of          |
|                                   | applicant, the Schooling A record |
|                                   | must always be cancelled before   |
|                                   | the new applicant's details are   |
|                                   | updated, to ensure that a         |
|                                   | cancellation letter is generated  |
|                                   | to the applicant who is no longer |
|                                   | the student's carer. A new claim  |
|                                   | can then be processed with the    |
|                                   | appropriate changes and new       |
|                                   | applicant details.                |
|                                   |                                   |
|                                   | Has the applicant lodged a claim  |
|                                   | through the ABSTUDY Claiming      |
|                                   | Tool?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, applicant has lodged a    |
|                                   |     paper claim. Go to Step 6     |
+-----------------------------------+-----------------------------------+
| 2                                 | ABSTUDY Claiming Tool + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Applicants completing a claim     |
|                                   | through the ABSTUDY Claiming Tool |
|                                   | can be offered a verbal           |
|                                   | declaration. Applicants who do    |
|                                   | not accept or have not been       |
|                                   | offered a verbal declaration are  |
|                                   | required to return a signed       |
|                                   | Customer Declaration Form (CDF)   |
|                                   | unless the applicant is receiving |
|                                   | Family Tax Benefit (FTB) for that |
|                                   | child, and:                       |
|                                   |                                   |
|                                   | -   the applicant or applicant's  |
|                                   |     partner has already           |
|                                   |     identified as an Australian   |
|                                   |     Aboriginal or Torres Strait   |
|                                   |     Islander person, or           |
|                                   | -   an ABSTUDY claim has          |
|                                   |     previously been lodged        |
|                                   |     indicating that the student   |
|                                   |     is an Australian Aboriginal   |
|                                   |     or Torres Strait Islander     |
|                                   |     person                        |
|                                   |                                   |
|                                   | An applicant's acceptance to a    |
|                                   | verbal declaration is recorded in |
|                                   | the Verbal Declaration DOC, and   |
|                                   | in the Progress of ABSTUDY claim  |
|                                   | DOC.                              |
|                                   |                                   |
|                                   | Note: on 28 November 2016, a      |
|                                   | verbal declaration replaced the   |
|                                   | requirement for the applicant to  |
|                                   | return a signed CDF. All          |
|                                   | applicants who completed a claim  |
|                                   | through the ABSTUDY Claiming Tool |
|                                   | prior to 28 November 2016 are     |
|                                   | still required to return their    |
|                                   | signed CDF unless the applicant   |
|                                   | is receiving FTB for that child,  |
|                                   | and:                              |
|                                   |                                   |
|                                   | -   the applicant or applicant's  |
|                                   |     partner has already           |
|                                   |     identified as an Australian   |
|                                   |     Aboriginal or Torres Strait   |
|                                   |     Islander person, or           |
|                                   | -   an ABSTUDY claim has          |
|                                   |     previously been lodged        |
|                                   |     indicating that the student   |
|                                   |     is an Australian Aboriginal   |
|                                   |     or Torres Strait Islander     |
|                                   |     person                        |
|                                   |                                   |
|                                   | Go to Step 3.                     |

| 3                                 | Checking the new claim (NCL)      |
|                                   | activity + Read more ...          |
|                                   |                                   |
|                                   | The ABSTUDY NCL activity must be  |

|                                   | Source is recorded and a Claim    |
|                                   | Type and verbal declaration       |
|                                   | keyword has been added.           |
|                                   |                                   |

|                                   | already been indexed              |
|                                   |                                   |

|                                   |                                   |
|                                   | -   Select the started ABSTUDY    |
|                                   |     NCL activity from the         |
|                                   |     Activity List (AL) screen     |
|                                   | -   Check the Source is recorded  |
|                                   |     as 'SCT'. If an incorrect     |
|                                   |     Source is recorded, the NCL   |
|                                   |     activity will need to be      |
|                                   |     cancelled and the claim will  |
|                                   |     need to be reindexed using    |
|                                   |     the correct Source of 'SCT'   |
|                                   |                                   |

|                                   |                                   |

|                                   | screen, select Keywords           |

|                                   |     accepted a verbal declaration |

|                                   |     supporting documents before   |
|                                   |     their ABSTUDY claim can be    |
|                                   |     assessed                      |
|                                   | -   'VERBNO' - if the applicant   |
|                                   |     did not accept a verbal       |
|                                   |     declaration                   |
|                                   |                                   |
|                                   | If the 'ABTNCL' claim type and    |
|                                   | verbal declaration keywords have  |
|                                   | not been added:                   |
|                                   |                                   |
|                                   | Select the ABSTUDY NCL activity   |
|                                   | from the AL screen                |
|                                   |                                   |
|                                   | From the Selection Options        |
|                                   | screen, select (O) transfer the   |
|                                   | activity to another site/region   |
|                                   |                                   |
|                                   | Key the 'ABTNCL' claim type       |
|                                   | keyword                           |
|                                   |                                   |
|                                   | Key the appropriate verbal        |
|                                   | declaration keyword:              |
|                                   |                                   |
|                                   | -   'VERBNIL' - if the applicant  |
|                                   |     accepted a verbal declaration |
|                                   |     and was not required to       |
|                                   |     provide any supporting        |
|                                   |     documents before their        |
|                                   |     ABSTUDY claim can be assessed |
|                                   | -   'VERBDOC' - if the applicant  |
|                                   |     accepted a verbal declaration |
|                                   |     and was required to provide   |
|                                   |     supporting documents before   |
|                                   |     their ABSTUDY claim can be    |
|                                   |     assessed                      |
|                                   | -   'VERBNO' - if the applicant   |
|                                   |     did not accept a verbal       |
|                                   |     declaration                   |
|                                   |                                   |
|                                   | If no ABSTUDY NCL activity is on  |
|                                   | the AL screen                     |

|                                   | Indexing, re-indexing and         |
|                                   | cancelling claim activities.      |
|                                   |                                   |
|                                   | Note: it is most likely that the  |
|                                   | student will already be a Family  |
|                                   | Tax Benefit (FTB) child. This     |
|                                   | Customer Reference Number (CRN)   |
|                                   | should be picked up and used in   |
|                                   | the indexing process.             |
|                                   |                                   |
|                                   | Select the ABSTUDY NCL activity   |
|                                   | from the AL screen.               |
|                                   |                                   |
|                                   | From the Selection Options        |
|                                   | screen, select (O) transfer the   |
|                                   | activity to another site/region   |
|                                   |                                   |
|                                   | Key the 'ABTNCL' claim type       |
|                                   | keyword                           |
|                                   |                                   |
|                                   | Key the appropriate verbal        |
|                                   | declaration keyword:              |
|                                   |                                   |
|                                   | -   'VERBNIL' - if the applicant  |
|                                   |     accepted a verbal declaration |
|                                   |     and was not required to       |
|                                   |     provide any supporting        |
|                                   |     documents before their        |
|                                   |     ABSTUDY claim can be assessed |

| 9                                 | Student or applicant not          |
|                                   | eligible + Read more ...          |
|                                   |                                   |
|                                   | Depending on the reason for the   |
|                                   | rejection, the ABSTUDY Schooling  |
|                                   | A claim may reject automatically  |
|                                   | (following the updating of        |
|                                   | information on the student's      |
|                                   | record for example, student       |
|                                   | attending primary school and is   |
|                                   | under 14 years of age) or may     |
|                                   | need to be rejected manually. See |
|                                   | Codes for ABSTUDY cancellation,   |
|                                   | suspension and rejection for a    |
|                                   | list of which rejection codes are |
|                                   | automatic and which are manual.   |
|                                   |                                   |
|                                   | Prior to rejection, the           |
|                                   | appropriate screens need to be    |
|                                   | updated according to the          |
|                                   | information provided in the Claim |
|                                   | for Schooling A.                  |
|                                   |                                   |
|                                   | Select the started ABSTUDY new    |
|                                   | claim activity from the Activity  |
|                                   | List (AL) screen in the student's |
|                                   | record.                           |
|                                   |                                   |
|                                   | Go to the ABSTUDY Task Selector   |
|                                   | (EYTS) screen and update the      |
|                                   | appropriate screens.              |
|                                   |                                   |
|                                   | When the appropriate screens have |
|                                   | been updated, go to the           |
|                                   | Assessment Results (AR) screen.   |
|                                   |                                   |
|                                   | Has the claim rejected            |
|                                   | automatically on the AR screen?   |
|                                   |                                   |
|                                   | -   Yes, finalise the activity    |
|                                   |     and record details on a DOC.  |
|                                   |     Contact the applicant and     |
|                                   |     provide an explanation of the |
|                                   |     decision. See Advising        |
|                                   |     verbally of an unfavourable   |
|                                   |     decision                      |
|                                   | -   No, the claim will need to be |
|                                   |     rejected manually. See        |
|                                   |     Rejecting an ABSTUDY claim    |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   | student to the applicant          |
|                                   |                                   |

|                                   |     new link start date           |

|                                   |     link correct applicant        |
|                                   |                                   |

|                                   | the applicant's record to         |

|                                   |                                   |
