This page contains information on assessing claims for emergency
payments from customers affected by the NSW Bushfires (Sep 2019).
Emergency payments include Australian Government Disaster Recovery
Payment (AGDRP), Disaster Recovery Allowance (DRA) and their associated
ex-gratia payments.

|                                   | reasonably close to the major     |

|                                   | will assist with the assessment   |

|                                   |     immediately next to your      |

|                                   |     kilometres of your home?      |
|                                   | -   Was the bushfire within 10    |
|                                   |     kilometres of your home?      |
|                                   |                                   |
|                                   | For customers with a home outside |
|                                   | the 10km radius the following     |
|                                   | questions may help to determine   |

|                                   | result of the designated          |
|                                   | bushfire:                         |

|                                   |     distance away from the fire   |

|                                   |     circumstances that led to the |
|                                   |     interior of your home         |
|                                   |     experiencing smoke and/or ash |
|                                   |     damage as a direct result of  |
|                                   |     this major disaster?          |
|                                   | -   Have you claimed insurance    |
|                                   |     for the damage and been       |
|                                   |     issued with a claim number?   |
|                                   | -   How would you describe the    |
|                                   |     circumstances that led to     |
|                                   |     smoke and/or ash damage to    |

AGDRP and ex-gratia DRP claims

For staff processing Australian Government Disaster Recovery Payment
(AGDRP) and ex-gratia Disaster Recovery Payment (DRP) claims only.

Table 3: This table describes the steps to assess eligibility for
AGDRP/ex-gratia DRP for customers affected by the NSW Bushfires (Sep
2019). Service Officers should not process a claim unless they have
completed formal training.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Disaster determined and basic     |
|                                   | eligibility + Read more ...       |
|                                   |                                   |
|                                   | A determination has been made     |
|                                   | that a major disaster has         |
|                                   | occurred for the purposes of      |
|                                   | AGDRP/ex-gratia DRP for the NSW   |
|                                   | Bushfires (Sep 2019).             |
+-----------------------------------+-----------------------------------+
| 2                                 | Assess eligibility for            |
|                                   | AGDRP/ex-gratia DRP + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | See Eligibility for AGDRP and NZ  |
|                                   | DRP                               |
|                                   |                                   |
|                                   | For the NSW Bushfires (Sep 2019)  |
|                                   | the assessment of major damage    |
|                                   | from smoke and/or ash has been    |
|                                   | updated. The following questions  |
|                                   | may be useful to determine if     |
|                                   | major damage from smoke and/or    |
|                                   | ash has occurred:                 |
|                                   |                                   |
|                                   | -   Please describe the damage to |
|                                   |     your home                     |
|                                   | -   Is the interior or exterior   |
|                                   |     damaged? Note: exterior       |
|                                   |     damage is not eligible        |
|                                   | -   Did you have to replace       |
|                                   |     contents?                     |
|                                   | -   Did you have to repair        |
|                                   |     interior damage? Note: soft   |
|                                   |     furnishings or electrical     |
|                                   |     appliances must need          |
|                                   |     repairing or replacing. These |
|                                   |     do not need to have already   |
|                                   |     been replaced or repaired for |
|                                   |     the customer to meet the      |
|                                   |     major damage criteria         |
|                                   | -   Could the smoke/ash damage be |
|                                   |     cleaned?                      |
|                                   | -   How could you demonstrate the |
|                                   |     damage occurred? For example, |
|                                   |     could you provide a photo or  |
|                                   |     invoice? Note: [] this        |
|                                   |     information can be forwarded  |
|                                   |     to Services Australia through |
|                                   |     a customer’s online account   |
|                                   |     via the upload documents      |
|                                   |     service                       |
|                                   | -   Where was the major asset     |
|                                   |     located on the property?      |
|                                   | -   Can the major asset be        |
|                                   |     repaired?                     |
|                                   | -   Did you own any major assets  |
|                                   |     on your property that were    |
|                                   |     damaged or destroyed? Please  |
|                                   |     describe the damage? Note: a  |
|                                   |     customer must own a major     |
|                                   |     asset or assets and the major |
|                                   |     asset or assets must be       |
|                                   |     located at the customer’s     |
|                                   |     principal place of residence  |
|                                   | -   What do you think is the      |
|                                   |     market value of these assets  |
|                                   |     that were damaged or          |
|                                   |     destroyed?                    |
|                                   |                                   |
|                                   | Major damage does not include     |
|                                   | cleaning an item or surface from  |
|                                   | smoke, soot, char or ash.         |
|                                   |                                   |
|                                   | Elements to be considered when    |
|                                   | assessing major damage from smoke |
|                                   | and/or ash can be found in Step 6 |
|                                   | of the Definitions for the        |
|                                   | adversely affected eligibility    |
|                                   | criteria table of Assessing       |
|                                   | eligibility for the AGDRP or      |
|                                   | ex-gratia DRP. The table also     |
|                                   | provides examples of repair work  |
|                                   | that needs to be considered when  |
|                                   | assessing claims.                 |
|                                   |                                   |
|                                   | Note: for ex-gratia DRP a         |
|                                   | customer must have participated   |
|                                   | in the Australian taxation system |
|                                   | for at least 1 year in the        |
|                                   | following past 3 financial years  |
|                                   | 2016-17, 2017-18, 2018-19 or will |
|                                   | do so before 5 February 2021.     |
+-----------------------------------+-----------------------------------+
| 3                                 | Code claim + Read more ...        |
|                                   |                                   |
|                                   | See Processing AGDRP and NZ DRP   |
|                                   | claims in Customer First.         |
|                                   |                                   |
|                                   | If a customer is not eligible for |
|                                   | AGDRP/ex-gratia DRP claims are    |
|                                   | still to be recorded and the      |
|                                   | outcome noted as rejected.        |
|                                   |                                   |
|                                   | Claims cannot be rejected until   |
|                                   | formal approval has been          |
|                                   | provided. Staff will be advised   |
|                                   | once they are able to reject      |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Claims should be held with the    |
|                                   | relevant keyword if formal        |
|                                   | approval has not been provided.   |
|                                   |                                   |
|                                   | For Minimum Activity Management   |
|                                   | Standards, see the Resources      |
|                                   | page.                             |
