| 7                                 | Residency assessment + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The customer's Australian         |
|                                   | residence qualification will be   |
|                                   | automatically assessed based on   |
|                                   | the residence information         |
|                                   | recorded. Residency information   |
|                                   | is recorded on the relevant       |
|                                   | customer's record.                |
|                                   |                                   |
|                                   | Residence information must still  |
|                                   | be recorded if the claim is to be |
|                                   | rejected for a reason other than  |
|                                   | residence. Otherwise, the system  |
|                                   | will override any other rejection |
|                                   | reason and the claim will reject  |
|                                   | for a residence related reason.   |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Residence assessment for      |
|                                   |     customers claiming FTB, PPL   |
|                                   |     and DAP                       |
|                                   | -   Residence and Portability     |
|                                   |     screens                       |
|                                   |                                   |
|                                   | The new claim process will        |
|                                   | attempt a datalink with the       |
|                                   | Department of Home Affairs. If    |
|                                   | successful, the datalink will     |
|                                   | automatically record a customer's |
|                                   | visa and movement information     |
|                                   | from 1 September 1994, and grants |
|                                   | of Australian citizenship where   |
|                                   | available.                        |
|                                   |                                   |
|                                   | Check the assessment and ensure   |
|                                   | PPL is only granted when the      |
|                                   | claimant has been residentially   |
|                                   | qualified for every day from the  |
|                                   | date of birth. Where the          |
|                                   | assessment result is incorrect,   |
|                                   | the details must be investigated  |
|                                   | through normal escalation         |
|                                   | processes, for example, referral  |
|                                   | to NIRO or SSO.                   |

|                                   | attempting to finalise the claim. |

|                                   | Debt offsetting - FTB             |
|                                   |                                   |
|                                   | From 10 June 2019, FTB top-ups,   |
|                                   | supplements, lump sum claim       |
|                                   | amounts and instalment arrears    |
|                                   | can be used to recover any family |
|                                   | assistance, social security,      |

|                                   | debts. This will occur even if    |

|                                   | recover FTB, Child Care Benefit   |
|                                   | (CCB) and Child Care Rebate (CCR) |
|                                   | debts.                            |
|                                   |                                   |
|                                   | Debt offsetting - PPL             |
|                                   |                                   |
|                                   | PPL arrears can be used to        |
|                                   | recover determined or due PPL     |
|                                   | debts, and offset PPL or ISP      |
|                                   | overpayments that were generated  |
|                                   | in the same activity. These       |
|                                   | amounts will be offset against    |
|                                   | each other so the payment does    |
|                                   | not issue and subsequently raise  |
|                                   | a debt on the same day.           |
|                                   |                                   |
|                                   | Note: PPL offsetting applies only |
|                                   | to payments of PPL or             |
|                                   | overpayments of an income support |
|                                   | payment (ISP) that result from    |
|                                   | the customer getting PPL for the  |
|                                   | same past period. This is because |
|                                   | PPL is treated as ordinary income |
|                                   | for calculating the rate of ISP.  |
|                                   |                                   |
|                                   | PPL arrears offsetting will occur |
|                                   | even if the customer has a        |
|                                   | current payment arrangement. Any  |
|                                   | residual amount will then be paid |
|                                   | to the customer. See Parental     |
|                                   | Leave Pay debt raising, debt      |
|                                   | offsetting and recovery.          |
