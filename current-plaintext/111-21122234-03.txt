+-----------------------------------+-----------------------------------+
| 1                                 | Before seeking technical          |
|                                   | support + Read more ...           |
|                                   |                                   |
|                                   | Research using a range of online  |
|                                   | tools and reference material      |
|                                   | before contacting TPS. For        |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   Operational Blueprint         |
|                                   | -   Network News Update (NNU)     |
|                                   | -   Office Locator                |
|                                   | -   mySupport                     |
|                                   | -   Roxy Digital Assistant        |
|                                   |     response                      |
|                                   | -   Process Direct current issues |
|                                   |                                   |
|                                   | Contact TPS if:                   |
|                                   |                                   |
|                                   | an activity results in arrears or |
|                                   | a negative adjustment and the     |
|                                   | Service Officer is not sure if    |
|                                   | this result is correct based on   |
|                                   | the reference material used       |
|                                   |                                   |
|                                   | relevant reference material does  |
|                                   | not provide a solution when:      |
|                                   |                                   |
|                                   | -   the status or rate of payment |
|                                   |     does not reflect information  |
|                                   |     on the record                 |
|                                   | -   the system will not allow     |
|                                   |     updates, or                   |
|                                   | -   coding results in an error    |
|                                   |                                   |
|                                   | Do not contact TPS to:            |
|                                   |                                   |
|                                   | -   confirm information found in  |
|                                   |     reference material, or        |
|                                   | -   check if an activity is       |
|                                   |     correct before completion     |
+-----------------------------------+-----------------------------------+
| 2                                 | Create a SharePoint profile when  |
|                                   | accessing the page for the first  |
|                                   | time + Read more ...              |
|                                   |                                   |
|                                   | -   Open the relevant boutique    |
|                                   |     SharePoint page from the      |
|                                   |     Smart Centre Operations       |
|                                   |     Division                      |
|                                   | -   Select Profile                |
|                                   | -   Fill in all required details  |
|                                   |     on the form                   |
|                                   | -   Select Save                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete the request for          |
|                                   | support + Read more ...           |
|                                   |                                   |
|                                   | -   Open the relevant boutique    |
|                                   |     SharePoint page from the      |
|                                   |     Smart Centre Operations       |
|                                   |     Division                      |
|                                   | -   Select New Item               |
|                                   | -   Fill in all required fields   |
|                                   |     on the form                   |
|                                   | -   Select Submit                 |
|                                   | -   Refresh [F5] the page to see  |
|                                   |     the status of the request     |
|                                   | -   Staff can view and monitor    |
|                                   |     requests in the ALL Items tab |
+-----------------------------------+-----------------------------------+
| 4                                 | Service Officer instructions      |
|                                   | while waiting for support + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Due to the complex nature of some |
|                                   | boutique work, staff can          |

|                                   | staff must follow the process     |
|                                   | below:                            |
|                                   |                                   |
|                                   | -   review all resource materials |
|                                   |     for operational instructions  |
|                                   |     outlining the steps required  |
|                                   |     to complete the update        |
|                                   | -   if a Service Officer has      |
|                                   |     waited more than 10 minutes   |
|                                   |     for support, the staff member |
|                                   |     must ‘hold’ the work item.    |
|                                   |     Select a new piece of work to |
|                                   |     complete while waiting for a  |
|                                   |     TPS response                  |

Technical Peer Support Officer (TPSO)

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | View items submitted for          |
|                                   | technical support + Read more ... |
|                                   |                                   |
|                                   | -   Open the relevant boutique    |
|                                   |     SharePoint page from the      |
|                                   |     Smart Centre Operations       |
|                                   |     Division                      |
|                                   | -   Refresh [F5] the screen to    |
|                                   |     view the most up-to-date      |
|                                   |     items                         |
|                                   | -   Select Admin view             |
|                                   | -   New items display at the top  |
|                                   |     of the list, then items Under |
|                                   |     review, then items Finalised  |
+-----------------------------------+-----------------------------------+
| 2                                 | Select item for action + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the New item using the     |
|                                   | Edit icon and change the status   |
|                                   | to Under Review. This will:       |
|                                   |                                   |
|                                   | -   give staff a visual on the    |
|                                   |     progress of their request     |
|                                   | -   prevent another TPSO from     |
|                                   |     selecting the item            |
+-----------------------------------+-----------------------------------+
| 3                                 | Provide support to a Service      |
|                                   | Officer + Read more ...           |
|                                   |                                   |
|                                   | TPSOs will support the Service    |
|                                   | Officer to resolve the enquiry by |
|                                   | guiding them to the relevant      |
|                                   | resource or escalating the        |
|                                   | enquiry using the appropriate     |
|                                   | channel.                          |
|                                   |                                   |
|                                   | If the Service Officer has not    |
|                                   | checked any reference material,   |
|                                   | the TPSO still needs to resolve   |
|                                   | the enquiry.                      |
|                                   |                                   |
|                                   | The TPSO must give feedback on    |
|                                   | every contact using the Technical |
|                                   | Support Centre (TSC).             |
|                                   |                                   |
|                                   | If the support contact can be     |
|                                   | completed by the TPS, go to Step  |
|                                   | 4.                                |
|                                   |                                   |
|                                   | If the TPSO is not able to        |
|                                   | contact the service officer       |
|                                   | immediately, they must:           |
|                                   |                                   |
|                                   | -   leave the item Under Review   |
|                                   |     and move on to the next       |
|                                   |     enquiry                       |
|                                   | -   periodically attempt to       |
|                                   |     contact the service officer   |
|                                   |     throughout the day            |
|                                   |                                   |
|                                   | Any enquiries remaining Under     |
|                                   | Review at the end of the day must |
|                                   | be returned to the pool.          |
|                                   | Procedure ends here.              |
|                                   |                                   |
|                                   | If the TPSO cannot complete the   |
|                                   | enquiry because it requires       |
|                                   | escalation, go to Step 6.         |
+-----------------------------------+-----------------------------------+
| 4                                 | Finalise a support contact via    |
|                                   | the Technical Support Centre      |
|                                   | (TSC) + Read more ...             |
|                                   |                                   |
|                                   | The TSC must be completed for     |
|                                   | every TPS contact completed.      |
|                                   |                                   |
|                                   | TPSOs will get the following      |
|                                   | information from the SharePoint   |
|                                   | entry submitted by the Service    |
|                                   | Officer. This needs to be         |
|                                   | transposed to the TSC form:       |
|                                   |                                   |
|                                   | -   Service Officer name          |
|                                   | -   Service Officer log on        |
|                                   | -   What they need help with      |
|                                   | -   Any reference material        |
|                                   |     referred to                   |
|                                   |                                   |
|                                   | Note: the CRN will not be         |
|                                   | available in SharePoint. This     |
|                                   | needs to be given verbally by the |
|                                   | Service Officer when required.    |
|                                   |                                   |
|                                   | When completing the TSC form, be  |
|                                   | accurate, clear and succinct.     |
|                                   | Choose the correct drop down      |
|                                   | options.                          |
|                                   |                                   |
|                                   | Include the correct resources for |
|                                   | future reference by the Service   |
|                                   | Officer. If an Operational        |
|                                   | Blueprint file has been           |
|                                   | referenced, include the number in |
|                                   | Reference/s field in the TSC.     |
|                                   |                                   |
|                                   | Include an accurate summary of    |
|                                   | the contact in the Comments field |
|                                   | in the TSC.                       |
|                                   |                                   |
|                                   | Note: TPSOs must make sure that   |
|                                   | no personal or customer details   |
|                                   | are recorded within the TSC. This |
|                                   | includes names, dates of birth    |
|                                   | (DOB) and any other customer      |
|                                   | details, Customer Reference       |
|                                   | Numbers (CRNs), SPARC, ACMPS,     |
|                                   | children detail, ex-partner       |
|                                   | detail in the free text fields.   |
|                                   |                                   |
|                                   | When completing the TSC, select   |
|                                   | the Finalise and notify SO option |
|                                   | in the Next action field. This    |
|                                   | sends an email to the Service     |
|                                   | Officer with a summary of the     |
|                                   | support contact.                  |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Finalise the SharePoint request   |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Finalised              |
|                                   | -   Select Close                  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Escalating a support contact to a |
|                                   | Service Support Officer using the |
|                                   | Technical Support Centre (TSC) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The TSC must be completed for     |
|                                   | every TPS contact.                |
|                                   |                                   |
|                                   | TPSOs will get the following      |
|                                   | information from the SharePoint   |
|                                   | entry submitted by the Service    |
|                                   | Officer. This information needs   |
|                                   | to be transposed to the TSC form: |
|                                   |                                   |
|                                   | -   Service Officer name          |
|                                   | -   Service Officer log on        |
|                                   | -   SharePoint entry ID           |
|                                   | -   CRN (will not be available in |
|                                   |     SharePoint and will need to   |
|                                   |     be given verbally by the      |
|                                   |     Service Officer when          |
|                                   |     required)                     |
|                                   | -   What they need help with      |
|                                   | -   Any reference material        |
|                                   |     referred to                   |
|                                   |                                   |
|                                   | Note: the TPS must include the    |
|                                   | SharePoint entry ID in the        |
|                                   | escalation comments. This         |
|                                   | information is required to        |
|                                   | finalise the SharePoint entry.    |
|                                   |                                   |
|                                   | When completing the TSC form, be  |
|                                   | accurate, clear and succinct.     |
|                                   | Choose the correct drop down      |
|                                   | options.                          |
|                                   |                                   |
|                                   | Include the correct resources for |
|                                   | future reference by the Service   |
|                                   | Officer. If an Operational        |
|                                   | Blueprint file has been           |
|                                   | referenced, include the number in |
|                                   | Reference/s field in the TSC.     |
|                                   |                                   |
|                                   | Include an accurate summary of    |
|                                   | the contact in the Comments field |
|                                   | in the TSC.                       |
|                                   |                                   |
|                                   | TPSOs must make sure that no      |
|                                   | personal or customer details are  |
|                                   | recorded within the TSC. This     |
|                                   | includes names, dates of birth    |
|                                   | (DOB) and any other customer      |
|                                   | details, Customer Reference       |
|                                   | Numbers (CRNs), SPARC, ACMPS,     |
|                                   | children detail, ex-partner       |
|                                   | detail in the free text fields.   |
|                                   |                                   |
|                                   | After completing the TSC entry,   |
|                                   | choose the Send to Service        |
|                                   | Support Officer option in the     |
|                                   | Next action field.                |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Update the customer record + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | TPSOs must complete a DOC on the  |
|                                   | customer's record detailing the   |
|                                   | escalation issue and action       |
|                                   | already taken. Use Fast Note -    |
|                                   | select Auto Text, use Generic >   |
|                                   | Escalation > Escalated to Service |
|                                   | Support Officer                   |
+-----------------------------------+-----------------------------------+
| 8                                 | Update the SharePoint request     |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Escalate to SSO        |
|                                   | -   Select Close                  |
+-----------------------------------+-----------------------------------+
| 9                                 | Finalise the SharePoint request   |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | Once the SSO has completed the    |
|                                   | escalation and the TPS receives   |
|                                   | the response via the TSC email.   |
|                                   | The TPS must navigate to the      |
|                                   | SharePoint page to finalise the   |
|                                   | entry.                            |
|                                   |                                   |
|                                   | -   Locate the correct entry      |
|                                   |     using the SharePoint entry ID |
|                                   |     number                        |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Finalised              |
|                                   | -   Select Close                  |
|                                   |                                   |
|                                   | Procedure ends here.              |

Service Support Officer

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complete the Technical Support    |
|                                   | Centre (TSC) + Read more ...      |
|                                   |                                   |
|                                   | SSOs must include an accurate     |
|                                   | summary of the resolution in the  |
|                                   | Comments field. When completing   |
|                                   | the TSC form, be accurate, clear  |
|                                   | and succinct.                     |
|                                   |                                   |
|                                   | When finalising the TSC entry,    |
|                                   | SSOs must select the Finalise and |
|                                   | Notify SO & TPS option in the     |
|                                   | Next action field. This sends an  |
|                                   | email to the Service Officer and  |
|                                   | TPS, allowing the TPS to finalise |
|                                   | the SharePoint entry.             |
|                                   |                                   |
|                                   | Note: SSOs must make sure that no |
|                                   | personal or customer details are  |
|                                   | recorded within the TSC. This     |
|                                   | includes names, dates of birth    |
|                                   | (DOB) and any other customer      |
|                                   | details, Customer Reference       |
|                                   | Numbers (CRNs), SPARC, ACMPS,     |
|                                   | children detail, ex-partner       |
|                                   | detail in the free text fields.   |
+-----------------------------------+-----------------------------------+
| 2                                 | System issues, policy referrals,  |
|                                   | complex or unable to resolve      |
|                                   | issues (e.g. ICT/Policy           |
|                                   | referrals) + Read more ...        |
|                                   |                                   |
|                                   | When an SSO is unable to complete |
|                                   | the request, they should escalate |
|                                   | using the appropriate channel for |
|                                   | the work type (e.g. Policy        |
|                                   | helpdesk/ICT/Portfolio Operations |
|                                   | Team).                            |
|                                   |                                   |
|                                   | SSOs must complete a DOC on the   |
|                                   | customer's record detailing the   |
|                                   | escalation issue and action       |
|                                   | already taken. Use Fast Note -    |
|                                   | select Auto Text, use Generic >   |
|                                   | Escalation > (Select relevant     |
|                                   | escalation template)              |
+-----------------------------------+-----------------------------------+
