For Family Assistance Manual Intervention (MIV) Smart Centre Processing
staff only.

This page explains how to complete FTB reconciliation for a customer
after a manual or special adjustment.

On this page:

Action to complete FTB reconciliation (#RACADJ or #RAOADJ)

|                                   |     amount, or                    |

|                                   |     screen                        |
|                                   |                                   |
|                                   | The result should equal the       |
|                                   | reconciliation result calculated  |
|                                   | in the activity. This is after    |
|                                   | any adjustments to the RAC        |
|                                   | Adjustment (RACADJ).              |
+-----------------------------------+-----------------------------------+
| 9                                 | Amend and/or accept the adjusted  |
|                                   | amount + Read more ...            |
|                                   |                                   |
|                                   | On the FCRAS screen, select the   |
|                                   | RAC adjustment line:              |
|                                   |                                   |
|                                   | -   If the adjustment amount      |
|                                   |     should be included, do not    |
|                                   |     change the default amount.    |
|                                   |     Code 'AAR' (accept auto rate  |
|                                   |     calculated) in the Reason     |
|                                   |     field                         |
|                                   | -   If amending the amount,       |
|                                   |     record the new adjustment     |
|                                   |     amount. Code 'NAJ' (Include   |
|                                   |     new adjustment amount) in the |
|                                   |     Reason field                  |
|                                   | -   If reducing the default       |
|                                   |     amount to nil, record the     |
|                                   |     amount of $0.00. Code 'RIA'   |
|                                   |     (Respect Initial Adjustment   |
|                                   |     made) in the Reason field.    |
+-----------------------------------+-----------------------------------+
| 10                                | Action and finalise the           |
|                                   | activity + Read more ...          |
|                                   |                                   |
|                                   | Complete normal unpacking of      |
|                                   | reconciliation results.           |
|                                   |                                   |
|                                   | Once satisfied with the outcome,  |
|                                   | finalise the activity via         |
|                                   | Assessment Results (AR) screen:   |
|                                   |                                   |
|                                   | For activities holding:           |
|                                   |                                   |
|                                   | -   RACADJ keyword, record a DOC  |
|                                   |     using the #RACADJ Fast Note   |
|                                   |     template in Customer First    |
|                                   | -   #RAOADJ keyword, record a DOC |
|                                   |     using the #ROA/DRA Fast Note  |
|                                   |     template in Customer First    |
|                                   | -   Check for and complete any    |
|                                   |     associated Customer First SAP |
|                                   |     Interaction Record/s          |
|                                   |                                   |
|                                   | Debt offsetting can automatically |
|                                   | apply during claim calculation if |
|                                   | the customer has an outstanding   |
|                                   | debt.                             |
