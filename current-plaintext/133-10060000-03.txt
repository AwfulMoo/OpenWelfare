This page explains how customers can use their online account to
register, amend and remove consent or opt out of the Australian Organ
Donor Register (AODR).

On this page:

Register organ donation decision

Amend organ donor decision

Remove me from register

Register organ donation decision

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access service + Read more ...    |
|                                   |                                   |
|                                   | Sign in to myGov and select       |
|                                   | Medicare services.                |
|                                   |                                   |
|                                   | Select Proceed to ONLINE ACCOUNT. |
|                                   |                                   |
|                                   | From the homepage, select the     |
|                                   | Organ donor details menu option.  |
|                                   |                                   |
|                                   | -   Select Register me on the     |
|                                   |     Donor Register                |
|                                   | -   Select NEXT                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Donor decision + Read more ...    |
|                                   |                                   |
|                                   | Does the customer want to be an   |
|                                   | organ donor?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Register consent to be organ      |
|                                   | donor + Read more ...             |
|                                   |                                   |
|                                   | Select the register my consent    |
|                                   | radio button                      |
|                                   |                                   |
|                                   | Select organs they want to        |
|                                   | donate:                           |
|                                   |                                   |
|                                   | -   select ALL, or                |
|                                   | -   select one or more organs     |
|                                   |     from the list                 |
|                                   |                                   |
|                                   | To get a donor card, select the   |
|                                   | box. Customers can use the        |
|                                   | Express Plus Medicare mobile app  |
|                                   | to view their donor card in the   |
|                                   | Digital Wallet                    |
|                                   |                                   |
|                                   | Answer How did you find out about |
|                                   | the Australian Organ Donor        |
|                                   | Register?                         |
|                                   |                                   |
|                                   | Read and agree to the             |
|                                   | declarations                      |
|                                   |                                   |
|                                   | Select NEXT                       |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Register decision not to be an    |
|                                   | organ donor + Read more ...       |
|                                   |                                   |
|                                   | -   Select the register my        |
|                                   |     decision radio button         |
|                                   | -   Answer How did you find out   |
|                                   |     about the Australian Organ    |
|                                   |     Donor Register?               |
|                                   | -   Read and agree to the         |
|                                   |     declarations                  |
|                                   | -   Select NEXT                   |
+-----------------------------------+-----------------------------------+
| 5                                 | Registration summary + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Review all decisions in the   |
|                                   |     summary                       |
|                                   | -   Select SUBMIT                 |
+-----------------------------------+-----------------------------------+
| 6                                 | Confirmation + Read more ...      |
|                                   |                                   |
|                                   | A confirmation screen confirms    |
|                                   | their registration.               |
|                                   |                                   |
|                                   | Select RETURN TO ONLINE ACCOUNT   |
|                                   | to go to the homepage.            |
|                                   |                                   |
|                                   | If they have finished their       |
|                                   | session, select Return to myGov.  |
