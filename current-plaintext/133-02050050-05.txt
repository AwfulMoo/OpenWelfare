Rules that apply for new locations submitted via HPOS

+-----------------------------------+-----------------------------------+
| Topic                             | Addressing rule/edit check        |
+-----------------------------------+-----------------------------------+
| Mandatory address components      | On the address tab, as well as    |
|                                   | street name, locality and         |
|                                   | postcode, it is mandatory to      |
|                                   | include one of the following:     |
|                                   |                                   |
|                                   | -   street or lot number          |
|                                   | -   practice or building name     |
|                                   |                                   |
|                                   | Note: as with PDS intranet        |
|                                   | location addresses, address       |
|                                   | details entered by a provider may |
|                                   | be corrected to conform with      |
|                                   | addressing records held by        |
|                                   | Australia Post to allow           |
|                                   | allocation of a Delivery Point    |
|                                   | Identifier (DPID).                |
+-----------------------------------+-----------------------------------+
| Physical address required         | The Street, Practice              |
|                                   | name/Building or                  |
|                                   | Property/Department fields on the |
|                                   | Address tab do not accept any of  |
|                                   | the following:                    |
|                                   |                                   |
|                                   | -   PO                            |
|                                   | -   PO Box                        |
|                                   | -   Locked Bag                    |
|                                   | -   Box                           |
|                                   |                                   |
|                                   | For example: If any of the above  |
|                                   | fields contains the word ‘Box’    |
|                                   | for example ‘Box hill avenue’ the |
|                                   | system will not accept it, the    |
|                                   | provider must submit an           |
|                                   | application for an additional     |
|                                   | location, to have the need for    |
|                                   | the new practice location         |
|                                   | considered.                       |
+-----------------------------------+-----------------------------------+
| Location start date               | Mandatory field on the Contact    |
|                                   | tab.                              |
|                                   |                                   |
|                                   | Cannot be earlier than the date   |
|                                   | the provider allocates the        |
|                                   | location. If a:                   |
|                                   |                                   |
|                                   | -   Provider attempts to enter a  |
|                                   |     date that is earlier than the |
|                                   |     date of processing, the       |
|                                   |     message ‘The additional       |
|                                   |     location start date cannot be |
|                                   |     prior to today's date. If you |
|                                   |     require the start date to be  |
|                                   |     back-dated, please continue   |
|                                   |     to add the location then      |
|                                   |     contact Medicare on 132 150   |
|                                   |     to update the start date’ is  |
|                                   |     displayed.                    |
|                                   | -   Back-dated start date is      |
|                                   |     required the provider should  |
|                                   |     not be asked to complete an   |
|                                   |     application but needs to      |
|                                   |     submit a written request to   |
|                                   |     have the date changed.        |
+-----------------------------------+-----------------------------------+
| Telephone number                  | Mandatory field on the Contact    |
|                                   | tab.                              |
|                                   |                                   |
|                                   | It is mandatory to include either |
|                                   | a:                                |
|                                   |                                   |
|                                   | -   Landline number, or           |
|                                   | -   Mobile phone number           |
+-----------------------------------+-----------------------------------+
| Duplicate locations               | An error message advising the     |
|                                   | provider that they need to        |
| (Street name, locality and        | contact Medicare is displayed:    |
| postcode match street name,       |                                   |
| locality and postcode on one of   | -   the provider must submit an   |
| the provider's existing           |     application for an additional |
| locations)                        |     location, to have the need    |
|                                   |     for the new practice location |
|                                   |     considered.                   |
+-----------------------------------+-----------------------------------+
| Postcode and locality do not      | The message "The locality you     |
| match                             | have entered does not match the   |
|                                   | postcode. Please check the        |
|                                   | details you have supplied and     |
|                                   | correct any errors. You may need  |
|                                   | to contact Medicare on 132 150"   |
|                                   | is displayed:                     |
|                                   |                                   |
|                                   | -   check the location and        |
|                                   |     postcode on Medicare postcode |
|                                   |     file using either EPCI,       |
|                                   |     postcode or name of location  |
|                                   | -   if correct location           |
|                                   |     name/postcode is not in the   |
|                                   |     system, advise Services       |
|                                   |     Australia at                  |
|                                   |                                   |
|                                   |  CO.Medicare.Provider.Eligibility |
+-----------------------------------+-----------------------------------+
| Organisation Details              | Health professionals who wish to  |
|                                   | claim a Medicare benefits are not |
|                                   | required to provide the           |
|                                   | information on this tab prior to  |
|                                   | 1 September 2018.                 |
+-----------------------------------+-----------------------------------+
| Banking Details                   | Services Australia makes payments |
|                                   | for services by Electronic Funds  |
|                                   | Transfer.                         |
|                                   |                                   |
|                                   | The provider must answer the      |
|                                   | following question:               |
|                                   |                                   |
|                                   | Will you be claiming for services |
|                                   | provided at this new location?    |
+-----------------------------------+-----------------------------------+
