Recording a compliance investigation

Table 3: Using workflow during a compliance investigation.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Accessing the Compliance          |
|                                   | Investigation workflow + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To access this workflow, select   |
|                                   | the Compliance investigation      |
|                                   | button for the event date of the  |
|                                   | compliance action from the        |
|                                   | Outstanding Compliance Actions    |
|                                   | section of the Participation      |
|                                   | Compliance Hub.                   |
|                                   |                                   |
|                                   | The workflow is used to undertake |
|                                   | compliance investigations for     |
|                                   | both Targeted Compliance          |
|                                   | Framework and Job Seeker          |
|                                   | Compliance Framework.             |
|                                   |                                   |
|                                   | A workflow menu displays on the   |
|                                   | left of the workflow window. This |
|                                   | includes a list of workflow pages |
|                                   | with the current page             |
|                                   | highlighted.                      |
|                                   |                                   |
|                                   | The workflow contains the         |
|                                   | following pages:                  |
|                                   |                                   |
|                                   | -   Other Related Information     |
|                                   | -   Key Information               |
|                                   | -   Investigation                 |
|                                   | -   Decision and Penalty          |
|                                   | -   Re-engagement (Job Seeker     |
|                                   |     Compliance Framework only)    |
|                                   | -   Save and Exit                 |
|                                   |                                   |
|                                   | If appropriate, the workflow may  |
|                                   | also link to the Circumstances    |
|                                   | Impacting Compliance page.        |
|                                   |                                   |
|                                   | The workflow is intuitive,        |
|                                   | displaying information,           |
|                                   | discussion points and questions   |
|                                   | tailored to the failure reason    |
|                                   | and the responses in the          |
|                                   | workflow. This ensures staff      |
|                                   | discuss, consider and capture     |
|                                   | relevant details.                 |
|                                   |                                   |
|                                   | All staff have general view       |
|                                   | access to the workflow. However,  |
|                                   | a compliance investigation can    |
|                                   | only be recorded by Service       |
|                                   | Officers with a WNPPD, UNPP or    |
|                                   | PFDEC IT security resource.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Other Related Information + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This page displays links to       |
|                                   | information relevant to the       |
|                                   | compliance investigation.         |
|                                   |                                   |
|                                   | Select this page from the link on |
|                                   | the Compliance Investigation      |
|                                   | workflow menu.                    |
|                                   |                                   |
|                                   | Select links from this page to    |
|                                   | open another window displaying    |
|                                   | the required page or information. |
|                                   | The Compliance Investigation      |
|                                   | workflow will remain open.        |
|                                   |                                   |
|                                   | The following links display:      |
|                                   |                                   |
|                                   | -   Provider Report               |
|                                   | -   Job Plan                      |
|                                   | -   Provider Recorded Information |
|                                   | -   Online Diary                  |
|                                   | -   Job Seeker Calendar (Targeted |
|                                   |     Compliance Framework provider |
|                                   |     reported compliance action    |
|                                   |     only)                         |
|                                   | -   Recent non-compliance         |
|                                   | -   Services Australia Rates      |
|                                   |     Estimator                     |
|                                   | -   Address History               |
|                                   | -   View ESAt                     |
|                                   | -   Recent requirements and       |
|                                   |     exemptions                    |
|                                   | -   Circumstances impacting       |
|                                   |     compliance                    |
|                                   | -   Medical certificates          |
+-----------------------------------+-----------------------------------+
| 3                                 | Key Information + Read more ...   |
|                                   |                                   |
|                                   | This page is the starting point   |
|                                   | for a compliance investigation.   |
|                                   | It displays the information       |
|                                   | relevant to the job seeker and    |
|                                   | failure details on the failure    |
|                                   | event date.                       |
|                                   |                                   |
|                                   | Information includes:             |
|                                   |                                   |
|                                   | -   Provider type                 |
|                                   | -   Payment type                  |
|                                   | -   Participation category        |
|                                   | -   Indigenous Identifier         |
|                                   | -   Existing circumstances        |
|                                   |     impacting compliance: link    |
|                                   |     'View circumstances' is       |
|                                   |     active where there are        |
|                                   |     current circumstances         |
|                                   |     recorded and the circumstance |
|                                   |     category will be listed       |
|                                   | -   Failure event date            |
|                                   | -   Failure reason                |
|                                   | -   Potential penalty             |
|                                   |                                   |
|                                   | The job seeker's current contact  |
|                                   | information is displayed, along   |
|                                   | with any language support         |
|                                   | identifiers.                      |
|                                   |                                   |
|                                   | If an interpreter has been        |
|                                   | engaged during the compliance     |
|                                   | investigation this must be        |
|                                   | recorded, and contact details     |
|                                   | confirmed.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Investigation Page + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | This page includes these          |
|                                   | sections:                         |
|                                   |                                   |
|                                   | -   Reason failure submitted      |
|                                   | -   Excuse for failure            |
|                                   | -   Prior contact                 |
|                                   | -   Further evidence              |
|                                   |                                   |
|                                   | The questions are based on the    |
|                                   | type of compliance action under   |
|                                   | investigation, the job seeker     |
|                                   | compliance system applies and if  |
|                                   | the action was generated by the   |
|                                   | provider or Services Australia.   |
|                                   |                                   |
|                                   | Reason failure submitted + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The non-compliance event, the     |
|                                   | compulsory requirement and the    |
|                                   | potential penalty display, with a |
|                                   | prompt to discuss this with the   |
|                                   | job seeker.                       |
|                                   |                                   |
|                                   | Read the Privacy statement as     |
|                                   | displayed in this section.        |
|                                   |                                   |
|                                   | Excuse for failure + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the provider generated the     |
|                                   | compliance action, refer to the   |
|                                   | information reported by the       |
|                                   | provider as the starting point    |
|                                   | for the investigation:            |
|                                   |                                   |
|                                   | -   For Targeted Compliance       |
|                                   |     Framework failures, the       |
|                                   |     workflow will display the     |

|                                   |     seeker if there is anything   |

|                                   |     'Provider Report Summary'     |
|                                   |     link                          |
|                                   |                                   |

|                                   | record the job seeker's reason    |
|                                   | for the failure.                  |
|                                   |                                   |

|                                   | suggests ongoing circumstances    |
|                                   | directly impacted their capacity  |

|                                   | Circumstances Impacting           |
|                                   | Compliance page from this section |
|                                   | to assess the impact. The         |
|                                   | Compliance Investigation workflow |
|                                   | will then resume from this        |
|                                   | section.                          |

|                                   | seeker's reason for               |
|                                   | non-compliance.                   |
|                                   |                                   |
|                                   | Depending on the reason selected, |
|                                   | more information or questions may |
|                                   | appear in relation to:            |
|                                   |                                   |
|                                   | -   Using the Rate Estimator      |
|                                   | -   Relocation assistance         |
|                                   | -   Employment Separation         |
|                                   |     Certificate (SU1)             |
|                                   | -   Requirement in                |
|                                   |     Job/Participation Plan        |
|                                   | -   Requirement in job seeker     |
|                                   |     calendar                      |
|                                   | -   Date and method of            |
|                                   |     notification                  |
|                                   | -   Provider confirmation of      |
|                                   |     suitability of work           |
|                                   | -   Employment Income reported    |
|                                   | -   Medical certificates recorded |
|                                   |     in last 12 months             |
|                                   | -   Exemptions recorded in last 3 |
|                                   |     months                        |
|                                   |                                   |

|                                   | presents questions that prompt    |

|                                   |     was not established           |
|                                   |                                   |
|                                   | If reasonable excuse is not       |
|                                   | established, the workflow will go |
|                                   | to the Decision and Penalty page. |
|                                   | A decision can be made to apply   |
|                                   | the compliance action.            |
|                                   |                                   |
|                                   | See Assessing reasonable excuse   |
|                                   | for non-compliance with mutual    |
|                                   | obligation requirements.          |
|                                   |                                   |
|                                   | Prior contact + Read more ...     |
|                                   |                                   |
|                                   | This section will display only    |
|                                   | when an assessment is required.   |
|                                   |                                   |
|                                   | Failure types requiring an        |
|                                   | assessment of prior contact are:  |
|                                   |                                   |
|                                   | Targeted Compliance Framework:    |
|                                   |                                   |
|                                   | -   PPA - Failure to attend       |
|                                   |     provider appointment (or      |
|                                   |     attend on time)               |
|                                   | -   P3A - Failure to attend other |
|                                   |     provider appointment (or      |
|                                   |     attend on time)               |
|                                   | -   PAY - Failure to attend       |
|                                   |     activity (or attend on time)  |
|                                   |                                   |
|                                   | Job Seeker Compliance Framework:  |
|                                   |                                   |
|                                   | -   RFA - Failure to attend       |
|                                   |     provider reconnection         |
|                                   |     appointment                   |
|                                   | -   COP - Failure to attend other |
|                                   |     provider appointment          |
|                                   | -   FAR - Failure to attend       |
|                                   |     regular provider appointment  |
|                                   |                                   |
|                                   | Questions allow the recording of  |
|                                   | whether a job seeker made prior   |
|                                   | contact or if they had reasonable |
|                                   | excuse for not doing so.          |
|                                   |                                   |
|                                   | If the job seeker has not         |
|                                   | provided a reasonable excuse, the |
|                                   | workflow will go to the Decision  |
|                                   | and Penalty page for a decision   |
|                                   | to apply the compliance action.   |
|                                   |                                   |
|                                   | See Assessing reasonable excuse   |
|                                   | for non-compliance with mutual    |
|                                   | obligation requirements.          |
|                                   |                                   |
|                                   | Further evidence + Read more ...  |
|                                   |                                   |
|                                   | This section displays if          |
|                                   | responses in the workflow         |
|                                   | indicate there is enough evidence |
|                                   | to make a decision. Further       |
|                                   | evidence should only be gathered  |
|                                   | if it will affect the decision.   |
|                                   | If possible, complete the         |
|                                   | compliance investigation on the   |
|                                   | day of contact.                   |
|                                   |                                   |
|                                   | Only 1 person can be selected. If |
|                                   | more than 1 person has been       |
|                                   | contacted for further evidence,   |
|                                   | record the details of the primary |
|                                   | contact, e.g. provider, employer  |
|                                   | or other. Capture all other       |
|                                   | contact details in the in the     |

|                                   | requested/provided field, e.g.    |
|                                   | organisation/business name,       |
|                                   | contact person, contact number    |
|                                   | and a summary of the evidence     |
|                                   | provided.                         |

|                                   | available evidence. All other     |
|                                   | investigations may be held        |
|                                   | pending relevant evidence.        |
|                                   |                                   |
|                                   | If it is appropriate to hold an   |
|                                   | investigation pending further     |
|                                   | evidence, record in this section: |
|                                   |                                   |
|                                   | -   what evidence has been        |
|                                   |     requested                     |
|                                   | -   why this evidence will        |
|                                   |     influence the outcome of the  |
|                                   |     decision                      |
|                                   | -   the evidence due date         |
|                                   |                                   |
|                                   | See Gathering information and     |
|                                   | weighing evidence to establish    |
|                                   | facts for a compliance            |
|                                   | investigation.                    |
