Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | When to remove an Emergency       |
|                                   | Response Indicator + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Staff must remove an Emergency    |
|                                   | Response Indicator, and           |
|                                   | associated emergency event        |
|                                   | details, if the customer:         |
|                                   |                                   |
|                                   | has an Emergency Response         |
|                                   | Indicator with a past review      |
|                                   | date, and                         |
|                                   |                                   |
|                                   | -   is Private Collect, or        |
|                                   | -   is payee, or                  |
|                                   | -   has a debt of $10 or less     |
|                                   |                                   |
|                                   | Staff may choose to remove the    |
|                                   | Emergency Response Indicator, and |
|                                   | associated emergency event        |
|                                   | details, if the customer:         |
|                                   |                                   |
|                                   | has an Emergency Response         |
|                                   | Indicator with a past review      |
|                                   | date, and                         |
|                                   |                                   |
|                                   | -   is a payer, and               |
|                                   | -   has a debt of more than $10   |
|                                   |                                   |
|                                   | Staff must review the customer’s  |
|                                   | circumstances to determine if the |
|                                   | indicator must stay. Staff must   |
|                                   | use their discretion. If the      |
|                                   | indicator must stay, set the next |
|                                   | date of review.                   |
+-----------------------------------+-----------------------------------+
| 2                                 | How to remove an Emergency        |
|                                   | Response Indicator + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | To remove an indicator and        |
|                                   | associated emergency event        |
|                                   | details:                          |
|                                   |                                   |
|                                   | Go to the Client Profile screen   |
|                                   | in Cuba:                          |
|                                   |                                   |
|                                   | -   Remove the tick next to the   |
|                                   |     Sensitive Issue 'Emergency    |
|                                   |     Response' (this should remove |
|                                   |     both the 'Emergency Response' |
|                                   |     and the Review date)          |
|                                   | -   Change Special Contact to     |
|                                   |     None, if applicable           |
|                                   |                                   |
|                                   | Add Notepad with details:         |
|                                   |                                   |
|                                   | -   Subject: Emergency Response   |
|                                   |     Exit Strategy                 |
|                                   | -   Text: Emergency Response and  |
|                                   |     Special Contact removed as    |
|                                   |     part of Child Support [insert |
|                                   |     name of event and date]       |
|                                   | -   Save and Exit                 |

|                                   | relevant Estimate Deeming         |
