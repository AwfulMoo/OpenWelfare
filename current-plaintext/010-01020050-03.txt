Forms/Modules - Q to Z

Table 3: This table describes a list of forms/modules beginning from Q
to Z and describes when to issue these particular forms/ modules in
conjunction with the claim forms for ABSTUDY, Isolated Children (AIC),
Austudy, Pensioner Education Supplement (PES) and Youth Allowance (YA)
so that a customer’s entitlement can be accurately assessed.

+-----------------------------------+-----------------------------------+
| Real estate                       | Module R (Real Estate Details) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Issue to customers who own real   |

|                                   |                                   |
|                                   | Note: when triggered from an      |
|                                   | online new claim, Mod R can be    |
|                                   | launched from the What You Must   |
|                                   | Do page and lodged online.        |
+-----------------------------------+-----------------------------------+
| Separation                        | Module S (Separation details) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Issue to customers who have       |
|                                   | separated from their partner. See |
|                                   | Partner details.                  |
+-----------------------------------+-----------------------------------+
| Short Term Boarding Allowance     | Assistance for Isolated Children  |
| (AIC)                             | (AIC) - Application for Short     |
|                                   | Term Boarding Allowance (SY068) + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Issue to AIC applicants to allow  |
|                                   | them to claim AIC boarding        |
|                                   | allowance for the student for     |
|                                   | additional short term board       |
|                                   | periods.                          |
+-----------------------------------+-----------------------------------+
| Study details                     | Module ST (Study Details) + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Issue to customers who are        |

| Unreasonable to live at home      | Unreasonable to live at home +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For over 18 year olds issue:      |
|                                   |                                   |
|                                   | -   SY015 - Youth Allowance or    |
|                                   |     ABSTUDY 'Unreasonable to live |
|                                   |     at home' Statement by Young   |
|                                   |     Person                        |
|                                   | -   SY016 - Youth Allowance or    |
|                                   |     ABSTUDY 'Unreasonable to live |
|                                   |     at home' Statement by         |
|                                   |     Parent(s)/Guardians(s)        |
|                                   |                                   |
|                                   | Note: if the Statement by young   |
|                                   | person form (SY015) and Statement |

|                                   | the UTLAH Over 18 Assessment      |

|                                   | Party form (SY017) is not         |
|                                   | required and should not be        |
|                                   | requested to grant UTLAH (O18).   |
