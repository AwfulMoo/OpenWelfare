Frequently asked questions

Table 1: these questions are designed to assist staff to use the F2F QOL
and QMA Disputes SharePoint page and track the progress of the dispute.

+-----------------------------------+-----------------------------------+
| Question                          | Answer                            |
+-----------------------------------+-----------------------------------+
| What does the status mean?        | -   New: this is the initial      |
|                                   |     status of new disputes        |
|                                   | -   Under review: a request has   |
|                                   |     been assigned to an           |
|                                   |     independent reviewer and is   |
|                                   |     being reviewed                |
|                                   | -   Awaiting reply: the           |
|                                   |     independent reviewer is       |
|                                   |     waiting for the original QOL  |
|                                   |     checker's/QMO's right of      |
|                                   |     reply                         |
|                                   | -   Reply received: the original  |
|                                   |     QOL checker's/QMO's reply has |
|                                   |     been received                 |
|                                   | -   Escalated: the dispute has    |
|                                   |     been escalated to Level 2     |
|                                   |     Helpdesk                      |
|                                   | -   Withdrawn: the dispute has    |
|                                   |     been withdrawn                |
|                                   | -   Finalised: the dispute has    |
|                                   |     been finalised                |
+-----------------------------------+-----------------------------------+
| What does the dispute outcome     | -   Pending: the dispute is still |
| mean?                             |     under review                  |
|                                   | -   Upheld: the independent       |
|                                   |     reviewer agrees with original |
|                                   |     check result                  |
|                                   | -   Upheld for a different        |
|                                   |     reason: the checking outcome  |
|                                   |     is correct but for a          |
|                                   |     different reason to the       |
|                                   |     original decision             |
|                                   | -   Overturned: the independent   |
|                                   |     reviewer agrees with the      |
|                                   |     dispute                       |
|                                   | -   Activity not located: the     |
|                                   |     activity or Work Item was     |
|                                   |     unable to be located          |
|                                   | -   Insufficient evidence:        |
|                                   |     insufficient evidence to      |
|                                   |     support the QOL/QMA dispute   |
|                                   |     has been provided             |
+-----------------------------------+-----------------------------------+
| How do I edit my dispute?         | Service Officers can update a     |
|                                   | QOL/QMA dispute after it has been |
|                                   | lodged.                           |
|                                   |                                   |
|                                   | Is the status of the dispute New? |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   read and acknowledge the      |
|                                   |     Privacy statement             |
|                                   | -   make the required updates     |
|                                   | -   scroll to the bottom of the   |
|                                   |     form and select Update        |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   contact the independent       |
|                                   |     reviewer to update the        |
|                                   |     dispute                       |
|                                   | -   if the independent reviewer   |
|                                   |     is not available email        |
|                                   |     F2F.QUALITY                   |
+-----------------------------------+-----------------------------------+
| How do I withdraw a dispute?      | Service Officers can withdraw a   |
|                                   | QOL/QMA dispute after it has been |
|                                   | lodged.                           |
|                                   |                                   |
|                                   | Is the status of the dispute New? |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   read and acknowledge the      |
|                                   |     Privacy statement             |
|                                   | -   scroll to the bottom of the   |
|                                   |     form and select Withdraw      |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   contact the independent       |
|                                   |     reviewer to withdraw the      |
|                                   |     dispute                       |
|                                   | -   if the independent reviewer   |
|                                   |     is not available email        |
|                                   |     SSW.QUALITY                   |
+-----------------------------------+-----------------------------------+
| Can I seek a further review if I  | No, once QOL/QMA dispute decision |
| disagree with QOL/QMA dispute     | has been finalised, no further    |
| decision?                         | escalation steps are available.   |
