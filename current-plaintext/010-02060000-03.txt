Assessing eligibility for the away from home rate for ABSTUDY

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claim for the away from home      |
|                                   | rate + Read more ...              |
|                                   |                                   |
|                                   | When a student or Australian      |
|                                   | Apprentice lodges a claim for the |
|                                   | away from home rate, or advises   |
|                                   | of a change of circumstances,     |
|                                   | (for example change of address),  |
|                                   | the rate of ABSTUDY must be       |
|                                   | determined or reassessed.         |
|                                   |                                   |
|                                   | To assess a claim for the away    |
|                                   | from home rate, check whether the |
|                                   | student or Australian Apprentice  |
|                                   | is dependent or independent. This |
|                                   | information can be found in       |
|                                   | Customer First on the             |
|                                   | Independent/Homeless/Away from    |
|                                   | Home Details (NIH) screen for     |
|                                   | students or Australian            |
|                                   | Apprentices under 22 years of     |
|                                   | age.                              |
|                                   |                                   |
|                                   | Is the student or Australian      |
|                                   | Apprentice independent?           |
|                                   |                                   |
|                                   | -   Yes, this process does not    |
|                                   |     apply. Procedure ends here    |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Living away from the permanent    |
|                                   | home + Read more ...              |
|                                   |                                   |
|                                   | Is the dependent student or       |
|                                   | Australian Apprentice living away |
|                                   | from the permanent home?          |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the student or Australian |
|                                   |     Apprentice will be paid the   |
|                                   |     at home rate. Procedure ends  |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Criteria for living away from     |
|                                   | home + Read more ...              |
|                                   |                                   |
|                                   | Identify if the student or        |
|                                   | Australian Apprentice is living   |
|                                   | away from home due to:            |
|                                   |                                   |
|                                   | -   Travel time and access        |
|                                   |     between the permanent home    |
|                                   |     and the education institution |
|                                   |     or employer location for an   |
|                                   |     Australian Apprentice, makes  |
|                                   |     daily travel impossible or    |
|                                   |     unreasonable, see Assessing   |
|                                   |     ABSTUDY travel time and       |
|                                   |     access for the away from home |
|                                   |     rate                          |
|                                   | -   Home conditions are impacting |
|                                   |     on educational or training    |
|                                   |     progress. See Assessing       |
|                                   |     ABSTUDY home conditions for   |
|                                   |     the away from home rate       |
|                                   | -   Student or Australian         |
|                                   |     Apprentice is in State care   |
|                                   |     and a foster parent is not    |
|                                   |     receiving a foster care or    |
|                                   |     similar allowance. See        |
|                                   |     Assessing ABSTUDY student in  |
|                                   |     state care for the away from  |
|                                   |     home rate                     |
|                                   |                                   |
|                                   | Is the student or Australian      |
|                                   | Apprentice living away from home  |
|                                   | due to any of the above reasons?  |
|                                   |                                   |
|                                   | Yes, procedure ends here          |
|                                   |                                   |
|                                   | No, if the application is for a:  |
|                                   |                                   |
|                                   | -   secondary school student, go  |
|                                   |     to Step 4                     |
|                                   | -   tertiary student or an        |
|                                   |     Australian Apprentice, go to  |
|                                   |     Step 5                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Secondary student away from       |
|                                   | home + Read more ...              |
|                                   |                                   |
|                                   | If a secondary student lives away |
|                                   | from home they may be eligible    |
|                                   | for the away from home rate for   |
|                                   | one of the following reasons:     |
|                                   |                                   |
|                                   | -   Student is studying an        |
|                                   |     approved special secondary    |
|                                   |     course at a government        |
|                                   |     school. See Assessing special |
|                                   |     courses for ABSTUDY away from |
|                                   |     home rate                     |
|                                   | -   Student's academic needs are  |
|                                   |     not met by local secondary    |
|                                   |     facilities. See Assessing     |
|                                   |     limited local school          |
|                                   |     facilities for ABSTUDY away   |
|                                   |     from home rate                |
|                                   | -   Student has a disability. See |
|                                   |     Assessing disability for      |
|                                   |     ABSTUDY away from home rate   |
|                                   | -   Student is from an itinerant  |
|                                   |     family. See Assessing         |
|                                   |     itinerant family for ABSTUDY  |
|                                   |     away from home rate           |
|                                   | -   Student has been excluded     |
|                                   |     from attending local schools. |
|                                   |     See Assessing exclusion from  |
|                                   |     local schooling ABSTUDY away  |
|                                   |     from home rate                |
|                                   | -   Student has been subjected to |
|                                   |     racial discrimination of a    |
|                                   |     serious and continuing nature |
|                                   |     at the local schools. See     |
|                                   |     Assessing racial              |
|                                   |     discrimination for ABSTUDY    |
|                                   |     away from home rate           |
|                                   | -   Student has previously been   |
|                                   |     approved to live away from    |
|                                   |     home and it would be          |
|                                   |     unreasonable for the student  |
|                                   |     to break continuity of study. |
|                                   |     See Assessing continuity of   |
|                                   |     study for ABSTUDY away from   |
|                                   |     home rate                     |
|                                   | -   Student is receiving an       |
|                                   |     independent school            |
|                                   |     scholarship. See Assessing    |
|                                   |     scholarships and mobility     |
|                                   |     provisions for secondary      |
|                                   |     students for ABSTUDY away     |
|                                   |     from home rate                |
|                                   |                                   |
|                                   | If the above does not apply the   |
|                                   | eligibility for the away from     |
|                                   | home rate has not been met.       |
|                                   | Advise the student or Australian  |
|                                   | Apprentice of the decision. If    |
|                                   | this is due to insufficient       |
|                                   | evidence, when the required       |
|                                   | documents are provided the claim  |
|                                   | can be reviewed, See Making an    |
|                                   | Unfavourable decision (CLK).      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Compulsory requirement to reside  |
|                                   | at the educational institution +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Students may be approved for the  |
|                                   | away from home rate where their   |
|                                   | course of study requires          |
|                                   | compulsory residence.             |
|                                   |                                   |
|                                   | Where an Australian Apprentices   |
|                                   | has to live away from home for    |
|                                   | the study component of their      |
|                                   | Australian Apprenticeship, this   |
|                                   | is a block release period only    |
|                                   | and the away from home rate is    |
|                                   | not available in these            |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Students that are required to     |
|                                   | attend block release may qualify  |
|                                   | for Away from Base assistance,    |
|                                   | see, Away from Base assistance    |
|                                   | for ABSTUDY                       |
|                                   |                                   |
|                                   | If the student is living away     |
|                                   | from home because it is a         |
|                                   | compulsory requirement of the     |
|                                   | course to reside at the education |
|                                   | institution while studying, see   |
|                                   | Assessing course of study         |
|                                   | requires compulsory residence for |
|                                   | ABSTUDY away from home rate       |
|                                   |                                   |
|                                   | If the above does not apply the   |
|                                   | eligibility for the away from     |
|                                   | home rate has not been met.       |
|                                   | Advise the student or Australian  |
|                                   | Apprentice of the decision. If    |
|                                   | this is due to insufficient       |
|                                   | evidence, when the required       |
|                                   | documents are provided the claim  |
|                                   | can be reviewed, See Making an    |
|                                   | Unfavourable decision (CLK).      |
