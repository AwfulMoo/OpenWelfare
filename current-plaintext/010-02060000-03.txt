Assessing eligibility

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claim for the away from home      |
|                                   | rate + Read more ...              |
|                                   |                                   |
|                                   | Determine or reassess a student   |
|                                   | or Australian Apprentice's rate   |
|                                   | of ABSTUDY if they:               |
|                                   |                                   |
|                                   | -   lodge a claim for the Away    |
|                                   |     from Home rate, or            |
|                                   | -   advise of a change of         |
|                                   |     circumstances, (for example   |
|                                   |     change of address)            |
|                                   |                                   |
|                                   | To assess a claim for the Away    |
|                                   | from Home rate:                   |
|                                   |                                   |
|                                   | -   Go to the                     |
|                                   |     Independent/Homeless/Away     |
|                                   |     from Home Details (NIH)       |
|                                   |     screen                        |
|                                   | -   Check if the customer is      |
|                                   |     dependent or independent      |
|                                   |                                   |
|                                   | Is the customer independent?      |
|                                   |                                   |
|                                   | -   Yes, this process does not    |
|                                   |     apply. Procedure ends here    |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Living away from the permanent    |
|                                   | home + Read more ...              |
|                                   |                                   |
|                                   | Will the dependent customer live  |
|                                   | away from the permanent home?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the customer will be paid |
|                                   |     the at home rate. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Travel time and access + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the travel time and access  |
|                                   | between the permanent home and:   |
|                                   |                                   |
|                                   | -   the education institution for |
|                                   |     a student or                  |
|                                   | -   employer location for an      |
|                                   |     Australian Apprentice         |
|                                   |                                   |
|                                   | Is daily travel impossible or     |
|                                   | unreasonable?                     |
|                                   |                                   |
|                                   | -   Yes, see Assessing ABSTUDY    |
|                                   |     travel time and access for    |
|                                   |     the away from home rate       |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Home conditions + Read more ...   |
|                                   |                                   |
|                                   | Do the home conditions impact the |
|                                   | educational or training progress  |
|                                   | for the student or Australian     |
|                                   | Apprentice?                       |
|                                   |                                   |
|                                   | -   Yes, see Assessing ABSTUDY    |
|                                   |     home conditions for the away  |
|                                   |     from home rate                |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | State care + Read more ...        |
|                                   |                                   |
|                                   | Is the student or Australian      |
|                                   | Apprentice in State care and a    |
|                                   | foster parent is not receiving a  |
|                                   | foster care or similar allowance? |
|                                   |                                   |
|                                   | Yes, see ABSTUDY customer in      |
|                                   | state care                        |
|                                   |                                   |
|                                   | No, if the application is for a:  |
|                                   |                                   |
|                                   | -   secondary school student, go  |
|                                   |     to Step 6                     |
|                                   | -   tertiary student or an        |
|                                   |     Australian Apprentice, go to  |
|                                   |     Step 14                       |
+-----------------------------------+-----------------------------------+
| 6                                 | Approved special secondary        |
|                                   | course + Read more ...            |
|                                   |                                   |
|                                   | Is the student studying a special |
|                                   | secondary course at a government  |
|                                   | school?                           |
|                                   |                                   |
|                                   | -   Yes, See Assessing special    |
|                                   |     courses for ABSTUDY away from |
|                                   |     home rate                     |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Limited local school facilities + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Are the student's academic needs  |
|                                   | not met by local secondary        |
|                                   | facilities?                       |
|                                   |                                   |
|                                   | -   Yes, see Assessing limited    |
|                                   |     local school facilities for   |
|                                   |     ABSTUDY away from home rate   |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 8                                 | Disability requirement + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check if the student:             |
|                                   |                                   |
|                                   | -   has a physical or             |
|                                   |     intellectual disability       |
|                                   | -   has a psychological,          |
|                                   |     emotional or behavioural      |
|                                   |     problem                       |
|                                   | -   has a medical condition or    |
|                                   | -   is pregnant                   |
|                                   |                                   |
|                                   | Do they meet any of the above?    |
|                                   |                                   |
|                                   | -   Yes, see Assessing disability |
|                                   |     for ABSTUDY away from home    |
|                                   |     rate                          |
|                                   | -   No, go to Step 9              |
+-----------------------------------+-----------------------------------+
| 9                                 | Itinerant family + Read more ...  |
|                                   |                                   |
|                                   | Is the student’s family           |
|                                   | itinerant?                        |
|                                   |                                   |
|                                   | -   Yes, see Assessing itinerant  |
|                                   |     family for ABSTUDY away from  |
|                                   |     home rate                     |
|                                   | -   No, go to Step 10             |
+-----------------------------------+-----------------------------------+
| 10                                | Exclusion from local schooling +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Has the student has been excluded |
|                                   | from attending local school/s?    |
|                                   |                                   |
|                                   | -   Yes, see Assessing exclusion  |
|                                   |     from local schooling ABSTUDY  |
|                                   |     away from home rate           |
|                                   | -   No, go to Step 11             |
+-----------------------------------+-----------------------------------+
| 11                                | Racial discrimination + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Has the student has been          |
|                                   | subjected to racial               |
|                                   | discrimination of a serious and   |
|                                   | continuing nature at the local    |
|                                   | school/s?                         |
|                                   |                                   |
|                                   | -   Yes, see Assessing racial     |
|                                   |     discrimination for ABSTUDY    |
|                                   |     away from home rate           |
|                                   | -   No, go to Step 12             |
+-----------------------------------+-----------------------------------+
| 12                                | Continuity of study + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Has the student been approved to  |
|                                   | live away from home previously    |
|                                   | and it would be unreasonable for  |
|                                   | the student to break continuity   |
|                                   | of study?                         |
|                                   |                                   |
|                                   | -   Yes, see Assessing continuity |
|                                   |     of study for ABSTUDY away     |
|                                   |     from home rate                |
|                                   | -   No, go to Step 13             |
+-----------------------------------+-----------------------------------+
| 13                                | Scholarships and mobility         |
|                                   | provisions + Read more ...        |
|                                   |                                   |
|                                   | Is the student receiving a        |
|                                   | scholarship to attend a secondary |
|                                   | school or lives in Aurukun, Coen, |
|                                   | Hope Vale, Mossman Gorge or       |
|                                   | Doomadgee?                        |
|                                   |                                   |
|                                   | Yes, see Assessing scholarships   |
|                                   | and mobility provisions for       |
|                                   | secondary students for ABSTUDY    |
|                                   | away from home rate               |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   The customer is not eligible  |
|                                   |     for the Away from Home rate   |
|                                   | -   Tell the student or           |
|                                   |     Australian Apprentice of the  |
|                                   |     decision                      |
|                                   | -   A review of the decision can  |
|                                   |     occur if the customer         |
|                                   |     provides further evidence     |
|                                   | -   See Making an Unfavourable    |
|                                   |     decision (CLK)                |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 14                                | Compulsory requirement to reside  |
|                                   | at the educational institution +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Students may be approved for the  |
|                                   | Away from Home rate where their   |
|                                   | course of study requires          |
|                                   | compulsory residence. See         |
|                                   | Assessing course of study         |
|                                   | requires compulsory residence for |
|                                   | ABSTUDY away from home rate.      |
|                                   |                                   |
|                                   | Students or Australian            |
|                                   | Apprentices who need to live away |
|                                   | from home due to a block release  |
|                                   | are not eligible for the Away     |
|                                   | from Home rate. However, students |
|                                   | may qualify for Away from Base    |
|                                   | assistance for ABSTUDY.           |
|                                   |                                   |
|                                   | If the none of the above apply:   |
|                                   |                                   |
|                                   | -   Tell the student or           |
|                                   |     Australian Apprentice they    |
|                                   |     are not eligible for the Away |
|                                   |     from Home rate                |
|                                   | -   A review of the decision can  |
|                                   |     occur if the customer         |
|                                   |     provides further evidence     |
|                                   | -   See Making an Unfavourable    |
|                                   |     decision (CLK)                |
