Check the Homeless and Indigenous Australian care recipient form is complete

Table 2: this table describes how to check the Homelessness and
Indigenous Australian care recipient Notification for permanent care –
Confirmation section (pages 3 and 4) within the AC024 form is complete.

Note: use this table to check the AC024 form for the homeless supplement
on any day that is before 1 October 2022. There is no standard letter
template for this task.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check the form is complete + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Contact the residential aged care |
|                                   | service if any of the following   |
|                                   | information is not complete or    |
|                                   | unclear:                          |
|                                   |                                   |
|                                   | Items 1 and 2 Service details:    |
|                                   |                                   |
|                                   | -   Service Name                  |
|                                   | -   Service ID (NAPS ID)          |
|                                   |                                   |
|                                   | Items 3 to 6 Care Recipient       |
|                                   | Details:                          |
|                                   |                                   |
|                                   | -   surname/family name           |
|                                   | -   first given names             |
|                                   | -   second given name             |
|                                   | -   date of birth                 |
|                                   | -   gender                        |
|                                   | -   care recipient ID             |
|                                   |                                   |
|                                   | Item 7 Date of Entry (incorrect,  |
|                                   | blank or future date):            |
|                                   | Note: if the entry is a future    |
|                                   | date, the AC024 must be returned  |
|                                   | to the Service/Provider           |
|                                   |                                   |
|                                   | Item 8 Homelessness and/or        |
|                                   | Indigenous Australian status:     |
|                                   |                                   |
|                                   | -   homelessness                  |
|                                   | -   Indigenous Australian         |
|                                   | -   both                          |
|                                   |                                   |
|                                   | Item 10 Declaration by            |
|                                   | Residential Aged Care             |
|                                   | Service/Provider:                 |
|                                   |                                   |
|                                   | -   printed name (person          |
|                                   |     authorised by the             |
|                                   |     Service/Provider)             |
|                                   | -   telephone number              |
|                                   | -   signature                     |
|                                   | -   date                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Return the form to RACS + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The AC024 form is to be           |
|                                   | resubmitted by the residential    |
|                                   | aged care service if any of the   |
|                                   | following apply:                  |
|                                   |                                   |
|                                   | -   The form is incomplete        |
|                                   | -   The form has a future entry   |
|                                   |     date, and/or                  |
|                                   | -   The form is missing the       |
|                                   |     declaration/signature         |
|                                   |                                   |
|                                   | Contact the residential aged care |
|                                   | service outlining the reasons     |
|                                   | via:                              |
|                                   |                                   |
|                                   | -   phone, or                     |
|                                   | -   email                         |
|                                   |                                   |
|                                   | Record a note in PaNDA detailing  |
|                                   | the action taken.                 |
|                                   |                                   |
|                                   | Note: once the resubmitted        |
|                                   | application is received it will   |
|                                   | be treated as a new application.  |
+-----------------------------------+-----------------------------------+

Process Homeless and Indigenous supplements in ACSP

Table 3: this table describes how a Service Officer processes the
Homelessness and Indigenous Australian care recipient Notification for
permanent care – Confirmation section (pages 3 and 4) within the AC024
form in the Aged Care Staff Portal (ACSP).

Note: use this table to check the AC024 form for the homeless supplement
on any day that is before 1 October 2022.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Ensure all information required   |
|                                   | for processing has been           |
|                                   | provided + Read more ...          |
|                                   |                                   |
|                                   | See the Check the Homeless and    |
|                                   | Indigenous Australian Care        |
|                                   | Recipient form is complete table. |
+-----------------------------------+-----------------------------------+
| 2                                 | Check entry details in the Aged   |
|                                   | Care Staff Portal (ACSP) + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | The View entry - Step 4 screen of |
|                                   | the permanent entry event         |
|                                   | contains the                      |
|                                   | Homelessness/Indigenous details   |
|                                   | field                             |
|                                   |                                   |
|                                   | If the Homelessness/Indigenous    |
|                                   | details field does not display,   |
|                                   | the residential aged care service |
|                                   | is not approved to receive the    |
|                                   | supplement                        |
|                                   |                                   |
|                                   | Contact the service to advise     |
|                                   | them that the form cannot be      |
|                                   | processed due to the service's    |
|                                   | ineligibility to receive the      |
|                                   | supplement                        |
|                                   |                                   |
|                                   | -   Add a care recipient note in  |
|                                   |     the Aged Care Staff Portal    |
|                                   |     detailing the action taken.   |
|                                   |     See the Using the Aged Care   |
|                                   |     Staff Portal to manage care   |
|                                   |     recipient notes (Care         |
|                                   |     Recipient in Context (CRiC))  |
|                                   |     table                         |
|                                   | -   Record a note in PaNDA        |
|                                   |     detailing the action taken    |
|                                   |                                   |
|                                   | The coding of the care            |
|                                   | recipient’s eligibility for the   |
|                                   | supplement is completed in the    |
|                                   | ACSP as part of the care          |
|                                   | recipient’s entry event in        |
|                                   | Service in Context view.          |
|                                   |                                   |
|                                   | The entry event may already be    |
|                                   | registered in the ACSP. See Step  |
|                                   | 13 in the Using the Aged Care     |
|                                   | Staff Portal to search for a care |
|                                   | recipient (Service in Context)    |
|                                   | table.                            |
|                                   |                                   |
|                                   | Is there an entry event in ACSP?  |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Contact the residential aged  |
|                                   |     care service either by email  |
|                                   |     or phone and advise the AC024 |
|                                   |     form is received but there is |
|                                   |     no entry event for the        |
|                                   |     relevant care recipient       |
|                                   | -   Request the residential aged  |
|                                   |     care service to submit the    |
|                                   |     entry event either by using   |
|                                   |     the Aged Care Provider Portal |
|                                   |     (ACPP) (online), or the Aged  |
|                                   |     Care Entry Record form        |
|                                   |     (AC021) (manual)              |
|                                   | -   Advise the AC024 form cannot  |
|                                   |     not be processed until the    |
|                                   |     residential aged care service |
|                                   |     has submitted the entry event |
|                                   | -   Add a care recipient note in  |
|                                   |     the Aged Care Staff Portal    |
|                                   |     detailing the action taken.   |
|                                   |     See the Using the Aged Care   |
|                                   |     Staff Portal to manage care   |
|                                   |     recipient notes (Care         |
|                                   |     Recipient in Context (CRiC))  |
|                                   |     table                         |
|                                   | -   Record a note in PaNDA        |
|                                   |     detailing the action taken    |
|                                   | -   Procedure ends here until     |
|                                   |     required details are provided |
|                                   |                                   |
|                                   | Note: once the resubmitted        |
|                                   | application is received it will   |
|                                   | be treated as a new application.  |
+-----------------------------------+-----------------------------------+
| 3                                 | Process AC024 form using the Aged |
|                                   | Care Staff Portal (ACSP) + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To process the Homelessness and   |
|                                   | Indigenous Australian care        |
|                                   | recipient Notification for        |
|                                   | permanent care – Confirmation     |
|                                   | section (pages 3 and 4) within    |
|                                   | the AC024 form, from the ACSP     |
|                                   | Service in Context home page:     |
|                                   |                                   |
|                                   | -   Select Care recipient search  |
|                                   |     (service) from the Quick find |
|                                   |     search bar                    |
|                                   | -   The Care recipient search     |
|                                   |     (service) screen displays     |
|                                   | -   Complete the relevant fields  |
|                                   |     to generate search results    |
|                                   | -   Select the hyperlink for the  |
|                                   |     relevant Care recipient ID    |
|                                   | -   The care recipient summary    |
|                                   |     screen (Service in Context)   |
|                                   |     displays                      |
|                                   | -   Select Entry under the        |
|                                   |     Circumstances menu tab        |
|                                   | -   The Entry summary screen      |
|                                   |     displays                      |
|                                   | -   Select the hyperlink Status   |
|                                   |     in the Entry table to display |
|                                   |     the entry event               |
|                                   | -   The View entry - Step 1       |
|                                   |     screen displays               |
|                                   | -   Keep selecting Next until the |
|                                   |     View entry - Step 4 screen    |
|                                   |     displays                      |
|                                   | -   Select Update to display the  |
|                                   |     Update entry - Step 4 screen  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Code the care recipient’s         |
|                                   | Homeless/Indigenous eligibility + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Note: if the Neither homeless nor |
|                                   | Indigenous option is selected,    |
|                                   | key an Entry note detailing the   |
|                                   | action taken, for example: AC024  |
|                                   | form received DD/MM/YY, unable to |
|                                   | confirm care recipient            |
|                                   | eligibility for homelessness      |
|                                   | and/or Indigenous care needs,     |
|                                   | service notified by phone/email   |
|                                   | on DD/MM/YY. Go to Table 2 for    |
|                                   | the process to contact the        |
|                                   | service.                          |
|                                   |                                   |
|                                   | In the Update entry - Step 4      |
|                                   | screen:                           |
|                                   |                                   |
|                                   | Select the drop down box on the   |
|                                   | Homeless/Indigenous description   |
|                                   | field to display the following    |
|                                   | options:                          |
|                                   |                                   |
|                                   | -   Both homeless and Indigenous  |
|                                   | -   Homelessness                  |
|                                   | -   Indigenous Australian         |
|                                   | -   Neither homeless nor          |
|                                   |     Indigenous                    |
|                                   |                                   |
|                                   | Select the relevant option as per |
|                                   | the details provided at item 8 on |
|                                   | the form                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Add Entry notes + Read more ...   |
|                                   |                                   |
|                                   | Key clear and concise notes in    |
|                                   | the Entry notes field detailing   |
|                                   | the action taken.                 |
|                                   |                                   |
|                                   | Key a note (500 character limit)  |
|                                   | with the action taken and why,    |
|                                   | for example:                      |
|                                   |                                   |
|                                   | -   AC024 form received DD/MM/YY, |
|                                   |     care recipient identified as  |
|                                   |     having Homeless and/or        |
|                                   |     Indigenous related care needs |
|                                   |                                   |
|                                   | Select Submit                     |
|                                   |                                   |
|                                   | The SUCCESS Permanent residential |
|                                   | care entry is being processed     |
|                                   | message displays                  |
