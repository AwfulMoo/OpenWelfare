This page contains the procedure for creating a Contracted Service
Provider (CSP) Officer record in the Healthcare Identifier (HI) Service.

CSP Officer record in the HI Service

Create a CSP Officer record in the HI Service

This table describes how Service Officers create a CSP Officer record in
the HI Service.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on and perform a search +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Log on to the eBusiness Home      |
|                                   | Page.                             |
|                                   |                                   |
|                                   | Result: the Search for Individual |
|                                   | screen displays.                  |
|                                   |                                   |
|                                   | Perform a search for the CSP      |
|                                   | Officer.                          |
|                                   |                                   |
|                                   | Does a CSP Officer record exist?  |
|                                   |                                   |
|                                   | -   Yes, provide the CSP Officer  |
|                                   |     with the unique number        |
|                                   |     allocated by the system. The  |
|                                   |     process ends here             |
|                                   | -   No, click Create CSP Officer  |
|                                   |                                   |
|                                   | The Create CSP Officer screen     |
|                                   | displays defaulting to the Name   |
|                                   | Details tab.                      |
|                                   |                                   |
|                                   | Note: a maximum of 3 CSP Officers |
|                                   | can be linked to a CSP            |
|                                   | organisation at any one time.     |
|                                   |                                   |
|                                   | See also: Searching for records   |
|                                   | in the Healthcare Identifiers     |
|                                   | Service using the eHealth Program |
|                                   | (EHP).                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Key or select name details for    |
|                                   | the CSP Officer + Read more ...   |
|                                   |                                   |
|                                   | Key or select name details for    |
|                                   | the CSP Officer and select Add.   |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   In the Name Usage field,      |
|                                   |     always select Registered Name |
|                                   |     (Legal Name)                  |
|                                   | -   Name Usage Start Date         |
|                                   |     defaults to today's date      |
|                                   | -   Start Date Accuracy Indicator |
|                                   |     defaults to Day, month and    |
|                                   |     year are accurate             |
|                                   |                                   |
|                                   | Leave the following fields blank: |
|                                   |                                   |
|                                   | -   Name Usage End Date           |
|                                   | -   End Date Accuracy Indicator   |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   Invalid, messages are         |
|                                   |     returned which must be        |
|                                   |     actioned                      |
|                                   | -   Valid, go to Step 3           |
|                                   |                                   |
|                                   | Note: to add additional name      |
|                                   | details, select Create New Name   |
|                                   | and repeat this step.             |
+-----------------------------------+-----------------------------------+
| 3                                 | Evidence of identity details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Select the EOI Details tab    |
|                                   | -   In the EOI Documents field,   |
|                                   |     select the name of a document |
|                                   |     provided by the CSP Officer   |
|                                   |     and select Add                |
|                                   |                                   |
|                                   | Note: unless the CSP Officer is a |
|                                   | known customer, they must provide |
|                                   | 1 document each from both the     |
|                                   | primary and secondary groups.     |
|                                   |                                   |
|                                   | Note: if the CSP Officer is a     |
|                                   | known customer, select Known      |
|                                   | Medicare customer in the EOI      |
|                                   | Documents field.                  |
|                                   |                                   |
|                                   | The:                              |
|                                   |                                   |
|                                   | -   EOI Type field is not         |
|                                   |     populated                     |
|                                   | -   EOI option selected is        |
|                                   |     displayed at the bottom of    |
|                                   |     the screen                    |
|                                   |                                   |
|                                   | See also: Evidence of identity    |
|                                   | for Healthcare Identifiers (HI)   |
|                                   | Service                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Add evidence of identity          |
|                                   | documents + Read more ...         |
|                                   |                                   |
|                                   | Continue to add EOI documents to  |
|                                   | confirm evidence of identity:     |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Create New EOI                |
|                                   | -   the name of a document        |
|                                   |     provided by the CSP Officer   |
|                                   | -   Add                           |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   Invalid, messages are         |
|                                   |     returned which must be        |
|                                   |     actioned                      |
|                                   | -   Valid and evidence of         |
|                                   |     identity has been confirmed,  |
|                                   |     go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Activate personal details + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select Personal Details tab   |
|                                   | -   Key or select personal        |
|                                   |     details for the CSP Officer   |
|                                   | -   Select Activate               |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   Invalid, messages are         |
|                                   |     returned which must be        |
|                                   |     actioned                      |
|                                   | -   Valid, the screen rebuilds    |
|                                   |     and returns personal details  |
|                                   |     fields. Go to Step 6          |
+-----------------------------------+-----------------------------------+
| 6                                 | Address details + Read more ...   |
|                                   |                                   |
|                                   | Select Address Details tab        |
|                                   |                                   |
|                                   | The Unstructured Address          |
|                                   | Indicator tick box will be        |
|                                   | automatically selected. See       |
|                                   | Address formatting in Healthcare  |
|                                   | Identifiers                       |
|                                   |                                   |
|                                   | Enter the address in the Address  |
|                                   | Search box, starting with the     |
|                                   | number of the address. The rapid  |
|                                   | addressing tool will predict the  |
|                                   | address and display it as the     |
|                                   | number and street name starts to  |
|                                   | be entered. If the address that   |
|                                   | appears is the correct address    |
|                                   | select it                         |
|                                   |                                   |
|                                   | The following information will    |
|                                   | pre-populate into the relevant    |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Address Lines                 |
|                                   | -   Suburb/Town/Locality          |
|                                   | -   State                         |
|                                   | -   Postcode                      |
|                                   |                                   |
|                                   | Select Address Purpose            |
|                                   |                                   |
|                                   | Address Start Date (defaults to   |
|                                   | today’s date)                     |
|                                   |                                   |
|                                   | Start Date Accuracy Indicator     |
|                                   | (defaults to Day, month and year  |
|                                   | are accurate)                     |
|                                   |                                   |
|                                   | Select Add                        |
+-----------------------------------+-----------------------------------+
| 7                                 | Electronic communications         |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | -   Select Electronic             |
|                                   |     Communications Details tab    |
|                                   | -   Key or select electronic      |
|                                   |     communications details        |
|                                   | -   Select Add                    |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   Missing or invalid, messages  |
|                                   |     are returned which must be    |
|                                   |     actioned                      |
|                                   | -   Valid, the electronic         |
|                                   |     communications details        |
|                                   |     display at the bottom of the  |
|                                   |     screen. Go to Step 8          |
|                                   |                                   |
|                                   | Note: to add additional           |
|                                   | electronic communications         |
|                                   | details, select Create New        |
|                                   | Contact and repeat this step.     |
+-----------------------------------+-----------------------------------+
| 8                                 | Validate and submit + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Validate.                  |
|                                   |                                   |
|                                   | If there are:                     |
|                                   |                                   |
|                                   | -   Errors, messages are returned |
|                                   |     which must be actioned        |
|                                   | -   No errors, select Submit. The |
|                                   |     Confirm Create CSP Officer    |
|                                   |     screen displays               |
+-----------------------------------+-----------------------------------+
| 9                                 | Check for errors and amend + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If details are:                   |
|                                   |                                   |
|                                   | incorrect, select the following:  |
|                                   |                                   |
|                                   | -   Edit to return the Amend CSP  |
|                                   |     Officer screen                |
|                                   | -   Amend in the Action column    |
|                                   |     for the tab where the details |
|                                   |     are in error                  |
|                                   |                                   |
|                                   | correct, select Save              |
|                                   |                                   |
|                                   | Confirmation message displays the |
|                                   | CSP Officer ID.                   |
|                                   |                                   |
|                                   | Note: a letter will be manually   |
|                                   | prepared and addressed to the CSP |
|                                   | Officer to notify them of the     |
|                                   | record created for them. Where    |
|                                   | there is more than 1 name and     |
|                                   | address on the record, the mail   |
|                                   | out will be sent to the preferred |
|                                   | name and address.                 |
+-----------------------------------+-----------------------------------+
