|                                   |     location provider number may  |
|                                   |     be released to the nurse      |
|                                   |     practitioner or appropriate   |
|                                   |     third party                   |

|                                   |     activate the prescriber       |
|                                   |     number. The specialty code    |
|                                   |     651 is added to the stem and  |
|                                   |     location with the location    |
|                                   |     service ruled to prevent      |
|                                   |     claiming of Medicare benefits |
|                                   |     when not eligible. The        |

|                                   |     only                          |
