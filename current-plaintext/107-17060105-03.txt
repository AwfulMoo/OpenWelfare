This page contains information on the activation and retention processes
for a Departure Prohibition Order (DPO) or Departure Authorisation
Certificate (DAC).

DPO or DAC activation on departure of Australia

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Person attempts to leave          |
|                                   | Australian + Read more ...        |
|                                   |                                   |
|                                   | If a person attempts to depart    |
|                                   | Australia when a DPO is in force, |
|                                   | this is called an 'Activation'.   |
|                                   |                                   |
|                                   | -   If a DPO is in force and      |
|                                   |     there is no DAC, go to Step 2 |
|                                   | -   If a DPO is in force and      |
|                                   |     there is a DAC, go to Step 3  |
+-----------------------------------+-----------------------------------+
| 2                                 | Activation of a DPO + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Department of Home Affairs    |
|                                   | will raise an alert with the      |
|                                   | Australian Federal Police (AFP).  |
|                                   |                                   |
|                                   | The AFP will detain the person    |
|                                   | and contact Services Australia    |
|                                   | via the DPO Team AFP phone        |
|                                   | number. The AFP must be able to   |
|                                   | contact the DPO team 24/7.        |
|                                   |                                   |
|                                   | When the AFP phones following an  |
|                                   | activation, the DPO Officers      |
|                                   | confirms if the DPO is to be      |
|                                   | enforced. The DPO list (OPDPL)    |
|                                   | screen and DAC list (OPDAL)       |
|                                   | screens are checked.              |
|                                   |                                   |
|                                   | The AFP advise the DPO team:      |
|                                   |                                   |
|                                   | -   that the alert will be        |
|                                   |     retained for a further 360    |
|                                   |     days, and                     |
|                                   | -   the new 'Alert Expiry date'   |
|                                   |                                   |
|                                   | This occurs whether or not the    |
|                                   | person is allowed to depart.      |
|                                   | Note: the AFP do not send a       |
|                                   | confirmation email to the DPO     |
|                                   | team.                             |
|                                   |                                   |
|                                   | If the DPO is in force, the AFP   |
|                                   | advise the person:                |
|                                   |                                   |
|                                   | -   they cannot travel, and       |
|                                   | -   to contact the DPO team       |
|                                   |     during business hours         |
|                                   |                                   |
|                                   | If the AFP are unable to contact  |
|                                   | Services Australia and there is   |
|                                   | no DAC, they will deny departure  |
|                                   |                                   |
|                                   | If the person advises they have   |
|                                   | an urgent need to travel, the DPO |
|                                   | Team must confirm the reason See  |
|                                   | Departure Authorisation           |
|                                   | Certificate (DAC) processing      |
|                                   |                                   |
|                                   | For persons requesting urgent     |
|                                   | revoking of a DPO because they    |
|                                   | have no outstanding debts. See    |
|                                   | Issue, vary or revoke a Departure |
|                                   | Prohibition Order (DPO)           |
|                                   |                                   |
|                                   | Record the details on a DOC.      |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Activation of a DAC + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The person with a DAC will be     |
|                                   | able to leave Australia.          |
|                                   |                                   |
|                                   | The Department of Home Affairs    |
|                                   | will notify the AFP of the        |
|                                   | departure. The AFP will then      |
|                                   | email the DPO team to advise of   |
|                                   | the activation (departure) and    |
|                                   | provide the new DPO 'Alert Expiry |
|                                   | date'.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Recording the activation of a DPO |
|                                   | or DAC + Read more ...            |
|                                   |                                   |
|                                   | After the AFP advises that a      |
|                                   | person has either attempted to    |
|                                   | leave Australia (for a DPO) or    |
|                                   | has left Australia (for a DAC),   |
|                                   | record the details of the         |
|                                   | activation in the DPO system:     |
|                                   |                                   |
|                                   | -   navigate to the DPO list      |
|                                   |     (OPDPL) screen and select the |
|                                   |     appropriate DPO               |
|                                   | -   enter or overwrite the Last   |
|                                   |     Activation Date as the date   |
|                                   |     the person left or attempted  |
|                                   |     to leave Australia            |
|                                   | -   enter the logon ID of the DPO |
|                                   |     Officer contacted             |
|                                   | -   update the Alert Expiry date  |
|                                   |     as advised                    |
|                                   |                                   |
|                                   | Note: if there are multiple       |
|                                   | current DPOs or DACs each record  |
|                                   | will need to be updated.          |
|                                   |                                   |
|                                   | The future activity required for  |
|                                   | retaining a DPO needs to have the |
|                                   | due date updated:                 |
|                                   |                                   |
|                                   | -   cancel the existing future    |
|                                   |     activity on the Future        |
|                                   |     Activity List (FAL) screen    |
|                                   | -   create a new Manual Follow-up |
|                                   |     (MFU) activity. Record the    |
|                                   |     due date as 14 days before    |
|                                   |     the new 'Alert Expiry date'.  |
|                                   |     See How to code and action a  |
|                                   |     manual review                 |
+-----------------------------------+-----------------------------------+

Extension of time for retention of existing DPO

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Retaining a DPO + Read more ...   |
|                                   |                                   |
|                                   | The Australian Federal Police     |
|                                   | (AFP) set an expiry of 360 days   |
|                                   | when recording a DPO Alert on     |
|                                   | their system.                     |
|                                   |                                   |
|                                   | Note: Services Australia has 30   |
|                                   | days from the Alert expiry date   |
|                                   | to advise the AFP to retain the   |
|                                   | DPO. The AFP remove the alert if  |
|                                   | Services Australia does not       |
|                                   | advise the decision to retain the |
|                                   | DPO.                              |
|                                   |                                   |
|                                   | When a DPO is issued, a Manual    |
|                                   | Follow-up (MFU) activity is       |
|                                   | manually created on the Future    |
|                                   | Activity List (FAL) screen. The   |
|                                   | due date of the MFU is 14 days    |
|                                   | before the AFP expiry date.       |
|                                   |                                   |
|                                   | When the MFU matures a DPO        |
|                                   | Officer must review the case.     |
|                                   | They decide whether the DPO is    |
|                                   | still appropriate or whether it   |
|                                   | should be revoked. The DPO        |
|                                   | officer is to record the review   |
|                                   | and outcome details on a DOC.     |
|                                   |                                   |
|                                   | If a decision is made to keep the |
|                                   | DPO in place the AFP must be      |
|                                   | notified of the decision. The     |
|                                   | 'DHS Centrelink DPO Retention'    |
|                                   | email template is used to advise  |
|                                   | the AFP of the DPO retention.     |
|                                   |                                   |
|                                   | AFP will confirm they have        |
|                                   | received the retention email and  |
|                                   | advise the new expiry date to the |
|                                   | DPO Team. Record the details of   |
|                                   | the retention in the DPO system:  |
|                                   |                                   |
|                                   | -   navigate to the DPO list      |
|                                   |     (OPDPL) screen and select the |
|                                   |     appropriate DPO               |
|                                   | -   update the 'Alert Expiry      |
|                                   |     Date'                         |
|                                   | -   code a new future activity    |
|                                   |     for 14 days before the new    |
|                                   |     expiry date. See How to code  |
|                                   |     and action a manual review    |
|                                   |                                   |
|                                   | Note: if there are multiple DPOs  |
|                                   | current each record will need to  |
|                                   | be updated.                       |
|                                   |                                   |
|                                   | If the DPO should be revoked, see |
|                                   | Issue, vary or revoke a Departure |
|                                   | Prohibition Order (DPO).          |
