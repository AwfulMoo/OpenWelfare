|                                   |     medical practitioner’s        |
|                                   |     additional training in MHST - |
|                                   |     Non Vocationally Registered   |
|                                   |     (664)                         |
|                                   | -   If the medical practitioner   |
|                                   |     is only eligible to receive   |
|                                   |     VR benefits (see Step 5) add  |
|                                   |     specialty code 649 to the     |

|                                   |     medical practitioner’s        |
|                                   |     additional training in MHST – |
|                                   |     Vocationally Registered (649) |
|                                   | -   If the medical practitioner   |
|                                   |     is eligible to receive both   |
|                                   |     non-VR and VR benefits (at    |
|                                   |     different locations) add      |
|                                   |     specialty codes 664 and 649   |
|                                   |     to the stem. Go to Processing |
|                                   |     a medical practitioner’s      |
|                                   |     additional training in MHST – |
|                                   |     Non Vocationally Registered   |
|                                   |     (664). Once the steps in this |
|                                   |     table have been completed     |
|                                   |     then go to Processing a       |
|                                   |     health professional’s         |
|                                   |     additional training in MHST – |
|                                   |     Vocationally Registered (649) |
|                                   |                                   |
|                                   | Note: medical practitioners who   |
|                                   | have multiple placement programs  |
|                                   | are often eligible for both       |
|                                   | non-VR and VR benefits depending  |
|                                   | on their location and 3GA         |
|                                   | program. It is important to       |
|                                   | correctly assess eligibility and  |
|                                   | complete the correct processing   |
|                                   | table(s).                         |
