This page contains the process to follow when updating a Minor ID in the
Provider Directory System (PDS) via an inbound call.

On this Page:

Check a Minor ID is registered in the PDS

Update a Minor ID in the PDS

Check a Minor ID is registered in the PDS

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm Minor ID with customer +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   A Minor ID will (usually)     |
|                                   |     consist of 3 letters followed |
|                                   |     by 5 numbers                  |
|                                   | -   See the Major and Minor       |
|                                   |     customer IDs for Medicare     |
|                                   |     Online table for a full list  |
|                                   |     of Minor ID prefixes          |
+-----------------------------------+-----------------------------------+
| 2                                 | Check Minor ID is registered as   |
|                                   | an organisation in the PDS + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Access the PDS                |
|                                   | -   Select Organisation           |
|                                   | -   Select View                   |
|                                   | -   Select Basic Search           |
|                                   | -   Enter the Minor ID provided   |
|                                   |     from the customer in the      |
|                                   |     Search ID field               |
|                                   | -   Select MINOR ID in the ID     |
|                                   |     Type field                    |
|                                   | -   Select Search                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Assess search results + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the search results:            |
|                                   |                                   |
|                                   | -   populate with organisation    |
|                                   |     details, and                  |
|                                   | -   the Minor ID has been         |
|                                   |     previously registered in the  |
|                                   |     PDS, and                      |
|                                   | -   the Minor ID can be updated   |
|                                   |     via an inbound call           |
|                                   | -   go to Update a Minor ID in    |
|                                   |     the PDS table                 |
|                                   |                                   |
|                                   | If the search results:            |
|                                   |                                   |
|                                   | -   return the error message      |
|                                   |     ‘error.0224 No results found  |
|                                   |     for the search criteria’      |
|                                   | -   go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Minor ID not registered in the    |
|                                   | PDS + Read more ...               |
|                                   |                                   |
|                                   | Advise the caller:                |
|                                   |                                   |
|                                   | that the Minor ID has not been    |
|                                   | previously registered with        |
|                                   | Medicare, and                     |
|                                   |                                   |
|                                   | to submit the relevant banking    |
|                                   | details form, either:             |
|                                   |                                   |
|                                   | -   Online claiming provider      |
|                                   |     agreement form (HW027)        |
|                                   | -   Provider registration for     |
|                                   |     Electronic Funds Transfer     |
|                                   |     payments form (HW029), or     |
|                                   | -   Banking details online        |
|                                   |     claiming form (HW052)         |
|                                   |                                   |
|                                   | The Resources page includes links |
|                                   | to the forms.                     |
+-----------------------------------+-----------------------------------+

Update a Minor ID in the PDS

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check provider details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Conduct an eBusiness security     |
|                                   | check to confirm the providers    |
|                                   | name and practice address, see    |
|                                   | Verify caller identity for        |
|                                   | eBusiness Service Centre          |
|                                   | enquiries for information         |
|                                   |                                   |
|                                   | Confirm the caller’s details are  |
|                                   | correct:                          |
|                                   |                                   |
|                                   | -   Access the PDS                |
|                                   | -   Select Provider Location      |
|                                   | -   Select View                   |
|                                   | -   Select Basic Search           |
|                                   | -   Enter the provider’s stem     |
|                                   |     number in the Provider Stem   |
|                                   |     field                         |
|                                   | -   Enter the provider’s location |
|                                   |     in the Provider Location      |
|                                   |     field                         |
|                                   | -   Select Search                 |
|                                   | -   Assess the search results and |
|                                   |     confirm the details match the |
|                                   |     caller’s details              |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirm banking details + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Confirm banking details           |
|                                   | registered against the provider   |
|                                   | number.                           |
|                                   |                                   |
|                                   | From the Provider Location        |
|                                   | screen:                           |
|                                   |                                   |
|                                   | -   Select Program Registration   |
|                                   | -   Select View against the       |
|                                   |     relevant Program              |
|                                   |                                   |
|                                   | The caller must confirm the       |
|                                   | Current Payment Instruction       |
|                                   | details are correct, ask the      |
|                                   | caller for the BSB and account    |
|                                   | number                            |
|                                   |                                   |
|                                   | If the caller:                    |
|                                   |                                   |
|                                   | -   can confirm banking details,  |
|                                   |     go to Step 3                  |
|                                   | -   cannot confirm banking        |
|                                   |     details, go to Step 4         |
+-----------------------------------+-----------------------------------+
| 3                                 | Update Minor ID + Read more ...   |
|                                   |                                   |
|                                   | In the current screen:            |
|                                   |                                   |
|                                   | Select Edit                       |
|                                   |                                   |
|                                   | Enter the Minor ID provided by    |
|                                   | the customer in the MINOR ID      |
|                                   | field                             |
|                                   |                                   |
|                                   | Press [Enter]                     |
|                                   |                                   |
|                                   | The MAJOR ID will automatically   |
|                                   | populate                          |
|                                   |                                   |
|                                   | Note: if the MAJOR ID does not    |
|                                   | populate, see the Major and Minor |
|                                   | customer IDs for Medicare Online  |
|                                   | table                             |
|                                   |                                   |
|                                   | Select Validate                   |
|                                   |                                   |
|                                   | One of 3 information messages     |
|                                   | will appear, if:                  |
|                                   |                                   |
|                                   | -   ‘information.8801 Validation  |
|                                   |     completed successfully. DATA  |
|                                   |     NOT SAVED. (Validation        |
|                                   |     completed successfully and no |
|                                   |     errors or warnings            |
|                                   |     generated.)’. Select Save to  |
|                                   |     update Minor ID. Procedure    |
|                                   |     ends here                     |
|                                   | -   ‘warning.003 Check Minor Id.  |
|                                   |     Minor ID location is not      |
|                                   |     registered in ECLIPSE         |
|                                   |     Registration’. Select Save to |
|                                   |     update Minor ID. Procedure    |
|                                   |     ends here                     |
|                                   | -   ‘error.0005 Minor Id is       |
|                                   |     invalid. (Enter a valid       |
|                                   |     value)’. Go to Step 4         |
+-----------------------------------+-----------------------------------+
| 4                                 | Unable to update Minor ID + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Minor ID cannot be updated    |
|                                   | over the phone, if:               |
|                                   |                                   |
|                                   | -   the caller is unable to       |
|                                   |     confirm banking details, or   |
|                                   | -   an information message        |
|                                   |     ‘error.0005 Minor Id is       |
|                                   |     invalid. (Enter a valid       |
|                                   |     value)’ is returned when      |
|                                   |     attempting to update the      |
|                                   |     Minor ID                      |
|                                   |                                   |
|                                   | Advise the caller to submit the   |
|                                   | relevant banking details form,    |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   Online claiming provider      |
|                                   |     agreement form (HW027)        |
|                                   | -   Provider registration for     |
|                                   |     Electronic Funds Transfer     |
|                                   |     payments form (HW029), or     |
|                                   | -   Banking details online        |
|                                   |     claiming form (HW052)         |
|                                   |                                   |
|                                   | The Resources page includes links |
|                                   | to the forms.                     |
+-----------------------------------+-----------------------------------+
