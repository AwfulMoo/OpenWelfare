Determining exemption from requirement for payments to be made to parent/guardian

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Away from home rate + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the student approved for away  |
|                                   | from home rate of payments?       |
|                                   |                                   |
|                                   | Check the students away from home |
|                                   | status on the NIH screen.         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Does the parent/guardian receive  |
|                                   | all, or any, of the away from     |
|                                   | home rate of Living Allowance? +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A student who is approved for     |
|                                   | away from home rate has strict    |
|                                   | rules governing who the payee can |
|                                   | be, depending on the student's    |
|                                   | living arrangements. See Payment  |
|                                   | of away from home rate for        |
|                                   | ABSTUDY to determine who can be   |
|                                   | the payee for away from home      |
|                                   | living allowances.                |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, advise the student that   |
|                                   |     ABSTUDY provisions do not     |
|                                   |     allow for the change of payee |
|                                   |     to occur for students         |
|                                   |     approved for away from home   |
|                                   |     rate. Record a DOC on the     |
|                                   |     student's record. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Exemptions from having payments   |
|                                   | paid to parent(s)/guardian(s) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | ABSTUDY can be paid directly to   |
|                                   | the dependent 16-17 year old      |
|                                   | student, if:                      |
|                                   |                                   |
|                                   | -   the student is not benefiting |
|                                   |     from the allowances because   |
|                                   |     the payee is using it for     |
|                                   |     other purposes                |
|                                   | -   the payee leaves Australia    |
|                                   |     temporarily and the young     |
|                                   |     person cannot gain access to  |
|                                   |     ABSTUDY payments, or          |
|                                   | -   the parent(s)/guardian(s)     |
|                                   |     choose to have ABSTUDY paid   |
|                                   |     directly to the young person  |
|                                   |     and have authorised the       |
|                                   |     change in payee either        |
|                                   |     verbally or in writing. The   |
|                                   |     parent/guardian does not have |
|                                   |     to explain their decision.    |
|                                   |     Note: where a parent/guardian |
|                                   |     has requested payments be     |
|                                   |     directed to their 16-17 year  |
|                                   |     old dependant they can also   |
|                                   |     request that payments revert  |
|                                   |     back to their own account -   |
|                                   |     no explanation or reason is   |
|                                   |     needed                        |
|                                   |                                   |
|                                   | Has the student advised that they |
|                                   | meet any of the above exemption   |
|                                   | requirements for ABSTUDY payments |
|                                   | to be paid to                     |
|                                   | parent(s)/guardian(s)?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 4                                 | Substantiation of claim + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Advise the student that they will |
|                                   | be required to substantiate their |

|                                   | writing where the exemption       |
|                                   | reason is:                        |
|                                   |                                   |
|                                   | -   the student is not benefiting |
|                                   |     from the allowances because   |
|                                   |     the payee is using it for     |
|                                   |     other purposes, or            |
|                                   | -   the payee leaves Australia    |
|                                   |     temporarily and the young     |
|                                   |     person cannot gain access to  |
|                                   |     ABSTUDY payments              |
|                                   |                                   |
|                                   | Advise the student that a         |
|                                   | supporting statement from an      |

|                                   | Services Australia to make a      |
|                                   | decision.                         |

|                                   | Lodgement of written statements   |
|                                   |                                   |

|                                   | written statement(s) online via   |
|                                   | the [] Upload documents service,  |

|                                   | centre or an ABSTUDY Smart Centre |
|                                   | Processing team. Look up local    |
|                                   | Processing Service details for:   |
|                                   | ABSTUDY and ABSTUDY PES.          |

|                                   | supporting statements, whether    |
|                                   | ABSTUDY payments can be directed  |
|                                   | to the student.                   |
|                                   |                                   |
|                                   | -   Yes, see Step 1 in the        |
|                                   |     Updating payment destination  |
|                                   |     and nominee arrangements      |
|                                   |     table                         |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
