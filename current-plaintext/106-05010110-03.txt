Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | 1 January start date + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Use the start date of 1 January   |
|                                   | in the year the student commences |
|                                   | studies if they:                  |
|                                   |                                   |
|                                   | -   are eligible for ABSTUDY, and |
|                                   | -   have applied by the relevant  |
|                                   |     closing date                  |
|                                   |                                   |
|                                   | If the customer was paid Family   |
|                                   | Tax Benefit (FTB) (instead of     |
|                                   | ABSTUDY when student living at    |
|                                   | home) and they completed the      |
|                                   | school year, FTB entitlements may |
|                                   | not always pay up to 31 December. |
|                                   | ABSTUDY can be backdated to the   |
|                                   | student’s 16th birthday, which    |
|                                   | may create an FTB overpayment.    |
|                                   | The FTB customer and the student  |
|                                   | will need to decide which payment |
|                                   | is more beneficial before         |
|                                   | backdating ABSTUDY. In these      |
|                                   | cases, ABSTUDY is granted from    |
|                                   | the date paid +1 of FTB. See      |
|                                   | Study requirements for Family Tax |
|                                   | Benefit aged 16 years or over. If |
|                                   | the FTB customer cannot be        |
|                                   | contacted, ABSTUDY is only to be  |
|                                   | granted from date paid +1, unless |
|                                   | it is obvious that ABSTUDY would  |
|                                   | be more beneficial.               |
|                                   |                                   |
|                                   | Therefore, to ensure there is no  |
|                                   | gap in payments, a customer can   |
|                                   | transfer to ABSTUDY and be paid   |
|                                   | from date paid +1 of FTB (where   |
|                                   | the secondary student is living   |
|                                   | at home), as long as they:        |
|                                   |                                   |
|                                   | -   are intending to enrol in a   |
|                                   |     full-time or concessional     |
|                                   |     study load, in an approved    |
|                                   |     course, in the next available |
|                                   |     study period, and             |
|                                   | -   lodge a claim for ABSTUDY by  |
|                                   |     the closing date, which is 31 |
|                                   |     December in the year          |
|                                   |     secondary school was          |
|                                   |     finished, and                 |
|                                   | -   are eligible for ABSTUDY      |
|                                   |                                   |
|                                   | From 27 April 2020 customers      |
|                                   | entitled to receive ABSTUDY       |
|                                   | Living Allowance were eligible to |
|                                   | receive the fortnightly           |
|                                   | Coronavirus Supplement (CVS). See |
|                                   | Assessing a claim for ABSTUDY     |
|                                   | Schooling B Award, Tertiary Award |
|                                   | and Masters and Doctorate Award   |
|                                   | for information on CVS and when a |
|                                   | customer is in receipt of         |
|                                   | ABSTUDY.                          |
|                                   |                                   |
|                                   | If the secondary student is aged  |
|                                   | 16 years or over and is living    |
|                                   | away from home to study and       |
|                                   | boarding, FTB can continue to be  |
|                                   | paid in addition to the ABSTUDY.  |
|                                   |                                   |
|                                   | Service Officers will not be      |
|                                   | required to manually cancel FTB.  |
|                                   | The system will automatically     |
|                                   | cease FTB for the student from    |
|                                   | the grant date of the ABSTUDY (if |
|                                   | applicable). Where the parent has |
|                                   | no other dependent children, FTB  |
|                                   | will auto-cancel due to reason    |
|                                   | 'No dependent eligible children'  |
|                                   | (NDE).                            |
|                                   |                                   |
|                                   | Record details on a DOC.          |
|                                   |                                   |
|                                   | Go to Step 10.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | 1 July start date + Read more ... |

|                                   | year in which they submitted      |
