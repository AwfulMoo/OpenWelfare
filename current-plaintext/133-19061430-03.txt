Medicare online accounts

Risks to a customer's privacy have been identified. See Separating
Safely - protecting personal details to make sure the customer's
personal details are safe before progressing.

Staff training simulations are available for some services.

Make a Medicare claim

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Make a Medicare claim             |
|                                   |                                   |
|                                   | Customers can claim all Medicare  |
|                                   | services except for bulk-billed   |
|                                   | or simplified billing services.   |
|                                   |                                   |
|                                   | Customers need to have current    |
|                                   | Medicare eligibility and have     |
|                                   | bank details recorded. If the     |
|                                   | customer has an ‘Invalid bank     |
|                                   | details’ flag on their record,    |
|                                   | they will be presented with a     |
|                                   | task to confirm their bank        |
|                                   | details. Tasks will be displayed  |
|                                   | on the home page. The customer    |
|                                   | cannot submit any claims online   |
|                                   | until the task is completed.      |
|                                   |                                   |
|                                   | They will need the invoice or     |
|                                   | receipt from their health         |
|                                   | professional and details of:      |
|                                   |                                   |
|                                   | -   the service they are claiming |
|                                   |     (the service must have been   |
|                                   |     provided for someone on their |
|                                   |     Medicare card)                |
|                                   | -   referral details (if needed)  |
|                                   | -   the amount they paid          |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Claiming Medicare benefits    |
|                                   |     online                        |
|                                   | -   Online guide Medicare online  |
|                                   |     account help - Submit a claim |
|                                   | -   User help guide ‘Help         |
|                                   |     customers to make a Medicare  |
|                                   |     claim online’ on the          |
|                                   |     Resources page                |
|                                   | -   The Resources page for a link |
|                                   |     to the Services Australia     |
|                                   |     website for proof of payment  |
|                                   |     and advice on taking/scanning |
|                                   |     clear photos and documents    |
