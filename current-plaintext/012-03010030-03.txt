This page contains information on the how to use Health Professionals
(HP) Records Manager for Healthcare Identifiers (HI) Service.

Save completed HI applications in HP Records Manager

HP Records Manager - save completed HI applications

Table 1: This table describes the process Service Officers follow when
saving completed HI Service applications in HP Records Manager.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Navigate to network folder + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Open the network folder           |
|                                   | containing the Healthcare         |
|                                   | Identifiers (HI) application      |
|                                   | which needs to be stored in       |
|                                   | Health Professionals (HP) Records |
|                                   | Manager, for example:             |
|                                   | R:\NAT                            |
|                                   | \HEALTHEBUS\ECLAIM\UHI_Shared\000 |
|                                   | Tier 2 Melbourne Team\01.         |
|                                   | National Office to                |
|                                   | register\Applications for QC      |
|                                   | 131213                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Select appropriate application +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   right click on the Healthcare |
|                                   |     Identifiers (HI) application  |
|                                   |     which requires electronic     |
|                                   |     filing                        |
|                                   | -   select Send to then HP        |
|                                   |     Records Manager               |
|                                   | -   ensure 'Document' option is   |
|                                   |     selected then select OK       |
|                                   |     button                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Amend name + Read more ...        |
|                                   |                                   |
|                                   | In the New Record Document        |
|                                   | screen, amend the title name in   |
|                                   | the Title (Free Text Part) field. |
|                                   |                                   |
|                                   | Ensure the naming is in line with |
|                                   | the naming convention. See        |
|                                   | Administration for Healthcare     |
|                                   | Identifiers (HI) Service.         |
+-----------------------------------+-----------------------------------+
| 4                                 | Complete Health Professionals     |
|                                   | (HP) Records Manager container    |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | In Put On What File? field, type  |
|                                   | the HP Records Manager container  |
|                                   | the electronic document needs to  |
|                                   | be filed into. See Administration |
|                                   | for Healthcare Identifiers (HI)   |
|                                   | Service.                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Finalise + Read more ...          |
|                                   |                                   |
|                                   | Select OK.                        |
|                                   |                                   |
|                                   | Result: the Healthcare            |
|                                   | Identifiers (HI) application is   |
|                                   | saved in Health Professionals     |
|                                   | (HP) Records Manager with a HP    |
|                                   | Record Manager Record Number (for |
|                                   | example, 13/199999).              |
+-----------------------------------+-----------------------------------+

Record HP Records Manager record number

Record HP Records Manager record number in the HI service Applications Tracker

Table 2: This table describes how Tier 2 Service Officers record the
Health Professionals (HP) Records Manager Record Number in the
Healthcare Identifiers (HI) Service Applications Tracker. The HI Service
Tracker is maintained by Tier 2 and is used to record all applications
received for processing.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Search for Healthcare Identifiers |
|                                   | (HI) application + Read more ...  |
|                                   |                                   |
|                                   | -   open the HI Service           |
|                                   |     Applications Tracker          |
|                                   | -   search for and select the     |
|                                   |     required HI application       |
|                                   |     record                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Complete Health Professionals     |
|                                   | (HP) Records Manager Record       |
|                                   | Number + Read more ...            |
|                                   |                                   |
|                                   | In the Edit Application From      |
|                                   | Details screen, key the HP        |
|                                   | Records Manager Record Number     |
|                                   | into the HP Records Manager       |
|                                   | Reference field in the HI Service |
|                                   | Applications Tracker (for example |
|                                   | 13/199999).                       |
|                                   |                                   |
|                                   | Select the Close Form button (in  |
|                                   | the top right hand corner).       |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalise + Read more ...          |
|                                   |                                   |
|                                   | In the Close Form screen, select  |
|                                   | Yes. The Health Professionals     |
|                                   | (HP) Records Manager Record       |
|                                   | Number is recorded and saved in   |
|                                   | the Healthcare Identifiers (HI)   |
|                                   | Service Applications Tracker for  |
|                                   | the HI application.               |
|                                   |                                   |
|                                   | Note: an HI application may have  |
|                                   | more than 1 document requiring    |
|                                   | storage. Where an application has |
|                                   | multiple documents, enter the HP  |
|                                   | Records Manager Record Numbers    |
|                                   | as: '12/196274, 12/196275,        |
|                                   | 12/196280'.                       |
+-----------------------------------+-----------------------------------+

Search for stored HI applications

HP Records Manager - Searching for HI applications

Table 3: This table describes how Service Officers search for a
Healthcare Identifiers (HI) Service application once stored in Health
Professionals (HP) Records Manager using either a record number or the
applicant's name.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Open application and begin        |
|                                   | search + Read more ...            |
|                                   |                                   |
|                                   | -   Open the Health Professionals |
|                                   |     (HP) Records Manager Desktop  |
|                                   |     application                   |
|                                   | -   Select the Search by field    |
|                                   |     and select Record Number      |
+-----------------------------------+-----------------------------------+
| 2                                 | Code Record number + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | In the Equal to field, key the    |
|                                   | record number (for example,       |
|                                   | 13/199999) or the applicant's     |
|                                   | name as appropriate.              |
|                                   |                                   |
|                                   | Select [Enter].                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Open document + Read more ...     |
|                                   |                                   |
|                                   | When the stored record displays   |
|                                   | in a new window, double click on  |
|                                   | the document to open it. Close    |
|                                   | the window when when no longer    |
|                                   | required.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Check status + Read more ...      |
|                                   |                                   |
|                                   | Refer back to the Health          |
|                                   | Professionals (HP) Records        |
|                                   | Manager window where the document |
|                                   | was found and where the 'Edit     |
|                                   | Status' in the Properties preview |
|                                   | pane below says 'Checked In'.     |
|                                   |                                   |
|                                   | Note: when a document is opened   |
|                                   | in HP Records Manager the status  |
|                                   | reads 'Checked out' and the       |
|                                   | document is locked. The name of   |
|                                   | the person who opened the         |
|                                   | document is displayed. Other      |
|                                   | staff members are unable to open  |
|                                   | the same document.                |
+-----------------------------------+-----------------------------------+
