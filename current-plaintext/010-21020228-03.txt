Identifying enrolment reviews

Table 1:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Enrolment check + Read more ...   |
|                                   |                                   |
|                                   | Enrolment checks can be received  |
|                                   | either electronically or via a    |
|                                   | paper form, depending on the      |
|                                   | education provider recorded on    |
|                                   | the Education Details (EDC)       |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Where:                            |
|                                   |                                   |
|                                   | a paper enrolment check has been  |
|                                   | received:                         |
|                                   |                                   |
|                                   | -   the Study Load Compliance     |
|                                   |     processing team uploads the   |
|                                   |     review onto the education     |
|                                   |     provider's organisation CRN   |
|                                   |     (Org CRN) as a Q790 letter,   |
|                                   |     see Scanning Centrelink       |
|                                   |     documents using an MFD        |
|                                   | -   Workload Management (WLM)     |
|                                   |     creates a ZDMS_Q790 work item |
|                                   |     for allocation                |
|                                   |                                   |
|                                   | an electronic enrolment check has |
|                                   | been received in the Integrated   |
|                                   | Review System (IRS):              |
|                                   |                                   |
|                                   | -   an AIC Review Received DOC is |
|                                   |     recorded, and                 |
|                                   | -   a ZIAM_AIC_RVW work item is   |
|                                   |     created in WLM for allocation |
|                                   |                                   |
|                                   | If:                               |
|                                   |                                   |
|                                   | the applicant has provided        |
|                                   | documents in response to a        |
|                                   | request for information for an    |
|                                   | enrolment check, or an open DOC   |
|                                   | has fallen due, see Step 4 in     |
|                                   | Table 2                           |
|                                   |                                   |
|                                   | a ZDMS_Q790 work item has been    |
|                                   | allocated, go to Step 2           |
|                                   |                                   |
|                                   | a ZIAM_AIC_RVW work item has been |
|                                   | allocated, go to Step 3           |
|                                   |                                   |
|                                   | an applicant is enquiring about   |
|                                   | an enrolment check, and the       |
|                                   | student is recorded as attending  |
|                                   | a provider subject to:            |
|                                   |                                   |
|                                   | -   a paper enrolment check, go   |
|                                   |     to Step 2                     |
|                                   | -   an electronic enrolment       |
|                                   |     check, go to Step 3           |
+-----------------------------------+-----------------------------------+
| 2                                 | Paper enrolment checks + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Enrolment checks that contain     |
|                                   | information on multiple students, |
|                                   | are scanned onto the education    |
|                                   | provider's Organisation CRN (Org  |
|                                   | CRN).                             |
|                                   |                                   |
|                                   | To locate the Org CRN in Customer |
|                                   | First, note the student's         |
|                                   | Institution ID code, from the     |
|                                   | Education Course Details (EDC)    |
|                                   | screen:                           |
|                                   |                                   |
|                                   | -   enter Codes in the Next field |
|                                   | -   select Enter                  |
|                                   | -   select the Education          |
|                                   |     Institution Summary line by   |
|                                   |     placing an 'S' in the field   |
|                                   | -   select Continue, to display   |
|                                   |     the Education Institution     |
|                                   |     Summary (EIS) screen          |
|                                   | -   Code: field, enter the        |
|                                   |     Institution ID from the       |
|                                   |     student's EDC screen          |
|                                   | -   select Continue               |
|                                   | -   copy the Org CRN from the     |
|                                   |     Educational Inst Details      |
|                                   |     (EID) screen                  |
|                                   | -   select Home                   |
|                                   | -   enter the Org CRN into the    |
|                                   |     CRN/BP field                  |
|                                   | -   select Enter                  |
|                                   |                                   |
|                                   | Locate the scanned paper          |
|                                   | enrolment check via Document      |
|                                   | Tools, from Customer First        |
|                                   | workspace.                        |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Electronic enrolment checks +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Reviews returned electronically   |
|                                   | create an individual IRS review   |
|                                   | on each student record.           |
|                                   |                                   |
|                                   | There are four review types in    |
|                                   | IRS:                              |
|                                   |                                   |
|                                   | -   WSE - Withdrawn Secondary     |
|                                   | -   NSE - No Record - Secondary   |
|                                   | -   PSE - Part Time - Secondary   |
|                                   | -   LSE - Late Starter Secondary  |
|                                   |                                   |
|                                   | To locate the electronic          |
|                                   | enrolment check in the student's  |
|                                   | record:                           |
|                                   |                                   |
|                                   | -   enter IRRS in the Next field  |
|                                   | -   select Enter                  |
|                                   | -   code 'S' against the REG or   |
|                                   |     STA review to view the Review |
|                                   |     Details (IRRD) screen, which  |
|                                   |     includes information about    |
|                                   |     the dates relevant to the     |
|                                   |     review                        |
|                                   | -   code 'S' in the Review        |
|                                   |     Actions field, to view the    |
|                                   |     Review Action History (IRRA)  |
|                                   |     screen, which includes a      |
|                                   |     history of the review actions |
|                                   | -   select Continue to go back to |
|                                   |     the IRRD screen               |
|                                   | -   code 'S' in the Match Data    |
|                                   |     field, to view the Match Data |
|                                   |     (IRMD) screen, which includes |
|                                   |     information about the term in |
|                                   |     review, the education         |
|                                   |     provider, and the review type |
|                                   | -   select Continue to go back to |
|                                   |     the IRRD screen               |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Review enrolment check details +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Compare the information provided  |
|                                   | within the enrolment check        |
|                                   | against the information available |
|                                   | on the student's record,          |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   Education Course Details      |
|                                   |     (EDC) screen                  |
|                                   | -   Document List (DL), for any   |
|                                   |     information recorded in a DOC |
|                                   | -   scanned documents via         |
|                                   |     Document Tools                |
|                                   |                                   |
|                                   | Is there a discrepancy between    |
|                                   | the enrolment check and the       |
|                                   | student's record, or are any      |
|                                   | updates required?                 |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, see Step 1 in Table 3     |
+-----------------------------------+-----------------------------------+
| 5                                 | Outbound contact + Read more ...  |
|                                   |                                   |
|                                   | Where the information provided in |
|                                   | the enrolment check shows a       |
|                                   | change in circumstances, attempt  |
|                                   | to contact the applicant to       |
|                                   | verbally confirm the information  |
|                                   | provided.                         |
|                                   |                                   |
|                                   | Where the contact attempt is      |
|                                   | successful, see Step 1 in Table 2 |
|                                   |                                   |
|                                   | If no contact details are         |
|                                   | available, or contact is          |
|                                   | unsuccessful:                     |
|                                   |                                   |
|                                   | issue a Q781 letter requesting    |
|                                   | the applicant to provide          |
|                                   | verification of enrolment and/or  |
|                                   | study load                        |
|                                   |                                   |
|                                   | Record details in an open Fast    |
|                                   | Note. Select Auto Text option >   |
|                                   | Students, ABSTUDY & Apprentices > |
|                                   | AIC > AIC Enrolment Check         |
|                                   |                                   |
|                                   | Hold the Fast Note to the due     |
|                                   | date from the Q781 letter. Note:  |
|                                   | the open DOC acts as the review   |
|                                   | for return of requested documents |
|                                   |                                   |
|                                   | Where the enrolment check is:     |
|                                   |                                   |
|                                   | -   a paper review, go to Step 6  |
|                                   | -   an electronic review, go to   |
|                                   |     Step 7                        |
+-----------------------------------+-----------------------------------+
| 6                                 | Unsuccessful contact for paper    |
|                                   | review + Read more ...            |
|                                   |                                   |
|                                   | Have all student records          |
|                                   | identified on the paper enrolment |
|                                   | check been actioned?              |
|                                   |                                   |
|                                   | -   Yes, mark the scanned         |
|                                   |     document as Completed         |
|                                   | -   No, repeat steps for all      |
|                                   |     relevant student records      |
|                                   |     until all records are         |
|                                   |     actioned                      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | Update IRS with unsuccessful      |
|                                   | contact + Read more ...           |
|                                   |                                   |
|                                   | Update the IRS review to show     |
|                                   | unsuccessful contact with the     |
|                                   | applicant.                        |
|                                   |                                   |
|                                   | In the student's record:          |
|                                   |                                   |
|                                   | -   enter IRRS in the Next field  |
|                                   | -   'S'elect the REG or STA       |
|                                   |     review                        |
|                                   | -   enter IRCS in the Next field  |
|                                   | -   select Enter, to go to the    |
|                                   |     Review Contact Summary (IRCS) |
|                                   |     screen                        |
|                                   | -   Meth: field, PHO              |
|                                   | -   Date Contacted: field, the    |
|                                   |     date the attempt to contact   |
|                                   |     the applicant was made        |
|                                   | -   select Continue               |
|                                   | -   Confirm: field, Y             |
|                                   | -   select Continue               |
|                                   | -   record all relevant fields on |
|                                   |     the Review Contact Details    |
|                                   |     (IRCD) screen                 |
|                                   |                                   |
|                                   | Note: once a contact is           |
|                                   | initiated, the review status      |
|                                   | changes from Registered (REG) to  |
|                                   | started (STA).                    |
|                                   |                                   |
|                                   | Procedure ends here.              |

Assessment and coding of an enrolment check

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine impact of the enrolment |
|                                   | check + Read more ...             |
|                                   |                                   |
|                                   | Has the applicant confirmed the   |
|                                   | information provided in the       |
|                                   | enrolment check, or can confirm   |
|                                   | ongoing AIC entitlement without   |
|                                   | the need for further evidence?    |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Issue a Q781 letter           |
|                                   |     requesting the applicant      |
|                                   |     provide verification of       |
|                                   |     enrolment and/or study load   |
|                                   | -   Suspend the AIC payment       |
|                                   |     'OTH' - Other, to prevent     |
|                                   |     overpayment, see Suspension   |
|                                   |     of payments of Assistance for |
|                                   |     Isolated Children (AIC)       |
|                                   |     Scheme                        |
|                                   | -   Record details in an open     |
|                                   |     Fast Note. Select Auto Text   |
|                                   |     option > Students, ABSTUDY &  |
|                                   |     Apprentices > AIC > AIC       |
|                                   |     Enrolment Check               |
|                                   | -   Hold the Fast Note to the due |
|                                   |     date from the Q781 letter     |
|                                   | -   Go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Update student record + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Update the student's record:      |
|                                   |                                   |
|                                   | -   Complete any changes to       |
|                                   |     relevant screens. For         |
|                                   |     example, EDC                  |
|                                   | -   Restore the student's payment |
|                                   |     via the Benefit Action (BA)   |
|                                   |     screen, if there is current   |
|                                   |     entitlement. See Restoration  |
|                                   |     of Assistance for Isolated    |
|                                   |     Children (AIC) Scheme         |
|                                   |     payments                      |
|                                   | -   Update and Finalise the AIC   |
|                                   |     Enrolment Check DOC, see Step |
|                                   |     1 in Table 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Letter action + Read more ...     |
|                                   |                                   |
|                                   | Once the AIC payment is           |
|                                   | suspended, issue the correct      |
|                                   | suspension letter to the          |
|                                   | applicant:                        |
|                                   |                                   |
|                                   | -   go to the Letter Display (LD) |
|                                   |     screen                        |
|                                   | -   Letter Code: field, Q134      |
|                                   | -   select Enter                  |
|                                   | -   Service Reason: field, EIC    |
|                                   | -   REASON FOR DECISION field -   |
|                                   |     use free text to explain      |
|                                   | -   Phone Officer: field, 1 - Yes |
|                                   | -   Phone Number: field, 13 23 18 |
|                                   | -   complete all other relevant   |
|                                   |     fields                        |
|                                   | -   Command: field, FIN -         |
|                                   |     Finalise                      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Enrolment check activity falls    |
|                                   | due + Read more ...               |
|                                   |                                   |
|                                   | If the applicant has:             |
|                                   |                                   |
|                                   | -   supplied all necessary        |
|                                   |     evidence/information, go to   |
|                                   |     Step 5                        |
|                                   | -   not supplied all necessary    |
|                                   |     evidence/information, go to   |
|                                   |     Step 7                        |
|                                   | -   not responded to the letter   |
|                                   |     by the due date, go to Step 7 |
+-----------------------------------+-----------------------------------+
| 5                                 | Applicant responds + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Does the information/evidence     |
|                                   | supplied confirm ongoing AIC      |
|                                   | entitlement?                      |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   make any necessary updates to |
|                                   |     the student's circumstances   |
|                                   | -   Record details on a DOC. See  |
|                                   |     Step 1 in Table 3             |
|                                   |                                   |
|                                   | No, go to Step 6                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Eligibility not confirmed + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where AIC eligibility has ceased: |
|                                   |                                   |
|                                   | -   Make any necessary updates to |
|                                   |     the student's circumstances   |
|                                   | -   Cancel the AIC payment via    |
|                                   |     the Benefit Action (BA)       |
|                                   |     screen (if a manual           |
|                                   |     cancellation is required)     |
|                                   | -   Record details on a DOC, see  |
|                                   |     Step 1 in Table 3             |
+-----------------------------------+-----------------------------------+
| 7                                 | Cancellation of payment + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where the applicant has not       |
|                                   | responded to the request for      |
|                                   | information, or has not verified  |
|                                   | ongoing entitlement:              |
|                                   |                                   |
|                                   | -   Update the relevant screens   |
|                                   |     with the information from the |
|                                   |     enrolment check               |
|                                   | -   Where the updates do not      |
|                                   |     result in an automatic        |
|                                   |     cancellation, cancel the AIC  |
|                                   |     payment via the Benefit       |
|                                   |     Action (BA) screen. Use the   |
|                                   |     Reason Code: FRC - Failure to |
|                                   |     reply to correspondence       |
|                                   | -   Record details on a DOC, see  |
|                                   |     Step 1 in Table 3             |
+-----------------------------------+-----------------------------------+

Finalise enrolment check

Table 3:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identify debt, rate reduction or  |
|                                   | cancellation has occurred + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Has the enrolment check or        |
|                                   | assessment resulted in:           |
|                                   |                                   |
|                                   | a debt,                           |
|                                   |                                   |
|                                   | rate reduction, or                |
|                                   |                                   |
|                                   | cancellation of payment?          |
|                                   |                                   |
|                                   | -   Yes, note the debt period and |
|                                   |     amount. See Raising debts for |
|                                   |     students. To conduct          |
|                                   |     unfavourable decision         |
|                                   |     processing, see Advising      |
|                                   |     verbally of an unfavourable   |
|                                   |     decision (CLK). Go to Step 2  |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Finalise the enrolment check +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Written advice must be given to   |
|                                   | the applicant when contact has    |
|                                   | been made, but the applicant has  |
|                                   | not:                              |
|                                   |                                   |
|                                   | -   been advised of the           |
|                                   |     intervention finalisation, or |
|                                   | -   had any debt advice issued,   |
|                                   |     where appropriate             |
|                                   |                                   |
|                                   | Issue a Q765 letter to the        |
|                                   | applicant and, if required, the   |
|                                   | nominee, to advise the enrolment  |
|                                   | check is now complete.            |
|                                   |                                   |
|                                   | Finalise or cancel the review, as |
|                                   | appropriate. For:                 |
|                                   |                                   |
|                                   | electronic enrolment checks, see  |
|                                   | Finalising and attributing        |
|                                   | actions to a Service Update or    |
|                                   | Review in the Integrated Review   |
|                                   | System (IRS)                      |
|                                   |                                   |
|                                   | paper enrolment checks, have all  |
|                                   | student records identified on the |
|                                   | paper enrolment check been        |
|                                   | actioned?                         |
|                                   |                                   |
|                                   | -   Yes, mark the scanned         |
|                                   |     document as Completed         |
|                                   | -   No, repeat steps for all      |
|                                   |     relevant student records      |
|                                   |     until all records are         |
|                                   |     actioned                      |
