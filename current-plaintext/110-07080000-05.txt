Examples of when a student will not receive a review letter

Table 5

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | ABSTUDY Schooling A + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | An ABSTUDY Schooling A student    |
|                                   | whose current study details are   |
|                                   | recorded as undertaking Year 8 at |
|                                   | Hopetown High School. The system  |
|                                   | will automatically update the     |
|                                   | student to Year 9 at Hopetown     |
|                                   | High School.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Boarding + Read more ...          |
|                                   |                                   |
|                                   | An ABSTUDY student is recorded as |
|                                   | boarding and undertaking Year 10  |
|                                   | at Worawa College. The system     |
|                                   | will automatically update the     |
|                                   | student's study details to Year   |
|                                   | 11 at Worawa College. It will     |
|                                   | also update the boarding details  |
|                                   | and charges to the following      |
|                                   | year.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Tertiary + Read more ...          |
|                                   |                                   |
|                                   | An ABSTUDY tertiary student is    |
|                                   | recorded as undertaking the       |
|                                   | second year in a three year       |
|                                   | Bachelor Degree at Melbourne      |
|                                   | University. They will not need to |
|                                   | confirm if they are moving into   |
|                                   | third year of the course. Their   |
|                                   | payments will continue into the   |
|                                   | next calendar year.               |
