| 1                                 | Undertake proof of record         |
|                                   | ownership                         |
|                                   |                                   |
|                                   | Always check the customer's       |
|                                   | identity when dealing with        |
|                                   | requests to change payment        |
|                                   | destination and issuing refunds   |
|                                   | to former customers.              |
|                                   |                                   |
|                                   | As well as the normal             |
|                                   | authentication checks, ask the    |
|                                   | customer:                         |
|                                   |                                   |
|                                   | to state their current payment    |
|                                   | destination details (before the   |
|                                   | requested change). This can be    |
|                                   | either BSB code and account       |
|                                   | number, or branch name and        |
|                                   | account number                    |
|                                   |                                   |
|                                   | the approximate amount of their   |
|                                   | last Services Australia payment   |
|                                   | to that account. There are extra  |
|                                   | checks for customers turning 18   |
|                                   | who want to change payment into   |
|                                   | their own bank account            |
|                                   |                                   |
|                                   | if they are a JobSeeker Payment   |
|                                   | customer, if they had casual      |
|                                   | earnings on their last report:    |
|                                   |                                   |
|                                   | -   employer's name               |
|                                   | -   wages declared (accept        |
|                                   |     approximate figures)          |
|                                   |                                   |
|                                   | if applicable, their child’s      |
|                                   | middle name and date of birth     |
|                                   |                                   |
|                                   | There are no set questions. Take  |
|                                   | extra care and make sure the      |
|                                   | customer can answer questions     |
|                                   | that are not common knowledge to  |
|                                   | another person.                   |
|                                   |                                   |
|                                   | The customer may not know the     |
|                                   | organisation's account details if |
|                                   | the current payment destination   |
|                                   | is a rehabilitation centre,       |
|                                   | correctional centre trust account |
|                                   | or respite nursing home. The      |
|                                   | customer can state their previous |
|                                   | payment destination details and   |
|                                   | the amount of their last payment  |
|                                   | to that account. This does not    |
|                                   | include by group schedule.        |
|                                   |                                   |
|                                   | If the customer cannot provide    |
|                                   | these details or the details are  |
|                                   | incorrect, ask more questions     |
|                                   | before updating the account       |
|                                   | details. See Step 7 of            |
|                                   | Authenticating a customer.        |
