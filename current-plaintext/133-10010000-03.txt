Medicare claims history

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access service + Read more ...    |
|                                   |                                   |
|                                   | Sign in to myGov and select       |
|                                   | Medicare services.                |
|                                   |                                   |
|                                   | From the Home page, select        |
|                                   | History and statements > Claims   |
|                                   | history.                          |
|                                   |                                   |
|                                   | The Medicare Claims History page  |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Select date range + Read more ... |
|                                   |                                   |
|                                   | From the drop down list, the      |
|                                   | customer can choose to get a      |
|                                   | statement for:                    |
|                                   |                                   |
|                                   | -   Last 3 months                 |
|                                   | -   Last 6 months                 |
|                                   | -   Last 12 months                |
|                                   | -   Last 2 years                  |
|                                   | -   Last 3 years                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Select person + Read more ...     |
|                                   |                                   |
|                                   | Using the drop down menu, select  |
|                                   | the person the statement is for.  |
|                                   |                                   |
|                                   | The page only lists dependents    |
|                                   | who are under 14 years of age and |
|                                   | listed on the Medicare card.      |
|                                   |                                   |
|                                   | Note: customers need to make a    |
|                                   | separate request for each person. |
+-----------------------------------+-----------------------------------+
| 4                                 | Submit declaration + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Accept declaration box to  |
|                                   | agree to the declaration.         |
|                                   |                                   |
|                                   | The Medicare Claims History page  |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Select option + Read more ...     |
|                                   |                                   |
|                                   | Download history                  |
|                                   |                                   |
|                                   | Customers may require a statement |
|                                   | for the Compensation Recovery     |
|                                   | Program.                          |
|                                   |                                   |
|                                   | They need to request a Medicare   |
|                                   | Claims History Statement using    |
|                                   | the link on the Medicare Claims   |
|                                   | History page.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Return to Home page + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select Home.                      |
|                                   |                                   |
|                                   | The Home page displays.           |
+-----------------------------------+-----------------------------------+
