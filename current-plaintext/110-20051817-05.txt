+-----------------------------------+-----------------------------------+
| 1                                 | Earnings reporting Obligations    |
|                                   | (not for self-employed customers) |
|                                   |                                   |
|                                   | 'You need to report the gross     |
|                                   | income your employer paid you in  |
|                                   | your 14 day Centrelink reporting  |
|                                   | period. Your gross income is the  |
|                                   | amount your employer pays you     |
|                                   | before tax and other deductions.  |
|                                   | You can find your gross pay       |
|                                   | amount on your payslip.'          |
|                                   |                                   |
|                                   | 'If you don't get payslips, you   |
|                                   | should ask your employer to       |
|                                   | provide them or tell you the      |
|                                   | gross income. It's a requirement  |
|                                   | that all employers should provide |
|                                   | a payslip to their employees.'    |
|                                   |                                   |
|                                   | 'There are no changes to how you  |
|                                   | report the number of hours you    |
|                                   | worked, keep telling us how many  |
|                                   | hours you actually worked in your |
|                                   | reporting period, which may not   |
|                                   | match your payslip.'              |
|                                   |                                   |
|                                   | If they have a partner who is     |
|                                   | also working: 'It is important    |
|                                   | that you report any income you    |
|                                   | and your partner get from         |
|                                   | employers.'                       |
|                                   |                                   |
|                                   | 'If you are paid an allowance for |
|                                   | out of pocket expenses, for       |
|                                   | example, laundry, uniform or      |
|                                   | travel allowance, you do not need |
|                                   | to include this when reporting    |
|                                   | your gross income to us.'         |
|                                   |                                   |
|                                   | 'If you are paid any type of      |
|                                   | employment lump sums such as      |
|                                   | commissions, bonuses, leave       |
|                                   | and/or termination payments, or   |
|                                   | your income includes salary       |
|                                   | sacrificing and/or Fringe         |
|                                   | Benefits, these may be assessed   |
|                                   | differently and it is best to     |
|                                   | contact Services Australia.'      |
|                                   |                                   |
|                                   | 'If you get any compensation,     |
|                                   | WorkCover or income protection    |
|                                   | payments, you must advise         |
|                                   | Services Australia within 14      |
|                                   | days.'                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Earnings reporting obligations    |
|                                   | for self-employed customers       |
|                                   |                                   |
|                                   | Self-employed customers getting   |
|                                   | the JobKeeper Payment must report |
|                                   | changes to business profit by     |
|                                   | submitting an updated Profit and  |
|                                   | Loss (SU580) statement where      |
|                                   | there is a change to the overall  |
|                                   | business profit. JobKeeper will   |
|                                   | be assessed as part of the gross  |
|                                   | business income and must not be   |
|                                   | coded as earnings on the EANS     |
|                                   | screen.                           |
|                                   |                                   |
|                                   | 'If there has been a change in    |
|                                   | your overall business profit at   |
|                                   | any time from when you commenced  |
|                                   | receiving JobKeeper Payment, you  |
|                                   | must provide an amended Profit &  |
|                                   | Loss statement and include the    |
|                                   | JobKeeper Payment as business     |
|                                   | income.'                          |
|                                   |                                   |
|                                   | 'If there has been no change in   |
|                                   | the overall business profit, you  |
|                                   | do not need to provide an amended |
|                                   | Profit & Loss.'                   |
|                                   |                                   |
|                                   | 'If JobKeeper Payment income is   |
|                                   | not reported as business income   |
|                                   | and it is subsequently identified |
|                                   | that there has been an increase   |
|                                   | in profit, you may get a debt.'   |
|                                   |                                   |
|                                   | 'While you remain on a Centrelink |
|                                   | income support payment, you must  |
|                                   | provide us with a copy of your    |
|                                   | business and personal income tax  |
|                                   | return within 14 days from the    |
|                                   | date you lodge them with the      |
|                                   | ATO.'                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Where to report earnings          |
|                                   |                                   |
|                                   | 'Are you aware that you can       |
|                                   | report your employment income by  |
|                                   | going to my.gov.au, signing in    |
|                                   | and selecting Centrelink to       |
|                                   | access your online account or     |
|                                   | using the Express Plus Centrelink |
|                                   | mobile app?'                      |
|                                   |                                   |
|                                   | 'The agency understands that      |
|                                   | personal factors or circumstances |
|                                   | may sometimes prevent you from    |
|                                   | declaring the correct amount of   |
|                                   | your employment income for the    |
|                                   | days worked within your           |
|                                   | Centrelink fortnight period. If   |
|                                   | this happens:                     |
|                                   |                                   |
|                                   | 'You can correct your previously  |
|                                   | reported income for up to 6       |
|                                   | fortnights in the past through    |
|                                   | your Centrelink online account,   |
|                                   | or if you are unable to do this,  |
|                                   | you can contact Services          |
|                                   | Australia and we will assist you  |
|                                   | to correct the income.'           |
+-----------------------------------+-----------------------------------+
| 4                                 | Recovery Fee                      |
|                                   |                                   |
|                                   | A Recovery Fee is not applied to  |
|                                   | customers in receipt of AGE       |
|                                   | pension, Carer Payment (CP) or    |
|                                   | ABSTUDY.                          |
|                                   |                                   |
|                                   | The Recovery Fee will not apply   |
|                                   | to partner income even if the     |
|                                   | partner income is incorrectly     |
|                                   | declared or not declared.         |
|                                   |                                   |
|                                   | Suggested dialogue:               |
|                                   |                                   |
|                                   | 'Are you aware if you incorrectly |
|                                   | declare your income from          |
|                                   | employment, Services Australia    |
|                                   | may impose a 10% recovery fee to  |
|                                   | the debt?'                        |
|                                   |                                   |
|                                   | 'To avoid being incorrectly paid  |
|                                   | and having a 10% recovery fee     |
|                                   | imposed, it is important that you |
|                                   | declare all your employment       |
|                                   | income correctly on your          |
|                                   | reporting day.'                   |
+-----------------------------------+-----------------------------------+
| 5                                 | Working Credit/Income Bank/Work   |
|                                   | Bonus scheme                      |
|                                   |                                   |
|                                   | 'Are you aware of the Working     |
|                                   | Credit/Income Bank/Work Bonus     |
|                                   | scheme?'                          |
|                                   |                                   |
|                                   | 'The working credit/income        |
|                                   | bank/work bonus scheme allows you |
|                                   | to build up credits in each       |
|                                   | fortnight. When you have income   |
|                                   | from employment you can draw on   |
|                                   | your working credits/income       |
|                                   | bank/work bonus to reduce the     |
|                                   | effect of that income on your     |
|                                   | income support payment. Only      |
|                                   | income from your employment can   |
|                                   | be used to affect these credits   |
|                                   | in this way'.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Employment Diary and Worksheet    |
|                                   | (SU505)                           |
|                                   |                                   |
|                                   | 'To help your keep a track of     |
|                                   | your employment income, you can   |
|                                   | download an earnings worksheet    |
|                                   | (SU505) from Services Australian  |
|                                   | website                           |
|                                   | (www.servicesaustralia.gov.au).   |
|                                   | You can fill it out on your       |
|                                   | device or print it and compete it |

|                                   |                                   |
|                                   | 'This information will help you   |
