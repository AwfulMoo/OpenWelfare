Seasonal employment

Assisting job seekers seeking seasonal employment

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Seasonal employment + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If a job seeker is seeking        |
|                                   | information on seasonal           |
|                                   | employment, take the following    |
|                                   | action:                           |
|                                   |                                   |
|                                   | -   Refer to job search           |
|                                   |     facilities. See the Resources |
|                                   |     page for more information.    |
|                                   | -   Refer to local                |
|                                   |     industry/businesses/focus     |
|                                   |     points which support seasonal |
|                                   |     employment. See the Resources |
|                                   |     page for more information.    |
|                                   | -   Refer the job seeker to a     |
|                                   |     Harvest Labour Services (HLS) |
|                                   |     provider, or telephone the    |
|                                   |     National Harvest Labour       |
|                                   |     Information Service (NHLIS)   |
|                                   |     on 1800 062 332.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Moving to an Area of Lower        |
|                                   | Employment Prospects (MALEP) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Ensure to discuss the job         |
|                                   | seeker's mutual obligation        |
|                                   | requirements, particularly Moving |
|                                   | to an Area of Lower Employment    |
|                                   | Prospects (MALEP) that may be     |
|                                   | imposed.                          |
|                                   |                                   |
|                                   | For more information see Moving   |
|                                   | to an Area of Lower Employment    |
|                                   | Prospect (MALEP) employment       |
|                                   | related exclusion period.         |
+-----------------------------------+-----------------------------------+
| 3                                 | Income test + Read more ...       |
|                                   |                                   |
|                                   | Advise job seeker about the       |
|                                   | income test and the effect it     |
|                                   | will have on payments. See Income |
|                                   | Test for single allowance         |
|                                   | customers.                        |
|                                   |                                   |
|                                   | Note: job seeker's should be made |
|                                   | aware of the fact that upon       |
|                                   | reclaiming, a Seasonal Work       |
|                                   | Preclusion Period may apply.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Record details on a DOC + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Record details of the discussion  |
|                                   | with the customer about Seasonal  |
|                                   | Work on a DOC.                    |
