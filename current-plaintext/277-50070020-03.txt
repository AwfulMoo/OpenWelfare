| 3                                 | Assess if the customer is a       |
|                                   | member of an illness separated    |
|                                   | couple + Read more ...            |
|                                   |                                   |
|                                   | A couple is considered to be an   |
|                                   | illness separated couple where:   |
|                                   |                                   |
|                                   | they are unable to live together  |
|                                   | in their home, and                |
|                                   |                                   |
|                                   | the inability to live together:   |
|                                   |                                   |
|                                   | -   is due to illness or          |
|                                   |     infirmity of either or both   |
|                                   |     of them, and                  |
|                                   | -   results in their living       |
|                                   |     expenses being greater or are |
|                                   |     likely to be greater than     |
|                                   |     otherwise, and is likely to   |
|                                   |     continue indefinitely         |
|                                   |                                   |
|                                   | Aspects to consider when          |
|                                   | assessing if the customer is a    |
|                                   | member of an illness separated    |
|                                   | couple:                           |
|                                   |                                   |
|                                   | Inability to live at home due to  |
|                                   | illness or infirmity:             |
|                                   |                                   |
|                                   | -   it is not the intention that  |
|                                   |     medical evidence is required  |
|                                   |     for the majority of           |
|                                   |     assessments                   |
|                                   | -   for entry to an aged care     |
|                                   |     home, the RCA Institution     |
|                                   |     (RIS) screen can be used to   |
|                                   |     verify entry into residential |
|                                   |     care by checking Admission    |
|                                   |     Date and Care Type:           |
|                                   |     Residential                   |
|                                   | -   for a Supported Residential   |

|                                   |     be requested to confirm the   |
|                                   |     level of care provided to the |
|                                   |     customer                      |
|                                   | -   retirement villages do not    |

|                                   |     customer advises they are     |
|                                   |     receiving care from a         |

|                                   |     the care plan should be       |
|                                   |     requested                     |

| 4                                 | There is sufficient information   |
|                                   | to assess the illness             |
|                                   | separation + Read more ...        |
|                                   |                                   |
|                                   | When assessing the customer's     |
|                                   | circumstances it may become       |
|                                   | apparent that the customer is not |
|                                   | simply an illness separated       |
|                                   | couple but they may be either     |
|                                   | living separately and apart due   |
|                                   | to 1 member of a couple being     |
|                                   | institutionalised due to a severe |
|                                   | and debilitating illness or they  |
|                                   | are genuinely living separately   |
|                                   | and apart on a permanent or       |
|                                   | indefinite basis.                 |
|                                   |                                   |
|                                   | Living separately and apart due   |
|                                   | to 1 member of a couple being     |
|                                   | institutionalised due to a severe |
|                                   | and debilitating illness. An      |
|                                   | assessment of living separately   |
|                                   | and apart in this circumstance    |
|                                   | can only be made if:              |
|                                   |                                   |
|                                   | -   the incapacity of the         |
|                                   |     institutionalised partner is  |
|                                   |     severe, debilitating and of a |
|                                   |     permanent nature, for         |
|                                   |     example, Huntington's         |
|                                   |     disease, advanced stages of   |
|                                   |     Alzheimer's disease, and      |
|                                   | -   the institutionalised partner |
|                                   |     is totally and permanently    |
|                                   |     incapable of providing their  |
|                                   |     partner with such things as   |
|                                   |     companionship, comfort,       |
|                                   |     physical, intellectual or     |
|                                   |     emotional support             |
|                                   | -   evidence of the permanent or  |
|                                   |     indefinite separation must be |
|                                   |     provided from the nursing     |
|                                   |     home or approved institution, |
|                                   |     together with medical         |
|                                   |     evidence on the type and      |
|                                   |     severity of the illness       |
|                                   | -   separation can be a sensitive |
|                                   |     topic for some customers,     |
|                                   |     particularly elderly couples  |
|                                   |     who may not be able to        |
|                                   |     emotionally disconnect from   |
|                                   |     their partner, regardless of  |
|                                   |     the financial impacts. If a   |
|                                   |     customer still strongly       |
|                                   |     believes and states that they |
|                                   |     are a member of a couple with |
|                                   |     their partner, the records    |
|                                   |     are not to be unlinked. In    |
|                                   |     these situations, the         |
|                                   |     customer and partner will     |
|                                   |     still be considered to be an  |
|                                   |     illness separated couple      |
|                                   |                                   |
|                                   | Genuinely living separately and   |
|                                   | apart on a permanent or           |
|                                   | indefinite basis as there has     |
|                                   | been an estrangement or breakdown |
|                                   | in their relationship - this      |
|                                   | requirement must consider both a  |
|                                   | physical and emotional separation |
|                                   | between the customer and their    |
|                                   | partner                           |
|                                   |                                   |
|                                   | Considerations when assessing a   |
|                                   | customer as living separately and |
|                                   | apart due to 1 member of a couple |
|                                   | being institutionalised due to a  |
|                                   | severe and debilitating illness   |
|                                   |                                   |
|                                   | -   'Living separately and apart  |
|                                   |     on a permanent or indefinite  |
|                                   |     basis' should not generally   |
|                                   |     be accepted if the couple are |
|                                   |     living apart solely due to    |
|                                   |     ill health. All other options |
|                                   |     should be explored before it  |
|                                   |     has been determined that the  |
|                                   |     couple are living separately  |
|                                   |     and apart. References has     |
|                                   |     links to the Social Security  |
|                                   |     Guide                         |
|                                   | -   Careful consideration must be |
|                                   |     given to correctly assess the |
|                                   |     customer's circumstances as   |
|                                   |     it directly impacts on their  |
|                                   |     entitlements and rate of      |
|                                   |     income support payment        |
|                                   | -   Care must be taken to ensure  |
|                                   |     the consequences of the       |
|                                   |     decision when assessing a     |
|                                   |     customer as living separately |
|                                   |     and apart due to 1 member of  |
|                                   |     a couple being                |
|                                   |     institutionalised due to a    |
|                                   |     severe and debilitating       |
|                                   |     illness are discussed with    |
|                                   |     the customer prior to         |
|                                   |     unlinking. This may include   |
|                                   |     consultation with the FIS     |
|                                   |     Officer                       |
|                                   |                                   |
|                                   | The decision to assess a couple   |
|                                   | as Living separately and apart -  |
|                                   | 1 member of a couple              |
|                                   | institutionalised due to a severe |
|                                   | and debilitating illness, as      |
|                                   | outlined above, should only be    |
|                                   | undertaken if:                    |
|                                   |                                   |
|                                   | -   the couple's circumstances    |
|                                   |     have been fully investigated, |
|                                   |     and                           |
|                                   | -   the couple do not meet the    |
|                                   |     overall 5 factors that        |
|                                   |     determine a member of a       |
|                                   |     couple as per the Social      |
|                                   |     Security Guide. References    |
|                                   |     has a link to the Social      |
|                                   |     Security Guide, 1.1.M.120,    |
|                                   |     Member of a couple, and       |
|                                   | -   all scenarios and effects of  |
|                                   |     the income / assets tests     |
|                                   |     have been explored and        |
|                                   |     explained for now, for 2      |
|                                   |     years in the future and       |
|                                   |     beyond that                   |
|                                   |                                   |
|                                   | Resources has examples of the     |
|                                   | effect on the income and asset    |
|                                   | tests for a customer assessed as  |
|                                   | living separately and apart.      |
