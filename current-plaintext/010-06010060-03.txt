Explanation for questions on the Youth Allowance (YA) and ABSTUDY Statement by Young Person

This table describes an explanation for most questions on the Youth
Allowance and ABSTUDY: 'Unreasonable to live at home' Statement by Young
Person (SY015). Simple questions are not described.

+-----------------------------------+-----------------------------------+
| Category                          | Description                       |
+-----------------------------------+-----------------------------------+
| Question 1-6                      | Your Details + Read more ...      |
|                                   |                                   |
|                                   | The customer's present address.   |
|                                   | This is important as it enables   |
|                                   | the Service Officer within the    |
|                                   | O18 UTLAH Assessment Team to      |
|                                   | contact the customer if           |
|                                   | necessary.                        |
+-----------------------------------+-----------------------------------+
| Question 8                        | Why are you living away from your |
|                                   | parent(s)/guardian(s)' home? +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Need to select reason from list   |
|                                   | but not supply an explanation at  |
|                                   | this question.                    |
|                                   |                                   |
|                                   | If the reason is not in the list, |
|                                   | the customer can tick 'Other' and |
|                                   | state the reason why.             |
+-----------------------------------+-----------------------------------+
| Question 9-10                     | What date did you move out of     |
|                                   | your parent(s)/guardian(s)'       |
|                                   | home? + Read more ...             |
|                                   |                                   |
|                                   | Need to indicate the date that    |
|                                   | the customer left home; if they   |
|                                   | have left home previously, they   |
|                                   | can indicate the previous dates   |
|                                   | or estimates of dates as well.    |
+-----------------------------------+-----------------------------------+
| Question 15-17                    | Do your parent(s)/guardian(s)     |
|                                   | live together? + Read more ...    |
|                                   |                                   |
|                                   | If the parent(s)/guardian(s) are  |
|                                   | not living together, the customer |
|                                   | needs to indicate if they have    |
|                                   | had contact with the 'other       |
|                                   | parent/guardian' (the             |
|                                   | parent/guardian the customer did  |
|                                   | not normally live with).          |
|                                   |                                   |
|                                   | If they:                          |
|                                   |                                   |
|                                   | -   have not had contact with the |
|                                   |     'other parent/guardian', they |
|                                   |     need to state why not and     |
|                                   |     whether any attempts were     |
|                                   |     made to contact them          |
|                                   | -   have had contact with the     |
|                                   |     'other parent/guardian', the  |
|                                   |     customer needs to indicate if |
|                                   |     they get on with them, and if |
|                                   |     they would allow the customer |
|                                   |     to live with them. This       |
|                                   |     should include any conditions |
|                                   |     on such living arrangements,  |
|                                   |     or indications as to why they |
|                                   |     cannot live with the 'other   |
|                                   |     parent/guardian'              |
+-----------------------------------+-----------------------------------+
| Question 18                       | What incident(s) and/or           |
|                                   | problem(s) made you leave home +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The customer needs to explain why |
|                                   | s/he left home; they should       |
|                                   | include details or examples where |
|                                   | possible.                         |
|                                   |                                   |
|                                   | The customer can attach extra     |
|                                   | paper if there is not enough room |
|                                   | on the form to explain.           |
+-----------------------------------+-----------------------------------+
| Questions 19 - 28                 | Incident(s) and/or problem(s)     |
|                                   | that made you leave home + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The customer needs to explain     |
|                                   | what the problems are and include |
|                                   | details and examples where        |
|                                   | possible including:               |
|                                   |                                   |
|                                   | -   how long the problems have    |
|                                   |     been occurring                |
|                                   | -   whether they've tried to sort |
|                                   |     out their difficulties and if |
|                                   |     so, what was attempted or     |
|                                   |     reasons why they haven't      |
|                                   |     tried                         |
|                                   | -   the sort of contact they      |
|                                   |     currently have with parents   |
|                                   | -   whether the customer is       |
|                                   |     allowed to return home and if |
|                                   |     not, why not                  |
|                                   | -   whether the customer wants to |
|                                   |     return home and if not, why   |
|                                   |     not                           |
|                                   |                                   |
|                                   | The customer can attach extra     |
|                                   | paper if there is not enough room |
|                                   | on the form and should include as |
|                                   | much detail and/or examples as    |
|                                   | possible.                         |
+-----------------------------------+-----------------------------------+
| Question 29                       | Is there any other information    |
|                                   | you can provide that will help    |
|                                   | Centrelink decide if it is        |
|                                   | unreasonable for you to live at   |
|                                   | home with your                    |
|                                   | parent(s)/guardian(s)? + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The customer can add any extra    |
|                                   | information they feel may help    |
|                                   | them with their application and   |
|                                   | can add extra paper or any other  |
|                                   | documents or reports that they    |
|                                   | wish to provide.                  |
+-----------------------------------+-----------------------------------+
| Question 30                       | Declaration + Read more ...       |
|                                   |                                   |
|                                   | The customer needs to sign to     |
|                                   | declare that the information is   |
|                                   | all correct, and that s/he is     |
|                                   | aware there are penalties for     |
|                                   | giving false information and that |
|                                   | personal information is protected |
|                                   | by law.                           |
+-----------------------------------+-----------------------------------+
