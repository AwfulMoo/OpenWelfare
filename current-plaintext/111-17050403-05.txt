Common errors

Common errors, why they occur and how to resolve them.

+-----------------------------------+-----------------------------------+
| Item                              | Common errors                     |
+-----------------------------------+-----------------------------------+
| 1                                 | Image directory not found or      |
|                                   | cannot be opened                  |
|                                   |                                   |
|                                   | Reason: infrastructure issue with |
|                                   | File share                        |
|                                   |                                   |
|                                   | Solution: right click the batch   |
|                                   | and select Unreject               |
+-----------------------------------+-----------------------------------+
| 2                                 | Kofax Export Connector for IBM    |
|                                   | FileNet Content Manager - unable  |
|                                   | to connect                        |
|                                   |                                   |
|                                   | Reason: FileNet server is not     |
|                                   | available or connection to        |
|                                   | FileNet cannot be established     |
|                                   |                                   |
|                                   | Solution: ensure FileNet services |
|                                   | are up and running (contact your  |
|                                   | FileNet support team) – right     |
|                                   | click the batch and select        |
|                                   | Unreject to retry export          |
+-----------------------------------+-----------------------------------+
| 3                                 | File Path Cannot be opened        |
|                                   |                                   |
|                                   | Reason: file share or network     |
|                                   | issue                             |
|                                   |                                   |
|                                   | Solution: right click the batch   |
|                                   | and select Unreject               |
+-----------------------------------+-----------------------------------+
| 4                                 | Empty batch with No Name or       |
|                                   | System ID as Batch name           |
|                                   |                                   |
|                                   | Reason: this is due to an         |
|                                   | incorrect settings in KIC import  |
|                                   |                                   |
|                                   | Solution: note down the batch     |
|                                   | number, redirect to               |
|                                   | MIDRANGE.SYSTEMS.OPERATIONS with  |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
