| 2                                 | Both were long-term social        |
|                                   | security recipients + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | A Lump Sum Bereavement Payment    |
|                                   | (LBP) is payable to the surviving |
|                                   | partner where their new single    |
|                                   | rate is less than the couple's    |
|                                   | member of a couple rate at the    |
|                                   | time of death. The LBP represents |
|                                   | the difference between these 2    |
|                                   | rates for the duration of the     |
|                                   | bereavement period. Therefore, if |
|                                   | the member of a couple rate of    |
|                                   | payment has continued from the    |
|                                   | date of death, the LBP will be    |
|                                   | reduced                           |
|                                   |                                   |
|                                   | When a member of an allowee       |
|                                   | couple dies the LBP will need to  |
|                                   | be worked out manually, however,  |
|                                   | no Manual Follow-up (MFU) or      |
|                                   | activity is generated on the      |
|                                   | survivor's record to do this      |
|                                   |                                   |
|                                   | Follow the process to manually    |
|                                   | calculate the LBP:                |
|                                   |                                   |
|                                   | -   go to the One Off Payment     |
|                                   |     (OOP) screen                  |
|                                   | -   in the Effect Date field,     |
|                                   |     code the date of death        |
|                                   | -   in the Payment Type field,    |
|                                   |     select 'bls' (bereavement     |
|                                   |     lump sum)                     |
|                                   | -   press [Enter]                 |
|                                   | -   The Assessment Results (AR)   |
|                                   |     screen is displayed           |
|                                   | -   from the left hand menu       |
|                                   |     select Optional               |
|                                   |     Actions>Partner/E             |
|                                   | x-Partner>Pensions/Allowances>One |
|                                   |     Off Payments (OOP)            |
|                                   | -   Code the LBP amount in the    |
|                                   |     Amount column in the          |
|                                   |     Bereavement Payment line      |
|                                   | -   press [Enter] twice to return |
|                                   |     to the AR screen              |
|                                   | -   finalise activity and record  |
|                                   |     details on a DOC              |
|                                   |                                   |
|                                   | Is the surviving partner in       |
|                                   | receipt of JobSeeker Payment or   |
|                                   | Youth Allowance?                  |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No,                               |
