Q888 or Q999 example letter text

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | PPR not applied                   |
|                                   |                                   |
|                                   | When a decision is made not to    |
|                                   | apply PPR, issue a Q888 or Q999   |
|                                   | letter with the following         |
|                                   | information:                      |
|                                   |                                   |
|                                   | 'The decision has been made not   |
|                                   | to continue payment of Disability |
|                                   | Support Pension pending the       |
|                                   | outcome of your review.'          |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+
| 2                                 | PPR applied                       |
|                                   |                                   |
|                                   | When a decision is made to apply  |
|                                   | PPR, issue a Q888 or Q999 letter  |
|                                   | with the following information:   |
|                                   |                                   |
|                                   | 'The decision has been made to    |
|                                   | continue payment of Disability    |
|                                   | Support Pension pending the       |
|                                   | outcome of your review.'          |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+
| 3                                 | PPR ended                         |
|                                   |                                   |
|                                   | When a decision is made to end    |
|                                   | PPR, issue a Q888 or Q999 letter  |
|                                   | with the following information:   |
|                                   |                                   |
|                                   | 'We have been paying your         |
|                                   | Disability Support Pension under  |
|                                   | payment pending review provisions |
|                                   | whilst your review is being       |
|                                   | finalised. As the review is now   |
|                                   | complete (enter outcome of review |
|                                   | for example 'your payments have   |
|                                   | ceased' or 'your payments will    |
|                                   | continue under normal             |
|                                   | provisions').'                    |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+
