This page contains a list of approved activities that result in exit or
suspension from Workforce Australia/Community Development Program (CDP).

Suspension codes

Suspension code for approved activities

This table describes the automatic suspension codes from Workforce
Australia for job seekers who have mutual obligation requirements. These
activities are displayed on the Activity and Exemption Summary (AEX)
screen, which will result in automatic suspension.

+-----------------------------------+-----------------------------------+
| AEX code                          | Explanation                       |
+-----------------------------------+-----------------------------------+
| DFR                               | Defence Force Reserves - will     |
|                                   | suspend principal carers from     |
|                                   | Employment Provider Services.     |
+-----------------------------------+-----------------------------------+
| PFT                               | Parenting - Fully Approved Study  |
+-----------------------------------+-----------------------------------+
| PTW                               | Part Time Work                    |
+-----------------------------------+-----------------------------------+
| PWS                               | Paid Work or Self Employment -    |
|                                   | will suspend principal carers,    |
|                                   | partial capacity to work or       |
|                                   | mature age job seekers.           |
+-----------------------------------+-----------------------------------+
| SEL                               | Self Employment                   |
+-----------------------------------+-----------------------------------+
| SHD                               | Short Course -only for principal  |
|                                   | carers and job seekers who are    |
|                                   | assessed as having a partial      |
|                                   | capacity to work.                 |
+-----------------------------------+-----------------------------------+
| SPT                               | Job Seeker Part Time Student      |
|                                   | -only applies to those job        |
|                                   | seekers transferring from         |
|                                   | Disability Support Pension (DSP)  |
|                                   | (transitional) to JobSeeker       |
|                                   | Payment (JSP) / Youth Allowance   |
|                                   | (YAL) and continues to study on   |
|                                   | the same course on part-time      |
|                                   | basis, that is, at least          |
|                                   | equivalent to 50% of the          |
|                                   | full-time course load, which they |
|                                   | commenced while on DSP.           |
+-----------------------------------+-----------------------------------+
| SPW                               | Combination of paid work and      |
|                                   | study - only applies to principle |
|                                   | carers.                           |
+-----------------------------------+-----------------------------------+
| SSE                               | Combination of study and self     |
|                                   | employment - only applies to      |
|                                   | principal carers.                 |
+-----------------------------------+-----------------------------------+
| STF                               | Job Seeker Full Time Student      |
|                                   | -only applies to those job        |
|                                   | seekers transferring from DSP     |
|                                   | (transitional) to JSP/YA as       |
|                                   | having a partial capacity to work |
|                                   | or from Parenting Payment (PP) to |
|                                   | JSP/YA as a principal carer, and  |
|                                   | continue to study on the same     |
|                                   | course on a full-time basis, i.e. |
|                                   | at least equivalent to 75% of the |
|                                   | full-time course load, which they |
|                                   | commenced while on DSP or PP.     |
+-----------------------------------+-----------------------------------+
| VPE                               | Voluntary and Paid Work           |
|                                   | (Combined)                        |
+-----------------------------------+-----------------------------------+
| VWK                               | Voluntary Work                    |
+-----------------------------------+-----------------------------------+
| YAC                               | Youth Activities                  |
+-----------------------------------+-----------------------------------+

Note: any approved activities which are not listed above, will not
result in an automatic suspension of Employment Provider Services.
