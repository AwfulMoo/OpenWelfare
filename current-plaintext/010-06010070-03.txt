UTLAH Statement by Parent(s)

Explanation for questions on the Youth Allowance and ABSTUDY Statement by Parent(s) form

This table describes an explanation for most questions on the Youth
Allowance and ABSTUDY: 'Unreasonable to live at home' Statement by
Parent(s) (SY016). Simple questions have not been included.

+-----------------------------------+-----------------------------------+
| Category                          | Description                       |
+-----------------------------------+-----------------------------------+
| Question 1-2                      | Your details + Read more ...      |
|                                   |                                   |
|                                   | There are separate fields to      |
|                                   | capture parent(s)/guardian(s)     |
|                                   | details; if the parents/guardians |
|                                   | are not living together, they     |
|                                   | each need to complete a separate  |
|                                   | statement form.                   |
+-----------------------------------+-----------------------------------+
| Question 5                        | Does your child currently live    |
|                                   | with you? + Read more ...         |
|                                   |                                   |
|                                   | -   Yes, there is no need to      |
|                                   |     complete the form. The        |
|                                   |     parent(s)/guardian(s) can     |
|                                   |     sign and return the form to   |
|                                   |     Services Australia without    |
|                                   |     proceeding further            |
|                                   | -   No, parent/guardian needs to  |
|                                   |     indicate date that child last |
|                                   |     lived at home                 |
+-----------------------------------+-----------------------------------+
| Question 7                        | What incident(s) and/or           |
|                                   | problem(s) led your child to      |
|                                   | leave home? + Read more ...       |
|                                   |                                   |
|                                   | The parent/guardian needs to      |
|                                   | explain from their perspective    |
|                                   | why the child left home. Extra    |
|                                   | paper can be attached if there is |
|                                   | not enough room on the form.      |
+-----------------------------------+-----------------------------------+
| Questions 10-15                   | Problems between the family and   |
|                                   | the child. + Read more ...        |
|                                   |                                   |
|                                   | The parent/guardian needs to      |
|                                   | explain what sort of problems are |
|                                   | occurring with as many details or |
|                                   | examples where possible,          |
|                                   | including:                        |
|                                   |                                   |
|                                   | what the problems are and how     |
|                                   | long they've been occurring       |
|                                   |                                   |
|                                   | if anything has been tried to     |
|                                   | sort out the difficulties for     |
|                                   | example, counselling, mediation:  |
|                                   |                                   |
|                                   | -   No, list the reasons why not  |
|                                   | -   Yes, describe what has been   |
|                                   |     tried and whether it helped   |
|                                   |     or not                        |
|                                   |                                   |
|                                   | whether the child is allowed to   |
|                                   | return home to live               |
|                                   |                                   |
|                                   | -   Not allowed, list the reasons |
|                                   |     why                           |
|                                   | -   Allowed, list any conditions  |
|                                   |     that may apply                |
|                                   |                                   |
|                                   | how they are getting along with   |
|                                   | their child since they left home  |
|                                   | and what sort of contact they     |
|                                   | have with them                    |
+-----------------------------------+-----------------------------------+
| Question 16                       | Are you, or any other person      |
|                                   | providing your child with money   |
|                                   | or other support? + Read more ... |
|                                   |                                   |
|                                   | If the child is being provided    |
|                                   | with money or other support (e.g. |
|                                   | food, paying school fees, paying  |
|                                   | bills, living at a reduced rent   |
|                                   | in accommodation which the parent |
|                                   | owns), they need to provide       |
|                                   | details of what type of support   |
|                                   | is being provided, how often and  |
|                                   | who is providing it.              |
+-----------------------------------+-----------------------------------+
| Question 20                       | Are you living with your child's  |
|                                   | other parent/guardian? + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Yes, continue to Question 25  |
|                                   | -   No, they need to indicate if  |
|                                   |     the child has had any contact |
|                                   |     with the other                |
|                                   |     parent/guardian in the last   |
|                                   |     two years. If so, they need   |
|                                   |     to describe the relationship  |
|                                   |     between the child and the     |
|                                   |     other parent/guardian         |
+-----------------------------------+-----------------------------------+
| Question 25                       | Is there any other information    |
|                                   | you can provide that will help    |
|                                   | Services Australia decide if your |
|                                   | child is entitled to the          |
|                                   | Unreasonable to live at home      |
|                                   | rate? + Read more ...             |
|                                   |                                   |
|                                   | The parent(s)/guardian(s) can add |
|                                   | anything extra here that they     |
|                                   | feel may assist the agency decide |
|                                   | about the claim; extra pages can  |
|                                   | be attached if there is not       |
|                                   | enough room, or they can attach   |
|                                   | any other documents/reports. they |
|                                   | wish to provide.                  |
+-----------------------------------+-----------------------------------+
