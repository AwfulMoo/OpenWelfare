| 1                                 | Check clock days + Read more ...  |
|                                   |                                   |
|                                   | In preparation for the review,    |
|                                   | check the FHA clock to decide how |
|                                   | many days of payment the customer |
|                                   | has already received.             |
|                                   |                                   |
|                                   | Consider the customer's remaining |
|                                   | period of FHA entitlement in      |
|                                   | preparing the review discussion.  |
|                                   |                                   |
|                                   | If the customer is nearing or has |
|                                   | reached 730 days on payment,      |
|                                   | consider additional topics for    |
|                                   | discussion, including:            |
|                                   |                                   |
