Transferring a service in aged care

Table 3: This table describes the process for transferring a service
from one approved provider to another approved provider.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Email advice received by Services |
|                                   | Australia + Read more ...         |
|                                   |                                   |
|                                   | The Department of Health notifies |
|                                   | Services Australia of services    |
|                                   | that are transferring in the      |
|                                   | coming month.                     |
|                                   |                                   |
|                                   | Email notification is sent        |
|                                   | through to the Aged Care liaison  |
|                                   | inbox.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Checking the claim history + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Service Officers must check the   |
|                                   | claim history of the aged care    |
|                                   | service that is to be             |
|                                   | transferred. This service must be |
|                                   | up to date with their claims.     |
|                                   |                                   |
|                                   | For example, if the transfer date |
|                                   | is 1 June 2019, the aged care     |
|                                   | service is up to date with their  |
|                                   | claiming when the service has     |
|                                   | submitted the May 2019 claim.     |
|                                   |                                   |
|                                   | Is the aged care service up to    |
|                                   | date with their claims?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, contact the aged care     |
|                                   |     service and request to lodge  |
|                                   |     the remaining claims for the  |
|                                   |     transfer process to continue. |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Process last monthly claim + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To process the last monthly claim |
|                                   | in:                               |
|                                   |                                   |
|                                   | -   SPARC - see Process a claim   |
|                                   |     form table in Residential     |
|                                   |     Care monthly claim form       |
|                                   | -   ACMPS - see Process claim     |
|                                   |     summary table in Home Care    |
|                                   |     monthly claim form            |
+-----------------------------------+-----------------------------------+
| 4                                 | Suspend monthly advance + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Suspend the future monthly        |
|                                   | advance for the service that is   |
|                                   | due to be transferred. This       |
|                                   | ensures the correct payment of    |
|                                   | subsidies to the approved         |
|                                   | provider who is operating the     |
|                                   | service.                          |
|                                   |                                   |
|                                   | To suspend the advance in:        |
|                                   |                                   |
|                                   | SPARC - see Suspend an auto       |
|                                   | advance table in Advances in the  |
|                                   | System for the Payment of Aged    |
|                                   | Care Residential Care (SPARC)     |
|                                   |                                   |
|                                   | ACMPS - see Suspend and unsuspend |
|                                   | advances table in Advances in the |
|                                   | Aged Care Management Payments     |
|                                   | System (ACMPS)                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Check for updated provider        |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | The Department of Health sends    |
|                                   | updated approved provider details |
|                                   | via the National Approved         |
|                                   | Provider System (NAPS).           |
|                                   |                                   |
|                                   | This information flows through to |
|                                   | SPARC and/or ACMPS.               |
|                                   |                                   |
|                                   | Check SPARC and/or ACMPS daily to |
|                                   | ensure that the new approved      |
|                                   | provider details have been        |
|                                   | uploaded in the data exchange.    |
|                                   |                                   |
|                                   | Has the new provider details been |
|                                   | updated in SPARC and/or ACMPS?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, Service Officers will     |
|                                   |     need to ensure the escalation |
|                                   |     process is actioned           |
|                                   |     appropriately prior to        |
|                                   |     forwarding queries to the     |
|                                   |     Aged Care Support positional  |
|                                   |     mailbox. See Team Leader for  |
|                                   |     the escalation process.       |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 6                                 | Provider Statement and update     |
|                                   | bank account details + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The new approved provider must:   |
|                                   |                                   |
|                                   | -   complete the Aged Care        |
|                                   |     Provider Statement, and       |
|                                   | -   advise the bank account       |
|                                   |     details                       |
|                                   |                                   |
|                                   | These details are received via    |
|                                   | PaNDA (email, post and fax).      |
|                                   |                                   |
|                                   | To update the bank account        |
|                                   | details in:                       |
|                                   |                                   |
|                                   | -   SPARC - see Add bank account  |
|                                   |     details in SPARC table in     |
|                                   |     Bank account details for aged |
|                                   |     care services                 |
|                                   | -   ACMPS - see Maintain banking  |
|                                   |     details in ACMPS table in     |
|                                   |     Bank account details for aged |
|                                   |     care services                 |
+-----------------------------------+-----------------------------------+
| 7                                 | Lift suspension on advance + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Lift the suspension (e.g.         |
|                                   | unsuspend the advance suspension) |
|                                   | on the advance and pay any        |
|                                   | outstanding advances.             |
|                                   |                                   |
|                                   | To unsuspend advances in:         |
|                                   |                                   |
|                                   | -   SPARC - see Unsuspend an      |
|                                   |     advance table in Advances in  |
|                                   |     the System for the Payment of |
|                                   |     Aged Care Residential Care    |
|                                   |     (SPARC)                       |

|                                   |                                   |

|                                   |                                   |

|                                   |     an individual service advance |
|                                   |     table in Advances in the      |
|                                   |     System for the Payment of     |
|                                   |     Aged Care Residential Care    |
|                                   |     (SPARC)                       |
|                                   | -   ACMPS - see Verify and        |
|                                   |     approve an advance table in   |
|                                   |     Advances in the Aged Care     |
|                                   |     Management Payments System    |
|                                   |     (ACMPS)                       |
+-----------------------------------+-----------------------------------+
| 8                                 | Reply to the Department of Health |
|                                   | when processed + Read more ...    |
|                                   |                                   |
|                                   | Reply the Department of Health    |
|                                   | when all actions have been        |
|                                   | processed.                        |

Checking variations to aged care approved places

Table 4: This table describes how Team Leaders and Subject Matter
Experts can check if a variation to aged care approved places has been
made. Service Australia does not have the system access to make these
changes.

Note: approved places are not applicable to home care.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Email advice received by Services |
|                                   | Australia + Read more ...         |
|                                   |                                   |
|                                   | The Department of Health notifies |
|                                   | Services Australia of services    |
|                                   | whose approved places have either |
|                                   | increased or decreased. This      |
|                                   | could vary in format depending on |
|                                   | the state/territory office of the |
|                                   | Department of Health.             |
|                                   |                                   |
|                                   | Services Australia receive email  |
|                                   | notification from the Department  |
|                                   | of Health through the Aged Care   |
|                                   | liaison inbox.                    |

| 2                                 | Identify increase/decrease in     |
|                                   | approved places + Read more ...   |
|                                   |                                   |
|                                   | Is there an increase/decrease in  |
|                                   | places (> 10%) for the service?   |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, procedure ends here       |
|                                   |                                   |
|                                   | Note: this ensures government     |
|                                   | aged care subsidies are paid      |
|                                   | correctly.                        |

|                                   | more ...                          |

|                                   |     Service Details (Page 1 of 2) |

|                                   | -   Press [Enter]. The Browse     |
|                                   |     Service Details (Page 2 of 2) |
|                                   |     screen will display           |

+-----------------------------------+-----------------------------------+
| 5                                 | Checking approved places in       |
|                                   | ACMPS + Read more ...             |
|                                   |                                   |
|                                   | The Department of Health will     |
|                                   | update the number of approved     |
