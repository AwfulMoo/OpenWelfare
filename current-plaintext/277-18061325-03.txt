This page contains information about the process of checking and
collecting Child Support overpayments. It also contains information
about other payee debts and the associated collection actions.

On this page:

|                                   | has to repay some or all of the   |
|                                   | overpayment.                      |

|                                   | The action taken to address an    |
|                                   | overpayment depends on how it was |
|                                   | created. Decide if the            |

|                                   | discussing with the payer and/or  |
|                                   | payee. Do not recover             |
|                                   | overpayments from retrospective   |
|                                   | variations caused when:           |
|                                   |                                   |

|                                   |     with their legal obligation   |
|                                   |     to lodge their tax return on  |
|                                   |     time, go to Step 4            |
|                                   | -   the payer failed to advise us |
|                                   |     of a change of address which  |
|                                   |     resulted in a terminating     |
|                                   |     event, go to Step 5           |
|                                   | -   the assessment ended as a     |
|                                   |     result of an objection, AAT   |
|                                   |     decision or court order which |
|                                   |     determined that the           |
|                                   |     application for assessment    |
|                                   |     should not have been          |
|                                   |     accepted, go to Step 7        |
|                                   | -   a court made a declaration    |
|                                   |     under section 107 of the      |
|                                   |     Child Support (Assessment)    |
|                                   |     Act 1989, go to Step 6        |
|                                   |                                   |

|                                   | new and existing overpayments.    |
|                                   |                                   |

|                                   | policy was applied. If not,       |

|                                   | should be applied to the          |
|                                   | remaining overpayment.            |
|                                   |                                   |
|                                   | If it is determined that an       |
|                                   | overpayment is not or may not be  |
|                                   | recoverable, action must be taken |

|                                   | overpayment. This applies even if |
|                                   | Service Support Officer (SSO)     |

|                                   |                                   |
|                                   | -   use the subject line          |
|                                   |     Overpayment Policy            |
|                                   | -   document that the payment was |

|                                   |     overpayment policy and not    |

|                                   |     waive the debt                |

| 4                                 | Overpayment caused by late        |
|                                   | lodgement of tax return + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the payer lodges a tax return  |
|                                   | later than the timeframes set by  |
|                                   | the Australian Taxation Office    |
|                                   | (ATO), the new income may replace |
|                                   | a higher income being used in the |
|                                   | assessment, this may create an    |
|                                   | overpayment. Because the payer    |
|                                   | has failed to comply with their   |
|                                   | legal obligation and this has     |
|                                   | resulted in an overpayment, Child |
|                                   | Support will not recover the      |
|                                   | overpayment.                      |
|                                   | Note: this usually applies to     |
|                                   | periods prior to July 2008 as new |
|                                   | ATO income rules generally        |
|                                   | prevent overpayments occurring    |
|                                   | due to late lodgement.            |
|                                   |                                   |
|                                   | If the customer lodged a late tax |
|                                   | return after June 2008 and it is  |
|                                   | a reassessment based on Section   |
|                                   | 11 of the Child Support           |
|                                   | (Assessment) Regulations 2018,    |
|                                   | refer the case to Program Advice  |
|                                   | via a Service Support Officer.    |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Income processing for Child   |
|                                   |     Support customers             |
|                                   | -   Technical and policy          |
|                                   |     escalation process            |
|                                   |                                   |
|                                   | Prevent automatic recovery of the |
|                                   | overpayment while technical       |
|                                   | advice is being sought. Where the |
|                                   | payer has asked the ATO to        |
|                                   | retrospectively record that they  |
|                                   | are ‘not required to lodge’ for a |
|                                   | past year and this results in an  |
|                                   | overpayment, Child Support will   |
|                                   | not recover the overpayment.      |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+
| 5                                 | Overpayment caused by residency   |
|                                   | decision + Read more ...          |
|                                   |                                   |
|                                   | When a payer changes their        |
|                                   | address, they are required to     |
|                                   | notify Child Support within       |
|                                   | fourteen days.                    |
|                                   |                                   |
|                                   | If a payer changes their address  |
|                                   | to a non-reciprocating            |
|                                   | jurisdiction and they have ceased |
|                                   | to be an Australian resident a    |
|                                   | terminating event will occur. The |
|                                   | terminating event date will most  |
|                                   | likely have taken place from a    |
|                                   | date in the past, this will cause |
|                                   | an overpayment.                   |
|                                   |                                   |
|                                   | Child Support will not recover    |
|                                   | the overpayment if the payer did  |
|                                   | not advise of the address change  |
|                                   | (within fourteen days) resulting  |
|                                   | in a delayed residency decision   |
|                                   | and terminating event. Remove the |
|                                   | overpayment at the same time      |
|                                   | residency decision is made. See   |
|                                   | Residency decisions for Child     |
|                                   | Support customers - Specialist    |
|                                   | Service Officer.                  |
|                                   |                                   |
|                                   | Note: extra time can be allowed   |
|                                   | to allow for normal passage of    |
|                                   | the post or any special           |
|                                   | circumstances. The customer must  |
|                                   | advise Child Support as soon as   |
|                                   | is reasonably practical.          |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+
| 6                                 | Section 107 court orders + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If an overpayment occurred due to |
|                                   | a declaration under section 107   |
|                                   | of the Child Support (Assessment) |
|                                   | Act 1989 there is no longer a     |
|                                   | registered maintenance liability. |
|                                   |                                   |
|                                   | The overpayment is not a debt due |
|                                   | to the Commonwealth and therefore |
|                                   | not recoverable.                  |
|                                   |                                   |
|                                   | For more information, see Section |
|                                   | 107 order received by Child       |
|                                   | Support and follow this process.  |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+
| 7                                 | Assessment ends due to objection, |
|                                   | AAT or court order + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | When the outcome of an objection, |
|                                   | AAT decision or court order       |
|                                   | (other than an order made under   |
|                                   | section 107) means that an        |
|                                   | application for a child support   |
|                                   | assessment should not have been   |
|                                   | accepted:                         |
|                                   |                                   |
|                                   | -   the assessment ends from the  |
|                                   |     start date                    |
|                                   | -   there is no longer a          |
|                                   |     registered maintenance        |
|                                   |     liability                     |
|                                   |                                   |
|                                   | See Ending case or liability.     |
|                                   |                                   |
|                                   | The overpayment is not a debt due |
|                                   | to the Commonwealth. Therefore,   |
|                                   | Child Support cannot recover the  |
|                                   | overpayment from the payee. The   |
|                                   | overpayment must be removed at    |
|                                   | the same time that the            |
|                                   | review/appeal decision is         |
|                                   | implemented to prevent any        |
|                                   | automated debt recovery.          |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |

+-----------------------------------+-----------------------------------+

|                                   | more ...                          |

|                                   |                                   |
|                                   | the amount of the overpayment     |

|                                   |                                   |

|                                   |     the Election to waive table   |

|                                   |     apply against future          |

|                                   |     they are willing to continue  |
|                                   |     to pay an amount for child    |
|                                   |     support if the payee requests |
|                                   |     recovery at less than 100% of |
|                                   |     the ongoing liability, see WA |
|                                   |     Ex-nuptial child support      |
|                                   |     cases - Negotiating a Child   |
|                                   |     Support debt repayment and    |
|                                   |     payee payment arrangement     |
|                                   |     defaults                      |
|                                   | -   attempt collection from the   |
|                                   |     payee when there is no        |
|                                   |     ongoing liability, see WA     |
|                                   |     Ex-nuptial child support      |
|                                   |     cases - Negotiating a Child   |
|                                   |     Support debt repayment and    |
|                                   |     payee payment arrangement     |
|                                   |     defaults                      |
|                                   |                                   |
|                                   | Child Support is not obliged to   |
|                                   | refund the overpaid amount to     |
|                                   | them until the amount is          |
|                                   | recovered from the payee          |
|                                   |                                   |
|                                   | Refunding the overpayment with    |
|                                   | monies from consolidated revenue  |
|                                   | before recovery from the payee    |
|                                   | may be considered certain         |
|                                   | circumstances. See Creating       |
|                                   | consolidated revenue debts.       |
|                                   | Creating a consolidated revenue   |
|                                   | debt is not the preferred method  |
|                                   | of managing an overpayment and    |
|                                   | careful consideration must be     |
|                                   | given before offering this as an  |
|                                   | option.                           |
|                                   |                                   |
|                                   | If the payer elects to waive the  |
|                                   | overpayment, see Step 1 in Table  |
|                                   | 3.                                |
|                                   |                                   |
|                                   | If the payer declines to waive    |
|                                   | the overpayment, see Step 6 in    |
|                                   | Table 3.                          |

|                                   | advise:                           |
|                                   |                                   |

|                                   |                                   |

|                                   |     resulting in terminating      |
|                                   |     event                         |
|                                   |                                   |
|                                   | the overpayment is not a debt due |

Customer elects or declines to waive an overpayment

Table 3: This table describes the process for speaking with a payer
about the option of waiving an overpayment.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Payer elects to waive             |
|                                   | overpayment + Read more ...       |
|                                   |                                   |
|                                   | If the payer elects to partially  |
|                                   | or fully waive the overpayment    |
|                                   | advise that:                      |
|                                   |                                   |
|                                   | -   they will receive an          |
|                                   |     Overpayment waiver cooling    |
|                                   |     off - Payer letter providing  |
|                                   |     seven days from the date the  |
|                                   |     letter is received to         |
|                                   |     withdraw their election to    |
|                                   |     waive the overpayment         |
|                                   | -   after the cooling off period  |
|                                   |     they will no longer be able   |
|                                   |     to ask Child Support to       |
|                                   |     recover the overpayment or    |
|                                   |     have the overpayment credited |
|                                   |     against their child support   |
|                                   |     liabilities                   |
|                                   | -   the waived amount cannot be   |
|                                   |     reclaimed or used to offset a |
|                                   |     future debt, and              |
|                                   | -   in the case of a partial      |
|                                   |     waiver Child Support will:    |
|                                   | -   contact the payee to make a   |
|                                   |     payment arrangement for the   |
|                                   |     remaining amount, and         |
|                                   | -   contact the payer again to    |
|                                   |     advise of their ongoing child |
|                                   |     support obligations, go to    |
|                                   |     Step 6                        |
|                                   |                                   |
|                                   | Document the payer’s decision to  |
|                                   | waive the overpayment in the      |
|                                   | Manage Gift and Overpayment       |
|                                   | window. See Documenting Child     |
|                                   | Support information.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Process overpayment waiver + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Process the overpayment waiver    |
|                                   | immediately to prevent the        |
|                                   | overpayment automatically being   |
|                                   | recovered from the payee's next   |
|                                   | entitlement. See Overpayments     |
|                                   | Cuba Process Help.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Issue overpayment waiver cooling  |
|                                   | off letter + Read more ...        |
|                                   |                                   |
|                                   | Complete the Overpayment waiver   |
|                                   | cooling off - Payer letter        |
|                                   | template and paste it into a      |
|                                   | unique letter in Cuba (MX0-1      |
|                                   | letter). See Letters for Child    |
|                                   | Support customers.                |
|                                   |                                   |
|                                   | The letter asks the payer to      |
|                                   | contact Child Support within      |
|                                   | seven days of receipt if they     |
|                                   | elect to reverse their decision.  |
|                                   | Allow the customer time to        |
|                                   | respond. See Standard response    |
|                                   | times.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Payee contact + Read more ...     |
|                                   |                                   |
|                                   | If the payer waives the           |
|                                   | overpayment, the payee is not     |
|                                   | required to repay the overpaid    |
|                                   | amount. Contact the payee and     |
|                                   | discuss the following:            |
|                                   |                                   |
|                                   | how the overpayment occurred      |
|                                   |                                   |
|                                   | the amount of the overpayment     |
|                                   |                                   |
|                                   | payer election to waive the       |
|                                   | overpayment                       |
|                                   |                                   |
|                                   | the payer has a seven day cooling |
|                                   | off period to reconsider their    |
|                                   | decision to waive, after which    |
|                                   | Child Support will contact the    |
|                                   | payee to either:                  |
|                                   |                                   |
|                                   | -   confirm the waiver            |
|                                   | -   negotiate a payment           |
|                                   |     arrangement if the payer      |
|                                   |     reverses the decision to      |
|                                   |     waive the overpayment         |
|                                   |                                   |
|                                   | Document all contact attempts and |
|                                   | the details of the discussion     |
|                                   | with the payee in the manage Gift |
|                                   | and Overpayment window. See       |
|                                   | Contact with Child Support        |
|                                   | customers.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Expiry of cooling off period +    |
|                                   | Read more ...                     |
