Issuing FTB clean skin letters

This table contains information about clean skin letters that can be
manually issued to FTB customers if required.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Issuing a cleanskin letter + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Clean skin letters show basic     |
|                                   | entitlement information and are   |
|                                   | used when an automatic letter is  |
|                                   | not available.                    |
|                                   |                                   |
|                                   | To issue clean skin letters:      |
|                                   |                                   |
|                                   | Go to the Issue Letter - Clean    |
|                                   | skin (ILC) screen.                |
|                                   |                                   |
|                                   | On ILC screen, enter letter type  |
|                                   | 'FTB' - Family Tax Benefit.       |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | The Main Menu (MM) screen will    |
|                                   | display. The letter will be       |
|                                   | printed and posted centrally to   |
|                                   | the customer.                     |
|                                   |                                   |
|                                   | Once generated, letters can be    |
|                                   | viewed on the History Summary     |
|                                   | (HS) screen.                      |
+-----------------------------------+-----------------------------------+
