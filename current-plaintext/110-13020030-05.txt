Earnings Apportionment Tool (EAT) calculations sheets

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Uploading EAT calculations sheets |
|                                   |                                   |
|                                   | All EAT calculation sheets used   |
|                                   | must be saved as an .XLSX file    |
|                                   | for them to be uploaded to the    |
|                                   | customer record.                  |
|                                   |                                   |
|                                   | To save the EAT as an .XLSX file: |
|                                   |                                   |
|                                   | Save the file to the H Drive as   |
|                                   | an 'Excel workbook' rather than   |
|                                   | an 'Excel Macro-Enabled           |
|                                   | Workbook'. To do this, select:    |
|                                   |                                   |
|                                   | -   File                          |
|                                   | -   Save As                       |
|                                   | -   Browse to locate the H Drive  |
|                                   | -   Change the Save as type to    |
|                                   |     Excel workbook to save the    |
|                                   |     file as an .XLSX file         |
|                                   |                                   |
|                                   | Upload the sheets to the          |
|                                   | customer's record using the       |
|                                   | Upload Document function on the   |
|                                   | Customer First Tools screen.      |
|                                   |                                   |
|                                   | Upload all EAT calculation sheets |
|                                   | using the reason code DBTCAL.     |
|                                   |                                   |
|                                   | After uploading all calculation   |
|                                   | sheets, delete the saved files    |
|                                   | from the H drive.                 |
|                                   |                                   |
|                                   | For multiple employers, save each |
|                                   | calculation with the Customer     |
|                                   | Reference Number (CRN) and        |
|                                   | employer number as below for      |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   EAT sheet for 1st employer    |
|                                   |     save as - 123456789A01        |
|                                   | -   EAT sheet for 2nd employer    |
|                                   |     save as - 123456789A02        |
+-----------------------------------+-----------------------------------+
| 2                                 | Deleting EAT calculation sheets   |
|                                   | returned in Quality Online (QOL)  |
|                                   |                                   |
|                                   | If a decision is changed because  |
|                                   | of the QOL checking process:      |
|                                   |                                   |
|                                   | -   Upload the corrected          |
|                                   |     calculation sheet to the      |
|                                   |     customer record               |
|                                   | -   Arrange for the incorrect     |
|                                   |     calculation sheet to be       |
|                                   |     deleted by lodging a ECRM     |
|                                   |     Image Deletion Request form   |
|                                   |                                   |
|                                   | This makes sure only the correct  |
|                                   | calculation sheet remains         |
|                                   | attached to the customer record.  |
|                                   | This is important as customers    |
|                                   | can access this information under |
|                                   | Freedom Of Information (FOI).     |
+-----------------------------------+-----------------------------------+
| 3                                 | Reassessment or review of         |
|                                   | decision                          |
|                                   |                                   |
|                                   | If a calculation sheet needs to   |
|                                   | be amended because of a           |
|                                   | reassessment or review of         |
|                                   | decision:                         |
|                                   |                                   |
|                                   | -   Copy information from the     |
|                                   |     calculation sheet,            |
|                                   |     (previously uploaded to the   |
|                                   |     customer record)              |
|                                   | -   Paste the information into a  |
|                                   |     new Excel Macro-Enabled       |
|                                   |     Workbook                      |
|                                   |                                   |
|                                   | Once the reassessment has been    |
|                                   | completed, upload the new         |
|                                   | calculation sheet to the customer |
|                                   | record as outlined above.         |
|                                   |                                   |
|                                   | There is no need to delete the    |
|                                   | original calculation sheet. This  |
|                                   | is a new decision resulting from  |
|                                   | the reassessment.                 |
+-----------------------------------+-----------------------------------+
