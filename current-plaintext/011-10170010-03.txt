This page contains processes related to setting up links on child
records between Centrelink and the Australian Immunisation Register
(AIR). This includes how to respond to enquiries and how Families
Override Officers sever a link set up in error.

|                                   | How the link between Centrelink   |
|                                   | and the AIR is established        |
|                                   |                                   |
|                                   | Data is transmitted to the AIR    |
|                                   | once the activity is completed –  |
|                                   | this can take up to 14 days to    |

|                                   | 14 days ends and no match has     |
|                                   | been established the child will   |
|                                   | commence a 63 day grace period.   |
|                                   |                                   |
|                                   | The AIR link status for FTB will  |
|                                   | not update in real time. If an    |
|                                   | exact, unique match exists, a     |
|                                   | link is established and the       |

|                                   | overnight. If a match is not      |
|                                   | successful, a mismatch reason     |
|                                   | code is received overnight.       |
|                                   |                                   |
|                                   | If a possible match or multiple   |
|                                   | matches exist, the AIR staff will |
|                                   | attempt to match the child        |
|                                   | manually. This process can also   |
|                                   | take up to 14 days to complete.   |
|                                   |                                   |

|                                   | link status and immunisation      |
