Eligibility and notification for the Youth Bonus Wage Subsidy (YBWS) 26 week suspension period

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for the YBWS + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Department of Education,      |
|                                   | Skills and Employment will be     |
|                                   | responsible for determining a job |
|                                   | seeker's eligibility for the      |
|                                   | YBWS.                             |
|                                   |                                   |
|                                   | A job seeker will be eligible to  |
|                                   | participate in the YBWS           |
|                                   | initiative if they satisfy all of |
|                                   | the following:                    |
|                                   |                                   |
|                                   | -   they are under 25 years of    |
|                                   |     age at the time of entering   |
|                                   |     the YBWS initiative           |
|                                   | -   in receipt of JobSeeker       |
|                                   |     Payment (JSP) or Youth        |
|                                   |     Allowance (YA), and           |
|                                   | -   is provider managed by a      |
|                                   |     jobactive or Transition to    |
|                                   |     Work (TTW) provider           |
|                                   |                                   |
|                                   | Job seekers participating in the  |
|                                   | Community Development Program     |
|                                   | (CDP) or with Disability          |
|                                   | Employment Services (DES) are     |
|                                   | excluded from this initiative.    |
|                                   |                                   |
|                                   | Note: customers undertaking an    |
|                                   | apprenticeship are ineligible for |
|                                   | YBWS as they are not considered   |
|                                   | to be job seekers.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Notification for the YBWS + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Department of Education,      |
|                                   | Skills and Employment will send   |
|                                   | through an electronic             |
|                                   | notification to Services          |
|                                   | Australia when a job seeker is    |
|                                   | placed into employment under the  |
|                                   | YBWS initiative:                  |
|                                   |                                   |
|                                   | -   information will display on   |
|                                   |     the Participation Summary     |
|                                   |     screen in Customer First,     |
|                                   |     under the Subsidy heading and |
|                                   |     will include the job seeker's |
|                                   |     start date with the           |
|                                   |     initiative                    |
|                                   | -   the YBWS information will     |
|                                   |     display as 'no current        |
|                                   |     information' where the job    |
|                                   |     seeker is not a current YBWS  |
|                                   |     participant. This could be    |
|                                   |     because they have had payment |
|                                   |     restored within the 26 week   |
|                                   |     suspension period or where    |
|                                   |     their income support payment  |
|                                   |     is cancelled at the end of    |
|                                   |     the 26 week suspension period |
|                                   | -   previous periods in the YBWS  |
|                                   |     will be included under the    |
|                                   |     History heading               |
|                                   | -   YBWS information may be       |
|                                   |     displayed on the              |
|                                   |     Participation Summary screen  |
|                                   |     even though the customer may  |
|                                   |     not be suspended (SUS) or     |
|                                   |     cancelled (CAN), that is,     |
|                                   |     they could be reporting       |
|                                   |     income within a 6 fortnight   |
|                                   |     nil rate period               |

|                                   | provider managed until their      |
|                                   | income support payment cancels or |
|                                   | they return to their income       |
|                                   | support payment.                  |
|                                   |                                   |
|                                   | Has the job seeker contacted      |

|                                   | week suspension period?           |

|                                   | -   is participating in the YBWS, |

|                                   | period - no job seeker contact +  |
|                                   | Read more ...                     |

|                                   | job seeker at the end of the 26   |

|                                   |                                   |

|                                   | formal review of the cancellation |

|                                   | reviews and appeals, see Request  |
|                                   | for an explanation, quality check |
|                                   | or review (CLK).                  |
