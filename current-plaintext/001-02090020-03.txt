Making a referral to SEE

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine if the customer         |
|                                   | receives an eligible income       |
|                                   | support payment + Read more ...   |
|                                   |                                   |
|                                   | Is the customer in receipt of an  |
|                                   | eligible income support payment   |
|                                   | (i.e. JobSeeker Payment, Youth    |
|                                   | Allowance (job seeker),           |
|                                   | Disability Support Pension,       |
|                                   | Parenting Payment (Single or      |
|                                   | Partnered) or Special Benefit)?   |
|                                   |                                   |
|                                   | Yes, and they are aged:           |
|                                   |                                   |
|                                   | -   15 years to under 65 years?   |
|                                   |     Go to Step 2                  |
|                                   | -   65 years or older? Go to Step |
|                                   |     5                             |
|                                   |                                   |
|                                   | No, go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine if job seeker is        |
|                                   | Centrelink or provider managed +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Customer Information field on |
|                                   | the Activity and Exemption        |
|                                   | Summary (AEX) screen shows if a   |
|                                   | customer is Centrelink or         |
|                                   | provider managed. For:            |
|                                   |                                   |
|                                   | -   Centrelink managed job        |
|                                   |     seekers, go to Step 3         |
|                                   | -   Provider managed job seekers, |
|                                   |     go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Centrelink managed job seekers +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Service Officer must check    |
|                                   | the following screens before      |
|                                   | starting the referral process.    |
|                                   | This assists with determining     |
|                                   | eligibility:                      |
|                                   |                                   |
|                                   | -   Legal Residence Details       |
|                                   |     (RSLEG) screen - for          |
|                                   |     Citizenship and Visa sub      |
|                                   |     class                         |
|                                   | -   Residence Results Display     |
|                                   |     (RSRD) screen - for permanent |
|                                   |     visa details                  |
|                                   | -   Period of Australian Presence |
|                                   |     (RSPAP) screen - for end date |
|                                   |     of the Newly Arrived          |
|                                   |     Resident's Waiting Period     |
|                                   |     (NARWP)                       |
|                                   |                                   |
|                                   | For job seekers without mutual    |
|                                   | obligation requirements, go to    |
|                                   | Step 5                            |
|                                   |                                   |
|                                   | For job seekers with mutual       |
|                                   | obligation requirements:          |
|                                   |                                   |
|                                   | -   advise these job seekers of   |
|                                   |     their obligation to           |
|                                   |     participate in SEE, if        |
|                                   |     assessed as suitable for      |
|                                   |     training at the Pre-Training  |
|                                   |     Assessment                    |
|                                   | -   use the Referral Workflow to  |
|                                   |     refer them to SEE and arrange |
|                                   |     a Pre-Training Assessment     |
|                                   |     appointment with the SEE      |
|                                   |     provider while the job seeker |
|                                   |     is present                    |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+
| 4                                 | Provider managed job seekers +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customer is eligible for SEE.     |
|                                   |                                   |
|                                   | -   Tell the job seeker they are  |
|                                   |     eligible to be referred to    |
|                                   |     SEE. If the job seeker agrees |
|                                   |     to the referral, contact the  |
|                                   |     job seeker's Employment       |
|                                   |     Services Provider             |
|                                   | -   Discuss any impact on the     |
|                                   |     activities already in the Job |
|                                   |     Plan, if a referral is made   |
|                                   |     to SEE. This discussion can   |
|                                   |     be with any staff member of   |
|                                   |     the provider, not just the    |
|                                   |     job seeker's Case Manager     |
|                                   | -   If the provider has           |
|                                   |     reasonable grounds to oppose  |
|                                   |     a referral, tell the job      |
|                                   |     seeker to contact their Case  |
|                                   |     Manager. Do not proceed with  |
|                                   |     the referral                  |
|                                   |                                   |
|                                   | If the provider agrees, continue  |
|                                   | with the referral:                |
|                                   |                                   |
|                                   | -   Obtain the jobseeker's        |
|                                   |     consent to pass on their      |
|                                   |     personal details to the SEE   |
|                                   |     provider                      |
|                                   | -   Use the Referral Workflow to  |
|                                   |     refer the job seeker to SEE   |
|                                   |                                   |
|                                   | See Step 1 in Table 2.            |
+-----------------------------------+-----------------------------------+
| 5                                 | System limitations + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Job seekers aged 65 years or      |
|                                   | older, customers not in receipt   |
|                                   | of eligible income support        |
|                                   | payments, and customers who are   |
|                                   | not in receipt of any income      |
|                                   | support payment (volunteer job    |
|                                   | seekers), are eligible for SEE    |
|                                   | but are unable to be referred to  |
|                                   | SEE through Services Australia    |
|                                   | systems.                          |
|                                   |                                   |
|                                   | When the job seeker is in receipt |
|                                   | of an eligible income support     |
|                                   | payment and aged 65 years or      |
|                                   | older, they should be advised to  |
|                                   | ask a provider for a referral to  |
|                                   | SEE. Procedure ends here.         |
|                                   |                                   |
|                                   | The following customers will only |
|                                   | be able to access SEE as a        |
|                                   | volunteer job seeker when the     |
|                                   | Services Australia system does    |
|                                   | not allow a referral to SEE:      |
|                                   |                                   |
|                                   | -   the customer is in receipt of |
|                                   |     an income support payment     |
|                                   |     other than JobSeeker Payment, |
|                                   |     Youth Allowance (job seeker), |
|                                   |     Disability Support Pension,   |
|                                   |     Parenting Payment (single or  |
|                                   |     partnered), or Special        |
|                                   |     Benefit, or                   |
|                                   | -   the customer is not in        |
|                                   |     receipt of any income support |
|                                   |     payment                       |
|                                   |                                   |
|                                   | If they want to be referred to    |
|                                   | SEE:                              |
|                                   |                                   |
|                                   | -   register them as a volunteer  |
|                                   |     job seeker. This will         |
|                                   |     automatically refer them to   |
|                                   |     Online Employment Services    |
|                                   | -   tell the volunteer job seeker |
|                                   |     they can only access a        |
|                                   |     maximum of 12 months of       |
|                                   |     Online Employment Services as |
|                                   |     a volunteer. Their access to  |
|                                   |     the service offer will stop   |
|                                   |     after that time, however,     |
|                                   |     they will be able to continue |
|                                   |     in SEE until they complete    |
|                                   |     the activity                  |
|                                   | -   tell the volunteer job seeker |
|                                   |     to contact the Digital        |
|                                   |     Services Contact Centre and   |
|                                   |     request a referral to SEE     |
|                                   | -   if the customer cannot, or    |
|                                   |     does not want to be referred  |
|                                   |     as a volunteer job seeker,    |
|                                   |     they cannot be referred to    |
|                                   |     SEE. Procedure ends here      |

Referral Workflow

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start the Referral Workflow +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | From Workspace, select Referral   |
|                                   | Summary. Ensure there is no       |
|                                   | current SEE referral, then select |
|                                   | Create new referral:              |
|                                   |                                   |
|                                   | -   Selecting Create new referral |
|                                   |     (or Create Referral from      |
|                                   |     Workspace) launches the       |
|                                   |     Referrals Workflow and        |
|                                   |     displays the Referral Type    |
|                                   |     screen                        |
|                                   | -   Enter the referral date of    |
|                                   |     effect                        |
|                                   | -   Select the Referral Type:     |
|                                   |     'Skills for Education and     |
|                                   |     Employment', then Continue    |
|                                   |                                   |
|                                   | The Skills for Education and      |
|                                   | Employment Eligibility screen is  |
|                                   | displayed. This screen requires   |
|                                   | completion of a SEE eligibility   |
|                                   | checklist to ensure the job       |
|                                   | seeker is eligible for a referral |
|                                   | to SEE.                           |
|                                   |                                   |
|                                   | Answer each question. The         |
|                                   | Eligibility Result displays at    |
|                                   | the end of the checklist          |
|                                   |                                   |
|                                   | -   For the question Aged 15      |
|                                   |     years or more and looking for |
|                                   |     full-time work (dependent on  |
|                                   |     individual mutual obligation  |
|                                   |     requirements), select Yes for |
|                                   |     Principal Carers, or job      |
|                                   |     seekers with a reduced work   |
|                                   |     capacity                      |
|                                   |                                   |
|                                   | Enter the Source and Receipt      |
|                                   | date, then Continue               |
|                                   |                                   |
|                                   | If the job seeker is:             |
|                                   |                                   |
|                                   | -   eligible, go to Step 2        |
|                                   | -   not eligible, go to Step 3    |
+-----------------------------------+-----------------------------------+
| 2                                 | Job seeker is eligible for SEE +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Service Provider screen       |
|                                   | displays all SEE service          |
|                                   | providers and distance education  |
|                                   | providers in the job seeker's     |
|                                   | region. The Resources page        |
|                                   | contains a list of distance       |
|                                   | education providers.              |
|                                   |                                   |
|                                   | Is a suitable provider or         |
|                                   | distance educator available?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Job seeker is not eligible for    |
|                                   | SEE + Read more ...               |
|                                   |                                   |
|                                   | A job seeker is not eligible when |
|                                   | they:                             |
|                                   |                                   |
|                                   | are in New Enterprise Incentive   |
|                                   | Scheme (NEIS)                     |
|                                   |                                   |
|                                   | are in Adult Migrant Education    |
|                                   | Program (AMEP)                    |
|                                   |                                   |
|                                   | have recently participated in SEE |
|                                   | and the last program exit date is |
|                                   | within 12 weeks of the next       |
|                                   | referral date:                    |
|                                   |                                   |
|                                   | -   When it's a new provider, the |
|                                   |     workflow completes and        |
|                                   |     creates a DOC advising the    |
|                                   |     date when the job seeker can  |
|                                   |     be referred back to SEE       |
|                                   | -   Job seekers wanting to go     |
|                                   |     back to their previous SEE    |
|                                   |     provider can contact the      |
|                                   |     provider themselves to        |
|                                   |     re-start SEE                  |
|                                   | -   When a Service Officer        |
|                                   |     attempts to code another      |
|                                   |     referral to the same provider |
|                                   |     within the 12 week period     |
|                                   |     will receive the following    |
|                                   |     error ''E409RR - Not          |
|                                   |     Eligible, can be referred to  |
|                                   |     SEE after <Due Date>'. These  |
|                                   |     job seekers are not eligible  |
|                                   |     for a new referral by         |
|                                   |     Services Australia before the |
|                                   |     date recorded against the     |
|                                   |     previous finalisation         |
|                                   |                                   |
|                                   | successful completion. This is    |
|                                   | shown by the latest SEE status of |
|                                   | Lang/Lit/Num Successful on the    |
|                                   | Referral Summary screen.          |
|                                   | Customers who have completed the  |
|                                   | program will be eligible again    |
|                                   | after a 12 week break             |
|                                   |                                   |
|                                   | other reasons, the workflow       |
|                                   | completes and creates a DOC with  |
|                                   | the text Customer is not eligible |
|                                   | for SEE                           |
|                                   |                                   |
|                                   | Job seeker has changed address:   |
|                                   |                                   |
|                                   | The job seeker can:               |
|                                   |                                   |
|                                   | -   contact their previous        |
|                                   |     provider to advise they have  |
|                                   |     relocated. They should        |
|                                   |     request their record be       |
|                                   |     updated in the provider's     |
|                                   |     system with Transfer to new   |
|                                   |     provider. Once updated, a     |
|                                   |     referral to the new provider  |
|                                   |     is possible                   |
|                                   | -   wait until 12 weeks from      |
|                                   |     their last exit date, then    |
|                                   |     seek a new referral, or       |
|                                   | -   resume with the previous SEE  |
|                                   |     provider by contacting the    |
|                                   |     provider directly             |
|                                   |                                   |
|                                   | Job seeker wants to change        |
|                                   | providers:                        |
|                                   |                                   |
|                                   | -   wait until 12 weeks from      |
|                                   |     their last exit date, then    |
|                                   |     seek a new referral           |
|                                   |                                   |
|                                   | If launched from the Job Plan,    |
|                                   | the Referral Workflow then        |
|                                   | returns to the Job Plan to        |
|                                   | continue. This is for job seekers |
|                                   | with mutual obligation            |
|                                   | requirements. See Negotiating Job |
|                                   | Plans.                            |
|                                   |                                   |
|                                   | If the referral is not being made |
|                                   | via the Job Plan, return to the   |
|                                   | Referral Summary screen to        |
|                                   | finalise the activity.            |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | No suitable providers are         |
|                                   | available + Read more ...         |
|                                   |                                   |
|                                   | Ask the job seeker if they can    |
|                                   | attend a SEE provider at a nearby |
|                                   | location. A job seeker may choose |
|                                   | to undertake the program at       |
|                                   | another location due to:          |
|                                   |                                   |
|                                   | -   transportation difficulties   |
|                                   | -   a disability                  |
|                                   | -   caring responsibilities       |
|                                   | -   cultural constraints          |
|                                   | -   no accessible SEE provider    |
|                                   |                                   |
|                                   | To search for other SEE providers |
|                                   | in nearby locations, place a tick |
|                                   | in the Include nearby regions     |
|                                   | checkbox. The list expands to     |
|                                   | include SEE providers in          |
|                                   | surrounding regions.              |
|                                   |                                   |
|                                   | If the job seeker wishes to       |
|                                   | choose a provider outside their   |
|                                   | surrounding region, select        |
|                                   | Provider Search to manually       |
|                                   | search by region code, postcode   |
|                                   | or town/suburb.                   |
|                                   |                                   |
|                                   | Is a referral to another suitable |
|                                   | SEE provider an option for the    |
|                                   | job seeker?                       |
|                                   |                                   |
|                                   | Yes, go to Step 5                 |
|                                   |                                   |
|                                   | No, when there are no suitable    |
|                                   | service providers available for   |
|                                   | the job seeker, a service gap is  |
|                                   | to be recorded:                   |
|                                   |                                   |
|                                   | -   Click in the Record service   |
|                                   |     gap checkbox                  |
|                                   | -   Select a service gap reason   |
|                                   |     from the drop down list       |
|                                   |     options. When this option is  |
|                                   |     selected, the list of service |
|                                   |     providers is disabled         |
|                                   | -   Continue, then Finalise       |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Suitable provider is available +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Ask the job seeker if they are    |
|                                   | interested in undertaking SEE by  |
|                                   | distance education, if the        |
|                                   | suitable provider list includes   |
|                                   | distance education providers. If  |
|                                   | not, only select local providers. |
|                                   |                                   |
|                                   | Select the job seeker's preferred |
|                                   | SEE provider. Select More Info... |
|                                   | to obtain provider contact        |
|                                   | details.                          |
|                                   |                                   |
|                                   | Arrange an appointment with the   |
|                                   | SEE provider for the Pre-Training |
|                                   | Assessment. This is mandatory for |
|                                   | all SEE referrals, whether the    |
|                                   | referral is coded through the Job |
|                                   | Plan or the Create Referral       |
|                                   | workflow. Ensure the job seeker   |
|                                   | knows of the compliance           |
|                                   | consequences if they do not       |
|                                   | attend their Pre-Training         |
|                                   | Assessment appointment.           |
|                                   |                                   |
|                                   | Service Officers should not       |
|                                   | discuss the job seeker's          |
|                                   | circumstances with the provider.  |
|                                   | Only disclose sufficient          |
|                                   | information to book the           |
|                                   | appointment.                      |
|                                   |                                   |
|                                   | If referring to a distance        |
|                                   | education provider, go to Step 6. |
|                                   |                                   |
|                                   | If referring to local providers:  |
|                                   |                                   |
|                                   | -   check if the job seeker has a |
|                                   |     silent phone number – if so,  |
|                                   |     seek their consent to pass    |
|                                   |     the number to the provider    |
|                                   |     over the phone as the         |
|                                   |     referral system will exclude  |
|                                   |     it from the online referral   |
|                                   | -   phone the provider and make a |
|                                   |     Pre-Training Assessment       |
|                                   |     appointment                   |
|                                   | -   advise the job seeker of the  |
|                                   |     date, time and location of    |
|                                   |     the appointment               |
|                                   | -   once the SEE provider's       |
|                                   |     contact details have been     |
|                                   |     noted, select Back , then     |
|                                   |     Continue                      |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Referral to distance education    |
|                                   | provider + Read more ...          |
|                                   |                                   |

|                                   | undertake SEE by distance         |
|                                   | learning.                         |
|                                   |                                   |
|                                   | For all SEE courses through a     |
|                                   | distance education provider, job  |
|                                   | seekers require:                  |

|                                   | Go to Step 7.                     |

|                                   | screen.                           |

|                                   | Go to Step 8.                     |

| 8                                 | Job Plan Workflow + Read more ... |
|                                   |                                   |
|                                   | For Centrelink managed job        |
|                                   | seekers with mutual obligation    |
|                                   | requirements, record the          |

|                                   |     appointment                   |
|                                   | -   Start date and End date       |
|                                   |     fields should automatically   |

|                                   |                                   |
|                                   | Select Add and continue through   |
|                                   | the Job Plan workflow as          |
|                                   | required. See Negotiating Job     |
|                                   | Plans.                            |
|                                   |                                   |
|                                   | See Participation in Skills for   |
|                                   | Education and Employment (SEE)    |
|                                   | for details of the process to     |
|                                   | update the Job Plan when a job    |
|                                   | seeker commences SEE:             |
