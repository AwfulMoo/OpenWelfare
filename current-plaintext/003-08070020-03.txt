| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessment of Income + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer single?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2.            |
|                                   | -   No, if their partner          |
|                                   |     receives:                     |
|                                   | -   a pension, go to Step 3       |
|                                   | -   Partner Allowance (PA) or     |
|                                   |     SpB, go to Step 4             |
|                                   | -   another allowance or benefit, |
|                                   |     or does not receive a         |
|                                   |     payment, go to Step 5         |
+-----------------------------------+-----------------------------------+
| 2                                 | Special Benefit customer is       |
|                                   | single + Read more ...            |
|                                   |                                   |
|                                   | Advise customer there is no       |
|                                   | income free area for SpB, and     |
|                                   | payment is reduced dollar for     |
|                                   | dollar under the Special Benefit  |
|                                   | income test.                      |
|                                   |                                   |
|                                   | Customer’s entitlement = Current  |
|                                   | rate of SpB minus total of gross  |
|                                   | income from employment.           |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | SpB customer with partner on a    |
|                                   | pension + Read more ...           |
|                                   |                                   |
|                                   | If below Age Pension age, partner |
|                                   | may have access to working        |
|                                   | credits to offset their           |
|                                   | employment income.                |
|                                   |                                   |
|                                   | They may remain on payment for up |
|                                   | to 6 fortnights and retain        |
|                                   | certain benefits if their rate is |
|                                   | reduced to nil by income and some |
|                                   | of that income is employment      |
|                                   | income.                           |
|                                   |                                   |
|                                   | Advise the customer that half of  |
|                                   | the assessable income will be     |
|                                   | applied to the pension customer   |
|                                   | under the Pensions Income and     |
|                                   | Assets Tests rules, and half will |
|                                   | reduce the SpB customer's payment |
|                                   | dollar for dollar.                |
|                                   |                                   |
|                                   | SpB customer’s entitlement =      |
|                                   | Current rate of SpB minus (total  |
|                                   | of gross income from employment   |
|                                   | divided by 2).                    |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | SpB customer with partner on SpB  |
|                                   | or PA + Read more ...             |
|                                   |                                   |
|                                   | Neither customer is covered by    |
|                                   | the Working Credit provisions as  |
|                                   | a different income test applies.  |
|                                   | The partner's qualification for   |
|                                   | PA depends on the customer's      |
|                                   | qualification for SpB.            |
|                                   |                                   |
|                                   | The customer's SpB payment is     |
|                                   | reduced dollar for dollar under   |
|                                   | the Special Benefit income test   |
|                                   | and any excess income reduces     |
|                                   | their partner’s payment dollar    |
|                                   | for dollar.                       |
|                                   |                                   |
|                                   | Customer’s entitlement = Current  |
|                                   | rate of SpB minus total of gross  |
|                                   | income from employment.           |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | SpB customer with partner not     |
|                                   | receiving pension, SpB or PA +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Partner may have access to        |
|                                   | working credits to offset their   |
|                                   | employment income.                |
|                                   |                                   |
|                                   | They may remain on payment for up |
|                                   | to 6 fortnights and retain        |
|                                   | certain benefits if their rate is |
|                                   | reduced to nil by income and some |
|                                   | of that income is employment      |
|                                   | income.                           |
|                                   |                                   |
|                                   | Income will be applied to         |

|                                   | 60c in $1.00.                     |
|                                   |                                   |
|                                   | SpB customer’s entitlement =      |
|                                   | Current SpB rate minus total of   |
|                                   | gross income from employment.     |
