|                                   | if the customer cannot:           |
|                                   |                                   |
|                                   | -   access their Child Support    |
|                                   |     online account                |
|                                   | -   update their personal details |
|                                   | -   complete transactions like    |
|                                   |     advise extraordinary          |
|                                   |     (non-agency) payments or      |
|                                   |     update income                 |
|                                   | -   upload a document             |
|                                   | -   view payment history          |
|                                   | -   view online letters in the    |
|                                   |     myGov Inbox (e.g. they cannot |
|                                   |     open the PDF document from    |
|                                   |     their Inbox, or they are      |
|                                   |     notified of a new letter but  |
|                                   |     it is not in their Inbox)     |
+-----------------------------------+-----------------------------------+
| 2                                 | Obtain information + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Obtain the following information  |
|                                   | from the customer:                |
|                                   |                                   |
|                                   | -   Customer's full name          |
|                                   | -   Child Support Reference       |
|                                   |     Number (CSRN)                 |
|                                   | -   Date and time issue occurred  |
|                                   | -   What the issue is             |
|                                   | -   If an error message is        |
|                                   |     appearing, the error number   |
|                                   |     or exact wording of the       |
|                                   |     message                       |
|                                   | -   If the error is related to a  |
|                                   |     letter, the letter name or    |
|                                   |     type and date of the letter   |
|                                   | -   What type of computer they    |
|                                   |     are using - Mac or PC         |
|                                   | -   What operating system they    |
|                                   |     are using, e.g. Windows 10,   |
|                                   |     Vista                         |
|                                   | -   What browser they are using,  |
|                                   |     e.g. Internet Explorer        |
|                                   |     version 11                    |
|                                   | -   If they are able to, take a   |
|                                   |     screenshot of the issue and   |
|                                   |     upload the document           |
+-----------------------------------+-----------------------------------+
| 3                                 | Escalate to SSO + Read more ...   |
|                                   |                                   |
|                                   | All non-Multicultural and         |
|                                   | Tailored Services Branch Service  |

|                                   |                                   |
|                                   | -   Escalate the issue to         |
|                                   |     technical support. See        |
|                                   |     Technical support in Child    |
|                                   |     Support                       |

|                                   | -   Follow Branch request         |
