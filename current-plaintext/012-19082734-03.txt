The QI process

Table 1

+-----------------------+-----------------------+-----------------------+
| Stage                 | Who                   | Action                |
+-----------------------+-----------------------+-----------------------+
| 1                     | Practice              | Participate in the    |
|                       |                       | PIP + Read more ...   |
|                       |                       |                       |
|                       |                       | Practices need to     |
|                       |                       | register for the PIP. |
|                       |                       | They do this through  |
|                       |                       | HPOS using their      |
|                       |                       | PRODA Account.        |
+-----------------------+-----------------------+-----------------------+
| 2                     | Practice              | Register for the QI   |
|                       |                       | Incentive + Read more |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | Practices register    |
|                       |                       | online using their    |
|                       |                       | PRODA account.        |
+-----------------------+-----------------------+-----------------------+
| 3                     | Practice              | Provide data to the   |
|                       |                       | local PHN + Read more |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | Practices must        |
|                       |                       | provide data to their |
|                       |                       | local Primary Health  |
|                       |                       | Network (PHN). They   |
|                       |                       | must do this by the   |
|                       |                       | 15th day of the       |
|                       |                       | month, prior to the   |
|                       |                       | calculation month.    |
|                       |                       |                       |
|                       |                       | Aboriginal Community  |
|                       |                       | Controlled Health     |
|                       |                       | Services (ACCHS) and  |
|                       |                       | other organisations   |
|                       |                       | funded under the      |
|                       |                       | Indigenous            |
|                       |                       | Australians’ Health   |
|                       |                       | Program (IAHP)        |
|                       |                       | provide their data    |
|                       |                       | directly to Health.   |
|                       |                       | They do not need to   |
|                       |                       | provide this to their |
|                       |                       | PHN.                  |
+-----------------------+-----------------------+-----------------------+
| 4                     | Primary Health        | Provide data to       |
|                       | Network (PHN)         | Health + Read more    |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | The PHN will:         |
|                       |                       |                       |
|                       |                       | -   assess the data   |
|                       |                       |     and               |
|                       |                       | -   send confirmation |
|                       |                       |     to Health. This   |
|                       |                       |     will determine    |
|                       |                       |     the practice’s    |
|                       |                       |     eligibility       |
+-----------------------+-----------------------+-----------------------+
| 5                     | Department of Health  | Send payment          |
|                       | and Aged Care         | eligibility data to   |
|                       |                       | Services Australia +  |
|                       |                       | Read more ...         |
|                       |                       |                       |
|                       |                       | Health send payment   |
|                       |                       | eligibility           |
|                       |                       | confirmation files to |
|                       |                       | Services Australia.   |
+-----------------------+-----------------------+-----------------------+
| 6                     | Services Australia    | Make the payment +    |
|                       |                       | Read more ...         |
|                       |                       |                       |
|                       |                       | Make QI payments      |
|                       |                       | under the PIP to      |
|                       |                       | practices each        |
|                       |                       | quarter.              |
+-----------------------+-----------------------+-----------------------+

How to apply for the QI Incentive

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to apply for the QI        |
|                                   | Incentive + Read more ...         |
|                                   |                                   |
|                                   | Log on to PRODA:                  |
|                                   |                                   |
|                                   | -   From My linked services,      |
|                                   |     select Go to Service on the   |
|                                   |     HPOS tile                     |
|                                   | -   Select My Programs            |
+-----------------------------------+-----------------------------------+
| 2                                 | Apply for the Quality Improvement |
|                                   | Incentive + Read more ...         |
|                                   |                                   |
|                                   | To assist a practice owner:       |
|                                   |                                   |
|                                   | Go to the PIP tile:               |
|                                   |                                   |
|                                   | -   If this is unavailable they   |
|                                   |     can link directly to ‘your    |
|                                   |     PRODA Registration Authority  |
|                                   |     (RA) number’, or              |
|                                   | -   Contact Incentive programs    |
|                                   |     (Medicare)                    |
|                                   |                                   |
|                                   | Select the Update button          |
|                                   |                                   |
|                                   | -   If multiple practices         |
|                                   |     display, select the correct   |
|                                   |     one                           |
|                                   | -   If a PIP information screen   |
|                                   |     appears, select the Exit      |
|                                   |     button at the bottom of the   |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Go to the Main menu (top-left     |
|                                   | hand side of screen) and select   |
|                                   | Incentive summary                 |
|                                   |                                   |
|                                   | Go to the Quality Stream section  |
|                                   |                                   |
|                                   | -   In Quality Improvement        |
|                                   |     Incentive select the Apply    |
|                                   |     link                          |
|                                   |                                   |
|                                   | Read the terms and conditions     |
|                                   |                                   |
|                                   | Select the Participation Payment  |
|                                   | option                            |
|                                   |                                   |
|                                   | Select Submit                     |
|                                   |                                   |
|                                   | Note: a message will show that    |
|                                   | says the QI Incentive details     |
|                                   | have been updated.                |

Check QI payments

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log into systems and practice     |
|                                   | search + Read more ...            |
|                                   |                                   |
|                                   | Log on to the PIP Online system   |
|                                   |                                   |
|                                   | Search for the practice in PIP    |
|                                   | Online.                           |
|                                   |                                   |
|                                   | On the Practice Search screen:    |
|                                   |                                   |
|                                   | Key the Practice ID               |
|                                   |                                   |
|                                   | Select Search                     |
|                                   |                                   |
|                                   | -   The Search Results will show  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check practice payment status +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select the Practice ID hyperlink  |
|                                   | from the Search results.          |
|                                   |                                   |
|                                   | On the Practice Summary screen:   |
|                                   |                                   |
|                                   | Check if an information message   |
|                                   | ‘This practice is on hold. Check  |
|                                   | payment details screen for        |
|                                   | details.’ shows                   |
|                                   |                                   |
|                                   | -   If the information message    |
|                                   |     does not show, go to step 3   |
|                                   |                                   |
|                                   | If the practice is on hold, check |
|                                   | why the payments are on hold:     |
|                                   |                                   |
|                                   | From the Main menu:               |
|                                   |                                   |
|                                   | Select Practice Details           |
|                                   |                                   |
|                                   | Select Payment Details from the   |
|                                   | drop down menu                    |
|                                   |                                   |
|                                   | -   The Payment Details screen is |
|                                   |     displayed                     |
|                                   |                                   |
|                                   | Check the Reason for the payment  |
|                                   | On Hold under the PIP Payments    |
|                                   |                                   |
|                                   | Advise the practice why their     |
|                                   | payment is on hold. Procedure     |
|                                   | ends here.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Check practice’s QI payment +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | From the Main menu:               |
|                                   |                                   |
|                                   | Select Payments                   |
|                                   |                                   |
|                                   | Select Payment History from the   |
|                                   | drop down menu                    |
|                                   |                                   |
|                                   | -   The Payment History screen    |
|                                   |     will show                     |
|                                   |                                   |
|                                   | Select the View PDF for the       |
|                                   | payment quarter in the Payment    |
|                                   | Advice column                     |
|                                   |                                   |
|                                   | The payment advice PDF will open  |
|                                   |                                   |
|                                   | Check the Quality Improvement     |
|                                   | Participation payment amount      |
|                                   | listed under Schedule A on the    |
|                                   | payment advice.                   |
|                                   |                                   |
|                                   | If the payment amount is $0.00    |
|                                   |                                   |
|                                   | -   Check if the practice has a   |
|                                   |     zero SWPE on the first page   |
|                                   |     of the payment advice. The QI |
|                                   |     payment is based on SWPE and  |
|                                   |     new practices who have not    |
|                                   |     generated a SWPE yet will not |
|                                   |     receive a payment             |
|                                   | -   If the practice has an        |
|                                   |     established SWPE go to Step   |
|                                   |     4.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Check QI status + Read more ...   |
|                                   |                                   |
|                                   | From the Main menu:               |
|                                   |                                   |
|                                   | Select Incentive Summary          |
|                                   |                                   |
|                                   | Check the Status column for       |
|                                   | Quality Improvement Incentive     |
|                                   | under the Quality Stream heading  |
|                                   |                                   |
|                                   | If the status is:                 |
|                                   |                                   |
|                                   | -   Approved, and the Date        |
|                                   |     Approved is before the        |
|                                   |     point-in-time, go to Step 5   |
|                                   | -   Not applied, procedure ends   |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Services tab + Read more ...      |
|                                   |                                   |
|                                   | On the Services tab:              |
|                                   |                                   |
|                                   | Select Quality Improvement        |
|                                   | Transaction Search screen         |
|                                   |                                   |
|                                   | -   The Search screen will show   |
|                                   |                                   |
|                                   | Key the Practice ID in the        |
|                                   | Practice ID field                 |
|                                   |                                   |
|                                   | Select the Month and Year in the  |
|                                   | Calculation quarter fields        |
|                                   |                                   |
|                                   | The Search results will show      |
|                                   |                                   |
|                                   | Note: if the error message ‘2542  |
|                                   | – No results found. Please change |
|                                   | the search criteria and try       |
|                                   | again’ shows, see the Resources   |
|                                   | page for assistance.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Check File Type + Read more ...   |
|                                   |                                   |
|                                   | Check the File Type column. If    |
|                                   | the file type is:                 |
|                                   |                                   |
|                                   | PRACPY or CONFPY, go to Step 7    |
|                                   |                                   |
|                                   | ADJTPY, go to Step 8              |
|                                   |                                   |
|                                   | VERIPY, we are waiting for        |
|                                   | confirmation from Health          |
|                                   |                                   |
|                                   | -   Advise the practice to        |
|                                   |     confirm their details with    |
|                                   |     their PHN                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Transaction status + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The status of the QI payment for  |
|                                   | the calculation quarter will show |
|                                   | in the Transaction status column: |
|                                   |                                   |
|                                   | Calculated – Positive a payment   |
|                                   | has been calculated               |
|                                   |                                   |
|                                   | Calculated – Zero a payment has   |
|                                   | been calculated as zero           |
|                                   |                                   |
|                                   | -   A zero payment will be        |
|                                   |     calculated if the practice is |
|                                   |     new and has not yet generated |
|                                   |     a SWPE                        |
|                                   |                                   |
|                                   | Initial a CONFPY, PRACPY or       |
|                                   | ADJTPY file has been received,    |
|                                   | but not processed                 |
|                                   |                                   |
|                                   | Rejected – Admin a record on an   |
|                                   | ADJTPY file was rejected due to   |
|                                   | the QI Incentive payment          |
|                                   | type/tier that the adjustment     |
|                                   | payment relates to not being      |
|                                   | active                            |
|                                   |                                   |
|                                   | Rejected – Invalid a record was   |
|                                   | rejected due to invalid data      |
|                                   |                                   |
|                                   | Rejected – Ineligible a record    |
|                                   | was rejected due to the practice  |
|                                   | being ineligible for a QI         |
|                                   | Incentive payment                 |
+-----------------------------------+-----------------------------------+
| 8                                 | Adjustment reason + Read more ... |
|                                   |                                   |
|                                   | If the file type column shows     |
|                                   | ADJTPY, the reason for the        |
|                                   | adjustment payment will show in   |
|                                   | the Adjustment reason column.     |
|                                   |                                   |
|                                   | The Transaction Status column     |
|                                   | will show:                        |
|                                   |                                   |
|                                   | -   Rejected – Manual a record on |
|                                   |     an ADJTPY file is rejected    |
|                                   |     due to the adjustment payment |
|                                   |     amount being less than zero   |
|                                   | -   Rejected – Duplicate a record |
|                                   |     on an ADJTPY file is rejected |
|                                   |     due to the practice already   |
|                                   |     having received an adjustment |
|                                   |     payment for the quarterly QI  |
|                                   |     Incentive payment (i.e.       |
|                                   |     cannot auto adjust an auto    |
|                                   |     adjustment)                   |
|                                   | -   Adjusted – Negative a QI      |
|                                   |     Incentive payment is adjusted |
|                                   |     and the result of the         |
|                                   |     adjustment is lower than the  |
|                                   |     initial payment (i.e.         |
|                                   |     recovery action is required)  |
|                                   | -   Adjusted – Zero a QI          |
|                                   |     Incentive payment is adjusted |
|                                   |     and the result of the         |
|                                   |     adjustment does not change    |
|                                   |     the initial payment           |
|                                   | -   Adjusted – Positive a QI      |
|                                   |     Incentive payment is adjusted |
|                                   |     and the result of the         |
|                                   |     adjustment is more than the   |
|                                   |     initial payment               |
|                                   |                                   |
|                                   | Adjustment payments are made to   |
|                                   | practices on an adhoc basis. The  |
|                                   | practice will receive an          |
|                                   | adjustment payment advice         |
|                                   | detailing the payment when it is  |
|                                   | made.                             |
