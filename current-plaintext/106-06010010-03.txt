Assessing residing in Australia

Table 4: assessing if a person is living in Australia.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Consider the balance between the  |
|                                   | person's ties in Australia and    |
|                                   | their ties with another country + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Contact the customer when         |
|                                   | considering their circumstances   |
|                                   | to make a correct assessment.     |
|                                   | While a factor may indicate they  |
|                                   | are not residing in Australia,    |
|                                   | exceptional circumstances may     |
|                                   | prove otherwise. Contacting the   |
|                                   | customer will give them the       |
|                                   | chance to comment. It will also   |
|                                   | give the chance to advise them of |
|                                   | any adverse decision and the      |
|                                   | potential impact on their         |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Determining if a customer is a    |
|                                   | resident of Australia is a        |
|                                   | discretionary decision. Document  |
|                                   | the reasons for the decision,     |
|                                   | including any discussion with the |
|                                   | customer. Do not change           |
|                                   | discretionary decisions unless    |
|                                   | there is evidence the customer    |
|                                   | misrepresented their              |
|                                   | circumstances when the original   |
|                                   | decision was made, or a           |
|                                   | significant error about facts     |
|                                   | requires correction. Changes in   |
|                                   | circumstances or discovery of     |
|                                   | factual error may be a reason to  |
|                                   | make a new decision about where   |
|                                   | the customer is residing.         |
|                                   | However, do not backdate these    |
|                                   | changes unless there is evidence  |
|                                   | the customer misrepresented their |
|                                   | circumstances when the original   |
|                                   | decision was made.                |
|                                   |                                   |
|                                   | A customer's presence in          |
|                                   | Australia or their statement they |
|                                   | intend to reside in Australia is  |
|                                   | not sufficient evidence that they |
|                                   | are residing in Australia. If     |
|                                   | their intent is to stay for a     |
|                                   | period to lodge a valid claim and |
|                                   | meet qualification criteria for a |
|                                   | social security payment and then  |
|                                   | leave Australia to resume         |
|                                   | residence in another country, do  |
|                                   | not regard them as being an       |
|                                   | Australian resident.              |
|                                   |                                   |
|                                   | Consider the customer's answers   |
|                                   | in each of the steps below as     |
|                                   | well as any other relevant        |
|                                   | information when making a         |
|                                   | decision.                         |
|                                   |                                   |
|                                   | For help, use the Residence       |
|                                   | assessment templates on the       |
|                                   | International Programme home page |
|                                   | under the Residence heading. The  |
|                                   | Resources page contains a link    |
|                                   | and examples of decisions.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Frequency and duration of travel  |
|                                   | outside Australia + Read more ... |
|                                   |                                   |
|                                   | Where is the person spending most |
|                                   | of their time?                    |
|                                   |                                   |
|                                   | What is the reason for being      |
|                                   | overseas for long periods?        |
|                                   |                                   |
|                                   | A person does not have to be      |
|                                   | continuously present in Australia |
|                                   | to be residing here. A person     |
|                                   | holidaying or working temporarily |
|                                   | overseas does not necessarily     |
|                                   | cease to reside in Australia      |
|                                   | while they are away.              |
|                                   |                                   |
|                                   | Find the reason for being         |
|                                   | overseas and look closely at the  |
|                                   | pattern and time spent outside    |
|                                   | Australia to decide if a person   |
|                                   | continues to reside in Australia. |
|                                   | To maintain Australian residence  |
|                                   | during an absence, a person must  |
|                                   | demonstrate continued physical    |
|                                   | ties to Australia:                |
|                                   |                                   |
|                                   | -   the absence must be for a     |
|                                   |     short time                    |
|                                   | -   there must be a purpose for   |
|                                   |     the absence, and              |
|                                   | -   there must be a proposed end  |
|                                   |     date for the absence          |
|                                   |                                   |
|                                   | For example, a person working on  |
|                                   | a 2 year overseas contract        |
|                                   | posting is considered to reside   |
|                                   | in Australia as long as they have |
|                                   | demonstrated ongoing physical     |
|                                   | ties to Australia and a           |
|                                   | commitment to return to Australia |
|                                   | at the end of the posting.        |
|                                   |                                   |
|                                   | Note: long periods outside        |
|                                   | Australia do not preclude a       |
|                                   | person from still being a         |
|                                   | resident of Australia. However,   |
|                                   | taken in isolation a 3 year       |
|                                   | continuous absence is regarded as |
|                                   | an upper limit to still being     |
|                                   | considered residing in Australia, |
|                                   | unless there are special          |
|                                   | circumstances delaying a return.  |
|                                   | When looking at the pattern and   |
|                                   | duration of time spent outside    |
|                                   | Australia, if a person regularly  |
|                                   | spends more than 6 months a year  |
|                                   | outside Australia, their          |
|                                   | residence in Australia is         |
|                                   | questionable.                     |
