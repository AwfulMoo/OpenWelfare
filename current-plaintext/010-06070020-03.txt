Assessing a continuing/intending student's full-time study requirements for YA

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer is intending to study    |
|                                   | and will be claiming/is receiving |
|                                   | YA + Read more ...                |
|                                   |                                   |
|                                   | Does the YA customer meet the     |
|                                   | definition of a                   |
|                                   | continuing/intending student:     |
|                                   |                                   |
|                                   | Yes, and                          |
|                                   |                                   |
|                                   | -   knows the course they are     |
|                                   |     intending to commence, or     |
|                                   |     knows their first course      |
|                                   |     preference if they have       |
|                                   |     applied for more than one     |
|                                   |     course, see Step 1 in Table 2 |
|                                   | -   is unsure of the course they  |
|                                   |     will commence, see Step 2 in  |
|                                   |     Table 2                       |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | New student current on YA or      |
|                                   | JobSeeker Payment (JSP) + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers who are not             |
|                                   | continuing/intending students are |
|                                   | considered to be 'new students'   |
|                                   | and only satisfy the full-time    |
|                                   | study requirements for students   |
|                                   | from the date they                |
|                                   | commence/resume full-time study.  |
|                                   |                                   |
|                                   | Is the new student customer       |
|                                   | currently receiving either YA or  |
|                                   | JSP?                              |
|                                   |                                   |
|                                   | -   Receiving YA as a student, go |
|                                   |     to Step 6                     |
|                                   | -   Receiving JSP or YA as a job  |
|                                   |     seeker, go to Step 3          |
|                                   | -   Not receiving YA or JSP,      |
|                                   |     assess their YA claim.        |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | New students do not satisfy the   |
|                                   | full-time study requirements      |
|                                   | until their course commences +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | These customers must satisfy      |
|                                   | mutual obligation requirements by |
|                                   | undertaking other approved        |
|                                   | activities, no matter how short   |
|                                   | the period is before they         |
|                                   | commence their course.            |
|                                   |                                   |
|                                   | Is the customer under 22 years of |
|                                   | age?                              |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer is under 22 years of     |
|                                   | age + Read more ...               |
|                                   |                                   |
|                                   | Advise customer they can satisfy  |
|                                   | YA mutual obligation requirements |
|                                   | by undertaking approved           |
|                                   | activities, for example,          |
|                                   | voluntary work, training course   |
|                                   | or job search.                    |
|                                   |                                   |
|                                   | Note: customers who have not      |
|                                   | completed Year 12 (or equivalent) |
|                                   | or a Certificate III or higher    |
|                                   | qualification may undertake other |
|                                   | approved activities of at least   |
|                                   | 25 hours per week to satisfy      |
|                                   | their mutual obligation           |
|                                   | requirements. See Mutual          |
|                                   | obligation requirements for Youth |
|                                   | Allowance job seekers under 22    |
|                                   | years of age who have not         |
|                                   | completed Year 12 or an           |
|                                   | equivalent level of education     |
|                                   | (Certificate III or higher)       |
|                                   | qualification.                    |
|                                   |                                   |
|                                   | If the customer is required to    |
|                                   | satisfy mutual obligation         |
|                                   | requirements by job search, they  |
|                                   | are required to look for any      |
|                                   | suitable work which will not      |
|                                   | affect their ability to commence  |
|                                   | their course (for example,        |
|                                   | temporary, part-time or casual    |
|                                   | work).                            |
|                                   |                                   |
|                                   | See Coding new study details.     |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Customer is 22 or over + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Advise customer they must remain  |
|                                   | on JSP and satisfy the JSP mutual |
|                                   | obligation requirements during    |
|                                   | the period prior to commencing    |
|                                   | their course.                     |
|                                   |                                   |
|                                   | They can satisfy mutual           |
|                                   | obligation requirements by        |
|                                   | looking for casual, temporary or  |
|                                   | part-time work that will not      |
|                                   | affect their ability to commence  |
|                                   | their course.                     |
|                                   |                                   |
|                                   | [] To assess the YA (student)     |
|                                   | claim, see Assessing and coding a |
|                                   | transfer to Austudy or Youth      |
|                                   | Allowance (YA) student claim from |
|                                   | another payment.                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Customer receiving YA (student) + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a YA student cannot be         |
|                                   | considered a continuing/intending |
|                                   | student, end the course of study. |
|                                   |                                   |
|                                   | See Student to job seeker         |
|                                   | transfers.                        |
+-----------------------------------+-----------------------------------+

Coding the education details and intention to study for a YA student

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record course details for a YA    |
|                                   | customer who is a                 |
|                                   | continuing/intending student +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Process Direct:                   |
|                                   |                                   |
|                                   | Key 'START' in the Super Key and  |
|                                   | select the Study and Scholarship  |
|                                   | Details workflow from the Task    |
|                                   | Selector                          |
|                                   |                                   |
|                                   | Select Customer Study Details     |
|                                   | task. Select Start                |
|                                   |                                   |
|                                   | Select Add to open the Create     |
|                                   | Education Details screen          |
|                                   |                                   |
|                                   | Participation Status Event Date:  |
|                                   |                                   |
|                                   | -   Student End Date +1 from the  |
|                                   |     previous line of Study        |
|                                   |     Details for YA students       |
|                                   |     continuing from one course to |
|                                   |     a new course, or              |
|                                   | -   Deemed date of claim if       |
|                                   |     processing a YA new claim, or |
|                                   | -   Previous study period end     |
|                                   |     date + 1 for YA job seeker    |
|                                   |     undertaking part time study   |
|                                   |     and returning to full time    |
|                                   |     study                         |
|                                   |                                   |
|                                   | Participation Status: select from |
|                                   | drop down the appropriate details |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | Complete the Receipt Date and     |
|                                   | Channel. Select Save              |
|                                   |                                   |
|                                   | Select Assess. Address any        |
|                                   | errors/warnings on the Errors     |
|                                   | (SWE) screen                      |
|                                   |                                   |
|                                   | Select Assess                     |
|                                   |                                   |
|                                   | Select Finish. Record details of  |
|                                   | the update. Select Finalise       |
|                                   |                                   |
|                                   | Customer First:                   |
|                                   |                                   |
|                                   | Run the Update Study              |
|                                   | Details guided procedure          |
|                                   |                                   |
|                                   | Ensure the Participation Status   |
|                                   | Event Date is:                    |
|                                   |                                   |
|                                   | -   the Student End Date +1 from  |
|                                   |     the previous line of Study    |
|                                   |     Details for YA students       |
|                                   |     continuing from one course to |
|                                   |     a new course, or              |
|                                   | -   the deemed date of claim if   |
|                                   |     processing a YA new claim, or |
|                                   | -   the course start date for     |
|                                   |     current YA job seekers        |
|                                   |                                   |
|                                   | If the course details have been   |
|                                   | provided for a new claim, and the |
|                                   | customer is under 22 years of     |
|                                   | age, determine whether the        |
|                                   | customer wishes to apply for YA   |
|                                   | and satisfy mutual obligation     |
|                                   | requirements by undertaking       |
|                                   | approved activities until their   |
|                                   | course commences. See Mutual      |
|                                   | obligation requirements for Youth |
|                                   | Allowance job seekers under 22    |
|                                   | years of age who have not         |
|                                   | completed Year 12 or an           |
|                                   | equivalent level of education     |
|                                   | (Certificate III or higher)       |
|                                   | qualification.                    |
|                                   |                                   |
|                                   | If a new claimant only wishes for |
|                                   | payment to be granted from the    |
|                                   | course start date, ensure the     |
|                                   | claim is being assessed from the  |
|                                   | course start date (taking into    |
|                                   | account any relevant waiting      |
|                                   | periods that may apply).          |
|                                   |                                   |
|                                   | Record details of customer's      |
|                                   | intended course on a Note/DOC.    |
|                                   |                                   |
|                                   | Note: if the student has left the |
|                                   | course and started a new course,  |
|                                   | the original course must be end   |
|                                   | dated. The end date will be the   |
|                                   | date the customer left the        |
|                                   | course. Procedure ends here.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Record the intention to study +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | To record the intention to study  |
|                                   | in:                               |
|                                   |                                   |
|                                   | -   Process Direct, go to Step 3  |
|                                   | -   Customer First, go to Step 4  |
+-----------------------------------+-----------------------------------+
| 3                                 | Process Direct - record intention |
|                                   | to study + Read more ...          |
|                                   |                                   |
|                                   | Key 'START' in the Super Key and  |
|                                   | select the Study and Scholarship  |
|                                   | Details workflow from the Task    |
|                                   | Selector                          |
|                                   |                                   |
|                                   | Select Customer Study Details.    |
|                                   | Select Start                      |
|                                   |                                   |
|                                   | Select Add. Key the Create        |
|                                   | Education Details fields:         |
|                                   |                                   |
|                                   | -   Participation Status Event    |
|                                   |     Date: key the previous course |
|                                   |     end date +1                   |
|                                   | -   Participation Status: select  |
|                                   |     from drop down                |
|                                   |     INS - Intention to Study      |
|                                   |     Secondary, if the customer is |
|                                   |     intending to study at         |
|                                   |     secondary level               |
|                                   |     INT - Intention to Study      |
|                                   |     Tertiary, if the customer is  |
|                                   |     intending to study at         |
|                                   |     tertiary level                |
|                                   | -   Course Start Date: key the    |
|                                   |     same date as the              |
|                                   |     Participation Status Event    |
|                                   |     Date                          |
|                                   | -   Course End Date: key in the   |
|                                   |     day before the expected start |
|                                   |     date of the intended course.  |
|                                   |     Note: amend this date if      |
|                                   |     required, provided the date   |
|                                   |     is not after the normal       |
|                                   |     course start date for the     |
|                                   |     institution. For example, if  |
|                                   |     the customer misses out on    |
|                                   |     first round university offers |
|                                   |     but expects to pick something |
|                                   |     up in the second round, or is |
|                                   |     investigating other options   |
|                                   |     such as TAFE or private       |
|                                   |     colleges                      |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | Key the Receipt Date and Channel. |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | Select Assess. Address any        |
|                                   | errors/warnings on the Errors     |
|                                   | (SWE) screen                      |
|                                   |                                   |
|                                   | Select Assess                     |
|                                   |                                   |
|                                   | Select Finish. Record details of  |
|                                   | the update. Select Finalise       |
|                                   |                                   |
|                                   | The system will issue a letter    |
|                                   | and set up a review for 3 weeks   |
|                                   | prior to the end of the intention |
|                                   | period, when a Course details     |
|                                   | required for Youth Allowance      |
|                                   | letter will be sent to the        |
|                                   | student for further study         |
|                                   | details. For further details on   |
|                                   | the letter, see Processing end of |
|                                   | course and intention to study     |
|                                   | reviews for Austudy and Youth     |
|                                   | Allowance (YA).                   |
|                                   |                                   |
|                                   | Note: when full course details    |
|                                   | are provided, study details must  |
|                                   | be recorded using the Study and   |
|                                   | Scholarship Details workflow.     |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer First - record intention |
|                                   | to study + Read more ...          |
|                                   |                                   |
|                                   | Run the Update Study              |
|                                   | Details guided procedure          |
|                                   |                                   |
|                                   | Complete only the following       |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Participation Status Event    |
|                                   |     Date - key the previous       |
|                                   |     course end date +1            |
|                                   | -   Participation Status - key:   |
|                                   |     'INS' if the customer is      |
|                                   |     intending to study at         |
|                                   |     secondary level               |
|                                   |     'INT' if the customer is      |
|                                   |     intending to study at         |
|                                   |     tertiary level                |
|                                   | -   Inst Course Start Date - key  |
|                                   |     the same date as              |
|                                   |     the Participation Status      |
|                                   |     Event Date field              |
|                                   | -   Inst Course End Date - key    |
|                                   |     the day before the expected   |
|                                   |     start date of the intended    |
|                                   |     course. Note: this date can   |
|                                   |     be amended if required,       |
|                                   |     provided the date is not      |
|                                   |     after the normal course start |
|                                   |     date for the institution. For |
|                                   |     example, if the customer      |
|                                   |     misses out on first round     |
|                                   |     university offers but expects |
|                                   |     to pick something up in the   |
|                                   |     second round, or is           |
|                                   |     investigating other options   |
|                                   |     such as TAFE or private       |
|                                   |     colleges                      |
|                                   |                                   |
|                                   | Finalise the activity             |
|                                   |                                   |
|                                   | Record details of customer's      |
|                                   | intended course on a DOC          |
|                                   |                                   |
|                                   | The system will issue a letter    |
|                                   | and set up a review for 3 weeks   |
|                                   | prior to the end of the intention |
|                                   | period, when a Course details     |
|                                   | required for Youth Allowance      |
|                                   | letter will be sent to the        |
|                                   | student for further study         |
|                                   | details. For further details on   |
|                                   | the letter, see Processing end of |
|                                   | course and intention to study     |
|                                   | reviews for Austudy and Youth     |
|                                   | Allowance (YA).                   |
|                                   |                                   |
|                                   | Note: when full course details    |
|                                   | are provided, study details must  |
|                                   | be recorded using the Update      |
|                                   | Study Details guided procedure.   |
