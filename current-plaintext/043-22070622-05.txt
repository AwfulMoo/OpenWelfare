Examples -calculating freight costs using TAS016

Table1: Freight assistance is paid on eligible products. The total
freight paid may include an ineligible portion of the vehicle. The
TAS016 provides details to calculate the cost of the freight for the
eligible product.

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Vans and panel vans               |
|                                   |                                   |
|                                   | A sportsperson claimant ships     |
|                                   | eligible sporting equipment in    |
|                                   | the cargo area of a transit van.  |
|                                   | The claimant provides a TAS016    |
|                                   | showing the following             |
|                                   | measurements of the vehicle:      |
|                                   |                                   |
|                                   | -   XL – 592 cms                  |
|                                   | -   XH – 228 cms                  |
|                                   | -   XW – 195 cms                  |
|                                   | -   IL – 234 cms                  |
|                                   | -   IH – 198 cms                  |
|                                   | -   IW – 165 cms                  |
|                                   |                                   |
|                                   | To calculate the entire volume of |
|                                   | the vehicle:                      |
|                                   |                                   |
|                                   | -   Multiply XL by XH and XW      |
|                                   | -   592cm X 228cm X 195cm =       |
|                                   |     26.32m3                       |
|                                   |                                   |
|                                   | To calculate the eligible cargo   |
|                                   | area of the vehicle:              |
|                                   |                                   |
|                                   | -   Multiply IL by IH and IW      |
|                                   | -   234 cm X 198 cm X 165 cm =    |
|                                   |     10.403m3                      |
|                                   |                                   |
|                                   | The tax invoice shows a freight   |
|                                   | charge for the entire van of      |
|                                   | $1000 (ex. GST). To calculate the |
|                                   | freight charge for the eligible   |
|                                   | portion:                          |
|                                   |                                   |
|                                   | -   Divide the total freight      |
|                                   |     charged by the entire volume  |
|                                   |     of the vehicle                |
|                                   | -   $1000/26.32m3 = $38 per m3    |
|                                   | -   Multiple $38 by the eligible  |
|                                   |     volume of the vehicle         |
|                                   | -   $38 X 10.403m3 = $395.31 is   |
|                                   |     the eligible portion of the   |
|                                   |     freight cost for the shipment |
+-----------------------------------+-----------------------------------+
| 2                                 | Trailers and Prime Movers         |
|                                   |                                   |
|                                   | A professional entertainer        |
|                                   | claimant ships eligible equipment |
|                                   | used for performing. The          |
|                                   | equipment is transported by a     |
|                                   | prime mover with a trailer. The   |
|                                   | claimant provides a TAS016        |
|                                   | showing the following             |
|                                   | measurements of the prime mover   |
|                                   | and trailer:                      |
|                                   |                                   |
|                                   | -   XL – 750 cms                  |
|                                   | -   IL – 600 cms                  |
|                                   |                                   |
|                                   | The tax invoice shows one freight |
|                                   | charge of $6000 (ex. GST) for the |
|                                   | prime mover and the trailer       |
|                                   | combined. To calculate the        |
|                                   | freight charge for the eligible   |
|                                   | portion:                          |
|                                   |                                   |
|                                   | -   Divide the total freight      |
|                                   |     charge by the entire length   |
|                                   |     of the vehicle                |
|                                   | -   $6000/750 = $8 per cm         |
|                                   | -   Multiple $8 by the eligible   |
|                                   |     length of the vehicle         |
|                                   | -   $8 X 600 = $4800 is the       |
|                                   |     eligible portion of the       |
|                                   |     freight cost for the shipment |
+-----------------------------------+-----------------------------------+
