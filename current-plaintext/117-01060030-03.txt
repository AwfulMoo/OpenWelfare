For Compensation Recovery Teams (CRT) only.

Reactivate a finalised compensation claim to allow updates

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | A claim for compensation is to be |
|                                   | reactivated + Read more ...       |
|                                   |                                   |
|                                   | A claim for compensation has a    |
|                                   | status of finalised on the        |
|                                   | Compensation Management System    |
|                                   | (CMS).                            |
|                                   |                                   |
|                                   | -   Select Claim Summary twisty   |
|                                   | -   Select Claim ID               |
|                                   | -   Select Reactivate Claim       |
|                                   | -   The Reactivate screen will    |
|                                   |     show                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Update the Reactivate Claim       |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | -   Select Reactivate Reason:     |
|                                   |     field. Select valid option    |
|                                   |     from drop down box            |
|                                   | -   Complete the Source and       |
|                                   |     Receipt Date fields           |
|                                   | -   Select Continue. The          |
|                                   |     Compensation Management       |
|                                   |     Summary screen will show.     |
|                                   |     Changes made on the           |
|                                   |     Reactivate Claim screen will  |
|                                   |     save                          |
|                                   | -   Select Back. The changes made |
|                                   |     on the Reactivate Claim       |
|                                   |     screen will not save          |
|                                   | -   The claim will revert to the  |
|                                   |     status that it had prior to   |
|                                   |     having been finalised         |
+-----------------------------------+-----------------------------------+
| 3                                 | Create or annotate a DOC + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Complete a Fast Note noting the   |
|                                   | action taken.                     |
+-----------------------------------+-----------------------------------+
