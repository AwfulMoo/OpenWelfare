Referral to Australian Government Actuary (AGA)

Documents and supporting information required for referral to AGA

This table contains a list of assets and the information required when a
referral to the AGA is made.

+-----------------------------------+-----------------------------------+
| Asset                             | Details required                  |
+-----------------------------------+-----------------------------------+
| Properties                        | -   address                       |
|                                   | -   whether it is a residential,  |
|                                   |     commercial or farming         |
|                                   |     property                      |
|                                   | -   whether the customer resides  |
|                                   |     at the property               |
|                                   | -   whether there is any reason   |
|                                   |     to expect subdued rental      |
|                                   |     return (and if so, why)       |
|                                   | -   current rental return will be |
|                                   |     useful but not necessary      |
|                                   | -   property valuation report as  |
|                                   |     at (or within 6 months to)    |
|                                   |     the valuation dates (if a     |
|                                   |     customer has a life interest  |
|                                   |     in a farming property that    |
|                                   |     he/she resides at, then       |
|                                   |     please provide a separate     |
|                                   |     value for house and           |
|                                   |     curtilage)                    |
+-----------------------------------+-----------------------------------+
| Shares                            | -   names and quantities          |
|                                   | -   portfolio statements as at    |
|                                   |     (or within 6 months to) the   |
|                                   |     valuation dates               |
+-----------------------------------+-----------------------------------+
| Cash and term deposits            | -   account type                  |
|                                   | -   term of the investment (if    |
|                                   |     available)                    |
|                                   | -   bank statement as at (or      |
|                                   |     within 6 months to) the       |
|                                   |     valuation dates               |
+-----------------------------------+-----------------------------------+
| Managed portfolio                 | -   name of the managed portfolio |
|                                   | -   portfolio asset allocation    |
|                                   |     (useful but not necessary)    |
|                                   | -   portfolio statements as at    |
|                                   |     (or within 6 months to) the   |
|                                   |     valuation dates               |
+-----------------------------------+-----------------------------------+
