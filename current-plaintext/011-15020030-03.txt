Request a manual care costs check

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claimant or injured person has    |
|                                   | declared ‘yes’ to the care costs  |
|                                   | question on MO028 + Read more ... |
|                                   |                                   |
|                                   | A manual care cost check is       |
|                                   | required:                         |
|                                   |                                   |
|                                   | -   if the case has settled and   |
|                                   |     the end date of the care      |
|                                   |     costs calculation is after    |
|                                   |     the date of judgment or       |
|                                   |     settlement. The authorised    |
|                                   |     staff member will complete a  |
|                                   |     recalculation                 |
|                                   | -   if there are special          |
|                                   |     circumstances for the case.   |
|                                   |     For example, the case is      |
|                                   |     about to settle and an urgent |
|                                   |     issue of the notice of past   |
|                                   |     benefits has been requested   |
|                                   | -   after settlement and the case |
|                                   |     is nearing the end of the 3   |
|                                   |     month period in which a       |
|                                   |     notice of charge can be       |
|                                   |     issued (nearing the s33D or   |
|                                   |     s24(4) date)                  |
|                                   | -   if there have been previous   |
|                                   |     care costs included for a     |
|                                   |     different case for the same   |
|                                   |     compensable person            |
|                                   | -   to follow up a previous care  |
|                                   |     cost request that was not     |
|                                   |     responded to                  |
|                                   |                                   |
|                                   | Does the case meet one of the     |
|                                   | above reasons for a manual care   |
|                                   | cost request?                     |
|                                   |                                   |
|                                   | -   Yes, go to step 2             |
|                                   | -   No, request a care cost check |
|                                   |     in CCMS. See Processing       |
|                                   |     statement by claimant table.  |
+-----------------------------------+-----------------------------------+
| 2                                 | Manual care costs check + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the 'care costs check         |
|                                   | request' - email template to      |
|                                   | request a manual care costs       |
|                                   | check. The Resources page         |
|                                   | contains a link to the template.  |
+-----------------------------------+-----------------------------------+
| 3                                 | Add a case note + Read more ...   |
|                                   |                                   |
|                                   | Select the Add notes button from  |
|                                   | the list on the right hand side   |
|                                   | of the main CCMS screen.          |
|                                   |                                   |
|                                   | -   The top box shows all notes   |
|                                   |     previously recorded on this   |
|                                   |     case                          |
|                                   | -   In the blank bottom box write |
|                                   |     'Manual care costs check      |
|                                   |     requested'                    |
|                                   | -   Select Add to save notes. A   |
|                                   |     confirmation box will appear, |
|                                   |     select 'Yes' to add case note |
|                                   | -   Press [F4] to release the     |
|                                   |     work item, and the case will  |
|                                   |     suspend on CCMS               |

Refer a request for a review of care costs to Programme Management

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Escalate to Programme Management  |
|                                   | (PM) email template + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Access the 'Escalation to     |
|                                   |     Programme Management via TL5  |
|                                   |     or PSO' - email template from |
|                                   |     the Resources page            |
|                                   | -   Complete and send to the      |
|                                   |     Compensation Team Leaders     |
|                                   |     mailbox for review and        |
|                                   |     discussion with a Team        |
|                                   |     Leader. The supporting        |
|                                   |     documents must be attachedIf  |
|                                   |     a Team Leader agrees with the |
|                                   |     escalation, the Team Leader   |
|                                   |     will forward the email        |
|                                   |     template and attachments to   |
|                                   |     the Programme Management      |
|                                   |     mailbox for action            |
+-----------------------------------+-----------------------------------+
| 2                                 | Add a case note + Read more ...   |
|                                   |                                   |
|                                   | Select the Add notes button from  |
|                                   | the list on the right hand side   |
|                                   | of the main CCMS screen.          |
|                                   |                                   |
|                                   | The top box shows all notes       |
|                                   | previously recorded on this case  |
|                                   |                                   |
|                                   | In the blank bottom box enter a   |
|                                   | description about:                |
|                                   |                                   |
|                                   | -   details of the document/s     |
|                                   |     received that dispute the     |
|                                   |     care costs, and who has       |
|                                   |     submitted the dispute and     |
|                                   |     supporting evidence, and      |
|                                   | -   the case will be escalated to |
|                                   |     PM for a decision             |
|                                   |                                   |
|                                   | Select Add to save notes. A       |
|                                   | confirmation box will appear,     |
|                                   | select 'Yes' to add case note     |
|                                   |                                   |
|                                   | Press [F4] to release the work    |
|                                   | item, and the case will suspend   |
|                                   | on CCMS                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Scan copy to case on CCMS + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Print a copy of the completed     |
|                                   | 'Escalation to PM' email          |
|                                   | template.                         |
|                                   |                                   |
|                                   | Print copies of any/all           |
|                                   | supporting documents for example, |
|                                   | letter from the solicitor, letter |
|                                   | from treating medical             |
|                                   | practitioner.                     |
|                                   |                                   |
|                                   | Write '11' enclosed in a circle   |
|                                   | in the top right hand corner of   |
|                                   | the first page and deliver it to  |
|                                   | the scanning room.                |
+-----------------------------------+-----------------------------------+
| 4                                 | Decision by Programme Management  |
|                                   | and advice sent to Service        |
|                                   | Delivery + Read more ...          |
|                                   |                                   |
|                                   | Programme Management may consult  |
|                                   | Department of Health and Aged     |
|                                   | Care for complex cases. Programme |
|                                   | Management will review the        |
|                                   | evidence and make a decision to   |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   reduce the care costs charge  |
|                                   |     (in part or whole), or        |
|                                   | -   keep the care costs charge    |
|                                   |     unchanged                     |
|                                   |                                   |
|                                   | Programme Management will then    |
|                                   | send a reply email to the         |
|                                   | Compensation Team Leaders mailbox |
|                                   | and the requesting officer to     |
|                                   | advise of the decision.           |
+-----------------------------------+-----------------------------------+
| 5                                 | Service Officer receives the      |
|                                   | escalation outcome and takes      |
|                                   | further action + Read more ...    |
|                                   |                                   |
|                                   | Upon receipt of the escalation    |
|                                   | outcome:                          |
|                                   |                                   |
|                                   | -   print the email to scan to    |
|                                   |     the case (write 11 enclosed   |
|                                   |     in a circle at the top of the |
|                                   |     document and take it to the   |
|                                   |     scanning room for scanning)   |
|                                   | -   where the decision is to not  |
|                                   |     to amend the charge,          |
|                                   |     communicate the decision by   |
|                                   |     telephone to the person who   |
|                                   |     submitted the dispute         |
|                                   | -   where the person remains      |
|                                   |     unsatisfied with this         |
|                                   |     decision, advise of their     |
|                                   |     option to submit an appeal to |
|                                   |     the Ombudsman or the Minister |
|                                   |     for Services Australia        |
|                                   | -   where the decision is to      |
|                                   |     amend the care costs amount,  |
|                                   |     amend the notice of charge by |
|                                   |     entering case notes then      |
|                                   |     continue with the process     |
|                                   |     steps in Amended notices and  |
|                                   |     reviews for Medicare          |
|                                   |     Compensation Recovery         |
+-----------------------------------+-----------------------------------+
| 6                                 | Add a further case note + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the Add notes button from  |
|                                   | the list on the right hand side   |
|                                   | of the main CCMS screen.          |
|                                   |                                   |
|                                   | -   The top box shows all notes   |
|                                   |     previously recorded on this   |
|                                   |     case                          |
|                                   | -   In the blank bottom box enter |
|                                   |     details of the PM decision    |
|                                   |     and the communication with    |
|                                   |     the person who made the       |
|                                   |     request                       |
|                                   | -   Select Add to save notes. A   |
|                                   |     confirmation box will appear, |
|                                   |     select 'Yes' to add case note |
|                                   | -   Press [F4] to release the     |
|                                   |     work item, and the case will  |
|                                   |     suspend on CCMS               |
