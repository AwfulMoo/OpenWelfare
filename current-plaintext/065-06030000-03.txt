Checking customer details and the claim

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Lodgement of claim or             |
|                                   | documentation + Read more ...     |
|                                   |                                   |
|                                   | Service Officers must stream the  |
|                                   | claim on the day it is received   |
|                                   | and check that a claim assessment |
|                                   | can be completed.                 |
|                                   |                                   |
|                                   | See Claiming Commonwealth Seniors |
|                                   | Health Card (CSHC) for what to do |
|                                   | upon receipt of a paper claim or  |
|                                   | documentation to support a claim, |
|                                   | and [] how to help customers      |
|                                   | upload documents for an online or |
|                                   | Assisted Customer Claim (ACC).    |
|                                   |                                   |
|                                   | An online or ACC for CSHC cannot  |
|                                   | be submitted until all required   |
|                                   | tasks have been completed         |
|                                   | (exceptions apply).               |
|                                   |                                   |
|                                   | Verbal claims completed using ACC |
|                                   | must not be submitted or          |
|                                   | finalised until a verbal          |
|                                   | declaration has been accepted by  |
|                                   | the customer or a signed Customer |
|                                   | Declaration Form (CDF) has been   |
|                                   | scanned to their record. See the  |
|                                   | Using Assisted Customer Claim     |
|                                   | (ACC) for CSHC table for more     |
|                                   | information.                      |
|                                   |                                   |
|                                   | If a rejected claim needs to be   |
|                                   | opened and reassessed, create a   |
|                                   | SOA shell in Process Direct. For  |
|                                   | more information see Cancelling   |
|                                   | claim activities and re-indexing  |
|                                   | claims.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Locate and review claim + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Request or search for the claim   |
|                                   | work item via Inbox:              |
|                                   |                                   |
|                                   | On the Transaction Summary Page   |
|                                   | (TS) select [Open left slider;    |
|                                   | Close right slider] to review the |
|                                   | customer’s claim responses and    |
|                                   | scanned documents.                |
|                                   |                                   |
|                                   | Online claim/ACC submitted        |
|                                   |                                   |
|                                   | When an online or ACC is          |
|                                   | submitted, a CSHC claim work item |
|                                   | is created in Process Direct      |
|                                   | containing the claim information. |
|                                   | Go to Step 4                      |
|                                   |                                   |
|                                   | Paper claim lodged                |
|                                   |                                   |
|                                   | Where a paper CSHC claim (SA296,  |
|                                   | SA296A, SA443) has been uploaded  |
|                                   | view the scanned claim form. See  |
|                                   | Viewing Centrelink customers'     |
|                                   | digital images.                   |
|                                   |                                   |
|                                   | For partnered customers, take     |
|                                   | note if lodged as a joint claim.  |
|                                   | Where a joint claim has been      |
|                                   | lodged, the assessment process    |
|                                   | has to be followed for both       |
|                                   | customer and partner.             |
+-----------------------------------+-----------------------------------+
| 3                                 | SOA shell created + Read more ... |
|                                   |                                   |
|                                   | When a paper claim is scanned to  |
|                                   | the customer's record, a SOA      |
|                                   | shell work item is generated on   |
|                                   | the customer's record. This       |
|                                   | contains no claim information     |
|                                   | apart from the:                   |
|                                   |                                   |
|                                   | -   Customer Reference Number     |
|                                   |     (CRN)                         |
|                                   | -   Claim ID                      |
|                                   | -   Claim dates and               |
|                                   | -   personal details.             |
|                                   |                                   |
|                                   | The claim information is added    |
|                                   | into the work item using the      |
|                                   | scanned claim and documentation.  |
|                                   |                                   |
|                                   | If the customer and partner have  |
|                                   | lodged a joint claim, when the    |
|                                   | claim is ready to be processed a  |
|                                   | manual SOA shell for the partner  |
|                                   | will be required.                 |
|                                   |                                   |
|                                   | Common screens that may need      |
|                                   | updating via SWE:                 |
|                                   |                                   |
|                                   | -   personal information (name,   |
|                                   |     address, phone number)        |
|                                   | -   interpreter information and   |
|                                   |     language details (OCD)        |
|                                   | -   Office Code (OC)              |
|                                   | -   Payment Destination Details   |
|                                   |     (PAS/PAD/PASB/PA/PAN) screen. |
|                                   |     If provided, select to add or |
|                                   |     update a customer's payment   |
|                                   |     destination. Update           |
|                                   |     customer's BSB, account       |
|                                   |     number, account title for     |
|                                   |     Benefit System: SHC – Seniors |
|                                   |     Health Card                   |
|                                   | -   Alternative identity (POI) if |
|                                   |     required                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Check Customer Summary + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Before contacting the customer or |
|                                   | processing the claim, staff must  |
|                                   | check the Customer Summary in     |
|                                   | Process Direct and/or Customer    |
|                                   | First for customer specific       |
|                                   | warnings.                         |
|                                   |                                   |
|                                   | This includes checking:           |
|                                   |                                   |
|                                   | -   the Customer Incident         |
|                                   |     Management System (CIMS), and |
|                                   | -   the Managed Service Plan      |
|                                   |     (MSP). See Customer           |
|                                   |     aggression - Managed Service  |
|                                   |     Plan (MSP) for more           |
|                                   |     information                   |
|                                   |                                   |
|                                   | This alerts staff to the actions  |
|                                   | they can complete for the         |
|                                   | customer, and what action they    |
|                                   | advise customers to take.         |
+-----------------------------------+-----------------------------------+
| 5                                 | Qualification on date of claim +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | On the day the claim was lodged   |
|                                   | was the customer Age Pension age? |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, the customer is not       |
|                                   |     qualified and claim must be   |
|                                   |     rejected. Go to Processing    |
|                                   |     Commonwealth Seniors Health   |
|                                   |     Card (CSHC) claims. Procedure |
|                                   |     ends here                     |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the Age Pension age rules.     |
|                                   |                                   |
|                                   | Note: early claim provisions do   |
|                                   | not apply to the CSHC. If the     |
|                                   | person is not qualified for CSHC  |
|                                   | on the date of claim lodgement    |
|                                   | the claim must be rejected. This  |
|                                   | includes the following            |
|                                   | situations:                       |
|                                   |                                   |
|                                   | -   A customer has claimed CSHC   |
|                                   |     prior to becoming Age Pension |
|                                   |     age                           |
|                                   | -   Before the Age Pension        |
|                                   |     cancellation date             |
|                                   | -   Before the end of a NARWP     |
+-----------------------------------+-----------------------------------+
| 6                                 | Residence requirements + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | On the day the claim was lodged,  |
|                                   | does the customer satisfy the     |
|                                   | residence requirements? See       |
|                                   | Residence assessment for          |
|                                   | customers claiming Commonwealth   |
|                                   | Seniors Health Card (CSHC).       |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, the customer is not       |
|                                   |     qualified and claim must be   |
|                                   |     rejected. See Processing      |
|                                   |     Commonwealth Seniors Health   |
|                                   |     Card (CSHC) claims. Procedure |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Customer located in Australia +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check if the customer was         |
|                                   | physically present in Australia   |
|                                   | on the date they lodged their     |
|                                   | claim, by checking the customer’s |
|                                   | travel movements (if available)   |
|                                   | using the Travel Outside          |
|                                   | Australia Summary (TOAS) and      |
|                                   | Immigration Movements (RSIM)      |
|                                   | screens.                          |
|                                   |                                   |
|                                   | On the day the claim was lodged,  |
|                                   | is there evidence that the        |
|                                   | customer was not in Australia?    |
|                                   |                                   |
|                                   | -   Yes, the customer is not      |
|                                   |     qualified and claim must be   |
|                                   |     rejected. See Processing      |
|                                   |     Commonwealth Seniors Health   |
|                                   |     Card (CSHC) claims. Procedure |
|                                   |     ends here                     |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 8                                 | Customer receives a pension or    |
|                                   | allowance + Read more ...         |
|                                   |                                   |
|                                   | On the day the claim was lodged,  |
|                                   | was the customer in receipt       |
|                                   | either of a:                      |
|                                   |                                   |
|                                   | Social Security pension, or       |
|                                   |                                   |
|                                   | Social Security benefit           |
|                                   |                                   |
|                                   | -   Yes, the customer is not      |
|                                   |     qualified and claim must be   |
|                                   |     rejected Go to Processing     |
|                                   |     Commonwealth Seniors Health   |
|                                   |     Card (CSHC) claims. Procedure |
|                                   |     ends here                     |
|                                   | -   No, go to Step 9              |
+-----------------------------------+-----------------------------------+
| 9                                 | Customer receives a DVA pension   |
|                                   | or allowance + Read more ...      |
|                                   |                                   |
|                                   | On the day the claim was lodged,  |
|                                   | was the customer in receipt any   |
|                                   | of:                               |
|                                   |                                   |
|                                   | -   Department of Veterans'       |
|                                   |     Affairs (DVA) Service         |
|                                   |     pension,                      |
|                                   | -   DVA Age Pension,              |
|                                   | -   Veteran Payment               |
|                                   | -   Income Support Supplement     |
|                                   |     (ISS)                         |
|                                   | -   DVA CSHC                      |
|                                   |                                   |
|                                   | See Income from Department of     |
|                                   | Veterans' Affairs (DVA) for more  |
|                                   | information.                      |
|                                   |                                   |
|                                   | -   Yes, the customer is not      |
|                                   |     qualified and claim must be   |
|                                   |     rejected. See Processing      |
|                                   |     Commonwealth Seniors Health   |
|                                   |     Card (CSHC) claims. Procedure |
|                                   |     ends here                     |
|                                   | -   No, go to Step 10             |
+-----------------------------------+-----------------------------------+
| 10                                | Identity Confirmation + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The claimant must meet the        |
|                                   | Identity Confirmation             |
|                                   | requirements. Identity            |
|                                   | Confirmation for a partner is     |
|                                   | only needed if the partner is     |
|                                   | also applying for CSHC.           |
|                                   |                                   |
|                                   | On the TS screen, check the       |
|                                   | customer's Identity Status.       |
|                                   |                                   |
|                                   | Has the customer met the Identity |
|                                   | Confirmation requirements?        |
|                                   |                                   |
|                                   | Yes, go to Step 11                |
|                                   |                                   |
|                                   | No, consider the following        |
|                                   | options:                          |
|                                   |                                   |
|                                   | -   provide identity documents,   |
|                                   |     see Alternative Identity      |
|                                   | -   provide proof of age, see     |
|                                   |     Verifying the date of birth   |
|                                   |     or age of a customer or child |
|                                   | -   attend in person to confirm   |
|                                   |     identity, see Supporting      |
|                                   |     Identity Confirmation for     |
|                                   |     customers unable to attend in |
|                                   |     person                        |
|                                   |                                   |
|                                   | Where the identification          |
|                                   | requirements cannot be met, the   |
|                                   | claim will be rejected. See       |
|                                   | Processing Commonwealth Seniors   |
|                                   | Health Card (CSHC) claims.        |
|                                   | Procedure ends here               |
+-----------------------------------+-----------------------------------+
| 11                                | Tax File Number (TFN) + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Has the customer provided their   |
|                                   | (and their partner's) Tax File    |
|                                   | Number (TFN) (unless an exemption |
|                                   | is granted)?                      |
|                                   |                                   |
|                                   | See Tax File Number (TFN) for     |
|                                   | Commonwealth Seniors Health Card  |
|                                   | (CSHC) for more information.      |
|                                   |                                   |
|                                   | -   Yes, go to Step 12            |
|                                   | -   No, a request for information |
|                                   |     is required, take note of the |
|                                   |     required information, go to   |
|                                   |     Step 12                       |
+-----------------------------------+-----------------------------------+
| 12                                | CSHC income test + Read more ...  |
|                                   |                                   |
|                                   | Has the customer declared         |
|                                   | Adjusted Taxable Income (ATI) for |

|                                   | Commonwealth Seniors Health Card  |
|                                   | (CSHC) - Taxable Income Limit?    |
|                                   |                                   |
|                                   | Note: this should include any     |
|                                   | deemed income from account based  |
|                                   | income streams.                   |
|                                   |                                   |
|                                   | For details of Adjusted Taxable   |
|                                   | Income (ATI) for CSHC, see        |
|                                   | Commonwealth Seniors Health Card  |

|                                   | See Rates and Thresholds for      |
|                                   | Commonwealth Seniors Health Card  |
|                                   | (CSHC) - Taxable Income Limits    |
|                                   |                                   |
|                                   | -   Yes, go to Step 13            |
|                                   | -   No, the claim will be         |
|                                   |     rejected due to income. See   |
|                                   |     Processing Commonwealth       |
|                                   |     Seniors Health Card (CSHC)    |
|                                   |     claims. Procedure ends here   |
+-----------------------------------+-----------------------------------+
| 13                                | Evidence of income + Read more    |
|                                   | ...                               |
|                                   |                                   |

|                                   | evidence of income for            |

|                                   |                                   |
