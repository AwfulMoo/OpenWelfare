This page contains the process for reinstating and reactivating
Healthcare Provider Identifier - Organisation (HPI-O) organisations in
the Healthcare Identifiers (HI) Service.

Reinstate HPI-O record

This procedure reinstates a HPI-O to the status it held before it was
retired (that is, either 'Active' or 'Deactivated').

Reinstate HPI-O record that has been retired in error

This table describes how to reinstate a HPI-O record that has been
'retired' in the HI Service in error.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Perform a search for the          |
|                                   | organisation being reinstated +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | From the Healthcare Identifier    |
|                                   | eHealth program (EHP), perform a  |
|                                   | search for the organisation being |
|                                   | reinstated.                       |
|                                   |                                   |
|                                   | See Searching for records in the  |
|                                   | Healthcare Identifiers Service    |
|                                   | using the eHealth Program (EHP)   |
+-----------------------------------+-----------------------------------+
| 2                                 | Change Organisation Activity      |
|                                   | Status + Read more ...            |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Amend in the Action column of |
|                                   |     the relevant record           |
|                                   | -   the Organisation Details tab  |
|                                   | -   Reinstate                     |
|                                   |                                   |
|                                   | The Organisation Activity Status  |
|                                   | changes to Active or Deactivated, |
|                                   | depending on its status before    |
|                                   | retirement.                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Key comment + Read more ...       |
|                                   |                                   |
|                                   | Key a comment into the Reason for |
|                                   | Activity Status change field.     |
+-----------------------------------+-----------------------------------+
| 4                                 | Select Submit + Read more ...     |
|                                   |                                   |
|                                   | If there are:                     |
|                                   |                                   |
|                                   | -   validation errors, correct    |
|                                   |     errors as prompted            |
|                                   | -   no validation errors, the     |
|                                   |     Confirm Amend HPI-O Network   |
|                                   |     Details screen is displayed   |
+-----------------------------------+-----------------------------------+
| 5                                 | Check details + Read more ...     |
|                                   |                                   |
|                                   | If the details are:               |
|                                   |                                   |
|                                   | -   incorrect, select Edit to     |
|                                   |     return to the Amend           |
|                                   |     Seed/Networked HPI-O screen   |
|                                   | -   correct, select Save          |
|                                   |                                   |
|                                   | A confirmation message is         |
|                                   | displayed. Go to Step 6.          |
+-----------------------------------+-----------------------------------+
| 6                                 | Contact Tier 3 support + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Contact Tier 3 support to arrange |
|                                   | for a letter of notification to   |
|                                   | be mailed to the organisation.    |
