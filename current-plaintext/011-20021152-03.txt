Assessing claims notated with ‘not for comparison’

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | The claim has the text/notation   |
|                                   | ‘not for comparison’ (or to that  |
|                                   | effect)                           |
|                                   |                                   |
|                                   | ‘Not for comparison’ (or to that  |
|                                   | effect) can only be used if the   |
|                                   | claim is for multiple items for   |
|                                   | the same body area (for example,  |
|                                   | 2 x 57506 or 57506 and 57512).    |
|                                   |                                   |
|                                   | Is the claim for multiple         |
|                                   | diagnostic radiology (Category 5, |
|                                   | Group I3) items for the same body |
|                                   | area?                             |
|                                   |                                   |
|                                   | -   Yes, pay the claim using      |
|                                   |     PI 42. The system will apply  |
|                                   |     the DIMSR and calculate the   |
|                                   |     benefit amounts for each item |
|                                   | -   No (it is for items not for   |
|                                   |     the same body area for        |
|                                   |     example, leg and arm). Assess |
|                                   |     the claim based on the        |
|                                   |     assessing rules and           |
|                                   |     information for the items     |
|                                   |     being claimed. ‘Not for       |
|                                   |     comparison’ is not            |
|                                   |     appropriate text for the      |
|                                   |     claim to be paid              |
+-----------------------------------+-----------------------------------+
