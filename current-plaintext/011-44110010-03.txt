|                                   | -   Birth certificates for        |
|                                   |     Northern Territory (NT) with  |
|                                   |     Aboriginal Population Record  |
|                                   |     (APR)                         |

|                                   | Australian visa holders who do    |
|                                   | not have Australian citizenship,  |
|                                   | who ‘return’ to Australia and ask |
|                                   | to change their legal name must   |
|                                   | provide:                          |
|                                   |                                   |
|                                   | -   A foreign passport with a     |
|                                   |     current Australian visa       |
|                                   |     (verified)                    |
|                                   |                                   |
|                                   | See Escalation process for        |
|                                   | Medicare Eligibility for help.    |
|                                   |                                   |
|                                   | The name must be changed at both  |
|                                   | Personal and Group unless the     |
|                                   | applicant produces documents to   |
|                                   | support a preferred name change   |
|                                   | at Group. See Table 4 Change of   |
|                                   | legal name in Personal.           |
|                                   |                                   |
|                                   | If this customer is the group     |
|                                   | contact, the name of the group    |
|                                   | contact must also be changed, see |
|                                   | Amend customer details in the     |
|                                   | Consumer Directory Maintenance    |
|                                   | System (CDMS) for Medicare.       |
|                                   |                                   |
|                                   | Where a name has been previously  |
|                                   | changed at Group and the Comment  |
|                                   | field indicates that the          |
|                                   | documents sighted would be        |
|                                   | sufficient to allow a name to be  |
|                                   | amended at Personal (for example, |
|                                   | a marriage certificate), a name   |
|                                   | at Personal can be changed.       |
|                                   |                                   |
|                                   | See Table 4 Change of legal name  |
|                                   | in Personal.                      |
|                                   |                                   |
|                                   | Note: if a customer provides an   |
|                                   | Australian Passport or Australian |
|                                   | Citizenship Certificate in their  |
|                                   | new name along with a foreign     |
|                                   | Marriage certificate or foreign   |
|                                   | birth certificate, these matters  |
|                                   | need to be escalated to Medicare  |
|                                   | Consumers Program Support.        |
|                                   |                                   |
|                                   | See Escalation process for        |
|                                   | Medicare Eligibility.             |

|                                   | sighted to action the change of   |

Register a customer with a long name in Group

Table 7

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Long name + Read more ...         |
|                                   |                                   |
|                                   | A name is considered long when    |
|                                   | any of the following apply:       |
|                                   |                                   |
|                                   | -   the person's last name        |
|                                   |     contains more than 18         |
|                                   |     characters                    |
|                                   | -   their first given name        |
|                                   |     contains more than 12         |
|                                   |     characters, or                |
|                                   | -   the combination of their      |
|                                   |     first name, initial and       |
|                                   |     family name have more than 25 |
|                                   |     characters (including spaces  |
|                                   |     between the names)            |
+-----------------------------------+-----------------------------------+
| 2                                 | Reduce a long name + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Register a new customer       |
|                                   | -   Register customer’s Medicare  |
|                                   |     entitlement                   |
|                                   |                                   |
|                                   | After a customer’s entitlement is |
|                                   | registered:                       |
|                                   |                                   |
|                                   | -   Select Group on the           |
|                                   |     Registration Type screen      |
|                                   | -   The New Medicare Card Details |
|                                   |     screen displays               |
|                                   |                                   |
|                                   | The New Medicare Card Details     |
|                                   | screen is displayed. It shows how |
|                                   | the name will look on the         |
|                                   | Medicare card. Some of the        |
|                                   | characters will be missing.       |
|                                   |                                   |
|                                   | Reduce the name:                  |
|                                   |                                   |
|                                   | -   In the Name field (which has  |
|                                   |     the long name) delete the     |
|                                   |     characters at the end. Do     |
|                                   |     this until the name fits in   |
|                                   |     the field                     |
