|                                   | record in Process Direct record   |

|                                   |                                   |

+-----------------------------------+-----------------------------------+
| 3                                 | Requesting the retrieval of an    |
|                                   | archived claim + Read more ...    |
|                                   |                                   |
|                                   | To retrieve an archived claim, 1  |
|                                   | of the 2 mandatory criteria of    |
|                                   | CRN or Transaction ID must be     |
|                                   | entered. The Transaction ID field |
|                                   | is used to input the Claim ID.    |
|                                   | The Date From and Date To are     |
|                                   | optional fields and may be        |
|                                   | entered to refine the retrieved   |
|                                   | results.                          |
|                                   |                                   |
|                                   | Press Enter or select the         |
|                                   | ‘Search’ icon at the top right of |
|                                   | the application.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Retrieved claim display + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The below details for the Claim   |
|                                   | are displayed upon successful     |
|                                   | search:                           |
|                                   |                                   |
|                                   | -   Customer (CRN) Header Summary |
|                                   | -   Claims Details (Attachments)  |
|                                   | -   Claim Status Change Documents |
|                                   | -   Follow on Activities (if any) |
+-----------------------------------+-----------------------------------+
| 5                                 | Restricted Access (RA)            |
|                                   | customers + Read more ...         |
|                                   |                                   |
|                                   | The archived claim records for    |
|                                   | Restricted Access (RA) customers  |
|                                   | will not be available for search  |
|                                   | and view in Process Direct unless |
|                                   | the user accessing the archive    |
|                                   | for RA customers has one of the   |
|                                   | below roles:                      |
|                                   |                                   |
|                                   | -   ZC.XX.REST.ACC.ADMIN          |
|                                   | -   ZC.XX.REST.ACC.MGR            |
|                                   | -   ZC.XX.REST.ACC.OFF.L1         |
|                                   |                                   |
|                                   | When there is a need to retrieve  |
|                                   | an archived claim for a RA        |
|                                   | customer and the required         |
|                                   | resource is not held, contact the |
|                                   | National Restricted Access line   |
|                                   | on 1800 413 955 or                |
|                                   | NATIONAL.RESTRICTED.ACCESS        |
|                                   | mailbox.                          |
