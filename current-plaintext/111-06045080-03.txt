Review actions

Review actions in Customer First SAP native

This table explains how to locate reviews, change the review date, and
complete reviews for payments that use Customer First SAP native
processes.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Select review + Read more ...     |
|                                   |                                   |
|                                   | In the customer's record:         |
|                                   |                                   |
|                                   | -   select the CRN hyperlink to   |
|                                   |     go to the Customer Details    |
|                                   |     screen                        |
|                                   | -   select the Cases tab          |
|                                   | -   select the Benefit case       |
|                                   |     hyperlink                     |
|                                   | -   select the Related cases tab  |
|                                   | -   select the review             |
+-----------------------------------+-----------------------------------+
| 2                                 | Change date for ongoing review +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | To change the date:               |
|                                   |                                   |
|                                   | -   Select the review as above    |
|                                   | -   Select Edit                   |
|                                   | -   Click on the Calendar to      |
|                                   |     select a new date             |
|                                   | -   Tick the All Day event and    |
|                                   |     Save. This will automatically |
|                                   |     adjust consequential actions  |
|                                   |     such as letters or grace      |
|                                   |     period date                   |
|                                   | -   Select Back                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete review + Read more ...   |
|                                   |                                   |
|                                   | See the relevant procedure for    |
|                                   | payment specific information      |
|                                   | about reviews, including          |
|                                   | selection criteria, frequency,    |
|                                   | the assessment process and how to |
|                                   | conduct the review.               |
|                                   |                                   |
|                                   | Once the review has been          |
|                                   | completed:                        |
|                                   |                                   |
|                                   | -   select the review as above    |
|                                   | -   select Edit                   |
|                                   | -   select Complete from the      |
|                                   |     Review Status drop down menu  |
|                                   | -   select Save                   |
|                                   |                                   |
|                                   | The review status should now show |
|                                   | as Complete.                      |
+-----------------------------------+-----------------------------------+
