| 2                                 | Advantages of Centrelink          |
|                                   | deductions + Read more ...        |
|                                   |                                   |
|                                   | There are a number of advantages  |
|                                   | of Centrelink deductions:         |
|                                   |                                   |
|                                   | Payment of deductions is prompt - |
|                                   | they are sent on a daily basis    |
|                                   |                                   |
|                                   | Automatic suspension period:      |
|                                   |                                   |
|                                   | -   If the customer no longer     |
|                                   |     receives a pension or benefit |
|                                   |     the deduction is active but   |
|                                   |     suspended for 2 weeks         |
|                                   | -   If the pension or benefit     |
|                                   |     resumes in this period the    |
|                                   |     deduction resumes             |
|                                   | -   If the pension or benefit     |
|                                   |     does not resume, the          |
|                                   |     deduction request will stop   |
|                                   |     automatically                 |
|                                   |                                   |
|                                   | If the customer moves from a      |
|                                   | pension or benefit to another,    |
|                                   | the deduction automatically       |
|                                   | transfers to the new pension or   |
|                                   | benefit                           |
|                                   |                                   |
|                                   | The deduction stops automatically |
|                                   | once the child support            |
|                                   | (maintenance) debt is paid        |

| 3                                 | Limit to deduction amount + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Section 20 of the Child Support   |
|                                   | (Registration and Collection)     |
|                                   | Regulations 2018 sets the amount  |
|                                   | that can be deducted from ISPs    |
|                                   | per week. See the References page |
|                                   | for a link to the Regulations.    |
|                                   |                                   |
|                                   | If the Child Support Period       |
|                                   | commenced in 2019, this amount is |
|                                   | 3 times the minimum weekly rate   |
|                                   | of $8.34 per week, or $25.01 per  |
|                                   | week. This amount is increased    |
|                                   | periodically to take inflation    |
|                                   | into account.                     |
|                                   |                                   |
|                                   | If the Child Support Period       |
|                                   | commenced in an earlier year, see |
|                                   | the Basic Values for that year to |
|                                   | determine the deduction amount.   |
|                                   |                                   |
|                                   | Customers receiving ISP who have  |
|                                   | overdue child support will        |
|                                   | automatically have 3 times the    |
|                                   | minimum weekly rate of child      |
|                                   | support deducted.                 |
|                                   |                                   |
|                                   | Note: PLP is not an ISP and is    |
|                                   | not subject to Section 20. For    |
|                                   | deduction amounts, go to Step 6.  |
+-----------------------------------+-----------------------------------+

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers on an ISP who have      |
|                                   | overdue child support that are    |
|                                   | having difficulty meeting other   |
|                                   | life costs can seek a deduction   |
|                                   | concession, as long as they are   |
|                                   | not assessed to pay at least 3    |
|                                   | times the minimum annual rate.    |
|                                   |                                   |
|                                   | Income Support Payment (ISP)      |
|                                   | concession for Centrelink         |
|                                   | deductions outlines how to assess |
|                                   | and action applications from a    |
|                                   | customer to have the amount of    |
|                                   | deduction reduced.                |
|                                   |                                   |
|                                   | Note: if a customer is receiving  |
|                                   | PLP and wants to adjust the       |
|                                   | amount of PLP deductions being    |

|                                   | not required to seek an ISP       |
|                                   | Concession. If their ongoing      |
|                                   | liability is less than the amount |
|                                   | Child Support can collect from    |
|                                   | Centrelink, they can negotiate a  |
|                                   | payment arrangement.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Parental Leave Payment (PLP)      |
|                                   | deductions + Read more ...        |
|                                   |                                   |
|                                   | Deductions can be requested from  |
|                                   | customers who receive PLP from    |
|                                   | Centrelink. The 2019 maximum      |
|                                   | gross amount of PLP a customer    |
|                                   | can receive is $740.60 a week,    |
|                                   | equivalent to the weekly national |
|                                   | minimum wage. This amount will be |
|                                   | indexed at the start of each      |
|                                   | financial year. The maximum       |
|                                   | deduction is the PLP maximum      |
|                                   | gross amount minus the Protected  |
|                                   | Earnings Amount (PEA).            |
|                                   |                                   |
|                                   | Centrelink will not make any      |
|                                   | deductions if the customer’s PLP  |
|                                   | instalment is less than the PEA.  |
|                                   | If the customer is already        |
|                                   | receiving an ISP when they        |
|                                   | commence receiving PLP,           |
|                                   | Centrelink deductions can be      |
|                                   | requested from both entitlements. |
|                                   |                                   |
|                                   | If the customer is receiving PLP  |
|                                   | from their employer, see Employer |
|                                   | Withholdings (linking) for Child  |
|                                   | Support customers.                |
|                                   |                                   |
|                                   | Note: Centrelink may make other   |
|                                   | deductions, for example tax from  |
|                                   | PLP which could affect the        |
|                                   | deduction amount.                 |

Automatic Centrelink deductions

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Conditions when automatic         |
|                                   | deductions will commence + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Cuba and Centrelink's computer    |
|                                   | systems exchange information to   |
|                                   | establish which customers will    |
|                                   | automatically have deductions     |
|                                   | made from their Centrelink        |
|                                   | benefit.                          |
|                                   |                                   |
|                                   | Automatic deductions will         |
|                                   | commence for all customers who    |
|                                   | meet at least 1 of the following  |
|                                   | conditions:                       |
|                                   |                                   |
|                                   | -   the case has an ongoing       |
|                                   |     liability                     |
|                                   | -   the customer has a child      |
|                                   |     support debt of at least $25  |
|                                   | -   the customer has no debt      |
|                                   |     offset in any case            |
|                                   | -   the customer owes an          |
|                                   |     overpayment of at least $25   |
|                                   | -   the customer is in receipt of |
|                                   |     certain Centrelink benefits   |
|                                   |     or Parental Leave Payment     |
|                                   |     (PLP) (Cuba will seek to      |
|                                   |     collect 100% of any PLP       |
|                                   |     entitlement)                  |
|                                   | -   the customer does/does not    |
|                                   |     have a Special Recovery       |
|                                   |     Condition of hardship, stay   |
|                                   |     order, or customer agreement, |
|                                   |     (a special recovery condition |
|                                   |     will only prevent             |
|                                   |     disbursement, not collection) |
|                                   |     and the case has an ongoing   |
|                                   |     liability or arrears          |
|                                   |     registered for collection     |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   For Child Support liabilities |
|                                   |     and arrears, automatic        |
|                                   |     Centrelink deductions will    |
|                                   |     commence 7 days after the     |
|                                   |     amount becomes due            |
|                                   | -   For overpayments, automatic   |
|                                   |     Centrelink deductions will    |
|                                   |     commence 21 days after the    |
|                                   |     overpayment is created        |
|                                   |                                   |
|                                   | Once a customer is determined to  |
|                                   | meet the requirements a batch     |
|                                   | process sends a deduction request |
|                                   | to Centrelink to commence         |
|                                   | deductions                        |
|                                   |                                   |
|                                   | If the customer is receiving PLP, |
|                                   | the maximum deduction will be     |
|                                   | automatically requested if:       |
|                                   |                                   |
|                                   | -   the ongoing liability is      |
|                                   |     equal to or higher than the   |
|                                   |     maximum deduction amount, or  |
|                                   | -   the ongoing liability is less |
|                                   |     than the maximum deduction    |
|                                   |     amount, but the customer has  |
|                                   |     arrears which makes their     |
|                                   |     account balance more than the |
|                                   |     maximum deduction amount      |
|                                   |                                   |
|                                   | The PLP deduction amount can be   |
|                                   | manually adjusted. PLP is not an  |
|                                   | ISP and therefore an ISP          |
|                                   | concession is not applicable. If  |
|                                   | the customer wants to reduce      |
|                                   | their PLP deductions for their    |
|                                   | child support arrears, they must  |
|                                   | negotiate a payment arrangement.  |
|                                   | See Debt repayment.               |
|                                   |                                   |
|                                   | To amend Centrelink deductions,   |
|                                   | see Centrelink Deductions Cuba    |
|                                   | Process Help.                     |
|                                   |                                   |
|                                   | Note: in certain circumstances,   |
|                                   | payments from PLP and Centrelink  |
|                                   | benefits may be deducted          |
|                                   | simultaneously. See Centrelink    |
|                                   | Deductions Cuba Process Help.     |

| 2                                 | Preventing automatic deductions + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | It may be appropriate to prevent  |
|                                   | Cuba from making automatic        |
|                                   | deduction attempts because of the |

|                                   | example due to an agreement with  |

|                                   | reason for preventing automatic   |
|                                   | deductions                        |
|                                   |                                   |
|                                   | -   If automatic deductions       |
|                                   |     should never be established   |
|                                   |     for the customer, key a       |
|                                   |     finalisation date of          |
|                                   |     31/12/4000                    |
|                                   | -   See TRIP Cuba Process Help -  |
|                                   |     Record a special recovery     |
|                                   |     condition.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Ending automatic deductions +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Centrelink Deductions will        |
|                                   | automatically stop when:          |
|                                   |                                   |
|                                   | -   the child support liability   |
|                                   |     and/or arrears has been paid  |
|                                   | -   the customer has stopped      |
|                                   |     receiving a Centrelink        |
|                                   |     pension, benefit or PLP       |
|                                   |                                   |
|                                   | This will generate a Centrelink   |
|                                   | Deduction Terminated intray.      |
|                                   |                                   |
|                                   | Where the debt has been paid and  |
|                                   | the customer has other debts,     |
|                                   | such as penalties or consolidated |
|                                   | revenue debt (which cannot be     |
|                                   | collected via Centrelink          |
|                                   | deductions), contact the customer |
|                                   | to negotiate payment of the       |
|                                   | remaining debt.                   |
|                                   |                                   |
|                                   | Where the customer has stopped    |
|                                   | receiving a pension or benefit    |
|                                   | there may be alternative          |
|                                   | collection avenues available.     |
|                                   | This may be an indication that    |
|                                   | the customer is employed.         |
|                                   |                                   |
|                                   | Where the customer is deceased    |
|                                   | the refer the case to the         |
|                                   | Deceased Customer Management      |
|                                   | (DCM) team. See Terminating       |
|                                   | Events (Stage 2).                 |
|                                   |                                   |
|                                   | To resolve problems with          |
|                                   | automatic Centrelink deductions,  |
|                                   | see Table 4 or Table 5.           |

| Step                              | Action                            |

| 1                                 | Establishing manual deductions +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Although the majority of          |
|                                   | Centrelink deductions are         |
|                                   | automated, it will sometimes be   |
|                                   | necessary to establish manual     |
|                                   | Centrelink or Parental Leave      |
|                                   | Payment (PLP) deductions.         |
|                                   |                                   |
|                                   | Manual deductions are             |
|                                   | appropriate:                      |
|                                   |                                   |
|                                   | -   Where a liability is raised,  |
|                                   |     not yet due and the customer  |
|                                   |     requests that Centrelink or   |
|                                   |     PLP deductions commence       |
|                                   | -   Where automatic deductions    |
|                                   |     have been prevented and a     |
|                                   |     decision is made to commence  |
|                                   |     Centrelink or PLP deductions  |
|                                   | -   When the customer is a payee  |
|                                   |     with an overpayment and       |
|                                   |     requests Centrelink           |
|                                   |     deductions be established     |
|                                   |     immediately. Centrelink       |
|                                   |     deductions for overpayments   |
|                                   |     will normally commence 21     |
|                                   |     days after the overpayment is |
|                                   |     raised                        |
|                                   | -   When the customer is a payee  |
|                                   |     with an overpayment and       |
|                                   |     automatic deductions did not  |
|                                   |     commence due to a debt offset |
|                                   |     in 1 of their cases. Note:    |
|                                   |     manual deductions can still   |
|                                   |     be set up in this scenario    |
|                                   |                                   |
|                                   | The conditions outlined in About  |
|                                   | Centrelink deductions must be met |
|                                   | in order to establish Centrelink  |
|                                   | or PLP deductions.                |
|                                   |                                   |
|                                   | See the Client Benefits window on |
|                                   | how to check which benefit or     |
|                                   | pension (if any) the customer     |
|                                   | receives.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Review the case + Read more ...   |
|                                   |                                   |
|                                   | Review the case to ensure that    |
|                                   | Centrelink or PLP deductions are  |
|                                   | appropriate:                      |
|                                   |                                   |
|                                   | -   The debt must be correct (or  |
|                                   |     corrected if possible)        |
|                                   | -   There must be no outstanding  |
|                                   |     intrays                       |
|                                   | -   Contact the Debt Enforcement  |
|                                   |     Service Officer if            |
|                                   |     enforcement litigation is     |
|                                   |     occurring, to discuss the     |
|                                   |     proposed deduction            |
+-----------------------------------+-----------------------------------+
| 3                                 | Action any outstanding intrays +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check both customers' records for |
|                                   | any outstanding intrays           |
|                                   | indicating pending actions that   |
|                                   | may change the liability, such as |
|                                   | a change in care. Ensure that     |
|                                   | these issues are finalised before |
|                                   | proceeding.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Correct the debt + Read more ...  |
|                                   |                                   |
|                                   | Check that the debt is correct.   |
|                                   |                                   |
|                                   | If there are outstanding NAP      |
|                                   | intrays, the NAPs must be         |
|                                   | accepted or refused before        |
|                                   | commencing deductions. See Non    |
|                                   | Agency Payments (NAPs)            |
|                                   |                                   |
|                                   | If the liability is set by a      |
|                                   | child support assessment, ensure  |
|                                   | that the incomes being used are   |
|                                   | accurate. See Income Processing   |
|                                   | for Child Support customers       |
|                                   |                                   |
|                                   | If the liability is set by a      |
|                                   | child support agreement, check    |
|                                   | the agreement to for unemployment |
|                                   | clauses, or other clauses that    |
|                                   | may affect the liability. See     |
|                                   | Agreements between Child Support  |
|                                   | customers                         |
|                                   |                                   |
|                                   | If the liability is set by a      |
|                                   | court order that overrides a      |
|                                   | child support assessment, check   |
|                                   | for unemployment clauses, or      |
|                                   | other clauses that may affect the |
|                                   | liability. See Court Varied       |
|                                   | Assessments (CVA) including lump  |
|                                   | sum orders                        |
|                                   |                                   |
|                                   | -   If there is an unemployment   |
|                                   |     clause in the order that has  |
|                                   |     not been actioned, see Stage  |
|                                   |     1-variations to activate the  |
|                                   |     clause                        |
|                                   | -   If there is no unemployment   |
|                                   |     clause in the order, the      |
|                                   |     customer may be able to apply |
|                                   |     for a Low Income Non          |
|                                   |     Enforcement Period (LINE).    |
|                                   |     Contact the customer if a     |
|                                   |     LINE is not in place. See     |
|                                   |     Stage 1- Low Income           |
|                                   |     non-enforcement periods       |
+-----------------------------------+-----------------------------------+
| 5                                 | Payment arrangements + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check if the customer has a       |
|                                   | payment arrangement on the        |
|                                   | Customer window and in the        |
|                                   | Payment Arrangement window. See   |
|                                   | Payment arrangements Cuba Process |
|                                   | Help on how to access these       |
|                                   | screens.                          |

|                                   | Where there is a payment          |
|                                   | arrangement in place, only        |
|                                   | request Centrelink deductions     |
|                                   | following negotiation with the    |
|                                   | customer. See Debt repayment.     |

| 8                                 | Starting manual deductions + Read |

| 1                                 | Statements + Read more ...        |
|                                   |                                   |
|                                   | Generally payers are sent a       |

|                                   | If a customer requests that       |
|                                   | statements be issued, this can be |
|                                   | overridden by keying the          |
|                                   | statement function and changing   |
|                                   | the frequency from Suppress to    |
|                                   | Always. Cuba will then continue   |
|                                   | to send statements while          |
|                                   | Centrelink deductions are being   |
|                                   | made.                             |
|                                   |                                   |
|                                   | Note: customers with an           |
|                                   | overpayment are sent a monthly    |
|                                   | overpayment summary.              |
