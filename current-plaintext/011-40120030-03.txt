Complete a warm transfer

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Advise caller of warm transfer +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Tell the caller you are not the   |
|                                   | right business area but will help |
|                                   | them find the right person or     |
|                                   | business area (not a voicemail).  |
+-----------------------------------+-----------------------------------+
| 2                                 | Initiate transfer + Read more ... |
|                                   |                                   |
|                                   | Call the right person or business |
|                                   | area.                             |
|                                   |                                   |
|                                   | Is contact made?                  |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, return to the caller and      |
|                                   | explain that the:                 |
|                                   |                                   |
|                                   | -   right area cannot be found,   |
|                                   |     or                            |
|                                   | -   transfer number is busy or    |
|                                   |     diverted to voicemail. If the |
|                                   |     caller is happy to go to      |
|                                   |     voicemail, transfer them. If  |
|                                   |     not, offer to take a message. |
|                                   |     Ensure the enquiry is         |
|                                   |     followed up                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete transfer + Read more ... |
|                                   |                                   |
|                                   | -   Summarise the caller's        |
|                                   |     enquiry                       |
|                                   | -   Advise that security checks   |
|                                   |     are complete                  |
|                                   | -   Transfer the call             |
+-----------------------------------+-----------------------------------+
