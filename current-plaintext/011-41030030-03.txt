This page has the process for viewing the current entitlement period,
answering enquiries from a child/teenager (or their parent/guardian)
when they have not received a Child Dental Benefits Schedule (CDBS)
notification and how to set and remove a CDBS Return Mail Flag
indicator.

View relevant current entitlement period

Table 1: This table describes the process to view the current
entitlement period in CDBS.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access CDMS + Read more ...       |
|                                   |                                   |
|                                   | Log on to Consumer Directory      |
|                                   | Maintenance System (CDMS).        |
+-----------------------------------+-----------------------------------+
| 2                                 | Inquire on Consumer Details +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Inquire on Consumer        |
|                                   | Details.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete details + Read more ...  |
|                                   |                                   |
|                                   | Complete the following fields:    |
|                                   |                                   |
|                                   | Program: Select Child Dental      |
|                                   | Benefits Schedule                 |
|                                   |                                   |
|                                   | ID Number: Key either the parents |
|                                   | or teenagers Consumer ID, CRN,    |
|                                   | DVA ID or DVA UIN                 |
|                                   |                                   |
|                                   | ID Number Type: Select either     |
|                                   | Consumer ID, CRN, DVA ID or DVA   |
|                                   | UIN from the drop down menu.      |
|                                   | Note: needs to relate to the      |
|                                   | number entered above)             |
|                                   |                                   |
|                                   | -   Select Entitlement            |
|                                   |                                   |
|                                   | The ‘Eligible person record’      |
|                                   | displays.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Searching using the               |
|                                   | Guardian/Parent details + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the radio button next to   |
|                                   | the Child/Teenager you are        |
|                                   | inquiring on.                     |
|                                   |                                   |
|                                   | Select the Eligible Person tab.   |
+-----------------------------------+-----------------------------------+
| 5                                 | Searching using the               |
|                                   | Child/Teenager’s details + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the Eligible Person tab.   |
+-----------------------------------+-----------------------------------+
| 6                                 | Entitlement details + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Entitlement details will      |
|                                   | display showing:                  |
|                                   |                                   |
|                                   | -   Currently Eligible            |
|                                   | -   Date of Inquiry               |
|                                   | -   Balance of Inquiry            |
|                                   | -   Current Entitlement Period    |
|                                   | -   CAP Amount                    |
|                                   | -   Entitlement Year (this is not |
|                                   |     the 2 year period of          |
|                                   |     eligibility)                  |
|                                   | -   Benefit Code                  |
+-----------------------------------+-----------------------------------+

Managing enquiries when CDBS notification not received

Table 2: This table describes how to answer enquiries from a
child/teenager (or their parent/guardian) when they have not received a
CDBS notification.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Get details and complete a        |
|                                   | security check + Read more ...    |
|                                   |                                   |
|                                   | Ask for the child/teenager's      |
|                                   | details and conduct a security    |
|                                   | check, see Authenticating a       |
|                                   | Medicare customer.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirm eligibility + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Confirm the child/teenager's      |
|                                   | eligibility in Consumer Directory |
|                                   | Maintenance System (CDMS), see:   |
|                                   |                                   |
|                                   | -   Amend CDMS tab fields for the |
|                                   |     Child Dental Benefits         |
|                                   |     Schedule (CDBS) in Medicare   |
|                                   | -   Inquire CDMS tab fields for   |
|                                   |     the Child Dental Benefits     |
|                                   |     Schedule (CDBS) in Medicare   |
|                                   |                                   |
|                                   | If CDBS eligibility details are   |
|                                   | not found in CDMS, please contact |
|                                   | CDBS Help desk and attach the     |
|                                   | CDBS Enquiry form. The Help desk  |
|                                   | will investigate the enquiry and  |
|                                   | advise the customer of the        |
|                                   | outcome.                          |
|                                   |                                   |
|                                   | The Resources page has a link to  |
|                                   | the CDBS Enquiry form.            |
+-----------------------------------+-----------------------------------+

Setting a CDBS Return Mail Flag indicator

Table 3: This table describes how to set a CDBS and Return Mail Flag
indicator in CDMS production.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access CDMS + Read more ...       |
|                                   |                                   |
|                                   | Log on to Consumer Directory      |
|                                   | Maintenance System (CDMS).        |
+-----------------------------------+-----------------------------------+
| 2                                 | Amend Consumer Details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select Amend Consumer Details.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete details + Read more ...  |
|                                   |                                   |
|                                   | Complete the following fields:    |
|                                   |                                   |
|                                   | -   Program: Select Child Dental  |
|                                   |     Benefits Schedule             |
|                                   | -   ID Number: Key either the     |
|                                   |     parents or teenagers Consumer |
|                                   |     ID, CRN, DVA ID or DVA UIN    |
|                                   | -   ID Number Type: Select either |
|                                   |     Consumer ID, CRN, DVA ID or   |
|                                   |     DVA UIN from the drop down    |
|                                   |     menu. Note: needs to relate   |
|                                   |     to the number entered above)  |
|                                   | -   Select Entitlement            |
|                                   |                                   |
|                                   | The ‘Eligible person record’ is   |
|                                   | displayed.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | If the notification was sent to   |
|                                   | the Child/Teenager + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Was the notification sent to the  |
|                                   | Child/Teenager? (Who is receiving |
|                                   | the government payments)          |
|                                   |                                   |
|                                   | -   Yes, go to Eligible Person    |
|                                   |     tab                           |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | If the notification was sent to   |
|                                   | the Guardian/Parent (who is       |
|                                   | receiving the government          |
|                                   | payments) + Read more ...         |
|                                   |                                   |
|                                   | Go to Guardian/Parents tab.       |
|                                   |                                   |
|                                   | Note: if it is an Individual then |
|                                   | the Guardian/Parents tab is not   |
|                                   | populated and the Return Mail     |
|                                   | Flag is updated on the Eligible   |
|                                   | Person tab only with the same     |
|                                   | process.                          |
+-----------------------------------+-----------------------------------+
| 6                                 | Set Return Mail Flag indicator +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |
|                                   | -   Return Mail Flag indicator    |
|                                   | -   Validate                      |
+-----------------------------------+-----------------------------------+
| 7                                 | Finalise process + Read more ...  |
|                                   |                                   |
|                                   | Select Update.                    |
|                                   |                                   |
|                                   | The consumer record has been      |
|                                   | successfully updated in the       |
|                                   | Consumer Directory with the       |
|                                   | amended details.                  |
+-----------------------------------+-----------------------------------+
