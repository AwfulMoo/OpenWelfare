Calculating the start day for a claim made after a partner's claim

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check timeframe of claim          |
|                                   | lodgement + Read more ...         |
|                                   |                                   |
|                                   | Has the customer made the claim   |
|                                   | within 14 days after their        |
|                                   | partner's claim?                  |
|                                   | Note: day 1 of the 14 day period  |
|                                   | commences the day after partner’s |
|                                   | claim was lodged.                 |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the start date for the    |
|                                   |     customer is the first day     |
|                                   |     they become qualified for     |
|                                   |     payment. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 2                                 | Will the customer's partner’s     |
|                                   | claim be granted? + Read more ... |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the start date for the    |
|                                   |     customer is the first day     |
|                                   |     they become qualified for     |
|                                   |     payment. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 3                                 | Determine start day for both      |
|                                   | customer and partner + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Determine the start day for both  |
|                                   | the customer's and their          |
|                                   | partner's claims. See Start day   |
|                                   | (CLK) for definition.             |
|                                   |                                   |
|                                   | Note: start dates may be affected |
|                                   | where the customer and/or partner |
|                                   | are subject to RapidConnect       |
|                                   | requirements. See Calculating the |
|                                   | start day general rule for        |
|                                   | further information.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Compare start day of both         |
|                                   | customer and partner + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the partner's start day        |
|                                   | earlier than the customer's start |
|                                   | day?                              |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, the start date for the    |
|                                   |     customer is the first day     |
|                                   |     they become qualified for     |
|                                   |     payment. Procedure ends here  |
+-----------------------------------+-----------------------------------+
| 5                                 | The start day + Read more ...     |
|                                   |                                   |
|                                   | The customer's start day is the   |
|                                   | first day they are qualified for  |
|                                   | the payment on or after their     |
|                                   | partner's start day.              |
+-----------------------------------+-----------------------------------+
