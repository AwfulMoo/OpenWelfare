Lodging Medicare Two-way claims

Table 1

| Item                              | Description                       |
+-----------------------------------+-----------------------------------+

|                                   | to claim in person + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The claimant must provide service |
|                                   | centre staff with:                |
|                                   |                                   |
|                                   | -   a completed and signed        |
|                                   |     Medicare Two-way claim form   |

|                                   |     receipts when paid            |

|                                   | form, ask for the missing         |

|                                   | to resubmit when they have all    |
|                                   | the required information.         |
|                                   |                                   |
|                                   | Note: when the claimant provides  |
|                                   | a Medicare Two-way claim form at  |
|                                   | a service centre, it must be      |
|                                   | processed as a Two-way claim. See |
|                                   | Table 2.                          |
+-----------------------------------+-----------------------------------+

|                                   | claim at a service centre, or by  |

|                                   |                                   |
|                                   | The claimant must provide service |
|                                   | centre staff with:                |

+-----------------------------------+-----------------------------------+
| 3                                 | Claimant lodges their Medicare    |
|                                   | Two-way claim by Medicare online  |

|                                   |                                   |
|                                   | The claimant must have uploaded:  |
|                                   |                                   |
|                                   | -   completed and signed Medicare |
|                                   |     Two-way claim form (MS001) or |

|                                   |     website), and                 |
|                                   | -   photos of original accounts   |
|                                   |     and receipts when paid        |
|                                   |                                   |
|                                   | See also: Process digital self-   |
|                                   | service claims                    |
|                                   |                                   |
|                                   | Note: where an account is over    |
|                                   | $9,999.99 a completed and signed  |
|                                   | Medicare Claim form (MS014) must  |
|                                   | also be uploaded.                 |

|                                   | Two-way claim with their private  |

Process a Medicare Two-way claim

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Process the Medicare claim as     |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | Reprocess a claim                 |
|                                   |                                   |
|                                   | If a Medicare Two-way claim is to |
|                                   | be reprocessed, select the        |
|                                   | services for transmission via the |
|                                   | PHEB screen after reprocessing    |
|                                   | the Medicare component.           |
|                                   |                                   |
|                                   | Where more than one item line is  |
|                                   | included in the Medicare          |
|                                   | component of a Medicare Two-way   |
|                                   | claim, delete and reprocess the   |
|                                   | entire claim. This avoids         |
|                                   | problems when the benefit is      |
|                                   | subject to multiple service       |
|                                   | rules.                            |
|                                   |                                   |
|                                   | For information on how to process |
|                                   | a claim, see:                     |
|                                   |                                   |
|                                   | -   Claims processing in Medicare |
|                                   | -   Process digital self service  |
|                                   |     claims                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Check if the private health       |
|                                   | insurer participates in Medicare  |
|                                   | Two-way + Read more ...           |
|                                   |                                   |
|                                   | Check the list of private health  |
|                                   | insurers in Medicare.             |
|                                   | Participants in Medicare Two-way  |
|                                   | have a Two-way Fund Brand ID      |
|                                   | listed in this contact list.      |
|                                   |                                   |
|                                   | Do they participate?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the claim cannot be       |
|                                   |     submitted electronically. See |
|                                   |     Step 3 in Table 5             |
+-----------------------------------+-----------------------------------+
| 3                                 | Access PHEB through Mainframe +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Key control line PHEB         |
|                                   | -   Press [Enter]                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Enter claim form fields + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Key information from the Medicare |
|                                   | Two-way claim form (MS001) or     |
|                                   | private health insurer’s claim    |
|                                   | form into the appropriate fields: |
|                                   |                                   |
|                                   | FUND BRAND ID – see the List of   |
|                                   | private health insurers in        |
|                                   | Medicare on the Resources page    |
|                                   |                                   |
|                                   | MEMBERSHIP NUMBER – if membership |
|                                   | number is incorrect or not        |
|                                   | provided, see Table 3             |
|                                   |                                   |
|                                   | CARD NUMBER and PATIENT (first    |
|                                   | name)                             |
|                                   |                                   |
|                                   | COMPENSATION CLAIM – key 'Y'      |
|                                   | (yes), 'N' (no), or ‘?’ (if       |
|                                   | unknown)                          |
|                                   |                                   |
|                                   | HOSP FROM DATE – (date of         |
|                                   | admission) and TO – (date of      |
|                                   | discharge)                        |
|                                   |                                   |
|                                   | TODAY CLAIM. If the claims were:  |
|                                   |                                   |
|                                   | -   paid today, leave 'Y'         |
|                                   | -   not paid today, key 'N'       |
|                                   |                                   |
|                                   | HOSPITAL NAME                     |
|                                   |                                   |
|                                   | Press [Enter]                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Populate the PHEB screen + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | PHEB screen is populated from the |
|                                   | patient’s claims history with:    |
|                                   |                                   |
|                                   | -   TRANS TYPE (transmission      |
|                                   |     type) – EDI or ECL (ECLIPSE)) |
|                                   | -   DOL (date of lodgement –      |
|                                   |     populates with date of        |
|                                   |     processing)                   |
|                                   | -   TRADING NAME (private health  |
|                                   |     insurer name)                 |
|                                   | -   DOB (date of birth)           |
|                                   | -   CLAIMANT'S FAMILY NAME        |
|                                   | -   FIRST NAME (claimant's fist   |
|                                   |     name)                         |
|                                   | -   AD (address)                  |
|                                   |                                   |
|                                   | Note: the PAYEE TYPE field        |
|                                   | displays either P (Account has    |
|                                   | been paid in favour of claimant)  |
|                                   | or U (unpaid account. The payment |
|                                   | is made in favour of the          |
|                                   | provider).                        |
+-----------------------------------+-----------------------------------+
| 6                                 | Permanent address field + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The PERMANENT ADDRESS field is    |
|                                   | now unprotected.                  |
|                                   |                                   |
|                                   | Key either:                       |
|                                   |                                   |
|                                   | -   'P' (permanent) enter new     |
|                                   |     permanent address if required |
|                                   | -   'T' (temporary) enter new     |
|                                   |     temporary address if required |
+-----------------------------------+-----------------------------------+
| 7                                 | Selection field + Read more ...   |
|                                   |                                   |
|                                   | -   Key 'S' in the selection      |
|                                   |     field to select the items     |
|                                   |     lines to be transmitted.      |
|                                   | -   Press [F3] to update          |
|                                   |                                   |
|                                   | The screen displays showing the   |
|                                   | message I-2 WAY OK                |
|                                   |                                   |
|                                   | The item lines and other patient  |
|                                   | and claimant identification       |
|                                   | details are automatically         |
|                                   | transmitted to the private health |
|                                   | insurer after the close of        |
|                                   | business.                         |
|                                   |                                   |
|                                   | Note: miscellaneous items (e.g.   |
|                                   | facility fees - non MBS items)    |
|                                   | cannot be transmitted.            |
+-----------------------------------+-----------------------------------+
| 8                                 | Record successful Two-way         |
|                                   | transmission + Read more ...      |
|                                   |                                   |
|                                   | Service centre – Service Officers |
|                                   | must notate on the Medicare       |
|                                   | Two-way claim form (MS001)        |
|                                   | 'successfully transmitted'.       |
|                                   |                                   |
|                                   | PaNDA claims – Service Officers   |
|                                   | must add processing notes 'Two    |
|                                   | way information successfully      |
|                                   | transmitted'.                     |
|                                   |                                   |
|                                   | Digital claims (Medicare online   |
|                                   | account/App claims) – Service     |
|                                   | Officers must leave comments in   |
|                                   | CDMS, stating 'successful Two way |
|                                   | transmitted electronically for    |
|                                   | MOA/App claim DOL DDMMYY'.        |
+-----------------------------------+-----------------------------------+
| 9                                 | Clear screen + Read more ...      |
|                                   |                                   |
|                                   | Press [Pause–Break] to clear the  |
|                                   | screen.                           |
+-----------------------------------+-----------------------------------+

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Contact claimant to obtain name   |

|                                   |                                   |

|                                   |                                   |
|                                   | return the Medicare Two-way claim |

|                                   | -   For PaNDA claims - upload a   |

|                                   | -   For digital claims (Medicare  |
|                                   |     online account/App claims) –  |
|                                   |     add notes in CDMS 'Details    |
|                                   |     missing health fund details   |
|                                   |     on Two-way claim form, Z1485  |

|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the Medicare letter       |
|                                   | template.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Private health insurer membership |
|                                   | number + Read more ...            |
|                                   |                                   |

|                                   | claim form, contact claimant to   |
|                                   | obtain the membership number.     |
|                                   |                                   |
|                                   | If the membership number cannot   |
|                                   | be identified and the fund is     |
|                                   | known:                            |

|                                   |                                   |

|                                   |                                   |
|                                   | -   Medicare Two-way claim form   |
|                                   |     (MS001)                       |
|                                   | -   Medicare Gap Statement of     |
|                                   |     Benefits                      |
|                                   |                                   |
|                                   | Forward to the private health     |
|                                   | insurer by external post          |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the forms and contact     |
|                                   | details in the List of private    |
|                                   | health insurers.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Permanent address indication +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the claimant has not indicated |
|                                   | whether the address on the claim  |
|                                   | form is permanent or temporary,   |
|                                   | key '?' in the PERMANENT ADDRESS  |
|                                   | field to move to the next tab.    |
+-----------------------------------+-----------------------------------+
| 5                                 | Compensation indication + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In the COMPENSATION CLAIM field,  |

|                                   | to move to the next tab.          |
+-----------------------------------+-----------------------------------+
| 6                                 | Hospital name + Read more ...     |
|                                   |                                   |
|                                   | If the claimant has not written   |
|                                   | the hospital name on the claim    |
|                                   | form, key '?' in the HOSP NAME    |
|                                   | field to move to the next tab.    |
+-----------------------------------+-----------------------------------+
| 7                                 | Hospital admission date + Read    |
|                                   | more ...                          |

Transmitting a duplicate service

Table 4

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Duplicate services can be         |
|                                   | transmitted + Read more ...       |
|                                   |                                   |
|                                   | if the same service lines exist   |
|                                   | in the Medicare Two-way database: |
|                                   |                                   |
|                                   | -   and the same service lines    |
|                                   |     with a different Fund Brand   |
|                                   |     ID exists on the PHEB screen, |
|                                   |     or                            |
|                                   | -   with the same Fund Brand ID   |
|                                   |     and the original service      |
|                                   |     lines have a rejected or      |
|                                   |     transmission failed status    |
|                                   |                                   |
|                                   | if this line and the associated   |
|                                   | SELECTION field is displayed      |
|                                   | yellow with the message W-DUP     |
|                                   | SELN                              |
+-----------------------------------+-----------------------------------+
| 2                                 | Duplicate services cannot be      |
|                                   | transmitted + Read more ...       |
|                                   |                                   |
|                                   | if the same service lines exists  |
|                                   | in the Medicare Two-way database, |
|                                   | and:                              |
|                                   |                                   |
|                                   | -   the same service lines with   |
|                                   |     the same Fund Brand ID exists |
|                                   |     on the PHEB screen, and       |
|                                   | -   the lines display as          |
|                                   |     protected blue with no        |
|                                   |     associated SELECTION field    |

Ancillary claims and Two-way claims that cannot be submitted electronically

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Does the private health insurer   |
|                                   | participate in Medicare           |
|                                   | Two-way? + Read more ...          |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | list of private health insurers   |
|                                   | in Medicare. Participants in      |
|                                   | Medicare Two-way have a Two-way   |
|                                   | Fund Brand ID on this list.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Claim cannot be submitted         |
|                                   | electronically + Read more ...    |
|                                   |                                   |
|                                   | If the claim is an ancillary      |
|                                   | claim or claim that cannot be     |
|                                   | transmitted electronically via    |
|                                   | the PHEB screen, complete and     |
|                                   | print the Unable to transmit      |
|                                   | Medicare Two-way claim (HW038)    |
|                                   | form.                             |
|                                   |                                   |
|                                   | -   Complete all sections of the  |
|                                   |     form. Section 2 reference     |
|                                   |     number refers to the service  |
|                                   |     operator mainframe log on     |
|                                   |     this identification number,   |
|                                   |     and must be filled out        |
|                                   | -   Ensure the appropriate reason |
|                                   |     is ticked. If the box 'Other' |
|                                   |     is ticked, give a short       |
|                                   |     explanation of the reason.    |
|                                   |     E.g., change of address with  |
|                                   |     the private health insurer.   |
|                                   |     Do not use abbreviations such |
|                                   |     as 'E-NO SVC FND'             |
|                                   | -   Attach the Medicare Two-way   |
|                                   |     claim form (MS001) or         |
|                                   |     participating health fund     |
|                                   |     claim form with Medicare Gap  |
|                                   |     Benefit Statement, or if it   |
|                                   |     is an ancillary claim, attach |
|                                   |     the documents to the          |
|                                   |     completed HW038 form          |
|                                   | -   Forward to the participating  |
|                                   |     health fund by external post  |
|                                   | -   Go to Step 4                  |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the Unable to transmit    |
|                                   | Medicare Two-way claim form       |
|                                   | (HW038).                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete Medicare Two-way claim   |
|                                   | rejected or incomplete (Z1485)    |
|                                   | letter + Read more ...            |
|                                   |                                   |
|                                   | Advise the claimant their claim   |
|                                   | cannot be forwarded to the        |
|                                   | private health insurer:           |
|                                   |                                   |
|                                   | Use the Medicare Two way claim    |
|                                   | rejected or incomplete (Z1485)    |
|                                   | letter                            |
|                                   |                                   |
|                                   | In Letter Content, select 'No     |
|                                   | two-way agreement with health     |
|                                   | insurance provider' to populate   |
|                                   | the text 'We cannot forward your  |
|                                   | claim information because we do   |
|                                   | not have a Two-way agreement with |
|                                   | your private health insurer'      |
|                                   |                                   |
|                                   | Print the letter                  |
|                                   |                                   |
|                                   | Attach the:                       |
|                                   |                                   |
|                                   | -   Medicare Two-way form or      |
|                                   |     Private Health fund form      |
|                                   | -   Medicare Gap Statement of     |
|                                   |     Benefit / Ancillary claim and |
|                                   |     return to the customer        |
|                                   |                                   |
|                                   | Go to step 4                      |
|                                   |                                   |
|                                   | Note: the Unable to transmit      |
|                                   | Two-way form (HW038) is not       |
|                                   | required. This form is an         |
|                                   | internal form used to send claim  |
|                                   | details to a participating health |
|                                   | fund only. This form is not sent  |
|                                   | to the customer.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Record Service Officer actions +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Service centre – no notation      |
|                                   | required by Service Officers. The |
|                                   | Service Officer must forward the  |
|                                   | claim to the private health fund  |
|                                   | by external post.                 |
|                                   |                                   |
|                                   | PaNDA claims – Service Officers   |
