AHS search query fields

Table 2 Note: ‘*’ denotes at least 1 of these fields is required.

+-----------------------------------+-----------------------------------+
| Field                             | Action                            |
+-----------------------------------+-----------------------------------+
| AHS registration number *         | Key the AHS registration number   |
|                                   | for an AHS centre or community    |
+-----------------------------------+-----------------------------------+
| Approved supplier number *        | Key the approved supplier number  |
+-----------------------------------+-----------------------------------+
| Date of supply                    | These fields can be used to enter |
|                                   | either a specific supply date (in |
|                                   | the first date field) or a date   |
|                                   | range                             |
+-----------------------------------+-----------------------------------+
| Claim reference number            | Key the Claim reference number if |
|                                   | available. Approved suppliers     |
|                                   | provide a claim reference number  |
|                                   | to Services Australia with each   |
|                                   | claim                             |
+-----------------------------------+-----------------------------------+
| AHS community name                | -   Select from the drop down     |
|                                   |     list the AHS community or     |
|                                   |     outreach station that is      |
|                                   |     affiliated with the           |
|                                   |     Aboriginal Health Service     |
|                                   |     centre holding the AHS        |
|                                   |     registration number           |
|                                   | -   If required, also select the  |
|                                   |     main AHS centre by choosing   |
|                                   |     the ‘Main AHS centre – no     |
|                                   |     community’ option. By doing   |
|                                   |     this the search is narrowed   |
|                                   |     to a particular community or  |
|                                   |     Main centre                   |
|                                   | -   The drop down list can remain |
|                                   |     on ‘Please Select’. This will |
|                                   |     ensure claims for all         |
|                                   |     communities/outreach stations |
|                                   |     which meet the search         |
|                                   |     criteria will display         |
+-----------------------------------+-----------------------------------+
| AHS claim number *                | Key the AHS claim number          |
+-----------------------------------+-----------------------------------+
| Status                            | Select from the drop down list of |
|                                   | possible claim status values. The |
|                                   | possible values are Registered,   |
|                                   | In Progress, Completed and        |
|                                   | Abandoned                         |
