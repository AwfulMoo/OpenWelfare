This page contains the process Compliance Officers (CO) must follow when
undertaking BIBI interventions within the Customer Compliance Division.

On this page:

Discuss match data with customer

Customer responds to Request to contact letter

| Step                              | Action                            |

| 1                                 | Initiation + Read more ...        |
|                                   |                                   |
|                                   | A request to contact letter is    |
|                                   | issued when a BIBI Intervention   |
|                                   | has been initiated.               |
|                                   |                                   |
|                                   | The letter requires the customer  |
|                                   | to contact to discuss information |
|                                   | received relating to their        |
|                                   | business income or bank interest. |
|                                   |                                   |
|                                   | The letter asks the customer to   |
|                                   | contact within 28 days.           |
|                                   |                                   |
|                                   | Customers may receive an SMS      |
|                                   | reminder 5 days after they        |
|                                   | receive the contact letter. If    |
|                                   | they do not contact within 14     |
|                                   | days, they will also be sent a    |
|                                   | reminder letter.                  |
|                                   |                                   |
|                                   | For current customers, the letter |
|                                   | will advise their payments may be |
|                                   | suspended if they do not contact. |

Discuss match data with customer

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Discuss match data + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | A match has occurred because the  |
|                                   | information on the customer's     |
|                                   | record differs from the           |
|                                   | information received from the     |
|                                   | Australian Taxation Office (ATO). |
|                                   |                                   |
|                                   | Check the business income or bank |
|                                   | interest details on the           |
|                                   | customer's record. Discuss the    |
|                                   | discrepancy with the customer.    |
|                                   | For help, refer to the relevant   |
|                                   | screen column in the Assessing    |
|                                   | and Coding Income and Assets      |
|                                   | attachment on the Resources page. |
|                                   |                                   |
|                                   | The scope start date:             |
|                                   |                                   |
|                                   | -   the start date of the income, |
|                                   |     or                            |
|                                   | -   if the start date is greater  |
|                                   |     than 7 years, 1 July 7 years  |
|                                   |     before the current financial  |
|                                   |     year. For example, for the    |
|                                   |     2020-21 financial year, the   |
|                                   |     scope start date is 1 July    |
|                                   |     2013                          |
|                                   |                                   |
|                                   | The scope end date:               |
|                                   |                                   |
|                                   | -   If the customer is still      |
|                                   |     receiving income from         |
|                                   |     business income or bank       |
|                                   |     interest, Date Paid To        |
|                                   | -   If the customer has ceased    |
|                                   |     receiving income, the         |
|                                   |     cessation date                |
|                                   |                                   |
|                                   | Use questioning skills to obtain  |
|                                   | as much information as possible   |
|                                   | from the customer over the phone. |
+-----------------------------------+-----------------------------------+
| 2                                 | Check anomaly or mismatch + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the customer indicates the     |
|                                   | information is incorrect,         |
|                                   | thoroughly review the customer    |
|                                   | record to check for an anomaly or |
|                                   | mismatch.                         |
|                                   |                                   |
|                                   | Say to the customer:              |
|                                   |                                   |
|                                   | 'Would you mind holding the line  |
|                                   | while I check your record.'       |
|                                   |                                   |
|                                   | Preliminary assessment            |
|                                   |                                   |
|                                   | Use all information available to  |
|                                   | decide if an anomaly or mismatch  |
|                                   | exists between the customer and   |
|                                   | the match data.                   |
|                                   |                                   |
|                                   | If there are at least 2           |
|                                   | commonalities (e.g. name and date |
|                                   | of birth matches, or name and     |
|                                   | address matches) between the      |
|                                   | customer and the match data, it   |
|                                   | the match data is not a mismatch  |
|                                   | and should be continued. Take     |
|                                   | extra care with twin or           |
|                                   | intertwined records.              |
|                                   |                                   |
|                                   | Alternative spelling or           |
|                                   | misspelling of a customer's name  |
|                                   | does not automatically mean a     |
|                                   | mismatch.                         |
|                                   |                                   |
|                                   | Can an anomaly or mismatch be     |
|                                   | identified from the customer      |
|                                   | record?                           |
|                                   |                                   |
|                                   | Yes, advise the customer:         |
|                                   |                                   |
|                                   | -   More investigation is needed. |
|                                   |     Someone will contact them to  |
|                                   |     advise the outcome and next   |
|                                   |     steps                         |
|                                   | -   If they cannot be contacted,  |
|                                   |     they will need to call back   |
|                                   |     to complete their             |
|                                   |     intervention within 14 days.  |
|                                   |     If they fail to contact       |
|                                   |     within 14 days, the match     |
|                                   |     data may be applied to their  |
|                                   |     record and this may result in |
|                                   |     a debt                        |
|                                   | -   Go to Step 3                  |
|                                   |                                   |
|                                   | No, continue the intervention:    |
|                                   |                                   |
|                                   | -   To assess bank interest, go   |
|                                   |     to Step 6                     |
|                                   | -   To assess business income, go |
|                                   |     to Step 4                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Anomaly or mismatch identified +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Cancel the BIBI Integrated Review |
|                                   | in Customer First to enable       |
|                                   | Manual IRS Processing:            |
|                                   |                                   |
|                                   | -   select Edit from the          |
|                                   |     Integrated Review (ZIRV)      |
|                                   |     screen in Customer First      |
|                                   | -   select Cancelled from the     |
|                                   |     Activity Status drop down     |
|                                   |     menu                          |
|                                   | -   select the Manual IRS         |
|                                   |     Processing Required from the  |
|                                   |     Reason drop down menu         |
|                                   | -   select Save                   |
|                                   | -   add the keyword 'BIIAMM' on   |
|                                   |     the Manage Reviews (IRMR)     |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Record a contact and place the    |
|                                   | intervention on hold for 14 days  |
|                                   |                                   |
|                                   | Refer to a Quality Development    |
|                                   | Officer (QDO).                    |
|                                   |                                   |
|                                   | QDO action                        |
|                                   |                                   |
|                                   | Investigate the mismatch to       |
|                                   | validate the referral. These can  |
|                                   | be located by undertaking a       |
|                                   | National Search (IRNS) screen     |
|                                   | search using the keyword BIIAMM.  |
|                                   |                                   |
|                                   | Is the mismatch referral          |
|                                   | supported?                        |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   delete the keyword BIIAMM     |
|                                   | -   add the keyword 'AMMYES'      |
|                                   | -   ensure the intervention Notes |
|                                   |     reflect validation of the     |
|                                   |     anomaly/mismatch              |
|                                   | -   forward these details via the |
|                                   |     CADO Service Desk webform     |
|                                   |     - Customer Reference Number   |
|                                   |     (CRN)                         |
|                                   |     - details of the mismatch     |
|                                   |     - any extra comments that may |
|                                   |     help the service desk assess  |
|                                   |     the review                    |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   delete the keyword BIIAMM     |
|                                   | -   return the referral to the CO |
|                                   |     with feedback explaining the  |
|                                   |     referral error                |
