Determine limits of assistance for ABSTUDY

|                                   |     Award                         |

|                                   | customer's record. This can be    |
|                                   | checked on the Education Course   |
|                                   | History (EDCH) screen.            |
|                                   |                                   |

|                                   | reasonable time or limit of       |
