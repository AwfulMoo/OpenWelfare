|                                   |                                   |

| 3                                 | Customer under 22 years of age +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer is under 22 years |
|                                   | of age, the course may be an      |
|                                   | approved activity for job         |
|                                   | seekers.                          |
|                                   |                                   |
|                                   | Consider the ways the customer    |
|                                   | may satisfy the YA activity test. |
|                                   | For example, as a job seeker      |
|                                   | undertaking:                      |
|                                   |                                   |
|                                   | -   part-time or full-time study  |
|                                   |     as an approved activity, or   |
