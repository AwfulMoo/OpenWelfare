This page contains information relating to requesting and receiving
information from Inland Revenue, New Zealand (NZIR).

Request and receive information

Request and receive information from NZIR

This table describes the process for requesting and receiving
information from NZIR.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Requesting information + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Email the following information   |
|                                   | to the International NZ           |
|                                   | Administration with the heading   |
|                                   | of PHONE EXCHANGE:                |
|                                   |                                   |
|                                   | -   Customer’s name               |
|                                   | -   Customer’s CSID               |
|                                   | -   Customer’s Inland Revenue     |
|                                   |     Department (IRD) number       |
|                                   | -   A brief summary of what has   |
|                                   |     occurred                      |
|                                   | -   What information you would    |
|                                   |     like clarified by NZIR        |
|                                   |                                   |
|                                   | Document request in Cuba before   |
|                                   | emailing to NZ Admin Unit         |
+-----------------------------------+-----------------------------------+
| 2                                 | Responses from NZIR + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The International CA team will    |
|                                   | document the NZIR response on     |
|                                   | Cuba and send an email advising   |
|                                   | that a response has been          |
|                                   | received.                         |
|                                   |                                   |
|                                   | Note: if there have been previous |
|                                   | communications with NZIR via the  |
|                                   | Daily Exchange spreadsheet, give  |
|                                   | reference to the date and the tab |
|                                   | used.                             |
|                                   |                                   |
|                                   | If the request relates to the     |
|                                   | payer the information will be     |
|                                   | documented in the Payer           |
|                                   | Collection window                 |
|                                   |                                   |
|                                   | If the request relates to the     |
|                                   | payee the information will be     |
|                                   | documented in Communication       |
|                                   | window.                           |
+-----------------------------------+-----------------------------------+
