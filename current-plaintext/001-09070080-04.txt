Social Security Guide, 3.11.2, Job Plans

Policy Decision regarding Drug Court participants

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Decision + Read more ...          |
|                                   |                                   |
|                                   | Unemployed people participating   |
|                                   | in Drug Court programs may be     |
|                                   | approved to meet their mutual     |
|                                   | obligation requirements through a |
|                                   | Job Plan negotiated with either   |
|                                   | Services Australia or their       |
|                                   | provider, if they are a Provider  |
|                                   | managed job seeker.               |
|                                   |                                   |
|                                   | Job Plans are to be renegotiated  |
|                                   | every 6 months and are to take    |
|                                   | into consideration the person's   |
|                                   | individual Drug Court program     |
|                                   | obligations and their ability to  |
|                                   | look for and take up work.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Effective Date + Read more ...    |
|                                   |                                   |
|                                   | 1 November 1999                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Background + Read more ...        |
|                                   |                                   |
|                                   | The objectives of Drug Court are: |
|                                   |                                   |
|                                   | -   to reduce crime associated    |
|                                   |     with the illicit use of drugs |
|                                   | -   to reduce recidivism          |
|                                   | -   to have the participants      |
|                                   |     cease illicit drug use while  |
|                                   |     participating in the program, |
|                                   |     and                           |
|                                   | -   to improve the participant's  |
|                                   |     health and social functioning |
+-----------------------------------+-----------------------------------+
| 4                                 | Description + Read more ...       |
|                                   |                                   |
|                                   | Once accepted into the program,   |
|                                   | participants will be required to  |
|                                   | follow an individual case         |
|                                   | management plan which will set    |
|                                   | out the requirements of the 12    |
|                                   | months program. This program will |
|                                   | be strictly monitored by the      |
|                                   | court and any deviance from the   |
|                                   | agreed program may result in the  |
|                                   | imposition of sanctions and in    |
|                                   | the last resort, the expulsion of |
|                                   | the program. The following        |
|                                   | activities will be mandatory for  |
|                                   | all participants:                 |
|                                   |                                   |
|                                   | -   attendance at court on a      |
|                                   |     weekly, fortnightly or        |
|                                   |     monthly basis                 |
|                                   | -   regular attendance at         |
|                                   |     counselling sessions          |
|                                   |     organised by the Department   |
|                                   |     of Health and Probation &     |
|                                   |     Parole Service (P&PS)         |
|                                   | -   attendance at programs        |
|                                   |     organised by the Department   |
|                                   |     of Health                     |
|                                   | -   TAFE programs such as         |
|                                   |     numeracy and literacy,        |
|                                   |     English as a second language, |
|                                   |     anger management,             |
|                                   |     self-esteem, preparing for    |
|                                   |     employment etc                |
|                                   | -   urinalysis testing, and       |
|                                   | -   home visits by P&PS           |
+-----------------------------------+-----------------------------------+
| 5                                 | Issues + Read more ...            |
|                                   |                                   |
|                                   | As participants are required to   |
|                                   | undertake set activities at set   |
|                                   | times, there may be limits to     |
|                                   | their ability to look for and     |
|                                   | accept full time work. In these   |
|                                   | cases the Job Plan will need to   |
|                                   | reflect a requirement to look for |
|                                   | and accept work at a level        |
|                                   | consistent with their Drug Court  |
|                                   | activities.                       |
|                                   |                                   |
|                                   | On rare occasions the Drug Court  |
|                                   | may direct employed participants  |
|                                   | to temporarily stop work for a    |
|                                   | duration so that they can         |
|                                   | undertake intensive               |
|                                   | rehabilitation to help them       |
|                                   | overcome their drug addiction.    |
|                                   |                                   |
|                                   | Most people in this situation     |
|                                   | will not qualify for JobSeeker    |
|                                   | Payment or Youth Allowance but    |
|                                   | where they do qualify, a penalty  |
|                                   | for voluntary unemployment must   |
|                                   | not be applied.                   |
