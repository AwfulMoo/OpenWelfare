|                                   |                                   |
|                                   | A Service Officer may receive an  |

|                                   | supplementary task when a         |
|                                   | customer advises in their claim:  |

|                                   |     close relative living in      |
|                                   |     their home                    |
|                                   |                                   |
|                                   | The customer provides the carer   |
|                                   | and/or close relative assessment  |
|                                   | instructions to their carer       |
|                                   | and/or close relative.            |
|                                   |                                   |
|                                   | The carer and/or close relative   |
|                                   | confirms their details online     |
|                                   | using the Services Australia      |
|                                   | website:                          |
|                                   |                                   |
|                                   | -   they have 21 days to respond  |
|                                   |     to the request                |
|                                   | -   the means assessment needs to |
|                                   |     be held for 21 days from date |
|                                   |     of issue                      |
|                                   |                                   |
|                                   | Current status of the carer       |
|                                   | and/or close relative assessment  |
|                                   |                                   |
|                                   | Carer and/or close relative       |
|                                   | assessment is outstanding         |
|                                   |                                   |
|                                   | -   A claim number and Temporary  |
|                                   |     Access Code (TAC) has issued  |
|                                   |     as part of an online          |
|                                   |     calculation of ‘your cost of  |
|                                   |     care’. Go to Step 2           |
|                                   |                                   |
|                                   | No carer and/or close relative    |
|                                   | assessment issued                 |
|                                   |                                   |
|                                   | -   Customer has lodged an online |
|                                   |     calculation of ‘your cost of  |
|                                   |     care’. They are now stating   |
|                                   |     that there is a carer and/or  |
|                                   |     close relative living in      |
|                                   |     their principal home, Go to   |
|                                   |     Step 7                        |
+-----------------------------------+-----------------------------------+
| 2                                 | ACC claim or customer enquiry +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a carer and/or close relative  |
|                                   | assessment issues, an assessment  |
|                                   | task for this presents within the |
|                                   | online claim.                     |
|                                   |                                   |
|                                   | Is the carer and/or close         |
|                                   | relative able to confirm their    |
|                                   | details online?                   |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   unable to, or chooses not to  |
|                                   |     use online services, go to    |
|                                   |     Step 4                        |
|                                   | -   carer and/or close relative   |
|                                   |     assessment is no longer       |
|                                   |     required, go to Step 8        |
+-----------------------------------+-----------------------------------+

|                                   | complete online assessment + Read |

|                                   | assessment:                       |
|                                   |                                   |
|                                   | Online through their myGov        |
|                                   | account                           |
|                                   |                                   |
|                                   | -   sign in to myGov              |
|                                   | -   select Centrelink             |
|                                   | -   select Income and assets      |
|                                   | -   select income and assets      |
|                                   |     details                       |
|                                   | -   select Carer and/or close     |
|                                   |     relative assessment for aged  |
|                                   |     care                          |
|                                   |                                   |
|                                   | Online through website            |
|                                   |                                   |
|                                   | -   Go to Services Australia      |
|                                   |     website                       |
|                                   | -   Search for Aged Care Carer    |
|                                   |     and/or close Relative         |
|                                   |     Assessment                    |
|                                   |                                   |
|                                   | Once they access the online       |
|                                   | assessment, they enter the claim  |
|                                   | number and their TAC and complete |
|                                   | the questions.                    |

|                                   | Once complete, the response is    |
|                                   | able to be viewed as a PDF        |
|                                   | document in Document Tools.       |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   | elects not to use online          |
|                                   | services + Read more ...          |

|                                   |     request                       |
|                                   | -   If the carer and/or close     |
|                                   |     relative cannot provide the   |
|                                   |     claim number and TAC, the     |

|                                   |     permission to confirm details |

|                                   |                                   |
|                                   | confirm the claim number and TAC  |

|                                   |                                   |
|                                   | give consent to access their      |
|                                   | Centrelink record after reading   |
|                                   | the following:                    |
|                                   |                                   |
|                                   | -   By providing consent, you are |
|                                   |     allowing Services Australia   |
|                                   |     to use relevant information   |
|                                   |     collected from your record to |

|                                   |     person’s assets under the     |
|                                   |     Aged Care Act 1997. Do you    |

|                                   | Has the carer and/or close        |
|                                   | relative met all of these         |
|                                   | criteria?                         |

|                                   | -   Yes, go to Step 5             |
