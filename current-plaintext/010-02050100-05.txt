Scenario - student turns 16 years of age prior to school commencing in Term 1

+-----------------------------------+-----------------------------------+
| Scenario                          | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Sally was granted ABSTUDY         |
|                                   | Schooling A award from 1 January  |
|                                   | but was cancelled from 7 January  |
|                                   | as Sally turned 16 years of age.  |
|                                   | Sally commenced Year 10 on 30     |
|                                   | January. As Sally had eligibility |
|                                   | for at least one day of Schooling |
|                                   | A Award and commenced study in    |
|                                   | the same academic year, Sally is  |
|                                   | eligible for SFA.                 |
+-----------------------------------+-----------------------------------+
