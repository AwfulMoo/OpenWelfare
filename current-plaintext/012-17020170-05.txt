This page contains some common enquires and suggested responses for
Service Officers as well as links to external websites for information
related to Department of Veterans’ Affairs Social Workers.

Common enquiry types

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: What is the difference  |
|                                   | between a Social Worker and       |
|                                   | Mental Health Social Worker? +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Answer: Social Workers assess a   |
|                                   | patient’s total social and        |
|                                   | psychological situation to assist |
|                                   | them to overcome problems caused  |
|                                   | by illness, social and financial  |
|                                   | change.                           |
|                                   |                                   |
|                                   | Social workers in mental health   |
|                                   | help individuals with mental      |
|                                   | health disorders to resolve       |
|                                   | associated psychosocial problems  |
|                                   | and improve their quality of      |
|                                   | life.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Can I claim for a group |
|                                   | therapy and subsequent            |
|                                   | consultation on the same day? +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Answer: Social workers (mental    |
|                                   | health) claiming US21-29, US97    |
|                                   | cannot claim SW items on the same |
|                                   | day for the same customer.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Can a Social Worker     |
|                                   | claim kilometres for a home       |
|                                   | visit? + Read more ...            |
|                                   |                                   |
|                                   | Answer: Kilometres are payable    |
|                                   | for Psychologist/Social           |
|                                   | Worker/Occupational therapist     |
|                                   | without the need for extra        |
|                                   | registration.                     |
+-----------------------------------+-----------------------------------+
