Archiving, culling, and reactivating of compensation claims

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | How automatic archiving occurs +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Compensation claims cannot be |
|                                   |     manually amended to show a    |
|                                   |     status of 'Archived'          |
|                                   | -   A 'Claim will be Archived'    |
|                                   |     (CMM/ARC) review will show on |
|                                   |     the Future Activity List      |
|                                   |     (FAL) screen                  |
|                                   | -   This review will show on the  |
|                                   |     Activity List (AL) screen as  |
|                                   |     protected                     |
|                                   | -   The status of the claim will  |
|                                   |     change from 'Finalised' to    |
|                                   |     'Archived'                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Impact when claim reactivating +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the claim has been reactivated |
|                                   | since the CMM/ARC review. A       |
|                                   | review will be produced to        |
|                                   | convert the claim to 'Archived'   |
|                                   | at the appropriate date.          |
+-----------------------------------+-----------------------------------+
| 3                                 | How culling affects screens +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | All screens on a culled record    |
|                                   | will be protected but can still   |
|                                   | be browsed.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Culling processing + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The system will set up a CMM/CUL  |
|                                   | review that will:                 |
|                                   |                                   |
|                                   | -   mature two years after the    |
|                                   |     claim has been archived, or   |
|                                   | -   four weeks after the claim    |
|                                   |     has been deleted              |
|                                   |                                   |
|                                   | The 'Claim will be Culled' review |
|                                   | will show on the FAL screen.      |
|                                   |                                   |
|                                   | When review matures, it will show |
|                                   | on the AL screen.                 |
|                                   |                                   |
|                                   | The status of the claim will      |
|                                   | change from 'Deleted' or          |
|                                   | 'Archived' to 'Culled'. The       |
|                                   | activity will show on the HAL     |
|                                   | screen as completed.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Reactivating process + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | To allow updates, a culled        |
|                                   | Compensation claim can be         |
|                                   | reactivated.                      |
|                                   |                                   |
|                                   | See Reactivating a compensation   |
|                                   | claim.                            |
+-----------------------------------+-----------------------------------+
