| 7                                 | Why am I receiving three payment  |
|                                   | statements instead of one?        |
|                                   |                                   |
|                                   | Each payment statement has a      |
|                                   | specific purpose:                 |
|                                   |                                   |
|                                   | -   Payment Advice – provides a   |
|                                   |     summary of the claim payments |
|                                   |     you can expect to receive in  |
|                                   |     your nominated bank account   |
|                                   |     and allows for a quick review |
|                                   |     to check the total            |
|                                   |     reimbursement is within the   |
|                                   |     expected amount of your claim |
|                                   | -   Recipient Created Tax Invoice |
|                                   |     – provides GST applicable     |
|                                   |     information to assist with    |
|                                   |     your financial reporting and  |
|                                   |     reconciling payments          |
|                                   | -   Reconciliation Statement –    |
|                                   |     assists with reconciling      |
|                                   |     payments. Items are listed    |
|                                   |     individually and detail the   |
|                                   |     supplies paid, rejected, or   |
|                                   |     adjusted by Services          |
|                                   |     Australia                     |
+-----------------------------------+-----------------------------------+
| 8                                 | Why do I have multiple PBS        |
|                                   | Payment ID’s on my statement? I   |
|                                   | don’t understand what they mean?  |
|                                   |                                   |
|                                   | All PBS Payment ID’s will         |
|                                   | correspond on each payment        |
|                                   | statement. If you look at page 2  |
|                                   | of the Reconciliation Statement,  |
|                                   | this provides a breakdown of the  |
|                                   | items paid, rejected or adjusted  |
|                                   | by Services Australia.            |
|                                   |                                   |
|                                   | The first PBS Payment ID noted on |
|                                   | the statement represents the      |
|                                   | automated claim assessment        |
|                                   | outcomes of the electronic claim  |
|                                   | load and lists serial numbers,    |
|                                   | item codes, amount paid and any   |
|                                   | associated reason codes for the   |
|                                   | item.                             |
|                                   |                                   |
|                                   | Any subsequent PBS Payment ID’s   |
|                                   | represent adjustments made        |
|                                   | following claim assessment review |
|                                   | by us. The relevant reason codes  |
|                                   | assigned to the claim/s and their |
|                                   | explanation are noted on page 3   |
|                                   | of the statement.                 |
|                                   |                                   |
|                                   | Service Officer - if there is a   |
|                                   | particular item to be discussed,  |
|                                   | go to next question.              |
+-----------------------------------+-----------------------------------+
| 9                                 | Why has this item been rejected?  |
|                                   |                                   |
|                                   | The rejection codes and           |
|                                   | explanation are noted on page 3   |
|                                   | of the Reconciliation Statement.  |
|                                   |                                   |
|                                   | Service Officer - retrieve the    |
|                                   | item record in the system through |
|                                   | Query Ostomy Item screen. Key in  |
|                                   | the association’s approval ID,    |
|                                   | relevant claim period number and  |
|                                   | other supporting information such |
|                                   | as patient identifier or serial   |
|                                   | number.                           |
|                                   |                                   |
|                                   | Discuss the concerns regarding    |
|                                   | the error. Use the Stoma and      |
|                                   | ParaQuad Reason Codes and fix     |
|                                   | instructions as guide to assist   |
|                                   | the enquiry. Advise the           |
|                                   | association of the correct        |
|                                   | claiming requirements and to      |
|                                   | resubmit the item in their next   |
|                                   | claim if necessary.               |
|                                   |                                   |
|                                   | Note: if looking at the payment   |
|                                   | lines for the item, the payment   |
|                                   | amount noted on the PBS Claims    |
|                                   | Processing System does not        |
|                                   | include GST.                      |
+-----------------------------------+-----------------------------------+
| 10                                | Why is there a W or warning       |
|                                   | reason code on my statement?      |
|                                   |                                   |
|                                   | Warning codes do not affect       |
|                                   | payment.                          |
|                                   |                                   |
|                                   | All reason codes provide valuable |
|                                   | feedback to reconcile claims.     |
|                                   | They also help association’s to   |
|                                   | improve payment integrity and     |
|                                   | capture useful information for    |
|                                   | managing the Scheme.              |
+-----------------------------------+-----------------------------------+
