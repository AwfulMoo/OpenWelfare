Leave impacts on home care subsidies, supplements and care recipient fees

Table 1

The amount of home care subsidies and supplements is dependent on the
leave type and leave duration. For home care recipients, leave balances
resets at 2 specific events:

-   on 1 July each year, or
-   when their package level changes at any time

+-----------------------------------+-----------------------------------+
| Leave type                        | The payment impacts are           |
+-----------------------------------+-----------------------------------+
| Hospital                          | Provider:                         |
|                                   |                                   |
|                                   | For 28 consecutive days for each  |
|                                   | hospital stay, the full basic     |
|                                   | subsidy including any supplements |
|                                   | are paid                          |
|                                   |                                   |
|                                   | From day 29, the subsidy reduces. |
|                                   | The subsidy:                      |
|                                   |                                   |
|                                   | -   is payable at 25% of the      |
|                                   |     basic subsidy rate            |
|                                   | -   does not include any primary  |
|                                   |     supplements*                  |
|                                   | -   continues to include other    |
|                                   |     supplements                   |
|                                   |                                   |
|                                   | Care recipient:                   |
|                                   |                                   |
|                                   | Care recipients continue to pay   |
|                                   | their basic daily fee             |
|                                   |                                   |
|                                   | They continue to pay their        |
|                                   | income-tested care fee for up to  |
|                                   | 28 days in a row for each         |
|                                   | hospital stay                     |
|                                   |                                   |
|                                   | From day 29, the income-tested    |
|                                   | care fee they pay is the lower    |
|                                   | amount of either:                 |
|                                   |                                   |
|                                   | -   Their income-tested care fee  |
|                                   | -   25% of the basic subsidy rate |
|                                   |                                   |
|                                   | Fees are different for care       |
|                                   | recipients who entered home care  |
|                                   | before 1 July 2014. If they take  |
|                                   | hospital leave, they can continue |
|                                   | to pay both the basic daily fee   |
|                                   | and the income-tested care fee.   |
|                                   |                                   |
|                                   | *Primary supplements are oxygen,  |
|                                   | enteral feeding, dementia and     |
|                                   | cognition, and veterans’. Other   |
|                                   | eligible supplements are the      |
|                                   | viability and hardship            |
|                                   | supplements.                      |
+-----------------------------------+-----------------------------------+

Impact of leave on the residential care subsidy, supplements and care recipient fees

Table 2

The amount of residential care subsidies and supplements is dependent on
the leave type and leave duration. For residential care recipients,
leave balances only reset on 1 July each year.

+-----------------------------------+-----------------------------------+
| Type of leave                     | Impact on payment of residential  |
|                                   | subsidy to the services and       |
|                                   | impact to care recipient          |
+-----------------------------------+-----------------------------------+
| Hospital                          | Provider:                         |
|                                   |                                   |
|                                   | Residential care recipients are   |
|                                   | entitled to unlimited days of     |
|                                   | hospital leave                    |
|                                   |                                   |
|                                   | For 28 consecutive days for each  |
|                                   | hospital stay, the full basic     |
|                                   | subsidy is paid                   |
|                                   |                                   |
|                                   | From day 29, the subsidy will     |
|                                   | reduce to 50% of the basic        |
|                                   | subsidy rate                      |
|                                   |                                   |
|                                   | For the 2020-2021 financial year  |
|                                   | only, the basic subsidy will not  |
|                                   | be reduced from day 29, for care  |
|                                   | recipients on extended hospital   |
|                                   | leave                             |
|                                   |                                   |
|                                   | For all other financial years,    |
|                                   | the basic subsidy is reduced to   |
|                                   | 50% for permanent residential     |
|                                   | care recipients on extended       |
|                                   | hospital leave                    |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Extended hospital leave that  |
|                                   |     commenced on 20 May 2021      |
|                                   |     financial year and ends on 17 |
|                                   |     July 2021                     |
|                                   | -   The subsidy will reduce to    |
|                                   |     50% of the basic subsidy rate |
|                                   |     for the period 1 to 17 July   |
|                                   |     2021                          |
|                                   |                                   |
|                                   | Care recipient:                   |
|                                   |                                   |
|                                   | -   Current agreed fees and daily |
|                                   |     accommodation costs continue  |
|                                   | -   From day 29, the means tested |
|                                   |     care fee may be reduced       |
+-----------------------------------+-----------------------------------+
| Transition care                   | Provider:                         |
|                                   |                                   |
|                                   | -   Transition care is part of    |
|                                   |     hospital leave                |
|                                   | -   For 28 consecutive days for   |
|                                   |     each transition care period,  |
|                                   |     the full basic subsidy is     |
|                                   |     paid                          |
|                                   | -   From day 29, the subsidy will |
|                                   |     reduce to 50% of the basic    |
|                                   |     subsidy rate                  |
|                                   |                                   |
|                                   | Care recipient:                   |
|                                   |                                   |
|                                   | -   No impact on current agreed   |
|                                   |     fees and daily accommodation  |
|                                   |     costs                         |
+-----------------------------------+-----------------------------------+
| Social leave                      | Provider:                         |
|                                   |                                   |
|                                   | -   Care recipients are entitled  |
|                                   |     to 52 days of social leave in |
|                                   |     a financial year              |
|                                   | -   Subsidy is not payable for    |
|                                   |     care recipients who exceeds   |
|                                   |     the 52 days                   |
|                                   |                                   |
|                                   | Care recipient:                   |
|                                   |                                   |
|                                   | -   Current agreed fees and daily |
|                                   |     accommodation costs continue  |
|                                   | -   Care recipients who go over   |
|                                   |     their 52 days of social leave |
|                                   |     can be asked to pay a fee to  |
|                                   |     reserve their place in the    |
|                                   |     service in addition to their  |
|                                   |     agreed fees and daily         |
|                                   |     accommodation costs           |
+-----------------------------------+-----------------------------------+
| Emergency leave                   | Provider:                         |
|                                   |                                   |
|                                   | -   Permanent residential care    |
|                                   |     recipients are entitled to    |
|                                   |     take emergency leave without  |
|                                   |     exhausting their social leave |
|                                   |     entitlements                  |
|                                   | -   Aged care services will       |
|                                   |     continue to be paid           |
|                                   |     residential care subsidy when |
|                                   |     care recipients are on        |
|                                   |     emergency leave               |
|                                   |                                   |
|                                   | Care recipient:                   |
|                                   |                                   |
|                                   | -   Current agreed fees and daily |
|                                   |     accommodation costs continue  |
|                                   | -   Care recipients cannot be     |
|                                   |     asked to pay a fee to reserve |
|                                   |     their place in the service    |
|                                   |     whilst on leave               |
+-----------------------------------+-----------------------------------+
