Search for prescriber record

Search for prescriber record in database

This table describes how Service Officers display a prescriber record in
the complex drugs databases.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Open Prescriber Search window +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Open any speciality database and  |
|                                   | select Prescriber Details.        |
|                                   |                                   |
|                                   | Result: the Prescriber Search     |
|                                   | window displays.                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for record + Read more ... |
|                                   |                                   |
|                                   | Key either all or part of:        |
|                                   |                                   |
|                                   | -   prescriber number             |
|                                   | -   last name                     |
|                                   | -   first name                    |
|                                   |                                   |
|                                   | Select OK. If search returns:     |
|                                   |                                   |
|                                   | -   One record, the Prescriber    |
|                                   |     Master List window displays   |
|                                   |     the prescriber record. Go to  |
|                                   |     Step 5                        |
|                                   | -   More than one record, Search  |
|                                   |     Results field lists possible  |
|                                   |     matching records. Go to Step  |
|                                   |     3                             |
|                                   | -   No records, No Records Found  |
|                                   |     message displays              |
+-----------------------------------+-----------------------------------+
| 3                                 | Highlight best match + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Highlight the best match in the   |
|                                   | Search Results field, and double  |
|                                   | click.                            |
|                                   |                                   |
|                                   | Result: the Prescriber Master     |
|                                   | List window displays the          |
|                                   | prescriber record.                |
+-----------------------------------+-----------------------------------+
| 4                                 | Action failed search + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the record does not match:     |
|                                   |                                   |
|                                   | -   select Return Search, and     |
|                                   |     repeat Step 5, or             |
|                                   | -   select Search, and go back to |
|                                   |     Step 2 to create a new search |
|                                   |                                   |
|                                   | If there are no matches, see the  |
|                                   | Create prescriber record table.   |
+-----------------------------------+-----------------------------------+
| 5                                 | Exit database + Read more ...     |
|                                   |                                   |
|                                   | Select Exit database to exit.     |
+-----------------------------------+-----------------------------------+

Create prescriber record

Create prescriber record in database

This table describes how Service Officers create a prescriber record in
the complex drugs databases.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Search for record + Read more ... |
|                                   |                                   |
|                                   | Complete Search for prescriber    |
|                                   | record table.                     |
|                                   |                                   |
|                                   | If no record found, go to Step 2  |
|                                   | to create a new prescriber        |
|                                   | record.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter prescriber details + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select [Enter] in New Prescriber  |
|                                   |                                   |
|                                   | Key Prescriber Number             |
|                                   |                                   |
|                                   | Select Title from dropdown list   |
|                                   |                                   |
|                                   | Key the following:                |
|                                   |                                   |
|                                   | -   Last Name                     |
|                                   | -   First Name                    |
|                                   | -   Phone Number                  |
|                                   | -   Phone Number (2) if alternate |
|                                   |     number is available           |
|                                   | -   Mobile number                 |
|                                   | -   Fax number                    |
|                                   | -   Primary Contact Name          |
|                                   | -   Comments, if needed           |
+-----------------------------------+-----------------------------------+
| 3                                 | Select speciality codes + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select prescribing specialty      |
|                                   | codes.                            |
|                                   |                                   |
|                                   | Result: Prescriber information    |
|                                   | will be shared with selected      |
|                                   | speciality databases.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Key preferred address + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key Preferred Address against     |
|                                   | Specialty Code if applicable.     |
|                                   |                                   |
|                                   | Result: address displays in the   |
|                                   | selected speciality database.     |
+-----------------------------------+-----------------------------------+
| 5                                 | Key primary address + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key primary address in address    |
|                                   | fields for AddressA.              |
|                                   |                                   |
|                                   | Result: displays as the default   |
|                                   | address in speciality databases   |
|                                   | if no preferred address is keyed. |
|                                   |                                   |
|                                   | Note: additional addresses are    |
|                                   | keyed as AddressB, AddressC and   |
|                                   | so on.                            |
+-----------------------------------+-----------------------------------+
| 6                                 | Complete PAH Contact Details      |
|                                   | section if applicable + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If Pulmonary Arterial             |
|                                   | Hypertension specialty is         |
|                                   | selected, the PAH Contact Details |
|                                   | section may be completed.         |
|                                   |                                   |
|                                   | Result: details appear in the     |
|                                   | Pulmonary Arterial Hypertension   |
|                                   | database.                         |
+-----------------------------------+-----------------------------------+
| 7                                 | Exit database + Read more ...     |
|                                   |                                   |
|                                   | Select Exit Database to exit.     |
+-----------------------------------+-----------------------------------+

Amend prescriber record

Amend prescriber record in database

This table describes how Service Officers amend a prescriber record in
the complex drugs databases.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Search for record + Read more ... |
|                                   |                                   |
|                                   | Complete Search for prescriber    |
|                                   | record table.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Make changes to record + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To:                               |
|                                   |                                   |
|                                   | -   Change an address or contact  |
|                                   |     details, overtype existing    |
|                                   |     information.                  |
|                                   | -   Add an address line, select a |
|                                   |     new address line. For         |
|                                   |     example, Address B.           |
|                                   | -   Add or change a speciality    |
|                                   |     preferred address, overtype   |
|                                   |     or key the address.           |
|                                   | -   Add a speciality code, select |
|                                   |     from the prescribing codes.   |
|                                   |     Note: more than one code may  |
|                                   |     be selected. Prescriber       |
|                                   |     information will be shared    |
|                                   |     with selected speciality      |
|                                   |     databases.                    |
|                                   | -   Remove a speciality code,     |
|                                   |     highlight the code and press  |
|                                   |     [Delete]. Note: prescriber    |
|                                   |     information is not displayed  |
|                                   |     in the deleted speciality     |
|                                   |     database. Information is      |
|                                   |     retained in the complex drugs |
|                                   |     database.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Exit database + Read more ...     |
|                                   |                                   |
|                                   | Select Exit Database to exit.     |
+-----------------------------------+-----------------------------------+
