This page contains information about how to process accounts for
assistant surgeons.

On this Page:

Assistant surgeon bills the patient directly

Assistant surgeon bills patient direct, surgeon account not submitted

Surgeon bills on behalf of the assistant

Assistant surgeon bills the patient directly

The assistant surgeon bills the patient on a separate account.

Process surgeon and assistant surgeon accounts where assistant surgeon bills patient directly

Table 1: This table describes how to process an account for an assistant
surgeon and surgeon where there is one surgeon and one assistant surgeon
and the assistant surgeon bills the patient on a separate account.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Items for surgeon + Read more ... |
|                                   |                                   |
|                                   | Process items for surgeon.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Process items for assistant       |
|                                   | surgeon + Read more ...           |
|                                   |                                   |
|                                   | Process items for assistant       |
|                                   | surgeon using details on          |
|                                   | assistant surgeon's account.      |
|                                   |                                   |
|                                   | Message 3-ASSIST is displayed.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Key stroke + Read more ...        |
|                                   |                                   |
|                                   | Key AST in RSN field on           |
|                                   | assistant’s item line and press   |
|                                   | [Enter].                          |
|                                   |                                   |
|                                   | If the account is unpaid the      |
|                                   | assistant’s benefit is paid to    |
|                                   | the assistant surgeon.            |
+-----------------------------------+-----------------------------------+

Assistant surgeon bills patient direct, surgeon account not submitted

Process assistant surgeon accounts where assistant surgeon bills patient directly and the surgeons account has not yet been submitted

Table 2: This table describes how to process an account for an assistant
surgeon when the surgeons account has not yet been submitted.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Process items for assistant       |
|                                   | surgeon + Read more ...           |
|                                   |                                   |
|                                   | Process items for assistant       |
|                                   | surgeon.                          |
|                                   |                                   |
|                                   | The message 7- No SURG is         |
|                                   | displayed                         |
|                                   |                                   |
|                                   | The item being claimed cannot be  |
|                                   | automatically processed as there  |
|                                   | is no corresponding item on       |
|                                   | history.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Action warning message + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Review and apply assessing        |
|                                   | restrictions by following warning |
|                                   | messages.                         |
|                                   |                                   |
|                                   | Warning message must be actioned. |
|                                   |                                   |
|                                   | Further explanations can be       |
|                                   | viewed by placing the cursor at   |
|                                   | the beginning of the return       |
|                                   | message and pressing [F1].        |
+-----------------------------------+-----------------------------------+

Surgeon bills on behalf of the assistant

The surgeon bills the patient on behalf of the assistant on the same
account.

Process surgeon or assistant surgeon accounts where the surgeon bills on behalf of the assistant

Table 3: This table describes how to process an account for an assistant
surgeon where there is one surgeon and one assistant surgeon and the
surgeon bills on behalf of the assistant (both on the same account).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Items for surgeon + Read more ... |
|                                   |                                   |
|                                   | Process items for surgeon.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Process items for assistant       |
|                                   | surgeon + Read more ...           |
|                                   |                                   |
|                                   | Process items for assistant       |
|                                   | surgeon using details on          |
|                                   | surgeon's account.                |
|                                   |                                   |
|                                   | Note: ensure that the assistant   |
|                                   | surgeon's provider number is used |
|                                   | for the assistant items.          |
|                                   |                                   |
|                                   | Message 3-ASSIST is displayed.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Key stroke + Read more ...        |
|                                   |                                   |
|                                   | Key SRG in RSN: field on          |
|                                   | assistance item line and press    |
|                                   | [Enter].                          |
|                                   |                                   |
|                                   | If the account is unpaid the      |
|                                   | assistant’s benefit is paid to    |
|                                   | the surgeon.                      |
+-----------------------------------+-----------------------------------+
