This page contains information for eServices staff to action Medicare
and DVA Government Direct Entry Services (GDES) reports for failed
electronic funds transfer (EFT) payments.

On this page:

Veterans' Affairs (DVA) payment that has been written back.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Banking details provided -        |
|                                   | redirect payment + Read more ...  |
|                                   |                                   |
|                                   | If new banking details are        |
|                                   | provided after the payment has    |
|                                   | been written back to DVA, action  |
|                                   | a redirection.                    |
|                                   |                                   |
|                                   | In Mainframe, go to the DVA       |
|                                   | Payment History Enquiry (BRVA)    |
|                                   | screen:                           |
|                                   |                                   |
|                                   | -   Key BRVA and press [Enter]    |
|                                   | -   In Payee Type, key 'P'        |
|                                   | -   In the Payee ID field, enter  |
|                                   |     the full provider number      |
|                                   |     (e g. 1234567AB)              |
|                                   | -   In Search Criteria, enter Pay |
|                                   |     Date (found on the GDES       |
|                                   |     report under the heading      |
|                                   |     'GDES Pay Date’)              |
|                                   | -   In the Selection field, enter |
|                                   |     claim line number where       |
|                                   |     Status is to be amended (e.g. |
|                                   |     1)                            |
|                                   | -   Press [F9] to update          |
|                                   | -   The EFT Payment Status Update |
|                                   |     screen displays               |
|                                   | -   The Status field will display |
|                                   |     REJECT from the earlier step  |
|                                   | -   Over-key 'REDIRECT' in Status |
|                                   |     field                         |
|                                   | -   Key a new message under the   |
|                                   |     previous message in the Text  |
|                                   |     field, for example, WRITEBACK |
|                                   |     REDIRECT, Date, P number      |
|                                   | -   Press [F5] to update          |
+-----------------------------------+-----------------------------------+
| 2                                 | Email Treasury Medicare and       |
|                                   | Department of Veterans' Affairs   |
|                                   | (DVA) Helpdesk to advise          |
|                                   | previously written back payment   |
|                                   | is redirected + Read more ...     |
|                                   |                                   |
|                                   | The Service Officer actioning the |
|                                   | Write Back redirection emails     |
|                                   | Treasury Medicare and DVA         |
|                                   | Helpdesk to inform them of the    |
|                                   | redirection of the written back   |
|                                   | payment. A sample template is     |
|                                   | under ‘Stationery eServices’ in   |
|                                   | the eServices positional mailbox. |
|                                   |                                   |
|                                   | Include in the email:             |
|                                   |                                   |
|                                   | -   Subject: Failed DVA EFT       |
|                                   |     Payment Write Back Reversal   |
|                                   | -   Text in email body: Please    |
|                                   |     reverse the Write Back of the |
|                                   |     below failed EFT Payment as   |
|                                   |     banking details have been     |
|                                   |     updated and redirection has   |
|                                   |     been actioned                 |
|                                   | -   Provider Name                 |
|                                   | -   Provider Number               |
|                                   | -   Screenshot of updated BRVS    |
|                                   |     screen                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalise entries in GDES spread   |
|                                   | sheet + Read more ...             |
|                                   |                                   |
|                                   | Update any relevant fields and    |
|                                   | ensure to add comments regarding  |
|                                   | actions taken.                    |
+-----------------------------------+-----------------------------------+

Action GDES spreadsheet and email daily statistics

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check pended GDES payments for    |
|                                   | updated bank details + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | After actioning the GDES reports, |
|                                   | finalise the GDES line items for  |
|                                   | the day and save the spreadsheet. |
|                                   |                                   |
|                                   | In the GDES spreadsheet:          |
|                                   |                                   |
|                                   | -   Turn on the filter            |
|                                   | -   In the Status column, select  |
|                                   |     ‘pended'                      |
|                                   | -   Go through each of the pended |
|                                   |     GDES payments and check if    |
|                                   |     bank details have been        |
|                                   |     updated for these providers   |
+-----------------------------------+-----------------------------------+
| 2                                 | Redirect payments + Read more ... |
|                                   |                                   |
|                                   | Redirect payments only for        |
|                                   | providers whose banking details:  |
|                                   |                                   |
|                                   | -   have been updated from the    |
|                                   |     incorrect details, and        |
|                                   | -   are currently effective with  |
|                                   |     today’s date or previous date |
|                                   |                                   |
|                                   | See Step 11 in Table 1 for how to |
|                                   | redirect payments.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Update status + Read more ...     |
|                                   |                                   |
|                                   | Change the Status to ‘closed’ and |
|                                   | update the other fields for       |
|                                   | payments which have:              |
|                                   |                                   |
|                                   | -   been redirected               |
|                                   | -   reached KPI date              |
|                                   | -   been written back             |
|                                   |                                   |
|                                   | Put the filter on ‘pended’ once   |
|                                   | again. This displays the          |
|                                   | remaining number of pended GDES   |
|                                   | payments awaiting redirection.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Email GDES statistics + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Email GDES statistics to the Team |
|                                   | Leader at the end of each day. A  |
|                                   | sample template for this is       |
|                                   | located under ‘Stationery         |
|                                   | eServices’ in the eServices       |
|                                   | positional mailbox.               |
|                                   |                                   |
|                                   | Include in the email, the:        |
|                                   |                                   |
|                                   | -   date                          |
|                                   | -   number of GDES payments       |
|                                   |     received on the day           |
|                                   | -   number of payments            |
|                                   |     redirected/closed             |
|                                   | -   number of pended payments     |
|                                   |     awaiting redirection          |
+-----------------------------------+-----------------------------------+
| 5                                 | Remove filters and save + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Remove all filters and save the   |
|                                   | spreadsheet.                      |
|                                   |                                   |
|                                   | Also save a copy in the folder    |
|                                   | named ‘Archived’ located at       |
|                                   | G:\eServices\GDES\archived, in    |
|                                   | the current month folder.         |
+-----------------------------------+-----------------------------------+

Action outstanding DVA payment report

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Cross reference report with GDES  |
|                                   | spreadsheet + Read more ...       |
|                                   |                                   |
|                                   | The Department of Veterans'       |
|                                   | Affairs (DVA) sends requests to   |
|                                   | investigate outstanding DVA       |
|                                   | payments to the eServices inbox.  |
|                                   | The Team Leader forwards requests |
|                                   | on to the Service Officer who is  |
|                                   | on GDES processing that day for   |
|                                   | action.                           |
|                                   |                                   |
|                                   | Cross-reference each provider     |
|                                   | number and payment amount with    |
|                                   | the GDES spreadsheet (saved at    |
|                                   | G:\eServices\GDES\GDES[current    |
|                                   | year]) to check if they are on    |
|                                   | GDES reports received by          |
|                                   | eServices.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter comments in spreadsheet +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In the spreadsheet of outstanding |
|                                   | payments sent from DVA, enter     |
|                                   | comments against each provider    |
|                                   | number to be checked about the    |
|                                   | status of each failed payment.    |
|                                   |                                   |
|                                   | Example comments:                 |
|                                   |                                   |
|                                   | -   in GDES report dated          |
|                                   |     ‘dd/mm/yy’ – currently being  |
|                                   |     followed up                   |
|                                   | -   payment redirected on         |
|                                   |     ‘dd/mm/yy’                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Save and email spreadsheet + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Save the spreadsheet with         |
|                                   | comments. Attach this saved copy  |
|                                   | to the email and forward to DVA   |
|                                   | (cc to Team Leader).              |
