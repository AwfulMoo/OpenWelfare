Transferring SRSS debts to the Department of Home Affairs

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Transferring a SRSS debt to the   |
|                                   | Department of Home Affairs + Read |

|                                   |                                   |
|                                   | In circumstances where an SRSS    |
|                                   | customer has a Department of Home |
|                                   | Affairs deduction in place:       |
|                                   |                                   |
|                                   | -   go to the Debt List (OPDL)    |
|                                   |     screen and select the         |
|                                   |     relevant debt                 |
|                                   | -   go to the Debt Amount (OPDA)  |
|                                   |     screen and key XBP in the     |
|                                   |     Change reason field, then     |
|                                   |     press [Enter]                 |
|                                   | -   the system will confirm the   |
|                                   |     amount to be transferred,     |
|                                   |     press [Enter]                 |
|                                   | -   Do not change or manually     |
|                                   |     enter amounts                 |
|                                   | -   the debt status will change   |
|                                   |     to FXP (Transferred to        |
|                                   |     External Department)          |
|                                   | -   select and complete activity  |
|                                   |     in the Assessment             |
|                                   |     Finalisation (OPAF) screen.   |
|                                   |     Key Y in the Finished field   |
|                                   | -   DOC using Fast Note - select  |
|                                   |     Auto text, use                |
|                                   |     Multicultural > SRSS Debts >  |
|                                   |     SRSS Debt Transferred         |
|                                   |                                   |
|                                   | Repeat coding for all determined  |
|                                   | debts to be transferred.          |
|                                   |                                   |
|                                   | Note: use the one DOC for         |
|                                   | multiple debts transferred        |
