Checking the transition care extension form

Table 1: This table describes how to check the transition care extension
request for completion.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Transition care extension         |
|                                   | requests + Read more ...          |
|                                   |                                   |
|                                   | The transition care extension     |
|                                   | requests are available in 2       |
|                                   | formats:                          |
|                                   |                                   |
|                                   | -   emails from My Aged Care, go  |
|                                   |     to Step 2                     |
|                                   | -   paper format (e.g. mail, fax  |
|                                   |     or post), go to Step 3        |
+-----------------------------------+-----------------------------------+
| 2                                 | Checking My Aged Care’s emails    |
|                                   | for completion + Read more ...    |
|                                   |                                   |
|                                   | Check the following information   |
|                                   | has been provided:                |
|                                   |                                   |
|                                   | care recipient ID                 |
|                                   |                                   |
|                                   | care recipient name               |
|                                   |                                   |
|                                   | date of birth                     |
|                                   |                                   |
|                                   | gender                            |
|                                   |                                   |
|                                   | date of original entry to         |
|                                   | transition care                   |
|                                   |                                   |
|                                   | goals and action details          |
|                                   |                                   |
|                                   | delegation details:               |
|                                   |                                   |
|                                   | -   number of days of extension   |
|                                   | -   name of ACAT                  |
|                                   | -   delegate ID                   |
|                                   | -   date                          |
|                                   |                                   |
|                                   | If any of the above information   |
|                                   | is incomplete or unclear, Service |
|                                   | Officers must advise the aged     |
|                                   | care service and give reasons why |
|                                   | the form cannot be processed.     |
|                                   | Record a note in PaNDA advising   |
|                                   | the action taken. See View, add   |
|                                   | and edit processing notes to a    |
|                                   | work item table of Manage work    |
|                                   | items in PaNDA.                   |
|                                   |                                   |
|                                   | End of process until the required |
|                                   | details have been provided.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Checking the form for             |
|                                   | completion + Read more ...        |
|                                   |                                   |
|                                   | Service Officers must check the   |
|                                   | below sections of the Aged care   |
|                                   | application for an extension of   |
|                                   | Transition Care form (AC012) for  |
|                                   | completion:                       |
|                                   |                                   |
|                                   | -   Transition care service       |
|                                   |     details                       |
|                                   | -   Care recipient details        |
|                                   | -   Extension details             |
|                                   | -   Care recipient’s declaration  |
|                                   | -   Transition care service’s     |
|                                   |     declaration                   |
|                                   | -   Delegation details            |
|                                   |                                   |
|                                   | If any of the above information   |
|                                   | is incomplete or unclear, Service |
|                                   | Officers must advise the aged     |
|                                   | care service and give reasons why |
|                                   | the form cannot be processed.     |
|                                   | Record a note in PaNDA advising   |
|                                   | the action taken. See View, add   |
|                                   | and edit processing notes to a    |
|                                   | work item table of Manage work    |
|                                   | items in PaNDA.                   |
|                                   |                                   |
|                                   | End of process until the required |
|                                   | details have been provided.       |
+-----------------------------------+-----------------------------------+

Processing the transition care extension requests

Table 2: This table describes the process to enter transition care
extension requests in the Aged Care Staff Portal (ACSP).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | The Aged Care Staff Portal        |
|                                   | (ACSP) + Read more ...            |
|                                   |                                   |
|                                   | Note: all fields are optional     |
|                                   | unless marked with an asterisk.   |
|                                   |                                   |
|                                   | Transition care extension         |
|                                   | requests are processed in the     |
|                                   | ACSP.                             |
|                                   |                                   |
|                                   | Log on to the ACSP. The Welcome   |
|                                   | to the Aged Care Staff portal     |
|                                   | launch screen displays            |
|                                   |                                   |
|                                   | Select the Service Search tile    |
|                                   | from the All searches menu tab.   |
|                                   | The Service Search screen         |
|                                   | displays                          |
|                                   |                                   |
|                                   | Complete the Enter service        |
|                                   | details section                   |
|                                   |                                   |
|                                   | -   Select the relevant option in |
|                                   |     the Search by field -         |
|                                   |     defaults to NAPS ID. Service  |
|                                   |     Officers can search for       |
|                                   |     services by BP ID, NAPS ID,   |
|                                   |     RACS ID or Service details    |
|                                   | -   Complete the NAPS ID field -  |
|                                   |     this is a mandatory field if  |
|                                   |     the NAPS ID option has been   |
|                                   |     selected in the Search by     |
|                                   |     field ID                      |
|                                   | -   Complete the BP ID field -    |
|                                   |     this is a mandatory field if  |
|                                   |     the BP ID option has been     |
|                                   |     selected in the Search by     |
|                                   |     field ID                      |
|                                   | -   Complete the RACS ID - this   |
|                                   |     is a mandatory field if the   |
|                                   |     RACS ID option has been       |
|                                   |     selected in the Search by     |
|                                   |     field ID                      |
|                                   | -   Complete the Service name     |
|                                   |     field - all other fields are  |
|                                   |     optional if this field is     |
|                                   |     completed                     |
|                                   |                                   |
|                                   | Select the Search button. The     |
|                                   | search results will display       |
|                                   |                                   |
|                                   | -   Select the relevant           |
|                                   |     hyperlinked aged care service |
|                                   |     ID in the search results      |
|                                   |     table. The ACSP landing page  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Find the care recipient + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the Care recipient search     |
|                                   | screen to search for a care       |
|                                   | recipient.                        |
|                                   |                                   |
|                                   | For further information on how to |
|                                   | navigate to this screen, see      |
|                                   | Registering events in the Aged    |
|                                   | Care Staff Portal.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Registering Additional days +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Use the Register additional       |
|                                   | transitional care days screen to  |
|                                   | register transition care          |
|                                   | extension requests (i.e.          |
|                                   | additional days).                 |
|                                   |                                   |
|                                   | For further information on how to |
|                                   | navigate to this screen, see      |
|                                   | Registering events in the Aged    |
|                                   | Care Staff Portal.                |

Processing a transition care monthly claim form

Table 3: This table describes how to process a transition care monthly
claim form.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Channel option + Read more ...    |
|                                   |                                   |
|                                   | Regardless of how a claim is      |
|                                   | lodged, all transition care       |
|                                   | monthly claim forms are:          |
|                                   |                                   |
|                                   | -   finalised in the Aged Care    |
|                                   |     Provider Portal (ACPP) for    |
|                                   |     online claims and the Aged    |
|                                   |     Care Staff Portal (ACSP) for  |
|                                   |     manual claims                 |
|                                   | -   approved in either ACSP or    |
|                                   |     Aged Care Management Payment  |
|                                   |     System (ACMPS)                |
|                                   |                                   |
|                                   | The transition care monthly claim |
|                                   | forms can be lodged via 2         |
|                                   | channels:                         |
|                                   |                                   |
|                                   | -   online via the ACPP           |
|                                   | -   manual via PaNDA (email, fax  |
|                                   |     or post)                      |
|                                   |                                   |
|                                   | Is this is a manual claim?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 11             |
+-----------------------------------+-----------------------------------+
| 2                                 | Checking the manual claim form    |
|                                   | for completion + Read more ...    |
|                                   |                                   |
|                                   | Check the manual transition care  |
|                                   | monthly claim form for            |
|                                   | completion.                       |
|                                   |                                   |
|                                   | The required information          |
|                                   | includes:                         |
|                                   |                                   |
|                                   | Care recipient details            |
|                                   |                                   |
|                                   | -   care recipient ID             |
|                                   | -   surname                       |
|                                   | -   first name                    |
|                                   | -   sex                           |
|                                   | -   date of birth                 |
|                                   | -   Functional Capacity Score     |
|                                   |     (MBI) on Entry                |
|                                   | -   Functional Capacity Score     |
|                                   |     (MBI) on Departure (if        |
|                                   |     applicable)                   |
|                                   | -   care setting days             |
|                                   |                                   |
|                                   | Declaration                       |
|                                   |                                   |
|                                   | -   name of person authorised to  |
|                                   |     sign                          |
|                                   | -   signature of authorised       |
|                                   |     person                        |
|                                   | -   date                          |
|                                   |                                   |
|                                   | Is the manual claim form          |
|                                   | complete?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Missing information + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If any of the information in Step |
|                                   | 2 is missing, the form must be    |
|                                   | returned to the transition care   |
|                                   | service.                          |
|                                   |                                   |
|                                   | Contact the transition care       |
|                                   | service via:                      |
|                                   |                                   |
|                                   | -   email                         |
|                                   | -   post                          |
|                                   |                                   |
|                                   | To contact via post, access the   |
|                                   | letter templates from the         |
|                                   | Operational Blueprint letters and |
|                                   | electronic messaging:             |
|                                   |                                   |
|                                   | -   under Medicare, select        |
|                                   |     Provider and Third Party      |
|                                   |     letters                       |
|                                   | -   select Aged Care              |
|                                   | -   select Aged Care More         |
|                                   |     information required to       |
|                                   |     process application template  |
|                                   |                                   |
|                                   | Record a note in PaNDA advising   |
|                                   | the action taken. See View, add   |
|                                   | and edit processing notes to a    |
|                                   | work item table of Manage work    |
|                                   | items in PaNDA.                   |
|                                   |                                   |
|                                   | Repeat Step 1 of this table when  |
|                                   | the missing information is        |
|                                   | provided.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Aged Care Staff Portal (ACSP) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Note: all fields are optional     |
|                                   | unless marked with an asterisk.   |
|                                   |                                   |
|                                   | Log on to the ACSP. The Welcome   |
|                                   | to the Aged Care Staff portal     |
|                                   | launch screen displays            |
|                                   |                                   |
|                                   | Select the Service Search tile    |
|                                   | from the All searches menu tab.   |
|                                   | The Service Search screen         |
|                                   | displays                          |
|                                   |                                   |
|                                   | Complete the Enter service        |
|                                   | details section                   |
|                                   |                                   |
|                                   | -   Select the relevant option in |
|                                   |     the Search by* field -        |
|                                   |     defaults to NAPS ID. Service  |
|                                   |     Officers can search for       |
|                                   |     services by BP ID, NAPS ID,   |
|                                   |     RACS ID or Service details    |
|                                   | -   Complete the NAPS ID field -  |
|                                   |     this is a mandatory field if  |
|                                   |     the NAPS ID option has been   |
|                                   |     selected in the Search by*    |
|                                   |     field ID                      |
|                                   | -   Complete the BP ID field -    |
|                                   |     this is a mandatory field if  |
|                                   |     the BP ID option has been     |
|                                   |     selected in the Search by*    |
|                                   |     field ID                      |
|                                   | -   Complete the RACS ID - this   |
|                                   |     is a mandatory field if the   |
|                                   |     RACS ID option has been       |
|                                   |     selected in the Search by*    |
|                                   |     field ID                      |
|                                   | -   Complete the Service name     |
|                                   |     field - all other fields are  |
|                                   |     optional if this field is     |
|                                   |     completed                     |
|                                   |                                   |
|                                   | Select the Search button. The     |
|                                   | search results will display       |
|                                   |                                   |
|                                   | Select the relevant hyperlinked   |
|                                   | aged care service ID in the       |
|                                   | search results table. The ACSP    |
|                                   | landing page displays             |
+-----------------------------------+-----------------------------------+
| 5                                 | Updating existing care            |
|                                   | recipients’ details + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Update existing care recipients’  |
|                                   | details via two methods:          |
|                                   |                                   |
|                                   | -   the individual care           |
