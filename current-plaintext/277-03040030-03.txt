| 1                                 | Change in care percentage but not |
|                                   | the cost percentage + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Services Australia has the        |
|                                   | discretion not to change the      |
|                                   | existing care decision if the     |
|                                   | reported change in care:          |
|                                   |                                   |

|                                   |                                   |

|                                   | all care change notifications     |
|                                   | even if there would be no change  |
|                                   | to the cost percentage.           |
|                                   |                                   |
|                                   | The agency will make a new care   |
|                                   | percentage decision despite the   |
|                                   | fact this will not change the     |
|                                   | cost percentage if:               |
|                                   |                                   |
|                                   | -   the reported care change is   |
|                                   |     due to a written care         |
|                                   |     arrangement (court order,     |
|                                   |     parenting plan or written     |
|                                   |     care agreement) not           |
|                                   |     previously recorded in Cuba.  |
|                                   |     This is mandatory. Services   |
|                                   |     Australia will always make a  |
|                                   |     new care decision to reflect  |
|                                   |     a previously unrecorded       |
|                                   |     written care arrangement, or  |
|                                   | -   the reported care change is   |
|                                   |     due to a written care         |
|                                   |     arrangement ceasing to be     |
|                                   |     followed, or                  |
|                                   | -   either customer has expressed |
|                                   |     a desire to have the care     |
|                                   |     arrangement reflected despite |
|                                   |     there being no change to the  |
|                                   |     cost percentage, or           |
|                                   | -   the reported care change does |
|                                   |     not require any further       |

|                                   |     decision. That is, all the    |
|                                   |     information needed to make a  |

|                                   |     initial point of contact with |
|                                   |     the notifier or respondent,   |
|                                   |     or                            |

|                                   | Refusal decision to be made       |
