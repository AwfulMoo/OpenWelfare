| 5                                 | CSHC Income + Read more ...       |
|                                   |                                   |
|                                   | The income test for the CSHC uses |
|                                   | adjusted taxable income (ATI)     |
|                                   | for:                              |
|                                   |                                   |
|                                   | -   the customer and their        |
|                                   |     partner (if relevant) and     |
|                                   | -   the applicable financial year |
|                                   |                                   |
|                                   | This includes deemed income from  |
|                                   | account-based income streams      |
|                                   |                                   |
|                                   | See Commonwealth Seniors Health   |
|                                   | Card (CSHC) income test and       |
|                                   | reference tax year, and Lodgement |
|                                   | of Notice of Assessment (NOA) and |
|                                   | other evidence of income for      |
|                                   | Commonwealth Seniors Health Card  |
|                                   | (CSHC).                           |
|                                   |                                   |
|                                   | ATI is the sum of:                |
|                                   |                                   |
|                                   | -   taxable income, and           |
|                                   | -   total net investment losses,  |
|                                   |     and                           |
|                                   | -   target foreign income, and    |
|                                   | -   reportable superannuation     |
|                                   |     contributions, and            |
|                                   | -   employer provided fringe      |
|                                   |     benefits in excess of $1,000  |
|                                   |                                   |
|                                   | Deemed income is calculated from  |
|                                   | the current balance of income     |
|                                   | streams. The deemed income is     |
|                                   | added to the ATI amount, with the |
|                                   | combined total used in the CSHC   |
|                                   | income test.                      |
|                                   |                                   |
|                                   | Deeming will only apply to        |
|                                   | account-based-income steam(s) and |
|                                   | not any other financial assets    |
|                                   | they own.                         |

|                                   | Commonwealth Seniors Health Card  |
|                                   | (CSHC) claims                     |

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Before starting Assisted Customer |
|                                   | Claim (ACC) + Read more ...       |
|                                   |                                   |
|                                   | If required update the customer’s |
|                                   | contact details, such as Address, |
|                                   | or interpreter details via the    |
|                                   | Other Contact Details (OCD)       |
|                                   | screen,                           |
|                                   |                                   |
|                                   | If ACC was started before the     |
|                                   | updates were made, the claim can  |
|                                   | be exited and the updates made    |
|                                   | outside the claim before          |
|                                   | submitting it. The updates will   |
|                                   | appear on the Review page in ACC. |
|                                   |                                   |
|                                   | Electronic messaging and letters  |
|                                   |                                   |
|                                   | Customers who provide a mobile    |
|                                   | phone number or email address     |
|                                   | should be advised that they will  |
|                                   | automatically be subscribed to    |
|                                   | Electronic Messaging (EM)         |
|                                   | services unless they have         |
|                                   | previously declined EM. They will |
|                                   | be asked to nominate a preferred  |
|                                   | contact method (SMS or email) for |
|                                   | electronic messages. They will    |
|                                   | receive messages via their        |
|                                   | preferred contact method to       |
|                                   | complete all Required tasks, see  |
|                                   |  Electronic Messaging (EM) -      |
|                                   | Targeted notifications.           |
|                                   |                                   |
|                                   | Where the customer no longer has  |
|                                   | online access or wishes to change |
|                                   | their preferred contact method,   |
|                                   | See Centrelink letters, messages  |
|                                   | and notifications online          |
|                                   |                                   |
|                                   | Partner to claim                  |
|                                   |                                   |
|                                   | Establish whether the customer    |
|                                   | has a partner. If partner is      |
|                                   | eligible and:                     |
|                                   |                                   |
|                                   | -   available, a separate ACC can |
|                                   |     be started with the partner   |
|                                   | -   unavailable, use the FCSO     |
|                                   |     workflow to issue the         |
|                                   |     appropriate claim to the      |
|                                   |     partner                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Running Assisted Customer Claim   |
|                                   | (ACC) + Read more ...             |
|                                   |                                   |
|                                   | Assisted Customer Claim (ACC)     |
|                                   | replicates the questions in the   |
|                                   | online claim.                     |
|                                   |                                   |
|                                   | Service Officers can run ACC      |
|                                   | when:                             |
|                                   |                                   |
|                                   | -   the customer is available and |
|                                   |     able to answer the questions  |
|                                   |     in the claim, and             |
|                                   | -   a customer is unable or       |
|                                   |     unwilling to claim online, or |
|                                   |     helping a customer with       |
|                                   |     starting the online claim     |
|                                   |                                   |
|                                   | Note: If both members of a couple |
|                                   | wish to claim and require an ACC, |
|                                   | ACC will need to be completed for |
|                                   | both members of the couple.       |
|                                   |                                   |
|                                   | Claims started via ACC can        |
|                                   | submitted by the customer, in     |
|                                   | their online account (an active   |
|                                   | Centrelink online account is      |
|                                   | required).                        |
|                                   |                                   |
|                                   | When running ACC, tell the        |
|                                   | customer of any documents they    |
|                                   | need to provide or action they    |
|                                   | need to take to submit their      |
|                                   | claim. The customer will need to: |
|                                   |                                   |
|                                   | -   answer all mandatory          |
|                                   |     questions                     |
|                                   | -   provide all required          |
|                                   |     documents, and                |
|                                   | -   verify their identity if a    |
|                                   |     Confirmed Identify status is  |
|                                   |     not recorded                  |
|                                   |                                   |
|                                   | Some customers may not be able to |
|                                   | complete all of their 'Required'  |
|                                   | tasks before submitting their     |
|                                   | claim. Exceptions apply for some  |
|                                   | vulnerable customers.             |
+-----------------------------------+-----------------------------------+
| 3                                 | Submitting the claim + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Claims commenced online or via    |
|                                   | ACC cannot be submitted until all |
|                                   | required tasks are completed and  |
|                                   | documents are provided, unless a  |
|                                   | claim submission exception        |
|                                   | applies.                          |
|                                   |                                   |
|                                   | Advise the customer a claim is    |
|                                   | not deemed to have been made      |
|                                   | until it is submitted. Check the  |
|                                   | customer record for any documents |
|                                   | already provided that are         |
|                                   | relevant to a claim for CSHC.     |
|                                   |                                   |
|                                   | A claim started using ACC will    |
|                                   | expire after 13 weeks if not      |
|                                   | submitted. The customer must then |
|                                   | start a new claim. Services       |
|                                   | Australia can assess the claim in |
|                                   | some circumstances from the       |
|                                   | original start date. See          |
|                                   | Circumstance Change Monitor for   |
|                                   | more information.                 |
|                                   |                                   |
|                                   | To submit their claim, the        |
|                                   | customer will need to:            |
|                                   |                                   |
|                                   | -   access the draft claim to     |
|                                   |     upload their documents and    |
|                                   |     submit the claim. Service     |
|                                   |     Officers or the customer can  |
|                                   |     upload the required           |
|                                   |     documents. See Step 10 in the |
|                                   |     Assisted Customer Claim (ACC) |
|                                   |     Table of Claiming and         |
|                                   |     re-claiming Age Pension       |
|                                   | -   accept the verbal declaration |
+-----------------------------------+-----------------------------------+
| 4                                 | Completing Verbal Declaration +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Read the customer the verbal      |
|                                   | declaration script ensuring they  |
|                                   | accept and tick the statement     |
|                                   | declaration box.                  |
|                                   |                                   |
|                                   | Service Officers should tick the  |
|                                   | Declaration box on the Review     |
|                                   | your claim page to confirm they   |
|                                   | have read the declaration script  |
|                                   | and the customer accepts it.      |
|                                   |                                   |
|                                   | Once a verbal declaration is      |
|                                   | completed and the ACC submitted,  |
|                                   | a DOC will confirm that the       |
|                                   | verbal declaration was accepted.  |
|                                   | This DOC will display in:         |
|                                   |                                   |
|                                   | -   the Notes section of Process  |
|                                   |     Direct                        |
|                                   | -   Super Case (notes) of         |
|                                   |     Customer First                |
|                                   | -   the Customer Claim Summary    |
|                                   |                                   |
|                                   | If the customer is identified as  |
|                                   | vulnerable and they submit the    |
|                                   | claim with outstanding documents: |
|                                   |                                   |
|                                   | -   Check the customer knows what |
|                                   |     they need to provide          |
|                                   | -   Record on a DOC               |
|                                   |                                   |
|                                   | If a customer is unable or        |
|                                   | declines to make a verbal         |
|                                   | declaration, the Service Officer  |
|                                   | must:                             |
|                                   |                                   |
|                                   | -   invite the customer to log on |
|                                   |     to their online services,     |
|                                   |     accept the declaration and    |
|                                   |     submit their claim, or        |
|                                   | -   provide the customer with a   |
|                                   |     Customer Declaration Form     |
|                                   |     (CDF). Do not submit the      |
|                                   |     claim without the CDF being   |
|                                   |     completed and signed.         |
|                                   |                                   |
|                                   | Where the customer cannot sign    |
|                                   | the CDF:                          |
|                                   |                                   |
|                                   | -   Send a Q004 to customer with  |
|                                   |     a Customer Declaration Form   |
|                                   |     (CDF) using the template in   |
|                                   |     the Resources page.           |
|                                   |                                   |
|                                   | As this cannot be added as a      |
|                                   | required document on Next steps,  |
|                                   | record a detailed note on the     |
|                                   | customer’s record stating the     |
|                                   | Q004 claim and signature block    |
|                                   | was issued with the expected due  |
|                                   | date.                             |
|                                   |                                   |
|                                   | Reminder: if a partner is         |
|                                   | eligible, available and also      |
|                                   | wishes to claim CSHC, run ACC     |
|                                   | with the partner in their own     |
|                                   | record.                           |

|                                   |     due to system limitations or  |
|                                   | -   the customer is unable to     |
