Receiving Assistance for Isolated Children (AIC) End of Year (EOY) review forms

Table 1: this table outlines the process for Service Officers and AIC
Smart Centre staff to complete the Assistance for Isolated Children
(AIC) End of Year (EOY) review forms. Portions of this process are
completed by Assistance for Isolated Children (AIC) Scheme Smart Centre
Processing staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Receipt of (EOY) review + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | [] The completed Assistance for   |
|                                   | Isolated Children (AIC) End of    |
|                                   | Year (EOY) review form is         |
|                                   | normally received at the Student  |
|                                   | Services postal address. The      |
|                                   | scanned copy of the review form   |
|                                   | is referred to the relevant AIC   |
|                                   | Smart Centre. The receipt date of |
|                                   | the review is recorded.           |
+-----------------------------------+-----------------------------------+
| 2                                 | EOY review form received at       |
|                                   | service centre + Read more ...    |
|                                   |                                   |
|                                   | [] Has the completed form been    |
|                                   | lodged at, or sent to a service   |
|                                   | centre rather than sent to the    |
|                                   | Student Services postal address   |
|                                   | for the AIC Smart Centre?         |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, see Step 1 in Table 2     |
+-----------------------------------+-----------------------------------+
| 3                                 | Review form received at service   |
|                                   | centre + Read more ...            |
|                                   |                                   |
|                                   | [] Check that the form is         |
|                                   | completed correctly.              |
|                                   |                                   |
|                                   | Form received in person           |
|                                   |                                   |
|                                   | If the form is missing the        |
|                                   | following information return to   |
|                                   | the customer/applicant for        |
|                                   | completion:                       |
|                                   |                                   |
|                                   | -   missing questions or          |
|                                   |     documents                     |
|                                   | -   not signed (including the     |
|                                   |     applicant's partner's         |
|                                   |     signature where Additional    |
|                                   |     Boarding Allowance (ABA) is   |
|                                   |     being claimed)                |
|                                   |                                   |
|                                   | If the form is completed scan     |
|                                   | onto the student's record.        |
|                                   | Include the service centre and    |
|                                   | the receipt date recorded. The    |
|                                   | scanned copy is then available    |
|                                   | for processing by the relevant    |
|                                   | AIC Smart Centre.                 |
|                                   |                                   |
|                                   | Review form received by post or   |
|                                   | fax                               |
|                                   |                                   |
|                                   | The form is scanned onto the      |
|                                   | student's record by the service   |
|                                   | centre and the receipt date       |
|                                   | recorded. The scanned copy is     |
|                                   | then available for processing by  |
|                                   | the relevant AIC Smart Centre.    |
|                                   |                                   |
|                                   | Lookup local Smart Centre         |
|                                   | Processing details for:           |
|                                   | Assistance for Isolated Children  |
|                                   | (AIC) for the area. Record a DOC  |
|                                   | advising of the action taken.     |
|                                   | Procedure ends here.              |

Scrutinising Assistance for Isolated Children (AIC) End of Year (EOY) review forms

Table 2: this table describes the process for Service Officers and AIC
Smart Centre staff to follow when scrutinising Assistance for Isolated
Children (AIC) End of Year (EOY) review forms.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Is the review form fully          |
|                                   | complete? + Read more ...         |
|                                   |                                   |
|                                   | Have all questions been answered, |
|                                   | all supporting documents supplied |
|                                   | (if needed) and the form signed   |
|                                   | by the applicant (and the         |
|                                   | applicant's partner if Additional |
|                                   | Boarding Allowance (ABA) is being |
|                                   | claimed)?                         |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No;                               |
|                                   |                                   |
|                                   | -   attempt to call the           |
|                                   |     applicant, or                 |
|                                   | -   return a photocopy of the     |
|                                   |     form and request the missing  |
|                                   |     information be provided via a |
|                                   |     Q999 (this includes missing   |
|                                   |     signatures)                   |
|                                   | -   record details of the request |
|                                   |     on a DOC                      |
|                                   | -   procedure ends here           |

| 2                                 | Applicant details + Read more ... |
|                                   |                                   |
|                                   | [] Is the person still an         |
|                                   | approved applicant for AIC        |
|                                   | payments for the student?         |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |

| 3                                 | Applicant details (continued) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Does the form indicate the date   |

|                                   | longer an approved applicant?     |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Do not select the EOY review  |
|                                   |     activity on the Future        |
|                                   |     Activity List (FAL) screen.   |
|                                   |     Navigate to the Benefit       |
|                                   |     Action (BA) screen            |
|                                   | -   Code 'EIC' in the SVC RSN     |
|                                   |     field                         |
|                                   | -   Code 'REJ' in the Action      |
|                                   |     field                         |
|                                   | -   Code 'LCC' in the reason      |
|                                   |     field, and use the date the   |
|                                   |     EOY review has indicated that |
|                                   |     the applicant became          |
|                                   |     unapproved                    |
|                                   | -   Navigate to the AR screen and |
|                                   |     send any overpayment for      |
|                                   |     investigation. DOC the record |
|                                   |     and finalise the REJection    |
|                                   |     activity                      |
|                                   |                                   |
|                                   | No, attempt to call the applicant |
|                                   | to determine the date they became |
|                                   | unapproved. If uncontactable by   |
|                                   | phone, send a request for this    |
|                                   | information to the applicant via  |
|                                   | Q999. DOC the record. Procedure   |
|                                   | ends here                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Processing an AIC EOY review +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The form is returned from the     |
|                                   | applicant under the following     |
|                                   | groups:                           |
|                                   |                                   |
|                                   | -   Group 15: no expected changes |
|                                   |     or supporting documents are   |
|                                   |     required                      |
|                                   | -   Group 16: expected changes    |
|                                   |     and supporting documents are  |
|                                   |     required as the child moving  |
|                                   |     from primary to secondary     |
|                                   | -   Group 17: parental income may |
|                                   |     be required for Additional    |
|                                   |     Boarding Allowance to be paid |
|                                   |                                   |
|                                   | The EOY activity displays on FAL  |
|                                   | until 1 April. Any EOY reviews    |
|                                   | processed after 1 April must be   |
|                                   | indexed as a new claim.           |
|                                   |                                   |
|                                   | Is the EOY activity being         |
|                                   | assessed before 1 April?          |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Assessing the new   |
|                                   |     claim for Assistance for      |
|                                   |     Isolated Children (AIC)       |
|                                   |     (SY040)                       |
+-----------------------------------+-----------------------------------+
| 5                                 | Access the Future Activity List   |
|                                   | (FAL) screen in Customer Record + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | Process the form using the 'EOY'  |
|                                   | activity on the Future Activity   |
|                                   | List (FAL) screen.                |
|                                   |                                   |
|                                   | Access the FAL screen by coding   |
|                                   | 'FAL' in Nxt: field:              |
|                                   |                                   |
|                                   | -   Select the 'EOY' activity     |
|                                   | -   Source: field 'AICEY'         |
|                                   | -   DOR: field the date the       |
|                                   |     review was received. Note: do |
|                                   |     not leave the DOR blank as    |
|                                   |     the system will auto default  |
|                                   |     to the date the review was    |
|                                   |     sent. This affects timeliness |
|                                   |     standards                     |
|                                   | -   Confirm changes?: field 'Y'   |
|                                   | -   Select [Enter] to register    |
|                                   |     the EOY review activity       |
|                                   | -   Select the 'REA' activity     |
|                                   |     from the AL screen to start   |
|                                   |     processing the review         |
|                                   | -   Repeat this process for all   |
|                                   |     AIC siblings who have had     |
|                                   |     EOYs lodged and not assessed  |
|                                   |     yet. This makes sure all      |
|                                   |     family members are actioned   |
|                                   |     and assessed on the same day  |
|                                   |                                   |
|                                   | The system will navigate to the   |
|                                   | AIC Task Selector (EITS) screen.  |
|                                   | Select the screens that need      |
|                                   | updates.                          |
|                                   |                                   |
|                                   | Go to Table 3.                    |
