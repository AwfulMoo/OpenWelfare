Codes to automatically finalise a claim

This table lists the codes that will result in an automatic
finalisation. Use these codes to finalise a claim on the Clearance
Details screen. Do not use them to finalise a claim on the Finalise
Claim screen.

+-----------------------------------+-----------------------------------+
| Code                              | Description                       |
+-----------------------------------+-----------------------------------+
| SIC                               | Sporting injury (no offset        |
|                                   | clause).                          |
+-----------------------------------+-----------------------------------+
| CRM                               | Criminal Injuries Compensation    |
|                                   | (claim type must also be CRM).    |
+-----------------------------------+-----------------------------------+
| RAC                               | Compensation to Relatives Act     |
|                                   | 1897.                             |
+-----------------------------------+-----------------------------------+
| DBC                               | Death benefits.                   |
+-----------------------------------+-----------------------------------+
| ULC                               | Unlawful dismissal.               |
+-----------------------------------+-----------------------------------+
| SHC                               | Sexual harassment.                |
+-----------------------------------+-----------------------------------+
| RDC                               | Racial discrimination.            |
+-----------------------------------+-----------------------------------+
| NDC                               | Natural disaster.                 |
+-----------------------------------+-----------------------------------+
| PNC                               | Professional negligence.          |
+-----------------------------------+-----------------------------------+
| MAH                               | Medically acquired HIV trust.     |
+-----------------------------------+-----------------------------------+
| DVA                               | DVA disability payments.          |
+-----------------------------------+-----------------------------------+
| MEO                               | Medical expenses only             |
+-----------------------------------+-----------------------------------+
| NRA                               | No reasonable action taken to     |
|                                   | claim compensation. The claim     |
|                                   | will automatically finalise after |
|                                   | 28 days if no updates are         |
|                                   | completed on the CMS.             |
|                                   |                                   |
|                                   | These claims will remain with a   |
|                                   | status of finalised until the     |
|                                   | claim is reactivated and the      |
|                                   | clearance reason is changed.      |
+-----------------------------------+-----------------------------------+
| NOC                               | Nil Charge                        |
+-----------------------------------+-----------------------------------+
