This page contains information on how to action Work Items and messages
specific to Child Protection Income Management (CPIM). Portions of these
Work Items are to be completed by an Income Management Contact Officer
(IMCO) only.

outstanding Child Protection Income Management (CPIM) Work Item which
requires investigation.

+-----------------------------------+-----------------------------------+
| 1                                 | Confirm Work Item investigation   |
|                                   | required + Read more ...          |

|                                   | customer's record for the         |

|                                   |     last contact or the customer  |
|                                   |     was not present               |
|                                   | -   a Service Officer has         |
|                                   |     completed an Assessment       |

|                                   | -   The Resources page contains a |

|                                   |                                   |

+-----------------------------------+-----------------------------------+

|                                   | Item + Read more ...              |

|                                   | assist in determining appropriate |

|                                   | Income Management.                |

|                                   | -   Seek assistance from a        |

|                                   |     (if available) or Service     |

|                                   |     Management option) for        |

|                                   |     updates are required, and     |

|                                   |     DOC                           |
|                                   |                                   |

|                                   |                                   |
|                                   | Important:                        |
|                                   |                                   |
|                                   | -   Work Items must not be        |
|                                   |     removed without appropriate   |
|                                   |     investigation and action.     |
|                                   |     Inappropriately 'ticking off' |

|                                   |     completed or impact the       |
|                                   |     customer's Income Management  |

|                                   |     transferred to another office |

Processing Child Protection Income Management (CPIM) Work Items

Table 3: This table details the Work Items, messages and appropriate
action, specific to Child Protection Income Management (CPIM), listed by
number, which are to be actioned by IMCOs. These Work Items are to be
completed by an Income Management Contact Officer (IMCO) only.

+-----------------------------------+-----------------------------------+
| Work Item                         | When Work Item Raised             |
+-----------------------------------+-----------------------------------+
| I289Q1                            | []End in 35 days. IMCO to contact |
|                                   | referring ORG if applicable +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Income Management Contact Officer |
|                                   | (IMCO) only                       |
|                                   |                                   |
|                                   | When Work Item raised             |
|                                   |                                   |
|                                   | The customer's current Income     |
|                                   | Management Notice will end in 35  |
|                                   | days.                             |
|                                   |                                   |
|                                   | Note: a Work Item will be         |
|                                   | displayed regardless of the       |
|                                   | customer's Income Management      |
|                                   | status.                           |
|                                   |                                   |
|                                   | Action required:                  |
|                                   |                                   |
|                                   | IMCO to discuss with the          |
|                                   | referring State or Territory      |
|                                   | Child Protection Authority        |
|                                   | whether Income Management is to   |
|                                   | continue for the customer.        |
|                                   |                                   |
|                                   | The IMCO is to make an            |
|                                   | appointment with the referring    |
|                                   | organisation and, where relevant, |
|                                   | the DVA to review the customer's  |
|                                   | Income Management. The referring  |
|                                   | organisation will decide whether  |
|                                   | Income Management is to end or    |
|                                   | continue.                         |
|                                   |                                   |
|                                   | If the customer is not currently  |
|                                   | being income managed, the         |
|                                   | referring organisation will       |
|                                   | decide if a new notice is to be   |
|                                   | issued, for potential Income      |
|                                   | Management.                       |
+-----------------------------------+-----------------------------------+
| I291Q1                            | []Revoke received for current CP  |
|                                   | notice. Review IM + Read more ... |
|                                   |                                   |
|                                   | Income Management Contact Officer |
|                                   | (IMCO) only                       |
|                                   |                                   |
|                                   | When Work Item raised             |
|                                   |                                   |
|                                   | The referring organisation has    |
|                                   | lodged a Revocation of Income     |
|                                   | Management notice (revoke         |
|                                   | notice), and:                     |
|                                   |                                   |
|                                   | -   no new commencement notice    |
|                                   |     from another authority exists |
|                                   |     and the end date for the      |
|                                   |     notice is more than 28 days   |
|                                   |     in the future                 |
|                                   | -   no continuation notice exists |
|                                   |     and the end date for the      |
|                                   |     notice is more than 28 days   |
|                                   |     in the future                 |
|                                   | -   the end date for the current  |
|                                   |     notice is 28 days or less in  |
|                                   |     the future                    |
|                                   |                                   |
|                                   | Action required:                  |
|                                   |                                   |
|                                   | -   If the end date of the        |
|                                   |     current notice is more than   |
|                                   |     28 days in the future the     |
|                                   |     customer's Income Management  |
|                                   |     will automatically end in 28  |
|                                   |     days                          |
|                                   | -   If the end date of the        |
|                                   |     current notice is 28 days or  |
|                                   |     less in the future then the   |
|                                   |     pre-exit process should have  |
|                                   |     already started               |
|                                   | -   the IMCO should liaise with   |
|                                   |     the referring organisation to |
|                                   |     determine what action to take |
|                                   |     with the customer's Income    |
|                                   |     Management, as the revoke     |
|                                   |     notice has been received less |
|                                   |     than 28 days before Income    |
|                                   |     Management end date           |
|                                   | -   In circumstances where the    |
|                                   |     revoke notice is not          |
|                                   |     automatically processed,      |
|                                   |     IMCOs can manually record the |
|                                   |     revoke notice via the Child   |
|                                   |     Protection Assessment         |
|                                   |     workflow. For more            |
|                                   |     information see Actioning     |
|                                   |     Unified Government Gateway    |
|                                   |     (UGG) requests                |
|                                   | -   If the customer is eligible   |
|                                   |     for Income Management under a |
|                                   |     different measure when the    |
|                                   |     Child Protection notice is    |
|                                   |     revoked, Income Management    |
|                                   |     will continue under that      |
|                                   |     measure. For more information |
|                                   |     see Moving between Income     |
|                                   |     Management measures           |
