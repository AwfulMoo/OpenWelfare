| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   | ...                               |
|                                   |                                   |
|                                   | Make a verbal agreement with the  |
|                                   | employer outlining:               |
|                                   |                                   |
|                                   | the actions they have stated they |
|                                   | will take; and                    |
|                                   |                                   |
|                                   | evidence/details that they will   |
|                                   | supply, such as confirmation of   |
|                                   | insolvency/liquidation, or PPL    |
|                                   | payments paid/not paid            |
|                                   |                                   |
|                                   | Document the record:              |
|                                   |                                   |
|                                   | -   Use Fast Note - select Auto   |
|                                   |     Text, use Families >          |
|                                   |     Updates > ADR - Q836 issued,  |
|                                   |     and                           |
|                                   | -   confirm all fields in the     |
|                                   |     text                          |
|                                   |                                   |
|                                   | Hold the open DOC for 14 days     |
|                                   | from the date the Q836 is issued  |
|                                   |                                   |
|                                   | Issue a manual Notice of          |
|                                   | Resolution (Q836) letter to the   |
|                                   | employer confirming agreed        |
|                                   | actions:                          |
|                                   |                                   |
|                                   | -   Go to the Letter Display (LD) |
|                                   |     screen. Enter Q836 in the     |
|                                   |     letter code                   |
|                                   | -   Select the relevant options,  |
|                                   |     then Next to scroll to the    |
|                                   |     next page. Select continue to |
|                                   |     confirm each option           |
|                                   | -   Enter the agreed actions in   |
|                                   |     the free text field. Adhere   |
|                                   |     to the Plain English Policy   |
|                                   |     and Writing Guides. See the   |
|                                   |     Resources page for links      |
|                                   |                                   |
|                                   | The letter must include:          |
|                                   |                                   |
|                                   | -   Employee details              |
|                                   | -   The role of the employer or   |
|                                   |     where insolvency or other     |
|                                   |     circumstances have been       |
|                                   |     identified, include relevant  |
|                                   |     payability determination      |
|                                   |     information                   |
|                                   | -   Agreed resolution             |
|                                   | -   The evidence required. This   |
|                                   |     is to be faxed, or posted to  |
|                                   |     the PPLEPT within 14 days to  |
|                                   |     avoid further action          |
|                                   | -   FWO warning statement         |
|                                   |                                   |
|                                   | Obtain a duplicate of the Q836    |
|                                   | sent to the employer by printing  |
|                                   | a local copy from the History     |
|                                   | Summary (HS) screen               |
|                                   |                                   |
|                                   | On the employee's record, go to   |
|                                   | the Letter Display (LD) screen to |
|                                   | issue and print locally, a Q837   |
|                                   |                                   |
