Job seeker registration

When a student is eligible to register as a job seeker

This table describes information on circumstances when a student is
eligible to register as a job seeker and for what services they are
eligible for.

+-----------------------------------+-----------------------------------+
| Item                              | Eligibility                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Full-time student currently       |
|                                   | studying, on a semester break or  |
|                                   | between two consecutive years of  |
|                                   | their course + Read more ...      |
|                                   |                                   |
|                                   | A full-time ongoing student is    |
|                                   | not eligible to register as a job |
|                                   | seeker, however they can still    |
|                                   | access Self Service Terminals at  |
|                                   | Services Australia.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Full-time student who has just    |
|                                   | left secondary school and has     |
|                                   | applied for tertiary study + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers who have completed      |
|                                   | their secondary school studies    |
|                                   | and have applied to go on to      |
|                                   | tertiary studies are not          |
|                                   | considered to be full-time        |
|                                   | students for the purposes of      |
|                                   | Employment Services Provider      |
|                                   | eligibility until their           |
|                                   | application has been approved by  |
|                                   | the tertiary institution.         |
|                                   | However, they are still           |
|                                   | considered to be full-time        |
|                                   | students for the purposes of      |
|                                   | income support. See Employment    |
|                                   | assistance for students moving    |
|                                   | from secondary to tertiary        |
|                                   | studies for more information.     |
+-----------------------------------+-----------------------------------+
| 3                                 | Part-time student or job seeker + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers may be eligible for     |
|                                   | employment assistance through     |
|                                   | providers, see Eligibility to     |
|                                   | access employment assistance      |
|                                   | services through Workforce        |
|                                   | Australia for further             |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
