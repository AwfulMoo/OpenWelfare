| 1                                 | Debts + Read more ...             |
|                                   |                                   |
|                                   | A Schoolkids Bonus (SKB) debt     |
|                                   | could arise if the FTB customer   |
|                                   | was not entitled to the payment   |
|                                   | for any reason. This includes a   |
|                                   | retrospective assessment where it |
|                                   | is determined that they were not  |
|                                   | entitled to the payment already   |
|                                   | received for any relevant bonus   |
|                                   | test day.                         |
|                                   |                                   |
|                                   | For example, the customer is paid |
|                                   | for the child for bonus test day  |
|                                   | of 1 January 2014. They then      |
|                                   | advise the child left care on 29  |
|                                   | December 2013. They are not       |
|                                   | entitled to FTB Part A or SKB     |
|                                   | payment for bonus test day 1      |
|                                   | January 2014.                     |
|                                   |                                   |
|                                   | FTB customers who lose FTB        |
|                                   | eligibility as a result of FTB    |
|                                   | non-lodger processing will also   |
|                                   | lose their entitlement to SKB     |
|                                   | payments for the bonus test days  |
|                                   | in the relevant financial year.   |
|                                   | See Item 4.                       |
|                                   |                                   |
|                                   | If FTB Part A entitlement is      |
|                                   | lost:                             |
|                                   |                                   |
|                                   | -   on a date after the relevant  |
|                                   |     bonus test day, a debt is not |
|                                   |     raised, e.g. SKB is paid for  |
|                                   |     1 January 2016 and the FTB    |
|                                   |     customer then advises the     |
|                                   |     child left care on 6 January  |
|                                   |     2016                          |
|                                   | -   retrospectively, on a date on |
|                                   |     or before an SKB bonus test   |
|                                   |     day, the FTB customer is not  |
|                                   |     entitled to any SKB paid and  |
|                                   |     the SKB will be raised as an  |
|                                   |     overpayment (this may not     |
|                                   |     apply where FTB is cancelled  |
|                                   |     retrospectively due to a      |
|                                   |     Youth Allowance backdated     |
|                                   |     claim)                        |
|                                   |                                   |
|                                   | If SKB is paid to an FTB customer |
|                                   | for a child for a bonus test day  |
|                                   | and the child then claims and is  |
|                                   | granted a qualifying income       |
|                                   | support payment (ISP) for a       |
|                                   | period covering the bonus test    |
|                                   | day, see Item 10 in Table 3.      |
|                                   |                                   |
|                                   | Debts are to be raised as         |
|                                   | standalone SKB debts as it is a   |
|                                   | separate payment, not a component |
|                                   | of FTB. SKB debts should not be   |
|                                   | combined with family assistance   |
|                                   | debts, e.g. reconciliation or FTB |
|                                   | qualification debts.              |
|                                   |                                   |
|                                   | SKB debts can be recovered from   |
|                                   | an FTB top up or through          |
|                                   | withholding of ongoing FTB        |
|                                   | fortnightly payments.             |
|                                   |                                   |
|                                   | The SKB may be used to offset     |
|                                   | family assistance debts.          |

| 2                                 | FTB reconciliation process + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Schoolkids Bonus (SKB) is         |
|                                   | impacted by FTB reconciliation    |
|                                   | outcomes.                         |
|                                   |                                   |
|                                   | For 2013 and 2014 bonus test days |
|                                   |                                   |
|                                   | To maintain SKB entitlement, a    |
|                                   | person must remain entitled to    |
|                                   | FTB Part A at reconciliation. If  |
|                                   | entitlement to FTB Part A is lost |
|                                   | at reconciliation for any reason, |
|                                   | an automatic SKB debt is raised.  |
|                                   | SKB debts may be recovered from   |
|                                   | an FTB top-up.                    |
|                                   |                                   |
|                                   | An FTB customer not eligible for  |
|                                   | SKB during the financial year     |
|                                   | because their estimated adjusted  |
|                                   | taxable income was greater than   |
|                                   | the FTB Part A income test is     |
|                                   | automatically paid SKB when their |
|                                   | FTB is reconciled and they gain   |
|                                   | entitlement to FTB Part A and all |
|                                   | other eligibility criteria are    |
|                                   | met. Note: this includes cases    |
|                                   | where income has been confirmed   |
|                                   | late, e.g. after the end of the   |
|                                   | lodgement year.                   |
|                                   |                                   |
|                                   | For 2015 and 2016 bonus test days |
|                                   |                                   |
|                                   | To maintain SKB entitlement, a    |
|                                   | person must remain entitled to    |
|                                   | FTB Part A at reconciliation and  |
|                                   | have an adjusted taxable income   |
|                                   | of $100,000 or less.              |
|                                   |                                   |
|                                   | An FTB customer not eligible for  |
|                                   | SKB during the financial year     |
|                                   | because their estimated adjusted  |
|                                   | taxable income was greater than   |
|                                   | $100,000 is automatically paid    |
|                                   | SKB when their FTB is reconciled  |
|                                   | if their actual adjusted taxable  |
|                                   | income was $100,000 or less and   |
|                                   | all other eligibility criteria    |
|                                   | are met. Note: this includes      |
|                                   | cases where income has been       |
|                                   | confirmed late, e.g. after the    |
|                                   | end of the lodgement year.        |
|                                   |                                   |
|                                   | SKB top-ups are paid immediately  |
|                                   | and do not depend on receipt of   |
|                                   | an ATO wash-up transaction. FTB   |
|                                   | top-ups and SKB top-ups may be    |
|                                   | paid at different times.          |
|                                   |                                   |
|                                   | A letter is not sent advising the |
|                                   | SKB payment.                      |
|                                   |                                   |
|                                   | Prohibition of FTB instalments    |
|                                   |                                   |
|                                   | If SKB was not paid during the    |
|                                   | financial year due to Prohibition |
|                                   | of FTB instalments (PIP), the SKB |
|                                   | will be paid when FTB reconciles  |
|                                   | and Part A eligibility is         |
|                                   | confirmed and all other           |
|                                   | eligibility criteria are met,     |
|                                   | e.g. income test. Note: this      |
|                                   | includes cases where income has   |
|                                   | been confirmed late, e.g. after   |
|                                   | the end of the lodgement year.    |
|                                   |                                   |
|                                   | Debts and debt recovery           |
|                                   |                                   |
|                                   | If SKB was paid to an FTB         |
|                                   | customer during the financial     |
|                                   | year based on estimated income of |
|                                   | $100,000 or less and their actual |
|                                   | adjusted income for the financial |
|                                   | year was more than $100,0000 or   |
|                                   | they were not entitled to FTB     |
|                                   | Part A on the bonus test day, an  |
|                                   | automatic SKB debt is raised. SKB |
|                                   | debts may be recovered from an    |
|                                   | FTB top-up.                       |
