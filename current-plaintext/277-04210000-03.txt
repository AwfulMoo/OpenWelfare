| 4                                 | Informal vs formal information    |
|                                   | gathering + Read more ...         |
|                                   |                                   |
|                                   | Informal information gathering    |
|                                   | does not require a formal legal   |
|                                   | notice to be issued. The          |
|                                   | information is usually gathered   |
|                                   | via phone or by accessing         |
|                                   | electronic resources such as      |
|                                   | Centrelink.                       |
|                                   |                                   |
|                                   | Where possible, information       |
|                                   | should be sought informally as it |
|                                   | can be faster and easier for both |
|                                   | Child Support and the customer.   |
|                                   |                                   |
|                                   | If Service Officers are concerned |
|                                   | that a person:                    |
|                                   |                                   |
|                                   | -   may not respond to an         |
|                                   |     informal information          |
|                                   |     gathering request             |
|                                   | -   states that they will comply, |
|                                   |     but have not after a specific |
|                                   |     time                          |
|                                   |                                   |
|                                   | Issue a formal notice, see Table  |
|                                   | 3.                                |
|                                   |                                   |
|                                   | This includes issuing a notice on |
|                                   | the customer directly, if they    |
|                                   | have refused or failed:           |
|                                   |                                   |
|                                   | -   to comply with an informal    |
|                                   |     request on this and/or        |
|                                   |     previous occasions, and       |
|                                   | -   are identified as the likely  |
|                                   |     sole holder of the required   |
|                                   |     information                   |
|                                   |                                   |
|                                   | If there is a formal arrangement  |
|                                   | for requests between Child        |
|                                   | Support and a source, Service     |
|                                   | Officers must follow the          |
|                                   | established procedures. Examples  |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   issuing a legal notice to     |
|                                   |     formally seek information     |
|                                   |     from a bank or financial      |
|                                   |     institution                   |
|                                   | -   before accessing ATO systems  |
|                                   |     directly for information      |
|                                   |     relevant to a particular      |
|                                   |     customer, issue an electronic |
|                                   |     ATO legal notice. See Legal   |
|                                   |     notice for Child Support      |
|                                   |     access to Australian Taxation |
|                                   |     Office (ATO) data and systems |
|                                   |                                   |
|                                   | See also Table 2 or Table 3.      |

|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | When seeking information          |
|                                   | informally from a customer:       |
|                                   |                                   |
|                                   | -   Tell the customer why the     |
|                                   |     information is required       |
|                                   |     including the type of         |
|                                   |     information that will be      |
|                                   |     relied upon to make a         |
|                                   |     decision                      |
|                                   | -   Consider the customer's       |
|                                   |     cultural background and       |
|                                   |     individual needs when         |
|                                   |     supporting them to comply     |
|                                   |     with the request for          |

|                                   | -   Make sure the customer is     |

|                                   |     opportunity to provide        |
|                                   |     relevant information taking   |
|                                   |     into account individual       |
|                                   |     circumstances                 |
|                                   | -   Ensure the customer           |
|                                   |     understands the potential     |
|                                   |     consequences of not complying |
|                                   |     with the request for          |
|                                   |     information, that is, the     |
|                                   |     information may be sought     |
|                                   |     from elsewhere and/or a       |
|                                   |     formal notice may be issued   |
|                                   | -   Consider all relevant         |
|                                   |     available information before  |
|                                   |     making a decision, including  |
|                                   |     whether to undertake more     |

|                                   |                                   |

Formal information gathering via issuing a notice

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Formal information gathering +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Formal information gathering      |
|                                   | means that a notice is issued in  |
|                                   | writing requiring specified       |
|                                   | information be provided in a      |
|                                   | certain manner by a certain date. |
|                                   |                                   |
|                                   | Set clear and reasonable          |
|                                   | timeframes for information to be  |
|                                   | provided. In limited              |
|                                   | circumstances, discretion may be  |
|                                   | applied to allow more time to     |
|                                   | respond, if necessary. See        |
|                                   | Standard response times.          |
|                                   |                                   |
|                                   | Issuing multiple notices to a     |
|                                   | source within a short period,     |
|                                   | requesting the same information,  |
|                                   | could delay the response. Service |
|                                   | Officers must not collect         |
|                                   | information that is unnecessary   |
|                                   | or intrudes unreasonably on a     |
|                                   | person's privacy. This includes   |
|                                   | seeking information already held, |
|                                   | or requesting information already |
|                                   | provided by a third party, which  |
|                                   | is unlikely to result in a        |
|                                   | different outcome for the purpose |
|                                   | of assessment or collection.      |
|                                   | Document the reasons for issuing  |
|                                   | a second notice within a short    |
|                                   | period.                           |
|                                   |                                   |
|                                   | Notices must be generated and     |
|                                   | issued in Cuba.                   |
|                                   |                                   |
|                                   | When issuing a notice, the        |
|                                   | customer must be locked to an     |
|                                   | officer as per Customer           |
|                                   | Management Approach (CMA) for     |
|                                   | Child Support. This is to ensure  |
|                                   | that a contact officer is listed  |
|                                   | on the notice.                    |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Notices are tools to gather   |
|                                   |     information. They must not be |
|                                   |     used or issued to intimidate  |
|                                   |     or coerce a customer          |
|                                   | -   A warning message will appear |
|                                   |     when issuing s120 and s161    |
|                                   |     notices if the customer has   |
|                                   |     current aliases which may     |
|                                   |     need to be included on the    |
|                                   |     notice/s                      |
+-----------------------------------+-----------------------------------+

|                                   |     electronically via Child      |
|                                   |     Support Business Online       |
|                                   |     Services (CSBOS)              |

|                                   | (EW) linking. Issuing a s120      |

|                                   | the linking process.              |
|                                   |                                   |
|                                   | See External searches guide,      |

|                                   | Support customers.                |
