|                                   | certain manner by a certain date. |
|                                   | Informal information gathering    |
|                                   | does not require a formal notice  |
|                                   | to be issued. Informal            |
|                                   | information will usually be       |

|                                   | accessing electronic resources    |
|                                   | such as Centrelink.               |
|                                   |                                   |
|                                   | Note: prior to accessing ATO      |

|                                   | data and systems.                 |
|                                   |                                   |

|                                   | can be faster and easier for both |
|                                   | Child Support and the customer.   |

Informal information gathering via phone or electronic searches

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Informal information gathering +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Informal information gathering    |
|                                   | will usually be via telephone or  |
|                                   | through electronic searches.      |
|                                   |                                   |
|                                   | Often the best informal source is |
|                                   | the customer. Contact the         |
|                                   | customer first unless it may      |
|                                   | jeopardise collection action.     |
|                                   | Make genuine efforts to contact   |
|                                   | the customer with methods         |
|                                   | specific to the customer. See     |
|                                   | Contact with Child Support        |
|                                   | customers.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Seeking information from a        |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | When seeking information from a   |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   Explain to the customer why   |
|                                   |     the information is required   |
|                                   |     including the type of         |
|                                   |     information that will be      |
|                                   |     relied upon to make a         |
|                                   |     decision                      |
|                                   | -   Consider the customer's       |
|                                   |     cultural background and       |
|                                   |     individual needs when         |
|                                   |     supporting them to comply     |
|                                   |     with the request for          |
|                                   |     information                   |
|                                   | -   Ensure customers are given    |
|                                   |     reasonable opportunities to   |
|                                   |     provide relevant information  |
|                                   |     taking into account           |
|                                   |     individual circumstances      |
|                                   | -   Ensure customers understand   |
|                                   |     the potential consequences of |
|                                   |     not complying with the        |
|                                   |     request for information, that |
|                                   |     is, a formal notice may be    |
|                                   |     issued                        |
|                                   | -   Ensure all relevant available |
|                                   |     information is considered     |
|                                   |     before making a decision and  |
|                                   |     whether the decision is to    |
|                                   |     undertake more formal         |
|                                   |     information gathering or      |
|                                   |     otherwise                     |
|                                   | -   Other sources that may        |
|                                   |     provide information           |
|                                   |     informally are the other      |
|                                   |     customer in the case,         |
|                                   |     employers, or business        |
|                                   |     associates of the customer.   |
|                                   |     As informal information       |
|                                   |     gathering from the other      |
|                                   |     parent or third party often   |
|                                   |     occurs in a very              |
|                                   |     conversational way, be        |
|                                   |     careful not to give away      |
|                                   |     information about the         |
|                                   |     customer that would breach    |
|                                   |     their privacy                 |
|                                   |                                   |
|                                   | If information is obtained, go to |
|                                   | Documentation.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Seeking information from any      |
|                                   | other source + Read more ...      |
|                                   |                                   |
|                                   | When seeking information          |
|                                   | informally from any source:       |
|                                   |                                   |
|                                   | -   Allow a reasonable timeframe  |
|                                   |     to provide the information    |
|                                   | -   Be careful not to breach the  |
|                                   |     customer's privacy            |
|                                   | -   If the source is the customer |
|                                   |     and they cannot or will not   |
|                                   |     provide the information,      |
|                                   |     advise them the intention is  |
|                                   |     to seek the information       |
|                                   |     elsewhere                     |
|                                   | -   If the source refuses to      |
|                                   |     provide information or is     |
|                                   |     uncomfortable providing the   |
|                                   |     details informally, or there  |
|                                   |     is reason to believe the      |
|                                   |     source will not comply with   |
|                                   |     an informal request, issue a  |
|                                   |     formal notice requiring the   |
|                                   |     information be provided       |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Many organisations must       |
|                                   |     comply with privacy           |
|                                   |     legislation and issuing a     |
|                                   |     formal notice ensures that    |
|                                   |     the source is not in breach   |
|                                   |     of their own privacy          |
|                                   |     considerations                |
|                                   | -   If seeking detailed, complex, |
|                                   |     or extensive information it   |
|                                   |     is preferable to issue a      |
|                                   |     formal notice so that the     |
|                                   |     information required is       |
|                                   |     clearly communicated to the   |
|                                   |     source and the source         |
|                                   |     provides the response in      |
|                                   |     writing                       |
|                                   |                                   |
|                                   | If information is obtained, go to |
|                                   | Documentation.                    |
|                                   |                                   |
|                                   | If other sources may provide      |
|                                   | information informally, continue  |
|                                   | seeking information from those    |
|                                   | sources.                          |

Formal information gathering via issuing a notice

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Formal information gathering +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Notices must be generated and     |
|                                   | issued through Cuba.              |
|                                   |                                   |
|                                   | When issuing a notice, Service    |
|                                   | Officer(s) must select Melbourne  |
|                                   | Central from the Office dropdown  |
|                                   | list.                             |
|                                   |                                   |
|                                   | There is currently no capacity    |
|                                   | for Cuba to issue s161 notices    |
|                                   | seeking information about a child |
|                                   | earnings (who does not have a     |
|                                   | role as a payer) in a change of   |
|                                   | assessment Reason 4 application.  |
|                                   |                                   |
|                                   | The notice must be issued         |
|                                   | manually. Where a notice cannot   |
|                                   | be issued through Cuba see        |
|                                   | Letters for Child Support         |
|                                   | customers.                        |
|                                   |                                   |
|                                   | If a notice is issued manually    |
|                                   | the schedule sent must be copied  |
|                                   | into a notepad. Also a hard copy  |
|                                   | of the actual notice must either  |
|                                   | be:                               |
|                                   |                                   |
|                                   | -   batch filed or                |
|                                   | -   stored in the file relevant   |
|                                   |     to the proceedings. For       |
|                                   |     example COA or OBJ            |
|                                   |                                   |
|                                   | Notices may be issued by          |
|                                   | facsimile in certain              |
|                                   | circumstances. See Letters for    |
|                                   | Child Support customers if there  |
|                                   | is a need to serve a notice by    |
|                                   | fax:                              |
|                                   |                                   |
|                                   | -   Financial institutions must   |
|                                   |     consent to a notice being     |
|                                   |     served by fax, see Letters    |
|                                   |     for Child Support customers   |
|                                   | -   Employers and some financial  |
|                                   |     institutions can elect to     |
|                                   |     have their notices sent       |
|                                   |     electronically via Child      |
|                                   |     Support Business Online       |
|                                   |     Services                      |
|                                   |                                   |
|                                   | Service Officer(s) should         |
|                                   | consider issuing a notice on the  |
|                                   | customer if they have refused or  |
|                                   | failed to comply with an informal |
|                                   | request on this and/or previous   |
|                                   | occasions and are identified as   |
|                                   | the likely sole holder of the     |
|                                   | required information.             |
|                                   |                                   |
|                                   | Set clear and reasonable          |
|                                   | timeframes for information to be  |
|                                   | provided, generally not less than |
|                                   | 14 days. Consider the individual  |
|                                   | circumstances of the source when  |
|                                   | setting the response date. A      |
|                                   | source in a regional or remote    |
|                                   | area may need 21 or 28 days to    |
|                                   | return a response.                |
|                                   |                                   |
|                                   | Allow the customer time to        |
|                                   | respond. See Standard response    |
|                                   | times.                            |
|                                   |                                   |
|                                   | Section 120 notices seeking       |
|                                   | information about customers are   |
|                                   | issued through the Collection     |
|                                   | window.                           |
|                                   |                                   |
|                                   | If information from an employer   |
|                                   | is required, use an employer      |
|                                   | questionnaire (EF1). Issuing a    |
|                                   | s120 notice to an employer could  |
|                                   | delay the linking process.        |
|                                   |                                   |
|                                   | Section 161 notices are issued    |
|                                   | through the Collection/Capacity   |
|                                   | to Pay window for both payees and |
|                                   | payers.                           |
|                                   |                                   |
|                                   | If issuing a s120 or s161 notice  |
|                                   | with a standard schedule will     |
|                                   | result in obtaining information   |
|                                   | that cannot be used for the       |
|                                   | purposes of collecting, making    |
|                                   | and varying child support         |
|                                   | assessments, a unique schedule    |
|                                   | should be used.                   |
|                                   |                                   |
|                                   | Endorsed schedules for a range of |
|                                   | organisations are available via   |
|                                   | Cuba or unique schedules in       |
|                                   | Separated parents letter and      |
|                                   | email templates, Schedules - S161 |
|                                   | and S120.                         |
|                                   |                                   |
|                                   | Clearly document:                 |
|                                   |                                   |
|                                   | -   reasons for issuing a s120 or |
|                                   |     s161 notice                   |
|                                   | -   reasons for requesting a      |
|                                   |     search                        |
|                                   | -   the information being         |
|                                   |     requested                     |
|                                   | -   whether the customer was      |
|                                   |     asked to provide the          |
|                                   |     information, and if not, the  |
|                                   |     reason the customer was not   |
|                                   |     asked, and                    |
|                                   | -   how the lead was obtained,    |
|                                   |     for example, bank account     |
|                                   |     details supplied by employer  |
|                                   |                                   |
|                                   | Issuing a notice will generate an |
|                                   | intray due 21 days after the      |
|                                   | notice issued. For large          |
|                                   | organisations (such as banks) it  |
|                                   | is usual to allow the full 21 day |
|                                   | response timeframe and only       |
|                                   | contact them if no response is    |
|                                   | received after this time.         |
|                                   |                                   |
|                                   | When issuing a notice, the        |
|                                   | customer must be locked to an     |
|                                   | officer as per Customer           |
|                                   | Management Approach (CMA) for     |
|                                   | Child Support. This is to ensure  |
|                                   | that a contact officer is listed  |
|                                   | on the notice.                    |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   notices are tools to gather   |
|                                   |     information. They must not be |
|                                   |     used or issued to intimidate  |
|                                   |     or coerce a customer          |
|                                   | -   a warning message will appear |
|                                   |     when issuing s120 and s161    |
|                                   |     notices if the customer has   |
|                                   |     current aliases which may     |
|                                   |     need to be included on the    |
|                                   |     notice/s                      |
|                                   |                                   |
|                                   | The Child Support Guide outlines  |
|                                   | the information that can be       |
|                                   | sought under section 120 of the   |
|                                   | Registration and Collection Act   |
|                                   | and section 161 of the Assessment |
|                                   | Act.                              |

| 2                                 | Requesting information from       |
|                                   | Department of Veterans' Affairs + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When there is reason to believe a |
|                                   | customer is in receipt of a       |
|                                   | Department of Veterans' Affairs   |
|                                   | (DVA) pension or benefit and      |
|                                   | attempts are being made to update |
|                                   | the customer's income or explore  |
|                                   | avenues for collecting child      |
|                                   | support, it may be appropriate to |
|                                   | issue a request to DVA for more   |
|                                   | detailed information. There are 2 |
|                                   | ways this can be done:            |
|                                   |                                   |
|                                   | -   Child Support can directly    |
|                                   |     request information from DVA  |
|                                   |     when it relates to            |
|                                   |     determining the tax free      |
|                                   |     pension or benefit component  |
|                                   |     of a customer's adjusted      |
|                                   |     taxable income (see the Child |
|                                   |     Support Guide 2.4.4: child    |
|                                   |     support income) or to         |
|                                   |     determine if the customer     |
|                                   |     receives 1 of the 6 DVA       |
|                                   |     payments from which Child     |
|                                   |     Support can collect debt (see |
|                                   |     the Child Support Guide       |
|                                   |     5.2.7: Collection from        |
|                                   |     veterans pensions and         |
|                                   |     allowances). For direct       |
|                                   |     requests, see Direct Requests |
|                                   |     for Information from          |
|                                   |     Department of Veterans'       |
|                                   |     Affairs (DVA)                 |
|                                   | -   For all other information     |
|                                   |     requests, Child Support must  |
|                                   |     issue a notice under s120 or  |
|                                   |     s161 using the appropriate    |
|                                   |     schedule                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Schedules for requesting          |
|                                   | information + Read more ...       |
|                                   |                                   |
|                                   | There are a variety of standard   |
|                                   | schedules which have been         |
|                                   | designed to request the most      |
|                                   | commonly required information.    |
|                                   |                                   |
|                                   | Endorsed schedules for a range of |
|                                   | organisations are available via   |
|                                   | Cuba or manual schedules in       |
|                                   | Separated parents letter and      |
|                                   | email templates, Schedules - S161 |
|                                   | and S120.                         |
|                                   |                                   |
|                                   | If it is necessary to ask for     |
|                                   | information which is not covered  |
|                                   | by questions in the Cuba          |
|                                   | schedules, generate a unique      |
|                                   | schedule This must be authorised  |
|                                   | by TL. Document the reasons for   |
|                                   | the unique schedule in the        |
|                                   | notepad.                          |
|                                   |                                   |
|                                   | See Issue section 120/161 notice  |
|                                   | or External searches guide.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Payment for information + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Some authorities may request      |
|                                   | payment of a fee before they      |
|                                   | comply with our request for       |
|                                   | information. These may be         |
|                                   | government authorities or         |
|                                   | 'information brokers' such as     |
|                                   | Veda Advantage. In this case,     |
|                                   | look for other ways to get the    |
|                                   | information. If unsuccessful, ask |
|                                   | a Service Support Officer (SSO)   |
|                                   | to help decide if the information |
|                                   | is critical enough to justify the |
|                                   | fee.                              |
|                                   |                                   |
|                                   | However, not all organisations    |
|                                   | are entitled to charge a fee for  |
|                                   | complying with Child Support      |
|                                   | notices. For example,             |
|                                   | accountants, lawyers and medical  |
|                                   | professionals ordinarily charge   |
|                                   | their clients at a scheduled rate |
|                                   | for their services. These         |
|                                   | professionals may ask Child       |
|                                   | Support to pay at a similar rate  |
|                                   | for what they see as a service.   |
|                                   | Complying with Child Support      |
|                                   | notices is their legal obligation |
|                                   | so Child Support will not pay for |
|                                   | their compliance. Service         |
|                                   | Officers should be careful not to |
|                                   | make unreasonable requests for    |
|                                   | information in these cases. For   |
|                                   | example, a request for            |
|                                   | information from an accountant    |
|                                   | should not require the accountant |
|                                   | to produce 'special purpose       |
|                                   | financial statements' – documents |
|                                   | created solely for the purpose of |
|                                   | complying with Child Support      |
|                                   | notice.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Simultaneous issue of s120 and    |
|                                   | s72A notices to banks + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Child Support will usually seek   |
|                                   | information to identify capacity  |
|                                   | to pay to improve Child Support's |
|                                   | negotiating power, prior to       |
|                                   | taking garnishee action.          |
|                                   |                                   |
|                                   | Some financial institutions, for  |
|                                   | example, Commonwealth Bank, will  |
|                                   | advise a mutual customer that     |
|                                   | Services Australia has made       |
|                                   | enquiries which could result in   |
|                                   | the customer withdrawing funds to |
|                                   | avoid garnishee action.           |
|                                   |                                   |
|                                   | In limited circumstances where    |
|                                   | this is known to be the case, it  |
|                                   | may be appropriate to issue s72A  |
|                                   | and s120 notices at the same time |
|                                   | to that financial institution.    |
|                                   |                                   |
|                                   | Service Officers must be certain  |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   the customer is non-compliant |
|                                   | -   the customer has no intention |
|                                   |     of meeting their debt         |
|                                   |     responsibilities, and         |
|                                   | -   the s72A notice should not    |
|                                   |     leave the customer without    |
|                                   |     access to funds to meet       |
|                                   |     immediate essential living    |
|                                   |     expenses and expected regular |
|                                   |     payments. Contact a SSO for   |
|                                   |     support with this decision,   |
|                                   |     see Technical support in      |
|                                   |     Child Support                 |
|                                   |                                   |
|                                   | Note: the customer will be unable |
|                                   | to access their account where the |
|                                   | balance of the account is less    |
|                                   | than the amount requested in the  |
|                                   | notice.                           |
|                                   |                                   |
|                                   | Fully document the reasons for    |
|                                   | the decision and evidence that    |
|                                   | supports the decision in the s120 |
|                                   | notepad and the s72A notepad.     |
|                                   |                                   |
|                                   | Do not issue s72A notices to:     |
|                                   |                                   |
|                                   | -   a bank account receiving      |
|                                   |     National Disability Insurance |
|                                   |     Scheme (NDIS) payments or     |
|                                   |     which holds a known payment   |
|                                   |     from the National Redress     |
|                                   |     Scheme.                       |
|                                   |     This includes NDIS or Redress |
|                                   |     payments made to a nominee or |
|                                   |     beneficiary.                  |
|                                   |     See National Intensive        |
|                                   |     Collection Services           |
|                                   | -   recover relevant debts        |
|                                   |     directly from a Status        |
|                                   |     Resolution Support Services   |
|                                   |     (SRSS) Program payment        |
|                                   |                                   |
|                                   | To issue s72A notices, see        |
|                                   | Collection of child support debts |
|                                   | through third parties (s72A and   |
|                                   | s72AC).                           |
|                                   |                                   |
|                                   | See References for a link to The  |
|                                   | Child Support Guide 5.2.9:        |
|                                   | Collection from third parties.    |
