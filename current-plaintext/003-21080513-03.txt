| 1                                 | Basic eligibility + Read more ... |
|                                   | There are modal links to rows in  |

|                                   |                                   |
|                                   | Customers may be eligible for an  |
|                                   | urgent payment if they:           |
|                                   |                                   |
|                                   | are in severe financial hardship: |
|                                   |                                   |
|                                   | -   as a single person, their     |
|                                   |     liquid assets are less than   |
|                                   |     the maximum fortnightly       |
|                                   |     payment rate (including       |
|                                   |     add-ons) for the payment they |
|                                   |     are receiving                 |
|                                   | -   as a member of a couple,      |
|                                   |     their liquid assets are less  |
|                                   |     than twice the maximum        |
|                                   |     fortnightly payment rate      |
|                                   |     (including add-ons) for the   |
|                                   |     payment they are receiving    |
|                                   |                                   |
|                                   | will not be placed in further     |
|                                   | financial hardship by the urgent  |
|                                   | payment                           |
|                                   |                                   |
|                                   | are receiving an eligible         |
|                                   | Centrelink payment. If they are   |
|                                   | receiving an ineligible payment,  |
|                                   | they cannot get an urgent payment |
|                                   | and their request should be       |
|                                   | denied                            |
|                                   |                                   |
|                                   | are not receiving weekly payments |
|                                   |                                   |
|                                   | have not received an urgent       |
|                                   | payment on the same day           |
|                                   |                                   |
|                                   | have not requested one urgent     |
|                                   | payment and one reconsideration   |
|                                   | of the decision on the same day   |
|                                   |                                   |
|                                   | have not received 2 urgent        |
|                                   | payments in the last 12 months.   |
|                                   | Note: this only applies to urgent |
|                                   | payment requests for exceptional  |
|                                   | and unforeseen circumstances.     |
|                                   | Customers may be eligible for an  |
|                                   | extra payment in extraordinary    |
|                                   | circumstances. Do not count       |
|                                   | urgent payments due to funerals,  |
|                                   | Centrelink delay or holiday       |
|                                   | processing                        |
|                                   |                                   |
|                                   | do not have a record in           |
|                                   | Environment 'I' (overseas)        |
|                                   |                                   |
|                                   | See the relevant procedure for    |
|                                   | other eligibility criteria:       |
|                                   |                                   |
|                                   | -   urgent payments due to        |
|                                   |     exceptional and unforeseen or |
|                                   |     extraordinary circumstances   |
|                                   | -   urgent payments due to        |
|                                   |     Centrelink delay              |
|                                   | -   urgent payments due to        |
|                                   |     expenses associated with a    |
|                                   |     funeral                       |
|                                   | -   urgent payments due to        |
|                                   |     holiday processing            |

| 2                                 | Delegation for decision making +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The decision to approve or deny   |
|                                   | an urgent payment is made by a    |
|                                   | Service Officer at the APS4 level |
|                                   | or above, Service Delivery        |
|                                   | Partner Service Officers or       |
|                                   | specific Smart Centre staff.      |
|                                   | These are staff fully trained in  |
|                                   | urgent payment processing. This   |
|                                   | applies to Smart Centre           |
|                                   | telephony, service centre and     |
|                                   | processing team staff.            |
|                                   |                                   |
|                                   | During periods of holiday         |
|                                   | processing only, when a request   |
|                                   | is made for an urgent payment due |
|                                   | to departmental delay, this       |
|                                   | decision can be made at the APS3  |
|                                   | level.                            |
|                                   |                                   |
|                                   | Staff assessing urgent payments   |
|                                   | must first complete all urgent    |
|                                   | payment training.                 |
|                                   |                                   |
|                                   | Staff not trained in assessing    |
|                                   | urgent payment requests (or for   |
|                                   | FHA customers):                   |
|                                   |                                   |
|                                   | -   cold transfer the customer    |
|                                   |     via Services Australia        |
|                                   |     Workspace to the urgent       |
|                                   |     payment queue for the payment |
|                                   |     type (or the Farmer           |
|                                   |     Assistance hotline)           |
|                                   | -   document the customer record  |
|                                   |     with any details obtained     |
|                                   |     during the contact            |
|                                   |                                   |
|                                   | If the customer has a Managed     |
|                                   | Service Plan (MSP), the details   |
|                                   | of the MSP may influence who the  |
|                                   | decision maker will be. For       |
|                                   | example, customers with a full or |
|                                   | partial phone restriction in      |
|                                   | place should be transferred to    |
|                                   | their One Main Contact (OMC) or   |
|                                   | referred to the Personalised      |
|                                   | Services Officer (PSSO) if        |
|                                   | available, or advised to contact  |
|                                   | as per the MSP arrangements.      |
|                                   |                                   |
|                                   | It may be appropriate for service |
|                                   | centre or processing team staff   |
|                                   | to assess an urgent payment       |
|                                   | request. Examples include when a  |
|                                   | customer:                         |
|                                   |                                   |
|                                   | is being assisted by a staff      |
|                                   | member on another matter (such as |
|                                   | restoration of payment) and the   |
|                                   | customer requests an urgent       |
|                                   | payment                           |
|                                   |                                   |
|                                   | presents as distressed or         |
|                                   | vulnerable and it would not be    |
|                                   | appropriate to put them on the    |
|                                   | phone                             |
|                                   |                                   |
|                                   | attends a service centre to       |
|                                   | request an urgent payment and     |
|                                   | they have a Managed Service Plan  |
|                                   | (MSP) in place on their record    |
|                                   | prohibiting them from attending   |
|                                   | face-to-face and a one-off        |
|                                   | variation has not been approved   |
|                                   |                                   |
|                                   | -   An authorised delegate should |
|                                   |     direct the customer to leave  |
|                                   |     the premises and to contact   |
|                                   |     their OMC/PSSO as per the MSP |
|                                   |     arrangements. Tell the        |
|                                   |     customer's OMC/PSSO of the    |
|                                   |     contact so they can follow up |
|                                   |     with the customer. For more   |
|                                   |     information, see Customer not |
|                                   |     complying with am MSP         |
|                                   | -   If the MSP is managed by      |
|                                   |     Personalised Services, the    |
|                                   |     Service Officer must contact  |
|                                   |     the Personalised Services     |
|                                   |     phone queue                   |

| 3                                 | Customers on weekly payments +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers cannot receive an       |
|                                   | urgent payment of their income    |
|                                   | support payment while they are on |
|                                   | a weekly payment cycle. They also |
|                                   | cannot receive an urgent payment  |
|                                   | from any other eligible payment   |
|                                   | they receive.                     |
|                                   |                                   |
|                                   | Any request for an urgent payment |
|                                   | should be rejected. Customers     |
|                                   | cannot be removed from a weekly   |
|                                   | payment cycle to allow the urgent |
|                                   | payment to be made and then       |
|                                   | placed back on a weekly payment   |
|                                   | cycle in the same pay period. A   |
|                                   | Service Officer trained in weekly |
|                                   | payments should talk to the       |
|                                   | customer before removing them     |
|                                   | from weekly payments. See         |
|                                   | Assessing, coding and reviewing   |
|                                   | weekly payments.                  |
|                                   |                                   |
|                                   | Customers issued a hardship       |
|                                   | advance or urgent payment in the  |
|                                   | current entitlement period cannot |
|                                   | start on the weekly payment cycle |
|                                   | until the current period has      |
|                                   | ended.                            |

| 6                                 | Payment methods + Read more ...   |
|                                   |                                   |
|                                   | Urgent payments can be delivered  |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   priority direct credit via    |
|                                   |     the New Payments Platform     |
|                                   |     (NPP) or Real Time Gross      |
|                                   |     Settlement (RTGS), or         |
|                                   | -   direct credit (standard       |
|                                   |     processing)                   |
|                                   | -   Electronic Benefits Transfer  |
|                                   |     (EBT) card only if all other  |
|                                   |     payment options are not       |
|                                   |     suitable or will place the    |
|                                   |     customer in further hardship  |
|                                   |                                   |
|                                   | Use NPP and RTGS only when the    |
|                                   | customer has had a regular        |
|                                   | payment issued to the payment     |
|                                   | destination (except for Crisis    |
|                                   | Payment and AGDRP). Customers     |
|                                   | will not be able to use these     |
|                                   | services if they changed their    |
|                                   | payment destination before        |
|                                   | requesting the urgent payment and |
|                                   | have not yet received a regular   |
|                                   | payment into this account.        |
|                                   |                                   |
|                                   | NPP is the preferred method of    |
|                                   | delivery for urgent payments.     |
|                                   | Payment by RTGS or direct credit  |
|                                   | will delay the customer's access  |
|                                   | to the payment by 1 to 2 business |
|                                   | days.                             |
|                                   |                                   |
|                                   | When offering RTGS, make sure the |
|                                   | customer understands that the     |
|                                   | bank may charge a fee for         |
|                                   | processing the payment and        |
|                                   | timeframes vary depending on the  |
|                                   | bank. Services Australia does not |
|                                   | have details of timeframes or     |
|                                   | fees charged.                     |
|                                   |                                   |
|                                   | Rural Agency/Remote Service       |
|                                   | Centres do not have access to EBT |
|                                   | cards. Do not refer customer or   |
|                                   | their representative to these     |
|                                   | sites to collect a payment. Due   |
|                                   | to the remote location of these   |
|                                   | sites, the method must be NPP or  |
|                                   | direct credit.                    |
|                                   |                                   |
|                                   | Farm Household Allowance (FHA)    |
|                                   | and Status Resolution Support     |
|                                   | Services (SRSS) customers can     |
|                                   | only be paid an urgent payment    |
|                                   | via RTGS or direct Credit.        |
|                                   |                                   |
|                                   | If a customer has a Managed       |
|                                   | Service Plan (MSP) in place on    |
|                                   | their record, consider this when  |
|                                   | choosing how to pay an urgent     |
|                                   | payment. If the customer is       |
|                                   | prohibited from attending a       |
|                                   | service centre, other methods of  |
|                                   | delivery such as direct credit,   |
|                                   | RTGS or payment nominee may be    |
|                                   | more appropriate.                 |

| 7                                 | Customers with a history of       |
|                                   | urgent payment requests + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This may indicate that a customer |
|                                   | is having trouble managing their  |
|                                   | finances, or there are other      |
|                                   | ongoing issues. A holistic        |
|                                   | approach to servicing will        |
|                                   | support customers to manage their |
|                                   | finances better.                  |
|                                   |                                   |
|                                   | When a customer has a history of  |
|                                   | requesting urgent payments, make  |
|                                   | sure they are not being placed in |
|                                   | a cycle of ongoing hardship,      |
|                                   | unable to meet their ongoing      |
|                                   | expenses. If an urgent payment    |
|                                   | would mean the customer cannot    |
|                                   | meet their regular expenses and   |
|                                   | would increase their financial    |
|                                   | hardship, the request can be      |
|                                   | refused. The history could        |
|                                   | include multiple requests for     |
|                                   | each type of urgent payment.      |
|                                   |                                   |
|                                   | Customers are limited to one      |
|                                   | urgent payment request            |
|                                   | (self-service or staff assisted), |
|                                   | and one reconsideration of the    |
|                                   | decision on the same day. After   |
|                                   | this, all requests are to be      |
|                                   | rejected.                         |
|                                   |                                   |
|                                   | Explore other assistance options  |
|                                   | with the customer. Customers      |
|                                   | should be highly encouraged to    |
|                                   | access these services to manage   |
|                                   | finances better. The Resources    |
|                                   | page has a link for examples of   |
|                                   | phrasing and questioning to       |
|                                   | support unfavourable discussions  |
|                                   | with customers.                   |
|                                   |                                   |
|                                   | If the customer does not accept   |
|                                   | this offer, discuss with Team     |
|                                   | Leader to explore other options   |
|                                   | to provide holistic service.      |
|                                   |                                   |
|                                   | Customers who have a history of   |
|                                   | repeated urgent payment requests  |
|                                   | should be considered for a        |
|                                   | Managed Service Plan (MSP). The   |
|                                   | history can include urgent        |
|                                   | payments because of expenses      |
|                                   | associated with a funeral. These  |
|                                   | provide extra support when there  |
|                                   | are vulnerabilities or barriers.  |
