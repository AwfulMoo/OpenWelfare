Re-referring debts to an ECA

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Has the debt previously been      |
|                                   | referred to an ECA? + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the Collection Agent (OPMC) |
|                                   | screen.                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure does not apply  |
+-----------------------------------+-----------------------------------+
| 2                                 | Update the re-referral indicator  |
|                                   | and ECA on the OPMC screen + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Do not send a debt to the same    |
|                                   | ECA where it was previously.      |
|                                   | Re-refer the debt to a different  |
|                                   | ECA. Re-referral to the same ECA  |
|                                   | would only be appropriate in      |
|                                   | unusual circumstances. An example |
|                                   | is recalling a debt to correct a  |
|                                   | system problem.                   |
|                                   |                                   |
|                                   | Check the OPMC screen. Make note  |
|                                   | of the previous ECA in the        |
|                                   | Collection agent: field.          |
|                                   |                                   |
|                                   | If the customer has more than one |
|                                   | debt, refer each debt separately  |
|                                   | to the ECA.                       |
|                                   |                                   |
|                                   | Go to the OPMC screen             |
|                                   |                                   |
|                                   | To complete:                      |
|                                   |                                   |
|                                   | -   Re-referral: field 'Y'        |
|                                   | -   Press [Enter]                 |
|                                   | -   Re-referral Collection agent: |
|                                   |     field - will populate with    |
|                                   |     the appropriate ECA code. If  |
|                                   |     the code is to be             |
|                                   |     overwritten, use field help   |
|                                   |     ('?') to select the other ECA |
|                                   |     code                          |
|                                   | -   Press [Enter]                 |
|                                   |                                   |
|                                   | Record all details on a DOC       |
