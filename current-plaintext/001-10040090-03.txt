|                                   |                                   |
|                                   | An automatic CCA should be        |
|                                   | rejected when:                    |
|                                   |                                   |
|                                   | -   the job seeker is currently   |
|                                   |     serving an Unemployment       |
|                                   |     Non-Payment Period (UNPP) or  |
|                                   |     other serious failure period  |
|                                   | -   a determination on            |
|                                   |     persistence would not be      |
|                                   |     possible as the job seeker's  |
|                                   |     record is cancelled           |
|                                   | -   the CCA was generated within  |
|                                   |     a period of conditional       |
|                                   |     waiver, or                    |
|                                   | -   there has been a nationally   |
|                                   |     declared disaster and the CCA |
|                                   |     cannot be conducted           |
|                                   |                                   |
|                                   | A CCA should not be rejected or   |
|                                   | finalised without penalty simply  |
|                                   | because the person is in a        |
|                                   | Compliance Activity.              |
|                                   |                                   |
|                                   | Failures lodged after a CCA has   |
|                                   | commenced should only be          |
|                                   | investigated if the event date of |
|                                   | the failure is later than the     |
|                                   | date the CCA is finalised.        |
