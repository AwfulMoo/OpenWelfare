feeding supplement events for aged care services.

Assessment of the oxygen and/or enteral feeding supplement form (AC011)

Table 2: This table describes how Service Officers determine the rate of
oxygen and/or enteral feeding supplement.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessing the supplement payment  |
|                                   | rates + Read more ...             |
|                                   |                                   |
|                                   | There are 2 payment rates of the  |
|                                   | oxygen and/or enteral feeding     |
|                                   | supplement:                       |
|                                   |                                   |
|                                   | -   The standard oxygen and/or    |
|                                   |     enteral feeding supplement    |
|                                   | -   The higher oxygen and/or      |
|                                   |     enteral feeding supplement    |
|                                   |                                   |
|                                   | The higher oxygen and/or enteral  |
|                                   | feeding supplement is payable     |
|                                   | when the cost of administering is |
|                                   | 125% or more than the standard    |
|                                   | rate and tax invoices covering a  |
|                                   | 3 month period have been          |
|                                   | accompanied for the higher oxygen |
|                                   | rate with the application form.   |
|                                   |                                   |
|                                   | For the higher oxygen,            |
|                                   |                                   |
|                                   | -   (Concentrator hire + Cylinder |
|                                   |     delivery cost + Cylinder      |
|                                   |     hire + Kit hire)              |
|                                   |                                   |
|                                   | For the higher enteral feeding,   |
|                                   |                                   |
|                                   | -   [(mls per day/Tin size) * Tin |
|                                   |     cost] + Giving set +          |
|                                   |     Flexitainer + Hire of pump    |
|                                   |     cost                          |
|                                   |                                   |
|                                   | Service Officers must manually    |
|                                   | calculate the payable amount from |
|                                   | the application form by using     |
|                                   | above formula.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Higher and standard rate + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Are the actual costs greater than |
|                                   | or equal to 125% of the standard  |
|                                   | rate and have tax invoices been   |
|                                   | included for the higher oxygen    |
|                                   | supplement?                       |
|                                   |                                   |
|                                   | Yes, the higher oxygen and/or     |
|                                   | enteral feeding supplement is     |
|                                   | payable                           |
|                                   |                                   |
|                                   | -   Service Officers must         |
|                                   |     manually add up all cost      |
|                                   |     items from the tax invoices   |
|                                   |     (covering a 3 month period)   |
|                                   |     and calculate the daily cost  |
|                                   |     for the higher oxygen         |
|                                   |     supplement for each month     |
|                                   | -   The higher enteral feeding    |
|                                   |     supplement costings are       |
|                                   |     required on a daily basis     |
|                                   |                                   |
|                                   | No, the standard oxygen and/or    |
|                                   | enteral feeding supplement is     |
|                                   | payable                           |
|                                   |                                   |
|                                   | -   If the total cost is less     |
|                                   |     than the higher rate, the     |
|                                   |     Service Officers must approve |
|                                   |     standard rate only            |
|                                   |                                   |
|                                   | Note: the current and historical  |
|                                   | rates of the oxygen and enteral   |
|                                   | feeding supplement are published  |
|                                   | on the Department of Health’s     |
|                                   | website. The Resources page       |
|                                   | contains a link.                  |

|                                   | system + Read more ...            |

|                                   |     feeding supplement events in  |
|                                   |     the ACSP, see the Using the   |
|                                   |     Aged Care Staff Portal to     |
|                                   |     manage enteral feeding        |
|                                   |     supplement events table       |
|                                   | -   home care oxygen events in    |
|                                   |     ACMPS, go to Table 3          |

|                                   |     supplement events in ACMPS,   |
|                                   |     go to Table 4                 |
