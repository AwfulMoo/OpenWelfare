Preparing for Initial Assessment interview

Table 1: This table outlines the steps required before proceeding with
the Initial Assessment interview for VIM.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Is the customer eligible for      |
|                                   | Voluntary Income Management       |
|                                   | (VIM)? + Read more ...            |
|                                   |                                   |
|                                   | To be eligible for VIM, customers |
|                                   | must meet all of the following    |
|                                   | criteria:                         |
|                                   |                                   |
|                                   | -   receive a Category H welfare  |
|                                   |     payment                       |
|                                   | -   reside in a declared Income   |
|                                   |     Management area               |
|                                   | -   not have an excluded payment  |
|                                   |     nominee for Income Management |
|                                   |     purposes                      |
|                                   | -   not be on a higher measure in |
|                                   |     the hierarchy of IM measures  |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Voluntary Income Management       |
|                                   | eligibility.                      |
|                                   |                                   |
|                                   | Does the customer meet all        |
|                                   | eligibility criteria?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Table 6             |
+-----------------------------------+-----------------------------------+
| 2                                 | Is the customer in receipt of     |
|                                   | ABSTUDY? + Read more ...          |
|                                   |                                   |
|                                   | 100% of ABSTUDY (with Living      |
|                                   | Allowance or Pensioner Education  |
|                                   | Supplement (PES)) payments are    |
|                                   | Income Managed. For more          |
|                                   | information, see Voluntary Income |
|                                   | Management eligibility            |
|                                   |                                   |
|                                   | Is the customer in receipt of     |
|                                   | ABSTUDY when it includes an       |
|                                   | amount identified as Living       |
|                                   | Allowance?                        |
|                                   |                                   |
|                                   | Yes, ensure the customer is aware |
|                                   | that 100% of this payment will be |
|                                   | Income Managed.                   |
|                                   |                                   |
|                                   | -   If the customer wants to      |
|                                   |     proceed with their VIM        |
|                                   |     assessment, go to Step 3      |
|                                   | -   If not, procedure ends here   |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Initial assessment interview must |
|                                   | only be conducted by              |
|                                   | appropriately trained Service     |
|                                   | Officers + Read more ...          |
|                                   |                                   |
|                                   | Service Officers not trained in   |
|                                   | IM must cold transfer customers   |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   IM line                       |
|                                   | -   Indigenous Services line (if  |
|                                   |     appropriate), or              |
|                                   | -   Multilingual call (if         |
|                                   |     appropriate).                 |
|                                   |     See Resources for a link to   |
|                                   |     the National Transfer         |
|                                   |     numbers. Procedure ends here  |
|                                   |                                   |
|                                   | IM trained Service Officers, go   |
|                                   | to Step 4                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Initial Assessment Interview +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers must have an Initial    |
|                                   | Assessment interview before       |
|                                   | commencing VIM.                   |
|                                   |                                   |
|                                   | This interview is to:             |
|                                   |                                   |
|                                   | -   explain the terms and         |
|                                   |     conditions of VIM             |
|                                   | -   explain how IM works          |
|                                   | -   determine how income managed  |
|                                   |     funds can be best used to     |
|                                   |     meet the priority needs of    |
|                                   |     the customer and their        |
|                                   |     family, and                   |
|                                   | -   provide information about     |
|                                   |     other services which may      |
|                                   |     assist them, including        |
|                                   |     Centrepay                     |
|                                   |                                   |
|                                   | If the customer requires an       |
|                                   | interpreter, only continue with   |
|                                   | the Initial Assessment interview  |
|                                   | when an interpreter is present.   |
|                                   | See Booking an interpreter for an |
|                                   | appointment.                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Determine if Voluntary Income     |
|                                   | Management (VIM) is suitable for  |
|                                   | the customer + Read more ...      |
|                                   |                                   |
|                                   | To determine if VIM is            |
|                                   | appropriate, advise the customer: |
|                                   |                                   |
|                                   | -   how IM works                  |
|                                   | -   when they are exited from an  |
|                                   |     agreement, a 21 day waiting   |
|                                   |     period will apply from the    |
|                                   |     date they are exited, before  |
|                                   |     any new agreement can be      |
|                                   |     entered into, and             |
|                                   | -   they can only apply for VIM   |
|                                   |     four times in any 12 month    |
|                                   |     period                        |
|                                   |                                   |
|                                   | Consider the customer's net       |
|                                   | payment when discussing options   |
|                                   | with the customer.                |
|                                   |                                   |
|                                   | If the customer is receiving an   |
|                                   | insufficient amount of income     |
|                                   | support payment, VIM may not      |
|                                   | benefit the customer. For         |
|                                   | example, a customer may only      |
|                                   | receive income managed funds of   |
|                                   | $50 due to earnings and the only  |
|                                   | expense that can be covered is    |
|                                   | electricity. In this case,        |
|                                   | Centrepay would be a better       |
|                                   | choice for the customer.          |
|                                   |                                   |
|                                   | Discuss alternative options with  |
|                                   | the customer about managing their |
|                                   | money. For example:               |
|                                   |                                   |
|                                   | -   Have they considered          |
|                                   |     Centrepay deductions?         |
|                                   | -   Have they considered a        |
|                                   |     payment nominee?              |
|                                   | -   Have they considered          |
|                                   |     participating in Financial    |
|                                   |     Wellbeing and Capability      |
|                                   |     Activities or financial       |
|                                   |     counselling?                  |
|                                   | -   Tell the customer they can    |
|                                   |     visit                         |
|                                   |     servi                         |
|                                   | cesaustralia.gov.au/managingmoney |
|                                   |     for useful information and    |
|                                   |     resources                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Provide information to customer + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Before progressing with the       |
|                                   | interview, provide the customer   |
|                                   | with an overview of a VIM         |
|                                   | Agreement, including:             |
|                                   |                                   |
|                                   | -   the mandatory 13 week period  |
|                                   |     of VIM                        |
|                                   | -   how they can request to end   |
|                                   |     IM any time after 13 weeks,   |
|                                   |     or                            |
|                                   | -   the agency can end IM if they |
|                                   |     lose eligibility or it is no  |
|                                   |     longer feasible               |
|                                   | -   review any payment nominee    |
|                                   |     arrangement to determine if   |
|                                   |     it is still in the customer's |
|                                   |     best interests. See Reviewing |
|                                   |     nominee arrangements          |
|                                   |                                   |
|                                   | Does the customer wish to         |
|                                   | participate in Voluntary Income   |
|                                   | Management (VIM)?                 |
|                                   |                                   |
|                                   | Yes, go to Table 2, Step 1        |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   record details on a DOC       |
|                                   | -   if applicable, code           |
|                                   |     Centrepay, and/or             |
|                                   | -   if requested, provide the     |
|                                   |     customer with contact details |
|                                   |     of their nearest Financial    |
|                                   |     Wellbeing and Capacity        |
|                                   |     Activity provider             |
|                                   | -   procedure ends here           |

Reviewing a customer's payment nominee arrangements

Table 2: This table contains information on how to review a customer's
payment nominee arrangement for IM purposes.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check customer's nominee status + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Go to the Nominee Link Summary    |
|                                   | (NOLS) screen to see if the       |
|                                   | customer has or is a payment      |
|                                   | nominee (NOP).                    |
|                                   |                                   |
|                                   | Note: Correspondence nominee      |
|                                   | (NOC) arrangements do not exclude |
|                                   | a customer from IM.               |
|                                   |                                   |
|                                   | Is there a payment nominee        |
|                                   | arrangement in place?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Table 3 Step 1      |
+-----------------------------------+-----------------------------------+
| 2                                 | Review Nominee arrangement + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers identified/referred for |
|                                   | IM should have their payment      |
|                                   | nominee relationship reviewed.    |
|                                   | For more information, see Nominee |
|                                   | arrangements under Income         |
|                                   | Management.                       |
|                                   |                                   |
|                                   | Service Officers are to review    |
|                                   | nominee arrangements to confirm   |
|                                   | the arrangement is appropriate    |
|                                   | and in the customer's best        |
|                                   | interests.                        |
|                                   |                                   |
|                                   | Is the payment nominee            |
|                                   | arrangement to continue?          |
|                                   |                                   |
|                                   | Note: if the customer is a        |
|                                   | payment nominee for another       |
|                                   | person, tell the customer the     |
|                                   | person, for whom they are a       |
|                                   | payment nominee, may now be       |
|                                   | eligible for IM. See Nominee      |
|                                   | arrangements under Income         |
|                                   | Management.                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, cancel the nominee        |
|                                   |     arrangement. Go to Table 3    |
|                                   |     Step 2                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Does the customer have an         |
|                                   | excluded payment nominee? + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Yes, the customer is not      |
|                                   |     eligible for IM if they have  |
|                                   |     an excluded payment nominee   |
|                                   |     for Income Management         |
|                                   |     purposes. Go to Step 4        |
|                                   | -   No, tell the customer their   |
|                                   |     payment nominee must be       |
|                                   |     present for the Initial       |
|                                   |     Assessment interview as they  |
|                                   |     are responsible for managing  |
|                                   |     their funds. Go to Step 5     |
+-----------------------------------+-----------------------------------+
| 4                                 | Excluded payment nominee + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Tell the customer their       |
|                                   |     payments will not be income   |
|                                   |     managed due to their payment  |
|                                   |     nominee arrangement. However, |
|                                   |     if the arrangement changes,   |
|                                   |     they may become eligible for  |
|                                   |     IM                            |
|                                   | -   Create a DOC on the           |
|                                   |     customer's record saying      |
|                                   |     'Customer has an excluded     |
|                                   |     payment nominee and is not    |
|                                   |     eligible for Income           |
|                                   |     Management'                   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Is the payment nominee present? + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Yes, see Table 3 Step 1       |
|                                   | -   No, book another appointment  |
|                                   |     at a suitable time for the    |
|                                   |     payment nominee as it is      |
|                                   |     mandatory for the payment     |
|                                   |     nominee to attend an Initial  |
|                                   |     Assessment interview. It is   |
|                                   |     preferred, although not       |
|                                   |     compulsory, for the customer  |
|                                   |     to attend the interview to    |
|                                   |     ensure they are aware of how  |
|                                   |     IM will affect them.          |
|                                   |     Procedure ends here           |

Conducting the Initial Assessment interview

Table 3: This table contains steps to conduct the Initial Assessment
interview.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Conduct Initial Assessment        |
|                                   | interview + Read more ...         |
|                                   |                                   |
|                                   | During the interview, the         |
|                                   | following must be covered:        |
|                                   |                                   |
|                                   | Explain IM and how delivery of    |
|                                   | their payments will change        |
|                                   |                                   |
|                                   | Explain what happens if the       |
|                                   | customer moves out of an Income   |
|                                   | Management area                   |
|                                   |                                   |
|                                   | Explain how income managed money  |
|                                   | is accessed. For more             |
|                                   | information, see Coding Income    |
|                                   | Management expenses               |
|                                   |                                   |
|                                   | Explain what income managed money |
|                                   | can be used for, that is, School  |
|                                   | Meals Program (if applicable),    |
|                                   | food, rent/housing, utilities,    |
|                                   | clothing, medical needs. See      |
|                                   | Determining a person's priority   |
|                                   | needs for Income Management       |
|                                   |                                   |
|                                   | Explain what goods cannot be      |
|                                   | purchased from income managed     |
|                                   | funds, that is, prohibited goods  |
|                                   | and services - alcohol, home brew |
|                                   | concentrate, home brew kits,      |
|                                   | tobacco products, pornographic    |
|                                   | material or gambling              |
|                                   |                                   |
|                                   | Explain how money can be          |
|                                   | allocated to their BasicsCard:    |
|                                   |                                   |
|                                   | -   Residual Payment method       |
|                                   |     (default), or                 |
|                                   | -   they choose the Partial       |
|                                   |     Payment method, and           |
|                                   | -   immediate transfers (when     |
|                                   |     Income Management funds are   |
|                                   |     available)                    |
|                                   |                                   |
|                                   | If applicable, discuss converting |
|                                   | existing Centrepay deductions to  |
|                                   | an income managed expense.        |
|                                   | Explain the impact of IM on any   |
|                                   | ongoing Centrepay deductions      |
|                                   |                                   |
|                                   | Review and discuss any existing   |
|                                   | Rent Deduction Scheme payment and |
|                                   | whether the customer would like   |
|                                   | to pay their State Housing        |
|                                   | Authority (SHA) rent from:        |
|                                   |                                   |
|                                   | -   their income managed funds    |
|                                   | -   their discretionary funds, or |
|                                   | -   a combination of both         |
|                                   |                                   |
|                                   | Explain how money can be accessed |
|                                   | for Income Management customers   |
|                                   | accessing funds while travelling  |
|                                   |                                   |
|                                   | Explain how customers can contact |
|                                   | Services Australia, including     |
|                                   | self service, in regards to IM    |
|                                   | and BasicsCard. See Income        |
|                                   | Management and BasicsCard self    |
|                                   | service options. Offer to         |
|                                   | register the customer for         |
|                                   | self-service                      |
|                                   |                                   |
|                                   | Explain how common changes in     |
|                                   | circumstances may affect IM and   |
|                                   | the importance of notifying the   |
|                                   | agency of any changes in their    |
|                                   | expenses                          |
|                                   |                                   |
|                                   | Advise customer of impact on      |
|                                   | expense payments if income        |
|                                   | support payments are reduced due  |
|                                   | to earning from employment or     |
|                                   | other income and the priority     |
|                                   | needs payment order when there    |
|                                   | are insufficient funds to cover   |
|                                   | an expense                        |
|                                   |                                   |
|                                   | Explain how lump sum payments     |
|                                   | such as Advances, arrears and     |
|                                   | Family Tax Benefit (FTB)          |
|                                   | reconciliation payments are 100%  |
|                                   | income managed                    |
|                                   |                                   |
|                                   | Explain what happens if they are  |
|                                   | identified for a higher measure   |
|                                   | of IM and how this may impact     |
|                                   | ongoing eligibility for VIM. See  |
|                                   | Hierarchy and movement between    |
|                                   | Income Management (IM) measures - |
|                                   | Overview                          |
|                                   |                                   |
|                                   | When IM ends, all IM expenses     |
|                                   | stop. They must make arrangements |
|                                   | to continue paying these expenses |
|                                   | themselves, or contact the agency |
|                                   | to discuss alternative payment    |
|                                   | options, such as Centrepay        |
|                                   |                                   |
|                                   | Advise the customer of their      |
|                                   | right to review Income Management |
|                                   | decisions                         |
|                                   |                                   |
|                                   | Note: if during the interview,    |
|                                   | the Service Officer identifies    |
|                                   | vulnerability factors (on the     |
|                                   | record or verbally), offer a      |
|                                   | social worker referral as part of |
|                                   | normal business processes.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Discuss with customer how         |
|                                   | Priority Needs will be met + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Regular expense allocations will  |
|                                   | be established to meet their      |
|                                   | priority needs and those of their |
|                                   | family.                           |
|                                   |                                   |
|                                   | Discuss with the customer what    |
|                                   | ongoing expenses they have (for   |
|                                   | example, rent, food, utility      |
|                                   | bills etc.). If the expense meets |
|                                   | a priority need, this can be paid |
|                                   | from their income managed funds.  |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Determining a person's priority   |
|                                   | needs for Income Management.      |
+-----------------------------------+-----------------------------------+
| 3                                 | Voluntary Income Management (VIM) |
|                                   | Assessment workflow + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Launch the Voluntary IM           |
|                                   | Assessment workflow.              |
|                                   |                                   |
|                                   | Enter the following information   |
|                                   | on the Voluntary Agreement        |
|                                   | Details page:                     |
|                                   |                                   |
|                                   | Agreement                         |
|                                   |                                   |
|                                   | Receipt Date: field will display  |
|                                   | as today's date. This should only |
|                                   | be changed where the IM interview |
|                                   | was held on a previous date       |
|                                   | Note: although the interview may  |
|                                   | be coded for a previous date the  |
|                                   | date of effect will be the        |
|                                   | Entitlement Period End Date       |
|                                   | (EPED) following the assessment   |
|                                   | being completed.                  |
|                                   |                                   |
|                                   | Start date: field is protected    |
|                                   | and defaults to the day the       |
|                                   | workflow is being completed       |
|                                   |                                   |
|                                   | Status: Select 'Active' from the  |
|                                   | drop down list                    |
|                                   |                                   |
|                                   | Customer referred by: Select the  |
|                                   | appropriate option from the drop  |
|                                   | down list:                        |
|                                   |                                   |
|                                   | -   Non State authority           |
|                                   | -   Not referred                  |
|                                   | -   State authority               |
|                                   |                                   |
|                                   | If 'Non State authority' is       |
|                                   | selected, select the              |
|                                   | Organisation/office from the drop |
|                                   | down list:                        |
|                                   |                                   |
|                                   | -   Financial Management Program  |
|                                   |     Services                      |
|                                   | -   Housing Authorities           |
|                                   | -   Welfare Authorities           |
|                                   |                                   |
|                                   | If 'State authority' is selected, |
|                                   | select State/territory and        |
|                                   | Organisation/office from the drop |
|                                   | down menu                         |
|                                   |                                   |
|                                   | Northern Territory:               |
|                                   |                                   |
|                                   | -   DCF Head Office               |
|                                   | -   DCF Katherine                 |
|                                   |                                   |
|                                   | Western Australia:                |
|                                   |                                   |
|                                   | -   Department for Child          |
|                                   |     Protection                    |
|                                   |                                   |
|                                   | Payment nominee details           |
|                                   |                                   |
|                                   | This information only presents    |
|                                   | when the customer is a payment    |
|                                   | nominee for another person.       |
|                                   |                                   |
|                                   | -   Principal/s: This defaults to |
|                                   |     the name/s of any person/s    |
|                                   |     for whom the customer is a    |
|                                   |     payment nominee               |
|                                   | -   Customer will continue acting |
|                                   |     in each principal's best      |
|                                   |     interests/spending payments   |
|                                   |     for their benefit:            |
|                                   | -   Customer agrees? Select Yes   |
|                                   |     or No radio button as         |
|                                   |     appropriate. If:              |
|                                   | -   Yes, explain that the         |
|                                   |     principal may become eligible |
|                                   |     for IM in their own right     |
|                                   | -   No, refer to reviewing        |
|                                   |     nominee arrangements.         |
|                                   |     Customers are not prevented   |
|                                   |     from entering into a VIM      |
|                                   |     Agreement when they no longer |
|                                   |     wish to act as a payment      |
|                                   |     nominee, however Service      |
|                                   |     Officers must ensure all      |
|                                   |     appropriate action is taken   |
|                                   |     when the arrangement is to be |
|                                   |     cancelled                     |
|                                   |                                   |
|                                   | For more information, see Nominee |
|                                   | arrangements under Income         |
|                                   | Management.                       |
|                                   |                                   |
|                                   | Select Continue.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Continue workflow: Voluntary      |
|                                   | Agreement Details - Interpreter   |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Interpreter Requirements          |
|                                   |                                   |
|                                   | Details of the interpreter        |
|                                   | requirements must be recorded for |
|                                   | interview purposes.               |
|                                   |                                   |
|                                   | Where no information is recorded  |
|                                   | on the Other Contact Details      |
|                                   | (OCD) screen, the Service Officer |
|                                   | must exit the assessment workflow |
|                                   | and update the OCD screen before  |
|                                   | proceeding with the interview.    |
|                                   | See Booking an interpreter for an |
|                                   | appointment.                      |
|                                   |                                   |
|                                   | Customer's Spoken Language: This  |
|                                   | will default to the interpreter   |
|                                   | language recorded on the OCD      |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Interpreter Required: This will   |
|                                   | default to:                       |
|                                   |                                   |
|                                   | -   Yes if the OCD screen         |
|                                   |     indicates an interpreter is   |
|                                   |     required                      |
|                                   | -   No if interpreter language is |
|                                   |     recorded as English, and      |
|                                   |     cannot be changed. If it is   |
|                                   |     determined an interpreter is  |
|                                   |     needed for the interview, the |
|                                   |     Service Officer must exit the |
|                                   |     workflow and update the OCD   |
|                                   |     screen as appropriate         |
|                                   |                                   |
|                                   | Is an interpreter being used?     |
|                                   | This will only present if the     |
|                                   | customer's language is not        |
|                                   | English. Select Yes, No, Not      |
|                                   | Available or Not Required from    |
|                                   | the dropdown list as appropriate. |
|                                   |                                   |
|                                   | Reason: If an interpreter is not  |
|                                   | available, not being used or not  |
|                                   | required, the reason the          |
|                                   | assessment continued is to be     |
|                                   | selected from the applicable      |
|                                   | dropdown list.                    |
|                                   |                                   |
|                                   | If Yes is selected, no further    |
|                                   | information is required. Select   |
|                                   | Continue and go to Step 5         |
|                                   |                                   |
|                                   | If No is selected, select the     |
|                                   | applicable reason from the        |
|                                   | dropdown menu:                    |
|                                   |                                   |
|                                   | -   Appointment not booked        |
|                                   | -   Conflict of interest          |
|                                   | -   Customer declined interpreter |
|                                   | -   Customer wishes to use agent  |
|                                   |     or family member              |
|                                   | -   Deaf or hearing impairment    |
|                                   | -   Services Australia does not   |
|                                   |     have an interpreter who       |
|                                   |     speaks the customer's         |
|                                   |     language                      |
|                                   |                                   |
|                                   | If Not available is selected,     |
|                                   | select the applicable reason from |
|                                   | the dropdown menu:                |
|                                   |                                   |
|                                   | -   Interpreter did not attend    |
|                                   |     interview                     |
|                                   | -   Interpreter service for that  |
|                                   |     language not available        |
|                                   | -   Same gender interpreter not   |
|                                   |     available                     |
|                                   |                                   |
|                                   | If Not required is selected,      |
