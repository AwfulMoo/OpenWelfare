Suspension process for YA

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Decision to suspend payment +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Confirm the customer’s continuing |
|                                   | eligibility for payment if they   |
|                                   | have:                             |
|                                   |                                   |
|                                   | -   changed course or             |
|                                   |     institutions, or              |
|                                   | -   reduced their study load      |
|                                   |                                   |
|                                   | See Changing or correcting course |
|                                   | of study for Youth Allowance,     |
|                                   | Austudy and Pensioner Education   |
|                                   | Supplement students.              |
|                                   |                                   |
|                                   | Only suspend Youth Allowance (YA) |
|                                   | payments if:                      |
|                                   |                                   |
|                                   | -   there is sufficient evidence  |
|                                   |     the customer no longer        |
|                                   |     qualifies for payment or      |
|                                   |     payment is not payable        |
|                                   | -   the period of ineligibility   |
|                                   |     is for a short time. For      |
|                                   |     example, there is reason to   |
|                                   |     believe the customer will     |
|                                   |     resolve non-compliance with   |
|                                   |     any legislative/policy        |
|                                   | -   all required actions such as  |
|                                   |     requesting details and        |
|                                   |     contacting customer are       |
|                                   |     completed                     |
|                                   |                                   |
|                                   | Is the decision to suspend the    |
|                                   | payment correct?                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, record a DOC on why a     |
|                                   |     manual suspension is not      |
|                                   |     required. See Creating        |
|                                   |     documents (Fast Note,         |
|                                   |     Callbacks and ACTDOCs).       |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Suspend payment + Read more ...   |
|                                   |                                   |
|                                   | Update the customer record with   |
|                                   | the relevant details to allow the |
|                                   | system to make an automatic       |
|                                   | determination if possible.        |
|                                   |                                   |
|                                   | If the system does not            |
|                                   | automatically suspend the         |
|                                   | payment, manually suspend the     |
|                                   | payment in the customer’s record. |
|                                   | Go to the Benefit Action (BA)     |
|                                   | screen and code the following     |
|                                   |                                   |
|                                   | Svc Rsn field: YAL for Youth      |
|                                   | Allowance                         |
|                                   |                                   |
|                                   | Action field: SUS'                |
|                                   |                                   |
|                                   | Reason field: code the            |
|                                   | appropriate suspension reason.    |
|                                   | For a list of suspension codes,   |
|                                   | see:                              |
|                                   |                                   |
|                                   | -   Cancellation, suspension and  |
|                                   |     rejection codes for Newstart  |
|                                   |     System (NSS), or              |
|                                   | -   use field help (?) to view    |
|                                   |     options                       |
|                                   |                                   |
|                                   | Effect Date field: select Enter.  |
|                                   | The date of effect will           |
|                                   | automatically enter the date of   |
|                                   | effect of the suspension in the   |
|                                   | Effect Date field. For more       |
|                                   | information, see Suspension of    |
|                                   | payments (CLK)                    |
|                                   |                                   |
|                                   | Notes field and Optional field.   |
|                                   | include any additional details    |
|                                   | relevant to this customer's       |
|                                   | circumstances                     |
|                                   |                                   |
|                                   | Source: Date of Receipt: and      |
|                                   | Action: fields                    |
|                                   |                                   |
|                                   | Issue a manual letter when        |
|                                   | manually actioning a suspension   |
|                                   | activity. For a manual suspension |
|                                   | with a reason of OTH (other)      |
|                                   | issue the Q134 letter in the      |
|                                   | suspension activity. This is to   |
|                                   | prevent an Manual Follow-up (MFU) |
|                                   | activity generating. See Creating |
|                                   | a Q134 letter.                    |
|                                   |                                   |
|                                   | Select This decision has been     |
|                                   | made under the Student Assistance |
|                                   | Act 1973 as the relevant act      |
|                                   |                                   |
|                                   | Finalise the activity on the      |
|                                   | Assessment Results (AR) screen.   |
|                                   | Record details of decision and    |
|                                   | any action taken on a DOC using   |
|                                   | the Suspend Cancel Reason Fast    |
|                                   | Note. Locate the Fast Note via:   |
|                                   |                                   |
|                                   | -   Auto Text                     |
|                                   | -   Category 1: Students, ABSTUDY |
|                                   |     and Apprentices               |
|                                   | -   Category 2: SUS/CAN/RES       |
|                                   | -   Template: SUSP DOV            |
|                                   |                                   |
|                                   | Complete the Fast Note with:      |
|                                   |                                   |
|                                   | -   Decision made, payment type   |
|                                   |     and relevant Act/Guide        |
|                                   |     reference . See Creating      |
|                                   |     documents (Fast Note,         |
|                                   |     Callbacks and ACTDOCs)        |
|                                   | -   Date of effect of decision    |
|                                   | -   Reasons for decision. Include |
|                                   |     reasons for customer not      |
|                                   |     qualifying or why the payment |
|                                   |     is not payable for decisions  |
|                                   |     made under section 80         |
|                                   | -   Follow up action required     |
|                                   |     before restoration can take   |
|                                   |     place                         |

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Send manual advice + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If a manual Q134 or Q999 letter   |
|                                   | is required for a suspension      |
|                                   | activity, a Manual Follow Up      |
|                                   | (MFU) activity generates, usually |
|                                   | overnight. Workload Manager (WLM) |
|                                   | allocates the MFU for action.     |
|                                   |                                   |
|                                   | Has a manual suspension letter    |
|                                   | been issued to the customer at    |
|                                   | suspension?                       |
|                                   |                                   |
|                                   | Yes, finalise the MFU. Go to Step |
|                                   | 2                                 |
|                                   |                                   |
|                                   | No, send a manual letter. Select  |
|                                   | the MFU from the Activity List    |
|                                   | (AL) screen and manually issue a  |
|                                   | Q134 letter. Include in the       |
|                                   | letter:                           |
|                                   |                                   |
|                                   | -   payment type                  |
|                                   | -   decision                      |
|                                   | -   date of effect of the         |
|                                   |     decision                      |
|                                   | -   an explanation of the reason  |
|                                   |     for the decision              |
|                                   | -   an explanation of what they   |
|                                   |     must do to have their payment |
|                                   |     restored and the consequences |
|                                   |     for not doing this. For       |
|                                   |     example, cancellation of      |
|                                   |     payment                       |
|                                   | -   a reasonable time frame to    |
|                                   |     take required action to avoid |
|                                   |     cancellation                  |
|                                   | -   their review and appeal       |
|                                   |     rights                        |
|                                   | -   the relevant section of the   |
|                                   |     Legislation or Policy under   |
|                                   |     which the decision was made   |
+-----------------------------------+-----------------------------------+
| 2                                 | Set up review + Read more ...     |
|                                   |                                   |
|                                   | When a manual suspension occurs,  |
|                                   | set a review date to check:       |
|                                   |                                   |
|                                   | -   if the customer qualifies for |
|                                   |     the payment                   |
|                                   | -   the payment is payable        |
|                                   |                                   |
|                                   | This should mature at the time    |
|                                   | the decision to suspend must be   |
|                                   | re-examined, usually 13 weeks     |
|                                   | later. See How to code and action |
|                                   | a manual review.                  |
|                                   |                                   |
|                                   | Update the DOC on the customer    |
|                                   | record with the date the review   |
|                                   | is due.                           |
|                                   |                                   |
|                                   | Customers need manual             |
|                                   | intervention if they are:         |
|                                   |                                   |
|                                   | -   A vulnerable or at risk       |
|                                   |     customer                      |
|                                   | -   without income support        |
|                                   |     payment for a minimum of 6    |
|                                   |     weeks due to an automatic     |
|                                   |     suspension or cancellation,   |
|                                   |     and                           |
|                                   | -   not overseas permanently      |
|                                   |                                   |
|                                   | Cancel any MFUs generated for     |
|                                   | customers in this situation.      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Person contacts due to            |
|                                   | suspension, or review date is     |
|                                   | reached + Read more ...           |
|                                   |                                   |
|                                   | At the review due date, review    |
|                                   | the person's qualification for    |
|                                   | and payability of the payment.    |
|                                   |                                   |
|                                   | If a customer contacts regarding  |
|                                   | their suspension, check the       |
|                                   | suspension DOC . The customer is  |
|                                   | eligible to have their payment    |
|                                   | restored if:                      |
|                                   |                                   |
|                                   | -   they have already complied    |
|                                   |     with the action required or   |
|                                   |     are able to comply at contact |
|                                   |     they do not need to supply    |
|                                   |     documents                     |
|                                   | -   they do not need to attend    |
|                                   |     the service centre and the    |
|                                   |     payment is one which is able  |
|                                   |     to be restored, and           |
|                                   | -   there is no note on a DOC to  |
|                                   |     prevent restoration           |
|                                   |                                   |
|                                   | Is the customer eligible for      |
|                                   | payment?                          |
|                                   |                                   |
|                                   | -   Yes, restore payment.         |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Unfavourable decision + Read more |
|                                   | ...                               |

|                                   |     appeal rights                 |
