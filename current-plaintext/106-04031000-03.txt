|                                   | calculate the holiday allowance:  |
|                                   |                                   |
|                                   | -   Combine all income, including |
|                                   |     Slovenian, Australian and     |
|                                   |     third country pensions, plus  |
|                                   |     any other income amounts      |
|                                   | -   Using this value, determine   |
|                                   |     the ‘base’ rate of Slovenian  |
|                                   |     holiday allowance that is     |
|                                   |     payable using the Slovenian   |
|                                   |     Holiday Allowance Summary     |
|                                   |     (see Resources for            |
|                                   |     instructions on accessing the |
|                                   |     Summary)                      |
|                                   | -   Proportionalise the holiday   |
|                                   |     allowance amount using the    |
|                                   |     same proportion as Slovenia   |
|                                   |     apply to the base rate of     |
|                                   |     payment, where the base rate  |
|                                   |     of payment is                 |
|                                   |     proportionalised              |
|                                   | -   Divide this proportionalised  |
|                                   |     amount by 12                  |
