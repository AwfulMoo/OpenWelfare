Re-route to the termination queue and terminate incorrectly registered case

This table helps Service Officers with the process in CCMS.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Suspended or active case + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the case is:                   |
|                                   |                                   |
|                                   | -   suspended, go to Step 2       |
|                                   | -   active, go to Step 3          |
+-----------------------------------+-----------------------------------+
| 2                                 | For a suspended case + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Re-route case to termination      |
|                                   | queue in CCMS:                    |
|                                   |                                   |
|                                   | In the Wake up field, select:     |
|                                   |                                   |
|                                   | -   activate from the drop down   |
|                                   |     list in Case is to be         |
|                                   | -   Case termination from the     |
|                                   |     drop down list in New wake up |
|                                   |     step                          |
|                                   | -   update                        |
|                                   |                                   |
|                                   | In the case note box that opens,  |
|                                   | enter detailed notes about the    |
|                                   | reason for termination            |
+-----------------------------------+-----------------------------------+
| 3                                 | For an active case + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | If untrained in case              |
|                                   | terminations:                     |
|                                   |                                   |
|                                   | -   send a ‘Follow up work to be  |
|                                   |     actioned’ email to the Team   |
|                                   |     Leaders mailbox. A trained    |
|                                   |     Service Officer will complete |
|                                   |     the termination in CCMS       |
|                                   | -   add detailed notes about the  |
|                                   |     reason for termination        |
|                                   |                                   |
|                                   | If trained in case terminations,  |
|                                   | complete steps in Terminate       |
|                                   | Medicare Compensation Recovery    |
|                                   | cases                             |
