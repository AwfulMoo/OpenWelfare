BAC and CAC subsidies

Table 1:

+-----------------+-----------------+-----------------+-----------------+
| Measure         | BAC             | CAC             | CAC             |
+-----------------+-----------------+-----------------+-----------------+
| Time period     | 0 – 12 months   | 12 -24 months   | 24 – 36 months  |

| Subsidy         | 50% of wages    | 10% of wages    | 5% of wages     |

|                 | quarter         | quarter         | quarter         |
|                 |                 |                 |                 |
|                 | $28,000 for 12  | $6,000 for 12   | $3,000 for 12   |
|                 | months          | months          | months          |
