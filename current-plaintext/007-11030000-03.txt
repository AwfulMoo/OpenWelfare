| 11                                | Parked payment or debt + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Following the receipt of income   |
|                                   | from the ATO, information about   |
|                                   | reconciliation entitlements and   |
|                                   | outstanding family assistance     |
|                                   | debts may be reported to the ATO. |
|                                   |                                   |
|                                   | FTB reconciliation is occurring   |
|                                   |                                   |
|                                   | If FTB reconciliation is          |
|                                   | occurring following the receipt   |
|                                   | of income for the customer or     |
|                                   | their consenting partner for the  |
|                                   | relevant financial year,          |
|                                   | information about arrears or      |
|                                   | debts for the relevant financial  |
|                                   | year and outstanding FTB/CCS/CCB  |
|                                   | debt amounts for prior financial  |
|                                   | years is reported to the ATO.     |
|                                   |                                   |
|                                   | The ATO may:                      |
|                                   |                                   |
|                                   | -   offset FTB reconciliation     |
|                                   |     top-ups against outstanding   |
|                                   |     taxation debts, and           |
|                                   | -   use tax refunds against       |
|                                   |     outstanding family assistance |
|                                   |     debts                         |
|                                   |                                   |
|                                   | The reconciliation result will be |
|                                   | placed on hold, or parked until   |
|                                   | the ATO has advised offsetting    |
|                                   | action. The ATO response          |
|                                   | transaction is called a wash-up.  |
|                                   |                                   |
|                                   | FTB reconciliation is not         |
|                                   | occurring                         |
|                                   |                                   |
|                                   | If reconciliation is not          |
|                                   | occurring (reconciliation already |
|                                   | COMplete, partner lodgement       |
|                                   | outstanding) as a result of       |
|                                   | actual income being received,     |
|                                   | Services Australia will send a    |
|                                   | transaction to the ATO if the     |
|                                   | customer has:                     |
|                                   |                                   |
|                                   | -   an outstanding FTB            |
|                                   |     non-lodger,                   |
|                                   | -   a qualification or            |
|                                   |     reconciliation debt, or       |
|                                   | -   CCB reconciliation or CCR     |
|                                   |     debt determined after 1 July  |
|                                   |     2006                          |
|                                   |                                   |
|                                   | When a FTB reconciliation result  |
|                                   | or tax offset request is sent to  |
|                                   | the ATO, a future review is       |
|                                   | created on the customer's record. |
|                                   | The review will appear on the FAL |
|                                   | screen and will have a maturity   |
|                                   | date of one year from the date of |
|                                   | registration                      |
|                                   |                                   |
|                                   | FTB payment or FTB reconciliation |
|                                   | debt is parked                    |
|                                   |                                   |
|                                   | It may take the ATO up to 28 days |
|                                   | to send a wash up transaction     |
|                                   | after completing:                 |
|                                   |                                   |
|                                   | -   a FTB reconciliation or       |
|                                   | -   lump sum claim assessment     |
|                                   |                                   |
|                                   | They will send the wash up        |
|                                   | transaction after issuing the     |
|                                   | NOA.                              |
|                                   |                                   |
|                                   | View the date the payment was     |
|                                   | parked on the FAO Reconciliation  |
|                                   | and Claim Transaction (FRCT)      |
|                                   | screen. Key 'T' against the Reco  |
|                                   | Completed line on the FAO         |
|                                   | Reconciliation and Claim Summary  |
|                                   | (FRCS) screen to go to the FRCT   |
|                                   | screen                            |
|                                   |                                   |
|                                   | Check the customer has actually   |
|                                   | received the NOA. If they did not |
|                                   | receive the NOA, tell them that   |
|                                   | the reconciliation of lump sum    |
|                                   | claim assessment will not be      |
|                                   | implemented until:                |
|                                   |                                   |
|                                   | -   the NOA has been issued and   |
|                                   | -   a wash up transaction has     |
|                                   |     been received                 |
|                                   |                                   |
|                                   | If the FRCS screen shows that the |
|                                   | reconciliation has been           |
|                                   | cancelled, go to Step 12          |
|                                   |                                   |
|                                   | Send a request to the ATO Liaison |
