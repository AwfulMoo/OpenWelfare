|                                   |                                   |
|                                   | Note: updates to the partnerâ€™s    |

|                                   |     deselected                    |
|                                   |                                   |
|                                   | Tasks can be shown or hidden by   |
|                                   | selecting the twisty Hide         |
|                                   | Customer's transferable tasks.    |

|                                   | DL/Note will be created with DOC  |
