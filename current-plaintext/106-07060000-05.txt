Acceptance of supporting documentation

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Responsibility of the determining |
|                                   | officer (delegate)                |
|                                   |                                   |
|                                   | The determining officer           |
|                                   | (delegate) must decide whether    |
|                                   | supporting documentation/evidence |
|                                   | provided is acceptable and/or if  |
|                                   | further information should be     |
|                                   | requested. The information        |
|                                   | provided in the examples of       |
|                                   | supporting documents tables is to |
|                                   | be used as a guide in making this |
|                                   | determination.                    |
|                                   |                                   |
|                                   | For assistance with:              |
|                                   |                                   |
|                                   | -   documents not listed - if the |
|                                   |     information being sought is   |
|                                   |     not listed then request the   |
|                                   |     customer to provide all the   |
|                                   |     relevant documentation they   |
|                                   |     have                          |
|                                   | -   unusual cases - follow normal |
|                                   |     escalation procedures as per  |
|                                   |     individual payment processing |
|                                   |     procedures where further      |
|                                   |     information for acceptable    |
|                                   |     documentation in unusual      |
|                                   |     cases is required. For Smart  |
|                                   |     Centre staff, see Service     |
|                                   |     Officer roles when seeking    |
|                                   |     Technical Peer Support (TPS)  |
|                                   |     in Smart Centres              |
|                                   |                                   |
|                                   | Where the supporting              |
|                                   | documentation/evidence supplied   |
|                                   | requires further explanation as   |
|                                   | to why it was accepted (or not    |
|                                   | requested/provided), the          |
|                                   | determining staff member must     |
|                                   | include full details in the       |
|                                   | decision DOC to support that      |
|                                   | decision.                         |

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Aboriginality                     |
|                                   |                                   |
|                                   | The customer's or nominee's       |
|                                   | declaration on the claim form     |
|                                   | will normally be sufficient to    |

|                                   | Australian Aboriginal or Torres   |

+-----------------------------------+-----------------------------------+
| 2                                 | Age                               |
|                                   |                                   |
|                                   | The age of all customers must be  |
|                                   | verified, except for those in     |
|                                   | receipt of family payments,       |
|                                   | Parenting Payment (PP) and Carer  |
|                                   | Allowance (CA). Proof of age is   |
|                                   | different to Proof of Birth (POB) |
|                                   | for identity purposes because     |
|                                   | some, but not all, proof of age   |
|                                   | documents can be used for         |
|                                   | identity.                         |
|                                   |                                   |
|                                   | Service Officers must check that  |
|                                   | the details on the document       |
|                                   | provided match the details        |
|                                   | recorded on the claim and the     |
|                                   | customer record.                  |
|                                   |                                   |
|                                   | Note: a customer’s official date  |
|                                   | of birth (DOB) for Centrelink     |
|                                   | purposes is the date indicated    |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   State or Territory Registrar  |
|                                   |     of Births, Deaths and         |
|                                   |     Marriages (BDM) if born in    |
|                                   |     Australia, or                 |
|                                   | -   Department of Home Affairs if |
|                                   |     born outside Australia        |
|                                   |                                   |
|                                   | Only the dates identified on      |
|                                   | documents issued by these         |
|                                   | agencies can be accepted as the   |
|                                   | official DOB - see Verifying the  |
|                                   | date of birth or age of a         |
|                                   | customer or child.                |
|                                   |                                   |
|                                   | For Assistance for Isolated       |
|                                   | Children (AIC) Scheme claims,     |
|                                   | proof of age is required before   |
|                                   | benefits can commence for         |
|                                   | students who have not qualified   |
|                                   | for AIC in a previous year.       |
|                                   |                                   |
|                                   | Note: if Family Tax Benefit (FTB) |
|                                   | is/has been paid for the student, |
|                                   | then no proof of age is required. |
|                                   |                                   |
|                                   | Otherwise the normal requirement  |
|                                   | is for a customer's:              |
|                                   |                                   |
|                                   | -   Birth certificate, or         |
|                                   | -   Valid Australian passport     |
|                                   |                                   |
|                                   | If neither is available,          |
|                                   | acceptable alternatives include:  |
|                                   |                                   |
|                                   | -   official citizenship or       |
|                                   |     residency papers (such as an  |
|                                   |     Australian visa or ImmiCard)  |
|                                   |     which show the customer’s     |
|                                   |     date of birth, or             |
|                                   |                                   |
|                                   | If a customer is unable to supply |
|                                   | proof of age, details of this     |
|                                   | must be recorded in a DOC.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Accommodation - residential aged  |
|                                   | care                              |
|                                   |                                   |
|                                   | -   Aged care accommodation       |
|                                   |     payments agreement for        |
|                                   |     residential aged care         |
|                                   |     facility. This should include |
|                                   |     the detail of any daily fees  |
|                                   |     paid or lump sum amounts      |
|                                   |     including date paid. See Aged |
|                                   |     care fees and charges -       |
|                                   |     accommodation payments        |
|                                   | -   Contract or agreement with    |
|                                   |     retirement village. If an     |
|                                   |     entry contribution was paid   |
|                                   |     this should include the       |
|                                   |     amount paid and date of       |
|                                   |     entry. See Assessment of      |
|                                   |     entry contribution            |
|                                   |                                   |
|                                   | For verification for Rent         |
|                                   | Assistance purposes - see ‘Rent   |
|                                   | Assistance (RA)’ in L to R        |
|                                   | examples.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | Approved Activity                 |
|                                   |                                   |
|                                   | Job seekers with mutual           |
|                                   | obligation requirements who are   |
|                                   | already engaged in an activity    |
|                                   | and want to have it approved and  |
|                                   | included in a Job Plan may need   |
|                                   | to provide evidence. Youth        |
|                                   | Allowance (YA) job seekers who    |
|                                   | have not attained Year 12 (or     |
|                                   | equivalent) or a Certificate III  |
|                                   | or higher qualification must      |
|                                   | undertake approved activities to  |
|                                   | meet their mutual obligation      |
|                                   | requirements.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Assets                            |
|                                   |                                   |
|                                   | Life insurance policies           |
|                                   |                                   |
|                                   | The latest statement for each     |
|                                   | life insurance policy that shows  |
|                                   | the ownership and the             |
|                                   | surrender/withdrawal value for    |
|                                   | that policy.                      |
|                                   |                                   |
|                                   | Foreign assets                    |
|                                   |                                   |
|                                   | A current statement or            |
|                                   | verification documents for each   |
|                                   | asset held outside Australia      |
|                                   | showing the applicable foreign    |
|                                   | currency value and/or balance.    |
|                                   |                                   |
|                                   | Other assets                      |
|                                   |                                   |
|                                   | Customers may need to supply      |
|                                   | evidence of the value or sale of  |
|                                   | assets where the Service Officer  |
|                                   | has doubts about the stated       |
|                                   | value. For verification of        |
|                                   | investments, see 'Investments' in |
|                                   | D to I examples.                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Away from home                    |
|                                   |                                   |
|                                   | It is necessary to verify that a  |
|                                   | dependent customer is genuinely   |
|                                   | required to live away from home   |
|                                   | for a specific period of time,    |
|                                   | for the purpose of study,         |
|                                   | training, or searching for work.  |
|                                   | See The away from home rate for   |
|                                   | Youth Allowance (YA) customers.   |
|                                   |                                   |
|                                   | Customers who claim that their    |
|                                   | parents' home is an inadequate    |
|                                   | place for study may need to       |
|                                   | supply a floor plan of the house, |
|                                   | or other evidence depending on    |
|                                   | their circumstances. See          |
|                                   | Assessing parental home           |
|                                   | inadequate for study, work or job |
|                                   | search for Youth Allowance (YA)   |
|                                   | at the away from home rate.       |
|                                   |                                   |
|                                   | Customers who claim the away from |
|                                   | home rate of ABSTUDY will need to |
|                                   | provide evidence supporting their |
|                                   | circumstances. This may also need |
|                                   | to be supported by an Authority   |
|                                   | in the community.                 |
+-----------------------------------+-----------------------------------+
| 7                                 | Bank accounts and balances        |
|                                   |                                   |
|                                   | Customers submitting new claims   |
|                                   | for income support payments or    |
|                                   | services may need to supply       |
|                                   | supporting documents of all       |
|                                   | income and assets held:           |
|                                   |                                   |
|                                   | -   by them                       |
|                                   | -   by their partner              |
|                                   | -   jointly                       |
|                                   |                                   |
|                                   | If supporting documents from a    |
|                                   | financial institution (bank,      |
|                                   | building society or credit union) |
|                                   | are required as part of the       |
|                                   | claim, they must show:            |
|                                   |                                   |
|                                   | -   account holder names          |
|                                   | -   account number                |
|                                   | -   BSB or name of financial      |
|                                   |     institution                   |
|                                   | -   the current account balance   |
|                                   |                                   |
|                                   | Note: Supporting documents that   |
|                                   | do not include some of the        |
|                                   | information may be acceptable, as |
|                                   | long as the delegate is satisfied |
|                                   | that the information is true and  |
|                                   | correct.                          |
|                                   |                                   |
|                                   | Exceptions for customers          |
|                                   | reclaiming                        |
|                                   |                                   |
|                                   | Some exceptions may apply to      |
|                                   | former customers who are          |
|                                   | reclaiming a payment or           |
|                                   | concession. These can include:    |
|                                   |                                   |
|                                   | -   when they are close to the    |
|                                   |     asset limit                   |
|                                   | -   they have previously been     |
|                                   |     rejected because they were    |
|                                   |     over the asset limit          |
|                                   | -   if there is a significant     |
|                                   |     change in circumstances       |
|                                   |                                   |
|                                   | In cases such as these, they may  |
|                                   | be exempt from verifying their    |
|                                   | income and assets if they         |
|                                   | acknowledge in their claim that   |
|                                   | the information is still true and |
|                                   | correct.                          |
|                                   |                                   |
|                                   | Service Officers can process      |
|                                   | these claims without asking for   |
|                                   | further bank account              |
|                                   | verification, unless there is     |
|                                   | doubt about the validity of the   |
|                                   | information provided in the       |
|                                   | claim. When in doubt, make a      |
|                                   | request for information to the    |
|                                   | customer.                         |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Requesting information (CLK)  |
|                                   | -   Verifying income and assets   |
|                                   |                                   |
|                                   | Claims subject to Liquid Assets   |
|                                   | Waiting Period (LAWP)             |
|                                   |                                   |
|                                   | JobSeeker payment, Youth          |
|                                   | Allowance, and Austudy are all    |
|                                   | subject to a LAWP. If they are    |
|                                   | claiming one of these payments,   |
|                                   | see Liquid Assets Waiting Period  |
|                                   | (LAWP).                           |
|                                   |                                   |
|                                   | Claims not subject to LAWP        |
|                                   |                                   |
|                                   | In new claims for payments that   |
|                                   | are not subject to LAWP, the      |
|                                   | documentation does not have to    |
|                                   | include the exact balance         |
|                                   | declared within the claim.        |
|                                   | However, the delegate must be     |
|                                   | satisfied that the supporting     |
|                                   | documents show the overall        |
|                                   | financial position the customer   |
|                                   | has declared in their claim.      |
|                                   |                                   |
|                                   | If the delegate conducts          |
|                                   | verification then the delegate    |
|                                   | has three courses of action:      |
|                                   |                                   |
|                                   | -   they accept the customer      |
|                                   |     declaration, no further       |
|                                   |     action is required            |
|                                   | -   the account balance differs   |
|                                   |     from the declared financial   |
|                                   |     position in the claim, the    |
|                                   |     account balance can be        |
|                                   |     corrected as long as the      |
|                                   |     supporting documentation is   |
|                                   |     either dated before or on the |
|                                   |     date of the claim             |
|                                   | -   they are not satisfied the    |
|                                   |     declaration reflects the      |
|                                   |     customers financial position  |
|                                   |     at time of claim, an updated  |
|                                   |     statement or transaction      |
|                                   |     record must be requested      |
|                                   |                                   |
|                                   | If undertaking verification,      |
|                                   | staff should also check if any    |
|                                   | withdrawals do not appear to be   |
|                                   | for usual living expenses.        |
|                                   |                                   |
|                                   | Term deposits                     |
|                                   |                                   |
|                                   | The supporting documents required |
|                                   | for term deposits differ to other |
|                                   | bank accounts and must include:   |
|                                   |                                   |
|                                   | -   current balance of the term   |
|                                   |     deposit                       |
|                                   | -   customer’s name               |
|                                   |                                   |
|                                   | The latest statement should       |
|                                   | confirm the current balance. A    |
|                                   | reinvestment or rollover          |
|                                   | statement is acceptable, if an    |
|                                   | automatic reinvestment has        |
|                                   | occurred.                         |
|                                   |                                   |
|                                   | For overseas financial            |
|                                   | institutions supporting           |
|                                   | documentation is required that    |
|                                   | shows the applicable foreign      |
|                                   | currency value, current account   |
|                                   | balance and the account holder    |
|                                   | details, i.e. account name,       |
|                                   | account number and financial      |
|                                   | institution.                      |
|                                   |                                   |
|                                   | Direct Investments                |
|                                   |                                   |
|                                   | For any Debentures or Bonds the   |
|                                   | latest statement or internet      |
|                                   | printout showing the current      |
|                                   | balance and ownership of the      |
|                                   | investment.                       |

|                                   | retirement or recently ceased     |
|                                   | working in the business).         |

| 2                                 | Employment income                 |
|                                   |                                   |
|                                   | Ceased work                       |
|                                   |                                   |
|                                   | Customers applying for a payment  |
|                                   | that requires verification they   |
|                                   | have ceased work need to verify   |
|                                   | that they are unemployed through  |
|                                   | one of the following ways:        |
|                                   |                                   |
|                                   | Employer Separation Certificate   |
|                                   | (SU1)                             |
|                                   |                                   |
|                                   | Letter from employer including -  |
|                                   | date last worked, reason for      |
|                                   | ceasing work and leave            |
|                                   | entitlements paid, the amount of  |
|                                   | the person's final payment, the   |
|                                   | amount of any money still owed by |
|                                   | the employer to the person, any   |
|                                   | personal compensation details     |
|                                   |                                   |
|                                   | Phone call to employer to obtain  |
|                                   | date last worked, reason for      |
|                                   | ceasing work and leave            |
|                                   | entitlements paid, the amount of  |
|                                   | the person's final payment, the   |
|                                   | amount of any monies still owed   |
|                                   | by the employer to the person,    |
|                                   | any personal compensation details |
|                                   |                                   |
|                                   | Note: Service Officers should let |
|                                   | the employer know that            |
|                                   | information obtained over the     |
|                                   | phone will be confirmed in        |
|                                   | writing. Follow up contact by     |
|                                   | sending a letter to the employer. |
|                                   | Use a Q999 (free text) letter:    |
|                                   |                                   |
|                                   | -   restate the relevant details  |
|                                   |     provided by the employer      |
|                                   | -   advise employer to contact if |
|                                   |     any material information is   |
|                                   |     incorrect                     |
|                                   |                                   |
|                                   | If the customer was               |
|                                   | self-employed, a letter from      |
|                                   | their accountant or solicitor     |
|                                   | detailing that the business has   |
|                                   | ceased operating should be        |
|                                   | provided                          |
|                                   |                                   |
|                                   | There may be situations where     |
|                                   | this verification may not be      |
|                                   | required, see Employment          |
|                                   | Separation Certificate (SU1).     |
|                                   |                                   |
|                                   | Overseas work: for a customer who |
|                                   | ceased work overseas, a letter or |
|                                   | statement from their employer     |
|                                   | providing details of their        |
|                                   | income, cessation date, reasons   |
|                                   | etc. and any holiday or other     |
|                                   | entitlements is suitable. If the  |
|                                   | customer is not able to provide   |
|                                   | such a letter, then a declaration |
|                                   | statement should be done instead. |
|                                   |                                   |
|                                   | Note: verification of             |
|                                   | unemployment may also allow the   |
|                                   | correct application of            |
|                                   | non-payment and other waiting     |
|                                   | periods that may be applicable.   |
|                                   |                                   |
|                                   | Still working                     |
|                                   |                                   |
|                                   | If a customer and/or their        |
|                                   | partner is still employed         |
|                                   | full-time, part-time or casual    |
|                                   | then request the following:       |
|                                   |                                   |
|                                   | -   two most recent payslips; or  |
|                                   | -   letter from employer stating  |
|                                   |     number of hours worked and    |
|                                   |     gross amount paid for the two |
|                                   |     most recent pay periods; or   |
|                                   | -   employment letter of offer or |
|                                   |     contract stating number of    |
|                                   |     hours worked and gross amount |
|                                   |     paid                          |
|                                   |                                   |
|                                   | The Income and Asset form (Mod    |
|                                   | IA) and some online claims only   |
|                                   | requests the two most recent      |
|                                   | payslips. Subsequent and ongoing  |
|                                   | employment periods do not need    |
|                                   | verification, and details of the  |
|                                   | employment income can be reported |
|                                   | verbally. If the customer is      |
|                                   | unable to be contacted by phone,  |
|                                   | or if there is doubt over the     |
|                                   | employment details or income      |
|                                   | amount, then formal verification  |
|                                   | may still be requested where      |
|                                   | necessary.                        |
|                                   |                                   |
|                                   | This applies to new claims where  |
|                                   | additional employment periods     |
|                                   | have elapsed between claim        |
|                                   | lodgement and finalisation.       |
|                                   |                                   |
|                                   | Note: evidence of employment      |
|                                   | income is not required for job    |
|                                   | seeker claims as these customers  |
|                                   | are automatically profiled to     |
|                                   | report their employment income,   |
|                                   | any change of circumstances and   |
|                                   | mutual obligation activities      |
|                                   | before any payment is released.   |
|                                   |                                   |
|                                   | For sales commission a document   |
|                                   | giving details, including the     |
|                                   | amount and date of receipt, is    |
|                                   | required.                         |
|                                   |                                   |
|                                   | Note: where a customer and/or     |
|                                   | partner are currently receiving   |
|                                   | an income support payment,        |
|                                   | documentation may be requested to |
|                                   | assist in coding of the income,   |
|                                   | even though it may not be         |
|                                   | required for verification         |
|                                   | purposes.                         |
|                                   |                                   |
|                                   | Overseas work for foreign         |
|                                   | employment income, the documents  |
|                                   | should:                           |
|                                   |                                   |
|                                   | -   show the income amount        |
|                                   |     received in foreign currency, |
|                                   |     rather than Australian        |
|                                   |     dollars, and                  |
|                                   | -   be issued by the overseas     |
|                                   |     person, organisation or       |
|                                   |     authority making the payment  |
|                                   |                                   |
|                                   | Salary sacrificing/salary         |
|                                   | packaging                         |
|                                   |                                   |
|                                   | For salary sacrificing into       |
|                                   | superannuation, a statement from  |
|                                   | the employer, on the employer's   |
|                                   | letterhead, advising the amount   |
|                                   | of the contributions which have   |
|                                   | been salary sacrificed and the    |
|                                   | name of the superannuation fund   |
|                                   | and member to which the payments  |
|                                   | are made.                         |
|                                   |                                   |
|                                   | For salary sacrificing or salary  |
|                                   | packaging for benefits other than |
|                                   | superannuation:                   |
|                                   |                                   |
|                                   | payslips are acceptable evidence  |
|                                   | as long as they show the total    |
|                                   | gross income, that is, the amount |
|                                   | before any salary sacrifice       |
|                                   | deductions                        |
|                                   |                                   |
|                                   | if the payslips do not show this  |
|                                   | or, if it is unclear whether the  |
|                                   | payslip shows the non-grossed-up  |
|                                   | amount, the customer should       |
|                                   | provide:                          |
|                                   |                                   |
|                                   | -   a copy of the salary          |
|                                   |     sacrificing agreement, or     |
|                                   | -   a statement from their        |
|                                   |     employer on the employer's    |
|                                   |     letterhead advising the type  |
|                                   |     and amount of benefit         |
|                                   |     received and whether or not   |
|                                   |     the amount is grossed up or   |
|                                   |     non-grossed up                |
|                                   |                                   |
|                                   | See Assessing salary              |
|                                   | sacrificing/salary packaging      |
|                                   | arrangements for employment       |
|                                   | income                            |
|                                   |                                   |
|                                   | For employer provided fringe      |
|                                   | benefits paid in addition to a    |
|                                   | wage or salary, the customer      |
|                                   | should provide verification in    |
|                                   | the form of:                      |
|                                   |                                   |
|                                   | a letter from their employer, on  |
|                                   | the employer's letterhead,        |
|                                   | stating:                          |
|                                   |                                   |

|                                   | (or equivalent) or a Certificate  |
|                                   | III or higher must have a study   |
|                                   | exemption coded on NES to qualify |
|                                   | for YA, even when they are        |
|                                   | undertaking other approved        |
|                                   | activities to satisfy mutual      |
|                                   | obligation requirements. The      |
|                                   | system will display a warning     |
|                                   | message for these job seekers     |
|                                   | when no study exemption has been  |
|                                   | coded and their payment will      |
|                                   | cancel.                           |

| 4                                 | Superannuation and income streams |
|                                   |                                   |
|                                   | Superannuation                    |
|                                   |                                   |
|                                   | Details about a customer’s (or    |
|                                   | partner’s) superannuation is only |
|                                   | required if the owner of the      |
|                                   | investment is over Age Pension    |
|                                   | age. See Assessing                |
|                                   | superannuation.                   |
|                                   |                                   |
|                                   | For a Self-Managed Superannuation |
|                                   | Fund (SMSF) or Small APRA Fund    |
|                                   | (SAF) the customer must provide   |
|                                   | the latest annual member          |
|                                   | statement and the annual          |
|                                   | financial returns for their       |
|                                   | superannuation fund.              |
|                                   |                                   |
|                                   | For other superannuation          |
|                                   | investments (non-SMSF or SAF) the |
|                                   | customer must provide the latest  |
|                                   | statement showing the following:  |
|                                   |                                   |
|                                   | -   the name of the fund manager  |
|                                   | -   full name of the investment   |
|                                   |     product                       |
|                                   | -   the SPIN or APIR codes        |
|                                   | -   the value of the investment   |
|                                   |     or the number of units owned  |
|                                   |     (if a unitised product)       |
|                                   |                                   |
|                                   | Most providers issue an annual    |
|                                   | statement and the most recent     |
|                                   | statement should be provided by   |
|                                   | the customer. Further             |
|                                   | documentation should only be      |
|                                   | requested where:                  |
|                                   |                                   |
|                                   | -   the statement is more than 12 |
|                                   |     months old at time of claim   |
|                                   |     lodgement, and                |
|                                   | -   the customer confirms a more  |
|                                   |     recent statement is available |
|                                   |                                   |
|                                   | However, if there has been a      |
|                                   | change in the superannuation      |
|                                   | investment since claim lodgement, |
|                                   | or there is doubt about the       |
|                                   | accuracy of the information       |
|                                   | provided by the customer, further |
|                                   | documentation can be requested.   |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   it is not deemed to be a      |
|                                   |     change in circumstance where  |
|                                   |     the customer has made         |
|                                   |     contributions to their        |
|                                   |     superannuation fund due to    |
|                                   |     employment since the date the |
|                                   |     statement was issued          |
|                                   | -   if all or part of a           |
|                                   |     superannuation investment has |
|                                   |     been converted into an income |
|                                   |     stream - see income streams   |
|                                   |     below                         |
|                                   |                                   |
|                                   | Income streams                    |
|                                   |                                   |
|                                   | For income streams, including     |
|                                   | defined benefit superannuation    |
|                                   | pensions, the customer must       |
|                                   | provide the latest:               |
|                                   |                                   |
|                                   | -   Centrelink/DVA schedule       |
|                                   | -   similar schedule completed by |
|                                   |     the income stream provider or |
|                                   |     trustee of SMSF/SAF showing   |
|                                   |     the income stream details     |
|                                   |     including current gross       |
|                                   |     income and account balance    |
|                                   |     (if applicable) of the        |
|                                   |     product                       |
|                                   | -   Defined Benefit schedule for  |
|                                   |     defined benefits, or          |
|                                   | -   details of Income Stream form |
|                                   |     (SA330) completed and signed  |
|                                   |     by the income stream provider |
|                                   |     or trustee of the SMSF/SAF    |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   a more recent                 |
|                                   |     schedule/statement should be  |
|                                   |     requested, if the document    |
|                                   |     provided by the customer is   |
|                                   |     more than 12 months old       |
|                                   |                                   |
|                                   | for SMSF, documents must not be   |
|                                   | requested more than once per      |
|                                   | year.                             |
