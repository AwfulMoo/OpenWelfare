|                                   | separating couples to resolve     |
|                                   | Family Law matters without having |
|                                   | to go to Court.                   |
|                                   |                                   |
|                                   | Customers must not be referred,   |
|                                   | if any of the following apply:    |
|                                   |                                   |
|                                   | family and domestic violence      |
|                                   |                                   |
|                                   | time limits for Family Court      |
|                                   | finalisations. That is:           |
|                                   |                                   |
|                                   | -   if divorced and it has been 9 |
|                                   |     months or more since divorce  |
|                                   |     was finalised                 |
|                                   | -   are in a de facto             |
|                                   |     relationship and it has been  |
|                                   |     more than one year and 9      |
|                                   |     months since the breakdown of |
|                                   |     the relationship              |
|                                   |                                   |
|                                   | the relationship is not           |
|                                   | recognised under Australian       |
|                                   | family law                        |
|                                   |                                   |
|                                   | are on a visa or currently        |
|                                   | applying for a visa which is      |
|                                   | dependent on the relationship     |
|                                   | with their former partner         |
|                                   |                                   |
|                                   | children. If:                     |
|                                   |                                   |
|                                   | -   over the age of 18            |
|                                   | -   the former partner has        |
|                                   |     removed or threatened to      |
|                                   |     remove the children from      |
|                                   |     their home                    |
|                                   | -   there are any existing court  |
|                                   |     orders in place about         |
