Conduct a security check on a representative of a private health insurer

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Request private health insurer    |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Request from the caller:          |
|                                   |                                   |
|                                   | -   name of private health        |
|                                   |     insurer                       |
|                                   | -   Minor ID                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Access private health insurers    |
|                                   | list + Read more ...              |
|                                   |                                   |
|                                   | Confirm the private health        |
|                                   | insurer details provided.         |
|                                   |                                   |
|                                   | The Resources page contains       |
|                                   | private health insurer lists.     |
|                                   |                                   |
|                                   | Note: private health insurers can |
|                                   | be searched for directly by       |
|                                   | entering the Minor ID as a Search |
|                                   | term.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Confirm details provided matches  |
|                                   | that held by Services Australia   |
|                                   | (the agency) + Read more ...      |
|                                   |                                   |
|                                   | Are the details provided by the   |
|                                   | caller the same as the details    |
|                                   | recorded with the agency?         |
|                                   |                                   |
|                                   | -   Yes, proceed with the enquiry |
|                                   | -   No, advise the caller that    |
|                                   |     the security check cannot be  |
|                                   |     continued and end the call.   |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
