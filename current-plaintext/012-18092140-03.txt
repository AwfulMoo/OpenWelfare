This page contains the process to handle a request to get a:

-   HW018 – Application to Register a Seed Organisation form
-   HW021 – Application to request a National Authentication Service for
    Health Public Key Infrastructure Certificate for Healthcare Provider
    Organisations form
-   HW037 – Application to amend a Healthcare Organisation record form
-   HW040 – Application to replace a Responsible Officer or add/remove
    an Organisation Maintenance Officer for an organisation form

Handle a request for a caller to get a HW018 or HW021 form via the intranet

Table 1: This table describes how Service Officers handle requests to
get a HW018 or HW021 form via the intranet. Initial security check is
not required for this procedure.

Note: if the caller is abusive, warm transfer them to the NASH
Operations Team.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Is the caller requesting a HW018  |
|                                   | or a HW021 + Read more ...        |
|                                   |                                   |
|                                   | -   HW018 or both, go to Step 2   |
|                                   | -   HW021 only, go to Step 7      |
+-----------------------------------+-----------------------------------+
| 2                                 | Does the caller know there is an  |
|                                   | online process to register a      |
|                                   | seed? + Read more ...             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, advise the caller that    |
|                                   |     registering a seed            |
|                                   |     organisation online takes     |
|                                   |     less than a week to receive a |
|                                   |     result, and submitting a hard |
|                                   |     copy can take 4 weeks to      |
|                                   |     receive a result. The caller  |
|                                   |     needs a Provider Digital      |
|                                   |     Access (PRODA) account to use |
|                                   |     the Healthcare Identifiers –  |
|                                   |     Register Seed Organisation    |
|                                   |     application process in the    |
|                                   |     Health Professional Online    |
|                                   |     Services (HPOS). Go to Step 3 |
+-----------------------------------+-----------------------------------+
| 3                                 | Is the caller willing to use the  |
|                                   | online process? + Read more ...   |
|                                   |                                   |
|                                   | -   Yes, ask them if they have a  |
|                                   |     PRODA account to access HPOS, |
|                                   |     go to Step 4                  |
|                                   | -   No, ask for the caller’s      |
|                                   |     email address so the form can |
|                                   |     be sent via email, go to Step |
|                                   |     7                             |
+-----------------------------------+-----------------------------------+
| 4                                 | Does the caller have a PRODA      |
|                                   | account? + Read more ...          |
|                                   |                                   |
|                                   | -   Yes, they need to log on to   |
|                                   |     HPOS using their PRODA        |
|                                   |     account. They can log on from |
|                                   |     the HPOS page. Direct the     |
|                                   |     caller to the Services        |
|                                   |     Australia website homepage,   |
|                                   |     and they can type HPOS into   |
|                                   |     the search bar to find the    |
|                                   |     page. Go to Step 5            |
|                                   | -   No, direct the caller to the  |
|                                   |     Services Australia PRODA      |
|                                   |     page. From the Services       |
|                                   |     Australia website homepage,   |
|                                   |     they can type in PRODA in the |
|                                   |     search bar to find the page.  |
|                                   |     They need to follow the       |
|                                   |     instructions for getting a    |
|                                   |     PRODA account and accessing   |
|                                   |     HPOS. Thank the caller.       |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Does the caller want help in      |
|                                   | completing the Register Seed      |
|                                   | Organisation online registration  |
|                                   | form? + Read more ...             |
|                                   |                                   |
|                                   | -   Yes, see Process online       |
|                                   |     registration for Healthcare   |
|                                   |     Provider Identifier –         |
|                                   |     Organisation (HPI-O) to help  |
|                                   |     applicants step through and   |
|                                   |     complete the online           |
|                                   |     registration. Go to Step 6    |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 6                                 | Was the caller successful in      |
|                                   | completing the online process? +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Yes, thank the caller and     |
|                                   |     advise them they can request  |
|                                   |     a NASH PKI certificate in     |
|                                   |     HPOS. Procedure ends here     |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Does the caller want a HW021? +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Yes, advise the caller:           |
|                                   |                                   |
|                                   | -   they must be registered in    |
|                                   |     the HI Service before they    |
|                                   |     can request a NASH PKI        |
|                                   |     certificate                   |
|                                   | -   even if they apply for a NASH |
|                                   |     PKI certificate using a paper |
|                                   |     form, the certificate can     |
|                                   |     only be downloaded through    |
|                                   |     HPOS                          |
|                                   | -   if the caller also wants a    |
|                                   |     HW018, attach both forms in   |
|                                   |     an email to the caller        |
|                                   |                                   |
|                                   | No, attach the HW018 in an email  |
|                                   | to the caller                     |
|                                   |                                   |
|                                   | The Resources page includes links |
|                                   | to the HW018 and HW021 forms.     |
+-----------------------------------+-----------------------------------+

Handle a request for a caller to get a HW037 or HW040 form via the intranet

Table 2: This table describes how Service Officers handle requests to
get a HW037 or HW040 form via the intranet. Initial security check is
not required for this procedure.

Note: if the caller is abusive, warm transfer them to the NASH
Operations Team.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Is the caller requesting a HW037  |
|                                   | and/or a HW040? + Read more ...   |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 2                                 | Check if caller knows of online   |
|                                   | process + Read more ...           |
|                                   |                                   |
|                                   | Does the caller know there is an  |
|                                   | online process to amend a         |
|                                   | healthcare organisation record,   |
|                                   | replace a Responsible Officer     |
|                                   | (RO) or add/remove an             |
|                                   | Organisation Maintenance Officer  |
|                                   | (OMO)?                            |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, advise the caller:            |
|                                   |                                   |
|                                   | -   of online form                |
|                                   | -   application takes less than a |
|                                   |     week to receive a result      |
|                                   | -   submitting a hard copy can    |
|                                   |     take 4 weeks to receive a     |
|                                   |     result                        |
|                                   | -   they need a Provider Digital  |
|                                   |     Access (PRODA) account to     |
|                                   |     access these functions        |
|                                   | -   go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Is the caller willing to use the  |
|                                   | online process? + Read more ...   |
|                                   |                                   |
|                                   | -   Yes, ask them if they have a  |
|                                   |     PRODA account to access       |
|                                   |     Health Professional Online    |
|                                   |     Services (HPOS), go to Step 4 |
|                                   | -   No, ask for the caller’s      |
|                                   |     email address so the form can |
|                                   |     be sent via email, go to Step |
|                                   |     7                             |
+-----------------------------------+-----------------------------------+
| 4                                 | Does the caller have a PRODA      |
|                                   | account? + Read more ...          |
|                                   |                                   |
|                                   | -   Yes, they need to log on to   |
|                                   |     HPOS using their PRODA        |
|                                   |     account. They can log on from |
|                                   |     the HPOS page. Direct the     |
|                                   |     caller to the Services        |
|                                   |     Australia website homepage,   |
|                                   |     and they can type HPOS into   |
|                                   |     the search bar to find the    |
|                                   |     page. Go to Step 5            |
|                                   | -   No, direct the caller to the  |
|                                   |     Services Australia PRODA      |
|                                   |     page. From the Services       |
|                                   |     Australia website homepage,   |
|                                   |     they can type in PRODA in the |
|                                   |     search bar to find the page.  |
|                                   |     They need to follow the       |
|                                   |     instructions for getting a    |
|                                   |     PRODA account and accessing   |
|                                   |     HPOS. Thank the caller.       |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Does the caller want help in      |
|                                   | completing the online process? +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Yes, to help applicants step      |
|                                   | through and complete the online   |
|                                   | registration for:                 |
|                                   |                                   |
|                                   | -   change of ownership of a seed |
|                                   |     or network organisation, see  |
|                                   |     Process organisation change   |
|                                   |     of ownership in the           |
|                                   |     Healthcare Identifiers (HI)   |
|                                   |     Service for help              |
|                                   | -   change of RO for a seed       |
|                                   |     organisation, see Amend       |
|                                   |     Responsible Officer (RO) or   |
|                                   |     Organisation Maintenance      |
|                                   |     Officer (OMO) record in the   |
|                                   |     Healthcare Identifiers (HI)   |
|                                   |     Service for help              |
|                                   | -   go to Step 6                  |
|                                   |                                   |

+-----------------------------------+-----------------------------------+
| 6                                 | Was the caller successful in      |
|                                   | completing the online process? +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Yes, thank the caller.        |
|                                   |     Procedure ends here           |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 7                                 | Does the caller want a HW040,     |
|                                   | HW037 or both? + Read more ...    |
|                                   |                                   |

|                                   |     amendment only                |
|                                   | -   HW040, send if updating       |
|                                   |     responsible officer (RO) or   |
|                                   |     organisation maintenance      |
|                                   |     office (OMO) only             |
|                                   | -   HW037 and HW040, send if      |
|                                   |     organisation amendment        |
|                                   |     includes updates to RO or OMO |
|                                   |     details                       |
