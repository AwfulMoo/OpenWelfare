|                                   | Redress Scheme websites.          |

|                                   | into an arrangement to address    |

|                                   | possible time.                    |
|                                   |                                   |
|                                   | For child support debts only, not |

|                                   | -   discuss the Late Payment      |

|                                   |     overpayment debts). See Child |
|                                   |     Support late payment          |
|                                   |     penalties and estimate        |
|                                   |     penalties                     |

| 7                                 | Negotiate payment within 3        |
|                                   | instalments + Read more ...       |
|                                   |                                   |
|                                   | Negotiate payment within 3        |

|                                   | to 3 instalments.                 |
|                                   |                                   |

|                                   | fortnightly payments), or monthly |
|                                   | (3 monthly payments).             |
|                                   |                                   |
|                                   | The repayment amounts will be     |
|                                   | reduced if an initial lump sum is |
|                                   | paid, and these can be of equal   |
|                                   | or unequal amounts to repay the   |
|                                   | debt in the shortest time.        |
|                                   |                                   |
|                                   | If the customer:                  |
|                                   |                                   |
|                                   | -   agrees to pay within 3        |
|                                   |     instalments, go to Step 10    |
|                                   | -   cannot pay within 3           |
|                                   |     instalments, go to Step 8     |
