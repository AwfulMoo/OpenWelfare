Delete advance payments

Table 2: This table describes how to delete an advance payment in ACMPS.

Note: only Subject Matter Experts (SME) and Team Leaders from the Aged
Care Payments Team (Aged Care and Allied Health Branch) have the
authority to delete advance payments. Advances can only be deleted when
in a processed status. Approved advances cannot be deleted.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Home screen + Read more ...       |
|                                   |                                   |
|                                   | Log on to ACMPS                   |
|                                   |                                   |
|                                   | On the Home screen, select        |
|                                   | Payments Additions option from    |
|                                   | the left side menu                |
|                                   |                                   |
|                                   | -   The Payment Additions screen  |
|                                   |     is displayed                  |
|                                   |                                   |
|                                   | Key the Service ID in the Service |
|                                   | ID field. Select the icon to open |
|                                   | the search dialog box and         |
|                                   | manually key the Service ID       |
|                                   |                                   |
|                                   | Select the relevant category type |
|                                   | in the Category Type field e.g.   |
|                                   | Manual advance - Home Care for    |
|                                   | home care manual advances etc.    |
|                                   |                                   |
|                                   | Select Search                     |
|                                   |                                   |
|                                   | -   The search results are        |
|                                   |     displayed                     |
|                                   |                                   |
|                                   | Select the relevant advance from  |
|                                   | the search results                |
|                                   |                                   |
|                                   | -   The relevant advance details  |
|                                   |     are displayed                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Deleting the advance + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Note: once a transaction status   |
|                                   | is set to Deleted, no further     |
|                                   | change can be made to it. All     |
|                                   | options are disabled.             |
|                                   |                                   |
|                                   | In relevant advance details       |
|                                   | screen:                           |
|                                   |                                   |
|                                   | Select the Status tab             |
|                                   |                                   |
|                                   | -   The advance’s status tab      |
|                                   |     screen is displayed           |
|                                   |                                   |
|                                   | Select Set New Status             |
|                                   |                                   |
|                                   | -   The advance’s New status      |
|                                   |     screen is displayed           |
|                                   |                                   |
|                                   | Select Deleted status from the    |
|                                   | New Status dialog box             |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | -   The status is displayed as    |
|                                   |     Deleted                       |
|                                   |                                   |
|                                   | Procedure ends here.              |
