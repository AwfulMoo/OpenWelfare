|                                   | entitled to the following away    |
|                                   | from home rate allowances,        |
|                                   | including across vacation         |
|                                   | periods. Entitlement continues    |

|                                   | Code the AD screen to match the   |

|                                   | qualification for RAA, matches    |
|                                   | the eligibility period for the    |
|                                   | away from home rate.              |

|                                   | (home) address.                   |
|                                   |                                   |
|                                   | If the student's home address is: |
|                                   |                                   |
|                                   | not in a RAA specified area,      |
|                                   | record the TER (term) address of  |
|                                   | where the student is residing     |
|                                   | whilst studying                   |
|                                   |                                   |
|                                   | in a RAA specified area, record   |
|                                   | the location where the student    |
|                                   | resides while they are studying:  |
|                                   | Code the ADH screen as follows:   |
|                                   |                                   |
|                                   | -   Address Type: TER             |
|                                   | -   Start Date:                   |
|                                   |     - Private Boarder:            |
|                                   |     commencement date of the      |
|                                   |     board arrangement             |
|                                   |     - Boarding School/Hostel:     |
|                                   |     determined away from home     |
|                                   |     commencement date             |
|                                   | -   End Date: start Date + 2 days |
|                                   | -   Address: students term        |
|                                   |     address                       |
|                                   | -   Is Rent Assistance being      |
|                                   |     claimed on this Address: Yes  |
|                                   | -   Is this address in a Remote   |
|                                   |     Area for the purpose of       |
|                                   |     Remote Area Allowance: No     |
|                                   | -   Should this address be used   |
|                                   |     for all mail: Yes             |
|                                   |                                   |
|                                   | Note: if the student wants to     |
|                                   | receive mail at their term        |
|                                   | address, code a POS (postal)      |

|                                   | details.                          |
|                                   |                                   |
