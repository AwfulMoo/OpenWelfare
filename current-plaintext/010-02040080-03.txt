This page contains information to assist staff in making a determination
if a student or Australian Apprentice is eligible for independence due
to age.

Assessing the ABSTUDY customer's eligibility for independent status

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Australian Apprentice aged 22     |
|                                   | years or over + Read more ...     |
|                                   |                                   |
|                                   | Is the student or Australian      |
|                                   | Apprentice 22 years of age or     |
|                                   | older?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Not eligible for independence due |
|                                   | to age + Read more ...            |
|                                   |                                   |
|                                   | The student or Australian         |
|                                   | Apprentice is not eligible for    |
|                                   | independence under the age        |
|                                   | criteria.                         |
|                                   |                                   |
|                                   | Consider whether the customer is  |
|                                   | independent on other grounds. See |
|                                   | Independence for ABSTUDY          |
|                                   | customers.                        |
|                                   |                                   |
|                                   | If appropriate, refer the         |
|                                   | customer for other assistance,    |
|                                   | for example, to a Centrelink      |
|                                   | social worker, or Indigenous      |
|                                   | Service Officers.                 |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Proof of age + Read more ...      |
|                                   |                                   |
|                                   | The student or Australian         |
|                                   | Apprentice is eligible for        |
|                                   | independence under the age        |
|                                   | criteria. There is no requirement |
|                                   | to request additional             |
|                                   | documentation to prove            |
|                                   | independence. Proof of age would  |
|                                   | have been provided as part of the |
|                                   | Identity Confirmation             |
|                                   | requirements.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Auto grant + Read more ...        |
|                                   |                                   |
|                                   | The system will automatically     |
|                                   | grant the independent status      |
|                                   | based on age from the customer's  |
|                                   | 22^(nd) birthday or the date of   |
|                                   | grant of ABSTUDY, whichever is    |
|                                   | the later.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Record decision + Read more ...   |
|                                   |                                   |
|                                   | Record details of decision or     |
|                                   | discussion on a DOC.              |
+-----------------------------------+-----------------------------------+
