Reassessing Additional Board Allowance due to change in parental income

This table describes how to reassess entitlement to Additional Boarding
Allowance (ABA) when an AIC student’s parents/guardians commence or
return to full time work. Portions of the process can only be completed
by Assistance for Isolated Children (AIC) Smart Centre staff. Look up
local Processing Services details for: Assistance for Isolated Children
(AIC).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Income details + Read more ...    |
|                                   |                                   |
|                                   | Is the AIC applicant receiving    |
|                                   | Additional Boarding Allowance     |
|                                   | (ABA)?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, this procedure does not   |
|                                   |     apply                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Processing ABA + Read more ...    |
|                                   |                                   |
|                                   | -   AIC Smart Centre staff, go to |
|                                   |     Step 3                        |
|                                   | -   Advise the AIC applicant they |
|                                   |     may be required to provide    |
|                                   |     additional income details if  |
|                                   |     their circumstances change    |
|                                   |     and that their entitlement to |
|                                   |     ABA may vary. Record date of  |
|                                   |     commencement of employment on |
|                                   |     a DOC and forward to relevant |
|                                   |     AIC Smart Centre. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | ABA Assessment + Read more ...    |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | -   Consider whether special      |
|                                   |     assessment is affected or     |
|                                   |     reverse current income        |
|                                   |     assessment applies and        |
|                                   |     request relevant income       |
|                                   |     details from the applicant.   |
|                                   |     See Requesting information    |
|                                   |     (CLK)                         |
|                                   | -   Select Information to be      |
|                                   |     supplied within appropriate   |
|                                   |     timeframe                     |
|                                   | -   Check with the applicant      |
|                                   |     whether the job has resulted  |
|                                   |     in a change of address        |
|                                   | -   Follow up any possible        |
|                                   |     overpayment                   |
|                                   | -   If there are other siblings,  |
|                                   |     ensure that their records are |
|                                   |     also updated with the above   |
|                                   |     information                   |
|                                   | -   Record relevant details on a  |
|                                   |     DOC                           |
+-----------------------------------+-----------------------------------+
