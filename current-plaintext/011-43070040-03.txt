For Service Officers authorised to perform Simplified Billing adjustment
on adjustment only

This page contains the steps to follow when checking the National LDA
spreadsheet to verify if a previous manual adjustment has been performed
and to register a Simplified Billing adjustment on adjustment on the
FHSI screen.

On this Page:

Check National LDA spreadsheet

Register an adjustment on adjustment on FHSI screen

Check National LDA spreadsheet

Before performing a Simplified Billing adjustment on adjustment check
the National LDA spreadsheet to verify if a previous manual adjustment
has been performed. This enables a fully informed evaluation of the
claim to be conducted.

Process to check National LDA spreadsheet

Table 1: This table describes the process to complete when checking the
National LDA spreadsheet to verify a previous manual adjustment.

Register an adjustment on adjustment on FHSI screen

Process to register Simplified Billing adjustment on adjustment

Table 2: This table describes the process to complete to register a
Simplified Billing adjustment on adjustment on the FHSI screen and
provides the required fields to be completed.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access FHSI screen + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | -   FHSI on the control line      |
|                                   | -   press [Enter]                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Key details + Read more ...       |
|                                   |                                   |
|                                   | Key the following details into    |
|                                   | corresponding fields:             |
|                                   |                                   |
|                                   | -   minor customer ID (Minor ID   |
|                                   |     or Registration number)       |
|                                   | -   original claim ID or previous |
|                                   |     adjusted claim ID             |
|                                   | -   original or previous          |
|                                   |     adjustment DOL                |
|                                   | -   new adjustment claim ID       |
|                                   | -   new adjustment DOL            |
|                                   | -   ADJ on ADJ                    |
|                                   |                                   |
|                                   | Note: a secret modifier is        |
|                                   | required for the ADJ on ADJ field |
|                                   | to enable a Service Officer to    |
|                                   | perform an adjustment on          |
|                                   | adjustment electronically. This   |
|                                   | information is privileged         |
|                                   | information and will only be      |
|                                   | provided to authorised Service    |
|                                   | Officers.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Suppress action field + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The suppress action field default |
|                                   | is 'N'                            |
|                                   |                                   |
|                                   | Only change to 'Y' if performing  |
|                                   | a suppressed adjustment, see      |
|                                   | Process Simplified Billing Latter |
|                                   | Day Adjustment (LDA) claims in    |
|                                   | Medicare.                         |
|                                   |                                   |
|                                   | -   press [Enter]. The claim      |
|                                   |     registration details are      |
|                                   |     validated and the name and    |
|                                   |     address of the billing entity |
|                                   |     displayed for verification.   |
|                                   |                                   |
|                                   | -   press [Enter]                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Process adjustment + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   to process the adjustment see |
|                                   |     Process Simplified Billing    |
|                                   |     Latter Day Adjustments (LDA)  |
|                                   |     claims in Medicare            |
|                                   |                                   |
|                                   | Note: for information about       |
|                                   | screens, see Register a           |
|                                   | Simplified Billing Latter Day     |
|                                   | Adjustment (LDA) in Medicare      |
