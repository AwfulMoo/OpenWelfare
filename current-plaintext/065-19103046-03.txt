Investigate annual or lifetime caps

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm the start date and        |
|                                   | history of care + Read more ...   |
|                                   |                                   |
|                                   | To decide the start date of aged  |
|                                   | care, access ACMPS and SPARC for  |
|                                   | periods of care including         |
|                                   | pre-entry leave.                  |
|                                   |                                   |
|                                   | For home care, access ACMPS:      |
|                                   |                                   |
|                                   | Assessments & Events tab:         |
|                                   |                                   |
|                                   | -   Entry Details, including      |
|                                   |     Service ID information        |
|                                   | -   Leave Details                 |
|                                   | -   Departure Details             |
|                                   |                                   |
|                                   | Fees and Charges tab to confirm   |
|                                   | ongoing fees and annual and       |
|                                   | lifetime caps                     |
|                                   |                                   |
|                                   | For residential care, access      |
|                                   | SPARC:                            |
|                                   |                                   |
|                                   | -   Client Profile (screen 6) for |
|                                   |     a current ADM_P item type     |
|                                   | -   the Pre-entry leave date can  |
|                                   |     be viewed in SPARC Client     |
|                                   |     Profile (screen 6) by         |
|                                   |     expanding the Adm P line      |
|                                   | -   View Means Tested Fees and    |
|                                   |     Refunds (screen 30) to        |
|                                   |     confirm ongoing fees          |
|                                   | -   Means Tested Cap Details      |
|                                   |     Screen (screen 33) for annual |
|                                   |     or lifetime caps              |
|                                   |                                   |
|                                   | Note: care recipients with an     |
|                                   | entry date prior to November 2018 |
|                                   | will have a new start date set    |
|                                   | when they change care types.      |
|                                   |                                   |
|                                   | Further investigation is required |
|                                   | when a care recipient:            |
|                                   |                                   |
|                                   | -   is paid MTCF in residential   |
|                                   |     care or ITCF in home care and |
|                                   | -   changed to a different type   |
|                                   |     of care prior to November     |
|                                   |     2018 and                      |
|                                   | -   in SPARC has reached an       |
|                                   |     annual cap or a lifetime cap  |
|                                   |     based on current fees         |
|                                   |                                   |
|                                   | Does the care recipient meet all  |
|                                   | of the above criteria?            |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   create a Fast Note in         |
|                                   |     Customer First. Select Auto   |
|                                   |     Text option Aged Care >       |
|                                   |     Enquiry> Aged Care            |
|                                   |     Annual/Lifetime Cap Enquiry   |
|                                   | -   advise that case has been     |
|                                   |     referred for investigation    |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check for previous manual         |
|                                   | adjustments + Read more ...       |
|                                   |                                   |
|                                   | In ACMPS, check:                  |
|                                   |                                   |
|                                   | -   Reconciliation Data tab for   |
|                                   |     any refunds that were not     |
|                                   |     paid                          |
|                                   | -   Payments Additions screen for |
|                                   |     any manual adjustments that   |
|                                   |     were paid                     |
|                                   | -   Payment Deductions screen for |
|                                   |     any manual recoveries that    |
|                                   |     were deducted                 |
|                                   | -   Notes tab for details of any  |
|                                   |     manual refunds that were paid |
|                                   |     and/or manual adjustments     |
|                                   |     that were made                |
|                                   |                                   |
|                                   | In SPARC, check:                  |
|                                   |                                   |
|                                   | -   Client Notes (screen 20.3)    |
|                                   |     for details of previous       |
|                                   |     refunds held/released         |
|                                   | -   Income Testing Letter Reprint |
|                                   |     Facility (screen 13) for any  |
|                                   |     letters not sent              |
|                                   | -   Manual Adjustments (screen    |
|                                   |     5.8) for details of manual    |
|                                   |     adjustments that were made    |
|                                   |                                   |
|                                   | Check Customer First:             |
|                                   |                                   |
|                                   | -   Document Tools                |
|                                   | -   Document List                 |
|                                   |                                   |
|                                   | Has a manual adjustment been      |
|                                   | completed previously for this     |
|                                   | care recipient?                   |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   create a Fast Note in         |
|                                   |     Customer First. Select Auto   |
|                                   |     Text option > Aged Care >     |
|                                   |     Enquiry > Aged Care           |
|                                   |     Annual/Lifetime Cap Enquiry   |
|                                   | -   advise that case has been     |
|                                   |     referred for investigation    |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Confirm SPARC /ACMPS ongoing      |
|                                   | fees + Read more ...              |
|                                   |                                   |
|                                   | The ongoing MTCF/ITCF fee should  |
|                                   | be $0 after an annual or lifetime |
|                                   | cap is reached and applied. If    |
|                                   | the ongoing fee has not been      |
|                                   | updated to $0, the assessment may |
|                                   | be pending the lodgement of a     |
|                                   | monthly claim by the service      |
|                                   | provider.                         |
|                                   |                                   |
|                                   | If a cap is reached in a month    |
|                                   | that has not been claimed by the  |
|                                   | service provider, the ongoing fee |
|                                   | will not have been updated. Once  |
|                                   | the monthly claim is lodged, the  |
|                                   | ongoing fee will then be set to   |
|                                   | $0 and any refund will be         |
|                                   | calculated at Quarterly Review.   |
|                                   |                                   |
|                                   | In SPARC check the Means Tested   |
|                                   | Cap Detail (screen 33).           |
|                                   |                                   |
|                                   | In ACMPS check the Fees and       |
|                                   | Charges tab.                      |
|                                   |                                   |
|                                   | Has an annual or lifetime cap     |
|                                   | been applied in ACMPS/SPARC?      |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   if the service provider's     |
|                                   |     claim month has not yet been  |
|                                   |     finalised, advise that the    |
|                                   |     ongoing fee will be adjusted  |
|                                   |     in due course/shortly,        |
|                                   |     procedure ends here           |
|                                   | -   go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Have annual or lifetime cap       |
|                                   | letters been issued + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | In SPARC check Income Testing     |
|                                   | Letter Reprint Facility (screen   |
|                                   | 13).                              |
|                                   |                                   |
|                                   | In ACMPS, check Communication     |
|                                   | History tab.                      |
|                                   |                                   |
|                                   | Has a letter been issued that the |
|                                   | care recipient has reached an     |
|                                   | annual or lifetime cap?           |
|                                   |                                   |
|                                   | Yes, reprint letter if required,  |
|                                   | procedure ends here               |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   create a Fast Note in         |
|                                   |     Customer First. Select Auto   |
|                                   |     Text option > Aged Care >     |
|                                   |     Enquiry > Aged Care           |
|                                   |     Annual/Lifetime Cap Enquiry   |
|                                   | -   advise that case has been     |
|                                   |     referred for investigation    |
|                                   | -   procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Calculate total MTCF/ITCF paid +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In SPARC, calculate the paid MTCF |
|                                   | for residential care:             |
|                                   |                                   |
|                                   | Client Summary (screen 37) to     |
|                                   | confirm:                          |
|                                   |                                   |
|                                   | -   current entry date            |
|                                   | -   anniversary date, and         |
|                                   | -   current fees                  |
|                                   |                                   |
|                                   | Means Tested Cap Details (screen  |
|                                   | 33) to see start date and history |
|                                   | of annual caps                    |
|                                   |                                   |
|                                   | Maintain Home Care Payments       |
|                                   | (screen 34) to confirm previous   |
|                                   | home care payments populated from |
|                                   | ACMPS (for entry after November   |
|                                   | 2018)                             |
|                                   |                                   |
|                                   | View Means Tested Fees and        |
|                                   | Refunds (screen 30) to confirm    |
|                                   | ongoing fees                      |
|                                   |                                   |
|                                   | Client Notes (screen 20.3) for    |
|                                   | details of previous refunds       |
|                                   | held/released                     |
|                                   |                                   |
|                                   | In ACMPS, calculate the paid ITCF |
|                                   | for home care fees:               |
|                                   |                                   |
|                                   | -   Fees and Charges tab under    |
|                                   |     Means Tested Fee for fees     |
|                                   |     that were charged and Annual  |
|                                   |     and Lifetime Caps tab for     |
|                                   |     accrued fees                  |
|                                   |                                   |
|                                   | Use the Aged Care Manual          |
|                                   | Adjustment calculator if          |
|                                   | required.                         |
|                                   |                                   |
|                                   | When the latest entry date is     |
|                                   | prior to November 2018, the       |
|                                   | following are not transferred     |
|                                   | into ACMPS/SPARC for care         |
|                                   | recipients who were previously in |
|                                   | a different care type:            |
|                                   |                                   |
|                                   | -   start date, and               |
|                                   | -   fees paid                     |
|                                   |                                   |
|                                   | A link to Department of Health -  |
|                                   | Schedules of Fees and Charges     |
|                                   | containing annual and lifetime    |
|                                   | cap rates can be found in         |
|                                   | Resources page.                   |
|                                   |                                   |
|                                   | Does this investigation show that |
|                                   | this care recipient may have      |
|                                   | reached an annual or lifetime     |
|                                   | cap?                              |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   create a Fast Note in         |
|                                   |     Customer First. Select Auto   |
|                                   |     Text option > Aged Care >     |
|                                   |     Enquiry > Aged Care           |
|                                   |     Annual/Lifetime Cap Enquiry   |
|                                   | -   advise that case has been     |
|                                   |     referred for investigation    |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   advise current annual and     |
|                                   |     lifetime cap amounts and      |
|                                   |     create a Fast Note in         |
|                                   |     Customer First to record      |
|                                   |     discussion                    |
|                                   | -   procedure ends here           |
