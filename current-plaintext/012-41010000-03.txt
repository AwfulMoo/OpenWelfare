Checking 90 Day cheque payments via mainframe

Table 6

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Cheque details screen + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Where 90 day cheques remain       |
|                                   | unpresented on the 91st day after |
|                                   | the date of issue, the cheques    |
|                                   | are automatically cancelled and   |
|                                   | the payment made by Electronic    |
|                                   | Funds Transfer (EFT). Payment     |
|                                   | progress can be checked via:      |
|                                   |                                   |
|                                   | -   NHSI/NHOI                     |
|                                   | -   BBEI/BBCI/BATI                |
+-----------------------------------+-----------------------------------+
| 2                                 | Security check and information    |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | -   Conduct a security check      |
|                                   | -   Get the Medicare card number  |
|                                   |     from the medical practitioner |
|                                   |     or the medial practitioner's  |
|                                   |     staff                         |
|                                   | -   Get any available payment     |
|                                   |     reference (Medicare number)   |
|                                   |     from the medical              |
|                                   |     practitioner's staff          |
|                                   |                                   |
|                                   | MC number details (for example    |
|                                   | MC123456789N021014):              |
|                                   |                                   |
|                                   | -   MC - a Medicare payment       |
|                                   | -   9 digit cheque number (such   |
|                                   |     as 12345678) - number of the  |
|                                   |     original cheque that has      |
|                                   |     since been cancelled          |
|                                   | -   state code (such as N) - the  |
|                                   |     state the cheque was issued   |
|                                   |     in                            |
|                                   | -   issue date (such as 191005) - |
|                                   |     the date the cheque was       |
|                                   |     originally issued             |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   The above example is used to  |
|                                   |     determine if a PDVC cheque    |
|                                   |     has been cancelled and an     |
|                                   |     Electronic Funds Transfer     |
|                                   |     (EFT) deposit made under the  |
|                                   |     90 Day PDC Scheme             |
|                                   | -   It is important the complete  |
|                                   |     reference number is used for  |
|                                   |     reconciliation                |
|                                   | -   90 Day Cheque Cancellation    |
|                                   |     Statements are printed        |
|                                   |     fortnightly                   |
+-----------------------------------+-----------------------------------+

|                                   |                                   |

|                                   |     9                             |
