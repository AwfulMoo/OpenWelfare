Child Dental Benefits Schedule (CDBS) frequently asked questions

This table contains frequently asked questions from Service Officers,
and answers relating to Child Dental Benefits Schedule (CDBS) enquiries.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: How do I know who I can |
|                                   | release information to?           |
|                                   |                                   |
|                                   | Answer: Service Officers can      |
|                                   | release CDBS eligibility and      |
|                                   | balance information to a parent   |
|                                   | or guardian:                      |
|                                   |                                   |
|                                   | -   that is recorded in CDMS      |
|                                   |     (CDBS) in the parent/guardian |
|                                   |     tab, or                       |
|                                   | -   if there are nominee or       |
|                                   |     Person Permitted to Enquire   |
|                                   |     (PPE) arrangements are        |
|                                   |     recorded in Customer First on |
|                                   |     the home screen               |
|                                   |                                   |
|                                   | Service Officers can release this |
|                                   | information to a parent or        |
|                                   | guardian until the eligible child |
|                                   | turns 18 years old. No claiming   |
|                                   | or other history information can  |
|                                   | be released without the consent   |
|                                   | of the child if they are 14 years |
|                                   | or older.                         |
|                                   |                                   |
|                                   | A non-recipient parent or         |
|                                   | guardian who is the current       |
|                                   | partner of an FTB(A) recipient    |
|                                   | cannot be given information       |
|                                   | unless nominee or PPE is          |
|                                   | recorded.                         |
|                                   |                                   |
|                                   | Service Officers cannot disclose  |
|                                   | information to a third party      |
|                                   | unless there is nominee or PPE    |
|                                   | recorded.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Can I give information  |
|                                   | about a child’s eligibility to a  |
|                                   | partner?                          |
|                                   |                                   |
|                                   | Answer: Yes, in some instances.   |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Roger calls to see if his child   |
|                                   | Tony, who is 15, is eligible for  |
|                                   | CDBS and enquire on Tony’s        |
|                                   | remaining balance.                |
|                                   |                                   |
|                                   | Tony is eligible for CDBS as his  |
|                                   | mum, Sarah, is receiving a Family |
|                                   | Tax Benefit FTB (A).              |
|                                   |                                   |
|                                   | A check of CDBS in CDMS shows     |
|                                   | Sarah is listed as the parent or  |
|                                   | guardian. When checking Customer  |
|                                   | First it shows Roger and Sarah    |
|                                   | are in a relationship. Roger has  |
|                                   | been recorded as a Person         |
|                                   | Permitted to Enquire (PPE).       |
|                                   |                                   |
|                                   | In this instance, Service         |
|                                   | Officers can release this         |
|                                   | information to Roger.             |
|                                   |                                   |
|                                   | Note: if Customer First shows     |
|                                   | there was no PPE indicator or     |
|                                   | they are separated, the only      |
|                                   | information Services Australia    |
|                                   | can release to Roger is general   |
|                                   | eligibility criteria for CDBS.    |
|                                   | Services Australia cannot release |
|                                   | any other information.            |
+-----------------------------------+-----------------------------------+
