Approved Care Organisation (ACO) Spreadsheet

Once an organisation has been approved an Approved Care Organisation
(ACO), the ACO Smart Centre Processing team will issue the admission,
transfer or discharge spreadsheet by email. The spreadsheet must be used
by the organisation to update admission (claim), transfer or discharge
details of a young person. The completed spreadsheet is then emailed to
the ACO Smart Centre Processing team.

Example of a referral of Special Benefit (SpB) claim to social worker

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Example of Service Officer        |
|                                   | referral of SpB claim to social   |
|                                   | worker                            |
|                                   |                                   |
|                                   | Trevor is an unaccompanied        |
|                                   | humanitarian minor (UHM) aged 14  |
|                                   | years 2 months. Trevor shares a   |
|                                   | house with other UHMs he met      |
|                                   | after arriving in Australia. The  |
|                                   | guardianship of Trevor has been   |
|                                   | transferred to a state based      |
|                                   | support organisation. A case      |
|                                   | worker for this support           |
|                                   | organisation contacts Services    |
|                                   | Australia advising they only      |
|                                   | provide case management and are   |
|                                   | not funded to financially support |
|                                   | Trevor. Other UHM living in the   |
|                                   | house with Trevor are in receipt  |
|                                   | of Youth Allowance (YA). The case |
|                                   | worker advises Trevor is          |
|                                   | experiencing hardship and wants   |
|                                   | to claim for financial            |
|                                   | assistance.                       |
|                                   |                                   |
|                                   | The Service Officer makes         |
|                                   | necessary enquiries to determine  |
|                                   | if an authorised party, either a  |
|                                   | foster carer or an Approved Care  |
|                                   | Organisation (ACO) can make a     |
|                                   | claim for Family Tax Benefit      |
|                                   | (FTB) for Trevor. The Service     |
|                                   | Officer asks if other material    |
|                                   | support is being provided to      |
|                                   | Trevor. This can be state         |
|                                   | government welfare payments, or   |
|                                   | UHM Program Support Allowance     |
|                                   | (UPSA, previously known as        |
|                                   | Maintenance Allowance for         |
|                                   | Refugees) from the Department of  |
|                                   | Home Affairs. (The Services       |
|                                   | Australia ICT system still        |
|                                   | displays the name Maintenance     |
|                                   | Allowance for Refugees).          |
|                                   |                                   |
|                                   | The Service Officer is advised    |
|                                   | there is no state/territory       |
|                                   | support or UPSA provided and no   |
|                                   | authorised party can make an FTB  |
|                                   | claim for him.                    |
|                                   |                                   |
|                                   | Service Officer refers Trevor and |
|                                   | his case worker to a Services     |
|                                   | Australia social worker to assess |
|                                   | if Trevor is eligible for SpB.    |
+-----------------------------------+-----------------------------------+

Examples in determining if customer has sufficient funds

This table shows some examples to consider if a customer has sufficient
funds, but not limited too.

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer has sufficient funds     |
|                                   | available for regular             |
|                                   | accommodation costs such as rent  |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer is able to pay regular   |
|                                   | bills such as gas, electricity,   |
|                                   | telephone or water bills          |
+-----------------------------------+-----------------------------------+
| 3                                 | Customer can afford food and      |
|                                   | general living expenses           |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer can afford public and    |
|                                   | private transport costs           |
+-----------------------------------+-----------------------------------+
| 5                                 | If customer has a vehicle,        |
|                                   | customer can afford car           |
|                                   | maintenance, registration or      |
|                                   | petrol costs                      |
+-----------------------------------+-----------------------------------+
