Aged Care Staff Portal (ACSP) frequently asked questions and answers

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: What is the Aged Care   |
|                                   | Staff Portal (ACSP)?              |
|                                   |                                   |
|                                   | Answer: Service Officers can use  |
|                                   | the Aged Care Staff Portal (ACSP) |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   process events and claims for |
|                                   |     home care and flexible care   |
|                                   |     services                      |
|                                   | -   search for services within    |
|                                   |     all care types                |
|                                   | -   register invoice amounts,     |
|                                   |     report Commonwealth (CW)      |
|                                   |     unspent amounts and view a    |
|                                   |     care recipient’s home care    |
|                                   |     account for home care         |
|                                   |     purposes only                 |
|                                   | -   approve home and flexible     |
|                                   |     care claims                   |
|                                   | -   manage and request            |
|                                   |     registration of service user  |
|                                   |     details, and                  |
|                                   | -   manage and register service   |
|                                   |     contact details               |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   A flexible care service can   |
|                                   |     either be a short-term        |
|                                   |     restorative care or a         |
|                                   |     transition care service       |
|                                   | -   The processing of residential |
|                                   |     care events and claims must   |
|                                   |     continue to occur in the      |
|                                   |     System for the Payment of     |
|                                   |     Aged Residential Care (SPARC) |
|                                   | -   This ensures the correct      |
|                                   |     processing of aged care       |
|                                   |     subsidies and supplements for |
|                                   |     all care types                |
|                                   | -   Service Officers can use the  |
|                                   |     ACSP via 2 different views -  |
|                                   |     Service in Context and Care   |
|                                   |     Recipient in Context (CRiC).  |
|                                   |     The Service in Context view   |
|                                   |     providers Service Officers    |
|                                   |     information from a service’s  |
|                                   |     perspective. The CRiC view    |
|                                   |     provides Service Officers a   |
|                                   |     more holistic view from a     |
|                                   |     care recipient’s perspective  |
|                                   |     compared to the Service in    |
|                                   |     Context view. That is, they   |
|                                   |     are presented with care       |
|                                   |     recipient information across  |
|                                   |     their journey within various  |
|                                   |     aged care services            |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Who can use the ACSP?   |
|                                   |                                   |
|                                   | Answer: To access the ACSP,       |
|                                   | Service Officers must have the    |
|                                   | below role and resource:          |
|                                   |                                   |
|                                   | -   The AGED CARE - SERVICE       |
|                                   |     OFFICER role (security role)  |
|                                   |     via ESSentials                |
|                                   | -   The ac_staff_portal Tivoli    |
|                                   |     Access Manager (TAM) resource |
|                                   |     via mySupport                 |
|                                   |                                   |
|                                   | See Table 1 on the Process page   |
|                                   | for how to request access to the  |
|                                   | Aged Care Staff Portal and the    |
|                                   | access request process.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How can I access the    |
|                                   | ACSP?                             |
|                                   |                                   |
|                                   | Answer: Service Officers can      |
|                                   | access the ACSP via the below     |
|                                   | link:                             |
|                                   |                                   |
|                                   | Aged Care Staff Portal            |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Where do I go for       |
|                                   | support if I cannot access the    |
|                                   | ACSP?                             |
|                                   |                                   |
|                                   | Answer: Service Officers can see  |
|                                   | Table 2 on the Process page for   |
|                                   | trouble-shooting the Aged Care    |
|                                   | Staff Portal process to resolve   |
|                                   | ACSP access issues.               |
