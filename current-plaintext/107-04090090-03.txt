Actioning an undetermined qualification debt

Note: due to the pause on debt raising and recover, Steps 1, 2 and 3
will not apply until further notice unless it is necessary to raise the
qualification debt to apply a waiver.

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Refer to Debt Raising Team + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Refer to the Debt Management Team |
|                                   | to finalise the undetermined FTB  |
|                                   | qualification debt(s). Send an    |
|                                   | ACTDOC using the Debt Raising MIV |
|                                   | FFRECO Request Fast Note          |
|                                   | Template:                         |
|                                   |                                   |
|                                   | -   Include the relevant debt ID  |
|                                   |     number(s)                     |
|                                   | -   Transfer to region code XWA   |
|                                   |                                   |
|                                   | The Debt Management Team will     |
|                                   | determine the debt shell within 3 |
|                                   | working days. An ACTDOC will be   |
|                                   | returned to the Smart Centre for  |
|                                   | the original activity to be       |
|                                   | re-coded and finalised.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Advise the customer: + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   the timeframe to have their   |
|                                   |     FTB reconciliation finalised  |
|                                   | -   they will be sent a letter    |
|                                   |     once their FTB reconciliation |
|                                   |     has been completed or they    |
|                                   |     can view the results using    |
|                                   |     Online Services               |
+-----------------------------------+-----------------------------------+
| 3                                 | Check for an ACTDOC from Debt     |
|                                   | Raising Team + Read more ...      |
|                                   |                                   |
|                                   | Regularly check Workload          |
|                                   | Management. Finalise the          |
|                                   | reconciliation activity as soon   |
|                                   | as the XWA team have determined   |
|                                   | the debt/s.                       |
|                                   |                                   |
|                                   | The Service Officer/Service       |
|                                   | Support Officer is required to    |
|                                   | regularly check Workload          |
|                                   | Management in Customer First to   |
|                                   | ensure that the reconciliation    |
|                                   | activity is finalised as soon as  |
|                                   | the Debt Raising Team has         |
|                                   | determined the debt/s.            |
+-----------------------------------+-----------------------------------+
| 4                                 | Debt waiver provisions + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Raise a manual debt if the debt   |
|                                   | meets waiver provisions contained |
|                                   | in Section 97 or 101 of the       |
|                                   | Family Assistance                 |
|                                   | (Administration) Act 1999.        |
|                                   | Include the same information as   |
