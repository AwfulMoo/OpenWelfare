Changing study details to cancel payment when ceasing studies

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notification requirements + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Students must advise they have    |
|                                   | ceased study within their         |
|                                   | notification period.              |
|                                   |                                   |
|                                   | Statement reporters must notify   |
|                                   | changes on or before the          |
|                                   | Entitlement Period End Date       |
|                                   | (EPED) in which the change        |
|                                   | occurred.                         |
|                                   |                                   |
|                                   | Notification reporters must       |
|                                   | notify changes within 14 days.    |
|                                   |                                   |
|                                   | Has the customer advised within   |
|                                   | their notification provisions?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
|                                   |                                   |
|                                   | Note: customers who contact the   |
|                                   | agency early and wish to advise   |
|                                   | that they are ceasing studies on  |
|                                   | a future date, will be always     |
|                                   | considered to have met their      |
|                                   | notification provisions.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Notification: customer has not    |
|                                   | advised in accordance with their  |
|                                   | notification period that they     |
|                                   | have ceased study + Read more ... |
|                                   |                                   |
|                                   | As the customer has failed to     |
|                                   | notify a change in circumstances  |
|                                   | within their notification period, |
|                                   | they are no longer eligible for   |
|                                   | payment. Update the customer      |
|                                   | record as follows:                |
|                                   |                                   |
|                                   | navigate to Education Course      |
|                                   | History (EDCH) screen and correct |
|                                   | the Student End Date: field, to:  |
|                                   |                                   |
|                                   | -   the date last studied, or     |
|                                   | -   the date the customer ceased  |
|                                   |     to have an intention to       |
|                                   |     return to study               |
|                                   |                                   |
|                                   | insert a Newstart Education       |
|                                   | Status (NES) entry:               |
|                                   |                                   |
|                                   | -   Event Date: field, code the   |
|                                   |     student end date plus 1 day   |
|                                   | -   Student Status: field, code   |
|                                   |     'NST' (not a full-time        |
|                                   |     student)                      |
|                                   | -   Education Level Attained:     |
|                                   |     field, update if necessary    |
|                                   |                                   |
|                                   | For customers under 22 years:     |
|                                   |                                   |
|                                   | -   Navigate to the Benefit       |
|                                   |     Action (BA) screen            |
|                                   | -   Service Reason: field, select |
|                                   |     the customers payment type    |
|                                   | -   Action: field, CAN - Cancel   |
|                                   | -   Reason: field 'NST' - not a   |
|                                   |     full-time student             |
|                                   | -   Effect Date: field, new study |
|                                   |     end date plus 1 day.          |
|                                   |                                   |
|                                   | Note: customers 22 years of age   |
|                                   | or over will not require manual   |
|                                   | cancellation via the BA screen.   |
|                                   | These customers will have their   |
|                                   | payment automatically cancelled   |
|                                   | from the updates made to NES and  |
|                                   | EDCH.                             |
|                                   |                                   |
|                                   | Customers wishing to receive      |
|                                   | further financial assistance must |
|                                   | lodge a new claim.                |
|                                   |                                   |
|                                   | Record the details of the         |
|                                   | decision to cancel on a DOC.      |
|                                   |                                   |
|                                   | Go to Step 8.                     |
