Administering the Citizenship test 18 to 59 years (inclusive)

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access test (applicant aged       |
|                                   | 18-59) + Read more ...            |
|                                   |                                   |
|                                   | Access the test by logging on to  |
|                                   | the Department of Home Affairs    |
|                                   | system.                           |
|                                   |                                   |
|                                   | Department of Home Affairs        |
|                                   | provide a URL address, staff log  |
|                                   | on and password during staff      |
|                                   | training.                         |
|                                   |                                   |
|                                   | Each applicant has a separate log |
|                                   | on and ID number as detailed in   |
|                                   | the Daily Regional Attendees List |
|                                   | provided in the Department of     |
|                                   | Home Affairs weekly email.        |
|                                   |                                   |
|                                   | Staff can log into the test       |
|                                   | before the applicant comes into   |
|                                   | the citizenship testing area so   |
|                                   | it is ready for them to begin the |
|                                   | test.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Applicant to confirm name on      |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Once logged in, the applicant's   |
|                                   | name will be visible on the       |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Turn the screen so the applicant  |
|                                   | can see the test. Ask them to     |
|                                   | confirm it is their name showing  |
|                                   | on the screen.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Explain test rules (applicant     |
|                                   | aged 18-59) + Read more ...       |
|                                   |                                   |
|                                   | -   there are 20 randomly         |
|                                   |     generated questions with 3    |
|                                   |     possible answers              |
|                                   | -   they are not allowed to refer |
|                                   |     to any resources              |
|                                   | -   mobiles should be on silent   |
|                                   |     or switched off               |
|                                   | -   they have 45 minutes to       |
|                                   |     complete the test             |
|                                   | -   they can review their answers |
|                                   |     by moving backwards and       |
|                                   |     forwards in the test          |
|                                   | -   they need to achieve 75%,     |
|                                   |     including correct answers for |
|                                   |     all 5 mandatory Australian    |
|                                   |     values questions to pass the  |
|                                   |     test                          |
|                                   | -   if they ask to listen to the  |
|                                   |     test questions, provide       |
|                                   |     headphones                    |
|                                   | -   provide assistance if they    |
|                                   |     need help with operating the  |
|                                   |     iPad, but the applicant must  |
|                                   |     indicate the answer they      |
|                                   |     think is correct              |
|                                   | -   advise the applicant to stop  |
|                                   |     immediately if they attempt   |
|                                   |     to cheat. Do not proceed with |
|                                   |     the test                      |
|                                   | -   they should notify you when   |
|                                   |     the test is completed         |
|                                   |                                   |
|                                   | While the applicant is taking the |
|                                   | test, staff must supervise the    |
|                                   | test and must not leave the room. |
+-----------------------------------+-----------------------------------+
| 4                                 | On completion of test + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Upon completion of the test,      |
|                                   | advise the applicant if they are  |
|                                   | satisfied with their answers, to  |
|                                   | select Submit then OK.            |
|                                   |                                   |
|                                   | The results will display on the   |
|                                   | applicant's screen. Select        |
|                                   | finish, the results will be       |
|                                   | transmitted to Department of Home |
|                                   | Affairs.                          |
|                                   |                                   |
|                                   | If the applicant fails the test,  |
|                                   | offer a resit if time permits.    |
|                                   | Applicants can sit the test a     |
|                                   | maximum of 3 times at each        |
|                                   | appointment.                      |
|                                   |                                   |
|                                   | Failing the citizenship test does |
|                                   | not affect their permanent        |
|                                   | residence visa or their ability   |
|                                   | to continue living in Australia.  |
|                                   |                                   |
|                                   | To continue with interview see    |
|                                   | Step 1 in Table 4.                |
