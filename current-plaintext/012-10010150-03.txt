This page contains the process when Specialised Provider Services (SPS)
action a privacy incident for Incentive programs Practice Incentive
Program (PIP) and Workforce Incentive Program (WIP) - Practice Stream.

Privacy incident identified by SPS

Action a privacy incident identified by SPS

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Complete and submit a Privacy     |
|                                   | Incident form + Read more ...     |
|                                   |                                   |
|                                   | When SPS identify a privacy       |
|                                   | incident, a Service Officer must  |
|                                   | complete a Report a Privacy       |
|                                   | Incident form. See the Resources  |
|                                   | page for a link.                  |
|                                   |                                   |
|                                   | To ensure the form is completed   |
|                                   | correctly, see Privacy Incidents  |
|                                   | FAQ 14b in Privacy and Secrecy.   |
|                                   |                                   |
|                                   | Note: Team Leader/Manager email   |
|                                   | address must be included.         |
+-----------------------------------+-----------------------------------+
| 2                                 | Programme Advice and Privacy      |
|                                   | Branch + Read more ...            |
|                                   |                                   |
|                                   | The privacy team will:            |
|                                   |                                   |
|                                   | -   assess the information to     |
|                                   |     determine if a privacy        |
|                                   |     incident has occurred, and    |
|                                   | -   determine the level and       |
|                                   |     severity (for example no      |
|                                   |     incident at all, minor,       |
|                                   |     moderate or major)            |
|                                   | -   notify the staff member who   |
|                                   |     submitted the form of the     |
|                                   |     outcome and any further       |
|                                   |     action that needs to be       |
|                                   |     taken.                        |
|                                   |                                   |
|                                   | Note: if the Privacy team require |
|                                   | additional information they will  |
|                                   | contact the service officer or    |
|                                   | the manager. The Resources page   |
|                                   | contains to the contact details   |
|                                   | for Privacy.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Notify Program Management + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Send a copy of the Report a       |
|                                   | Privacy Incident form and advice  |
|                                   | from the privacy team to Program  |
|                                   | Management for information. This  |
|                                   | is to be sent by email.           |
+-----------------------------------+-----------------------------------+
