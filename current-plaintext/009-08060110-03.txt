Review process

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | This review affects the carer's   |
|                                   | payment but may also involve the  |
|                                   | care receiver + Read more ...     |
|                                   |                                   |
|                                   | [                                 |
|                                   | \\INTERNAL.DEPT.LOCAL\Shared\NAT\ |

|                                   |                                   |
|                                   | The care receiver may need to     |
|                                   | claim Carer Allowance (CA) for    |
|                                   | their dependent child if          |
|                                   | appropriate, when the child turns |
|                                   | 6 years for the CP to continue to |
|                                   | be paid to the carer.             |
|                                   |                                   |
|                                   | If CP is to be cancelled, the     |
|                                   | carer may need to consider        |
|                                   | another payment.                  |
|                                   |                                   |
|                                   | The review will present even if   |
|                                   | the dependent child (or another   |
|                                   | dependent child of the care       |
|                                   | receiver) already attracts CA, or |
|                                   | if the care receiver has another  |
|                                   | dependent child under the age of  |
|                                   | 6 years.                          |
|                                   |                                   |
|                                   | A CAR/AAP activity will appear on |
|                                   | the carer's Activity List (AL)    |
|                                   | screen when the child is 5 years  |
|                                   | and 9 months of age.              |
|                                   |                                   |
|                                   | The notes will read:              |
|                                   |                                   |
|                                   | -   'Lower ADAT Adult Caree. The  |
|                                   |     Adult Care Receiver must be   |
|                                   |     paid CA for a child over six. |
|                                   |     Contact the CAR customer to   |
|                                   |     explain review.               |
|                                   |                                   |
|                                   | Once the child turns 6, another   |
|                                   | activity will mature from the     |
|                                   | Future Activity List (FAL) screen |
|                                   | and reassess the record. If there |
|                                   | is no qualifying child either     |
|                                   | under 6 or over 6 and attracting  |
|                                   | CA for the care receiver, CP for  |
|                                   | the carer will be cancelled with  |
|                                   | Reason:                           |
|                                   |                                   |
|                                   | -   FAD, Caree did not meet ADAT  |
|                                   |     score, or                     |
|                                   | -   HPA, if the HPA score is less |
|                                   |     than 8                        |
|                                   |                                   |
|                                   | An auto advice will be sent.      |

|                                   |                                   |
|                                   | -   a dependent child that        |

|                                   | -   another dependent child that  |
|                                   |     already attracts CA, or       |
|                                   | -   another dependent child under |
|                                   |     the age of 6 years            |
|                                   |                                   |
|                                   | Otherwise, go to Step 3.          |

|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Advise customer of new claim      |
|                                   | process + Read more ...           |

|                                   |                                   |
|                                   | Go to Step 6.                     |

| 5                                 | Deterioration in medical          |
|                                   | condition + Read more ...         |
|                                   |                                   |

|                                   | No, the CP will automatically     |
