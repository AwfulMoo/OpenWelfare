Business rules community nursing

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Rule                              |
+-----------------------------------+-----------------------------------+
| 1                                 | The provider specialty code C83   |
|                                   | 205 will be used for              |
|                                   | DVA-contracted community nursing  |
|                                   | (CN) providers extending their    |
|                                   | current Deed of Standing Offers   |
|                                   | past 1 October 2014.              |
+-----------------------------------+-----------------------------------+
| 2                                 | CN providers are paid through a   |
|                                   | Schedule of Item Numbers and      |
|                                   | Fees.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | A referral is required from a     |
|                                   | General Practitioner (GP),        |
|                                   | Specialist (treating doctor in    |
|                                   | hospital), hospital discharge     |
|                                   | planner, Nurse Practitioner, or   |
|                                   | Veterans’ Home Care (VHC)         |
|                                   | assessment agency, on admission   |
|                                   | per episode of care.              |
+-----------------------------------+-----------------------------------+
| 4                                 | An episode of care is defined as  |
|                                   | the period between the date of    |
|                                   | admission to and the date of      |
|                                   | discharge from community nursing  |
|                                   | services. There may be several    |
|                                   | episodes of care in one year, or  |
|                                   | an episode of care may be         |
|                                   | indefinite.                       |
+-----------------------------------+-----------------------------------+
| 5                                 | Payments are based on an ‘episode |
|                                   | of care’ which is divided into    |
|                                   | 28-day claim periods.             |
+-----------------------------------+-----------------------------------+
| 6                                 | CN providers may be paid an       |
|                                   | additional travel allowance when  |
|                                   | claimed through NA10.             |
+-----------------------------------+-----------------------------------+
| 7                                 | If an entitled person transfers   |
|                                   | between CN providers, the new CN  |
|                                   | provider is required to obtain a  |
|                                   | referral.                         |
+-----------------------------------+-----------------------------------+
| 8                                 | Only one item from either the     |
|                                   | Clinical Care Core (NL01 - NL29)  |
|                                   | or Personal Care Core (NP01 -     |
|                                   | NP31) items may be claimed in a   |
|                                   | 28-day claim period per provider  |
|                                   | (maximum 2 providers in a claim   |
|                                   | period). Where appropriate and in |
|                                   | accordance with the business      |
|                                   | rules, add-ons may be claimed in  |
|                                   | addition to the core item.        |
+-----------------------------------+-----------------------------------+
| 9                                 | An add-on may be claimed based    |
|                                   | upon the visit count or upon the  |
|                                   | time (hours and/or minutes).      |
|                                   | Given the visit bandings from the |
|                                   | opposing core and add-on          |
|                                   | schedules do not align, there     |
|                                   | will be instances where the       |
|                                   | bandings 'overlap', these will be |
|                                   | monitored through DVA post        |
|                                   | payment monitoring (PPM) of MDS,  |
|                                   | rather than through item          |
|                                   | restrictions.                     |
+-----------------------------------+-----------------------------------+
| 10                                | Additional/other items may be     |
|                                   | claimed with core and exceptional |
|                                   | case items as required, in        |
|                                   | accordance with the business      |
|                                   | rules.                            |
+-----------------------------------+-----------------------------------+
| 11                                | Classifications (item numbers)    |
|                                   | can change during an episode of   |
|                                   | care.                             |
+-----------------------------------+-----------------------------------+
