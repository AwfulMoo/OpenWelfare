Table 5: when patient claims submitted for payment are over 2 years old
from the date of services.

+-----------------------------------+-----------------------------------+
| Patient claim…                    | Action                            |
+-----------------------------------+-----------------------------------+
| over 2 years old (less than 5     | Medicare benefit less than $100   |
| years old)                        |                                   |
|                                   | Claim must be supported by an     |
| Medicare benefits less than $100  | original or duplicate account or  |
| (paid for 1 claim for 1 Medicare  | receipt for payment.              |
| card number)                      |                                   |
|                                   | The claim can then be processed   |
|                                   | and paid without reference to     |
|                                   | culled history.                   |
+-----------------------------------+-----------------------------------+
| over 2 years old (less than 5     | Medicare benefit more than $100   |
| years old)                        |                                   |
|                                   | Check the culled history for any  |
| Medicare benefit more than $100   | duplicate payments. If:           |
| (paid for 1 claim for 1 Medicare  |                                   |
| card number)                      | there is no evidence of a         |
|                                   | duplicate payment on the culled   |
|                                   | history:                          |
|                                   |                                   |
|                                   | -   the claim can be processed    |
|                                   |     and paid                      |
|                                   |                                   |
|                                   | if cannot be determined whether a |
|                                   | previous payment exists:          |
|                                   |                                   |
|                                   | -   If it is a manual paper       |
|                                   |     claim, scan and email all the |
|                                   |     claim documents to MPS        |
|                                   |     Assessing where it will be    |
