|                                   |                                   |
|                                   | Note: batches should not exceed   |
|                                   | 300 pages. For very large         |
|                                   | documents, scan as a separate     |
|                                   | batch or add to a batch with      |
|                                   | fewer pages.                      |

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |

| 1                                 | Image correspondence (postal or   |
|                                   | hand delivered) + Read more ...   |
|                                   |                                   |
|                                   | Paper documents must be scanned   |
|                                   | within 48 hours of arrival in an  |
|                                   | office. Image documents using the |
|                                   | Fuji Xerox AP6000 multi-function  |
|                                   | device (MFD).                     |
|                                   |                                   |
|                                   | Clean MFD                         |
|                                   |                                   |
|                                   | Clean the MFD scanner areas       |
|                                   | daily:                            |
|                                   |                                   |
|                                   | -   follow the instructions on    |
|                                   |     the cover of the MFD          |
|                                   | -   dust off any paper 'crumbs'   |
|                                   |     from guillotined, spliced and |
|                                   |     perforated documents that may |
|                                   |     build up in the lid of the    |
|                                   |     MFD. These may impact the     |
|                                   |     image quality. For example,   |
|                                   |     the inside upper left hand    |
|                                   |     strip of the MFD              |
|                                   | -   clear away any documents left |
|                                   |     behind from other people      |
|                                   |     photocopying etc, to ensure   |
|                                   |     they do not accidentally get  |
|                                   |     scanned                       |
|                                   | -   check that there are no error |
|                                   |     messages on either machine    |
+-----------------------------------+-----------------------------------+
| 2                                 | Check documents before scanning + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Before scanning commences:        |
|                                   |                                   |
|                                   | sort and match like size          |
|                                   | documents into the following      |
|                                   | batches:                          |
|                                   |                                   |
|                                   | -   standard A4 paged documents   |
|                                   | -   any other mixed               |
|                                   |     correspondence/forms          |
|                                   |                                   |
|                                   | remove staples/splice forms       |
|                                   |                                   |
|                                   | If the documents are being        |
|                                   | scanned on the date of receipt:   |
|                                   |                                   |
|                                   | -   ensure the front page is face |
|                                   |     up, and                       |
|                                   | -   place a barcode separator     |
|                                   |     sheet between each document   |
|                                   |                                   |
|                                   | If the documents are being        |
|                                   | scanned on a day after the date   |
|                                   | of receipt:                       |
|                                   |                                   |
|                                   | -   collate documents by the date |
|                                   |     they were received            |
|                                   | -   place a barcode separator     |
|                                   |     sheet between each document   |
|                                   | -   place an 'older date' barcode |
|                                   |     sheet at the beginning of the |
|                                   |     batch. See folder within DST  |
|                                   |     team or seek TL support       |
|                                   | -   scan the cover sheet as the   |
|                                   |     first page because it informs |
|                                   |     the software what date to     |
|                                   |     apply to the documents in     |
|                                   |     that batch                    |

|                                   |                                   |

|                                   | press to turn the MFD on          |
|                                   |                                   |
|                                   | Note: if scanning a relatively    |
|                                   | small document, be ready to       |
|                                   | record the document number on the |
|                                   | batch file cover sheet. This      |
|                                   | number will be displayed on the   |
|                                   | top left hand side of the screen  |
|                                   | when scanning commences.          |
