|                                   | conducted by the provider when a  |
|                                   | job seeker gets 3 demerits in 6   |
|                                   | active months. In this interview  |
|                                   | the provider and job seeker       |
|                                   | review:                           |
|                                   |                                   |
|                                   | -   activities in the Job Plan,   |
|                                   |     and                           |
|                                   | -   the job seekers circumstances |
|                                   |                                   |
|                                   | The Capability Interview ensures  |
|                                   | the provider tailors the Job Plan |
|                                   | to ensure it is achievable given  |
|                                   | the job seeker's circumstances.   |
|                                   |                                   |

|                                   | the PST-skilled Service Officer   |
|                                   | must discuss the outcomes from    |
|                                   | the Capability Interview.         |
|                                   |                                   |
|                                   | The Service Officer selects view  |
|                                   | summary of Capability Interview   |
|                                   | in the Capability Interview       |
|                                   | summary page in order to review   |
|                                   | the outcomes of that              |
|                                   | conversation. If the Service      |
|                                   | Officer needs any clarification   |
|                                   | about the details of the          |
|                                   | Capability Interview they should  |
|                                   | contact the provider.             |
|                                   |                                   |
|                                   | Contact with the provider must be |
|                                   | recorded on the Capability        |
|                                   | Assessment Outcomes page.         |
|                                   |                                   |
|                                   | Note: job seekers who query the   |
|                                   | recent non-compliance and demerit |

|                                   | Capability Interview must be      |

|                                   | to discuss. See Non-compliance    |

|                                   | review and appeals.               |

|                                   | requirements. These may be        |
|                                   | circumstances which:              |
|                                   |                                   |
|                                   | -   are newly identified: the job |
|                                   |     seeker has not disclosed      |
|                                   |     these circumstances to their  |
|                                   |     provider in the Capability    |
|                                   |     Interview or during other     |
|                                   |     contacts, or                  |
|                                   | -   are ongoing: the              |
|                                   |     circumstances may have been   |
|                                   |     disclosed previously and are  |
|                                   |     continuing to, or have        |
|                                   |     changed in the way they       |
|                                   |     impact the job seeker's       |
|                                   |     capacity                      |
|                                   |                                   |
|                                   | If the job seeker has any         |
|                                   | circumstances impacting their     |
|                                   | capacity to comply, the           |
|                                   | PST-skilled Service Officer must: |

|                                   |     free text box that presents   |
|                                   |     in this section of the        |
|                                   |     workflow                      |
|                                   |                                   |
|                                   | Completing 'Yes - has ongoing     |
|                                   | difficulties meeting              |
