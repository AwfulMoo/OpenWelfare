| 8                                 | Accepting the customer's estimate |
|                                   | for each assessable property +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Risk based model                  |
|                                   |                                   |
|                                   | A risk based model for assessment |
|                                   | of real estate values             |
|                                   | automatically assesses a          |
|                                   | customer’s individual             |
|                                   | circumstances to determine the    |
|                                   | appropriate ongoing method of     |
|                                   | assessment for their property.    |
|                                   | This may be:                      |
|                                   |                                   |
|                                   | -   acceptance of the customer's  |
|                                   |     estimated value               |
|                                   | -   a check of the customer's     |
|                                   |     estimated value               |
|                                   |                                   |
|                                   | The risk based model will be      |
|                                   | applied to most properties, such  |
|                                   | as houses, units and townhouses,  |
|                                   | and the system will automatically |
|                                   | initiate the process to determine |
|                                   | if a customer’s estimate is       |
|                                   | acceptable or a valuation is      |
|                                   | required.                         |
|                                   |                                   |
|                                   | Other properties are also         |
|                                   | included such as:                 |
|                                   |                                   |
|                                   | -   RET: retail premises          |
|                                   | -   CMP: commercial premises      |
|                                   | -   IDP: industrial premises      |
|                                   | -   FRM: farm                     |
|                                   | -   MGD: market garden            |
|                                   | -   HBF: hobby farm               |
|                                   | -   BHB: bush block               |
|                                   | -   HMB: home block on greater    |
|                                   |     than 2 hectares               |
|                                   | -   VAC: vacant blocks            |
|                                   |                                   |
|                                   | Not included in the risk based    |
|                                   | model                             |
|                                   |                                   |
|                                   | Some property is not included in  |
|                                   | the risk based model and will     |
|                                   | require a valuation request to    |
|                                   | ensure the customers assets are   |
|                                   | assessed correctly. These can     |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   overseas properties           |
|                                   | -   property that has been gifted |
|                                   | -   a persons home when they      |
|                                   |     enter residential care        |
|                                   | -   property considered as Other  |
|                                   |     (OTH) such as car parks and   |
|                                   |     marina berths                 |
|                                   | -   properties that have never    |
|                                   |     been advised and historical   |
|                                   |     valuations are required       |
|                                   | -   self contained properties     |
|                                   |     (SCF)                         |
|                                   |                                   |
|                                   | Is the customer's property type   |
|                                   | included in the risk based model? |
|                                   |                                   |
|                                   | -   Yes, update the estimated     |
|                                   |     value on the RE screen with   |
|                                   |     an Asset Valuation Source:    |
|                                   |     CLI. Go to Step 11            |
|                                   | -   No, a valuation is required,  |
|                                   |     go to Step 10                 |
|                                   |                                   |
|                                   | Note: if the customer's claim is  |
|                                   | to be rejected or an RFI is       |
|                                   | required for the claim, do not    |
|                                   | update REBS until all details for |
|                                   | the claim have been returned.     |
|                                   | Valuations are at a cost to the   |
|                                   | agency and do not need to be      |
|                                   | requested for customers that are  |
|                                   | not receiving or entitled to      |
|                                   | receive a payment.                |
