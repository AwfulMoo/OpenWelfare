|                                   | Term Allowance in Term 1 + Read   |

|                                   | The first term School Term        |
|                                   | Allowance is automatically paid   |
|                                   | if the student is recorded as     |
|                                   | enrolled and attending a          |
|                                   | secondary school in Term 1 of the |
|                                   | year. To pay School Term          |
|                                   | Allowance for Term 1, ensure the  |
|                                   | date used at the Student Start    |
|                                   | Date and Course Start Date fields |
|                                   | on the Customer Study Details     |
|                                   | (EDC) screen in Process Direct or |
|                                   | Education Course Details (EDC)    |
|                                   | screen in Customer First is a     |
|                                   | date valid for Term 1 of the      |
|                                   | school year.                      |
|                                   |                                   |
|                                   | Note: the term entitlement period |
|                                   | is used to assess for             |
|                                   | qualification for School Term     |
|                                   | Allowance in a particular term.   |
|                                   | However, the Student Start Date   |
|                                   | coded should still be the actual  |
|                                   | date the student commenced study  |
|                                   | for the school year.              |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Attendance requirements for Terms |
|                                   | 2, 3 or 4 + Read more ...         |
|                                   |                                   |
|                                   | A student must attend at least    |
|                                   | 85% of the required days in the   |
|                                   | term prior to that for which      |
|                                   | payment is assessed. Evidence of  |
|                                   | attendance of 85 per cent or more |
|                                   | in the previous term(s) must be   |
|                                   | received by Services Australia    |
|                                   | before payments will be made.     |
|                                   |                                   |
|                                   | Students who do not have an       |
|                                   | attendance requirement for their  |
|                                   | study (home schooling, distance   |
|                                   | education or non-school           |
|                                   | institutions such as TAFE) will   |
|                                   | be required to establish they     |
|                                   | have completed the workload       |
|                                   | requirements of the course.       |
|                                   | Students undertaking Home         |
|                                   | Schooling in NSW will be required |
|                                   | to confirm a current registration |
|                                   | in order to meet the 85%          |
|                                   | attendance requirement for        |
|                                   | payment of ABSTUDY School Term    |
|                                   | Allowance in the relevant term.   |
|                                   |                                   |
|                                   | Note: due to coronavirus          |
|                                   | (COVID-19) disrupting normal      |
|                                   | study arrangements for many       |
|                                   | students during 2020, payment of  |
|                                   | STA in Term 3 and 4 2020 requires |
|                                   | a student to have attendance of   |
|                                   | 85% or more in Term 1 of 2020. An |
|                                   | attendance check for attendance   |
|                                   | in Term 2 and 3 2020 was not      |
|                                   | undertaken. If the student has    |
|                                   | not attended 85% in Term 1 2020,  |
|                                   | the student will not be eligible  |
|                                   | for payment of STA in Terms 2, 3  |
|                                   | and 4 in 2020.                    |
|                                   |                                   |
|                                   | See Processing ABSTUDY School     |
|                                   | Term Allowance payments for       |
|                                   | guidance.                         |
|                                   |                                   |
|                                   | Has the school advised the        |
|                                   | student attended 85% of the       |
|                                   | required school days in the term  |
|                                   | previous to the one for which     |
|                                   | payment is being assessed?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, see Step 2 in Table 2     |

|                                   | School Term Allowance in Term 1 + |

|                                   | School Term Allowance paying for  |

|                                   | the school year.                  |

|                                   | School Term Allowance + Read more |
|                                   | ...                               |

|                                   | -   Key the Receipt Date and      |
|                                   |     Channel. Select Save          |
|                                   | -   Select Assess and address any |
|                                   |     errors/warnings               |
|                                   | -   Select Assess                 |
|                                   | -   Select Finish. Record details |
|                                   |     of the update                 |
|                                   | -   Select Finalise               |

|                                   |     relevant term                 |

|                                   |     Assessment Result (AR) screen |

|                                   | 3 and 4 in 2020.                  |
|                                   |                                   |
