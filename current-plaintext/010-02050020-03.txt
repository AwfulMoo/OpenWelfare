Processing when a customer is not eligible for School Term Allowance

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer is not eligible for      |
|                                   | School Term Allowance in Term 1 + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the student is not enrolled    |
|                                   | and attending school in Term 1,   |
|                                   | School Term Allowance is not      |
|                                   | payable for that term. To prevent |
|                                   | School Term Allowance paying for  |
|                                   | Term 1, ensure the Student Start  |
|                                   | Date on the Education Course      |
|                                   | Details (EDC) screen is a date    |
|                                   | after the end of Term 1 in the    |
|                                   | school year.                      |
|                                   |                                   |
|                                   | Note: the Student Start Date      |
|                                   | should still be the actual date   |
|                                   | the student commenced study for   |
|                                   | the school year.                  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer is not eligible for      |
|                                   | School Term Allowance + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the student has not attended   |
|                                   | 85% in either Term 2, 3 or 4 of   |
|                                   | the school year, code UNSAT on    |
|                                   | the Education Payments School     |
|                                   | Attendance (EYSA) screen for the  |
|                                   | relevant term(s).                 |
|                                   |                                   |
|                                   | Note: as a result of COVID-19     |
|                                   | disrupting normal study           |
|                                   | arrangements for many students    |
|                                   | during 2020, qualification for    |
|                                   | payment of STA in Term 3 and 4    |
|                                   | 2020 requires a student to have   |
|                                   | attendance of 85% or more in Term |
|                                   | 1 of 2020. If the student has not |
|                                   | attended 85% in Term 1 2020, code |
|                                   | UNSAT at EYSA screen for Terms 2, |
|                                   | 3 and 4 in 2020.                  |
|                                   |                                   |
|                                   | Record details of the decision on |
|                                   | a DOC. Procedure ends here.       |
+-----------------------------------+-----------------------------------+
| 3                                 | Student is not eligible for       |
|                                   | School Term Allowance + Read more |
|                                   | ...                               |
|                                   |                                   |

|                                   | are any changes in their          |
|                                   | circumstances, they should inform |

|                                   | reviewed.                         |
|                                   |                                   |
