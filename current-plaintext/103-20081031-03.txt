| 4                                 | Consideration of Disability and   |
|                                   | Caring Responsibilities + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The Delegate must consider the    |
|                                   | impact of a CDC participant’s     |
|                                   | disability or non-parental caring |
|                                   | responsibilities on their ability |
|                                   | to demonstrate reasonable and     |
|                                   | responsible management of their   |
|                                   | affairs, as it relates to the     |
|                                   | assessment criteria.              |
|                                   |                                   |
|                                   | If a participant is unable to     |
|                                   | demonstrate this, the Delegate    |
|                                   | can still make a determination    |
|                                   | that the participant is to be     |
|                                   | exited from the CDC program, if   |
|                                   | the Delegate is satisfied the     |
|                                   | participant has reasonable and    |
|                                   | responsible management of their   |
|                                   | affairs.                          |
|                                   |                                   |
|                                   | Service Officers will need to     |
|                                   | clearly document how this         |
|                                   | provision applies in the Delegate |
|                                   | Report.                           |
|                                   |                                   |
|                                   | This provision will be assessed   |
|                                   | on a case-by-case basis, and will |
|                                   | need to be supported by evidence. |
|                                   | It is not enough for the          |
|                                   | participant to be receiving a     |
|                                   | carer or disability support       |
|                                   | payment, they will be required to |
|                                   | demonstrate how their disability  |
|                                   | or caring responsibilities impact |
|                                   | their ability to show they can    |
|                                   | manage their affairs against each |
|                                   | assessment criteria.              |
|                                   |                                   |
|                                   | This may be through the inability |
|                                   | to provide the supporting         |
|                                   | information required by the form, |
|                                   | or where the participant          |
|                                   | transfers funds to another person |
|                                   | to meet their primary needs.      |
|                                   |                                   |
|                                   | Evidence that could be considered |
|                                   | under this provision may include  |
|                                   | information recorded on the       |
|                                   | participant’s record including,   |
|                                   | but not limited to:               |
|                                   |                                   |
|                                   | -   assessment and / or receipt   |
|                                   |     of Disability Support         |
|                                   |     Payment, Carers payments,     |
|                                   |     Mobility Allowance, exemption |
|                                   |     from mutual obligation due to |
|                                   |     temporary incapacity          |
|                                   |     (including when they have a   |
|                                   |     job to return to)             |
|                                   | -   medical documentation         |
|                                   |     confirming the participant or |
|                                   |     the person the participant    |
|                                   |     has caring responsibility for |
|                                   |     has a disability or is        |
|                                   |     elderly                       |
|                                   | -   NDIA assessment and funded    |
|                                   |     supports, including if the    |
|                                   |     participant has a             |
|                                   |     self-managed or agency        |
|                                   |     managed NDIA plan             |
|                                   | -   any other evidence that would |
|                                   |     satisfy the Delegate that the |
|                                   |     participant’s disability or   |
|                                   |     caring responsibilities has a |
|                                   |     sufficient impact against the |
|                                   |     assessment criteria           |
|                                   |                                   |
|                                   | The Service Officer should:       |
|                                   |                                   |
