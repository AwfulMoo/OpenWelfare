| 6                                 | Shared care percentage for social |
|                                   | security payments + Read more ... |
|                                   |                                   |
|                                   | If Child Support have determined  |
|                                   | the percentage of care, there is  |
|                                   | no need to calculate the          |
|                                   | percentage of care. The           |
|                                   | percentage advised by Child       |
|                                   | Support can be used for the       |
|                                   | principal carer assessment and    |
|                                   | determination of Social Security  |
|                                   | payment entitlements for Income   |
|                                   | Support Payments, see Step 1 in   |
|                                   | Table 2.                          |
|                                   |                                   |
|                                   | If FCC Processing or Child        |
|                                   | Support staff have already        |
|                                   | determined the percentage of care |
|                                   | for FTB purposes this percentage  |
|                                   | can be used for principal carer   |
|                                   | assessment and determination of   |
|                                   | Social Security payment           |
|                                   | entitlements, see Step 1 in Table |
|                                   | 2.                                |
|                                   |                                   |
|                                   | Determine the assessment period   |
|                                   | to use. This may depend on the    |
|                                   | payment the customer is           |
|                                   | receiving.                        |
|                                   |                                   |
|                                   | If carers have agreed on the      |
|                                   | percentage of care, there is no   |
|                                   | need to calculate the percentage  |
|                                   | based on the agreed pattern of    |
|                                   | care or the available evidence.   |
|                                   |                                   |
|                                   | To calculate the percentage:      |
|                                   |                                   |
|                                   | -   add up the number of nights   |
|                                   |     the child is in the           |
|                                   |     customer's care during the    |
|                                   |     assessment period based on    |
|                                   |     the pattern of care           |
|                                   | -   if the number of nights does  |
|                                   |     not accurately reflect the    |
|                                   |     time in care, the number of   |
|                                   |     hours may be used. Add the    |
|                                   |     number of hours for each part |
|                                   |     day the child is in care.     |
|                                   |     Divide the total hours by 24  |
|                                   |     to get the number of days,    |
|                                   |     and add this to the number of |
|                                   |     whole days in care to get the |
|                                   |     total days in care during the |
|                                   |     assessment period             |
|                                   | -   divide the number of nights   |
|                                   |     in care in the assessment     |
|                                   |     period by the total number of |
|                                   |     days in the assessment period |
|                                   | -   multiply the result by 100 to |
|                                   |     get the shared care           |
|                                   |     percentage                    |
|                                   | -   round to the nearest whole    |
|                                   |     percentage (0.5 is rounded    |
|                                   |     up) except where the          |
|                                   |     percentage is close to the    |
|                                   |     eligibility percentage, for   |
|                                   |     example where a customer has  |
|                                   |     13.9% it is not rounded up to |
|                                   |     14%. Exception for principal  |
|                                   |     carer assessment: Percentages |
|                                   |     between 45.1 and 45.9 should  |
|                                   |     all be rounded up to 46%.     |
|                                   |     Having at least 45.1 of care  |
|                                   |     means there is less than a    |
|                                   |     10% difference, therefore     |
|                                   |     equal care policy applies.    |
|                                   |     The system will only allow    |
|                                   |     coding of whole numbers, so   |
|                                   |     code 46%                      |
|                                   |                                   |
|                                   | Is the percentage of care 55% and |
|                                   | above, or 45% and below?          |
|                                   |                                   |
|                                   | Yes, go to Step 8                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Create a Fast Note, select    |
