Impacts and links on how to code changes of care for income support payments

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Impact of change of care on       |
|                                   | income support payments + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A change of care may cause a      |
|                                   | change of entitlement to existing |
|                                   | income support payments:          |
|                                   |                                   |
|                                   | -   Parenting Payment - the       |
|                                   |     customer will no longer be    |
|                                   |     qualified if principal carer  |
|                                   |     status is lost for all        |
|                                   |     children aged under 14 years  |
|                                   |     (if single) or under 6 (if    |
|                                   |     partnered). If appropriate    |
|                                   |     offer the customer a transfer |
|                                   |     to JSP, see Transfer to       |
|                                   |     JobSeeker Payment (JSP) from  |
|                                   |     another payment               |
|                                   | -   JSP or YA - an increase in    |
|                                   |     care may mean customers can   |
|                                   |     be treated as a principal     |
|                                   |     carer and qualify for a       |
|                                   |     higher rate of payment,       |
|                                   |     additional concessions and    |
|                                   |     reduced mutual obligation     |
|                                   |     requirements, see Principal   |
|                                   |     carer of a dependent child    |
|                                   |                                   |
|                                   | For customers who gain principal  |
|                                   | care of a child:                  |
|                                   |                                   |
|                                   | -   Parenting Payment (PP) should |
|                                   |     be offered if a child aged    |
|                                   |     under 14 (if single), or      |
|                                   |     under 6 (if partnered) and    |
|                                   |     customer is not already PP    |
|                                   |     current. See Claiming         |
|                                   |     Parenting Payment and         |
|                                   |     Processing Parenting Payment  |
|                                   |     (PP) claims                   |
|                                   | -   Single JobSeeker Payment      |
|                                   |     (JSP) and Youth Allowance     |
|                                   |     (YA) customers may qualify    |
|                                   |     for a higher rate of payment, |
|                                   |     additional concessions and    |
|                                   |     reduced mutual obligation     |
|                                   |     requirements. See Principal   |
|                                   |     carer of a dependent child    |
|                                   |     and Principal carer           |
|                                   |     determination in shared care  |
|                                   |     arrangements                  |
|                                   |                                   |
|                                   | Customers who do not have, or     |
|                                   | lose principal care of a child,   |
|                                   | However, have shared care.        |
|                                   |                                   |
|                                   | If Parenting Payment (PP)         |
|                                   | principal carer status is lost:   |
|                                   |                                   |
|                                   | -   for all children aged under 8 |
|                                   |     years (if single) or under 6  |
|                                   |     (if partnered), the customer  |
|                                   |     will no longer be qualified.  |
|                                   |     If appropriate, offer the     |
|                                   |     customer a transfer to JSP.   |
|                                   |     See Transfer to JobSeeker     |
|                                   |     Payment (JSP) from another    |
|                                   |     payment, and                  |
|                                   | -   the customer still has shared |
|                                   |     care of another PP child in   |
|                                   |     principal care this child may |
|                                   |     be included on their          |
|                                   |     concession card and the       |
|                                   |     customer may be eligible for  |
|                                   |     the additional child income   |
|                                   |     free area                     |
|                                   | -   see Principal carer of a      |
|                                   |     dependent child and Principal |
|                                   |     carer determination in shared |
|                                   |     care arrangements             |
|                                   |                                   |
|                                   | If JobSeeker Payment (JSP), Youth |
|                                   | Allowance (YA), Austudy, Special  |
|                                   | Benefit (SpB) and Farm Household  |
|                                   | Allowance (FH) principal carer    |
|                                   | status is lost:                   |
|                                   |                                   |
|                                   | single customers may be entitled  |
|                                   | to the 'with child' rate if they  |
|                                   | have at least 14% care of a       |
|                                   | child. As per 'With child' rate   |
|                                   | of benefit for non-principal      |
|                                   | carers with shared care, a single |
|                                   | JSP or YA-JSK customer must have  |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   legal responsibility (alone   |
|                                   |     or jointly with another       |
|                                   |     person) for the day to day    |
|                                   |     care, welfare and development |
|                                   |     of a child or a family law    |
|                                   |     order, or                     |
|                                   | -   parenting plan which states   |
|                                   |     the child is to supposed to   |
|                                   |     live with the customer        |
