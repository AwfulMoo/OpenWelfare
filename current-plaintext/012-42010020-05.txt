Provider location claiming activity indicators

Table 1: This table describes the indicators used after an automated
fortnightly system update has calculated the claiming activity for each
open practice location.

+-----------------------------------+-----------------------------------+
| Indicator                         | Description                       |
+-----------------------------------+-----------------------------------+
| P                                 | Principal practice (more than 50  |
|                                   | services in a 6 month period).    |
|                                   |                                   |
|                                   | -   principal practice            |
|                                   |     identification is based on    |
|                                   |     volume of claims in the last  |
|                                   |     month, with the practice      |
|                                   |     recording the highest claims  |
|                                   |     activity set as the principal |
|                                   |     location                      |
+-----------------------------------+-----------------------------------+
| S                                 | Secondary practice (more than 50  |
|                                   | services in a 6 month period).    |
|                                   |                                   |
|                                   | -   secondary practices are set   |
|                                   |     where more than one practice  |
|                                   |     location for a health         |
|                                   |     professional has more than 50 |
|                                   |     services in the previous 6    |
|                                   |     months                        |
+-----------------------------------+-----------------------------------+
| L                                 | Low activity practice (less than  |
|                                   | 50 services in a 6 month period). |
+-----------------------------------+-----------------------------------+
| O                                 | Ordering only (pathology,         |
|                                   | radiology, or using prescriber    |
|                                   | number).                          |
+-----------------------------------+-----------------------------------+
| I                                 | Inactive practice.                |
+-----------------------------------+-----------------------------------+
