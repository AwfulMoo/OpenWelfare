For Complex Assessment Officer (CAO) use only

This page contains information on the process for Complex Assessment
Officers (CAO) to follow when creating links between customers and
private trusts or companies to assess income or assets.

Creating links between customers and organisations

This table describes the steps to follow when creating links between
customers and organisations to assess the income and assets from a
private trust or private company.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Checking organisation records     |
|                                   | exist + Read more ...             |
|                                   |                                   |
|                                   | Search for the organisation's     |
|                                   | record, using the Reference       |
|                                   | Number Search (RNS) or            |
|                                   | Organisation Name Search (ON)     |
|                                   | screens.                          |
|                                   |                                   |
|                                   | Does the entity have a record?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Index the entity record + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For help, see Searching for and   |
|                                   | indexing organisations.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Check current links on the        |
|                                   | customer record + Read more ...   |
|                                   |                                   |
|                                   | Go to Link Summary (LS) screen in |
|                                   | customer record.                  |
|                                   |                                   |
|                                   | To display Trust and Company      |
|                                   | (TAC) type links only, code 'TAC' |
|                                   | in the Link Type: field.          |
|                                   |                                   |
|                                   | Does link between customer and    |
|                                   | entity already exist?             |
|                                   |                                   |
|                                   | -   Yes, procedure ends here      |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Create new link + Read more ...   |
|                                   |                                   |
|                                   | Ensure that the trust or          |
|                                   | company's details are in the      |
|                                   | header as the link has to be      |
|                                   | created from the organisation     |
|                                   | record.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | The OCL screen + Read more ...    |
|                                   |                                   |
|                                   | Go to the Other Customer Links    |
|                                   | (OCL) screen. If no links exist,  |
|                                   | the OCL screen will be blank.     |
|                                   |                                   |
|                                   | If there are already customer     |
|                                   | links on the organisation record, |
|                                   | the LS screen will display.       |
|                                   |                                   |
|                                   | -   If the blank OCL screen is    |
|                                   |     presented, go to Step 7       |
|                                   | -   If the LS screen is           |
|                                   |     presented, go to Step 6       |
+-----------------------------------+-----------------------------------+
| 6                                 | The LS screen + Read more ...     |
|                                   |                                   |
|                                   | At the bottom of the LS screen    |
|                                   | code 'Y' in the Add new link:     |
|                                   | field. A blank OCL screen will    |
|                                   | then be presented.                |
+-----------------------------------+-----------------------------------+
| 7                                 | Completing the OCL screen + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Complete the following fields:    |
|                                   |                                   |
|                                   | -   Link type: code 'TAC'         |
|                                   | -   From SVC Rsn: code 'TRF'      |
|                                   | -   To: code 'TRP'                |
|                                   | -   Start date: code the start    |
|                                   |     date of the link              |
|                                   |                                   |
|                                   | The Link Person (LP) screen will  |
|                                   | display.                          |
+-----------------------------------+-----------------------------------+
| 8                                 | The LP screen + Read more ...     |
|                                   |                                   |
|                                   | Search for the required customer  |
|                                   | by entering customer's data on    |
|                                   | the LP screen.                    |
|                                   |                                   |
|                                   | The Link Confirmation (LC) screen |
|                                   | will display.                     |
+-----------------------------------+-----------------------------------+
| 9                                 | The LC screen + Read more ...     |
|                                   |                                   |
|                                   | -   Code 'Y' in Confirm           |
|                                   |     Selection?: field on the LC   |
|                                   |     screen to verify the          |
|                                   |     selection of the customer     |
|                                   | -   If more than one link is to   |
|                                   |     be added in the same          |
|                                   |     activity, key 'Y' in the Add  |
|                                   |     another link of this type?:   |
|                                   |     field                         |
|                                   | -   Finalise the activity         |
+-----------------------------------+-----------------------------------+
