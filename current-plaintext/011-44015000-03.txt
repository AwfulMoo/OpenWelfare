Face to Face (F2F) Service Officer escalation

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Try to resolve the enquiry + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A F2F Service Officer must try to |
|                                   | resolve the customer enquiry      |
|                                   | first. They can do this using:    |
|                                   |                                   |
|                                   | -   Operational Blueprint         |
|                                   | -   staff messages such as Daily  |
|                                   |     Comms                         |
|                                   | -   comments in the Consumer      |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   a VEVO system check           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact the HSDD Eligibility team |
|                                   | for help + Read more ...          |
|                                   |                                   |
|                                   | Call the HSDD Eligibility team    |
|                                   | for help if unable to resolve the |
|                                   | customer’s enquiry.               |
|                                   |                                   |
|                                   | If the HSDD Service Officer is    |
|                                   | not able to help with the         |
|                                   | enquiry, they will call an        |
|                                   | Eligibility PSO. The F2F Service  |
|                                   | Officer is put on hold while this |
|                                   | occurs. The:                      |
|                                   |                                   |
|                                   | -   PSO gives advice to the HSDD  |
|                                   |     Service Officer               |
|                                   | -   HSDD Service Officer gives    |
|                                   |     this advice to the F2F        |
|                                   |     Service Officer               |
|                                   | -   F2F Service Officer can then  |
|                                   |     resolve the enquiry with the  |
|                                   |     customer                      |
+-----------------------------------+-----------------------------------+
| 3                                 | The PSO is unable to help with    |
|                                   | the enquiry + Read more ...       |
|                                   |                                   |
|                                   | If it is not urgent, the F2F      |
|                                   | Service Officer escalates the     |
|                                   | enquiry to the HSDD Eligibility   |
|                                   | Team. To do this they will:       |
|                                   |                                   |
|                                   | Scan all of the enrolment         |
|                                   | documents                         |
|                                   |                                   |
|                                   | In the CDMS:                      |
|                                   |                                   |
|                                   | -   Add a Comment at the Group    |
|                                   |     level (if the customer has a  |
|                                   |     Medicare number). For         |
|                                   |     example, the enrolment has    |
|                                   |     been escalated                |
|                                   | -   Add the HSDD Service Officer  |
|                                   |     login ID details in the       |
|                                   |     comment                       |
|                                   |                                   |
|                                   | This work will be finalised by an |
|                                   | HSDD Service Officer or PSO.      |
|                                   |                                   |
|                                   | If the enquiry is urgent, go to   |
|                                   | Step 4                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Escalate when the query is        |
|                                   | urgent + Read more ...            |
|                                   |                                   |
|                                   | If an enquiry requires an         |
|                                   | immediate response the PSO        |
|                                   | should:                           |
|                                   |                                   |
|                                   | -   call the Medicare Enrolment   |
|                                   |     and Eligibility Helpdesk for  |
|                                   |     advice                        |
|                                   |                                   |
|                                   | Note: if the helpdesk cannot      |
|                                   | resolve the enquiry over the      |
|                                   | phone refer to Escalations from a |
|                                   | PSO to the helpdesk               |
+-----------------------------------+-----------------------------------+

Escalations from a HSDD Service Officer to the PSO

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Escalating to the PSO + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The HSDD Service Officer must try |
|                                   | to resolve the enquiry first.     |
|                                   | They do this by using all of      |
|                                   | their available resources.        |
|                                   |                                   |
|                                   | If they cannot resolve the        |
|                                   | enquiry, the HSDD Service Officer |
|                                   | will call an Eligibility PSO for  |
|                                   | help.                             |
|                                   |                                   |
|                                   | If the PSO asks the HSDD Service  |
|                                   | Officer to send them the enquiry, |
|                                   | they must:                        |
|                                   |                                   |
|                                   | Give the PSO all of the details.  |
|                                   | This may include:                 |
|                                   |                                   |
|                                   | -   The file name Work Item ID    |
|                                   | -   A Medicare number             |
|                                   | -   The applicant's name          |
|                                   | -   Applications/documents/VEVOs  |
|                                   |     if relevant                   |
|                                   |                                   |
|                                   | Forward the details to the PSO    |
|                                   | via the digitised workflow        |
+-----------------------------------+-----------------------------------+
| 2                                 | PSO response to HSDD Service      |
|                                   | Officer + Read more ...           |
|                                   |                                   |
|                                   | For a phone enquiry:              |
|                                   |                                   |
|                                   | -   The PSO will provide a verbal |
|                                   |     response to the HSDD Service  |
|                                   |     Officer. They will complete   |
|                                   |     the PSO tool template and     |
|                                   |     email it to the HSDD Service  |
|                                   |     Officer                       |
|                                   | -   The HSDD Service Officer will |
|                                   |     tell the F2F Service Officer  |
|                                   |     how to complete the enrolment |
|                                   |                                   |
|                                   | For a workflow enquiry:           |
|                                   |                                   |
|                                   | The PSO will:                     |
|                                   |                                   |
|                                   | -   assess the enquiry            |
|                                   | -   send a written response back  |
|                                   |     to the HSDD Service Officer   |
|                                   |     via the workflow              |
|                                   | -   complete the PSO tool         |
|                                   |     template and email it to the  |
|                                   |     HSDD Service Officer          |
|                                   |                                   |
|                                   | The HSDD Service Officer will:    |
|                                   |                                   |
|                                   | retrieve the piece of work from   |
|                                   | the workflow, and                 |
|                                   |                                   |
|                                   | complete the enquiry              |
+-----------------------------------+-----------------------------------+
| 3                                 | Escalate an urgent query to the   |
|                                   | PSO + Read more ...               |
|                                   |                                   |
|                                   | Some examples of an urgent        |
|                                   | enquiry are:                      |
|                                   |                                   |
|                                   | a Ministerial or Ombudsman        |
|                                   | complaint                         |
|                                   |                                   |
|                                   | a potential breach of privacy     |
|                                   |                                   |
|                                   | a Child Protection Agency request |
|                                   | for a Medicare card for a child   |
|                                   | that needs urgent medical         |
|                                   | attention                         |
|                                   |                                   |
|                                   | any enrolment errors that         |
|                                   |                                   |
|                                   | -   have a negative impact on the |
|                                   |     customer, or                  |
|                                   | -   stop them from claiming a     |
|                                   |     Medicare benefit              |
|                                   |                                   |
|                                   | When a HSDD Service Officer       |
|                                   | cannot resolve an urgent query,   |
|                                   | they call an Eligibility PSO.     |
|                                   | They will explain that the query  |
|                                   | is urgent, and give the PSO all   |
|                                   | of the details.                   |
|                                   |                                   |
|                                   | The PSO will either:              |
|                                   |                                   |
|                                   | -   tell them how to resolve the  |
|                                   |     enquiry, or                   |
|                                   | -   ask them to send the details  |
|                                   |     to the PSO via PaNDA using    |
|                                   |     work type Medicare            |
|                                   |     Eligibility escalations       |
|                                   |                                   |
|                                   | If they cannot contact the        |
|                                   | Eligibility PSO, they should      |
|                                   | direct the urgent query to their  |
|                                   | Team Leader.                      |

|                                   | A F2F Service Officer must try to |
|                                   | resolve the customer enquiry. See |
|                                   | Patient eligibility for Child     |
|                                   | Dental Benefits Schedule (CDBS)   |
|                                   | in Medicare.                      |
|                                   |                                   |
