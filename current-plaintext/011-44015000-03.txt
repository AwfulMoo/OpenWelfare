Face to Face (F2F) Service Officer escalation

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Try to resolve the enquiry + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A F2F Service Officer must try to |
|                                   | resolve the customer enquiry      |
|                                   | first by using:                   |
|                                   |                                   |
|                                   | -   Operational Blueprint         |
|                                   | -   staff messages such as Daily  |
|                                   |     Comms                         |
|                                   | -   comments in the Consumer      |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   a VEVO system check           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact the HSD Eligibility team  |
|                                   | for help + Read more ...          |
|                                   |                                   |
|                                   | If the F2F Service Officer is     |
|                                   | unable to resolve the customer’s  |
|                                   | enquiry call the HSD Eligibility  |
|                                   | team.                             |
|                                   |                                   |
|                                   | If the HSDD Eligibility Service   |
|                                   | Officer is not able to help with  |
|                                   | the enquiry the HSDD Eligibility  |
|                                   | Service Officer can call an       |
|                                   | Eligibility PSO. F2F Service      |
|                                   | Officer is put on hold while this |
|                                   | occurs. The following process     |
|                                   | will take place:                  |
|                                   |                                   |
|                                   | -   PSO gives advice to the HSDD  |
|                                   |     Eligibility Service Officer   |
|                                   | -   HSDD Service Officer gives    |
|                                   |     this advice to the F2F        |
|                                   |     Service Officer               |
|                                   | -   F2F Service Officer can then  |
|                                   |     resolve the enquiry with the  |
|                                   |     customer                      |
+-----------------------------------+-----------------------------------+
| 3                                 | The PSO is unable to help with    |
|                                   | the enquiry + Read more ...       |
|                                   |                                   |
|                                   | If it is not urgent, the F2F      |
|                                   | Service Officer escalates the     |
|                                   | enquiry to the HSDD Eligibility   |
|                                   | Team. To do this the F2F Service  |
|                                   | Officer will:                     |
|                                   |                                   |
|                                   | Scan all of the enrolment         |
|                                   | documents                         |
|                                   |                                   |
|                                   | In the CDMS:                      |
|                                   |                                   |
|                                   | -   Add a Comment at the Group    |
|                                   |     level (if the customer has a  |
|                                   |     Medicare number). For         |
|                                   |     example, the enrolment has    |
|                                   |     been escalated                |
|                                   | -   Add the HSDD Eligibility      |
|                                   |     Service Officer login ID      |
|                                   |     details in the comment        |
|                                   |                                   |
|                                   | This work will be finalised by an |
|                                   | HSDD Eligibility Service Officer  |
|                                   | or PSO.                           |
|                                   |                                   |
|                                   | If the enquiry is urgent, go to   |
|                                   | Step 4                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Escalate when the enquiry is      |
|                                   | urgent + Read more ...            |
|                                   |                                   |
|                                   | If an enquiry requires an         |
|                                   | immediate response the PSO will   |
|                                   | call the Medicare Enrolment and   |
|                                   | Eligibility Helpdesk for advice.  |
|                                   |                                   |
|                                   | Note: if the helpdesk cannot      |
|                                   | resolve the enquiry over the      |
|                                   | phone refer to Escalations from a |
|                                   | PSO to the helpdesk               |

Escalations from a HSDD Eligibility Service Officer to the PSO

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Escalating to the PSO + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The HSDD Eligibility Service      |
|                                   | Officer must try to resolve the   |
|                                   | enquiry first by using all        |
|                                   | available resources.              |
|                                   |                                   |
|                                   | If the enquiry is unable to be    |
|                                   | resolved, the HSDD Eligibility    |
|                                   | Service Officer will call an      |
|                                   | Eligibility PSO for help.         |
|                                   |                                   |
|                                   | If the PSO asks the HSDD          |
|                                   | Eligibility Service Officer to    |
|                                   | send them the enquiry, the        |
|                                   | following details may be included |
|                                   | in the request:                   |
|                                   |                                   |
|                                   | -   The file name Work Item ID    |
|                                   | -   A Medicare number             |
|                                   | -   The applicant’s name          |
|                                   | -   Applications/documents/VEVOs  |
|                                   |     if relevant                   |
|                                   |                                   |
|                                   | Note: Forward the details to the  |
|                                   | PSO via the digitised workflow    |
+-----------------------------------+-----------------------------------+
| 2                                 | PSO response to HSDD Eligibility  |
|                                   | Service Officer + Read more ...   |
|                                   |                                   |
|                                   | For a phone enquiry:              |
|                                   |                                   |
|                                   | -   The PSO will provide a verbal |
|                                   |     response to the HSDD          |
|                                   |     Eligibility Service Officer   |
|                                   |     to complete the PSO tool      |
|                                   |     template and email it to the  |
|                                   |     HSDD Eligibility Service      |
|                                   |     Officer                       |
|                                   | -   The HSDD Eligibility Service  |
|                                   |     Officer will tell the F2F     |
|                                   |     Service Officer how to        |
|                                   |     complete the enrolment        |
|                                   |                                   |
|                                   | For a workflow enquiry:           |
|                                   |                                   |
|                                   | The PSO will:                     |
|                                   |                                   |
|                                   | -   assess the enquiry            |
|                                   | -   send a written response back  |
|                                   |     to the HSDD Eligibility       |
|                                   |     Service Officer via the       |
|                                   |     workflow                      |
|                                   | -   complete the PSO tool         |
|                                   |     template and email it to the  |
|                                   |     HSDD Eligibility Service      |
|                                   |     Officer                       |
|                                   |                                   |
|                                   | The HSDD Eligibility Service      |
|                                   | Officer will:                     |
|                                   |                                   |
|                                   | retrieve the piece of work from   |
|                                   | the workflow, and                 |
|                                   |                                   |
|                                   | complete the enquiry              |
+-----------------------------------+-----------------------------------+
| 3                                 | Escalate an urgent query to the   |
|                                   | PSO + Read more ...               |
|                                   |                                   |
|                                   | Some examples of an urgent        |
|                                   | enquiry are:                      |
|                                   |                                   |
|                                   | a Ministerial or Ombudsman        |
|                                   | complaint                         |
|                                   |                                   |
|                                   | a potential breach of privacy     |
|                                   |                                   |
|                                   | a Child Protection Agency request |
|                                   | for a Medicare card for a child   |
|                                   | that needs urgent medical         |
|                                   | attention                         |
|                                   |                                   |
|                                   | any enrolment errors that         |
|                                   |                                   |
|                                   | -   have a negative impact on the |
|                                   |     customer, or                  |
|                                   | -   stop them from claiming a     |
|                                   |     Medicare benefit              |
|                                   |                                   |
|                                   | When a HSDD Eligibility Service   |
|                                   | Officer cannot resolve an urgent  |
|                                   | query, they call an Eligibility   |
|                                   | PSO. They will explain that the   |
|                                   | query is urgent, and give the PSO |
|                                   | all of the details.               |
|                                   |                                   |
|                                   | The PSO will either:              |
|                                   |                                   |
|                                   | -   tell them how to resolve the  |
|                                   |     enquiry, or                   |
|                                   | -   ask them to send the details  |
|                                   |     to the PSO via PaNDA using    |
|                                   |     work type Medicare            |
|                                   |     Eligibility escalations       |
|                                   |                                   |
|                                   | If they cannot contact the        |
|                                   | Eligibility PSO, they should      |
|                                   | direct the urgent query to their  |
|                                   | Team Leader.                      |

Escalate a third Medicare card request

Table 7

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | A third card request + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | F2F or HSDD Eligibility Service   |
|                                   | Officers must escalate all third  |
|                                   | card requests to Eligibility PSO  |
|                                   | team via PaNDA. PSOs do not       |
|                                   | decide and grant third card       |
|                                   | approvals verbally.               |
|                                   |                                   |
|                                   | Service Officers must ensure a    |
|                                   | completed application and all     |
|                                   | relevant documentation has been   |
|                                   | provided by the applicant.        |
|                                   |                                   |
|                                   | Service Officers will escalate to |
|                                   | Eligibility PSO via PaNDA under   |
|                                   | the work type:                    |
|                                   |                                   |
|                                   | -   DCP 3^(rd) card request to    |
|                                   |     Medicare Eligibility DCP      |
|                                   |     Escalations.                  |
|                                   | -   Other 3^(rd) card requests to |
|                                   |     Medicare Eligibility          |
|                                   |     Escalations                   |
|                                   |                                   |
|                                   | The PSO will:                     |
|                                   |                                   |
|                                   | -   assess and confirm that the   |
|                                   |     request is valid              |
|                                   | -   include approval/rejection in |
|                                   |     PaNDA work item notes         |
|                                   | -   re-assign the third card      |
|                                   |     request back to the           |
|                                   |     processing queue for the next |
|                                   |     HSDD Eligibility SO to        |
|                                   |     complete                      |
|                                   |                                   |
|                                   | All 4^(th) card (or more) must be |
|                                   | escalated to the Medicare Public  |
|                                   | Eligibility Helpdesk. The PSO is  |
