| Item                  | Description           | Role                  |
+-----------------------+-----------------------+-----------------------+
| 1                     | Request to withdraw a | Service Officer       |
|                       | child support         |                       |
|                       | assessment from       |                       |
|                       | NZIR + Read more ...  |                       |
|                       |                       |                       |
|                       | Locate the Withdrawal |                       |
|                       | OUT spreadsheet at    |                       |
|                       | G:\Int\NZ.            |                       |
|                       |                       |                       |
|                       | Note: for Service     |                       |
|                       | Officers without      |                       |
|                       | access to the         |                       |
|                       | spreadsheet, email    |                       |
|                       | the details to        |                       |
|                       | International NZ      |                       |
|                       | Administration Team   |                       |
|                       | for action.           |                       |
|                       |                       |                       |
|                       | Select the Withdrawal |                       |
|                       | to NZ tab.            |                       |
|                       |                       |                       |
|                       | Enter the following   |                       |
|                       | information on the    |                       |
|                       | spreadsheet:          |                       |
|                       |                       |                       |
|                       | -   is this request   |                       |
|                       |     urgent? Y/N       |                       |
|                       | -   reason for        |                       |
|                       |     withdrawal        |                       |
|                       | -   payer Inland      |                       |
|                       |     Revenue           |                       |
|                       |     Department (IRD)  |                       |
|                       |     number            |                       |
|                       | -   payer CSRN        |                       |
|                       | -   payer name        |                       |
|                       |     (surname, first   |                       |
|                       |     and middle) and   |                       |
|                       |     date of birth     |                       |
|                       |     (DOB)             |                       |
|                       | -   all children’s    |                       |
|                       |     full names, DOB,  |                       |
|                       |     Start Date of     |                       |
|                       |     Liability (SDOL)  |                       |
|                       |     and End Date of   |                       |
|                       |     Liability (EDOL)  |                       |
|                       |     for all existing  |                       |
|                       |     and ended cases.  |                       |
|                       |     Note: for the     |                       |
|                       |     same child, there |                       |
|                       |     may be multiple   |                       |
|                       |     SDOL and EDOL     |                       |
|                       | -   save the entry    |                       |
|                       |     and close the     |                       |
|                       |     spreadsheet       |                       |
|                       |                       |                       |
|                       | In Cuba, update the   |                       |
|                       | front screen of the   |                       |
|                       | International         |                       |
|                       | Transmittals window   |                       |
|                       | with the date         |                       |
|                       | withdrawn and the     |                       |
|                       | reason for the        |                       |
|                       | withdrawal then paste |                       |
|                       | the document into a   |                       |
|                       | Notepad in the        |                       |
|                       | International         |                       |
|                       | Transmittals window.  |                       |
|                       |                       |                       |
|                       | The Central Authority |                       |
|                       | Team email the        |                       |
|                       | spreadsheets to NZIRD |                       |
|                       | weekly.               |                       |

| 2                     | Case Returns to Child | International Central |
|                       | Support (NZ Out -     | Authority Team        |
|                       | Australian Cases) +   |                       |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | A child support case  |                       |
|                       | is returned to Child  |                       |
|                       | Support when:         |                       |
|                       |                       |                       |
|                       | -   NZIR has          |                       |
|                       |     exhausted all     |                       |
|                       |     collection        |                       |
|                       |     avenues           |                       |
|                       | -   NZIR is not able  |                       |
|                       |     to locate the     |                       |
|                       |     customer          |                       |
|                       | -   the customer is   |                       |
|                       |     deceased and      |                       |
|                       |     there is no       |                       |
|                       |     collection        |                       |
|                       |     opportunity       |                       |
|                       |     through the       |                       |
|                       |     customer’s        |                       |
|                       |     estate, or        |                       |
|                       | -   the customer is   |                       |
|                       |     no longer         |                       |
|                       |     habitually        |                       |
|                       |     resident of New   |                       |
|                       |     Zealand           |                       |
|                       |                       |                       |
|                       | This request is       |                       |
|                       | received from NZIR on |                       |
|                       | the 'Returns'         |                       |
|                       | spreadsheet via the   |                       |
|                       | weekly exchange. The  |                       |
|                       | spreadsheet includes  |                       |
|                       | details of the case,  |                       |
|                       | including the reason  |                       |
|                       | for the return of the |                       |
|                       | child support case.   |                       |
|                       |                       |                       |
|                       | Note: for cases       |                       |
|                       | Returned as ‘Non      |                       |
|                       | Collectable’, 'Non    |                       |
|                       | Traceable' or         |                       |
|                       | 'Customer No Longer   |                       |
|                       | Habitually Resident'  |                       |
|                       | check the ‘Potential  |                       |
|                       | Returns Response      |                       |
|                       | Date’ has been        |                       |
|                       | provided by NZIR and  |                       |
|                       | is within the last    |                       |
|                       | calendar month.       |                       |
