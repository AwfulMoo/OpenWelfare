|                                   |                                   |

|                                   |                                   |

|                                   |                                   |
|                                   | Result: the Administration Menu   |
|                                   | is displayed.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Select option 3 + Read more ...   |

|                                   | Maintenance Menu) and press       |
|                                   | [Enter].                          |
|                                   |                                   |
|                                   | Result: the System Constants      |
|                                   | Maintenance Menu is displayed.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Select option 68 + Read more ...  |
|                                   |                                   |
|                                   | Select option 68 (Workforce       |
|                                   | Supplement Percentage) and press  |
|                                   | [Enter].                          |
|                                   |                                   |
|                                   | Result: the Workforce Supplement  |
|                                   | Percentage is displayed.          |
+-----------------------------------+-----------------------------------+
| 5                                 | Key details + Read more ...       |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | -   Date of effect and press      |
|                                   |     [Tab]. Result: Validates and  |
|                                   |     the cursor moves to           |
|                                   |     percentage field.             |
|                                   | -   Percentage rate and press     |
|                                   |     [Enter]. Result: Validates    |
|                                   |     the percentage rate.          |
+-----------------------------------+-----------------------------------+
| 6                                 | Save percentage rate + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Press [F5] (Commit) twice to save |
|                                   | the percentage rate.              |

| Step                              | Action                            |

|                                   | Key the:                          |

|                                   | -   DSS LAN ID                    |

|                                   | Result: the Solve Access -        |
|                                   | Primary Menu is displayed.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Key details + Read more ...       |
|                                   |                                   |
|                                   | Key M (Access Services) and press |
|                                   | [Enter]. Result: the MAI Primary  |
|                                   | Menu is displayed.                |
|                                   |                                   |
|                                   | Select TSO and press [Enter].     |
|                                   | Result: The TSO/E Logon menu is   |
|                                   | displayed.                        |
|                                   |                                   |
|                                   | Key the:                          |
|                                   |                                   |
|                                   | -   DSS LAN ID                    |
|                                   | -   password                      |
|                                   |                                   |
|                                   | Press [Enter]. Result: The        |
|                                   | 'Ready' prompt is displayed.      |
+-----------------------------------+-----------------------------------+
| 3                                 | Select as required + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select:                           |
|                                   |                                   |

|                                   |     on menu bar). Result: the     |
|                                   |     Send Files to Host - Session  |
|                                   |     A window is displayed.        |
|                                   | -   Browse from the PC File Name  |

|                                   |     window is displayed.          |
|                                   |                                   |
|                                   | From the Browse window, select:   |
|                                   |                                   |
|                                   | -   and double click required     |
|                                   |     file in the network drive or  |
|                                   |     removal storage               |
|                                   | -   Open                          |
|                                   |                                   |
|                                   | Result: the Send Files to Host -  |
|                                   | Session A window is displayed.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Send Files to Host + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the Send Files to Host -     |
|                                   | Session A window:                 |
|                                   |                                   |
|                                   | Click on selected file in the     |

|                                   | -   Text                          |
