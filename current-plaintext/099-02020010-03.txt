Notification of the death of a single pensioner

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm if death has previously   |
|                                   | been notified + Read more ...     |
|                                   |                                   |
|                                   | A Service Officer may either      |
|                                   | receive:                          |
|                                   |                                   |
|                                   | -   a work item if the death has  |
|                                   |     previously been notified but  |
|                                   |     unable to be finalised at     |
|                                   |     first point, or               |
|                                   | -   multiple contacts from        |
|                                   |     different parties notifying   |
|                                   |     the death                     |
|                                   |                                   |
|                                   | Has the death action been taken   |
|                                   | on the deceased customer's        |
|                                   | record?                           |
|                                   |                                   |
|                                   | Yes, no further action is         |
|                                   | required. Procedure ends here     |
|                                   |                                   |
|                                   | No, and:                          |
|                                   |                                   |
|                                   | -   this is the initial advice    |
|                                   |     received via phone or         |
|                                   |     lodgement of an Advice of     |
|                                   |     death form (SA116), go to     |
|                                   |     Step 2                        |
|                                   | -   the death has not been        |
|                                   |     confirmed yet and an          |
|                                   |     additional person has         |
|                                   |     contacted to notify of the    |
|                                   |     death, every effort should be |
|                                   |     made to confirm the death at  |
|                                   |     this contact, go to Step 2    |
|                                   | -   any outstanding activities    |
|                                   |     which were preventing the     |
|                                   |     death action have now been    |
|                                   |     finalised, go to Step 7       |
|                                   | -   a Death Notified but not      |
|                                   |     confirmed - Adult or Child    |
|                                   |     work item has been received,  |
|                                   |     contact the funeral director  |
|                                   |     or the hospital where the     |
|                                   |     deceased passed away to       |
|                                   |     confirm the death. If this is |
|                                   |     not an option and contact     |
|                                   |     with relatives is proposed,   |
|                                   |     social worker involvement may |
|                                   |     be necessary. Once the death  |
|                                   |     has been confirmed, go to     |
|                                   |     Step 7                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Notification of the death of a    |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | When a person contacts Services   |
|                                   | Australia to advise of the death  |
|                                   | of a customer, staff must offer:  |
|                                   |                                   |
|                                   | -   a social worker referral in   |
|                                   |     the first instance or at any  |
|                                   |     time the person shows signs   |
|                                   |     of distress during the        |
|                                   |     contact                       |
|                                   | -   the pamphlet 'Needing help    |
|                                   |     after someone has died?'      |
|                                   |     (HD001) if the person is      |
|                                   |     attending a service centre,   |
|                                   |     and/or direct them to 'Death  |
|                                   |     of a loved one' on the        |
|                                   |     Services Australia website.   |
|                                   |     See Resources page for the    |
|                                   |     link                          |
|                                   |                                   |
|                                   | Consider and respect the          |
|                                   | following cultural issues when    |
|                                   | being advised of the death of an  |
|                                   | Indigenous customer.              |
|                                   |                                   |
|                                   | -   The name of the deceased      |
|                                   |     person cannot be mentioned by |
|                                   |     immediate family members.     |
|                                   |     This means they may have to   |
|                                   |     ask other relatives/friends   |
|                                   |     to call on their behalf or    |
|                                   |     attend the interview with the |
|                                   |     customer                      |
|                                   | -   It may also be offensive to   |
|                                   |     the person for the Service    |
|                                   |     Officer to say the name of    |
|                                   |     the deceased customer. If     |
|                                   |     unsure, sensitively ask the   |
|                                   |     person if it is all right to  |
|                                   |     use the deceased customer's   |
|                                   |     name. If the person is not    |
|                                   |     comfortable with this, avoid  |
|                                   |     using the deceased customer's |
|                                   |     name wherever possible. See   |
|                                   |     Cultural Considerations when  |
|                                   |     someone dies                  |
|                                   |                                   |
|                                   | Does the customer's record        |
|                                   | indicate they were income         |
|                                   | managed, as shown by INM on the   |
|                                   | benefit status line?              |
|                                   |                                   |
|                                   | -   Yes, follow process to update |
|                                   |     record for death of an income |
|                                   |     managed customer, then return |
|                                   |     to this procedure and go to   |
|                                   |     Step 3                        |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Confirm the death of the          |
|                                   | deceased + Read more ...          |
|                                   |                                   |
|                                   | Run the Notification of death     |
|                                   | guided procedure in the deceased  |
|                                   | customerâ€™s record to determine if |
|                                   | death can be confirmed.           |
|                                   |                                   |
|                                   | Confirmation can be accepted      |
|                                   | where the deceased customer's     |
|                                   | full name and date of death is    |
|                                   | provided along with 2 of the      |
|                                   | following:                        |
|                                   |                                   |
|                                   | -   deceased customer's last      |
|                                   |     address in full               |
|                                   | -   date of birth                 |
|                                   | -   Customer Reference Number     |
|                                   |     (CRN)                         |
|                                   |                                   |
|                                   | Confirmation of death also        |
|                                   | requires the following            |
|                                   | information about the informant:  |
|                                   |                                   |
|                                   | -   name                          |
|                                   | -   address                       |
|                                   | -   phone number                  |
|                                   |                                   |
|                                   | Confirm if the person notifying   |
|                                   | of the death is the executor of   |
|                                   | the estate. If the informant is   |
|                                   | not the executor, but knows the   |
|                                   | executor details, obtain:         |
|                                   |                                   |
|                                   | -   the name and address of the   |
|                                   |     executor, and                 |
|                                   | -   the name of the funeral       |
|                                   |     director or hospital          |
|                                   |                                   |
|                                   | Nominee arrangements and Power of |
|                                   | Attorney (POA) stop at the time   |
|                                   | of death. Any correspondence will |
|                                   | be sent to the executor or        |
|                                   | administrator of an estate. This  |
|                                   | includes the payment summary and  |
|                                   | possible debt recovery action.    |
|                                   |                                   |
|                                   | If the executor or administrator  |
|                                   | details are incorrectly recorded  |
|                                   | or not recorded at all when       |
|                                   | details have been provided, this  |
|                                   | will impact the agency's ability  |
|                                   | to finalise any outstanding       |
|                                   | matters.                          |
|                                   |                                   |
|                                   | If executor details are not known |
|                                   | when notifying the death, leave   |
|                                   | blank the executor details fields |
|                                   | in the Notification of Death      |
|                                   | guided procedure. The details can |
|                                   | be recorded later on the          |
|                                   | Death/Executor (DEA) screen after |
|                                   | completing the death action.      |
|                                   |                                   |
|                                   | Can death be confirmed at first   |
|                                   | contact?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Death cannot be confirmed + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Using the Notification of Death   |
|                                   | guided procedure obtain and       |
|                                   | record all relevant information   |
|                                   | from the informant including      |
|                                   | their own name, address, contact  |
|                                   | number and relationship to the    |
|                                   | deceased, and if known, details   |
|                                   | of the executor, the name of the  |
|                                   | funeral director, and if          |
|                                   | applicable the name of the        |
|                                   | hospital.                         |
|                                   |                                   |
|                                   | The guided procedure will not     |
|                                   | code the death, but will generate |
|                                   | a DOC on the record which is an   |
|                                   | open work item to be allocated to |
|                                   | a Service Officer with the        |
|                                   | required skill set to confirm the |
|                                   | death.                            |
|                                   |                                   |
|                                   | If the guided procedure is not    |
|                                   | available, create an open work    |
|                                   | item. Use Fast Note. Select Auto  |
|                                   | Text use > Generic > Update >     |
|                                   | Death Notification (Adult) -      |
|                                   | ACTION REQUIRED.                  |
|                                   |                                   |
|                                   | The guided procedure presents a   |
|                                   | suspension option if the death is |
|                                   | unable to be confirmed:           |
|                                   |                                   |
|                                   | -   use the reason code 'OTH'     |
|                                   |     (Other) the date of effect    |
|                                   |     should be the unconfirmed     |
|                                   |     date of death or date paid to |
|                                   |     plus one (DPT+1), whichever   |
|                                   |     is the later                  |
|                                   | -   ensure the DOC meets Online   |
|                                   |     Document Recording (ODR)      |
|                                   |     standards for significant     |
|                                   |     decisions and including what  |
|                                   |     action is required to confirm |
|                                   |     the death                     |
|                                   | -   do not issue a suspension     |
|                                   |     letter. Cancel the advice     |
|                                   |     activity on the AL screen     |
|                                   |     triggered by the 'OTH'        |
|                                   |     suspension                    |
|                                   | -   if the guided procedure is    |
|                                   |     not available or the          |
|                                   |     suspension option is not      |
|                                   |     presented, suspend the        |
|                                   |     deceased customerâ€™s payments  |
|                                   |     manually                      |
|                                   |                                   |
|                                   | When the death is confirmed, go   |
|                                   | to Step 5.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Outstanding activities + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A death action cannot be          |
|                                   | processed with activities on the  |
|                                   | Activity List (AL) screen. The    |
|                                   | Notification of death guided      |
|                                   | procedure will prompt if there is |
|                                   | an activity on the AL screen.     |
|                                   |                                   |
|                                   | Note: if a deceased customerâ€™s    |
|                                   | record was suspended while the    |
|                                   | death was being confirmed and     |
|                                   | their death has now been          |
|                                   | confirmed, manually check for     |
|                                   | started activities on the AL      |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Where possible ensure any         |
|                                   | activities are completed and run  |
|                                   | the daily driver (DDR):           |
|                                   |                                   |
|                                   | -   if there is a 'STA' debt      |
|                                   |     activity on the AL screen,    |
|                                   |     create an open work item for  |
|                                   |     region XWA advising them of   |
|                                   |     the death and requesting the  |
|                                   |     debt activity be completed.   |
|                                   |     Use Fast Note. Select Auto    |
|                                   |     Text use > Debt > DMB         |
|                                   |     Raising > Debt Raising ACTion |
|                                   |     DOC Request. See Requesting   |
|                                   |     refunds and raising debts for |
|                                   |     deceased customer             |
|                                   | -   if there is an RCA/NCL or     |
|                                   |     RCA/HOM activity on the AL    |
|                                   |     screen, add the keyword       |
|                                   |     RCAPRI. The RCA team will     |
|                                   |     action the activity and       |
|                                   |     complete the death action     |
|                                   | -   for other outstanding         |
|                                   |     activities, immediately       |
|                                   |     contact the team or Service   |
|                                   |     Officer who needs to complete |
|                                   |     the outstanding activity - if |
|                                   |     unsure who to contact, refer  |
|                                   |     to existing escalation        |
|                                   |     channels                      |
|                                   |                                   |
|                                   | Have any started activities been  |
|                                   | completed?                        |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | When it is not possible to        |
|                                   | complete an activity + Read more  |

|                                   | death action:                     |

|                                   | manually suspend using the reason |
|                                   | 'OTH' (other) pending completion  |
|                                   | of the outstanding activity       |
|                                   |                                   |
|                                   | the suspension date of effect     |
|                                   | should be the later of:           |
|                                   |                                   |
|                                   | -   date of death, or             |
|                                   | -   date paid to plus one (DPT +  |
|                                   |     1)                            |
|                                   |                                   |
|                                   | record a DOC on the customerâ€™s    |
|                                   | record, ensuring that it meets    |
|                                   | Online Document Recording (ODR)   |
|                                   | minimum standards for significant |
|                                   | decisions and including what      |
|                                   | action must occur so the death    |

|                                   | triggered by the 'OTH' suspension |
|                                   |                                   |

|                                   | either:                           |
|                                   |                                   |
|                                   | -   run the Notification of death |
|                                   |     guided procedure to generate  |

|                                   |     relevant staff to complete    |
|                                   |     the death action once the     |
|                                   |     outstanding activities have   |
|                                   |     been completed, or            |
|                                   | -   if the guided procedure is    |
|                                   |     not available create an open  |
|                                   |     work item using Fast Note.    |
|                                   |     Select Auto Text use >        |
|                                   |     Generic > Update > Death      |
|                                   |     Notification (Adult) - ACTION |
|                                   |     REQUIRED                      |
|                                   |                                   |
|                                   | When activity is completed, go to |
|                                   | Step 7.                           |
