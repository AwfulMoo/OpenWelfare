If an incident is occurring:
• for sites providing face to face services, follow Emergency Response
Procedures - Aggression (ERPAs)
• for non-customer facing sites, follow Customer Aggression Emergency
Response Procedures (CA ERPs)
• for agents, follow Emergency Response Procedures - Aggression (ERPAs)
for agents
• with customers at risk of suicide or self-harm, see Customer talking
about suicide or self-harm

This page has information on how to respond to threats, customer
aggression and counterproductive behaviour.

|                                   | Code Grey is the response to a    |

|                                   | other people are involved in a    |
|                                   | direct physical threat that could |
|                                   | result in injury. These include:  |
|                                   |                                   |

|                                   |     weapon (other than a firearm, |
|                                   |     or flammable substance). This |
|                                   |     includes if they claim to     |

|                                   |     have one. If it is a firearm  |

|                                   |     Code Black                    |
|                                   | -   A customer is threatening to  |
|                                   |     use a weapon, including an    |
|                                   |     improvised weapon             |
|                                   | -   Physical assault such as      |
|                                   |     pushing, fighting, spitting   |
|                                   |     and shoving between customers |
|                                   |     and staff                     |
|                                   | -   Violence between customers    |
|                                   |     where the safety of others is |
|                                   |     compromised                   |

|                                   |     site, including pepper sprays |
|                                   |     or batons                     |

|                                   |                                   |

|                                   |     should try to keep furniture  |

| 6                                 | Code Grey - non-customer facing   |
|                                   | environment + Read more ...       |
|                                   |                                   |
|                                   | In a non-customer facing          |
|                                   | environment, a Code Grey is a     |
|                                   | building or floor lockdown in     |
|                                   | response to:                      |
|                                   |                                   |
|                                   | -   A violent threat              |
|                                   | -   Danger, or                    |
|                                   | -   An armed person near the      |
|                                   |     building that poses a threat  |
|                                   |     to people inside              |
|                                   |                                   |
|                                   | Staff response                    |
|                                   |                                   |
|                                   | -   Stay in the work area and     |
|                                   |     wait for further instructions |
|                                   |     from the ART                  |
|                                   | -   Do not go near windows or     |
|                                   |     doors, into any lift foyers,  |
|                                   |     or near the entrance          |
|                                   | -   Do not leave the building or  |
|                                   |     floor unless told to do so by |
|                                   |     the ART                       |
|                                   |                                   |
|                                   | ART response                      |
|                                   |                                   |
|                                   | -   Upon identifying or being     |
|                                   |     notified of the incident      |
|                                   |     assess the risk and confirm   |
|                                   |     what response is required     |
|                                   | -   Once it is assessed that a    |
|                                   |     Code Grey response is         |
|                                   |     required, make sure that      |
|                                   |     everyone on the site receives |
|                                   |     the Code Grey instruction.    |
|                                   |     For multi-tenanted sites,     |
|                                   |     contact the Chief Warden who  |
|                                   |     will tell other tenants as    |
|                                   |     necessary                     |
|                                   | -   Tell staff to shelter in      |
|                                   |     place and stay away from the  |
|                                   |     windows and doors             |
|                                   | -   Arrange for the police to be  |
|                                   |     called if this has not        |
|                                   |     already happened              |
|                                   | -   Advise Agency Security as     |
|                                   |     soon as it is safe and        |
|                                   |     practicable to do so          |
|                                   | -   Update staff (as appropriate) |
|                                   |     on the status of the incident |
|                                   | -   Continually reassess the risk |
|                                   |     to ensure that Code Grey      |
|                                   |     remains the appropriate       |
|                                   |     response. For multi-tenanted  |
|                                   |     sites keep in contact with    |
|                                   |     the Chief Warden.             |
|                                   |                                   |
|                                   | Once the threat has been removed, |
|                                   | issue an ‘all clear’ and resume   |
|                                   | normal site access.               |

| 7                                 | Code Black - face-to-face         |
|                                   | environment + Read more ...       |
|                                   |                                   |
|                                   | Code Black is the response to an  |
|                                   | extreme threat level that could   |
|                                   | result in serious injury or death |
|                                   | to customers and staff. These can |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   hostage scenarios             |
|                                   | -   customer with a firearm       |
|                                   | -   customer brandishing          |
|                                   |     flammable substance within    |
|                                   |     the site                      |
|                                   | -   a gas is released within the  |
|                                   |     site                          |
|                                   |                                   |
|                                   | Staff response                    |
|                                   |                                   |
|                                   | -   Activate the duress alarm if  |
|                                   |     it is safe to do so           |
|                                   | -   Tell customers to evacuate    |
|                                   |     through the safest exit       |
|                                   | -   If it is safe to do so, move  |
|                                   |     directly to the Code Black    |
|                                   |     assembly area                 |
|                                   | -   If it is not safe to          |
|                                   |     evacuate, find a place to     |
|                                   |     hide out of the aggressor's   |
|                                   |     sight                         |
|                                   | -   Whenever possible, avoid      |
|                                   |     restricting the option to     |
|                                   |     evacuate                      |
|                                   | -   If staff choose to hide, they |
|                                   |     may also need to consider     |
|                                   |     options to disrupt or disable |
|                                   |     the customer if they are      |
|                                   |     located. Any action should    |
|                                   |     only be taken as a last       |
|                                   |     resort, and to protect the    |
|                                   |     life of the individual or     |
|                                   |     others in the area            |
|                                   | -   If staff are unable to        |
|                                   |     evacuate or find a place to   |
|                                   |     hide, comply with the         |
|                                   |     aggressor's instructions. Do  |
|                                   |     only as told and nothing more |
|                                   |                                   |
|                                   | ART response                      |
|                                   |                                   |
|                                   | -   Announce a Code Black         |
|                                   | -   Once outside, remain vigilant |
|                                   |     as the aggressor could move   |
|                                   |     to your location. Be aware of |
|                                   |     the exit points and watch for |
|                                   |     any threat coming from inside |
|                                   |     the site                      |
|                                   | -   Make sure police have been    |
|                                   |     called and wait for them to   |
|                                   |     arrive                        |
|                                   | -   Make sure all staff and       |
|                                   |     customers are accounted for.  |
|                                   |     Consider contacting staff who |
|                                   |     are offsite and to tell them  |
|                                   |     about the Code Black          |
|                                   | -   Liaise with the police once   |
|                                   |     they arrive                   |
|                                   | -   Encourage staff and customers |
|                                   |     to stay in the Code Black     |
|                                   |     assembly area                 |
|                                   | -   If other customers on the     |
|                                   |     site refuse to move, staff    |
|                                   |     should not remain with them   |
|                                   |     and must move to the Code     |
|                                   |     Black location                |
|                                   |                                   |
|                                   | For a very serious firearm or     |
|                                   | active shooter incident, the ART  |
|                                   | may instruct staff to disperse    |
|                                   | from the site. If this happens,   |
|                                   | staff and site leaders should try |
|                                   | to contact one another to confirm |
|                                   | their location and safety.        |
|                                   |                                   |
|                                   | Refer to the Resources page for a |
|                                   | link to the Emergency Response    |
|                                   | Procedures – Aggression (ERPAs)   |
|                                   | including Code Grey and Code      |
|                                   | Black for a link to the Code Grey |
|                                   | and Code contact card.            |

| 8                                 | Code Black - non-customer facing  |
|                                   | environment + Read more ...       |
|                                   |                                   |
|                                   | In a non-customer facing          |
|                                   | environment, a Code Black         |
|                                   | (evacuation) is a response to an  |
|                                   | extreme threat in the building    |
|                                   | which could result in serious     |
|                                   | injury or death, such as a lethal |
|                                   | weapon.                           |
|                                   |                                   |
|                                   | Staff response                    |
|                                   |                                   |
|                                   | Have an evacuation route in mind. |
|                                   | Move away from any sounds of      |
|                                   | violence and evacuate regardless  |
|                                   | of whether others agree to follow |
|                                   |                                   |
|                                   | Look for any threat affecting the |
|                                   | nearest evacuation route          |
|                                   |                                   |
|                                   | If it is safe to do so, move away |
|                                   | from the building towards the     |
|                                   | Code Black assembly point.        |
|                                   |                                   |
|                                   | Do not assemble at the fire       |
|                                   | evacuation assembly points        |
|                                   |                                   |
|                                   | If it is not safe to evacuate     |
|                                   | (for example, if the offender is  |
|                                   | close by) find a place to hide    |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   remains out of the offender's |
|                                   |     line of sight                 |
|                                   | -   does not restrict your        |
|                                   |     potential escape route        |
|                                   | -   provides cover                |
|                                   |                                   |
|                                   | Note: turn mobile phones to       |
|                                   | silent and turn off the vibration |
|                                   |                                   |
|                                   | As a last resort, staff should    |
|                                   | consider options for arming       |
|                                   | themselves with improvised        |
|                                   | weapons for defence in the event  |
|                                   | they are located by the offender  |
|                                   |                                   |
|                                   | If staff are still in the         |
|                                   | building when the police arrive,  |
|                                   | staff should:                     |
|                                   |                                   |
|                                   | -   follow police instructions    |
|                                   | -   avoid quick movements or      |
|                                   |     shouting and keep their hands |
|                                   |     in clear view. This is to     |
|                                   |     help the police distinguish   |
|                                   |     staff from the offender       |
|                                   |                                   |
|                                   | ART response                      |
|                                   |                                   |
|                                   | Upon identifying or being         |
|                                   | notified of the incident assess   |
|                                   | the risk and confirm what         |
|                                   | response is required              |
|                                   |                                   |
|                                   | Once it is assessed that a Code   |
|                                   | Black response is required, make  |
|                                   | sure that all people on site      |
|                                   | receive the Code Black            |
|                                   | instruction. For multi-tenanted   |
|                                   | sites, contact the Chief          |
|                                   | Warden/Emergency Control          |
|                                   | Organisation as soon as possible  |
|                                   | so that they can tell other       |
|                                   | tenants in the building           |
|                                   |                                   |
|                                   | Direct staff to:                  |
|                                   |                                   |
|                                   | -   evacuate the building         |
|                                   |     immediately if it is safe to  |
|                                   |     do so, look for any threat    |
|                                   |     that might affect the nearest |
|                                   |     evacuation route, and move to |
|                                   |     the Code Black assembly point |
|                                   | -   if the incident is an active  |
|                                   |     shooter incident, staff       |
|                                   |     should be directed to         |
|                                   |     disperse, not move to the     |
|                                   |     Code Black location           |
|                                   |                                   |
|                                   | ART may need to tell staff to     |
|                                   | disperse from the Code Black      |
|                                   | assembly point, such as if an     |
|                                   | offender follows staff to the     |
|                                   | Code Black assembly point         |
|                                   |                                   |
|                                   | Make sure that the police have    |
|                                   | been called                       |
|                                   |                                   |
|                                   | Advise Physical Security as soon  |
|                                   | as it is safe and practicable to  |
|                                   | do so                             |

Responding to virtual threats to harm staff or damage agency property

Table 3: actions staff and site leaders take in response to phone
threats to harm staff or property owned by Services Australia.

For threats to Employment Service Providers and other third parties see
Customer aggression – escalating incidents, table 2.

When a threat is received through a telephone call, email, internet
message, letter etc, use the following process.

|                       |     possible so the   |                       |

|                       |     be traced         |                       |

|                       |     Malicious Call    |                       |

|                       |     procedures, go to |                       |
