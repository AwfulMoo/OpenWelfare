This page contains the process to complete and submit the file template
to upload authorised employee records.

Completing and submitting the file template

Completing and submitting the file template with authorised employee details

This table describes the process to complete and submit the file
template.

+-----------------------+-----------------------+-----------------------+
| Stage                 | Action                | Who                   |
+-----------------------+-----------------------+-----------------------+
| 1                     | Access employee       | Organisation          |
|                       | details file          | Maintenance Officer   |
|                       | template + Read more  | (OMO)                 |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | Accesses the          |                       |
|                       | authorised employee   |                       |
|                       | details file template |                       |
|                       | via their existing    |                       |
|                       | Health Professional   |                       |
|                       | Online Services       |                       |
|                       | (HPOS) account.       |                       |
+-----------------------+-----------------------+-----------------------+
| 2                     | Complete file         | OMO                   |
|                       | template + Read more  |                       |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | Completes the file    |                       |
|                       | template and enters   |                       |
|                       | identity details and  |                       |
|                       | secret questions and  |                       |
|                       | answers about         |                       |
|                       | authorised employees  |                       |
|                       | that work at their    |                       |
|                       | premises.             |                       |
+-----------------------+-----------------------+-----------------------+
| 3                     | Submit file           | OMO                   |
|                       | template + Read more  |                       |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | Submits the completed |                       |
|                       | file template via     |                       |
|                       | HPOS email facility.  |                       |
+-----------------------+-----------------------+-----------------------+
| 4                     | Receive file          | Tier 3 Service        |
|                       | template + Read more  | Officer               |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | Receives the file     |                       |
|                       | template via the Tier |                       |
|                       | 3 positional email    |                       |
|                       | account.              |                       |
+-----------------------+-----------------------+-----------------------+
| 5                     | Upload employee       | Tier 3 Service        |
|                       | details file + Read   | Officer               |
|                       | more ...              |                       |
|                       |                       |                       |
|                       | Uploads the           |                       |
|                       | authorised employee   |                       |
|                       | details file to the   |                       |
|                       | eHealth Program       |                       |
|                       | interface (EHP)       |                       |
|                       | system.               |                       |
+-----------------------+-----------------------+-----------------------+
| 6                     | Advises OMO + Read    | Tier 3 Service        |
|                       | more ...              | Officer               |
|                       |                       |                       |
|                       | Advises the OMO that  |                       |
|                       | details were          |                       |
|                       | successfully          |                       |
|                       | uploaded, or where    |                       |
|                       | necessary, advises    |                       |
|                       | the OMO that further  |                       |
|                       | information is        |                       |
|                       | required before the   |                       |
|                       | details can be        |                       |
|                       | successfully loaded.  |                       |
