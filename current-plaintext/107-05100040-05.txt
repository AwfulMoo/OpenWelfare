Recovery Fee codes in DOC

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Recovery Fee codes in DOCs        |
|                                   |                                   |
|                                   | The Debt Action script creates a  |
|                                   | DOC when application of the       |
|                                   | Recovery Fee is considered. The   |
|                                   | script places a single alpha code |
|                                   | before the debt amount in the DOC |
|                                   | Extra Details summary line. The   |
|                                   | full details are available in the |
|                                   | DOC text.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Recovery Fee is imposed           |
|                                   |                                   |
|                                   | I = Recovery Fee Imposed          |
+-----------------------------------+-----------------------------------+
| 3                                 | Recovery Fee is not imposed       |
|                                   |                                   |
|                                   | L = Q175 Warning letter           |
|                                   |                                   |
|                                   | V = Verbal Warning                |
|                                   |                                   |
|                                   | N = No Warning has been issued    |
|                                   |                                   |
|                                   | A DOC detailing the decision      |
|                                   | about the Recovery Fee is still   |
|                                   | required. These codes are a quick |
|                                   | reference.                        |

Examples of when a Recovery Fee should and should not be applied for online compliance interventions

Table 4: These examples are a guide. This list is not exhaustive.

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Examples - when a Recovery Fee    |
|                                   | should not be applied             |
|                                   |                                   |
|                                   | The following examples are        |
|                                   | situations where a customer has   |
|                                   | engaged to the best of their      |
|                                   | ability and a recovery fee should |
|                                   | not be applied:                   |
|                                   |                                   |
|                                   | Customer actions the intervention |
|                                   | online and accepts the following: |
|                                   |                                   |
|                                   | -   'By accepting this result you |
|                                   |     are declaring that the        |
|                                   |     information you provided or   |
|                                   |     confirmed is correct, and if  |
|                                   |     your information changed from |
|                                   |     what you told us previously,  |
|                                   |     there was a good reason'      |
|                                   |                                   |
|                                   | The customer does not action the  |
|                                   | intervention online or does not   |
|                                   | contact to confirm or update      |
|                                   | their employment circumstances,   |
|                                   | however they have provided        |
|                                   | documentary evidence              |
|                                   |                                   |
|                                   | The customer supplies documentary |
|                                   | evidence, but the evidence        |
|                                   | received is not sufficient to     |
|                                   | make an update                    |
|                                   |                                   |
|                                   | The customer contacts, or a       |
|                                   | Compliance Officer contacts the   |
|                                   | customer, at any time to discuss  |
|                                   | the intervention and the customer |
|                                   | either confirms the match data or |
|                                   | provides further information to   |
|                                   | update their employment           |
|                                   | circumstances. This includes      |
|                                   | where the customer is agreeable   |
|                                   | to the application of the         |
|                                   | Australian Taxation Office (ATO)  |
|                                   | match data                        |
|                                   |                                   |
|                                   | The customer contacts, or a       |
|                                   | Compliance Officer contacts the   |
|                                   | customer, to discuss the          |
|                                   | intervention and the customer     |
|                                   | advises they are unable to        |
|                                   | provide the information, for      |
|                                   | example the employer has ceased   |
|                                   | to operate or the customer cannot |
|                                   | locate the information            |
|                                   |                                   |
|                                   | The intervention drops out of the |
|                                   | online process due to an          |
|                                   | Assessment Error and contact with |
|                                   | the customer is unsuccessful      |
|                                   |                                   |
|                                   | Where the customer is providing   |
|                                   | new information or evidence as    |
|                                   | part of a reassessment, and the   |
|                                   | customer had a recovery fee in    |
|                                   | the original debt, the recovery   |
|                                   | fee must be removed               |
+-----------------------------------+-----------------------------------+
| 2                                 | Examples - when a Recovery Fee    |
|                                   | should be applied                 |
|                                   |                                   |
|                                   | The following examples are        |
|                                   | situations where a customer has   |
|                                   | not engaged to the best of their  |
|                                   | ability and a recovery fee should |
|                                   | be applied:                       |
|                                   |                                   |
|                                   | -   Customer contacts but refuses |
|                                   |     to discuss/action the         |
|                                   |     intervention                  |
|                                   | -   Customer engages online but   |
|                                   |     does not complete the         |
|                                   |     intervention and attempts to  |
|                                   |     contact the customer are      |
|                                   |     unsuccessful                  |
|                                   | -   Customer engages online and   |
|                                   |     is requested by the online    |
|                                   |     process to provide            |
|                                   |     documentary evidence, or they |
|                                   |     are advised to contact, but   |
|                                   |     they do not provide the       |
|                                   |     information and contact with  |
|                                   |     the customer is unsuccessful  |
