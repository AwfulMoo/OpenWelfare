Identifying if FAA is applicable

Table 1: This table describes the process for when an ABSTUDY secondary
school student is applying for FAA for vacation travel during the study
year.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | FAA eligibility + Read more ...   |
|                                   |                                   |
|                                   | Note: an Australian Apprentice is |
|                                   | not entitled to FAA for the       |
|                                   | Australian Apprenticeship.        |
|                                   |                                   |
|                                   | Is the customer eligible for FAA  |
|                                   | for ABSTUDY?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, see Step 3 in the         |
|                                   |     Assessment table              |
+-----------------------------------+-----------------------------------+
| 2                                 | Check level of study + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the student undertaking study  |
|                                   | at secondary school level?        |
|                                   |                                   |
|                                   | Check the Course Level field on   |
|                                   | the Education Course Details      |
|                                   | (EDC) screen to determine level   |
|                                   | of course.                        |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, see Step 3 in the         |
|                                   |     Assessment table              |
+-----------------------------------+-----------------------------------+
| 3                                 | Check where student is residing + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the student residing away from |
|                                   | their permanent home address in   |
|                                   | order to undertake an Away from   |
|                                   | Base activity?                    |
|                                   |                                   |
|                                   | Note: vacation travel is only     |
|                                   | available to students who are     |
|                                   | approved for commencement and end |
|                                   | of study travel to move away from |
|                                   | their permanent home to undertake |
|                                   | a course of study. Students       |
|                                   | undertaking Away from Base        |
|                                   | activities may qualify for        |
|                                   | alternative assistance for Away   |
|                                   | from Base travel.                 |
|                                   |                                   |
|                                   | -   Yes, see Step 3 in the        |
|                                   |     Assessment table              |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Check if student has commenced    |
|                                   | study in the course for the       |
|                                   | year + Read more ...              |
|                                   |                                   |
|                                   | Has the student commenced study   |
|                                   | in the course for the year?       |
|                                   |                                   |
|                                   | Note: vacation travel can only be |
|                                   | accessed after the student has    |
|                                   | commenced study in the course for |
|                                   | the year.                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, see Step 3 in the         |
|                                   |     Assessment table              |
+-----------------------------------+-----------------------------------+
| 5                                 | Check scheduled school term       |
|                                   | breaks + Read more ...            |
|                                   |                                   |
|                                   | Does the journey coincide with a  |
|                                   | scheduled school term break in    |
|                                   | that State/Territory?             |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to school term dates for each     |
|                                   | State/Territory.                  |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, see Step 3 in the         |
|                                   |     Assessment table              |

|                                   | travelling + Read more ...        |
|                                   |                                   |

|                                   | destination + Read more ...       |

Assessment

Approve or reject FAA

Table 2: This table describes the steps to follow to confirm if the
ABSTUDY secondary school student is eligible for FAA for vacation
travel.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | The student is eligible for FAA   |
|                                   | for vacation travel + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check if the student has          |
|                                   | previously claimed FAA for this   |
|                                   | travel for this term as vacation  |
|                                   | travel may only be claimed once   |
|                                   | per term.                         |
|                                   |                                   |
|                                   | Has FAA for vacation travel       |
|                                   | already been claimed for the      |
|                                   | term?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Approval + Read more ...          |
|                                   |                                   |
|                                   | Advise the student they are       |
|                                   | approved and entitled to          |
|                                   | undertake vacation travel         |
|                                   |                                   |
|                                   | Determine if the student requires |
|                                   | FAA to be:                        |
|                                   |                                   |
|                                   | -   authorised and booked in      |
|                                   |     advance of travel, or         |
|                                   | -   costs reimbursed after the    |
|                                   |     travel has been undertaken    |
|                                   |                                   |
|                                   | Students claiming reimbursement   |
|                                   | will be required to complete an   |
|                                   | ABSTUDY Fares Allowance Claim     |
|                                   | (SY032). The Resources page       |
|                                   | contains a link to this form.     |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Ineligible + Read more ...        |
|                                   |                                   |
|                                   | Advise the student they are not   |
|                                   | eligible for FAA for vacation     |
|                                   | travel.                           |
|                                   |                                   |
|                                   | If the travel request has been    |
|                                   | received by ABSTUDY Smart Centre  |
|                                   | telephony, record details of      |
|                                   | discussion on a DOC. Explain      |
|                                   | their review and appeal rights.   |
|                                   | See Initial contact about a       |
|                                   | decision and the review of        |
|                                   | decision process.                 |
|                                   |                                   |
|                                   | If the travel request has been    |
|                                   | received by the ABSTUDY Smart     |
|                                   | Centre Travel team, finalise the  |
|                                   | Travel Booking status as          |
|                                   | 'Rejected' in the Travel          |
|                                   | Management Application (TMA).     |
|                                   | Record details of the decision in |
|                                   | the Notes field. See Coding       |
|                                   | pre-booked Fares Allowance (FAA)  |
|                                   | entitlements for ABSTUDY and      |
|                                   | ABSTUDY Pensioner Education       |
|                                   | Supplement (PES).                 |
|                                   |                                   |
|                                   | Check if the student's            |
|                                   | circumstances will entitle them   |
|                                   | to access alternate FAA travel    |
|                                   | types.                            |
+-----------------------------------+-----------------------------------+
