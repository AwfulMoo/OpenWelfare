Implementation or non-compliance

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Implement the garnishee notice +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Confirm when the first payment    |
|                                   | will occur.                       |
|                                   |                                   |
|                                   | Confirm the employer's intended   |
|                                   | method and frequency of           |
|                                   | remittance.                       |
|                                   |                                   |
|                                   | Update the Pay Start Date &       |
|                                   | Frequency: field on the Garnishee |
|                                   | Details (OPGD) screen. This       |
|                                   | should be:                        |
|                                   |                                   |
|                                   | -   the date the garnishee        |
|                                   |     payment is expected, or       |
|                                   | -   two weeks from today's date   |
|                                   |     (whichever is the later)      |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Non-compliance + Read more ...    |
|                                   |                                   |
|                                   | Advise the employer or third      |
|                                   | party in writing they did not     |
|                                   | comply with a garnishee notice.   |
|                                   |                                   |
|                                   | Calculate the amount recoverable  |
|                                   | to date                           |
|                                   |                                   |
|                                   | Send a 'Garnishee-warning re      |
|                                   | non-compliance-employer & FI'     |
|                                   | (Q276) letter                     |
|                                   |                                   |
|                                   | Code the Review Registration      |
|                                   | (RVR) screen, completing the      |
|                                   | following fields:                 |
|                                   |                                   |
|                                   | -   ServRsn: key 'DMN'            |
|                                   | -   RevRsn: key 'REV'             |
|                                   | -   Due Date: key the date 14     |
|                                   |     days from the date the Q276   |
|                                   |     letter was sent               |
|                                   | -   Text: key 'follow up wage     |
|                                   |     garnishee non-compliance'     |
|                                   |     Coding the review for 14 days |
|                                   |     in the future gives the       |
|                                   |     employer or third party time  |
|                                   |     to respond to the Q276        |
|                                   |                                   |
|                                   | If the employer or third party    |
|                                   | does not respond to the Q276      |
|                                   | letter after 14 days, make a      |
|                                   | final attempt to contact employer |
|                                   | or third party by phone.          |
|                                   |                                   |
|                                   | If the employer or third party    |
|                                   | still refuses to comply with the  |
|                                   | notice, or cannot be contacted:   |
|                                   |                                   |
|                                   | -   cease the garnishee. See      |
|                                   |     Varying or ceasing a          |
|                                   |     garnishee notice to recover   |
|                                   |     debts                         |
|                                   | -   record details on a DOC, and  |
|                                   | -   pursue other                  |
|                                   |     recovery or non-recovery      |
|                                   |     action                        |
|                                   |                                   |
|                                   | Process ends here.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Garnishee notice has not been     |
|                                   | received + Read more ...          |
|                                   |                                   |
|                                   | -   Reprint the garnishee notice  |
|                                   | -   Discuss with the employer the |
|                                   |     best way to get notice to     |
|                                   |     them (for example, mailing    |
|                                   |     address, fax, Registered      |
|                                   |     Mail, PGP secure webmail)     |
|                                   | -   Update the Pay Start Date:    |
|                                   |     field on the Garnishee        |
|                                   |     Details (OPGD) screen. This   |
|                                   |     should be to the date         |
|                                   |     garnishee payment is expected |
|                                   |     or 2 weeks from today's date  |
|                                   |     (whichever is later)          |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Record all details on a DOC using |
|                                   | Fast Note located in Customer     |
|                                   | First + Read more ...             |
|                                   |                                   |
|                                   | On the summary line, key          |
|                                   | 'Recovery: Rang Empl re           |
|                                   | Garnishee'.                       |
|                                   |                                   |
|                                   | In the Txt: field, record all     |
|                                   | details of the conversation with  |
|                                   | the employer or third party.      |
