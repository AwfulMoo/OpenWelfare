This document outlines information on how to request access and navigate
the Aged Care Staff Portal (ACSP) home page.

Aged Care Staff Portal (ACSP) access

Service Officers must request to have the relevant business role in
ESSentials and access to the relevant Tivoli Access Manager (TAM) group.
Service Officers can start using the ACSP home page once access has been
granted.

Service Officers have the ability to manage provider details and service
user details via the ACSP home page. See Aged Care Staff Portal –
Managing provider and service user details for the process on how to do
this.

Service Officers can also use the ACSP via 2 different views below:

Service in Context - which provides information from a service’s
perspective:

-   See the Aged Care Staff Portal (ACSP) - Service in Context for
    further information on how to use the ACSP via the Service in
    Context view

Care Recipient in Context (CRiC) - provides a more holistic view from a
care recipient’s perspective compared to the Service in Context view:

-   See the Aged Care Staff Portal (ACSP) - Care Recipient in Context
    (CRiC) for further information on how to use the ACSP via the CRiC
    view

The Process page describes how Service Officers can request access to
the ACSP and use the ACSP home page.

The Resources page contains question and answers about the ACSP, a link
to the ACSP and mySupport.
