|                                   |                                   |
|                                   | Where an ineligible person has    |
|                                   | been enrolled in Medicare and the |
|                                   | information has been provided by  |
|                                   | another person (in person, over   |
|                                   | the phone or in correspondence),  |
|                                   | further investigation is          |
|                                   | necessary:                        |
|                                   |                                   |
|                                   | -   recall documents from         |
|                                   |     archives (where applicable)   |
|                                   | -   write to the person informing |
|                                   |     them it is necessary to       |
|                                   |     confirm their Medicare        |
|                                   |     eligibility                   |
|                                   | -   add a comment in the Personal |

|                                   | that details regarding a          |
|                                   | suspected incorrect enrolment     |
|                                   | cannot be disclosed to anyone     |
|                                   | other than the person concerned.  |

|                                   | alerted to an incorrect enrolment |
|                                   | by a member of the public.        |
|                                   |                                   |
|                                   | For further assistance see        |
