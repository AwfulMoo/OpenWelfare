| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Compensation and insurance        |
|                                   | payments + Read more ...          |
|                                   |                                   |
|                                   | Compensation for loss of wages is |
|                                   | not covered in this procedure.    |
|                                   | See Treatment of compensation     |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Compensation and insurance        |
|                                   | payments for damage to property   |
|                                   | and personal effects are not      |
|                                   | treated as income.                |
|                                   |                                   |
|                                   | If the payment is invested, do    |
|                                   | not record funds in any           |
|                                   | savings/investment accounts.      |
|                                   | Money earned on the investment is |
|                                   | not deemed under the Income Test  |
|                                   | for up to 12 months after the     |
|                                   | person receives the payment. The  |
|                                   | exemption from the Income Test    |
|                                   | deeming rules may be extended     |
|                                   | past 12 months if the customer    |
|                                   | can show they meant to spend the  |
|                                   | compensation or insurance payment |
|                                   | on repairs within 12 months, but  |
|                                   | were not able to do so for        |
|                                   | reasons beyond their control.     |
|                                   |                                   |
|                                   | See Exempt lump sums for further  |
|                                   | information about this.           |
|                                   |                                   |
|                                   | Exception: If the payment is not  |
|                                   | invested, but spent for a purpose |
|                                   | other than for which it was paid, |
|                                   | it may become immediately         |
|                                   | assessable under the Income and   |
|                                   | Assets Tests.                     |
|                                   |                                   |
|                                   | Advise customer of outcome.       |
|                                   |                                   |
|                                   | Procedure ends here.              |

| 2                                 | Blocked income + Read more ...    |
|                                   |                                   |
|                                   | Blocked income is not treated as  |
|                                   | income.                           |
|                                   |                                   |
|                                   | The term 'blocked income' is used |
|                                   | if a customer has severely        |
|                                   | limited or no access to income    |
|                                   | they have a legal entitlement to. |
|                                   |                                   |
|                                   | If a customer has blocked income  |
|                                   | not from an overseas source,      |
|                                   | clarify through local Policy or   |
|                                   | Helpdesk contacts to ensure       |
|                                   | correct assessment.               |
|                                   |                                   |
|                                   | Overseas income                   |
|                                   |                                   |
|                                   | In certain circumstances it may   |
|                                   | be accepted an overseas pension   |
|                                   | is not income for social security |
|                                   | purposes, if the prospect of      |
|                                   | receiving the money is so remote  |
|                                   | the moneys are not 'earned,       |
|                                   | derived or received for the       |
|                                   | person's own use or benefit',     |
|                                   | that is, effectively 'blocked'.   |
|                                   |                                   |
|                                   | See the References page for a     |
|                                   | link to the Guide to Social       |
|                                   | Security Law, section 4.3.6.10,   |
|                                   | Income from overseas pension.     |
|                                   | Overseas income is not considered |
|                                   | blocked just because the country  |
|                                   | of payment will not allow funds   |
|                                   | to be transferred out of that     |
|                                   | country unless it is unreasonable |
|                                   | for the customer to travel to     |
|                                   | that country, for example,        |
|                                   | medical reason preventing travel  |
|                                   | or civil unrest in that country.  |
|                                   | International Programme must      |
|                                   | determine if a payment is         |
|                                   | blocked.                          |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the International Programme    |
|                                   | contacts.                         |
|                                   |                                   |
|                                   | Review cases where blocked income |
|                                   | is being disregarded regularly.   |
|                                   | The timing of the review depends  |
|                                   | on the particular situation.      |
|                                   |                                   |
|                                   | -   Advise customer of outcome    |
|                                   | -   Record details on a DOC or    |
|                                   |     select [] to add NOTES        |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   |     outcome. If necessary, adjust |

| 5                                 | New Enterprise Incentive Scheme   |
|                                   | (NEIS) + Read more ...            |
|                                   |                                   |
|                                   | NEIS payments are treated as a    |
|                                   | direct deduction for the          |
|                                   | following payments:               |
|                                   |                                   |
|                                   | -   Age Pension                   |
|                                   | -   Carer Payment (CP)            |
|                                   | -   Disability Support Pension    |
|                                   |     (DSP)                         |
|                                   | -   JobSeeker Payment (JSP)       |
|                                   | -   Parenting Payment (Single     |
|                                   |     (PPS) or Partnered (PPP))     |
|                                   | -   Partner Allowance (PA)        |
|                                   | -   Rehabilitation Allowance      |
|                                   | -   Special Benefit (SpB)         |
|                                   | -   Widow Allowance (WA)          |
|                                   |                                   |
|                                   | Recipients of CP, PPS and DSP can |
|                                   | elect to remain on their          |
|                                   | Centrelink payment while          |
|                                   | participating in NEIS, up to the  |
|                                   | maximum of 52 weeks, providing    |
|                                   | all other pension/payment and     |
|                                   | NEIS eligibility criteria is met. |
|                                   |                                   |
|                                   | Effect of NEIS payment on         |
|                                   | partner's payments                |
|                                   |                                   |
|                                   | If a DSP or Age Pension customer  |
|                                   | is receiving NEIS, and their      |
|                                   | partner is receiving CP (caring   |
|                                   | for partner or spouse), treat 50% |
|                                   | of the payments made to the       |
|                                   | participant as a direct deduction |
|                                   | against both members of the       |
|                                   | couple.                           |
|                                   |                                   |
|                                   | If their partner is receiving a   |
|                                   | payment other than CP, the NEIS   |
|                                   | payment does not affect their     |
|                                   | payment rate under section        |
|                                   | 1187(2) and 8(8)(t) of the Social |
|                                   | Security Act 1991.                |

| 7                                 | Coding equity or merit based      |
|                                   | scholarship income + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The gross amount of the equity or |
|                                   | merit based scholarship is coded  |
|                                   | on the Other Income (OIN).        |
|                                   |                                   |
|                                   | If completing in Process Direct   |
|                                   | type Start into Super Key select  |
|                                   | Income and Assets Update workflow |
|                                   |                                   |
|                                   | Lump sum and non-periodic         |
|                                   | payments:                         |
|                                   |                                   |
|                                   | -   coded over a 12 month period  |
|                                   |     from the date the payment is  |
|                                   |     received                      |
|                                   | -   code the gross amount of the  |
|                                   |     scholarship using the code    |
|                                   |     'EMS' in the Type: field      |
|                                   | -   the system automatically      |
|                                   |     deducts the exempted portion  |
|                                   |     from the total and applies    |
|                                   |     the Income Test to the        |
|                                   |     remaining amount              |
|                                   |                                   |
|                                   | Fortnightly or periodic payments: |
|                                   |                                   |
|                                   | -   Coded over the period which   |
|                                   |     the payment represents from   |
|                                   |     the first day of the          |
|                                   |     assessable period             |
|                                   | -   Code 'EMS' in the Type: field |
|                                   |     and the actual amounts and    |
|                                   |     frequency in the relevant     |
|                                   |     fields                        |
|                                   | -   Note: when an equity or merit |
|                                   |     based scholarship is coded as |
|                                   |     a fortnightly or periodic     |
|                                   |     payment, the exempt income    |
|                                   |     amount is spread over the     |
|                                   |     calendar year. The exempt     |
|                                   |     income is divided by 26       |
|                                   |     fortnights and this is        |
|                                   |     deducted from the income      |
|                                   |     coded on OIN. The relevant    |
|                                   |     Income Test is then applied   |
|                                   |     to any remaining amounts each |
|                                   |     fortnight or period           |
|                                   | -   Set a review for the          |
|                                   |     appropriate date to cease     |
|                                   |     scholarship assessment. It is |
|                                   |     important to code a review to |
|                                   |     remove recorded scholarship   |
|                                   |     income                        |
|                                   |                                   |
|                                   | Review annual payments 12 months  |
|                                   | after their receipt. Set periodic |
|                                   | payments for review in Customer   |
|                                   | First when they are expected to   |
|                                   | cease. Record all information     |
|                                   | regarding customer contact and    |
|                                   | scholarship on a DOC or NOTE.     |
|                                   |                                   |
|                                   | Note: scholarships that are not   |
|                                   | equity or merit based must be     |
|                                   | coded separately on OIN screen as |
|                                   | income type 'OTH'.                |
