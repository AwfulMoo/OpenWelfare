| 8                                 | Carer in gaol or psychiatric      |
|                                   | confinement in connection with an |
|                                   | offence + Read more ...           |
|                                   |                                   |
|                                   | From 27 September 2006, respite   |
|                                   | days cannot be used where the     |
|                                   | carer is in gaol, or in           |
|                                   | psychiatric confinement in        |
|                                   | connection with an offence.       |
|                                   |                                   |
|                                   | If a carer is in periodic         |
|                                   | detention or weekend detention    |
|                                   | for a whole 24 hour period        |
|                                   | (midnight to midnight), they lose |
|                                   | qualification for CA as respite   |
|                                   | cannot be used for this absence   |
|                                   | and therefore, daily care is not  |
|                                   | being provided.                   |
|                                   |                                   |
|                                   | Suspend CA from the day after the |
|                                   | carer enters custody. The carer   |
|                                   | does not lose entitlement until   |
|                                   | the absence is a full 24 hour     |
|                                   | period (midnight to midnight). A  |
|                                   | 13 week review will be set up     |
|                                   | automatically once the suspension |
|                                   | is coded.                         |
|                                   |                                   |
|                                   | If the carer remains in gaol or   |
|                                   | psychiatric confinement for 13    |
|                                   | weeks or more, when the review    |
|                                   | matures it will cancel CA from    |
|                                   | the day after the date the carer  |
|                                   | had entered custody. The carer    |
|                                   | does not lose entitlement until   |
|                                   | the absence is a full 24 hours    |
|                                   | period (midnight to midnight).    |
|                                   |                                   |
|                                   | If the carer has already been     |
|                                   | confined for 13 weeks or more     |
|                                   | when Centrelink is notified of    |
|                                   | the absence, then CA is cancelled |
|                                   | immediately, without suspension,  |
|                                   | from the day after the carer had  |
|                                   | entered custody. The carer does   |
|                                   | not lose entitlement until the    |
|                                   | absence is a full 24 hour period  |
|                                   | (midnight to midnight).           |
|                                   |                                   |
|                                   | Note: this rule does not apply to |
|                                   | Health Care Card (HCC) only CA    |
|                                   | customers. HCC only carers do not |
|                                   | receive a payment - the HCC is    |
|                                   | issued in the child's name only   |
|                                   | and will continue to remain       |
|                                   | current, unless another carer     |
|                                   | lodges a claim.                   |
|                                   |                                   |
|                                   | Has the carer already been        |
|                                   | confined for 13 weeks or more?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 10            |
|                                   | -   No, go to Step 9              |
