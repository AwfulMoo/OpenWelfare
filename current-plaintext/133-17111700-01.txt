myGov Code Generator app

The myGov Code Generator app gives myGov customers a secure option to
sign in to their myGov account. The app creates a one-time access code
when signing in to myGov. When the app is downloaded and linked to an
active myGov account, the code generated may replace the security codes
by SMS or secret questions people currently use.
