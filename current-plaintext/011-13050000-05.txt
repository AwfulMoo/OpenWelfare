Examples of EBPRP reimbursement amounts

Table 2: This table describes examples of various amounts a claimant can
be reimbursed, including when amounts are received from private health
insurers.

+-----------------------------------+-----------------------------------+
| Example                           | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | If a claimant purchases a single  |
|                                   | external breast prosthesis for an |
|                                   | amount of $500, they will only be |
|                                   | reimbursed the maximum amount     |
|                                   | payable for each external breast  |
|                                   | prosthesis for each side. Which   |
|                                   | is a total of $400.               |
+-----------------------------------+-----------------------------------+
| 2                                 | If a claimant purchases a single  |
|                                   | external breast prosthesis for    |
|                                   | $450 and indicates on their claim |
|                                   | form that they have received an   |
|                                   | amount of $50 from their private  |
|                                   | health insurance, then the        |
|                                   | claimant will be reimbursed an    |
|                                   | amount of $350.                   |
+-----------------------------------+-----------------------------------+
| 3                                 | If a claimant purchases external  |
|                                   | breast prostheses for both breast |
|                                   | sides and the total amount paid   |
|                                   | was $950, the claimant will only  |
|                                   | be reimbursed an amount of $800.  |
|                                   | This is because the maximum       |
|                                   | amount a claimant can be          |
|                                   | reimbursed for each prostheses    |
|                                   | for each side is $400.            |
+-----------------------------------+-----------------------------------+
