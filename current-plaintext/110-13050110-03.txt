| 1                                 | Customer current on income        |
|                                   | support payment + Read more ...   |
|                                   |                                   |
|                                   | If the customer is current,       |
|                                   | ensure the intervention is        |
|                                   | started before any updates to the |
|                                   | record, Changes made when the     |
|                                   | intervention is not STA cannot be |
|                                   | attributed and claimed in IRS.    |
|                                   |                                   |
|                                   | Note: current customers include   |
|                                   | customers with suspended          |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Selecting the IRS intervention    |
|                                   |                                   |
|                                   | Before any updates on the         |
|                                   | customer's record, make sure the  |
|                                   | IRS intervention has a status of  |
|                                   | STA (started) and be in the       |
|                                   | Review Image.                     |
|                                   |                                   |
|                                   | Is the customer's payment         |
|                                   | current?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Suspend or cancel payment + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Use the Corrective Services       |
|                                   | script for processing all COR     |
|                                   | interventions.                    |
|                                   |                                   |
|                                   | -   Change the customer's postal  |
|                                   |     address to the prison address |
|                                   | -   Suspend or cancel the         |
|                                   |     customer's payment as         |
|                                   |     appropriate. Assessing a      |
|                                   |     member of a couple for a      |
|                                   |     customer with a partner in    |
|                                   |     prison                        |
|                                   | -   Check the Debt List (OPDL)    |
|                                   |     screen to see if a current    |
|                                   |     debt exists                   |
|                                   | -   Customers currently           |
|                                   |     incarcerated are to have all  |
|                                   |     debts temporarily written off |
|                                   |                                   |
|                                   | Outstanding Job seeker            |
|                                   | participation suspension          |
|                                   |                                   |
|                                   | If a job seeker participation     |
|                                   | failure prevents suspending or    |
|                                   | cancelling the payment IMP,       |
|                                   | contact the local Participation   |
|                                   | Solutions Team (PST) to have the  |
|                                   | record updated correctly. See     |
|                                   | Outstanding compliance action for |
|                                   | imprisoned job seekers.           |
|                                   |                                   |
|                                   | Mobility Allowance (MOB)          |
|                                   |                                   |
|                                   | If the customer gets MOB, cancel  |
|                                   | the payment IMP via the Benefit   |
|                                   | Action (BA) screen. IMP is not a  |
|                                   | valid suspension code for MOB.    |
|                                   |                                   |
|                                   | Date of release is unclear        |
|                                   |                                   |
|                                   | If the date of release is         |
|                                   | unclear, in all cases treat the   |
|                                   | customer as though the            |
|                                   | incarceration period is less than |
|                                   | 13 weeks and suspend payment if   |
|                                   | possible.                         |
|                                   |                                   |
|                                   | Youth Allowance (job seeker)      |
|                                   | turning 22                        |
|                                   |                                   |
|                                   | Cancel YA if the date of          |
|                                   | imprisonment is within 13 weeks   |
|                                   | of the customer turning 22. If    |
|                                   | only suspended, the system will   |
|                                   | transfer them to JobSeeker        |
|                                   | Payment.                          |
|                                   |                                   |
|                                   | Q134 MFU activities               |
|                                   |                                   |
|                                   | Action MFU activities that need a |
|                                   | manual Q134 letter to be sent to  |
|                                   | the customer after suspension.    |
|                                   | This is because the customer will |
|                                   | not get an automatic letter       |
|                                   | advising of the decision and      |
|                                   | their review and appeal rights.   |
|                                   |                                   |
|                                   | -   The text about why payments   |
|                                   |     have been suspended should    |
|                                   |     state:                        |
|                                   |     'people remanded in custody   |
|                                   |     awaiting trial are not        |
|                                   |     entitled to receive…'         |
|                                   |                                   |
|                                   | Note: as customers in custody     |
|                                   | cannot be contacted by phone,     |
|                                   | there is no need to make genuine  |
|                                   | attempts to contact to discuss    |
|                                   | the adverse decision. Clearly DOC |
|                                   | that the customer cannot be       |
|                                   | contacted by phone and proceed    |
|                                   | with the decision. The customer   |
|                                   | will receive a Q134 letter        |
|                                   | advising of the decision and      |
|                                   | their review and appeal rights.   |

| 3                                 | Check if customer has a current   |

|                                   | customer has a linked carer. The  |

|                                   |                                   |
|                                   | -   CRP care receiver (Carer      |
|                                   |     Payment), and/or              |
|                                   | -   CRA care receiver (Carer      |
|                                   |     Allowance)                    |
|                                   |                                   |
|                                   | To identify if there is a current |
|                                   | linked carer, check the Links     |
|                                   | Summary (LS) screen to see if     |
|                                   | there is a CRP and/or CRA link    |
|                                   | with no end date recorded.        |

|                                   | Virtual Carers nation team (CD1)  |
|                                   | need to be advised to make sure   |
|                                   | appropriate action is taken to    |
|                                   | reassess the carer’s entitlement  |

|                                   | screen, or                        |
|                                   |                                   |
|                                   | Key the CRN of the current carer  |
|                                   | and complete the following        |
|                                   | actions:                          |

|                                   | -   on the carer's record, send a |

|                                   |     (Virtual Carers National)     |
|                                   |     - use Fast Note - select      |
|                                   |     workflow use Carers >         |
|                                   |     SUS/CAN/RES > CP/CA Care      |
|                                   |     Receiver in custody           |
|                                   | -   on the care receivers record, |
|                                   |     go to the Review Summary      |
|                                   |     (IRRS) screen:                |
|                                   |     - key 'M' in the field next   |
|                                   |     to the COR (Correctives)      |
|                                   |     Intervention                  |
|                                   |     - on the Manage Review (IRMR) |

|                                   |                                   |
|                                   | Once the Fast Note is completed   |
|                                   | in the carer's record and the     |
