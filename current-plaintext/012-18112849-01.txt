This sub-topic outlines information about making changes and updates to
practice details in the Practice Incentives Program (PIP) and in the
Workforce Incentive Program (WIP) - Practice Stream. The information is
for Service Officers to use when processing updates to practice details
and confirmation statements.

The Resources page contains links to contact details for Practice
Incentives Program (PIP) and Workforce Incentive Program (WIP) -
Practice Stream.
