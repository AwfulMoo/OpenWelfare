This page contains information for restoring Partner Allowance (PA) and
Widow Allowance (WA) payments following suspension or cancellation.

On this page:

Making a decision on whether PA and WA can be restored

Restoring PA and WA within 13 weeks of suspension or cancellation

Making a decision on whether PA and WA can be restored

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Investigate the record. There may |
|                                   | be more than one reason for a     |
|                                   | suspension or cancellation.       |
|                                   |                                   |
|                                   | Suspension due to non-lodgement   |
|                                   | of TTA or SA383                   |
|                                   |                                   |
|                                   | 13 weeks prior to Age Pension     |
|                                   | age, eligible customers are       |
|                                   | invited to submit the online      |

|                                   | superannuation products must be   |

|                                   | information/action within the     |

|                                   | payment may be suspended.         |

|                                   | For all other                     |

|                                   | For suspension codes, see         |
|                                   | Cancellation, suspension and      |
|                                   | rejection reason codes for        |
|                                   | Newstart System (NSS).            |

Restoring PA and WA within 13 weeks of suspension or cancellation

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Suspension/cancellation period is |
|                                   | less than 13 weeks (3 months) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Determine and/or check:           |
|                                   |                                   |
|                                   | -   Check the reason for          |
|                                   |     suspension/cancellation via   |
|                                   |     the SRS screen                |
|                                   | -   Check the list of suspension  |
|                                   |     codes, find the relevant code |
|                                   |     and check if restorable. See  |
|                                   |     Cancellation, suspension and  |
|                                   |     rejection reason code for     |
|                                   |     Newstart System (NSS)         |
|                                   | -   Check the Document List (DL)  |
|                                   |     screen in the customer's      |
|                                   |     record and read any DOCs      |
|                                   |     regarding the                 |
|                                   |     suspension/cancellation       |
|                                   | -   Check the record to make sure |
|                                   |     everything that is required   |
|                                   |     for restoration has been      |
|                                   |     lodged/actioned               |
|                                   |                                   |
|                                   | Is the customer required to       |
|                                   | provide any information or        |
|                                   | documentation or meet any other   |
|                                   | requirement before payment can be |
|                                   | restored? Or is there any other   |
|                                   | information that prevents         |
|                                   | customer's record being restored  |
|                                   | and actioned at first contact?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Do not restore on initial         |
|                                   | contact + Read more ...           |
|                                   |                                   |
|                                   | WA or PA is suspended for         |
|                                   | non-lodgement of a Transfer to    |
|                                   | Age Pension request               |
|                                   |                                   |
|                                   | Service Officer's must discuss    |
|                                   | transferring to Age Pension. PA   |
|                                   | and WA payments will cease on 1   |
|                                   | January 2022. To avoid a break in |
|                                   | payment, customers will need to   |
|                                   | transfer to Age Pension on or     |
|                                   | before this date                  |
|                                   |                                   |
|                                   | -   If the customer wants to      |
|                                   |     transfer to Age Pension,      |
|                                   |     complete an ACC at the first  |
|                                   |     point of contact and advise   |
|                                   |     what documentation is         |
|                                   |     required. See Transfer to Age |
|                                   |     Pension                       |
|                                   |     Go to Step 3                  |
|                                   | -   If the customer declines to   |
|                                   |     transfer to Age Pension,      |
|                                   |     superannuation products must  |
|                                   |     still be provided before the  |
|                                   |     payment can be restored.      |
|                                   |     Advise the customer WA and PA |
|                                   |     payments will cease on the 01 |
|                                   |     January 2022                  |
|                                   |     If the customer does not have |
|                                   |     any superannuation            |
|                                   |     investments, their payment    |
|                                   |     can be restored               |
|                                   |                                   |
|                                   | Please annotate the 'Urgent       |
|                                   | WDA/PA Discussion REQ' display on |
|                                   | access DOC with details of the    |
|                                   | discussion. If there is no        |
|                                   | display on access DOC, record a   |
|                                   | new DOC with details of the       |
|                                   | discussion                        |
|                                   |                                   |
|                                   | All other reasons                 |
|                                   |                                   |
|                                   | Do not restore payment. Advise    |
|                                   | the customer what is required for |
|                                   | restoration. Payment cannot be    |
|                                   | restored until all information    |
|                                   | has been provided and/or action   |
|                                   | has been undertaken as required   |
|                                   | and has been verified.            |
|                                   |                                   |
|                                   | Record details on a DOC including |
|                                   | advice given to the customer. If  |
|                                   | there is an 'Urgent WDA/PA        |
|                                   | Discussion REQ' display on access |
|                                   | DOC on the record please annotate |
|                                   | this with details of the          |
|                                   | discussion.                       |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Restore Payments + Read more ...  |
|                                   |                                   |
|                                   | PA and WA can only be restored    |
|                                   | where the customer has continuous |
|                                   | entitlement. If there has been    |
|                                   | any break in entitlement during   |
|                                   | the suspension period, payment    |
|                                   | cannot be restored. The customer  |
|                                   | will need to test their           |
|                                   | eligibility for a different       |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Confirm with the customer if      |
|                                   | there have been any changes of    |
|                                   | circumstances they should         |
|                                   | advise - update accordingly.      |
|                                   |                                   |
|                                   | TTA lodged and has declined to    |
|                                   | transfer to Age Pension           |
|                                   |                                   |
|                                   | Service Officers must discuss     |
|                                   | transferring to Age Pension. PA   |
|                                   | and WA payments cease on 1        |
|                                   | January 2022. To avoid a break in |
|                                   | payment, customers will need to   |
|                                   | transfer to Age Pension on or     |
|                                   | before this date. If the customer |
|                                   | wants to transfer to Age Pension, |
|                                   | complete ACC at the first point   |
|                                   | of contact. See Transfer to Age   |
|                                   | Pension.                          |
|                                   |                                   |
|                                   | Income management                 |
|                                   |                                   |
|                                   | If the customer is under income   |
|                                   | management (they have 'INM' on    |
|                                   | the benefit status line) and they |
|                                   | have an arrears or lump sum       |
|                                   | payment entitlement, do not make  |
|                                   | any changes to the customer's     |
|                                   | record to allow this payment to   |
|                                   | be made. Contact the Income       |
|                                   | Management Team to allow income   |
|                                   | management to be turned on for    |
|                                   | the arrears payment before it is  |
|                                   | made.                             |
|                                   |                                   |
|                                   | Rent assistance paid with FTB     |
|                                   |                                   |
|                                   | If the customer's Family Tax      |
|                                   | Benefit (FTB) is cancelled as a   |
|                                   | result of the income support      |
|                                   | payment being suspended/cancelled |
|                                   | restore FTB first where Rent      |
|                                   | Assistance is paid with the FTB.  |
|                                   |                                   |
|                                   | To update record:                 |
|                                   |                                   |
|                                   | -   Go to the Benefit Action (BA) |
|                                   |     screen                        |
|                                   | -   Enter the type of benefit in  |
|                                   |     the Svc Rsn: field and enter  |
|                                   |     the action 'RES' in the       |
|                                   |     Action: field                 |
|                                   | -   Press [Enter] and the effect  |
|                                   |     date will default or enter    |
|                                   |     the effect date of the        |
|                                   |     restoration in the Event      |
|                                   |     Date: field                   |
|                                   | -   Complete Source: and DOR:     |
|                                   |     fields                        |
|                                   | -   If the customer is income     |
|                                   |     managed under the Voluntary   |
|                                   |     Income Management measure,    |
|                                   |     and the agreement was in      |
|                                   |     place for more than 13 weeks, |
|                                   |     the Voluntary Income          |
|                                   |     Management Continuation       |
|                                   |     Assessment page is presented. |
|                                   |     The purpose of this screen is |
|                                   |     to record whether the         |
|                                   |     customer wishes to continue   |
|                                   |     or terminate their VIM        |
|                                   |     agreement. Refer to Effect on |
|                                   |     Income Management when        |
|                                   |     payment is suspended, on zero |
|                                   |     rate, cancelled, or           |
|                                   |     Unemployment Non-Payment      |
|                                   |     Period applied                |
|                                   | -   Annotate the 'Urgent WDA/PA   |
|                                   |     Discussion REQ' display on    |
|                                   |     access DOC or record a new    |
|                                   |     DOC including action taken    |
|                                   |     and advice given to customer, |
|                                   |     and of any changes in         |
|                                   |     circumstances that have been  |
|                                   |     recorded                      |
|                                   | -   Go to the Assessment Results  |
|                                   |     (AR) screen                   |
|                                   | -   Check if customer is entitled |
|                                   |     to any arrears and that they  |
|                                   |     appear correctly on the AR    |
|                                   |     screen, and                   |
|                                   | -   Finalise the activity         |
|                                   |                                   |
|                                   | The payment will be restored from |
|                                   | the date of effect and a letter   |
|                                   | will be generated advising the    |
|                                   | customer that their payments have |
|                                   | been restored.                    |
