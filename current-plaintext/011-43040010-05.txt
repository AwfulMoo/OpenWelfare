This page contains a list of adjustment types and their descriptions for
bulk bill latter day adjustment (LDA) claims in Medicare.

Adjustment group items

Adjustment types and descriptions

This table lists what the system displays when an adjustment is
processed for an item that is grouped with other items (for example,
multiple operations and composites).

+-----------------------------------+-----------------------------------+
| Adjustment type                   | Description                       |
+-----------------------------------+-----------------------------------+
| Keyed over with corrected details | All fields are protected, except  |
|                                   | the A field that contains an O    |
|                                   | for original line:                |
|                                   |                                   |
|                                   | -   that was adjusted             |
|                                   | -   for all other items in the    |
|                                   |     group                         |
|                                   |                                   |
|                                   | All keying fields are unprotected |
|                                   | and R is displayed in the A field |
|                                   | for restated line for:            |
|                                   |                                   |
|                                   | -   the corrected line            |
|                                   | -   all other items in the group  |
+-----------------------------------+-----------------------------------+
| Deleted line                      | All fields are protected, except  |
|                                   | the A field which contains an O   |
|                                   | for the original line for:        |
|                                   |                                   |
|                                   | -   the deleted item              |
|                                   | -   all other items in the group  |
|                                   |                                   |
|                                   | All fields are protected, except  |
|                                   | the A field that contains an R    |
|                                   | for:                              |
|                                   |                                   |
|                                   | -   a restated line for the       |
|                                   |     deleted item                  |
|                                   | -   all other items in the group  |
|                                   |                                   |
|                                   | The BENEFIT field contains a zero |
|                                   | amount and the PAYEE code is      |
|                                   | change to H to indicate the line  |
|                                   | is recorded, for history purposes |
|                                   | only, as a total overpayment.     |
+-----------------------------------+-----------------------------------+
| Inserted line                     | All fields are protected, except  |
|                                   | the A field which contains and O  |
|                                   | for the original line for all     |
|                                   | items in the group where the line |
|                                   | was inserted.                     |
|                                   |                                   |
|                                   | All keying fields are unprotected |
|                                   | and the A field contains an:      |
|                                   |                                   |
|                                   | -   N for the new (inserted) line |
|                                   | -   R for restated lines for all  |
|                                   |     items in the group            |
+-----------------------------------+-----------------------------------+
