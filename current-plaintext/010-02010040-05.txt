ABSTUDY request for private board provider details form

If a secondary student is boarding privately, the ABSTUDY request for
private board provider details form must be issued to the person
responsible for making payment decisions.

The person responsible for redirecting payments to a private board
provider is either a:

-   parent/guardian for a dependant student under 18 years of age
-   student 18 years of age or over
-   student claiming independence

ABSTUDY Claiming Tool (ACT) FAQ's

Table 1: This table provides information on the ABSTUDY Claiming Tool
(ACT) processes and handy hints and tips.

+-----------------------------------+-----------------------------------+
| Question                          | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | What happens to the ACT once the  |
|                                   | claim is finalised?               |
|                                   |                                   |
|                                   | The ACT retains the answers       |
|                                   | provided. The processing copy is  |
|                                   | uploaded to the student's record, |
|                                   | except for Schooling A claims.    |
|                                   | These claims are uploaded to the  |
|                                   | applicant's record. The           |
|                                   | processing copy can be located    |
|                                   | via Document Tools in Customer    |
|                                   | First menu.                       |
|                                   |                                   |
|                                   | A NCL activity will be indexed    |
|                                   | and the details recorded in a NCL |
|                                   | progress DOCument. The activity   |
|                                   | is then transferred (O) to the    |
|                                   | ABSTUDY Smart Centre teams for    |
|                                   | finalisation.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Can there be more than one claim  |
|                                   | at a time?                        |
|                                   |                                   |
|                                   | Yes - multiple claims can now be  |
|                                   | completed.                        |
|                                   |                                   |
|                                   | Schooling A claims for more than  |
|                                   | one student can be recorded       |
|                                   | within the same ACT workflow.     |
|                                   |                                   |
|                                   | For all other ACT claims, the     |
|                                   | instructions in Table 4 must be   |
|                                   | completed before commencing       |
|                                   | another ACT claim.                |
+-----------------------------------+-----------------------------------+
| 3                                 | How do I re-print the Customer    |
|                                   | Declaration Form (CDF)?           |
|                                   |                                   |
|                                   | The CDF can be re-printed from    |
|                                   | the History Summary (HS) screen.  |
|                                   |                                   |
|                                   | To re-print and identify the CDF, |
|                                   | use the letter code:              |
|                                   |                                   |
|                                   | -   ABY - Student claiming -      |
|                                   |     ABYYBALA                      |
|                                   | -   ABY - Parent claiming -       |
|                                   |     ABYYBALP                      |
|                                   | -   ABT - Applicant claiming -    |
|                                   |     ABYYBALS                      |
|                                   | -   PES - Student claiming -      |
|                                   |     ABYYBALA                      |
|                                   |                                   |
|                                   | Select the twisty next to the     |
|                                   | letter required.                  |
|                                   |                                   |
|                                   | Select Print Local.               |
|                                   |                                   |
|                                   | On the Letter Print (LPA) screen: |
|                                   |                                   |
|                                   | -   Copy: field - code 'Y'        |
|                                   | -   Printer: field - code the     |
|                                   |     required printer number       |
|                                   | -   select Continue               |
+-----------------------------------+-----------------------------------+
| 4                                 | How do I access a Processing copy |
|                                   | after the NCL has been cancelled? |
|                                   |                                   |
|                                   | If the NCL activity has been      |
|                                   | cancelled, the processing copy    |
|                                   | can be accessed via Customer      |
|                                   | First - Document Tools menu.      |
|                                   | Staff can view the information    |
|                                   | recorded in the processing copy   |
|                                   | prior to re-indexing the NCL      |
|                                   | activity.                         |
+-----------------------------------+-----------------------------------+
