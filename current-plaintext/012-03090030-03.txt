This page contains information to assist Service Officers in creating a
Responsible Officer (RO) record in the Healthcare Identifiers (HI)
Service.

Create RO records in the HI Service

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Logon and search + Read more ...  |
|                                   |                                   |
|                                   | Log on to the eBusiness Home      |
|                                   | Page. The Search for Individual   |
|                                   | screen displays.                  |
|                                   |                                   |
|                                   | Perform a search for the RO. See  |
|                                   | Searching for records in the      |
|                                   | Healthcare Identifiers Service    |
|                                   | using the eHealth Program (EHP).  |
|                                   |                                   |
|                                   | Does a record exist for an        |
|                                   | existing eHealth Program (EHP)    |
|                                   | individual?                       |
|                                   |                                   |
|                                   | Note: this may be an existing RO, |
|                                   | Organisation Maintenance Officer  |
|                                   | (OMO) or Healthcare Provider      |
|                                   | Identifier – Individual (HPI-I)   |
|                                   | record?                           |
|                                   |                                   |
|                                   | -   Yes, do not create a new      |
|                                   |     record                        |
|                                   |     Note: the existing record     |
|                                   |     must be used and linked as a  |
|                                   |     RO when creating a seed       |
|                                   |     organisation. Procedure ends  |
|                                   |     here                          |
|                                   | -   No, select Create PO          |
|                                   |                                   |
|                                   | The Create Public Officer screen  |
|                                   | displays defaulting to the Name   |
|                                   | Details tab.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Check name details + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Key or select name details for    |
|                                   | the RO and select Add.            |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | in the Name Usage field, always   |
|                                   | select Registered Name (Legal     |
|                                   | Name)                             |
|                                   |                                   |
|                                   | Name Usage Start Date (will       |
|                                   | automatically default to today’s  |
|                                   | date)                             |
|                                   |                                   |
|                                   | leave the below fields blank:     |
|                                   |                                   |
|                                   | -   Name Usage End Date           |
|                                   | -   End Date Accuracy Indicator   |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   invalid, action the return    |
|                                   |     messages                      |
|                                   | -   valid, go to Step 3           |
|                                   |                                   |
|                                   | Note: to add additional name      |
|                                   | details, select Create New Name   |
|                                   | and repeat this step.             |
+-----------------------------------+-----------------------------------+
| 3                                 | Select EOI Details tab + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In the EOI Documents field,       |
|                                   | select:                           |
|                                   |                                   |
|                                   | -   the name of a document        |
|                                   |     provided by the OMO           |
|                                   | -   Add                           |
|                                   |                                   |
|                                   | See also: Evidence of Identity    |
|                                   | for Healthcare Identifiers (HI)   |
|                                   | Service.                          |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   unless the RO is a known      |
|                                   |     customer, they must provide 1 |
|                                   |     document from the Primary     |
|                                   |     group and a combination       |
|                                   |     enough to make up the         |
|                                   |     remaining points from the     |
|                                   |     Secondary group               |
|                                   | -   if the RO is a known          |
|                                   |     customer, select known        |
|                                   |     Medicare customer in the EOI  |
|                                   |     Documents field               |
|                                   | -   always enter all documents    |
|                                   |     provided by the applicant     |
|                                   |                                   |
|                                   | -   EOI Type field is not         |
|                                   |     populated                     |
|                                   | -   EOI option selected displays  |
|                                   |     at the bottom of the screen   |
+-----------------------------------+-----------------------------------+
| 4                                 | Evidence of identity (EOI) + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Add EOI documents to confirm      |
|                                   | evidence of identity. Select:     |
|                                   |                                   |
|                                   | -   Create New EOI                |
|                                   | -   the name of a document        |
|                                   |     provided by the RO            |
|                                   | -   Add                           |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   invalid, action the returned  |
|                                   |     messages                      |
|                                   | -   valid and evidence of         |
|                                   |     identity has been confirmed,  |
|                                   |     go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Select Personal Details tab +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Key or select personal details    |
|                                   | for the RO and select Validate.   |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   invalid, action the returned  |
|                                   |     messages                      |
|                                   | -   valid, the screen rebuilds    |
|                                   |     and returns address details   |
|                                   |     fields, go to Step 6          |
+-----------------------------------+-----------------------------------+
| 6                                 | Address details + Read more ...   |
|                                   |                                   |
|                                   | Select Address Details tab        |
|                                   |                                   |
|                                   | The Unstructured Address          |
|                                   | Indicator tick box is             |
|                                   | automatically selected. See       |
|                                   | Address formatting in Healthcare  |
|                                   | Identifiers                       |
|                                   |                                   |
|                                   | Enter the address in the Address  |
|                                   | Search box, start with the number |
|                                   | of the address. The rapid         |
|                                   | addressing tool predicts the      |
|                                   | address and displays it as the    |
|                                   | number and street name starts to  |
|                                   | be entered. If the address that   |
|                                   | appears is the correct address,   |
|                                   | select it                         |
|                                   |                                   |
|                                   | The below information             |
|                                   | pre-populates into the relevant   |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Address Lines                 |
|                                   | -   Suburb/Town/Locality          |
|                                   | -   State                         |
|                                   | -   Postcode                      |
|                                   |                                   |
|                                   | Select Address Purpose            |
|                                   |                                   |
|                                   | Address Start Date (defaults to   |
|                                   | today’s date)                     |
|                                   |                                   |
|                                   | Start Date Accuracy Indicator     |
|                                   | (defaults to Day, month and year  |
|                                   | are accurate)                     |
|                                   |                                   |
|                                   | Select Add                        |
+-----------------------------------+-----------------------------------+
| 7                                 | Select Electronic Communications  |
|                                   | Details tab + Read more ...       |
|                                   |                                   |
|                                   | Key or select electronic          |
|                                   | communications details and select |
|                                   | Add.                              |
|                                   |                                   |
|                                   | If the information is:            |
|                                   |                                   |
|                                   | -   missing or invalid, action    |
|                                   |     returned messages             |
|                                   | -   valid, the electronic         |
|                                   |     communications details        |
|                                   |     display at the bottom of the  |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Note: to add additional           |
|                                   | electronic communications details |
|                                   | select Create New Contact and     |
|                                   | repeat this step.                 |
+-----------------------------------+-----------------------------------+
| 8                                 | Select Certificates tab + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If there is:                      |
|                                   |                                   |
|                                   | -   no certificate information on |
|                                   |     the page, go to Step 9        |
|                                   | -   certificate information see   |
|                                   |     Authentication required to    |
|                                   |     access digital health         |
+-----------------------------------+-----------------------------------+
| 9                                 | Select Validate + Read more ...   |
|                                   |                                   |
|                                   | Are there any errors?             |
|                                   |                                   |
|                                   | -   Yes, action returned messages |
|                                   | -   No, select Submit. The        |
|                                   |     Confirm Create Public Officer |
|                                   |     screen displays               |
+-----------------------------------+-----------------------------------+
| 10                                | Finalise + Read more ...          |
|                                   |                                   |
|                                   | If details are:                   |
|                                   |                                   |
|                                   | -   correct, select Save          |
|                                   | -   incorrect, select Edit to     |
|                                   |     return the Amend Public       |
|                                   |     Officer screen, or select     |
|                                   |     Amend in the Action column    |
|                                   |     for the tab where the details |
|                                   |     are in error                  |
|                                   |                                   |
|                                   | Confirmation message displays the |
|                                   | RO ID.                            |
+-----------------------------------+-----------------------------------+
