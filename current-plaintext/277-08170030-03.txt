Actioning bulk income requests

Required tasks when bulk income request received

This table describes the process Quality Improvement Officers follow
when actioning a bulk income request from NZIR.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Request information from ATO +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Email the spreadsheet to the      |
|                                   | Business Improvement team, they   |
|                                   | will liaise with the ATO for the  |
|                                   | provision of incomes.             |
|                                   |                                   |
|                                   | Income information will be        |
|                                   | returned from the ATO with one of |
|                                   | the following confidence level    |
|                                   | ratings:                          |
|                                   |                                   |
|                                   | -   High Confidence Level -       |
|                                   |     considered reliable without   |
|                                   |     further investigation         |
|                                   | -   Medium Confidence Level -     |
|                                   |     manually search to verify     |
|                                   |     information, and              |
|                                   | -   Low                           |
|                                   |                                   |
|                                   |    Confidence/Unresolved/Multiple |
|                                   |     Matches                       |
|                                   | -   The spreadsheet is saved as   |
|                                   |     ‘ATO Income supplied NZBAA    |
|                                   |     dd.mm.yy’.                    |
|                                   |                                   |
|                                   | The customers with a Medium Level |
|                                   | rating are forwarded for review   |
|                                   | to determine if the ATO details   |
|                                   | are a match. These are forwarded  |
|                                   | to the Program Manager in         |
|                                   | International Child Support for   |
|                                   | consideration for which business  |
|                                   | area will action this workload.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Send income information to NZIR + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Once all the customers with       |
|                                   | Incomes from ATO have been        |
|                                   | confirmed as High, send the       |
|                                   | spreadsheet with the requested    |
|                                   | information to NZIR via secure    |
|                                   | messaging.                        |
|                                   |                                   |
|                                   | Bulk incomes are provided to NZIR |
|                                   | to enable them to create their    |
|                                   | new assessments (which run from 1 |
|                                   | April to 31 March in line with    |
|                                   | their fiscal year). Once NZIR     |
|                                   | have the incomes and create their |
|                                   | new assessments NZIR provide      |
|                                   | Child Support with bulk annual    |
|                                   | assessments.                      |
+-----------------------------------+-----------------------------------+

Actioning bulk annual assessments

Required tasks to update assessment information

This table describes the process Quality Improvement Officers follow in
order to have assessment information updated following a transfer of
information between NZIR and Child Support.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check spreadsheet + Read more ... |
|                                   |                                   |
|                                   | Check if the spreadsheet provides |
|                                   | the following information:        |
|                                   |                                   |
|                                   | -   arrears as at 31/03/YY        |
|                                   | -   includes payments received    |
|                                   |     date is the last payment      |
|                                   |     schedule date                 |
|                                   | -   monthly assessed amount is    |
|                                   |     ongoing from 01/04/YY, and    |
|                                   | -   there are no blank fields in  |
|                                   |     any of the columns            |
|                                   | -   there are no duplicate IRDs   |
|                                   |     or CSIDs on the sheet         |
|                                   |                                   |
|                                   | If no, request information from   |
|                                   | the nominated NZIR Team Leader    |
|                                   | via email and/or phone.           |
|                                   |                                   |
|                                   | If yes, go to Step 2.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Save spreadsheet + Read more ...  |
|                                   |                                   |
|                                   | Save the ‘BAA Run to DHS dd mm yy |
|                                   | sent’ spreadsheet to Hobart’s     |
|                                   | shared drive as follows:          |
|                                   |                                   |
|                                   | -   create a folder titled ‘BAA’  |
|                                   |     in the folder titled Int / NZ |
|                                   |     / 20{YY}                      |
|                                   | -   save the spreadsheet in its   |
|                                   |     original format, rename       |
|                                   |     ‘20{YY} BAA Master List – do  |
|                                   |     not touch’ and password       |
|                                   |     protect, open as read only    |
|                                   |     unless password entered       |
|                                   |                                   |
|                                   | Do not make any changes to this   |
|                                   | spreadsheet, it should remain as  |
|                                   | a clean copy of the exact data    |
|                                   | sent by NZIR                      |
|                                   |                                   |
|                                   | -   create a copy of the master   |
|                                   |     sheet and title ‘NZ BAA       |
|                                   |     20{YY} Working list’          |
|                                   |                                   |
|                                   | This is the end of the exchange   |
|                                   | of information process. To        |
|                                   | process the bulk annual           |
|                                   | assessments, see New Zealand Case |
|                                   | Management - Bulk Annual          |
|                                   | Assessments of Child Support –    |
|                                   | NZIR to Child Support.            |
