Notification that a child support payee is deceased

Table 1: this table describes the initial and immediate steps when
notification of a payee's death is received.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notification of death + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the customer is Indigenous,    |
|                                   | obtain permission before          |
|                                   | mentioning the deceased by name.  |
|                                   | Check the indicators on the       |
|                                   | Customer window for an Indigenous |
|                                   | indicator.                        |
|                                   |                                   |
|                                   | Deceased payees are managed       |
|                                   | within the Deceased Customer      |
|                                   | Management (DCM) team. If advised |
|                                   | of a payee's death, warm transfer |
|                                   | the call to the DCM team using    |
|                                   | Interaction Workspace (iWS)       |
|                                   | Softphone.                        |
|                                   |                                   |
|                                   | Child Support can be notified of  |
|                                   | a customer's death by:            |
|                                   |                                   |
|                                   | -   Centrelink, go to Step 2      |
|                                   | -   the payer or a third party,   |
|                                   |     go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Centrelink notification + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When Centrelink are informed of a |
|                                   | mutual customer's death the       |
|                                   | information is automatically      |
|                                   | uploaded to the Deceased Client   |
|                                   | window and an intray is           |
|                                   | generated.                        |
|                                   |                                   |
|                                   | Note: Centrelink notification     |
|                                   | alone is not sufficient proof of  |
|                                   | a customer's death, DCM Service   |
|                                   | Officers must obtain additional   |
|                                   | proof. Go to Step 4.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Notification from payer or third  |
|                                   | party + Read more ...             |
|                                   |                                   |
|                                   | When a payer or third party       |
|                                   | notifies of a payee's death,      |
|                                   | obtain the following information: |
|                                   |                                   |
|                                   | -   name and contact details of   |
|                                   |     the caller, if a third party  |
|                                   | -   sufficient detail to identify |
|                                   |     the deceased, such as name,   |
|                                   |     address and date of birth     |
|                                   | -   date of death                 |
|                                   | -   country/state/town the death  |
|                                   |     occurred in                   |
|                                   | -   if the payee had a will       |
|                                   | -   name and contact details of   |
|                                   |     the Executor or person who    |
|                                   |     will be finalising the        |
|                                   |     affairs of the deceased       |
|                                   | -   the name of the funeral       |
|                                   |     director                      |
|                                   | -   where the children are        |
|                                   |     residing, and                 |
|                                   | -   if the caller does not know   |
|                                   |     these details, try to obtain  |
|                                   |     contact details of another    |
|                                   |     person who may                |
|                                   |                                   |
|                                   | Note: if the caller is a third    |
|                                   | party do not disclose any         |
|                                   | information about either parent   |
|                                   | or the case to the caller. Accept |
|                                   | and record any information they   |
|                                   | provide.                          |
|                                   |                                   |
|                                   | If the caller is upset, check if  |
|                                   | there is someone available that   |
|                                   | the caller would prefer Child     |
|                                   | Support to speak with or if they  |
|                                   | would like to continue the call   |
|                                   | at a later date.                  |
|                                   |                                   |
|                                   | If the caller wishes to continue  |
|                                   | the call at a later date,         |
|                                   | document the conversation on the  |
|                                   | Deceased Client window and offer  |
|                                   | to transfer them to the Family    |
|                                   | Relationship Advice Line or       |
|                                   | Social worker and other service   |
|                                   | referrals for child support       |
|                                   | customers. For more information,  |
|                                   | see Products and Services Cuba    |
|                                   | Process Help. Go to Step 4.       |
|                                   |                                   |
|                                   | When speaking to a person who now |
|                                   | has care of the children, advise  |
|                                   | them they can lodge an            |
|                                   | application for child support.    |
|                                   |                                   |
|                                   | If the caller is the payer and    |
|                                   | wishes to continue the call,      |
|                                   | explain:                          |
|                                   |                                   |
|                                   | -   once the death has been       |
|                                   |     confirmed, the case will be   |
|                                   |     ended from the day before the |
|                                   |     date of death                 |
|                                   | -   any child support debt or     |
|                                   |     overpayment on the case and   |
|                                   |     if the amount is likely to    |
|                                   |     change                        |
|                                   | -   options available for         |
|                                   |     addressing the child support  |
|                                   |     debt or overpayment. See Step |
|                                   |     5 in Table 2                  |
|                                   |                                   |
|                                   | Support services are available    |
|                                   | and, if appropriate, offer to     |
|                                   | transfer them to the Family       |
|                                   | Relationship Advice Line or       |
|                                   | Social worker and other service   |
|                                   | referrals for child support       |
|                                   | customers. See Products and       |
|                                   | Services Cuba process help.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Immediate action required + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When notification of death is     |
|                                   | received the follow these steps   |
|                                   | for all cases until proof of      |
|                                   | death is received and the case is |
|                                   | finalised:                        |
|                                   |                                   |
|                                   | -   lock the deceased customer    |
|                                   |     and any associated customers  |
|                                   | -   record the death in the       |
|                                   |     Deceased Client window. See   |
|                                   |     Deceased customer management  |
|                                   |     Cuba Process Help for more    |
|                                   |     information                   |
|                                   | -   do not review or reconcile    |
|                                   |     any estimates. If a review or |
|                                   |     reconciliation is underway    |
|                                   |     suspend activity until a case |
|                                   |     management strategy is        |
|                                   |     decided                       |
|                                   | -   add a Sensitive Issue         |
|                                   |     Indicator to the deceased     |
|                                   |     customer and all associated   |
|                                   |     customers, see Table 2 in     |
|                                   |     Sensitive issue management    |
|                                   |     Cuba Process Help             |
|                                   | -   if the payee owes has an      |
|                                   |     overpayment or consolidated   |
|                                   |     revenue debt add a Special    |
|                                   |     Recovery Condition            |
|                                   | -   if notification of death is   |
|                                   |     received via Centrelink       |
|                                   |     exchange of information or    |
|                                   |     through the business (for a   |
|                                   |     customer who uses Centrelink  |
|                                   |     services) and more            |
|                                   |     information is required,      |
|                                   |     check Customer First for the  |
|                                   |     required information          |
|                                   | -   suspend payee overpayment     |
|                                   |     summaries in the Payment      |
|                                   |     Options window                |
|                                   |                                   |
|                                   | See the Deceased Customer         |
|                                   | Management Key Performance        |
|                                   | Indicator Checklist located on    |
|                                   | the Resources page of the         |
|                                   | Deceased Child Support Customer   |
|                                   | Management procedure.             |
|                                   |                                   |
|                                   | Is the payee involved in an       |
|                                   | International case (identified by |
|                                   | an 'INT' indicator on either of   |
|                                   | the parent's customer screens)?   |
|                                   |                                   |
|                                   | Yes, go to Step 5                 |
|                                   |                                   |
|                                   | No. If the case is:               |
|                                   |                                   |
|                                   | -   private collect with no       |
|                                   |     arrears, procedure ends here  |
|                                   | -   a single collect case (or     |
|                                   |     private collect with          |
|                                   |     arrears), go to Step 6        |
+-----------------------------------+-----------------------------------+
| 5                                 | International case + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Go to the Case screen to see what |
|                                   | type of application the case is   |
|                                   | based on.                         |
|                                   |                                   |
|                                   | If the case type is ASSESSMENT,   |
|                                   | action the deceased notification  |
|                                   | in the usual way. Go to Step 6.   |
|                                   |                                   |
|                                   | If the case type is INTERL CO/AGR |
|                                   | or INTERL ASSMNT, contact the     |
|                                   | Central Authority Team:           |
|                                   |                                   |
|                                   | -   create a STAKEHOLDER FOLLOW   |
|                                   |     UP intray                     |
|                                   | -   reset the intray due date to  |
|                                   |     2 business days from today's  |
|                                   |     date (instead of 14 days)     |
|                                   | -   add the Service Officer's     |
|                                   |     contact details to the intray |
|                                   |     notepad with the reason for   |
|                                   |     contacting the Central        |
|                                   |     Authority Team, for example   |
|                                   |     DCM International case –      |
|                                   |     INTERL CO/AGR or INTERL       |
|                                   |     ASSMENT), and                 |
|                                   | -   route to MC S003 Cent Auth    |
|                                   |     Tm1-none                      |
|                                   |                                   |
|                                   | A Central Authority Service       |
|                                   | Officer will work collaboratively |
|                                   | with the Child Support Service    |
|                                   | Officer and undertake any         |
|                                   | necessary liaison with the        |
|                                   | Overseas Authority. For example,  |
|                                   | the Overseas Authority may assist |
|                                   | Child Support in establishing     |
|                                   | proof of death and if there is a  |
|                                   | will.                             |
|                                   |                                   |
|                                   | The Overseas Authority must be    |
|                                   | consulted on whether collection   |
|                                   | of any arrears is required and    |
|                                   | the date the case should be       |
|                                   | ended.                            |
|                                   |                                   |
|                                   | The DCM officer will be kept      |
|                                   | informed and all actions will be  |
|                                   | recorded on the customer’s Cuba   |
|                                   | record.                           |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | A single case - Collect, or       |
|                                   | Private Collect with arrears +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the case is a single case -    |
|                                   | Collect, or Private Collect with  |
|                                   | arrears:                          |
|                                   |                                   |
|                                   | suspend payer statements, in the  |
|                                   | Payment Options window            |
|                                   |                                   |
|                                   | place all payments on hold        |
|                                   |                                   |
|                                   | suspend all collection action     |
|                                   | (stopping any EW and Centrelink   |
|                                   | deductions), and                  |
|                                   |                                   |
|                                   | suspend the accrual of late       |
|                                   | payment penalties, see Table 3 in |
|                                   | Penalties Cuba Process Help, and  |
|                                   |                                   |
|                                   | record a Special Recovery         |
|                                   | Condition on the payer, to:       |
|                                   |                                   |
|                                   | -   generate a TRIP exception,    |
|                                   |     should their tax refund be    |
|                                   |     intercepted, and              |
|                                   | -   prevent automatic Centrelink  |
|                                   |     deductions. See Tax Refund    |
|                                   |     Intercept Process (TRIP)      |
|                                   |     exceptions Cuba Process Help  |
+-----------------------------------+-----------------------------------+
| 7                                 | Multi case - Collect or Private   |
|                                   | Collect with arrears + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the case is a Multi case -     |
|                                   | Collect or Private Collect with   |
|                                   | arrears, place a hold on the case |
|                                   | with the deceased payee.          |
+-----------------------------------+-----------------------------------+
| 8                                 | Obtain proof of death + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Proof of death must be obtained   |
|                                   | before processing the terminating |
|                                   | event. The following can be used  |
|                                   | as proof of death:                |
|                                   |                                   |
|                                   | -   External Searches Guide -     |
|                                   |     search by the state the       |
|                                   |     customer died in              |
|                                   | -   funeral notice from a         |
|                                   |     newspaper - citing the        |
|                                   |     customer's name, date of      |
|                                   |     birth, date of death or other |
|                                   |     identifying detail            |
|                                   | -   confirmation from the Public  |
|                                   |     Trustee, Coroner's office,    |
|                                   |     hospital or Police            |
|                                   | -   death certificate from the    |
|                                   |     Registry of Births, Deaths    |
|                                   |     and Marriages (BDM), or       |
|                                   | -   confirmation from the         |
|                                   |     Department of Foreign         |
|                                   |     Affairs, if the person died   |
|                                   |     overseas                      |
|                                   |                                   |
|                                   | Note: due to the cost involved,   |
|                                   | obtaining a death certificate     |
|                                   | from BDM should only be pursued   |
|                                   | if no other proof can be          |
|                                   | obtained. BDM take a minimum of 6 |
|                                   | weeks (12 weeks where there is a  |
|                                   | Coroner's enquiry) to produce a   |
|                                   | death certificate.                |
|                                   |                                   |
|                                   | Proof of death should generally   |
|                                   | be in writing. If unable to       |
|                                   | obtain written evidence, or       |
|                                   | unable to obtain any proof of     |
|                                   | death but believe the customer is |
|                                   | deceased, escalate the query, via |
|                                   | a SSO, to a Program Support       |
|                                   | Manager to assist in the decision |
|                                   | making process. See Technical     |
|                                   | support in Child Support.         |
|                                   |                                   |
|                                   | In the event an Indigenous        |
|                                   | customer passed away in a remote  |
|                                   | location, the community manager   |
|                                   | or elder may be able to assist.   |
|                                   | Contact the local Indigenous      |
|                                   | Relations Officer who will draw   |
|                                   | on their networks to progress the |
|                                   | matter.                           |
|                                   |                                   |
|                                   | Centrelink notification alone is  |
|                                   | not sufficient to process a       |
|                                   | customer's death. Additional      |
|                                   | proof must be obtained.           |
|                                   |                                   |
|                                   | Once the proof of death has been  |
|                                   | obtained, see Table 2.            |

Update Cuba and contact the payer

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Process death + Read more ...     |
|                                   |                                   |
|                                   | Once proof of death has been      |
|                                   | obtained process the death in the |
|                                   | Deceased Client window. See       |
|                                   | Deceased customer management Cuba |
|                                   | process help.                     |
|                                   |                                   |
|                                   | Before running Eligibility:       |
|                                   |                                   |
|                                   | -   update care records, if the   |
|                                   |     payer now has care of the     |
|                                   |     children change the care to   |
|                                   |     reflect this                  |
|                                   | -   if there is an A18            |
|                                   |     application, change the last  |
|                                   |     day of secondary education to |
|                                   |     the date/day before the date  |
|                                   |     of death                      |
|                                   | -   correct incomes - check if    |
|                                   |     there are any estimates or    |
|                                   |     provisional incomes and       |
|                                   |     correct if possible. See      |
|                                   |     Income Processing for Child   |
|                                   |     Support customers, Child      |
|                                   |     support estimates review      |
|                                   |     (post 1 July 2010) or Child   |
|                                   |     support estimates - Reconcile |
|                                   |     (from 1 July 2011)            |
|                                   | -   suspend any future LPP (if    |
|                                   |     this is the payer's only      |
|                                   |     child support case. See       |
|                                   |     Penalties Cuba Process Help   |
|                                   | -   if an executor has been       |
|                                   |     identified, ensure they have  |
|                                   |     been added as the customer    |
|                                   |     representative, and           |
|                                   | -   check if a Security Incident  |
|                                   |     Report is appropriate - a     |
|                                   |     security incident report      |
|                                   |     should be completed when the  |
|                                   |     death of the customer was due |
|                                   |     to homicide or suicide. The   |
|                                   |     completed security report     |
|                                   |     should be sent to the team    |
|                                   |     leader                        |
|                                   |                                   |
|                                   | Once these actions are complete,  |
|                                   | run Eligibility.                  |
|                                   |                                   |
|                                   | If Cuba is making unexpected      |
|                                   | changes, do not save eligibility, |
|                                   | see Eligibility window help. If   |
|                                   | unable to correct the Eligibility |
|                                   | changes, seek technical support   |
|                                   | from a Service Support Officer    |
|                                   | (SSO).                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer contact + Read more ...  |
|                                   |                                   |
|                                   | Each call must be approached in a |
|                                   | sensitive and considerate manner. |
|                                   | Each customer contact will differ |
|                                   | so gauge the situation at the     |
|                                   | time of the call. If unsure how   |
|                                   | to proceed, speak to a Team       |
|                                   | Leader or Service Support Officer |
|                                   | (SSO).                            |
|                                   |                                   |
|                                   | Before contacting the payer,      |
|                                   | determine if there has been:      |
|                                   |                                   |
|                                   | -   previous contact regarding    |
|                                   |     the death, or                 |
|                                   | -   no previous contact regarding |
|                                   |     the death                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Previous contact + Read more ...  |
|                                   |                                   |
|                                   | If Child Support has already      |
|                                   | spoken to the payer about the     |
|                                   | death, confirm:                   |
|                                   |                                   |
|                                   | -   the case has ended and the    |
|                                   |     date it ended from            |
|                                   | -   letters will be issued        |
|                                   |     confirming the case has been  |
|                                   |     ended, and                    |
|                                   | -   any amounts owing and the     |
|                                   |     options available for         |
|                                   |     addressing any child support  |
|                                   |     debt or overpayment. For      |
|                                   |     available options, go to Step |
|                                   |     5                             |
|                                   |                                   |
|                                   | Once the conversation has been    |
|                                   | documented, go to Step 5.         |
+-----------------------------------+-----------------------------------+
| 4                                 | No previous contact + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The payer may not be aware of the |
|                                   | payee's death, approach the       |
|                                   | conversation with extreme         |
|                                   | caution.                          |
|                                   |                                   |
|                                   | If Child Support has not had any  |
|                                   | contact with the payer since the  |
|                                   | payee's death allow a reasonable  |
|                                   | amount of time (14 days from date |
|                                   | of death) before contacting the   |
|                                   | payer. Before contacting the      |
|                                   | payer check previous              |
|                                   | communications to see if there    |
|                                   | are any records indicating that   |
|                                   | the customers were in contact     |
|                                   | with each other. Decide if the    |
|                                   | payer is likely to be aware of    |
|                                   | the payee's death.                |
|                                   |                                   |
|                                   | Contact the payer and advise that |
|                                   | Child Support has been notified   |
|                                   | of the death of the payee and     |
|                                   | that their child support case has |
|                                   | ended.                            |
|                                   |                                   |
|                                   | If the payer is upset, check if   |
|                                   | there is someone available the    |
|                                   | payer would like Child Support to |
|                                   | speak with or whether they would  |
|                                   | prefer to continue the call at a  |
|                                   | later date. If the payer wishes   |
|                                   | to continue the call at a later   |
|                                   | date document the conversation in |
|                                   | the Deceased Client window and    |
|                                   | offer to transfer them to the     |
|                                   | Family Relationship Advice Line   |
|                                   | or Social worker and other        |
|                                   | service referrals for child       |
|                                   | support customers. See Products   |
|                                   | and services Cuba process help.   |
|                                   | Once contact has been documented, |
|                                   | go to Step 5.                     |
|                                   |                                   |
|                                   | If the payer wishes to continue   |
|                                   | the call, provide this advice:    |
|                                   |                                   |
|                                   | -   the case has ended, from the  |
|                                   |     day before the date of death  |
|                                   | -   any amounts owing and the     |
|                                   |     options available for         |
|                                   |     addressing any child support  |
|                                   |     debt or overpayment. For      |
|                                   |     available options go to Step  |
|                                   |     5                             |
|                                   | -   the support service available |
|                                   |     and, if appropriate, offer to |
|                                   |     transfer them to the Family   |
|                                   |     Relationship Advice Line or   |
|                                   |     Social worker and other       |
|                                   |     service referrals for child   |
|                                   |     support customers. See also   |
|                                   |     Products and services Cuba    |
|                                   |     process help                  |
+-----------------------------------+-----------------------------------+
| 5                                 | End case + Read more ...          |
|                                   |                                   |
|                                   | Before ending the case and        |
|                                   | issuing letters, consider if the  |
|                                   | case is:                          |
|                                   |                                   |
|                                   | -   Private Collect (no arrears), |
|                                   |     go to Step 6                  |
|                                   | -   Collect or Private Collect    |
|                                   |     with arrears, go to Step 7    |
+-----------------------------------+-----------------------------------+
| 6                                 | Private Collect (no arrears) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the case is private collect    |
|                                   | and there are no arrears or other |
|                                   | outstanding issues, such as       |
|                                   | consolidated revenue debt, excess |
|                                   | cash or overpayments use the DCM  |
|                                   | letters macro to issue a letter   |
|                                   | to the payer advising that the    |
|                                   | case has been finalised. In this  |
|                                   | circumstance, there is no need to |
|                                   | locate or contact the Executor    |
|                                   | for the payee's estate.           |
|                                   |                                   |
|                                   | If the case is Private Collect    |
|                                   | but has a consolidated revenue    |
|                                   | debt, excess cash or overpayment  |
|                                   | go to Collect or Private Collect  |
|                                   | with arrears below.               |
+-----------------------------------+-----------------------------------+
| 7                                 | Collect or Private Collect with   |
|                                   | arrears + Read more ...           |
|                                   |                                   |
|                                   | If there are any arrears, or      |
|                                   | other outstanding issues, check   |
|                                   | the notes in the Deceased Client  |
|                                   | window to determine if the payee  |
|                                   | had a will and if they are:       |
|                                   |                                   |
|                                   | -   testate - died with a valid   |
|                                   |     will, see Table 3             |
|                                   | -   intestate - died without a    |
|                                   |     valid will, see Table 4       |
+-----------------------------------+-----------------------------------+

Collection options for a testate payee

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Collection options for testate    |
|                                   | payees + Read more ...            |
|                                   |                                   |
|                                   | Use the DCM letters macro to      |
|                                   | issue a letter to the payer       |
|                                   | advising that the case has been   |
|                                   | ended and detailing any           |
|                                   | outstanding issues.               |
|                                   |                                   |
|                                   | The payee's Will should name an   |
|                                   | executor.                         |
|                                   |                                   |
|                                   | The Executor must supply a copy   |
|                                   | of the Will, or letters from the  |
|                                   | court naming them as the Executor |
|                                   | or Administrator before any       |
|                                   | information can be released to    |
|                                   | them. Child Support does not      |
|                                   | require Executors to produce a    |
|                                   | grant of probate. Once the        |
|                                   | Executor has supplied proof of    |
|                                   | Executorship record them as the   |
|                                   | payee's representative.           |
|                                   |                                   |
|                                   | Once the Executor has been        |
|                                   | recorded as a representative,     |
|                                   | check if the case has any of the  |
|                                   | following issues:                 |
|                                   |                                   |
|                                   | -   child support owed to payee,  |
|                                   |     go to Step 2                  |
|                                   | -   overpayment, go to Step 3     |
