Technical escalations

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Technical escalation calls        |
|                                   |                                   |
|                                   | Where a customer asks to speak to |
|                                   | a supervisor, Technical Peer      |
|                                   | Support Officers (TPSOs) should   |
|                                   | identify themselves to the        |
|                                   | customer as a technical expert    |
|                                   | who can help them. Technical Peer |
|                                   | Support (TPS) should only handle  |
|                                   | the call if the reason for the    |
|                                   | request is technical and needs a  |
|                                   | more in-depth explanation than    |
|                                   | the Service Officer could give.   |
|                                   |                                   |
|                                   | If the customer is not satisfied  |
|                                   | with a Service Officer            |
|                                   | explanation of a decision, refer  |
|                                   | the Service Officer to Request    |
|                                   | for an explanation or application |
|                                   | for a formal review.              |
|                                   |                                   |
|                                   | Supervisor requests involving     |
|                                   | customer service issues should be |
|                                   | handled by a Team Leader.         |
|                                   |                                   |
|                                   | If the customer wants to lodge a  |
|                                   | complaint, see Managing customer  |
|                                   | complaints and feedback.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Q999 letter requests              |
|                                   |                                   |
|                                   | For information about Q999 letter |
|                                   | requests, see Assessing           |
|                                   | suitability to issue a Q999       |
|                                   | letter.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Finalised activity generates      |
|                                   | incorrect arrears - stop payment  |
|                                   |                                   |
|                                   | Requests to prevent incorrect     |
|                                   | arrears must be referred to the   |
|                                   | ICT Service Desk before close of  |
|                                   | business.                         |
|                                   |                                   |
|                                   | Service Officers are to escalate  |
|                                   | this directly to their local      |
|                                   | Service Support Officer (SSO) or  |
|                                   | Team Leader for immediate action. |
+-----------------------------------+-----------------------------------+
