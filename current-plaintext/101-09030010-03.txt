|                                   |     Online Services (not relevant |
|                                   |     for BasicsCard). DaC usage    |
|                                   |     report will show the Log-On   |
|                                   |     Id of users and usage. Check  |
|                                   |     passwords/log-on ID are not   |
|                                   |     shared                        |
|                                   |                                   |
|                                   | If the entity is not correct or   |
|                                   | the ABN has been cancelled:       |
|                                   |                                   |
|                                   | -   Issue a correction of         |
|                                   |     registration letter or an     |

|                                   |     is confirmed by the business  |
|                                   |     or Program Assurance team     |
|                                   |     that a termination/withdrawal |
|                                   |     of approval letter should be  |
|                                   |     issued                        |

|                                   |     organisation finder. Refer to |

|                                   |     and confirmation service      |

|                                   |     retaining the same CRN, see   |

|                                   |     Management deductions, EVoR   |

|                                   | example, Name (Finance Officer).  |
|                                   |                                   |

|                                   | with. The Authorised Officer can  |

|                                   | Format > Auto Fit column width.   |
|                                   | Then PDF the whole excel          |
|                                   | spreadsheet.                      |
|                                   |                                   |
|                                   | The usage report can be used to   |
|                                   | check consent for EVoR (if there  |
|                                   | is CCeS access)                   |
|                                   |                                   |
|                                   | Request a recent consent record   |
|                                   | from the business, even if there  |
|                                   | is no usage by the business in    |
|                                   | the past 12 months. The focus of  |
|                                   | these checks for assurance        |
|                                   | reviews is to confirm that the    |
|                                   | business:                         |
|                                   |                                   |
|                                   | -   has an effective method for   |
|                                   |     collecting and storing        |
|                                   |     consent, and                  |
|                                   | -   is using the appropriate      |
|                                   |     consent wording               |
|                                   |                                   |
|                                   | If there is a complaint or tip    |
|                                   | off, or data identifies a         |
|                                   | potential issue of                |
|                                   | non-compliance, check 25 consent  |
|                                   | forms as standard or what the     |
|                                   | usage report returns. If the      |
|                                   | usage report returns usage fewer  |
|                                   | than 25, DOC this for quality     |
|                                   | assurance. For low usage, see     |
|                                   | Table 2                           |
|                                   |                                   |
|                                   | If the consent forms have been    |
|                                   | provided due to a complaint or    |
|                                   | tip-off, and were all signed      |
|                                   | before the March 2015 transition, |
|                                   | make sure the current version of  |
|                                   | the consent meets requirements    |
|                                   |                                   |
|                                   | The usage report can also be used |
|                                   | for Centrepay to identify User    |
|                                   | Logon IDs.                        |
|                                   |                                   |

|                                   | report is Australian Eastern      |
|                                   | Standard Time (AEST). For states  |
|                                   | not in this time zone, subtract   |
|                                   | the time relevant to the          |
|                                   | state/time zone.                  |
|                                   |                                   |

|                                   | spreadsheet before forwarding the |
|                                   | usage report to the business.     |
|                                   |                                   |
|                                   | Old consent                       |

| 6                                 | Non-compliance excluding a        |

|                                   | questions sets or other           |
|                                   | documentation provided. Use the   |
|                                   | referral template on the shared   |
|                                   | drive.                            |
