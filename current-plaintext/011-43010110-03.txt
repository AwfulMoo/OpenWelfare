This page contains information for Service Officers about the process to
follow when a Medicare patient claim account includes a charge for Goods
and Services Tax (GST).

On this Page:

Account or receipt presented at service centre

Medicare account or receipt with GST charged is presented at service centre

Table 1: This table describes the process to follow when a claimant
presents their Medicare claim account or receipt at a service centre for
a Medicare benefit and they have been charged GST.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Contact health professional +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Can contact be made with the      |
|                                   | health professional?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Process Medicare patient claim +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Process the Medicare patient      |
|                                   | claim with the confirmed service  |
|                                   | details and amended charge        |
|                                   | (deducting the GST amount).       |
|                                   |                                   |
|                                   | If the health professional cannot |
|                                   | confirm or refuses to update the  |
|                                   | charged amount, either pay the    |
|                                   | account or receipt minus the GST  |
|                                   | amount, or go to Step 3.          |
|                                   |                                   |
|                                   | Note: Service Officers must       |
|                                   | complete a VG4 or sticky note and |
|                                   | attach to claim.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Return Medicare patient claim to  |
|                                   | claimant + Read more ...          |
|                                   |                                   |
|                                   | Return the Medicare account or    |
|                                   | receipt to the claimant to follow |
|                                   | up with the health professional   |
|                                   | as Medicare payments and services |
|                                   | are GST free.                     |
+-----------------------------------+-----------------------------------+

Account or receipt lodged by other method

Medicare account or receipt with GST charged is lodged by Medicare drop box, online account, Express Plus Medicare mobile app, phone or mail

Table 2: This table describes the process to follow when a Medicare
account or receipt that includes a GST charge is lodged by Medicare drop
box, online account, Express Plus Medicare mobile app, phone or mail.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Process Medicare patient claim +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Process the Medicare patient      |
|                                   | claim with the charge minus the   |
|                                   | GST amount and pay the Medicare   |
|                                   | benefit.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Copy and endorse Medicare account |
|                                   | or receipt + Read more ...        |
|                                   |                                   |
|                                   | Photocopy or print the Medicare   |
|                                   | account or receipt and endorse    |
|                                   | the copy with 'Medicare benefit   |
|                                   | paid'.                            |
|                                   |                                   |
|                                   | Endorsing the account avoids any  |
|                                   | possible confusion over whether   |
|                                   | the benefit has been paid once    |
|                                   | the GST component has been        |
|                                   | refunded to the claimant.         |
+-----------------------------------+-----------------------------------+
| 3                                 | Return copy of Medicare account   |
|                                   | or receipt by post + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Return by post to the claimant:   |
|                                   |                                   |
|                                   | -   the copy of the account or    |
|                                   |     receipt, and                  |
|                                   | -   a letter from Services        |
|                                   |     Australia advising Medicare   |
|                                   |     benefits and services are GST |
|                                   |     free                          |
|                                   |                                   |
|                                   | The Resources page contains link  |
|                                   | to the letter Your claim for      |
|                                   | Medicare benefits has been paid - |
|                                   | Medicare services are GST free.   |
+-----------------------------------+-----------------------------------+
