Care changes notified up to 14 May 2019 for WA Ex-nuptial child support
cases must be managed under the legislation that was in place prior to
the amendments effective 1 July 2018, see For WA Ex-nuptial cases -
Notification of change in child support care (post 1 July 2010).

|                                   | not occurring in line with a      |

| 1                                 | Determine likely date of event +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The term 'likely' means           |
|                                   | information gathered from the     |
|                                   | initial customer only. Identify   |
|                                   | the date of event by asking the   |
|                                   | customer what circumstances led   |
|                                   | to the change in care. When       |
|                                   | identifying the date of event,    |
|                                   | consider whether the care that is |
|                                   | actually occurring for the child  |
|                                   | has changed.                      |
|                                   |                                   |
|                                   | Do not pend a care change if a    |
|                                   | determination relating to this    |
|                                   | care event has already been made  |
|                                   | (original care decision,          |
|                                   | objection or appeal). Provide     |
|                                   | customers with options to seek    |
|                                   | review of the decision(s).        |
|                                   |                                   |
|                                   | The date of event is either:      |
|                                   |                                   |
|                                   | -   the date both customers       |
|                                   |     confirm the actual care       |
|                                   |     change occurred               |
|                                   | -   a date determined by Child    |
|                                   |     Support based on the          |
|                                   |     information and evidence      |
|                                   |     provided by both customers    |
|                                   | -   the date customers made a new |
|                                   |     care arrangement for example  |
|                                   |     the made on date for a new    |
|                                   |     court order                   |
|                                   |                                   |
|                                   | Note: the date of event used to   |
|                                   | record the care change will be    |
|                                   | tentative until the other         |
|                                   | customer has been contacted.      |
|                                   |                                   |
|                                   | Questions that may assist in      |
|                                   | determining the likely date of    |
|                                   | event include:                    |
|                                   |                                   |
|                                   | -   which child does the change   |
|                                   |     relate to                     |
|                                   | -   what date did the             |
|                                   |     care/pattern of care change   |
|                                   | -   what caused the change in     |
|                                   |     care                          |
|                                   | -   was the care change discussed |
|                                   |     with the other parent         |
|                                   | -   is the change intended to be  |
|                                   |     permanent                     |
|                                   | -   if the care change is         |
|                                   |     temporary, what is the period |
|                                   |     of the change                 |
|                                   |                                   |
|                                   | If a one-off care event is missed |
|                                   | that is likely to be made up/the  |
|                                   | care is likely to resume to the   |
|                                   | standard pattern, this means a    |
|                                   | care change may not have          |
|                                   | occurred.                         |
|                                   |                                   |
|                                   | If unable to determine the date   |
|                                   | of event, see Step 2 in the       |
|                                   | Investigate change in care table  |
|                                   | in Investigation of change in     |
|                                   | child support care (post 1 July   |
|                                   | 2010).                            |

| 2                                 | Determine likely care period +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When a customer notifies Child    |
|                                   | Support of a change in care,      |
|                                   | determine the likely care period. |
|                                   | A care period is generally the 12 |
|                                   | month period starting from the    |
|                                   | date of event.                    |

| 3                                 | Calculate care period + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | A care period is generally the 12 |
|                                   | month period starting from the    |
|                                   | date of event. In some            |
|                                   | circumstances, a shorter or       |
|                                   | longer care period may be         |
|                                   | appropriate, however judge each   |
|                                   | case on its own merits. If        |
|                                   | determining a care period other   |
|                                   | than 12 months, seek technical    |
|                                   | support. See Technical support in |
|                                   | Child Support.                    |
|                                   |                                   |
|                                   | Care periods longer than 12       |
|                                   | months                            |
|                                   |                                   |
|                                   | A care period of more than 12     |
|                                   | months may be appropriate where   |
|                                   | the care arrangement follows a    |
|                                   | cycle over a period greater than  |
|                                   | 12 months, for example every 2    |
|                                   | years.                            |
|                                   |                                   |
|                                   | See Resources page for examples   |
|                                   | of care periods longer than 12    |
|                                   | months.                           |
|                                   |                                   |
|                                   | Care periods shorter than 12      |
|                                   | months                            |
|                                   |                                   |
|                                   | A care period of less than 12     |
|                                   | months might be appropriate where |
|                                   | the pattern of care will          |
|                                   | gradually change over definable   |
|                                   | periods in a specific and         |
|                                   | measurable way, for example one   |
|                                   | parent is to increase their care  |
|                                   | by one day per week every 3       |
|                                   | months over the upcoming period.  |
|                                   |                                   |
|                                   | A shorter care period may also be |
|                                   | appropriate where a carer has     |
|                                   | unexpectedly gained 100% care of  |
|                                   | a child for a temporary period    |
|                                   | (generally 4 weeks minimum). In   |
|                                   | these situations, the Registrar   |
|                                   | will determine the care over a    |
|                                   | short care period related to the  |
|                                   | unexpected circumstance and make  |
|                                   | a one-off block of 100% care      |
|                                   | determination.                    |
|                                   |                                   |
|                                   | See Resources page for examples   |
|                                   | of care periods shorter than 12   |
|                                   | months, and References page in    |
|                                   | Change in child support care      |
|                                   | levels (post 1 July 2010) for     |
|                                   | further information.              |
|                                   |                                   |
|                                   | Note: tell the customer Child     |
|                                   | Support will not give effect to   |
|                                   | the future events until they      |
|                                   | notify the change has occurred.   |
|                                   | If neither customer advises of    |
|                                   | the subsequent change, the        |
|                                   | existing percentage of care will  |
|                                   | continue.                         |

| 4                                 | Determine likely percentage of    |
|                                   | care + Read more ...              |
|                                   |                                   |
|                                   | The term 'likely' means           |
|                                   | information gathered from the     |
|                                   | initial customer only. Care is    |
|                                   | expressed as a percentage of care |
|                                   | (generally based on a number of   |
|                                   | nights) that a customer is likely |
|                                   | to have during the care period as |
|                                   | shown in the care percentage      |
|                                   | table.                            |
|                                   |                                   |
|                                   | See the Resources page for the    |
|                                   | Care percentage table.            |

| 5                                 | Establish pattern of care + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Ask the customer to provide the   |
|                                   | pattern of care based on what     |
|                                   | they actually expect to occur     |
|                                   | during the 12 month care period.  |
|                                   | In most cases, care is determined |
|                                   | by the actual care each parent    |
|                                   | has. Where a pattern has been     |
|                                   | established in the past and the   |
|                                   | Service Officer is satisfied that |
|                                   | it will continue in the future,   |
|                                   | use that pattern to determine the |
|                                   | likely percentage of care for the |
|                                   | care period.                      |
|                                   |                                   |
|                                   | If the customer:                  |
|                                   |                                   |
|                                   | -   requests that care be         |
|                                   |     calculated in a form other    |
|                                   |     than nights, see Step 2 in    |
|                                   |     the Calculate and determine   |
|                                   |     care percentage table in      |
|                                   |     Child support care outcomes   |
|                                   |     and decision making (post 1   |
|                                   |     July 2010)                    |
|                                   | -   has unexpectedly gained 100%  |
|                                   |     care of a child for a         |
|                                   |     temporary period (generally 4 |
|                                   |     weeks minimum), see Step 3 in |
|                                   |     the Calculate and determine   |
|                                   |     care percentage table in      |
|                                   |     Child support care outcomes   |
|                                   |     and decision making (post 1   |
|                                   |     July 2010)                    |
|                                   | -   notifies Child Support within |
|                                   |     a reasonable time and informs |
|                                   |     that the level of care for    |
|                                   |     the other parent (who has at  |
|                                   |     least regular care) has       |
|                                   |     fallen below 14% despite the  |
|                                   |     child being made available to |
|                                   |     them, see Step 7 in the       |
|                                   |     Calculate and determine care  |
|                                   |     percentage table in Child     |
|                                   |     support care outcomes and     |
|                                   |     decision making (post 1 July  |
|                                   |     2010)                         |
|                                   |                                   |
|                                   | If a customer wants to estimate   |
|                                   | future or possible care changes   |
|                                   | themselves, refer them to the     |
|                                   | Care Estimator available on       |
|                                   | servicesaustralia.gov.au.         |
|                                   |                                   |
|                                   | Questions that may assist in      |
|                                   | determining the likely percentage |
|                                   | of care include:                  |
|                                   |                                   |
|                                   | -   have you and the other parent |
|                                   |     established a pattern of care |
|                                   |     for the child                 |
|                                   | -   how many nights do you and    |
|                                   |     the other parent expect to    |
|                                   |     care for your child over the  |
|                                   |     12 month care period from the |
|                                   |     date the care changed         |
|                                   | -   how did you calculate the     |
|                                   |     expected care (for example,   |
|                                   |     is there a pattern of care)   |
|                                   | -   do you or the other parent    |
|                                   |     provide extensive care during |
|                                   |     the day but little or no      |
|                                   |     overnight care? See Step 2 in |
