+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Code customer's absence + Read    |

|                                   | Australia                         |

|                                   | Record the overseas telephone     |
|                                   | number and/or an email address if |
|                                   | available                         |

|                                   | -   Code date of departure in the |
|                                   |     Event date: field             |
|                                   | -   Code the destination country  |
|                                   |     in the Country: field. If the |
|                                   |     customer is travelling to     |
|                                   |     multiple destinations or the  |
|                                   |     destination country is        |
|                                   |     unknown, or they are          |
|                                   |     travelling outside Australia  |
|                                   |     on a cruise, code 'XT'        |
