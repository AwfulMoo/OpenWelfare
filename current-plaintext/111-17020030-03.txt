Storing and handling compensation recovery future preclusion files

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Compensation recovery future      |
|                                   | preclusion files are no longer    |
|                                   | created + Read more ...           |
|                                   |                                   |
|                                   | All documents relating to         |
|                                   | compensation lump sum future      |
|                                   | preclusion cases are scanned to   |
|                                   | the customer's record.            |
|                                   |                                   |
|                                   | For information on scanning, see  |
|                                   | Scanning Centrelink documents     |
|                                   | using an MFD.                     |

| 2                                 | Storing and handling a future     |
|                                   | preclusion file + Read more ...   |
|                                   |                                   |
|                                   | File requests                     |
|                                   |                                   |
|                                   | Request the filefrom the local    |
|                                   | Records Management Unit (RMU) if  |
|                                   | needed for:                       |
|                                   |                                   |
|                                   | -   review of a decision          |
|                                   | -   Freedom of Information (FOI)  |
|                                   |     request                       |
|                                   |                                   |
|                                   | Use the Records Management        |
|                                   | service request webform.          |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the form.                 |
|                                   |                                   |
|                                   | File transfer                     |
|                                   |                                   |
|                                   | Notify the RMU via the Records    |
|                                   | Management service request        |
|                                   | webform. Responsibility for the   |
|                                   | file will remain with the         |
|                                   | original assignee.                |
