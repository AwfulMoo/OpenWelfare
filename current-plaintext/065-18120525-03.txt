|                                   | the sanction. For example, an     |
|                                   | exception is when a care          |
|                                   | recipient who has already entered |
|                                   | into a care agreement before the  |
|                                   | sanction period but not yet       |
|                                   | entered into care.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Identify the relevant payment     |
|                                   | system + Read more ...            |
|                                   |                                   |
|                                   | If the sanction applies to:       |
|                                   |                                   |
|                                   | -   a home care and flexible care |
|                                   |     services, see Table 3         |
|                                   | -   a residential care service,   |
|                                   |     see Table 4                   |

|                                   | Select Save                       |

| 3                                 | Monitor monthly claims + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a sanction is in place,        |
|                                   | monitor the status of monthly     |
|                                   | claims for the duration of the    |
|                                   | sanction. This includes           |
|                                   | identifying (any) new care        |
|                                   | recipients who entered into care  |
|                                   | during the sanction period. This  |
|                                   | must be done before the claim     |
|                                   | being approved.                   |

|                                   | Read more ...                     |
|                                   |                                   |

|                                   |     Service field                 |

|                                   |                                   |
|                                   | Identify the existing care        |
|                                   | recipients from new care          |
|                                   | recipients by cross checking the  |
|                                   | dates from the Active From column |

|                                   |                                   |
|                                   | Care recipients whose Active From |
|                                   | date:                             |
|                                   |                                   |
|                                   | -   falls within the sanction     |
|                                   |     period, including the         |
|                                   |     commencement date, are        |
|                                   |     considered to be new care     |
|                                   |     recipients, go to Step 5      |
|                                   | -   is prior to the commencement  |
|                                   |     of the sanction period are    |
|                                   |     considered to be existing     |
|                                   |     care recipients, go to Step 6 |
+-----------------------------------+-----------------------------------+
| 5                                 | New care recipients entered into  |
|                                   | care + Read more ...              |
|                                   |                                   |
|                                   | -   Contact the Aged Care Quality |
|                                   |     and Safety Commission via     |
|                                   |     email or phone to determine   |
|                                   |     why new care recipients were  |
|                                   |     entered during the sanction   |
|                                   |     period (as this means         |
|                                   |     sanction requirements may not |
|                                   |     been met)                     |
|                                   | -   The Aged Care Quality and     |
|                                   |     Safety Commission will give   |
|                                   |     instructions if any other     |
|                                   |     action is required            |
|                                   | -   Once actioned, go to Step 6   |
|                                   |     to approve claim              |
+-----------------------------------+-----------------------------------+
| 6                                 | Care recipients existing + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If all the care recipients are    |
|                                   | existing, the claim can be        |
|                                   | approved.                         |
|                                   |                                   |
|                                   | Advise the Aged Care Payments     |
|                                   | Team within the Aged Care and     |
|                                   | Allied Health Branch that claim   |
|                                   | can be approved as sanction       |
|                                   | requirements have been met.       |
