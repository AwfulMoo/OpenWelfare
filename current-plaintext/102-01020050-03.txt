PQPD screen

Predictor Details screen information

This table describes the PQPD screen.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Predictor Details (PQPD) screen   |
|                                   |                                   |
|                                   | The PQPD screen shows the details |
|                                   | for the selected Predictor for a  |
|                                   | Profile.                          |
|                                   |                                   |
|                                   | The details available are:        |
|                                   |                                   |
|                                   | -   Predictor Description         |
|                                   | -   (Predictor) Start Date        |
|                                   | -   (Predictor) End Date          |
|                                   |                                   |
|                                   | This screen is for view purposes  |
|                                   | only and no action can be taken.  |
|                                   | It displays details of a selected |
|                                   | Predictor for a Particular        |
|                                   | Profile.                          |
|                                   |                                   |
|                                   | Page scrolling by [F7]/[F8] or    |
|                                   | over striking the page number is  |
|                                   | enabled to view all the           |
|                                   | Predictors on this screen.        |
|                                   |                                   |
|                                   | The PQPD screen can only be       |
|                                   | accessed by selecting ‘S’ against |
|                                   | Predictor from the Profile        |
|                                   | Details (PQD) screen. Note: this  |
|                                   | screen cannot be accessed by      |
|                                   | entering PQPD in the Nxt: field.  |
+-----------------------------------+-----------------------------------+
