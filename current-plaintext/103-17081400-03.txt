Selecting a suitable payment method

Table 3: This table describes the process for selecting a suitable
payment method for the deduction during conversation with the third
party.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Discussing payment methods and    |
|                                   | Income Management terms + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Uncontracted organisations can be |
|                                   | paid in a variety of ways.        |
|                                   |                                   |
|                                   | Organisations should be asked if  |
|                                   | they offer Bpay in the first      |
|                                   | instance                          |
|                                   |                                   |
|                                   | Organisations that offer travel   |
|                                   | where the customer may require    |
|                                   | immediate use of the service must |
|                                   | be paid using Credit Card over    |
|                                   | the phone. This ensures the       |
|                                   | payment is immediate and the      |
|                                   | customer can access the service.  |
|                                   | If the organisation is a travel   |
|                                   | provider, go to Step 4.           |
|                                   |                                   |
|                                   | Is the organisation registered    |
|                                   | for Bpay?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Bpay + Read more ...              |
|                                   |                                   |
|                                   | Confirm the TPOs:                 |
|                                   |                                   |
|                                   | -   biller code for the purpose   |
|                                   |     of receiving Bpay payments.   |
|                                   |     Bpay payments can be          |
|                                   |     identified on the TPOs bank   |
|                                   |     statement as well as any      |
|                                   |     information Bpay provide the  |
|                                   |     TPO directly                  |
|                                   | -   the length/characteristics of |
|                                   |     the TPOs Bpay reference       |
|                                   |     number                        |
|                                   |                                   |
|                                   | See Step 1 in Setting up the new  |
|                                   | TPO records to receive deductions |
|                                   | table.                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Other payment methods + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Services Australia can make       |
|                                   | payment into a nominated bank     |
|                                   | account for the purpose of        |
|                                   | payment for goods or services.    |
|                                   |                                   |
|                                   | Such deposits are known as        |
|                                   | 'Scheduled Transfer' payments.    |
|                                   | These payments are identifiable   |
|                                   | on the organisations bank account |
|                                   | and can include any reference the |
|                                   | organisation nominates to assist  |
|                                   | with identification.              |
|                                   |                                   |
|                                   | If the business requires access   |
|                                   | to the money immediately, or is   |
|                                   | not able to accept a Scheduled    |
|                                   | Transfer payment, a credit card   |
|                                   | payment over the phone can be     |
|                                   | made to the business.             |
|                                   |                                   |
|                                   | If proceeding with Scheduled      |
|                                   | Transfer payment,                 |
|                                   |                                   |
|                                   | Confirm the following details     |
|                                   | with the organisation:            |
|                                   |                                   |
|                                   | -   Account Name                  |
|                                   | -   BSB                           |
|                                   | -   Account number                |
|                                   |                                   |
|                                   | Reference number (to be included  |
|                                   | for identification on             |
|                                   | organisation bank statement) this |
|                                   | might be a name, number (not the  |
|                                   | Customer Reference Number (CRN)), |
|                                   | address or unique code.           |
|                                   |                                   |
|                                   | Is Scheduled Transfer being used  |
|                                   | to pay the business?              |
|                                   |                                   |
|                                   | -   Yes, see Step 1 Setting up    |
|                                   |     the new TPO record to receive |
|                                   |     deductions table              |
|                                   | -   No, a credit card payment is  |
|                                   |     required, go to Step 4        |
+-----------------------------------+-----------------------------------+
| 4                                 | Making a credit card payment +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Where an organisation requires a  |
|                                   | credit card payment, a payment    |
|                                   | can be made over the phone        |
|                                   | directly for the amount requested |
|                                   | by the customer.                  |
|                                   |                                   |
|                                   | Advise the business they will be  |
|                                   | contacted to arrange the payment. |
|                                   | If the customer presents at the   |
|                                   | business to access goods or       |
|                                   | services, the business can call   |
|                                   | the processing team to arrange    |
|                                   | payment. See contact details for  |
|                                   | the processing team in Income     |
|                                   | Management Payment Contact        |
|                                   | (IMPACT) team.                    |
|                                   |                                   |
|                                   | See Step 1 Setting up the new TPO |
|                                   | record to receive deductions      |
|                                   | table.                            |
