Create file

Table 1: This table describes the steps for creating the initial AAT
file.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record review application in Cuba |
|                                   | (Service Officer) + Read more ... |
|                                   |                                   |
|                                   | Create a notepad with a subject   |
|                                   | heading ‘File location’ on the    |
|                                   | SSAT Maintenance window. Document |
|                                   | the location of the file and if   |
|                                   | applicable, the RACS level (1, 2  |
|                                   | or 3) of the case.                |
|                                   |                                   |
|                                   | Documentation to be saved in Cuba |
|                                   | (SSAT Maintenance window)         |
|                                   |                                   |
|                                   | -   ‘Request for applicant        |
|                                   |     details’                      |
|                                   | -   ‘Request for statement’, and  |
|                                   | -   Response emails               |
|                                   |                                   |
|                                   | Note: if the matter is a care     |
|                                   | appeal and cannot be loaded into  |
|                                   | the SSAT Maintenance window due   |
|                                   | to a known system error, a        |
|                                   | stacking document should be       |
|                                   | started in the communications     |
|                                   | window. All future notes relating |
|                                   | to this appeal, which would       |
|                                   | normally be added to the SSAT     |
|                                   | Maintenance window, are to be     |
|                                   | added to this notepad.            |
|                                   |                                   |
|                                   | -   Type: Electronic In           |
|                                   | -   Category: SSAT                |
|                                   | -   Issue: OTHER                  |
|                                   | -   Reason: Objection             |
|                                   | -   With: EXTERNAL AGENCY         |
+-----------------------------------+-----------------------------------+
| 2                                 | Create initial AAT file (Service  |
|                                   | Officer) + Read more ...          |
|                                   |                                   |
|                                   | To assist with creating and       |
|                                   | compiling the AAT file, read the  |
|                                   | objection decision; this will     |
|                                   | allow a full understanding of the |
|                                   | documents required. The initial   |
|                                   | file is the unedited compilation  |
|                                   | of all documents potentially      |
|                                   | relevant to the review. The       |
|                                   | initial file forms the basis of   |
|                                   | the final AAT file.               |
|                                   |                                   |
|                                   | Note: the final AAT file must be  |
|                                   | sent to the AAT within 28 days of |
|                                   | the date of the 'Request for      |
|                                   | statement'.                       |
|                                   |                                   |
|                                   | Relevant documents to be saved in |
|                                   | the electronic file include:      |
|                                   |                                   |
|                                   | -   ‘Record of Application’       |
|                                   | -   All evidence and attachments  |
|                                   |     submitted by the applicant    |
|                                   |     with their record of          |
|                                   |     application                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Retrieve paper and electronic     |
|                                   | records + Read more ...           |
|                                   |                                   |
|                                   | Commence gathering documents and  |
|                                   | files within the first week of    |
|                                   | being assigned the review. Allow  |
|                                   | time for documents and files to   |
|                                   | be sent by courier if necessary.  |
|                                   |                                   |
|                                   | Notes:                            |
|                                   |                                   |
|                                   | -   It may be necessary to        |
|                                   |     retrieve paper files from     |
|                                   |     batch filing or Archives in   |
|                                   |     the process of building the   |
|                                   |     file. Paper files will need   |
|                                   |     to be scanned and saved as a  |
|                                   |     PDF into the electronic file. |
|                                   | -   Complete the file within 14   |
|                                   |     days of the AAT's 'Request    |
|                                   |     for statement' email.         |
|                                   |                                   |
|                                   | Determine if the documents and    |
|                                   | files relevant to the appeal are  |
|                                   | paper or electronic. Retrieve any |
|                                   | paper files that have been        |
|                                   | created for example, Objection or |
|                                   | Change of Assessment (COA) file.  |
|                                   |                                   |
|                                   | Save all documents and            |
|                                   | sub-screens relevant to the:      |
|                                   |                                   |
|                                   | -   initial decision              |
|                                   | -   objection decision, and       |
|                                   | -   post decision contact and     |
|                                   |     developments                  |
|                                   |                                   |
|                                   | Note: where the question of       |
|                                   | whether contact was made is       |
|                                   | relevant, include a copy of the   |
|                                   | Customer Receipt Number window in |
|                                   | the file. Edit the actual receipt |
|                                   | numbers to protect the customer's |
|                                   | privacy.                          |
|                                   |                                   |
|                                   | Documents to be included in the   |
|                                   | AAT file:                         |
|                                   |                                   |
|                                   | -   Any documents specifically    |
|                                   |     referred to in the objection  |
|                                   |     statement of reasons report   |
|                                   | -   Any information relied upon   |
|                                   |     in the decision must be       |
|                                   |     included. If information was  |
|                                   |     provided in a phone call or   |
|                                   |     CoA conference include the    |
|                                   |     record (Cuba Notepad) of that |
|                                   |     contact                       |
|                                   | -   All materials received from   |
|                                   |     the customers, in relation to |
|                                   |     the objection decision, must  |
|                                   |     be included. While the        |
|                                   |     initial decision maker or     |
|                                   |     objection officer may not     |
|                                   |     have considered the materials |
|                                   |     relevant, they may still be   |
|                                   |     relevant in the AAT's view    |
|                                   | -   Any relevant records of       |
|                                   |     contact, before, during and   |
|                                   |     after the decision was made,  |
|                                   |     and                           |
|                                   | -   Any developments regarding    |
|                                   |     the issue the AAT is          |
|                                   |     considering                   |
|                                   |                                   |
|                                   | Note: do not include all contact  |
|                                   | with the customer, only           |
|                                   | contact/developments relevant to  |
|                                   | the issue which the AAT is        |
|                                   | considering.                      |
|                                   |                                   |
|                                   | Check the Feedback and Review     |
|                                   | activity (Complaints) and the CSA |
|                                   | online windows for any relevant   |
|                                   | documents.                        |
|                                   |                                   |
|                                   | See the relevant AAT document     |
|                                   | checklists for the specific       |
|                                   | documents required for each       |
|                                   | review type.                      |
|                                   |                                   |
|                                   | Note: the AAT document checklists |
|                                   | are the minimum required.         |
+-----------------------------------+-----------------------------------+
| 4                                 | Review correspondence to and from |
|                                   | AAT + Read more ...               |
|                                   |                                   |
|                                   | Copy emails to or from the AAT    |
|                                   | containing information about the  |
|                                   | review to the SSAT Cuba notepad,  |
|                                   | noting that they are an exact     |
|                                   | copy of the original. These       |
|                                   | documents are not included in the |
|                                   | document bundle sent to customers |
|                                   | and the AAT as they are for       |
|                                   | administrative/internal purposes  |
|                                   | and do not need to be attached to |
|                                   | the file.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Family and domestic violence +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Check the case for information    |
|                                   | regarding family and domestic     |
|                                   | violence, including:              |
|                                   |                                   |
|                                   | -   Family Violence / Sensitive   |
|                                   |     Issues indicator              |
|                                   | -   information (including Cuba   |
|                                   |     notes) or communication       |
|                                   |     indicating a history of       |
|                                   |     violence between the parents  |
|                                   | -   current family violence       |
|                                   |     orders                        |
|                                   | -   intervention or restraining   |
|                                   |     orders against one or both    |
|                                   |     parents, and/or               |
|                                   | -   either parent expressing      |
|                                   |     concerns for the safety of    |
|                                   |     themselves or their children  |
|                                   |                                   |
|                                   | See Family and domestic violence. |
|                                   |                                   |
|                                   | Note: Information relating to     |
|                                   | family violence should be         |
|                                   | escalated to a Service Support    |
|                                   | Officer (SSO) to assess if        |
|                                   | exception handling is necessary.  |
+-----------------------------------+-----------------------------------+
| 6                                 | Review objection file + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Read the Statement of Reasons     |
|                                   | report to identify the decision   |
|                                   | under review and the information  |
|                                   | used or considered by the         |
|                                   | decision maker. Save (or scan if  |
|                                   | paper files are being reviewed)   |
|                                   | the relevant documents to the     |
|                                   | initial electronic file.          |
|                                   |                                   |
|                                   | If:                               |
|                                   |                                   |
|                                   | -   a paper file was not created, |
|                                   |     or                            |
|                                   | -   relevant documents are not on |
|                                   |     the objection file or were    |
|                                   |     received after the objection  |
|                                   |     decision                      |
|                                   |                                   |
|                                   | check the View Correspondence     |
|                                   | window and/or request the         |
|                                   | documents from batch filing. See  |
|                                   | Correspondence received in Child  |
|                                   | Support for further information   |
|                                   | on how to gather paper records.   |
|                                   |                                   |
|                                   | When emailing Batch Filing with a |
|                                   | request for documents, ensure the |
|                                   | subject line of the email clearly |
|                                   | details the file is required for  |
|                                   | the AAT review process, for       |
|                                   | example, 'File required for AAT - |
|                                   | URGENT'. Use the AAT File         |
|                                   | Location email template for the   |
|                                   | body of the email. If the Service |
|                                   | Officer receives advice that a    |
|                                   | requested document cannot be      |
|                                   | located escalate this to the SSO. |
|                                   | Save a copy of the document       |
|                                   | request on the AAT file.          |
|                                   |                                   |
|                                   | If documents are stored in        |
|                                   | Archives, send a request to the   |
|                                   | Records Management Unit (RMU)     |
|                                   | using the AAT File Location email |
|                                   | template.                         |
|                                   |                                   |
|                                   | Review COA file                   |
|                                   |                                   |
|                                   | If the review relates to a COA    |
|                                   | decision, review the notice of    |
|                                   | decision and the information used |
|                                   | or considered by the Decision     |
|                                   | Maker. Save the notice of         |
|                                   | decision and any relevant         |
|                                   | documents not already part of the |
|                                   | initial file and add to the       |
|                                   | electronic file.                  |
+-----------------------------------+-----------------------------------+
| 7                                 | Review Cuba records + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Use the relevant AAT document     |
|                                   | guide to ensure that all relevant |
|                                   | information is identified.        |
|                                   |                                   |
|                                   | Review Cuba records to identify   |
|                                   | any other relevant information.   |
|                                   | Print the following and add to    |
|                                   | the initial file:                 |
|                                   |                                   |
|                                   | -   relevant documents from the   |
|                                   |     View Correspondence window    |
|                                   | -   records of contact attempts   |
|                                   |     relevant to the review, and   |
|                                   | -   letters to the customers      |
|                                   |     regarding the decision under  |
|                                   |     review, and                   |
|                                   | -   Cuba screens relevant to the  |
|                                   |     review                        |
|                                   |                                   |
|                                   | Note: where a document has been   |
|                                   | receipted on Cuba, save a copy of |
|                                   | the original document to the AAT  |
|                                   | file.                             |
|                                   |                                   |
|                                   | Risk Point: ‘Suspicious Matter    |
|                                   | Reports’ (SMRs) from AUSTRAC must |
|                                   | never be disclosed. This means    |
|                                   | they must not be included when    |
|                                   | making a submission – they cannot |
|                                   | be disclosed to any court or      |
|                                   | tribunal and are not admissible   |
|                                   | as evidence.                      |
|                                   |                                   |
|                                   | You must review AUSTRAC search    |
|                                   | results to ensure they do not     |
|                                   | contain an SMR/SUSTR – these must |
|                                   | not be recorded in the system. If |
|                                   | a record needs to be removed,     |
|                                   | take immediate action by sending  |
|                                   | an email the National.ICS         |
|                                   | mailbox.                          |
|                                   |                                   |
|                                   | Full details regarding AUSTRAC    |
|                                   | disclosure of information can be  |
|                                   | found on the Resources page.      |
+-----------------------------------+-----------------------------------+
| 8                                 | Organise file + Read more ...     |
|                                   |                                   |
|                                   | Consider the ‘Preparing documents |
|                                   | for disclosure to the             |
|                                   | Administrative Appeals Tribunal’  |
|                                   | and identify if there is any      |
|                                   | information that is relevant to   |
|                                   | the review of the decision, but   |
|                                   | which may require redaction. See  |
|                                   | the Exception handling table      |
|                                   | below for more information.       |
|                                   |                                   |
|                                   | The documents are saved           |
|                                   | electronically in the AST shared  |
|                                   | drive using the agreed naming     |
|                                   | convention. This should ensure    |
|                                   | that when the files are           |
|                                   | ultimately combined into a single |
|                                   | PDF document they will be in the  |
|                                   | correct order.                    |
|                                   |                                   |
|                                   | The agreed correct order for      |
|                                   | files lodged with the AAT as at 1 |
|                                   | July 2015 is:                     |
|                                   |                                   |
|                                   | 1.  AAT Record of application     |
|                                   | 2.  Written AAT application if    |
|                                   |     provided by applicant to AAT  |
|                                   | 3.  Customer letters and          |
|                                   |     Statement of Reasons          |
|                                   |     (objection decision)          |
|                                   | 4.  CS Assessment notice (if      |
|                                   |     objection was allowed and     |
|                                   |     assessment varied)            |
|                                   | 5.  Relevant other documents in   |
|                                   |     reverse chronological order   |
|                                   |     (Oldest to Newest)            |
|                                   | 6.  Cuba Screenshots (which are   |
|                                   |     not dated)                    |

Exception handling

Table 2: This table describes the process for identifying case where
additional action is required to be taken.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Exception handling + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Exception handling is the process |
|                                   | of identifying cases that require |
|                                   | additional actions to be taken,   |
|                                   | outside of the routine AAT review |
|                                   | process. The action required will |
|                                   | fall into the following           |
|                                   | categories:                       |
|                                   |                                   |
|                                   | Documents relating to an          |
|                                   | Ombudsman or Ministerial          |
|                                   | investigation                     |
|                                   |                                   |
|                                   | The Service Officer identifies a  |
|                                   | problem with the objection        |
|                                   | decision or there are issues      |
|                                   | regarding case management or      |
|                                   | customer service                  |
|                                   |                                   |
|                                   | There is something unusual about  |
|                                   | the case, application or          |
|                                   | objection decision and it may be  |
|                                   | appropriate for the Senior        |
|                                   | Advisor to make written and/or    |
|                                   | oral submissions to the AAT about |
|                                   | these issues                      |
|                                   |                                   |
|                                   | There are documents which are     |
|                                   | confidential or sensitive, for    |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   medical issues                |
|                                   | -   witness protection            |
|                                   | -   evidence from children, or    |
|                                   | -   conflict between the parents  |
|                                   |                                   |
|                                   | There is a history of family and  |
|                                   | domestic violence, or a Family    |
|                                   | Violence order. See Family and    |
|                                   | domestic violence.                |
|                                   |                                   |
|                                   | The above list is a guide only.   |
|                                   | If a Service Officer is unsure    |
|                                   | whether exception handling is     |
|                                   | required, seek advice from an SSO |
|                                   | or Team Leader.                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Information relating to an        |
|                                   | Ombudsman or Ministerial          |
|                                   | investigation + Read more ...     |
|                                   |                                   |
|                                   | Documents relating to an          |
|                                   | Ombudsman or Ministerial          |
|                                   | investigation should not be       |
|                                   | included in the AAT file unless   |
|                                   | it is directly related to the     |
|                                   | decision under review. In these   |
|                                   | situations if the Service Officer |
|                                   | considers that information        |
|                                   | relating to an Ombudsman or       |
|                                   | Ministerial investigation is      |
|                                   | relevant to the review, they      |
|                                   | should first consult with the     |
|                                   | SSO/TL.                           |
|                                   |                                   |
|                                   | The SSO/TL will contact the       |
|                                   | Ombudsman's or Ministerial office |
|                                   | to advise that the information    |
|                                   | may be sent to the AAT and both   |
|                                   | parties.                          |
|                                   |                                   |
|                                   | The SSO/TL will consider any      |
|                                   | request from the Ombudsman or     |
|                                   | Minister to submit a s35(4)       |
|                                   | non-disclosure application due to |
|                                   | the sensitivity of the            |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Identify if exception handling is |
|                                   | necessary (Service Officer) +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the Service Officer thinks     |
|                                   | that any personal information may |
|                                   | be relevant to the review, this   |
|                                   | must be brought to the attention  |
|                                   | of an SSO to consider whether     |
|                                   | Exception handling may be         |
|                                   | required. For example, a          |
|                                   | customer’s address may help the   |
|                                   | tribunal make a decision about    |
|                                   | the care of the child or the fate |
|                                   | that a parent was notified of a   |
|                                   | certain decision.                 |
|                                   |                                   |
|                                   | Consider the Guidelines for the   |
|                                   | preparation of documents to the   |
|                                   | AAT and determine what action is  |
|                                   | potentially necessary regarding   |
|                                   | the documents identified for      |
|                                   | exception handling.               |
|                                   |                                   |
|                                   | If the case meets one of the      |
|                                   | above criteria:                   |
|                                   |                                   |
|                                   | -   complete the AAT file         |
|                                   |     exception Handling Form and   |
|                                   |     record the potential need for |
|                                   |     action                        |
|                                   | -   Save the form to the AAT file |
|                                   | -   Copy and save all documents   |
|                                   |     relevant to the Exception     |
|                                   |     Handling Form in the          |
|                                   |     electronic AAT file           |
|                                   | -   record the exception request  |
|                                   |     on the SSAT Maintenance       |
|                                   |     window and include the date   |
|                                   |     the matter was referred to    |
|                                   |     the SSO, and                  |
|                                   | -   provide the AAT file to the   |
|                                   |     SSO and update the AAT        |
|                                   |     database                      |
|                                   |                                   |
|                                   | The Service Officer will make a   |
|                                   | recommendation and the SSO will   |
|                                   | consider that and advise the      |
|                                   | Service Officer of the outcome.   |
+-----------------------------------+-----------------------------------+
| 4                                 | Exception handling request        |
|                                   | considered (SSO) + Read more ...  |
|                                   |                                   |
|                                   | The SSO will consider the Service |
|                                   | Officer’s recommendation.         |
|                                   |                                   |
|                                   | The SSO will determine if contact |
|                                   | with the customer is required to  |
|                                   | explain the AAT open exchange of  |
|                                   | information process or            |
|                                   | consultation with the customer is |
|                                   | needed.                           |
|                                   |                                   |
|                                   | The SSO will assess the Service   |
|                                   | Officer’s exception handling      |
|                                   | request within 72 hours of        |
|                                   | receipt of the file and determine |
|                                   | if exception handling is          |
|                                   | appropriate. The SSO may seek     |
|                                   | advice from the Program Support   |
|                                   | Manager (PSM).                    |
|                                   |                                   |
|                                   | Note: if a document has already   |
|                                   | been exchanged with the other     |
|                                   | party, this may negate the need   |
|                                   | for a subsection 35(4)            |
|                                   | non-disclosure request. The SSO   |
|                                   | should seek advice from the PSM   |
|                                   | is there is any uncertainty.      |
|                                   |                                   |
|                                   | The SSO will advise the Service   |
|                                   | Officer of the outcome.           |
|                                   |                                   |
|                                   | If exception handling is          |
|                                   | required, go to Step 5.           |
|                                   |                                   |
|                                   | If exception handling is not      |
|                                   | required, go to Step 8.           |
+-----------------------------------+-----------------------------------+
| 5                                 | Exception handling required +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If exception handling is required |
|                                   | the SSO will determine if the     |
|                                   | information is both 'relevant'    |
|                                   | and 'sensitive' and they will     |
|                                   | make a non-disclosure             |
|                                   | application.                      |
|                                   |                                   |
|                                   | The SSO will:                     |
|                                   |                                   |
|                                   | prepare the AAT Non-disclosure of |
|                                   | document request - to AAT         |
|                                   | tribunal letter                   |
|                                   |                                   |
|                                   | complete the 'Schedule of         |
|                                   | documents subject to the s35(4)   |
|                                   | of the AAT Act application        |
|                                   | template. This will assist the    |
|                                   | AAT with the reason the           |
|                                   | application has been made for     |
|                                   | every page under consideration    |
|                                   |                                   |
|                                   | update the SSAT Maintenance       |
|                                   | window in Cuba with the exception |
|                                   | handling details                  |
|                                   |                                   |
|                                   | update the AAT database           |
|                                   |                                   |
|                                   | attach the AAT Non-disclosure of  |
|                                   | document request - to AAT         |
|                                   | tribunal letter and any Schedule  |
|                                   | of Documents                      |
|                                   |                                   |
|                                   | save the completed s35(4)         |
|                                   | application (AAT Non-disclosure   |
|                                   | of document request – to AAT) to  |
|                                   | the initial AAT file              |
|                                   |                                   |
|                                   | send the following documents to   |
|                                   | the AAT:                          |
|                                   |                                   |
|                                   | -   the original non-disclosure   |
|                                   |     application                   |
|                                   | -   the Schedule of Documents,    |
|                                   |     and                           |
|                                   | -   copies of the documents which |
|                                   |     are the subject of the        |
|                                   |     non-disclosure application    |
|                                   |     (as an adobe attachment), and |
|                                   | -   rename the file in line with  |
|                                   |     the AAT naming conventions    |
|                                   |     for non-disclosure            |
|                                   |     applications                  |
|                                   |                                   |
|                                   | Do not send the Schedule of       |
|                                   | Documents and copies of the       |
|                                   | documents which are the subject   |
|                                   | of the application or a copy of   |
|                                   | the application to the customers. |
+-----------------------------------+-----------------------------------+
| 6                                 | AAT to be advised of family       |
|                                   | violence risk + Read more ...     |
|                                   |                                   |
|                                   | Information about family and      |
|                                   | domestic violence can be provided |
|                                   | to the AAT if:                    |
|                                   |                                   |
|                                   | -   there is information or       |
|                                   |     communication indicating a    |
|                                   |     history of violence between   |
|                                   |     the parents which may affect  |
|                                   |     their conduct during the AAT  |
|                                   |     hearing                       |
|                                   | -   there is a current family     |
|                                   |     violence order in place       |
|                                   | -   there is an intervention or   |
|                                   |     restraining order against one |
|                                   |     or both parents               |
|                                   | -   either parent has expressed   |
|                                   |     concerns for the safety of    |
|                                   |     themselves or their children, |
|                                   |     and/or                        |
|                                   | -   we obtained or became aware   |
|                                   |     of the information within the |
|                                   |     last 2 years                  |
|                                   |                                   |
|                                   | In this instance the SSO should   |
|                                   | escalate the matter to their Team |
|                                   | Leader who will seek assistance   |
|                                   | from the Freedom of Information   |
|                                   | (FOI) team.                       |
|                                   |                                   |
|                                   | For more information on family    |
|                                   | and domestic violence see:        |
