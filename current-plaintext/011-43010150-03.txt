This page contains process information to enable Service Officers to
action radiation oncology accounts with multiple fields and keying the
account fee charged.

On this page:

Process accounts with multiple fields

Key the fee charged from the account

Process accounts with multiple fields

Table 1: This table describes how to pay the claim with 2 or more fields
indicated on the account.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identify fields + Read more ...   |
|                                   |                                   |
|                                   | Identify the:                     |
|                                   |                                   |
|                                   | -   single field base item on the |
|                                   |     account                       |
|                                   | -   matching additional field     |
|                                   |     derived fee item              |
|                                   | -   number of additional fields   |
|                                   |     on the account for the        |
|                                   |     additional field derived fee  |
|                                   |     item                          |
|                                   |                                   |
|                                   | Note: this information is         |
|                                   | populated by the health           |
|                                   | professional for online claims.   |
|                                   |                                   |
|                                   | The Resources page has a list of  |
|                                   | items.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Key details + Read more ...       |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | -   the details for the first     |
|                                   |     single field base item, and   |
|                                   | -   the details for the matching  |
|                                   |     additional field derived fee  |
|                                   |     item on the next line, then   |
|                                   | -   key the number of additional  |
|                                   |     fields in the NO field for    |
|                                   |     the additional field derived  |
|                                   |     fee item. The maximum number  |
|                                   |     of additional fields is 5     |
|                                   |                                   |
|                                   | Note: multiple treatments on the  |
|                                   | same day must be keyed in         |
|                                   | separate claims.                  |
|                                   |                                   |
|                                   | Repeat this step for the          |
|                                   | remainder of items.               |
|                                   |                                   |
|                                   | For:                              |
|                                   |                                   |
|                                   | -   manual claims, go to Step 3   |
|                                   | -   online claims, go to Step 4   |
+-----------------------------------+-----------------------------------+
| 3                                 | Key LSPN and equipment number +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Key the Location Specific         |
|                                   | Practice Number (LSPN) and        |
|                                   | equipment number in the FEE       |
|                                   | field. Preceding zeros are not    |
|                                   | required for the LSPN. Example:   |
|                                   | L0125.                            |
|                                   |                                   |
|                                   | Note: this information will be    |
|                                   | populated by the health           |
|                                   | professional for online claims.   |
+-----------------------------------+-----------------------------------+
| 4                                 | Return messages + Read more ...   |
|                                   |                                   |
|                                   | Check all return messages in      |
|                                   | full:                             |
|                                   |                                   |
|                                   | -   press [F1] to display         |
|                                   |     multiple pages and actions    |
|                                   |     where necessary               |
|                                   | -   press [F1] again to display   |
|                                   |     multiple pages and actions    |
|                                   |     within each return message    |
|                                   |                                   |
|                                   | The Resources page includes       |
|                                   | radiotherapy return messages and  |
|                                   | resolutions.                      |
+-----------------------------------+-----------------------------------+

Key the fee charged from the account

Table 2: This table lists the actions taken to record the fee charged.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Single fee charged for each       |
|                                   | item + Read more ...              |
|                                   |                                   |
|                                   | Key each fee in the charge field  |
|                                   | against each item.                |
+-----------------------------------+-----------------------------------+
| 2                                 | One fee charged for combined      |
|                                   | single field base item and        |
|                                   | matching additional field derived |
|                                   | fee item + Read more ...          |
|                                   |                                   |
|                                   | Key the fee charged against the   |
|                                   | single field base item and key    |
|                                   | zero in the charge field for the  |
|                                   | additional field derived fee      |
|                                   | item.                             |
|                                   |                                   |
|                                   | The Resources page has a list of  |
|                                   | items.                            |
+-----------------------------------+-----------------------------------+
| 3                                 | One fee charged for the           |
|                                   | combination of all items on the   |
|                                   | account + Read more ...           |
|                                   |                                   |
|                                   | Key the fee charged against the   |
|                                   | first single field base item as a |
|                                   | notional charge and key zero in   |
|                                   | the charge field for all          |
|                                   | remaining items.                  |
+-----------------------------------+-----------------------------------+
