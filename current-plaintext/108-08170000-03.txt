|                                   |                                   |
|                                   | Newborn Upfront Payment (NBU)     |
|                                   |                                   |
|                                   | The system will use arrears to    |
|                                   | offset a debt if an activity      |
|                                   | causes an NBU debt due to loss of |
|                                   | eligibility. For example, FTB A   |
|                                   | cancels and then an activity pays |
|                                   | NBU arrears. To allow the         |
|                                   | automated withholdings process to |
|                                   | offset arrears, do not suppress   |
|                                   | the arrears and do not code FMAD. |
|                                   |                                   |
|                                   | Withholdings will automatically   |
|                                   | apply from the NBU payment to the |
|                                   | outstanding amount of an existing |
|                                   | NBU debt. This displays on the PS |
|                                   | and PD screens before the         |
|                                   | activity is completed. It         |
|                                   | displays as a withholding for     |
|                                   | Family Assistance.                |
|                                   |                                   |
|                                   | This offsetting also occurs when  |
|                                   | processing multiple activities on |
|                                   | one day. The first paying arrears |
|                                   | of NBU and then an activity       |
|                                   | causing a loss of eligibility     |
|                                   | resulting in a debt. If these     |
|                                   | activities are on the same day,   |
|                                   | the offset will occur so the      |
|                                   | agency does not issue a payment   |
|                                   | and then raise a debt on the same |
|                                   | day.                              |
|                                   |                                   |
|                                   | Parental Leave Pay (PPL)          |
|                                   |                                   |
|                                   | PPL arrears will offset against   |
|                                   | any PPL overpayments or income    |
|                                   | support payment (ISP)             |
|                                   | overpayments that are generated   |
|                                   | in the same activity. This only   |
|                                   | applies to ISP overpayments that  |
|                                   | occur due to the payment of PPL   |
|                                   | for the past period.              |
|                                   |                                   |
|                                   | This offset will occur to avoid   |
|                                   | the issue of a payment and debt   |
|                                   | raised on the same day.           |
