This page contains an example of recording ex-partner details for Family
Tax Benefit (FTB), Child Care Subsidy (CCS) and Child Care Benefit
(CCB). The dates used are examples only. Links on this page may be
unavailable to external audiences. All CCB/CCR references relate to
financial years prior to 2018/2019.

Example - Recording ex-partner details

This table contains an example of recording ex-partner details for
Family Tax Benefit (FTB).

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Jody has claimed Family Tax       |
|                                   | Benefit (FTB) for a past period   |
|                                   |                                   |
|                                   | Jody is currently single but was  |
|                                   | partnered to David on 6 June      |
|                                   | 2000. They separated on 9         |
|                                   | September 2000.                   |
|                                   |                                   |
|                                   | The Service Officer needs to      |
|                                   | 'I'nsert Jody's relationship      |
|                                   | history onto the Marital Status   |
|                                   | (MS) screen. This is done when    |
|                                   | recording the claim. Jody's       |
|                                   | changes in relationship status    |
|                                   | need to be recorded in reverse    |
|                                   | order, starting with the          |
|                                   | 'SEP'aration on the 9 September   |
|                                   | 2000.                             |
|                                   |                                   |
|                                   | Jody's relationship status is     |
|                                   | currently 'SIN'gle with the date  |
|                                   | of event as Jody's date of birth. |
|                                   |                                   |
|                                   | Jody's MS screen appears as       |
|                                   | below. Coding is in bold and      |
|                                   | numbered to show the order in     |
|                                   | which the lines are inserted.     |
|                                   |                                   |
|                                   | When relationship status is       |
|                                   | updated to 'MAR'ried, the Link    |
|                                   | Partner (LP) screen will display. |
|                                   |                                   |
|                                   | As this is a previous             |
|                                   | relationship, David's record is   |
|                                   | not updated automatically. A      |
|                                   | message to this effect is         |
|                                   | displayed.                        |
+-----------------------------------+-----------------------------------+
