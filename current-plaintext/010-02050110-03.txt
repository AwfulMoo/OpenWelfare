| 1                                 | Payment method for SFA (Group 2)  |
|                                   | payable directly to a school +    |

|                                   | student is:                       |

|                                   | -   boarding at a school, in an   |
|                                   |     approved ABSTUDY Homestay     |
|                                   |     arrangement, or at a hostel   |

|                                   |     ABSTUDY Hostel Agreement. Go  |

|                                   | -   boarding in a private         |
|                                   |     arrangement, or at a hostel   |

|                                   |     Go to Step 3                  |
|                                   | -   eligible for the Away from    |
|                                   |     Home rate due to limited      |
|                                   |     local school facilities or    |
|                                   |     travel time and access        |
|                                   |     criteria but elects to remain |
|                                   |     at home and attend a local    |

|                                   | -   eligible for the payment of   |
|                                   |     SFA (Group 2) while living    |

|                                   |     (Group 2) rate of SFA (but    |

|                                   |     changes to within reasonable  |
|                                   |     travel time or distance of    |
|                                   |     the school the student is     |
|                                   |     studying at, and the student  |
|                                   |     continues studying at the     |
|                                   |     same school as a day student. |
|                                   |     Go to Step 6                  |

|                                   |                                   |
|                                   | Go to Step 5.                     |
