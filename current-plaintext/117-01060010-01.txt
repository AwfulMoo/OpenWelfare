This document outlines how the Compensation Management System (CMS) will
finalise a claim.

Automatic finalisation

This will occur in the following instances:

-   A clearance using a certain clearance code for compensatory payments
-   A nil charge
-   A repaid debt

The Resources page contains a full list of finalisation codes.
