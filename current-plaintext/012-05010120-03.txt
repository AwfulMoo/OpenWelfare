This page contains the process for using email templates for outgoing
correspondence from the eBusiness Service Centre.

Use eBusiness Service Centre email templates

Create email using eBusiness Service Centre email templates

This table describes how to use email templates used by eBusiness
Service Centres for outgoing correspondence.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access eBusiness email            |
|                                   | templates + Read more ...         |
|                                   |                                   |
|                                   | Open the National eBusiness       |
|                                   | Service Centre email box.         |
|                                   |                                   |
|                                   | In the left hand pane, select     |
|                                   | Stationery. A list of email       |
|                                   | templates are displayed in the    |
|                                   | Stationery column.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Select email template + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Select the relevant email         |
|                                   | template. Highlight the template, |
|                                   | right click and select Forward.   |
|                                   | Use:                              |
|                                   |                                   |
|                                   | -   Application for a PKI Site    |
|                                   |     Certificate for a practice or |
|                                   |     site wishing to register for  |
|                                   |     a PKI Site Certificate        |
|                                   | -   Application for a PKI         |
|                                   |     Individual Certificate for an |
|                                   |     applicant wishing to register |
|                                   |     for a PKI individual          |
|                                   |     certificate.                  |
|                                   | -   Application for a PKI         |
|                                   |     Individual Certificate for    |
|                                   |     DVA Organisations wishing to  |
|                                   |     register for a PKI individual |
|                                   |     certificate                   |
|                                   | -   Request to Revoke and Reissue |
|                                   |     a PKI Individual Certificate  |
|                                   |     – for an individual           |
|                                   |     certificate holder where the  |
|                                   |     applicant requires a          |
|                                   |     replacement PKI certificate.  |
+-----------------------------------+-----------------------------------+
| 3                                 | Add details to the template +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Key the applicable details into   |
|                                   | the template using the field      |
|                                   | guides within the template.       |
+-----------------------------------+-----------------------------------+
| 4                                 | Send and file email + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   CC eBusiness into the email   |
|                                   | -   Select Send and email         |
|                                   |     classification as             |
|                                   |     Unclassified, select OK. A    |
|                                   |     copy of the sent email is     |
|                                   |     sent to the eBusiness inbox.  |
|                                   | -   Select the email from the     |
|                                   |     Inbox, highlight and click    |
|                                   |     and drag email to the         |
|                                   |     relevant folder under Emails  |
|                                   |     sent by staff.                |
+-----------------------------------+-----------------------------------+
