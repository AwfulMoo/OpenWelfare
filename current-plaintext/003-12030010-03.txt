| 1                                 | Finalise the UTLAH claim + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To finalise a UTLAH claim         |
|                                   | complete the following:           |
|                                   |                                   |
|                                   | check that all procedures have    |
|                                   | been followed correctly and there |
|                                   | is sufficient verification to     |
|                                   | support any recommendation to     |
|                                   | grant or reject the claim         |
|                                   |                                   |
|                                   | clearly outline the reason for    |
|                                   | granting or rejecting independent |
|                                   | status on the basis of UTLAH      |
|                                   | criteria                          |
|                                   |                                   |
|                                   | DOC what nominee or Person        |
|                                   | Permitted to Enquire (PPE)        |
|                                   | arrangements have been discussed, |
|                                   | including current arrangements    |
|                                   |                                   |
|                                   | make sure all confidential        |
|                                   | material is stored appropriately  |
|                                   |                                   |
|                                   | record all information in the     |
|                                   | SWIS report, and                  |
|                                   |                                   |
|                                   | determine any need for follow-up  |
|                                   | and/or referral for social        |
|                                   | casework and/or refer customer to |
|                                   | appropriate services if needed    |
|                                   |                                   |
|                                   | if granting the claim, update the |
|                                   | customer's payment destination if |
|                                   | before grant of UTLAH the         |
|                                   | payments were received by a       |
|                                   | parent or guardian                |
|                                   |                                   |
|                                   | if rejecting the claim, refer the |
|                                   | application for payment as a      |
|                                   | dependent back to the processing  |
|                                   | team                              |
|                                   |                                   |
|                                   | -   discuss any issues that may   |
|                                   |     exist around the relationship |
|                                   |     between the customer and the  |
|                                   |     parents that may impact on    |
|                                   |     the customer's ability to get |
|                                   |     Parental Income details       |
|                                   | -   consider recommending the     |
|                                   |     away from home rate of Youth  |
|                                   |     Allowance (if applicable)     |
+-----------------------------------+-----------------------------------+
| 2                                 | Complete the Fast Note + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Complete the relevant Fast Note   |
|                                   | in Customer First. The Fast Note  |
|                                   | will automatically transfer to    |
|                                   | the correct processing team once  |
|                                   | completed. For:                   |
|                                   |                                   |
|                                   | -   Youth Allowance Student, use  |
|                                   |     UTLAH STUDENT GRANT/REJECT    |
|                                   |     Fast Note                     |
|                                   | -   Youth Allowance Jobseeker,    |
|                                   |     use UTLAH JOBSEEKER           |
|                                   |     GRANT/REJECT Fast Note        |
|                                   | -   Tertiary Access Payment, use  |
|                                   |     the UTLAH TAP GRANT/REJECT    |
|                                   |     Fast Note                     |
|                                   | -   Disability Support Pension    |
|                                   |     (DSP), use UTLAH DSP          |
|                                   |     GRANT/REJECT Fast Note        |
|                                   | -   ABSTUDY, use UTLAH ABSTUDY    |
|                                   |     GRANT/REJECT Fast Note        |
|                                   | -   Special Benefit (SpB), use SW |
|                                   |     recommendation - SpB Under 16 |
|                                   |     Fast Note                     |

|                                   | For more detail on creating these |
|                                   | Fast Notes using the Auto text    |
|                                   | function, see Assessing           |
|                                   | unreasonable to live at home      |
|                                   | (UTLAH) for customers claiming or |
|                                   | receiving Youth Allowance (YA),   |
|                                   | Disability Support Pension (DSP), |

|                                   | Access Payment (TAP), (Process    |
