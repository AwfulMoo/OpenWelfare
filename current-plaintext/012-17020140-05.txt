DVA physiotherapy service common enquiries

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: My Physiotherapy item   |
|                                   | hasn’t been paid because another  |
|                                   | provider in the practice has      |
|                                   | claimed for a chiropractic        |
|                                   | consultation on the same day. Why |
|                                   | hasn’t this been paid?            |
|                                   |                                   |
|                                   | Answer: Concurrent physiotherapy  |
|                                   | and chiropractic is not permitted |
|                                   | for the same entitled patient.    |
|                                   | Check the Notes for Allied Health |
|                                   | Providers on the DVA website for  |
|                                   | more information.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: I have claimed 2        |
|                                   | physiotherapy items for the same  |
|                                   | patient on the same day and only  |
|                                   | 1 item has been paid. Why?        |
|                                   |                                   |
|                                   | Answer: 2 subsequent              |
|                                   | physiotherapy consultation items  |
|                                   | cannot be claimed on the same     |
|                                   | day.                              |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: I have claimed a        |
|                                   | postage item PH98 but it hasn’t   |
|                                   | been paid, why is that?           |
|                                   |                                   |
|                                   | Answer: The item PH98 requires    |
|                                   | the item PH94 to be claimed on    |
|                                   | the same day. If there is nothing |
|                                   | on history, the provider will     |
|                                   | need to re-submit a claim for     |
|                                   | PH94 & PH98 for the same DOS in   |
|                                   | the same claim                    |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: I am claiming a         |
|                                   | physiotherapy consultation item   |
|                                   | and an aquatic physiotherapy      |
|                                   | item. Will I be paid for both?    |
|                                   |                                   |
|                                   | Answer: No, only 1 item is        |
|                                   | payable per veteran, per day.     |
+-----------------------------------+-----------------------------------+
