This page contains the process for calculating the derived fee item
amount and total benefit for a specialist Telehealth consultation.

Calculating derived fee item amount

Calculating derived fee item amount and total benefit for specialist Telehealth consultation

This table describes how to calculate the derived fee item amount and
total benefit for a specialist Telehealth consultation.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Identify schedule fee + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Identify the base item schedule   |
|                                   | fee.                              |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Base Item 104                 |
|                                   | -   Schedule fee = $83.95         |
+-----------------------------------+-----------------------------------+
| 2                                 | Divide schedule fee + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Divide the schedule fee by 50%.   |
|                                   |                                   |
|                                   | Result: this will be the derived  |
|                                   | fee amount.                       |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Base Item 104                 |
|                                   | -   Schedule fee = $83.95         |
|                                   | -   50% = $41.97                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Add base item and derived fee     |
|                                   | amounts + Read more ...           |
|                                   |                                   |
|                                   | Add the base item schedule fee    |
|                                   | and the derived fee amount.       |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Base Item 104                 |
|                                   | -   Schedule fee = $83.95         |
|                                   | -   Derived fee = $41.97          |
|                                   | -   $83.95 + $41.97 = $125.92     |
+-----------------------------------+-----------------------------------+
| 4                                 | Divide total + Read more ...      |
|                                   |                                   |
|                                   | Divide the total from Step 4 by   |
|                                   | 85%.                              |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   $125.92 x 85% = $107.03       |
+-----------------------------------+-----------------------------------+
| 5                                 | Apply rounding + Read more ...    |
|                                   |                                   |
|                                   | Round up/down to the nearest 5    |
|                                   | cents.                            |
|                                   |                                   |
|                                   | Result: this will be the total    |
|                                   | benefit amount.                   |
|                                   |                                   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Benefit amount = $107.05      |
+-----------------------------------+-----------------------------------+
