Eligibility for CrP under other extreme circumstances

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Qualification for CrP for Other   |
|                                   | Extreme Circumstances + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | In order to qualify for payment   |
|                                   | of CrP under other extreme        |
|                                   | circumstances, a customer must:   |
|                                   |                                   |
|                                   | -   meet the common criteria for  |
|                                   |     CRP, including severe         |
|                                   |     financial hardship            |
|                                   | -   make a claim for CrP within 7 |
|                                   |     days after the extreme        |
|                                   |     circumstance occurred         |
|                                   | -   have left their home due to   |
|                                   |     the extreme circumstance, and |
|                                   |     it is unreasonable to expect  |
|                                   |     the person to remain in or    |
|                                   |     return to their home          |
|                                   | -   have established a new home,  |
|                                   |     or are intending to establish |
|                                   |     a new home                    |
|                                   | -   have been in Australia when   |
|                                   |     the extreme circumstance      |
|                                   |     occurred and lodged a claim   |
|                                   |     for CrP within Australia      |
|                                   | -   not have been paid more than  |
|                                   |     4 payments of CrP for extreme |
|                                   |     circumstances combined        |
|                                   |     (family and domestic violence |
|                                   |     and/or other extreme          |
|                                   |     circumstances) in any 12      |
|                                   |     month period                  |
|                                   |                                   |
|                                   | Some common reasons customers may |
|                                   | claim CrP-EXO include a house     |
|                                   | fire, flooding (for example, due  |
|                                   | a burst water main), structural   |
|                                   | damage, or where the customer     |
|                                   | feels unsafe (for example, due to |
|                                   | community violence or repeated    |
|                                   | break-ins/burglaries).            |
|                                   |                                   |
|                                   | More examples of CrP other        |
|                                   | extreme circumstances are         |
|                                   | available on the Resources page.  |
|                                   |                                   |
|                                   | A customer cannot get a CrP-EXO   |
|                                   | if they:                          |
|                                   |                                   |
|                                   | -   intentionally brought about   |
|                                   |     the extreme circumstance with |
|                                   |     a view to getting a payment   |
|                                   |     of CrP                        |
|                                   | -   were evicted from their home  |
|                                   |     due to not paying their       |
|                                   |     mortgage or rent              |
|                                   | -   had to move out because their |
|                                   |     landlord is selling the       |
|                                   |     property or moving back in    |
|                                   | -   decided to move to another    |
|                                   |     home                          |
|                                   | -   are living in government      |
|                                   |     housing and have applied for  |
|                                   |     a transfer to a different     |
|                                   |     location, but are still       |
|                                   |     living at the current         |
|                                   |     location or intend to return  |
|                                   |     while they await the transfer |
|                                   |     (such as have not established |
|                                   |     a new home)                   |
|                                   |                                   |
|                                   | A customer does not qualify to    |
|                                   | receive a CrP - EXO if a disaster |
|                                   | relief payment administered by    |
|                                   | the local, State or Commonwealth  |
|                                   | Government for the same extreme   |
|                                   | circumstance is available.        |
|                                   |                                   |
|                                   | Note: if the AGDRP or DRP (or any |
|                                   | other disaster relief payment) is |
|                                   | announced as being available:     |
|                                   |                                   |
|                                   | -   Crisis Payment is only        |
|                                   |     payable to people who lodged  |
|                                   |     a claim for CrP before the    |
|                                   |     day of the announcement of    |
|                                   |     the disaster relief payment,  |
|                                   |     given they meet all other     |
|                                   |     eligibility conditions        |
|                                   | -   Customers who have been       |
|                                   |     granted CrP prior to the      |
|                                   |     announcement could also test  |
|                                   |     their eligibility for a       |
|                                   |     disaster relief payment       |
|                                   |                                   |
|                                   | Has the customer met all the      |
|                                   | qualifications for CrP other      |
|                                   | extreme circumstance?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer qualifies for CrP + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where the customer appears to     |
|                                   | have met all the qualifications   |
|                                   | for CrP, a new claim should be    |
|                                   | invited. For CrP other extreme    |
|                                   | circumstances claims, see New     |
|                                   | claim for Crisis Payment Other    |

|                                   | Note: claims for most Crisis      |
|                                   | Payments - other extreme          |

|                                   | referral to a social worker may   |

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   |                                   |
|                                   | A customer (or someone on their   |
|                                   | behalf) contacts Services         |
|                                   | Australia seeking help, as they   |
|                                   | are due to be, or have recently   |
|                                   | been, released from prison or     |
|                                   | psychiatric confinement.          |
+-----------------------------------+-----------------------------------+
| 2                                 | In prison or psychiatric          |
|                                   | confinement for 14 days and       |
|                                   | charged with committing an        |
|                                   | offence + Read more ...           |
|                                   |                                   |
|                                   | To be eligible for a CrP-PRI, the |
|                                   | customer must:                    |
|                                   |                                   |
|                                   | -   meet the common criteria for  |
|                                   |     CRP, including severe         |
|                                   |     financial hardship            |
|                                   | -   have been in prison or        |
|                                   |     psychiatric confinement for   |
|                                   |     14 or more days               |
|                                   | -   have been charged with        |
|                                   |     committing an offence         |
|                                   |                                   |
|                                   | For the purposes of CrP, when     |
|                                   | determining if a customer has     |
|                                   | been detained for 14 days, the    |
|                                   | first day of entry into           |
|                                   | imprisonment/confinement or       |
|                                   | detention is included in the      |
|                                   | calculation of days. The day of   |
|                                   | release is not included.          |
|                                   |                                   |
|                                   | A customer is not eligible for    |
|                                   | CrP-PRI if they:                  |
|                                   |                                   |
|                                   | -   were held in an immigration   |
|                                   |     detention centre              |
|                                   | -   have been in a mental health  |
|                                   |     facility or psychiatric       |
|                                   |     confinement but have not been |
|                                   |     charged with committing an    |
|                                   |     offence                       |
|                                   |                                   |
|                                   | Different rules apply when        |
|                                   | suspending/cancelling payments    |
|                                   | for customers admitted into       |
|                                   | custody. See:                     |
|                                   |                                   |
|                                   | -   Confirming prison admission   |
|                                   |     and release dates and         |
|                                   | -   Admission process for         |
|                                   |     incarcerated customers        |
|                                   |                                   |
|                                   | Was the person detained for 14    |
|                                   | days or more as a result of being |
|                                   | charged with an offence?          |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 3                                 | Additional qualifications for     |
|                                   | CrP + Read more ...               |
|                                   |                                   |
|                                   | In addition to being charged with |
|                                   | an offence and being detained for |
|                                   | 14 days or more CrP may be        |
|                                   | payable to the customer if:       |
|                                   |                                   |
|                                   | -   the common criteria for CrP   |
|                                   |     are met, including severe     |
|                                   |     financial hardship            |
|                                   | -   a claim for CrP is made       |
|                                   |     within 7 days after the       |
|                                   |     extreme circumstance occurred |
|                                   |                                   |
|                                   | The customer can have been in     |
|                                   | prison or psychiatric confinement |
|                                   | overseas, but must make a claim   |
|                                   | from within Australia.            |
|                                   |                                   |
|                                   | Claims for CrP-PRI can also be    |
|                                   | made up to 21 days before the     |
|                                   | customerâ€™s expected release date  |
|                                   | from prison or psychiatric        |
|                                   | confinement. However, these       |
|                                   | customers are usually assisted by |
|                                   | Prison Liaison Officers who       |
|                                   | assess the claim while the        |
|                                   | customer is still in the          |
|                                   | facility.                         |
|                                   |                                   |
