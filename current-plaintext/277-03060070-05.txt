Examples for date of effect for family assistance assessments

This table provides examples to assist with determining the date of
effect for family assistance assessments.

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Example 1                         |
|                                   |                                   |
|                                   | Child moves from 100% care of Mum |
|                                   | to 100% care of Dad and advise    |
|                                   | this within 28 days. This is a    |
|                                   | change of care not a terminating  |
|                                   | event. This change in care will   |
|                                   | take effect for the:              |
|                                   |                                   |
|                                   | -   child support assessment from |
|                                   |     the date the change occurred  |
|                                   | -   family assistance assessment  |
|                                   |     from the date the change      |
|                                   |     occurred                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Example 2                         |
|                                   |                                   |
|                                   | Child Jill moved from Fran’s      |
|                                   | (Mum) 100% care to Dave’s (Dad)   |
|                                   | 100% care on 2 February 2018.     |
|                                   | Fran and Dave do not have a care  |
|                                   | arrangement for Jill. Fran        |
|                                   | advised the agency of this change |
|                                   | in care on 10 July 2018. Both     |
|                                   | parties confirm the event date    |
|                                   | and the new pattern of care. This |
|                                   | change in care will take effect   |
|                                   | for the:                          |
|                                   |                                   |
|                                   | -   child support assessment - as |
|                                   |     a result of this care change  |
|                                   |     all eligible carers (Fran)    |
|                                   |     ceased to be eligible carers  |
|                                   |     and the parent who was not an |
|                                   |     eligible carer (Dave) would   |
|                                   |     have become an eligible carer |
|                                   |     except for the care date of   |
|                                   |     effect rules. Therefore, the  |
|                                   |     child support assessment is   |
|                                   |     suspended from 2 February to  |
|                                   |     9 July 2018. The new care     |
|                                   |     arrangement will be used in   |
|                                   |     the assessment from 10 July   |
|                                   |     2018, the date of             |
|                                   |     notification                  |
|                                   | -   family assistance assessment  |
|                                   |     from the date of event- 2     |
|                                   |     February 2018                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Example 3                         |
|                                   |                                   |
|                                   | Justin moved from Kevin’s (Dad)   |
|                                   | 100% care to Jodie’s (Mum) 100%   |
|                                   | care on 1 August 2018. Jodie      |
|                                   | advises the agency of this change |
|                                   | in care on 8 March 2019. Both     |
|                                   | parties confirm the date of event |
|                                   | and the new pattern of care.      |
|                                   |                                   |
|                                   | For the family assistance         |
|                                   | assessment this change will take  |
|                                   | effect from the date of event,    |
|                                   | being 1 August 2018.              |
|                                   |                                   |
|                                   | For the child support assessment, |
|                                   | as Jodie's care reduces below 35% |
|                                   | from 1 August 2018 (date of       |
|                                   | event) and Kevin's increase       |
|                                   | percentage of care cannot be used |
|                                   | in the assessment until 8 March   |
|                                   | 2019 (date of notification), a    |
|                                   | terminating event has occurred on |
|                                   | 1 August 2018 because neither     |
|                                   | parent has a care percentage of   |
|                                   | 35% or more. When the agency is   |
|                                   | advised of the change in care on  |
|                                   | 8 March 2019, a new application   |
|                                   | for child support is required if  |
|                                   | either parent wants to re-start   |
|                                   | the assessment for Justin.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Example 4                         |
|                                   |                                   |
|                                   | Grandparent has 40% care of their |
|                                   | grandchild and has an existing    |
|                                   | child support assessment with the |
|                                   | child’s mother (who has the       |
|                                   | remaining 60% care). The child    |
|                                   | then moves into the               |
|                                   | grandparents’s full-time care     |
|                                   | (100% care).                      |
|                                   |                                   |
|                                   | The care change was advised with  |
|                                   | 28 days. Both parties confirm the |
|                                   | event date and the new pattern of |
|                                   | care. This is a change of care    |
|                                   | not a terminating event. The      |
|                                   | child is still in the care of one |
|                                   | of the parties to the child       |
|                                   | support assessment. The change    |
|                                   | will take effect for the:         |
|                                   |                                   |
|                                   | -   child support assessment from |
|                                   |     the date the change occurred  |
|                                   | -   family assistance assessment  |
|                                   |     from the date the change      |
|                                   |     occurred                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Example 5                         |
|                                   |                                   |
|                                   | Tommy’s grandmother, Suzie,       |
|                                   | applied for a child support       |
|                                   | assessment on 1 October 2011 with |
|                                   | Tommy in Suzie’s care for 40% of  |
|                                   | the time, and 60% of the time in  |
|                                   | Kate’s (mother) care.             |
|                                   |                                   |
|                                   | Tommy moved from Kate’s (parent)  |
|                                   | 60% care to Suzie’s (grandparent) |
|                                   | 100% care on 3 March 2019. Kate   |
|                                   | advised the agency of the care    |
|                                   | change on 19 April 2019. Both     |
|                                   | parties confirm the event date    |
|                                   | and the new pattern of care. This |
|                                   | change in care will take effect   |
|                                   | for the:                          |
|                                   |                                   |
|                                   | -   child support assessment -    |
|                                   |     Kate’s care has decreased so  |
|                                   |     it will take effect in the    |
|                                   |     assessment from 3 March 2019. |
|                                   |     Suzie’s care has increased so |
|                                   |     Suzie’s care will increase in |
|                                   |     the assessment from 19 April  |
|                                   |     2019 because the care change  |
|                                   |     was notified more than 28     |
|                                   |     days after the care changed   |
|                                   | -   family assistance assessment  |
|                                   |     from 3 March 2019 being the   |
|                                   |     date the change of care       |
|                                   |     occurred                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Example 6                         |
|                                   |                                   |
|                                   | Mum and Dad equally share the     |
|                                   | care of 1 child (50% care each).  |
|                                   | The child then moves into 100%    |
|                                   | care of an aunt, who is not a     |
|                                   | party to the current assessment.  |
|                                   | This is a terminating event for   |
|                                   | the current child support         |
|                                   | assessment as both Mum and Dad    |
|                                   | cease to be eligible carers       |
|                                   | because their percentage of care  |
|                                   | has fallen to 0%.                 |
|                                   |                                   |
|                                   | When a terminating event occurs,  |
|                                   | the assessment must change from   |
|                                   | the date of event even if the     |
|                                   | agency is notified after 28 days. |
|                                   | Mum and Dad would also lose their |
|                                   | entitlement to FTB and the aunt   |
|                                   | will have the option of making a  |
|                                   | claim.                            |
+-----------------------------------+-----------------------------------+
