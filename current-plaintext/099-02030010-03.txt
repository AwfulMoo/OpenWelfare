This page contains information about manually reprinting the Bereavement

Manually reprinting the Bereavement Statement

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | To reprint the Bereavement        |
|                                   | Statement + Read more ...         |
|                                   |                                   |
|                                   | go to the History Summary (HS)    |
|                                   | screen                            |
|                                   |                                   |
|                                   | select the drop down box next to  |
|                                   | the Bereavement Statement         |
|                                   | (PENBIAST)                        |
|                                   |                                   |
|                                   | select the option:                |
|                                   |                                   |
|                                   | -   'Print Central', or           |
|                                   | -   'Print Local'                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Availability of the Bereavement   |
|                                   | Statement to reissue + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The statement is available from   |
|                                   | the HS screen for 62 days from    |
|                                   | initial issue. After this time,   |
|                                   |                                   |
|                                   | -   it will be available from the |
|                                   |     Archiving and Culling Engine  |
|                                   |     (ACE) system if it was        |
|                                   |     delivered via Online Letters  |
|                                   |     (OLL), or                     |
|                                   | -   a copy of the letter text can |
|                                   |     be obtained from the          |
|                                   |     Centrelink Archive Repository |
|                                   |     (CAR) system if it was sent   |
|                                   |     by post                       |
+-----------------------------------+-----------------------------------+
| 3                                 | Helpdesk + Read more ...          |
|                                   |                                   |
|                                   | To report any problems and/or     |
|                                   | feedback relating to the          |
|                                   | Bereavement Statement, see        |
|                                   | Bereavement for contact details   |
+-----------------------------------+-----------------------------------+
