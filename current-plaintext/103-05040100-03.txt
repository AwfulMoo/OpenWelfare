| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Fraudulent advance claim          |

|                                   | been reported on a customer       |
|                                   | record?                           |

|                                   |     be deleted from the AVCS      |

|                                   | investigation complete            |
