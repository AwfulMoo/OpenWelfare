For ACCS Processing Team (ACCPT) staff only.

This page contains information about how to review and actioning appeals
for Additional Child Care Subsidy (ACCS) (Child Wellbeing).

On this page:

determination

ACCS (Child Wellbeing) appeal

(SME)

Services Australia is identified as the Original Decision maker

|                                   | notified of the decision. This is |

|                                   | applying for a review.            |

|                                   | notes on the record).             |
|                                   |                                   |

|                                   |     rejected. See Step 3 in Table |

|                                   | -   No, the reassessment is to be |
|                                   |     rejected as the evidence does |
|                                   |     not support the child being   |
|                                   |     assessed as at risk. See Step |
|                                   |     3 in Table 3                  |
