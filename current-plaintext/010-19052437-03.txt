| 2                                 | Customer details + Read more ...  |
|                                   |                                   |
|                                   | Australian Apprenticeship         |
|                                   |                                   |
|                                   | If the customer is an Australian  |
|                                   | Apprentice, they must provide     |
|                                   | details of their Australian       |
|                                   | Apprenticeship. Services          |
|                                   | Australia will then confirm this  |
|                                   | information with the Department   |
|                                   | of Employment and Workplace       |
|                                   | Relations.                        |
|                                   |                                   |
|                                   | Aboriginality                     |
|                                   |                                   |
|                                   | For ABSTUDY purposes, an          |
|                                   | Australian Aboriginal or Torres   |
|                                   | Strait Islander is someone who:   |
|                                   |                                   |
|                                   | -   is of Australian Aboriginal   |
|                                   |     or Torres Strait Islander     |
|                                   |     descent                       |
|                                   | -   identifies themselves as an   |
|                                   |     Australian Aboriginal or      |
|                                   |     Torres Strait Islander, and   |
|                                   | -   is accepted as such by the    |
|                                   |     community in which they live  |
|                                   |                                   |
|                                   | The declaration on the claim will |
|                                   | normally suffice, however for     |
|                                   | more information, see             |
|                                   | Aboriginality or Torres Strait    |
|                                   | Islander status for ABSTUDY       |
|                                   | purposes.                         |
|                                   |                                   |
|                                   | Born overseas                     |
|                                   |                                   |
|                                   | Customers must be an Australian   |
|                                   | citizen to qualify for ABSTUDY.   |
|                                   | Proof of citizenship is only      |
|                                   | required for customers:           |
|                                   |                                   |
|                                   | -   born overseas, and            |
|                                   | -   who have not provided this    |
|                                   |     proof to Centrelink           |
|                                   |                                   |
|                                   | See Residence requirements for    |
|                                   | ABSTUDY.                          |
|                                   |                                   |
|                                   | Name                              |
|                                   |                                   |
|                                   | Customers must provide all names  |
|                                   | they are currently known by, or   |
|                                   | have been known by in the past.   |
|                                   |                                   |
|                                   | Address                           |
|                                   |                                   |
|                                   | The permanent family home address |
|                                   | must be provided.                 |
|                                   |                                   |
|                                   | Optional questions                |
|                                   |                                   |
|                                   | Customers who are under 22 years  |
|                                   | of age and have a disability      |
|                                   | preventing them from working up   |
|                                   | to 30 hours per week may be       |
|                                   | eligible for the Youth Disability |
|                                   | Supplement (YDS).                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Other government assistance +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A customer is generally           |
|                                   | ineligible for ABSTUDY if they    |
|                                   | receive other government          |
|                                   | assistance to study or complete   |
|                                   | an Australian Apprenticeship. For |
|                                   | more information, see Not         |
|                                   | qualified for ABSTUDY if          |
|                                   | receiving other assistance to     |
|                                   | study or as an Australian         |
|                                   | Apprentice.                       |
|                                   |                                   |
|                                   | Receiving a social security       |
|                                   | pension or benefit                |
|                                   |                                   |
|                                   | Students studying a full-time     |
|                                   | course that is less than 12       |
|                                   | months duration, may choose to    |
|                                   | remain on JobSeeker Payment       |
|                                   | (JSP), Youth Allowance or Austudy |
|                                   | or transfer to ABSTUDY. For       |
|                                   | further information, see          |
|                                   | Identifying the most beneficial   |
|                                   | payment for students and          |
|                                   | Australian Apprentices.           |
+-----------------------------------+-----------------------------------+
| 4                                 | Payment details + Read more ...   |
|                                   |                                   |
|                                   | ABSTUDY payees                    |
|                                   |                                   |
|                                   | There are specific payee          |
|                                   | requirements for ABSTUDY          |
|                                   | payments. See How and to whom     |
|                                   | ABSTUDY payments are made.        |
|                                   |                                   |
|                                   | Customers must provide bank       |
|                                   | account details that are held in  |
|                                   | their name.                       |
|                                   |                                   |
|                                   | Customers who have genuine        |
|                                   | barriers in providing bank        |
|                                   | account details can be granted a  |
|                                   | 28 day grace period. See Payment  |
|                                   | destination for more information. |
+-----------------------------------+-----------------------------------+
| 5                                 | Accommodation details + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The customer's accommodation and  |
|                                   | living arrangements must be       |
|                                   | provided.                         |
|                                   |                                   |
|                                   | Rent Assistance may be payable    |
|                                   | depending on the customer's       |
|                                   | circumstances. For further        |
|                                   | information, see Qualification    |
|                                   | for payment of Rent Assistance    |
|                                   | (RA).                             |
|                                   |                                   |
|                                   | Residential costs option          |
|                                   |                                   |
|                                   | Eligible students that live away  |
|                                   | from home and who are living in a |
|                                   | hostel, or residential college    |
|                                   | attached to a university may      |
|                                   | choose to apply for the           |
|                                   | Residential Costs Option. For     |
|                                   | further information, see Payment  |
|                                   | of Residential Costs Option (RCO) |
|                                   | for ABSTUDY and ABSTUDY Pensioner |
|                                   | Education Supplement (PES).       |
+-----------------------------------+-----------------------------------+
| 6                                 | Income and assets details + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Income                            |
|                                   |                                   |
|                                   | All assessable ABSTUDY customer   |
|                                   | income is taken into account for  |
|                                   | customers claiming ABSTUDY.       |
|                                   |                                   |
|                                   | Australian Apprentices need to    |
|                                   | provide details of their          |
|                                   | employment as an Apprentice under |
|                                   | Job 1. If they have a second job, |
|                                   | details should be provided under  |
|                                   | Job 2.                            |
|                                   |                                   |
|                                   | Customers may subtract any        |
|                                   | maintenance expenditure from      |
|                                   | their estimated income.           |
|                                   |                                   |
|                                   | The customer must complete either |
|                                   | of the following forms if:        |
|                                   |                                   |
|                                   | -   they have been self-employed  |
|                                   |     or a subcontractor in the     |
|                                   |     last 12 months - Business     |
|                                   |     details form (Mod F)          |
|                                   | -   they own any freehold land or |
|                                   |     buildings other than the home |
|                                   |     they live in - Real estate    |
|                                   |     details form (Mod R)          |
|                                   | -   in addition to providing      |
|                                   |     statements for superannuation |
|                                   |     or rollover funds, is         |
|                                   |     receiving a pension from      |
|                                   |     this - Income and Assets form |
|                                   |     (Mod IA)                      |
|                                   |                                   |
|                                   | The Resources page contains links |
|                                   | to these forms.                   |
|                                   |                                   |
|                                   | Assets                            |
|                                   |                                   |
|                                   | All assessable ABSTUDY customer   |
|                                   | assets are taken into account for |
|                                   | customers claiming ABSTUDY.       |
|                                   |                                   |
|                                   | Customers should not include      |
|                                   | their principle family home and   |
|                                   | any surrounding land of 2         |
|                                   | hectares or less.                 |
|                                   |                                   |
|                                   | The current market value of the   |
|                                   | assets must be used to determine  |
|                                   | its value.                        |
|                                   |                                   |
|                                   | Additional income or assets       |
|                                   |                                   |
|                                   | If the customer and/or partner    |
|                                   | earns any income from other       |
|                                   | sources or owns assets not        |
|                                   | covered by any questions in the   |
|                                   | form, an Income and Assets form   |
|                                   | (Mod IA) should be issued. The    |
|                                   | Resources page contains a link to |
|                                   | this form.                        |
+-----------------------------------+-----------------------------------+
| 7                                 | Study details + Read more ...     |
|                                   |                                   |
|                                   | The customer must provide details |
|                                   | of their study. The customer can  |
|                                   | obtain this information from      |
|                                   | their education institution.      |
|                                   | Tertiary, Masters and Doctorate   |
|                                   | level students must also provide  |
|                                   | details of all non-school study   |
|                                   | undertaken in the last 10 years.  |
|                                   |                                   |
|                                   | Secondary student unable to       |
|                                   | provide school details            |
|                                   |                                   |
|                                   | A secondary boarding facility may |
|                                   | need to conduct an interview or   |
|                                   | selection exercise prior to       |
|                                   | accepting a student. Therefore,   |
|                                   | it may not be possible at the     |
|                                   | time of claiming to confirm the   |
|                                   | school the customer will be       |
|                                   | attending. If this is the case,   |
|                                   | the claim can be accepted without |
|                                   | study details. The customer must  |
|                                   | tell the agency of their study    |
|                                   | details as soon as possible.      |
|                                   |                                   |
|                                   | Startup Year Course               |
|                                   |                                   |
|                                   | Students undertaking a Startup    |
|                                   | Year course, to be eligible for   |
|                                   | ABSTUDY Pensioner Education       |
|                                   | Supplement must:                  |
|                                   |                                   |
|                                   | -   provide evidence they are     |
|                                   |     enrolled in the course, and   |
|                                   | -   have been selected for a      |
|                                   |     STARTUP-HELP loan by their    |
|                                   |     higher education provider     |
|                                   |                                   |
|                                   | See Startup Year Courses for more |
|                                   | information.                      |

| 11                                | Living away from home + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | For ABSTUDY, living away from     |
|                                   | home means the student or         |
|                                   | apprentice is not living at the   |
|                                   | permanent family home to          |
|                                   | undertake their study or          |
|                                   | apprenticeship. This can include: |
|                                   |                                   |
|                                   | -   boarding in another person's  |
|                                   |     home, boarding at a hostel or |
|                                   |     residential college           |
|                                   | -   living alone or sharing a     |
|                                   |     house, flat or caravan with   |
|                                   |     other students or friends     |
|                                   |                                   |
|                                   | A dependant customer must be      |
|                                   | living away from home as a result |
|                                   | of 1 of the circumstances listed  |
|                                   | in this section to qualify for    |
|                                   | the away from home rate of        |
|                                   | ABSTUDY. The form advises         |
|                                   | appropriate statements, documents |
|                                   | or letters required to support    |
|                                   | this claim.                       |
|                                   |                                   |
|                                   | Parents of students living in     |
|                                   | private board accommodation       |
|                                   | during term must complete all the |
|                                   | Away from home details question   |
|                                   | set. If parents are unable to     |
|                                   | complete all the required         |
|                                   | questions, issue the ABSTUDY      |
|                                   | request for private board         |
|                                   | provider details form.            |
+-----------------------------------+-----------------------------------+
| 12                                | Independence + Read more ...      |

|                                   | independent as a result of 1 of   |

|                                   | The form advises appropriate      |
|                                   | statements, documents or letters  |
|                                   | required to support this claim.   |
|                                   | See Independence for ABSTUDY      |
|                                   | customers.                        |
|                                   |                                   |
|                                   | If the customer in under 18 years |
|                                   | of age and is applying under the  |
|                                   | Unreasonable to Live at Home      |
|                                   | criteria, an appointment with an  |
|                                   | agency social worker will be      |
|                                   | required. See ABSTUDY             |
|                                   | unreasonable to live at home.     |
|                                   |                                   |
|                                   | Where a customer does not meet    |
|                                   | the independent criteria, the     |
|                                   | parental income test must be      |
|                                   | applied regardless of receiving   |
|                                   | parental support or not.          |

|                                   | more ...                          |
|                                   |                                   |
|                                   | The customer and their partner,   |
|                                   | the customer's parent and/or      |
|                                   | their partner must sign the       |
|                                   | declaration in order for their    |
|                                   | claim to be accepted and          |
