Letters - Q888/999 example text - instalment claim required

This table contains standard text for a Q888/999 letter requesting an
instalment claim be made for Rent Assistance (RA) to be assessed.

+-----------------------------------+-----------------------------------+
| Item                              | Letter text                       |
+-----------------------------------+-----------------------------------+
| 1                                 | 'Thank you for your claim for     |
|                                   | Family Tax Benefit for the        |
|                                   | xxxx/xxxx financial year. To be   |
|                                   | paid Rent Assistance for the      |
|                                   | xxxx/xxxx financial year, you     |
|                                   | will also need to lodge a Family  |
|                                   | Tax Benefit instalment (current   |
|                                   | financial year) claim.            |
|                                   |                                   |
|                                   | You can lodge an instalment claim |
|                                   | online at                         |
|                                   | servicesaustralia.gov.au then     |
|                                   | sign in to myGov and select       |
|                                   | Centrelink, or by completing the  |
|                                   | Claim for Parental Leave Pay and  |
|                                   | Family Tax Benefit form (FA100).  |
|                                   |                                   |
|                                   | You need to do this within xx     |
|                                   | days of the date of receiving     |
|                                   | this letter or you will not be    |
|                                   | able to be paid Rent Assistance.  |
|                                   |                                   |
|                                   | This request has been made under  |
|                                   | Schedule 1, Part 5, Division 2B,  |
|                                   | Subdivision A, section 38C, A New |
|                                   | Tax System (Family Assistance)    |
|                                   | Act 1999.'                        |
+-----------------------------------+-----------------------------------+
