This page details the process for dealing with a Level 2 complaint.

Throughout this procedure the role:

-   Service Officer (APS6) refers to either a Complaints Officer or
    Personalised Services Service Officer (PSSO)
-   Complaints Team Leader refers to a Complaints Service Support
    Manager (SSM)

|                                   |                                   |
|                                   | If a customer lodges a complaint  |
|                                   | regarding the decision to         |
|                                   | restrict their service:           |
|                                   |                                   |
|                                   | -   The customer needs to adhere  |
|                                   |     to their restriction in       |
|                                   |     lodging the complaint for     |

|                                   | Manager, Customer Service.        |
