or documents held by Services Australia.

| 3                                 | A person unable to access IPS     |
|                                   | document + Read more ...          |
|                                   |                                   |
|                                   | Upon request, IPS documents not   |
|                                   | available on the Service          |
|                                   | Australia website can be made     |
|                                   | available to the person in hard   |
|                                   | copy.                             |
|                                   |                                   |
|                                   | Requests can be:                  |
|                                   |                                   |
|                                   | -   emailed to                    |
|                                   |     freedomofinf                  |
|                                   | ormation@servicesaustralia.gov.au |
|                                   | -   posted to:                    |
|                                   |                                   |
|                                   | FOI Section                       |
|                                   |                                   |
|                                   | Freedom of Information and        |
|                                   | Information Release Legal Branch  |
|                                   |                                   |
|                                   | PO Box 7820                       |
|                                   |                                   |
|                                   | Canberra Mail Centre ACT 2610     |
|                                   |                                   |
|                                   | The agency may charge a person    |
|                                   | for accessing IPS documents that  |
|                                   | have not yet been published       |
|                                   | online. The charge will be the    |
|                                   | lowest reasonable amount to       |
|                                   | recover costs incurred in         |
|                                   | producing the document. A person  |
|                                   | may request that charges be       |
|                                   | waived on the grounds of public   |
|                                   | interest or hardship.             |
+-----------------------------------+-----------------------------------+
| 4                                 | FOI requests + Read more ...      |
|                                   |                                   |
|                                   | To request access to non-personal |
|                                   | information not published under   |
|                                   | the IPS, the person must make an  |
|                                   | application in writing (by email, |
|                                   | post or in person) via:           |
|                                   |                                   |
|                                   | the Freedom of Information I want |
|                                   | to access or change document(s)   |
|                                   | (Si031), or                       |
|                                   |                                   |
|                                   | a statement or letter that:       |
|                                   |                                   |
|                                   | -   states the request is made    |
|                                   |     under the FOI Act             |
|                                   | -   provides information about    |
|                                   |     the documents                 |
|                                   | -   provides an address for reply |
|                                   |                                   |
|                                   | Requests can be:                  |
|                                   |                                   |
|                                   | -   emailed to                    |
|                                   |     freedomofinf                  |
|                                   | ormation@servicesaustralia.gov.au |
|                                   | -   posted to:                    |
|                                   |                                   |
|                                   | Services Australia                |
|                                   |                                   |
|                                   | PO Box 7820                       |
|                                   |                                   |
|                                   | Canberra BC ACT 2610              |
