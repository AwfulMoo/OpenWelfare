Disputing or requesting to delete an evaluation

Dispute and deletion process

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Dispute process + Read more ...   |
|                                   |                                   |
|                                   | Staff may dispute the outcome of  |
|                                   | a quality check if they think it  |
|                                   | is incorrect or inappropriate.    |
|                                   |                                   |
|                                   | When a staff member disagrees     |
|                                   | with the evaluation outcome, they |
|                                   | should discuss it with their      |
|                                   | immediate supervisor.             |
|                                   |                                   |
|                                   | Supervisors will then follow the  |
|                                   | escalation process in line with   |
|                                   | the business team’s dispute       |
|                                   | management process.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Requests to delete an evaluation  |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | The Quality Checker who conducted |
|                                   | the check can request deletion:   |
|                                   |                                   |
|                                   | within 20 calendar days of the    |
|                                   | date the evaluation was submitted |
|                                   |                                   |
|                                   | by completing the Request to      |
|                                   | Delete an Evaluation form. This   |
|                                   | is on the Quality Call Framework  |
|                                   | Portal                            |
|                                   |                                   |
|                                   | -   select the reason for the     |
|                                   |     deletion from the drop down   |
|                                   |     menu                          |
|                                   | -   include enough detail to      |
|                                   |     identify the evaluation       |
|                                   |                                   |
|                                   | Business teams can request        |
|                                   | deletion:                         |
|                                   |                                   |
|                                   | within 20 calendar days of the    |
|                                   | date the evaluation was submitted |
|                                   |                                   |
|                                   | by emailing the Quality Call      |
|                                   | Framework mailbox                 |
|                                   |                                   |
|                                   | for the following reasons:        |
|                                   |                                   |
|                                   | -   Outside call recording        |
|                                   |     parameters                    |
|                                   | -   Outside sampling variation    |
|                                   |     parameters                    |
|                                   | -   Quality Checker absence       |
|                                   | -   Privacy breach                |
