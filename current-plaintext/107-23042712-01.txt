This document outlines how to use the Process Direct Bulk Load Tool to
write off or waive multiple debts. Access to this tool is for authorised
users only.

For appropriately trained staff only.

Process Direct Bulk Load tool

The Bulk Load tool allows multiple debt waivers or write offs to be
actioned at once by uploading relevant Customer Reference Numbers (CRNs)
and Process Direct debt identifications.

Use the tool to:

-   add and remove keywords on Debt Management and Information System
    (DMIS) and Public Sector Collections and Disbursements (PSCD) debts
    and debtors
-   add, restart and remove the end date for write offs for DMIS and
    PSCD debts
-   create non-standard withholdings and remove withholding arrangements
    for DMIS and PSCD debts
-   apply bulk waivers to DMIS and PSCD debts, and
-   apply bulk write offs to DMIS and PSCD debts

See the Resources page for access requirements.
