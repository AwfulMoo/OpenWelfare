Holding a work item

Table 3: This table describes how to hold a work item.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Hold a work item + Read more ...  |
|                                   |                                   |
|                                   | Where a staff member determines a |
|                                   | Job Plan (CoC Interaction) cannot |
|                                   | be completed on allocation, staff |
|                                   | must follow these instructions to |
|                                   | hold the following work items     |
|                                   | correctly:                        |
|                                   |                                   |
|                                   | In Customer First, CoC            |
|                                   | Interaction Records are listed on |
|                                   | the Customer Summary screen.      |
|                                   |                                   |
|                                   | -   Select the CoC Interaction    |
|                                   |     Record by clicking on the ID  |
|                                   | -   Select Edit from the top menu |
|                                   | -   Go to the Dates tab and click |
|                                   |     on Expand                     |
|                                   | -   Find the On Hold Expiry Date  |
|                                   |     line, update the hold date by |
|                                   |     clicking on the calendar icon |
|                                   |     in the date column and select |
|                                   |     the next calendar day         |
|                                   | -   Select the activity status    |
|                                   |     and change status to On Hold  |
|                                   |     from the drop down list       |
|                                   | -   Select Save from the top menu |
|                                   |                                   |
|                                   | Note: do not place Interaction    |
|                                   | Record on hold without an expiry  |
|                                   | date. This will place the work    |
|                                   | item on Hold indefinitely,        |
|                                   | resulting in the item never       |
|                                   | reallocating.                     |

|                                   |                                   |
|                                   | The PAA Review Contact is about   |
|                                   | assessing outcomes from a job     |
|                                   | seeker's participation in an      |
|                                   | approved activity or program. It  |
|                                   | allows a Service Officer to       |
|                                   | review a job seeker's mutual      |
|                                   | obligation requirements. Job      |
|                                   | seekers who are selected to       |
|                                   | attend this appointment are       |
|                                   | required to:                      |
|                                   |                                   |
|                                   | -   attend the face-to-face       |
|                                   |     appointment as scheduled      |
|                                   | -   contact the Services          |
|                                   |     Australia to book an          |
|                                   |     appointment (where the system |
|                                   |     has been unable to            |
|                                   |     automatically book an         |
|                                   |     appointment), and             |
|                                   | -   for a Centrelink managed job  |
|                                   |     seeker, negotiate and sign a  |
|                                   |     new Job Plan                  |

|                                   | ...                               |
|                                   |                                   |
|                                   | Job seekers may be excluded from  |
|                                   | selection for the PAA Review      |
|                                   | Contact if they:                  |
|                                   |                                   |
|                                   | -   have commenced another        |
|                                   |     approved activity (for        |
|                                   |     example, Short Course (SHD))  |
|                                   |     within 28 days of ceasing to  |
|                                   |     participate in their previous |
|                                   |     activity                      |
|                                   | -   have become exempt from       |
|                                   |     mutual obligation             |
|                                   |     requirements within 28 days   |
|                                   |     of ceasing to participate in  |
|                                   |     their previous activity       |
|                                   | -   have had a follow-up Job Plan |
|                                   |     review within the previous 3  |
|                                   |     months                        |
|                                   | -   are a job seeker with a       |
|                                   |     partial capacity to work      |
|                                   |     (PCW) who has suffered the    |
|                                   |     loss or temporary incapacity  |
|                                   |     of a carer (if granted for 4  |
|                                   |     or more weeks)                |
|                                   | -   are a job seeker whose part   |
|                                   |     time work activity has been   |
|                                   |     automatically updated on the  |
|                                   |     AEX screen and their Job Plan |
|                                   |     has been updated accordingly  |

| 4                                 | System process + Read more ...    |
|                                   |                                   |
|                                   | System process once job seeker    |
|                                   | completes an activity that        |
|                                   | attracts PAA Review Contact       |
|                                   |                                   |
|                                   | -   Once the job seeker completes |
|                                   |     an approved program or        |
|                                   |     activity which attracts the   |
|                                   |     PAA Review Contact, the       |
|                                   |     system will not select the    |
|                                   |     job seeker for an appointment |
|                                   |     for at least 21 days          |
|                                   | -   The system will then check    |
|                                   |     for activity type of          |
|                                   |     Jobsearch (JSE) and searches  |
|                                   |     for an available appointment  |
|                                   |     14 to 28 days in the future   |
|                                   | -   PAA reviews will not be       |
|                                   |     created for CDP provider      |
|                                   |     managed job seekers           |
|                                   | -   If an appointment is made,    |
|                                   |     the job seeker is sent an     |
|                                   |     auto appointment notice       |
|                                   | -   If no available appointment   |
|                                   |     is found, an auto advice is   |
|                                   |     issued requesting the job     |
|                                   |     seeker to contact and make an |
|                                   |     appointment                   |
|                                   | -   If the job seeker fails to    |
|                                   |     contact to make an            |
|                                   |     appointment the job seeker's  |
|                                   |     payment will be suspended.    |
|                                   |     Automatic suspension occurs   |
|                                   |     when a PAA Review has not     |
|                                   |     been completed within 70 days |
|                                   |     from the date that the job    |
|                                   |     seeker exited the program or  |
|                                   |     activity                      |
|                                   | -   The job seeker's payment will |
|                                   |     be restorable upon completion |
|                                   |     of the PAA Review             |

| 5                                 | Appointment requirements + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | An Action Item 'Action Post       |
|                                   | Approved Activity Review' will    |
|                                   | appear on the job seeker's        |
|                                   | Participation Summary screen.     |
|                                   | Once this Action Item is          |
|                                   | selected, the Service Officer     |
|                                   | will be able to complete or       |
|                                   | cancel the PAA Review as required |
|                                   | via the Post Approved Activity    |
|                                   | Review screen, which supports the |
|                                   | process through discussion points |
|                                   | and links to other workflows      |
|                                   | which may be required.            |
|                                   |                                   |
|                                   | Actions and discussion points for |
|                                   | PAA Review appointments include:  |
|                                   |                                   |
|                                   | -   Assess outcomes from          |
|                                   |     participation in a recent     |
|                                   |     program or activity, and any  |
|                                   |     follow up action required     |
|                                   | -   Investigate whether the job   |
|                                   |     seeker's circumstances have   |
|                                   |     changed since the last review |
|                                   |     was undertaken (for example,  |
|                                   |     now homeless, have an illness |
|                                   |     or changed family             |
|                                   |     circumstances resulting in a  |
|                                   |     possible exemption for a      |
|                                   |     period)                       |
|                                   |                                   |
|                                   | Where necessary re-run the Job    |
|                                   | Seeker Classification Instrument  |
|                                   | (JSCI) and refer for an           |
|                                   | Employment Services Assessment    |
|                                   | (ESAt) as required, including     |
|                                   | where                             |
|                                   |                                   |
|                                   | there has been a change of        |
|                                   | circumstances                     |
|                                   |                                   |
|                                   | -   the QTI activity has expired, |
|                                   |     and the job seeker is likely  |
|                                   |     to be eligible for Quarterly  |
|                                   |     Participation Interview, as   |
|                                   |     it appears they still have a  |
|                                   |     partial capacity to work      |
|                                   |     (PCW) or temporary reduced    |
|                                   |     work capacity (TRWC) of less  |
|                                   |     than 15 hours per week        |
|                                   |                                   |
|                                   | Consider whether a referral is    |
|                                   | required for a Job Capacity       |
|                                   | Assessment (JCA)                  |
|                                   |                                   |
|                                   | Investigate whether all barriers  |
|                                   | to employment have been addressed |
|                                   |                                   |
|                                   | Discuss with the job seeker other |
|                                   | options in terms of further       |
|                                   | activities the job seeker could   |
|                                   | undertake to improve employment   |
|                                   | prospects                         |
|                                   |                                   |
|                                   | For Centrelink managed job        |
|                                   | seekers, review the existing Job  |
|                                   | Plan and update it as necessary   |
|                                   |                                   |
|                                   | If not already referred and it is |
|                                   | appropriate, refer the job seeker |
|                                   | to a suitable Employment Services |
|                                   | Provider. If the customer has a   |
|                                   | current and valid ESAt/JCA, this  |
|                                   | action should take into account   |
|                                   | any referral recommendations in   |
|                                   | the ESAt/JCA. If the customer has |
|                                   | already completed a period of     |
|                                   | participation in the recommended  |
|                                   | program - for example, exited     |
|                                   | from Disability Employment        |
|                                   | Services (DES) with an employment |
|                                   | or education outcome - the        |
|                                   | customer would generally not be   |
|                                   | referred back to the same program |
|                                   | on the basis of the original      |
|                                   | ESAt/JCA                          |
|                                   |                                   |
|                                   | Ensure the job seeker is engaged  |
|                                   | with their provider. If not, and  |
|                                   | no future appointments already    |
|                                   | exist, encourage the job seeker   |
|                                   | to contact their provider as soon |
|                                   | as possible to book an            |
|                                   | appointment to re-negotiate their |
|                                   | Job Plan                          |

| 6                                 | Minimum outcomes + Read more ...  |
|                                   |                                   |
|                                   | As a minimum outcome, each job    |
|                                   | seeker required to attend a       |
|                                   | face-to-face PAA Review           |
|                                   | appointment will:                 |
|                                   |                                   |
|                                   | -   receive written advice        |
|                                   |     regarding the requirement to  |
|                                   |     negotiate an Job Plan         |
|                                   | -   have their JSCI updated where |
|                                   |     necessary                     |
|                                   | -   if they are a Centrelink      |
|                                   |     managed job seeker, they will |
|                                   |     have their Job Plan reviewed. |
|                                   |     A new Job Plan should be      |
|                                   |     negotiated to reflect the     |
|                                   |     current situation and         |
|                                   |     encompass further obligations |
|                                   |     and follow-up activities as   |
|                                   |     agreed to by both the job     |
|                                   |     seeker and the Service        |
|                                   |     Officer, and                  |
|                                   | -   if not already connected and  |
|                                   |     if it is appropriate they     |
|                                   |     will be connected with a      |
|                                   |     suitable Employment Services  |
|                                   |     Provider                      |

| 7                                 | Failure to attend interview       |
|                                   | suspension processes + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | A manual suspension process will  |
|                                   | apply if selected job seekers     |
|                                   | fail to attend scheduled          |
|                                   | interviews without prior contact  |
|                                   | with Services Australia.          |
|                                   |                                   |
|                                   | If a job seeker fails to attend   |
|                                   | the interview, two genuine        |
|                                   | attempts must be made to contact  |
|                                   | them and if they cannot be        |
|                                   | contacted, their payment should   |
|                                   | be suspended with the reason      |
|                                   | 'Failed to Attend Interview'      |
|                                   | (FAI).                            |
|                                   |                                   |
|                                   | Following successful contact with |
|                                   | Services Australia, the interview |
|                                   | must be rescheduled and when the  |
|                                   | job seeker attends their          |
|                                   | appointment, payment can be       |
|                                   | restored. Service Officers must   |
|                                   | record a DOC containing reasons   |
|                                   | for non-attendance.               |
|                                   |                                   |
|                                   | A fail-safe mechanism also        |
|                                   | operates to automatically suspend |
|                                   | the payments of job seekers       |
|                                   | (suspension code 'PRO') who do    |
|                                   | not have a Job Plan negotiated    |
|                                   | within 70 days from the date they |
|                                   | became eligible for the PAA       |
|                                   | Review. A suspension letter will  |
|                                   | be issued to the job seeker.      |
