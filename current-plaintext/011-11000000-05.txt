Definitions for terms intent and consent

Table 3: This table provides definitions for the terms intent and
consent. See also: Register online for AODR (Process).

+-----------------------------------+-----------------------------------+
| Registration type                 | Definition                        |
+-----------------------------------+-----------------------------------+
| Intent                            | -   registrations without a       |
|                                   |     signature (completed online   |
|                                   |     or transferred to Medicare    |
|                                   |     from State Transport          |
|                                   |     Authorities)                  |
|                                   | -   registrations from 16 - 17    |
|                                   |     year old registrants          |
|                                   | -   Intent registration received  |
|                                   |     via previous form codes       |
+-----------------------------------+-----------------------------------+
| Consent                           | -   a completed new consent       |
|                                   |     registration form with a      |
|                                   |     signature from registrants    |
|                                   |     aged 18 and over              |
|                                   | -   confirmation and/or consent   |
|                                   |     letter with a signature from  |
|                                   |     registrants aged 18 and over  |
|                                   | -   registrations received via    |
|                                   |     Medicare online accounts in   |
|                                   |     myGov and the Express Plus    |
|                                   |     Medicare mobile app           |
+-----------------------------------+-----------------------------------+

AODR registration age restrictions

Table 4

+-----------------------------------+-----------------------------------+
| Age group                         | Restrictions                      |
+-----------------------------------+-----------------------------------+
| 16-17 years                       | Can register their intent or      |
|                                   | objection on the Donor Register,  |
|                                   | but consent will be required from |
|                                   | their families for                |
|                                   | transplantation to take place.    |
+-----------------------------------+-----------------------------------+
| 18 years and over                 | Can register their intent,        |
|                                   | consent or objection to donate    |
|                                   | organs and/or tissue for          |
|                                   | transplantation on the Donor      |
|                                   | Register.                         |
+-----------------------------------+-----------------------------------+
| Under 16 years of age             | Cannot be registered on the Donor |
|                                   | Register. Consent will be         |
|                                   | required from their families for  |
|                                   | donation of organs and/or tissue  |
|                                   | for transplantation to take       |
|                                   | place.                            |
+-----------------------------------+-----------------------------------+
