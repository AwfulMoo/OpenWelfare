Determining payment recipient

Payments to parents for ABSTUDY students under 16 years of age

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Is the student receiving ABSTUDY  |
|                                   | as an independent person? + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Note: check customer's            |
|                                   | independent status on the         |
|                                   | Independent/Homeless/Away From    |
|                                   | Home Details (NIH) screen. For    |
|                                   | more information see Independence |
|                                   | for ABSTUDY customers.            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Payment directed to independent   |
|                                   | person + Read more ...            |
|                                   |                                   |
|                                   | If independent, the payment       |
|                                   | should already be directed to the |
|                                   | student's own bank account.       |
|                                   |                                   |
|                                   | If the payment is not already     |
|                                   | going to the student, update on   |
|                                   | the Payment Destination Summary   |
|                                   | (PAS) screen.                     |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Is the student receiving ABSTUDY  |
|                                   | Schooling Allowance? + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Payment destination + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Payment of ABSTUDY is determined  |
|                                   | by the student's living           |
|                                   | arrangements. The payee can be    |
|                                   | one of the following:             |
|                                   |                                   |
|                                   | -   the parent/guardian           |
|                                   | -   the boarding school (if       |
|                                   |     requested by the parent)      |
|                                   | -   the private board provider    |
|                                   |     (if requested by the parent), |
|                                   |     or                            |
|                                   | -   the boarding hostel that is a |
|                                   |     signatory to the Standard     |
|                                   |     Hostels Agreement (if         |
|                                   |     requested by the parent)      |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Payment to go to parents + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Advise the student that as they   |
|                                   | are in receipt of ABSTUDY School  |
|                                   | Term Allowance their payments     |
|                                   | must go to the bank account of    |
|                                   | the parent(s)/guardian(s) who is  |
|                                   | the applicant for ABSTUDY.        |
+-----------------------------------+-----------------------------------+
| 6                                 | Record the decision + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Record the decision and           |
|                                   | discussion with the customer on a |
|                                   | DOC.                              |
+-----------------------------------+-----------------------------------+
| 7                                 | Nominee arrangements in effect    |
|                                   | from 1 January 2013 + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Payment and correspondence    |
|                                   |     nominees are now recognised   |
|                                   |     in Chapter 1.6 of the ABSTUDY |
|                                   |     Policy Manual                 |
|                                   | -   The guidelines, policy and    |
|                                   |     procedures for correspondence |
|                                   |     and payment nominee           |
|                                   |     arrangements for ABSTUDY are  |
|                                   |     based on those for Family     |
|                                   |     Assistance and Social         |
|                                   |     Security customers. For       |
|                                   |     further information see       |
|                                   |     Nominees                      |
+-----------------------------------+-----------------------------------+
