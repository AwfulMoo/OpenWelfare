PP040 - Customer enquiry form

mySupport

If a payment is not displaying on the customer record, processing staff
are to complete and submit the mySupport webform Medicare Online
Accounts General Enquiries issues:

Go to mySupport

Search using key words, error message or application name and select the
relevant webform

Complete the webform providing:

-   Consumer ID
-   Medicare card number
-   Medicare PIN
-   Date of missing service
-   Provider number
-   Item number service was for

Submit the required webform taking note of any receipt or reference code

Complete and Submit the required webform taking note of any receipt or
reference code produced
