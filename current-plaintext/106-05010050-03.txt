Calculating the start day for a claim made after a partner's death

This table describes the process steps to calculate the start date of a
claim, when the customer has lodged their claim after their partner has
died.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claim lodged + Read more ...      |
|                                   |                                   |
|                                   | Has the customer lodged a claim   |
|                                   | for a Social Security payment?    |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, identify if they are      |
|                                   |     eligible for income support   |
|                                   |     and other products, see       |
|                                   |     Contact in relation to an     |
|                                   |     intended claim (CLK).         |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Check marital status + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer claiming after    |
|                                   | recently becoming widowed?        |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, this process does not     |
|                                   |     apply. Procedure ends here    |
+-----------------------------------+-----------------------------------+
| 3                                 | Check timeframe of claim          |
|                                   | lodgement + Read more ...         |
|                                   |                                   |
|                                   | Did the customer make the claim   |
|                                   | within 4 weeks after the death of |
|                                   | their partner?                    |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 4                                 | Check qualification + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer qualified for the |
|                                   | payment?                          |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, this provision does not   |
|                                   |     apply, reject the claim.      |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Start day may be backdated + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If a customer makes a claim       |
|                                   | within four weeks after the death |
|                                   | of their partner, their claim may |
|                                   | be backdated to the first day     |
|                                   | they qualified for payment on or  |
|                                   | after the day of their partner's  |
|                                   | death.                            |
|                                   |                                   |
|                                   | Record the relevant start date    |
|                                   | for payment.                      |
|                                   |                                   |
|                                   | See the Resources page for an     |
|                                   | example on calculating the start  |
|                                   | date with other factors affecting |
|                                   | qualifications.                   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | The start day + Read more ...     |
|                                   |                                   |
|                                   | The start day is the day on which |
|                                   | the claim is made unless other    |
|                                   | backdating provisions apply. For  |
|                                   | more information, see the         |
|                                   | References page, in particular    |
|                                   | Policy Guideline 8.3.3.10,        |
|                                   | Backdated Start Days - General    |
|                                   | Provisions.                       |
+-----------------------------------+-----------------------------------+
