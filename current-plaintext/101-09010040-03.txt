Assessing the Business Application

Table 3: This table describes how to assess a Business Application and
associated coding where necessary.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check websites + Read more ...    |
|                                   |                                   |
|                                   | All Australian Business Numbers   |
|                                   | (ABN)/Australian Company Names    |
|                                   | (ACN) must be checked using the   |
|                                   | ABN Lookup and ASIC websites. The |
|                                   | Resources page contains a link to |
|                                   | these websites.                   |
|                                   |                                   |
|                                   | The ABN/ACN should be current and |
|                                   | the entity type should be         |
|                                   | checked.                          |
|                                   |                                   |
|                                   | If the business is:               |
|                                   |                                   |
|                                   | -   an unincorporated entity, go  |
|                                   |     to Step 2                     |
|                                   | -   a trust, go to Step 3         |
|                                   | -   a partnership, go to Step 4   |
|                                   | -   a sole trader, trustee of a   |
|                                   |     trust, company,               |
|                                   |     State/Territory or            |
|                                   |     Commonwealth Government       |
|                                   |     Organisation, go to Step 5    |
+-----------------------------------+-----------------------------------+
| 2                                 | Unincorporated entities + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Contact the business and advise   |
|                                   | that in order to enter into a     |
|                                   | contract the options are:         |
|                                   |                                   |
|                                   | -   For a member of the business  |
|                                   |     (with an ABN) to contract     |
|                                   |     with the agency (the member   |
|                                   |     signs an individual           |
|                                   |     undertaking and enters into   |
|                                   |     the contract in their         |
|                                   |     personal capacity and as a    |
|                                   |     representative of the         |
|                                   |     business)                     |
|                                   | -   For the Third Party           |
|                                   |     Organisation (TPO) to         |
|                                   |     register as an incorporated   |
|                                   |     entity                        |
|                                   | -   If there is an associated     |
|                                   |     trust, a trustee can lodge an |
|                                   |     application on behalf of the  |
|                                   |     trust                         |
|                                   | -   Having an overarching legal   |
|                                   |     entity to take                |
|                                   |     responsibility - such as an   |
|                                   |     'umbrella'                    |
|                                   |     business/organisation         |
|                                   | -   Check the Business            |
|                                   |     Application and make sure the |
|                                   |     individual who signed the     |
|                                   |     individual undertaking has    |
|                                   |     not changed and is still      |
|                                   |     liable for the business. If   |
|                                   |     the Business Application is   |
|                                   |     not signed, contact the TPO   |
|                                   |     with a request to send a      |
|                                   |     signed Business Application   |
|                                   |     or send a Request for         |
|                                   |     information (Q823) letter     |
|                                   |                                   |
|                                   | For Indigenous Businesses check   |
|                                   | the Office of the Registrar of    |
|                                   | Indigenous Corporations website.  |
|                                   | The Resources page contains a     |
|                                   | link to this website. Some        |
|                                   | Indigenous Businesses may be      |
|                                   | registered under the Corporations |
|                                   | (Aboriginal and Torres Strait     |
|                                   | Islander) Act (CATSI Act).        |
|                                   |                                   |
|                                   | If they are registered under the  |
|                                   | CATSI Act, create a Display on    |
|                                   | Access (DOA) DOC and record this. |
|                                   |                                   |
|                                   | Scan the document to the TPOs     |
|                                   | record that states they are       |
|                                   | incorporated under this Act.      |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Trusts + Read more ...            |
|                                   |                                   |
|                                   | Trustee details cannot be         |
|                                   | established from ABN Lookup or    |

|                                   | To code trustee details in        |

|                                   |     of Notification of            |
|                                   |     application outcome table     |

|                                   | If programme advice is required,  |

Finalising Business Applications for Income Management, EVoR and CCeS

Table 4: This table describes the process for finalising the
registration of Business Applications for Income Management deductions,
EVoR and CCeS.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Update service reasons for the    |
|                                   | agreement + Read more ...         |
|                                   |                                   |
|                                   | -   At the Agreement Details      |
|                                   |     (OSAS) screen, 'S'elect the   |
|                                   |     service reason to attach to a |
|                                   |     new agreement                 |
|                                   | -   To attach to an existing      |
|                                   |     agreement, select the         |
|                                   |     existing Agreement ID         |
|                                   | -   Add or ‘S’elect the service   |
|                                   |     reasons                       |
|                                   | -   Press [Enter]                 |
|                                   | -   At the Organisation Service   |
|                                   |     Agreement Details (OSAD)      |
|                                   |     screen for new agreements,    |
|                                   |     the service reason will be    |
|                                   |     attached to the first         |
|                                   |     agreement for the business    |
|                                   | -   To attach to an existing      |
|                                   |     agreement, add the new        |
|                                   |     service reason in the blank   |
|                                   |     field and change the date of  |
|                                   |     effect to 'T'oday             |
|                                   | -   Check the default details are |
|                                   |     correct and code 'I' to       |
|                                   |     insert in the Action: field   |
|                                   |     and press [Enter]             |
|                                   |                                   |
|                                   | For CCeS only (additional step)   |
|                                   |                                   |
|                                   | Businesses can only be approved   |
|                                   | to use CCeS under one of the      |
|                                   | business categories defined in    |
|                                   | the CCeS policy. Where a business |
|                                   | appears to qualify for several    |
|                                   | business categories, staff should |
|                                   | determine the most suitable one   |
|                                   | based on the main activities of   |
|                                   | the business.                     |
|                                   |                                   |
