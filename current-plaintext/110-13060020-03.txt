| 1                                 | Recording contact via Outbound    |
|                                   | Dialler (OBD) + Read more ...     |
|                                   | There are modal links to rows in  |
|                                   | this table from compliance        |
|                                   | intervention procedures.          |
|                                   |                                   |
|                                   | Before starting the conversation  |
|                                   | with the customer, the IRS        |
|                                   | intervention must be initiated    |
|                                   | and display a STA (started)       |
|                                   | status. This will make sure all   |
|                                   | updates made to the customer's    |
|                                   | record are correctly attributed   |
|                                   | to the intervention.              |
|                                   |                                   |
|                                   | If the IRS review has not been    |
|                                   | initiated, it will show:          |
|                                   |                                   |
|                                   | -   Review type: EII              |
|                                   | -   Review method: PHO or FIL     |
|                                   | -   Review status: REG            |
|                                   | -   Review origin: e.g. JBK       |
|                                   |                                   |
|                                   | To initiate the IRS review:       |
|                                   |                                   |
|                                   | Go to the Review Summary (IRRS)   |
|                                   | screen, key 'K' next to the       |
|                                   | review and press [Enter]          |
|                                   |                                   |
|                                   | The Review Contact Summary (IRCS) |
|                                   | screen displays, key:             |
|                                   |                                   |
|                                   | -   Method type: 'PHO'            |
|                                   | -   Date Contacted: 'T' for today |
|                                   | -   Press [Enter]. When the       |
|                                   |     Confirm Y/N box appears, key  |
|                                   |     'Y' and press [Enter]         |
|                                   |                                   |
|                                   | The Review Contact Details (IRCD) |
|                                   | screen displays. Key:             |
|                                   |                                   |
|                                   | -   Person contacted: customer or |
|                                   |     nominee                       |
|                                   | -   Person responded: the name of |
|                                   |     the person                    |
|                                   | -   Response initiated by: CUS    |
|                                   |     (customer) or REP (customer   |
|                                   |     representative, including     |
|                                   |     nominee)                      |
|                                   | -   Tel/Mob/Fax: the phone number |
|                                   |     of the person who contacted   |
|                                   | -   Press [Enter] to go back to   |
|                                   |     the IRCS screen               |
|                                   |                                   |
|                                   | The IRS review status will now    |
|                                   | show STA.                         |
