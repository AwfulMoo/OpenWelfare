Manager - actions required

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Feedback about staff members +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Managers get feedback about a     |
|                                   | staff member by:                  |
|                                   |                                   |
|                                   | -   warm transfer or              |
|                                   | -   through a Complaint Action    |
|                                   |     required or Complaint FYI     |
|                                   |     email sent from the Customer  |
|                                   |     Feedback Tool (CFT)           |
|                                   |                                   |
|                                   | The email provides a link to the  |
|                                   | completed service request in the  |
|                                   | CFT. Select the link to view the  |
|                                   | feedback and action taken by the  |
|                                   | staff member managing the         |
|                                   | feedback.                         |
|                                   |                                   |
|                                   | If the complaint or compliment is |
|                                   | about a phone conversation,       |
|                                   | consider requesting and listening |
|                                   | to the call recording.            |
|                                   |                                   |
|                                   | For more details:                 |
|                                   |                                   |
|                                   | -   contact the customer          |
|                                   | -   contact the Manager of the    |
|                                   |     staff member, or the staff    |
|                                   |     member who recorded the staff |
|                                   |     complaint                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Providing complaint feedback to a |
|                                   | staff member + Read more ...      |
|                                   |                                   |
|                                   | Discuss the customer's feedback   |
|                                   | with the staff member who is      |
|                                   | subject of the complaint. Allow   |
|                                   | the staff member:                 |
|                                   |                                   |
|                                   | -   to discuss what they observed |
|                                   |     during the call or visit      |
|                                   | -   to recognise any              |
|                                   |     opportunities for improvement |
|                                   |                                   |
|                                   | Consider:                         |
|                                   |                                   |
|                                   | -   how to communicate the        |
|                                   |     feedback e.g. as part of the  |
|                                   |     staff member's regular        |
|                                   |     coaching discussion           |
|                                   | -   if there is an identified     |
|                                   |     training or development need. |
|                                   |     If so, how best to give this  |
|                                   | -   listening to the call         |
|                                   |     recording together (if        |
|                                   |     available) to find out        |
|                                   |     perceptions and future        |
|                                   |     learnings                     |
|                                   |                                   |
|                                   | After giving feedback to the      |
|                                   | staff member, update the CFT      |
|                                   | record including:                 |
|                                   |                                   |
|                                   | -   feedback has been reviewed    |
|                                   | -   staff member has received     |
|                                   |     feedback. Do not include      |
|                                   |     specific details              |
|                                   | -   the date action and the       |
|                                   |     manager identifier. For       |
|                                   |     example '00/00/0000 ABC123    |
|                                   |     provided feedback to the      |
|                                   |     staff member' or '00/00/0000  |
|                                   |     ABC123 determined feedback    |
|                                   |     not required'                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Update status + Read more ...     |
|                                   |                                   |
|                                   | If feedback is provided to the    |
|                                   | staff member, check the Has TL    |
|                                   | provided feedback to staff?       |
|                                   | checkbox.                         |
|                                   |                                   |
|                                   | This checkbox is located in the   |
|                                   | Status section of the Details tab |
|                                   | in the CFT.                       |
