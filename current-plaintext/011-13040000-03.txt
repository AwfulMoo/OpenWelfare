Apply for reassessment and determination within 2 year claim again period

Table 1: This table describes the process for applying for reassessment
and determination for EBPRP within the 2 year claim again period which
commences after the claim is rejected and the customer has been asked to
obtain a letter from their health professional.

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer obtains a letter from a  |
|                                   | health professional + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | A health professional can         |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   a general practitioner        |
|                                   | -   medical specialist            |
|                                   | -   nurse                         |
|                                   | -   clinical psychologist, or     |
|                                   | -   physiotherapist involved in   |
|                                   |     the woman's treatment         |
|                                   |                                   |
|                                   | The signed letter from the health |
|                                   | professional must include:        |
|                                   |                                   |
|                                   | -   the nature of the             |
|                                   |     illness/treatment             |
|                                   | -   how this gave rise to them    |
|                                   |     requiring a subsequent        |
|                                   |     external breast prosthesis    |
|                                   |     within the 2 year claim again |
|                                   |     period                        |
|                                   | -   the health professional's     |
|                                   |     stated opinion that a         |
|                                   |     subsequent external breast    |
|                                   |     prosthesis is required        |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer sends documents to       |
|                                   | Services Australia + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the documents include a:    |
|                                   |                                   |
|                                   | -   signed claim form, marked     |
|                                   |     with 'Resubmitted'            |
|                                   | -   copy of the prosthesis        |
|                                   |     purchase                      |
|                                   | -   signed and dated letter from  |
|                                   |     customer’s health             |
|                                   |     professional in support of    |
|                                   |     their subsequent purchase     |
|                                   |     within the 2 year claim again |
|                                   |     period                        |
|                                   |                                   |
|                                   | Send the information to the EBPRP |
|                                   | Processing Team.                  |
|                                   |                                   |
|                                   |                                   |

Process application for reassessment for EBPRP within 2 year claim period

Table 2: This table describes how EBPRP Processing Team Service Officers
process an application for reassessment for EBPRP.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claim submitted + Read more ...   |
|                                   |                                   |
|                                   | Receipt claims with a letter from |
|                                   | a health professional requesting  |
|                                   | reimbursement toward the cost of  |
|                                   | a subsequent external breast      |
|                                   | prosthesis within the 2 year      |
|                                   | claim again period, in the        |
|                                   | Consumer Directory Maintenance    |
|                                   | System (CDMS) and email the daily |
|                                   | batch to the EBPRP Processing     |
|                                   | Team.                             |
|                                   |                                   |
|                                   | Note: these claims must be        |
|                                   | forwarded to the EBPRP Processing |
|                                   | Team on the same day they are     |
|                                   | received.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Receipt claim in CDMS + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the claim has come directly to |
|                                   | the EBPRP Processing Team,        |
|                                   | receipt in CDMS:                  |
|                                   |                                   |
|                                   | -   check customer restrictions   |
|                                   |     in CDMS when verifying the    |
|                                   |     Medicare card                 |
|                                   | -   follow the procedure relevant |
|                                   |     to the message displayed      |
|                                   |                                   |
|                                   | If the claim has already been     |
|                                   | receipted in CDMS by Services     |
|                                   | Australia Service Officers, there |
|                                   | is no need to receipt the claim   |
|                                   | twice. See Validate claim form.   |
+-----------------------------------+-----------------------------------+
| 3                                 | Check information + Read more ... |
|                                   |                                   |
|                                   | Check that all the information is |
|                                   | attached to the requests,         |
|                                   | including the:                    |
|                                   |                                   |
|                                   | completed claim form              |
|                                   |                                   |
|                                   | letter from the health            |
|                                   | professional, and                 |
|                                   |                                   |
|                                   | invoice/receipt from the          |
|                                   | supplier/retailer showing:        |
|                                   |                                   |
|                                   | -   the prosthesis has been fully |
|                                   |     paid                          |
|                                   | -   a description of the external |
|                                   |     breast prosthesis             |
|                                   | -   the date of purchase          |
+-----------------------------------+-----------------------------------+
| 4                                 | Missing information + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If any information is missing,    |
|                                   | contact the customer and:         |
|                                   |                                   |
|                                   | confirm identity                  |
|                                   |                                   |
|                                   | request the missing information   |
|                                   |                                   |
|                                   | tell them to supply the           |
|                                   | information:                      |
|                                   |                                   |
|                                   | -   by email                      |
|                                   | -   by post                       |
|                                   | -   at any service centre         |
|                                   |                                   |
|                                   | All the details are on the form.  |
|                                   |                                   |
|                                   | When all missing information has  |
|                                   | been supplied, go to Step 5.      |
+-----------------------------------+-----------------------------------+
| 5                                 | Send all information to Program   |
|                                   | Management + Read more ...        |
|                                   |                                   |
|                                   | Scan and email the claim form and |
|                                   | all supporting documentation to   |
|                                   | EBPRP Program Management.         |
|                                   |                                   |
|                                   | Note: Program Management then     |
|                                   | assess the discretionary claim.   |
+-----------------------------------+-----------------------------------+

Reassessment determinations

Table 4: This table describes the process of reassessment determinations
as completed by Service Officers and Program Management.

+-----------------------+-----------------------+-----------------------+
| Stage                 | Who                   | Description           |
+-----------------------+-----------------------+-----------------------+
| 1                     | Program Management    | Notification of       |
|                       |                       | decision + Read more  |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | If the claim is       |
|                       |                       | Approved, EBPRP       |
|                       |                       | Program Management    |
|                       |                       | send an email to      |
|                       |                       | EBPRP Processing Team |
|                       |                       | notifying them that   |
|                       |                       | the claim has been    |
|                       |                       | approved and the      |
|                       |                       | customer can be       |
|                       |                       | reimbursed.           |
|                       |                       |                       |
|                       |                       | Note: the reference   |
|                       |                       | number must be        |
|                       |                       | included in the       |
|                       |                       | email.                |
|                       |                       |                       |
|                       |                       | If the claim is       |
|                       |                       | Rejected, EBPRP       |
|                       |                       | Program Management    |
|                       |                       | send an email to      |
|                       |                       | EBPRP Processing Team |
|                       |                       | notifying them that   |
|                       |                       | the claim has been    |
|                       |                       | rejected:             |
|                       |                       |                       |
|                       |                       | -   the email needs   |
|                       |                       |     to include the    |
|                       |                       |     reference number  |
|                       |                       |     and clearly state |
|                       |                       |     the reason for    |
|                       |                       |     rejection         |
|                       |                       | -   EBPRP Program     |
|                       |                       |     Management will   |
|                       |                       |     issue a letter of |
|                       |                       |     rejection to the  |
|                       |                       |     customer          |
|                       |                       |     explaining why    |
|                       |                       |     the discretionary |
|                       |                       |     claim was         |
|                       |                       |     rejected as well  |
|                       |                       |     as their options  |
|                       |                       |     to proceed        |
|                       |                       |     further and       |
|                       |                       |     re-submit another |
|                       |                       |     discretionary     |
|                       |                       |     claim             |
+-----------------------+-----------------------+-----------------------+
| 2                     | EBPRP Processing Team | Discretionary claim   |
|                       |                       | approved + Read more  |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | To process a          |
|                       |                       | discretionary claim   |
|                       |                       | in the EBPRP system,  |
|                       |                       | a payment override is |
|                       |                       | required. See Table 7 |
|                       |                       | in Processing claims  |
|                       |                       | for External Breast   |
|                       |                       | Prostheses            |
|                       |                       | Reimbursement Program |
|                       |                       | (EBPRP).              |
+-----------------------+-----------------------+-----------------------+
| 3                     | EBPRP Processing Team | Discretionary claim   |
|                       |                       | rejected + Read more  |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | If the discretionary  |
|                       |                       | claim is rejected,    |
|                       |                       | update the EBPRP      |
|                       |                       | claim screen stating  |
|                       |                       | the discretionary     |
|                       |                       | request has been      |
|                       |                       | rejected.             |
+-----------------------+-----------------------+-----------------------+
| 4                     | Program Management    | Letter of rejection + |
|                       |                       | Read more ...         |
|                       |                       |                       |
|                       |                       | Issue a letter of     |
|                       |                       | rejection to the      |
|                       |                       | customer and merge    |
|                       |                       | all the relevant      |
|                       |                       | discretionary claim   |
|                       |                       | documentation         |
|                       |                       | together and file:    |
|                       |                       |                       |
|                       |                       | -   EBPRP Program     |
|                       |                       |     Management will   |
|                       |                       |     send a copy of    |
|                       |                       |     the reject letter |
|                       |                       |     to the EBPRP      |
|                       |                       |     Processing Team   |
|                       |                       |     to attach to the  |
|                       |                       |     customer’s file   |
|                       |                       |     for future        |
|                       |                       |     reference         |
|                       |                       | -   EBPRP Processing  |
|                       |                       |     Team do not issue |
|                       |                       |     letters of        |
|                       |                       |     rejection to      |
|                       |                       |     customers for     |
|                       |                       |     discretionary     |
|                       |                       |     requests          |
+-----------------------+-----------------------+-----------------------+
| 5                     | Program Management    | Update register with  |
|                       |                       | decision + Read more  |
|                       |                       | ...                   |
|                       |                       |                       |
|                       |                       | Recording the         |
|                       |                       | decision to approve   |
|                       |                       | or reject the claim   |
|                       |                       | on the discretionary  |
|                       |                       | request spreadsheet.  |
+-----------------------+-----------------------+-----------------------+

Pre-approval of discretionary request only

Table 5: This table describes the process when a discretionary request
is granted for pre-approval.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notification + Read more ...      |
|                                   |                                   |
|                                   | EBPRP Program Management will     |
|                                   | send an email to the EBPRP        |
|                                   | Processing Team stating that the  |
|                                   | pre-approval request has been     |
|                                   | approved. This pre-approval       |
|                                   | enables a customer to proceed in  |
|                                   | making their prosthesis purchase. |
+-----------------------------------+-----------------------------------+
| 2                                 | Record reference number + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the email for the           |
|                                   | pre-approval reference number and |
|                                   | record this in the EBPRP Master   |
|                                   | Tracking spreadsheet.             |
+-----------------------------------+-----------------------------------+
| 3                                 | EBPRP Processing Team contact     |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | EBPRP Processing Team contacts    |
|                                   | the customer via phone, confirms  |
|                                   | identity and tells them they are  |
|                                   | pre-approved to make the breast   |
|                                   | prostheses purchase.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Note pre-approval + Read more ... |
|                                   |                                   |
|                                   | In the EBPRP claim history        |
|                                   | screen, document that             |
|                                   | pre-approval has been granted and |
|                                   | include the date accepted. This   |
|                                   | note will be beneficial when      |
|                                   | referencing any future claiming   |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
