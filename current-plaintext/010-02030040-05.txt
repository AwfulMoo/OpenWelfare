List of known cadetships and effect on ABSTUDY payments

Escalate an enquiry to the Level 2 Policy Helpdesk to include a
cadetship in the listing.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Indigenous Cadetship Support      |
|                                   | Programs                          |
|                                   |                                   |
|                                   | Indigenous Cadetship Support      |
|                                   | Programs are funded by the        |
|                                   | Australian Government. Employers  |
|                                   | that offer these types of         |
|                                   | cadetships are CSIRO and AFL      |
|                                   | sport Ready Ltd.                  |
|                                   |                                   |
|                                   | These programmes includes         |
|                                   | assistance such as a:             |
|                                   |                                   |
|                                   | -   a study allowance per week    |
|                                   |     during periods of full-time   |
|                                   |     study                         |
|                                   | -   financial assistance of per   |
|                                   |     semester to assist with study |
|                                   |     costs                         |
|                                   | -   12 week paid work placement   |
|                                   |     during each year of the       |
|                                   |     cadetship                     |
|                                   | -   payment of HECS fees by CSIRO |
|                                   |     on successful completion of   |
|                                   |     study                         |
|                                   |                                   |
|                                   | Customers in receipt of these     |
|                                   | cadetships are eligible for       |
|                                   | ABSTUDY.                          |
|                                   |                                   |
|                                   | Assess study allowance and income |
|                                   | from the paid placement under the |
|                                   | ABSTUDY personal income test.     |
|                                   |                                   |
|                                   | Treat the financial assistance    |
|                                   | paid per semester for study costs |
|                                   | as exempt income.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Indigenous Cadetship Scholarship  |
|                                   | (ICS)                             |
|                                   |                                   |
|                                   | Indigenous Cadetship Scholarships |
|                                   | (ICS) is a Commonwealth           |
|                                   | Government funded program         |
|                                   | provided by the Department of the |
|                                   | Prime Minister and Cabinet        |
|                                   | (PM&C). ICS aims to provide       |
|                                   | financial support for Indigenous  |
|                                   | university students who are       |
|                                   | undertaking an internship during  |
|                                   | their studies to gain             |
|                                   | professional qualifications and   |
|                                   | experience and move into          |
|                                   | employment on completion of their |
|                                   | studies.                          |
|                                   |                                   |
|                                   | ICS provides financial support in |
|                                   | the form of fortnightly payments  |
|                                   | during the academic calendar year |
|                                   | to assist in covering living      |
|                                   | expenses.                         |
|                                   |                                   |
|                                   | Customers in receipt of an ICS    |
|                                   | are not eligible for ABSTUDY.     |
+-----------------------------------+-----------------------------------+
| 3                                 | CareerTrackers internship program |
|                                   |                                   |
|                                   | CareerTrackers links Aboriginal   |
|                                   | and Torres Strait Islander        |
|                                   | students with employers related   |
|                                   | to their aspirations for paid,    |
|                                   | multi-year internships. It        |
|                                   | includes a 12-week paid           |
|                                   | internship each year throughout   |
|                                   | the student’s degree.             |
|                                   |                                   |
|                                   | Customers undertaking a           |
|                                   | CareerTrackers internship program |
|                                   | who do not receive an Indigenous  |
|                                   | Cadetship Scholarship are         |
|                                   | eligible for ABSTUDY. Assess the  |
|                                   | student’s income from the 12 week |
|                                   | paid internship under the ABSTUDY |
|                                   | Personal Income Test.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Tailored Assistance Employment    |
|                                   | Grants (TAEG)                     |
|                                   |                                   |
|                                   | Grants approved under the TAEG    |
|                                   | program, are awarded to           |
|                                   | organisations providing support   |
|                                   | to Indigenous Australians, rather |
|                                   | than individuals.                 |
|                                   |                                   |
|                                   | This means that the individual    |
|                                   | may not see any of the funding    |
|                                   | for the milestones that the       |
|                                   | business receives.                |
|                                   |                                   |
|                                   | A TAEG does not exclude a         |
|                                   | customer from ABSTUDY, however    |
|                                   | assess any income received under  |
|                                   | the ABSTUDY personal income test. |
+-----------------------------------+-----------------------------------+
| 5                                 | NSW Health Aboriginal Nursing and |
|                                   | Midwifery Cadetship Program       |
|                                   |                                   |
|                                   | The NSW Aboriginal Nursing and    |
|                                   | Midwifery Cadetship Program       |
|                                   | provides support and assistance   |
|                                   | to Aboriginal people studying an  |
|                                   | undergraduate nursing or          |
|                                   | midwifery degree at university.   |
|                                   | Student are required to do 12     |
|                                   | weeks paid work placement in a    |
|                                   | local public hospital or Justice  |
|                                   | Health facility.                  |
|                                   |                                   |
|                                   | The students are not regarded as  |
|                                   | an employee of a government       |
|                                   | agency while receiving this       |
|                                   | Cadetship. Therefore students are |
|                                   | eligible for ABSTUDY while        |
|                                   | receiving this cadetship.         |
|                                   |                                   |
|                                   | Assess study allowance and income |
|                                   | from the paid placement under the |
|                                   | ABSTUDY Personal Income Test.     |
