Payment under International Agreement

Cancellation of payment under former resident provisions

This table describes information relating to former residents whose
pension has been cancelled who may qualify for payment under an
International Agreement.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Cancellation of payment under     |
|                                   | former resident provisions + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers whose payment is to be  |
|                                   | cancelled under former resident   |
|                                   | provisions may be able to         |
|                                   | transfer to an international      |
|                                   | agreement if going to an          |
|                                   | agreement country provided they   |
|                                   | are qualified and able to meet    |
|                                   | the scope and lodgement           |
|                                   | requirements of the Agreement.    |
|                                   | For detailed information on each  |
|                                   | individual agreement, refer to    |
|                                   | the relevant country file in      |
|                                   | International Social Security     |
|                                   | Agreements.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Payments made under International |
|                                   | agreements + Read more ...        |
|                                   |                                   |
|                                   | Payments made under an            |
|                                   | international agreement are not   |
|                                   | only affected by income and       |
|                                   | assets but are also paid a        |
|                                   | special rate determined by the    |
|                                   | relevant agreement. The rate paid |
|                                   | in most cases is a proportional   |
|                                   | rate based on their Australian    |
|                                   | Working Life Residence (WLR). The |
|                                   | New Zealand Agreement however,    |
|                                   | has different rules. Under the    |
|                                   | New Zealand Agreement the rate    |
|                                   | paid will be based on Working Age |
|                                   | Residence (WAR).See Australian    |
|                                   | pensions paid outside Australia,  |
|                                   | the proportional rate and the     |
|                                   | relevant country file in          |
|                                   | International Social Security     |
|                                   | Agreements.                       |
|                                   |                                   |
|                                   | Note: Customers paid using the    |
|                                   | agreements with Greece or         |
|                                   | Republic of North Macedonia may   |
|                                   | also need to use a different      |
|                                   | working life residence            |
|                                   | calculation. For details, see:    |
|                                   |                                   |
|                                   | -   Greece Agreement and foreign  |
|                                   |     pension information, or       |
|                                   | -   North Macedonia Agreement and |
|                                   |     foreign pension information   |
|                                   |                                   |
|                                   | Possible Agreement cases are to   |
|                                   | be referred to International      |
|                                   | Services (CIS). Contact details   |
|                                   | are available in C useful contact |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
