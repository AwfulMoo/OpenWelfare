Process in assessing Crisis Payment (CrP) due to extreme circumstances family and domestic violence

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claim due to extreme              |
|                                   | circumstances + Read more ...     |
|                                   |                                   |
|                                   | Note: if claim is for Crisis      |
|                                   | Payment (CrP) for Other Extreme   |
|                                   | Circumstance (EXO), see New claim |
|                                   | for Crisis Payment Other Extreme  |
|                                   | Circumstances.                    |
|                                   |                                   |
|                                   | Customer claiming Crisis Payment  |
|                                   | (CrP) for extreme circumstances   |
|                                   | family and domestic violence      |
|                                   | (EXV) has been referred to the    |
|                                   | social worker.                    |
|                                   |                                   |
|                                   | If possible, the Service Officer  |
|                                   | who referred the customer should  |
|                                   | check:                            |
|                                   |                                   |
|                                   | -   the common criteria for CrP   |
|                                   |     are met                       |
|                                   | -   the customer is in severe     |
|                                   |     financial hardship            |
|                                   | -   the customer was in Australia |
|                                   |     at the time of the extreme    |
|                                   |     circumstance                  |
|                                   | -   no more than four CrP EXV/EXO |
|                                   |     have been paid to them within |
|                                   |     the previous 12 month period  |
|                                   | -   they have not intentionally   |
|                                   |     brought about the extreme     |
|                                   |     circumstance to obtain a CrP  |
|                                   |                                   |
|                                   | If the customer has attended a    |
|                                   | service centre, the service       |
|                                   | officer should warm hand off the  |
|                                   | customer to the social worker.    |
|                                   |                                   |
|                                   | For online or Assisted Customer   |
|                                   | Claim (ACC) CrP claims or if the  |
|                                   | customer has completed the SU510, |
|                                   | the Service Officer should tell   |
|                                   | the customer to upload all        |
|                                   | available verification to their   |
|                                   | record immediately if it is safe  |
|                                   | to do so. If the customer is      |
|                                   | unable to upload verification,    |
|                                   | the social worker can obtain      |
|                                   | verbal verification of the        |
|                                   | customer's circumstances from     |
|                                   | third parties, using verbal       |
|                                   | declarations and/or make a        |
|                                   | professional assessment based on  |
|                                   | the information available.        |
|                                   |                                   |
|                                   | For all claims for CrP relating   |
|                                   | to family and domestic violence,  |
|                                   | the social worker needs to:       |
|                                   |                                   |
|                                   | -   review the details within the |
|                                   |     claim and, if possible        |
|                                   |     discuss what has happened to  |
|                                   |     determine if it meets the     |
|                                   |     definition of family and      |
|                                   |     domestic violence             |
|                                   | -   determine if the customer     |
|                                   |     made contact within seven     |
|                                   |     days after the extreme        |
|                                   |     circumstance occurred         |
|                                   | -   check that everyone is now    |
|                                   |     safe, and                     |
|                                   | -   provide immediate support and |
|                                   |     counselling to the customer,  |
|                                   |     and refer for appropriate     |
|                                   |     community support and         |
|                                   |     services, depending on the    |
|                                   |     person's circumstances        |
|                                   |                                   |
|                                   | Note: if the customer requests no |
|                                   | social work contact, a social     |
|                                   | work referral and social work     |
|                                   | contact is still needed for the   |
|                                   | purposes of the assessment. When  |
|                                   | contacting the customer, the      |
|                                   | social worker must confirm if it  |
|                                   | is safe for the customer to       |
|                                   | discuss their circumstances.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine if a claim has been     |
|                                   | lodged + Read more ...            |
|                                   |                                   |
|                                   | A CrP claim is considered lodged  |
|                                   | if:                               |
|                                   |                                   |
|                                   | -   the customer submits an       |
|                                   |     online claim                  |
|                                   | -   an Assisted Customer Claim    |

|                                   | possible outcomes to the claim    |
|                                   | are:                              |
|                                   |                                   |
|                                   | -   granted                       |
|                                   | -   rejected, or                  |
|                                   | -   not pursued                   |
|                                   |                                   |
|                                   | Once the claim is lodged, but     |
|                                   | before the claim determination,   |
|                                   | the customer can withdraw the     |
|                                   | claim.                            |
|                                   |                                   |
|                                   | If it is determined during the    |
|                                   | social work interview that the    |
|                                   | CrP EXV claim is to be withdrawn  |
|                                   | as the customer requests to       |
|                                   | withdraw their claim, the social  |
|                                   | worker must:                      |
|                                   |                                   |
|                                   | -   complete the Crisis payment   |
|                                   |     EXV Claim Withdrawn Fast      |
|                                   |     Note, answering all fields    |
|                                   |     and make sure the Fast Note   |
|                                   |     is closed                     |
|                                   | -   complete the social work      |
|                                   |     referral work item as this    |
|                                   |     will allocate the claim to a  |
|                                   |     Crisis Payment skill tagged   |
|                                   |     Service Officer for           |
|                                   |     processing                    |
|                                   | -   tell the customer they have   |
|                                   |     completed a request to        |
|                                   |     withdraw their claim, and     |
|                                   |     once processed, they will not |
|                                   |     have review and appeal rights |
|                                   |     as no social work             |
|                                   |     determination was made        |
|                                   |                                   |
|                                   | The claim is considered 'not      |
|                                   | pursued' for SWIS purposes.       |
|                                   | Clearly DOC this on the customers |
|                                   | record.                           |
|                                   |                                   |
|                                   | See New claim for Crisis Payment  |
|                                   | under extreme circumstances       |
|                                   | family and domestic violence,     |
|                                   | Resources page, for more          |
|                                   | information.                      |
|                                   |                                   |
|                                   | No current claim for CrP          |
|                                   |                                   |
|                                   | 'Crisis Payment Enquiry' is used  |
|                                   | if there is no current claim for  |
|                                   | Crisis Payment on PD at the time  |
|                                   | of the social work assessment.    |
|                                   |                                   |
|                                   | Either:                           |
|                                   |                                   |
|                                   | -   In discussions with the       |
|                                   |     social worker the customer    |
|                                   |     advises they not want to      |
|                                   |     proceed with a Crisis Payment |
|                                   |     claim, or                     |
|                                   | -   The social worker was unable  |
|                                   |     to make contact with the      |
|                                   |     customer. This will occur in  |
|                                   |     situations where a Service    |
|                                   |     Officer (by SW Line or WLM)   |
|                                   |     has referred the customer to  |
|                                   |     a social worker and no online |
|                                   |     claim or ACC has been         |
|                                   |     completed prior. As there is  |
|                                   |     no claim lodged, no decision  |
|                                   |     can be made. The customers    |
|                                   |     contact is considered a       |
|                                   |     'Crisis Payment Enquiry'      |
|                                   |     only.                         |
