Personal call received for staff member

This table describes the steps to follow if a personal call is received
for a staff member.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Call received + Read more ...     |
|                                   |                                   |
|                                   | A personal call for another       |
|                                   | member of staff is received.      |
+-----------------------------------+-----------------------------------+
| 2                                 | Obtain information from caller +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Ask the following:                |
|                                   |                                   |
|                                   | -   Caller's name                 |
|                                   | -   Name of the staff member they |
|                                   |     are trying to contact         |
|                                   | -   Where the staff member works  |
|                                   | -   Why the caller is ringing     |
|                                   | -   What is the staff member's    |
|                                   |     phone number                  |
|                                   |                                   |
|                                   | Note: do not acknowledge that the |
|                                   | person is a Services Australia    |
|                                   | staff member or the office where  |
|                                   | they work.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Assess urgency + Read more ...    |
|                                   |                                   |
|                                   | Is the call of an urgent nature   |
|                                   | (e.g. hospital, doctor, school,   |
|                                   | etc.) or another appropriate      |
|                                   | reason?                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Advise caller + Read more ...     |
|                                   |                                   |
|                                   | The caller should be advised to   |
|                                   | contact the staff member after    |
|                                   | hours at home.                    |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Staff member available + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Attempt to locate the staff       |
|                                   | member and advise them who is     |
|                                   | calling.                          |
|                                   |                                   |
|                                   | If the staff member is in another |
|                                   | Smart Centre call site, please    |
|                                   | locate the persons team leader    |
|                                   | (via the Oganization tab in Lync) |
|                                   | and pass the message along to     |
|                                   | them. They will take the call and |
|                                   | follow up with the staff member.  |
|                                   |                                   |
|                                   | If the staff member does not wish |
|                                   | to accept the call, the caller    |
|                                   | should be advised that they are   |
|                                   | unable to be contacted.           |
|                                   |                                   |
|                                   | Under no circumstances should the |
|                                   | call be transferred unannounced.  |
+-----------------------------------+-----------------------------------+
| 6                                 | Staff member not available + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the staff member is not        |
|                                   | available, take the contact       |
|                                   | details of the caller and advise  |
|                                   | that if the staff member can be   |
|                                   | located, their message will be    |
|                                   | passed on as soon as possible.    |
|                                   |                                   |
|                                   | The caller's details are to be    |
|                                   | passed onto the local team leader |
|                                   | who will continue to try and      |
|                                   | locate the staff member.          |
+-----------------------------------+-----------------------------------+
