| 7                                 | Confirm supporting evidence is    |
|                                   | current + Read more ...           |
|                                   |                                   |
|                                   | Any evidence provided must not be |
|                                   | more than 6 months old from the   |
|                                   | commencement date of a            |
|                                   | determination.                    |
|                                   |                                   |
|                                   | The six month period should be    |
|                                   | looked at in terms of months      |
|                                   | rather than specific dates. For   |
|                                   | example, if a determination       |
|                                   | application period commences      |
|                                   | anytime in November 2021, it is   |
|                                   | reasonable to accept any evidence |
|                                   | dated May 2021 and onwards.       |
|                                   |                                   |
|                                   | Evidence could be dated 6 months  |
|                                   | and 1 day up to 6 months and 31   |
|                                   | days.                             |
|                                   |                                   |
|                                   | If the document is a Word         |
|                                   | document incorrectly displaying   |
|                                   | today’s date, check the currency  |
|                                   | of the evidence by selecting File |
|                                   | and confirming the 'last          |
|                                   | modified' date.                   |
|                                   |                                   |
|                                   | The                               |
|                                   |                                   |
|                                   | Resources page has further        |
|                                   | information and scenarios on      |
|                                   | currency of evidence.             |
|                                   |                                   |

| 11                                | Confirm supporting evidence child |
|                                   | is or was at risk + Read more ... |
|                                   |                                   |
|                                   | Confirm supporting evidence       |
|                                   | contains sufficient information   |
|                                   | the child is or was taken to be   |
|                                   | at risk of serious abuse or       |
|                                   | neglect.                          |
|                                   |                                   |
|                                   | A child is taken to be at risk of |
|                                   | serious abuse or neglect if the   |
|                                   | child is at risk of suffering     |
|                                   | harm as a result of being subject |
|                                   | to, or exposed to one or more of  |
|                                   | the following events:             |
|                                   |                                   |
|                                   | -   serious physically, emotional |
|                                   |     or psychological abuse        |
|                                   | -   sexual abuse                  |
|                                   | -   domestic or family violence   |
|                                   | -   neglect                       |
|                                   | -   Harm, means any detriment to  |
|                                   |     the child wellbeing           |
|                                   | -   Neglect, means a failure to   |
|                                   |     be provided with the basic    |
|                                   |     needs essential for the       |
|                                   |     child’s physical and          |
|                                   |     emotional wellbeing           |
|                                   |                                   |
|                                   | The evidence provided should:     |
|                                   |                                   |
|                                   | -   focus on the presence of risk |
|                                   |     of serious abuse or neglect   |
|                                   |     (or evidence of actual abuse  |
|                                   |     or neglect). For example by   |
|                                   |     referencing indicators of     |
|                                   |     abuse or neglect or risk      |
|                                   |     factors                       |
|                                   | -   describe the child's          |
|                                   |     circumstances in a way they   |
|                                   |     can be clearly linked to the  |
|                                   |     definition of 'at risk'       |
|                                   | -   explain how the circumstances |
|                                   |     of the family are linked to   |
|                                   |     the risk to the child (where  |
|                                   |     applicable)                   |
|                                   | -   highlight how the family's    |
|                                   |     issues are a barrier to the   |
|                                   |     family caring appropriately   |
|                                   |     for the child thereby leaving |
|                                   |     the child 'at risk'           |
|                                   |                                   |
|                                   | There should be enough            |
|                                   | information to determine the      |
|                                   | child is currently suffering harm |
|                                   | from any of these events,         |
|                                   | regardless of if:                 |
|                                   |                                   |
|                                   | -   they occurred in the past     |
|                                   | -   are currently happening, or   |
|                                   | -   they are likely to occur in   |
|                                   |     the future?                   |
|                                   |                                   |
|                                   | There are certain circumstances   |
|                                   | that, on their own, do not mean a |
|                                   | child should be taken to be at    |
|                                   | serious risk of abuse or neglect. |
|                                   | These include:                    |
|                                   |                                   |
|                                   | -   the income of the individual  |
|                                   |     or individuals who have care  |
|                                   |     of the child                  |
|                                   | -   the ethnic, cultural,         |
|                                   |     religious or racial           |
|                                   |     background of the child or    |
|                                   |     the child's immediate family  |
|                                   | -   the geographical location in  |
|                                   |     which the child and the       |
|                                   |     child's immediate family      |
|                                   |     resides                       |
|                                   | -   the child's place of          |
|                                   |     residence is statistically an |
|                                   |     area of socio-economic        |
|                                   |     disadvantage                  |
|                                   | -   the child is likely to        |
|                                   |     benefit from early childhood  |
|                                   |     education and care programs   |
|                                   | -   the child has a disability,   |
|                                   |     or                            |
|                                   | -   the child is in a (informal)  |
|                                   |     foster care or kinship care   |
|                                   |     arrangement                   |
|                                   |                                   |
|                                   | The                               |
|                                   |                                   |
|                                   | Resources page has detailed       |
|                                   | information and scenarios about   |
|                                   | ACCS (child wellbeing) evidence   |
|                                   | requirements.                     |
|                                   |                                   |
|                                   | Note: historical evidence, which  |
|                                   | is dated within the last 6 months |
|                                   | can be viewed/used where:         |
|                                   |                                   |
|                                   | -   the determination being       |
|                                   |     assessed is going to be       |

|                                   | -   the service has submitted an  |
|                                   |     appeal for a rejected ACCS    |
|                                   |     (child wellbeing)             |
|                                   |     determination                 |
|                                   |                                   |
|                                   | Does the evidence contain enough  |

|                                   |                                   |
