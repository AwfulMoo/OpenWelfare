Questions to ask approved supplier

Questions the approved supplier must answer to receive Centrelink data

This table lists questions the approved supplier must answer to receive
Centrelink data.

+-----------------------------------+-----------------------------------+
| Number                            | Questions and comments            |
+-----------------------------------+-----------------------------------+
| 1                                 | What is your name?                |
+-----------------------------------+-----------------------------------+
| 2                                 | What is your pharmacy approval    |
|                                   | number?                           |
+-----------------------------------+-----------------------------------+
| 3                                 | What is the name of the pharmacy  |
|                                   | or surgery for dispensing         |
|                                   | doctors? + Read more ...          |
|                                   |                                   |
|                                   | Cross reference this name with    |
|                                   | the pharmacy approval             |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Do you have the patient's         |
|                                   | permission to obtain and store    |
|                                   | the number? + Read more ...       |
|                                   |                                   |
|                                   | Yes is required. The patient or   |
|                                   | agent must give consent before    |
|                                   | Centrelink data is released.      |
+-----------------------------------+-----------------------------------+
| 5                                 | What is the patient's Medicare    |
|                                   | number, or name and address?      |
+-----------------------------------+-----------------------------------+
| 6                                 | (Optional) What is the patient's  |
|                                   | date of birth? + Read more ...    |
|                                   |                                   |
|                                   | If the name and address provided  |
|                                   | does not uniquely identify the    |
|                                   | patient, ask for the patient's    |
|                                   | date of birth.                    |
+-----------------------------------+-----------------------------------+
