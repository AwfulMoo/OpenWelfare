This page contains information on how to find, create and update
customer records. It also provides the processes for adding and deleting
passwords and setting a customer record to Restricted Access Customers
(RACS).

On this page:

Find customer record

Create a new customer record

Update individual customer details

Add or update a password

Add Restricted Access Customers (RACS)

Table 2: this table describes how to create a new customer record in

| Step                              | Action                            |

|                                   |                                   |
|                                   | -   enter the Tax File Number     |
|                                   |     (TFN) in the TFN: field, if   |
|                                   |     known                         |
|                                   | -   enter the Customer Reference  |
|                                   |     Number (CRN) in the CRN:      |
|                                   |     field, if known               |
|                                   | -   enter any foreign reference   |
|                                   |     numbers in the OS ID: field,  |
|                                   |     if known                      |

| 4                                 | Update customer name details +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In the Name group box:            |
|                                   |                                   |
|                                   | -   leave the Type as the default |
|                                   |     selection Legal Name          |

|                                   |     This is not a mandatory field |

|                                   | errors in data matching.          |

|                                   | birth and marital status + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In the Other Details > Gender     |
|                                   | group box:                        |
|                                   |                                   |
|                                   | -   select the appropriate Male,  |
|                                   |     Female or Unknown radio       |
|                                   |     button                        |
|                                   |                                   |
|                                   | Note: currently there is no       |
|                                   | non-binary (X) field available    |
|                                   | for customers in Cuba. Where a    |

|                                   |                                   |

|                                   | details + Read more ...           |
|                                   |                                   |

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - Update individual   |
|                                   | customer details + Read more ...  |
|                                   |                                   |

|                                   | more ...                          |
|                                   |                                   |
