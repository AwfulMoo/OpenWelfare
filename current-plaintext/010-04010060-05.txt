FAQs - generic online claims

Table 1

+-----------------------------------+-----------------------------------+
| Generic                           | Question and Answer               |
+-----------------------------------+-----------------------------------+
| Customer has a partner            | Will a MOD P be requested when a  |
|                                   | customer is identified as having  |
|                                   | a partner?                        |
|                                   |                                   |
|                                   | If the customer has indicated in  |
|                                   | their online claim they can       |
|                                   | accurately provide their          |
|                                   | partner's information, a MOD P is |
|                                   | not requested (regardless of the  |
|                                   | partner claiming/receiving        |
|                                   | payment or not). Both the         |
|                                   | customer's and partner's          |
|                                   | information will be collected in  |
|                                   | the customer's claim.             |
+-----------------------------------+-----------------------------------+
| Partner contact details           | Will the partner's contact        |
|                                   | details be updated automatically? |
|                                   |                                   |
|                                   | This depends on the partner's     |
|                                   | circumstances (for example, the   |
|                                   | partner's record may have a       |
|                                   | started activity that prevents    |
|                                   | details updating automatically).  |
|                                   | It is important to ensure the     |
|                                   | partner's contact details,        |
|                                   | including their address, are      |
|                                   | correct and update this manually  |
|                                   | if required.                      |
+-----------------------------------+-----------------------------------+
| Documents & Forms                 | Can there be cases where nil      |
|                                   | documentation is requested of     |
|                                   | customer?                         |
|                                   |                                   |
|                                   | Yes.                              |
+-----------------------------------+-----------------------------------+

FAQs - Student and Apprentices online claims

Table 2

+-----------------------------------+-----------------------------------+
| Students & Apprentices            | Question and Answer               |
+-----------------------------------+-----------------------------------+
| Online Mod JY                     | How does a parent provide         |
|                                   | information if an A2115 was       |
|                                   | requested within the online       |
|                                   | claim?                            |
|                                   |                                   |
|                                   | Customer is advised as part of    |
|                                   | Next Steps of the claim to have   |
|                                   | parents provide parental income   |
|                                   | details via the Online 'Advise    |
|                                   | Parental Income' service (Mod     |
|                                   | JY). This functionality has 2     |
|                                   | advantages over requesting an     |
|                                   | A2115 form.                       |
|                                   |                                   |
|                                   | -   Use of an online service      |
|                                   |     versus paper A2115 form       |
|                                   | -   If independence cannot be     |
|                                   |     proven parental income        |
|                                   |     information will be available |
|                                   |     to access claim under         |
|                                   |     dependent rules               |
|                                   |                                   |
|                                   | Note: customer tasks will display |
|                                   | if an A2115 is required but as    |
|                                   | per above is not actually         |
|                                   | requested of the customer. The    |
|                                   | A2115 in Customer Tasks is stated |
|                                   | as an alert to Service Officer of |
|                                   | the fact customer is claiming     |
|                                   | Independence.                     |
+-----------------------------------+-----------------------------------+
| Documents & Forms                 | Can the customer print a Mod JY   |
|                                   | from Next Steps?                  |
|                                   |                                   |
|                                   | No. To encourage parents to use   |
|                                   | the online service the Mod JY is  |
|                                   | not available from Next Steps.    |
|                                   | Instructions are provided to      |
|                                   | access the Advise Parental Income |
|                                   | (API) service.                    |
|                                   |                                   |
|                                   | If required, parents can still    |
|                                   | print these forms from the        |
|                                   | website.                          |
+-----------------------------------+-----------------------------------+
| Independence                      | Is the A2115 form requested?      |
|                                   |                                   |
|                                   | No. Parent(s) complete Mod JY     |
|                                   | online. This provides parental    |
|                                   | income details for independence   |
|                                   | or for dependence assessment      |
|                                   | should the customer not meet      |
|                                   | independence criteria.            |
+-----------------------------------+-----------------------------------+
| Payment Destination               | Why was a dependent customer      |
|                                   | under 18 asked to provide their   |
|                                   | own account for the payment       |
|                                   | destination?                      |
|                                   |                                   |
|                                   | These customers are asked to      |
|                                   | provide their bank details but    |
|                                   | are advised this is pending their |
|                                   | parent authorising the payment to |
|                                   | be made directly to the customer. |
|                                   | If they do not have their own     |
|                                   | account, they can enter their     |
|                                   | parent's account to continue      |
|                                   | through the claim.                |
|                                   |                                   |
|                                   | Parents are asked in the Online   |
|                                   | Mod JY that will load the payment |
|                                   | destination with either the       |
|                                   | customer or parent's bank account |
|                                   | based on the parent's response.   |
+-----------------------------------+-----------------------------------+
