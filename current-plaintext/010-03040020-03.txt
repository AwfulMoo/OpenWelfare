| 6                                 | Calculating the pro-rata rate of  |
|                                   | DED + Read more ...               |
|                                   |                                   |
|                                   | Where the percentage of           |
|                                   | home-based study between 20% and  |
|                                   | 74.9%, and is consistent across a |
|                                   | term instalment period.           |
|                                   |                                   |
|                                   | The calculation of entitlement    |
|                                   | for that term instalment is as    |
|                                   | follows:                          |
|                                   |                                   |
|                                   | -   $ annual DED rate/days in the |
|                                   |     year x days in the term       |
|                                   |     instalment x percentage of    |
|                                   |     home-based study              |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | Round to the 3rd decimal point,   |
|                                   | for example 0.3948 becomes 0.395  |
|                                   |                                   |
|                                   | Days in the year are 365 or 366   |
|                                   | in a leap year                    |
|                                   |                                   |
|                                   | Term instalment periods have a    |
|                                   | different number of days as       |
|                                   | follows:                          |
|                                   |                                   |
|                                   | -   Term 1: 90 days or 91 days in |
|                                   |     a leap year                   |
|                                   | -   Term 2: 91 days               |
|                                   | -   Term 3: 92 days               |
|                                   | -   Term 4: 92 days               |
|                                   |                                   |
|                                   | See Table 4 on the Resources page |
|                                   | for Examples 1 and 2.             |
|                                   |                                   |
|                                   | Where the percentage of           |
|                                   | home-based study is between 20%   |
|                                   | and 74.5% and is not consistent   |
|                                   | across a term instalment period.  |
|                                   |                                   |
|                                   | The calculation of entitlement    |
|                                   | should be done for each portion   |
|                                   | of the instalment period as       |
|                                   | follows:                          |
|                                   |                                   |
|                                   | -   $ annual DED rate/days in the |
|                                   |     year x number of affected     |
