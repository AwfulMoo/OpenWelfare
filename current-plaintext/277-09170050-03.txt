|                                   | percentage decisions with         |
|                                   | notification dates on or after 1  |

|                                   |                                   |
|                                   | For care percentage decisions     |
|                                   | with notification dates on or     |
|                                   | after 1 July 2010 (3 March 2011   |
|                                   | for WA Ex nup) to determine:      |
|                                   |                                   |
|                                   | -   if a new care determination   |
|                                   |     can be made, go to Step 2     |
|                                   | -   the date of effect of the     |
|                                   |     decision, go to Step 3        |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine if new care             |
|                                   | determination can be made + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For care percentage decisions     |
|                                   | made after 1 July 2010 (3 March   |
|                                   | 2011 for WA ex nup) a new care    |
|                                   | determination can only be made    |
|                                   | where the existing determination  |
|                                   | can be revoked. The new care      |
|                                   | determination commences the day   |
|                                   | after the existing care           |
|                                   | determination is revoked.         |
|                                   |                                   |
|                                   | Circumstances where an existing   |
|                                   | care determination will be        |
|                                   | revoked are:                      |
|                                   |                                   |
|                                   | -   he new care percentage        |
|                                   |     determination would change    |
|                                   |     the cost percentage and any   |
|                                   |     interim care period has ended |
|                                   | -   one of the parents who was    |
|                                   |     previously assessed to have   |
|                                   |     at least regular care of the  |
|                                   |     child is now determined to    |
|                                   |     have less than regular care   |
|                                   |     despite the child being made  |
|                                   |     available by the other parent |
|                                   | -   the new care percentage       |
|                                   |     determination would cause a   |
|                                   |     change to the existing care   |
|                                   |     percentage but not the cost   |
|                                   |     percentage and any interim    |
|                                   |     care period has ended         |
+-----------------------------------+-----------------------------------+
| 3                                 | Date of effect + Read more ...    |
|                                   |                                   |
|                                   | If the objection was lodged       |
|                                   | within 28 days of the person      |
|                                   | being notified of the care        |
|                                   | percentage decision, then the     |
|                                   | objection decision will replace   |
|                                   | the original care percentage      |
|                                   | decision.                         |
|                                   |                                   |
|                                   | If the objection was lodged       |
|                                   | outside the 28 day timeframe (90  |
|                                   | days for customers in a           |
|                                   | reciprocating jurisdiction), the  |
|                                   | customer may ask Child Support to |
|                                   | consider special circumstances    |
|                                   | that impacted on their ability to |
|                                   | lodge the objection earlier. See  |
|                                   | Child Support objection received  |
|                                   | out of time - Extension of time   |
|                                   | and Special Circumstance          |
|                                   | application, Care decisions.      |
|                                   |                                   |
|                                   | When determining the date:        |
|                                   |                                   |
|                                   | -   If the customer was notified  |
|                                   |     of the care percentage        |
|                                   |     decision, generally use the   |
|                                   |     letter issued by Child        |
|                                   |     Support as the first time the |
|                                   |     customer is notified, even    |
|                                   |     when Centrelink made the      |
|                                   |     original care decision        |
|                                   | -   Centrelink notifies shared    |
|                                   |     care decisions in writing to  |
|                                   |     their customers in very       |
|                                   |     limited situations. They do   |
|                                   |     not always notify the other   |
|                                   |     party of the outcome of an    |
|                                   |     original care decision if     |
|                                   |     that person is not getting    |
|                                   |     FTB for the child or the      |
|                                   |     decision is not changed.      |
|                                   | -   Therefore, the first          |
|                                   |     notification of a care        |
|                                   |     decision a customer may       |
|                                   |     receive may be the Child      |
|                                   |     Support letter                |
|                                   | -   If the customer indicates     |
|                                   |     that Centrelink notified them |
|                                   |     of the decision before they   |
|                                   |     got the first letter from     |
|                                   |     Child Support (e.g. they were |
|                                   |     notified by phone) then that  |
|                                   |     date is the date of           |
|                                   |     notification                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Care suspension periods and       |
|                                   | objections + Read more ...        |
|                                   |                                   |
|                                   | From 1 July 2018, a care          |
|                                   | suspension period may apply if we |
|                                   | are advised of a change in care   |
|                                   | outside of 28 days but within 26  |
|                                   | weeks of the care change, for     |
|                                   | more information see Notification |
|                                   | of change in child support care   |
|                                   | (post 1 July 2010).               |

|                                   |                                   |
|                                   | If an objection is lodged outside |
|                                   | 28 days (or 90 days for a         |
|                                   | customer in a reciprocating       |
|                                   | jurisdiction) and there are no    |
|                                   | special circumstances, the        |
|                                   | objection decision will apply     |
|                                   | from the date the objection was   |
|                                   | lodged even if the care           |
|                                   | determination from the objection  |
|                                   | would otherwise change the        |
|                                   | decision about suspension of the  |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | If an objection is lodged outside |
|                                   | 28 days (or 90 days for customer  |

|                                   | care determination:               |
|                                   |                                   |
|                                   | -   suspended the assessment, the |

|                                   |     determination from the        |
|                                   |     objection would have given    |

|                                   |     decision, a suspension will   |
|                                   |     not be applied as there is no |
|                                   |     period of time when all       |
|                                   |     parents/cares have less than  |
|                                   |     35% care in the assessment    |
|                                   |                                   |
|                                   | See the Resources page for        |
|                                   | examples of care suspension       |
|                                   | periods and objections.           |

|                                   |                                   |
|                                   | If an objection is lodged outside |
|                                   | 28 days (90 days for customers    |
|                                   | living in a reciprocating         |

|                                   | would otherwise change the        |

|                                   |                                   |
|                                   | If an objection is lodged outside |
|                                   | 28 days (90 days for customers    |
|                                   | living in a reciprocating         |
|                                   | jurisdiction) and there are no    |

|                                   | original care determination:      |

|                                   | terminated the assessment the     |

|                                   | events and objections.            |
