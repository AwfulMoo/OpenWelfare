Contact the customer

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer contact + Read more ...  |
|                                   |                                   |
|                                   | Make genuine attempts to contact  |
|                                   | the customer to negotiate         |
|                                   | repayment of the debt. See        |
|                                   | Telephone standards.              |
|                                   |                                   |
|                                   | The Resources page for Quality    |
|                                   | Call Framework Quality Call       |
|                                   | Standards at a Glance.            |
|                                   |                                   |
|                                   | Was contact successful?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Authenticate the customer before  |
|                                   | speaking with them. Suggested     |
|                                   | authentication questions for:     |
|                                   |                                   |
|                                   | providers and practices, ask them |
|                                   | to provide:                       |
|                                   |                                   |
|                                   | -   full name of provider,        |
|                                   | -   practice location/address as  |
|                                   |     listed on SAP, and            |

|                                   |     and contact number            |
|                                   |                                   |
|                                   | organisations (Simplified Billing |

|                                   | debts), ask them to provide:      |
|                                   |                                   |

|                                   | -   current address as listed on  |
|                                   |     SAP, and                      |
|                                   | -   details, contact number       |
|                                   |     and/or email of the contact   |
|                                   |     person                        |
|                                   |                                   |
|                                   | There is no requirement for a     |
|                                   | Service Officer to disclose their |
|                                   | full name to customers, but they  |
|                                   | must:                             |
|                                   |                                   |
|                                   | -   identify themselves, and      |
|                                   | -   explain they are calling from |
|                                   |     Services Australia - Medicare |
|                                   |                                   |
|                                   | Use the following greetings as a  |
|                                   | guide:                            |

Instalment plans

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Paying by instalments or          |
|                                   | deferring the date of payment +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A customer may request to:        |
|                                   |                                   |
|                                   | -   pay the debt in instalments,  |
|                                   |     or                            |
|                                   | -   defer the date of payment     |
|                                   |                                   |
|                                   | To determine an acceptable        |
|                                   | repayment amount, consider the:   |
|                                   |                                   |
|                                   | -   customer's capacity to pay,   |
|                                   |     and                           |
|                                   | -   the value of the debt.        |
|                                   |                                   |
|                                   | Evidence may be required to       |
|                                   | support the debtor's claim that   |
|                                   | they cannot repay in full. This   |
|                                   | can include:                      |
|                                   |                                   |
|                                   | -   latest tax returns (ATO -     |
|                                   |     Notice of Assessment)         |
|                                   | -   financial statements or       |
|                                   |     Evidence of Assets and        |
|                                   |     Liabilities                   |
|                                   | -   reasons for their claim       |
|                                   | -   a statement of financial      |
|                                   |     position, showing their       |
|                                   |     current and expected future   |
|                                   |     financial positions           |
|                                   | -   their genuine commitment to   |
|                                   |     repay the debt in the         |
|                                   |     prescribed timeframe          |
|                                   | -   any independent assessment    |
|                                   |     and/or advice received        |
|                                   | -   proof showing that enforcing  |
|                                   |     the debtor to discharge their |
|                                   |     debt in 1 year or less will   |
|                                   |     cause financial hardship      |
|                                   |                                   |
|                                   | Debts should be repaid within 12  |
|                                   | months. The delegate may approve  |
|                                   | a longer period after considering |
|                                   | the debtor's circumstances.       |
|                                   |                                   |
|                                   | Note: only Medicare Debt Recovery |
|                                   | staff have delegation to accept   |
|                                   | instalments for Medicare debts.   |
|                                   |                                   |
|                                   | When necessary, email the         |
|                                   | delegate (APS5 or 6 in Medicare   |
|                                   | Recovery) for advice.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Delegate decision + Read more ... |
|                                   |                                   |
|                                   | A delegate can accept the         |
|                                   | instalment plan if:               |
|                                   |                                   |
|                                   | the debt amount is within their   |
|                                   | delegation                        |
|                                   |                                   |
|                                   | they consider the amount of the   |
|                                   | instalment appropriate            |
|                                   |                                   |
|                                   | the debtor agrees to the          |
|                                   | repayment schedule, and           |
|                                   |                                   |
|                                   | the debtor understands the        |
|                                   | consequences if they default on   |
|                                   | their payments before the debt    |
|                                   | (including interest) is fully     |
|                                   | repaid. This may include:         |
|                                   |                                   |
|                                   | -   legal action,                 |
|                                   | -   applying interest and         |
|                                   | -   referring the debt to an      |
|                                   |     external collection agency    |
+-----------------------------------+-----------------------------------+
| 3                                 | Instalment plan approval + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Open the Instalment Plan          |
|                                   | spreadsheet template from the     |
|                                   | shared drive > Finance >          |
|                                   | Instalment Plan. See Resources    |
|                                   | for a link to the Instalment      |
|                                   | Schedule template.                |
|                                   |                                   |
|                                   | To set instalment plan up:        |
|                                   |                                   |
|                                   | In the Data Entry tab complete    |
|                                   | details for the payment plan:     |
|                                   |                                   |
|                                   | -   Contact Date: key the date    |
|                                   |     the customer contacted        |
|                                   |     (dd/mm/yyyy)                  |
|                                   | -   Debt Amount: Due amount as on |
|                                   |     SAP                           |
|                                   | -   Due Date: As per SAP          |
|                                   | -   Repayment: Agreed payment     |
|                                   |     amount per frequency          |
|                                   | -   1st Repayment Date: Agreed    |
|                                   |     date                          |
|                                   | -   No. of Payments: Auto filled  |
|                                   | -   Interest Type: Seek advice    |
|                                   |     from Operational Team Leader  |
|                                   |     (OTL)                         |
|                                   | -   Interest Rate: Seek advice    |
|                                   |     from OTL                      |
|                                   |                                   |
|                                   | A letter auto generates from      |
|                                   | details entered in the Data Entry |
|                                   | tab. Letter can be printed from   |
|                                   | the Letter tab in the spreadsheet |
|                                   |                                   |
|                                   | Save the instalment plan to the   |
|                                   | shared drive > Finance >          |
|                                   | Instalment Plan using the         |
|                                   | customer name and customer        |
|                                   | number. For example,              |
|                                   | Jones_BB000000XX                  |
|                                   |                                   |
|                                   | Print the Letter and Instalment   |
|                                   | Schedule from tabs in the         |
|                                   | spreadsheet                       |
|                                   |                                   |
|                                   | Email or mail the Letter and      |
|                                   | Instalment Schedule to the        |
|                                   | customer                          |
|                                   |                                   |
|                                   | In SAP, run Transaction FBL5N -   |
|                                   | Customer Line Item Display:       |
|                                   |                                   |
|                                   | -   Attach the documents to SAP   |
|                                   | -   Update the Assignment field.  |
|                                   |     For example, IPF/IPM/IPW. See |
|                                   |     Resources for a list of the   |
|                                   |     Assignment field values       |
|                                   | -   Go to the Dunning Block field |
|                                   |     and key 'A'                   |

Recovery of Medicare debt - deceased customer

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Recovery of debts for deceased    |
|                                   | customers + Read more ...         |
|                                   |                                   |
|                                   | Exercise sensitivity when         |
|                                   | discussing a debt with:           |
|                                   |                                   |
|                                   | -   the nominated executor        |
|                                   | -   Public Trustee                |
|                                   | -   Court, or                     |
|                                   | -   administrator with authority  |
|                                   |     to deal with the proceeds of  |
|                                   |     the estate                    |
|                                   |                                   |
|                                   | If notification is received via   |
|                                   | telephone or email advising the   |
|                                   | customer is deceased, obtain the  |
|                                   | following:                        |
|                                   |                                   |
|                                   | -   name of caller                |
|                                   | -   relationship to the deceased  |
|                                   | -   contact telephone number      |
|                                   | -   details of the executor (if   |
|                                   |     known)                        |
|                                   |                                   |
|                                   | Send an advice to the deceased    |
|                                   | customer's estate for all legally |
|                                   | recoverable outstanding debts.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact + Read more ...           |
|                                   |                                   |
|                                   | Do not pursue any debt until 4    |
|                                   | weeks after the customer’s death. |
|                                   |                                   |
|                                   | If pursuing recovery of a debt:   |
|                                   |                                   |
|                                   | -   the debt is only recoverable  |
|                                   |     from the debtors' estate      |
|                                   | -   do not accept payment of a    |
|                                   |     debt from any source other    |
|                                   |     than the deceased customer's  |
|                                   |     estate                        |
|                                   | -   do not garnishee any          |
|                                   |     financial institution         |
|                                   |     account/s belonging to a      |
|                                   |     deceased customer's estate    |
+-----------------------------------+-----------------------------------+
| 3                                 | Write off a debt due to no estate |
|                                   | or insufficient funds + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Write off the debt if:            |
|                                   |                                   |
|                                   | an executor or third party        |
|                                   | notifies:                         |
|                                   |                                   |
|                                   | -   there is no estate, or        |
|                                   | -   the estate has been           |
|                                   |     distributed                   |
|                                   |                                   |
|                                   | the estate has:                   |
|                                   |                                   |
|                                   | -   partially repaid the debt,    |
|                                   |     and                           |
|                                   | -   insufficient funds remaining  |
|                                   |     to pay the balance            |
|                                   |                                   |
|                                   | the executor or any               |
|                                   | assets/property cannot be located |
|                                   |                                   |
|                                   | Obtain written advice from the    |
|                                   | executor, a relative or carer     |
|                                   | advising there is no estate. Add  |
|                                   | these details to SAP notes and    |
|                                   | update the Assignment field.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Not cost effective + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Make reasonable effort to recover |
|                                   | the amount from the estate.       |
|                                   |                                   |
|                                   | Write off a debt where it is not  |
|                                   | cost effective to recover (where  |
|                                   | the outstanding debt balance is   |
|                                   | less than $100).                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Action a write off + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | See Debt for write off table in   |
|                                   | Financial Adjustments for         |
|                                   | Medicare debt for the write off   |
|                                   | process.                          |
|                                   |                                   |
|                                   | Update the Assignment field and   |
|                                   | create a note on the customer     |
|                                   | record. For example:              |
|                                   |                                   |
|                                   | -   After further investigation,  |
|                                   |     customer passed away on       |
|                                   |     <date>                        |
|                                   | -   Advice received that there is |
|                                   |     no estate                     |
|                                   | -   Estate has insufficient funds |
|                                   |     available, or                 |
|                                   | -   No details in relation to the |
|                                   |     executor of the will          |
|                                   |                                   |
|                                   | Include in the SAP note:          |
|                                   |                                   |
|                                   | -   “Debt of <amount> written     |
|                                   |     off - (not legally            |
|                                   |     recoverable or not cost       |
|                                   |     effective to recover) under   |
|                                   |     the Public Governance,        |
|                                   |     Performance and               |
|                                   |     Accountability Rule 2014,     |
|                                   |     rule 11, Recovery of debts”.  |

Bankruptcy

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | General bankruptcy information +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Where a customer advises they     |
|                                   | have applied for bankruptcy or    |
|                                   | been declared bankrupt, check     |
|                                   | with Australian Financial         |
|                                   | Security Authority (ASFA) for the |
|                                   | bankruptcy details.               |
|                                   |                                   |
|                                   | When a customer is declared       |
|                                   | bankrupt, some Medicare debts     |
|                                   | cease to be payable during the    |
|                                   | bankruptcy. The Bankruptcy Act    |
|                                   | does not release a bankrupt from  |
|                                   | a debt incurred by means of       |
|                                   | fraud.                            |
|                                   |                                   |
|                                   | Where required, the submit a      |
|                                   | proof of debt to the              |
|                                   | Administrator to be registered as |
|                                   | a creditor. This allows any       |
|                                   | potential dividends from the      |
|                                   | trustee to go towards the debt.   |
|                                   |                                   |
|                                   | If a customer is bankrupted seek  |
|                                   | advice from a Team leader. If     |
|                                   | necessary, seek advice by email   |
|                                   | from the Legal Services Branch    |
|                                   | before taking legal action.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Customer contact + Read more ...  |
|                                   |                                   |
|                                   | Advise customers considering      |
|                                   | bankruptcy or who are bankrupted: |
|                                   |                                   |
|                                   | -   recovery of debts covered by  |
|                                   |     bankruptcy will stop for the  |
|                                   |     bankruptcy period             |
|                                   | -   they should include all       |
|                                   |     Medicare debts when informing |
|                                   |     their bankruptcy trustee      |
|                                   | -   do not give advice about      |
|                                   |     debts which may be due to     |
|                                   |     fraud                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Debts not due to fraud + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For debts that are not due to     |
|                                   | fraud, permanently write off the  |
|                                   | debt/s. See Financial adjustments |
|                                   | for Medicare debt.                |
|                                   |                                   |
|                                   | Advise the customer in writing    |
|                                   | that recovery does not continue   |
|                                   | after they are discharged from    |
|                                   | bankruptcy.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Debt raised due to fraud + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | For debts raised due to fraud:    |
|                                   |                                   |
|                                   | Apply a temporary write off up to |
|                                   | the date the customer is          |
|                                   | scheduled to be discharged. See   |
|                                   | Financial adjustments for         |
|                                   | Medicare debt                     |
|                                   |                                   |
|                                   | Send a letter to the customer 1   |
|                                   | month before the end of           |
|                                   | bankruptcy to advise the debt is  |
|                                   | recoverable                       |
|                                   |                                   |
|                                   | After the customer is discharged: |
|                                   |                                   |
|                                   | -   the temporary write off ends, |
|                                   |     and                           |
|                                   | -   recovery of the debt          |
|                                   |     recommences                   |
