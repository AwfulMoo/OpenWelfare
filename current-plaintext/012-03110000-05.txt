Description of organisation name details

Description of HPI-O name details for the HI Service

This table lists and describes various features of HPI-O names for the
HI Service.

+-----------------------------------+-----------------------------------+
| Organisation detail               | Description                       |
+-----------------------------------+-----------------------------------+
| Organisation Name                 | Commonly the name of the business |
|                                   | entity, for example, Robert J.    |
|                                   | Green Unit Trust.                 |
+-----------------------------------+-----------------------------------+
| Trading Name                      | Commonly the name of the surgery, |
|                                   | for example, Greens Medical       |
|                                   | Centre.                           |
+-----------------------------------+-----------------------------------+
| Australian Standard               | Australian Standard (AS4846)      |
|                                   | definitions for recording of      |
|                                   | organisation names:               |
|                                   |                                   |
|                                   | -   "Business name - Business     |
|                                   |     name used only for trading    |
|                                   |     purposes".                    |
|                                   | -   "Enterprise name - generally, |
|                                   |     the complete organisation     |
|                                   |     name should be used to avoid  |
|                                   |     any ambiguity in              |
|                                   |     identification. This should   |
|                                   |     usually be the same as        |
|                                   |     company registration name".   |
+-----------------------------------+-----------------------------------+
| Other                             | Used when the organisation name   |
|                                   | does not fit into any one of the  |
|                                   | categories listed above.          |
+-----------------------------------+-----------------------------------+
