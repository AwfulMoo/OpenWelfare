Item categories

Categories of pharmaceutical benefits

This table describes the categories of pharmaceutical benefits listed in
the Schedule of Pharmaceutical Benefits.

+-----------------------------------+-----------------------------------+
| Benefits                          | Description                       |
+-----------------------------------+-----------------------------------+
| Unrestricted benefits             | Have no Pharmaceutical Benefits   |
|                                   | Scheme (PBS) restrictions on      |
|                                   | therapeutic uses.                 |
+-----------------------------------+-----------------------------------+
| Restricted benefits               | Can only be prescribed for        |
|                                   | specific therapeutic uses. They   |
|                                   | are noted as restricted benefit   |
|                                   | in the Schedule.                  |
+-----------------------------------+-----------------------------------+
| Authority required benefits       | Are restricted and require prior  |
|                                   | approval from Services Australia  |
|                                   | or the Department of Veteran’s    |
|                                   | Affairs (DVA). They are noted as  |
|                                   | authority required in the         |
|                                   | Schedule.                         |
+-----------------------------------+-----------------------------------+
| Authority required (streamlined)  | These Authority required items do |
| benefits                          | not require prior approval from   |
|                                   | Services Australia or the DVA.    |
|                                   | These items require a 4 digit     |
|                                   | streamlined authority code        |
|                                   | obtained from the Schedule of     |
|                                   | Pharmaceutical Benefits.          |
|                                   |                                   |
|                                   | For increase quantities and/or    |
|                                   | additional repeats above those    |
|                                   | specified in the Schedule of      |
|                                   | Pharmaceutical Benefits prior     |
|                                   | approval from Services Australia  |
|                                   | or DVA must be obtained.          |
+-----------------------------------+-----------------------------------+
