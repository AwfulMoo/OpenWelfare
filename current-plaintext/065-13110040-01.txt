This document outlines how Service Officers request access to the Aged
Care Management Payment System (ACMPS) and use the historical user
guide. The historical user guide provides basic information about common
tabs/screens displayed in ACMPS.

This document also contains information on how the carer status types,
leave types and the departure reasons are displayed in ACMPS. This
information is only for home care purposes.

The Resources page contains:

-   the ACMPS User Guide
-   a task card to request ACMPS access
-   a task card to create or update and Aged Care Client Record
-   information on how the carer status, leave types and departure
    reasons are displayed in ACMPS. These tables are relevant to home
    care only
