This page contains general information on scholarship assessment and how
amounts received are coded on the system for Centrelink income support
payments only, not for Family Tax Benefit (FTB)/Child Care Subsidy
(CCS). Approval for the exemption from the income test for social
security payments is obtained by the appropriate Services Australia
Helpdesk, through the Department of Social Services (DSS) Means Test
Policy Section.

|                                   |                                   |
|                                   | For fortnightly or periodic       |
|                                   | payments:                         |
|                                   |                                   |
|                                   | -   Type field, select 'Equity    |
|                                   |     and Merit Scholarship'        |
|                                   | -   Description field, key the    |
|                                   |     name of the scholarship       |
|                                   | -   Event Date field, key the     |
|                                   |     first day of the assessable   |
|                                   |     period                        |
|                                   | -   Amount field, key the amount  |
|                                   |     of the scholarship            |
|                                   | -   Freq field, select the        |
|                                   |     relevant frequency of the     |
|                                   |     payments so the system        |
|                                   |     auto-deducts the exempt       |
|                                   |     portion from the total and    |
|                                   |     applies the income test to    |
|                                   |     the remaining amount. For     |
|                                   |     other frequencies, for        |
|                                   |     example, once per trimester,  |
|                                   |     annualise the amount and code |
|                                   |     as 'ANN'                      |
|                                   |                                   |
|                                   | Note: when an Equity or Merit     |
|                                   | based Scholarship is coded as a   |
|                                   | fortnightly or periodic payment,  |
|                                   | the exempt income amount is       |
|                                   | spread out over the calendar      |
|                                   | year. The exempt income is        |
|                                   | divided by 26 fortnights and this |
|                                   | is deducted from the Equity or    |
|                                   | Merit based Scholarship income    |
|                                   | coded on the Other Income Summary |
|                                   | (OINS) screen. The relevant       |
|                                   | income test is then applied to    |
|                                   | any remaining amounts each        |
|                                   | fortnight or period. See          |
|                                   | Personal/Partner Income Tests for |
|                                   | students, trainees or Australian  |
|                                   | Apprentices.                      |
|                                   |                                   |
|                                   | For a customer to receive the     |
|                                   | total benefit of the exempt       |
|                                   | income amount:                    |
|                                   |                                   |
|                                   | -   they must be on payments for  |
|                                   |     the whole calendar year, and  |
|                                   | -   the Equity or Merit based     |
|                                   |     Scholarship income must be    |
|                                   |     affecting for the whole       |
|                                   |     calendar year                 |
|                                   |                                   |
|                                   | If both of the above criteria are |
|                                   | not met, the customer is only     |
|                                   | entitled to the equivalent        |
|                                   | proportion of the exempt income   |
|                                   | over the period that the Equity   |
|                                   | or Merit based Scholarship is     |
|                                   | coded, or the period that they    |
|                                   | are in receipt of the student     |
|                                   | payment. The Resources page       |
|                                   | contains an exempt income         |
|                                   | reduction example.                |
