This page has the process to record the outcome of an urgent payment
request on the customer's record and issue payment if approved. Staff
must first follow the relevant procedure to consider the request and
make a decision to approve or deny it.

On this page:

Using the script to approve an urgent payment request

Issuing an urgent payment for Farm Household Allowance (FHA) customers

Using the script to deny a request for urgent payment

Customer attends service centre to collect urgent payment

Issuing an urgent payment for Status Resolution Support Services (SRSS)
payment

Using the script to approve an urgent payment request
