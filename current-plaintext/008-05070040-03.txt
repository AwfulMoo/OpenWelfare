Legend of code table elements

Table 1: This table describes the acronyms and elements of the code
tables.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Code                              |
|                                   |                                   |
|                                   | 3 letter reason code.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Description                       |
|                                   |                                   |
|                                   | Expanded meaning of 3 letter      |
|                                   | reason codes.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Action                            |
|                                   |                                   |
|                                   | REJ = rejection                   |
|                                   |                                   |
|                                   | SUS = suspension                  |
|                                   |                                   |
|                                   | CAN = cancellation                |
+-----------------------------------+-----------------------------------+
| 4                                 | Initiation                        |
|                                   |                                   |
|                                   | Manual = reason codes that can be |
|                                   | manually input on the Benefit     |
|                                   | Action (BA) screen.               |
|                                   |                                   |
|                                   | Auto = cancellation, suspension   |
|                                   | or rejection codes that           |
|                                   | automatically occur on the system |
|                                   | and cannot be entered manually in |
|                                   | most cases.                       |
+-----------------------------------+-----------------------------------+
| 5                                 | Who can restore                   |
|                                   |                                   |
|                                   | This column indicates whether the |
|                                   | payment can be restored at a      |
|                                   | Smart Centre or only at a service |
|                                   | centre. Always check the Document |
|                                   | List (DL) screen for information  |
|                                   | about the suspension because it   |
|                                   | may not be appropriate to restore |
|                                   | at the first point of contact.    |
