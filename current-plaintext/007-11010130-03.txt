| 6                                 | Bereavement deferral + Read more  |
|                                   |                                   |
|                                   | A bereavement deferral can be     |
|                                   | recorded after:                   |
|                                   |                                   |
|                                   | -   the death of a customer who   |
|                                   |     may have entitlement to FTB   |
|                                   |     arrears payments, or          |
|                                   | -   the death of a customer's     |
|                                   |     child to issue an FTB         |
|                                   |     bereavement payment           |
|                                   |                                   |
|                                   | A bereavement deferral is not     |
|                                   | required after the death of a     |
|                                   | partner because they are then     |
|                                   | treated as a former partner. PIP  |
|                                   | does not apply because of a       |
|                                   | former partner's outstanding      |
|                                   | income. If PIP solely applies due |
|                                   | to deceased partner's outstanding |
|                                   | income tax return, it will be     |
|                                   | automatically lifted once the     |
|                                   | partner's death is recorded.      |
|                                   |                                   |
|                                   | Bereavement deferrals cannot be   |
|                                   | rejected but the period of the    |
|                                   | deferral can be reduced if        |
|                                   | another deferral is granted. If   |
|                                   | more than1 person has died,       |
|                                   | separate bereavement deferrals    |
|                                   | can be recorded.                  |
|                                   |                                   |
|                                   | Run the Deferral Exemption        |
|                                   | workflow. For the death of a      |
|                                   | child in a blended family, run    |
|                                   | the workflow on both records      |
|                                   |                                   |
|                                   | Select the Bereavement Deferral   |
|                                   | option to stop PIP being applied  |
|                                   | to the bereavement instalment     |
|                                   | payments. Code the following      |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Start date: Date the deferral |
|                                   |     starts to allow the           |
|                                   |     bereavement payment to be     |
|                                   |     made                          |
|                                   | -   End date: Date the deferral   |
|                                   |     ends                          |
|                                   | -   Activity Note: Record         |
|                                   |     relevant information          |
|                                   | -   Source and DOR: fields        |
|                                   |                                   |
|                                   | A letter automatically issues to  |
|                                   | the customer to confirm the       |
|                                   | outcome of the deferral           |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | Procedure ends here.              |

| 7                                 | Severe illness (including mental  |
|                                   | illness) + Read more              |
|                                   |                                   |
|                                   | A deferral assessment:            |
|                                   |                                   |
|                                   | -   can occur if a severe         |
|                                   |     physical or mental illness    |
|                                   |     prevents the required lodger  |
|                                   |     from lodging the required     |
|                                   |     income tax return/s           |
|                                   | -   cannot occur if the person    |
|                                   |     who did not lodge is the      |
|                                   |     partner of the person         |
|                                   |     suffering the illness         |
|                                   |                                   |
|                                   | To assess the deferral a doctor's |
|                                   | certificate is required to be     |
|                                   | lodged that indicates:            |
|                                   |                                   |
|                                   | -   the illness preventing        |
|                                   |     lodgement of the income tax   |
|                                   |     return/s                      |
|                                   | -   how long the condition is     |
|                                   |     expected to last              |
|                                   |                                   |
|                                   | If the customer is distressed due |
|                                   | to the illness, consider referral |
|                                   | to a social worker for support.   |
|                                   |                                   |
|                                   | Tell the customer to upload the   |
|                                   | doctor's certificate or take it   |
|                                   | to their local service centre for |
|                                   | assessment. If PIP has already    |
|                                   | started, remind the customer that |
|                                   | it will continue to apply until   |
|                                   | assessment of the deferral.       |
|                                   | Record details on a DOC.          |
|                                   |                                   |
|                                   | If the customer has provided the  |
|                                   | doctor's certificate, assess the  |
|                                   | evidence and run the Deferral     |
|                                   | Exemption workflow. For a blended |
|                                   | family (both customer and partner |
|                                   | FTB current), code details on     |
|                                   | both records for the deferral to  |
|                                   | apply to the family.              |
|                                   |                                   |
|                                   | Select the Severe Illness option. |
|                                   |                                   |
|                                   | Is the deferral being granted?    |
|                                   |                                   |
|                                   | Yes, code the following fields.   |
|                                   |                                   |
|                                   | -   Severely ill person: Select   |
|                                   |     the correct person from the   |
|                                   |     drop down box                 |
|                                   | -   Notification Date: The date   |
|                                   |     of notification. This can be  |
|                                   |     backdated at the request of   |
|                                   |     the customer however the      |
|                                   |     total period for the severe   |
|                                   |     illness deferral is up to 6   |
|                                   |     months, and the Doctor's      |
|                                   |     certificate must verify that  |
|                                   |     the condition existed for the |
|                                   |     backdated period              |
|                                   | -   Start date: Date the deferral |
|                                   |     starts (no back or future     |
|                                   |     dating provisions are         |
|                                   |     available unless a system     |
|                                   |     delay occurs)                 |
|                                   | -   End date: Date the deferral   |
|                                   |     ends                          |
|                                   | -   Activity Note: Record         |
|                                   |     relevant information          |
|                                   | -   Source and DOR: fields        |
|                                   |                                   |
|                                   | No, see Step 7 in Table 2         |
|                                   |                                   |
|                                   | Tell the customer that a letter   |
|                                   | will issue to them with the       |
|                                   | outcome and details about what    |
|                                   | they need to do.                  |
|                                   |                                   |
|                                   | A Severe Illness deferral for the |
|                                   | same person cannot overlap, but   |
|                                   | there can be multiple deferrals   |
|                                   | or exemptions for different types |
|                                   | or a Severe Illness deferral for  |
|                                   | their partner. The deferral       |
|                                   | period can also be reduced if     |
|                                   | another deferral has already been |
|                                   | granted.                          |
|                                   |                                   |
|                                   | A deferral may be granted for up  |
|                                   | to 6 months (subsequent deferrals |
|                                   | will need to be assessed by       |
|                                   | Centrelink before the current end |
|                                   | date, otherwise PIP will be       |
|                                   | reapplied).                       |
|                                   |                                   |
|                                   | A permanent deferral can be       |
|                                   | applied if a customer and partner |
|                                   | have separate consecutive Severe  |
|                                   | Illness deferrals (or extensions) |
|                                   | totalling 6 years for the same    |
|                                   | deferral reason. A permanent      |
|                                   | exemption will be granted and PIP |
|                                   | will no longer be assessable. In  |
|                                   | this case the system will         |
|                                   | automatically trigger the         |
|                                   | permanent exemption.              |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   |                                   |

|                                   | financial year, which may be      |

|                                   |                                   |
|                                   | The References page contains a    |

|                                   | -   The customer has advised they |

|                                   |     base rate and/or on income    |

|                                   |     Rate Explanation FTBA         |

|                                   |     Summary (PS) screen           |
|                                   |                                   |
|                                   | Run the Deferral Exemption        |
|                                   | workflow. For a blended family    |

|                                   | both records for the deferral to  |
|                                   | apply to the family.              |
|                                   |                                   |
|                                   | Select the Financial Hardship     |
|                                   | option.                           |
|                                   |                                   |
|                                   | Is the deferral being granted?    |
|                                   |                                   |

|                                   | -   Start date: The date the      |
|                                   |     deferral starts. This can be  |
|                                   |     backdated at the request of   |
|                                   |     the customer however the      |

|                                   |     financial hardship deferral   |

|                                   | -   End date: The date the        |
|                                   |     deferral ends                 |
|                                   | -   Activity Note: Record         |
|                                   |     relevant information          |
|                                   | -   Source and DOR: fields        |
|                                   |                                   |

|                                   |                                   |

|                                   | However, there can be multiple    |
|                                   | deferrals or exemptions for       |
|                                   | different deferral types. A       |
|                                   | deferral period can be reduced if |
|                                   | another deferral type has already |
|                                   | been granted.                     |
|                                   |                                   |

Special circumstances PIP deferral

Table 2: this table describes the process of assessing special
circumstances in the deferral of PIP. Portions of the process can only
be completed by social workers and Families and Child Care Smart Centre
Processing staff.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Special circumstances + Read more |
|                                   |                                   |
|                                   | Special circumstances PIP         |
|                                   | deferral                          |
|                                   |                                   |
|                                   | Check the Document List (DL)      |
|                                   | screen to ensure that this        |
|                                   | deferral has not already been     |
|                                   | considered/processed by the       |
|                                   | National Families Processing Team |
|                                   | (NFP).                            |
|                                   |                                   |
|                                   | If the deferral has not already   |
|                                   | been considered/processed:        |
|                                   |                                   |
|                                   | -   Smart Centre Call Service     |
|                                   |     Officers should follow normal |
|                                   |     escalation procedures         |
|                                   | -   Families Service Officer,     |
|                                   |     refer the customer to a       |
|                                   |     Families Service Support      |
|                                   |     Officer (SSO) for assessment. |
|                                   |     Referral should take place    |
|                                   |     within 24 hours. The Service  |
|                                   |     Officer should not promise    |
|                                   |     that the deferral will be     |
|                                   |     granted. Procedure ends here  |
|                                   | -   Families Service Support      |
|                                   |     Officer (SSO), make a         |
|                                   |     decision about the special    |
|                                   |     circumstances PIP deferral    |
|                                   |     within 24 hours of receiving  |
|                                   |     the initial referral from the |
|                                   |     Service Officer. Record       |
|                                   |     details on a DOC for example  |
|                                   |     type of deferral granted/not  |
|                                   |     granted and why/why not etc.  |
|                                   |                                   |
|                                   | Is the decision to grant the      |
|                                   | deferral?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 7              |
+-----------------------------------+-----------------------------------+
| 2                                 | Families Service Support Officer  |
|                                   | (SSO) only - deferral granted +   |
|                                   | Read more                         |
|                                   |                                   |
|                                   | Send an email to the National     |
|                                   | Families Processing (NFP) team.   |
|                                   | The Resources page contains       |
|                                   | contact details for the team in   |
|                                   | Families Reconciliation team.     |
|                                   |                                   |
|                                   | Record 'PIP Special Circ.         |
|                                   | Request' in the subject line and  |
|                                   | in the email include:             |
|                                   |                                   |
|                                   | -   Customer's Customer Reference |
|                                   |     Number (CRN)                  |
|                                   | -   type of deferral requested -  |
|                                   |     see Deferral Exemption        |
|                                   |     workflow                      |
|                                   | -   period of deferral (start and |
|                                   |     end dates)                    |
|                                   |                                   |
|                                   | NFP will return an email to       |
|                                   | advise the deferral coding has    |
|                                   | been done.                        |
|                                   |                                   |
|                                   | A Special Circumstances deferral  |
|                                   | can only be granted for up to 6   |
|                                   | months at a time.                 |
|                                   |                                   |
|                                   | Set a review after the deferral   |
|                                   | end date and contact the customer |
|                                   | to see if the deferral should be  |
|                                   | extended.                         |
|                                   |                                   |
|                                   | Customers must resolve their      |
|                                   | non-lodger circumstances by the   |
|                                   | end of the deferral period by:    |
|                                   |                                   |
|                                   | -   lodging all outstanding tax   |

| 3                                 | Family and domestic violence,     |
|                                   | coded as a social worker          |
|                                   | Deferral + Read more              |
|                                   |                                   |
|                                   | A customer can be assessed for a  |
|                                   | social worker deferral if family  |
|                                   | and domestic violence exists      |
|                                   | within the current partner        |
|                                   | relationship and:                 |
|                                   |                                   |
|                                   | -   the customer and/or family    |
|                                   |     may experience harm by        |
|                                   |     requesting the partner to     |
|                                   |     lodge their income tax        |
|                                   |     return/s, or                  |
|                                   | -   family and domestic violence  |
|                                   |     indicators are present        |
|                                   | -   a customer cannot lodge their |
|                                   |     tax return because of the     |
|                                   |     actions of their partner or   |
|                                   |     ex-partner, it is possible to |
|                                   |     grant a deferral for a        |
|                                   |     specified period              |
|                                   |                                   |
|                                   | Social worker deferrals can       |
|                                   | include where the violence was    |
|                                   | threatened or carried out against |
|                                   | a child, or another member of the |
|                                   | family who resides in the         |
|                                   | household, and this has meant the |
|                                   | customer has been unable to lodge |
|                                   | their tax return                  |
|                                   |                                   |
|                                   | If the customer is distressed     |
|                                   | (due to domestic violence) refer  |
|                                   | them directly to a social worker  |
|                                   | for a full assessment and         |
|                                   | support.                          |
|                                   |                                   |
|                                   | Service Officers should assess    |
|                                   | the referral with care and        |
|                                   | sensitivity due to the nature of  |
|                                   | the issue by:                     |
|                                   |                                   |
|                                   | -   investigating customer and    |
|                                   |     partner DOCs for information  |
|                                   |     relating to existing family   |
|                                   |     and domestic violence in the  |
|                                   |     relationship                  |
|                                   | -   discussing the circumstances  |
|                                   |     with the customer to help     |
|                                   |     consider the need for a       |
|                                   |     referral                      |
|                                   |                                   |
|                                   | Blended families and social       |
|                                   | worker deferrals                  |
|                                   |                                   |
|                                   | PIP is applied to both members of |
|                                   | the blended family but deferrals  |
|                                   | can be applied to customers       |
|                                   | individually. If 1 member of the  |
|                                   | couple has not lodged their       |
|                                   | income tax return, PIP is applied |
|                                   | to FTB for both customers.        |
|                                   |                                   |
|                                   | In these cases, the customer is   |
|                                   | legally entitled to 99% of the    |
|                                   | total amount of the household     |
|                                   | FTB, leaving only 1% that has the |
|                                   | PIP applied. Lift the PIP on the  |
|                                   | customer's record (only in this   |
|                                   | situation).                       |
|                                   |                                   |
|                                   | Be aware that the partner will    |
|                                   | get minimal amount paid if they   |
|                                   | eventually lodge which may        |
|                                   | possibly generate more family and |
|                                   | domestic violence on the customer |
|                                   | and/or family.                    |
