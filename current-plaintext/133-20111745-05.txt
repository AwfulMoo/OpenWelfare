FAQs from health professionals about a patient care plan history search

Table 1:

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
|                                   | What items appear when I use the  |
|                                   | care plan search?                 |
|                                   |                                   |
|                                   | Health Professional Online        |
|                                   | Services (HPOS) displays items    |
|                                   | that are part of:                 |
|                                   |                                   |
|                                   | -   General Practitioner          |
|                                   |     Management Plans              |
|                                   | -   Team Care Arrangements        |
|                                   | -   Multidisciplinary Care Plans  |
|                                   | -   General Practitioner Mental   |
|                                   |     Health Treatment Plans        |
+-----------------------------------+-----------------------------------+
|                                   | I cannot access specific details  |
|                                   | about my patient’s care plan      |
|                                   |                                   |
|                                   | This resource is not intended to  |
|                                   | provide detailed clinical         |
|                                   | information. It is available for  |
|                                   | checking whether a care plan      |
|                                   | exists to assess whether          |
|                                   | additional treatment is           |
|                                   | clinically appropriate. Patients  |
|                                   | may share relevant clinical       |
|                                   | details with health professionals |
|                                   | using their My Health Record, or  |
|                                   | getting a previous health         |
|                                   | professional to send their        |
|                                   | records over.                     |
+-----------------------------------+-----------------------------------+
|                                   | How do I access the care plan     |
|                                   | checker?                          |
|                                   |                                   |
|                                   | You need to have access to Health |
|                                   | Professional Online Services      |
|                                   | (HPOS). Instructions are          |
|                                   | available on the services         |
|                                   | Australia website.                |
+-----------------------------------+-----------------------------------+
|                                   | Can my practice manager access    |
|                                   | this information on my behalf?    |
|                                   |                                   |
|                                   | No. Delegates are not able to     |
|                                   | access this service on your       |
|                                   | behalf. Due to patient privacy,   |
|                                   | only the treating health          |
|                                   | professional can access this      |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+

FAQs from customers about care plans

Table 2:

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | I am on a care plan. Can a GP     |
|                                   | other than my regular GP check    |
|                                   | the details?                      |
|                                   |                                   |
|                                   | No, they can only see whether you |
|                                   | are on a plan or not. For further |
|                                   | information, the health           |
|                                   | professional who has created the  |
|                                   | care plan will be able to provide |
|                                   | more information. Alternatively,  |
|                                   | you can allow different health    |
|                                   | professionals access to your      |
|                                   | clinical records using My Health  |
|                                   | Record.                           |
|                                   |                                   |
|                                   | Note: Service Officers, see       |
|                                   | Medicare online account - Care    |
|                                   | plan search history screen for    |
|                                   | more information about the care   |
|                                   | plan search history screen.       |

FAQs from health professionals about MBS Items Online Checker

Table 3:

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | What items are available on the   |
|                                   | items online checker?             |
|                                   |                                   |
|                                   | The MBS Items Online Checker      |
|                                   | displays a full list of the items |
|                                   | available. Select the relevant    |
|                                   | item range tab to see if your     |
|                                   | item number is listed. The list   |
|                                   | is updated regularly.             |
|                                   |                                   |
|                                   | Note: Service Officers, see the   |
|                                   | screenshot of the MBS Items       |
|                                   | Online Checker screen in HPOS.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Why would I use the Item checker? |
|                                   |                                   |
|                                   | The MBS Items Online Checker      |
|                                   | enables health professionals to   |
|                                   | check eligibility to claim an     |
|                                   | item for a patient.               |
+-----------------------------------+-----------------------------------+
| 3                                 | What will happen when a patient   |
|                                   | has seen another doctor who has   |
|                                   | provided the same service on an   |
|                                   | earlier date, Medicare has not    |
|                                   | received and assessed the claim,  |
|                                   | and/or the previous claim is      |
|                                   | pending and not yet in the        |
|                                   | patient’s history?                |
|                                   |                                   |
|                                   | The MBS Item Online Checker will  |
|                                   | return a ‘Yes’ response.          |
+-----------------------------------+-----------------------------------+
| 4                                 | Can I use the MBS Online Item     |
|                                   | Checker without my provider       |
|                                   | numbers and/or the patients       |
|                                   | Medicare information?             |
|                                   |                                   |
|                                   | Without this information, the     |
|                                   | system cannot determine if the    |
|                                   | search has returned the correct   |
|                                   | response.                         |
+-----------------------------------+-----------------------------------+
| 5                                 | Does a 'Yes' answer indicate the  |
|                                   | patient is eligible to bill       |
|                                   |                                   |
|                                   | Yes. The patient is eligible at   |
|                                   | the time of checking the item,    |
|                                   | however it is important to bill   |
|                                   | the item close to the time that   |
|                                   | the check was undertaken.         |
+-----------------------------------+-----------------------------------+
| 6                                 | I am searching 5 items at a time, |
|                                   | why are the result through the    |
|                                   | MBS Online Item Checker returning |
|                                   | incorrect results?                |
|                                   |                                   |
|                                   | It is best to search for 1 item   |
|                                   | number at a time. This is because |
|                                   | some numbers restrict with each   |
|                                   | other, which may show incorrect   |
|                                   | results.                          |
+-----------------------------------+-----------------------------------+
| 7                                 | Can I check items well in advance |
|                                   | of the service being provided?    |
|                                   |                                   |
|                                   | No, it is important to verify the |
|                                   | patient eligibility using the MBS |
|                                   | Online Item Checker close to the  |
|                                   | date of the service. If the check |
|                                   | is done too far in advance, there |
|                                   | is a risk of other claims being   |
|                                   | lodged and paid.                  |
+-----------------------------------+-----------------------------------+
| 8                                 | Can I check a patient’s           |
|                                   | eligibility for the Child Dental  |
|                                   | Benefits Schedule (CDBS)?         |
|                                   |                                   |
|                                   | Yes, however, restrictions that   |
|                                   | apply to a particular tooth       |
|                                   | cannot be checked through this    |
|                                   | service.                          |
+-----------------------------------+-----------------------------------+
