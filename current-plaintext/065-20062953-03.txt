|                                   | be generated + Read more ...      |
|                                   |                                   |
|                                   | Income tested fee (ITF) refunds   |
|                                   | may generate for pre 1 July 2014  |
|                                   | care recipients where:            |
|                                   |                                   |
|                                   | a care recipient’s circumstances  |
|                                   | change and reduce the ITF they    |
|                                   | need to pay                       |
|                                   |                                   |
|                                   | the cost of care (Aged Care       |
|                                   | Funding Instrument - ACFI)        |
|                                   | changes, resulting in a reduction |
|                                   | of the ITF                        |
|                                   |                                   |
|                                   | -   Check current ACFI on Client  |
|                                   |     Profile (screen 2.6)          |
|                                   |                                   |
|                                   | a service provider has not lodged |
|                                   | a valid ACFI within the required  |
|                                   | time frame                        |
|                                   |                                   |
|                                   | -   Confirm dates of ACFI on      |
|                                   |     Client Profile (screen 2.6)   |
|                                   | -   ACFI lodged between 2 and 5   |
|                                   |     months from date of entry     |
|                                   |     results in a care subsidy     |
|                                   |     reduction of $25 per day (may |
|                                   |     affect maximum MTCF)          |
|                                   | -   ACFI lodged after 5 months    |
|                                   |     from date of entry results in |
|                                   |     no care subsidy paid to the   |
|                                   |     provider until the day prior  |
|                                   |     to lodgement (and therefore   |
|                                   |     no ITF payable)               |
|                                   |                                   |
|                                   | a care recipient's status changes |
|                                   | from self-funded retiree to an    |
|                                   | income support recipient          |
|                                   |                                   |
|                                   | a care recipient's status was RCA |
|                                   | Rejected - means not disclosed    |
|                                   | (MND),the care recipient has      |
|                                   | since completed a means           |
|                                   | assessment (backdated)            |
|                                   |                                   |
|                                   | -   Confirm on View Assistance,   |
|                                   |     Income and Asset Details      |
|                                   |     (screen 2.11)                 |
|                                   | -   Check that income/asset       |
|                                   |     details in SPARC from date of |
|                                   |     entry into care               |
|                                   |                                   |
|                                   | incorrect fees were charged due   |
