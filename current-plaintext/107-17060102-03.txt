| 1                                 | Requesting a substitute DAC       |
|                                   | return date + Read more ...       |
|                                   |                                   |
|                                   | A person can apply to have the    |
|                                   | return date on their DAC          |
|                                   | substituted for a later date. A   |
|                                   | person must apply for a DAC       |
|                                   | either in writing or verbally.    |
|                                   |                                   |
|                                   | Ensure to record the following    |
|                                   | information in a DOC, the:        |
|                                   |                                   |
|                                   | proposed new return date          |
|                                   |                                   |
|                                   | the reason for requesting a       |
|                                   | variation of travel. Include      |
|                                   | relevant information that assists |
|                                   | in verifying the reason, for      |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   if the person requests to     |
|                                   |     travel for a dying relative,  |
|                                   |     request the treating doctors  |
|                                   |     details                       |
|                                   | -   if the person requires a DAC  |
|                                   |     for work purpose they will    |
|                                   |     need to provide business      |
|                                   |     contacts for verification     |
|                                   |                                   |
|                                   | Extend the return date if the     |
|                                   | delegate deems the change         |
|                                   | appropriate. Only an appropriate  |
|                                   | delegate can reject an            |
|                                   | application for a DAC.            |
|                                   |                                   |
|                                   | The person may need to provide    |
|                                   | further security before extending |
|                                   | the return date. Even if there    |
|                                   | was no initial requirement for    |
|                                   | security.                         |
|                                   |                                   |
|                                   | Security will be forfeited if,    |
|                                   |                                   |
|                                   | -   the delegate does not approve |
|                                   |     the request,                  |
|                                   | -   and the debtor does not       |
|                                   |     return by the due date        |
|                                   |                                   |
|                                   | Discuss the reasons for the       |
|                                   | delayed return with the person.   |
|                                   | Depending on the reason the       |
|                                   | person provided, this may affect  |
|                                   | future DAC applications           |
|                                   |                                   |
|                                   | Note: you cannot make changes to  |
|                                   | a current DAC, however the person |
|                                   | can request a new DAC.            |
|                                   |                                   |
|                                   | Does the person wish to request   |
|                                   | to have the return date on a DAC  |
|                                   | extended?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure ends here       |
