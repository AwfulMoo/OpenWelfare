This page contains information on assessing claims for Parental Leave
Pay (PPL) and/or Family Tax Benefit (FTB) from birth mothers who
relinquish the care of their child.

PPL and FTB claims from birth mothers relinquishing care of child

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Birth mothers who relinquish the  |
|                                   | care of their child may be        |
|                                   | eligible for PPL and/or FTB +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | FTB can include Newborn           |
|                                   | Supplement (NBS) and Newborn      |
|                                   | Upfront Payment (NBU) if the      |
|                                   | customer is not eligible for PPL. |
|                                   |                                   |
|                                   | Has the customer lodged a claim   |
|                                   | for payment?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | How to claim + Read more ...      |
|                                   |                                   |
|                                   | [Self service option is           |
|                                   | available] Encourage customers to |
|                                   | complete an online claim for      |
|                                   | family assistance and PPL. This   |
|                                   | is the preferred service offer.   |
|                                   | It gives customers a streamlined  |
|                                   | experience.                       |
|                                   |                                   |
|                                   | Help the customer register for    |
|                                   | online services if necessary.     |
|                                   | Make sure they can access an      |
|                                   | 'active' online account before    |
|                                   | directing them to claim online.   |
|                                   |                                   |
|                                   | If it is unreasonable for the     |
|                                   | customer to claim online, Service |
|                                   | Officers can issue a Claim for    |
|                                   | Parental Leave Pay and Family Tax |
|                                   | Benefit (FA100).                  |
|                                   |                                   |
|                                   | Extra assistance is available to  |
|                                   | help remote Indigenous customers  |
|                                   | make a claim.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Primary carer + Read more ...     |
|                                   |                                   |
|                                   | Will the customer be the primary  |
|                                   | carer of the child after the      |
|                                   | birth before they relinquish care |
|                                   | of the child?                     |
|                                   |                                   |
|                                   | -   Yes, they may be eligible for |
|                                   |     PPL and FTB for the child.    |
|                                   |     Note: the birth               |
|                                   |     mother/primary carer may      |
|                                   |     receive either PPL or NBS for |
|                                   |     the newborn child. Customers  |
|                                   |     cannot be paid both PPL and   |
|                                   |     NBS for the same child. Go to |
|                                   |     Step 4                        |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Cessation of care of the child +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Birth mothers who cease to have   |
|                                   | care of their child immediately   |
|                                   | after birth may be eligible for   |
|                                   | PPL for maternal recovery         |
|                                   | purposes.                         |
|                                   |                                   |
|                                   | Birth mothers may receive PPL up  |
|                                   | to 18 weeks following the birth   |
|                                   | of their child if they:           |
|                                   |                                   |
|                                   | -   nominate a start date no      |
|                                   |     later than the child's date   |
|                                   |     of birth                      |
|                                   | -   claim and provide relevant    |
|                                   |     documents within the 28 days  |
|                                   |     following the birth           |
|                                   |                                   |
|                                   | This includes Flexible PPL days   |
|                                   | available for children that are   |
|                                   | born or adopted on or after 1     |
|                                   | July 2020. As the birth mother    |
|                                   | cannot receive PPL more than 18   |
|                                   | weeks after the childâ€™s birth,    |
|                                   | they should be advised to either: |
|                                   |                                   |
|                                   | -   connect all of their Flexible |
|                                   |     PPL days to the end of their  |
|                                   |     PPL period, or                |
|                                   | -   claim consecutive Flexible    |
|                                   |     PPL days after their PPL      |
|                                   |     Period                        |
|                                   |                                   |
|                                   | If they claim and provide         |
|                                   | relevant documents more than 28   |
|                                   | days following the birth, the PPL |
|                                   | period may only start from the    |
|                                   | date all information is received. |
|                                   | This means they will receive less |
|                                   | than the full 18 weeks of PPL. In |
|                                   | these cases, it may benefit the   |
|                                   | customer to claim consecutive     |
|                                   | Flexible PPL days after their PPL |
|                                   | Period ends. This can include     |
|                                   | claiming Flexible PPL days for    |
|                                   | weekend days. See Claiming and    |
|                                   | managing Flexible Paid Parental   |
|                                   | Leave (PPL) days, and the         |
|                                   | Resources page for examples.      |
|                                   |                                   |
|                                   | Has the customer lodged an        |
|                                   | effective claim and provided all  |
|                                   | relevant documents within 28 days |
|                                   | of the birth of the child?        |
|                                   |                                   |
|                                   | Yes, confirm the customer has     |
|                                   | selected the child's date of      |
|                                   | birth as their nominated start    |
|                                   | date for PPL. Go to Step 5        |
|                                   |                                   |
|                                   | No, the customer may not receive  |
|                                   | the full 18 weeks PPL             |
|                                   |                                   |
|                                   | -   Confirm the customer has      |
|                                   |     selected the date of claim as |
|                                   |     their nominated start date    |
|                                   | -   If the customer has selected  |
|                                   |     a later date, try to call the |
|                                   |     customer as a later start     |
|                                   |     date will prevent them        |
|                                   |     receiving the full 18 weeks   |
|                                   |     of PPL                        |
|                                   | -   Go to Step 5                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Return to work + Read more ...    |
|                                   |                                   |
|                                   | Has the customer returned to work |
|                                   | since giving birth to the child?  |
|                                   |                                   |
|                                   | The PPL payment will cease if the |
|                                   | birth mother advises a return to  |
|                                   | work:                             |
|                                   |                                   |
|                                   | -   before the end of the PPL     |
|                                   |     period, or                    |
|                                   | -   during the connected period   |
|                                   |                                   |
|                                   | Tell the customer to claim        |
|                                   | Flexible PPL days before the      |
|                                   | child is 18 weeks old. See        |
|                                   | Parental Leave Pay (PPL) customer |
|                                   | returns to work.                  |
|                                   |                                   |
|                                   | Any Flexible PPL days that are    |
|                                   | rejected or withdrawn after the   |
|                                   | customer has returned to work are |
|                                   | returned to the customerâ€™s        |
|                                   | balance of unclaimed Flexible PPL |
|                                   | days. These can be claimed on     |
|                                   | another eligible date within 18   |
|                                   | weeks of the childâ€™s birth or     |
|                                   | adoption. See Claiming and        |
|                                   | managing Flexible Paid Parental   |
|                                   | Leave (PPL) days.                 |
|                                   |                                   |
|                                   | -   Yes, they may not be eligible |
|                                   |     for PPL. See Parental Leave   |
|                                   |     Pay (PPL) customer returns to |
|                                   |     work. Procedure ends here     |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Child details + Read more ...     |
|                                   |                                   |
|                                   | Children must be named, but if    |
|                                   | care of the child has been        |
|                                   | relinquished as part of the       |
|                                   | process of adoption or a          |
|                                   | surrogacy arrangement and the     |
|                                   | child was never in the care of    |
|                                   | the birth mother, a claim can be  |
|                                   | accepted for the child without a  |
|                                   | name being given.                 |
|                                   |                                   |
|                                   | Has the customer provided a name  |
|                                   | for the child?                    |
|                                   |                                   |
|                                   | -   Yes, add the child to the     |
|                                   |     customer's record with the    |
|                                   |     child's name details          |
|                                   |     provided. Go to Step 7        |
|                                   | -   No, add the child to the      |
|                                   |     customer's record with a      |
|                                   |     first name and surname of     |
|                                   |     'surrogate' (this applies for |
|                                   |     both adopted and surrogate    |
|                                   |     children). Go to Step 7       |
+-----------------------------------+-----------------------------------+
| 7                                 | Check/complete coding             |
|                                   |                                   |
|                                   | For claim processing              |
|                                   | requirements, see Assessing       |
|                                   | family assistance and Paid        |
|                                   | Parental Leave scheme claims.     |
|                                   |                                   |
|                                   | Once this has been completed      |
|                                   |                                   |
|                                   | For claim processing in Customer  |
|                                   | First: + Read more ...            |
|                                   |                                   |
|                                   | Select the new claim activity     |
|                                   | from the Activity List (AL)       |
|                                   | screen and complete any further   |
|                                   | coding required.                  |
|                                   |                                   |
|                                   | 'S'elect the child from the Child |
|                                   | Selection (CHS) screen. On the    |
|                                   | Child Task Selector (CHTS)        |
|                                   | screen, select the following      |
|                                   | screens to check/update details:  |
|                                   |                                   |
|                                   | On the Child in Customer Care     |
|                                   | (CHC) screen, code 'ICC' (In      |
|                                   | Customer Care) in the Care        |
|                                   | Status: field.                    |
|                                   |                                   |
|                                   | Child in Primary Care (!CiPC)     |
|                                   | screen. This page should be       |
|                                   | updated for PPL claims with:      |
|                                   |                                   |
|                                   | -   Date of event                 |
|                                   | -   Carer: claimant               |
|                                   | -   In Primary Care: No           |
|                                   | -   Reason: Surrogacy/Adoption.   |
|                                   |     This will still allow the     |
|                                   |     payment to issue.             |
|                                   |                                   |
|                                   | There is no need to update:       |
|                                   |                                   |
|                                   | -   Child Relationship (!CHRE)    |
|                                   |     screen                        |
|                                   | -   Care Alignment Details        |
|                                   |     (!CADDS) screen               |
|                                   | -   Child Miscellaneous           |
|                                   |     Circumstances (CHM) screen    |
|                                   | -   Medicare Information (MEI)    |
|                                   |     screen                        |
|                                   |                                   |
|                                   | For claim processing in Process   |
|                                   | Direct + Read more ...            |
|                                   |                                   |
|                                   | Within the claim activity select  |
|                                   | the record to be updated from the |
|                                   | []                                |
|                                   |                                   |
|                                   | On the Child in Customer Care     |
|                                   | (CHC) screen, in Care Status      |
|                                   | select In Customer Care           |
|                                   |                                   |
|                                   | Use Super Key to go to the Child  |
|                                   | in Primary Care (CIPC) screen.    |
|                                   | This page should be updated for   |
|                                   | PPL claims with:                  |
|                                   |                                   |
|                                   | -   Date of event                 |
|                                   | -   Carer: Claimant               |
|                                   | -   In Primary Care: No           |
|                                   | -   Left Primary Care Reason:     |
|                                   |     Surrogacy/Adoption. This will |
|                                   |     still allow the payment to    |
|                                   |     issue                         |
|                                   |                                   |
|                                   | There is no need to update:       |
|                                   |                                   |
|                                   | -   Child Relationship (CHRE)     |
|                                   |     screen                        |
|                                   | -   Care Alignment Details        |
|                                   |     (CADDS) screen                |
|                                   | -   Child Miscellaneous           |
|                                   |     Circumstances (CHM) screen    |
|                                   | -   Medicare Information (MEI)    |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Note: Surrogacy arrangements +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A surrogacy arrangement includes  |
|                                   | the formal surrogacy agreement or |
|                                   | supporting evidence of a          |
|                                   | surrogacy agreement. If unable to |
|                                   | make a determination using the    |
|                                   | information provided, access      |
|                                   | technical support who may         |
|                                   | determine that a referral to the  |
|                                   | Level 2 Policy Helpdesk via the   |
|                                   | Online Form is required. See the  |
|                                   | Resources page for a link to the  |
|                                   | form.                             |
+-----------------------------------+-----------------------------------+
| 8                                 | Refer to Families and Child Care  |
|                                   | Portfolio Operations + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Once the claim has been assessed  |
|                                   | for the child, refer the child's  |
|                                   | Customer Reference Number (CRN)   |
|                                   | on the birth mother's record to   |
|                                   | Families and Child Care Portfolio |
|                                   | Operations.                       |
|                                   |                                   |
|                                   | Insert 'New adopted child record' |
|                                   | in the subject line.              |
|                                   |                                   |
|                                   | The CRN will be promoted to a     |
|                                   | 'PER'son record (if it was a      |
|                                   | 'CHI'ld record) and will remain   |
|                                   | visible. Action by Families and   |
|                                   | Child Care Portfolio Operations   |
|                                   | will prevent any future use of    |
|                                   | that CRN.                         |
|                                   |                                   |
|                                   | Referral for other issues:        |
|                                   |                                   |
|                                   | Other CRN issues to refer to the  |
|                                   | Families and Child Care helpdesk  |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   pre-adoptive care             |
|                                   | -   child names                   |
|                                   | -   aliases                       |
|                                   | -   privacy issues                |
