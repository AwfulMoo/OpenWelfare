Enquiry about decision to approve/reject an authority approval - PA

Table 7 This table describes the process for Pharmaceutical Advisers
(PA) when an approved prescriber enquires about a decision to approve or
reject an authority approval.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Approved prescriber is not        |
|                                   | satisfied with the response and   |
|                                   | requests a review of the          |
|                                   | decision - PA + Read more ...     |
|                                   |                                   |
|                                   | The Pharmaceutical Adviser (PA)   |
|                                   | will:                             |
|                                   |                                   |
|                                   | Investigate the outcome of the    |
|                                   | previous decision and consider    |
|                                   | whether the original decision was |
|                                   | appropriate including a review of |
|                                   | any additional information or     |
|                                   | circumstances                     |
|                                   |                                   |
|                                   | Communicate to the approved       |
|                                   | prescriber an overview of the     |
|                                   | reasons the PBS medicine          |
|                                   | restrictions have not been met in |
|                                   | the individual case               |
|                                   |                                   |
|                                   | Outline the role of Services      |
|                                   | Australia in administering the    |
|                                   | PBS in line with the legislation  |
|                                   | and policies for which the        |
|                                   | Department of Health is           |
|                                   | responsible                       |
|                                   |                                   |
|                                   | Confirm that Services Australia   |
|                                   | has no discretion to grant        |
|                                   | special access to subsidised      |
|                                   | medicines outside of the          |
|                                   | restriction criteria              |
|                                   |                                   |
|                                   | Use information available on the  |
|                                   | record and/or authority           |
|                                   | application                       |
|                                   |                                   |
|                                   | Provide advice of other options   |
|                                   | for exploring access to           |
|                                   | subsidised medicines. These may   |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   approaching the sponsor       |
|                                   |     pharmaceutical company        |
|                                   | -   enquire regarding             |
|                                   |     compassionate access programs |
|                                   | -   patient access through the    |
|                                   |     public hospital               |
|                                   | -   consulting with the approved  |
|                                   |     prescriberâ€™s professional     |
|                                   |     body to lobby for a change to |
|                                   |     the restrictions              |
|                                   | -   a private prescription may be |
|                                   |     written                       |
|                                   |                                   |
|                                   | Does the approved prescriber have |
|                                   | information for exceptional       |
|                                   | circumstances that may exist for  |
|                                   | the patient to meet the           |
|                                   | restriction criteria?             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 2                                 | Exceptional circumstances may     |
|                                   | apply + Read more ...             |
|                                   |                                   |
|                                   | If the approved prescriber        |
|                                   | presents information that         |
|                                   | demonstrates that exceptional     |
|                                   | circumstances prevent the patient |
|                                   | from meeting the restriction      |
|                                   | criteria, the PA may consider     |
|                                   | referral to the Pharmaceutical    |
|                                   | Benefit's Branch (PBB) for        |
|                                   | consideration and advice on the   |
|                                   | intent of the restriction. PBB    |
|                                   | may escalate to the Department of |
|                                   | Health if appropriate, where the: |
|                                   |                                   |
|                                   | -   PA is satisfied that facts    |
|                                   |     are/can be substantiated.     |
|                                   |     These may include a           |
|                                   |     contraindication, inability   |
|                                   |     to perform a required test or |
|                                   |     other legitimate compelling   |
|                                   |     reason                        |
|                                   | -   Approved prescriber is        |
|                                   |     prepared to submit a written  |
|                                   |     request including any         |
|                                   |     additional relevant evidence  |
|                                   | -   Approved prescriber is        |
|                                   |     advised that Services         |
|                                   |     Australia has no              |
|                                   |     administrative discretion to  |
|                                   |     authorise the request nor can |
|                                   |     provide comment on the likely |
|                                   |     outcome, however, the PA will |
|                                   |     commit to escalating the      |
|                                   |     request for consideration and |
|                                   |     will provide a response in    |
|                                   |     due course                    |
|                                   |                                   |
|                                   | Is the case suitable for referral |
|                                   | to PBB?                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 3                                 | Referral to Pharmaceutical        |
|                                   | Benefits Branch (PBB) + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Once the PA receives the          |
|                                   | correspondence from the approved  |
|                                   | prescriber, they must summarise   |
|                                   | the reason the case should be     |
|                                   | referred to PBB, including the    |
|                                   | details outlining why the         |
|                                   | application for authority         |
