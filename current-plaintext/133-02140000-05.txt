mySupport referrals

Service Officers who deal with organisations need to help with common
online problems. They can seek help from their local Technical Peer
Support (TPS) or Service Support Officer (SSO).

SSOs can escalate CSBOS issues by completing and submitting the Child
Support System issues webform:

-   Go to mySupport
-   Search using key words, error message or application name and select
    the relevant webform
-   Complete and Submit the webform taking note of any receipt or
    reference code produced
