|                                   | -   death of a partner            |

|                                   | when an exemption from MOR may    |

|                                   |                                   |
|                                   | -   a major personal disruption   |
|                                   |     to a person's home            |
|                                   | -   a major personal crisis       |
|                                   | -   affected by a declared        |
|                                   |     natural disaster              |
|                                   | -   performing temporary carer    |
|                                   |     duties                        |
|                                   | -   fulfilling a Community        |

|                                   | -   performing jury duty          |

|                                   |     emergency                     |

|                                   |     Exempting a job seeker from   |
|                                   |     mutual obligation             |

|                                   | Note: for this exemption, a child |
|                                   | can only be the dependent child   |
|                                   | of one parent.                    |
|                                   |                                   |
|                                   | Does the customer meet the        |
|                                   | criteria above?                   |
|                                   |                                   |
|                                   | Yes, the customer satisfies the   |
|                                   | MOR                               |
|                                   |                                   |
|                                   | -   For customers subject to the  |
|                                   |     SpB MOR (policy), make sure   |
|                                   |     the Payable Under: field on   |

| Step                              | Action                            |

| 1                                 | Customer is subjected to MOR for  |
|                                   | SpB + Read more ...               |
|                                   |                                   |
|                                   | Explain MOR to the customer,      |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   attending Employment Services |
|                                   |     Provider (ESP) appointments   |
|                                   | -   undertaking job search or     |
|                                   |     meeting a monthly points      |
|                                   |     target                        |
|                                   | -   acting on referrals to jobs   |
|                                   | -   undertaking an Annual         |
|                                   |     Activity Requirement          |
|                                   | -   participating in an activity  |
|                                   |     that fully meets their MOR    |
|                                   | -   be actively seeking and       |
|                                   |     willing to undertake suitable |
|                                   |     paid work                     |
|                                   | -   lodging a Reporting Statement |
|                                   |     via []Self Service (online    |
|                                   |     account and phone self        |
|                                   |     service), in person or in     |
|                                   |     writing (SU19)                |
|                                   |                                   |
|                                   | Customers confirm compliance with |
|                                   | their MO or participation         |
|                                   | requirements when completing      |
|                                   | fortnightly reporting.            |
|                                   |                                   |
|                                   | Note: NVH customers will not be   |
|                                   | paid SpB if they do not comply    |
|                                   | with their MOR without a          |
|                                   | reasonable reason. SpB recipients |
|                                   | with MO requirements which are    |
|                                   | based on legislation are subject  |

|                                   | set out in Division 3A of the     |
|                                   | Social Security (Administration)  |
|                                   | Act.                              |

|                                   |     the Process Direct landing    |
|                                   |     page                          |
|                                   | -   enter the customer's CRN,     |
|                                   |     select Go                     |
|                                   | -   in the customer's record      |
|                                   |     select [more options]         |
|                                   |     >Participation Summary>Job    |
|                                   |     seeker registration &         |
|                                   |     provider referral             |
|                                   |     information>Start job seeker  |
|                                   |     registration                  |
|                                   | -   Complete the Job Seeker       |
|                                   |     Snapshot if the customer is   |
|                                   |     present                       |
|                                   | -   Note: if unable to undertake  |
|                                   |     the Job Seeker Snapshot with  |
|                                   |     the customer a JSR/MFU or a   |
|                                   |     JSR UPD activity will be auto |
