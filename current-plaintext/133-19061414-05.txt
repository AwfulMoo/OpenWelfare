OLS1 account history codes used in Consumer Directory Maintenance System (CDMS)

+-----------------------------------+-----------------------------------+
| Code                              | Explanation                       |
+-----------------------------------+-----------------------------------+
| Added program password - program  | Completed registration for        |
| OLS reason FIRST                  | Medicare online account.          |
+-----------------------------------+-----------------------------------+
| Program interaction - program OLS | Status changes to 'deleted' once  |
| type PREQD                        | PSENT process completed.          |
+-----------------------------------+-----------------------------------+
| Program interaction - program OLS | Password sent to customer by      |
| type PSENT                        | mail.                             |
+-----------------------------------+-----------------------------------+
| Amend program password - program  | Password reset by customer:       |
| OLS reason CRESET                 |                                   |
|                                   | -   they created the password     |
|                                   |     during logon or in their      |
|                                   |     Medicare online account       |
|                                   | -   they requested a new password |
|                                   |     letter online via the         |
|                                   |     Forgotten Password page       |
+-----------------------------------+-----------------------------------+
| Added program questions and       | Secret question set up.           |
| answer - program OLS              |                                   |
+-----------------------------------+-----------------------------------+
| Program interaction - program OLS | Email sent to customer to reset   |
| type ESENT                        | password.                         |
+-----------------------------------+-----------------------------------+
| Program interaction - program OLS | Password reset by customer using  |
| type ERESET                       | instructions in email.            |
+-----------------------------------+-----------------------------------+
| Deleted program password -        | Password reset by staff member.   |
| program OLS reason ORESET         | 'P' number shows in the next      |
|                                   | column.                           |
+-----------------------------------+-----------------------------------+
| Deleted program entitlement -     | Medicare online account           |
| program OLS type OLS              | registration cancelled.           |
+-----------------------------------+-----------------------------------+

OLS2 account history codes used in CDMS

+-----------------------------------+-----------------------------------+
| Code                              | Explanation                       |
+-----------------------------------+-----------------------------------+
| Added program registration -      | The customer linked Medicare      |
| program OLS2                      | services to their myGov account   |
|                                   | by confirming their existing      |
|                                   | Medicare online account.          |
+-----------------------------------+-----------------------------------+
| Added program entitlement -       | The customer linked Medicare      |
| program OLS2 type OLS2            | services to their myGov account   |
|                                   | by confirming their Medicare      |
|                                   | record.                           |
+-----------------------------------+-----------------------------------+
| Amended program entitlement -     | The customer linked Medicare      |
| program OLS2 type OLS2            | services to their myGov account.  |
| (CANCELOLSREGISTRATION)           | They have since cancelled their   |
|                                   | Medicare online account.          |
|                                   |                                   |
|                                   | They can reactivate their account |
|                                   | by re-linking Medicare services   |
|                                   | to myGov.                         |
|                                   |                                   |
|                                   | Note: if they chose to cancel and |
|                                   | prevent access, a staff member    |
|                                   | will need to reset access before  |
|                                   | they can re-link Medicare         |
|                                   | services to myGov.                |
+-----------------------------------+-----------------------------------+
