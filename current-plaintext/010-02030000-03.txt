|                                   |     Schooling A Award             |

|                                   | school + Read more ...            |
|                                   |                                   |
|                                   | Is the customer 15 years of age   |
|                                   | or under, and:                    |
|                                   |                                   |
|                                   | -   they had already completed    |
|                                   |     the final year of primary     |
|                                   |     school and moved away from    |
|                                   |     home to undertake secondary   |
|                                   |     schooling but instead needs   |
|                                   |     to repeat the final year of   |

|                                   |     Schooling B Award             |

|                                   |                                   |
|                                   | Is the customer?                  |

|                                   |                                   |
|                                   | in State Care, or                 |
|                                   |                                   |
|                                   | independent as a result of one of |
|                                   | the following circumstances:      |

|                                   | -   they are an orphan            |
|                                   | -   they have, or have had, a     |

|                                   |     to live at home (UTLAH)       |
|                                   | -   their parents cannot exercise |
|                                   |     their responsibilities        |
|                                   |     because they are in prison,   |

|                                   | See ABSTUDY eligibility criteria  |
|                                   | for Schooling B Award             |
|                                   |                                   |
|                                   | No, the customer is not entitled  |
|                                   | to any ABSTUDY award. Check       |
|                                   | customer's entitlement to other   |
|                                   | payments                          |
|                                   |                                   |
|                                   | -   Procedure ends here           |
