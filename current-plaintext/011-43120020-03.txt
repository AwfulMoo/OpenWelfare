This page contains information that Service Officers must follow when
storing, confirming or end dating bank account details for the purpose
of making Medicare payments.

customers

Bank account details provided verbally or on a Medicare form

Table 5: this table describes what to do when a customer provides bank
account details verbally, either over the phone or at a service centre,
or where they are provided on a Medicare form and an error is displayed.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | How is the customer providing     |
|                                   | their bank account details? +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Over the phone or verbally in a   |
|                                   | service centre, go to Step 2      |
|                                   |                                   |
|                                   | Using a Medicare form and an      |
|                                   | error is displayed indicating the |
|                                   | bank account details provided are |
|                                   | invalid. Attempt to call the      |
|                                   | customer at least once. If the    |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   can be contacted, go to Step  |
|                                   |     2                             |
|                                   | -   cannot be contacted, record   |
|                                   |     the contact attempt,          |
|                                   |     including phone number used,  |
|                                   |     on the CDMS Personal tab and  |
|                                   |     send the form back to the     |
|                                   |     customer together with the    |
|                                   |     appropriate standard letter.  |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to standard letters on the        |
|                                   | agency's intranet.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Public security check + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Service Officers are to perform a |

|                                   | Authenticating a Medicare         |
|                                   | customer.                         |
|                                   |                                   |
|                                   | Has the customer passed the       |
|                                   | security check?                   |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, ask them to provide their     |
|                                   | bank account details either by:   |
|                                   |                                   |
|                                   | -   [] accessing their Medicare   |
|                                   |     online account via myGov      |
|                                   | -   [] using their Express Plus   |
|                                   |     Medicare app                  |
|                                   | -   by attending a Service Centre |
|                                   |     with acceptable               |
|                                   |     identification                |
|                                   | -   by calling back later with    |
|                                   |     the required information      |
|                                   | -   by completing a Bank account  |
|                                   |     details collection form       |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | The Resources page contains a     |
|                                   | link to the myGov website and the |
|                                   | Bank account details collection   |
|                                   | form.                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Security check passed + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the customer has been          |
|                                   | contacted or is contacting:       |
|                                   |                                   |
|                                   | -   due to an error indicating    |
|                                   |     the bank account details      |
|                                   |     provided are invalid, go to   |
|                                   |     Step 8                        |
|                                   | -   to provide their bank account |
|                                   |     details or are present at a   |
|                                   |     service centre, go to Step 4  |
+-----------------------------------+-----------------------------------+
| 4                                 | Privacy note + Read more ...      |
|                                   |                                   |
|                                   | Service Officers on the general   |
|                                   | enquiries line are not required   |
|                                   | to read out the privacy note, see |
|                                   | Privacy note and principles on    |
|                                   | the Background page for more      |
|                                   | information. Service Officers     |
|                                   | must tick the box under the       |
|                                   | Essential and Privacy information |

|                                   | -   No, go to Step 6              |

|                                   | -   key “NPBI, card number” and   |
|                                   |     press [Enter]                 |
|                                   | -   go to Step 6                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Update or record bank account     |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Update or record bank account     |
|                                   | details in CDMS.                  |
|                                   |                                   |
|                                   | Service Officers must ask the     |
|                                   | customer to provide the following |
|                                   | information and key their         |
|                                   | responses into CDMS in the        |
|                                   | appropriate fields:               |
|                                   |                                   |
|                                   | -   BSB number                    |
|                                   | -   Account number                |
|                                   | -   Account name                  |
|                                   |                                   |
|                                   | Confirm the following with the    |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   Repeat the BSB number, ask    |
|                                   |     the customer to confirm these |
|                                   |     details are correct           |
|                                   | -   Repeat the account number,    |
|                                   |     ask the customer to confirm   |
|                                   |     these details are correct     |
|                                   | -   Ask the customer 'Do you      |
|                                   |     understand that giving false  |
|                                   |     or misleading information is  |
|                                   |     an offence?'                  |
|                                   |                                   |
|                                   | Service Officers must record      |
|                                   | comments in CDMS, confirming      |
|                                   | that:                             |
|                                   |                                   |
|                                   | the privacy note was either       |
|                                   |                                   |
|                                   | -   read to the customer, or      |
|                                   | -   not read to the customer as   |
|                                   |     the call came through the     |
|                                   |     general enquiry line          |
|                                   |                                   |
|                                   | the customer accepted the bank    |
|                                   | declaration, and                  |
|                                   |                                   |
|                                   | the customer confirmed their BSB  |
|                                   | and account number                |
|                                   |                                   |
|                                   | Select Validate, OK, Validate,    |
|                                   | Update to confirm details updated |
|                                   |                                   |
|                                   | Note: it is critical to ensure    |
|                                   | the correct person has been       |
|                                   | selected when updating bank       |
|                                   | account details within CDMS.      |
|                                   | These details are linked to the   |
|                                   | individual and not the card.      |
|                                   |                                   |
|                                   | If the customer is making a       |
|                                   | claim, key bank account details   |
|                                   | in NPBI screen as appropriate,    |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   T (temporary)                 |
|                                   | -   P (permanent)                 |

|                                   | successfully stored?              |
|                                   |                                   |
|                                   | Yes, and they are:                |
|                                   |                                   |
|                                   | -   not making a claim, procedure |

|                                   | -   are making a claim, go to     |
|                                   |     Step 7                        |
|                                   |                                   |

|                                   | more ...                          |
|                                   |                                   |
|                                   | Continue assessing the Medicare   |

|                                   | Where a customer is making a      |

|                                   | Ask the customer whether their    |
|                                   | financial institution has changed |
|                                   | its BSB or if they wish to        |
|                                   | provide alternative bank account  |
|                                   | details.                          |
|                                   |                                   |
|                                   | Do they want to supply new        |
|                                   | details?                          |
|                                   |                                   |
|                                   | -   Yes, return to Step 5         |
|                                   | -   No, advise customer valid     |

|                                   | or cannot provide valid temporary |

|                                   | payment for the claim will be     |
|                                   | held until EFT details have been  |

Bank account details provided on Public/State Trustee letter for FMO customers

Table 6: this table describes what to do when a Public/State Trustee,
organisation or administration customer is under a Financial Management
Order and provides bank account details.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check the letter is complete +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Does the letter include the       |
|                                   | following?                        |
|                                   |                                   |
|                                   | -   Trustee letterhead            |
|                                   | -   Trustee reference             |
|                                   | -   Full name and DOB of the      |
|                                   |     customer                      |
|                                   | -   Financial Management Order    |
|                                   |                                   |
|                                   | Does the letter contain all of    |
|                                   | the above details?                |
|                                   |                                   |
|                                   | -   No, got to Step 2             |
|                                   | -   Yes, got to Step 3            |
+-----------------------------------+-----------------------------------+
| 2                                 | Letter not complete + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the letter is incomplete:      |
|                                   |                                   |
|                                   | -   send back to the customer     |
|                                   |     with the letter z0834 Bank    |
|                                   |     account details registration  |
|                                   |     – could not be processed      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Record or update bank account     |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Record (or update) bank account   |
|                                   | details in CDMS.                  |
|                                   |                                   |
|                                   | Key:                              |
|                                   |                                   |
|                                   | -   BSB code                      |
|                                   | -   Account number                |
|                                   | -   Account name, Ref             |
|                                   | -   Select Medicare Enrolment     |
|                                   |     Form under the Notification   |
|                                   |     Source                        |
|                                   | -   Add comments to Personal tab: |
|                                   |     bank details added from       |
|                                   |     Public Trustees notification  |
|                                   |                                   |
|                                   | Record confirmation in the        |
|                                   | customer's personal record in     |
|                                   | CDMS:                             |
|                                   |                                   |
|                                   | -   Scroll to the end of the      |
|                                   |     Essential and Privacy         |
|                                   |     information                   |
|                                   | -   Tick, if the application has  |
|                                   |     acknowledged they understand  |
|                                   |     the essential privacy         |
|                                   |     information                   |
|                                   | -   Validate, Validate, Update    |
|                                   |                                   |
|                                   | Procedure ends here.              |
