|                                   |                                   |
|                                   | -   Select Ready in the Customer  |

|                                   |     in Table 6                    |

|                                   | No, and                           |
|                                   |                                   |
|                                   | -   there is an ineligible reason |
|                                   |     displayed, see Step 1 in      |
|                                   |     Table 3                       |
|                                   | -   the ParentsNext Summary page  |
|                                   |     is blank (the Customer        |
|                                   |     Summary page will display a   |
|                                   |     status of Never been a        |
|                                   |     ParentsNext participant), see |
