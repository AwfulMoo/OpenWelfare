Work Bonus Balance screen

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Entitlement Period + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | The period for which the          |
|                                   | customer's rate is calculated and |
|                                   | paid. This is usually 14 days.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Work Bonus this period + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the period is 14 days, the     |
|                                   | Work Bonus is $300.               |
|                                   |                                   |
|                                   | If the period is less, it is the  |
|                                   | number of days x $300/14.         |
+-----------------------------------+-----------------------------------+
| 3                                 | Available Balance + Read more ... |
|                                   |                                   |
|                                   | This is the amount that the       |
|                                   | customer has available to         |
|                                   | discount their eligible income in |
|                                   | the Entitlement Period.           |
|                                   |                                   |
|                                   | It is the Balance at the end of   |
|                                   | the previous period plus the Work |
|                                   | Bonus this period.                |
+-----------------------------------+-----------------------------------+
| 4                                 | Eligible Income + Read more ...   |
|                                   |                                   |
|                                   | This is the eligible income       |
|                                   | earned in the Entitlement Period. |
|                                   |                                   |
|                                   | It includes wages, director's     |
|                                   | fees and leave (unless paid out   |
|                                   | on termination).                  |
|                                   |                                   |
|                                   | From 1 July 2019, Work Bonus also |
|                                   | includes self-employment income   |
|                                   | earned through personal exertion. |
+-----------------------------------+-----------------------------------+
| 5                                 | Assessed Eligible income + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This is the eligible Income less  |
|                                   | the Balance Reduction.            |
|                                   |                                   |
|                                   | If the available balance exceeds  |
|                                   | the eligible income, the Assessed |
|                                   | Income is nil.                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Remaining Work Bonus Balance +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | This is the amount the customer   |
|                                   | can carry into future periods.    |
|                                   |                                   |
|                                   | It is the previous balance        |
|                                   |                                   |
|                                   | -   plus the Work Bonus for the   |
|                                   |     period                        |
|                                   | -   less the balance reduction    |
|                                   |                                   |
|                                   | It is capped at $7,800.           |
+-----------------------------------+-----------------------------------+
