This page contains information for Service Officers about mutual
obligation failures and Participation Solutions Team (PST) skilled
Service Officers who have responsibility for investigating reasonable
excuse and determining if a financial penalty will apply as a result of
a mutual obligation committed by a job seeker in the financial penalty
zone.

On this page:

be applied

|                                   |                                   |

|                                   | warning zone, to avoid payment    |
|                                   | suspension the job seeker must    |

|                                   |     non-compliance, and           |
|                                   | -   meet any re-engagement        |
|                                   |     requirement issued            |
|                                   |                                   |
|                                   | For job seekers in the penalty    |
|                                   | zone, when the provider records   |
|                                   | the non-compliance event in       |
|                                   | ESSweb this will trigger an       |
|                                   | immediate participation           |
|                                   | suspension. This suspension       |
|                                   | prompts an SMS to the job seeker  |

|                                   | Reporting non-compliance -        |
|                                   | Targeted Compliance Framework.    |

|                                   | seeker is required to contact     |
|                                   | their provider:                   |
|                                   |                                   |
|                                   | -   The job seeker is notified of |

|                                   | job interview, fail to act on a   |

|                                   | manner such that employment would |
|                                   | not be offered to them. This      |
|                                   | recognises the seriousness of     |
|                                   | these types of mutual obligation  |
|                                   | failures.                         |

|                                   | seekers who contact to arrange a  |

|                                   |                                   |

Assessing reasonable excuse and determining if a financial penalty will be applied

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Focus of the compliance           |
|                                   | investigation + Read more ...     |
|                                   |                                   |
|                                   | The compliance investigation      |
|                                   | focuses on determining if a       |
|                                   | financial penalty will be applied |
|                                   | for a mutual obligation failure.  |
|                                   |                                   |
|                                   | The compliance investigation also |
|                                   | needs to incorporate the          |
|                                   | appropriate messages to build a   |
|                                   | clear understanding of the        |
|                                   | expectations of job seekers, and  |
|                                   | consequences of non-compliant     |
|                                   | behaviour. Ensuring the job       |
|                                   | seeker fully understands their    |
|                                   | requirements and the consequences |
|                                   | of not meeting them, gives them   |
|                                   | the best chance of complying and  |
|                                   | avoiding further penalties in the |
|                                   | future.                           |
|                                   |                                   |
|                                   | The PST-skilled Service Officer   |
|                                   | must ensure all of the evidence   |
|                                   | required to make a good decision  |
|                                   | is identified and appropriately   |
|                                   | considered by:                    |
|                                   |                                   |
|                                   | -   providing a clear explanation |
|                                   |     of the mutual obligation      |
|                                   |     failure that is under         |
|                                   |     investigation, the specific   |
|                                   |     matters that need to be       |
|                                   |     considered to reach a         |
|                                   |     decision and the potential    |
|                                   |     outcomes of the decision      |
|                                   | -   identifying the facts         |
|                                   |     relevant to the               |
|                                   |     non-compliance event and      |
|                                   |     conducting an investigation   |
|                                   |     focused on the relevant facts |
|                                   | -   giving the job seeker the     |
|                                   |     opportunity to respond to any |
|                                   |     adverse claims that have been |
|                                   |     made against them and to      |
|                                   |     disclose any further          |
|                                   |     information they would like   |
|                                   |     considered                    |
|                                   | -   making contact with the       |
|                                   |     provider or relevant third    |
|                                   |     party to assist in            |
|                                   |     establishing the facts, where |
|                                   |     evidence provided by the job  |
|                                   |     seeker contradicts the        |
|                                   |     reported non-compliance event |
|                                   | -   consultation with a           |
|                                   |     specialist officer or         |
|                                   |     Indigenous Cultural           |
|                                   |     Consultant. This may be       |
|                                   |     appropriate to establish the  |
|                                   |     impact of job seeker          |
|                                   |     circumstances on capacity to  |
|                                   |     comply. See Accessing a       |
|                                   |     PST-skilled social worker for |
|                                   |     information on when           |
|                                   |     consultation with a           |
|                                   |     PST-skilled social worker may |
|                                   |     be required or appropriate    |
|                                   |                                   |
|                                   | For information about specific    |
|                                   | non-compliance events, see:       |
|                                   |                                   |
|                                   | -   Refusing to enter into a Job  |
|                                   |     Plan or Participation Plan    |
|                                   | -   Not meeting job search        |
|                                   |     requirements                  |
|                                   | -   Failing to attend or          |
|                                   |     satisfactorily participate in |
|                                   |     a compulsory activity         |
|                                   | -   Failing to attend or          |
|                                   |     satisfactorily participate in |
|                                   |     a job interview               |
|                                   | -   Non-attendance at compulsory  |
|                                   |     provider managed appointments |
+-----------------------------------+-----------------------------------+
| 2                                 | The Compliance Investigation      |
|                                   | workflow + Read more ...          |
|                                   |                                   |
|                                   | All compliance investigations     |
|                                   | relating to mutual obligation     |
|                                   | failures must be recorded using   |
|                                   | the Compliance Investigation      |
|                                   | workflow. This workflow is        |
|                                   | accessed by selecting the active  |
|                                   | failure from the Participation    |
|                                   | Compliance workflow.              |
|                                   |                                   |
|                                   | On entering the workflow, contact |
|                                   | with the job seeker is            |
|                                   | automatically recorded and key    |
|                                   | information relevant to the       |
|                                   | compliance information presents.  |
|                                   |                                   |
|                                   | Talking points that relate to the |
|                                   | reason the failure was submitted  |
|                                   | should be discussed with the job  |
|                                   | seeker. PST-skilled Services      |
|                                   | Officers are expected to use      |
|                                   | these points to ensure that that  |
|                                   | the job seeker has a clear        |
|                                   | understanding of the              |
|                                   | non-compliance event being        |
|                                   | discussed and the potential       |
|                                   | consequences if a penalty is      |
|                                   | applied.                          |
|                                   |                                   |
|                                   | The PST-skilled Service Officer   |
|                                   | must read the privacy statement   |
|                                   | to the job seeker before          |
|                                   | commencing the investigation and  |
|                                   | ensure any concerns are addressed |
|                                   | before proceeding.                |
|                                   |                                   |
|                                   | The starting point for the        |
|                                   | investigation is the reported     |
|                                   | non-compliance event. The         |
|                                   | Compliance Investigation workflow |
|                                   | displays information recorded by  |
|                                   | the provider on the               |
|                                   | Non-Compliance Report, including  |
|                                   | the compulsory requirement, the   |
|                                   | reason the job seeker provided    |
|                                   | for non-compliance and why this   |
|                                   | was not considered valid.         |
|                                   |                                   |
|                                   | During the compliance             |
|                                   | investigation, the PST-skilled    |
|                                   | Service Officer must record:      |
|                                   |                                   |
|                                   | -   any relevant facts identified |
|                                   |     relating to the mutual        |
|                                   |     obligation failure that were  |
|                                   |     not already considered by the |
|                                   |     provider                      |
|                                   | -   the reasons provided by the   |
|                                   |     job seeker for not complying  |
|                                   |     with a compulsory requirement |
|                                   | -   any evidence collected that   |
|                                   |     supports the facts            |
|                                   | -   a decision to either apply,   |
|                                   |     or not apply a financial      |
|                                   |     penalty                       |
|                                   | -   the rationale for the         |
|                                   |     decision                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Job seeker's reason for           |
|                                   | non-compliance + Read more ...    |
|                                   |                                   |
|                                   | If a mutual obligation failure    |
|                                   | has been committed, the job       |
|                                   | seeker's reasons for committing   |
|                                   | this failure must be considered.  |
|                                   |                                   |
|                                   | Record a concise summary of any   |
|                                   | extra information provided by the |
|                                   | job seeker relevant to the        |
|                                   | non-compliance event, and any     |
|                                   | evidence they would be willing to |
|                                   | provide to support their          |
|                                   | statements.                       |
|                                   |                                   |
|                                   | Identify the main excuse for the  |
|                                   | failure. Select the appropriate   |
|                                   | category and reason from the drop |
|                                   | down options. It is essential     |
|                                   | that the PST-skilled Service      |
|                                   | Officer clearly identifies the    |
|                                   | main excuse. This is because the  |
|                                   | Compliance Investigation workflow |
|                                   | is intuitive and adjusts options  |
|                                   | available to the decision maker   |
|                                   | based on the main reason          |
|                                   | recorded.                         |
|                                   |                                   |
|                                   | Collect enough evidence to        |
|                                   | establish the facts on which a    |
|                                   | decision will be based.           |
|                                   |                                   |
|                                   | If a decision relates to          |
|                                   | misconduct, the investigation     |
|                                   | must focus on whether an incident |
|                                   | of misconduct occurred. A         |
|                                   | financial penalty would not apply |
|                                   | where the job seeker's actions    |
|                                   | were reasonable or not within     |
|                                   | their control.                    |
|                                   |                                   |
|                                   | For all other non-compliance      |
|                                   | events, the investigation must    |
|                                   | focus on whether the job seeker   |
|                                   | has reasonable excuse for the     |
|                                   | non-compliance event, and whether |
|                                   | prior contact occurred. Matters   |
|                                   | that did not significantly and    |
|                                   | directly impact at the time of    |
|                                   | non-compliance cannot be          |
|                                   | considered when assessing         |
|                                   | reasonable excuse. Generally,     |
|                                   | matters that are within the job   |
|                                   | seeker's control would not be     |
|                                   | considered reasonable excuse. For |
|                                   | more information, see Assessing   |
|                                   | reasonable excuse for             |
|                                   | non-compliance with mutual        |
|                                   | obligation requirements.          |
|                                   |                                   |
|                                   | Has the job seeker established    |
|                                   | reasonable excuse for the mutual  |
|                                   | obligation failure, including     |
|                                   | prior contact or misconduct has   |
|                                   | not occurred?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 4                                 | Decision not to apply a financial |
|                                   | penalty + Read more ...           |
|                                   |                                   |
|                                   | In the Participation Compliance   |
|                                   | workflow, a decision not to apply |
|                                   | a financial penalty as a result   |
|                                   | of reasonable excuse would        |
|                                   | generally not be appropriate      |
|                                   | where the following responses are |
|                                   | recorded:                         |
|                                   |                                   |
|                                   | -   Is this an acceptable reason  |
|                                   |     that warrants further         |
|                                   |     investigation: No             |
|                                   | -   Does the job seeker's excuse  |
|                                   |     indicate it had a direct and  |
|                                   |     significant impact on the     |
|                                   |     failure: No                   |
|                                   | -   Does the job seeker's excuse  |
|                                   |     indicate the failure could    |
|                                   |     have been avoided by the job  |
|                                   |     seeker taking appropriate     |
|                                   |     action? Yes                   |
|                                   |                                   |
|                                   | The Compliance Investigation      |
|                                   | workflow will only present        |
|                                   | rejection reasons that are        |
|                                   | relevant based on the             |
|                                   | non-compliance event and the job  |
|                                   | seeker's main reason for the      |
|                                   | failure which was selected.       |
|                                   |                                   |
|                                   | Decisions not to apply a          |
|                                   | financial penalty are actively    |
|                                   | monitored and care must be taken  |
|                                   | to ensure there is enough         |
|                                   | evidence to substantiate this     |
|                                   | decision.                         |
|                                   |                                   |
|                                   | Free text must be sent to the     |
|                                   | provider where appropriate.       |
|                                   |                                   |
|                                   | If reasonable excuse is           |
|                                   | established, a financial penalty  |
|                                   | is not applied.                   |
+-----------------------------------+-----------------------------------+
| 5                                 | Advise job seeker of the decision |
|                                   | not to apply a financial          |
|                                   | penalty + Read more ...           |
|                                   |                                   |
|                                   | It is important to discuss the    |
|                                   | reason for the decision with the  |
|                                   | job seeker in a way that          |
|                                   | consistently reinforces           |
|                                   | expectations that they:           |
|                                   |                                   |
|                                   | -   take personal responsibility  |
|                                   |     for meeting compulsory        |
|                                   |     requirements and engaging     |
|                                   |     with their provider           |
|                                   | -   talk to their provider about  |
|                                   |     any circumstances impacting   |
|                                   |     ability to comply with        |
|                                   |     requirements                  |
|                                   | -   actively participate with the |
|                                   |     compulsory requirements in    |
|                                   |     their Job/Participation Plan  |
|                                   | -   contact prior to a            |
|                                   |     requirement or appointment    |
|                                   |     scheduled start time if       |
|                                   |     unable to attend              |
|                                   |                                   |
|                                   | Talking points displayed in the   |
|                                   | Participation Compliance workflow |
|                                   | should be used to ensure          |
|                                   | consistent messages are provided. |
|                                   |                                   |
|                                   | If unable to verbally advise the  |
|                                   | decision and/or review and appeal |
|                                   | rights, the PST-skilled Service   |
|                                   | Officers should finalise the      |
|                                   | decision in the workflow.         |
|                                   | Finalising the decision will      |
|                                   | issue a letter to the job seeker  |
|                                   | advising of the adverse decision  |
|                                   | and review and appeal rights.     |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | Reasonable excuse not             |
|                                   | established + Read more ...       |
|                                   |                                   |
|                                   | In the Participation Compliance   |
|                                   | workflow, a decision to apply a   |
|                                   | financial penalty as a result of  |
|                                   | reasonable excuse would generally |
|                                   | be appropriate where any of the   |
|                                   | following responses are recorded: |
|                                   |                                   |
|                                   | -   Is this an acceptable reason  |
|                                   |     that warrants further         |
|                                   |     investigation: No             |
|                                   | -   Does the job seeker's excuse  |
|                                   |     indicate it had a direct and  |
|                                   |     significant impact on the     |
|                                   |     failure: No                   |
|                                   | -   Does the job seeker's excuse  |
|                                   |     indicate the failure could    |
|                                   |     have been avoided by the job  |
|                                   |     seeker taking appropriate     |
|                                   |     action? Yes                   |
|                                   |                                   |
|                                   | If reasonable excuse is not       |
|                                   | established, a financial penalty  |
|                                   | will be applied.                  |
+-----------------------------------+-----------------------------------+
| 7                                 | Advise job seeker of the decision |
|                                   | to apply a financial penalty +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A decision to apply a financial   |
|                                   | penalty as a result of a mutual   |

|                                   | verbally explained to the job     |

|                                   | compliance investigation.         |
|                                   |                                   |
|                                   | Talking points displayed in the   |
|                                   | Participation Compliance workflow |
|                                   | must be used to ensure consistent |
|                                   | messages are communicated         |
|                                   | regarding:                        |
|                                   |                                   |
|                                   | -   the decision                  |
|                                   | -   the financial penalty that    |
|                                   |     will be applied, for further  |
|                                   |     information see Targeted      |
|                                   |     Compliance Framework          |
|                                   |     financial penalties and       |
|                                   |     payment cancellations         |
|                                   |                                   |
|                                   | Note: the Participation           |
|                                   | Compliance workflow does not      |
|                                   | currently display the correct     |
|                                   | information about the job         |
|                                   | seeker’s review and appeal        |
|                                   | rights.                           |
|                                   |                                   |
|                                   | If unable to verbally advise the  |
|                                   | decision and/or review and appeal |
|                                   | rights, the PST-skilled Service   |
|                                   | Officers should finalise the      |
|                                   | decision in the workflow.         |
|                                   | Finalising the decision will      |
|                                   | issue a letter to the job seeker  |
|                                   | advising of the adverse decision  |
|                                   | and review and appeal rights.     |
+-----------------------------------+-----------------------------------+
| 8                                 | Job seeker circumstances          |
|                                   | impacting compliance or crisis    |
|                                   | intervention required + Read more |
