This page contains information on the procedure for Service Officers to
reverse the update of an adjusted claim.

DHRI procedures

Reverse the update of an adjusted claim

This table describes the procedure to reverse the update of an adjusted
claim.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Select claim + Read more ...      |
|                                   |                                   |
|                                   | -   Key DHRI,PROVIDER No,CLAIM    |
|                                   |     ID,DOL=DDMMYY                 |
|                                   | -   Press [Enter].                |
|                                   |                                   |
|                                   | The system returns a DHRC screen  |
|                                   | together with the warning message |
|                                   | R TO RVRSE.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Reverse the claim + Read more ... |
|                                   |                                   |
|                                   | -   Over key DHRC with DHRR       |
|                                   | -   Press [Enter].                |
|                                   |                                   |
|                                   | The system returns the DHRI       |
|                                   | control line and screen together  |
|                                   | with the message REVERSED and the |
|                                   | claim status of ADJUST in the     |
|                                   | STATUS field.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Continue processing + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Continue processing with one of   |
|                                   | the following options:            |
|                                   |                                   |
|                                   | -   Key the DHAI control line to  |
|                                   |     recall the claim from pend    |
|                                   |     and re-assess the claim.      |
|                                   | -   Control line DHAI,            |
|                                   |     providernumber, claim ID,     |
|                                   |     DOL=DDMMYY,delete             |
|                                   | -   Delete the claim and          |
|                                   |     recommence adjustment         |
|                                   |     processing from the           |
|                                   |     beginning.                    |
+-----------------------------------+-----------------------------------+
