This page contains the steps for Service Officers to action cases in all
queues in the Compensation Case Management System (CCMS) and determine
if a minor is entitled to a refund when they are the injured person.

Actioning a case for a minor in all queues in the CCMS

Table 1: This table describes the process for Service Officers to action
cases in all queues in the CCMS when a minor is the injured person.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check injured person is a minor + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the injured person under 14    |
|                                   | years of age (a minor)?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 2                                 | Check Medicare card details +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the minor on the same Medicare |
|                                   | card as their parent or guardian? |

|                                   | Is there legal documentation      |

|                                   | Declaration or a Guardianship or  |

|                                   | claimant can act on behalf of the |

Determining entitlement to refund for a minor

Table 2: This table is a guide used by Service Officers to determine the
entitlement to a refund for any payment for a case involving a minor.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Refund to a minor + Read more ... |
|                                   |                                   |
|                                   | If the refund is from a notice of |
|                                   | charge paid, see:                 |
|                                   |                                   |
|                                   | -   Actioning a notice of         |
|                                   |     judgment or settlement table  |
|                                   | -   Notice of judgment or         |
|                                   |     settlement for Medicare       |
|                                   |     Compensation Recovery         |
|                                   |                                   |
|                                   | Where the injured person is a     |
|                                   | minor who is entitled to a refund |
|                                   | from an intended advance payment, |
|                                   | issue the refund in the minor’s   |
|                                   | name. Send the refund to the      |
|                                   | claimant (parent or guardian) or  |
|                                   | the authorised financial          |
|                                   | administrator, usually the Public |
|                                   | Trustee.                          |
|                                   |                                   |
|                                   | Note: update the Claimant Name    |
|                                   | field to the minor’s name in the  |
|                                   | CCMS - Team 1 Supervisor queue    |
|                                   | before any refund is actioned.    |
|                                   | See Step 2 in the Actioning a     |
|                                   | case for a minor table.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Verify entitlement to an advance  |
|                                   | payment refund + Read more ...    |
|                                   |                                   |
|                                   | When Services Australia has not   |
|                                   | been notified who is              |
|                                   | administering the minor's         |
|                                   | financial affairs, verify who is  |
|                                   | legally entitled to receive an    |
|                                   | advance payment refund by         |
|                                   | telephoning the:                  |
|                                   |                                   |
|                                   | -   compensation payer (or their  |
|                                   |     solicitor), or                |
|                                   | -   the claimant’s authorised     |
|                                   |     third party (authorised       |
|                                   |     solicitor)                    |
|                                   |                                   |
|                                   | The Public Trustee in the minor's |
|                                   | State or Territory can only be    |
|                                   | contacted if documents have been  |
|                                   | received (including the notice of |
|                                   | judgment or settlement)           |
|                                   | confirming the Public Trustee is  |
|                                   | involved.                         |
|                                   |                                   |
|                                   | Note: if documents have not been  |
|                                   | received this is a breach of      |
|                                   | privacy.                          |
|                                   |                                   |
|                                   | Do not process a refund before    |
|                                   | receiving supporting documents.   |
|                                   | Refer to Payments, refunds and    |
|                                   | debts for Medicare Compensation   |
|                                   | Recovery.                         |
|                                   |                                   |
|                                   | Go to Step 3.                     |
|                                   |                                   |
|                                   | Note: if the Service Officer is   |
|                                   | unsure about these requirements,  |
|                                   | email Programme Support Officer   |
|                                   | (PSO) or Team Leader or contact   |
|                                   | the PSO hotline.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Issuing advance payment refunds   |
|                                   | after supporting documentation is |
|                                   | received + Read more ...          |
|                                   |                                   |
|                                   | In cases where there is no Public |
|                                   | Trustee, issue an advance payment |
|                                   | refund in the minor’s name, to be |
|                                   | sent to the:                      |
|                                   |                                   |
|                                   | -   Claimant (normally a parent   |
|                                   |     or guardian)                  |
|                                   | -   Legal representative          |
|                                   |     (including a Power of         |
|                                   |     Attorney or Litigation        |
|                                   |     Guardian)                     |
|                                   | -   Financial administrator or    |
|                                   |     executor (normally appointed  |
|                                   |     by the courts), or            |
|                                   | -   Minor, where no legal         |
|                                   |     representative exists         |
|                                   |                                   |
|                                   | Refer to Payments, refunds and    |
|                                   | debts for Medicare Compensation   |
|                                   | Recovery for more detailed        |
|                                   | information about when Electronic |
|                                   | Funds Transfer banking details    |
|                                   | are required, or when a cheque    |
|                                   | may be issued.                    |
