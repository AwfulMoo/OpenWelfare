Example of backdating provisions

+-----------------------------------+-----------------------------------+
| Example                           | Details                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Example - LIC claim lodged on 20  |
|                                   | March 2021                        |
|                                   |                                   |
|                                   | Fred lodges claim for LIC on 20   |
|                                   | March 2021. Fred is residentially |
|                                   | qualified for LIC. Fred provides  |
|                                   | evidence of a paid medical bill   |
|                                   | for treatment on 4 July 2020.     |
|                                   | Fred has provided proof of all    |
|                                   | income from the period 8 weeks    |
|                                   | before 4 July 2020 to 20 March    |
|                                   | 2021 Fred's income shows that     |
|                                   | Fred is qualified for LIC for the |
|                                   | entire period.                    |
|                                   |                                   |
|                                   | -   Date of treatment = 4 July    |
|                                   |     2020                          |
|                                   | -   40 weeks before date of claim |
|                                   |     = 13 June 2020                |
|                                   | -   Claim to be granted from 4    |
|                                   |     July 2020                     |
|                                   |                                   |
|                                   | Income would need to be coded     |
|                                   | from 10 May 2020 (8 weeks before  |
|                                   | 4 July 2020).                     |
|                                   |                                   |
|                                   | LSD would be coded as:            |
|                                   |                                   |
|                                   | -   Date: 4 July 2020             |
|                                   | -   Reason: MED                   |
+-----------------------------------+-----------------------------------+
