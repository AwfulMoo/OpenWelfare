Cost and fines imposed by a court on a customer

Steps for recording costs and fines

This table describes how to record a cost and/or fine.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - recording costs and |
|                                   | fines + Read more ...             |
|                                   |                                   |
|                                   | Start point - Costs and Fines     |
|                                   | window for either the customer or |
|                                   | non-individual (organisation).    |
+-----------------------------------+-----------------------------------+
| 2                                 | Create new file + Read more ...   |
|                                   |                                   |
|                                   | Click the  [New] icon, or click   |
|                                   | the File menu, then New (only     |
|                                   | required if there are existing    |
|                                   | costs or fines records).          |
+-----------------------------------+-----------------------------------+
| 3                                 | Update the Cost and Fine group    |
|                                   | box + Read more ...               |
|                                   |                                   |
|                                   | -   Type the reference number of  |
|                                   |     the court proceedings in the  |
|                                   |     Reference No field            |
|                                   | -   Type the date the cost or     |
|                                   |     fine was imposed in the Date  |
|                                   |     field                         |
|                                   | -   Type the amount of the cost   |
|                                   |     or fine in the Amount field   |
|                                   | -   Select the reason that the    |
|                                   |     cost or fine was imposed from |
|                                   |     the Reason drop down list     |
|                                   | -   Type the name of the court    |
|                                   |     that imposed the cost or fine |
|                                   |     in the Court Name field (up   |
|                                   |     to 50 characters)             |
+-----------------------------------+-----------------------------------+
| 4                                 | Save file + Read more ...         |
|                                   |                                   |
|                                   | Click the  [Save] icon.           |
+-----------------------------------+-----------------------------------+
| 5                                 | Documentation + Read more ...     |
|                                   |                                   |
|                                   | -   Highlight the created record  |
|                                   |     in the Costs and Fines List   |
|                                   |     box                           |
|                                   | -   Click the  [Note] icon to     |
|                                   |     open the  Notes window.       |
|                                   |      Create a new note to         |
|                                   |     document the decision, see    |
|                                   |     Documenting Child Support     |
|                                   |     information in Cuba for more  |
|                                   |     information                   |
|                                   | -   Click the Save button to save |
|                                   |     the note                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Close window + Read more ...      |
|                                   |                                   |
|                                   | Click the  [Close] icon to return |
|                                   | to the Costs and Fines window.    |
+-----------------------------------+-----------------------------------+

Update Costs and Fines

Steps for costs and fines update

This table describes how to update a cost or fine in the relevant
window.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point: for either the       |
|                                   | customer or non – individual      |
|                                   | (organisation) + Read more ...    |
|                                   |                                   |
|                                   | Start point:  Costs and Fines     |
|                                   | window for either the customer or |
|                                   | non – individual (organisation).  |
+-----------------------------------+-----------------------------------+
| 2                                 | Select the record to update +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   To update costs and fines     |
|                                   |     (excluding type), go to Step  |
|                                   |     3                             |
|                                   | -   To update costs and fines     |
|                                   |     (type), go to Step 4          |
+-----------------------------------+-----------------------------------+
| 3                                 | Update costs and fines (excluding |
|                                   | type) + Read more ...             |
|                                   |                                   |
|                                   | -   Select the record to be       |
|                                   |     updated in the Costs and      |
|                                   |     Fines List box                |
|                                   | -   Overtype the fields that      |
|                                   |     require updating              |
|                                   | -   Click the  [Save] icon        |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Update costs and fines (type) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Select the record to be       |
|                                   |     updated in the Costs and      |
|                                   |     Fines List box                |
|                                   | -   Change the amount in the      |
|                                   |     Amount field to $0.00         |
|                                   | -   Click the  [Save] icon        |
|                                   |                                   |
|                                   | Go to Record costs and fines, and |
|                                   | re-enter the record with the      |
|                                   | correct type.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Documentation + Read more ...     |
|                                   |                                   |
|                                   | -   Select the correct record in  |
|                                   |     the Costs and Fines List box  |
|                                   | -   Click the  This image could   |
|                                   |     not be displayed.             |
|                                   |     icon to open the  Notes       |
|                                   |     window.  Create a note to     |
|                                   |     document the decision, see    |
|                                   |      Documenting Child Support    |
|                                   |     information in Cuba for more  |
|                                   |     information                   |
|                                   | -   Click the Save button to save |
|                                   |     the note                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Close window + Read more ...      |
|                                   |                                   |
|                                   | Click the  [Close] icon to return |
|                                   | to the Costs and Fines window.    |
+-----------------------------------+-----------------------------------+
