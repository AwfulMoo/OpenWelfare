Keywords to use when a claim is placed on hold

This table provides the keywords that must be used when a claim is
placed on hold.

+-----------------------------------+-----------------------------------+
| Keyword                           | Description                       |
+-----------------------------------+-----------------------------------+
| AVTPOI                            | POI required                      |
+-----------------------------------+-----------------------------------+
| AVTRES                            | Residence assessment required     |
+-----------------------------------+-----------------------------------+
| AVTQA                             | Quality assessment to be          |
|                                   | completed                         |
+-----------------------------------+-----------------------------------+
| AVTPPL                            | New claim waiting on Pre-payment  |
|                                   | letter to be returned by customer |
+-----------------------------------+-----------------------------------+
| AVTPE                             | Claims awaiting policy            |
|                                   | clarification                     |
+-----------------------------------+-----------------------------------+
| AVTEVI                            | Further evidence required         |
+-----------------------------------+-----------------------------------+
| AVTMED                            | Medical evidence/death            |
|                                   | certificate required              |
+-----------------------------------+-----------------------------------+
| AVTIA                             | Life Impact Assessment required   |
+-----------------------------------+-----------------------------------+
| AVTRJ1                            | Claim is unsuccessful first       |
|                                   | rejection call attempt made and   |
|                                   | unsuccessful - follow up call     |
|                                   | required                          |
+-----------------------------------+-----------------------------------+
| AVTSVI                            | New secondary victim claim        |
|                                   | waiting for Secondary Victim      |
|                                   | Identification of all other Close |
|                                   | Family Members                    |
+-----------------------------------+-----------------------------------+
| AVTINV                            | New secondary victim claim        |
|                                   | waiting on invite to claim        |
|                                   | responses for other secondary     |
|                                   | victims                           |

Letters and electronic messages

The letter or electronic message template available via this link is
endorsed for use by Services Australia and is the latest version
available. Staff should not be using locally produced letters or
electronic messages.

All letters must be scanned and uploaded to the customer's, and if
applicable the nominee's, record. Refer to the AVTOP Processing task
card Steps 8 and 9 for help in uploading a PDF to a customer's record.
