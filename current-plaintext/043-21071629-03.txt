|                                   | cons\32w\icon-specialists.png]The |
|                                   | rate of assistance is based on    |
|                                   | whether goods are shipped in:     |
|                                   |                                   |
|                                   | -   Full Container Load (FCL), or |
|                                   | -   Less than a Container Load    |
|                                   |     (LCL)                         |
|                                   |                                   |
|                                   | Shipping units must be            |
|                                   | efficiently utilised with the     |
|                                   | claimant’s eligible goods to be   |
|                                   | considered full.                  |
|                                   |                                   |
|                                   | For the purpose of assessing a    |
|                                   | TFES claim, a full 6 meter        |
|                                   | container is considered to hold   |
|                                   | an average of 21 tonnes or 30     |
|                                   | cubic meters.                     |
|                                   |                                   |
|                                   | If a claimant utilises at least   |
|                                   | 80% of the container with their   |
|                                   | own, eligible product, it         |
|                                   | qualifies as a full load.         |
|                                   |                                   |
|                                   | Check Claimant Notes for          |
|                                   | information relating to freight   |
|                                   | load.                             |
|                                   |                                   |
|                                   | Review the supporting documents   |
|                                   | to determine if the goods are     |
|                                   | FCL, consider:                    |
|                                   |                                   |
|                                   | -   the shipping unit number      |
|                                   | -   if the container or trailer   |
|                                   |     is full                       |
|                                   | -   the weight indicates that the |
|                                   |     container or trailer used is  |
|                                   |     full                          |
|                                   | -   the volume indicates that the |
|                                   |     container or trailer used is  |
|                                   |     full                          |
|                                   | -   the price paid indicates that |
|                                   |     the claimant has been charged |
|                                   |     for a full container or       |
|                                   |     trailer                       |
|                                   |                                   |
|                                   | Consider accepted transport       |
|                                   | practices and the shipping        |
|                                   | characteristics of the goods.     |
|                                   | e.g. a container full of          |
|                                   | polystyrene would have low        |
|                                   | weight, but high volume.          |
|                                   |                                   |
|                                   | If unsure, contact the freight    |
|                                   | company by phone to confirm if    |
|                                   | the container was fully utilised. |
|                                   |                                   |
|                                   | If the goods were transported as  |
|                                   | a:                                |
|                                   |                                   |
|                                   | -   full container or trailer     |
