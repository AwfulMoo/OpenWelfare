Reviewing or appealing a Departure Prohibition Order (DPO) or Departure Authorisation Certificate (DAC) decision

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Review differences compared to    |
|                                   | other Centrelink decisions + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The review process for DPO or DAC |
|                                   | decisions is different to the     |
|                                   | standard internal review process. |
|                                   |                                   |
|                                   | The decision must be explained to |
|                                   | the customer.                     |
|                                   |                                   |
|                                   | There is no formal internal       |
|                                   | review process. Do not run the    |
|                                   | Internal Review/Explanation       |
|                                   | Script.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contacts regarding DPO/DAC        |
|                                   | decisions + Read more ...         |
|                                   |                                   |
|                                   | Decision notices will contain the |
|                                   | dedicated DPO Team number for the |
|                                   | person to call the DPO team       |
|                                   | directly.                         |
|                                   |                                   |
|                                   | If a person requests a review or  |
|                                   | further explanation of a          |
|                                   | decision, warm transfer them to   |
|                                   | the DPO team.                     |
|                                   |                                   |
|                                   | Customer wanting a review of the  |
|                                   | debt, not the DPO or DAC:         |
|                                   |                                   |
|                                   | -   normal review and appeal      |
|                                   |     processes apply               |
|                                   | -   advise the DPO team and       |
|                                   |     record the details on a DOC   |
+-----------------------------------+-----------------------------------+
| 3                                 | Reconsidering a decision about a  |
|                                   | DPO or DAC + Read more ...        |
|                                   |                                   |
|                                   | An explanation of a decision to   |
|                                   | issue a DPO or reject a DAC       |
|                                   | provides an opportunity to:       |
|                                   |                                   |
|                                   | -   clear up any misunderstanding |
|                                   | -   receive new information       |
|                                   | -   correct errors                |
|                                   |                                   |
|                                   | A decision to issue a DPO can     |
|                                   | only be appealed to:              |
|                                   |                                   |
|                                   | -   Federal Court of Australia,   |
|                                   |     or                            |
|                                   | -   Federal Circuit Court of      |
|                                   |     Australia                     |
|                                   | -   go to Step 5                  |
|                                   |                                   |
|                                   | The customer can only appeal to   |
|                                   | the Administrative Appeals        |
|                                   | Tribunal (AAT) for decisions:     |
|                                   |                                   |
|                                   | -   to reject an application to   |
|                                   |     revoke or vary a DPO          |
|                                   | -   to reject an application for  |
|                                   |     a DAC                         |
|                                   | -   refuse the issue a DAC with a |
|                                   |     substituted return date       |
|                                   | -   about the provision of        |
|                                   |     security                      |
|                                   | -   about the forfeit of security |
|                                   | -   go to Step 4                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Review by AAT + Read more ...     |
|                                   |                                   |
|                                   | Internal review by an Authorised  |
|                                   | Review Officer (ARO) is not       |
|                                   | available for any of these        |
|                                   | decisions.                        |
|                                   |                                   |
|                                   | The AAT will undertake an         |
|                                   | independent merits review of the  |
|                                   | decision. The person should       |
|                                   | contact the AAT about how to make |
|                                   | an application for review. They   |
|                                   | may find more information         |
|                                   | regarding the AAT via their       |
|                                   | website. For contact details, see |
|                                   | Administrative Appeals Tribunal   |
|                                   | (AAT).                            |
|                                   |                                   |
|                                   | These reviews are an AAT second   |
|                                   | (AAT2) review.                    |
|                                   |                                   |
|                                   | Record details on a DOC in the    |
|                                   | person's record.                  |
|                                   |                                   |
|                                   | Has the AAT advised the           |
|                                   | department of a review?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 5                                 | Review by the Federal Court of    |
|                                   | Australia or the Federal Circuit  |
|                                   | Court of Australia + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Advise the customer:              |
|                                   |                                   |
|                                   | -   there are considerable costs  |
|                                   |     involved                      |
|                                   | -   there are other options to    |
|                                   |     resolve the debt issue.       |
|                                   |     Guides are on the References  |
|                                   |     page                          |
|                                   | -   revoking the DPO could be     |
|                                   |     considered if criteria are    |
|                                   |     met                           |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the Federal and Circuit Court  |
|                                   | of Australia websites.            |
|                                   |                                   |
|                                   | Record details on a DOC in the    |
|                                   | person's record.                  |
|                                   |                                   |
|                                   | Is the agency aware of a review?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 6                                 | Record appeal + Read more ...     |
|                                   |                                   |
|                                   | To record a new appeal against a  |
|                                   | DPO or DAC decision:              |
|                                   |                                   |
|                                   | Access the APL system in Customer |
|                                   | Record                            |
|                                   |                                   |
|                                   | Begin a new appeal by entering    |
|                                   | ODM into the Nxt: field           |
|                                   |                                   |
|                                   | The Benefit Type will be either   |
|                                   | DPO or DAC as necessary           |
|                                   |                                   |
|                                   | The Subtype will refer to the     |
|                                   | legislation that the DPO was      |
|                                   | issued under: FAO, PPL, SSA, STU  |
|                                   |                                   |
|                                   | Select the entry for the Orig.    |
|                                   | Decision field from the           |
|                                   | following:                        |
|                                   |                                   |
|                                   | -   ISS: for the issuing of a DPO |
|                                   |     or DAC                        |
|                                   | -   NIS: for not issuing a DAC    |
|                                   | -   REV: for not revoking a DPO   |
|                                   | -   VAR: for not varying a DPO or |
|                                   |     DAC                           |
|                                   | -   SEC: for decisions related to |
|                                   |     DAC security                  |
|                                   |                                   |
|                                   | Enter the Original Decision Date  |
|                                   |                                   |
|                                   | The Reason for Orig. Decn. field  |
|                                   | for a DAC is the most appropriate |
|                                   | selection of the following:       |
|                                   |                                   |
|                                   | -   DA1 = DAC - other             |
|                                   | -   DA2 = failure to provide      |
|                                   |     security                      |
|                                   | -   DA3 = humanitarian grounds    |
|                                   |     not established               |
|                                   | -   DA4 = refusal not detrimental |
|                                   |     to Australia’s interests      |
|                                   | -   DA5 = unlikely to return by   |
|                                   |     required date                 |
|                                   | -   DA6 = security forfeited for  |
|                                   |     failing to return by required |
|                                   |     date                          |
|                                   |                                   |
|                                   | The Reason for Orig. Decn. field  |
|                                   | for a DPO is the most appropriate |
|                                   | selection of the following:       |
|                                   |                                   |
|                                   | -   DP1 = DPO - other             |
|                                   | -   DP2 = failure to meet agreed  |
|                                   |     repayment arrangement         |
|                                   | -   DP3 = failed to repay debt as |
|                                   |     agreed                        |
|                                   | -   DP4 = arrangement is not      |
|                                   |     acceptable                    |
|                                   | -   DP5 = no arrangement in place |
|                                   | -   DP6 = there are outstanding   |
|                                   |     debts                         |
|                                   |                                   |
|                                   | Enter the Reason for Contact      |
|                                   |                                   |
|                                   | Enter the Decision Maker and      |
|                                   | Region of Decision Maker          |
|                                   |                                   |
|                                   | Enter the Date Client Queried     |
|                                   | Decn.                             |
|                                   |                                   |
|                                   | In the Outcome field, enter BA1   |
|                                   | if the appeal is with the AAT or  |
|                                   | BA2 if the appeal is with the     |
|                                   | courts. This will bypass the SME, |
|                                   | ARO, AAT1 and possibly the AAT2   |
|                                   | steps of the appeal as they are   |
|                                   | not required.                     |
|                                   |                                   |
|                                   | All fields have help available by |
|                                   | entering ‘?’.                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Documents required for appeals +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Make the following documents      |
|                                   | available for a review of a       |
|                                   | decision about a DPO or DAC:      |
|                                   |                                   |
|                                   | -   the DPO                       |
|                                   | -   reasons for the decision to   |
|                                   |     make a DPO                    |
|                                   | -   the submission recommending   |
|                                   |     the making of a DPO           |
|                                   | -   notice and reasons for the    |
|                                   |     decision as to revoking or    |
|                                   |     varying a DPO (if applicable) |
|                                   | -   notice and reasons for the    |
|                                   |     decision not to issue a DAC   |
|                                   |     (if applicable)               |
|                                   | -   notice and reasons for the    |
|                                   |     decision not to accept        |
|                                   |     security for return to        |
|                                   |     Australia (if applicable)     |
|                                   | -   the documentary evidence      |
|                                   |     relied upon in making the     |
|                                   |     decision under review and any |
|                                   |     other documents relevant to   |
|                                   |     the application               |
|                                   | -   other documents requested by  |
|                                   |     the Litigation and            |
|                                   |     Information Release Branch    |
|                                   |     (LIR)                         |
|                                   |                                   |
|                                   | The Debt Management Branch        |
|                                   | Litigation Team (DMB LT) can      |
|                                   | request this information from the |
|                                   | DPO team. The DPO team must       |
|                                   | provide the documents to LIR      |
|                                   | within 2 business days of         |
|                                   | notification.                     |
|                                   |                                   |
|                                   | The appeal will be managed by     |
|                                   | FOIL with assistance and          |
|                                   | instructions from DMB LT.         |
