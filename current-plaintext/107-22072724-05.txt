Exemptions to the debt pause in disaster affected postcodes

Table 2: Exemptions apply to some debts where debt recovery continues.

+-----------------------------------+-----------------------------------+
| Exemption type                    | Reason                            |
+-----------------------------------+-----------------------------------+
| Fraud and serious non compliance  | Work relating to fraud and        |
|                                   | serious non-compliance will       |
|                                   | continue to maintain the          |
|                                   | integrity of the welfare system   |
+-----------------------------------+-----------------------------------+
| Assurance of Support              | Debt pause does not apply to an   |
|                                   | assurer with a bond in place, as  |
|                                   | recovery action can be taken      |
|                                   | against the bond. The debt pause  |
|                                   | will apply to an assurer if no    |
|                                   | bond is available                 |
+-----------------------------------+-----------------------------------+
| New Zealand Pension Arrears       | Debt activity is part of          |
|                                   | agreement with the New Zealand    |
|                                   | government, suspension of debt    |
|                                   | action requires agreement from    |
|                                   | New Zealand government authority  |
+-----------------------------------+-----------------------------------+
| Department of Veterans’ Affairs   | These debts are raised to offset  |
| (DVA) Clearance (customer and/or  | DVA impact payments. DVA has      |
| partner)                          | continued to make these payments. |
|                                   | Debts should continue to be       |
|                                   | raised to make the adjustments to |
|                                   | customer’s payment to minimise    |
|                                   | overpayment and future debt to    |
|                                   | the customer                      |
+-----------------------------------+-----------------------------------+
| Bankruptcy                        | Customers declaring bankruptcy    |
|                                   | must be notified of the outcome   |
|                                   | of their debt, and the            |
|                                   | implications on bankruptcy claims |
