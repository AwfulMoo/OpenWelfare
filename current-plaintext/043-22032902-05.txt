Apprenticeships Data Management System (ADMS)

Information for employers on gaining access to ADMS

Logging into ADMS

-   Select the ADMS hyperlink
-   On the ADMS landing page, select the Services Australia logon
    hyperlink
-   When prompted, select the text field and key 'Services Australia'.
    Select Services Australia from the menu of options, and press
    Continue
-   Select the Process Claim Applications button. The BAC/CAC or PWS
    forms page displays

Note: the ADMS system will time out after a period of inactivity

ADMS claim status

Table 2

+-----------------------------------+-----------------------------------+
| Status                            | Definition                        |
+-----------------------------------+-----------------------------------+
| Ready                             | Claim application has been        |
|                                   | generated and is ready to be      |
|                                   | actioned by the employer          |
+-----------------------------------+-----------------------------------+
| Draft                             | Employer has commenced updating   |
|                                   | the claim application, but has    |
|                                   | not yet submitted                 |
+-----------------------------------+-----------------------------------+
| Submitted                         | Claim application has been        |
|                                   | submitted by employer for         |
|                                   | assessment                        |
+-----------------------------------+-----------------------------------+
| Approved                          | Claim application approved        |
+-----------------------------------+-----------------------------------+
| Rejected                          | Claim application rejected        |
+-----------------------------------+-----------------------------------+
| Returned                          | Claim application has been        |
|                                   | returned to employer by a Service |
|                                   | Officer for more information      |
+-----------------------------------+-----------------------------------+
| Verified                          | Claim application marked eligible |
|                                   | (either through automation or by  |
|                                   | a Service Officer), pending       |
|                                   | completion of the DEWR delegation |
|                                   | approval process. Once this       |
|                                   | process completes, the status is  |
|                                   | updated to Approved               |
+-----------------------------------+-----------------------------------+
| Assessment                        | Claim application has been        |
|                                   | submitted and requires manual     |
|                                   | assessment by a Service Officer   |
+-----------------------------------+-----------------------------------+
