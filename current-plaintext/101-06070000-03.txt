|                                   | listed. There will be a blank     |

|                                   | concession entitlement.           |

|                                   | Concession Dependent Summary      |
|                                   | (MCDS) screen.                    |
|                                   |                                   |
|                                   | Note: Concession Card Issue       |
|                                   | Summary (CCIS) screen displays    |
|                                   | actual cards issued, not current  |
|                                   | entitlement.                      |

|                                   | Activating the Department of Home |
|                                   | Affairs datalink and contingency  |
|                                   | procedures if datalink is         |
|                                   | unavailable.                      |
|                                   |                                   |

|                                   | trouble with accessing            |
|                                   | concessional entitlements through |

|                                   | card number on the Medicare       |

|                                   | for the Pharmaceutical Benefits   |
|                                   | Scheme (PBS) and Medicare Safety  |
|                                   | Net for more details. Once the    |
|                                   | Medicare card number has been     |

|                                   | may be a child who has not yet    |
|                                   | developed a signature.            |
|                                   |                                   |
|                                   | Services Australia only provides  |
|                                   | the concession cards, not the     |
|                                   | concessions. It is ultimately the |
|                                   | concession provider who           |
|                                   | determines whether to accept a    |
|                                   | concession card which has not     |
|                                   | been signed by the person named   |
|                                   | on it.                            |

|                                   | Entitlement guided procedure      |
|                                   | completes a suitable DOC for the  |
|                                   | selected entitlement, as          |

|                                   | only if the guided procedure is   |

|                                   | Manual processing of concession   |
|                                   | card table.                       |

Manual processing of concession card

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Manual process to refresh         |
|                                   | concession entitlement + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Updates should be done via the    |
|                                   | Confirm Concession Entitlement    |
|                                   | guided procedure and manual       |
|                                   | updates should only be done if it |
|                                   | is not available. See Step 4 in   |
|                                   | the Confirming concession card    |
|                                   | entitlement and issuing interim   |
|                                   | vouchers table.                   |
|                                   |                                   |
|                                   | To manually refresh Concession    |
|                                   | Entitlements:                     |
|                                   |                                   |
|                                   | Go to the Concession Benefit      |
|                                   | Action (MCBA) screen and key:     |
|                                   |                                   |
|                                   | -   Action: RSH                   |
|                                   | -   Reason: HIC (Health Insurance |
|                                   |     Commission)                   |
|                                   | -   Effect date: Today's date     |
|                                   | -   Go to Assessment Results (AR) |
|                                   |     screen and finalise           |
|                                   |                                   |
|                                   | If a concession entitlement       |
|                                   | refresh has recently been         |
|                                   | performed and the concession      |
|                                   | entitled customer is still        |
|                                   | experiencing difficulties         |
|                                   | accessing PBS pharmaceuticals,    |
|                                   | refer the case to the             |
|                                   | Applications Service Desk via     |
|                                   | normal protocols.                 |
|                                   |                                   |
|                                   | Note: customers should not be     |
|                                   | provided with a Screen dump of    |
|                                   | the Card Details (CCID) screen    |
|                                   | when requesting proof of their    |
|                                   | eligibility for concessions.      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Manual process to issue interim   |
|                                   | voucher and/or replacement card   |
|                                   | required + Read more ...          |
|                                   |                                   |
|                                   | Issue a printed interim card via  |
|                                   | Concession Card Reissue guided    |
|                                   | procedure.See Step 7 in the       |
|                                   | Confirming concession card        |
|                                   | entitlement and issuing interim   |
|                                   | vouchers table.                   |
|                                   |                                   |
|                                   | If the Concession Card Reissue    |
|                                   | guided procedure is unavailable,  |
|                                   | a manual Confirmation of          |
|                                   | Concession Card Entitlement       |
|                                   | (SS055) can be issued. The SS055  |
|                                   | is an accountable form completed  |
|                                   | by hand by a Service Officer.     |
|                                   | This option should only be used   |
|                                   | in instances of system failure or |
|                                   | declared emergency situations     |
|                                   | where there is no print facility  |
|                                   | available to staff. A Service     |
|                                   | Officer should only issue a SS055 |
|                                   | manually if the customer's        |
|                                   | current concession entitlement    |
|                                   | has been verified via the         |
|                                   | Concession Entitlement Summary    |
|                                   | (MCCS) screen.                    |
+-----------------------------------+-----------------------------------+
| 3                                 | Manually record details + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To manually record the issue of   |
|                                   | Confirmation of Concession Card   |
|                                   | Entitlement (SS055):              |
|                                   |                                   |
|                                   | Go to the Concession Card Issue   |
|                                   | Summary (CCIS) screen             |
|                                   |                                   |
|                                   | Key 'I' next to the concession    |
|                                   | card required                     |
|                                   |                                   |
|                                   | The Concession Card Interim       |
|                                   | Voucher (CCIV) screen will        |
|                                   | display. Complete the following   |
|                                   | fields:                           |
|                                   |                                   |
|                                   | -   Serial no: use the serial     |
|                                   |     number on the SS055 form      |
|                                   | -   Valid To: default is 14 days. |
|                                   |     Can be extended for a maximum |
|                                   |     period of 3 months from the   |
|                                   |     date of issue (subject to     |
|                                   |     limit of expiry date)         |
|                                   | -   Reason for Issue: Use field   |
|                                   |     help ('?') for a list of      |
|                                   |     valid values                  |
|                                   |                                   |
|                                   | Complete the Source and DOR       |
|                                   | fields                            |
|                                   |                                   |
|                                   | Print the completed CCIV screen.  |
|                                   | The printout is to be given to    |
|                                   | the Accountable Forms Officer as  |
|                                   | proof of the issue                |
|                                   |                                   |
|                                   | Press [Enter]. The system updates |
|                                   | the customer’s record to show the |
|                                   | issue of an interim voucher on    |
|                                   | the CCIS screen. Ensure only the  |
|                                   | appropriate card illustration is  |
|                                   | date stamped and all applicable   |
|                                   | boxes on the form are filled. The |
|                                   | 'Date of Grant of concession      |
|                                   | card' can be found on the MCCS    |
|                                   | screen. The customer should be    |
|                                   | made aware of the 'Please note'   |
|                                   | provisions at the bottom of the   |
|                                   | SS055 and must sign the SS055     |
|                                   | immediately. The Service Officer  |
|                                   | must then sign the SS055 before   |
|                                   | issuing the completed form to the |
|                                   | customer. There are special       |
|                                   | requirements for what fields are  |
|                                   | to be completed for interim       |
|                                   | vouchers to be issued to          |
|                                   | custodial parents for use by      |
|                                   | non-custodial parents. See        |
|                                   | Concession cards for eligible     |
|                                   | children temporarily in           |
|                                   | non-custodial parent's care       |
|                                   |                                   |
|                                   | Finalise the activity and record  |
|                                   | the details on a DOC              |
|                                   |                                   |
|                                   | Note: if this process is applied  |
|                                   | a requested replacement card, may |
|                                   | arrive at the same time or even   |
|                                   | earlier than the interim voucher. |
|                                   |                                   |
|                                   | If necessary, issue an auto       |
|                                   | replacement card via the CCIS     |
|                                   | screen. See How to reissue        |
|                                   | concession cards.                 |
