Eligibility for Medicare

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for Medicare + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A person is eligible for Medicare |
|                                   | when they:                        |
|                                   |                                   |
|                                   | have lodged an application for a  |
|                                   | combined partner visa 309/100 or  |
|                                   | 820/801, and                      |
|                                   |                                   |
|                                   | are in Australia to live, and     |
|                                   |                                   |
|                                   | -   holds a valid visa            |
|                                   | -   has permission to work, or    |
|                                   | -   has a spouse, parent or child |
|                                   |     that is an Australian         |
|                                   |     citizen, New Zealand citizen  |
|                                   |     that lives in Australia, or   |
|                                   |     holds a permanent resident    |
|                                   |     visa                          |
|                                   |                                   |
|                                   | Eligibility commencement date is: |
|                                   |                                   |
|                                   | -   Temporary Partner Visa        |
|                                   |     Offshore 309: date of arrival |
|                                   |     in Australia (to live)        |
|                                   | -   Temporary Partner Visa        |
|                                   |     Onshore 820: date application |
|                                   |     was lodged with Department of |
|                                   |     Home Affairs                  |
|                                   |                                   |
|                                   | Where the temporary visa subclass |
|                                   | 309 or 820 was not granted at the |
|                                   | time the application for          |
|                                   | permanent residency was lodged,   |
|                                   | eligibility will commence from    |
|                                   | the date the:                     |
|                                   |                                   |
|                                   | -   applicant came to Australia   |
|                                   |     to live (offshore), or        |
|                                   | -   application for permanent     |
|                                   |     residency was lodged          |
|                                   |     (onshore)                     |
|                                   |                                   |
|                                   | The applicant does not need to    |
|                                   | wait until the visa subclass 309  |
|                                   | or 820 is granted to be eligible. |

|                                   |     Migrant                       |

|                                   |     Australia to live             |
|                                   | -   Entitlement End Date: 12      |
|                                   |     months                        |
|                                   | -   Entitlement End reason:       |
|                                   |     entitlement ceased            |
|                                   |                                   |

|                                   | registering customers Medicare    |
|                                   | entitlement.                      |
