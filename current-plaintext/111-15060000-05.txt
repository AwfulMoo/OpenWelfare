Security resources

This table outlines the different types of security resources needed to
undertake action in the Participation Compliance Hub. It includes the
corresponding Curriculum Codes for Services Officer to complete via the
Learning Portal.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| View Access                       | All Services Australia staff with |
|                                   | access to the Centrelink system   |
|                                   | automatically have Participation  |
|                                   | Compliance Hub 'view access'      |
|                                   |                                   |
|                                   | This enables staff to:            |
|                                   |                                   |
|                                   | -   generate compliance action,   |
|                                   |     and                           |
|                                   | -   view screens in an            |
|                                   |     outstanding or completed      |
|                                   |     compliance investigation      |
|                                   |                                   |
|                                   | This access also enables          |
|                                   | Comprehensive Compliance          |
|                                   | Assessment (CCA) Specialists to   |
|                                   | record a CCA, including           |
|                                   | completing the CCA Report.        |
|                                   |                                   |
|                                   | Curriculum codes                  |
|                                   |                                   |
|                                   | -   CLK00914                      |
+-----------------------------------+-----------------------------------+
| Participation Failure Decision    | PFDEC is the first level of       |
| (PFDEC)                           | security access to the workflow   |
|                                   | that requires Product Manager     |
|                                   | approval                          |
|                                   |                                   |
|                                   | This resource is for              |
|                                   | Participation Solutions Team      |
|                                   | (PST)-skilled Service Officers    |
|                                   | who deliver restricted Job Seeker |
|                                   | Compliance Framework services to  |
|                                   | Community Development Program     |
|                                   | (CDP) job seekers only.           |
|                                   |                                   |
|                                   | This resource enables staff to:   |
|                                   |                                   |
|                                   | -   record and finalise a         |
|                                   |     decision to apply or reject a |
|                                   |     No Show, No Pay failure,      |
|                                   |     connection failure,           |
|                                   |     non-attendance failure, or    |
|                                   |     reconnection failure          |
|                                   | -   access the Review workflow to |
|                                   |     revoke a previously-applied   |
|                                   |     No Show, No Pay failure,      |
|                                   |     connection failure,           |
|                                   |     non-attendance failure or     |
|                                   |     reconnection failure          |
|                                   |                                   |
|                                   | Curriculum codes                  |
|                                   |                                   |
|                                   | -   CLK00914                      |
|                                   | -   CLK00920                      |
+-----------------------------------+-----------------------------------+
| Unemployment Non-Payment Period   | UNPP is the second level of       |
| (UNPP)                            | security access to the workflow   |
|                                   | that requires Product Manager     |
|                                   | approval                          |
|                                   |                                   |
|                                   | This resource allows staff to:    |
|                                   |                                   |
|                                   | -   record a compliance           |
|                                   |     investigation for voluntary   |
|                                   |     unemployment or dismissal due |
|                                   |     to misconduct                 |
|                                   | -   Targeted Compliance           |
|                                   |     Framework - make a decision   |
|                                   |     to apply or reject an         |
|                                   |     Unemployment Preclusion       |
|                                   |     Period as a result of an      |
|                                   |     unemployment failure          |
|                                   | -   Job Seeker Compliance         |
|                                   |     Framework - finalise a        |
|                                   |     decision to apply or reject   |
|                                   |     an Unemployment Non-Payment   |
|                                   |     Period (UNPP)                 |
|                                   | -   Targeted Compliance Framework |
|                                   |     and Job Seeker Compliance     |
|                                   |     Framework - access the Review |
|                                   |     workflow to revoke or vary a  |
|                                   |     previously-applied            |
|                                   |     Unemployment Preclusion       |
|                                   |     Period or UNPP                |
|                                   | -   Job Seeker Compliance         |
|                                   |     Framework - grant Payment     |
|                                   |     Pending Review for a UNPP     |
|                                   | -   Job Seeker Compliance         |
|                                   |     Framework - make payments     |
|                                   |     during a UNPP under hardship  |
|                                   |     provisions                    |
|                                   |                                   |
|                                   | Curriculum codes                  |
|                                   |                                   |
|                                   | -   CLK00914                      |
|                                   | -   CLK00916                      |
+-----------------------------------+-----------------------------------+
| Eight Week Non-Payment Period     | WNPPD is the third and final      |
| Decision (WNPPD)                  | level of security access to the   |
|                                   | workflow that requires Product    |
|                                   | Manager approval                  |
|                                   |                                   |
|                                   | When delivering services to job   |
|                                   | seekers and ParentsNext           |
|                                   | participants managed under the    |
|                                   | Targeted Compliance Framework,    |
|                                   | this resource is used to:         |
|                                   |                                   |
|                                   | -   conduct a Capability          |
|                                   |     Assessment                    |
|                                   | -   conduct a compliance penalty  |
|                                   |     investigation                 |
|                                   | -   make a decision to apply a    |
|                                   |     financial penalty or cancel   |
|                                   |     payment as a result of an     |
|                                   |     unemployment failure, work    |
|                                   |     refusal failure or mutual     |
|                                   |     obligation failure under the  |
|                                   |     Targeted Compliance Framework |
|                                   | -   revoke or vary a previously   |
|                                   |     applied financial penalty or  |
|                                   |     non-compliance cancellation   |
|                                   |                                   |
|                                   | When delivering services to CDP   |
|                                   | job seekers managed under the Job |
|                                   | Seeker Compliance Framework, this |
|                                   | resource can be used to perform   |
|                                   | all the same functions as a PFDEC |
|                                   | security resource, but can also   |
|                                   | be used to:                       |
|                                   |                                   |
|                                   | -   record and finalise a         |
|                                   |     decision to apply an          |
|                                   |     Unemployment Non-Payment      |
|                                   |     Period (UNPP) or serious      |
|                                   |     failure                       |
|                                   | -   access the Review workflow to |
|                                   |     revoke a previously-applied   |
|                                   |     UNPP or serious failure       |
|                                   | -   grant Payment Pending Review  |
|                                   | -   waive a serious failure       |
|                                   |     period following agreement to |
|                                   |     participate in a compliance   |
|                                   |     activity, and                 |
|                                   | -   make payments during a        |
|                                   |     serious failure period or     |
|                                   |     UNPP under hardship           |
|                                   |     provisions                    |
|                                   |                                   |
|                                   | Curriculum codes                  |
|                                   |                                   |
|                                   | -   CLK00914                      |
|                                   | -   CLK00918                      |
|                                   | -   CLK00920                      |
+-----------------------------------+-----------------------------------+
