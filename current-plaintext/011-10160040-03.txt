This page contains information on how a Service Officer can identify a
vaccination provider's history on the Australian Immunisation Register
(AIR).

Note: please view PF in Definitions for more information on using these
functions.

On this page:

Access a vaccination provider's registration history

Search for vaccination provider registration and check mail history

Access a vaccination provider's registration history

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log into PDS + Read more ...      |
|                                   |                                   |
|                                   | See the Log on to Provider        |
|                                   | Directory System (PDS) table in   |
|                                   | Amend Medicare providers for AIR  |
|                                   | in PDS.                           |
|                                   |                                   |
|                                   | Does the vaccination provider     |
|                                   | have their immunisation provider  |
|                                   | number?                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, search for the            |
|                                   |     vaccination provider in PDS,  |
|                                   |     go to Step 2                  |
|                                   |                                   |
|                                   | If the vaccination provider       |
|                                   | registration cannot be located,   |
|                                   | see Verify provider identity.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for the vaccination        |
|                                   | provider in PDS + Read more ...   |
|                                   |                                   |
|                                   | In PDS:                           |
|                                   |                                   |
|                                   | -   select Provider Location from |
|                                   |     the main menu                 |
|                                   | -   select View                   |
|                                   | -   select the Detailed Search    |
|                                   |     tab                           |
|                                   |                                   |
|                                   | Key the vaccination provider’s    |
|                                   | details then select ‘Search’ to   |
|                                   | locate their provider number.     |
|                                   | When details return successfully, |
|                                   | go to Step 3.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Search for vaccination provider   |
|                                   | registration + Read more ...      |
|                                   |                                   |
|                                   | To check registration:            |
|                                   |                                   |
|                                   | select Program Registration from  |
|                                   | the main menu                     |
|                                   |                                   |
|                                   | select View                       |
|                                   |                                   |
|                                   | -   the Program Registration View |
|                                   |     page displays                 |
|                                   |                                   |
|                                   | From the Program Registration     |
|                                   | View page:                        |
|                                   |                                   |
|                                   | -   key vaccination provider stem |
|                                   |     (first 6 characters of        |
|                                   |     vaccination provider number)  |
|                                   |     in the Search ID field        |
|                                   | -   select ‘AIR Provider Number’  |
|                                   |     from the ID Type dropdown     |
|                                   |     menu                          |
|                                   | -   select ‘Program Registrations |
|                                   |     for this                      |
|                                   |                                   |
|                                   |   Provider/Practice/organisation’ |
|                                   |     using the radio button        |
|                                   | -   select Search                 |
|                                   |                                   |
|                                   | The PROGRAM REGISTRATIONS FOR     |
|                                   | THIS PROVIDER/PRACTICE page       |
|                                   | displays.                         |
|                                   |                                   |
|                                   | Select View in the Action column  |
|                                   | for AIR under the Program column  |
|                                   | to see the program registration   |
|                                   | details.                          |
+-----------------------------------+-----------------------------------+
| 4                                 | Check vaccination provider        |
|                                   | registration history + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the PROGRAM REGISTRATIONS    |
|                                   | FOR THIS PROVIDER/PRACTICE page:  |
|                                   |                                   |
|                                   | -   select History                |
|                                   | -   select View to see the        |
|                                   |     program registration details  |
|                                   |                                   |
|                                   | The Program Registration History  |
|                                   | page displays.                    |
+-----------------------------------+-----------------------------------+

Search for vaccination provider registration and check mail history

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Display vaccination provider      |
|                                   | enquiry screen + Read more ...    |
|                                   |                                   |
|                                   | Log into Mainframe and key 'YPEI' |
|                                   | and press [Enter].                |
|                                   |                                   |
|                                   | The YPEI - Provider Enquiry       |
|                                   | screen displays.                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check vaccination provider        |
|                                   | number + Read more ...            |
|                                   |                                   |
|                                   | Key the vaccination provider      |
|                                   | number and press [F19]-REG DET.   |
|                                   |                                   |
|                                   | The YPER - Provider Registration  |
|                                   | screen displays.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Vaccination provider displayed +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When the vaccination provider's   |
|                                   | record meets the search criteria  |
|                                   | and there is:                     |
|                                   |                                   |
|                                   | more than one match:              |
|                                   |                                   |
|                                   | -   key (/) the appropriate       |
|                                   |     vaccination provider and      |
|                                   |     press [PF19]-REG DET. The     |
|                                   |     YPER – Provider Registration  |
|                                   |     Details screen displays       |
|                                   |                                   |
|                                   | only one match:                   |
|                                   |                                   |
|                                   | -   key (/) the record returned   |
|                                   |     and press [PF19]-REG DET. The |
|                                   |     YPER - Provider Registration  |
|                                   |     Details screen displays       |
|                                   |                                   |
|                                   | no match found:                   |
|                                   |                                   |
|                                   | -   use a different search        |
|                                   |     criteria                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Check mail history + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the YPER - Provider          |
|                                   | Registration Details screen press |
|                                   | [PF15] – MAIL HIST.               |
|                                   |                                   |
|                                   | The YPEM – Provider Mail History  |
|                                   | Details screen displays.          |
+-----------------------------------+-----------------------------------+
