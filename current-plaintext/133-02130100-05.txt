This page contains links to contact details, and examples of escalating
privacy, security and unauthorised access incidents, fraud incidents and
system incidents for a user's Provider Digital Access (PRODA) account
and email templates.

Examples of escalating PRODA incidents

+-----------------------------------+-----------------------------------+
| Item                              | Incident example and action       |
+-----------------------------------+-----------------------------------+
| 1                                 | Escalating privacy, security and  |
|                                   | unauthorised access incidents     |
|                                   |                                   |
|                                   | User believes someone has gained  |
|                                   | access to their PRODA account and |
|                                   | suspicious activity has been      |
|                                   | undertaken using their            |
|                                   | credentials (system has been      |
|                                   | hacked) .                         |
|                                   |                                   |
|                                   | Tier 2 Program Support Officer    |
|                                   | (PSO4) to escalate by phone to    |
|                                   | Tier 3, see Help for staff        |
|                                   | Provider Digital Access (PRODA).  |
+-----------------------------------+-----------------------------------+
| 2                                 | Escalating fraud incidents        |
|                                   |                                   |
|                                   | User reports that their account,  |
|                                   | or their own information, has     |
|                                   | been made available to            |
|                                   | unauthorised sources.             |
|                                   |                                   |
|                                   | This is a fraud incident and the  |
|                                   | Tier 2 PSO4 must escalate via     |
|                                   | email to Tier 3, see Help for     |
|                                   | staff Provider Digital Access     |
|                                   | (PRODA).                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Escalating system incidents -     |
|                                   | user system outage                |
|                                   |                                   |
|                                   | Tier 2 PSO4 to escalate via email |
|                                   | to Tier 3, see Help for staff     |
|                                   | Provider Digital Access (PRODA).  |
|                                   |                                   |
|                                   | Include in email:                 |
|                                   |                                   |
|                                   | -   brief description of issue    |
|                                   |     including contact details for |
|                                   |     the user, dates and times of  |
|                                   |     the incident (and if possible |
|                                   |     provide screen shots)         |
|                                   | -   brief description of all your |
|                                   |     correspondence and response   |
|                                   |     to the user                   |
+-----------------------------------+-----------------------------------+
| 4                                 | Escalating system incidents -     |
|                                   | technical fault                   |
|                                   |                                   |
|                                   | Tier 2 PSO4 to escalate via email |
|                                   | to Tier 3, see Help for staff     |
|                                   | Provider Digital Access (PRODA).  |
|                                   |                                   |
|                                   | Include in email:                 |
|                                   |                                   |
|                                   | -   brief description of issue    |
|                                   |     including contact details for |
|                                   |     the user, dates and times of  |
|                                   |     the incident (and if possible |
|                                   |     provide screen shots)         |
|                                   | -   brief description of all your |
|                                   |     correspondence and response   |
|                                   |     to the user                   |
