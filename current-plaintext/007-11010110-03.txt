This page contains information about the non-lodger or late lodger
process which applies if a customer's correct Family Tax Benefit (FTB)
rate cannot be calculated because actual income details needed for the
customer or their current partner have not provided by the end of the
allowable period.

| 5                                 | Late lodgers + Read more ...      |
|                                   |                                   |
|                                   | If a customer or their partner is |
|                                   | required to lodge a tax return    |
|                                   | and lodges after the end of the   |
|                                   | allowable period:                 |
|                                   |                                   |
|                                   | The non-lodger debt will be set   |
|                                   | aside                             |
|                                   |                                   |
|                                   | FTB will be calculated using      |
|                                   | actual income details. If actual  |
|                                   | entitlement is lower than the     |
|                                   | amount they were paid based on    |
|                                   | their estimated income, a         |
|                                   | reconciliation debt will be       |
|                                   | raised                            |
|                                   |                                   |
|                                   | If they received less than their  |
|                                   | legislative rate of FTB due to a  |
|                                   | payment choice or prohibition     |
|                                   | applied during the year, they     |
|                                   | will be entitled to the lesser of |
|                                   | their entitlement based on the    |
|                                   | estimate and their actual         |
|                                   | entitlement based on their actual |
|                                   | adjusted taxable income (AATI).   |
|                                   | This is not a top-up              |
|                                   |                                   |
|                                   | FTB lump sum claims for the       |
|                                   | 2012-13 or later financial years  |
|                                   | will be made not effective        |
|                                   |                                   |
|                                   | FTB lump sum claims for the       |
|                                   | 2011-12 or earlier financial      |
|                                   | years will be paid the lower of   |
|                                   | the rate calculated based on      |
|                                   | estimated income and the rate     |
|                                   | based on their AATI               |
|                                   |                                   |
|                                   | Return to work (FTB Part B        |
|                                   | quarantine) assistance will not   |
|                                   | be included                       |
|                                   |                                   |
|                                   | Generally Part A and Part B       |
|                                   | supplements cannot be paid. If    |
|                                   | they were paid during interim     |
|                                   | reconciliation, the total amount  |
|                                   | paid is compared with entitlement |
|                                   | based on actual income, and the   |
|                                   | lesser amount paid. Depending on  |
|                                   | the circumstances, a debt may be  |
|                                   | raised for supplements paid       |
|                                   |                                   |
|                                   | -   FTB supplements can be paid   |
|                                   |     to late lodgers in some       |
|                                   |     circumstances. For example if |
|                                   |     a customer re-partners after  |
|                                   |     the lodgement year / extended |
|                                   |     lodgement year (for 2011-12   |
|                                   |     or earlier)                   |
|                                   | -   Cases can be referred to the  |
|                                   |     Level 2 Policy Helpdesk if it |
|                                   |     is unclear whether FTB        |
|                                   |     supplements have been paid    |
|                                   |     correctly                     |
|                                   | -   Requests to the MIV team to   |
|                                   |     have the reconciliation       |
|                                   |     result and the debt amended   |
|                                   |     in these cases via a FAO      |
|                                   |     Reconciliation Manual         |
|                                   |     Adjustment (FRMA) screen      |
|                                   |     adjustment must be referred   |
|                                   |     with approval from the        |
|                                   |     Families and Child Care Level |
|                                   |     2 Policy Helpdesk             |
|                                   | -   If actual entitlement is less |
|                                   |     than the amount paid during   |
|                                   |     the year, any deferred amount |
|                                   |     will be used to offset the    |
|                                   |     excess FTB paid               |
|                                   |                                   |
|                                   | If it is identified that the      |
|                                   | supplements have been paid        |
|                                   | incorrectly (or an activity is    |
|                                   | attempting to pay them            |
|                                   | incorrectly), these are to be     |
|                                   | referred directly to the MIV team |
|                                   | using Fast Note - select Auto     |
|                                   | text, use Families >              |
|                                   | Reconciliation > ACTDOC Manual    |
|                                   | Correction on FRMA action to      |
|                                   | raise them as a debt or suppress  |
|                                   | payment                           |

|                                   | (FMDA) screen + Read more ...     |
|                                   |                                   |
|                                   | Once the late lodger or late      |
|                                   | notifier FTB reconciliation       |
|                                   | completes, the FMDA screen        |
|                                   | displays the FTB entitlement the  |
|                                   | customer chose to defer until     |
|                                   | after the end of the relevant     |
|                                   | financial year and/or amounts     |
|                                   | unpaid due to Prohibition (PIP).  |
|                                   |                                   |
|                                   | Instalment amounts unpaid due to  |
|                                   | More Choice for Families (MCFF)   |
|                                   | or Prohibition (PIP) may be paid  |
|                                   | in late lodger or late notifier   |
|                                   | reconciliation and can be used    |
|                                   | for debt offsetting. The maximum  |
|                                   | amount payable is the lower of    |
|                                   | the amount calculated based on    |
|                                   | the estimate and the amount       |
|                                   | calculated based on the actual    |
|                                   | income. This is not a top-up.     |
|                                   |                                   |
|                                   | 'S'elect a period to view the FTB |
|                                   | Deferred Amounts Details (FDAD)   |
|                                   | screen.                           |

| 9                                 | Manual Intervention (MIV)         |
|                                   | activity - #RECLLM + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | From June 2014, if a late         |
|                                   | lodger/late notifier assessment   |
|                                   | is made for the customer or their |
|                                   | partner for the 2011-12 and later |
|                                   | financial years, manual           |
|                                   | intervention is required if the   |
|                                   | customer had Mandatory Continuous |
|                                   | Adjustment (MCA) deductions in    |
|                                   | the relevant year.                |
|                                   |                                   |
|                                   | The reconciliation activity will  |
|                                   | be automatically transferred to   |
|                                   | MIV for investigation.            |
|                                   |                                   |
|                                   | Manual intervention is necessary  |
|                                   | as reconciliation calculations    |
|                                   | cannot distinguish between MCA    |
|                                   | deductions, More Choice for       |
|                                   | Families (MCFF) deductions and    |
|                                   | prohibition (PIP) deductions when |
|                                   | determining if a late lodger is   |
|                                   | eligible for a payment.           |
|                                   |                                   |
|                                   | MIV staff will be required to     |
|                                   | manually check the result to      |
|                                   | ensure MCA deductions are         |
|                                   | adjusted and not paid as a        |
|                                   | top-up.                           |
