This page contains information about emergency messaging in emergency
situations and the procedures a Service Officer should follow in cases
where a complaint is not resolved. It also contains a link to mySupport.

Emergency provisions when the Authority prescription phone line is unavailable

Emergency provisions information for prescribers and pharmacists

This table describes the emergency provisions information the Service
Officer should provide the prescribers and pharmacists when Authority
prescription phone line is unavailable.

+-----------------------------------+-----------------------------------+
| Who                               | Emergency provision information   |
+-----------------------------------+-----------------------------------+
| Prescribers                       | Prescribers who contact the       |
|                                   | Authorities prescription phone    |
|                                   | line 1800 888 333 and hear the    |
|                                   | ‘Emergency message’, as stated    |
|                                   | below, must adhere to the         |
|                                   | instructions in order to          |
|                                   | prescribe an Authority required   |
|                                   | item.                             |
|                                   |                                   |
|                                   | Emergency Message:                |
|                                   |                                   |
|                                   | Current temporary measure until   |
|                                   | further notice                    |
|                                   |                                   |
|                                   | This is a COVID-19 specific       |
|                                   | arrangement, please ensure you    |
|                                   | listen to the full message. The   |
|                                   | PBS Authority Approval Telephone  |
|                                   | line is currently unavailable.    |
|                                   | For this prescription only, you   |
|                                   | may prescribe the required        |
|                                   | quantity and/or repeats as        |
|                                   | appropriate for clinical          |
|                                   | treatment and within PBS limits.  |
|                                   | Ensure you keep sufficient        |
|                                   | documentation for future          |
|                                   | compliance activities. Annotate   |
|                                   | the prescriptions with E2020CV,   |
|                                   | and include date and time of your |
|                                   | call. To hear these instructions  |
|                                   | again please hold, otherwise hang |
|                                   | up now.’                          |
|                                   |                                   |
|                                   | Emergency Message:                |
|                                   |                                   |
|                                   | ‘For increased quantities and/or  |
|                                   | repeats of an item you will need  |
|                                   | to call back later. For this      |
|                                   | prescription only, you may        |
|                                   | prescribe up to the maximum       |
|                                   | quantity and/or repeats for the   |
|                                   | item as listed in the current     |
|                                   | Pharmaceutical Benefits Scheme    |
|                                   | (PBS) schedule at pbs.gov.au. Do  |
|                                   | this by endorsing the             |
|                                   | prescription with the words       |
|                                   | ‘Emergency authorised by Services |
|                                   | Australia’ and the date and time  |
|                                   | of your call. For Highly          |
|                                   | Specialised Drug Authority        |
|                                   | required items please also        |
|                                   | endorse the prescription with the |
|                                   | letters ‘HSD’, the hospital       |
|                                   | provider number and number of     |
|                                   | days’ treatment.’                 |
|                                   |                                   |
|                                   | Key Points                        |
|                                   |                                   |
|                                   | Increased quantities and/or       |
|                                   | repeats above the listed PBS      |
|                                   | Schedule quantity and/or repeats  |
|                                   | cannot be written under the       |
|                                   | Emergency Provision arrangements. |
|                                   | You will need to either phone     |
|                                   | back later or send in the         |
|                                   | prescription by post.             |
|                                   |                                   |
|                                   | For Authority Required items      |
|                                   | written for up to the listed      |
|                                   | quantities and repeats, you need  |
|                                   | to endorse the Authority          |
|                                   | prescription with the following   |
|                                   | details:                          |
|                                   |                                   |
|                                   | -   ‘Emergency authorised by      |
|                                   |     Services Australia’.          |
|                                   | -   The date and time of the      |
|                                   |     call.                         |
|                                   |                                   |
|                                   | In addition, for Highly           |
|                                   | Specialised Drug Authority        |
|                                   | required items, you will also     |
|                                   | need to endorse the prescription  |
|                                   | and include:                      |
|                                   |                                   |
|                                   | -   ‘HSD’                         |
|                                   | -   the Hospital provider number, |
|                                   |     and                           |
|                                   | -   the number of days treatment  |
|                                   |     the quantity will provide.    |
+-----------------------------------+-----------------------------------+
| Pharmacists                       | When you receive an Authority     |
|                                   | prescription written under the    |
|                                   | Emergency Provision arrangements, |
|                                   | you need to ensure that:          |
|                                   |                                   |
|                                   | the quantity and repeats do not   |
|                                   | exceed the listed quantity and    |
|                                   | repeats in the PBS Schedule       |
|                                   |                                   |
|                                   | the prescription is endorsed      |
|                                   | with:                             |
|                                   |                                   |
|                                   | -   ‘Emergency authorised by      |
|                                   |     Services Australia’           |
|                                   | -   the date and time of the call |
|                                   |     are recorded on the           |
|                                   |     prescription                  |
|                                   |                                   |
|                                   | In addition, for Highly           |
|                                   | Specialised Drug Authority        |
|                                   | required items, the prescription  |
|                                   | must also be endorsed with:       |
|                                   |                                   |
|                                   | -   ‘HSD’                         |
|                                   | -   the hospital provider number, |
|                                   |     and                           |
|                                   | -   the number of days treatment  |
|                                   |                                   |
|                                   | When dispensing a prescription    |
|                                   | which meets all of the            |
|                                   | requirements as listed above you  |
|                                   | will need to enter the emergency  |
|                                   | provision authority approval      |
|                                   | number, P9999RX.                  |

Service Officer unable to resolve a complaint

This table describes the procedures a Service Officer should follow in
cases where a complaint is not resolved.

+-----------------------------------+-----------------------------------+
| Situation where the provider….    | Action                            |
+-----------------------------------+-----------------------------------+
| Wants the complaint recorded but  | -   Record as much detail as      |
| does not want to speak to a       |     possible.                     |
| supervisor.                       | -   Email the details to the Team |
|                                   |     Leader or Manager who should  |
|                                   |     record the details of the     |
|                                   |     complaint in the Feedback     |
|                                   |     Register.                     |
+-----------------------------------+-----------------------------------+

| supervisor.                       | Leader or Manager who should      |
