|                                   | Example 2:                        |
|                                   |                                   |
|                                   | Evan and Jamie both work 90 hours |
|                                   | per fortnight in a family         |

|                                   | per fortnight. As this is their   |
|                                   | normal work routine, there is no  |
|                                   | entitlement to more than 100      |
|                                   | hours of subsidised child care,   |
|                                   | and they would be responsible for |
|                                   | fees above 100 hours.             |
|                                   |                                   |
|                                   | Example 3:                        |
|                                   |                                   |
|                                   | Rebecca and Allan both work 110   |
|                                   | hours per fortnight (including    |
|                                   | travel time) running their own    |
|                                   | business. Due to one of the other |
|                                   | family members involved in        |
|                                   | running their business falling    |
|                                   | ill, they need to work an extra   |
|                                   | 20 hours (2 days) per fortnight   |
|                                   | work each. This means they are    |
|                                   | now working and travelling 134    |
|                                   | hours/fortnight.                  |
|                                   |                                   |
|                                   | After evidence is provided, both  |
|                                   | Rebecca and Allan are assessed as |
|                                   | eligible for an additional 24     |
|                                   | hours per fortnight (2 x 12       |
|                                   | (work/travel)) is granted to each |
|                                   | of them, giving them access to    |
|                                   | 124 hours per fortnight of        |
|                                   | subsidised care.                  |
|                                   |                                   |
|                                   | Note: the additional hours        |
|                                   | granted is for the extra hours    |
|                                   | they are required to work due to  |
|                                   | exceptional circumstances, not    |
|                                   | the total work/travel (134) they  |
|                                   | are undertaking. This exceptional |
|                                   | circumstance Activity Test result |
|                                   | would be limited to the time it   |
|                                   | takes for new staffing            |
|                                   | arrangements to be made to        |
|                                   | replace the ill family member.    |
|                                   |                                   |
|                                   | Example 4:                        |
|                                   |                                   |
|                                   | Sue works 90 hours per fortnight  |
|                                   | in a family business. Sue's       |
|                                   | child, Andrew, attends a local    |
|                                   | day care centre near the family   |
|                                   | home. It takes 75 minutes to      |
|                                   | travel from the day care centre   |
|                                   | to the family business. Although  |
|                                   | this means Sue is undertaking     |
|                                   | work and travel for a total of    |
|                                   | 115 hours per fortnight, this     |
|                                   | does not warrant an increase in   |
|                                   | subsidised hours as it is not due |
|                                   | to exceptional circumstances.     |
|                                   |                                   |
|                                   | Example 5:                        |
|                                   |                                   |
|                                   | Rodney is a single parent who     |
|                                   | works 80 hours per fortnight,     |
|                                   | travelling 2 hours per day from   |
|                                   | the child care service to work.   |
|                                   | Rodney is entitled to 100 hours   |
|                                   | subsidised child care per         |
|                                   | fortnight.                        |
|                                   |                                   |
|                                   | Rodney's house is damaged in a    |
|                                   | storm making it inhabitable,      |
|                                   | requiring Rodney to move in       |
|                                   | temporarily with relatives until  |
|                                   | repairs are completed (estimated  |
|                                   | to be 3 months). This temporary   |
|                                   | relocation means Rodney now has   |
|                                   | to travel 90 minutes each way     |
|                                   | from the child care service to    |
|                                   | work.                             |
|                                   |                                   |
|                                   | In this situation, after evidence |
|                                   | has been provided, Rodney is      |
|                                   | granted an additional 10 hours    |
|                                   | subsidised care (110 hours per    |
|                                   | fortnight) for 3 months, when     |
|                                   | repairs are expected to be        |
|                                   | completed. These additional hours |
|                                   | cover the additional travel hours |
|                                   | only due to the exceptional       |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | If the repairs are completed      |
|                                   | earlier than expected and Rodney  |
|                                   | moves back home before the        |
|                                   | 3-month period, Rodney should     |
|                                   | notify Services Australia and the |
|                                   | exceptional circumstances         |
|                                   | Activity Test result will be      |
|                                   | ended.                            |

payments, see Rates and thresholds.
