|                                   |     in Healthcare Identifiers     |

Verify an IHI via Medicare enrolment

Table 3: This table describes how Service Officers verify an IHI via a
Medicare enrolment. Where a person with unverified IHI is now entitled
to Medicare.

This function must be performed by an operator authorised to register a
person in Medicare.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to the system + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Log on to CDMS                |
|                                   | -   On the CDMS home page, select |
|                                   |     Register New Consumer         |
|                                   | -   The Registration Type screen  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Has the individual provided their |
|                                   | IHI + Read more ...               |
|                                   |                                   |
|                                   | Select Medicare in the Program    |
|                                   | field.                            |
|                                   |                                   |
|                                   | Has the individual provided their |
|                                   | IHI number?                       |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   enter the IHI in the ID       |
|                                   |     Number field                  |
|                                   | -   select IHI in the ID Number   |
|                                   |     Type field                    |
|                                   | -   select Get Details. The       |
|                                   |     Record New Consumer screen    |
|                                   |     displays defaulting to the    |
|                                   |     Personal Details tab.         |
|                                   | -   complete the Medicare         |
|                                   |     enrolment process             |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   select New Consumer. The      |
|                                   |     Search for Consumer           |
|                                   |     (Register) screen displays    |
|                                   | -   go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Search for the individual + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Perform a detailed search for the |
|                                   | individual.                       |
|                                   |                                   |
|                                   | Is a matching consumer record     |
|                                   | found?                            |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   select the radio button       |
|                                   |     beside the correct individual |
|                                   | -   select Add To List            |
|                                   | -   the Registration Type screen  |
|                                   |     displays                      |
|                                   |                                   |
|                                   | No, refine the search until the   |
|                                   | required record is matched        |
+-----------------------------------+-----------------------------------+
| 4                                 | Continue with the Medicare        |
|                                   | enrolment + Read more ...         |
|                                   |                                   |
|                                   | Select Entitlement and continue   |
|                                   | with the Medicare enrolment.      |
|                                   |                                   |
|                                   | See also: Register a customer for |
|                                   | Medicare in the Consumer          |
|                                   | Directory Maintenance System      |
|                                   | (CDMS).                           |
+-----------------------------------+-----------------------------------+
