How to re-categorise work and rename a reference ID

and rename a reference ID.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Re-categorising work + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Identify that the piece of work   |
|                                   | being held is sitting in a work   |
|                                   | type that does not match the      |
|                                   | location of the Service Officer.  |
|                                   | To re-categorise the work         |
|                                   | identified:                       |
|                                   |                                   |
|                                   | -   select Unassign               |
|                                   | -   key reason Re-categorised     |
|                                   | -   search for 'Auth'             |
|                                   | -   select the correct work type  |
|                                   |     from the list, for example    |
|                                   |     PBSBC_PBS_Auth_Mailed_SA      |
|                                   | -   select Submit                 |
|                                   |                                   |
|                                   | The file is now in the correct    |
|                                   | work stream to reflect its        |
|                                   | physical location.                |
+-----------------------------------+-----------------------------------+
| 2                                 | Renaming a reference ID + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Identify the piece of work has an |
|                                   | incorrect 'Primary Reference      |
|                                   | Field' (that is, the Authority    |
|                                   | script number is incorrect).      |
|                                   |                                   |
|                                   | Service Officers need to ensure   |
|                                   | the work is assigned to           |
|                                   | themselves.                       |
|                                   |                                   |
|                                   | In the Work Item Details screen:  |
|                                   |                                   |
|                                   | -   type over the Primary         |
|                                   |     Reference field (that is,     |
|                                   |     Authority Script Number)      |
|                                   | -   select Save                   |
