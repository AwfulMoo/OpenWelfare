Standard letters - Child Change in Care Arrangement script

This table describes the standard letters produced by the Child - Change
in Care Arrangements script, and when it is appropriate to send.

+-----------------------+-----------------------+-----------------------+
| Letter code           | Letter details        | When to use each      |
|                       |                       | letter                |
+-----------------------+-----------------------+-----------------------+
| Q805                  | Shared Care Grant     | Use this letter to    |
|                       | (Agreed full %)       | advise carers of the  |
|                       |                       | shared care decision  |
|                       |                       | based on either       |
|                       |                       | evidence provided     |
|                       |                       | from all carers or    |
|                       |                       | agreement between all |
|                       |                       | carers.               |
+-----------------------+-----------------------+-----------------------+
| Q806                  | Shared care Grant -   | Use this letter to    |
|                       | No third party        | advise carers of a    |
|                       | response              | shared care decision  |
|                       |                       | based on evidence     |
|                       |                       | provided by one of    |
|                       |                       | the carers only. That |
|                       |                       | is where the other    |
|                       |                       | carer has not         |
|                       |                       | responded to a        |
|                       |                       | request for evidence. |
+-----------------------+-----------------------+-----------------------+
| Q807                  | SSA Shared Care       | This letter must be   |
|                       | Principal Carer       | sent to all carers in |
|                       | Determination         | a shared care         |
|                       |                       | arrangement, where at |
|                       |                       | least one carer is    |
|                       |                       | claiming/receiving an |
|                       |                       | income support        |
|                       |                       | payment. The letter   |
|                       |                       | advises which carer   |
|                       |                       | is determined as the  |
|                       |                       | principal carer of    |
|                       |                       | the child/children.   |
+-----------------------+-----------------------+-----------------------+
| Q808                  | SSA Couple Principal  | This letter must be   |
|                       | Carer Determination   | sent to members of a  |
|                       |                       | couple advising which |
|                       |                       | parent is determined  |
|                       |                       | as the principal      |
|                       |                       | carer of the          |
|                       |                       | child/children.       |
+-----------------------+-----------------------+-----------------------+
| Q810                  | Shared Care Dispute - | Use this letter to    |
|                       | Request for           | request information   |
|                       | information           | and/or evidence about |
|                       |                       | shared care           |
|                       |                       | arrangements from any |
|                       |                       | carer.                |
+-----------------------+-----------------------+-----------------------+

You may be eligible for family assistance based on your circumstances.

To find out if you are eligible for other payments and services, please
go servicesaustralia.gov.au/paymentfinder.
