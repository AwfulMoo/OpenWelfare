|                                   |                                   |
|                                   | Sylvia, a full-time student, is   |
|                                   | entitled to a total of $544.40    |
|                                   | per fortnight (maximum            |
|                                   | Independent Away-From-Home rate   |

|                                   | Sylvia is paid $670 every         |
|                                   | fortnight. Sylvia's Income Bank   |

|                                   |                                   |
|                                   | To calculate the amount payable   |
|                                   | to Sylvia:                        |
|                                   |                                   |
|                                   | -   Earned ordinary Income (EI) = |
|                                   |     $670.00 before tax            |
|                                   | -   No Income Bank reduction as   |
|                                   |     balance is currently zero. No |
|                                   |     Income Bank accrual occurs as |
|                                   |     employment income is over the |
|                                   |     $452.00 Personal Income Test  |
|                                   |     lower threshold               |
|                                   | -   Sylvia's EI is greater than   |
|                                   |     $542.00, therefore 60c in the |
|                                   |     $1 reduction applies          |
|                                   | -   (EI - $542) multiplied by 60% |
|                                   |     then add $43.50 = Affecting   |
|                                   |     Income (AI)                   |
|                                   | -   ($670 - $542) x 60%) + $43.50 |
|                                   |     = $120.30                     |
|                                   | -   Sylvia's payment of $544.40   |
|                                   |     per fortnight will be reduced |
|                                   |     by $120.30                    |
|                                   | -   Sylvia is entitled to receive |
|                                   |     $424.10                       |

| 2                                 | YA student transfers to YA        |
|                                   | (other) jobseeker then to YA      |
|                                   | student                           |
|                                   |                                   |
|                                   | Rashid is 19 and defers from a    |
|                                   | Bachelor of Engineering degree    |
|                                   | half way through the second year. |
|                                   | Rashid attends a Services         |
|                                   | Australia office to register as a |
|                                   | job seeker. At the time of        |
|                                   | deferral from the course,         |
|                                   | Rashid's Income Bank balance was  |
|                                   | 3,400 credits.                    |
|                                   |                                   |
|                                   | As soon as Rashid's activity      |

|                                   | credits.                          |
|                                   |                                   |

|                                   | study the following year, any     |

|                                   | will be converted to the Income   |

|                                   | of 3,500 credits.                 |

| 3                                 | ABSTUDY, full time Australian     |
|                                   | Apprentice, single rate           |
|                                   |                                   |
|                                   | Jennifer is 27 years old and a    |
|                                   | single independent full-time      |
|                                   | Australian Apprentice in receipt  |
|                                   | of ABSTUDY. The maximum rate for  |
|                                   | Jennifer's circumstances is       |
|                                   | $638.30 (ABSTUDY maximum rate     |
|                                   | $629.50 plus the Energy           |
|                                   | Supplement $8.80) per fortnight.  |
|                                   | Jennifer has recently started     |
|                                   | being paid $850 per fortnight.    |
|                                   | Jennifer's Income Bank balance at |
|                                   | the beginning of the fortnight    |
|                                   | was $266.                         |
|                                   |                                   |
|                                   | Student Income Bank: Jennifer's   |
|                                   | income of $850 is greater than    |
|                                   | both the income free area and     |
|                                   | Income Bank balance ($266).       |
|                                   | Therefore, Jennifer is taken to   |
|                                   | have drawn down from the Income   |
|                                   | Bank the difference between the   |
|                                   | income and the $452 personal      |
|                                   | income lower threshold. This      |
|                                   | reduces the Income bank to zero   |
|                                   | ($266 - $452 = - $186 or zero).   |
|                                   |                                   |
|                                   | Ordinary income: Jennifer had     |
|                                   | $266 in the Student Income Bank.  |
|                                   | Jennifer's income of $850 is      |
|                                   | reduced by the Income Bank        |
|                                   | balance of $266 leaving an        |
|                                   | affecting income of $584. This    |
|                                   | will result in a reduction from   |
|                                   | the maximum rate of $79.50:       |
|                                   | ($584 - $542) X 0.6 + $43.50 =    |
|                                   | $68.70.                           |
|                                   |                                   |
|                                   | The rate for the fortnight is     |
|                                   | $550.10: ($638.30 - $68.70 =      |
|                                   | $569.60).                         |

| 4                                 | ABSTUDY full time student ceases  |
|                                   | study then recommences            |
|                                   |                                   |
|                                   | Mike is a full-time student in    |
|                                   | receipt of ABSTUDY payments. Mike |
|                                   | has been on ABSTUDY for some time |
|                                   | and has had no employment income  |
|                                   | to date, so the Student Income    |
|                                   | Bank balance is 11,300 credits.   |
|                                   |                                   |
|                                   | Mike then withdraws from the      |
|                                   | course of study, and the ABSTUDY  |
|                                   | payments are cancelled from the   |
|                                   | last day of study. Mike works for |
|                                   | 5 weeks, but then decides to      |
|                                   | return to full-time study. Mike   |
|                                   | reclaims ABSTUDY payments, which  |
|                                   | are granted from the date that    |
|                                   | study recommences.                |
|                                   |                                   |
|                                   | As there is a gap in ABSTUDY      |
|                                   | entitlement, Mike's Income Bank   |
|                                   | balance is set to zero from the   |
|                                   | date of recommenced ABSTUDY       |
|                                   | entitlement. Mike is not entitled |
|                                   | to carry over the previous Income |
|                                   | Bank balance of 11,300.           |
|                                   |                                   |
|                                   | If Mike had recommenced studies   |
|                                   | within 28 days of withdrawing     |
|                                   | from the previous course, Mike    |
|                                   | would be entitled to receive      |
|                                   | ABSTUDY through the gap between   |
|                                   | courses. This would mean the      |
|                                   | Income Bank balance would         |
|                                   | continue over this period, and    |
|                                   | both the Income Bank and any      |
|                                   | income from part-time employment  |
|                                   | would be taken into account when  |
|                                   | determining the rate of ABSTUDY   |
|                                   | for this time.                    |
+-----------------------------------+-----------------------------------+
| 5                                 | ABSTUDY full time student travels |
|                                   | overseas temporarily              |
|                                   |                                   |
|                                   | Sally is a full-time student in   |
|                                   | receipt of ABSTUDY payments.      |
|                                   | Sally has been on ABSTUDY for     |
|                                   | some time and has had no          |
|                                   | employment income to date, so the |
|                                   | Student Income Bank balance is    |
|                                   | 11,300 credits.                   |
|                                   |                                   |
|                                   | Sally travels to England in the   |
|                                   | mid semester break. While         |
|                                   | overseas, Sally's ABSTUDY         |
|                                   | payments are suspended for not    |
|                                   | meeting the portability rules. As |
|                                   | Sally's payments were suspended   |
|                                   | (and not cancelled), the student  |
|                                   | income bank balance should be     |
|                                   | restored to 11,300 credits, the   |
|                                   | amount it was immediately before  |
|                                   | payments were suspended (taking   |
|                                   | into account any income that may  |
|                                   | have normally affected Sally's    |
|                                   | payments).                        |
