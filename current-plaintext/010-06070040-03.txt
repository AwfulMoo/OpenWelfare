|                                   | evidence is provided.             |

|                                   |     activities or commence an     |
|                                   |     Australian Apprenticeship to  |

|                                   | -   Explain the consequences of   |
|                                   |     failing to notify changes,    |
|                                   |     or, of failing to satisfy     |
|                                   |     activity test requirements.   |

|                                   |     failures for students and     |
|                                   |     Australian Apprentices        |
|                                   | -   Record details on a DOC       |
|                                   |                                   |
|                                   | Procedure ends here.              |
