How to assess and finalise an EIIM

Table 2: This table describes the steps required to assess earnings,
raise debts and finalise an Earned Income Intervention (Mail based)
(EIIM).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Locate employment information +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Employment information can be     |
|                                   | supplied verbally or via          |
|                                   | documented verification.          |
|                                   |                                   |
|                                   | Verbal updates                    |
|                                   |                                   |
|                                   | Check the intervention DOC for    |
|                                   | the required information.         |
|                                   |                                   |
|                                   | Documented verification           |
|                                   |                                   |
|                                   | All returned employment           |
|                                   | information documents from the    |
|                                   | customer and/or employer will be  |
