This document outlines information about actioning a vaccination
provider’s enquiry using registration/mail history on the Australian
Immunisation Register (AIR).

Vaccination provider history enquiry

The Provider Registration History screen in Provider Directory System
(PDS) identifies if a change has been made to a recognised vaccination
provider's registration details.

Information displayed includes:

-   date and time
-   operator number
-   description of the change made

Note: if the change is system generated the operator number will display
the source code MPPD.

Vaccination provider registration enquiry

Complete a Mainframe search using the following criteria:

-   vaccination provider number (if known)
-   business name, postcode/postcode range
-   surname, postcode/postcode range
-   surname, given name, postcode/postcode range
