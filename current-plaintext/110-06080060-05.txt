This page contains a list of the Review Types which can be manually
started using the Manage Review (IRMR) screen.

Manually starting a review

The following are the applicable Review Types which can be manually
started using the Manage Review (IRMR) screen:

-   'IDF' (Identity Fraud)
-   'COR' (Corrective Services)
-   'LTO' (State Government Matching)
-   'RGO' (Registrar General's Office) reviews
