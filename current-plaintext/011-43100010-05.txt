Medicare Two-way patient claims field and message definitions

Table 2: This table describes the messages and fields on the Patient
Claim screen.

+-----------------------------------+-----------------------------------+
| Message                           | Description                       |
+-----------------------------------+-----------------------------------+
| 6-FUND STM                        | Displays on all patient claims    |
|                                   | screens prior to P TO PAY for all |
|                                   | in hospital claims providing the  |
|                                   | in-hospital service is payable.   |
+-----------------------------------+-----------------------------------+
| FUND STM                          | After the 6-FUND STM message      |
|                                   | displays, the cursor is           |
|                                   | automatically positioned under    |
|                                   | the highlighted FUND STM field.   |
|                                   | To complete entry, key 'Y' or 'N' |
|                                   | in the FUND STM field:            |
|                                   |                                   |
|                                   | -   Y - gap payment information   |
|                                   |     is to be forwarded to the     |
|                                   |     private health insurer        |
|                                   | -   N - no reason to forward gap  |
|                                   |     information to the private    |
|                                   |     health insurer                |
+-----------------------------------+-----------------------------------+
| 8-ADDR CHK                        | When the processing screen is     |
|                                   | first accessed, the cursor is     |
|                                   | positioned on the ADDRESS CHECK   |
|                                   | field and 8-ADDR CHK displays.    |
|                                   | Claim to be processed in the      |
|                                   | usual manner.                     |
|                                   |                                   |
|                                   | Note: FUND STM message is         |
|                                   | suppressed from the patient       |
|                                   | claims screen at this stage.      |
+-----------------------------------+-----------------------------------+
| 2-FUND PAT                        | Displays for a claim for services |
|                                   | for more than one patient and     |
|                                   | both consist of in-hospital       |
|                                   | items.                            |
|                                   |                                   |
|                                   | Note: claims which cover services |
|                                   | for more than one patient do not  |
|                                   | need to be split when only one    |
|                                   | patient claims in-hospital items: |
|                                   |                                   |
|                                   | -   the office copy receipt and   |
|                                   |     the customer copy cover all   |
|                                   |     patients within the claim     |
|                                   | -   the gap information receipt   |
|                                   |     shows only data for the       |
|                                   |     patient with in-hospital      |
|                                   |     services                      |
+-----------------------------------+-----------------------------------+
