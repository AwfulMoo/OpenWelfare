Methods to store documents lodged by customers

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Attached or batched documents +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | At the end of each day, documents |
|                                   | received in a service centre or   |
|                                   | Smart Centre that have been       |
|                                   | actioned must be stored in a      |
|                                   | secure location in the office.    |
|                                   | They are stored on site until     |
|                                   | they are forwarded to Records     |
|                                   | Management Unit Victoria (RMU     |
|                                   | Vic) in a tamper-proof satchel or |
|                                   | destroyed on site (scanned        |
|                                   | documents only).                  |
|                                   |                                   |
|                                   | The Resources page contains       |
|                                   | information on storing            |
|                                   | information.                      |
|                                   |                                   |
|                                   | Note: documents received in       |
|                                   | relation to a Restricted Access   |
|                                   | customer must be attached to the  |
|                                   | customer's file at the relevant   |
|                                   | service centre. Do not scan or    |
|                                   | send them to RMU Vic.             |
|                                   |                                   |
|                                   | Determine if the document to be   |
|                                   | stored is to be attached to the   |
|                                   | file (on-file storage), or        |
|                                   | batched (off-file storage). The   |
|                                   | Resources page contains the rules |
|                                   | for on-file and off-file storage. |
|                                   |                                   |
|                                   | If the document is to be:         |
|                                   |                                   |
|                                   | -   attached to the file, go to   |
|                                   |     Step 2                        |
|                                   | -   batch stored, go to Step 3    |
|                                   | -   destroyed on site (for a      |
|                                   |     scanned document only), go to |
|                                   |     Step 4                        |
|                                   | -   retrieved from a batch, go to |
|                                   |     Step 7                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Registered/ not registered        |
|                                   | paperwork + Read more ...         |
|                                   |                                   |
|                                   | Clearly print the individual's    |
|                                   | name and their Customer Reference |
|                                   | Number (CRN) in the top           |
|                                   | right-hand corner of the          |
|                                   | document:                         |
|                                   |                                   |
|                                   | ensure the location of the        |
|                                   | document is updated in their      |
|                                   | electronic record                 |
|                                   |                                   |
|                                   | if the paperwork was              |
|                                   |                                   |
|                                   | -   registered when received,     |
|                                   |     update the existing DOC with  |
|                                   |     the storage details           |
|                                   | -   not registered on receipt,    |
|                                   |     create a DOC to record the    |
|                                   |     storage details               |
|                                   |                                   |
|                                   | store the document in the         |
|                                   | office's centralised retention    |
|                                   | area for the 42 day holding       |
|                                   | period                            |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Batch storage + Read more ...     |
|                                   |                                   |
|                                   | Sort the documents for batch      |
|                                   | storage into scanned and          |
|                                   | unscanned documents. Sort each    |
|                                   | group alphabetically, regardless  |
|                                   | of payment type.                  |
|                                   |                                   |
|                                   | Place each set of documents       |
|                                   | (scanned and unscanned) into      |
|                                   | separate manila folders. Include  |
|                                   | a batch header/cover sheet as the |
|                                   | first document for each folder.   |
|                                   | This identifies the contents.     |
|                                   |                                   |
|                                   | If the thickness of the documents |
|                                   | in a folder exceeds 8cm, split    |
|                                   | the batch into sub-alpha sections |
|                                   | (for example, A-K, L-Z).          |
|                                   |                                   |
|                                   | Label the cover sheet(s) with:    |
|                                   |                                   |
|                                   | -   the service centre or Smart   |
|                                   |     Centre                        |
|                                   | -   Office Code                   |
|                                   | -   type of documents batched     |
|                                   | -   date and alpha range (if      |
|                                   |     applicable)                   |
|                                   |                                   |
|                                   | Have the documents in the batch   |
|                                   | been scanned?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Scanned documents + Read more ... |
|                                   |                                   |
|                                   | Store the scanned documents/batch |
|                                   | on site for 70 days from the date |
|                                   | the documents were scanned.       |
|                                   |                                   |
|                                   | When a batch has been stored for  |
|                                   | 70 days, they are ready for       |
|                                   | destruction:                      |
|                                   |                                   |
|                                   | -   remove the documents from     |
|                                   |     storage                       |
|                                   | -   place all the scanned         |
|                                   |     documents and any batch       |
|                                   |     headers into the secure waste |
|                                   |     container on site             |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Non-scanned documents + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | For documents that are not        |
|                                   | scanned:                          |
|                                   |                                   |
|                                   | -   Store the completed batch on  |
|                                   |     site for 42 calendar days.    |
|                                   |     This allows easy access to    |
|                                   |     the documents for review      |
|                                   |     purposes                      |
|                                   | -   Place documents in a          |
|                                   |     tamper-proof satchel and send |
|                                   |     to RMU Vic                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Document required from            |
|                                   | centralised retention area + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is a document required from the   |
|                                   | centralised retention area before |
|                                   | sending it to RMU Vic?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, go to Step 8              |
+-----------------------------------+-----------------------------------+
| 7                                 | Retrieval of document from the    |
|                                   | batch folder/attach to file       |
|                                   | documents + Read more ...         |
|                                   |                                   |
|                                   | Once the action on the document   |
|                                   | has been finalised, arrange for   |
|                                   | the document to be stored on-file |
|                                   | or in a new batch for the current |
|                                   | date.                             |
|                                   |                                   |
|                                   | Record the new location of the    |
|                                   | document on the customer's        |
|                                   | electronic record, either by      |
|                                   | annotating the original DOC or    |
|                                   | creating a new DOC.               |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 8                                 | Due date reached to send batch    |
|                                   | folder/attach to file documents   |
|                                   | to RMU Victoria + Read more ...   |
|                                   |                                   |
|                                   | On the due date, collect the      |
|                                   | batch /attach to file documents.  |
|                                   |                                   |
|                                   | For documents that are:           |
|                                   |                                   |
|                                   | attach to file                    |
|                                   |                                   |
|                                   | -   combine them into 1 group     |
|                                   | -   place the cover sheet on the  |
|                                   |     top of the group to identify  |
|                                   |     the contents. See the         |
|                                   |     Resources page for the cover  |
|                                   |     sheet                         |
|                                   | -   label the cover sheet with    |
|                                   |     - the service centre or Smart |
|                                   |     Centre                        |
|                                   |     - Office Code                 |
|                                   |     - type of documents batched   |
|                                   |     - date                        |
|                                   | -   place documents in a          |
|                                   |     tamper-proof satchel          |
|                                   |                                   |
|                                   | batched, combine them into the 1  |
|                                   | batch                             |
|                                   |                                   |
|                                   | The standard policy is for each   |
|                                   | office to send 1 batch per day.   |
|                                   | Except for larger offices which   |
|                                   | have a large volume of            |
|                                   | correspondence.                   |
|                                   |                                   |
|                                   | Note: the batch must be prepared  |
|                                   | at the end of each day and not    |
|                                   | left for the next day.            |
+-----------------------------------+-----------------------------------+
| 9                                 | Records Management Unit Victoria  |
|                                   | (RMU Vic) + Read more ...         |
|                                   |                                   |
|                                   | Send the documents/batches to t   |
|                                   | RMU Vic in a tamper-proof satchel |
|                                   | via the internal mail.            |
+-----------------------------------+-----------------------------------+
