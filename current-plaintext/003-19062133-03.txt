Customer/Applicant providing payment destination details

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligible customer/applicant       |
|                                   | contacts to provide payment       |
|                                   | destination details + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the payment destination        |
|                                   | details are in:                   |
|                                   |                                   |
|                                   | -   the AIC applicant’s           |
|                                   |     parent/guardian name(s), or   |
|                                   | -   a nominee’s name, where the   |
|                                   |     AIC applicant has a nominee   |
|                                   |     arrangement in place, or      |
|                                   | -   a parent’s name for a         |
|                                   |     dependent ABSTUDY student     |
|                                   |     under the age of 18 years, or |
|                                   | -   the student’s name, for       |
|                                   |     ABSTUDY students aged 18      |
|                                   |     years or older or who are     |
|                                   |     independent                   |
|                                   | -   Go to Step 2                  |
|                                   |                                   |
|                                   | If the payment details are in any |
|                                   | other name advise the             |
|                                   | customer/applicant Flood          |
|                                   | Education Supplement (FES) can    |
|                                   | only be paid into one of the      |
|                                   | above payment destinations.       |
|                                   | Procedure ends here.              |
|                                   |                                   |
|                                   | Note: where phone contact with    |
|                                   | the customer/applicant has been   |
|                                   | unsuccessful a letter may have    |
|                                   | been issued, and a Fast Note DOC  |
|                                   | (FES - Outbound Contact) will     |
|                                   | display on the record.            |
+-----------------------------------+-----------------------------------+
| 2                                 | To update payment destination for |
|                                   | FES + Read more ...               |
|                                   |                                   |
|                                   | -   Record payment destination    |
|                                   |     details by running the FES -  |
|                                   |     Action Required Fast Note.    |
|                                   |     Note: do not close this Fast  |
|                                   |     Note DOC upon completion      |
|                                   | -   advise the customer/applicant |
|                                   |     their payment destination     |
|                                   |     details will be provided to   |
|                                   |     the AIC Smart Centre          |
|                                   |     Processing Staff and if       |
|                                   |     eligible they will be paid    |
|                                   |     FES                           |
|                                   |                                   |
|                                   | Procedure ends here.              |
