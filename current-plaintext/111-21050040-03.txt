This page contains advice for Service Officers requiring special help
when using the Earnings and reporting workflow tool.

Once the generic help page is opened within the Earnings and reporting
Workflow by clicking on the help icon, Service Officers should click on
the 'Click Here' hyperlink to access Special Help.

Earnings and reporting workflow

Special help within Earnings and reporting workflow tool

This table outlines the special help available to Service Officers when
using the Earnings and reporting workflow tool.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Salary sacrifice + Read more ...  |
|                                   |                                   |
|                                   | Any salary sacrifice to the       |
|                                   | benefit of their partner's or a   |
|                                   | third party's superannuation,     |
|                                   | regardless of the age of the      |
|                                   | partner or third party is         |
|                                   | considered a reportable fringe    |
|                                   | benefit and is counted as income. |
|                                   |                                   |
|                                   | From 1 July 2009, any amount of   |
|                                   | salary voluntarily sacrificed     |
|                                   | into superannuation is income for |
|                                   | social security purposes for all  |
|                                   | income support payments and those |
|                                   | claiming a low income health care |
|                                   | card.                             |
|                                   |                                   |
|                                   | Prior to 1 July 2009, salary      |
|                                   | voluntarily sacrificed into       |
|                                   | superannuation:                   |
|                                   |                                   |
|                                   | -   by those under Age Pension    |
|                                   |     age was not assessable as     |
|                                   |     income, and                   |
|                                   | -   by those of Age Pension age   |
|                                   |     was assessable as income      |
|                                   |                                   |
|                                   | For all income support customers  |
|                                   | and those claiming a low income   |
|                                   | health care card:                 |
|                                   |                                   |
|                                   | Employer's contribution - when    |
|                                   | the employee is aged up to 70     |
|                                   | years:                            |
|                                   |                                   |
|                                   | -   contributions made by their   |
|                                   |     employer are not counted as   |
|                                   |     income if these contributions |
|                                   |     are required as part of their |
|                                   |     SGC obligations or part of an |
|                                   |     industry award                |
|                                   | -   employer contributions will   |
|                                   |     be counted as income if the   |
|                                   |     contribution is in excess of  |
|                                   |     the SGC or industry award.    |
|                                   |     Only the excess amount is     |
|                                   |     counted as income. The        |
|                                   |     current SGC is 9% of the      |
|                                   |     employee's salary             |
|                                   |                                   |
|                                   | When the employee is aged 70 and  |
|                                   | over, and the SGC has ceased, all |
|                                   | employer contributions to         |
|                                   | superannuation are counted as     |
|                                   | income                            |
|                                   |                                   |
|                                   | The employee's own contributions  |
|                                   | to superannuation - all voluntary |
|                                   | employee contributions will be    |
|                                   | counted as income                 |
|                                   |                                   |
|                                   | Salary sacrificing for benefits   |
|                                   | other than superannuation is      |
|                                   | regarded as fringe benefits and   |
|                                   | therefore 'valuable               |
|                                   | consideration'. They are usually  |
|                                   | included as income for all income |
|                                   | support payments (although some   |
|                                   | benefits such as free             |
|                                   | accommodation are exempt). The    |
|                                   | amount of fringe benefits taken   |
|                                   | into account in determining a     |
|                                   | customer's, or their partner's,   |
|                                   | entitlement is the non grossed up |
|                                   | value of fringe benefits.         |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Assessing salary                  |
|                                   | sacrificing/salary packaging      |
|                                   | arrangements for employment       |
|                                   | income.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Allowance income + Read more ...  |
|                                   |                                   |
|                                   | Payment of an allowance from an   |
|                                   | employer for reimbursement of     |
|                                   | work related expenses is not      |
|                                   | included in any income assessment |
|                                   | (for example, motor vehicle       |
|                                   | allowance, telephone allowance).  |
|                                   | However, if the employer pays     |
|                                   | more than the amount actually     |
|                                   | incurred by the customer, the     |
|                                   | extra is included in the income   |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | Allowances paid to a customer as  |
|                                   | part of their employment          |
|                                   | conditions are assessed as        |
|                                   | employment income. These          |
|                                   | allowances are not a              |
|                                   | reimbursement of expenses         |
|                                   | incurred by the employee.         |
|                                   | Examples may include (but are not |
|                                   | limited to) danger money, dirt    |
|                                   | money, service allowance.         |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Assessment of employment income   |
|                                   | for Centrelink payments.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Self-employed or employee? + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A self-employed person may: be a  |
|                                   | sole trader, be in a partnership, |
|                                   | be involved in a company or       |
|                                   | trust, contract their services or |
|                                   | equipment to other people, work   |
|                                   | on a commission basis. Customers  |
|                                   | who are self-employed may         |
|                                   | include: primary producers        |
|                                   | (including farmers and fishers),  |
|                                   | market gardeners, tradespeople,   |
|                                   | business owners, cottage/backyard |
|                                   | industries, seasonal workers,     |
|                                   | contractors, pieceworkers.        |
|                                   |                                   |
|                                   | Characteristics may include, but  |
|                                   | are not limited to:               |
|                                   |                                   |
|                                   | -   contract for services (to     |
|                                   |     produce a result) exists      |
|                                   | -   customer has a high level of  |
|                                   |     control and choice regarding  |
|                                   |     what work is to be done, how, |
|                                   |     when, where and what          |
|                                   |     resources are to be used      |
|                                   | -   risk of profit or loss rests  |
|                                   |     with the customer             |
|                                   | -   customer has to purchase      |
|                                   |     goods such as stationery,     |
|                                   |     sample products, and operates |
|                                   |     from their own home or        |
|                                   |     premises without              |
|                                   |     reimbursement for costs such  |
|                                   |     as electricity, postage and   |
|                                   |     transport                     |
|                                   |                                   |
|                                   | Employee:                         |
|                                   |                                   |
|                                   | An employee is someone who:       |
|                                   |                                   |
|                                   | -   is under a contract of        |
|                                   |     service (that is, contract    |
|                                   |     for labour)                   |
|                                   | -   has little control or         |
|                                   |     decision making regarding     |
|                                   |     what work is to be done, how, |
|                                   |     when, where and what          |
|                                   |     resources are to be used      |
|                                   | -   is in receipt of a payment    |
|                                   |     summary (group certificate)   |
|                                   |                                   |
|                                   | For making a determination on     |
|                                   | taxi drivers, commission earners, |
|                                   | commission only real estate sales |
|                                   | people and contractors, see       |
|                                   | Income for an independent         |
|                                   | contractor and commission income. |
