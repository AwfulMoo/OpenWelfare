
Compliance investigations and hardship provisions for job seekers affected by a natural disaster or declared emergency 001-10040020

This page contains information to assist in actioning outstanding
compliance action or applying hardship provisions for job seekers
affected by a natural disaster, declared emergency or assisting with a
recovery effort through a registered organisation. Note: arrangements
for specific events may differ.

On this page:

Considerations for managing compliance action during a natural disaster
or declared emergency

declared emergency

Job Seeker Compliance Model action during a natural disaster or declared
emergency for CDP participants

| 3                                 | Compliance investigation + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The impact of the natural         |
|                                   | disaster or declared emergency on |
|                                   | the job seeker's capacity to      |
|                                   | comply, including capacity to     |
|                                   | make prior contact, should be     |
|                                   | considered as part of a           |
|                                   | compliance investigation.         |
|                                   |                                   |
|                                   | Where the Department of           |

|                                   | have started contingency          |

|                                   |                                   |
|                                   | Where a compliance penalty        |
|                                   | investigation for a mutual        |
|                                   | obligation failure is already     |
|                                   | active, priority should be given  |
|                                   | to finalising these compliance    |
|                                   | investigations. Job seekers are   |
|                                   | unable to finalise reporting      |
|                                   | until a determination has been    |

|                                   | the financial penalty is          |

|                                   | made to reject a failure because  |
|                                   | of a disaster/emergency, the      |
|                                   | rejection code 'ERB' -Disastrous  |

|                                   | conditions - unable to comply,    |
|                                   | should be recorded.               |
|                                   |                                   |
|                                   | When an unemployment failure or   |
|                                   | work refusal failure is           |
|                                   | preventing the job seeker from    |
|                                   | reporting and they have been      |
|                                   | impacted by a natural disaster or |
|                                   | declared emergency, these         |
|                                   | failures will still require       |
|                                   | investigation. Services Australia |
|                                   | may temporarily lift the contact  |

|                                   | Exit option 'Unable to complete   |

|                                   | be completed. This save and exit  |

|                                   | the contact trigger on the day of |

|                                   | the trigger the next day.         |
|                                   |                                   |
|                                   | When the job seeker is not due to |
|                                   | report and they have contacted    |

|                                   | the contact trigger does not need |

|                                   | the job seeker to call back as    |
|                                   | soon as possible before or on     |
|                                   | their reporting day to discuss    |

Job Seeker Compliance Model action during a natural disaster or declared emergency for CDP participants

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Outstanding Non-Attendance        |
|                                   | Reports (NAR) + Read more ...     |
|                                   |                                   |
|                                   | Outstanding NARs are              |
|                                   | automatically finalised when an   |
|                                   | exemption from mutual obligation  |
|                                   | requirements is recorded and      |
|                                   | extends 3 or more working days    |
|                                   | into the future.                  |
|                                   |                                   |
|                                   | When a NAR is outstanding,        |
|                                   | Community Development Program     |
|                                   | (CDP) participants must be        |
|                                   | advised to contact their provider |
|                                   | after all other outstanding       |
|                                   | compliance action has been        |
|                                   | finalised.                        |
|                                   |                                   |
|                                   | When provider contact occurs, the |
|                                   | provider will consider if a       |
|                                   | re-engagement appointment is      |
|                                   | required: If a re-engagement      |
|                                   | appointment is:                   |
|                                   |                                   |
|                                   | -   required, payment will        |
|                                   |     automatically restore after   |
|                                   |     the CDP participant complies  |
|                                   |     with this requirement         |
|                                   | -   not required, payment will    |
|                                   |     automatically restore once    |
|                                   |     the provider finalises the    |
|                                   |     NAR                           |
|                                   |                                   |
|                                   | Once payment is restored, the CDP |
|                                   | participant will be able to       |
|                                   | finalise any outstanding          |
|                                   | reporting requirements.           |
|                                   |                                   |
|                                   | If the provider is unavailable    |
|                                   | because of the disaster or        |
|                                   | emergency, the CDP participant    |
|                                   | should be referred to the         |
|                                   | Department of Education, Skills   |
|                                   | and Employment for assistance.    |
|                                   |                                   |
|                                   | When the Department of Education, |
|                                   | Skills and Employment has made a  |
|                                   | decision that all outstanding     |
|                                   | NARs in an affected area will be  |
|                                   | finalised, a bulk update will be  |
|                                   | arranged by that department.      |
|                                   | Services Australia Service        |
|                                   | Officers should not individually  |
|                                   | finalise NARs for affected CDP    |
|                                   | participants.                     |

|                                   | non-attendance, reconnection and  |
|                                   | No Show, No Pay failures + Read   |

|                                   |                                   |

|                                   | for CDP participants affected by  |
|                                   | a natural disaster or declared    |
|                                   | emergency once contact is         |
|                                   | established and they agree to     |
|                                   | re-engage. When the PST-skilled   |
|                                   | Service Officer determines        |
|                                   | re-engagement is not possible or  |
|                                   | appropriate, payment may also be  |
|                                   | restored.                         |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Finalising Community Development  |
|                                   | Program (CDP) re-engagement       |
|                                   | requirements.                     |
|                                   |                                   |
|                                   | Failure investigations            |
|                                   |                                   |
|                                   | If a CDP participant was unable   |
|                                   | to comply with a requirement      |
|                                   | because of a natural disaster or  |
|                                   | declared emergency, their verbal  |
|                                   | statement provides the required   |
|                                   | level of evidence to make a       |
|                                   | determination of reasonable       |
|                                   | excuse. The impact of the         |

|                                   | discretion may be used to reject  |
|                                   | the failure.                      |
|                                   |                                   |
|                                   | Rejecting a failure as a result   |
|                                   | of a disaster/emergency           |
|                                   |                                   |
|                                   | When a decision is made to reject |
|                                   | a failure because of a            |
|                                   | disaster/emergency, the rejection |
|                                   | code 'ERB' - Disastrous event or  |

|                                   | unable to comply, should be       |
|                                   | recorded.                         |
|                                   |                                   |
|                                   | When a decision is made to reject |

| 3                                 | Re-engaging CDP participants      |
|                                   | during a disaster or emergency +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Sensitivity should be shown when  |
|                                   | a CDP participant's capacity to   |
|                                   | re-engage is affected by the      |
|                                   | disaster or emergency, including  |
|                                   | when a provider's premises is     |
|                                   | affected.                         |
|                                   |                                   |
|                                   | CDP participants who are unable   |
|                                   | to return to an activity because  |
|                                   | of a disaster/emergency should be |
|                                   | advised to speak with their       |
|                                   | provider as soon as possible.     |
|                                   |                                   |
|                                   | When a compliance investigation,  |
|                                   | including a Comprehensive         |
|                                   | Compliance Assessment (CCA),      |
|                                   | identifies a CDP participant's    |
|                                   | capacity to comply with mutual    |
|                                   | obligation requirements has been  |
|                                   | affected by a disaster/emergency, |
|                                   | an exemption from their mutual    |
|                                   | obligation requirements should be |
|                                   | considered.                       |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Exempting a job seeker from   |
|                                   |     mutual obligation             |
|                                   |     requirements due to special   |
|                                   |     circumstances, or             |
|                                   | -   Mutual obligation             |
|                                   |     requirements exemptions       |
|                                   |     during disasters              |
|                                   |                                   |
|                                   | If appropriate, an exemption from |
|                                   | mutual obligation requirements    |
|                                   | should be considered. If an       |
|                                   | exemption has been granted, a no  |
|                                   | re-engagement code of 'RNH' - No  |
|                                   | re-engagement, current exemption, |
|                                   | should be recorded if required.   |
|                                   |                                   |
|                                   | If the exemption from mutual      |
|                                   | obligation requirements will end: |
|                                   |                                   |
|                                   | -   within the next 14 days,      |
|                                   |     re-engagement is required     |
|                                   | -   more than 14 days in the      |
|                                   |     future, re-engagement is not  |
|                                   |     required                      |
|                                   |                                   |
|                                   | If the provider's premises are    |
|                                   | not impacted by the disaster or   |
|                                   | emergency and the CDP participant |
|                                   | has not been granted an exemption |
|                                   | from mutual obligation            |
|                                   | requirements, the impact of the   |
|                                   | disaster/emergency should still   |
|                                   | be considered. This will          |
|                                   | determine if it is appropriate to |
|                                   | re-engage the CDP participant     |
|                                   | back to their provider during the |
|                                   | disaster or emergency period.     |
|                                   |                                   |
|                                   | When a CDP participant would      |
|                                   | normally be issued with a         |
|                                   | reconnection requirement but it   |
|                                   | is clear the disaster/emergency   |
|                                   | has had a significant impact on   |
|                                   | the CDP participant, a            |
|                                   | no-re-engagement code of 'RNF' -  |
|                                   | No formal re-engagement, should   |
|                                   | be used. Advise the CDP           |
|                                   | participant to restart their      |
|                                   | activity/program as soon as       |
|                                   | possible or following contact by  |
|                                   | their provider and clearly        |
|                                   | document why no re-engagement has |
|                                   | occurred in the Participation     |
|                                   | Compliance workflow.              |

| 6                                 | Serious failures for refusing,    |
|                                   | failing to accept or commence     |
|                                   | suitable work + Read more ...     |
|                                   |                                   |
|                                   | If a CDP participant refuses,     |
|                                   | fails to accept or commence a     |
|                                   | suitable job offer, reasonable    |
|                                   | excuse will apply when this was   |
|                                   | due to a disaster/emergency. In   |
|                                   | these cases the CDP participant's |
|                                   | statement about how the           |
|                                   | disaster/emergency affected their |
|                                   | ability to accept or commence the |
|                                   | job should provide enough         |
|                                   | information to make a             |
|                                   | determination unless there is     |
|                                   | reason to doubt the CDP           |
|                                   | participant's statement. When     |
|                                   | reasonable excuse is established  |
|                                   | the serious failure should be     |
|                                   | rejected.                         |
|                                   |                                   |
|                                   | When the serious failure event    |
|                                   | occurred prior to the             |
|                                   | disaster/emergency, a serious     |
|                                   | failure will need to be           |
|                                   | investigated. However, when the   |
|                                   | CDP participant has been          |
|                                   | personally affected by the        |
|                                   | disaster/emergency, consideration |
|                                   | should be given to the impacts of |
|                                   | applying the failure given the    |
|                                   | CDP participant's current         |
|                                   | circumstances. In these cases,    |
|                                   | discretion may be used to reject  |
|                                   | the failure using the rejection   |
|                                   | reason code 'NDE' - Naturally     |
|                                   | declared disaster.                |
|                                   |                                   |
|                                   | When a decision is made to apply  |
|                                   | the failure and the CDP           |
|                                   | participant does not have         |
|                                   | capacity to participate in a      |
|                                   | Compliance Activity due to the    |
|                                   | disaster/emergency, (for example, |
|                                   | provider closed), consideration   |
|                                   | should be given to booking a      |
|                                   | Compliance Activity appointment   |
|                                   | at the first suitable time. If a  |
|                                   | suitable appointment is not       |
|                                   | available consideration should be |
|                                   | given to waiving the penalty. See |
|                                   | Waiving a serious failure period. |
|                                   |                                   |
|                                   | CDP participants currently        |
|                                   | serving a serious failure period  |

|                                   | eligible for payment under        |
|                                   | serious failure hardship          |

|                                   |     access an appropriate         |

|                                   | Any outstanding Unemployment Non  |
|                                   | Payment Period (UNPP)             |
|                                   | investigation that occurred       |
|                                   | before or during an emergency     |
|                                   | must be investigated.             |
|                                   |                                   |
|                                   | Voluntary unemployment has not    |
|                                   | occurred if a CDP participant     |
|                                   | left work because they are no     |
|                                   | longer being paid by their        |
|                                   | employer because of the           |
|                                   | disaster/emergency. For more      |
|                                   | information, see Unemployed       |
|                                   | eligibility for JobSeeker Payment |
|                                   | (JSP) and Youth Allowance (job    |
|                                   | seeker).                          |
|                                   |                                   |
|                                   | If an employer cannot be          |
|                                   | contacted because of the          |
|                                   | disaster/emergency, evidence      |
|                                   | provided by the CDP participant   |
|                                   | along with the Employment         |
|                                   | Separation Certificate (SU1) (if  |
|                                   | possible) needs to be carefully   |
|                                   | considered. When information from |
|                                   | the CDP participant conflicts     |
|                                   | with the information recorded on  |
|                                   | the SU1, the CDP participant's    |
|                                   | statement should provide enough   |
|                                   | information to make a             |
|                                   | determination on the UNPP.        |
|                                   |                                   |
|                                   | When a decision is made to apply  |
|                                   | a UNPP, the CDP participant's     |
|                                   | eligibility to have the UNPP      |
|                                   | ended under the hardship          |
|                                   | provisions should be assessed.    |
|                                   |                                   |
|                                   | CDP participants affected by a    |
|                                   | disaster/emergency who contact    |
|                                   | the agency and are currently      |
|                                   | serving a UNPP should be referred |
|                                   | to the Participation Solutions    |
|                                   | Team (PST) so that eligibility    |
|                                   | for payment under hardship        |
|                                   | provisions can be assessed.       |

| 8                                 | Restoring payment under           |
|                                   | Unemployment Non-Payment Period   |
|                                   | (UNPP) hardship provisions + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | An appropriate level of           |
|                                   | discretion should be applied when |
|                                   | considering the CDP participant's |
|                                   | financial circumstances. For      |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   When a CDP participant is     |
|                                   |     unable to provide evidence or |
|                                   |     advises that they are         |
|                                   |     homeless because of the       |
|                                   |     disaster/emergency, their     |
|                                   |     statement should be           |
|                                   |     considered appropriate        |
|                                   |     evidence, unless there is     |
|                                   |     reason to doubt the statement |
|                                   | -   CDP participants may be       |
|                                   |     unable to provide current     |
|                                   |     financial information. In     |
|                                   |     these cases, the Services     |
|                                   |     Australia record should be    |
|                                   |     used unless disputed by the   |
|                                   |     CDP participant               |
|                                   | -   When a CDP participant        |
|                                   |     advises that, because of the  |
|                                   |     disaster or emergency, they   |
|                                   |     have used a significant       |
|                                   |     amount of their savings and   |
|                                   |     investments, this expenditure |
|                                   |     should be considered to have  |
|                                   |     been unavoidable or           |
|                                   |     reasonable                    |
|                                   |                                   |
|                                   | Unless otherwise advised for a    |
|                                   | specific natural disaster,        |
|                                   | payment is reinstated under       |
|                                   | hardship provisions for CDP       |
|                                   | participants personally affected  |
|                                   | by a disaster/emergency from the  |
|                                   | date all the following are        |
|                                   | satisfied:                        |
|                                   |                                   |
|                                   | -   The CDP participant asks to   |
|                                   |     have the UNPP ended under     |
|                                   |     hardship provisions           |
|                                   | -   The CDP participant is a      |
|                                   |     vulnerable CDP participant    |
|                                   |     who falls within a class of   |
|                                   |     person specified by the       |
|                                   |     Secretary of the Department   |
|                                   |     of Education, Skills and      |
|                                   |     Employment, and               |
|                                   | -   The CDP participant would be  |
|                                   |     in severe financial hardship  |
|                                   |     as a result of the UNPP       |
|                                   |                                   |
|                                   | It is not appropriate to back     |
|                                   | date hardship provisions prior to |
|                                   | the CDP participant meeting all   |
|                                   | the above criteria.               |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Applying an Unemployment          |
|                                   | Non-Payment Period (UNPP) and     |
|                                   | determining hardship provisions.  |
