Service Officers may choose to cancel a Started ('STA') Service Update
by entering No Action Possible ('NAP') in the Status field of the
mandatory Service Action. Code a cancellation reason on the
Deferment/Cancellation Details (PQCD) screen, choosing an appropriate
code from the list below or via screen help ('?')

Note: Not Started Service Components cannot be cancelled.

Codes and descriptions for Service Component status

Service Component status codes and descriptions

Table 3: This table describes Service Component status codes and
descriptions.

+-----------------------+-----------------------+-----------------------+
| Code                  | Description           | Comments/Use          |
+-----------------------+-----------------------+-----------------------+
| NST                   | Not Started           | All Service Actions   |
|                       |                       | have a status of      |
|                       |                       | 'NST', indicating     |
|                       |                       | that the Service      |
|                       |                       | Component has not     |
|                       |                       | been actioned in any  |
|                       |                       | way.                  |
+-----------------------+-----------------------+-----------------------+
| INF                   | Information Only      | Display Only Service  |
|                       |                       | Components and        |
|                       |                       | 'Reporting' Service   |
|                       |                       | Components are        |
|                       |                       | created with a status |
|                       |                       | of 'INF' rather than  |
|                       |                       | 'NST'. All Service    |
|                       |                       | Actions and Reporting |
|                       |                       | Actions will have a   |
|                       |                       | status of 'INF' also. |
+-----------------------+-----------------------+-----------------------+
| STA                   | Started               | Indicates that the    |
|                       |                       | Service Component is  |
|                       |                       | currently in          |
|                       |                       | progress.             |
+-----------------------+-----------------------+-----------------------+
| DEF                   | Deferred              | One or more Service   |
|                       |                       | Actions have a status |
|                       |                       | of 'DEF' and no       |
|                       |                       | further action is     |
|                       |                       | required against the  |
|                       |                       | Service Component     |
|                       |                       | until the status is   |
|                       |                       | changed.              |
+-----------------------+-----------------------+-----------------------+
| WRP                   | Within Restoration    | The Service Component |
|                       | Period                | was in progress and   |
|                       |                       | the Profile Service   |
|                       |                       | Reason has been       |
|                       |                       | cancelled or          |
|                       |                       | suspended. It will be |
|                       |                       | finalised:            |
|                       |                       |                       |
|                       |                       | -   Automatically by  |
|                       |                       |     the system at the |
|                       |                       |     expiration of a   |
|                       |                       |     specified period, |
|                       |                       | -   Manually by the   |
|                       |                       |     user at any time  |
+-----------------------+-----------------------+-----------------------+
| CAN                   | Cancelled             | The Service Component |
|                       |                       | was in progress and   |
|                       |                       | at least one          |
|                       |                       | Mandatory Service     |
|                       |                       | Action has been       |
|                       |                       | updated to 'NAP'      |
|                       |                       | indicating that the   |
|                       |                       | core business         |
|                       |                       | objectives of the     |
|                       |                       | Service Component     |
|                       |                       | cannot be delivered.  |
+-----------------------+-----------------------+-----------------------+
| COM                   | Completed             | All Service Actions   |
|                       |                       | have a status of      |
|                       |                       | 'COM', 'NAP' or 'NAT' |
|                       |                       | but the Profile       |
|                       |                       | Bracket(s) that       |
|                       |                       | caused the Service    |
|                       |                       | Component to be       |
|                       |                       | created are still in  |
|                       |                       | effect.               |
+-----------------------+-----------------------+-----------------------+
| HST                   | Historical            | The Service Component |
|                       |                       | has been completed    |
|                       |                       | and the Profile       |
|                       |                       | Bracket(s) that       |
|                       |                       | caused the Service    |
|                       |                       | Component to be       |
|                       |                       | created have been     |
|                       |                       | ended.                |
+-----------------------+-----------------------+-----------------------+
| EXC                   | Excluded              | The Service Component |
|                       |                       | has been excluded by  |
|                       |                       | another Service       |
|                       |                       | Component.            |
+-----------------------+-----------------------+-----------------------+
| EXM                   | Exempt                | The Profile to which  |
|                       |                       | this Service          |
|                       |                       | Component is mapped   |
|                       |                       | is exempt for a       |
|                       |                       | specified period.     |
+-----------------------+-----------------------+-----------------------+

Codes and descriptions for Service Action Status

Service Action status codes and descriptions

Table 4: This table describes Service Action status codes and
descriptions.

+-----------------------+-----------------------+-----------------------+
| Code                  | Description           | Comments/Use          |
+-----------------------+-----------------------+-----------------------+
| NST                   | Not Started           | The Service Action    |
|                       |                       | has not been          |
|                       |                       | 'STA'rted.            |
|                       |                       |                       |
|                       |                       | -   Auto - set by     |
|                       |                       |     Service Manager   |
|                       |                       |     when Service      |
|                       |                       |     Component and     |
|                       |                       |     Actions are       |
|                       |                       |     created. May be   |
|                       |                       |     reset to 'NST' by |
|                       |                       |     another           |
|                       |                       |     application (e.g. |
|                       |                       |     Participation     |
|                       |                       |     when Deferment    |
|                       |                       |     ends)             |
+-----------------------+-----------------------+-----------------------+
| INF                   | Information Only      | The Service Action is |
|                       |                       | for information       |
|                       |                       | purposes only.        |
|                       |                       |                       |
|                       |                       | -   Auto - Set by the |
|                       |                       |     Service Manager   |
|                       |                       |     when the Service  |
|                       |                       |     Component and     |
|                       |                       |     Service Actions   |
|                       |                       |     are created, and  |
|                       |                       |     the Service       |
|                       |                       |     Component is for  |
|                       |                       |     Display Only      |
|                       |                       | -   Auto - Set by the |
|                       |                       |     Service Manager   |
|                       |                       |     for all 'on'      |
|                       |                       |     Reporting Actions |
+-----------------------+-----------------------+-----------------------+
| STA                   | Started               | The Service Action    |
|                       |                       | has been commenced.   |
|                       |                       |                       |
|                       |                       | -   Auto - set by     |
|                       |                       |     another           |
|                       |                       |     application when  |
|                       |                       |     the Service       |
|                       |                       |     Action is         |
|                       |                       |     commenced.        |
|                       |                       | -   Manual - set by   |
|                       |                       |     Service Officer   |
|                       |                       |     when manual       |
|                       |                       |     Service Action is |
|                       |                       |     commenced.        |
|                       |                       |                       |
|                       |                       | Note: not all Service |
|                       |                       | Actions will progress |
|                       |                       | from 'NST' to 'STA'   |
|                       |                       | to 'COM' - status may |
|                       |                       | be immediately set to |
|                       |                       | 'COM' where the       |
|                       |                       | action is completed   |
|                       |                       | in one step e.g.      |
|                       |                       | Issue Form.           |
+-----------------------+-----------------------+-----------------------+
| DEF                   | Deferred              | The Service Action    |
|                       |                       | has been Deferred and |
|                       |                       | no further action is  |
|                       |                       | required against the  |
|                       |                       | Service Action until  |
|                       |                       | the status is         |
|                       |                       | changed.              |
|                       |                       |                       |
|                       |                       | -   Auto - set by     |
|                       |                       |     Participation     |
|                       |                       |     Cluster when the  |
|                       |                       |     customer meets a  |
|                       |                       |     participation     |
|                       |                       |     Deferment         |
|                       |                       |     criteria          |
|                       |                       | -   Manual - set by   |
|                       |                       |     Service Officer   |
|                       |                       |     when deferring a  |
|                       |                       |     Service Component |
|                       |                       |                       |
|                       |                       | Note: manual entry of |
|                       |                       | this status will be   |
|                       |                       | controlled by screen  |
|                       |                       | edits.                |
+-----------------------+-----------------------+-----------------------+
| NAT                   | No Action Taken       | Can be keyed against  |
|                       |                       | an optional Service   |
|                       |                       | Action where the      |
|                       |                       | Service Officer       |
|                       |                       | determines that there |
|                       |                       | is no value in        |
|                       |                       | delivering the        |
|                       |                       | Service Action.       |
|                       |                       |                       |
|                       |                       | -   Manual (a) - set  |
|                       |                       |     by Service        |
|                       |                       |     Officer when      |
|                       |                       |     Service Action is |
|                       |                       |     not delivered     |
|                       |                       | -   Manual (b) - set  |
|                       |                       |     by user with      |
|                       |                       |     Profiling         |
|                       |                       |     National          |
|                       |                       |     Administrator     |
|                       |                       |     access where      |
|                       |                       |     Service Action    |
|                       |                       |     hasn't been       |
|                       |                       |     completed as a    |
|                       |                       |     result of error   |
|                       |                       |     in host area      |
|                       |                       |     processing or     |
|                       |                       |     Service Manager   |
|                       |                       |     processing        |
+-----------------------+-----------------------+-----------------------+
| NAP                   | No Action Possible    | The Service Action    |
|                       |                       | cannot be delivered   |
|                       |                       | e.g. the resources    |
|                       |                       | required to deliver   |
|                       |                       | the Service Action    |
|                       |                       | are not available to  |
|                       |                       | the site.             |
|                       |                       |                       |
|                       |                       | -   Manual (a) - set  |
|                       |                       |     by Service        |
|                       |                       |     Officer when      |
|                       |                       |     Service Action    |
|                       |                       |     cannot be         |
|                       |                       |     delivered         |
|                       |                       | -   Manual (b) - set  |
|                       |                       |     by user with      |
|                       |                       |     Profiling         |
|                       |                       |     National          |
|                       |                       |     Administrator     |
|                       |                       |     access where      |
|                       |                       |     Service Action    |
|                       |                       |     hasn't been       |
|                       |                       |     completed as a    |

|                       |                       |     processing or     |
|                       |                       |     Service Manager   |
|                       |                       |     processing        |
