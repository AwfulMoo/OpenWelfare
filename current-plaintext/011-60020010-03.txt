Assessing Power of Attorney (POA) details via CDMS

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Open CDMS                         |
|                                   |                                   |
|                                   | Open CDMS.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Select Inquire on Consumer        |
|                                   | Details                           |
|                                   |                                   |
|                                   | Enter Medicare number as the ID   |
|                                   | number                            |
|                                   |                                   |
|                                   | Program should be defaulted to    |
|                                   | Medicare and the ID Number Type   |
|                                   | should be Medicare Card Number    |
|                                   |                                   |
|                                   | Select the Entitlement button     |
|                                   |                                   |
|                                   | If a POA is recorded by Medicare  |
|                                   | then a POA tab will appear        |
|                                   |                                   |
|                                   | -   Select the tab to view POA    |
|                                   |     details                       |
|                                   | -   If no POA details are         |
|                                   |     recorded, then there is no    |
|                                   |     POA tab                       |
+-----------------------------------+-----------------------------------+
