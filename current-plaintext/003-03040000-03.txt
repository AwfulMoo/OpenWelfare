This page has the steps to decide a customer's rate of payment for
Disaster Recovery Allowance (DRA)/NZ DRA. These steps can be taken once
all other eligibility criteria have been assessed.

Determining a customer's rate of payment for DRA/NZ DRA

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Eligibility for DRA/NZ DRA + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the customer eligible for DRA  |
|                                   | and NZ DRA ?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 2                                 | Annualised disaster affected      |
|                                   | income + Read more ...            |
|                                   |                                   |
|                                   | Calculate the customer's          |
|                                   | annualised disaster affected      |
|                                   | income.                           |
|                                   |                                   |
|                                   | See Eligibility for DRA and NZ    |
|                                   | DRA.                              |
+-----------------------------------+-----------------------------------+
| 3                                 | Annualised average weekly         |
|                                   | income + Read more ...            |
|                                   |                                   |
|                                   | Using the Average Weekly Ordinary |
|                                   | Time Earnings (AWOTE) for a       |
|                                   | full-time adult, calculate the    |
|                                   | annualised average weekly income. |
|                                   |                                   |
|                                   | See Eligibility for DRA and NZ    |
|                                   | DRA.                              |
+-----------------------------------+-----------------------------------+
| 4                                 | Determine rate of payment + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the customer's annualised      |
|                                   | disaster affected income less     |
|                                   | than the annualised average       |
|                                   | weekly income?                    |
|                                   |                                   |
|                                   | Yes. the rate of payment depends  |
|                                   | on the customer's age and family  |
|                                   | situation. It is the relevant     |
|                                   | maximum basic rate of:            |
|                                   |                                   |
|                                   | -   JobSeeker Payment (JSP)       |
|                                   | -   Youth Allowance (YA)          |
|                                   |                                   |
|                                   | No. the rate of payment is nil    |
|                                   | and the customer is not eligible  |
|                                   | for DRA/NZ DRA                    |
+-----------------------------------+-----------------------------------+
