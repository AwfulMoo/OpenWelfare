This page contains the process Service Officers follow when a pharmacist
enquires about an authority on the Pharmaceutical Benefits Scheme (PBS)
Authorities line.

Pharmacist enquiries about an authority

Respond to pharmacist enquiry

This table describes how to respond to a call from a pharmacist on the
PBS Authorities Line.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record + Read more ...            |
|                                   |                                   |
|                                   | Record the approval number of the |
|                                   | pharmacist calling.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Notify + Read more ...            |
|                                   |                                   |
|                                   | Notify the pharmacist of the      |
|                                   | correct number to call. For       |
|                                   | contact details, see the PBS      |
|                                   | General Enquires line, or PBS     |
|                                   | Authorities line.                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Transfer + Read more ...          |
|                                   |                                   |
|                                   | Transfer the call to the correct  |
|                                   | number. For contact details, see  |
|                                   | the PBS General Enquires line, or |
|                                   | PBS Authorities line.             |
+-----------------------------------+-----------------------------------+
