This page contains the process required to terminate an existing case in
Compensation Case Management System (CCMS).

Terminate existing case in CCMS

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Answer questions in sequence in   |
|                                   | CCMS + Read more ...              |
|                                   |                                   |
|                                   | In the Case Termination screen in |
|                                   | CCMS, scroll to the bottom to     |
|                                   | finalise actions and answer the   |
|                                   | questions in sequence.            |
|                                   |                                   |
|                                   | Note: do not select or tab        |
|                                   | through the fields - Service      |
|                                   | Officers must double select a     |
|                                   | choice or Press [Enter] to answer |
|                                   | questions.                        |
|                                   |                                   |
|                                   | Refer case to Supervisor? -       |
|                                   | Select No and Press [Enter]       |
|                                   |                                   |
|                                   | Is this case a class action       |
|                                   | suit? - Select No and Press       |
|                                   | [Enter]                           |
|                                   |                                   |
|                                   | Verify claimant's address? -      |
|                                   | Select No and Press [Enter]       |
|                                   |                                   |
|                                   | Do you want to close this case? - |
|                                   | Select Yes and Press [Enter]      |
|                                   |                                   |
|                                   | Select finalisation type? From    |
|                                   | the drop down list, select one of |
|                                   | the following options and Press   |
|                                   | [Enter]:                          |
|                                   |                                   |
|                                   | -   incorrect registration (where |
|                                   |     wrong person registered)      |
|                                   | -   claim termination (where the  |
|                                   |     legal process was             |
|                                   |     discontinued, withdrawn,      |
|                                   |     dismissed or settled under    |
|                                   |     $5,000 where no notice of     |
|                                   |     settlement is supplied and    |
|                                   |     the information is received   |
|                                   |     in a letter or email,         |
|                                   |     claimant has advised they are |
|                                   |     not making a claim)           |
|                                   |                                   |
|                                   | Date the notice received? Enter   |
|                                   | the date stamp on the document    |
|                                   | received or the date the case was |
|                                   | incorrectly registered and Press  |
|                                   | [Enter]                           |
|                                   |                                   |
|                                   | Update CICS registration? -       |
|                                   | select the black arrow and Press  |
|                                   | [Enter]. Do not release the work  |
|                                   | form yet, go to Step 2            |
|                                   |                                   |
|                                   | A system message is returned to   |
|                                   | confirm 'completed'.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Re-register case on CCMS + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Does the case need to be          |
|                                   | re-registered on CCMS?            |
|                                   |                                   |
|                                   | Yes, use the notification         |
|                                   | document and/or the claim details |
|                                   | on the terminated case to         |
|                                   | register the case correctly:      |
|                                   |                                   |
|                                   | -   print a copy of all documents |
|                                   |     on the terminated case        |
|                                   | -   register the case and obtain  |
|                                   |     new Work Identification       |
|                                   |     Number (WIN)                  |
|                                   | -   scan all documents to the new |
|                                   |     WIN, go to Step 3             |
|                                   |                                   |
|                                   | See Register Medicare             |
|                                   | Compensation Recovery claims for  |
|                                   | assistance.                       |
|                                   |                                   |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Add a case note + Read more ...   |
|                                   |                                   |
|                                   | If the Case Notes screen has not  |
|                                   | opened automatically, select the  |
|                                   | Add notes button from the list on |
|                                   | the right hand side of the main   |
|                                   | CCMS screen.                      |
|                                   |                                   |
|                                   | -   The top box shows all notes   |
|                                   |     previously recorded on this   |
|                                   |     case                          |
|                                   | -   In the blank bottom box,      |
|                                   |     record the actions taken,     |
|                                   |     information requested and     |
|                                   |     from whom and reasons for a   |
|                                   |     decision. Use standard case   |
|                                   |     notes where possible          |
|                                   | -   Select Add to save notes. A   |
|                                   |     confirmation box will appear, |
|                                   |     select 'Yes' to add case note |
|                                   | -   Press [F4] to permanently     |
|                                   |     terminate the case            |
