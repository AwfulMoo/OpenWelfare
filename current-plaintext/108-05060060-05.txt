Event Date examples for current customers

+-----------------------------------+-----------------------------------+
| Type of pension                   | Description                       |
+-----------------------------------+-----------------------------------+
| Invalidity or TPS pension         | Rohan retired from work because   |
|                                   | of invalidity on 14 April 2005    |
|                                   | and applied for an Invalidity     |
|                                   | pension from their superannuation |
|                                   | fund on the same day.             |
|                                   |                                   |
|                                   | Rohan’s application for an        |
|                                   | invalidity pension was approved   |
|                                   | on 15 June 2019. Payments were    |
|                                   | backdated to 15 April 2005.       |
|                                   |                                   |
|                                   | Rohan's ongoing income stream     |
|                                   | payment of $2,000 per fortnight   |
|                                   | started on 15 June 2019. Rohan    |
|                                   | received a lump sum arrears       |
|                                   | payment of $500,000 on 18 June    |
|                                   | 2019, for the period from 15      |
|                                   | April 2005 to 14 June 2019.       |
|                                   |                                   |
|                                   | For social security income test   |
|                                   | purposes, the $500,000 paid for   |
|                                   | the period before 15 June 2019 is |
|                                   | apportioned as income for the     |
|                                   | following 12 months, beginning on |
|                                   | 18 June 2019. The ongoing income  |
|                                   | stream payment is assessed from   |
|                                   | 15 June 2019.                     |
|                                   |                                   |
|                                   | The Event Date for the income     |
|                                   | stream is 15 June 2019.           |
|                                   |                                   |
|                                   | Rohan will have to advise how he  |
|                                   | has used the funds to determine   |
|                                   | any further impact under the      |
|                                   | income and assets tests.          |
+-----------------------------------+-----------------------------------+
| Non-invalidity or TPD pension     | Larissa applies to their          |
|                                   | superannuation fund on 1 March    |
|                                   | 2019 to start the defined benefit |
|                                   | income stream. Larissa is         |
|                                   | eligible to receive an income     |
|                                   | stream from 1 March 2019.         |
|                                   |                                   |
|                                   | The superannuation fund starts    |
|                                   | Larissa’s ongoing income stream   |
|                                   | payments on 1 June 2019 and pays  |
|                                   | a lump sum arrears payment of     |
|                                   | $5,000 for the period 1 March     |
|                                   | 2019 to 31 May 2019.              |
|                                   |                                   |
|                                   | The lump sum arrears of $5,000 is |
|                                   | assessed as income from the       |
|                                   | income stream over the period 1   |
|                                   | March 2019 to 31 May 2019. The    |
|                                   | ongoing payments are assessed     |
|                                   | from 1 June 2019.                 |
|                                   |                                   |
|                                   | The Event Date for the income     |
|                                   | stream is 1 March 2019.           |
+-----------------------------------+-----------------------------------+
