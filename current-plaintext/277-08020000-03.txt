This page provides the information to correctly action applications to
register overseas maintenance liabilities where the application is
received from a payee, payer or an overseas authority.

| 4                                 | Scrutiny + Read more ...          |
|                                   |                                   |
|                                   | If the application is already     |
|                                   | recorded on Cuba, issue a unique  |
|                                   | letter to the customer advising a |
|                                   | previous application has been     |
|                                   | registered.                       |
|                                   |                                   |
|                                   | Review the application to         |
|                                   | determine if the child or spouse  |
|                                   | maintenance liability is a        |
|                                   | registrable overseas maintenance  |
|                                   | liability. To be registrable, the |
|                                   | liability must be:                |
|                                   |                                   |
|                                   | -   a maintenance order for a     |
|                                   |     periodic amount made by a     |
|                                   |     judicial authority of a       |
|                                   |     reciprocating jurisdiction    |
|                                   | -   a maintenance agreement for a |
|                                   |     periodic amount registered by |
|                                   |     a judicial or administrative  |
|                                   |     authority of a reciprocating  |
|                                   |     jurisdiction                  |
|                                   | -   a maintenance assessment for  |
|                                   |     a periodic amount issued by   |
|                                   |     an administrative authority   |
|                                   |     of a reciprocating            |
|                                   |     jurisdiction                  |
|                                   | -   arrears of periodic amounts   |
|                                   |     in relation to any of the     |
|                                   |     above liabilities             |
|                                   | -   an Agency Reimbursement       |
|                                   |     Liability (an amount that has |
|                                   |     been paid to a payee of a     |
|                                   |     child maintenance liability   |
|                                   |     by an overseas authority for  |
|                                   |     which the overseas authority  |
|                                   |     seeks reimbursement from the  |
|                                   |     payer)                        |
|                                   |                                   |
|                                   | For more information on the types |
|                                   | of liabilities that can be        |
|                                   | registered with Child Support,    |
|                                   | References has a link to the      |
|                                   | Child Support Guide 3.6.1:        |
|                                   | Registrable overseas maintenance  |
|                                   | liabilities.                      |
|                                   |                                   |
|                                   | Enforcement orders are not        |
|                                   | registrable                       |
|                                   |                                   |
|                                   | Some jurisdictions, particularly  |
|                                   | the U.K., make enforcement orders |
|                                   | after they have registered an     |
|                                   | Australian assessment.            |
|                                   | Enforcement orders enable the     |
|                                   | court to take action to collect   |
|                                   | the registered Australian         |
|                                   | liability if the paying parent    |
|                                   | refuses to pay. Enforcement       |
|                                   | orders are not maintenance orders |
|                                   | and are therefore not             |
|                                   | registrable.                      |
|                                   |                                   |
|                                   | An enforcement order will usually |
|                                   | clearly state that it is an order |
|                                   | for enforcement (rather than a    |
|                                   | maintenance order). The           |
|                                   | enforcement order may be for a    |
|                                   | different amount than the         |
|                                   | Australian assessment as the      |
|                                   | overseas court may make its own   |
|                                   | assessment of the paying parent’s |
|                                   | capacity to pay the registered    |
|                                   | Australian assessment.            |
|                                   |                                   |
|                                   | If it is not clear whether the    |
|                                   | order is a maintenance order or   |
|                                   | an enforcement order, seek help   |
|                                   | from an SSO, see Technical        |
|                                   | support Cuba Process Help.        |
|                                   |                                   |
|                                   | Overseas court order made in      |
|                                   | response to a transmitted         |
|                                   | Australian assessment             |
|                                   |                                   |
|                                   | The order may have been made in   |
|                                   | response to a transmitted         |
|                                   | Australian assessment. The        |
|                                   | domestic laws of some             |
|                                   | reciprocating jurisdictions       |
|                                   | require the transformation of     |
|                                   | Australian assessments into       |
|                                   | judicial maintenance liabilities  |
|                                   | (court orders) to enable that     |
|                                   | jurisdiction to register and      |
|                                   | enforce it. These orders are      |
|                                   | different to overseas enforcement |
|                                   | orders and can be registered if   |
|                                   | they are registrable overseas     |
|                                   | maintenance liabilities.          |
|                                   |                                   |
|                                   | The court in the reciprocating    |
|                                   | jurisdiction may set a level of   |
|                                   | maintenance that differs from the |
|                                   | Australian assessment.            |
|                                   |                                   |
|                                   | If the overseas order is          |
|                                   | registered, the Australian        |
|                                   | assessment will end from the day  |
|                                   | before the court order is         |
|                                   | registered. Any arrears that have |
|                                   | accrued under the assessment up   |
|                                   | to the date that the assessment   |
|                                   | ends, will remain payable.        |
|                                   |                                   |
|                                   | Liability is registrable          |
|                                   |                                   |
|                                   | If the order, agreement or        |
|                                   | assessment is a registrable       |
|                                   | overseas maintenance liability,   |
|                                   | check the case to determine if    |
|                                   | there is a registered maintenance |
|                                   | liability in place.               |
|                                   |                                   |
|                                   | The maintenance liability in the  |
|                                   | new application cannot be         |
|                                   | registered if there is a current  |
|                                   | liability registered:             |
|                                   |                                   |
|                                   | -   that relates to the same      |
|                                   |     payer, payee and child/ren,   |
|                                   |     and                           |
|                                   | -   the maintenance liability in  |
|                                   |     the new application was made  |
|                                   |     prior to that Child Support   |
|                                   |     has registered                |
|                                   |                                   |
|                                   | This is because Child Support     |
|                                   | only became aware of it after the |
|                                   | current liability was registered. |
|                                   | However, the arrears that arose   |
|                                   | under the earlier liability can   |
|                                   | be registered.                    |
|                                   |                                   |
|                                   | Liability is not registrable      |
|                                   |                                   |
|                                   | It the order, agreement or        |
|                                   | assessment is not a registrable   |
|                                   | overseas maintenance liability,   |
|                                   | go to Step 8.                     |
