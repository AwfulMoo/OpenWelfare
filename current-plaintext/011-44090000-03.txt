MES incomplete or missing information

Table 1: actions Service Officers are required to follow when missing
documents or incomplete information for MES applications has been
received.

+-----------------------------------+-----------------------------------+
| Item                              | Document type                     |
+-----------------------------------+-----------------------------------+
| 1                                 | Certified documents + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Medicare Entitlement Statement    |
|                                   | (MES) supporting documents do not |
|                                   | need to be certified.             |
|                                   |                                   |
|                                   | Do not return the application to  |
|                                   | the applicant.                    |
|                                   |                                   |
|                                   | Continue to assess and process    |
|                                   | the application.                  |
|                                   |                                   |
|                                   | Accepting non-certified documents |
|                                   | is an internal process only. All  |
|                                   | external reference material (MES  |
|                                   | form and website information)     |
|                                   | will refer to certified copies.   |
|                                   |                                   |
|                                   | When answering enquires, advise   |
|                                   | applicants to provide certified   |
|                                   | copies.                           |
+-----------------------------------+-----------------------------------+
| 2                                 | Missing supporting documents +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If it has been established that   |
|                                   | the application is incomplete or  |
|                                   | documents are missing, check if   |
|                                   | missing information can be        |
|                                   | verified via other sources before |
|                                   | returning the application.        |
|                                   | Sources include:                  |
|                                   |                                   |
|                                   | -   Visa Entitlement Verification |
|                                   |     Online (VEVO)                 |
|                                   | -   Immigration Movement Record   |
|                                   |     (IMR)                         |
|                                   | -   Consumer Directory            |
|                                   |     Maintenance System (CDMS)     |
|                                   | -   Processing and National       |
|                                   |     Demand Allocation (PaNDA)     |
|                                   | -   MES positional email box      |
|                                   |                                   |
|                                   | After all sources have been       |
|                                   | checked and an assessment cannot  |
|                                   | be made, return the application   |
|                                   | to the applicant with an          |
|                                   | appropriate letter, see Access    |
|                                   | VEVO (Visa Entitlement            |
|                                   | Verification Online) program.     |
+-----------------------------------+-----------------------------------+
| 3                                 | MES application forms + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Before processing an application  |
|                                   | check if the applicant has        |
|                                   | completed the acceptable version  |
|                                   | of the MES application forms      |
|                                   | below. Version numbers can be     |
|                                   | identified on the bottom left     |
|                                   | corner of the form:               |
|                                   |                                   |
|                                   | -   MS015.2205                    |
|                                   | -   MS015.2008                    |
|                                   |                                   |
|                                   | If an application is submitted on |
|                                   | an old MES application form (that |
|                                   | is, not listed above), return the |
|                                   | application to the applicant and: |
|                                   |                                   |
|                                   | -   complete standard letter      |
|                                   |     template z2456 – MES          |
|                                   |     application needs to be       |
|                                   |     resubmitted on the current    |
|                                   |     form                          |
|                                   | -   add copy of letter to the     |
|                                   |     application in PaNDA          |
|                                   |                                   |
|                                   | Do not assess the application or  |
|                                   | enter the applicant’s details     |
|                                   | into the MES database before      |
|                                   | returning the application.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Incomplete application forms +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The following questions on the    |
|                                   | form must be completed by the     |
|                                   | applicant:                        |
|                                   |                                   |
|                                   | applicant's name                  |
|                                   |                                   |
|                                   | questions relating to             |
|                                   | applications for permanent        |
|                                   | residency                         |
|                                   |                                   |
|                                   | questions relating to prior place |
|                                   | of residence. Exception- staff    |
|                                   | can verify questions 13 and 14 of |
|                                   | the MES application form by phone |
|                                   |                                   |
|                                   | declaration ticked (electronic or |
|                                   | hand written are permitted)       |
|                                   | including their:                  |
|                                   |                                   |
|                                   | -   full name (middle name/s not  |
|                                   |     mandatory), and               |
|                                   | -   date (not mandatory)          |
|                                   |                                   |
|                                   | Note: for previous form           |
|                                   | MS015.2008, the applicant must    |
|                                   | sign (electronic signatures are   |
|                                   | permitted).                       |
|                                   |                                   |
|                                   | If any of the above questions are |
|                                   | incomplete, check original        |
|                                   | application in email and upload   |
|                                   | to PaNDA.                         |
|                                   |                                   |
|                                   | If the questions listed above are |
|                                   | not completed:                    |
|                                   |                                   |
|                                   | -   return the application to the |
|                                   |     applicant with standard       |
|                                   |     letter template z0132; and    |
|                                   | -   add a copy of the letter and  |
|                                   |     attach it to the application  |
|                                   |     in PaNDA                      |
|                                   |                                   |
|                                   | If questions other than those     |
|                                   | listed above have not been        |
|                                   | completed:                        |
|                                   |                                   |
|                                   | -   make 2 attempts to contact    |
|                                   |     the applicant by phone to     |
|                                   |     verify details                |
|                                   |                                   |
|                                   | If contact has been made with     |
|                                   | applicant:                        |
|                                   |                                   |
|                                   | -   verify required information   |
|                                   |     with applicant                |
|                                   | -   add comments in the MES       |
|                                   |     database about the type of    |
|                                   |     information verified with     |
|                                   |     applicant                     |
|                                   | -   complete a separate document  |
|                                   |     stating type of information   |
|                                   |     verified, the date then add   |
|                                   |     to file in PaNDA              |
|                                   | -   continue assessing            |
|                                   |     application                   |
|                                   |                                   |
|                                   | If contact could not be made with |
|                                   | the applicant:                    |
|                                   |                                   |
|                                   | -   add comments in the MES       |
|                                   |     database ‘application form    |
|                                   |     not completed unable to make  |
|                                   |     contact with applicant’ see   |
|                                   |     Table 5                       |
|                                   | -   complete a separate document  |
|                                   |     stating application form not  |
|                                   |     completed, unable to contact  |
|                                   |     applicant. Then add to file   |
|                                   |     in PaNDA                      |
|                                   | -   return the application to the |
|                                   |     applicant with standard       |
|                                   |     letter template z0132         |
|                                   | -   add copy of letter to         |
|                                   |     application in PaNDA          |

MES and the end of a financial year

Table 2: process Service Officers follow for applications lodged before
the end of a financial year or on or after the commencement of the
financial year.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Applications received before 1    |
|                                   | June + Read more ...              |
|                                   |                                   |
|                                   | Applications received before the  |
|                                   | end of the current financial      |
|                                   | year, return to the applicant     |
|                                   | using the standard letter         |
|                                   | template z0132.                   |
|                                   |                                   |
|                                   | See Table 17 for the email        |
|                                   | response process.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Applications received on or after |
|                                   | 1 June + Read more ...            |
|                                   |                                   |
|                                   | Assess the application following  |
|                                   | the normal processing guidelines  |
|                                   | and checks.                       |
|                                   |                                   |
|                                   | -   If all the information is     |
|                                   |     provided, go to Step 3        |
|                                   | -   If the application form is    |
|                                   |     incomplete or further         |
|                                   |     documents are required, go to |
|                                   |     Step 4                        |
+-----------------------------------+-----------------------------------+
| 3                                 | All information provided - key    |
|                                   | application + Read more ...       |
|                                   |                                   |
|                                   | Key application into the MES      |
|                                   | database, see Table 5             |
|                                   |                                   |
|                                   | Force application into a ‘Pend’   |
|                                   | status by leaving the visa        |
|                                   | sighted and passport sighted tick |
|                                   | boxes blank                       |
|                                   |                                   |
|                                   | Add comment in MES Database –     |
|                                   | ‘EOFY YY-YY’                      |
|                                   |                                   |
|                                   | PaNDA – file is Pended            |
|                                   |                                   |
|                                   | -   Status – Pended               |
|                                   | -   Pend Reason – Future Release  |
|                                   |     Date                          |
|                                   | -   Release Date – dd/mm/yyyy     |
|                                   |                                   |
|                                   | Notate a comment in the ‘add      |
|                                   | processing notes’ section in      |
|                                   | PaNDA to advise why application   |
|                                   | is being pended – EOFY – YY-YY    |
+-----------------------------------+-----------------------------------+
| 4                                 | Application form incomplete - key |
|                                   | application + Read more ...       |
|                                   |                                   |
|                                   | -   Key application into the MES  |
|                                   |     database, see Table 5         |
|                                   | -   Outcome – Pend MES            |
|                                   |     Application                   |
|                                   | -   Return the application to the |
|                                   |     applicant with standard       |
|                                   |     letter template z0132         |
|                                   | -   Add copy of letter to         |
|                                   |     application in PaNDA          |
+-----------------------------------+-----------------------------------+

Medicare record in CDMS and Immigration data link

Table 3: this table describes the process Service Officers follow when
searching for an existing Medicare record in the Consumer Directory
Maintenance System (CDMS) and Immigration data link.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Register new consumer + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Register New Consumer  |
|                                   |     on the CDMS Home page         |
|                                   | -   The Registration Type screen  |
|                                   |     is displayed                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Enter consumer details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If an applicant has only one name |
|                                   | recorded on the passport, key the |
|                                   | only name in the Family name      |
|                                   | field and leave the First name    |
|                                   | field blank.                      |
|                                   |                                   |
|                                   | Make sure all variants of name    |
|                                   | searches are conducted, for       |
|                                   | instance, reversal of surname and |
|                                   | first name, split one name,       |
|                                   | removal of hyphens, adding or     |
|                                   | removing spaces in names and use  |
|                                   | common abbreviations of long      |
|                                   | names.                            |
|                                   |                                   |
|                                   | Select New Consumer               |
|                                   |                                   |
|                                   | Select Immigration Data search    |
|                                   | tick box                          |
|                                   |                                   |
|                                   | Enter applicant’s details in the  |
|                                   | following fields:                 |
|                                   |                                   |
|                                   | -   Family name                   |
|                                   | -   First name                    |
|                                   | -   Gender                        |
|                                   | -   Date of birth                 |
|                                   | -   Postcode or Postcode range    |
|                                   |     (leave field as default, for  |
|                                   |     example 0001-9999)            |
|                                   |                                   |
|                                   | Select Search                     |
|                                   |                                   |
|                                   | The warning message will display  |
|                                   | ‘01875 Greater than 12 months of  |
|                                   | age’                              |
|                                   |                                   |
|                                   | Select Yes to override warning    |
|                                   | message                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Returned results + Read more ...  |
|                                   |                                   |
|                                   | If no records are found, a        |
|                                   | message '00461 – no records       |
|                                   | found’ displays. See Table 4.     |
|                                   |                                   |
|                                   | If records are found              |
|                                   |                                   |
|                                   | If a Medicare record exists, the  |
|                                   | applicant’s details will display  |
|                                   | under the Matched Consumers tab.  |
|                                   | This tab also identifies a        |
|                                   | customer record which may be an   |
|                                   | AIR Record of IHI Record          |
|                                   |                                   |
|                                   | -   Select the consumer using the |
|                                   |     radio button. Select Amend    |
|                                   | -   Warning Message: 00382 NO     |
|                                   |     RECORDS FOUND                 |
|                                   | -   Proceed to assess MES ( Not a |
|                                   |     Medicare Record),             |
|                                   |     or                            |
|                                   | -   Select the consumer using the |
|                                   |     radio button. Select Amend    |
|                                   | -   Warning Message: 00075 CHANGE |
|                                   |     BUTTON SELECTION? Select NO   |
|                                   | -   Amend Personal Details page   |
|                                   |     will display                  |
|                                   | -   Scroll down. Select           |
|                                   |     ENTITLEMENT                   |
|                                   | -   Warning Message: 00306 NOT    |
|                                   |     REGISTERED FOR SELECTED       |
|                                   |     PROGRAM                       |
|                                   | -   Proceed to assess MES ( Not a |
|                                   |     Medicare Record)              |
|                                   |                                   |
|                                   | If the Immigration client tab is  |
|                                   | highlighted, this indicates       |
|                                   | possible matches for the          |
|                                   | applicant                         |
|                                   |                                   |
|                                   | The applicant can have a matched  |
|                                   | consumer and/or Immigration data  |
|                                   | link                              |
|                                   |                                   |
|                                   | If the Immigration data link      |
|                                   | results in Permanent Residency    |
|                                   | being lodged only after the MES   |
|                                   | application was received, the     |
|                                   | application does not need to be   |
|                                   | returned for documents and can be |
|                                   | processed                         |
|                                   |                                   |
|                                   | The action required depends on    |
|                                   | the Service Officer’s skill tag   |
|                                   | and whether or not the            |
|                                   | application needs to be escalated |
|                                   | based on the search results       |
|                                   |                                   |
|                                   | Determine if the search results   |
|                                   | impact the MES application being  |
|                                   | assessed                          |
|                                   |                                   |
|                                   | See Work types and escalations in |
|                                   | PaNDA table on the Resources      |
|                                   | page.                             |

MES database

Table 4: how Service Officers search the MES database to identify
existing records and determine the impacts to the MES application
outcome.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Search the MES database + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select File                       |
|                                   |                                   |
|                                   | Select Open                       |
|                                   |                                   |
|                                   | Key applicant’s details in the    |
|                                   | following fields:                 |
|                                   |                                   |
|                                   | -   Surname                       |
|                                   | -   Date of birth                 |
|                                   |                                   |
|                                   | Select Search                     |
|                                   |                                   |
|                                   | When searching punctuated names,  |
|                                   | replace punctuation with a space, |
|                                   | for example, O’Connor should be O |
|                                   | Connor.                           |
|                                   |                                   |
|                                   | To expand search parameters, the  |
|                                   | first 3 characters of the last    |
|                                   | name can be entered followed by   |
|                                   | percentage symbol (%), for        |
|                                   | example, for Smith, key Smi%.     |
|                                   |                                   |
|                                   | Make sure all variants of name    |
|                                   | searches are searched.            |
|                                   |                                   |
|                                   | Is a match returned?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the application is ready  |
|                                   |     to be processed. See Table 5  |
+-----------------------------------+-----------------------------------+
| 2                                 | Matched records + Read more ...   |
|                                   |                                   |
|                                   | Is the matched record for the     |
|                                   | same claim period as the current  |
|                                   | application?                      |
|                                   |                                   |
|                                   | Yes, further investigation is     |
|                                   | required. Go to Step 3            |
|                                   |                                   |
|                                   | No, if a previous MES application |
|                                   | has a status of:                  |
|                                   |                                   |
|                                   | -   Reject or Pend, open the      |
|                                   |     application in database and   |
|                                   |     further investigate by        |
|                                   |     reading the processing notes. |
|                                   |     Determine if the status of    |
|                                   |     previous applications will    |
|                                   |     impact the current            |
|                                   |     application                   |
|                                   | -   Approved, open to view        |
|                                   |     previous residence details,   |
|                                   |     if previous residence         |
|                                   |     indicated is RHCA country may |
|                                   |     impact current application,   |
|                                   |     further checks may be         |
|                                   |     required                      |
|                                   |                                   |
|                                   | If all checks have been completed |
|                                   | and there is no impact, the       |
|                                   | application is ready to be        |
|                                   | processed. See Table 5. Procedure |
|                                   | ends here                         |
|                                   |                                   |
|                                   | If trained in general layer MES   |
|                                   | processing or unable to determine |
|                                   | the impact, escalate the          |
|                                   | application to a more experienced |
|                                   | Service Officer by raising the    |
|                                   | proficiency or re-categorise      |
|                                   | correctly in PaNDA. See Work      |
|                                   | Optimiser for staff               |
|                                   |                                   |
|                                   | If there are impacts and the      |
|                                   | Service Officer has the required  |
|                                   | skill tag and all checks have     |
|                                   | been complete the application is  |
|                                   | ready to be processed, see Table  |
|                                   | 5                                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Determine the application         |
|                                   | outcome + Read more ...           |
|                                   |                                   |
|                                   | After all checks are complete and |
|                                   | the application does not have to  |
|                                   | be escalated or reassigned,       |
|                                   | determine the application         |
|                                   | outcome:                          |
|                                   |                                   |
|                                   | -   Certified (eligible for MES – |
|                                   |     full or partial period), go   |
|                                   |     to Step 4                     |
|                                   | -   Approved (eligible for MES -  |
|                                   |     full or partial period.       |
|                                   |     Application processed on the  |
|                                   |     same day of checking but not  |
|                                   |     yet certified.), go to Step 4 |
|                                   | -   Pend (further information or  |
|                                   |     missing information), go to   |
|                                   |     Step 7                        |
|                                   | -   Reject (not eligible for      |
|                                   |     MES), go to Step 8            |
+-----------------------------------+-----------------------------------+
| 4                                 | Duplicated application + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Determine if the application is a |
|                                   | duplicate.                        |
|                                   |                                   |
|                                   | A duplicate is where the below    |
|                                   | information is the same on the    |
|                                   | Certified or Approved application |
|                                   | and the application that is       |
|                                   | assessing:                        |
|                                   |                                   |
|                                   | -   financial year being claimed  |
|                                   | -   address information           |
|                                   | -   Tax Agent details are the     |
|                                   |     same (if applicable)          |
|                                   |                                   |
|                                   | Is the application a duplicate:   |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 5                                 | Verify the processing date + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the application is Certified,  |
|                                   | Service Officers must verify the  |
|                                   | processing date.                  |
|                                   |                                   |
|                                   | If the application was processed: |
|                                   |                                   |
|                                   | under 2 weeks from the processing |
|                                   | date:                             |
|                                   |                                   |
|                                   | -   no further action is          |
|                                   |     required, the MES is still in |
|                                   |     the mailing process           |
|                                   | -   notate the application number |
|                                   |     in the Case Id field in PaNDA |
|                                   | -   notate in the ‘add process    |
|                                   |     notes’ section ‘Dup           |
|                                   |     application –NFA’             |
|                                   | -   Note: comments not required   |
|                                   |     in MES database               |
|                                   |                                   |
|                                   | over 2 weeks from the processing  |
|                                   | date:                             |
|                                   |                                   |
|                                   | -   see Table 8                   |
|                                   | -   key comment ‘Dup application  |
|                                   |     MES reprinted and sent’ in    |
|                                   |     Comments field, in MES        |
|                                   |     database                      |
|                                   |                                   |
|                                   | If the application is Approved:   |
|                                   |                                   |
|                                   | -   no further action is          |
|                                   |     required, Duplicate           |
|                                   |     application processed on the  |
|                                   |     same day                      |
|                                   | -   notate the application number |
|                                   |     in the Case Id field in PaNDA |
|                                   | -   notate in the ‘add process    |
|                                   |     notes’ section ‘Dup           |
|                                   |     application – NFA’            |
|                                   |                                   |
|                                   | Comments not required in MES      |
|                                   | database.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Determine the next course of      |
|                                   | action + Read more ...            |
|                                   |                                   |
|                                   | Service Officers must check the   |
|                                   | following information to          |
|                                   | determine the next course of      |
|                                   | action:                           |
|                                   |                                   |
|                                   | If the postal address on the      |
|                                   | database is:                      |
|                                   |                                   |
|                                   | -   different to the application  |
|                                   |     form, see Table 8 (a new      |
|                                   |     statement will need to be     |
|                                   |     issued)                       |
|                                   | -   not different to the          |
|                                   |     application form, go to next  |
|                                   |     question                      |
|                                   |                                   |
|                                   | If the approved period on the     |
|                                   | database is:                      |
|                                   |                                   |
|                                   | -   different to the request      |
|                                   |     period on the application     |
|                                   |     form, further investigation   |
|                                   |     is required to determine MES  |
|                                   |     eligibility is correct, see   |
|                                   |     Table 12                      |
|                                   | -   not different to the request  |
|                                   |     period on the application     |
|                                   |     form, go to next question     |
|                                   |                                   |
|                                   | If the Tax agent details on the   |
|                                   | database are:                     |
|                                   |                                   |
|                                   | -   different to the application  |
|                                   |     form, see Table 8 (a new      |
|                                   |     statement will need to be     |
|                                   |     issued)                       |
|                                   | -   not different to the          |
|                                   |     application form, if there    |
|                                   |     are other discrepancies       |
|                                   |     escalate the application to a |
|                                   |     more experienced Service      |
|                                   |     Officer by raising the        |
|                                   |     proficiency in PaNDA. See     |
|                                   |     Work Optimiser for staff      |
+-----------------------------------+-----------------------------------+
| 7                                 | Application status is Pend + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check Comments tab to see why the |
|                                   | application was pended.           |
|                                   |                                   |
|                                   | Has the application been returned |
|                                   | with all the requested            |
|                                   | documentation?                    |
|                                   |                                   |
|                                   | Yes, proceed with processing, see |
|                                   | Table 5                           |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Search PaNDA and email to     |
|                                   |     make sure documents haven’t   |
|                                   |     been uploaded separately      |
|                                   | -   After searching PaNDA and     |
|                                   |     requested documents have not  |
|                                   |     been submitted, return the    |
|                                   |     application to applicant with |
|                                   |     letter z0132 re-requesting    |
|                                   |     the additional information    |
|                                   | -   If the missing information    |
|                                   |     relates to application for    |
|                                   |     Permanent Residency document  |
|                                   |     not being submitted, and the  |
|                                   |     application has been already  |
