Q888 or Q999 letter text

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Decision not to apply PPR         |
|                                   |                                   |
|                                   | 'The decision has been made not   |
|                                   | to continue payment of (insert    |
|                                   | relevant payment) pending the     |
|                                   | outcome of your review.'          |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Decision to apply PPRd            |
|                                   |                                   |
|                                   | 'The decision has been made to    |
|                                   | continue payment of (insert       |
|                                   | relevant payment) pending the     |
|                                   | outcome of your review.'          |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+
| 3                                 | Decision to end PPR               |
|                                   |                                   |
|                                   | 'We have been paying your (insert |
|                                   | relevant payment) under payment   |
|                                   | pending review provisions while   |
|                                   | your review is being finalised.   |
|                                   |                                   |
|                                   | 'As the review is now complete'   |
|                                   | (enter outcome of review and the  |
|                                   | effect on the customer's          |
|                                   | payment).For example:             |
|                                   |                                   |
|                                   | -   'and it has been found that   |
|                                   |     the original decision is      |
|                                   |     correct, your payments have   |
|                                   |     now ceased', or               |
|                                   | -   'and it has been found that   |
|                                   |     the original decision should  |
|                                   |     be changed, your payments     |
|                                   |     will continue under normal    |
|                                   |     provisions'                   |
|                                   |                                   |
|                                   | Include the Appeal paragraph.     |
+-----------------------------------+-----------------------------------+

Examples

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Parenting Payment Single (PPS)    |
|                                   | cancelled                         |
|                                   |                                   |
|                                   | PPS is cancelled due to a         |
|                                   | decision that the customer is a   |
|                                   | member of a couple.               |
|                                   |                                   |
|                                   | The customer contacts to apply    |
|                                   | for a formal review. They advise  |
|                                   | they cannot afford to buy         |
|                                   | medication for their child. They  |
|                                   | have provided letters from third  |
|                                   | parties saying they are separated |
|                                   | from their ex-partner.            |
|                                   |                                   |
|                                   | The Service Officers applies PPR  |
|                                   | and restores PPS.                 |
|                                   |                                   |
|                                   | A formal review is completed and  |
|                                   | the Authorised Review Officer     |
|                                   | (ARO) finds the customer is not a |
|                                   | member of a couple. They set      |
|                                   | aside the cancellation decision.  |
|                                   | PPS continues under normal        |
|                                   | provisions, not under PPR         |
|                                   | provisions.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Carer Payment (CP) cancelled      |
|                                   |                                   |
|                                   | CP is cancelled due to a decision |
|                                   | that care requirements are no     |
|                                   | longer satisfied                  |
|                                   |                                   |
|                                   | The customer applies for a formal |
|                                   | review of the decision. They      |
|                                   | provide a new medical report.     |
|                                   | They advise they are in financial |
|                                   | hardship.                         |
|                                   |                                   |
|                                   | The Service Officer applies PPR   |
|                                   | and restores CP.                  |
|                                   |                                   |
|                                   | The Authorised Review Officer     |
|                                   | (ARO) affirms the cancellation    |
|                                   | decision                          |
|                                   |                                   |
|                                   | The customer applies for a review |
|                                   | by the Administrative Appeals     |
|                                   | Tribunal (AAT). They ask that PPR |
|                                   | continue as cancellation of CP    |
|                                   | will cause financial hardship.    |
|                                   | PPR is applied and CP restored.   |
|                                   |                                   |
|                                   | The AAT affirms the cancellation  |
|                                   | decision. PPR is ended and CP     |
|                                   | cancelled.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Disability Support Pension (DSP)  |
|                                   | suspended                         |
|                                   |                                   |
|                                   | DSP is suspended for one          |
|                                   | fortnight because of              |
|                                   | non-compliance with participation |
|                                   | requirements.                     |
|                                   |                                   |
|                                   | PPR is not applied as the         |
|                                   | decision affects the customer for |
|                                   | a limited period.                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Age Pension rate reduced          |
|                                   |                                   |
|                                   | A customer's rate of Age Pension  |
|                                   | is reduced due to a decision that |
|                                   | the customer is a member of a     |
|                                   | couple.                           |
|                                   |                                   |
|                                   | The customer applies for a formal |
|                                   | review.                           |
|                                   |                                   |
|                                   | PPR is applied. Age Pension is    |
|                                   | paid at the single rate.          |
|                                   |                                   |
|                                   | Before an Authorised Review       |
|                                   | Officer (ARO) can complete the    |
|                                   | formal review, a real estate      |
|                                   | review is done. The reassessment  |
|                                   | values the customer’s assets      |
|                                   | above the limit for payment of    |
|                                   | Age Pension.                      |
|                                   |                                   |
|                                   | PPR is ended and Age Pension      |
|                                   | cancelled under the assets test.  |
+-----------------------------------+-----------------------------------+
| 5                                 | Jobseeker Payment (JSP) rate      |
|                                   | reduced                           |
|                                   |                                   |
|                                   | A customer's JSP rate reduces due |
|                                   | to their partner's income from    |
|                                   | employment.                       |
|                                   |                                   |
|                                   | PPR cannot apply as the decision  |
|                                   | is not discretionary or based on  |
|                                   | an opinion.                       |
+-----------------------------------+-----------------------------------+
