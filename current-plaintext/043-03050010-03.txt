This page contains information on the factors used in determining the
nature of business of commission sales persons, and how income is
assessed for commission sales persons who are considered to be
self-employed.

Employment assessment

Factors for determining if a commission sales person is self-employed

This table describes the factors used in determining the nature of
business of commission sales persons, and how income is assessed for
commission sales persons who are considered to be self-employed.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Employed on commission basis +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customer advises they are         |
|                                   | employed on a commission basis to |
|                                   | sell a product and/or recruit     |
|                                   | other commission sales people.    |
+-----------------------------------+-----------------------------------+
| 2                                 | Contract for service + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the customer under a contract  |
|                                   | for service (that is, to produce  |
|                                   | a result)?                        |
|                                   |                                   |
|                                   | -   Yes, customer is regarded as  |
|                                   |     self-employed. See Income for |
|                                   |     an independent contractor and |
|                                   |     commission income. Procedure  |
|                                   |     ends here                     |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Level of control + Read more ...  |
|                                   |                                   |
|                                   | Does the person have a high level |
|                                   | of control and choice regarding   |
|                                   | what work is to be done, how,     |
|                                   | when, where and what resources    |
|                                   | are to be used?                   |
|                                   |                                   |
|                                   | -   Yes, customer is regarded as  |
|                                   |     self-employed. See Income for |
|                                   |     an independent contractor and |
|                                   |     commission income. Procedure  |
|                                   |     ends here                     |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Profit or loss risk + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Does the risk of profit or loss   |
|                                   | rest with the person?             |
|                                   |                                   |
|                                   | -   Yes, customer is regarded as  |
|                                   |     self-employed. See Income for |
|                                   |     an independent contractor and |
|                                   |     commission income. Procedure  |
|                                   |     ends here                     |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Home based + Read more ...        |
|                                   |                                   |
|                                   | Does the person purchase goods    |
|                                   | such as stationery, sample        |
|                                   | products, etc., and operate from  |
|                                   | their own home or premises        |
|                                   | without reimbursement for costs   |
|                                   | such as electricity, postage and  |
|                                   | transport?                        |
|                                   |                                   |
|                                   | -   Yes, customer is regarded as  |
|                                   |     self-employed. See Income for |
|                                   |     an independent contractor and |
|                                   |     commission income. Procedure  |
|                                   |     ends here                     |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 6                                 | Receive a payment summary? + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Does the person receive a payment |
|                                   | summary?                          |
|                                   |                                   |
|                                   | Yes, customer is regarded as an   |
|                                   | employee                          |
|                                   |                                   |
|                                   | -   For more information, see     |
|                                   |     Commission sales persons      |
|                                   |     considered as employees       |
|                                   |                                   |
|                                   | No, customer may be regarded as   |
|                                   | self-employed. See Income for an  |
|                                   | independent contractor and        |
|                                   | commission income                 |
