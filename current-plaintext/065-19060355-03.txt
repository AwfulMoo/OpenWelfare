|                                   | more ...                          |
|                                   |                                   |

|                                   | customer has provided in their    |
|                                   | Scheme application.               |
|                                   |                                   |
|                                   | Note: if the Scheme application   |
|                                   | was lodged online or via Assisted |
|                                   | Customer Claim (ACC), most        |
|                                   | information provided by the       |
|                                   | customer will pre-populate to the |
|                                   | applicable screen. Check the      |
|                                   | claim information to ensure the   |

|                                   |                                   |
|                                   | For grant outcomes:               |

|                                   |     the pension assets test are   |

|                                   |                                   |

|                                   |     (BA) screen                   |

| 10                                | Project loan balance + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Home Equity Access Scheme     |
|                                   | grant and projection calculator   |
|                                   | is used to project the Maximum    |
|                                   | Loan Amount (MLA), loan balance   |
|                                   | and accrued interest. The purpose |
|                                   | of projecting the balance is to   |
|                                   | provide the customer with an      |
|                                   | estimate of future balance and    |
|                                   | repayment obligations to ensure   |
|                                   | they can make an informed         |
|                                   | decision before agreeing to       |
