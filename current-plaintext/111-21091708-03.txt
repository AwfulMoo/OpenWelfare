This page has the process of how to seek TPS for staff who are deaf or
have hearing loss.

On this page:

Service Officer

Technical Peer Support Officer (TPSO)

Service Officer

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Make sure the request meets the   |
|                                   | expectations + Read more ...      |
|                                   |                                   |
|                                   | Research thoroughly using a range |
|                                   | of online tools and reference     |
|                                   | material before contacting TPS.   |
|                                   | For example:                      |
|                                   |                                   |
|                                   | -   Operational Blueprint         |
|                                   | -   Network News Update (NNU)     |
|                                   | -   Office Locator                |
|                                   | -   mySupport                     |
|                                   | -   Roxy Digital Assistant        |
|                                   |     response                      |
|                                   | -   Process Direct current issues |
|                                   |                                   |
|                                   | Contact TPS if:                   |
|                                   |                                   |
|                                   | an activity results in arrears or |
|                                   | a negative adjustment and the     |
|                                   | Service Officer is not sure if    |
|                                   | this result is correct based on   |
|                                   | the reference material used       |
|                                   |                                   |
|                                   | relevant reference material does  |
|                                   | not provide a solution when       |
|                                   |                                   |
|                                   | -   the status or rate of payment |
|                                   |     does not reflect information  |
|                                   |     on the record                 |
|                                   | -   the system will not allow     |
|                                   |     updates, or                   |
|                                   | -   coding results in an error    |
|                                   |                                   |
|                                   | Do not contact TPS to confirm     |
|                                   | information found in reference    |
|                                   | material or that an activity is   |
|                                   | correct before completion.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Create a SharePoint Profile (only |
|                                   | required once) + Read more ...    |
|                                   |                                   |
|                                   | -   Open the Staff who are deaf   |
|                                   |     or have hearing loss          |
|                                   |     SharePoint                    |
|                                   | -   Select Profile                |
|                                   | -   Fill in all required details  |
|                                   |     on the form                   |
|                                   | -   Select Save                   |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete the request for          |
|                                   | support + Read more ...           |
|                                   |                                   |
|                                   | -   Open the Staff who are deaf   |
|                                   |     or have hearing loss          |
|                                   |     SharePoint                    |
|                                   | -   Select New Item               |
|                                   | -   Fill in all required fields   |
|                                   |     on the form                   |
|                                   | -   Select Submit                 |
|                                   | -   Refresh [F5] the page to see  |
|                                   |     the status of the request     |
|                                   |                                   |
|                                   | Staff can view and monitor their  |
|                                   | requests in the My Items tab.     |
+-----------------------------------+-----------------------------------+
| 4                                 | High demand in the SharePoint     |
|                                   | page + Read more ...              |
|                                   |                                   |
|                                   | If the Service Officer has waited |
|                                   | more than 10 minutes to have      |
|                                   | their enquiry acknowledged, they  |
|                                   | should:                           |
|                                   |                                   |
|                                   | -   seek technical support        |
|                                   |     locally                       |
|                                   | -   select the Edit icon          |
|                                   | -   select Withdraw               |
|                                   | -   send feedback via email to    |
|                                   |     the Smart Centre Quality      |
|                                   |     Portfolio Operations Team     |

Technical Peer Support Officer (TPSO)

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | View items submitted for          |
|                                   | Technical Support + Read more ... |
|                                   |                                   |
|                                   | When scheduled for TTY, open the  |
|                                   | Staff who are deaf or have        |
|                                   | hearing loss SharePoint.          |
|                                   |                                   |
|                                   | Always Refresh [F5] the screen to |
|                                   | view the most up-to-date items.   |
|                                   |                                   |
|                                   | Select Admin view.                |
|                                   |                                   |
|                                   | New items display at the top of   |
|                                   | the list, then items Acknowledged |
|                                   | or Under review, then items       |
|                                   | Finalised.                        |
|                                   |                                   |
|                                   | TPSOs need to place themselves    |
|                                   | into the not ready auxiliary      |
|                                   | code.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Select new item + Read more ...   |
|                                   |                                   |
|                                   | Select the Edit icon to action a  |
|                                   | New item                          |
|                                   |                                   |
|                                   | Select Acknowledge to show the    |
|                                   | TPSO is aware of the request.     |
|                                   | This will:                        |
|                                   |                                   |
|                                   | -   update the request status     |
|                                   | -   give staff a visual on the    |
|                                   |     progress of their request,    |
|                                   |     and                           |
|                                   | -   prevent another TPSO from     |
|                                   |     selecting the item            |
|                                   |                                   |
|                                   | Select Under Review to:           |
|                                   |                                   |
|                                   | -   update the request status     |
|                                   | -   give staff a visual on the    |
|                                   |     progress of their request     |
|                                   | -   prevent another TPSO from     |
|                                   |     selecting the item            |
+-----------------------------------+-----------------------------------+
| 3                                 | Can the TPSO help with the        |
|                                   | request under review? + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No, if the TPSO:                  |
|                                   |                                   |
|                                   | -   selects an item they do not   |
|                                   |     have the skillset to          |
|                                   |     complete, go to Step 6        |
|                                   | -   identifies the need to        |
|                                   |     escalate the request to a     |
|                                   |     Service Support Officer       |
|                                   |     (SSO), go to Step 7           |
+-----------------------------------+-----------------------------------+
| 4                                 | Technical Support Centre (TSC) +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The TSC must be completed for     |
|                                   | every TPS contact completed.      |
|                                   |                                   |
|                                   | Exception: when handing off the   |
|                                   | enquiry to another TPSO.          |
|                                   |                                   |
|                                   | When completing the TSC form, be  |
|                                   | accurate, clear and succinct.     |
|                                   | Choose the correct drop down      |
|                                   | options.                          |
|                                   |                                   |
|                                   | Do not record any personal or     |
|                                   | customer details in the TSC. This |
|                                   | includes names, dates of birth    |
|                                   | and any other customer details,   |
|                                   | CRNs, Aged Care Staff Portal      |
|                                   | (ACSP), ACMPS, children detail,   |
|                                   | ex-partner detail in the free     |
|                                   | text fields.                      |
|                                   |                                   |
|                                   | After completing the TSC record,  |
|                                   | choose the Finalise and notify SO |
|                                   | option.                           |
|                                   |                                   |
|                                   | This will send an email to the    |
|                                   | Service Officer with a summary of |
|                                   | the TPS contact.                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Finalise the SharePoint request   |
|                                   | form + Read more ...              |
|                                   |                                   |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Finalise               |
|                                   | -   Select Close                  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 6                                 | To refer to another TPSO + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Using the hard phone, contact |
|                                   |     the TPS line for the relevant |
|                                   |     business area (see Office     |
|                                   |     Locator for TPS numbers)      |
|                                   | -   Give an overview of the       |
|                                   |     enquiry, explaining the       |
|                                   |     Service Officer is deaf or    |
|                                   |     has hearing loss and their    |
|                                   |     preferred communication       |
|                                   |     method                        |
|                                   | -   Make sure there is a          |
|                                   |     connection between the other  |
|                                   |     TPSO and the Service Officer  |
|                                   |     before ending the call        |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Re-allocate to         |
|                                   |     alternate TPSO to finalise    |
|                                   |     the enquiry                   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 7                                 | To escalate to SSO + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Follow the current process in |
|                                   |     Step 6 of TPS role in Smart   |
|                                   |     Centres,                      |
|                                   | -   Select the Edit icon for the  |
|                                   |     item                          |
|                                   | -   Select Escalated to SSO to    |
|                                   |     finalise the enquiry          |
+-----------------------------------+-----------------------------------+
