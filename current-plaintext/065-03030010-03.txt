This page contains information on how to assess transfers to Age Pension
when the customer has completed the transfer process online, an Assisted
Customer Claim (ACC), or when they have lodged a Transfer to Age Pension
(SA383) form.

| 6                                 | Check if the customer will be     |
|                                   | disadvantaged + Read more ...     |
|                                   |                                   |
|                                   | Customers who are eligible to     |
|                                   | transfer to Age Pension are       |
|                                   | provided with details on the      |
|                                   | disadvantages and advantages      |
|                                   | using data from their record. See |

|                                   |                                   |
|                                   | Note: Widow Allowance (WA) and    |
|                                   | Partner Allowance (PA) will cease |

|                                   | need to transfer to Age Pension   |
|                                   | to continue receiving payment     |
|                                   | after this date.                  |
|                                   |                                   |
|                                   | If the customer's decision to     |
|                                   | transfer to Age Pension would     |
|                                   | disadvantage them make 2 genuine  |
|                                   | attempts to contact the customer: |
|                                   |                                   |
|                                   | -   If customer contact is        |
|                                   |     successful: advise the        |
|                                   |     advantages and disadvantages  |
|                                   |     of transferring. Confirm      |
|                                   |     their decision to transfer to |
|                                   |     Age Pension and DOC the       |
|                                   |     record                        |
|                                   | -   If customer contact is        |
|                                   |     unsuccessful: DOC the record  |
|                                   |     with details of that attempts |
|                                   |     and process the transfer      |
|                                   |                                   |
|                                   | The customer would be             |
|                                   | disadvantaged by transferring to  |
|                                   | Age Pension in the following      |
|                                   | scenarios, however, they may      |
|                                   | still prefer to access the        |
|                                   | advantages of Age Pension.        |
|                                   |                                   |
|                                   | Carer Supplement                  |
|                                   |                                   |
|                                   | -   This disadvantage applies to  |
|                                   |     all customers receiving Carer |
|                                   |     Payment (CP). If transferred  |
|                                   |     to Age Pension, they would no |
|                                   |     longer be eligible for an     |
|                                   |     annual Carer Supplement in    |
|                                   |     respect of CP. If they        |
|                                   |     receive Carer Allowance (CA)  |
|                                   |     they will continue to get an  |
|                                   |     annual Carer Supplement in    |
|                                   |     respect of CA.                |
|                                   |                                   |
|                                   | Bereavement assistance            |
|                                   |                                   |
|                                   | This disadvantage applies to all  |
|                                   | CP recipients if they are caring  |
|                                   | for someone who is not their      |
|                                   | partner and they transfer to Age  |
|                                   | Pension:                          |
|                                   |                                   |
|                                   | -   Customer will no longer       |
|                                   |     eligible for a CP Lump Sum    |
|                                   |     Bereavement Payment. This may |
|                                   |     be payable if the care        |
|                                   |     receiver is a child, is       |
|                                   |     single, or is partnered and   |
|                                   |     the care receiver's partner   |
|                                   |     (who is not the carer) is not |
|                                   |     getting an income support     |
|                                   |     payment.                      |
|                                   | -   If the customer gets CP       |
|                                   |     (XWP), they may be eligible   |
|                                   |     for a Lump Sum Bereavement    |
|                                   |     Payment (as above) if they    |
|                                   |     only get one payment level    |
|                                   |     Carer Allowance and the care  |
|                                   |     receiver dies.                |
|                                   | -   If the customer is only       |
|                                   |     caring for their partner,     |
|                                   |     they may still be eligible    |
|                                   |     for a Lump Sum Bereavement    |
|                                   |     Payment if they transfer to   |
|                                   |     Age Pension                   |
|                                   |                                   |
|                                   | Delaying the transfer             |
|                                   |                                   |
|                                   | -   If the customer still elects  |
|                                   |     to transfer, they may prefer  |
|                                   |     to delay it until their first |
|                                   |     Entitlement Period Start Date |
|                                   |     (EPSD) after 1 July if Carer  |
|                                   |     Supplement is due to be paid. |
|                                   |     This includes when they are   |
|                                   |     in a 14 week bereavement      |
|                                   |     period or in a 14 week        |
|                                   |     continuation period after the |
|                                   |     care receiver has been        |
|                                   |     admitted to an institution.   |
|                                   |     There is no need to delay it  |
|                                   |     if CP will stop before 1 July |
|                                   |     for any reason e.g. due to    |
|                                   |     respite or the maximum        |
|                                   |     portability period.           |
|                                   |                                   |
|                                   | Transferring from Disability      |
|                                   | Support Pension (DSP)             |
|                                   |                                   |
|                                   | Customers in receipt of Incentive |
|                                   | Allowance (IA) will lose          |
|                                   | eligibility to IA if they         |
|                                   | transfer to Age Pension, see the  |
|                                   | Pensions Rate Calculation (PRC)   |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Customers in receipt of IA who    |
|                                   | indicate they wish to transfer to |
|                                   | Age Pension must be contacted to  |
|                                   | advise that:                      |
|                                   |                                   |
|                                   | -   IA will cancel, and           |
|                                   | -   they may be eligible for Rent |
|                                   |     Assistance (RA) if they are   |
|                                   |     renting, and                  |
|                                   | -   if single, paying rent and    |
|                                   |     sharing accommodation, they   |
|                                   |     may wish to consider          |
|                                   |     remaining on DSP and claim RA |
|                                   |     instead of continuing to      |
|                                   |     receive IA as RA may be a     |
|                                   |     higher amount, see Sharer     |
|                                   |     Status field on the           |
|                                   |     Accommodation Details (AC)    |
|                                   |     screen                        |
|                                   |                                   |
|                                   | Customers whose partner is paid   |
|                                   | CP under the NZ agreement should  |
|                                   | be contacted as CP will cease if  |
|                                   | the DSP customer is transferred   |
|                                   | to Age Pension. The agreement     |
|                                   | country code is on the partner's  |
|                                   | ARD screen.                       |
|                                   |                                   |
|                                   | Check if the customer wishes to   |
|                                   | stay on DSP and not transfer to   |
|                                   | Age Pension at this time. See the |
|                                   | Resources page for information    |
|                                   | about Age Pension versus current  |
|                                   | payment.                          |
|                                   |                                   |
|                                   | If customer wants to change their |
|                                   | decision about transferring to    |
|                                   | Age Pension, continue with        |
|                                   | applicable checks ensuring that   |
|                                   | change of decision is recorded in |
|                                   | finalisation DOC along with any   |
|                                   | other relevant details.           |
