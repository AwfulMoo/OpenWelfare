|                                   |                                   |

|                                   | Note:                             |
|                                   |                                   |
|                                   | Address - will identify           |

|                                   |                                   |
|                                   | Service - can be used to find     |
|                                   | businesses and organisations that |
|                                   | have a service area identified on |
|                                   | the Addition Properties (OAP)     |
|                                   | screen.                           |
|                                   |                                   |
|                                   | Spatial - this search uses        |
|                                   | geospatial data to find           |
|                                   | businesses and organisations that |
|                                   | are physically located within the |
|                                   | boundary of the geographical area |
|                                   | that has been identified:         |
|                                   |                                   |
|                                   | -   Organisation code             |
|                                   | -   ID type                       |
|                                   | -   Organisation ID               |
|                                   | -   Indigenous community filter   |
|                                   | -   Name type filter              |
|                                   | -   Select 'Search' button        |
|                                   |                                   |
|                                   | Search results page is displayed  |
|                                   | showing all matches that meet the |
|                                   | search criteria entered.          |
|                                   |                                   |
|                                   | -   Select business or            |
|                                   |     organisation from search list |
|                                   |     to show further information   |
|                                   |     regarding that business or    |
|                                   |     organisation                  |
