Medicare Compensation Recovery Bank account details collection form (MO024)

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Refund is confirmed for the       |
|                                   | injured person + Read more ...    |
|                                   |                                   |
|                                   | Follow process for Table 1, and   |
|                                   | confirm if the refund is going    |
|                                   | directly to the injured person.   |
|                                   | For minors, see Minors and        |
|                                   | Medicare Compensation Recovery    |
|                                   |                                   |
|                                   | Check if the Medicare             |
|                                   | Compensation Recovery Bank        |
|                                   | account details collection form   |
|                                   | (MO024) has been received. At     |
|                                   | mail validation, see Mandatory    |
|                                   | checks for Medicare Compensation  |
|                                   | Recovery, Process page Table 4    |
|                                   |                                   |
|                                   | Does the form contain all         |
|                                   | mandatory information?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, make a refund by cheque.  |
|                                   |     Process ends here             |
+-----------------------------------+-----------------------------------+
| 2                                 | Instructions from management +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Are there instructions for the    |
|                                   | refund account/MO024 from         |
|                                   | Programme Management, a Team      |
|                                   | Leader, or a Programme Support    |
|                                   | Officer?                          |
|                                   |                                   |
|                                   | -   Yes, follow the instruction,  |
|                                   |     then go to Step 3 unless      |
|                                   |     instructed otherwise. For     |
|                                   |     example, instruction to phone |
|                                   |     to confirm account details    |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Bank details for the injured      |
|                                   | person in CDMS + Read more ...    |
|                                   |                                   |
|                                   | Check which account update is the |
|                                   | most recent.                      |
|                                   |                                   |
|                                   | If MO024 is the most recent:      |
|                                   |                                   |
|                                   | -   Refund to the MO024 account   |
|                                   |                                   |
|                                   | Note: if the MO024 is unclear due |
|                                   | to handwriting or the scan,       |
|                                   | confirm the MO024 account against |
|                                   | CDMS. If still unclear, issue a   |
|                                   | cheque.                           |
|                                   |                                   |
|                                   | If CDMS is the most recent:       |
|                                   |                                   |
|                                   | Issue a cheque                    |
|                                   |                                   |
|                                   | Do not:                           |
|                                   |                                   |
|                                   | -   call the injured person       |
|                                   | -   refund to the CMDS account    |
|                                   |     unless same as the MO024      |
|                                   |     account                       |
|                                   | -   phone to confirm details on   |
|                                   |     historic forms                |
|                                   |                                   |
|                                   | See:                              |
|                                   |                                   |
|                                   | -   Notice of judgment or         |
|                                   |     settlement for Medicare       |
|                                   |     Compensation Recovery,        |
|                                   |     Process page Table 2, or      |
|                                   | -   Statement by claimant for     |
|                                   |     Medicare Compensation         |
|                                   |     Recovery, Process page Table  |
|                                   |     3                             |
+-----------------------------------+-----------------------------------+
| 4                                 | Rejected EFT payment + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Compensation Finance advise       |
|                                   | Service Delivery if a EFT payment |

|                                   | account details:                  |

|                                   |                                   |

|                                   |     Please re-issue refund to     |
|                                   |     XXXXX XXXXX (being correct    |
|                                   |     account details)’ and include |
|                                   |     a snip of the MO024 account   |

|                                   |     respond. Use error code MCR15 |

Public Trustee refunds

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Public Trustee refund confirmed + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For Public Trustee refunds        |
|                                   | confirmed at Table 1.             |
|                                   |                                   |
|                                   | If the matter identification      |
|                                   | number is not provided on the     |
|                                   | settlement documents, contact the |
|                                   | insurer or Public Trustee:        |
|                                   |                                   |
|                                   | by phone, or                      |
|                                   |                                   |
|                                   | mail if the Public Trustee asked  |
|                                   | for a written enquiry:            |
|                                   |                                   |
|                                   | -   send the z2570 Medicare       |
|                                   |     Compensation Recovery funds   |
|                                   |     owing manual letter, and      |
|                                   |     select the business option    |
|                                   | -   enter a case note             |
|                                   |                                   |
|                                   | Go to Step 2 once the matter      |
|                                   | identification number is          |
|                                   | received.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Public Trustee matter             |
|                                   | identification number provided +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Refund to be made to for a Public |
|                                   | Trustee in:                       |
|                                   |                                   |
|                                   | Tasmania or the Australian        |
|                                   | Capital Territory (ACT):          |
|                                   |                                   |
|                                   | -   issue a refund by cheque      |
|                                   |                                   |
|                                   | All other states or territory:    |
|                                   |                                   |
|                                   | -   issue a refund by EFT         |
|                                   |                                   |
|                                   | Go to Step 3                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Issue a refund using the          |
|                                   | workflow + Read more ...          |
|                                   |                                   |
|                                   | To issue the refund follow the    |
|                                   | process in:                       |
|                                   |                                   |
|                                   | -   Notice of judgment or         |
|                                   |     settlement for Medicare       |
|                                   |     Compensation Recovery,        |
|                                   |     Process page Table 2, or      |
|                                   | -   Statement by claimant for     |
|                                   |     Medicare Compensation         |
|                                   |     Recovery, Process page Table  |
|                                   |     3, or                         |
|                                   | -   Amended notices and reviews   |
|                                   |     for Medicare Compensation     |
|                                   |     Recovery Tables 1 - 3 if      |
|                                   |     refunding after the case is   |
|                                   |     finalised, as this must be    |
|                                   |     done manually                 |
|                                   |                                   |
|                                   | In the Staffware work flow:       |
|                                   |                                   |
|                                   | Edit claimant and injured         |
|                                   | person’s fields in CCMS to remove |
|                                   | any other party, and enter:       |
|                                   |                                   |
|                                   | -   the injured person’s name     |
|                                   | -   c/- Public Trustee into the   |
|                                   |     address field                 |
|                                   | -   the matter identification     |
|                                   |     number into the same field    |
|                                   |                                   |
|                                   | Prepare email template:           |
|                                   |                                   |
|                                   | -   Go to Referring a case to     |
|                                   |     compensation finance - email  |
|                                   |     template                      |
|                                   | -   Enter the ‘matter             |
|                                   |     identification number’ as the |
|                                   |     reference number on the email |
|                                   | -   Complete all relevant fields  |
|                                   |     in the template               |
|                                   |                                   |
|                                   | Finalise the template:            |
|                                   |                                   |
|                                   | if a Tasmanian or ACT Public      |
|                                   | Trustee, issue a cheque refund    |
|                                   |                                   |
|                                   | all other state or territory      |
|                                   | Public Trustees, issue a EFT      |
|                                   | refund                            |
|                                   |                                   |
|                                   | -   Finance to enter the          |
|                                   |     applicable EFT account        |
|                                   |                                   |
|                                   | Send the email template to the    |
|                                   | Compensation finance team. The    |
|                                   | Resources page contains a link    |
|                                   |                                   |
|                                   | Print 1 copy of the email         |
|                                   | template, and scan to the WIN as  |
|                                   | backlog                           |
|                                   |                                   |
|                                   | Enter a case note                 |
|                                   |                                   |
|                                   | Complete all steps in the         |
|                                   | workflows                         |
