Veterans' Supplement in aged care questions and answers

Table 1: This table lists questions and answers.

+-----------------------------------+-----------------------------------+
| Item                              | Questions and answers             |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: When did the Veterans'  |
|                                   | Supplement for aged care          |
|                                   | providers come into effect?       |
|                                   |                                   |
|                                   | Answer: From 1 August 2013.       |
|                                   |                                   |
|                                   | Once a veteran or their legal     |
|                                   | representative has provided       |
|                                   | consent for information about     |
|                                   | their eligibility to be given to  |
|                                   | their aged care provider, payment |
|                                   | to their provider is automatic    |
|                                   | and backdated to either the date: |
|                                   |                                   |
|                                   | -   the supplement came into      |
|                                   |     effect                        |
|                                   | -   of entry into care            |
|                                   | -   the care recipient became     |
|                                   |     eligible for the supplement   |
|                                   |                                   |
|                                   | whichever is the latest date.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: Who is eligible for the |
|                                   | Veterans' Supplement?             |
|                                   |                                   |
|                                   | Answer: Any veteran:              |
|                                   |                                   |
|                                   | -   receiving a Home Care Package |
|                                   |     or residential aged care      |
|                                   |     service, and                  |
|                                   | -   who has a mental health       |
|                                   |     condition which has been      |
|                                   |     accepted by the Department of |
|                                   |     Veterans' Affairs as related  |
|                                   |     to their service              |
|                                   |                                   |
|                                   | See also: Eligibility on the      |
|                                   | Department of Health and Aged     |
|                                   | Care – Veterans’ supplement for   |
|                                   | aged care website.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How is eligibility be   |
|                                   | determined?                       |
|                                   |                                   |
|                                   | Answer: Eligibility is determined |
|                                   | automatically. However an         |
|                                   | eligible veteran, or their legal  |
|                                   | representative, must provide      |
|                                   | consent to disclose their         |
|                                   | eligibility information to their  |
|                                   | aged care provider before the     |
|                                   | supplement can be paid. There is  |
|                                   | no requirement for ongoing        |
|                                   | assessment of eligibility.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: What is an accepted     |
|                                   | mental health condition? How is   |
|                                   | this determined?                  |
|                                   |                                   |
|                                   | Answer: Accepted mental health    |
|                                   | condition refers to an injury or  |
|                                   | disease recognised under          |
|                                   | legislation as related to a       |
|                                   | veteran's military service and    |
|                                   | may include:                      |
|                                   |                                   |
|                                   | -   post-traumatic stress         |
|                                   |     disorder                      |
|                                   | -   other anxiety disorders       |
|                                   | -   depression                    |
|                                   | -   substance misuse disorders    |
|                                   |                                   |
|                                   | This will be determined by the    |
|                                   | Department of Veterans' Affairs   |
|                                   | (DVA) who will advise Services    |
|                                   | Australia.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Is there an application |
|                                   | form?                             |
|                                   |                                   |
|                                   | Answer: No.                       |
|                                   |                                   |
|                                   | Eligibility is determined         |
|                                   | automatically along with          |
|                                   | payments, however, a veteran, or  |
|                                   | their legal representative, must  |
|                                   | provide consent to disclose their |
|                                   | eligibility information to a      |
|                                   | provider before the supplement    |
|                                   | can be paid.                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: How much is the         |
|                                   | supplement?                       |
|                                   |                                   |
|                                   | The supplement is indexed on an   |
|                                   | annual basis and updated in the   |
|                                   | relevant aged care payment        |
|                                   | systems. Current and historical   |
|                                   | rates of the supplement are       |
|                                   | published on the Department of    |
|                                   | Health and Aged Care – Schedule   |
|                                   | of Subsidies and Supplements for  |
|                                   | Aged Care website.                |
+-----------------------------------+-----------------------------------+

Teams and responsibilities for the Veterans' Supplement file

Table 2: This table lists the teams and their relevant responsibilities
for the Veterans' supplement consent list.

|                                   |     Programmes Branch –           |

|                                   |     Australia                     |
|                                   | -   Policy and guidelines         |
+-----------------------------------+-----------------------------------+
