Lodging dispute

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Staff research + Read more ...    |
|                                   |                                   |
|                                   | Before lodging a dispute, Service |
|                                   | Officers must research            |
|                                   | Operational Blueprint to make     |
|                                   | sure their dispute is supported   |
|                                   | by a business process or policy.  |
|                                   |                                   |
|                                   | This evidence will need to be     |
|                                   | submitted when lodging the        |
|                                   | dispute.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Informal resolution + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the original Quality Checker   |
|                                   | located in the same service       |
|                                   | centre?                           |
|                                   |                                   |
|                                   | Yes, if the dispute can be        |
|                                   | resolved informally through       |
|                                   | discussion/explanation of         |
|                                   | decision with the Quality         |
|                                   | Checker:                          |
|                                   |                                   |
|                                   | -   Procedure ends here           |
|                                   | -   Otherwise, go to Step 3       |
|                                   |                                   |
|                                   | No, go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Lodge dispute + Read more ...     |
|                                   |                                   |
|                                   | Complete the F2F QOL and QMA      |
|                                   | Disputes form and include         |
|                                   | supporting evidence. See          |
|                                   | Resources for a link to the form. |
|                                   |                                   |
|                                   | Do not include any customer       |
|                                   | identifying data, such as         |
|                                   | Customer Reference Number (CRN)   |
|                                   | or names in the form.             |
|                                   |                                   |
|                                   | Open the F2F QOL and QMA Disputes |
|                                   | form                              |
|                                   |                                   |
|                                   | Select Submit a dispute and       |
|                                   | complete all fields               |
|                                   |                                   |
|                                   | Read and acknowledge the privacy  |
|                                   | statement                         |
|                                   |                                   |
|                                   | Fill in all fields:               |
|                                   |                                   |
|                                   | -   Mandatory fields are marked   |
|                                   |     with an asterisk (*)          |
|                                   | -   Enter QOL AMR or QMA Work     |
|                                   |     Item ID and the date activity |
|                                   |     was submitted exactly as they |
|                                   |     appear in QOLCheck tool or    |
|                                   |     Process Direct QMA Feedback   |
|                                   |     tile:                         |
|                                   |     QOL - open QOLCheck tool,     |
|                                   |     select Replay Work and        |
|                                   |     complete search criteria to   |
|                                   |     locate activity details (See  |
|                                   |     Resources for more details on |
|                                   |     the QOL Check and QOL Stat    |
|                                   |     Guide)                        |
|                                   |     QMA - select the QMA Feedback |
|                                   |     tile in Process Direct to     |
|                                   |     view results and comments     |
|                                   |     from Quality Management       |
|                                   |     Officers                      |
|                                   |                                   |
|                                   | Select Submit                     |
|                                   |                                   |
|                                   | Note: disputes will be returned   |
|                                   | to the staff member to resubmit   |
|                                   | or withdraw where:                |
|                                   |                                   |
|                                   | -   insufficient evidence         |
|                                   |     supports the dispute, or      |
|                                   | -   activity cannot be located    |
+-----------------------------------+-----------------------------------+
| 4                                 | Dispute Progress and outcome +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Service Officers will be able to  |
|                                   | view the progress of their        |
|                                   | dispute on the F2F QOL and QMA    |
|                                   | Disputes page. See Resources for  |
|                                   | a link to the page.               |
|                                   |                                   |
|                                   | A notification email is sent to   |
|                                   | all parties when the dispute is   |
|                                   | finalised.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Withdrawing QOL/QMA disputes +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Service Officers can withdraw a   |
|                                   | QOL/QMA dispute after it has been |
|                                   | lodged.                           |
|                                   |                                   |
|                                   | Is the status of the dispute New? |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   Read and acknowledge the      |
|                                   |     privacy statement             |
|                                   | -   Scroll to the bottom of the   |
|                                   |     form and select Withdraw      |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   Contact the independent       |
|                                   |     reviewer to withdraw the      |
|                                   |     dispute                       |
|                                   | -   If the independent reviewer   |
|                                   |     is not available email        |
|                                   |     SSW.Quality                   |

Independent review

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Independent Review - review and   |
|                                   | decision + Read more ...          |
|                                   |                                   |
|                                   | The Independent Reviewer locates  |
|                                   | the dispute form in F2F QOL and   |
|                                   | QMA Disputes inbox and makes an   |
|                                   | assessment.                       |
|                                   |                                   |
|                                   | -   Select the Edit icon to open  |
|                                   |     the F2F QOL and QMA dispute   |
|                                   |     form                          |
|                                   | -   Read and acknowledge the      |
|                                   |     Privacy statement             |
|                                   | -   Review the information        |
|                                   |     entered by the Service        |
|                                   |     Officer                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Has sufficient information been   |
|                                   | provided to complete the          |
|                                   | dispute? + Read more ...          |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, return the dispute to the     |
|                                   | Service Officer requesting more   |
|                                   | details. Select the appropriate   |
|                                   | option:                           |
|                                   |                                   |
|                                   | -   Insufficient evidence to      |
|                                   |     support the dispute, or       |
|                                   | -   The activity cannot be        |
|                                   |     located                       |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Undertake review + Read more ...  |
|                                   |                                   |
|                                   | Allocate the dispute by entering  |
|                                   | your logon in the Independent     |
|                                   | Reviewer field                    |
|                                   |                                   |
|                                   | Review the dispute and add        |
|                                   | comments supporting the decision  |
|                                   | in the Independent Reviewer       |
|                                   | comments field                    |
|                                   |                                   |
|                                   | If the Independent Reviewer       |
|                                   | recommendation is:                |
|                                   |                                   |
|                                   | -   Overturned, go to Step 4      |
|                                   | -   Upheld or Upheld for another  |
|                                   |     reason, go to Step 5          |
+-----------------------------------+-----------------------------------+
| 4                                 | Recommendation to overturn the    |
|                                   | original decision + Read more ... |
|                                   |                                   |
|                                   | Arrange access for the original   |
|                                   | Quality Checker to the F2F QOL    |
|                                   | and QMA disputes page             |
|                                   |                                   |
|                                   | Is the original Quality Checker   |
|                                   | located in F2F division?          |
|                                   |                                   |
|                                   | Yes, organise a suitable time     |
|                                   | (within 3 business days) to       |
|                                   | discuss your recommendation with  |
|                                   | the original checker via their    |
|                                   | Team Leader:                      |
|                                   |                                   |
|                                   | -   Provide the Team leader with  |
|                                   |     details so that the original  |
|                                   |     checker can locate and review |
|                                   |     the decision prior to the     |
|                                   |     contact                       |
|                                   | -   At the agreed time contact    |
|                                   |     the original Quality Checker  |
|                                   | -   Discuss the recommendation    |
|                                   | -   Give the original Quality     |
|                                   |     Checker the opportunity to    |
|                                   |     ask questions and provide an  |
|                                   |     alternative point of view     |
|                                   | -   Provide any additional        |
|                                   |     feedback. For example, about  |
|                                   |     the quality of the original   |
|                                   |     Quality Checkers feedback     |
|                                   | -   Explain the Right of Review   |
|                                   |     process                       |
|                                   | -   Go to Step 6                  |
|                                   |                                   |
|                                   | No, or if a suitable time cannot  |
|                                   | be arranged with the original     |
|                                   | Quality Checker:                  |
|                                   |                                   |
|                                   | -   Go to Step 6                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Decision is Upheld or Upheld for  |
|                                   | a different reason + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | In the Independent Reviewer       |
|                                   | determination field, select       |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   Upheld (agree with the        |
|                                   |     original check result)        |
|                                   | -   Upheld for a different reason |
|                                   |     (original checking decision   |
|                                   |     is correct but for a          |
|                                   |     different reason. For         |
|                                   |     example, different error      |
|                                   |     reason)                       |
|                                   |                                   |
|                                   | If further action is required by  |
|                                   | the Service Officer, complete the |
|                                   | Is further action required field  |
|                                   | detailing action to be taken.     |
|                                   |                                   |
|                                   | -   Once complete, select Submit  |
|                                   |                                   |
|                                   | An automatic email is sent to the |
|                                   | Service Officer, their Team       |
|                                   | Leader and the original Quality   |
|                                   | Checker explaining the outcome.   |
|                                   |                                   |
|                                   | Note: if the Independent Reviewer |
|                                   | identifies errors, they must      |
|                                   | update the customer record in     |
|                                   | line with their decision, as soon |
|                                   | as possible. Do not wait for the  |
|                                   | outcome of the right of reply     |
|                                   | process.                          |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Original decision to be           |
|                                   | overturned + Read more ...        |
|                                   |                                   |
|                                   | Locate the dispute in F2F QOL and |
|                                   | QMA Disputes page and select:     |
|                                   |                                   |
|                                   | -   Overturned (Original decision |
|                                   |     overturned. That is, agree    |
|                                   |     with dispute)                 |
|                                   |                                   |
|                                   | If further action is required by  |
|                                   | the Service Officer complete the  |
|                                   | Is further action required field  |
|                                   | detailing action to be taken.     |
|                                   |                                   |
|                                   | -   Once complete, select Submit  |
|                                   |                                   |
|                                   | An automatic email is sent to the |
|                                   | original Quality Checker with     |
|                                   | right of reply information        |
|                                   |                                   |
|                                   | Note: if the Independent Reviewer |
|                                   | identifies errors, they must      |
|                                   | update the customer record in     |
|                                   | line with the decision, as soon   |
|                                   | as possible. Do not wait for the  |
|                                   | outcome of the right of reply     |
|                                   | process.                          |
+-----------------------------------+-----------------------------------+
| 7                                 | Providing feedback to the Quality |
|                                   | Checker + Read more ...           |
|                                   |                                   |
|                                   | Does feedback (not covered by the |
|                                   | QOL/QMA process) need to be       |
|                                   | provided to the Quality Checker?  |
|                                   | For example, feedback about the   |
|                                   | quality of the QOL/QMA feedback.  |
|                                   |                                   |
|                                   | -   Yes, go to Step 8             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 8                                 | Feedback to Quality Checker via   |
|                                   | the Staff Feedback Tool + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Provide feedback via the Staff    |
|                                   | Feedback Tool, see Resources for  |
|                                   | a link to the tool:               |
|                                   |                                   |
|                                   | Under Consistency of Decision     |
|                                   | Making Review select Quality      |
|                                   | Management then the relevant tool |
|                                   | and feedback type:                |
|                                   |                                   |
|                                   | -   QMA                           |
|                                   | -   QOL                           |
|                                   |                                   |
|                                   | Update the F2F QOL and QMA        |
|                                   | Dispute form - National Feedback  |
|                                   | provided to Quality Checker and   |
|                                   | select the appropriate reason/s:  |
|                                   |                                   |
|                                   | -   Missing or incorrect          |
|                                   |     resources supporting the      |
|                                   |     decision such as Operational  |
|                                   |     Blueprint not provided        |
|                                   | -   Explanation of action to be   |
|                                   |     taken to correct the error    |
|                                   |     not included                  |
|                                   | -   Description of the error and  |
|                                   |     why it was incorrect not      |
|                                   |     provided                      |
|                                   | -   Feedback not clear            |
|                                   | -   Feedback not constructive     |

Right of reply

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Right of reply + Read more ...    |
|                                   |                                   |
|                                   | If the Independent Reviewer's     |
|                                   | recommendation is to change the   |
|                                   | original decision, the original   |
|                                   | Quality Checker will be given a   |
|                                   | right of reply.                   |
|                                   |                                   |
|                                   | -   Original Checker, go to Step  |
|                                   |     2                             |
|                                   | -   Independent Reviewer, go to   |
|                                   |     Step 3                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Original Quality Checker - review |
|                                   | of overturned recommendation +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Response to be completed within 3 |
|                                   | business days.                    |
|                                   |                                   |
|                                   | Note: if unable to respond within |
|                                   | 3 days and disagree with the      |
|                                   | dispute recommendation send an    |
|                                   | email to SSW.QUALITY.             |
|                                   |                                   |
|                                   | The original Quality checker      |
|                                   | reviews the overturned            |
|                                   | recommendation - right of reply   |
|                                   | email.                            |
|                                   |                                   |
|                                   | If they:                          |
|                                   |                                   |
|                                   | Agree with/accept the overturned  |
|                                   | recommendation:                   |
|                                   |                                   |
|                                   | -   select Agree with/accept with |
|                                   |     decision and enter supporting |
|                                   |     comments within the F2F QOL   |
|                                   |     and QMA Disputes form         |
|                                   | -   once completed, select Submit |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | Disagree with/don’t accept the    |
|                                   | overturned recommendation:        |
|                                   |                                   |
|                                   | -   select Disagree with          |
|                                   |     decision, enter supporting    |
|                                   |     comments within the F2F QOL   |
|                                   |     and QMA Disputes form         |
|                                   | -   once complete, select Submit  |
|                                   |                                   |
|                                   | The F2F QOL and QMA Disputes form |
|                                   | is automatically returned to the  |
|                                   | Independent Reviewer, who will    |
|                                   | review the details provided.      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Independent Reviewer – Right of   |
|                                   | reply + Read more ...             |
|                                   |                                   |
|                                   | Was the Right of Reply response   |
|                                   | received from original Quality    |
|                                   | Checker within 3 business days?   |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   finalise dispute decision     |
|                                   |     Overturned                    |
|                                   | -   procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Independent Reviewer – Right of   |
|                                   | reply received within 3 business  |
|                                   | days + Read more ...              |
|                                   |                                   |
|                                   | Review the additional details     |
|                                   | provided:                         |
|                                   |                                   |
|                                   | Agree with original Quality       |
|                                   | Checker                           |
|                                   |                                   |
|                                   | -   finalise dispute decision     |
|                                   |     Upheld                        |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | Disagree with original Quality    |
|                                   | Checker                           |
|                                   |                                   |
|                                   | -   If a mutual decision can be   |
|                                   |     reached, finalise dispute in  |
|                                   |     line with the mutual          |
|                                   |     decision. Procedure ends here |
|                                   | -   If mutual decision cannot be  |
|                                   |     reached, finalise the dispute |
|                                   |     referred to level 2. Go to    |
|                                   |     Step 5                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Independent Reviewer – Right of   |
|                                   | reply process referral to Level   |
|                                   | 2 + Read more ...                 |
|                                   |                                   |
|                                   | Within 2 business days:           |
|                                   |                                   |
|                                   | -   create the relevant Level 2   |
|                                   |     Helpdesk enquiry              |
|                                   | -   copy and paste responses from |
|                                   |     all parties involved into the |
|                                   |     Enquiry Description section   |
|                                   |     of the Level 2 Helpdesk       |
|                                   |     enquiry                       |
|                                   | -   in the Additional Contact     |
|                                   |     Name add the Original Checker |
|                                   | -   in the Positional Mailbox     |
|                                   |     Address add SSW.QUALITY       |
|                                   | -   save enquiry to submit to     |
|                                   |     Level 2 Helpdesk              |
+-----------------------------------+-----------------------------------+
| 6                                 | Independent Reviewer – Level 2    |
|                                   | response received + Read more ... |
|                                   |                                   |
|                                   | Response from Level 2 Helpdesk    |
|                                   | received:                         |
|                                   |                                   |
|                                   | -   Independent Reviewer to       |
|                                   |     review the response           |
|                                   | -   Add the Level 2 Helpdesk      |
|                                   |     comments to the QOL/QMA       |
|                                   |     Dispute form                  |
|                                   |                                   |
|                                   | Is recommendation to Overturn the |
|                                   | decision supported?               |
|                                   |                                   |
|                                   | -   Yes, finalise the dispute -   |
|                                   |     decision Overturned           |
|                                   | -   No, finalise the dispute -    |
|                                   |     decision Upheld               |
|                                   |                                   |
|                                   | Email is automatically sent to    |
|                                   | Service Officer, their Team       |
|                                   | Leader and the Quality Checker    |
|                                   | advising of the outcome.          |
