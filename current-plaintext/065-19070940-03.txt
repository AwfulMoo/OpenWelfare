Refresh means assessment data from Customer First to Aged Care Payment System

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Request a refresh of means        |
|                                   | assessment data + Read more ...   |
|                                   |                                   |
|                                   | In ACMPS, select Home to clear    |
|                                   | the care recipient record.        |
|                                   |                                   |
|                                   | In Customer First, locate the     |
|                                   | care recipient record. To request |
|                                   | a refresh of means assessment     |
|                                   | data, go to the Benefit Action    |
|                                   | (BA) screen in the RCA system and |
|                                   | complete the below fields:        |
|                                   |                                   |
|                                   | -   Benefit type: RCA             |
|                                   | -   Action: RSH                   |
|                                   | -   Effect Date: leave blank      |
|                                   | -   Source: INT                   |
|                                   | -   DOR: date of activity         |
|                                   | -   press [Enter]                 |
|                                   |                                   |
|                                   | Ensure the Effect Date populates  |
|                                   | with the relevant date of entry   |
|                                   | into care.                        |
|                                   |                                   |
|                                   | If it has a later date:           |
|                                   |                                   |
|                                   | -   correct the date of entry     |
|                                   | -   press [Enter]                 |
|                                   | -   confirm it does not change    |
|                                   |     back                          |
|                                   |                                   |
|                                   | If it auto updates to a later     |
|                                   | date, there may be an issue with  |
|                                   | a limiting date preventing        |
|                                   | transmission of data.             |
|                                   |                                   |
|                                   | If the reason for incorrect data  |
|                                   | transmission cannot be            |
|                                   | identified, contact Aged Care     |
|                                   | (RCA) Technical Peer Support      |
|                                   | (TPS). See Service Officer role   |
|                                   | when seeking Technical Peer       |
|                                   | Support (TPS) in Smart Centres.   |
|                                   |                                   |
|                                   | To complete the refresh activity: |
|                                   |                                   |
|                                   | -   key ‘AR’ in Next: field and   |
|                                   |     press [Enter]                 |
|                                   | -   select Save, then select Exit |
|                                   | -   on AL screen select the       |
|                                   |     started RCA/RSH activity      |
|                                   | -   key ‘XEV’ to go to Ev List    |
|                                   |     screen                        |
|                                   | -   select [Enter]                |
+-----------------------------------+-----------------------------------+
| 2                                 | Confirming means assessment data  |
|                                   | being transmitted + Read more ... |
|                                   |                                   |
|                                   | Check the RCA event on the EV     |
|                                   | List (XEV) screen. XEV screen     |
|                                   | displays income and asset data    |
|                                   | transmitted in the refresh        |
|                                   | activity.                         |
