
Identify and resolve duplicate Individual Healthcare Identifier (IHI) in Healthcare Identifiers (HI) Service 011-03010110

-   Background
-   Process Summary
-   Process

This page contains the procedures on how to flag duplicate IHI, remove
duplicate flags on IHIs and resolve duplicate IHIs in the Consumer
Directory Maintenance System (CDMS).

Flag duplicate IHIs in CDMS

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to the Consumer        |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   On the CDMS home page, select |
|                                   |     Amend Consumer Details        |
|                                   | -   The Amendment Search screen   |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for the individuals + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select Healthcare Identifiers |
|                                   |     from the Program drop down    |
|                                   |     list                          |
|                                   | -   Search for the individuals    |
|                                   |     that are to be flagged as     |
|                                   |     duplicate IHIs, see Search    |
|                                   |     for Individual Healthcare     |
|                                   |     Identifier (IHI) in           |
|                                   |     Healthcare Identifiers (HI)   |
|                                   |     Service                       |
|                                   | -   The Matched IHI screen        |
|                                   |     displays                      |
|                                   |                                   |
|                                   | Note: to be able to identify      |
|                                   | duplicate IHIs, all IHIs to be    |
|                                   | flagged must display on the       |
|                                   | Matched consumer screen.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Check if all the individuals are  |
|                                   | returned on the Matched IHI       |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Do the 2 or more individuals to   |
|                                   | be flagged return on the Matched  |
|                                   | IHI screen?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, go to Step 4              |
|                                   |     Note: it may be necessary to  |
|                                   |     change the demographic        |
|                                   |     details on 1 or more of the   |
|                                   |     IHIs, to ensure the           |
|                                   |     individuals return on the     |
|                                   |     same Matched IHI screen       |
+-----------------------------------+-----------------------------------+
| 4                                 | Select records that are potential |
|                                   | duplicates + Read more ...        |
|                                   |                                   |
|                                   | Select the check boxes in the     |
|                                   | Duplicate Flag column to select   |
|                                   | IHIs that are potential           |
|                                   | duplicates.                       |
|                                   |                                   |
|                                   | Note: more than 1 IHI must be     |
|                                   | selected in the Duplicate Flag    |
|                                   | column.                           |
+-----------------------------------+-----------------------------------+
| 5                                 | Check for validation errors +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select the Potential Duplicate    |
|                                   | button.                           |
|                                   |                                   |
|                                   | Are there any validation errors?  |
|                                   |                                   |
|                                   | -   Yes, correct errors as        |
|                                   |     prompted                      |
|                                   | -   No, the Report Potential      |
|                                   |     Duplicate IHI screen displays |
|                                   |     the duplicate IHIs            |
+-----------------------------------+-----------------------------------+
| 6                                 | Enter comments + Read more ...    |
|                                   |                                   |
|                                   | -   Enter comments into the free  |
|                                   |     text box                      |
|                                   | -   Example: ‘Merging of          |
|                                   |     duplicate IHI’s, Primary IHI  |
|                                   |     is (insert Primary IHI)'      |
|                                   | -   Select Add Comment            |
+-----------------------------------+-----------------------------------+
| 7                                 | Validate and check details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select Validate               |
|                                   | -   The Confirm Report Potential  |
|                                   |     Duplicate IHI screen displays |
|                                   |                                   |
|                                   | Are the details correct?          |
|                                   |                                   |
|                                   | -   Yes, select Update to flag    |
|                                   |     the IHIs as duplicates        |
|                                   | -   No, select Change to return   |
|                                   |     to the Report Potential       |
|                                   |     Duplicate IHI Records screen  |
|                                   |     to makes changes              |
|                                   |                                   |
|                                   | A confirmation message displays.  |
+-----------------------------------+-----------------------------------+

Remove duplicate flag on IHI in CDMS

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to the Consumer        |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   On the CDMS home page, select |
|                                   |     Amend Consumer Details        |
|                                   | -   The Amendment Search screen   |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for the IHI + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Healthcare Identifiers |
|                                   |     from the Program drop down    |
|                                   |     list                          |
|                                   | -   Conduct a basic, detailed or  |
|                                   |     address search to find the    |
|                                   |     IHI for the individual        |
+-----------------------------------+-----------------------------------+
| 3                                 | Check if there is a match + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is there a match?                 |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, refine search             |
|                                   |     Note: to refine search, add   |
|                                   |     or remove search criteria to  |
|                                   |     obtain a match for the        |
|                                   |     correct individual            |
+-----------------------------------+-----------------------------------+
| 4                                 | Select 1 of the individuals +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Select the radio button       |
|                                   |     beside one of the individuals |
|                                   |     whose IHI has been flagged as |
|                                   |     a potential duplicate         |
|                                   | -   Select Personal               |
|                                   | -   The Amend IHI Details screen  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Select Report/Resolve and Resolve |
|                                   | Duplicate + Read more ...         |
|                                   |                                   |
|                                   | -   Select Report/Resolve         |
|                                   | -   Select Resolve Duplicate      |
|                                   | -   The Resolve Duplicate IHI     |
|                                   |     screen displays               |
+-----------------------------------+-----------------------------------+
| 6                                 | Remove the duplicate flag + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select the radio button in    |
|                                   |     the Select/Primary Record     |
|                                   |     column of 1 of the IHI that   |
|                                   |     is not the duplicate          |
|                                   | -   Select Remove Duplicate Flag  |
|                                   | -   A warning message displays    |
|                                   | -   Select OK to action the       |
|                                   |     message                       |
|                                   | -   The Resolution Status changes |
|                                   |     to Resolved not duplicate     |
+-----------------------------------+-----------------------------------+
| 7                                 | Validate and check details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select Validate               |
|                                   | -   The Confirm Resolved IHI      |
|                                   |     Details screen is displays    |
|                                   |                                   |
|                                   | Are the details correct?          |
|                                   |                                   |
|                                   | -   Yes, select Update to remove  |
|                                   |     the duplicate flag. A         |
|                                   |     confirmation message displays |
|                                   | -   No, select Change to return   |
|                                   |     to the Resolve Duplicate IHI  |
|                                   |     screen to make changes        |
+-----------------------------------+-----------------------------------+

Resolve duplicate IHI in CDMS

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log on to CDMS + Read more ...    |
|                                   |                                   |
|                                   | -   Log on to the Consumer        |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   On the CDMS home page, select |
|                                   |     Amend Consumer Details        |
|                                   | -   The Amendment Search screen   |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for the individual's IHI + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Select Healthcare Identifiers |
|                                   |     from the Program drop down    |
|                                   |     list                          |
|                                   | -   Conduct a basic, detailed or  |
|                                   |     address search to find the    |
|                                   |     IHI for the individual        |
+-----------------------------------+-----------------------------------+
| 3                                 | Check if there is a match + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is there a match?                 |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, refine search             |
|                                   |     Note: to refine search, add   |
|                                   |     or remove search criteria to  |
|                                   |     obtain a match for the        |
|                                   |     correct individual            |
+-----------------------------------+-----------------------------------+
| 4                                 | Select 1 of the individuals +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select the radio button beside    |
|                                   | the individual whose IHI has been |
|                                   | flagged as a potential duplicate  |
|                                   | and will become the secondary IHI |
|                                   |                                   |
|                                   | -   Select Personal               |
|                                   | -   The Amend IHI Details screen  |
|                                   |     displays                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Select Report/Resolve and Resolve |
|                                   | Duplicate + Read more ...         |
|                                   |                                   |
|                                   | -   Select Report/Resolve         |
|                                   | -   Select Resolve Duplicate      |
|                                   | -   The Resolve Duplicate IHI     |
|                                   |     screen displays               |
+-----------------------------------+-----------------------------------+
| 6                                 | Select the Primary Record + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select the radio button in    |
|                                   |     the Select/Primary Record     |
|                                   |     column of the IHI that is to  |
|                                   |     become the primary IHI        |
|                                   | -   Note: this automatically      |
|                                   |     highlights the link box for   |
|                                   |     the IHI that is to become the |
|                                   |     secondary IHI (to be linked   |
|                                   |     to the primary IHI)           |
|                                   | -   Select the link box           |
+-----------------------------------+-----------------------------------+
| 7                                 | Add comments + Read more ...      |
|                                   |                                   |
|                                   | -   Enter comments into the Add   |
|                                   |     Comments box                  |
|                                   | -   Example: ‘Resolved duplicate  |
|                                   |     IHI’s, Primary IHI is (insert |
|                                   |     Primary IHI)’                 |
|                                   | -   Select Add Comment            |
+-----------------------------------+-----------------------------------+
| 8                                 | Action the message + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Select Link                   |
|                                   | -   A warning message displays    |
|                                   | -   Select Yes to action the      |
|                                   |     message                       |
|                                   | -   The Resolution Status changes |
|                                   |     to Resolved as duplicate      |
|                                   | -   The IHI resolved is greyed    |
|                                   |     out and unable to be selected |
+-----------------------------------+-----------------------------------+
| 9                                 | Validate and check details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select Validate               |
|                                   | -   The Confirm Resolved IHI      |
|                                   |     Details screen displays       |
|                                   |                                   |
|                                   | Are the details correct?          |
|                                   |                                   |
|                                   | -   Yes, select Update to flag    |
|                                   |     the IHIs as duplicates. A     |
|                                   |     confirmation message displays |
|                                   | -   No, select Change to return   |
|                                   |     to the Resolve Duplicate IHI  |
|                                   |     screen to make changes        |
+-----------------------------------+-----------------------------------+
| 10                                | Enter comments on both IHIs +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | When IHIs have been resolved, put |
|                                   | a comment on both IHIs in the     |
|                                   | Comments field as part of the     |
|                                   | Amend IHI process.                |
|                                   |                                   |
|                                   | ‘Primary IHI: PRIMARY IHI         |
|                                   | XXXXXXXXXXXXXXXX (THIS CONSUMER)  |
|                                   | RESOLVED FROM IHI                 |
|                                   | XXXXXXXXXXXXXXXX                  |
|                                   |                                   |
|                                   | Secondary IHI: SECONDARY IHI      |
|                                   | XXXXXXXXXXXXXXXX (THIS CONSUMER)  |
|                                   | RESOLVED TO IHI XXXXXXXXXXXXXXXX’ |
+-----------------------------------+-----------------------------------+
