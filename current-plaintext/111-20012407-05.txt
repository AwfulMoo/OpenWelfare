Worklist search criteria

This table outlines the worklist search criteria that appears as the
default setting for the following Saved Searches:

My Work search is available to all staff

My Team Work is available to Team Leaders and Analysts

All Work is available to Analysts

+-----------------------+-----------------------+-----------------------+
| Item                  | Description           | Default display       |
+-----------------------+-----------------------+-----------------------+
| Work Item ID          | This field can be     | -   My Work           |
|                       | used to search for a  | -   My Team Work      |
|                       | specific work item    | -   All Work          |
|                       | ID.                   |                       |
|                       |                       |                       |
|                       | Multiple work items   |                       |
|                       | can be entered to     |                       |
|                       | refine the search     |                       |
|                       | results.              |                       |
+-----------------------+-----------------------+-----------------------+
| Work Type             | This field can be     | -   My Work           |
|                       | used to search for a  | -   My Team Work      |
|                       | collection of work    | -   All Work          |
|                       | items of the same     |                       |
|                       | type.                 |                       |
|                       |                       |                       |
|                       | Search can search     |                       |
|                       | using a work type     |                       |
|                       | description, for      |                       |
|                       | example, CCS claims,  |                       |
|                       | or the work type ID   |                       |
|                       | if known.             |                       |
+-----------------------+-----------------------+-----------------------+
| Work Item Attributes  | This field can be     | -   All Work          |
|                       | used to search for    |                       |
|                       | work items with       |                       |
|                       | specific attributes.  |                       |
|                       |                       |                       |
|                       | Users can refine the  |                       |
|                       | results by entering   |                       |
|                       | ‘||’ into the field.  |                       |
|                       | For example,          |                       |
|                       | ‘PROCESS_TYPE || SHC’ |                       |
+-----------------------+-----------------------+-----------------------+
| Source Status         | Users can restrict    | -   All Work          |
|                       | the search results to |                       |
|                       | display work items by |                       |
|                       | a selected status in  |                       |
|                       | the source system.    |                       |
+-----------------------+-----------------------+-----------------------+
| Work Item Status      | Users can restrict    | -   My Work           |
|                       | search results to     | -   My Team Work      |
|                       | work items with a     | -   All Work          |
|                       | specific status in    |                       |
|                       | WLM. For example,     |                       |
|                       | OPEN, or Hold         |                       |
|                       | Retained (HTU)        |                       |
+-----------------------+-----------------------+-----------------------+
| Source System         | Used to filter work   | -   All Work          |
|                       | items that have       |                       |
|                       | originated in a       |                       |
|                       | specific source       |                       |
|                       | system, using a drop  |                       |
|                       | down menu.            |                       |
+-----------------------+-----------------------+-----------------------+
| Allocation Indicator  | Users can filter the  | -   All Work          |
|                       | results based on work |                       |
|                       | items that are        |                       |
|                       | allocated,            |                       |
|                       | unallocated, or both. |                       |
+-----------------------+-----------------------+-----------------------+
| Employee Responsible  | Staff logons can be   | -   My Team Work      |

|                       | to them.              |                       |
|                       |                       |                       |
|                       | Multiple logons can   |                       |
|                       | be entered to search  |                       |
|                       | for work assigned to  |                       |
|                       | groups of staff at    |                       |
|                       | one time.             |                       |
+-----------------------+-----------------------+-----------------------+
| Priority              | These fields are used | -   All Work          |

|                       | items within a        |                       |
