Requirements for taking or sending PBS medicine overseas

This table describes what members of the public need to do before taking
or sending Pharmaceutical Benefits Scheme (PBS) medicine overseas.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Carry letter from their doctor +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Carry a letter from their doctor  |
|                                   | explaining:                       |
|                                   |                                   |
|                                   | -   what the medicine is          |
|                                   | -   how much they are taking or   |
|                                   |     sending, and                  |
|                                   | -   that the medicine is for      |
|                                   |     personal use                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check medicine legal in           |
|                                   | destination country + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Patients are responsible for      |
|                                   | checking with the embassy, high   |
|                                   | commission or consulate, to make  |
|                                   | sure that the medication being    |
|                                   | taken is legal in the destination |
|                                   | country.                          |
|                                   |                                   |
|                                   | Note: always leave the medicine   |
|                                   | in its original packaging.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete Medicine export          |
|                                   | declaration form + Read more ...  |
|                                   |                                   |
|                                   | Include a completed Medicine      |
|                                   | export declaration form with the  |
|                                   | medicines. Patients will need to  |
|                                   | carry this form through customs.  |
|                                   | The Resources page has a link to  |
|                                   | the form.                         |
+-----------------------------------+-----------------------------------+
