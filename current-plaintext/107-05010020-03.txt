| 6                                 | Advance payments and debts        |
|                                   |                                   |
|                                   | Advance payments are repaid from  |
|                                   | a customer's ongoing payments.    |
|                                   | They do not become a debt unless: |
|                                   |                                   |
|                                   | -   the customer's entitlement    |
|                                   |     ceases before the advance has |
|                                   |     been fully repaid, or         |
|                                   | -   the level of advance          |
|                                   |     repayment is insufficient to  |
|                                   |     repay the advance in a        |
|                                   |     reasonable time               |
|                                   |                                   |
|                                   | It will become a debt where:      |
|                                   |                                   |
|                                   | -   the customer's FTB reduces    |
|                                   |     substantially                 |
|                                   | -   the customer's rate does not  |
|                                   |     change for 2 consecutive      |
|                                   |     fortnights, and               |
|                                   | -   the repayment of the advance  |
|                                   |     will take an unreasonable     |
|                                   |     amount of time. Equal to or   |
|                                   |     more than 2 years, or 731     |
|                                   |     days                          |
|                                   |                                   |
|                                   | To qualify for an advance         |
|                                   | payment, a customer must not owe  |
|                                   | money to the Commonwealth. This   |
|                                   | includes:                         |
|                                   |                                   |
|                                   | -   money that is being recovered |
|                                   |     or may be recovered by        |
|                                   |     withholdings from their       |
|                                   |     Centrelink payments, and      |
|                                   | -   some debts that have been     |
|                                   |     temporarily written off       |
|                                   |                                   |
|                                   | The Write-off (OPWO) screen will  |
|                                   | have the reason for the write.    |
|                                   |                                   |
|                                   | Temporary written off debts       |
|                                   |                                   |
|                                   | For Social Security advances, if  |
|                                   | the customer has a debt           |
|                                   | temporarily written off they are  |
|                                   | not entitled to an advance        |
|                                   | payment. This includes those with |
|                                   | non-lodger debts temporarily      |
|                                   | written off under the separated   |
|                                   | couples rule                      |
|                                   |                                   |
|                                   | For FTB advances, if the customer |
|                                   | has a debt temporarily written    |
|                                   | off they will not be entitled to  |
|                                   | an advance. The exception to this |
|                                   | is where the debt is temporarily  |
|                                   | written off due to:               |
|                                   |                                   |
|                                   | -   an ex-partner non-lodger, or  |
|                                   | -   bankruptcy                    |
|                                   |                                   |
|                                   | Permanently written-off debt      |
|                                   |                                   |
|                                   | A debt permanently written off is |
|                                   | no longer recoverable through     |
|                                   | deductions. The customer is       |
|                                   | entitled to an advance if they    |
|                                   | meet other qualifications.        |
|                                   |                                   |
|                                   | The Debt Programme is unable to   |
|                                   | provide any advice about          |
|                                   | qualification for an advance.     |
|                                   |                                   |
|                                   | Customers can make a repayment to |
|                                   | finalise the debt to be eligible  |
|                                   | for an advance.                   |
|                                   |                                   |
|                                   | Issuing an advance payment when   |
|                                   | the customer owes a balance of    |
|                                   | less than 5 cents                 |
|                                   |                                   |
|                                   | This may cause automatic          |
|                                   | rejection of any advance          |
|                                   | application.                      |
|                                   |                                   |
|                                   | Override the rejection in the     |
|                                   | workflow if the balance is less   |
|                                   | than 5 cents:                     |
|                                   |                                   |
|                                   | The advance can be paid if other  |
|                                   | conditions are met. The override  |
|                                   | option does not apply to advances |
|                                   | of:                               |
|                                   |                                   |
|                                   | -   FTB                           |
|                                   | -   Special Employment Advance    |
|                                   |     (SEA), or                     |
|                                   | -   Pharmaceutical Advance (PhA)  |
|                                   |                                   |
|                                   | The debt will be finalised when   |
|                                   | the outstanding rounded amount is |
|                                   | withheld on the customer's next   |
|                                   | payday                            |
|                                   |                                   |
|                                   | Compensation payer debts          |
|                                   |                                   |
|                                   | The insurer pays compensation     |
|                                   | payer debts. They do not          |
|                                   | disqualify a customer from an     |
|                                   | advance payment. The Advance      |
|                                   | Payment cluster cannot            |
|                                   | distinguish insurer debts from    |
|                                   | customer debts. It will reject    |
|                                   | advance payment applications if   |
|                                   | insurer debts are outstanding.    |
|                                   |                                   |
|                                   | The Advance Payments workflow has |
|                                   | an option to override rejections  |
|                                   | caused by insurer debts.          |
|                                   |                                   |
|                                   | Note: there is no override option |
|                                   | for FTB, SEA or PhA Advances.     |
|                                   | These cannot be granted if there  |
|                                   | is an outstanding debt.           |
|                                   |                                   |
|                                   | Repayment for an advance          |
|                                   |                                   |
|                                   | For information about accepting a |
|                                   | cash repayment for an advance,    |
|                                   | see Lump sum repayment of advance |
|                                   | payment.                          |
|                                   |                                   |
|                                   | A pension customer can receive    |
|                                   | multiple advance payments. This   |
|                                   | may result in several debts if    |
|                                   | the customer ceases to be         |
|                                   | entitled to any payments.         |
|                                   |                                   |
|                                   | For more information, see:        |
|                                   |                                   |
|                                   | -   Advances, or                  |
|                                   | -   Hardship Assistance Service   |
|                                   |     Level 2 Helpdesk. See the     |
|                                   |     Resources page.               |

| 24                                | Debt repayment hierarchy          |
|                                   |                                   |
|                                   | The debt repayment hierarchy      |
|                                   | determines the priority for the   |
|                                   | allocation of debt repayments.    |
|                                   |                                   |
|                                   | The repayment hierarchy is        |
|                                   | bypassed for payments made        |
|                                   | directly to:                      |
|                                   |                                   |
|                                   | -   a specific debt through       |
|                                   |     internal manual repayment     |
|                                   |     processing                    |
|                                   | -   compensation                  |
|                                   | -   Paid Parental Leave           |
|                                   | -   collection agent repayment    |
|                                   |     processing                    |
|                                   |                                   |
|                                   | Payments for a specific debt are  |
|                                   | applied based on the Debt ID used |
|                                   | when the customer makes the       |
|                                   | payment.                          |
|                                   |                                   |
|                                   | The order repayments are applied  |
|                                   | is stored in and automatically    |
|                                   | controlled by Debt Management and |
|                                   | Information System (DMIS). This   |
|                                   | determines the order in which     |
|                                   | debts are repaid when a customer  |
|                                   | has more than 1 outstanding debt. |
|                                   |                                   |
|                                   | Repayments are currently applied  |
|                                   | in the following category order:  |
|                                   |                                   |
|                                   | Family assistance (including      |
|                                   | Child Care Subsidy)               |
|                                   | reconciliation debts where the    |
|                                   | due date is in the past are       |
|                                   | collected first.                  |
|                                   |                                   |
|                                   | They are followed by all other    |
|                                   | debts, excluding:                 |
|                                   |                                   |
|                                   | Public Governance, Performance    |
|                                   | and Accountability (PGPA) Act     |
|                                   | 2013 debts                        |
|                                   |                                   |
|                                   | Debts temporarily written off     |
|                                   | (voluntary and automatic          |
|                                   | withholdings payments against     |
|                                   | these debts may be accepted and   |
|                                   | some write-off reasons will       |
|                                   | automatically end)                |
|                                   |                                   |
|                                   | Family assistance debts held for  |
|                                   | the reconciliation process        |
|                                   |                                   |
|                                   | Debts raised by other government  |
|                                   | departments                       |
|                                   |                                   |
|                                   | Debts raised internationally (for |
|                                   | example, New Zealand Government   |
|                                   | debts)                            |
|                                   |                                   |
|                                   | Family assistance reconciliation  |
|                                   | debts where the due date is in    |
|                                   | the future                        |
|                                   |                                   |
|                                   | Debts that have been previously   |
|                                   | finalised (creating an            |
|                                   | over-recovered debt), in the      |
|                                   | following priority order:         |
|                                   |                                   |
|                                   | -   Fully recovered               |
|                                   | -   Waived                        |
|                                   | -   Finalised as not a debt       |
|                                   | -   Transferred to Department of  |
|                                   |     Veterans' Affairs for         |
|                                   |     recovery                      |
|                                   | -   Permanently written-off       |
|                                   | -   Included in the family        |
|                                   |     assistance reconciliation     |
|                                   |     process                       |
|                                   |                                   |
|                                   | Potential debts requiring         |
|                                   | assessment                        |
|                                   |                                   |
|                                   | Compensation insurer debts        |
|                                   |                                   |
|                                   | Debts finalised as no-debt,       |
|                                   | waived or deleted on the same day |
|                                   | as the debt was raised, are       |
|                                   | excluded from the recovery        |
|                                   | hierarchy. Payments are not       |
|                                   | credited to those debts.          |

| 10                                | Waiver of debt                    |
|                                   |                                   |
|                                   | Waiver or part waiver of a debt   |
|                                   | means the right to recover the    |
|                                   | amount waived is relinquished. No |
|                                   | recovery action is possible on    |
|                                   | the waived amount unless there is |
|                                   | sufficient reason to believe the  |
|                                   | waiver was incorrect. Refer these |
|                                   | cases to the Level 2 Policy       |
|                                   | Helpdesk for further              |
|                                   | investigation and advice.         |
|                                   |                                   |
|                                   | A debt which has been:            |
|                                   |                                   |
|                                   | -   Fully waived, will display as |
|                                   |     'F-Waived' in the Status      |
|                                   |     field on the Debt List (OPDL) |
|                                   |     screen. No further recovery   |
|                                   |     action is possible            |
|                                   | -   Partially waived, will        |
|                                   |     display the partial waiver    |
|                                   |     amount on the Waiver (OPWA)   |
|                                   |     screen. Normal recovery       |
|                                   |     action continues              |
|                                   |                                   |
|                                   | See Waiving Centrelink debts.     |
