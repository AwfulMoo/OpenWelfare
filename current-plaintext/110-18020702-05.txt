Assessing the Educator Payment History Report (EPHR) from Family Day Care Provider

Table 4

+-----------------------------------+-----------------------------------+
| Item                              | Assessment                        |
+-----------------------------------+-----------------------------------+
| 1                                 | The EPHR divides the total child  |
|                                   | care fees billed by the Educator  |
|                                   | into two columns, 'total fees'    |
|                                   | and 'administration levy'.        |
|                                   |                                   |
|                                   | When using the EPHR as a source   |
|                                   | document for business income      |
|                                   | assessment, the Educator's gross  |
|                                   | income is the 'total fees' plus   |
|                                   | the 'administration levy'.        |
|                                   |                                   |
|                                   | Note: the administration levy is  |
|                                   | an allowable deduction for Child  |
|                                   | Care Business. The accumulative   |
|                                   | amount of the administration levy |
|                                   | can be deducted from the          |
|                                   | Educator's gross income for       |
|                                   | income tax purposes only.         |
