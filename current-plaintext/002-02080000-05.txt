ICT Service Desk referrals

If the Social Service Plan (SSP) result is incorrect, FHA Processing
staff can submit the Farm Household Allowance (FHA) Issue SSP Incident
webform:

-   Go to mySupport
-   In the Search field, enter the name of the webform or key words,
    select Search
-   Select SSP Incident in the Issues section
-   complete and submit the form noting any receipt or reference code
    provided
