This document outlines the processing instructions for staff on managing
Provider Digital Access (PRODA) accounts when users are unable to do
this themselves online.

Self service

Service Officers can help users self-manage their PRODA account online.

Users cannot complete the following updates online, they will need to be
completed by a Service Officer:

-   unlinking an identifier for healthcare professionals and
    administrators

Types of updates

Service Officers can complete the following updates on a PRODA account:

-   reset password
-   remove synchronised device
-   email and phone number changes
-   change security preferences
-   link identifiers for healthcare providers and administrators
-   unlink identifiers for healthcare providers and administrators
-   change current name

The Resources page contains error messages that may display in the PRODA
system, and identifiers for Provider Directory System (PDS) or eHealth
Program (EHP) data sources.
