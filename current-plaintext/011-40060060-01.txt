This document outlines the process used to amend program review details
for a group (Medicare card) in the Consumer Directory via the Consumer
Directory Maintenance System (CDMS).

Amendment of a group's program review details

The amendment of a group's program review details will involve one or
more of the following functions:

-   amending the address threshold
-   adding new address comments
-   amending end dates for existing address comments
-   amending the hold status of the group
-   adding or amending the review status of the group
-   adding new program review comments for the group
-   amending end dates for existing program review comments for the
    group
