For Assistance for Isolated Children (AIC) Smart Centre staff only.

This page contains information on assessing qualification for the AIC
Scheme.

Eligibility assessment

Testing eligibility requirements for AIC

This table describes the steps to correctly assess qualification for the
AIC Scheme.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, the student is not        |
|                                   |     qualified for AIC. Go to Step |

|                                   | requirements?                     |
