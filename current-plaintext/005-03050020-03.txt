Conducting assurer/s interview to determine if assuree is granted payment

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | At assurer/s interview (either by |
|                                   | phone or in person) + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Make sure that the assurer/s has  |
|                                   | completed the Assurance of        |
|                                   | Support (MOD AR) form. If not     |
|                                   | completed, assist the assurer's   |
|                                   | to complete the MOD AR            |
|                                   |                                   |
|                                   | -   Tell the assurer/s that any   |
|                                   |     AoS recoverable payment paid  |
|                                   |     to the assuree will become a  |
|                                   |     debt. This debt is legally    |
|                                   |     required to be paid in full   |
|                                   |     to the Australian Government. |
|                                   |     It should also be made clear  |
|                                   |     that approximately every 12   |
|                                   |     months Services Australia     |
|                                   |     will recover AoS debts from   |
|                                   |     the term deposit under the    |
|                                   |     bank guarantee they lodged.   |
|                                   |     If there is no AoS bank       |
|                                   |     guarantee and term deposit,   |
|                                   |     or this has been depleted,    |
|                                   |     the assurer/s are required to |
|                                   |     repay any outstanding debt    |
|                                   |     amount. For arrangements with |
|                                   |     joint-assurers, the debt will |
|                                   |     not be split between each     |
|                                   |     assurer                       |
|                                   | -   Tell the assurer/s that if    |
|                                   |     the assuree is granted a      |
|                                   |     recoverable payment, both     |
|                                   |     assurer/s and assuree are     |
|                                   |     required to submit a review   |
|                                   |     form every 3 months. See      |
|                                   |     Resources for AoS review      |
|                                   |     forms                         |
|                                   | -   If more than one assurer, an  |
|                                   |     interview must be completed   |
|                                   |     with each assurer             |
+-----------------------------------+-----------------------------------+
| 2                                 | Providing adequate support + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Ask the assurer/s if they are     |
|                                   | willing and able to provide       |
|                                   | adequate support to the assuree.  |
|                                   |                                   |
|                                   | Note: even if the assurer/s       |
|                                   | states that they are willing to   |
|                                   | support the assuree, they may not |
|                                   | be able to provide adequate       |
|                                   | support. Examine the              |
|                                   | circumstances thoroughly.         |
|                                   |                                   |
|                                   | Is the assurer/s willing and able |
|                                   | to provide adequate support to    |
|                                   | the assuree?                      |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Record outcome of the interview + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Record a DOC advising outcome of  |
|                                   | AoS interview and details of      |
|                                   | information provided to assurer/s |
|                                   | and assuree about implications of |
|                                   | granting an AoS recoverable       |
|                                   | payment to the assuree.           |
|                                   |                                   |
|                                   | To complete claim processing see  |
|                                   | Step 1 in Table 3                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Is the assuree refusing the       |
|                                   | assurer's support? + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, reject the claim.         |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 5                                 | Reasonable reason to refuse       |
|                                   | support + Read more ...           |
|                                   |                                   |
|                                   | Some instances where it would be  |
|                                   | reasonable for the assuree to     |
|                                   | refuse support are where:         |
|                                   |                                   |
|                                   | -   an assuree has not been       |
|                                   |     provided with a room that     |
|                                   |     they can access as required   |
|                                   |     on a continuing or indefinite |
|                                   |     basis                         |
|                                   | -   the assuree cannot rely on a  |
|                                   |     reasonable regular pattern of |
|                                   |     support, or                   |
|                                   | -   a social worker interview may |
|                                   |     be required to investigate    |
|                                   |     allegations of abuse          |
|                                   |                                   |
|                                   | If the Service Officer is unsure, |
|                                   | clearly DOC the record with       |
|                                   | information on the support the    |
|                                   | assurer/s indicate they provide   |
|                                   | and discuss with your SSO before  |
|                                   | making a decision. See Resources  |
|                                   | page for more scenarios.          |
|                                   |                                   |
|                                   | Is it reasonable for the assuree  |
|                                   | to refuse support?                |
|                                   |                                   |
|                                   | -   Yes, to complete claim        |
|                                   |     processing see Step 1 in      |
|                                   |     Table 3                       |
|                                   | -   No, reject the claim.         |
|                                   |     Procedure ends here.          |
