Maintenance codes assessed by social workers

Maintenance codes - full exemption

This table describes the codes that are recorded after a full exemption
has been granted by a social worker.

+-----------------------------------+-----------------------------------+
| Code                              | Description                       |
+-----------------------------------+-----------------------------------+
| IMP                               | Full exemption due to imposition. |
+-----------------------------------+-----------------------------------+
| IVF                               | In vitro non-biological parent.   |
|                                   | Used:                             |
|                                   |                                   |
|                                   | -   if the IVF is administered    |
|                                   |     through a medical clinic or   |
|                                   |     medical professional and the  |
|                                   |     customer does not have a      |
|                                   |     partner or the partner does   |
|                                   |     not agree to the procedure    |
|                                   |     (for example, sperm donor)    |
|                                   | -   if the IVF is                 |
|                                   |     self-administered, donor is   |
|                                   |     known to the woman and the    |
|                                   |     woman does not have a partner |
|                                   |     or the partner does not agree |
|                                   |     to the procedure and Child    |
|                                   |     Support decides that the      |
|                                   |     donor is not a parent for     |
|                                   |     child support purposes,       |
|                                   | -   if a couple use a surrogate   |
|                                   |     but the surrogacy agreement   |
|                                   |     is not recognised under       |
|                                   |     section 60HB of the Family    |
|                                   |     Law Act. These cases are      |
|                                   |     manually recorded from the    |
|                                   |     date exemption granted, or    |
|                                   | -   to record an IVF child        |
|                                   |     exemption, see Family Tax     |
|                                   |     Benefit (FTB) and child       |
|                                   |     support for children born as  |
|                                   |     a result of a surrogacy       |
|                                   |     arrangement or via artificial |
|                                   |     conception procedure          |
|                                   |     (including children of        |
|                                   |     same-sex couples)             |
|                                   |                                   |
|                                   | Note: since 1 July 2009, IVF      |
|                                   | exemptions should be reviewed for |
|                                   | children from same-sex            |
|                                   | relationships.                    |
+-----------------------------------+-----------------------------------+
| OCX                               | Other carer exemption. To record  |
|                                   | an OCX exemption, see How to      |
|                                   | group children on a customer's    |
|                                   | Family Tax Benefit record.        |
+-----------------------------------+-----------------------------------+
| PNK                               | Parentage unknown.                |
+-----------------------------------+-----------------------------------+
| VIO                               | Exemption due to violence. Sets   |
|                                   | up a future 12 month MFU review.  |
+-----------------------------------+-----------------------------------+
| XCC                               | Exemption due to exceptional      |
|                                   | cultural circumstances.           |
|                                   |                                   |
|                                   | Note: Aboriginal or Torres Strait |
|                                   | Islander Australians should be    |
|                                   | referred to a social worker for   |
|                                   | consideration about this          |
|                                   | exemption. A social worker may    |
|                                   | consult with an ISO to seek       |
|                                   | advice on cultural beliefs and    |
|                                   | other considerations in order to  |
|                                   | make an informed decision.        |
+-----------------------------------+-----------------------------------+

Maintenance codes - partial exemption

This table describes the codes that are recorded after a partial
exemption has been granted by a social worker.

+-----------------------------------+-----------------------------------+
| Code                              | Description                       |
+-----------------------------------+-----------------------------------+
| PMC                               | Not receiving full child support  |
|                                   | entitlement, partial exemption    |
|                                   | due to cultural reasons.          |
+-----------------------------------+-----------------------------------+
| PMI                               | Not receiving full child support  |
|                                   | entitlement, partial exemption    |
|                                   | due to imposition.                |
+-----------------------------------+-----------------------------------+
| PMV                               | Not receiving full child support  |
|                                   | entitlement, partial exemption    |
|                                   | due to violence. A Manual         |
|                                   | Follow-up (MFU) is created in 12  |
|                                   | months for social worker review.  |
+-----------------------------------+-----------------------------------+
