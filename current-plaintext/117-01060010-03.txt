For Compensation Recovery Teams (CRT) only.

Automatic finalisation of compensation claims

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Clearance processed on the        |
|                                   | Clearance Details screen + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | On the Clearance Details screen,  |
|                                   | check the Clearance Reason:       |
|                                   | field.                            |
|                                   |                                   |
|                                   | Is there an automatic             |
|                                   | finalisation code?                |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | The Clearance Details screen +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Compensation Management       |
|                                   | System (CMS) will finalise the    |
|                                   | claim when the CRT complete the   |
|                                   | processing of the manual          |
|                                   | compensation clearance. See       |
|                                   | Actioning a compensation          |
|                                   | clearance.                        |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Check to see if there is a nil    |
|                                   | charge + Read more ...            |
|                                   |                                   |
|                                   | Has the CRT completed a lump sum  |
|                                   | calculation and the result is a   |
|                                   | nil charge?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 4                                 | Nil charge + Read more ...        |
|                                   |                                   |
|                                   | When the lump sum calculation is  |
|                                   | complete, the Compensation        |
|                                   | Management System (CMS) will      |
|                                   | finalise the claim using the auto |
|                                   | finalisation code 'NOC'.          |
|                                   |                                   |
|                                   | Claims will not auto finalise if  |
|                                   | the settlement reason is 'IMP'.   |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Check if a debt has been raised   |
|                                   | and the charge fully repaid +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Has a debt been repaid in full?   |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 6                                 | Charge fully repaid + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Compensation Management       |
|                                   | System (CMS) will finalise the    |
|                                   | claim.                            |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
