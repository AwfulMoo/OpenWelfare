This page contains details of amounts payable for out-of-pocket
expenses, Medicare Safety Net letters for individuals and families,
links to Medicare Safety Net information on the Services Australia
website and the Medicare Safety Net user guide to Inquire on Medicare
Card Number.

Medicare Safety Net threshold letters for individuals and families

Table 2: This table lists the Safety Net threshold letters sent to
individuals and families.

+-----------------------+-----------------------+-----------------------+
| Customer              | Letter type           | Function              |
+-----------------------+-----------------------+-----------------------+
| Individuals           | Letter A              | Auto generated and    |
|                       |                       | sent once to          |
|                       |                       | individuals who are   |
|                       |                       | approaching a Safety  |
|                       |                       | Net threshold. This   |
|                       |                       | letter lists          |
|                       |                       | unverified services.  |
+-----------------------+-----------------------+-----------------------+
| Individuals           | Letter B              | Auto generated and    |
|                       |                       | sent to individuals   |
|                       |                       | who have reached the  |
|                       |                       | Safety Net threshold  |
|                       |                       | with verified         |
|                       |                       | services. This letter |
|                       |                       | lists unverified      |
|                       |                       | services over the     |
|                       |                       | threshold and is sent |
|                       |                       | each time a person    |
|                       |                       | claims until they     |
|                       |                       | verify the services.  |
+-----------------------+-----------------------+-----------------------+
| Families              | Letter F              | Auto generated and    |
|                       |                       | sent to the family    |
|                       |                       | contact when the      |
|                       |                       | registered family     |
|                       |                       | approaches a Safety   |
|                       |                       | Net threshold         |
|                       |                       | advising them to      |
|                       |                       | confirm the           |
|                       |                       | composition of the    |
|                       |                       | family in writing or  |
|                       |                       | over the phone.       |
+-----------------------+-----------------------+-----------------------+
| Families              | Letter U              | Customers can request |
|                       |                       | a listing of          |
|                       |                       | unverified services.  |
|                       |                       | This letter lists     |
|                       |                       | unverified services.  |
|                       |                       |                       |
|                       |                       | This letter can be    |
|                       |                       | requested by the      |
|                       |                       | Service Officer in    |
|                       |                       | CDMS Medicare Safety  |
|                       |                       | Net Programme.        |
|                       |                       |                       |
|                       |                       | Note: individual      |
|                       |                       | statements will be    |
|                       |                       | sent to persons over  |
|                       |                       | 14 years of age.      |
+-----------------------+-----------------------+-----------------------+
