Legislation provisions for different immunisation exemptions for the AIR

Table 5

+-----------------------------------+-----------------------------------+
| Type of exemption                 | Legislation provision             |
+-----------------------------------+-----------------------------------+
| Vaccination objection (formerly   | There is no longer provision for  |
| conscientious objection)          | vaccination objection (formerly   |
|                                   | conscientious objection) to       |
|                                   | immunisation in the Act. From 1   |
|                                   | January 2016, conscientious       |
|                                   | objections were renamed           |
|                                   | vaccination objections and are no |
|                                   | longer a valid immunisation       |
|                                   | exemption.                        |
+-----------------------------------+-----------------------------------+
| Medical contraindication          | An individual may have a          |
|                                   | contraindication recorded on the  |
|                                   | AIR if they are “… assessed by a  |
|                                   | general practitioner, a           |
|                                   | paediatrician, a public health    |
|                                   | physician, an infectious diseases |
|                                   | physician or a clinical           |
|                                   | immunologist as having a medical  |
|                                   | contraindication to a vaccine,    |
|                                   | and as a result should not        |
|                                   | receive a vaccination”.           |
+-----------------------------------+-----------------------------------+
| Natural immunity                  | An individual may have a natural  |
|                                   | immunity recorded on the AIR if   |
|                                   | they are “… assessed by a general |
|                                   | practitioner, a paediatrician, a  |
|                                   | public health physician, an       |
|                                   | infectious diseases physician or  |
|                                   | a clinical immunologist as not    |
|                                   | requiring a vaccination because   |
|                                   | the individual has contracted a   |
|                                   | disease or diseases, and as a     |
|                                   | result has developed a natural    |
|                                   | immunity”.                        |
