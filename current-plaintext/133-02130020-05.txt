Identity documents for PRODA accounts that can be verified online through DVS

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Online identity documents         |
|                                   |                                   |
|                                   | One of the following documents:   |
|                                   |                                   |
|                                   | -   Australian passport           |
|                                   | -   Australian birth certificate  |
|                                   | -   ImmiCard                      |
|                                   | -   Citizenship certificate       |
|                                   | -   Australian Visa (supported by |
|                                   |     a Foreign passport, which is  |
|                                   |     needed for verification)      |
|                                   | -   Certificate of registration   |
|                                   |     by descent                    |
|                                   |                                   |
|                                   | Two of the following documents:   |
|                                   |                                   |
|                                   | -   Medicare card                 |
|                                   | -   Australian driver's licence   |
|                                   | -   ImmiCard (can only be used    |
|                                   |     once)                         |
|                                   | -   Australian passport (can only |
|                                   |     be used once)                 |
|                                   |                                   |
|                                   | If a current name is different to |
|                                   | the name on any of the identity   |
|                                   | documents, the user will need to  |
|                                   | provide a change of name document |
|                                   | that can link the names:          |
|                                   |                                   |
|                                   | -   Marriage certificate          |
|                                   | -   Change of name certificate    |
