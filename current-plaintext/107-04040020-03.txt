For staff in International Services (CIS) only.

This page contains information to support International Services (CIS)
staff to finalise coding and reconcile recovery of Comparable Foreign
Payment (CFP) arrears debts by embargo from foreign pension arrears.

| 4                                 | Code the embargo + Read more ...  |
|                                   |                                   |
|                                   | Policy on exchange rates:         |
|                                   |                                   |
|                                   | Amounts paid by foreign           |
|                                   | authorities are converted to      |
|                                   | Australian dollars at a different |
|                                   | rate that the official exchange   |
|                                   | rate used in system calculations. |
|                                   | This means that the amount        |
|                                   | received does not match the       |
|                                   | calculated debt amount.           |
|                                   |                                   |
|                                   | Policy advice is that, where the  |
|                                   | delegate is satisfied that the    |
|                                   | variation between the debt amount |
|                                   | and the amount received is solely |
|                                   | due to exchange rates, any debt   |
|                                   | is considered to be fully repaid, |
|                                   | regardless of any shortfall or    |
|                                   | excess. Any bank fees deducted in |
|                                   | transactions are also ignored.    |
|                                   |                                   |
|                                   | In these cases, the coding of the |
|                                   | amount received must always be    |
|                                   | adjusted to match the debt amount |
|                                   | and recorded in the DOC.          |
|                                   |                                   |
|                                   | Coding:                           |
|                                   |                                   |
|                                   | For each record on the schedule,  |
|                                   | on the Foreign Claim Summary      |
|                                   | (FGS) screen, enter 'E' against   |
|                                   | the relevant foreign payment to   |
|                                   | navigate to the Foreign Indirect  |
|                                   | Embargo (FGIE) or Foreign Direct  |
|                                   | Embargo (FGDE) screen.            |
|                                   |                                   |
|                                   | For indirect embargo:             |
|                                   |                                   |
|                                   | On the FGIE screen, code:         |
|                                   |                                   |
|                                   | -   Amt Received $A:              |
|                                   |                                   |
|                                   | Notes:                            |
|                                   |                                   |
|                                   | -   Adjusted where relevant for   |
|                                   |     exchange rates, as above      |
|                                   | -   Code 0.00 for failed embargo  |
|                                   |                                   |
|                                   | -   Date Received:                |
|                                   | -   Receipt Num:                  |
|                                   |                                   |
|                                   | For direct embargo:               |
|                                   |                                   |
|                                   | On the FGDE screen, code:         |
|                                   |                                   |
|                                   | -   Date Grant Notice Received:   |
|                                   | -   Date Arrs Amt Received: as    |
|                                   |     advised by the Finance Team   |
|                                   | -   Amt Foreign Arrs Sent: in     |
|                                   |     source currency               |
|                                   | -   Exchange Rate: as advised by  |
|                                   |     the Finance Team              |
|                                   | -   Receipt Num:                  |
|                                   | -   Amt Received $A:              |
|                                   |                                   |
|                                   | Notes:                            |
|                                   |                                   |
|                                   | -   Adjusted where relevant for   |
|                                   |     exchange rates, as above,     |
|                                   |     including for recovery of     |
|                                   |     partner debts                 |
|                                   | -   Code 0.00 for failed embargo  |
|                                   |                                   |
|                                   | -   List Num:                     |
|                                   | -   Embargo Start Date:           |
|                                   | -   Embargo End Date:             |
|                                   | -   Arrears Start Date:           |
|                                   | -   Arrears End Date:             |
|                                   | -   Amt to be Recovered $A:       |
|                                   |                                   |
|                                   | Note: where partner's debt has    |
|                                   | also been recovered from the      |
|                                   | arrears, the amount coded is the  |
|                                   | total amount for both debts.      |
|                                   |                                   |
|                                   | Note: in all cases, unless        |
|                                   | otherwise recovered, any debt for |
|                                   | the partner must be raised at     |
