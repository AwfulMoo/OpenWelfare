Coding incoming NZ embargo schedules

Table 2: For Centrelink International Services (CIS) staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Generate the arrears debt + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Code the foreign pension arrears  |
|                                   | debt using the arrears amount and |
|                                   | arrears period from the NZ        |
|                                   | Service Decision and the          |
|                                   | applicable system exchange rate   |
|                                   | from the Foreign Pension Details  |
|                                   | screen. Do not use the exchange   |
|                                   | rate from the embargoes schedule. |
|                                   |                                   |
|                                   | When coding the arrears period to |
|                                   | ensure the debt is calculated for |
|                                   | the arrears period correctly, on  |
|                                   | the Residence Savings Screen      |
|                                   | (RSS) for the:                    |
|                                   |                                   |
|                                   | -   Long Term Country Code, key   |
|                                   |     AU                            |
|                                   | -   Date of Event, key the actual |
|                                   |     date the NZ Pension was       |
|                                   |     granted                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Compare arrears debt to NZ        |
|                                   | embargo amount + Read more ...    |
|                                   |                                   |
|                                   | Official exchange rates the       |
|                                   | agency uses to convert foreign    |
|                                   | currency recorded in the system   |
|                                   | are not the same as the exchange  |
|                                   | rate used to convert an embargo   |
|                                   | payment we receive from NZ on any |
|                                   | particular day. This means that   |
|                                   | the amount received from NZ will  |
|                                   | not reflect the amount of the     |
|                                   | arrears debt the agency           |
|                                   | calculates. Despite this          |
|                                   | difference, consider the          |
|                                   | customer’s debt to be fully       |
|                                   | repaid.                           |
|                                   |                                   |
|                                   | Under the Agreement with New      |
|                                   | Zealand, in Australia the rate of |
|                                   | NZ pension is ‘capped’ to the     |
|                                   | Australian Notional Rate and NZ   |
|                                   | pensions are always treated as a  |
|                                   | direct deduction, even for        |
|                                   | autonomous customers. Therefore   |
|                                   | the amount of the NZ embargoed    |
|                                   | arrears should be 100% of the     |
|                                   | Australian debt amount.           |
|                                   |                                   |
|                                   | There may still be cases where    |
|                                   | there are legitimate positive or  |
|                                   | negative adjustments remaining    |
|                                   | after the embargo. For example:   |
|                                   |                                   |
|                                   | -   the embargo amount received   |
|                                   |     from NZ is incorrect. For     |
|                                   |     example, NZ recovered their   |
|                                   |     own debt from the arrears,    |
|                                   |     there may still be some       |
|                                   |     negative adjustment remaining |
|                                   |     after the embargo is coded    |
|                                   | -   the customer is a member of a |
|                                   |     couple and their partner is   |
|                                   |     not in receipt of an income   |
|                                   |     support payment or the        |
|                                   |     customer has not authorised   |
|                                   |     recovery of their partner’s   |
|                                   |     debt. Half (50%) of the       |
|                                   |     amount received will be       |
|                                   |     embargoed and 50% will be     |
|                                   |     issued to the customer after  |
|                                   |     their debt is recovered       |
|                                   |                                   |
|                                   | A tolerance of 5% difference      |
|                                   | between the arrears debt and the  |
|                                   | embargo amount (AUD value) is     |
|                                   | accepted as being solely due to   |
|                                   | exchange rates.                   |
|                                   |                                   |
|                                   | The main processing file will     |
|                                   | automatically calculate if there  |
|                                   | is a (+ or -) 5% difference. The  |
|                                   | recovered column will show:       |
|                                   |                                   |
|                                   | -   Yes, if the difference is     |
|                                   |     within 5%                     |
|                                   | -   No, if the difference is over |
|                                   |     5%                            |
|                                   |                                   |
|                                   | Is the calculated arrears debt    |
|                                   | within 5% of the NZ embargo       |
|                                   | amount received?                  |
|                                   |                                   |
|                                   | -   Yes, consider the debt fully  |
|                                   |     recovered, go to Step 4       |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Check embargo amount + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | To ensure the arrears have been   |
|                                   | calculated correctly, check the   |
|                                   | arrears amount received from NZ   |
|                                   | matches the rate information      |
|                                   | provided on the original NZ       |
|                                   | Service Decision.                 |
|                                   |                                   |
|                                   | Multiply the NZ rates by the      |
|                                   | number of days in the arrears     |
|                                   | periods as provided in the NZ     |
|                                   | Service Decision and compare to   |
|                                   | the amount received in NZ         |
|                                   | dollars.                          |
|                                   |                                   |
|                                   | Check the agency’s arrears debt   |
|                                   | calculation is correct. See       |
|                                   | Calculating Centrelink            |
|                                   | entitlements and gathering        |
|                                   | information when investigating    |
|                                   | debts.                            |
|                                   |                                   |
|                                   | If the NZ arrears amount is       |
|                                   | established as:                   |
|                                   |                                   |
|                                   | correct, go to Step 4             |
|                                   |                                   |
|                                   | not correct, contact the NZ       |
|                                   | authorities to clarify. If the NZ |
|                                   | authorities:                      |
|                                   |                                   |
|                                   | -   will send the remaining       |
|                                   |     amounts at a later date, add  |
|                                   |     a comment (such as ‘Remaining |
|                                   |     arrears next schedule’) to    |
|                                   |     both the Main embargo         |
|                                   |     processing file and the       |
|                                   |     arrears debt activity. Put    |
|                                   |     the activity on hold for the  |
|                                   |     date the next schedule is     |
|                                   |     due. Procedure ends here      |
|                                   | -   have requested return of      |
|                                   |     amounts, contact the          |
|                                   |     International Finance Team to |
|                                   |     arrange return of funds. See  |
|                                   |     the Resources page table      |
|                                   | -   confirm some NZ embargo       |
|                                   |     amounts are remaining to      |
|                                   |     recover for their arrears     |
|                                   |     debt, go to Step 4            |
|                                   | -   confirm that no NZ amounts    |
|                                   |     remain for the debt, finalise |
|                                   |     the arrears debt as normal.   |
|                                   |     See Table 1 of Comparable     |
|                                   |     foreign payment lump sum      |
|                                   |     arrears debts                 |
|                                   | -   will not send any further     |
|                                   |     amounts, go to Step 4         |
+-----------------------------------+-----------------------------------+
| 4                                 | Partner debts + Read more ...     |
|                                   |                                   |
|                                   | If a member of a couple is        |
|                                   | granted a NZ pension and their    |
|                                   | partner receives an income        |
|                                   | support payment, the partner will |
|                                   | also have a debt. The Agreement   |
|                                   | does not give the authority to    |
|                                   | recover another persons’ debt     |
|                                   | from an embargoed amount. The     |
|                                   | customer must provide express     |
|                                   | authorisation before we can also  |
|                                   | recover the partner’s debt from   |
|                                   | their arrears.                    |
|                                   |                                   |
|                                   | An authorisation is sent to the   |
|                                   | customer in the NZ pension claim  |
|                                   | pack. If received, the            |
|                                   | authorisation is scanned to the   |
|                                   | customer record and also noted in |
|                                   | the relevant DOC when the NZ      |
|                                   | claim is sent overseas (OVS).     |
|                                   |                                   |
|                                   | Is there a partner debt?          |
|                                   |                                   |
|                                   | Yes, check the Document Tools and |
|                                   | the OVS DOC for the               |
|                                   | authorisation. If the customer:   |
|                                   |                                   |
|                                   | -   has not authorised the        |
|                                   |     recovery of the partner’s     |
|                                   |     debt from their arrears or    |
|                                   |     not returned the              |
|                                   |     authorisation form, go to     |
|                                   |     Step 5                        |
|                                   | -   has authorised the recovery   |
|                                   |     of their partner’s debt from  |
|                                   |     their arrears, go to Step 6   |
|                                   |                                   |
|                                   | No, go to Step 6                  |
+-----------------------------------+-----------------------------------+
| 5                                 | No partner authorisation, partner |
|                                   | debt to be raised + Read more ... |
|                                   |                                   |
|                                   | Raise the partner’s half of the   |
|                                   | debt as per normal debt raising   |
|                                   | procedures. See Table 1 of        |
|                                   | Comparable foreign payment lump   |
|                                   | sum arrears debts.                |
|                                   |                                   |
|                                   | The embargoed arrears can be used |
|                                   | to recover the customer’s debt    |
|                                   | and residual arrears released to  |
|                                   | the customer, go to Step 6.       |
+-----------------------------------+-----------------------------------+
| 6                                 | Raise the arrears debt and add    |
|                                   | embargo messaging + Read more ... |
|                                   |                                   |
|                                   | Before raising the debt:          |
|                                   |                                   |
|                                   | -   check the ADEX explanation    |
|                                   |     screen for debt information   |
|                                   |     by selecting ‘W’ on the Debt  |
|                                   |     List (OPDL) screen and        |
|                                   |     selecting the http link       |
|                                   | -   determine whether the debt is |
|                                   |     fully recovered according to  |
|                                   |     the policy above              |
|                                   | -   find the ‘Amount received in  |
|                                   |     debt period’                  |
|                                   |                                   |
|                                   | Run the Debt Action script.       |
